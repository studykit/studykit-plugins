# Arc2D.copy Method

Parent Object: [Arc2D](Arc2D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Arc2D.h>

## Description

Creates and returns an independent copy of this Arc2D object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"arc2D\_var" is a variable referencing an [Arc2D](Arc2D.htm) object.```` ``` returnValue = arc2D_var.copy() ``` ```` |

"arc2D\_var" is a variable referencing an [Arc2D](Arc2D.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| [Arc2D](Arc2D.htm) | Returns a new Arc2D object that is a copy of this Arc2D object. |
