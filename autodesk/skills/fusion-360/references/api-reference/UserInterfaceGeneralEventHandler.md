# UserInterfaceGeneralEventHandler Object

Derived from: [EventHandler](EventHandler.htm) Object

Defined in namespace "adsk::core" and the header file is <Core/UserInterface/UserInterfaceGeneralEventHandler.h>

## Description

The UserInterfaceGeneralEventHandler is a client implemented class that can be added as a handler to a UserInterfaceGeneralEvent event and which is triggered when the event is fired.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [notify](UserInterfaceGeneralEventHandler_notify.htm) | The function called by Fusion when the associated event is fired. |
