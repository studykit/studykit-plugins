# Canvas.flipVertical Method

Parent Object: [Canvas](Canvas.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Image/Canvas.h>

## Description

Flips the image along the vertical axis. This is a convenience method that flips the direction of the Y axis of the transform.

## Syntax

* [Python](#Python)
* [C++](#C++)

"canvas\_var" is a variable referencing a [Canvas](Canvas.htm) object.```` ``` returnValue = canvas_var.flipVertical() ``` ```` |

"canvas\_var" is a variable referencing a [Canvas](Canvas.htm) object. |

## Return Value

|  |  |

| Type | Description |
