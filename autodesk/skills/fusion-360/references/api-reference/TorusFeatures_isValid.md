# TorusFeatures.isValid Property

Parent Object: [TorusFeatures](TorusFeatures.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/TorusFeatures.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"torusFeatures\_var" is a variable referencing a TorusFeatures object. |

"torusFeatures\_var" is a variable referencing a TorusFeatures object. ```` ``` #include <Fusion/Features/TorusFeatures.h>  // Get the value of the property. boolean propertyValue = torusFeatures_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
