# Sphere.getData Method

Parent Object: [Sphere](Sphere.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Sphere.h>

## Description

Gets all of the data defining the sphere.

* [Python](#Python)
* [C++](#C++)
