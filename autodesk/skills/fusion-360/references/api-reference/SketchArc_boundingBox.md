# SketchArc.boundingBox Property

Parent Object: [SketchArc](SketchArc.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Sketch/SketchArc.h>

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

* [Python](#Python)
* [C++](#C++)

"sketchArc\_var" is a variable referencing a SketchArc object. |

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.htm).
