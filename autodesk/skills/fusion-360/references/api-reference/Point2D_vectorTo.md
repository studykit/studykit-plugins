# Point2D.vectorTo Method

Parent Object: [Point2D](Point2D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Point2D.h>

## Description

Returns a vector from this point to another point.

## Syntax

* [Python](#Python)
* [C++](#C++)

"point2D\_var" is a variable referencing a [Point2D](Point2D.htm) object.```` ``` returnValue = point2D_var.vectorTo(point) ``` ```` |

"point2D\_var" is a variable referencing a [Point2D](Point2D.htm) object. |

|  |  |
| --- | --- |
| Type | Description |

## Parameters

|  |  |  |
| --- | --- | --- |
