# RibFeature.deleteMe Method

Parent Object: [RibFeature](RibFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/RibFeature.h>

## Description

Deletes the feature. This works for both parametric and non-parametric features.

* [Python](#Python)
* [C++](#C++)
