# CAMExportFuture.isValid Property

Parent Object: [CAMExportFuture](CAMExportFuture.htm)
Defined in namespace "adsk::cam" and the header file is <Cam/CAM/CAMExportFuture.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"cAMExportFuture\_var" is a variable referencing a CAMExportFuture object. |

"cAMExportFuture\_var" is a variable referencing a CAMExportFuture object. ```` ``` #include <Cam/CAM/CAMExportFuture.h>  // Get the value of the property. boolean propertyValue = cAMExportFuture_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
