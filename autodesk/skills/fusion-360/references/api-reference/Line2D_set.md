# Line2D.set Method

Parent Object: [Line2D](Line2D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Line2D.h>

## Description

Sets all of the data defining the line segment.

## Syntax

* [Python](#Python)
* [C++](#C++)

"line2D\_var" is a variable referencing a [Line2D](Line2D.htm) object.```` ``` returnValue = line2D_var.set(startPoint, endPoint) ``` ```` |

"line2D\_var" is a variable referencing a [Line2D](Line2D.htm) object. |

|  |  |
| --- | --- |
| Type | Description |
