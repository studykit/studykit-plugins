# Viewport.refresh Method

Parent Object: [Viewport](Viewport.htm)
Defined in namespace "adsk::core" and the header file is <Core/Application/Viewport.h>

## Description

Forces the view to refresh. It is sometimes useful to force a refresh to be able to see edits that have been made using the API.

## Syntax

* [Python](#Python)
* [C++](#C++)

"viewport\_var" is a variable referencing a [Viewport](Viewport.htm) object.```` ``` returnValue = viewport_var.refresh() ``` ```` |

"viewport\_var" is a variable referencing a [Viewport](Viewport.htm) object. |

| boolean | Returns true if the operation was successful. |
