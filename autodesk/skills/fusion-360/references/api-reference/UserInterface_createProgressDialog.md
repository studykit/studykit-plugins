# UserInterface.createProgressDialog Method

Parent Object: [UserInterface](UserInterface.htm)
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/UserInterface.h>

## Description

Creates a new ProgressDialog object that you can use to display and control a progress dialog.

## Syntax

* [Python](#Python)
* [C++](#C++)

"userInterface\_var" is a variable referencing a [UserInterface](UserInterface.htm) object.```` ``` returnValue = userInterface_var.createProgressDialog() ``` ```` |

"userInterface\_var" is a variable referencing a [UserInterface](UserInterface.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| [ProgressDialog](ProgressDialog.htm) | Returns the created ProgressDialog object that you can use to define the contents of and display a progress dialog. |
