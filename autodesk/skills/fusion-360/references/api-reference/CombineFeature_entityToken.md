# CombineFeature.entityToken Property

Parent Object: [CombineFeature](CombineFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/CombineFeature.h>

## Description

Returns a token for the Feature object. This can be saved and used at a later time with the Design.findEntityByToken method to get back the same feature.

## Syntax

* [Python](#Python)
* [C++](#C++)

"combineFeature\_var" is a variable referencing a CombineFeature object.  ```` ``` # Get the value of the property. propertyValue = combineFeature_var.entityToken ``` ```` |

"combineFeature\_var" is a variable referencing a CombineFeature object. ```` ``` #include <Fusion/Features/CombineFeature.h>  // Get the value of the property. string propertyValue = combineFeature_var->entityToken(); ``` ```` |

## Property Value

This is a read only property whose value is a string.
