# Features.copyPasteBodies Property

Parent Object: [Features](Features.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/Features.h>

## Description

Returns the collection that provides access to the existing copy-paste features and supports the creation of new copy-paste features.

## Syntax

* [Python](#Python)
* [C++](#C++)

"features\_var" is a variable referencing a Features object. |

"features\_var" is a variable referencing a Features object. ```` ``` #include <Fusion/Features/Features.h>  // Get the value of the property. Ptr<CopyPasteBodies> propertyValue = features_var->copyPasteBodies(); ``` ```` |

## Property Value

This is a read only property whose value is a [CopyPasteBodies](CopyPasteBodies.htm).
