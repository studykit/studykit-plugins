# BRepEdge.objectType Property

Parent Object: [BRepEdge](BRepEdge.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/BRep/BRepEdge.h>

## Description

This property is supported by all objects in the API and returns a string that contains the full name (namespace::objecttype) describing the type of the object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"bRepEdge\_var" is a variable referencing a BRepEdge object.  ```` ``` # Get the value of the property. propertyValue = bRepEdge_var.objectType ``` ```` |

"bRepEdge\_var" is a variable referencing a BRepEdge object. ```` ``` #include <Fusion/BRep/BRepEdge.h>  // Get the value of the property. string propertyValue = bRepEdge_var->objectType(); ``` ```` |

## Property Value

This is a read only property whose value is a string.
