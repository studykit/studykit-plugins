# ConfigurationFeatureAspectStringCell Object

Derived from: [ConfigurationCell](ConfigurationCell.htm) Object

Defined in namespace "adsk::fusion" and the header file is <Fusion/Configurations/ConfigurationFeatureAspectStringCell.h>

## Description

Represents a cell from a column that defines a configuration feature aspect that is a String value.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [classType](ConfigurationFeatureAspectStringCell_classType.htm) | Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
