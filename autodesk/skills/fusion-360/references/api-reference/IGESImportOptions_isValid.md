# IGESImportOptions.isValid Property

Parent Object: [IGESImportOptions](IGESImportOptions.htm)
Defined in namespace "adsk::core" and the header file is <Core/Application/IGESImportOptions.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"iGESImportOptions\_var" is a variable referencing an IGESImportOptions object. |

"iGESImportOptions\_var" is a variable referencing an IGESImportOptions object. ```` ``` #include <Core/Application/IGESImportOptions.h>  // Get the value of the property. boolean propertyValue = iGESImportOptions_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.

Introduced in version September 2015
