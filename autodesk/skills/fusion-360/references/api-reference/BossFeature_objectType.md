# BossFeature.objectType Property

Parent Object: [BossFeature](BossFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Plastic/BossFeature.h>

## Description

This property is supported by all objects in the API and returns a string that contains the full name (namespace::objecttype) describing the type of the object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"bossFeature\_var" is a variable referencing a BossFeature object.  ```` ``` # Get the value of the property. propertyValue = bossFeature_var.objectType ``` ```` |

## Property Value

This is a read only property whose value is a string.
