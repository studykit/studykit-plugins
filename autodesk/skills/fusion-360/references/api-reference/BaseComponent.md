# BaseComponent Object

Derived from: [Base](Base.htm) Object

Defined in namespace "adsk::fusion" and the header file is <Fusion/Components/BaseComponent.h>

## Description

The BaseComponent object that defines all of the common design data and is the base class for the product specific components.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [allOccurrencesByComponent](BaseComponent_allOccurrencesByComponent.htm) | Returns all occurrences, at any level of the assembly, that reference the specified component. The returned list is read-only. |
| [classType](BaseComponent_classType.htm) | Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
| [findBRepUsingPoint](BaseComponent_findBRepUsingPoint.htm) | Finds all the entities of the specified type at the specified location. |
| [findBRepUsingRay](BaseComponent_findBRepUsingRay.htm) | Finds all the B-Rep entities that are intersected by the specified ray. This can return BRepFace, BrepEdge, and BRepVertex objects. |
| [occurrencesByComponent](BaseComponent_occurrencesByComponent.htm) | Returns all occurrences at the top-level of this component that reference the specified component. The returned list is read-only. |
