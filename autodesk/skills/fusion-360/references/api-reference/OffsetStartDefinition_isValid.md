# OffsetStartDefinition.isValid Property

Parent Object: [OffsetStartDefinition](OffsetStartDefinition.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/OffsetStartDefinition.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"offsetStartDefinition\_var" is a variable referencing an OffsetStartDefinition object. |

"offsetStartDefinition\_var" is a variable referencing an OffsetStartDefinition object. ```` ``` #include <Fusion/Features/OffsetStartDefinition.h>  // Get the value of the property. boolean propertyValue = offsetStartDefinition_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
