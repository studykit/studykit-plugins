# AccessibilityAnalysis.isValid Property

Parent Object: [AccessibilityAnalysis](AccessibilityAnalysis.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Fusion/AccessibilityAnalysis.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"accessibilityAnalysis\_var" is a variable referencing an AccessibilityAnalysis object. |

"accessibilityAnalysis\_var" is a variable referencing an AccessibilityAnalysis object. ```` ``` #include <Fusion/Fusion/AccessibilityAnalysis.h>  // Get the value of the property. boolean propertyValue = accessibilityAnalysis_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
