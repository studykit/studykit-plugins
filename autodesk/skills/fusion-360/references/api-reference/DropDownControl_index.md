# DropDownControl.index Property

Parent Object: [DropDownControl](DropDownControl.htm)
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/DropDownControl.h>

## Description

Gets the position of this control within the list of controls within the panel, toolbar, or drop-down control.
