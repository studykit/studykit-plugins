# CAM3MFExportMetadataOptions Object

Derived from: [Base](Base.htm) Object

Defined in namespace "adsk::cam" and the header file is <Cam/CAM/CAM3MFExportMetadataOptions.h>

## Description

Class providing read and write access to meta data of a 3MF file.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [classType](CAM3MFExportMetadataOptions_classType.htm) | Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
