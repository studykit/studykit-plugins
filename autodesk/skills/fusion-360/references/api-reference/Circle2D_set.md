# Circle2D.set Method

Parent Object: [Circle2D](Circle2D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Circle2D.h>

## Description

Sets all of the data defining the circle.

## Syntax

* [Python](#Python)
* [C++](#C++)

"circle2D\_var" is a variable referencing a [Circle2D](Circle2D.htm) object.```` ``` returnValue = circle2D_var.set(center, radius) ``` ```` |

"circle2D\_var" is a variable referencing a [Circle2D](Circle2D.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if redefining the circle is successful |

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| center | [Point2D](Point2D.htm) | A point that defines the center position of the circle. |
| radius | double | The radius of the circle. |
