# WebRequestEventHandler Object

Derived from: [EventHandler](EventHandler.htm) Object

Defined in namespace "adsk::core" and the header file is <Core/Application/WebRequestEventHandler.h>

## Description

The WebRequestEventHandler is a client implemented class that can be added as a handler to an WebRequestEvent.
