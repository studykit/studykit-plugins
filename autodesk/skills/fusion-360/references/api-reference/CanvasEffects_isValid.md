# CanvasEffects.isValid Property

Parent Object: [CanvasEffects](CanvasEffects.htm)
Defined in namespace "adsk::core" and the header file is <Core/Application/CanvasEffects.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"canvasEffects\_var" is a variable referencing a CanvasEffects object. |

"canvasEffects\_var" is a variable referencing a CanvasEffects object. ```` ``` #include <Core/Application/CanvasEffects.h>  // Get the value of the property. boolean propertyValue = canvasEffects_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
