# UserParameters.asArray Method

Parent Object: [UserParameters](UserParameters.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Fusion/UserParameters.h>

## Description

Returns the user parameters in the design as an array.

## Syntax

* [Python](#Python)
* [C++](#C++)

"userParameters\_var" is a variable referencing a [UserParameters](UserParameters.htm) object.```` ``` returnValue = userParameters_var.asArray() ``` ```` |

"userParameters\_var" is a variable referencing a [UserParameters](UserParameters.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| [UserParameter](UserParameter.htm)[] | Returns an array of the user parameters in the design. |
