# SweepFeature.timelineObject Property

Parent Object: [SweepFeature](SweepFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/SweepFeature.h>

## Description

Returns the timeline object associated with this feature.

## Syntax

This is a read only property whose value is a [TimelineObject](TimelineObject.htm).
