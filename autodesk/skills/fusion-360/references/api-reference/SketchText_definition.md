# SketchText.definition Property

Parent Object: [SketchText](SketchText.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Sketch/SketchText.h>

## Description

Gets the definition that is currently used to specify how the sketch text is defined.

## Syntax

* [Python](#Python)
* [C++](#C++)

"sketchText\_var" is a variable referencing a SketchText object. |

## Property Value

This is a read only property whose value is a [SketchTextDefinition](SketchTextDefinition.htm).
