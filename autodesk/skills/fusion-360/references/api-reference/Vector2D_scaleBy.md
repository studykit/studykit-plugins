# Vector2D.scaleBy Method

Parent Object: [Vector2D](Vector2D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Vector2D.h>

## Description

Scales the vector by specifying a scaling factor.

## Syntax

* [Python](#Python)
* [C++](#C++)

"vector2D\_var" is a variable referencing a [Vector2D](Vector2D.htm) object.```` ``` returnValue = vector2D_var.scaleBy(scale) ``` ```` |

"vector2D\_var" is a variable referencing a [Vector2D](Vector2D.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if successful. |

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| scale | double | The scale factor to multiple the vector by (i.e. 1.5). |
