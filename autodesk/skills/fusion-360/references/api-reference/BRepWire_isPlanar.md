# BRepWire.isPlanar Property

Parent Object: [BRepWire](BRepWire.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/BRep/BRepWire.h>

## Description

Indicates if this entities making up this wire body are planar and all lie on the same plane.

## Syntax

* [Python](#Python)
* [C++](#C++)

"bRepWire\_var" is a variable referencing a BRepWire object. |

"bRepWire\_var" is a variable referencing a BRepWire object. ```` ``` #include <Fusion/BRep/BRepWire.h>  // Get the value of the property. boolean propertyValue = bRepWire_var->isPlanar(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
