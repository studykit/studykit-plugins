# Vector3D.dotProduct Method

Parent Object: [Vector3D](Vector3D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Vector3D.h>

## Description

Returns the dot product between this vector and the specified vector.

## Syntax

* [Python](#Python)
* [C++](#C++)

"vector3D\_var" is a variable referencing a [Vector3D](Vector3D.htm) object.```` ``` returnValue = vector3D_var.dotProduct(vector) ``` ```` |

"vector3D\_var" is a variable referencing a [Vector3D](Vector3D.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| Double | Returns the dot product value. |

## Parameters

|  |  |  |
