# FlatPatternProduct.renderManager Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/SheetMetal/FlatPatternProduct.h>

## Description

Returns the RenderManager object associated with this design. Using the RenderManager you can access the same functionality that is available in the Render workspace.

## Syntax

* [Python](#Python)
* [C++](#C++)

"flatPatternProduct\_var" is a variable referencing a FlatPatternProduct object. |

"flatPatternProduct\_var" is a variable referencing a FlatPatternProduct object. ```` ``` #include <Fusion/SheetMetal/FlatPatternProduct.h>  // Get the value of the property. Ptr<RenderManager> propertyValue = flatPatternProduct_var->renderManager(); ``` ```` |

## Property Value

This is a read only property whose value is a [RenderManager](RenderManager.htm).
