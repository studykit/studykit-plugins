# FlatPatternProduct.activeComponent Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/SheetMetal/FlatPatternProduct.h>

## Description

Returns the component that is current being edited. This can return the root component or another component within the design.

## Syntax

* [Python](#Python)
* [C++](#C++)

"flatPatternProduct\_var" is a variable referencing a FlatPatternProduct object. |

"flatPatternProduct\_var" is a variable referencing a FlatPatternProduct object. ```` ``` #include <Fusion/SheetMetal/FlatPatternProduct.h>  // Get the value of the property. Ptr<Component> propertyValue = flatPatternProduct_var->activeComponent(); ``` ```` |

## Property Value

This is a read only property whose value is a [Component](Component.htm).
