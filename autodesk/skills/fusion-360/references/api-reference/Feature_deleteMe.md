# Feature.deleteMe Method

Parent Object: [Feature](Feature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/Feature.h>

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

* [Python](#Python)
* [C++](#C++)

"feature\_var" is a variable referencing a [Feature](Feature.htm) object.```` ``` returnValue = feature_var.deleteMe() ``` ```` |

"feature\_var" is a variable referencing a [Feature](Feature.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns a bool indicating if the delete was successful or not. |
