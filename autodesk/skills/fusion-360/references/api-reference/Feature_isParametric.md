# Feature.isParametric Property

Parent Object: [Feature](Feature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/Feature.h>

## Description

Indicates if this feature is parametric or not.

## Syntax

* [Python](#Python)
* [C++](#C++)

"feature\_var" is a variable referencing a Feature object. |

"feature\_var" is a variable referencing a Feature object. ```` ``` #include <Fusion/Features/Feature.h>  // Get the value of the property. boolean propertyValue = feature_var->isParametric(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
