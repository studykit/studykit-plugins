# CAMExportManager.isValid Property

Parent Object: [CAMExportManager](CAMExportManager.htm)
Defined in namespace "adsk::cam" and the header file is <Cam/CAM/CAMExportManager.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"cAMExportManager\_var" is a variable referencing a CAMExportManager object. |

"cAMExportManager\_var" is a variable referencing a CAMExportManager object. ```` ``` #include <Cam/CAM/CAMExportManager.h>  // Get the value of the property. boolean propertyValue = cAMExportManager_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
