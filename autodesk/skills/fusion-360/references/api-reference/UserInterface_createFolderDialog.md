# UserInterface.createFolderDialog Method

Parent Object: [UserInterface](UserInterface.htm)
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/UserInterface.h>

## Description

Creates a new FolderDialog object which provides the ability to show a standard folder selection dialog to the user.

## Syntax

* [Python](#Python)
* [C++](#C++)

"userInterface\_var" is a variable referencing a [UserInterface](UserInterface.htm) object.```` ``` returnValue = userInterface_var.createFolderDialog() ``` ```` |

"userInterface\_var" is a variable referencing a [UserInterface](UserInterface.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| [FolderDialog](FolderDialog.htm) | Returns the created FolderDialog object that you can use to define the contents of and display a standard folder dialog. |
