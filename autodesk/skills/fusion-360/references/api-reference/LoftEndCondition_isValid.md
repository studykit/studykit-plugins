# LoftEndCondition.isValid Property

Parent Object: [LoftEndCondition](LoftEndCondition.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/LoftEndCondition.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"loftEndCondition\_var" is a variable referencing a LoftEndCondition object. |

"loftEndCondition\_var" is a variable referencing a LoftEndCondition object. ```` ``` #include <Fusion/Features/LoftEndCondition.h>  // Get the value of the property. boolean propertyValue = loftEndCondition_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
