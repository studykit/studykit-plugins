# SketchPointsBossPositionDefinition Object

Derived from: [BossPositionDefinition](BossPositionDefinition.htm) Object

Defined in namespace "adsk::fusion" and the header file is <Fusion/Plastic/SketchPointsBossPositionDefinition.h>

## Description

Provides positioning information for a boss feature that is positioned by a sketch point(s).

## Methods

| --- | --- |
| Name | Description |
| [classType](SketchPointsBossPositionDefinition_classType.htm) | Static function that all classes support that returns the type of the class as a string. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
