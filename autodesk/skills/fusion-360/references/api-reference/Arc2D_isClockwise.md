# Arc2D.isClockwise Property

Parent Object: [Arc2D](Arc2D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Arc2D.h>

## Description

Gets if the sweep direction of the arc is clockwise or counterclockwise.

## Syntax

* [Python](#Python)
* [C++](#C++)

"arc2D\_var" is a variable referencing an Arc2D object. |

"arc2D\_var" is a variable referencing an Arc2D object. ```` ``` #include <Core/Geometry/Arc2D.h>  // Get the value of the property. boolean propertyValue = arc2D_var->isClockwise(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.

Introduced in version August 2014
