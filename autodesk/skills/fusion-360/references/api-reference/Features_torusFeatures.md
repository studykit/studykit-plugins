# Features.torusFeatures Property

Parent Object: [Features](Features.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/Features.h>

## Description

Returns the collection that provides access to the existing torus features.

## Syntax

* [Python](#Python)
* [C++](#C++)

"features\_var" is a variable referencing a Features object. |

"features\_var" is a variable referencing a Features object. ```` ``` #include <Fusion/Features/Features.h>  // Get the value of the property. Ptr<TorusFeatures> propertyValue = features_var->torusFeatures(); ``` ```` |

## Property Value

This is a read only property whose value is a [TorusFeatures](TorusFeatures.htm).

## Version

Introduced in version September 2015
