# BRepCoEdge.isOpposedToEdge Property

Parent Object: [BRepCoEdge](BRepCoEdge.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/BRep/BRepCoEdge.h>

## Description

Indicates if the orientation of this co-edge is in the same direction or opposed to its associated edge.

## Syntax

* [Python](#Python)
* [C++](#C++)

"bRepCoEdge\_var" is a variable referencing a BRepCoEdge object. |

## Property Value

This is a read only property whose value is a boolean.
