# ParallelConstraint.deleteMe Method

Parent Object: [ParallelConstraint](ParallelConstraint.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Sketch/ParallelConstraint.h>

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.
