# PathEntity.nativeObject Property

Parent Object: [PathEntity](PathEntity.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/PathEntity.h>

## Description

This property is not supported for a PathEntity object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"pathEntity\_var" is a variable referencing a PathEntity object. |

"pathEntity\_var" is a variable referencing a PathEntity object. ```` ``` #include <Fusion/Features/PathEntity.h>  // Get the value of the property. Ptr<PathEntity> propertyValue = pathEntity_var->nativeObject(); ``` ```` |

## Property Value

This is a read only property whose value is a [PathEntity](PathEntity.htm).
