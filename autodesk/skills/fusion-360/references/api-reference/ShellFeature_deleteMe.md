# ShellFeature.deleteMe Method

Parent Object: [ShellFeature](ShellFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/ShellFeature.h>

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

* [Python](#Python)
* [C++](#C++)

"shellFeature\_var" is a variable referencing a [ShellFeature](ShellFeature.htm) object.```` ``` returnValue = shellFeature_var.deleteMe() ``` ```` |

"shellFeature\_var" is a variable referencing a [ShellFeature](ShellFeature.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns a bool indicating if the delete was successful or not. |
