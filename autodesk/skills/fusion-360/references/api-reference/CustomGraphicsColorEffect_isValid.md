# CustomGraphicsColorEffect.isValid Property

Parent Object: [CustomGraphicsColorEffect](CustomGraphicsColorEffect.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Graphics/CustomGraphicsColorEffect.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
