# ExtendFeature.entityToken Property

Parent Object: [ExtendFeature](ExtendFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/ExtendFeature.h>

## Description

Returns a token for the Feature object. This can be saved and used at a later time with the Design.findEntityByToken method to get back the same feature.

## Syntax

* [Python](#Python)
* [C++](#C++)

"extendFeature\_var" is a variable referencing an ExtendFeature object.  ```` ``` # Get the value of the property. propertyValue = extendFeature_var.entityToken ``` ```` |

"extendFeature\_var" is a variable referencing an ExtendFeature object. ```` ``` #include <Fusion/Features/ExtendFeature.h>  // Get the value of the property. string propertyValue = extendFeature_var->entityToken(); ``` ```` |

## Property Value

This is a read only property whose value is a string.
