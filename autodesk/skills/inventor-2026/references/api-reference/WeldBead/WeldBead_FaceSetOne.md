# WeldBead.FaceSetOne Property

Parent Object: [WeldBead](../WeldBead/WeldBead.md)

## Description

Returns the first set of faces the weld bead is on.

## Syntax

WeldBead.**FaceSetOne**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2011
