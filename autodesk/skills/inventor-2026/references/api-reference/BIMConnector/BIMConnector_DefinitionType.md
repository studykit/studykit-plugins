# BIMConnector.DefinitionType Property

Parent Object: [BIMConnector](../BIMConnector/BIMConnector.md)

## Description

Read-only property that returns the type of definition associated with this connector. This property lets you determine what type of connector this object represents.

## Syntax

BIMConnector.**DefinitionType**() As [BIMConnectorDefinitionTypeEnum](../BIMConnectorDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [BIMConnectorDefinitionTypeEnum](../BIMConnectorDefinitionTypeEnum.md).

## Version

Introduced in version 2011
