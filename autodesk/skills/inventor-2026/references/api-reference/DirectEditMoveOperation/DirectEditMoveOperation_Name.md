# DirectEditMoveOperation.Name Property

Parent Object: [DirectEditMoveOperation](../DirectEditMoveOperation/DirectEditMoveOperation.md)

## Description

Gets the name of this direct edit operation within the scope of this Document.

## Syntax

DirectEditMoveOperation.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2015
