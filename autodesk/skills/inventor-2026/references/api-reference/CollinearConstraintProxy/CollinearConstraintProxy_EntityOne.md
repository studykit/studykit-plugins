# CollinearConstraintProxy.EntityOne Property

Parent Object: [CollinearConstraintProxy](../CollinearConstraintProxy/CollinearConstraintProxy.md)

## Description

Property that returns the first sketch entity being constrained.

## Syntax

CollinearConstraintProxy.**EntityOne**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
