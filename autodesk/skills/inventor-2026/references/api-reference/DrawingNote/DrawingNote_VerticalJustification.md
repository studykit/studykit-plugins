# DrawingNote.VerticalJustification Property

Parent Object: [DrawingNote](../DrawingNote/DrawingNote.md)

## Description

Gets/Sets vertical justification of the DrawingNote.

## Syntax

DrawingNote.**VerticalJustification**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 10
