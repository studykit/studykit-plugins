# GeneralDimension.Layer Property

Parent Object: [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Description

Gets and sets the layer applied to this dimension.

## Syntax

GeneralDimension.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
