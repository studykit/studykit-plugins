# GeneralDimension.Text Property

Parent Object: [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Description

Gets and sets the DimensionText setting.

## Syntax

GeneralDimension.**Text**() As [DimensionText](../DimensionText/DimensionText.md)

## Property Value

This is a read/write property whose value is a [DimensionText](../DimensionText/DimensionText.md).

## Version

Introduced in version 9
