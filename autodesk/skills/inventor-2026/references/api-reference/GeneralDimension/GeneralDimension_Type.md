# GeneralDimension.Type Property

Parent Object: [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

GeneralDimension.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
