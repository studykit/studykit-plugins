# GeneralDimension.SecondArrowheadInside Property

Parent Object: [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Description

Read-write property that gets and sets whether the second arrowhead is inside or outside.

## Syntax

GeneralDimension.**SecondArrowheadInside**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
