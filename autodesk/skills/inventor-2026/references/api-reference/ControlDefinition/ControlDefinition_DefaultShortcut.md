# ControlDefinition.DefaultShortcut Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Gets/Sets the default (internal) shortcut assigned to the command. Setting this property fails for built-in ControlDefinitions.

## Syntax

ControlDefinition.**DefaultShortcut**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
