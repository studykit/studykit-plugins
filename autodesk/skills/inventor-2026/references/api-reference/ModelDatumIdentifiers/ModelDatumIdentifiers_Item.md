# ModelDatumIdentifiers.Item Property

Parent Object: [ModelDatumIdentifiers](../ModelDatumIdentifiers/ModelDatumIdentifiers.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

ModelDatumIdentifiers.**Item**( ***Index*** As Variant ) As [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md)

## Property Value

This is a read only property whose value is a [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2018
