# ModelDatumIdentifiers.ExcludedDatumTextCharacters Property

Parent Object: [ModelDatumIdentifiers](../ModelDatumIdentifiers/ModelDatumIdentifiers.md)

## Description

Read-only property that returns an array of characters with comma as delimiter which will be excluded from the datum text list.

## Syntax

ModelDatumIdentifiers.**ExcludedDatumTextCharacters**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
