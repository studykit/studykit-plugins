# CosmeticBendFeatureProxy.AttributeSets Property

Parent Object: [CosmeticBendFeatureProxy](../CosmeticBendFeatureProxy/CosmeticBendFeatureProxy.md)

## Description

Property that returns the AttributeSets collection object associated with this object.

## Syntax

CosmeticBendFeatureProxy.**AttributeSets**() As [AttributeSets](../AttributeSets/AttributeSets.md)

## Property Value

This is a read only property whose value is an [AttributeSets](../AttributeSets/AttributeSets.md).

## Version

Introduced in version 2010
