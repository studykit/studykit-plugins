# CosmeticBendFeatureProxy.Parent Property

Parent Object: [CosmeticBendFeatureProxy](../CosmeticBendFeatureProxy/CosmeticBendFeatureProxy.md)

## Description

Property that returns the parent of the feature.

## Syntax

CosmeticBendFeatureProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
