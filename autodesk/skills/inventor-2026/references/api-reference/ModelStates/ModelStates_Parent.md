# ModelStates.Parent Property

Parent Object: [ModelStates](../ModelStates/ModelStates.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelStates.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2022
