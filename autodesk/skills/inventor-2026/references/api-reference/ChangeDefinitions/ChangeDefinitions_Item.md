# ChangeDefinitions.Item Property

Parent Object: [ChangeDefinitions](../ChangeDefinitions/ChangeDefinitions.md)

## Description

Returns the specified ChangeDefinition object from the collection.

## Syntax

ChangeDefinitions.**Item**( ***Index*** As Variant ) As [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md)

## Property Value

This is a read only property whose value is a [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the ChangeDefinition to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the ChangeDefinition name. If an out of range index or a name of a non-existent ChangeDefinition is provided, an error will occur. |

## Version

Introduced in version 9
