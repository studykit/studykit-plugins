# InteractionGraphics.PreviewClientGraphics Property

Parent Object: [InteractionGraphics](../InteractionGraphics/InteractionGraphics.md)

## Description

The top level object for graphics that are rendered with the main scene.

## Syntax

InteractionGraphics.**PreviewClientGraphics**() As [ClientGraphics](../ClientGraphics/ClientGraphics.md)

## Property Value

This is a read only property whose value is a [ClientGraphics](../ClientGraphics/ClientGraphics.md).
