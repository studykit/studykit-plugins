# InteractionGraphics.Type Property

Parent Object: [InteractionGraphics](../InteractionGraphics/InteractionGraphics.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

InteractionGraphics.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 8
