# SketchFillRegion.Parent Property

Parent Object: [SketchFillRegion](../SketchFillRegion/SketchFillRegion.md)

## Description

Property that gets the parent object from whom this object can logically be reached.

## Syntax

SketchFillRegion.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 10
