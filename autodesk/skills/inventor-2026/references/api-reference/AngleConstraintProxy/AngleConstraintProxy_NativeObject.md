# AngleConstraintProxy.NativeObject Property

Parent Object: [AngleConstraintProxy](../AngleConstraintProxy/AngleConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

AngleConstraintProxy.**NativeObject**() As [AngleConstraint](../AngleConstraint/AngleConstraint.md)

## Property Value

This is a read only property whose value is an [AngleConstraint](../AngleConstraint/AngleConstraint.md).

## Version

Introduced in version 11
