# AttributeManager.RevisionId Property

Parent Object: [AttributeManager](../AttributeManager/AttributeManager.md)

## Description

Gets the GUID that represents the last saved revision of this file. Works as a stamp of the contents of this file.

## Syntax

AttributeManager.**RevisionId**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 10
