# TableParameters Object

## Description

Provides access to the objects associated with the ParameterTable object the collection was obtained from.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../TableParameters/TableParameters_Count.md) | Property that returns the number of items in this collection. |
| [Item](../TableParameters/TableParameters_Item.md) | Returns the specified TableParameter object from the collection. This is the default property of the TableParameters collection object. |
| [Type](../TableParameters/TableParameters_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ParameterTable.TableParameters](../ParameterTable/ParameterTable_TableParameters.md)
