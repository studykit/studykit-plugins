# ClientFeatureElementProxy Object

Derived from: [ClientFeatureElement](../ClientFeatureElement/ClientFeatureElement.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../ClientFeatureElementProxy/ClientFeatureElementProxy_Delete.md) | Method that deletes this element of the client feature |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ClientFeatureElementProxy/ClientFeatureElementProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [BrowserVisible](../ClientFeatureElementProxy/ClientFeatureElementProxy_BrowserVisible.md) | Read-write property that gets and sets whether this element is visible in the browser under the ClientFeature node. |
| [ContainingOccurrence](../ClientFeatureElementProxy/ClientFeatureElementProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Element](../ClientFeatureElementProxy/ClientFeatureElementProxy_Element.md) |  |
| [HighlightWithFeature](../ClientFeatureElementProxy/ClientFeatureElementProxy_HighlightWithFeature.md) | Read-write property that gets and sets whether this element should be pre-highlight, highlighted or selected with the client feature. |
| [NativeObject](../ClientFeatureElementProxy/ClientFeatureElementProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../ClientFeatureElementProxy/ClientFeatureElementProxy_Parent.md) |  |
| [Type](../ClientFeatureElementProxy/ClientFeatureElementProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [UserEditable](../ClientFeatureElementProxy/ClientFeatureElementProxy_UserEditable.md) | Read-write property that gets and sets whether this element can be edited by the user. |
