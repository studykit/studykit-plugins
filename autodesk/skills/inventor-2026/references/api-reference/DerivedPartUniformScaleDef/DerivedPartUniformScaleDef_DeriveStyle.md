# DerivedPartUniformScaleDef.DeriveStyle Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Gets and sets how bodies are derived. Valid options are kDeriveAsSingleBodyWithSeams, kDeriveAsSingleBodyNoSeams, kDeriveAsMultipleBodies and kDeriveAsWorkSurface.

## Syntax

DerivedPartUniformScaleDef.**DeriveStyle**() As [DerivedComponentStyleEnum](../DerivedComponentStyleEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentStyleEnum](../DerivedComponentStyleEnum.md).

## Version

Introduced in version 2010
