# DerivedPartUniformScaleDef.SketchBlockDefinitions Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Property that gets the collection of sketch block definitions available in the selected part along with the inclusion option for each sketch block definition.

## Syntax

DerivedPartUniformScaleDef.**SketchBlockDefinitions**() As [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md)

## Property Value

This is a read only property whose value is a [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md).

## Version

Introduced in version 2010
