# DerivedPartUniformScaleDef.ReducedMemoryMode Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Read-write property that gets and sets the Reduced Memory Mode.

## Syntax

DerivedPartUniformScaleDef.**ReducedMemoryMode**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
