# MateiMateDefinition.ReferencedEntity Property

Parent Object: [MateiMateDefinition](../MateiMateDefinition/MateiMateDefinition.md)

## Description

Gets the object this entity is dependent on.

## Syntax

MateiMateDefinition.**ReferencedEntity**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version

Introduced in version 2009
