# UserCoordinateSystemDefinition.DefinitionType Property

Parent Object: [UserCoordinateSystemDefinition](../UserCoordinateSystemDefinition/UserCoordinateSystemDefinition.md)

## Description

Property that returns how the inputs for the coordinate system are defined. This can return either kTransformationDefinitionType or kThreePointsDefinitionType.

## Syntax

UserCoordinateSystemDefinition.**DefinitionType**() As [UCSDefinitionTypeEnum](../UCSDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [UCSDefinitionTypeEnum](../UCSDefinitionTypeEnum.md).

## Version

Introduced in version 2011
