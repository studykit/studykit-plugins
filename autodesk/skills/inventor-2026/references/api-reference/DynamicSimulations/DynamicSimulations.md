# DynamicSimulations Object

## Description

DynamicSimulations Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DynamicSimulations/DynamicSimulations_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DynamicSimulations/DynamicSimulations_Count.md) | Gets the number of items in this collection. |
| [Item](../DynamicSimulations/DynamicSimulations_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../DynamicSimulations/DynamicSimulations_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DSServer.GetDynamicSimulations](DSServer_GetDynamicSimulations.md), [SimulationManager.DynamicSimulations](../SimulationManager/SimulationManager_DynamicSimulations.md)
