# DrawingCurveSegments Object

## Description

The DrawingCurveSegments object provides access to the segments of a drawing curve.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingCurveSegments/DrawingCurveSegments_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DrawingCurveSegments/DrawingCurveSegments_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DrawingCurveSegments/DrawingCurveSegments_Item.md) | Method that returns the specified DrawingCurveSegment object from the collection. This is the default property of the DrawingCurveSegments collection object. |
| [Type](../DrawingCurveSegments/DrawingCurveSegments_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingCurve.Segments](../DrawingCurve/DrawingCurve_Segments.md)

## Version

Introduced in version 11
