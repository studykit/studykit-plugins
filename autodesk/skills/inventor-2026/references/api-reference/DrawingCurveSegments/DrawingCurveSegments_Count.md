# DrawingCurveSegments.Count Property

Parent Object: [DrawingCurveSegments](../DrawingCurveSegments/DrawingCurveSegments.md)

## Description

Property that returns the number of items in the collection.

## Syntax

DrawingCurveSegments.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 11
