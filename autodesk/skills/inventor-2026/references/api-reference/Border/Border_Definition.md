# Border.Definition Property

Parent Object: [Border](../Border/Border.md)

## Description

Property returning the border definition object that this border is an instance of.

## Syntax

Border.**Definition**() As [BorderDefinition](../BorderDefinition/BorderDefinition.md)

## Property Value

This is a read only property whose value is a [BorderDefinition](../BorderDefinition/BorderDefinition.md).

## Version

Introduced in version 5.3
