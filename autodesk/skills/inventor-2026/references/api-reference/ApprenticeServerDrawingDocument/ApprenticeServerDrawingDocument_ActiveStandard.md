# ApprenticeServerDrawingDocument.ActiveStandard Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Gets the currently active drafting standard that applies to all of the sheets.

## Syntax

ApprenticeServerDrawingDocument.**ActiveStandard**() As [DraftingStandard](../DraftingStandard/DraftingStandard.md)

## Property Value

This is a read only property whose value is a [DraftingStandard](../DraftingStandard/DraftingStandard.md).

## Version

Introduced in version 4
