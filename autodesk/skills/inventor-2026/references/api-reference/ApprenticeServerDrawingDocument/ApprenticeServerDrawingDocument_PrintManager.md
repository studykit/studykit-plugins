# ApprenticeServerDrawingDocument.PrintManager Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Property that returns the object for this document.

## Syntax

ApprenticeServerDrawingDocument.**PrintManager**() As [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Property Value

This is a read only property whose value is an [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md).

## Version

Introduced in version 9
