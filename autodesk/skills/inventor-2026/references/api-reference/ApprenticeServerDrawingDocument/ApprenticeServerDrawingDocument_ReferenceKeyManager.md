# ApprenticeServerDrawingDocument.ReferenceKeyManager Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Gets this Document's ReferenceKeyManager object.

## Syntax

ApprenticeServerDrawingDocument.**ReferenceKeyManager**() As [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md)

## Property Value

This is a read only property whose value is a [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md).

## Version

Introduced in version 2010
