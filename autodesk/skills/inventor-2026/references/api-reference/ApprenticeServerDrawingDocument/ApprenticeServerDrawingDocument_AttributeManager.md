# ApprenticeServerDrawingDocument.AttributeManager Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Returns the AttributeManager object.

## Syntax

ApprenticeServerDrawingDocument.**AttributeManager**() As [AttributeManager](../AttributeManager/AttributeManager.md)

## Property Value

This is a read only property whose value is an [AttributeManager](../AttributeManager/AttributeManager.md).

## Version

Introduced in version 5
