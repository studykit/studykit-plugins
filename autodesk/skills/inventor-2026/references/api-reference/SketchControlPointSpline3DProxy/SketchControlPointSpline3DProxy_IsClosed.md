# SketchControlPointSpline3DProxy.IsClosed Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Read-only property that returns whether the curve is closed.

## Syntax

SketchControlPointSpline3DProxy.**IsClosed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
