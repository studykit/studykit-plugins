# VertexProxy.NativeObject Property

Parent Object: [VertexProxy](../VertexProxy/VertexProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

VertexProxy.**NativeObject**() As [Vertex](../Vertex/Vertex.md)

## Property Value

This is a read only property whose value is a [Vertex](../Vertex/Vertex.md).

## Version

Introduced in version 4
