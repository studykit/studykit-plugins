# OrdinateDimensionSet.OriginMember Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets the origin for the ordinate dimension set.

## Syntax

OrdinateDimensionSet.**OriginMember**() As [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Property Value

This is a read/write property whose value is an [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md).

## Version

Introduced in version 2010
