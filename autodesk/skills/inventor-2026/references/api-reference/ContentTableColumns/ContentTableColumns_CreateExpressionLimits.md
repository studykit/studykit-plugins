# ContentTableColumns.CreateExpressionLimits Method

Parent Object: [ContentTableColumns](../ContentTableColumns/ContentTableColumns.md)

## Description

Method that creates an ExpressionLimits object. This object can be used to define the limits of a custom expression associated with a table column.

## Syntax

ContentTableColumns.**CreateExpressionLimits**() As [ExpressionLimits](../ExpressionLimits/ExpressionLimits.md)

## Version

Introduced in version 2010
