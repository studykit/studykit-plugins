# BIMComponentPropertySet.Type Property

Parent Object: [BIMComponentPropertySet](../BIMComponentPropertySet/BIMComponentPropertySet.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

BIMComponentPropertySet.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
