# ImportedRVTComponentDefinition Object

## Description

ImportedRVTComponentDefinition Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [FullFileName](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_FullFileName.md) | Read-only property that returns the full filename of the source file. |
| [Imported3DView](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_Imported3DView.md) | Read-write property that gets and sets the name of the 3D view of the Revit data to import. |
| [ImportedAssemblyOrganizationType](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_ImportedAssemblyOrganizationType.md) | Read-write property that gets and sets the imported assembly organization type. |
| [ImportedLanguage](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_ImportedLanguage.md) | Read-write property that gets and sets the imported Revit language type. |
| [Parent](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ReferenceModel](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_ReferenceModel.md) | Read-write property that gets and sets whether to import the component as linked reference. |
| [Type](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition_Type.md) | Gets the constant that indicates the type of this object. |
