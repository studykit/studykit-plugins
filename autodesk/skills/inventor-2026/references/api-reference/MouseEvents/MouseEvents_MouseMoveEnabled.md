# MouseEvents.MouseMoveEnabled Property

Parent Object: [MouseEvents](../MouseEvents/MouseEvents.md)

## Description

Gets/Sets the Boolean flag indicating whether a mouse move should fire an event (OnMouseMove). Default is FALSE.

## Syntax

MouseEvents.**MouseMoveEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
