# SketchPoints Object

## Description

The SketchPoints object provides access to all the objects in a sketch and provides methods to create additional sketch points. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchPoints/SketchPoints_Add.md) | Method that creates a new sketch point at the specified coordinate. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchPoints/SketchPoints_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchPoints/SketchPoints_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchPoints/SketchPoints_Item.md) | Returns the specified SketchPoint object from the collection. |
| [Type](../SketchPoints/SketchPoints_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
