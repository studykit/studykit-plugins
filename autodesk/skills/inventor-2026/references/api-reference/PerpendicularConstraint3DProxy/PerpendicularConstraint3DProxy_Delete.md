# PerpendicularConstraint3DProxy.Delete Method

Parent Object: [PerpendicularConstraint3DProxy](../PerpendicularConstraint3DProxy/PerpendicularConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

PerpendicularConstraint3DProxy.**Delete**()

## Version

Introduced in version 10
