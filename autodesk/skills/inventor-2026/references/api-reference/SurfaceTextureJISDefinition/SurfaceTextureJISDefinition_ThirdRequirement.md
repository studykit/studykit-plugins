# SurfaceTextureJISDefinition.ThirdRequirement Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets the third requirement value for the surface.

## Syntax

SurfaceTextureJISDefinition.**ThirdRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
