# SurfaceTextureJISDefinition.AllAroundSymbol Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets whether to add the all-around indicator to the symbol.

## Syntax

SurfaceTextureJISDefinition.**AllAroundSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
