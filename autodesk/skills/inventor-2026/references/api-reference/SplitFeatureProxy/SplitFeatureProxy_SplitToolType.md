# SplitFeatureProxy.SplitToolType Property

Parent Object: [SplitFeatureProxy](../SplitFeatureProxy/SplitFeatureProxy.md)

## Description

Property that returns the split tool type used to create the split feature.

## Syntax

SplitFeatureProxy.**SplitToolType**() As [SplitToolTypeEnum](../SplitToolTypeEnum.md)

## Property Value

This is a read only property whose value is a [SplitToolTypeEnum](../SplitToolTypeEnum.md).

## Version

Introduced in version 9
