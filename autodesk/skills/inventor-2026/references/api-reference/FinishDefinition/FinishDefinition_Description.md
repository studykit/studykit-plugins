# FinishDefinition.Description Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that gets and sets the description for the finish feature.

## Syntax

FinishDefinition.**Description**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2024
