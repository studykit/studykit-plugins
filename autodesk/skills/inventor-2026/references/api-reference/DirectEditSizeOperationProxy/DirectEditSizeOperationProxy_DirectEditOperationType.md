# DirectEditSizeOperationProxy.DirectEditOperationType Property

Parent Object: [DirectEditSizeOperationProxy](../DirectEditSizeOperationProxy/DirectEditSizeOperationProxy.md)

## Description

Gets the direct edit operation type.

## Syntax

DirectEditSizeOperationProxy.**DirectEditOperationType**() As [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md)

## Property Value

This is a read only property whose value is a [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md).

## Version

Introduced in version 2015
