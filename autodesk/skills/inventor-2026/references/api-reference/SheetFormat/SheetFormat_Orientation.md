# SheetFormat.Orientation Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that gets whether the sheet is oriented using landscape or portrait orientation.

## Syntax

SheetFormat.**Orientation**() As [PageOrientationTypeEnum](../PageOrientationTypeEnum.md)

## Property Value

This is a read only property whose value is a [PageOrientationTypeEnum](../PageOrientationTypeEnum.md).

## Version

Introduced in version 2009
