# SheetFormat.Size Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that gets the size of the sheet as defined in the format.

## Syntax

SheetFormat.**Size**() As [DrawingSheetSizeEnum](../DrawingSheetSizeEnum.md)

## Property Value

This is a read only property whose value is a [DrawingSheetSizeEnum](../DrawingSheetSizeEnum.md).

## Version

Introduced in version 2009
