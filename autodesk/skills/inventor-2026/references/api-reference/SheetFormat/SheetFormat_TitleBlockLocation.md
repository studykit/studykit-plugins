# SheetFormat.TitleBlockLocation Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that returns the position of the title block on the sheet as defined in the format.

## Syntax

SheetFormat.**TitleBlockLocation**() As [TitleBlockLocationEnum](../TitleBlockLocationEnum.md)

## Property Value

This is a read only property whose value is a [TitleBlockLocationEnum](../TitleBlockLocationEnum.md).

## Version

Introduced in version 2009
