# DrawingSketch.SketchEllipses Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Property that returns the SketchEllipses collection object.

## Syntax

DrawingSketch.**SketchEllipses**() As [SketchEllipses](../SketchEllipses/SketchEllipses.md)

## Property Value

This is a read only property whose value is a [SketchEllipses](../SketchEllipses/SketchEllipses.md).

## Version

Introduced in version 5.3
