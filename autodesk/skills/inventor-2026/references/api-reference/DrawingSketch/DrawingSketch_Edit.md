# DrawingSketch.Edit Method

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Method that causes the Sketch environment to be invoked with this sketch available for interactive edit.

## Syntax

DrawingSketch.**Edit**()
