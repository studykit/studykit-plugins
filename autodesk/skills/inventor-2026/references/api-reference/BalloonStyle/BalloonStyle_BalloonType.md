# BalloonStyle.BalloonType Property

Parent Object: [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Description

Property that gets and sets the shape of the balloon. Valid types are kCircularWithOneEntryBalloonType, kCircularWithTwoEntriesBalloonType, kHexagonBalloonType, kLinearBalloonType and kNoneBalloonType.

## Syntax

BalloonStyle.**BalloonType**() As [BalloonTypeEnum](../BalloonTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BalloonTypeEnum](../BalloonTypeEnum.md).

## Version

Introduced in version 2009
