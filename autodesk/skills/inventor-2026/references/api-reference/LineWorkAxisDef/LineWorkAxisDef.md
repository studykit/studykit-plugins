# LineWorkAxisDef Object

## Description

Object that allows you to get and set the information that specifies a work axis along a line.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LineWorkAxisDef/LineWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Line](../LineWorkAxisDef/LineWorkAxisDef_Line.md) | Property that returns the line of a line defined work axis. This object can be a linear Edge, WorkAxis, or SketchLine object. |
| [Parent](../LineWorkAxisDef/LineWorkAxisDef_Parent.md) | Property returning the parent WorkAxis object. |
| [Type](../LineWorkAxisDef/LineWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
