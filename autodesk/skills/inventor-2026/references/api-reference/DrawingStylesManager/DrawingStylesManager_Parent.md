# DrawingStylesManager.Parent Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

DrawingStylesManager.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 9
