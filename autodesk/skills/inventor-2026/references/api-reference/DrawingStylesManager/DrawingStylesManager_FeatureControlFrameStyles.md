# DrawingStylesManager.FeatureControlFrameStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the FeatureControlFrameStylesEnumerator object.

## Syntax

DrawingStylesManager.**FeatureControlFrameStyles**() As [FeatureControlFrameStylesEnumerator](../FeatureControlFrameStylesEnumerator/FeatureControlFrameStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [FeatureControlFrameStylesEnumerator](../FeatureControlFrameStylesEnumerator/FeatureControlFrameStylesEnumerator.md).

## Version

Introduced in version 2009
