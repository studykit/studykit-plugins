# DrawingStylesManager.ViewAnnotationStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Read-only property that returns the ViewAnnotationStylesEnumerator object.

## Syntax

DrawingStylesManager.**ViewAnnotationStyles**() As [ViewAnnotationStylesEnumerator](../ViewAnnotationStylesEnumerator/ViewAnnotationStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [ViewAnnotationStylesEnumerator](../ViewAnnotationStylesEnumerator/ViewAnnotationStylesEnumerator.md).

## Version

Introduced in version 2025.1
