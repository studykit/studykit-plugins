# DrawingStylesManager.PartsListStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the PartsListStylesEnumerator object. This object provides access to all the parts list styles.

## Syntax

DrawingStylesManager.**PartsListStyles**() As [PartsListStylesEnumerator](../PartsListStylesEnumerator/PartsListStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [PartsListStylesEnumerator](../PartsListStylesEnumerator/PartsListStylesEnumerator.md).

## Version

Introduced in version 2011
