# DrawingStylesManager Object

## Description

The DrawingStylesManager object provides utilities to manage the various styles in a drawing document.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [ReplaceStyles](../DrawingStylesManager/DrawingStylesManager_ReplaceStyles.md) | Method that replaces styles. |
