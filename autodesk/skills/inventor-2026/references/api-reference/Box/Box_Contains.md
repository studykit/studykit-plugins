# Box.Contains Method

Parent Object: [Box](../Box/Box.md)

## Description

Determines whether the specified point is contained within this Box.

## Syntax

Box.**Contains**( ***Point*** As [Point](../Point/Point.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input Point object that specifies the coordinate to check. |

## Version

Introduced in version 4
