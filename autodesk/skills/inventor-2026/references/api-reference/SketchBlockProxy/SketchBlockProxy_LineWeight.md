# SketchBlockProxy.LineWeight Property

Parent Object: [SketchBlockProxy](../SketchBlockProxy/SketchBlockProxy.md)

## Description

Gets and sets the line weight override for the sketch block.

## Syntax

SketchBlockProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
