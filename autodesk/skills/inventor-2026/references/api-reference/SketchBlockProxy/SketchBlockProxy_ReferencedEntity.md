# SketchBlockProxy.ReferencedEntity Property

Parent Object: [SketchBlockProxy](../SketchBlockProxy/SketchBlockProxy.md)

## Description

Property that returns the referenced sketch block, if there is one. Else, the property returns Nothing.

## Syntax

SketchBlockProxy.**ReferencedEntity**() As [SketchBlock](../SketchBlock/SketchBlock.md)

## Property Value

This is a read only property whose value is a [SketchBlock](../SketchBlock/SketchBlock.md).

## Version

Introduced in version 2010
