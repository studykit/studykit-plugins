# SketchBlockProxy.NativeObject Property

Parent Object: [SketchBlockProxy](../SketchBlockProxy/SketchBlockProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchBlockProxy.**NativeObject**() As [SketchBlock](../SketchBlock/SketchBlock.md)

## Property Value

This is a read only property whose value is a [SketchBlock](../SketchBlock/SketchBlock.md).

## Version

Introduced in version 2010
