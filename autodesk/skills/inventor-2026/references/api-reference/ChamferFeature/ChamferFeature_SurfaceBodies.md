# ChamferFeature.SurfaceBodies Property

Parent Object: [ChamferFeature](../ChamferFeature/ChamferFeature.md)

## Description

Property that returns the bodies that this feature has created or modified.

## Syntax

ChamferFeature.**SurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).

## Version

Introduced in version 2010
