# ChamferFeature.Appearance Property

Parent Object: [ChamferFeature](../ChamferFeature/ChamferFeature.md)

## Description

Gets and sets the current appearance of the feature.

## Syntax

ChamferFeature.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
