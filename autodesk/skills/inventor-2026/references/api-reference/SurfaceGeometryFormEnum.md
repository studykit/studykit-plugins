# SurfaceGeometryFormEnum Enumerator

## Description

Surface geometry form enum.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| SurfaceGeometryForm\_ClosedUVLoops | 1 | Surface geometry form is closed UV loops. |
| SurfaceGeometryForm\_Not\_ClosedUVLoops | 2 | Surface geometry form is not closed UV loops. |
| SurfaceGeometryForm\_Not\_NURBS | 8 | Surface geometry form is not NURBS (non-uniform rational B-spline). |
| SurfaceGeometryForm\_NURBS | 4 | Surface geometry form is NURBS (non-uniform rational B-spline). |
| SurfaceGeometryForm\_ProceduralToNURBS | 16 | Convert procedural surfaces to more accurate NURB approximations. This may result in splitting faces and edges as necessary. |

## Version

Introduced in version 4
