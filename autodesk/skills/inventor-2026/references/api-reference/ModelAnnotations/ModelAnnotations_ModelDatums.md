# ModelAnnotations.ModelDatums Property

Parent Object: [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md)

## Description

Returns the ModelDatums collection object.

## Syntax

ModelAnnotations.**ModelDatums**() As [ModelDatums](../ModelDatums/ModelDatums.md)

## Property Value

This is a read only property whose value is a [ModelDatums](../ModelDatums/ModelDatums.md).

## Version

Introduced in version 2023
