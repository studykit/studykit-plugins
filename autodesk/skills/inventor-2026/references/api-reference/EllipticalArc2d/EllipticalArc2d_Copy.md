# EllipticalArc2d.Copy Method

Parent Object: [EllipticalArc2d](../EllipticalArc2d/EllipticalArc2d.md)

## Description

Creates a copy of this EllipticalArc2d object. The result is entirely independent and can be edited without affecting the original EllipticalArc2d object.

## Syntax

EllipticalArc2d.**Copy**() As [EllipticalArc2d](../EllipticalArc2d/EllipticalArc2d.md)

## Version

Introduced in version 2013
