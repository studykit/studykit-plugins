# DrawingDocument.TitleBlockDefinitions Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the TitleBlockDefinitions collection object. This object provides access to all of the title block definition in the document and provides functionality to create new title block definitions.

## Syntax

DrawingDocument.**TitleBlockDefinitions**() As [TitleBlockDefinitions](../TitleBlockDefinitions/TitleBlockDefinitions.md)

## Property Value

This is a read only property whose value is a [TitleBlockDefinitions](../TitleBlockDefinitions/TitleBlockDefinitions.md).
