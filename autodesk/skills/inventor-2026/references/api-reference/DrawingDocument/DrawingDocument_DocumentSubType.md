# DrawingDocument.DocumentSubType Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the subtype of the document.

## Syntax

DrawingDocument.**DocumentSubType**() As [DocumentSubType](DocumentSubType.md)

## Property Value

This is a read only property whose value is a [DocumentSubType](DocumentSubType.md).

## Version

Introduced in version 6
