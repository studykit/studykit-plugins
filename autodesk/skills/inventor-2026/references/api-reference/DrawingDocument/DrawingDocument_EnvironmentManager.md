# DrawingDocument.EnvironmentManager Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the EnvironmentManager object.

## Syntax

DrawingDocument.**EnvironmentManager**() As [EnvironmentManager](../EnvironmentManager/EnvironmentManager.md)

## Property Value

This is a read only property whose value is an [EnvironmentManager](../EnvironmentManager/EnvironmentManager.md).

## Version

Introduced in version 10
