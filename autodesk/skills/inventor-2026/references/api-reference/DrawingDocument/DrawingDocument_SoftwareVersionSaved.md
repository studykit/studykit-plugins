# DrawingDocument.SoftwareVersionSaved Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Gets the object that encapsulates the version of the software with which this document was last saved.

## Syntax

DrawingDocument.**SoftwareVersionSaved**() As [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Property Value

This is a read only property whose value is a [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md).

## Version

Introduced in version 4
