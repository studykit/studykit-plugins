# SketchSplineHandleProxy.Spline Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Property that returns the spline associated with the handle.

## Syntax

SketchSplineHandleProxy.**Spline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Property Value

This is a read only property whose value is a [SketchSpline](../SketchSpline/SketchSpline.md).

## Version

Introduced in version 2008
