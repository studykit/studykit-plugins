# SketchSplineHandleProxy.Tangent Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Gets and sets the tangent vector at the fit point associated with this handle.

## Syntax

SketchSplineHandleProxy.**Tangent**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read/write property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).

## Version

Introduced in version 2008
