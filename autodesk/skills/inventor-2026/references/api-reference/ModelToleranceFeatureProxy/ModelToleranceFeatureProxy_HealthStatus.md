# ModelToleranceFeatureProxy.HealthStatus Property

Parent Object: [ModelToleranceFeatureProxy](../ModelToleranceFeatureProxy/ModelToleranceFeatureProxy.md)

## Description

Returns the current health status of the tolerance feature.

## Syntax

ModelToleranceFeatureProxy.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2018
