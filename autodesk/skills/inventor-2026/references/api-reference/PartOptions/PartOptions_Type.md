# PartOptions.Type Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

PartOptions.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
