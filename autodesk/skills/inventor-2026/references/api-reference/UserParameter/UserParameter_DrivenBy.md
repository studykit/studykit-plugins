# UserParameter.DrivenBy Property

Parent Object: [UserParameter](../UserParameter/UserParameter.md)

## Description

Method that returns the collection objects that the parameter is dependent on.

## Syntax

UserParameter.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
