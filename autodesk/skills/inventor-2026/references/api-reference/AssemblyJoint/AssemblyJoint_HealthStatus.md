# AssemblyJoint.HealthStatus Property

Parent Object: [AssemblyJoint](../AssemblyJoint/AssemblyJoint.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

AssemblyJoint.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2014
