# PresentationVertex.Edges Property

Parent Object: [PresentationVertex](../PresentationVertex/PresentationVertex.md)

## Description

Read-only property that returns the PresentationEdge objects that are adjacent to this vertex.

## Syntax

PresentationVertex.**Edges**() As [PresentationEdgesEnumerator](../PresentationEdgesEnumerator/PresentationEdgesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationEdgesEnumerator](../PresentationEdgesEnumerator/PresentationEdgesEnumerator.md).

## Version

Introduced in version 2018
