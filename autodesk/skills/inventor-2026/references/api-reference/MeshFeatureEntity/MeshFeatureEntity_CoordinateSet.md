# MeshFeatureEntity.CoordinateSet Property

Parent Object: [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Description

Read-only property that returns the coordinate set of the mesh feature entity.

## Syntax

MeshFeatureEntity.**CoordinateSet**() As [GraphicsCoordinateSet](../GraphicsCoordinateSet/GraphicsCoordinateSet.md)

## Property Value

This is a read only property whose value is a [GraphicsCoordinateSet](../GraphicsCoordinateSet/GraphicsCoordinateSet.md).

## Version

Introduced in version 2017
