# MeshFeatureEntity.NormalBinding Property

Parent Object: [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Description

Read-only property that returns the normal binding of the mesh feature entity.

## Syntax

MeshFeatureEntity.**NormalBinding**() As [NormalBindingEnum](../NormalBindingEnum.md)

## Property Value

This is a read only property whose value is a [NormalBindingEnum](../NormalBindingEnum.md).

## Version

Introduced in version 2017
