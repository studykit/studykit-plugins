# PointCloudPlane.Parent Property

Parent Object: [PointCloudPlane](../PointCloudPlane/PointCloudPlane.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PointCloudPlane.**Parent**() As [PointCloud](../PointCloud/PointCloud.md)

## Property Value

This is a read only property whose value is a [PointCloud](../PointCloud/PointCloud.md).

## Version

Introduced in version 2015
