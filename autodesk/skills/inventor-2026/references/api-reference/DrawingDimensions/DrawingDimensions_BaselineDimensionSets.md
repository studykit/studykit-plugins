# DrawingDimensions.BaselineDimensionSets Property

Parent Object: [DrawingDimensions](../DrawingDimensions/DrawingDimensions.md)

## Description

Property that returns the BaselineDimensionSets collection object.

## Syntax

DrawingDimensions.**BaselineDimensionSets**() As [BaselineDimensionSets](../BaselineDimensionSets/BaselineDimensionSets.md)

## Property Value

This is a read only property whose value is a [BaselineDimensionSets](../BaselineDimensionSets/BaselineDimensionSets.md).
