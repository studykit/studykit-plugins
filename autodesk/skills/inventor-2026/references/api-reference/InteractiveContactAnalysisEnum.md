# InteractiveContactAnalysisEnum Enumerator

## Description

Enum indicating the type of components which are used for interactive contact analysis.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllComponentsInteractiveContact | 64770 | All components are used as input. |
| kContactSetOnlyInteractiveContact | 64769 | Components in the contact set are used as input. |
| kSolverOffInteractiveContact | 64771 | None components are used as input. |

## Version

Introduced in version 2008
