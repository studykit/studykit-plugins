# FromToExtent.FromFace Property

Parent Object: [FromToExtent](../FromToExtent/FromToExtent.md)

## Description

Property that gets and sets the face that defines the 'from' extents of the feature.

## Syntax

FromToExtent.**FromFace**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 5
