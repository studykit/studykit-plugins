# LineAndPlaneWorkAxisDef.Parent Property

Parent Object: [LineAndPlaneWorkAxisDef](../LineAndPlaneWorkAxisDef/LineAndPlaneWorkAxisDef.md)

## Description

## Syntax

LineAndPlaneWorkAxisDef.**Parent**() As [WorkAxis](../WorkAxis/WorkAxis.md)

## Property Value

This is a read only property whose value is a [WorkAxis](../WorkAxis/WorkAxis.md).
