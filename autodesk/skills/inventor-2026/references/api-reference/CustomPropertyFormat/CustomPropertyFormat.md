# CustomPropertyFormat Object

## Description

The CustomPropertyFormat provides control over how the parameter value is formatted when it is exposed as a custom iProperty.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CustomPropertyFormat/CustomPropertyFormat_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../CustomPropertyFormat/CustomPropertyFormat_Parent.md) | Property that returns the parent Parameter object. |
| [Precision](../CustomPropertyFormat/CustomPropertyFormat_Precision.md) | Gets/Sets the precision to use when formatting the parameter value to be displayed as an iProperty. |
| [PropertyType](../CustomPropertyFormat/CustomPropertyFormat_PropertyType.md) | Gets/Sets whether the custom iProperty created for this parameter is a text (String) or number (Double) parameter. |
| [ShowLeadingZeros](../CustomPropertyFormat/CustomPropertyFormat_ShowLeadingZeros.md) | Gets/Sets if leading zeros should be shown in the formatted iProperty value. |
| [ShowTrailingZeros](../CustomPropertyFormat/CustomPropertyFormat_ShowTrailingZeros.md) | Gets/Sets if trailing zeros should be shown in the formatted iProperty value. |
| [ShowUnitsString](../CustomPropertyFormat/CustomPropertyFormat_ShowUnitsString.md) | Gets/Sets if the unit string should be shown in the formatted iProperty value. |
| [Type](../CustomPropertyFormat/CustomPropertyFormat_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Units](../CustomPropertyFormat/CustomPropertyFormat_Units.md) | Gets/Sets the units to use when formatting the parameter value to be displayed as an iProperty. |
