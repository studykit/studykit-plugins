# ConcentricConstraint3DProxy Object

Derived from: [GeometricConstraint3D](../GeometricConstraint3D/GeometricConstraint3D.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_Delete.md) | Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted. |
| [GetReferenceKey](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_Deletable.md) | Indicates whether this object can be deleted. |
| [NativeObject](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Type](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2010
