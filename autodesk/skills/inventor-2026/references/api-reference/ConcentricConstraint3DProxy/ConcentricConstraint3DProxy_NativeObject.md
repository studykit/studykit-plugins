# ConcentricConstraint3DProxy.NativeObject Property

Parent Object: [ConcentricConstraint3DProxy](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ConcentricConstraint3DProxy.**NativeObject**() As [ConcentricConstraint3D](../ConcentricConstraint3D/ConcentricConstraint3D.md)

## Property Value

This is a read only property whose value is a [ConcentricConstraint3D](../ConcentricConstraint3D/ConcentricConstraint3D.md).

## Version

Introduced in version 2010
