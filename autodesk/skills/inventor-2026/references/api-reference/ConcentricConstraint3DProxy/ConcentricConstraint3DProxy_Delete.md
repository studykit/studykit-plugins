# ConcentricConstraint3DProxy.Delete Method

Parent Object: [ConcentricConstraint3DProxy](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ConcentricConstraint3DProxy.**Delete**()

## Version

Introduced in version 2010
