# InventorVBAComponents.Parent Property

Parent Object: [InventorVBAComponents](../InventorVBAComponents/InventorVBAComponents.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

InventorVBAComponents.**Parent**() As [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Property Value

This is a read only property whose value is an [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md).

## Version

Introduced in version 6
