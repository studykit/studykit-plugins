# SketchFixedSplineProxy.Reference Property

Parent Object: [SketchFixedSplineProxy](../SketchFixedSplineProxy/SketchFixedSplineProxy.md)

## Description

Gets and sets whether this entity is a reference entity or not. It is only valid to set this property to False.

## Syntax

SketchFixedSplineProxy.**Reference**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
