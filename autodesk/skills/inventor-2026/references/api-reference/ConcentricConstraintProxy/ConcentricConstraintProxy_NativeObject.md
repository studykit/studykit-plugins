# ConcentricConstraintProxy.NativeObject Property

Parent Object: [ConcentricConstraintProxy](../ConcentricConstraintProxy/ConcentricConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ConcentricConstraintProxy.**NativeObject**() As [ConcentricConstraint](../ConcentricConstraint/ConcentricConstraint.md)

## Property Value

This is a read only property whose value is a [ConcentricConstraint](../ConcentricConstraint/ConcentricConstraint.md).

## Version

Introduced in version 6
