# CircularPatternFeatureProxy.Adaptive Property

Parent Object: [CircularPatternFeatureProxy](../CircularPatternFeatureProxy/CircularPatternFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

CircularPatternFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
