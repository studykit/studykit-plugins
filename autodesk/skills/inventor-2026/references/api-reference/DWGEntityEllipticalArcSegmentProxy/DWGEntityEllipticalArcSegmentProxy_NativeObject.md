# DWGEntityEllipticalArcSegmentProxy.NativeObject Property

Parent Object: [DWGEntityEllipticalArcSegmentProxy](../DWGEntityEllipticalArcSegmentProxy/DWGEntityEllipticalArcSegmentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGEntityEllipticalArcSegmentProxy.**NativeObject**() As [DWGEntityEllipticalArcSegment](../DWGEntityEllipticalArcSegment/DWGEntityEllipticalArcSegment.md)

## Property Value

This is a read only property whose value is a [DWGEntityEllipticalArcSegment](../DWGEntityEllipticalArcSegment/DWGEntityEllipticalArcSegment.md).

## Version

Introduced in version 2018
