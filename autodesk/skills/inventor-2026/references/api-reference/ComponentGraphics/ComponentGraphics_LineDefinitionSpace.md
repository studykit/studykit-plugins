# ComponentGraphics.LineDefinitionSpace Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Read-write property that gets and sets the line definition space for this primitive. This affects how the line weight and line scale are applied to the primitive. This defaults to kScreenSpace.

## Syntax

ComponentGraphics.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2011
