# ComponentGraphics.Parent Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

ComponentGraphics.**Parent**() As [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Property Value

This is a read only property whose value is a [GraphicsNode](../GraphicsNode/GraphicsNode.md).

## Version

Introduced in version 2011
