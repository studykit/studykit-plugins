# SheetMetalComponentDefinition.PointClouds Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Gets the PointClouds collection object that encapsulates all of the point clouds defined in this Component Definition.

## Syntax

SheetMetalComponentDefinition.**PointClouds**() As [PointClouds](../PointClouds/PointClouds.md)

## Property Value

This is a read only property whose value is a [PointClouds](../PointClouds/PointClouds.md).

## Version

Introduced in version 2016
