# SheetMetalComponentDefinition.ReferenceComponents Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the ReferenceComponents collection object.

## Syntax

SheetMetalComponentDefinition.**ReferenceComponents**() As [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Property Value

This is a read only property whose value is a [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md).

## Version

Introduced in version 5.3
