# SheetMetalComponentDefinition.SheetMetalStyles Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the SheetMetalStyles collection object.

## Syntax

SheetMetalComponentDefinition.**SheetMetalStyles**() As [SheetMetalStyles](../SheetMetalStyles/SheetMetalStyles.md)

## Property Value

This is a read only property whose value is a [SheetMetalStyles](../SheetMetalStyles/SheetMetalStyles.md).
