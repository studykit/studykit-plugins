# SheetMetalComponentDefinition.PartEvents Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the PartEvents source object.

## Syntax

SheetMetalComponentDefinition.**PartEvents**() As [PartEvents](../PartEvents/PartEvents.md)

## Property Value

This is a read only property whose value is a [PartEvents](../PartEvents/PartEvents.md).

## Version

Introduced in version 5.3
