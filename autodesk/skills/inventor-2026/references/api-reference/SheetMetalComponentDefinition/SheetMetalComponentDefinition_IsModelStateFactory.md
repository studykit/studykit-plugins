# SheetMetalComponentDefinition.IsModelStateFactory Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-only property that returns whether the document is a model state factory document or not.

## Syntax

SheetMetalComponentDefinition.**IsModelStateFactory**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2022
