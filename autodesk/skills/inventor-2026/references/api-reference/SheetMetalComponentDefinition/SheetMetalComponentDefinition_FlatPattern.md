# SheetMetalComponentDefinition.FlatPattern Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the FlatPattern object. If the sheet metal part hasn't been unfolded, this property returns Nothing.

## Syntax

SheetMetalComponentDefinition.**FlatPattern**() As [FlatPattern](../FlatPattern/FlatPattern.md)

## Property Value

This is a read only property whose value is a [FlatPattern](../FlatPattern/FlatPattern.md).

## Version

Introduced in version 6
