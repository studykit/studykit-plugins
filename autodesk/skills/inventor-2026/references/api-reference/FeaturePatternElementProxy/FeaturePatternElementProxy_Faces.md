# FeaturePatternElementProxy.Faces Property

Parent Object: [FeaturePatternElementProxy](../FeaturePatternElementProxy/FeaturePatternElementProxy.md)

## Description

Property that returns a collection object generated as a result of the feature.

## Syntax

FeaturePatternElementProxy.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 9
