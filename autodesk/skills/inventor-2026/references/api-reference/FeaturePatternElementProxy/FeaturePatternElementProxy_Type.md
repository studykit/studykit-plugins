# FeaturePatternElementProxy.Type Property

Parent Object: [FeaturePatternElementProxy](../FeaturePatternElementProxy/FeaturePatternElementProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FeaturePatternElementProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
