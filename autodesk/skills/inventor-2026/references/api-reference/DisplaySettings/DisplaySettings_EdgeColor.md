# DisplaySettings.EdgeColor Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets the color for the edge display (including silhouettes) for active components.

## Syntax

DisplaySettings.**EdgeColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2011
