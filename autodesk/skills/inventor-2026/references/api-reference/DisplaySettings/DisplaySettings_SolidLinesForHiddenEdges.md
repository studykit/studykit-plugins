# DisplaySettings.SolidLinesForHiddenEdges Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets whether the hidden model edges should be displayed as solid lines.

## Syntax

DisplaySettings.**SolidLinesForHiddenEdges**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
