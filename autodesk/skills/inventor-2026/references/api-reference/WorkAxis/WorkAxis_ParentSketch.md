# WorkAxis.ParentSketch Property

Parent Object: [WorkAxis](../WorkAxis/WorkAxis.md)

## Description

Property that returns the parent ComponentDefinition object.

## Syntax

WorkAxis.**ParentSketch**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2008
