# RectangularPatternFeature.Definition Property

Parent Object: [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md)

## Description

Read-write property that gets and sets the RectangularPatternFeatureDefinition object associated with this feature.

## Syntax

RectangularPatternFeature.**Definition**() As [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Property Value

This is a read/write property whose value is a [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md).

## Version

Introduced in version 2017
