# RectangularPatternFeature.PatternElements Property

Parent Object: [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md)

## Description

Property that returns the collection that contains the elements created by this pattern.

## Syntax

RectangularPatternFeature.**PatternElements**() As [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md)

## Property Value

This is a read only property whose value is a [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md).

## Version

Introduced in version 5
