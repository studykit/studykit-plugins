# AttributeSet.Copy Property

Parent Object: [AttributeSet](../AttributeSet/AttributeSet.md)

## Description

Gets the Boolean flag that indicates whether this set was created as a result of a operation that copied the owner Inventor Object.

## Syntax

AttributeSet.**Copy**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
