# DoubleHemDefinition Object

## Description

The DoubleHemDefinition defines all of the inputs that are unique to a double style of hem feature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Gap](../DoubleHemDefinition/DoubleHemDefinition_Gap.md) | Property that returns the parameter controlling the gap of a hem feature defined by a double bend style of bend. |
| [Length](../DoubleHemDefinition/DoubleHemDefinition_Length.md) | Property that returns the parameter controlling the length of a hem feature defined by a double bend style of bend. |
| [Parent](../DoubleHemDefinition/DoubleHemDefinition_Parent.md) | Property that returns the parent hem definition feature. |

## Version

Introduced in version 2009
