# FeatureControlFrameRows.Item Property

Parent Object: [FeatureControlFrameRows](../FeatureControlFrameRows/FeatureControlFrameRows.md)

## Description

Method that returns the specified FeatureControlFrameRow object from the collection.

## Syntax

FeatureControlFrameRows.**Item**( ***Index*** As Long ) As [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md)

## Property Value

This is a read only property whose value is a [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2009
