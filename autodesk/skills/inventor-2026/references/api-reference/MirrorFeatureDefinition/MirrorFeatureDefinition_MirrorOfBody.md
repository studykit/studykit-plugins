# MirrorFeatureDefinition.MirrorOfBody Property

Parent Object: [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md)

## Description

Read-only property that gets whether this mirror feature resulted from mirroring the SurfaceBody objects.

## Syntax

MirrorFeatureDefinition.**MirrorOfBody**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2017
