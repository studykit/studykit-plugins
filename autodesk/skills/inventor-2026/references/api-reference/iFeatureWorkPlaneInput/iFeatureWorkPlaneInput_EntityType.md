# iFeatureWorkPlaneInput.EntityType Property

Parent Object: [iFeatureWorkPlaneInput](../iFeatureWorkPlaneInput/iFeatureWorkPlaneInput.md)

## Description

Property that returns the type of geometry that are valid. The value returned is the sum of values specifying the valid entity types.

## Syntax

iFeatureWorkPlaneInput.**EntityType**() As [iFeatureEntityInputTypeEnum](../iFeatureEntityInputTypeEnum.md)

## Property Value

This is a read only property whose value is an [iFeatureEntityInputTypeEnum](../iFeatureEntityInputTypeEnum.md).

## Version

Introduced in version 6
