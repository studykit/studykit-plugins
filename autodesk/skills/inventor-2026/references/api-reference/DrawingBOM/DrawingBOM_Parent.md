# DrawingBOM.Parent Property

Parent Object: [DrawingBOM](../DrawingBOM/DrawingBOM.md)

## Description

Property that returns the parent DrawingDocument object.

## Syntax

DrawingBOM.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 2009
