# DrawingBOM.Level Property

Parent Object: [DrawingBOM](../DrawingBOM/DrawingBOM.md)

## Description

Property that returns the type of numbering for the BOM.

## Syntax

DrawingBOM.**Level**() As [PartsListLevelEnum](../PartsListLevelEnum.md)

## Property Value

This is a read only property whose value is a [PartsListLevelEnum](../PartsListLevelEnum.md).

## Version

Introduced in version 2009
