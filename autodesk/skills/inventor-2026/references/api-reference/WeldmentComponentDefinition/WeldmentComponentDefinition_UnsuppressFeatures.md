# WeldmentComponentDefinition.UnsuppressFeatures Method

Parent Object: [WeldmentComponentDefinition](../WeldmentComponentDefinition/WeldmentComponentDefinition.md)

## Description

Method that unsuppresses previously suppressed features.

## Syntax

WeldmentComponentDefinition.**UnsuppressFeatures**( ***Features*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Features | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection that contains the list of features to unsuppress. |

## Version

Introduced in version 2009
