# CentermarkStylesEnumerator Object

## Description

The CentermarkStylesEnumerator object provides access to the existing CentermarkStyle objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CentermarkStylesEnumerator/CentermarkStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CentermarkStylesEnumerator/CentermarkStylesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../CentermarkStylesEnumerator/CentermarkStylesEnumerator_Item.md) | Returns the specified CentermarkStyle object from the collection. |
| [Type](../CentermarkStylesEnumerator/CentermarkStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.CentermarkStyles](../DrawingStylesManager/DrawingStylesManager_CentermarkStyles.md)

## Version

Introduced in version 2010
