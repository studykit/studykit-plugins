# WeldSymbolStyle.SymbolSizeLinkToTextHeight Property

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Gets and sets whether the symbol size is linked to text height or not.

## Syntax

WeldSymbolStyle.**SymbolSizeLinkToTextHeight**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
