# WeldSymbolStyle.SetBackingSymbolsFilter Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Method that sets the backing symbol filter.

## Syntax

WeldSymbolStyle.**SetBackingSymbolsFilter**( ***BackingSymbolFilter***() As [BackingSymbolTypeEnum](../BackingSymbolTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| BackingSymbolFilter | [BackingSymbolTypeEnum](../BackingSymbolTypeEnum.md) | Input an array of BackingSymbolsTypeEnum values to set backing symbol filter. |

## Version

Introduced in version 2024
