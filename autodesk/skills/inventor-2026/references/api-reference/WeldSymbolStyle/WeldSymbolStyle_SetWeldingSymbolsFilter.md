# WeldSymbolStyle.SetWeldingSymbolsFilter Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Method that sets the welding symbol filter.

## Syntax

WeldSymbolStyle.**SetWeldingSymbolsFilter**( ***WeldingSymbolFilter***() As [WeldSymbolTypeEnum](../WeldSymbolTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| WeldingSymbolFilter | [WeldSymbolTypeEnum](../WeldSymbolTypeEnum.md) | Input an array of WeldSymbolTypeEnum values to set welding symbol filter. |

## Version

Introduced in version 2024
