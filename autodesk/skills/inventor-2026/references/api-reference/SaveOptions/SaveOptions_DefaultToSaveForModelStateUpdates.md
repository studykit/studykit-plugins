# SaveOptions.DefaultToSaveForModelStateUpdates Property

Parent Object: [SaveOptions](../SaveOptions/SaveOptions.md)

## Description

Gets/Sets the default save condition for a document that has model state updates.

## Syntax

SaveOptions.**DefaultToSaveForModelStateUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
