# SaveOptions.PromptSaveForModelStateUpdates Property

Parent Object: [SaveOptions](../SaveOptions/SaveOptions.md)

## Description

Gets/Sets whether a prompt to save the document should be displayed when a document that has model state updates.

## Syntax

SaveOptions.**PromptSaveForModelStateUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
