# EdgeUses.Item Property

Parent Object: [EdgeUses](../EdgeUses/EdgeUses.md)

## Description

Allows integer-indexed access to objects in the collection.

## Syntax

EdgeUses.**Item**( ***Index*** As Long ) As [EdgeUse](../EdgeUse/EdgeUse.md)

## Property Value

This is a read only property whose value is an [EdgeUse](../EdgeUse/EdgeUse.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 4
