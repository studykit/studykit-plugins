# MiniToolbarControl.Index Property

Parent Object: [MiniToolbarControl](../MiniToolbarControl/MiniToolbarControl.md)

## Description

Read-only property that returns the index this control is currently positioned at within the mini toolbar.

## Syntax

MiniToolbarControl.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2012
