# MiniToolbarControl.DisplayName Property

Parent Object: [MiniToolbarControl](../MiniToolbarControl/MiniToolbarControl.md)

## Description

Read-write property that gets and sets the display name for the control.

## Syntax

MiniToolbarControl.**DisplayName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
