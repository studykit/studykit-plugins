# DiameterModelDimensionProxy.Visible Property

Parent Object: [DiameterModelDimensionProxy](../DiameterModelDimensionProxy/DiameterModelDimensionProxy.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

DiameterModelDimensionProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
