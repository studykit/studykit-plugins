# DSDegreesOfFreedom.Count Property

Parent Object: [DSDegreesOfFreedom](../DSDegreesOfFreedom/DSDegreesOfFreedom.md)

## Description

Gets the number of items in this collection.

## Syntax

DSDegreesOfFreedom.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2013
