# DSDegreesOfFreedom.Item Property

Parent Object: [DSDegreesOfFreedom](../DSDegreesOfFreedom/DSDegreesOfFreedom.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

DSDegreesOfFreedom.**Item**( ***Index*** As Long ) As [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Property Value

This is a read only property whose value is a [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the DSDegreeOfFreedom to return. |

## Version

Introduced in version 2013
