# SimplifyDefinition.RemovedBodies Property

Parent Object: [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md)

## Description

Read-write property that gets and sets the bodies being removed. This is applicable when the RemoveBodiesBySize is set to True.

## Syntax

SimplifyDefinition.**RemovedBodies**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2026
