# SimplifyDefinition.RemovePocketsMaxDepthRange Property

Parent Object: [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md)

## Description

Read-write property that gets and sets the max depth of pockets in centimeters to remove for the simplify feature. The pockets which max depth is less than or equal to this value will be removed. This is only applicable when the RemovePocketsStyle is set to kS.

## Syntax

SimplifyDefinition.**RemovePocketsMaxDepthRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2026
