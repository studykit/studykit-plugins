# SketchFixedSpline.LineScale Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Gets and sets the LineScale applied to this sketch fixed spline.

## Syntax

SketchFixedSpline.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
