# SketchFixedSpline.LineWeight Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Gets and sets the LineWeight applied to this sketch fixed spline.

## Syntax

SketchFixedSpline.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
