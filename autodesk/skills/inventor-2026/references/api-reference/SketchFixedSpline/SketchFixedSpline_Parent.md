# SketchFixedSpline.Parent Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchFixedSpline.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 9
