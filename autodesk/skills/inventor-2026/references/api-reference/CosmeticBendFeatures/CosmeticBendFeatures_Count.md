# CosmeticBendFeatures.Count Property

Parent Object: [CosmeticBendFeatures](../CosmeticBendFeatures/CosmeticBendFeatures.md)

## Description

Property that returns the number of LoftedFlangeFeature objects in the collection.

## Syntax

CosmeticBendFeatures.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
