# AnnotationPlaneProxy.Type Property

Parent Object: [AnnotationPlaneProxy](../AnnotationPlaneProxy/AnnotationPlaneProxy.md)

## Description

Read-only property returning kAnnotationPlaneObject indicating this objects type.

## Syntax

AnnotationPlaneProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2018
