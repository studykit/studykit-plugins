# SketchFixedSpline3D.StartSketchPoint Property

Parent Object: [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Description

Property that returns the that defines the position of the start of the spline.

## Syntax

SketchFixedSpline3D.**StartSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 9
