# SketchFixedSpline3D.Geometry Property

Parent Object: [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Description

Gets and sets a BSplineCurve3d geometry object. The object returned represents a 'snapshot' view of the current state of the sketch fixed spline.

## Syntax

SketchFixedSpline3D.**Geometry**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read/write property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 9
