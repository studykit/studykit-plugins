# SketchSplineHandle3D.Length Property

Parent Object: [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchSplineHandle3D.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
