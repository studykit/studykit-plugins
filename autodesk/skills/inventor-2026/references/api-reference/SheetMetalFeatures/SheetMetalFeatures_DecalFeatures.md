# SheetMetalFeatures.DecalFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the DecalFeatures collection object. This collection provides access to existing DecalFeature objects.

## Syntax

SheetMetalFeatures.**DecalFeatures**() As [DecalFeatures](../DecalFeatures/DecalFeatures.md)

## Property Value

This is a read only property whose value is a [DecalFeatures](../DecalFeatures/DecalFeatures.md).

## Version

Introduced in version 6
