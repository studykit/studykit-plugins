# SheetMetalFeatures.MarkFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are MarkFeatures.

## Syntax

SheetMetalFeatures.**MarkFeatures**() As [MarkFeatures](../MarkFeatures/MarkFeatures.md)

## Property Value

This is a read only property whose value is a [MarkFeatures](../MarkFeatures/MarkFeatures.md).

## Version

Introduced in version 2023
