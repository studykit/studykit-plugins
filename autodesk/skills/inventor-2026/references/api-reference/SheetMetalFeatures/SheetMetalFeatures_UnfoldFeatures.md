# SheetMetalFeatures.UnfoldFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the UnfoldFeatures collection object. This collection provides access to existing UnfoldFeature objects and provides functionality to create new UnfoldFeature objects.

## Syntax

SheetMetalFeatures.**UnfoldFeatures**() As [UnfoldFeatures](../UnfoldFeatures/UnfoldFeatures.md)

## Property Value

This is a read only property whose value is a [UnfoldFeatures](../UnfoldFeatures/UnfoldFeatures.md).

## Version

Introduced in version 2010
