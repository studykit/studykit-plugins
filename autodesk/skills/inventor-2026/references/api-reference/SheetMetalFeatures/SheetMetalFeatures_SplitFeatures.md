# SheetMetalFeatures.SplitFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the SplitFeatures collection object. This collection provides access to existing SplitFeature objects.

## Syntax

SheetMetalFeatures.**SplitFeatures**() As [SplitFeatures](../SplitFeatures/SplitFeatures.md)

## Property Value

This is a read only property whose value is a [SplitFeatures](../SplitFeatures/SplitFeatures.md).

## Version

Introduced in version 5.3
