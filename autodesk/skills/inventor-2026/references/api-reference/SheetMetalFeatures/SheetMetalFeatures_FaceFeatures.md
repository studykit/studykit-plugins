# SheetMetalFeatures.FaceFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the FaceDraftFeatures collection object. This collection provides access to existing FaceFeature objects.

## Syntax

SheetMetalFeatures.**FaceFeatures**() As [FaceFeatures](../FaceFeatures/FaceFeatures.md)

## Property Value

This is a read only property whose value is a [FaceFeatures](../FaceFeatures/FaceFeatures.md).
