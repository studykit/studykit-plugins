# SheetMetalFeatures.CombineFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the CombineFeatures collection object.

## Syntax

SheetMetalFeatures.**CombineFeatures**() As [CombineFeatures](../CombineFeatures/CombineFeatures.md)

## Property Value

This is a read only property whose value is a [CombineFeatures](../CombineFeatures/CombineFeatures.md).

## Version

Introduced in version 2010
