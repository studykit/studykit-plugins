# SheetMetalFeatures.PunchToolFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the PunchToolFeatures collection object.

## Syntax

SheetMetalFeatures.**PunchToolFeatures**() As [PunchToolFeatures](../PunchToolFeatures/PunchToolFeatures.md)

## Property Value

This is a read only property whose value is a [PunchToolFeatures](../PunchToolFeatures/PunchToolFeatures.md).
