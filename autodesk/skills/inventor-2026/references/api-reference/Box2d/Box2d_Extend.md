# Box2d.Extend Method

Parent Object: [Box2d](../Box2d/Box2d.md)

## Description

Extends the Box to include the specified point.

## Syntax

Box2d.**Extend**( ***Point*** As [Point2d](../Point2d/Point2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point2d](../Point2d/Point2d.md) | Input Point2d object that specifies the coordinate to include in the box. |

## Version

Introduced in version 2008
