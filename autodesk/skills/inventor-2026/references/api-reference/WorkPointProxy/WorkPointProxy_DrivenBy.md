# WorkPointProxy.DrivenBy Property

Parent Object: [WorkPointProxy](../WorkPointProxy/WorkPointProxy.md)

## Description

Property that returns the collection of objects that the work point is dependent on.

## Syntax

WorkPointProxy.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 5
