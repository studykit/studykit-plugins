# WorkPointProxy.Dependents Property

Parent Object: [WorkPointProxy](../WorkPointProxy/WorkPointProxy.md)

## Description

Property that returns the collection of objects that have a direct dependency on the work point.

## Syntax

WorkPointProxy.**Dependents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 5
