# SketchPoint.Layer Property

Parent Object: [SketchPoint](../SketchPoint/SketchPoint.md)

## Description

Gets and sets the layer applied to this sketch entity.

## Syntax

SketchPoint.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
