# PresentationScene.DesignViewAssociative Property

Parent Object: [PresentationScene](../PresentationScene/PresentationScene.md)

## Description

Read-write property that gets and sets whether the publication design view is associative or not.

## Syntax

PresentationScene.**DesignViewAssociative**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
