# PromptMessages.Type Property

Parent Object: [PromptMessages](../PromptMessages/PromptMessages.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

PromptMessages.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2025.1
