# SketchArc.LineDefinitionSpace Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Gets and sets the lineDefinitionSpace applied to this sketch line.

## Syntax

SketchArc.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 9
