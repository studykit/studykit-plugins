# FaceShellProxy.Edges Property

Parent Object: [FaceShellProxy](../FaceShellProxy/FaceShellProxy.md)

## Description

Gets the referenced by this FaceShell.

## Syntax

FaceShellProxy.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 4
