# CollinearConstraint.UseEllipseTwoMajorAxis Property

Parent Object: [CollinearConstraint](../CollinearConstraint/CollinearConstraint.md)

## Description

Property used in the case where the entity returned by the EntityTwo property is an ellipse. This property specifies if the major or minor axis of the ellipse is collinear to EntityOne. True if it is the major axis.

## Syntax

CollinearConstraint.**UseEllipseTwoMajorAxis**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5
