# ApprenticePrintManager.PaperWidth Property

Parent Object: [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Description

Gets/Sets the width of the paper to be printed. Setting this, sets PaperSize to kPaperSizeCustom.

## Syntax

ApprenticePrintManager.**PaperWidth**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
