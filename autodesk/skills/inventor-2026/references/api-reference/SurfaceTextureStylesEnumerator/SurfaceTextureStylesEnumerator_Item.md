# SurfaceTextureStylesEnumerator.Item Property

Parent Object: [SurfaceTextureStylesEnumerator](../SurfaceTextureStylesEnumerator/SurfaceTextureStylesEnumerator.md)

## Description

Returns the specified SurfaceTextureStyle object from the collection.

## Syntax

SurfaceTextureStylesEnumerator.**Item**( ***Index*** As Variant ) As [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Property Value

This is a read only property whose value is a [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the SurfaceTextureStyle to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the SurfaceTextureStyle name. If an out of range index or a name of a non-existent SurfaceTextureStyle is provided, an error will occur. |

## Version

Introduced in version 2009
