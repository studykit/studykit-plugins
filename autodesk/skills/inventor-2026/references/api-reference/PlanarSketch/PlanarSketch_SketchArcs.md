# PlanarSketch.SketchArcs Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

PlanarSketch.**SketchArcs**() As [SketchArcs](../SketchArcs/SketchArcs.md)

## Property Value

This is a read only property whose value is a [SketchArcs](../SketchArcs/SketchArcs.md).

## Version

Introduced in version 5
