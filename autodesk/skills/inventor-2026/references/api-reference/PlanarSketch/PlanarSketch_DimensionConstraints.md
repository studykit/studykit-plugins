# PlanarSketch.DimensionConstraints Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets the collection of all dimension constraints on the sketch.

## Syntax

PlanarSketch.**DimensionConstraints**() As [DimensionConstraints](../DimensionConstraints/DimensionConstraints.md)

## Property Value

This is a read only property whose value is a [DimensionConstraints](../DimensionConstraints/DimensionConstraints.md).

## Version

Introduced in version 5
