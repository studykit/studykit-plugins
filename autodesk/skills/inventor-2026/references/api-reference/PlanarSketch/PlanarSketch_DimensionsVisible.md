# PlanarSketch.DimensionsVisible Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets and sets whether the dimensions on the sketch are visible.

## Syntax

PlanarSketch.**DimensionsVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
