# PlanarSketch.OriginPointGeometry Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that gets the geometry that describes the origin point.

## Syntax

PlanarSketch.**OriginPointGeometry**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).
