# DiameterGeneralDimension.OverrideModelValue Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Gets and sets the NominalValue setting.

## Syntax

DiameterGeneralDimension.**OverrideModelValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
