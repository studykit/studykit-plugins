# ModelAnnotationText.Color Property

Parent Object: [ModelAnnotationText](../ModelAnnotationText/ModelAnnotationText.md)

## Description

Property that gets and sets the color of the annotation text.

## Syntax

ModelAnnotationText.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2018
