# Sketch3DProxy.GeometricConstraints3D Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that returns a collection of all geometric constraints on the 3D sketch.

## Syntax

Sketch3DProxy.**GeometricConstraints3D**() As [GeometricConstraints3D](../GeometricConstraints3D/GeometricConstraints3D.md)

## Property Value

This is a read only property whose value is a [GeometricConstraints3D](../GeometricConstraints3D/GeometricConstraints3D.md).

## Version

Introduced in version 6
