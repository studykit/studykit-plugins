# Sketch3DProxy.HelicalCurves Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Gets the HelicalCurves collection object.

## Syntax

Sketch3DProxy.**HelicalCurves**() As [HelicalCurves](../HelicalCurves/HelicalCurves.md)

## Property Value

This is a read only property whose value is a [HelicalCurves](../HelicalCurves/HelicalCurves.md).

## Version

Introduced in version 2019
