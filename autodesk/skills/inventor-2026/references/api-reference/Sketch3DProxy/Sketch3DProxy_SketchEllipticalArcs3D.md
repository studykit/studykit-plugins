# Sketch3DProxy.SketchEllipticalArcs3D Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that returns the SketchEllipticalArcs3D collection object. This collection provides access to the existing elliptical arcs in the sketch.

## Syntax

Sketch3DProxy.**SketchEllipticalArcs3D**() As [SketchEllipticalArcs3D](../SketchEllipticalArcs3D/SketchEllipticalArcs3D.md)

## Property Value

This is a read only property whose value is a [SketchEllipticalArcs3D](../SketchEllipticalArcs3D/SketchEllipticalArcs3D.md).

## Version

Introduced in version 6
