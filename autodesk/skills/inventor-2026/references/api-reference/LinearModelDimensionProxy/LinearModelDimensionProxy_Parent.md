# LinearModelDimensionProxy.Parent Property

Parent Object: [LinearModelDimensionProxy](../LinearModelDimensionProxy/LinearModelDimensionProxy.md)

## Description

Read-only property that returns the parent component definition of the object.

## Syntax

LinearModelDimensionProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2018
