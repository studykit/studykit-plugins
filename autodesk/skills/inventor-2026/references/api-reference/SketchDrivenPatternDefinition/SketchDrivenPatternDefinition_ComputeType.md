# SketchDrivenPatternDefinition.ComputeType Property

Parent Object: [SketchDrivenPatternDefinition](../SketchDrivenPatternDefinition/SketchDrivenPatternDefinition.md)

## Description

Read-write property that indicates the method of solution for the pattern.

## Syntax

SketchDrivenPatternDefinition.**ComputeType**() As [PatternComputeTypeEnum](../PatternComputeTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PatternComputeTypeEnum](../PatternComputeTypeEnum.md).

## Version

Introduced in version 2017
