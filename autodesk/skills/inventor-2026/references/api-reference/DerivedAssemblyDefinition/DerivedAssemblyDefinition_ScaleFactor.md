# DerivedAssemblyDefinition.ScaleFactor Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Gets or sets the scale factor for the derived Assembly.

## Syntax

DerivedAssemblyDefinition.**ScaleFactor**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
