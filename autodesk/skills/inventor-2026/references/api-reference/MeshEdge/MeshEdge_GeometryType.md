# MeshEdge.GeometryType Property

Parent Object: [MeshEdge](../MeshEdge/MeshEdge.md)

## Description

Read-only property that returns a curve type of the curve that will be returned from the Geometry property.

## Syntax

MeshEdge.**GeometryType**() As [CurveTypeEnum](../CurveTypeEnum.md)

## Property Value

This is a read only property whose value is a [CurveTypeEnum](../CurveTypeEnum.md).

## Version

Introduced in version 2017
