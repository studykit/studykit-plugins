# MeshEdge.Edge Property

Parent Object: [MeshEdge](../MeshEdge/MeshEdge.md)

## Description

Read-only property that returns the transient Brep Edge for the MeshEdge.

## Syntax

MeshEdge.**Edge**() As [Edge](../Edge/Edge.md)

## Property Value

This is a read only property whose value is an [Edge](../Edge/Edge.md).

## Version

Introduced in version 2017
