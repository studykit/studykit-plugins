# RectangularPatternFeatures.AddByDefinition Method

Parent Object: [RectangularPatternFeatures](../RectangularPatternFeatures/RectangularPatternFeatures.md)

## Description

Method that creates a new rectangular pattern feature.

## Syntax

RectangularPatternFeatures.**AddByDefinition**( ***Definition*** As [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md) ) As [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md) | Input RectangularPatternFeatureDefinition objet that defines the RectangularPatternFeature you want to create. |
