# DSJoint.Definition Property

Parent Object: [DSJoint](../DSJoint/DSJoint.md)

## Description

Gets the definition object associated with this joint.

## Syntax

DSJoint.**Definition**() As [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md)

## Property Value

This is a read/write property whose value is a [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md).

## Version

Introduced in version 2013
