# GraphicsDataSets.Item Property

Parent Object: [GraphicsDataSets](../GraphicsDataSets/GraphicsDataSets.md)

## Description

Returns the specified object from the collection.

## Syntax

GraphicsDataSets.**Item**( ***Index*** As Long ) As [GraphicsDataSet](../GraphicsDataSet/GraphicsDataSet.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSet](../GraphicsDataSet/GraphicsDataSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index within the collection of the item to return. |
