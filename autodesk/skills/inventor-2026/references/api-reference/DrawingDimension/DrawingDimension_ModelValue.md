# DrawingDimension.ModelValue Property

Parent Object: [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Description

Property that gets the dimension value as defined in the model or as measured in the drawing.

## Syntax

DrawingDimension.**ModelValue**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 9
