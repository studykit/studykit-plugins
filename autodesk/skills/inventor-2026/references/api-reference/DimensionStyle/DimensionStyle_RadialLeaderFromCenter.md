# DimensionStyle.RadialLeaderFromCenter Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the leader line of a radial dimension extends from the center or not.

## Syntax

DimensionStyle.**RadialLeaderFromCenter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
