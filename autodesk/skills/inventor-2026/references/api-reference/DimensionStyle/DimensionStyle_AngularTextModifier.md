# DimensionStyle.AngularTextModifier Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the optional modifier that controls the controls the position of text in an angular dimension.

## Syntax

DimensionStyle.**AngularTextModifier**() As [DimensionTextModifierEnum](../DimensionTextModifierEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionTextModifierEnum](../DimensionTextModifierEnum.md).

## Version

Introduced in version 2014
