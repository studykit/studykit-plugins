# DimensionStyle.AlternateLeadingZeroDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the leading zeroes for alternate dimensions in the drawing.

## Syntax

DimensionStyle.**AlternateLeadingZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
