# DimensionStyle.OrdinateAlignment Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the type of alignment to use for ordinate dimensions.

## Syntax

DimensionStyle.**OrdinateAlignment**() As [OrdinateDimensionAlignmentEnum](../OrdinateDimensionAlignmentEnum.md)

## Property Value

This is a read/write property whose value is an [OrdinateDimensionAlignmentEnum](../OrdinateDimensionAlignmentEnum.md).

## Version

Introduced in version 2014
