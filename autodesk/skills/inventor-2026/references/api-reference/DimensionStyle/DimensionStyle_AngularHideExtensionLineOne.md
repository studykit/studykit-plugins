# DimensionStyle.AngularHideExtensionLineOne Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the first extension line of an angular dimension is hidden.

## Syntax

DimensionStyle.**AngularHideExtensionLineOne**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
