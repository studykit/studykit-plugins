# DimensionStyle.HoleThreadTypeList Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Returns the hole and thread type list.

## Syntax

DimensionStyle.**HoleThreadTypeList**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2025
