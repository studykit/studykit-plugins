# DimensionStyle.HorizontalDimensionTextOrientation Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Read-write property that gets and sets the orientation of the text in a horizontal dimension. Valid values are kParallelDimensionText and kInlineHorizontalText.

## Syntax

DimensionStyle.**HorizontalDimensionTextOrientation**() As [DimensionTextOrientationEnum](../DimensionTextOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionTextOrientationEnum](../DimensionTextOrientationEnum.md).

## Version

Introduced in version 2014
