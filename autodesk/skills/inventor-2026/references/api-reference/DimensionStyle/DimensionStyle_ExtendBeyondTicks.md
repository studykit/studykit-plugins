# DimensionStyle.ExtendBeyondTicks Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the distance extend the dimension line past the extension line in centimeters when the arrowheads is specified as oblique or no marks.

## Syntax

DimensionStyle.**ExtendBeyondTicks**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2026
