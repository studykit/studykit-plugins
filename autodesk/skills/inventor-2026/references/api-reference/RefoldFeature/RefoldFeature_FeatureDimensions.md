# RefoldFeature.FeatureDimensions Property

Parent Object: [RefoldFeature](../RefoldFeature/RefoldFeature.md)

## Description

Property that returns the FeatureDimensions collection object.

## Syntax

RefoldFeature.**FeatureDimensions**() As [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md)

## Property Value

This is a read only property whose value is a [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md).

## Version

Introduced in version 2010
