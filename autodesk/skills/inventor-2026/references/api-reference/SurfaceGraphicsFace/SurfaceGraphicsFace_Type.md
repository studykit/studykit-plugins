# SurfaceGraphicsFace.Type Property

Parent Object: [SurfaceGraphicsFace](../SurfaceGraphicsFace/SurfaceGraphicsFace.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SurfaceGraphicsFace.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
