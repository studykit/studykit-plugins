# SurfaceGraphicsFace.Appearance Property

Parent Object: [SurfaceGraphicsFace](../SurfaceGraphicsFace/SurfaceGraphicsFace.md)

## Description

Gets and sets appearance associated with the surface graphics face.

## Syntax

SurfaceGraphicsFace.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
