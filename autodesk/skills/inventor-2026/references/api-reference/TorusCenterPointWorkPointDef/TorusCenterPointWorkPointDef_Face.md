# TorusCenterPointWorkPointDef.Face Property

Parent Object: [TorusCenterPointWorkPointDef](../TorusCenterPointWorkPointDef/TorusCenterPointWorkPointDef.md)

## Description

Property that indicates the Face object whose geometry is a torus surface, the center point of which defines the position of the work point.

## Syntax

TorusCenterPointWorkPointDef.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2008
