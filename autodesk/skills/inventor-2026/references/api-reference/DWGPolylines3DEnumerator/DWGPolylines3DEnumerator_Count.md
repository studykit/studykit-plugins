# DWGPolylines3DEnumerator.Count Property

Parent Object: [DWGPolylines3DEnumerator](../DWGPolylines3DEnumerator/DWGPolylines3DEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGPolylines3DEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
