# ButtonDefinition.OverrideShortcut Property

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Gets/Sets the override shortcut assigned to the command. Setting this property to a null string clears the override.

## Syntax

ButtonDefinition.**OverrideShortcut**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
