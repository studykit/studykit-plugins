# EdgeUseDefinitions.Item Property

Parent Object: [EdgeUseDefinitions](../EdgeUseDefinitions/EdgeUseDefinitions.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

EdgeUseDefinitions.**Item**( ***Index*** As Long ) As [EdgeUseDefinition](../EdgeUseDefinition/EdgeUseDefinition.md)

## Property Value

This is a read only property whose value is an [EdgeUseDefinition](../EdgeUseDefinition/EdgeUseDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the EdgeUseDefinition object within the collection to return. |

## Version

Introduced in version 2011
