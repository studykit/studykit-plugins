# PresentationDocument.ReferencedDocuments Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns all the documents directly referenced by this document.

## Syntax

PresentationDocument.**ReferencedDocuments**() As [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md).

## Version

Introduced in version 11
