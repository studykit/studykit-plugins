# DynamicSimulation.IsInSimulationMode Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Toggles the simulation between simulation and construction modes.

## Syntax

DynamicSimulation.**IsInSimulationMode**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
