# DynamicSimulation.NumberOfTimeSteps Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Gets and sets the number of time steps for the entire simulation.

## Syntax

DynamicSimulation.**NumberOfTimeSteps**() As Long

## Property Value

This is a read/write property whose value is a Long.
