# SketchControlPointSplineProxy.StartSketchPoint Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Read-only property that returns the sketch point that defines the position of the start of the spline.

## Syntax

SketchControlPointSplineProxy.**StartSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 2014
