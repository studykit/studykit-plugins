# SketchControlPointSplineProxy.HasReferenceComponent Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchControlPointSplineProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
