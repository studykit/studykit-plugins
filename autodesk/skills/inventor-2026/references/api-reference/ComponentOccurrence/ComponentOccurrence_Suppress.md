# ComponentOccurrence.Suppress Method

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Suppresses this occurrence.

## Syntax

ComponentOccurrence.**Suppress**( [***SkipDocumentSave***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SkipDocumentSave | Boolean | Optional input Boolean value to specify if skip save the source document of the occurrence. This default to False if not specified. |
