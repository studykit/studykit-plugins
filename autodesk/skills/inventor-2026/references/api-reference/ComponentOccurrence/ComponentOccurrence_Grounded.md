# ComponentOccurrence.Grounded Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

'Put' is Inventor Only: Gets/Sets the Boolean flag that specifies whether this Occurrence is grounded or not.

## Syntax

ComponentOccurrence.**Grounded**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
