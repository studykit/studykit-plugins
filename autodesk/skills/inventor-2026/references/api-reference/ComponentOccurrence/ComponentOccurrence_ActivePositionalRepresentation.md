# ComponentOccurrence.ActivePositionalRepresentation Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Gets and sets the active Positional Representation for this occurrence.

## Syntax

ComponentOccurrence.**ActivePositionalRepresentation**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
