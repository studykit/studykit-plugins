# ComponentOccurrence.Constraints Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Property returns the collection of constraints that are acting on the occurrence.

## Syntax

ComponentOccurrence.**Constraints**() As [AssemblyConstraintsEnumerator](../AssemblyConstraintsEnumerator/AssemblyConstraintsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssemblyConstraintsEnumerator](../AssemblyConstraintsEnumerator/AssemblyConstraintsEnumerator.md).

## Version

Introduced in version 4
