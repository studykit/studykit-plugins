# ComponentOccurrence.Joints Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Inventor Only: Gets the enumerator that presents the joints applied on this Component Occurrence.

## Syntax

ComponentOccurrence.**Joints**() As [AssemblyJointsEnumerator](../AssemblyJointsEnumerator/AssemblyJointsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssemblyJointsEnumerator](../AssemblyJointsEnumerator/AssemblyJointsEnumerator.md).

## Version

Introduced in version 2014
