# ModelDatumDefinition.DatumTargetAreaDiameters Property

Parent Object: [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Description

Gets and sets the datum target area diameter values.

## Syntax

ModelDatumDefinition.**DatumTargetAreaDiameters**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2023
