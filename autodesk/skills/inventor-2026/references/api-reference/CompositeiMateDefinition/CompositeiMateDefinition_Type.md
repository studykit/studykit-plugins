# CompositeiMateDefinition.Type Property

Parent Object: [CompositeiMateDefinition](../CompositeiMateDefinition/CompositeiMateDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CompositeiMateDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
