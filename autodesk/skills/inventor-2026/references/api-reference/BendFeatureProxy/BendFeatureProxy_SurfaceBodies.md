# BendFeatureProxy.SurfaceBodies Property

Parent Object: [BendFeatureProxy](../BendFeatureProxy/BendFeatureProxy.md)

## Description

Property that returns the bodies that this feature has created or modified.

## Syntax

BendFeatureProxy.**SurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).

## Version

Introduced in version 2010
