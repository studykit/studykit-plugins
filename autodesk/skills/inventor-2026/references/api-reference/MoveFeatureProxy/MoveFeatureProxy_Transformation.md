# MoveFeatureProxy.Transformation Property

Parent Object: [MoveFeatureProxy](../MoveFeatureProxy/MoveFeatureProxy.md)

## Description

A transformation matrix that moves an extrude, revolve, or sweep feature to a new location.

## Syntax

MoveFeatureProxy.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2010
