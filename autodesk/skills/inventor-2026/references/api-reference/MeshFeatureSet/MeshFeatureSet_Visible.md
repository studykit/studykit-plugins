# MeshFeatureSet.Visible Property

Parent Object: [MeshFeatureSet](../MeshFeatureSet/MeshFeatureSet.md)

## Description

Read-write property that gets and sets whether all of the mesh features within this set are visible.

## Syntax

MeshFeatureSet.**Visible**() As [MeshFeaturesVisibilityEnum](../MeshFeaturesVisibilityEnum.md)

## Property Value

This is a read/write property whose value is a [MeshFeaturesVisibilityEnum](../MeshFeaturesVisibilityEnum.md).

## Version

Introduced in version 2017
