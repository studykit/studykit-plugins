# iFeatureProxy.NativeObject Property

Parent Object: [iFeatureProxy](../iFeatureProxy/iFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

iFeatureProxy.**NativeObject**() As [iFeature](../iFeature/iFeature.md)

## Property Value

This is a read only property whose value is an [iFeature](../iFeature/iFeature.md).

## Version

Introduced in version 2009
