# MoveFeatures.CreateMoveDefinition Method

Parent Object: [MoveFeatures](../MoveFeatures/MoveFeatures.md)

## Description

Creates a MoveDefinition object.

## Syntax

MoveFeatures.**CreateMoveDefinition**( ***Bodies*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) ) As [MoveDefinition](../MoveDefinition/MoveDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Bodies | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection object that contains the surface bodies to move. The WorkSurface is also valid to be added into the ObjectCollection to move. |
