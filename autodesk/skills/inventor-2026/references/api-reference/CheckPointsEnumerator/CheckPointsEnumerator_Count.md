# CheckPointsEnumerator.Count Property

Parent Object: [CheckPointsEnumerator](../CheckPointsEnumerator/CheckPointsEnumerator.md)

## Description

Property that returns the number of items in this collection.

## Syntax

CheckPointsEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5
