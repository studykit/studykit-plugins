# iAssemblyTableRow.Parent Property

Parent Object: [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md)

## Description

Property that returns the parent iAssemblyFactory object.

## Syntax

iAssemblyTableRow.**Parent**() As [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Property Value

This is a read only property whose value is an [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md).

## Version

Introduced in version 11
