# RibbonPanel.InternalName Property

Parent Object: [RibbonPanel](../RibbonPanel/RibbonPanel.md)

## Description

Property that returns the unique internal name of the ribbon panel.

## Syntax

RibbonPanel.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2010
