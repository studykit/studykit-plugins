# DimensionText.VerticalJustification Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Gets and sets the vertical justification of the text.

## Syntax

DimensionText.**VerticalJustification**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 9
