# ModelStateTableColumn.FormattedHeading Property

Parent Object: [ModelStateTableColumn](../ModelStateTableColumn/ModelStateTableColumn.md)

## Description

Read - only property that returns the heading of the column in XML format.

## Syntax

ModelStateTableColumn.**FormattedHeading**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
