# ModelGeneralNoteProxy.Parent Property

Parent Object: [ModelGeneralNoteProxy](../ModelGeneralNoteProxy/ModelGeneralNoteProxy.md)

## Description

Read-only property that returns the parent component definition of the object.

## Syntax

ModelGeneralNoteProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2018
