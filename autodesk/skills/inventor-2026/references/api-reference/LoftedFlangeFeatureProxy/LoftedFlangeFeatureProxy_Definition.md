# LoftedFlangeFeatureProxy.Definition Property

Parent Object: [LoftedFlangeFeatureProxy](../LoftedFlangeFeatureProxy/LoftedFlangeFeatureProxy.md)

## Description

Gets and sets the LoftedFlangeDefinition object associated with this lofted flange feature.

## Syntax

LoftedFlangeFeatureProxy.**Definition**() As [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Property Value

This is a read/write property whose value is a [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md).

## Version

Introduced in version 2011
