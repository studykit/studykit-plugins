# SoftwareVersion.ProductEdition Property

Parent Object: [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Description

Gets the product edition. Only valid when SoftwareVersion is obtained from the Application.

## Syntax

SoftwareVersion.**ProductEdition**() As [ProductEditionEnum](../ProductEditionEnum.md)

## Property Value

This is a read only property whose value is a [ProductEditionEnum](../ProductEditionEnum.md).

## Version

Introduced in version 2009
