# PartFeatures.SketchDrivenPatternFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that besides listing out the subset of part features that are SketchDrivenPatterns, allows the creations of new SketchDrivenPatterns.

## Syntax

PartFeatures.**SketchDrivenPatternFeatures**() As [SketchDrivenPatternFeatures](../SketchDrivenPatternFeatures/SketchDrivenPatternFeatures.md)

## Property Value

This is a read only property whose value is a [SketchDrivenPatternFeatures](../SketchDrivenPatternFeatures/SketchDrivenPatternFeatures.md).

## Version

Introduced in version 2017
