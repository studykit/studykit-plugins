# PartFeatures.CombineFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the CombineFeatures collection object.

## Syntax

PartFeatures.**CombineFeatures**() As [CombineFeatures](../CombineFeatures/CombineFeatures.md)

## Property Value

This is a read only property whose value is a [CombineFeatures](../CombineFeatures/CombineFeatures.md).

## Version

Introduced in version 2010
