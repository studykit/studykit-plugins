# PartFeatures.LipFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are LipFeatures.

## Syntax

PartFeatures.**LipFeatures**() As [LipFeatures](../LipFeatures/LipFeatures.md)

## Property Value

This is a read only property whose value is a [LipFeatures](../LipFeatures/LipFeatures.md).

## Version

Introduced in version 2010
