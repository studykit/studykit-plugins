# PartFeatures.CoilFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the CoilFeatures collection object. This collection provides access to existing CoilFeature objects and provides functionality to create new CoilFeature objects.

## Syntax

PartFeatures.**CoilFeatures**() As [CoilFeatures](../CoilFeatures/CoilFeatures.md)

## Property Value

This is a read only property whose value is a [CoilFeatures](../CoilFeatures/CoilFeatures.md).

## Version

Introduced in version 5
