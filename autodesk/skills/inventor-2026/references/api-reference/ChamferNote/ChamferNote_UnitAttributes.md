# ChamferNote.UnitAttributes Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Get the unit attributes associated with the LeaderNote.

## Syntax

ChamferNote.**UnitAttributes**() As [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Property Value

This is a read only property whose value is a [UnitAttributes](../UnitAttributes/UnitAttributes.md).

## Version

Introduced in version 2016
