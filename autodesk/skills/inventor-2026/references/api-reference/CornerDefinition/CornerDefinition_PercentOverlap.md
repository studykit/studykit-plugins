# CornerDefinition.PercentOverlap Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Property that specifies the overlap as a percentage of the flange thickness.

## Syntax

CornerDefinition.**PercentOverlap**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
