# CornerDefinition.CornerType Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Gets how the corner seam or miter is defined.

## Syntax

CornerDefinition.**CornerType**() As [CornerTypeEnum](../CornerTypeEnum.md)

## Property Value

This is a read/write property whose value is a [CornerTypeEnum](../CornerTypeEnum.md).

## Version

Introduced in version 2009
