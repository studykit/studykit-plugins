# MiniToolbarSlider.ControlType Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-only property that returns the control type.

## Syntax

MiniToolbarSlider.**ControlType**() As [MiniToolbarControlTypeEnum](../MiniToolbarControlTypeEnum.md)

## Property Value

This is a read only property whose value is a [MiniToolbarControlTypeEnum](../MiniToolbarControlTypeEnum.md).

## Version

Introduced in version 2013
