# LoftedFlangeFeature.IsOwnedByFeature Property

Parent Object: [LoftedFlangeFeature](../LoftedFlangeFeature/LoftedFlangeFeature.md)

## Description

Property that returns whether this object is owned by a feature. If True, the OwnedBy property returns the owning feature.

## Syntax

LoftedFlangeFeature.**IsOwnedByFeature**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
