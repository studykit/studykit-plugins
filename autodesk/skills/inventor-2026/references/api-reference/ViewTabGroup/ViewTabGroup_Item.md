# ViewTabGroup.Item Property

Parent Object: [ViewTabGroup](../ViewTabGroup/ViewTabGroup.md)

## Description

Read-only property that returns the specified ViewTab object from the collection.

## Syntax

ViewTabGroup.**Item**( ***Index*** As Long ) As [ViewTab](../ViewTab/ViewTab.md)

## Property Value

This is a read only property whose value is a [ViewTab](../ViewTab/ViewTab.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the ViewTab to return. |

## Version

Introduced in version 2022
