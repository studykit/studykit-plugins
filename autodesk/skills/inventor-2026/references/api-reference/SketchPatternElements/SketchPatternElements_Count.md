# SketchPatternElements.Count Property

Parent Object: [SketchPatternElements](../SketchPatternElements/SketchPatternElements.md)

## Description

Gets the number of items in this collection.

## Syntax

SketchPatternElements.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2025.1
