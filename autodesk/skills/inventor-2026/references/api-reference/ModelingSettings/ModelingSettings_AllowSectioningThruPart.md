# ModelingSettings.AllowSectioningThruPart Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets whether sectioning through part is allowed.

## Syntax

ModelingSettings.**AllowSectioningThruPart**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
