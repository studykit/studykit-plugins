# ModelingSettings Object

## Description

The ModelingSettings object provides access to properties that provide read and write access of the various modeling related document specific settings. This is somewhat equivalent to the Modeling tab of the Document Settings dialog.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AdaptivelyUsedInAssembly](../ModelingSettings/ModelingSettings_AdaptivelyUsedInAssembly.md) | Gets and sets the indicator that part is user adaptive in an assembly. |
| [AdvancedFeatureValidation](../ModelingSettings/ModelingSettings_AdvancedFeatureValidation.md) | Gets and sets the advanced feature validation setting. |
| [AllowSectioningThruPart](../ModelingSettings/ModelingSettings_AllowSectioningThruPart.md) | Gets and sets whether sectioning through part is allowed. |
| [Application](../ModelingSettings/ModelingSettings_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AutoFindErrorsAfterManualRepair](../ModelingSettings/ModelingSettings_AutoFindErrorsAfterManualRepair.md) | Gets and sets whether to find errors automatically after manual repair. |
| [CompactModelHistory](../ModelingSettings/ModelingSettings_CompactModelHistory.md) | Gets and sets whether the rollback history should be purged when a document is saved. |
| [InitialDrawingViewHeight](../ModelingSettings/ModelingSettings_InitialDrawingViewHeight.md) | Gets and sets the initial view height (in centimeters) to use when creating a drawing view from this model. |

| [UpdateLegacyHoles](../ModelingSettings/ModelingSettings_UpdateLegacyHoles.md) | Gets and sets the update legacy holes setting. |
