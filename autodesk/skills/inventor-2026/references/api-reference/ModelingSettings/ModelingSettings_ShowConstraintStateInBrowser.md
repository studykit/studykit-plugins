# ModelingSettings.ShowConstraintStateInBrowser Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets whether to show the constraint state in browser.

## Syntax

ModelingSettings.**ShowConstraintStateInBrowser**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
