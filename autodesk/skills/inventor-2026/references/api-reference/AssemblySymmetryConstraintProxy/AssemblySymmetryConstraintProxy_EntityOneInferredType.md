# AssemblySymmetryConstraintProxy.EntityOneInferredType Property

Parent Object: [AssemblySymmetryConstraintProxy](../AssemblySymmetryConstraintProxy/AssemblySymmetryConstraintProxy.md)

## Description

Read-only property that returns an enum indicating how the geometry of entity one is interpreted.

## Syntax

AssemblySymmetryConstraintProxy.**EntityOneInferredType**() As [InferredTypeEnum](../InferredTypeEnum.md)

## Property Value

This is a read only property whose value is an [InferredTypeEnum](../InferredTypeEnum.md).

## Version

Introduced in version 2014
