# FaceCollection.Add Method

Parent Object: [FaceCollection](../FaceCollection/FaceCollection.md)

## Description

Adds an object to the generic collection.

## Syntax

FaceCollection.**Add**( ***Object*** As Object )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Object | Object | Input Object to add to the collection. |
