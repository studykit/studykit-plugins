# UnwrapDefinition.InputFaces Property

Parent Object: [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md)

## Description

Read-write property that gets and sets the faces to unwrap.

## Syntax

UnwrapDefinition.**InputFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2020
