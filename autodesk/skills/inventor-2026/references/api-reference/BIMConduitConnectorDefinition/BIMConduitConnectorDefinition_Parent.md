# BIMConduitConnectorDefinition.Parent Property

Parent Object: [BIMConduitConnectorDefinition](../BIMConduitConnectorDefinition/BIMConduitConnectorDefinition.md)

## Description

Property that returns the parent BIMConnector that this definition is associated with. If the definition was created using one of the Create methods this property will return Nothing since the definition isn’t associated with a connector yet.

## Syntax

BIMConduitConnectorDefinition.**Parent**() As [BIMConnector](../BIMConnector/BIMConnector.md)

## Property Value

This is a read only property whose value is a [BIMConnector](../BIMConnector/BIMConnector.md).

## Version

Introduced in version 2011
