# DirectEditDeleteOperationProxy.NativeObject Property

Parent Object: [DirectEditDeleteOperationProxy](../DirectEditDeleteOperationProxy/DirectEditDeleteOperationProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

DirectEditDeleteOperationProxy.**NativeObject**() As [DirectEditDeleteOperation](../DirectEditDeleteOperation/DirectEditDeleteOperation.md)

## Property Value

This is a read only property whose value is a [DirectEditDeleteOperation](../DirectEditDeleteOperation/DirectEditDeleteOperation.md).

## Version

Introduced in version 2015
