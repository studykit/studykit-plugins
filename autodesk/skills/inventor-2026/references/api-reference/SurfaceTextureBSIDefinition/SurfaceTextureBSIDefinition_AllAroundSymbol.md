# SurfaceTextureBSIDefinition.AllAroundSymbol Property

Parent Object: [SurfaceTextureBSIDefinition](../SurfaceTextureBSIDefinition/SurfaceTextureBSIDefinition.md)

## Description

Gets and sets whether to add the all-around indicator to the symbol.

## Syntax

SurfaceTextureBSIDefinition.**AllAroundSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
