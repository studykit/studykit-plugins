# ViewFrame.ViewTabGroups Property

Parent Object: [ViewFrame](../ViewFrame/ViewFrame.md)

## Description

Read-only property that returns the ViewTabGroupsEnumerator collection object.

## Syntax

ViewFrame.**ViewTabGroups**() As [ViewTabGroupsEnumerator](../ViewTabGroupsEnumerator/ViewTabGroupsEnumerator.md)

## Property Value

This is a read only property whose value is a [ViewTabGroupsEnumerator](../ViewTabGroupsEnumerator/ViewTabGroupsEnumerator.md).

## Version

Introduced in version 2022
