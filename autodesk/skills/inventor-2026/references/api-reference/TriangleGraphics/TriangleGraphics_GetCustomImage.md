# TriangleGraphics.GetCustomImage Method

Parent Object: [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Description

Method that gets the image used for this TriangleGraphics object.

## Syntax

TriangleGraphics.**GetCustomImage**( ***ImageSet*** As [GraphicsImageSet](../GraphicsImageSet/GraphicsImageSet.md), ***ImageIndex*** As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ImageSet | [GraphicsImageSet](../GraphicsImageSet/GraphicsImageSet.md) | Output GraphicsImageSet that is currently assigned to this TriangleGraphics object. |
| ImageIndex | Long | Output Long that indicates which image in the GraphicsImageSet is currently being used by this TriangleGraphics object. |

## Version

Introduced in version 2017
