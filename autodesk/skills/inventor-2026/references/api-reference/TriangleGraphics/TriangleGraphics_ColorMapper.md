# TriangleGraphics.ColorMapper Property

Parent Object: [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Description

Read-write property that gets and sets the GraphicsColorMapper object associated with the set.

## Syntax

TriangleGraphics.**ColorMapper**() As [GraphicsColorMapper](../GraphicsColorMapper/GraphicsColorMapper.md)

## Property Value

This is a read/write property whose value is a [GraphicsColorMapper](../GraphicsColorMapper/GraphicsColorMapper.md).
