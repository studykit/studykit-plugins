# BendNotes.Item Property

Parent Object: [BendNotes](../BendNotes/BendNotes.md)

## Description

Returns the specified BendNote object from the collection.

## Syntax

BendNotes.**Item**( ***Index*** As Long ) As [BendNote](../BendNote/BendNote.md)

## Property Value

This is a read only property whose value is a [BendNote](../BendNote/BendNote.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
