# ViewList.Item Property

Parent Object: [ViewList](../ViewList/ViewList.md)

## Description

Returns the specified object in the collection.

## Syntax

ViewList.**Item**( ***Index*** As Long ) As [View](../View/View.md)

## Property Value

This is a read only property whose value is a [View](../View/View.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index within the collection of the item to return. |

## Version

Introduced in version 9
