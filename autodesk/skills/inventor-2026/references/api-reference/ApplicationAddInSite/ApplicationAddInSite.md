# ApplicationAddInSite Object

## Description

The ApplicationAddInSite object is an object that is only available through the Activate method of the ApplicationAddInServer interface.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ApplicationAddInSite/ApplicationAddInSite_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../ApplicationAddInSite/ApplicationAddInSite_Parent.md) | Property that returns the parent ApplicationAddIn object. |
| [Type](../ApplicationAddInSite/ApplicationAddInSite_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
