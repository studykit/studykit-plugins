# ModelFeatureControlFrame.TopToleranceFeature Property

Parent Object: [ModelFeatureControlFrame](../ModelFeatureControlFrame/ModelFeatureControlFrame.md)

## Description

Returns the top ModelToleranceFeature object.

## Syntax

ModelFeatureControlFrame.**TopToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2019
