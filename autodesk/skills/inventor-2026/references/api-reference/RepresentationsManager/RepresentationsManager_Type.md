# RepresentationsManager.Type Property

Parent Object: [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RepresentationsManager.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
