# Views.Item Property

Parent Object: [Views](../Views/Views.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

Views.**Item**( ***Index*** As Long ) As [View](../View/View.md)

## Property Value

This is a read only property whose value is a [View](../View/View.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the View to return. |

## Version

Introduced in version 4
