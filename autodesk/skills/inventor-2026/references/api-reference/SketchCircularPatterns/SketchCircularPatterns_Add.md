# SketchCircularPatterns.Add Method

Parent Object: [SketchCircularPatterns](../SketchCircularPatterns/SketchCircularPatterns.md)

## Description

Creates a new sketch circular pattern. The new SketchCircularPattern object is returned.

## Syntax

SketchCircularPatterns.**Add**( ***Definition*** As [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md) ) As [SketchCircularPattern](../SketchCircularPattern/SketchCircularPattern.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md) | Input SketchCircularPatternDefinition objet that defines the SketchCircularPattern to create. |

## Version

Introduced in version 2025.1
