# EdgeLoopDefinitions.Add Method

Parent Object: [EdgeLoopDefinitions](../EdgeLoopDefinitions/EdgeLoopDefinitions.md)

Method that creates a new EdgeLoopDefinition within the associated SurfaceBodyDefinition.

## Syntax

EdgeLoopDefinitions.**Add**() As [EdgeLoopDefinition](../EdgeLoopDefinition/EdgeLoopDefinition.md)
