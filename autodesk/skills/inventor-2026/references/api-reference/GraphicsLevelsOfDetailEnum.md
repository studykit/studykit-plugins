# GraphicsLevelsOfDetailEnum Enumerator

## Description

Bits indicating the Levels of Detail in the graphics system. It controls the fineness of the facets generated on the surfaces.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kCoarseRes | 128 | Estimated fineness required for a coarse picture. |
| kFullScreenHighRes | 2048 | Estimated fineness required for a good picture when you fit an 'average' model to full screen. |
| kMediumRes | 512 | Estimated fineness required for an averagely good picture. |
| kMereDotRes | 8 | Estimated fineness required for representing the model as just a blob. |
| kVeryCoarseRes | 32 | Estimated fineness required for a very coarse picture. |
| kZoomedInHighRes | 8192 | Estimated fineness required for a good picture when you are zoomed in on an 'average' model. |

## Version

Introduced in version 4
