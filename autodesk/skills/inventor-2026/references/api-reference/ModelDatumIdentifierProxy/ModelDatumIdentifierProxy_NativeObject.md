# ModelDatumIdentifierProxy.NativeObject Property

Parent Object: [ModelDatumIdentifierProxy](../ModelDatumIdentifierProxy/ModelDatumIdentifierProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ModelDatumIdentifierProxy.**NativeObject**() As [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md)

## Property Value

This is a read only property whose value is a [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md).

## Version

Introduced in version 2018
