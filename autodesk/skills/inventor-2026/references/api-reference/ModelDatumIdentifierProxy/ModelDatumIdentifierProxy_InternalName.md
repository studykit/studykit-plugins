# ModelDatumIdentifierProxy.InternalName Property

Parent Object: [ModelDatumIdentifierProxy](../ModelDatumIdentifierProxy/ModelDatumIdentifierProxy.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelDatumIdentifierProxy.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
