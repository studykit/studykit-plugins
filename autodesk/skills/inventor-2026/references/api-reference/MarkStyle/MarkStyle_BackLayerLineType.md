# MarkStyle.BackLayerLineType Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that gets and sets the line type for back Layer.

## Syntax

MarkStyle.**BackLayerLineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2023
