# LineSegment.Direction Property

Parent Object: [LineSegment](../LineSegment/LineSegment.md)

## Description

Property that returns the direction of the line segment.

## Syntax

LineSegment.**Direction**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 6
