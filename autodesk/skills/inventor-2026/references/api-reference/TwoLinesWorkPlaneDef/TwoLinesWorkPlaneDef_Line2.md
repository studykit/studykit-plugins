# TwoLinesWorkPlaneDef.Line2 Property

Parent Object: [TwoLinesWorkPlaneDef](../TwoLinesWorkPlaneDef/TwoLinesWorkPlaneDef.md)

## Description

Property that returns the second line of the two line defined work plane. This object can be a linear Edge, WorkAxis, or SketchLine object.

## Syntax

TwoLinesWorkPlaneDef.**Line2**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
