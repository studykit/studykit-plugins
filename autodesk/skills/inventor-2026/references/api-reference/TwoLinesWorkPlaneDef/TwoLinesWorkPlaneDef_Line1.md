# TwoLinesWorkPlaneDef.Line1 Property

Parent Object: [TwoLinesWorkPlaneDef](../TwoLinesWorkPlaneDef/TwoLinesWorkPlaneDef.md)

## Description

Property that returns the first line of the two line defined work plane. This line defines the X-axis of the work plane. This object can be a linear Edge, WorkAxis, or SketchLine object.

## Syntax

TwoLinesWorkPlaneDef.**Line1**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
