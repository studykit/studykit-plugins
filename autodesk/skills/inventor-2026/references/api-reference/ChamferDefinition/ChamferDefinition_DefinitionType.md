# ChamferDefinition.DefinitionType Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Gets the type of the definition object used to hold data for chamfer feature.

## Syntax

ChamferDefinition.**DefinitionType**() As [ChamferDefinitionTypeEnum](../ChamferDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [ChamferDefinitionTypeEnum](../ChamferDefinitionTypeEnum.md).

## Version

Introduced in version 2018
