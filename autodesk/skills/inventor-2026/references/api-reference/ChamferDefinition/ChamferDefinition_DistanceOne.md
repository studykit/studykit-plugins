# ChamferDefinition.DistanceOne Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Returns the parameter that controls the first distance used for the chamfer feature. This is applicable when the DefinitionType is kTwoDistances.

## Syntax

ChamferDefinition.**DistanceOne**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2018
