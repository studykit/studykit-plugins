# SheetSettings.Parent Property

Parent Object: [SheetSettings](../SheetSettings/SheetSettings.md)

## Description

Property that returns the parent DrawingDocument object.

## Syntax

SheetSettings.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 11
