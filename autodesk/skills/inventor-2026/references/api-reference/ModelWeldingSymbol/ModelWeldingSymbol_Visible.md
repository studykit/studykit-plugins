# ModelWeldingSymbol.Visible Property

Parent Object: [ModelWeldingSymbol](../ModelWeldingSymbol/ModelWeldingSymbol.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelWeldingSymbol.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
