# BOM.StructuredViewDelimiter Property

Parent Object: [BOM](../BOM/BOM.md)

## Description

Gets and sets the delimiter to use for numbering. This property applies only for an all-level structured view.

## Syntax

BOM.**StructuredViewDelimiter**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
