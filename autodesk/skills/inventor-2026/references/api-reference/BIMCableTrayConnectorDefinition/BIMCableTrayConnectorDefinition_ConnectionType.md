# BIMCableTrayConnectorDefinition.ConnectionType Property

Parent Object: [BIMCableTrayConnectorDefinition](../BIMCableTrayConnectorDefinition/BIMCableTrayConnectorDefinition.md)

## Description

Read-write property that gets and sets the connection type for this connector.

## Syntax

BIMCableTrayConnectorDefinition.**ConnectionType**() As [BIMCableTrayConnectionTypeEnum](../BIMCableTrayConnectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BIMCableTrayConnectionTypeEnum](../BIMCableTrayConnectionTypeEnum.md).

## Version

Introduced in version 2011
