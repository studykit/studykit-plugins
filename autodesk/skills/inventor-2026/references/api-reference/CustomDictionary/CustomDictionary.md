# CustomDictionary Object

## Description

CustomDictionary object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddWord](../CustomDictionary/CustomDictionary_AddWord.md) | Method that adds a word to this custom dictionary. |
| [DeleteWord](../CustomDictionary/CustomDictionary_DeleteWord.md) | Method that deletes a word from this custom dictionary. |
| [GetWords](../CustomDictionary/CustomDictionary_GetWords.md) | Method that gets all words in this custom dictionary. |
| [ImportWordsFromExistingDictionary](../CustomDictionary/CustomDictionary_ImportWordsFromExistingDictionary.md) | Method that imports words to this custom dictionary from an existing custom dictionary. |
| [Remove](../CustomDictionary/CustomDictionary_Remove.md) | Method that removes the CustomDictionary from the custom dictionaries. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CustomDictionary/CustomDictionary_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [FullFileName](../CustomDictionary/CustomDictionary_FullFileName.md) | Read-only property that returns the fully qualified file name of the CustomDictionary. |
| [Name](../CustomDictionary/CustomDictionary_Name.md) | Read-only property that returns the name of the CustomDictionary. |
| [Type](../CustomDictionary/CustomDictionary_Type.md) | Gets the constant that indicates the type of this object. |
