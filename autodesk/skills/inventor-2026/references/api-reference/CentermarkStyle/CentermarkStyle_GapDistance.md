# CentermarkStyle.GapDistance Property

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Gets and sets the gap distance between the center indicator and the extension line.

## Syntax

CentermarkStyle.**GapDistance**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
