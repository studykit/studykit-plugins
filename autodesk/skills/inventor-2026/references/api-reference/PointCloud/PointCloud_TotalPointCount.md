# PointCloud.TotalPointCount Property

Parent Object: [PointCloud](../PointCloud/PointCloud.md)

## Description

Read-only property that returns the total number of points in the point cloud taking no account of the crops.

## Syntax

PointCloud.**TotalPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
