# PositionalRepresentation.SetGroundingOverride Method

Parent Object: [PositionalRepresentation](../PositionalRepresentation/PositionalRepresentation.md)

## Description

Method that sets an override grounding state on the input occurrence. If not already overridden, the method has the effect of setting the 'Override' flag for grounding.

## Syntax

PositionalRepresentation.**SetGroundingOverride**( ***Occurrence*** As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md), ***Grounded*** As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Occurrence | [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md) | ComponentOccurrence object that specifies the occurrence to be overridden. |
| Grounded | Boolean | Boolean that specifies the override grounding state. |

## Version

Introduced in version 11
