# FlatPattern.WorkPlanes Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns that collection object that provides access to all of the existing work plane objects in the flat pattern and provides the functionality to create new work plane objects.

## Syntax

FlatPattern.**WorkPlanes**() As [WorkPlanes](../WorkPlanes/WorkPlanes.md)

## Property Value

This is a read only property whose value is a [WorkPlanes](../WorkPlanes/WorkPlanes.md).

## Version

Introduced in version 2010
