# FlatPattern.UnsuppressFeatures Method

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Method that unsuppresses the specified features. This is limited to the features created in the flat pattern.

## Syntax

FlatPattern.**UnsuppressFeatures**( ***Features*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Features | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection of flat pattern features to unsuppress. |

## Version

Introduced in version 2010
