# FlatPattern.FlatBendResults Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the FlatBendResults collection object that contains information about all the bends in the flat pattern.

## Syntax

FlatPattern.**FlatBendResults**() As [FlatBendResults](../FlatBendResults/FlatBendResults.md)

## Property Value

This is a read only property whose value is a [FlatBendResults](../FlatBendResults/FlatBendResults.md).

## Version

Introduced in version 2008
