# Sketch3DSettings.AutoBendRadius Property

Parent Object: [Sketch3DSettings](../Sketch3DSettings/Sketch3DSettings.md)

## Description

Gets and sets the default radius used for corner bends.

## Syntax

Sketch3DSettings.**AutoBendRadius**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 8
