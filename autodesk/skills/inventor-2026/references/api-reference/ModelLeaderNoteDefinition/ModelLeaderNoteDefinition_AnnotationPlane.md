# ModelLeaderNoteDefinition.AnnotationPlane Property

Parent Object: [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md)

## Description

Read-write property that gets and sets the annotation plane for the dimension.

## Syntax

ModelLeaderNoteDefinition.**AnnotationPlane**() As [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md).

## Version

Introduced in version 2018
