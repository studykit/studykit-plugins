# ModelLeaderNoteDefinition.AssociatedGeometries Property

Parent Object: [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md)

## Description

Read-write property that gets and sets the associated geometries. Valid geometries include faces, edges and vertices.

## Syntax

ModelLeaderNoteDefinition.**AssociatedGeometries**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2023
