# Arc3d.Evaluator Property

Parent Object: [Arc3d](../Arc3d/Arc3d.md)

## Description

Property that gets the CurveEvaluator for this arc.

## Syntax

Arc3d.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 6
