# Arc3d.Normal Property

Parent Object: [Arc3d](../Arc3d/Arc3d.md)

## Description

Specifies the normal for this arc.

## Syntax

Arc3d.**Normal**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 6
