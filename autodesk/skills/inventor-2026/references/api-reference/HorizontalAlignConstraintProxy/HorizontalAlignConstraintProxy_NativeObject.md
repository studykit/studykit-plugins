# HorizontalAlignConstraintProxy.NativeObject Property

Parent Object: [HorizontalAlignConstraintProxy](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

HorizontalAlignConstraintProxy.**NativeObject**() As [HorizontalAlignConstraint](../HorizontalAlignConstraint/HorizontalAlignConstraint.md)

## Property Value

This is a read only property whose value is a [HorizontalAlignConstraint](../HorizontalAlignConstraint/HorizontalAlignConstraint.md).

## Version

Introduced in version 6
