# CornerRoundFeature.Parent Property

Parent Object: [CornerRoundFeature](../CornerRoundFeature/CornerRoundFeature.md)

## Description

Property that returns the parent of the feature.

## Syntax

CornerRoundFeature.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 5.3
