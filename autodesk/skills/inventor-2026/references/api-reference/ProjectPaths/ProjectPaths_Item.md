# ProjectPaths.Item Property

Parent Object: [ProjectPaths](../ProjectPaths/ProjectPaths.md)

## Description

Returns the specified ProjectPath object from the collection.

## Syntax

ProjectPaths.**Item**( ***Index*** As Variant ) As [ProjectPath](../ProjectPath/ProjectPath.md)

## Property Value

This is a read only property whose value is a [ProjectPath](../ProjectPath/ProjectPath.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the ProjectPath to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the project path name. If an out of range index or a name of a non\-existent path is provided, an error occurs. |

## Version

Introduced in version 2011
