# Sketch3D.SketchEquationCurves3D Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Read-only property that returns the SketchEquationCurves3D collection object. This collection provides access to the existing equation curves in the sketch and provides functionality to create new equation curves.

## Syntax

Sketch3D.**SketchEquationCurves3D**() As [SketchEquationCurves3D](../SketchEquationCurves3D/SketchEquationCurves3D.md)

## Property Value

This is a read only property whose value is a [SketchEquationCurves3D](../SketchEquationCurves3D/SketchEquationCurves3D.md).
