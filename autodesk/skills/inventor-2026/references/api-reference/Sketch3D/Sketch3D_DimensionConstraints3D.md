# Sketch3D.DimensionConstraints3D Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Property that returns a collection of all dimension constraints on the 3D sketch.

## Syntax

Sketch3D.**DimensionConstraints3D**() As [DimensionConstraints3D](../DimensionConstraints3D/DimensionConstraints3D.md)

## Property Value

This is a read only property whose value is a [DimensionConstraints3D](../DimensionConstraints3D/DimensionConstraints3D.md).
