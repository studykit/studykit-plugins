# SectionDrawingView.IsFlatPatternView Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets whether the drawing view is of a sheet metal flat pattern.

## Syntax

SectionDrawingView.**IsFlatPatternView**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
