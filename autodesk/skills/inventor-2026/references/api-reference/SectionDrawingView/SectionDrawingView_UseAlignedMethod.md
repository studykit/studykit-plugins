# SectionDrawingView.UseAlignedMethod Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that gets and sets whether to use the projected or the aligned method.

## Syntax

SectionDrawingView.**UseAlignedMethod**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
