# SectionDrawingView.ViewType Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property returning an DrawingViewTypeEnum constant indicating the type of view.

## Syntax

SectionDrawingView.**ViewType**() As [DrawingViewTypeEnum](../DrawingViewTypeEnum.md)

## Property Value

This is a read only property whose value is a [DrawingViewTypeEnum](../DrawingViewTypeEnum.md).

## Version

Introduced in version 9
