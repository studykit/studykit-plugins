# SolidSweepDefinition.TwistAngle Property

Parent Object: [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md)

## Description

Read-write property that gets and sets the twist angle of sweep.

## Syntax

SolidSweepDefinition.**TwistAngle**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2020
