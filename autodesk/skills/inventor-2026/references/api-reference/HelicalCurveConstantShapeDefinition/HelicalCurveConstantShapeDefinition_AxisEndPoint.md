# HelicalCurveConstantShapeDefinition.AxisEndPoint Property

Parent Object: [HelicalCurveConstantShapeDefinition](../HelicalCurveConstantShapeDefinition/HelicalCurveConstantShapeDefinition.md)

## Description

Gets and sets the axis end point of the helical curve.

## Syntax

HelicalCurveConstantShapeDefinition.**AxisEndPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2019
