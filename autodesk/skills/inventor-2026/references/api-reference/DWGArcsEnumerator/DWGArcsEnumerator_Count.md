# DWGArcsEnumerator.Count Property

Parent Object: [DWGArcsEnumerator](../DWGArcsEnumerator/DWGArcsEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGArcsEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
