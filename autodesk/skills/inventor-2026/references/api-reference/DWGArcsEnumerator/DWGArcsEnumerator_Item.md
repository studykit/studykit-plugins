# DWGArcsEnumerator.Item Property

Parent Object: [DWGArcsEnumerator](../DWGArcsEnumerator/DWGArcsEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGArcsEnumerator.**Item**( ***Index*** As Variant ) As [DWGArc](../DWGArc/DWGArc.md)

## Property Value

This is a read only property whose value is a [DWGArc](../DWGArc/DWGArc.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
