# ImportedGenericComponentDefinition.ImportUnitsType Property

Parent Object: [ImportedGenericComponentDefinition](../ImportedGenericComponentDefinition/ImportedGenericComponentDefinition.md)

## Description

Read-write property that gets and sets the import units type.

## Syntax

ImportedGenericComponentDefinition.**ImportUnitsType**() As [ImportUnitsTypeEnum](../ImportUnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ImportUnitsTypeEnum](../ImportUnitsTypeEnum.md).

## Version

Introduced in version 2016
