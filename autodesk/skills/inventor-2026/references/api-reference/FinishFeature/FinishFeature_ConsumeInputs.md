# FinishFeature.ConsumeInputs Property

Parent Object: [FinishFeature](../FinishFeature/FinishFeature.md)

## Description

Gets and sets whether the inputs to this feature should be nested under this feature in the browser.

## Syntax

FinishFeature.**ConsumeInputs**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
