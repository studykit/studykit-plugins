# FinishFeature.Type Property

Parent Object: [FinishFeature](../FinishFeature/FinishFeature.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FinishFeature.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2024
