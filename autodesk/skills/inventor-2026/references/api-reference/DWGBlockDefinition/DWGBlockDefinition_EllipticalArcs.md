# DWGBlockDefinition.EllipticalArcs Property

Parent Object: [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Description

Read-only property that returns the DWGEllipticalArcsEnumerator collection object.

## Syntax

DWGBlockDefinition.**EllipticalArcs**() As [DWGEllipticalArcsEnumerator](../DWGEllipticalArcsEnumerator/DWGEllipticalArcsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEllipticalArcsEnumerator](../DWGEllipticalArcsEnumerator/DWGEllipticalArcsEnumerator.md).

## Version

Introduced in version 2016
