# DWGBlockDefinition.Polylines Property

Parent Object: [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Description

Read-only property that returns the DWGPolylinesEnumerator collection object.

## Syntax

DWGBlockDefinition.**Polylines**() As [DWGPolylinesEnumerator](../DWGPolylinesEnumerator/DWGPolylinesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGPolylinesEnumerator](../DWGPolylinesEnumerator/DWGPolylinesEnumerator.md).

## Version

Introduced in version 2016
