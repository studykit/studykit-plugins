# EdgeWidthExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The EdgeWidthExtent is used by sheet metal features when the width of the feature is defined by an edge.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../EdgeWidthExtent/EdgeWidthExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../EdgeWidthExtent/EdgeWidthExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Type](../EdgeWidthExtent/EdgeWidthExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2009
