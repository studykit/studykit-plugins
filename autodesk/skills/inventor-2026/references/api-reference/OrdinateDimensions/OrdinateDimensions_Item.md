# OrdinateDimensions.Item Property

Parent Object: [OrdinateDimensions](../OrdinateDimensions/OrdinateDimensions.md)

## Description

Method that returns the specified dimension object from the collection.

## Syntax

OrdinateDimensions.**Item**( ***Index*** As Long ) As [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Property Value

This is a read only property whose value is an [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2009
