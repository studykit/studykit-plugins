# FlatPunchResult.SketchRepresentationEdges Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns the edges in the flat pattern that belong to the punch result. The property will return Nothing if the punch tool feature that resulted in this punch result does not have a sketch-based alternate representation defined for it.

## Syntax

FlatPunchResult.**SketchRepresentationEdges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 2008
