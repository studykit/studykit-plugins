# UnfoldMethod.UnfoldMethodType Property

Parent Object: [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Description

Property that returns the type of unfold method used by this UnfoldMethod object. kLinearUnfoldMethod, kBendTableUnfoldMethod, and kCustomEquationUnfoldMethod are possible return values.

## Syntax

UnfoldMethod.**UnfoldMethodType**() As [UnfoldMethodTypeEnum](../UnfoldMethodTypeEnum.md)

## Property Value

This is a read only property whose value is a [UnfoldMethodTypeEnum](../UnfoldMethodTypeEnum.md).
