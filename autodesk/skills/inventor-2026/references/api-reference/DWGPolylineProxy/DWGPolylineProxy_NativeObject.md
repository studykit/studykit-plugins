# DWGPolylineProxy.NativeObject Property

Parent Object: [DWGPolylineProxy](../DWGPolylineProxy/DWGPolylineProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGPolylineProxy.**NativeObject**() As [DWGPolyline](../DWGPolyline/DWGPolyline.md)

## Property Value

This is a read only property whose value is a [DWGPolyline](../DWGPolyline/DWGPolyline.md).

## Version

Introduced in version 2016
