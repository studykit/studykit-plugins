# SketchOffsetSpline.LineWeight Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Gets and sets the LineWeight applied to this sketch spline.

## Syntax

SketchOffsetSpline.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
