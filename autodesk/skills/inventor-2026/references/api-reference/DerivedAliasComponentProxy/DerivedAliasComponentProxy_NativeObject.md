# DerivedAliasComponentProxy.NativeObject Property

Parent Object: [DerivedAliasComponentProxy](../DerivedAliasComponentProxy/DerivedAliasComponentProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DerivedAliasComponentProxy.**NativeObject**() As [DerivedAliasComponent](../DerivedAliasComponent/DerivedAliasComponent.md)

## Property Value

This is a read only property whose value is a [DerivedAliasComponent](../DerivedAliasComponent/DerivedAliasComponent.md).

## Version

Introduced in version 2010
