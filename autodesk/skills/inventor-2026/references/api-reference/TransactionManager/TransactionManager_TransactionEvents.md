# TransactionManager.TransactionEvents Property

Parent Object: [TransactionManager](../TransactionManager/TransactionManager.md)

## Description

Gets the object that will fire transaction events.

## Syntax

TransactionManager.**TransactionEvents**() As [TransactionEvents](../TransactionEvents/TransactionEvents.md)

## Property Value

This is a read only property whose value is a [TransactionEvents](../TransactionEvents/TransactionEvents.md).

## Version

Introduced in version 4
