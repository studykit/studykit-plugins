# ModelFeatureControlFrameRow.Tolerance Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the tolerance associated with this geometric characteristic. The value can be either in centimeters or radians depending on if the geometric characteristic defines a length or angle.

## Syntax

ModelFeatureControlFrameRow.**Tolerance**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
