# ModelFeatureControlFrameRow.ToleranceMaterialCondition Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the symbol to use as tolerance material condition. Valid values are kGDTNoSymbol, kGDTMaximumMaterialCondition and kGDTLeastMaterialCondition.

## Syntax

ModelFeatureControlFrameRow.**ToleranceMaterialCondition**() As [GDTSymbolEnum](../GDTSymbolEnum.md)

## Property Value

This is a read/write property whose value is a [GDTSymbolEnum](../GDTSymbolEnum.md).

## Version

Introduced in version 2018
