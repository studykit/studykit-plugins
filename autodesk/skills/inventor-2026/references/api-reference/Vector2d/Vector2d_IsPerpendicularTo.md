# Vector2d.IsPerpendicularTo Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Determine if this vector2d is perpendicular to the specified vector2d.

## Syntax

Vector2d.**IsPerpendicularTo**( ***Vector*** As [Vector2d](../Vector2d/Vector2d.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [Vector2d](../Vector2d/Vector2d.md) | Vector to use for perpendicular test. |
| Tolerance | Double | Tolerance value for test. |

## Version

Introduced in version 11
