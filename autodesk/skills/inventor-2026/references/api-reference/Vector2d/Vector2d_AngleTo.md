# Vector2d.AngleTo Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Determines the angle between this vector2d and the specified vector2d.

## Syntax

Vector2d.**AngleTo**( ***Vector*** As [Vector2d](../Vector2d/Vector2d.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [Vector2d](../Vector2d/Vector2d.md) | Vector to measure to. |

## Version

Introduced in version 11
