# DiameterDimConstraint.TextPoint Property

Parent Object: [DiameterDimConstraint](../DiameterDimConstraint/DiameterDimConstraint.md)

## Description

Gets and sets the position of the dimension text.

## Syntax

DiameterDimConstraint.**TextPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
