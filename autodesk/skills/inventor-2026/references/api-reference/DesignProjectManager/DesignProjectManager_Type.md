# DesignProjectManager.Type Property

Parent Object: [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DesignProjectManager.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
