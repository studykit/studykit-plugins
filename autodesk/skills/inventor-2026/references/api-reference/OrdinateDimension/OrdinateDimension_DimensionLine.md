# OrdinateDimension.DimensionLine Property

Parent Object: [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Description

Property that returns the dimension line geometry of the dimension.

## Syntax

OrdinateDimension.**DimensionLine**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2009
