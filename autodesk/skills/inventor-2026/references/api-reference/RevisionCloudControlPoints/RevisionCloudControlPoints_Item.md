# RevisionCloudControlPoints.Item Property

Parent Object: [RevisionCloudControlPoints](../RevisionCloudControlPoints/RevisionCloudControlPoints.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

RevisionCloudControlPoints.**Item**( ***Index*** As Long ) As [RevisionCloudControlPoint](../RevisionCloudControlPoint/RevisionCloudControlPoint.md)

## Property Value

This is a read only property whose value is a [RevisionCloudControlPoint](../RevisionCloudControlPoint/RevisionCloudControlPoint.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index number of the RevisionCloudControlPoint to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2024
