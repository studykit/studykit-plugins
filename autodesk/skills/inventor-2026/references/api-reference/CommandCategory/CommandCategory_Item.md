# CommandCategory.Item Property

Parent Object: [CommandCategory](../CommandCategory/CommandCategory.md)

## Description

Returns the specified object in the collection.

## Syntax

CommandCategory.**Item**( ***Index*** As Long ) As Object

## Property Value

This is a read only property whose value is an Object.

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index within the collection of the item to return. |

## Version

Introduced in version 9
