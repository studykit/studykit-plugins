# ThemeManager.ActiveTheme Property

Parent Object: [ThemeManager](../ThemeManager/ThemeManager.md)

## Description

Read-only property that returns the active theme.

## Syntax

ThemeManager.**ActiveTheme**() As [Theme](../Theme/Theme.md)

## Property Value

This is a read only property whose value is a [Theme](../Theme/Theme.md).

## Version

Introduced in version 2021
