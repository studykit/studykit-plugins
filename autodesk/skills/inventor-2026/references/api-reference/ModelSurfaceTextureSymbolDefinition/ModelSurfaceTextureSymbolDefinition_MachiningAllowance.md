# ModelSurfaceTextureSymbolDefinition.MachiningAllowance Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets the machining allowance. Setting this property will return an error if the SurfaceTextureType is not kMaterialRemovalRequiredSurfaceType. Denoted by E in the surface texture symbol dialog.

## Syntax

ModelSurfaceTextureSymbolDefinition.**MachiningAllowance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
