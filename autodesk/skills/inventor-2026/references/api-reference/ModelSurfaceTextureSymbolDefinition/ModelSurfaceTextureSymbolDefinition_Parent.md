# ModelSurfaceTextureSymbolDefinition.Parent Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelSurfaceTextureSymbolDefinition.**Parent**() As [ModelSurfaceTextureSymbol](../ModelSurfaceTextureSymbol/ModelSurfaceTextureSymbol.md)

## Property Value

This is a read only property whose value is a [ModelSurfaceTextureSymbol](../ModelSurfaceTextureSymbol/ModelSurfaceTextureSymbol.md).
