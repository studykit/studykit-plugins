# ModelSurfaceTextureSymbolDefinition.AnnotationPlane Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets the annotation plane for the symbol.

## Syntax

ModelSurfaceTextureSymbolDefinition.**AnnotationPlane**() As [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md).

## Version

Introduced in version 2018
