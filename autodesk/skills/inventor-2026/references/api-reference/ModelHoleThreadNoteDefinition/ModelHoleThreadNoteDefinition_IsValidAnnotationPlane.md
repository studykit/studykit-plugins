# ModelHoleThreadNoteDefinition.IsValidAnnotationPlane Method

Parent Object: [ModelHoleThreadNoteDefinition](../ModelHoleThreadNoteDefinition/ModelHoleThreadNoteDefinition.md)

## Description

Method that returns the wheather a planer object is valid to be used as an annotation plane for this hole or thread note.

## Syntax
