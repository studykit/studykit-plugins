# SurfaceBody.PreciseRangeBox Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Gets a tight fitting bounding box for this body.

## Syntax

SurfaceBody.**PreciseRangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2022
