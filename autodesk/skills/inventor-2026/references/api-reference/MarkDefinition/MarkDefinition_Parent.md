# MarkDefinition.Parent Property

Parent Object: [MarkDefinition](../MarkDefinition/MarkDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

MarkDefinition.**Parent**() As [MarkFeature](../MarkFeature/MarkFeature.md)

## Property Value

This is a read only property whose value is a [MarkFeature](../MarkFeature/MarkFeature.md).

## Version

Introduced in version 2023
