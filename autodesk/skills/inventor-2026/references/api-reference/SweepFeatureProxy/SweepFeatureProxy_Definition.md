# SweepFeatureProxy.Definition Property

Parent Object: [SweepFeatureProxy](../SweepFeatureProxy/SweepFeatureProxy.md)

## Description

Read-write property that gets and sets the SweepDefinition object associated with this sweep feature.

## Syntax

SweepFeatureProxy.**Definition**() As [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Property Value

This is a read/write property whose value is a [SweepDefinition](../SweepDefinition/SweepDefinition.md).

## Version

Introduced in version 11
