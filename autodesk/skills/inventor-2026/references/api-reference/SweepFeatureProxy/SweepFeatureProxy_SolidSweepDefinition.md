# SweepFeatureProxy.SolidSweepDefinition Property

Parent Object: [SweepFeatureProxy](../SweepFeatureProxy/SweepFeatureProxy.md)

## Description

Read-write property that gets and sets the SolidSweepDefinition object associated with this sweep feature.

## Syntax

SweepFeatureProxy.**SolidSweepDefinition**() As [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md)

## Property Value

This is a read/write property whose value is a [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md).

## Version

Introduced in version 2020
