# DWGBlockReferencesEnumerator.Item Property

Parent Object: [DWGBlockReferencesEnumerator](../DWGBlockReferencesEnumerator/DWGBlockReferencesEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGBlockReferencesEnumerator.**Item**( ***Index*** As Variant ) As [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md)

## Property Value

This is a read only property whose value is a [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
