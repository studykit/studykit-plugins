# DWGBlockReferencesEnumerator.Count Property

Parent Object: [DWGBlockReferencesEnumerator](../DWGBlockReferencesEnumerator/DWGBlockReferencesEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGBlockReferencesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
