# RipDefinition.Type Property

Parent Object: [RipDefinition](../RipDefinition/RipDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RipDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
