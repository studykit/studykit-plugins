# DrawingBOMColumns Object

## Description

The DrawingBOMColumns collection object represents the collection of all columns of a drawing BOM.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingBOMColumns/DrawingBOMColumns_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DrawingBOMColumns/DrawingBOMColumns_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DrawingBOMColumns/DrawingBOMColumns_Item.md) | Returns the specified DrawingBOMColumn object from the collection. |
| [Type](../DrawingBOMColumns/DrawingBOMColumns_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingBOM.DrawingBOMColumns](../DrawingBOM/DrawingBOM_DrawingBOMColumns.md)

## Version

Introduced in version 2009
