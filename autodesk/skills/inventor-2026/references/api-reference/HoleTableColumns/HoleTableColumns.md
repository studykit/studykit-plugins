# HoleTableColumns Object

## Description

The HoleTableColumns collection object provides access to the existing collection of HoleTableColumn objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../HoleTableColumns/HoleTableColumns_Add.md) | Method that creates a new HoleTableColumn based on a property. The newly created HoleTableColumn is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HoleTableColumns/HoleTableColumns_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../HoleTableColumns/HoleTableColumns_Count.md) | Property that returns the number of items in the collection. |
| [Item](../HoleTableColumns/HoleTableColumns_Item.md) | Returns the specified Column object from the collection. |
| [Type](../HoleTableColumns/HoleTableColumns_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[HoleTable.HoleTableColumns](../HoleTable/HoleTable_HoleTableColumns.md)

## Version

Introduced in version 10
