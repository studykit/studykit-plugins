# SketchCircularPatternDefinition.HasBoundarySettingConfigured Property

Parent Object: [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Description

Gets and sets whether this pattern has boundary setting configured or not.

## Syntax

SketchCircularPatternDefinition.**HasBoundarySettingConfigured**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
