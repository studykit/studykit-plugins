# ModelGeneralNotes.Add Method

Parent Object: [ModelGeneralNotes](../ModelGeneralNotes/ModelGeneralNotes.md)

## Description

Method that creates a general note.

## Syntax

ModelGeneralNotes.**Add**( ***Definition*** As [ModelGeneralNoteDefinition](../ModelGeneralNoteDefinition/ModelGeneralNoteDefinition.md) ) As [ModelGeneralNote](../ModelGeneralNote/ModelGeneralNote.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ModelGeneralNoteDefinition](../ModelGeneralNoteDefinition/ModelGeneralNoteDefinition.md) | Input ModelGeneralNoteDefinition object that defines the general note to be created. |

## Version

Introduced in version 2018
