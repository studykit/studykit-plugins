# ModelGeneralNotes.CreateEmbeddedFeatureControlFrameDefinition Method

Parent Object: [ModelGeneralNotes](../ModelGeneralNotes/ModelGeneralNotes.md)

## Description

Method that creates a ModelFeatureControlFrameDefinition. This ModelFeatureControlFrameDefinition is to define the model feature control frame symbol which is embedded in the model general note.

## Syntax

ModelGeneralNotes.**CreateEmbeddedFeatureControlFrameDefinition**() As [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Version

Introduced in version 2018
