# PartFeature.Adaptive Property

Parent Object: [PartFeature](../PartFeature/PartFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

PartFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
