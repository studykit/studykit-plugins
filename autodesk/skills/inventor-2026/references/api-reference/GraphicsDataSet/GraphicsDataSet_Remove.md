# GraphicsDataSet.Remove Method

Parent Object: [GraphicsDataSet](../GraphicsDataSet/GraphicsDataSet.md)

## Description

Method that removes a coordinate from the set.

## Syntax

GraphicsDataSet.**Remove**( ***Index*** As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Specifies the index you want to remove from the set. All coordinates above the Index will be shifted down. The coordinate set indices are one-based. |

## Version

Introduced in version 5
