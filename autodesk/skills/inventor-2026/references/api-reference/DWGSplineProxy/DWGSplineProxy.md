# DWGSplineProxy Object

Derived from: [DWGSpline](../DWGSpline/DWGSpline.md) Object

## Description

DWGSplineProxy Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DWGSplineProxy/DWGSplineProxy_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [ContainingOccurrence](../DWGSplineProxy/DWGSplineProxy_ContainingOccurrence.md) | Get the component occurrence context through which the native object is being seen through. |
| [EndPoint](../DWGSplineProxy/DWGSplineProxy_EndPoint.md) | Read-only property that returns the end point geometry. |
| [HandleID](../DWGSplineProxy/DWGSplineProxy_HandleID.md) | Read-only property that returns the HandleID for the DWGEntity. |
| [NativeObject](../DWGSplineProxy/DWGSplineProxy_NativeObject.md) | Get the object in the context of the definition instead of the containing assembly. |
| [Parent](../DWGSplineProxy/DWGSplineProxy_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ParentEntity](../DWGSplineProxy/DWGSplineProxy_ParentEntity.md) | Read-only property that returns the parent DWGEntity of the entity. |
| [StartPoint](../DWGSplineProxy/DWGSplineProxy_StartPoint.md) | Read-only property that returns the start point geometry. |
| [Type](../DWGSplineProxy/DWGSplineProxy_Type.md) | Gets the constant that indicates the type of this object. |
