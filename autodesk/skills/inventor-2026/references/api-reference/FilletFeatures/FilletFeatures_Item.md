# FilletFeatures.Item Property

Parent Object: [FilletFeatures](../FilletFeatures/FilletFeatures.md)

## Description

Returns the specified FilletFeature object from the collection.

## Syntax

FilletFeatures.**Item**( ***Index*** As Variant ) As [FilletFeature](../FilletFeature/FilletFeature.md)

## Property Value

This is a read only property whose value is a [FilletFeature](../FilletFeature/FilletFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the feature to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the feature name. If an out of range index or a name of a non-existent feature is provided, an error occurs. |

## Version

Introduced in version 5
