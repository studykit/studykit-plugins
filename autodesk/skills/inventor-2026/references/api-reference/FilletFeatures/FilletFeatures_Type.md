# FilletFeatures.Type Property

Parent Object: [FilletFeatures](../FilletFeatures/FilletFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FilletFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
