# SketchEquationCurves.Item Property

Parent Object: [SketchEquationCurves](../SketchEquationCurves/SketchEquationCurves.md)

## Description

Returns the specified SketchEquationCurve object from the collection

## Syntax

SketchEquationCurves.**Item**( ***Index*** As Long ) As [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Property Value

This is a read only property whose value is a [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. The first item in the collection has an index of 1. |

## Version

Introduced in version 2014
