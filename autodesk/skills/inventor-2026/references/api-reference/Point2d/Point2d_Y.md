# Point2d.Y Property

Parent Object: [Point2d](../Point2d/Point2d.md)

## Description

Specifies the Y coordinate of the point. If not supplied, the Y value will default to 0.

## Syntax

Point2d.**Y**() As Double

## Property Value

This is a read/write property whose value is a Double.
