# Point2d.VectorTo Method

Parent Object: [Point2d](../Point2d/Point2d.md)

## Description

Gets the vector of translation between this point and the specified point.

## Syntax

Point2d.**VectorTo**( ***Point*** As [Point2d](../Point2d/Point2d.md) ) As [Vector2d](../Vector2d/Vector2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point2d](../Point2d/Point2d.md) | Input Point2d object that specifies the point. |

## Version

Introduced in version 2008
