# Application.ChangeManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the ChangeManager object. The ChangeManager object manages the processes involved in making changes to data and recording the change process.

## Syntax

Application.**ChangeManager**() As [ChangeManager](../ChangeManager/ChangeManager.md)

## Property Value

This is a read only property whose value is a [ChangeManager](../ChangeManager/ChangeManager.md).

## Version

Introduced in version 9
