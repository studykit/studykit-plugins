# Application.GeneralOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the GeneralOptions object. The GeneralOptions object provides access to various application level options. This is somewhat equivalent to the General tab of the Application Options dialog.

## Syntax

Application.**GeneralOptions**() As [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Property Value

This is a read only property whose value is a [GeneralOptions](../GeneralOptions/GeneralOptions.md).

## Version

Introduced in version 8
