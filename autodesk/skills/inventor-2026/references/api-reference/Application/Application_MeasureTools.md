# Application.MeasureTools Property

Parent Object: [Application](../Application/Application.md)

## Description

Returns the MeasureTools object.

## Syntax

Application.**MeasureTools**() As [MeasureTools](../MeasureTools/MeasureTools.md)

## Property Value

This is a read only property whose value is a [MeasureTools](../MeasureTools/MeasureTools.md).
