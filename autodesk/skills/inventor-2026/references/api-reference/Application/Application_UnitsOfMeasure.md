# Application.UnitsOfMeasure Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the UnitsOfMeasure object.

## Syntax

Application.**UnitsOfMeasure**() As [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md)

## Property Value

This is a read only property whose value is a [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md).

## Version

Introduced in version 2009
