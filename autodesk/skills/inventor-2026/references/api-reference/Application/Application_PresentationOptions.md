# Application.PresentationOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the Presentation Options.

## Syntax

Application.**PresentationOptions**() As [PresentationOptions](../PresentationOptions/PresentationOptions.md)

## Property Value

This is a read only property whose value is a [PresentationOptions](../PresentationOptions/PresentationOptions.md).

## Version

Introduced in version 2011
