# Application.HardwareOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the Hardware Options.

## Syntax

Application.**HardwareOptions**() As [HardwareOptions](../HardwareOptions/HardwareOptions.md)

## Property Value

This is a read only property whose value is a [HardwareOptions](../HardwareOptions/HardwareOptions.md).

## Version

Introduced in version 11
