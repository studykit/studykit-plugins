# Application.HelpManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the HelpManager object that provides access to the help-related activity taking place in the system.

## Syntax

Application.**HelpManager**() As [HelpManager](../HelpManager/HelpManager.md)

## Property Value

This is a read only property whose value is a [HelpManager](../HelpManager/HelpManager.md).

## Version

Introduced in version 4
