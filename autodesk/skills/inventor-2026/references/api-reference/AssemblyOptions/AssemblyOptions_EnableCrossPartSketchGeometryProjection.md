# AssemblyOptions.EnableCrossPartSketchGeometryProjection Property

Parent Object: [AssemblyOptions](../AssemblyOptions/AssemblyOptions.md)

## Description

Enables or disables associative sketch geometry projection during in-place modeling.

## Syntax

AssemblyOptions.**EnableCrossPartSketchGeometryProjection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
