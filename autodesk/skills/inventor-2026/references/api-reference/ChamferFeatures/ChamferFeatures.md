# ChamferFeatures Object

## Description

The ChamferFeatures collection object provides access to all of the objects in a component definition and provides methods to create additional ChamferFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddUsingDistance](../ChamferFeatures/ChamferFeatures_AddUsingDistance.md) | Method that creates a new ChamferFeature defined by a specified distance. The new ChamferFeature is returned. |
| [AddUsingDistanceAndAngle](../ChamferFeatures/ChamferFeatures_AddUsingDistanceAndAngle.md) | Method that creates a new ChamferFeature that is defined by a distance and at a specified angle to an input face. The new ChamferFeature is returned. |
| [AddUsingTwoDistances](../ChamferFeatures/ChamferFeatures_AddUsingTwoDistances.md) | Method that creates a new ChamferFeature that is defined by two distances. The new ChamferFeature is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ChamferFeatures/ChamferFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ChamferFeatures/ChamferFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ChamferFeatures/ChamferFeatures_Item.md) | Method that returns the specified ChamferFeature object from the collection |
| [Type](../ChamferFeatures/ChamferFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
