# ImportedModelEntity.ParentEntity Property

Parent Object: [ImportedModelEntity](../ImportedModelEntity/ImportedModelEntity.md)

## Description

Read-only property that returns the parent ImportedModelEntity object.

## Syntax

ImportedModelEntity.**ParentEntity**() As [ImportedModelEntity](../ImportedModelEntity/ImportedModelEntity.md)

## Property Value

This is a read only property whose value is an [ImportedModelEntity](../ImportedModelEntity/ImportedModelEntity.md).

## Version

Introduced in version 2016
