# ImportedModelEntity.SubEntities Property

Parent Object: [ImportedModelEntity](../ImportedModelEntity/ImportedModelEntity.md)

## Description

Read-only property that returns the sub ImportedModelEntities collection.

## Syntax

ImportedModelEntity.**SubEntities**() As [ImportedModelEntities](../ImportedModelEntities/ImportedModelEntities.md)

## Property Value

This is a read only property whose value is an [ImportedModelEntities](../ImportedModelEntities/ImportedModelEntities.md).

## Version

Introduced in version 2016
