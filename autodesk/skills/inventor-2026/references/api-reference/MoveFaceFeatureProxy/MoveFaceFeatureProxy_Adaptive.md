# MoveFaceFeatureProxy.Adaptive Property

Parent Object: [MoveFaceFeatureProxy](../MoveFaceFeatureProxy/MoveFaceFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

MoveFaceFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
