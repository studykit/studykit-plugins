# MoveFaceFeatureProxy.Suppressed Property

Parent Object: [MoveFaceFeatureProxy](../MoveFaceFeatureProxy/MoveFaceFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

MoveFaceFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
