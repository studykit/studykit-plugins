# Balloon.BalloonValueSets Property

Parent Object: [Balloon](../Balloon/Balloon.md)

## Description

This property gets the collection of objects. See the BalloonValueSets and BalloonValueSet objects for more information.

## Syntax

Balloon.**BalloonValueSets**() As [BalloonValueSets](../BalloonValueSets/BalloonValueSets.md)

## Property Value

This is a read only property whose value is a [BalloonValueSets](../BalloonValueSets/BalloonValueSets.md).
