# LoftedFlangeDefinition.ProfileTwo Property

Parent Object: [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Description

Gets and sets the second of two profiles defining the lofted flange.

## Syntax

LoftedFlangeDefinition.**ProfileTwo**() As [Path](../Path/Path.md)

## Property Value

This is a read/write property whose value is a [Path](../Path/Path.md).

## Version

Introduced in version 2011
