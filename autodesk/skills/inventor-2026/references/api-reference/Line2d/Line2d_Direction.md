# Line2d.Direction Property

Parent Object: [Line2d](../Line2d/Line2d.md)

## Description

Specifies the direction of the line.

## Syntax

Line2d.**Direction**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read/write property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).

## Version

Introduced in version 4
