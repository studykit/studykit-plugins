# WorkSurfaceProxy.Parent Property

Parent Object: [WorkSurfaceProxy](../WorkSurfaceProxy/WorkSurfaceProxy.md)

## Description

Property that returns the parent  object from whom this object can logically be reached.

## Syntax

WorkSurfaceProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 6
