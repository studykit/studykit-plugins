# Lights.Item Property

Parent Object: [Lights](../Lights/Lights.md)

## Description

Returns the specified Light from the collection.

## Syntax

Lights.**Item**( ***Index*** As Variant ) As [Light](../Light/Light.md)

## Property Value

This is a read only property whose value is a [Light](../Light/Light.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Value that specifies the index number of the Light to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 10
