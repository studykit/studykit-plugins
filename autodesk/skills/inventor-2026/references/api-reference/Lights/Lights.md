# Lights Object

## Description

The Lights collection object allows for the creation of new Light objects and provides access to existing model space and view space lights.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Lights/Lights_Add.md) | Method that creates a new Light object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Lights/Lights_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Lights/Lights_Count.md) | Property that returns the number of Light objects in the collection. |
| [Item](../Lights/Lights_Item.md) | Returns the specified Light from the collection. |
| [Parent](../Lights/Lights_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../Lights/Lights_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[LightingStyle.Lights](../LightingStyle/LightingStyle_Lights.md)

## Version

Introduced in version 10
