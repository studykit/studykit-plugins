# MeshFeaturesVisibilityEnum Enumerator

## Description

Defines the visibility types of the Mesh Features .

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllMeshFeaturesVisible | 111105 | All mesh features are visiable. |
| kNoMeshFeaturesVisible | 111106 | No mesh features are visiable. |
| kSomeMeshFeaturesVisible | 111107 | Some mesh features are visiable. |

## Version

Introduced in version 2017
