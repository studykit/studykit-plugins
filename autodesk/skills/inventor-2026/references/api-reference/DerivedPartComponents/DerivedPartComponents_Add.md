# DerivedPartComponents.Add Method

Parent Object: [DerivedPartComponents](../DerivedPartComponents/DerivedPartComponents.md)

## Description

Method that creates a new DerivedPartDefinition using the information supplied by the input DerivedPartDefinition object. If successful, the new DerivedPartComponent is returned.

## Syntax

DerivedPartComponents.**Add**( ***Definition*** As [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md) ) As [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md) | Input object that defines the derived component. |
