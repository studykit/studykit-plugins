# DerivedPartComponents.CreateCoordinateSystemDef Method

Parent Object: [DerivedPartComponents](../DerivedPartComponents/DerivedPartComponents.md)

## Description

Method that creates a new DerivedPartCoordinateSystemDef object.

## Syntax

DerivedPartComponents.**CreateCoordinateSystemDef**( ***FullFileName*** As String ) As [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input string that specifies the name of the file to create the definition for. |

## Version

Introduced in version 6
