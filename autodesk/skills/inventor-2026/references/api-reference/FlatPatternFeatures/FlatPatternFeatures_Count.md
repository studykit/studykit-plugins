# FlatPatternFeatures.Count Property

Parent Object: [FlatPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures.md)

## Description

Property that returns the number of features in the flat pattern.

## Syntax

FlatPatternFeatures.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
