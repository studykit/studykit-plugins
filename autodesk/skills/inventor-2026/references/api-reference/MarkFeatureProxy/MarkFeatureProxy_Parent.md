# MarkFeatureProxy.Parent Property

Parent Object: [MarkFeatureProxy](../MarkFeatureProxy/MarkFeatureProxy.md)

## Description

Property that returns the parent of the feature.

## Syntax

MarkFeatureProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2023
