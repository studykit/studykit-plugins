# MarkFeatureProxy.Definition Property

Parent Object: [MarkFeatureProxy](../MarkFeatureProxy/MarkFeatureProxy.md)

## Description

Property that gets and sets the MarkDefinition object associated with this feature.

## Syntax

MarkFeatureProxy.**Definition**() As [MarkDefinition](../MarkDefinition/MarkDefinition.md)

## Property Value

This is a read/write property whose value is a [MarkDefinition](../MarkDefinition/MarkDefinition.md).

## Version

Introduced in version 2023
