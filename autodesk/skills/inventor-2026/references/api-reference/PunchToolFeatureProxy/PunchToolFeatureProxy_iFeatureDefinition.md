# PunchToolFeatureProxy.iFeatureDefinition Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Gets and sets the associated iFeatureDefinition.

## Syntax

PunchToolFeatureProxy.**iFeatureDefinition**() As [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md)

## Property Value

This is a read/write property whose value is an [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md).

## Version

Introduced in version 2009
