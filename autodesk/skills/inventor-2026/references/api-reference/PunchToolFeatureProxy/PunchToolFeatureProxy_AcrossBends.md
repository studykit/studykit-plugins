# PunchToolFeatureProxy.AcrossBends Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Gets and sets the property that defines the across bends.

## Syntax

PunchToolFeatureProxy.**AcrossBends**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
