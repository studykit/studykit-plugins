# PunchToolFeatureProxy.PunchRepresentationType Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Gets and sets punch tool feature representation type.

## Syntax

PunchToolFeatureProxy.**PunchRepresentationType**() As [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md).

## Version

Introduced in version 2017
