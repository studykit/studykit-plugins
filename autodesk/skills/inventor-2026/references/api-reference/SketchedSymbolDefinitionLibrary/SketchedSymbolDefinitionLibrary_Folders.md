# SketchedSymbolDefinitionLibrary.Folders Property

Parent Object: [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md)

## Description

Read-only property that returns the LibraryFolders collection object.

## Syntax

SketchedSymbolDefinitionLibrary.**Folders**() As [LibraryFolders](../LibraryFolders/LibraryFolders.md)

## Property Value

This is a read only property whose value is a [LibraryFolders](../LibraryFolders/LibraryFolders.md).

## Version

Introduced in version 2016
