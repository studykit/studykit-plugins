# UnitAttributes.PrimaryTrailingZerosDisplay Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets whether display the trailing zeros for primary values.

## Syntax

UnitAttributes.**PrimaryTrailingZerosDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
