# DSLoadDefinition.IsDirectionEntityNaturalDirection Property

Parent Object: [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Description

Gets and sets whether the direction of the load, when defined by an entity, is in the natural direction defined by the entity or reversed.

## Syntax

DSLoadDefinition.**IsDirectionEntityNaturalDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
