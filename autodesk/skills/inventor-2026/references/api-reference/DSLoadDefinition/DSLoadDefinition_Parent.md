# DSLoadDefinition.Parent Property

Parent Object: [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Description

Gets the parent DSLoad object this definition object is associated with. If this definition is not associated with a load then this property returns Nothing.

## Syntax

DSLoadDefinition.**Parent**() As [DSLoad](../DSLoad/DSLoad.md)

## Property Value

This is a read only property whose value is a [DSLoad](../DSLoad/DSLoad.md).

## Version

Introduced in version 2013
