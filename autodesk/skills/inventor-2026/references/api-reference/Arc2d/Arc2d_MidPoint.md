# Arc2d.MidPoint Property

Parent Object: [Arc2d](../Arc2d/Arc2d.md)

## Description

Read-only property that returns the mid point of the arc.

## Syntax

Arc2d.**MidPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2025
