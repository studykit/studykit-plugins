# DimensionConstraints3D.Item Property

Parent Object: [DimensionConstraints3D](../DimensionConstraints3D/DimensionConstraints3D.md)

## Description

Returns the specified 3D sketch dimension constraint object from the collection.

## Syntax

DimensionConstraints3D.**Item**( ***Index*** As Long ) As [DimensionConstraint3D](../DimensionConstraint3D/DimensionConstraint3D.md)

## Property Value

This is a read only property whose value is a [DimensionConstraint3D](../DimensionConstraint3D/DimensionConstraint3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Specifies the index of the object to return. |

## Version

Introduced in version 11
