# PunchToolFeature.PunchCenterPoints Property

Parent Object: [PunchToolFeature](../PunchToolFeature/PunchToolFeature.md)

## Description

Gets and sets the set of sketch points that define the origin points for the punch tool.

## Syntax

PunchToolFeature.**PunchCenterPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2009
