# PunchToolFeature.AppearanceSourceType Property

Parent Object: [PunchToolFeature](../PunchToolFeature/PunchToolFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

PunchToolFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
