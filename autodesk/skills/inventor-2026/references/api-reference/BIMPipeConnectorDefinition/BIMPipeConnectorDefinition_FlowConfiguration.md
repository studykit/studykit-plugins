# BIMPipeConnectorDefinition.FlowConfiguration Property

Parent Object: [BIMPipeConnectorDefinition](../BIMPipeConnectorDefinition/BIMPipeConnectorDefinition.md)

## Description

Read-write property that gets and sets the flow configuration used for this connector.

## Syntax

BIMPipeConnectorDefinition.**FlowConfiguration**() As [BIMPipeFlowConfigurationEnum](../BIMPipeFlowConfigurationEnum.md)

## Property Value

This is a read/write property whose value is a [BIMPipeFlowConfigurationEnum](../BIMPipeFlowConfigurationEnum.md).

## Version

Introduced in version 2011
