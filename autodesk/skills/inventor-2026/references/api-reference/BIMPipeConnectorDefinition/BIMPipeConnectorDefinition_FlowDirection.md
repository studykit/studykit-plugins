# BIMPipeConnectorDefinition.FlowDirection Property

Parent Object: [BIMPipeConnectorDefinition](../BIMPipeConnectorDefinition/BIMPipeConnectorDefinition.md)

## Description

Read-write property that gets and sets the flow direction for this pipe connector.

## Syntax

BIMPipeConnectorDefinition.**FlowDirection**() As [BIMFlowDirectionEnum](../BIMFlowDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [BIMFlowDirectionEnum](../BIMFlowDirectionEnum.md).

## Version

Introduced in version 2011
