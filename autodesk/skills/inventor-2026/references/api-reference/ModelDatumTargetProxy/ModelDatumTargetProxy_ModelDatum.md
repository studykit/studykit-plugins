# ModelDatumTargetProxy.ModelDatum Property

Parent Object: [ModelDatumTargetProxy](../ModelDatumTargetProxy/ModelDatumTargetProxy.md)

## Description

Returns the parent model datum object.

## Syntax

ModelDatumTargetProxy.**ModelDatum**() As [ModelDatum](../ModelDatum/ModelDatum.md)

## Property Value

This is a read only property whose value is a [ModelDatum](../ModelDatum/ModelDatum.md).

## Version

Introduced in version 2023
