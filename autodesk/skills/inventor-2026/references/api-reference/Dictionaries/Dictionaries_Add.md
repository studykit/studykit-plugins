# Dictionaries.Add Method

Parent Object: [Dictionaries](../Dictionaries/Dictionaries.md)

## Description

Method that creates a new Dictionary. The new created Dictionary is returned.

## Syntax

Dictionaries.**Add**( ***Name*** As String ) As [Dictionary](../Dictionary/Dictionary.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Name | String | Input String as the new Dictionary name. |

## Version

Introduced in version 2020
