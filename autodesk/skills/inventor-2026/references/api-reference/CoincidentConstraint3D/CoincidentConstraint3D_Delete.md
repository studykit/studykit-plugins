# CoincidentConstraint3D.Delete Method

Parent Object: [CoincidentConstraint3D](../CoincidentConstraint3D/CoincidentConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

CoincidentConstraint3D.**Delete**()

## Version

Introduced in version 6
