# Welds Object

## Description

The Welds object provides access to all of the existing welds within a weldment.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Edit](../Welds/Welds_Edit.md) | Method that activates this object for editing by the user. |
| [ExitEdit](../Welds/Welds_ExitEdit.md) | Method that causes the current editing environment to be closed and the user interface to return to weldment editing environment. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Welds/Welds_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [CosmeticWelds](../Welds/Welds_CosmeticWelds.md) | Property that returns the CosmeticWelds object. |
| [Count](../Welds/Welds_Count.md) | Property that returns the number of items in this collection. |
| [Item](../Welds/Welds_Item.md) | Returns the specified Weld object from the collection. |
| [Type](../Welds/Welds_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [WeldBeads](../Welds/Welds_WeldBeads.md) | Property that returns the WeldBeads object. |

## Accessed From

[WeldmentComponentDefinition.Welds](../WeldmentComponentDefinition/WeldmentComponentDefinition_Welds.md)

Introduced in version 8
