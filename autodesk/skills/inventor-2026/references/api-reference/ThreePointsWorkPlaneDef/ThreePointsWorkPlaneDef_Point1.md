# ThreePointsWorkPlaneDef.Point1 Property

Parent Object: [ThreePointsWorkPlaneDef](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef.md)

## Description

Property that returns the first point of a three point defined work plane. The first point defines the origin of the work plane. This object can be a WorkPoint, Vertex, or SketchPoint object.

## Syntax

ThreePointsWorkPlaneDef.**Point1**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
