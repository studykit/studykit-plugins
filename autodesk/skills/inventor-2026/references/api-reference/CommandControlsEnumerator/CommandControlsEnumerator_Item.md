# CommandControlsEnumerator.Item Property

Parent Object: [CommandControlsEnumerator](../CommandControlsEnumerator/CommandControlsEnumerator.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

CommandControlsEnumerator.**Item**( ***Index*** As Long ) As [CommandControl](../CommandControl/CommandControl.md)

## Property Value

This is a read only property whose value is a [CommandControl](../CommandControl/CommandControl.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long |  |

## Version

Introduced in version 2010
