# ClientFeatureProxy.Parent Property

Parent Object: [ClientFeatureProxy](../ClientFeatureProxy/ClientFeatureProxy.md)

## Description

Property that returns the parent of the feature.

## Syntax

ClientFeatureProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2008
