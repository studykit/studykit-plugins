# Transaction.State Property

Parent Object: [Transaction](../Transaction/Transaction.md)

## Description

Property that returns a TransactionStateEnum that describes the current state of the transaction.

## Syntax

Transaction.**State**() As [TransactionStateEnum](../TransactionStateEnum.md)

## Property Value

This is a read only property whose value is a [TransactionStateEnum](../TransactionStateEnum.md).

## Version

Introduced in version 7
