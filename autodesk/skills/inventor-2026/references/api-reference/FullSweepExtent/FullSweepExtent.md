# FullSweepExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The FullSweepExtent object suggests that a full, 360-degree sweep about an axis, is the definition of the extents on this feature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FullSweepExtent/FullSweepExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../FullSweepExtent/FullSweepExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Type](../FullSweepExtent/FullSweepExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 5
