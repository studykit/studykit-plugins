# ShrinkwrapDefinition.RemoveFilletsRadiusRange Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the radius of fillets in centimeters to remove for the shrinkwrap.

## Syntax

ShrinkwrapDefinition.**RemoveFilletsRadiusRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
