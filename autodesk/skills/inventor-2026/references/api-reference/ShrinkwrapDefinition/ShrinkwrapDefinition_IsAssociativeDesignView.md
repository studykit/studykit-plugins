# ShrinkwrapDefinition.IsAssociativeDesignView Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets if there is an associative link to the specified design view.

## Syntax

ShrinkwrapDefinition.**IsAssociativeDesignView**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
