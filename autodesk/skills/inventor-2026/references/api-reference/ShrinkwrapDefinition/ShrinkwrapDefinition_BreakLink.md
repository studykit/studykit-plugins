# ShrinkwrapDefinition.BreakLink Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets whether break link to the source assembly or not.

## Syntax

ShrinkwrapDefinition.**BreakLink**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
