# SketchEllipticalArcProxy.DisabledActionTypes Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets and sets whether specific user actions are blocked on this sketch geometry.

## Syntax

SketchEllipticalArcProxy.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 9
