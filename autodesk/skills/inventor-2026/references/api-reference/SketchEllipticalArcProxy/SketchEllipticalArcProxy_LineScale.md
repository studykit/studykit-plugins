# SketchEllipticalArcProxy.LineScale Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets and sets the LineScale applied to this elliptical arc.

## Syntax

SketchEllipticalArcProxy.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
