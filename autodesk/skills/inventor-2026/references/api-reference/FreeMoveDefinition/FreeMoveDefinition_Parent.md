# FreeMoveDefinition.Parent Property

Parent Object: [FreeMoveDefinition](../FreeMoveDefinition/FreeMoveDefinition.md)

## Description

Property that returns the parent MoveFaceDefinition object.

## Syntax

FreeMoveDefinition.**Parent**() As [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md)

## Property Value

This is a read only property whose value is a [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md).

## Version

Introduced in version 2011
