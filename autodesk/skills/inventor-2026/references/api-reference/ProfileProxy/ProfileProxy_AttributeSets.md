# ProfileProxy.AttributeSets Property

Parent Object: [ProfileProxy](../ProfileProxy/ProfileProxy.md)

## Description

Property that returns the AttributeSets collection object associated with this object.

## Syntax

ProfileProxy.**AttributeSets**() As [AttributeSets](../AttributeSets/AttributeSets.md)

## Property Value

This is a read only property whose value is an [AttributeSets](../AttributeSets/AttributeSets.md).

## Version

Introduced in version 6
