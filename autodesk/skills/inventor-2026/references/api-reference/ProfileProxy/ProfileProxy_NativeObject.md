# ProfileProxy.NativeObject Property

Parent Object: [ProfileProxy](../ProfileProxy/ProfileProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ProfileProxy.**NativeObject**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read only property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 6
