# ModelFeatureControlFrameDefinition.ArrowheadType Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Gets and sets the type of the arrowhead.

## Syntax

ModelFeatureControlFrameDefinition.**ArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2018
