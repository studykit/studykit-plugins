# ModelFeatureControlFrameDefinition.MergeSymbolOverridden Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Gets whether the OverrideMergeSymbol is set to override the MergeSymbol setting in FeatureControlFrameStyle. Set this to False will clear the override setting.

## Syntax

ModelFeatureControlFrameDefinition.**MergeSymbolOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
