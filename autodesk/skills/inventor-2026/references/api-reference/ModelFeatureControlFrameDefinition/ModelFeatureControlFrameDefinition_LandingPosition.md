# ModelFeatureControlFrameDefinition.LandingPosition Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Gets and sets the landing position of the note in the part or assembly.

## Syntax

ModelFeatureControlFrameDefinition.**LandingPosition**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
