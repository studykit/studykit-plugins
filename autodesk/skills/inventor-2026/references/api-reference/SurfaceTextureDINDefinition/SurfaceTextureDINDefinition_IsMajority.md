# SurfaceTextureDINDefinition.IsMajority Property

Parent Object: [SurfaceTextureDINDefinition](../SurfaceTextureDINDefinition/SurfaceTextureDINDefinition.md)

## Description

Gets and sets whether this symbol specifies the standard surface characteristics for the drawing.

## Syntax

SurfaceTextureDINDefinition.**IsMajority**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
