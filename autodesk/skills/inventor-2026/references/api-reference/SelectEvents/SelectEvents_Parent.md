# SelectEvents.Parent Property

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SelectEvents.**Parent**() As [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Property Value

This is a read only property whose value is an [InteractionEvents](../InteractionEvents/InteractionEvents.md).

## Version

Introduced in version 5
