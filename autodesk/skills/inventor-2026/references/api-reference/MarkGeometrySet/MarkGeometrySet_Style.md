# MarkGeometrySet.Style Property

Parent Object: [MarkGeometrySet](../MarkGeometrySet/MarkGeometrySet.md)

## Description

Property that gets and sets the mark Style.

## Syntax

MarkGeometrySet.**Style**() As [MarkStyle](../MarkStyle/MarkStyle.md)

## Property Value

This is a read/write property whose value is a [MarkStyle](../MarkStyle/MarkStyle.md).

## Version

Introduced in version 2023
