# PluginLicenseManagerEvents.Parent Property

Parent Object: [PluginLicenseManagerEvents](../PluginLicenseManagerEvents/PluginLicenseManagerEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PluginLicenseManagerEvents.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.
