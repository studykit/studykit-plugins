# DerivedAssemblyComponent.Parent Property

Parent Object: [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Description

Property that returns the parent object.

## Syntax

DerivedAssemblyComponent.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 5.3
