# DerivedAssemblyComponent.IsEmbedded Property

Parent Object: [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Description

Property that returns whether the reference component refers to an embedded document or a linked document.

## Syntax

DerivedAssemblyComponent.**IsEmbedded**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
