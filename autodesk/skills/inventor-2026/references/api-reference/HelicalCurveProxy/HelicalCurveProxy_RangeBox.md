# HelicalCurveProxy.RangeBox Property

Parent Object: [HelicalCurveProxy](../HelicalCurveProxy/HelicalCurveProxy.md)

## Description

Property that returns a Box object which contains the opposing points of a rectangular box that is guaranteed to enclose this object.

## Syntax

HelicalCurveProxy.**RangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2019
