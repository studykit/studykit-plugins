# iFeatureParameterInput.IsPunchToolDepth Property

Parent Object: [iFeatureParameterInput](../iFeatureParameterInput/iFeatureParameterInput.md)

## Description

Property specifies if this parameter input is used to define the custom depth of a punch tool.

## Syntax

iFeatureParameterInput.**IsPunchToolDepth**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
