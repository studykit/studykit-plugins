# SketchDrivenPatternFeatures.Type Property

## Syntax

SketchDrivenPatternFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
