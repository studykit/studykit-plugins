# iMateResultsEnumerator.Type Property

Parent Object: [iMateResultsEnumerator](../iMateResultsEnumerator/iMateResultsEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iMateResultsEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version
