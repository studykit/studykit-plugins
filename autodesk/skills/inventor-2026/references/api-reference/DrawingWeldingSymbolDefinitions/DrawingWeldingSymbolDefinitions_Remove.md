# DrawingWeldingSymbolDefinitions.Remove Method

Parent Object: [DrawingWeldingSymbolDefinitions](../DrawingWeldingSymbolDefinitions/DrawingWeldingSymbolDefinitions.md)

## Syntax

DrawingWeldingSymbolDefinitions.**Remove**( ***Index*** As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value indicating the index of DrawingWeldingSymbolDefinition to return. |
