# AutoCADBlocks.Type Property

Parent Object: [AutoCADBlocks](../AutoCADBlocks/AutoCADBlocks.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

AutoCADBlocks.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
