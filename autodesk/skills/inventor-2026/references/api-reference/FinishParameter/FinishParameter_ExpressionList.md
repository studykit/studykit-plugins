# FinishParameter.ExpressionList Property

Parent Object: [FinishParameter](../FinishParameter/FinishParameter.md)

## Description

Property that returns the expression list associated with this parameter.

## Syntax

FinishParameter.**ExpressionList**() As [ExpressionList](../ExpressionList/ExpressionList.md)

## Property Value

This is a read only property whose value is an [ExpressionList](../ExpressionList/ExpressionList.md).

## Version

Introduced in version 2024
