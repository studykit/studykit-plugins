# AssemblyJointDefinition.OriginTwoXOffset Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-only property that returns the offset of the OriginTwo along origin X-axis direction. This returns Empty if the OriginTwoDefinitonType is not equal to kOffsetOriginDefinitionType.

## Syntax

AssemblyJointDefinition.**OriginTwoXOffset**() As Variant

## Property Value

This is a read only property whose value is a Variant.

## Version

Introduced in version 2016
