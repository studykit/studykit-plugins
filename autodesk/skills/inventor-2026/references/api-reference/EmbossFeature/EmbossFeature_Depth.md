# EmbossFeature.Depth Property

Parent Object: [EmbossFeature](../EmbossFeature/EmbossFeature.md)

## Description

Property returns the parameter controlling the depth of the emboss feature.

## Syntax

EmbossFeature.**Depth**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2008
