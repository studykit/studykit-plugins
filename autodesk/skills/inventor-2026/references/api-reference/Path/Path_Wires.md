# Path.Wires Property

Parent Object: [Path](../Path/Path.md)

## Description

Property returning the Wires collection object associated with this Path.

## Syntax

Path.**Wires**() As [Wires](../Wires/Wires.md)

## Property Value

This is a read only property whose value is a [Wires](../Wires/Wires.md).

## Version

Introduced in version 2009
