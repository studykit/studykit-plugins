# DrawingWeldingSymbolDefinition.FieldWeldingSymbol Property

Parent Object: [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md)

## Description

Gets and sets whether to add a flag indicating a field or site weld to the selected reference line or not.

## Syntax

DrawingWeldingSymbolDefinition.**FieldWeldingSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
