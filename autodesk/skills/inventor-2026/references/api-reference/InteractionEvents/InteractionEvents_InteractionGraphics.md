# InteractionEvents.InteractionGraphics Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the Interaction Graphics object.

## Syntax

InteractionEvents.**InteractionGraphics**() As [InteractionGraphics](../InteractionGraphics/InteractionGraphics.md)

## Property Value

This is a read only property whose value is an [InteractionGraphics](../InteractionGraphics/InteractionGraphics.md).
