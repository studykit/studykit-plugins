# InteractionEvents.KeyboardEvents Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the Keyboard Events object.

## Syntax

InteractionEvents.**KeyboardEvents**() As [KeyboardEvents](../KeyboardEvents/KeyboardEvents.md)

## Property Value

This is a read only property whose value is a [KeyboardEvents](../KeyboardEvents/KeyboardEvents.md).

## Version

Introduced in version 5
