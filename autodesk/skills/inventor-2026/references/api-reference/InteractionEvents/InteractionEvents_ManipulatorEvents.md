# InteractionEvents.ManipulatorEvents Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the ManipulatorEvents object.

## Syntax

InteractionEvents.**ManipulatorEvents**() As [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md)

## Property Value

This is a read only property whose value is a [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md).

## Version

Introduced in version 2017
