# TangentiMateDefinition.ReferencedEntity Property

Parent Object: [TangentiMateDefinition](../TangentiMateDefinition/TangentiMateDefinition.md)

## Description

Gets the object this entity is dependent on.

## Syntax

TangentiMateDefinition.**ReferencedEntity**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version

Introduced in version 2009
