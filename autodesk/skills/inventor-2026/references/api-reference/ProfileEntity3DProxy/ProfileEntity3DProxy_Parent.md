# ProfileEntity3DProxy.Parent Property

Parent Object: [ProfileEntity3DProxy](../ProfileEntity3DProxy/ProfileEntity3DProxy.md)

## Description

Property that returns the parent ProfilePath3D of the entity.

## Syntax

ProfileEntity3DProxy.**Parent**() As [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Property Value

This is a read only property whose value is a [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md).

## Version

Introduced in version 6
