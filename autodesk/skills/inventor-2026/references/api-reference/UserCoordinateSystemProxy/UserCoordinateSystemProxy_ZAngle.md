# UserCoordinateSystemProxy.ZAngle Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Property that returns the parameter associated with the rotation angle about the z-axis.

## Syntax

UserCoordinateSystemProxy.**ZAngle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
