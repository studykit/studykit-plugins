# TorusMidPlaneWorkPlaneDef.Parent Property

Parent Object: [TorusMidPlaneWorkPlaneDef](../TorusMidPlaneWorkPlaneDef/TorusMidPlaneWorkPlaneDef.md)

## Description

Property that returns the parent WorkPlane object.

## Syntax

TorusMidPlaneWorkPlaneDef.**Parent**() As [WorkPlane](../WorkPlane/WorkPlane.md)

## Property Value

This is a read only property whose value is a [WorkPlane](../WorkPlane/WorkPlane.md).

## Version

Introduced in version 2008
