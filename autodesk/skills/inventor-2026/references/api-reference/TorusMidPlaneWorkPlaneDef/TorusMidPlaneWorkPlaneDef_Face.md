# TorusMidPlaneWorkPlaneDef.Face Property

Parent Object: [TorusMidPlaneWorkPlaneDef](../TorusMidPlaneWorkPlaneDef/TorusMidPlaneWorkPlaneDef.md)

## Description

Property that indicates the Face object whose geometry is a torus surface, the mid-plane of which defines the position of the work plane.

## Syntax

TorusMidPlaneWorkPlaneDef.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2008
