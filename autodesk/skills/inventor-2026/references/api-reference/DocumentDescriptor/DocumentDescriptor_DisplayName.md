# DocumentDescriptor.DisplayName Property

Parent Object: [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Description

Property that returns the display name of the referenced document as currently found (or the last known display name, if the reference is not found).

## Syntax

DocumentDescriptor.**DisplayName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 11
