# ArcLengthDimConstraint.DimensionCenterPoint Property

Parent Object: [ArcLengthDimConstraint](../ArcLengthDimConstraint/ArcLengthDimConstraint.md)

## Description

Gets the center of the dimension line.

## Syntax

ArcLengthDimConstraint.**DimensionCenterPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2013
