# DirectEditOperations Object

## Description

Part DirectEdit Operations Collection Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DirectEditOperations/DirectEditOperations_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../DirectEditOperations/DirectEditOperations_Count.md) | Gets the number of items in this collection. |
| [Item](../DirectEditOperations/DirectEditOperations_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../DirectEditOperations/DirectEditOperations_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[DirectEditFeature.DirectEditOperations](../DirectEditFeature/DirectEditFeature_DirectEditOperations.md), [DirectEditFeatureProxy.DirectEditOperations](../DirectEditFeatureProxy/DirectEditFeatureProxy_DirectEditOperations.md)
