# FileManager.GetRevitFileVersionCreated Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Method that returns the Revit version that the Revit file was created with.

## Syntax

FileManager.**GetRevitFileVersionCreated**( ***FullFileName*** As String ) As String

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input String value that specifies the full filename of a Revit file. Valid file extensions include:  * RVT - Revit Project * RTE - Revit Project Template * RFA - Revit Family File * RFT - Revit Family Templates |

## Version

Introduced in version 2026
