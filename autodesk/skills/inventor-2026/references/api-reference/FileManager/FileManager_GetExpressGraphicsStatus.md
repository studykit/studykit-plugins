# FileManager.GetExpressGraphicsStatus Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Returns a value indicating the current state of the express graphics in the specified assembly.

## Syntax

FileManager.**GetExpressGraphicsStatus**( ***AssemblyFullFilename*** As String ) As [CachedGraphicsStatusEnum](CachedGraphicsStatusEnum.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| AssemblyFullFilename | String | Input String that specifies the assembly full filename. |

## Version

Introduced in version 2014
