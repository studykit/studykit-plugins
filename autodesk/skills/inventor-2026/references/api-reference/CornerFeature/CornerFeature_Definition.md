# CornerFeature.Definition Property

Parent Object: [CornerFeature](../CornerFeature/CornerFeature.md)

## Description

Property that gets and sets the CornerDefinition object associated with this corner feature.

## Syntax

CornerFeature.**Definition**() As [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Property Value

This is a read only property whose value is a [CornerDefinition](../CornerDefinition/CornerDefinition.md).

## Version

Introduced in version 2009
