# ProgressiveToolTip.IsProgressive Property

Parent Object: [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md)

## Description

Gets and sets whether the second level content (ExpandedDescription and Image) is displayed after a delay.

## Syntax

ProgressiveToolTip.**IsProgressive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
