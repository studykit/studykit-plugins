# SketchCircularPattern.Definition Property

Parent Object: [SketchCircularPattern](../SketchCircularPattern/SketchCircularPattern.md)

## Description

Gets and sets the associative definition of the sketch patter.

## Syntax

SketchCircularPattern.**Definition**() As [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Property Value

This is a read/write property whose value is a [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md).

## Version

Introduced in version 2025.1
