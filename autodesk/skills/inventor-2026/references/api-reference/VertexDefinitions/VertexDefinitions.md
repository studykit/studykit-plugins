# VertexDefinitions Object

## Description

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../VertexDefinitions/VertexDefinitions_Add.md) | Method that creates a new VertexDefinition within the associated SurfaceBodyDefinition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../VertexDefinitions/VertexDefinitions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../VertexDefinitions/VertexDefinitions_Count.md) | Property that returns the number of items in this collection. |
| [Item](../VertexDefinitions/VertexDefinitions_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../VertexDefinitions/VertexDefinitions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
