# ShellDefinition.AffectedBodies Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Read-write property that specifies the solid bodies to be hollowed out.

## Syntax

ShellDefinition.**AffectedBodies**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2026
