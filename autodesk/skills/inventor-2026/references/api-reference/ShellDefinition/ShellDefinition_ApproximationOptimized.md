# ShellDefinition.ApproximationOptimized Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Gets and sets whether the approximation is optimized.

## Syntax

ShellDefinition.**ApproximationOptimized**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
