# ProjectAssetLibrary.Parent Property

Parent Object: [ProjectAssetLibrary](../ProjectAssetLibrary/ProjectAssetLibrary.md)

## Description

Read-only property that returns the parent DesignProject object.

## Syntax

ProjectAssetLibrary.**Parent**() As [DesignProject](../DesignProject/DesignProject.md)

## Property Value

This is a read only property whose value is a [DesignProject](../DesignProject/DesignProject.md).

## Version

Introduced in version 2014
