# ReferencedOLEFileDescriptors.Item Property

Parent Object: [ReferencedOLEFileDescriptors](../ReferencedOLEFileDescriptors/ReferencedOLEFileDescriptors.md)

## Description

Returns the specified ReferencedOLEFileDescriptor object from the collection. This is the default property of the ReferencedOLEFileDescriptors collection object.

## Syntax

ReferencedOLEFileDescriptors.**Item**( ***Index*** As Long ) As [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Property Value

This is a read only property whose value is a [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the ReferencedOLEFileDescriptor to return. |

## Version

Introduced in version 4
