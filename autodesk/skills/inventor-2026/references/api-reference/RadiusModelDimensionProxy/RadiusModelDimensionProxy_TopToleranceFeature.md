# RadiusModelDimensionProxy.TopToleranceFeature Property

Parent Object: [RadiusModelDimensionProxy](../RadiusModelDimensionProxy/RadiusModelDimensionProxy.md)

## Description

Returns the top ModelToleranceFeature object.

## Syntax

RadiusModelDimensionProxy.**TopToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2019
