# DirectionAndDistanceMoveDefinition.Distance Property

Parent Object: [DirectionAndDistanceMoveDefinition](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition.md)

## Description

Property that returns the distance Parameter object associated with the move.

## Syntax

DirectionAndDistanceMoveDefinition.**Distance**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2011
