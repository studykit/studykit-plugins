# FaceDefinition.EdgeLoopDefinitions Property

Parent Object: [FaceDefinition](../FaceDefinition/FaceDefinition.md)

## Description

Property that returns the collection of EdgeLoopDefinition objects associated with this FaceDefinition object.

## Syntax

FaceDefinition.**EdgeLoopDefinitions**() As [EdgeLoopDefinitions](../EdgeLoopDefinitions/EdgeLoopDefinitions.md)

## Property Value

This is a read only property whose value is an [EdgeLoopDefinitions](../EdgeLoopDefinitions/EdgeLoopDefinitions.md).
