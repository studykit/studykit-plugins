# TextBox.LineSpacingType Property

Parent Object: [TextBox](../TextBox/TextBox.md)

## Description

Gets and sets the method used to define the line spacing.

## Syntax

TextBox.**LineSpacingType**() As [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md)

## Property Value

This is a read/write property whose value is a [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md).

## Version

Introduced in version 5.3
