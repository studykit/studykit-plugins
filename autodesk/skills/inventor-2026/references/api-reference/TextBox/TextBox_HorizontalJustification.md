# TextBox.HorizontalJustification Property

Parent Object: [TextBox](../TextBox/TextBox.md)

## Description

Gets and sets the horizontal justification override of the text box.

## Syntax

TextBox.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).
