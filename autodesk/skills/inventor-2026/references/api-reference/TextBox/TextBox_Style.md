# TextBox.Style Property

Parent Object: [TextBox](../TextBox/TextBox.md)

## Description

Gets/Sets the text style associated with the text box.

## Syntax

TextBox.**Style**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).
