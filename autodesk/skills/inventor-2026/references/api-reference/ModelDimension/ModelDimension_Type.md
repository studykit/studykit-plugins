# ModelDimension.Type Property

Parent Object: [ModelDimension](../ModelDimension/ModelDimension.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ModelDimension.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2018
