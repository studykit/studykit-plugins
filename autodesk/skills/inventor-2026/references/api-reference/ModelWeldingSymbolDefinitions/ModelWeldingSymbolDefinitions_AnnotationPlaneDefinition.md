# ModelWeldingSymbolDefinitions.AnnotationPlaneDefinition Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Gets and sets the annotation plane definition for the symbol.

## Syntax

ModelWeldingSymbolDefinitions.**AnnotationPlaneDefinition**() As [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md).

## Version

Introduced in version 2024
