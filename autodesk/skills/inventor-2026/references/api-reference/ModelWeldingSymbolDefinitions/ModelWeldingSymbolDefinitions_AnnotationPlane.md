# ModelWeldingSymbolDefinitions.AnnotationPlane Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Gets and sets the annotation plane for the welding symbol.

## Syntax

ModelWeldingSymbolDefinitions.**AnnotationPlane**() As [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md).

## Version

Introduced in version 2024
