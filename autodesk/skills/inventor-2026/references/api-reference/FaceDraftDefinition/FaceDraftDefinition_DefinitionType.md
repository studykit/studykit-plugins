# FaceDraftDefinition.DefinitionType Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-only property that returns the FaceDraftDefinitionTypeEnum indicating which face draft type the definition is.

## Syntax

FaceDraftDefinition.**DefinitionType**() As [FaceDraftDefinitionTypeEnum](../FaceDraftDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [FaceDraftDefinitionTypeEnum](../FaceDraftDefinitionTypeEnum.md).

## Version

Introduced in version 2016
