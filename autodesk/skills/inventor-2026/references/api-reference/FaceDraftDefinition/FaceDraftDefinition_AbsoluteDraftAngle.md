# FaceDraftDefinition.AbsoluteDraftAngle Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-write property that gets and sets whether the draft angle of the face draft is absolute against the pull direction.

## Syntax

FaceDraftDefinition.**AbsoluteDraftAngle**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
