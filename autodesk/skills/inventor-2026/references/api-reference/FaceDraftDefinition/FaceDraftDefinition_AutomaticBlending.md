# FaceDraftDefinition.AutomaticBlending Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-write property that gets and sets whether include the blended features in the face draft feature.

## Syntax

FaceDraftDefinition.**AutomaticBlending**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
