# FaceDraftDefinition.Parent Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

FaceDraftDefinition.**Parent**() As [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md)

## Property Value

This is a read only property whose value is a [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md).

## Version

Introduced in version 2016
