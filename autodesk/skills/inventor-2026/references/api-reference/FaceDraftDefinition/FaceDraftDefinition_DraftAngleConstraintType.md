# FaceDraftDefinition.DraftAngleConstraintType Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-write property that gets and sets the draft angle constraint type for the face draft.

## Syntax

FaceDraftDefinition.**DraftAngleConstraintType**() As [DraftAngleConstraintTypeEnum](../DraftAngleConstraintTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DraftAngleConstraintTypeEnum](../DraftAngleConstraintTypeEnum.md).

## Version

Introduced in version 2016
