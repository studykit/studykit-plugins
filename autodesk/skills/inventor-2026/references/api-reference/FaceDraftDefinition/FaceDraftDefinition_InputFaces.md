# FaceDraftDefinition.InputFaces Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-write property that specifies the faces used to create the feature.

## Syntax

FaceDraftDefinition.**InputFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2016
