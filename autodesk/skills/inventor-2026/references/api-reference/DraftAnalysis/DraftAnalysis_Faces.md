# DraftAnalysis.Faces Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Gets/Sets the faces for the draft analysis.

## Syntax

DraftAnalysis.**Faces**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2009
