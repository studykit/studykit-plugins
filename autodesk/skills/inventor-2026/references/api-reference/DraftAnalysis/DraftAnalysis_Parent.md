# DraftAnalysis.Parent Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Property that returns the parent AnalysisManager object.

## Syntax

DraftAnalysis.**Parent**() As [AnalysisManager](../AnalysisManager/AnalysisManager.md)

## Property Value

This is a read only property whose value is an [AnalysisManager](../AnalysisManager/AnalysisManager.md).

## Version

Introduced in version 2009
