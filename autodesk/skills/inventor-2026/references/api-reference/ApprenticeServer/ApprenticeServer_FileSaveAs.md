# ApprenticeServer.FileSaveAs Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Gets the object that provides access to the helper object that provides the Save As facility.

## Syntax

ApprenticeServer.**FileSaveAs**() As [FileSaveAs](../FileSaveAs/FileSaveAs.md)

## Property Value

This is a read only property whose value is a [FileSaveAs](../FileSaveAs/FileSaveAs.md).

## Version

Introduced in version 4
