# ApprenticeServer.TransientGeometry Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Gets the object through which all transient geometry objects can be constructed.

## Syntax

ApprenticeServer.**TransientGeometry**() As [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Property Value

This is a read only property whose value is a [TransientGeometry](../TransientGeometry/TransientGeometry.md).

## Version

Introduced in version 4
