# ApprenticeServer.TransientObjects Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Property that returns the TransientObjects object.

## Syntax

ApprenticeServer.**TransientObjects**() As [TransientObjects](../TransientObjects/TransientObjects.md)

## Property Value

This is a read only property whose value is a [TransientObjects](../TransientObjects/TransientObjects.md).

## Version

Introduced in version 2008
