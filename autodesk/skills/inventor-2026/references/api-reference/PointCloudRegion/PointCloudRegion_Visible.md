# PointCloudRegion.Visible Property

Parent Object: [PointCloudRegion](../PointCloudRegion/PointCloudRegion.md)

## Description

Read-write property that gets and sets the visibility of the point cloud region object.

## Syntax

PointCloudRegion.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
