# FilletFeature.ExtendedName Property

Parent Object: [FilletFeature](../FilletFeature/FilletFeature.md)

## Description

Read-only property that returns the full feature name including any extended information.

## Syntax

FilletFeature.**ExtendedName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
