# EnvironmentList.Add Method

Parent Object: [EnvironmentList](../EnvironmentList/EnvironmentList.md)

## Description

Method that adds an Environment object to the list. The method returns an error if a built-in Environment is supplied.

## Syntax

EnvironmentList.**Add**( ***Environment*** As [Environment](../Environment/Environment.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Environment | [Environment](../Environment/Environment.md) | Input Environment object to add to the list. |
