# FileDialog.FileDialogEvents Property

Parent Object: [FileDialog](../FileDialog/FileDialog.md)

## Description

Property that returns the FileDialogEvents object.

## Syntax

FileDialog.**FileDialogEvents**() As [FileDialogEvents](../FileDialogEvents/FileDialogEvents.md)

## Property Value

This is a read only property whose value is a [FileDialogEvents](../FileDialogEvents/FileDialogEvents.md).

## Version

Introduced in version 2008
