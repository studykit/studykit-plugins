# ReferenceComponent.Type Property

Parent Object: [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ReferenceComponent.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5.3
