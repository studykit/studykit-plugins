# SketchFixedSplines Object

## Description

SketchFixedSplines object provides access to all the objects in a sketch. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchFixedSplines/SketchFixedSplines_Add.md) | Method that creates a fixed spline based on an input NURBS definition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchFixedSplines/SketchFixedSplines_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchFixedSplines/SketchFixedSplines_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchFixedSplines/SketchFixedSplines_Item.md) | Method that returns the specified SketchFixedSpline object from the collection. |
| [Type](../SketchFixedSplines/SketchFixedSplines_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
