# PartsListCell Object

## Description

The PartsListCell object represents an individual cell in the parts list table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PartsListCell/PartsListCell_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../PartsListCell/PartsListCell_Parent.md) | Property that gets the parent object from whom this object can logically be reached. |
| [Static](../PartsListCell/PartsListCell_Static.md) | Gets whether the value of the cell is static. |
| [Type](../PartsListCell/PartsListCell_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Value](../PartsListCell/PartsListCell_Value.md) | Gets or sets the value of the cell in the parts list table. |
