# CustomTable.MergedCells Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Returns all MergedCell objects in a table.

## Syntax

CustomTable.**MergedCells**() As [MergedCellsEnumerator](../MergedCellsEnumerator/MergedCellsEnumerator.md)

## Property Value

This is a read only property whose value is a [MergedCellsEnumerator](../MergedCellsEnumerator/MergedCellsEnumerator.md).

## Version

Introduced in version 2025.1
