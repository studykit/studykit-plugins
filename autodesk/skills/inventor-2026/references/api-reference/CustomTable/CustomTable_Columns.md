# CustomTable.Columns Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Property that returns the Columns collection object.

## Syntax

CustomTable.**Columns**() As [Columns](../Columns/Columns.md)

## Property Value

This is a read only property whose value is a [Columns](../Columns/Columns.md).

## Version

Introduced in version 9
