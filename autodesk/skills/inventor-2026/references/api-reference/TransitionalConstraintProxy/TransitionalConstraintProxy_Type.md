# TransitionalConstraintProxy.Type Property

Parent Object: [TransitionalConstraintProxy](../TransitionalConstraintProxy/TransitionalConstraintProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TransitionalConstraintProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
