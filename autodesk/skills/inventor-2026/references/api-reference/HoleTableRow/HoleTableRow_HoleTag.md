# HoleTableRow.HoleTag Property

Parent Object: [HoleTableRow](../HoleTableRow/HoleTableRow.md)

## Description

Property that returns the hole tag associated with this hole in the table.

## Syntax

HoleTableRow.**HoleTag**() As [HoleTag](../HoleTag/HoleTag.md)

## Property Value

This is a read only property whose value is a [HoleTag](../HoleTag/HoleTag.md).

## Version

Introduced in version 10
