# DerivedAliasComponents Object

## Description

The DerivedAliasComponents collection object provides access to all of the existing DerivedAliasComponent objects in a part and provides methods to create additional derived components.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DerivedAliasComponents/DerivedAliasComponents_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DerivedAliasComponents/DerivedAliasComponents_Count.md) | Property that returns the number of items in this collection. |
| [Item](../DerivedAliasComponents/DerivedAliasComponents_Item.md) | Method that returns the specified DerivedAliasComponent object from the collection. |
| [Parent](../DerivedAliasComponents/DerivedAliasComponents_Parent.md) | Property that returns the ComponentDefinition object. |
| [Type](../DerivedAliasComponents/DerivedAliasComponents_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ReferenceComponents.DerivedAliasComponents](../ReferenceComponents/ReferenceComponents_DerivedAliasComponents.md)

## Version

Introduced in version 2010
