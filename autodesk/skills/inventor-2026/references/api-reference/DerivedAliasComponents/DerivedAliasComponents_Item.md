# DerivedAliasComponents.Item Property

Parent Object: [DerivedAliasComponents](../DerivedAliasComponents/DerivedAliasComponents.md)

## Description

Method that returns the specified DerivedAliasComponent object from the collection.

## Syntax

DerivedAliasComponents.**Item**( ***Value*** As Variant ) As [DerivedAliasComponent](../DerivedAliasComponent/DerivedAliasComponent.md)

## Property Value

This is a read only property whose value is a [DerivedAliasComponent](../DerivedAliasComponent/DerivedAliasComponent.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | Variant | Input Variant value that specifies the DeriveAliasStudioComponent to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the derived component's name. If an out of range index or a name of a non-existent DerivedAssemblyComponent is provided, an error occurs. |

## Version

Introduced in version 2010
