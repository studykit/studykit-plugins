# SurfaceGraphicsVertex.Parent Property

Parent Object: [SurfaceGraphicsVertex](../SurfaceGraphicsVertex/SurfaceGraphicsVertex.md)

## Description

Read-only property that returns the parent SurfaceGraphics object.

## Syntax

SurfaceGraphicsVertex.**Parent**() As [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Property Value

This is a read only property whose value is a [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md).

## Version

Introduced in version 2013
