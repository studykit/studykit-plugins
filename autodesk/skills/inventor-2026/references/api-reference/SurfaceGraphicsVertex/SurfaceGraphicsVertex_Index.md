# SurfaceGraphicsVertex.Index Property

Parent Object: [SurfaceGraphicsVertex](../SurfaceGraphicsVertex/SurfaceGraphicsVertex.md)

## Description

Gets the index of the SurfaceGraphicsVertex object within the SurfaceGraphicsVertexList.

## Syntax

SurfaceGraphicsVertex.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2013
