# RectangularPatternFeatureProxy.SurfacesVisible Property

Parent Object: [RectangularPatternFeatureProxy](../RectangularPatternFeatureProxy/RectangularPatternFeatureProxy.md)

## Description

Gets and sets whether all the resulting surfaces are visible or not.

## Syntax

RectangularPatternFeatureProxy.**SurfacesVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
