# Camera.ModelToViewSpace Method

Parent Object: [Camera](../Camera/Camera.md)

## Description

Method that converts a point in model space to the equivalent point on the view.

## Syntax

Camera.**ModelToViewSpace**( ***ModelCoordinate*** As [Point](../Point/Point.md) ) As [Point2d](../Point2d/Point2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ModelCoordinate | [Point](../Point/Point.md) | Input model space point to convert to view space. |

## Version

Introduced in version 6
