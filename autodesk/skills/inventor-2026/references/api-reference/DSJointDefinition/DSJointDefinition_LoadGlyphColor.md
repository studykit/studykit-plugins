# DSJointDefinition.LoadGlyphColor Property

Parent Object: [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md)

## Description

Gets and sets the color of the load glyph as seen during the playback of the simulation.

## Syntax

DSJointDefinition.**LoadGlyphColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2013
