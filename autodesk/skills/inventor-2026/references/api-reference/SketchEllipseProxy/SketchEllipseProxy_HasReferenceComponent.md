# SketchEllipseProxy.HasReferenceComponent Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchEllipseProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
