# SketchEllipseProxy.MinorRadius Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Gets and sets the minor radius.

## Syntax

SketchEllipseProxy.**MinorRadius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 6
