# RibbonTabs.Type Property

Parent Object: [RibbonTabs](../RibbonTabs/RibbonTabs.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RibbonTabs.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
