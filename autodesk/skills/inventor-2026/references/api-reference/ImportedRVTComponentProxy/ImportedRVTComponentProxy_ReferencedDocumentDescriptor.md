# ImportedRVTComponentProxy.ReferencedDocumentDescriptor Property

Parent Object: [ImportedRVTComponentProxy](../ImportedRVTComponentProxy/ImportedRVTComponentProxy.md)

## Description

Property that returns the DocumentDescriptor object.

## Syntax

ImportedRVTComponentProxy.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 2021
