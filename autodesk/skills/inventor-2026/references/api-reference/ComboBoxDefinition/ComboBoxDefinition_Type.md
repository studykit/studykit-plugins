# ComboBoxDefinition.Type Property

Parent Object: [ComboBoxDefinition](../ComboBoxDefinition/ComboBoxDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ComboBoxDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
