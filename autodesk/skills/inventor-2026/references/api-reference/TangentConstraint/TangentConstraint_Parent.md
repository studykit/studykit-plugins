# TangentConstraint.Parent Property

Parent Object: [TangentConstraint](../TangentConstraint/TangentConstraint.md)

## Description

Property that returns the parent of the object.

## Syntax

TangentConstraint.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).

## Version

Introduced in version 4
