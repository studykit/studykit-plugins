# EmbossFeatureProxy.EmbossType Property

Parent Object: [EmbossFeatureProxy](../EmbossFeatureProxy/EmbossFeatureProxy.md)

## Description

Property that returns the type of emboss feature.

## Syntax

EmbossFeatureProxy.**EmbossType**() As [EmbossTypeEnum](../EmbossTypeEnum.md)

## Property Value

This is a read only property whose value is an [EmbossTypeEnum](../EmbossTypeEnum.md).

## Version

Introduced in version 2008
