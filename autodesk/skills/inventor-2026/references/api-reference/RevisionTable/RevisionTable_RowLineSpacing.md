# RevisionTable.RowLineSpacing Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets the spacing between the rows.

## Syntax

RevisionTable.**RowLineSpacing**() As [LineSpacingEnum](../LineSpacingEnum.md)

## Property Value

This is a read/write property whose value is a [LineSpacingEnum](../LineSpacingEnum.md).

## Version

Introduced in version 2012
