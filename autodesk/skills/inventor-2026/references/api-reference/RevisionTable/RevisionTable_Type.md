# RevisionTable.Type Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RevisionTable.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
