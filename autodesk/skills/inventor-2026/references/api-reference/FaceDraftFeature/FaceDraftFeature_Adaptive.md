# FaceDraftFeature.Adaptive Property

Parent Object: [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

FaceDraftFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
