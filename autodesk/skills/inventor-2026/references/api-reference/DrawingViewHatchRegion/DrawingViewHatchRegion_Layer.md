# DrawingViewHatchRegion.Layer Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that gets and sets the layer associated with the drawing view hatch region object.

## Syntax

DrawingViewHatchRegion.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2022
