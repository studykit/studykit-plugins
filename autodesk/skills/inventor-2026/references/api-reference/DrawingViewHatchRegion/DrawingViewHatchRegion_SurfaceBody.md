# DrawingViewHatchRegion.SurfaceBody Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-only property that returns the SurfaceBody or SurfaceBodyProxy that associated with the DrawingViewHatchRegion.

## Syntax

DrawingViewHatchRegion.**SurfaceBody**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2022
