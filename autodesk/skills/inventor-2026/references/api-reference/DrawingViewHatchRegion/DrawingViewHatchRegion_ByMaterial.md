# DrawingViewHatchRegion.ByMaterial Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that specifies if use the hatch pattern corresponds to the mapping between materials and hatch patterns defined in the active drawing standard style.

## Syntax

DrawingViewHatchRegion.**ByMaterial**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
