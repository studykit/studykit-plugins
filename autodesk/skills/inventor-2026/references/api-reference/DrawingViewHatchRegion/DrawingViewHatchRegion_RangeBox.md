# DrawingViewHatchRegion.RangeBox Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-only property that returns the range box of the DrawingViewHatchRegion.

## Syntax

DrawingViewHatchRegion.**RangeBox**() As [Box2d](../Box2d/Box2d.md)

## Property Value

This is a read only property whose value is a [Box2d](../Box2d/Box2d.md).

## Version

Introduced in version 2022
