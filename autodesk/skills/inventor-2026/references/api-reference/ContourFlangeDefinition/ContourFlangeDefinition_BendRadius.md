# ContourFlangeDefinition.BendRadius Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Gets and sets the bend radius of a contour flange feature.

## Syntax

ContourFlangeDefinition.**BendRadius**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2009
