# GeneralOptions.StartupNewFileTemplateName Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Gets and sets the name of the template for the new file that will be automatically created each time Autodesk Inventor is opened.

## Syntax

GeneralOptions.**StartupNewFileTemplateName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
