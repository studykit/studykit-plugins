# WebBrowserDialog.WindowState Property

Parent Object: [WebBrowserDialog](../WebBrowserDialog/WebBrowserDialog.md)

## Description

Read-write property that gets and sets the window state of this web browser dialog.

## Syntax

WebBrowserDialog.**WindowState**() As [WindowsSizeEnum](../WindowsSizeEnum.md)

## Property Value

This is a read/write property whose value is a [WindowsSizeEnum](../WindowsSizeEnum.md).

## Version

Introduced in version 2017
