# ImportedDataExchangeComponent.Parent Property

Parent Object: [ImportedDataExchangeComponent](../ImportedDataExchangeComponent/ImportedDataExchangeComponent.md)

## Description

Property that returns the parent object.

## Syntax

ImportedDataExchangeComponent.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2023
