# ImportedDataExchangeComponent.Definition Property

Parent Object: [ImportedDataExchangeComponent](../ImportedDataExchangeComponent/ImportedDataExchangeComponent.md)

## Description

Read-write property that gets and sets the ImportedComponentDefinition which defines the various inputs that were used to create the imported component.

## Syntax

ImportedDataExchangeComponent.**Definition**() As [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md)

## Property Value

This is a read/write property whose value is an [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md).

## Version

Introduced in version 2023
