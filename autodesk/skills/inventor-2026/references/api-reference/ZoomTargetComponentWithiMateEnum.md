# ZoomTargetComponentWithiMateEnum Enumerator

## Description

Constants signifying the state of a link.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kZoomTargetAll | 55811 | Zooms the assembly so that all elements in the model fit in the graphics window. |
| kZoomTargetNone | 55809 | No zoom is performed. |
| kZoomTargetPlacedComponent | 55810 | Zooms in on the placed part so that it fills the graphics window. |

## Version

Introduced in version 11
