# SymmetryConstraintProxy.Type Property

Parent Object: [SymmetryConstraintProxy](../SymmetryConstraintProxy/SymmetryConstraintProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SymmetryConstraintProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
