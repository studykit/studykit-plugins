# Edge.Evaluator Property

Parent Object: [Edge](../Edge/Edge.md)

## Description

Gets the CurveEvaluator for this Edge.

## Syntax

Edge.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 4
