# WorkPoints.Parent Property

Parent Object: [WorkPoints](../WorkPoints/WorkPoints.md)

## Description

Property returning the parent  object.

## Syntax

WorkPoints.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).
