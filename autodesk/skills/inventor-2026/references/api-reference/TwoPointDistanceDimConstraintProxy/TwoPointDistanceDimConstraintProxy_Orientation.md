# TwoPointDistanceDimConstraintProxy.Orientation Property

Parent Object: [TwoPointDistanceDimConstraintProxy](../TwoPointDistanceDimConstraintProxy/TwoPointDistanceDimConstraintProxy.md)

## Description

Property that indicates an enum indicating if the constraint is controlling the horizontal, vertical, or true distance between the points.

## Syntax

TwoPointDistanceDimConstraintProxy.**Orientation**() As [DimensionOrientationEnum](../DimensionOrientationEnum.md)

## Property Value

This is a read only property whose value is a [DimensionOrientationEnum](../DimensionOrientationEnum.md).

## Version

Introduced in version 6
