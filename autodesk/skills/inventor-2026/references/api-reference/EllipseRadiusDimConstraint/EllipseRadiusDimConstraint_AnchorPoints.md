# EllipseRadiusDimConstraint.AnchorPoints Property

Parent Object: [EllipseRadiusDimConstraint](../EllipseRadiusDimConstraint/EllipseRadiusDimConstraint.md)

## Description

Gets the anchor points of dimension.

## Syntax

EllipseRadiusDimConstraint.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
