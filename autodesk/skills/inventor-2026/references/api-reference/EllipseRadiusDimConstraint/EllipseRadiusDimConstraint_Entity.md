# EllipseRadiusDimConstraint.Entity Property

Parent Object: [EllipseRadiusDimConstraint](../EllipseRadiusDimConstraint/EllipseRadiusDimConstraint.md)

## Description

Property that returns the ellipse or elliptical arc being constrained.

## Syntax

EllipseRadiusDimConstraint.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
