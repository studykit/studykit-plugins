# RadiusGeneralDimension.OverrideModelValue Property

Parent Object: [RadiusGeneralDimension](../RadiusGeneralDimension/RadiusGeneralDimension.md)

## Description

Gets and sets the NominalValue setting.

## Syntax

RadiusGeneralDimension.**OverrideModelValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
