# RadiusGeneralDimension.Precision Property

Parent Object: [RadiusGeneralDimension](../RadiusGeneralDimension/RadiusGeneralDimension.md)

## Description

Gets and sets the Precision setting.

## Syntax

RadiusGeneralDimension.**Precision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 11
