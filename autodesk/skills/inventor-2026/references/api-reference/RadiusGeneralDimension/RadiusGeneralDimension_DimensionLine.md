# RadiusGeneralDimension.DimensionLine Property

Parent Object: [RadiusGeneralDimension](../RadiusGeneralDimension/RadiusGeneralDimension.md)

## Description

Property that returns the dimension line geometry of the dimension.

## Syntax

RadiusGeneralDimension.**DimensionLine**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2008
