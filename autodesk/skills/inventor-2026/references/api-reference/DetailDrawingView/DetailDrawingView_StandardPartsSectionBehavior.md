# DetailDrawingView.StandardPartsSectionBehavior Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets and sets the sectioning behavior for standard parts in drawing views of assemblies.

## Syntax

DetailDrawingView.**StandardPartsSectionBehavior**() As [StandardPartsSectionBehaviorEnum](../StandardPartsSectionBehaviorEnum.md)

## Property Value

This is a read/write property whose value is a [StandardPartsSectionBehaviorEnum](../StandardPartsSectionBehaviorEnum.md).

## Version

Introduced in version 2010
