# DetailDrawingView.ViewJustification Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets and sets the view justification.

## Syntax

DetailDrawingView.**ViewJustification**() As [ViewJustificationEnum](../ViewJustificationEnum.md)

## Property Value

This is a read/write property whose value is a [ViewJustificationEnum](../ViewJustificationEnum.md).

## Version

Introduced in version 2010
