# DetailDrawingView.HatchRegions Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-only property that returns the DrawingViewHatchRegions collection object.

## Syntax

DetailDrawingView.**HatchRegions**() As [DrawingViewHatchRegions](../DrawingViewHatchRegions/DrawingViewHatchRegions.md)

## Property Value

This is a read only property whose value is a [DrawingViewHatchRegions](../DrawingViewHatchRegions/DrawingViewHatchRegions.md).

## Version

Introduced in version 2022
