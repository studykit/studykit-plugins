# DetailDrawingView.CircularFence Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets whether the detail view uses a circular fence or a rectangular fence.

## Syntax

DetailDrawingView.**CircularFence**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
