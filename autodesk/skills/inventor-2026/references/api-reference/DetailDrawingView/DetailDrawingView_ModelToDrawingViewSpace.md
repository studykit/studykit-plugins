# DetailDrawingView.ModelToDrawingViewSpace Method

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Method that takes a 3d coordinate in model space, and returns a Point2d object containing the coordinate point in drawing view space.

## Syntax

DetailDrawingView.**ModelToDrawingViewSpace**( ***ModelCoordinate*** As [Point](../Point/Point.md) ) As [Point2d](../Point2d/Point2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ModelCoordinate | [Point](../Point/Point.md) | Input Point object of a point within model space. |

## Version

Introduced in version 11
