# DetailDrawingView.Include3DAnnotations Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that gets and sets whether to include the 3D annotations.

## Syntax

DetailDrawingView.**Include3DAnnotations**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
