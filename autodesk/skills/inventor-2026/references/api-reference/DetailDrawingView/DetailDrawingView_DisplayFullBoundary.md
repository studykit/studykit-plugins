# DetailDrawingView.DisplayFullBoundary Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that gets and sets whether to display full boundary of the detail view. This property can be set to True only if the IsBreakLineSmooth property is set to True.

## Syntax

DetailDrawingView.**DisplayFullBoundary**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
