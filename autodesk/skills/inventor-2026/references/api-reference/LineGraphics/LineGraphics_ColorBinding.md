# LineGraphics.ColorBinding Property

Parent Object: [LineGraphics](../LineGraphics/LineGraphics.md)

## Description

Gets and sets how colors are defined for the line.

## Syntax

LineGraphics.**ColorBinding**() As [ColorBindingEnum](../ColorBindingEnum.md)

## Property Value

This is a read/write property whose value is a [ColorBindingEnum](../ColorBindingEnum.md).

## Version

Introduced in version 5
