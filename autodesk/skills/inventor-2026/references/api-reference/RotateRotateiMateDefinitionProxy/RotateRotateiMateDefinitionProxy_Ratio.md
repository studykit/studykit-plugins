# RotateRotateiMateDefinitionProxy.Ratio Property

Parent Object: [RotateRotateiMateDefinitionProxy](../RotateRotateiMateDefinitionProxy/RotateRotateiMateDefinitionProxy.md)

## Description

Property that returns the Parameter object that controls the rotation ratio of the iMate definition.

## Syntax

RotateRotateiMateDefinitionProxy.**Ratio**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
