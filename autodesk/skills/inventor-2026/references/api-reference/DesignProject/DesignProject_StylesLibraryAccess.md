# DesignProject.StylesLibraryAccess Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets/Sets the status of the styles library access for the Design Project.

## Syntax

DesignProject.**StylesLibraryAccess**() As [StylesLibraryAccessEnum](../StylesLibraryAccessEnum.md)

## Property Value

This is a read/write property whose value is a [StylesLibraryAccessEnum](../StylesLibraryAccessEnum.md).

## Version

Introduced in version 10
