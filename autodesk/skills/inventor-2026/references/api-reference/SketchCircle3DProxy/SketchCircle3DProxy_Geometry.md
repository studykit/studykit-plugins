# SketchCircle3DProxy.Geometry Property

Parent Object: [SketchCircle3DProxy](../SketchCircle3DProxy/SketchCircle3DProxy.md)

## Description

Gets and sets a Circle geometry object. The object returned represents a 'snapshot' view of the current state of the sketch circle.

## Syntax

SketchCircle3DProxy.**Geometry**() As [Circle](../Circle/Circle.md)

## Property Value

This is a read/write property whose value is a [Circle](../Circle/Circle.md).

## Version

Introduced in version 6
