# SketchControlPointSpline.Length Property

Parent Object: [SketchControlPointSpline](../SketchControlPointSpline/SketchControlPointSpline.md)

## Description

Returns the length of the entity in centimeters.

## Syntax

SketchControlPointSpline.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2014
