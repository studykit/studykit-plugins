# SketchControlPointSpline.ConvertToSpline Method

Parent Object: [SketchControlPointSpline](../SketchControlPointSpline/SketchControlPointSpline.md)

## Description

Method that inserts a new control point into the existing control point spline.

## Syntax

SketchControlPointSpline.**ConvertToSpline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Version

Introduced in version 2014
