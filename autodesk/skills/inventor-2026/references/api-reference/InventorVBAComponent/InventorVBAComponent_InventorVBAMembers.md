# InventorVBAComponent.InventorVBAMembers Property

Parent Object: [InventorVBAComponent](../InventorVBAComponent/InventorVBAComponent.md)

## Description

Property that returns the InventorVBAMembers collection.

## Syntax

InventorVBAComponent.**InventorVBAMembers**() As [InventorVBAMembers](../InventorVBAMembers/InventorVBAMembers.md)

## Property Value

This is a read only property whose value is an [InventorVBAMembers](../InventorVBAMembers/InventorVBAMembers.md).

## Version

Introduced in version 6
