# FlushConstraint.EntityTwo Property

Parent Object: [FlushConstraint](../FlushConstraint/FlushConstraint.md)

## Description

Property that indicates the second of the geometric entities (Face, Axis, Edge, etc.) being constrained.

## Syntax

FlushConstraint.**EntityTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
