# MiniToolbarControls.AddSeparator Method

Parent Object: [MiniToolbarControls](../MiniToolbarControls/MiniToolbarControls.md)

## Description

Method that creates a new separator within the MiniToolbar.

## Syntax

MiniToolbarControls.**AddSeparator**() As [MiniToolbarControl](../MiniToolbarControl/MiniToolbarControl.md)

## Version

Introduced in version 2012
