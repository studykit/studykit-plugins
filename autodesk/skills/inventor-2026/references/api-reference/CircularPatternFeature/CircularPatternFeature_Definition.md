# CircularPatternFeature.Definition Property

Parent Object: [CircularPatternFeature](../CircularPatternFeature/CircularPatternFeature.md)

## Description

Read-write property that gets and sets the CircularPatternFeatureDefinition object associated with this feature.

## Syntax

CircularPatternFeature.**Definition**() As [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Property Value

This is a read/write property whose value is a [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md).

## Version

Introduced in version 2017
