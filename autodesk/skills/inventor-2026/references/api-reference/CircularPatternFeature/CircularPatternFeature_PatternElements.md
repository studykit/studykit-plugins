# CircularPatternFeature.PatternElements Property

Parent Object: [CircularPatternFeature](../CircularPatternFeature/CircularPatternFeature.md)

## Description

Property that returns the collection that contains the elements created by this pattern.

## Syntax

CircularPatternFeature.**PatternElements**() As [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md)

## Property Value

This is a read only property whose value is a [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md).

## Version

Introduced in version 5
