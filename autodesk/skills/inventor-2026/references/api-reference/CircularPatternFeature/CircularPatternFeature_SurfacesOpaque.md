# CircularPatternFeature.SurfacesOpaque Property

Parent Object: [CircularPatternFeature](../CircularPatternFeature/CircularPatternFeature.md)

## Description

Gets and sets whether all the resulting surfaces are opaque or transparent.

## Syntax

CircularPatternFeature.**SurfacesOpaque**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
