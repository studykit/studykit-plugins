# SketchFixedSpline3DProxy.Closed Property

Parent Object: [SketchFixedSpline3DProxy](../SketchFixedSpline3DProxy/SketchFixedSpline3DProxy.md)

## Description

Boolean property that gets whether the curve is closed. A periodic curve is closed and curvature continuous at the closure.

## Syntax

SketchFixedSpline3DProxy.**Closed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
