# DrawingViewHatchArea.Parent Property

Parent Object: [DrawingViewHatchArea](../DrawingViewHatchArea/DrawingViewHatchArea.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DrawingViewHatchArea.**Parent**() As [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Property Value

This is a read only property whose value is a [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md).

## Version

Introduced in version 2022
