# BalloonValueSets Object

## Description

The BalloonValueSets object represents a collection of objects. Note that this collection does not yet support an Add method. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../BalloonValueSets/BalloonValueSets_Add.md) | Method that creates a new BalloonValueSet. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BalloonValueSets/BalloonValueSets_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BalloonValueSets/BalloonValueSets_Count.md) | Property that returns the number of items in the collection. |
| [Item](../BalloonValueSets/BalloonValueSets_Item.md) | Returns the specified object from the collection. |
| [Type](../BalloonValueSets/BalloonValueSets_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Balloon.BalloonValueSets](../Balloon/Balloon_BalloonValueSets.md)

## Version

Introduced in version 9
