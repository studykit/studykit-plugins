# DrawingWeldingSymbol.Definitions Property

Parent Object: [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Description

Gets and sets the DrawingWeldingSymbolDefinitions object associated with this symbol.

## Syntax

DrawingWeldingSymbol.**Definitions**() As [DrawingWeldingSymbolDefinitions](../DrawingWeldingSymbolDefinitions/DrawingWeldingSymbolDefinitions.md)

## Property Value

This is a read/write property whose value is a [DrawingWeldingSymbolDefinitions](../DrawingWeldingSymbolDefinitions/DrawingWeldingSymbolDefinitions.md).

## Version

Introduced in version 2024
