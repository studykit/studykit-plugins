# UserInputEvents.OnDeleteKeyUp Event

Parent Object: [UserInputEvents](../UserInputEvents/UserInputEvents.md)

## Description

Fires when the user performs a delete key click when no command is active.

## Syntax

UserInputEvents.**OnDeleteKeyUp**( ***SelectedEntities*** As [ObjectsEnumerator](../ObjectsEnumerator/ObjectsEnumerator.md), ***SelectionDevice*** As [SelectionDeviceEnum](../SelectionDeviceEnum.md), ***View*** As [View](../View/View.md), ***AdditionalInfo*** As [NameValueMap](../NameValueMap/NameValueMap.md), ***HandlingCode*** As [HandlingCodeEnum](../HandlingCodeEnum.md) )
