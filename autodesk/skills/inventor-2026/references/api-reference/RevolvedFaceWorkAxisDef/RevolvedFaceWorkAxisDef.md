# RevolvedFaceWorkAxisDef Object

## Description

Object that allows you to get and set the information that specifies a work axis at the axis of a revolved face.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevolvedFaceWorkAxisDef/RevolvedFaceWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Face](../RevolvedFaceWorkAxisDef/RevolvedFaceWorkAxisDef_Face.md) | Property that returns the used to define the revolved face work axis. This object must be a Face object whose geometry is a revolved surface. Valid surface types include cylinders, cones, and toroids. |
| [Parent](../RevolvedFaceWorkAxisDef/RevolvedFaceWorkAxisDef_Parent.md) | Property returning the parent WorkAxis object. |
| [Type](../RevolvedFaceWorkAxisDef/RevolvedFaceWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
