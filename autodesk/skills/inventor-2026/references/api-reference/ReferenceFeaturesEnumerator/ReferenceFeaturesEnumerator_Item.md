# ReferenceFeaturesEnumerator.Item Property

Parent Object: [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md)

## Description

Returns the specified ReferenceFeature object from the collection.

## Syntax

ReferenceFeaturesEnumerator.**Item**( ***Index*** As Long ) As [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md)

## Property Value

This is a read only property whose value is a [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the index of the ReferenceFeature to return. |

## Version

Introduced in version 5.3
