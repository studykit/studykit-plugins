# ThreadFeatureProxy.Parent Property

Parent Object: [ThreadFeatureProxy](../ThreadFeatureProxy/ThreadFeatureProxy.md)

## Description

Property that returns the parent of the feature.

## Syntax

ThreadFeatureProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 9
