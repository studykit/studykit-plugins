# RotateRotateConstraint.Visible Property

Parent Object: [RotateRotateConstraint](../RotateRotateConstraint/RotateRotateConstraint.md)

## Description

Gets/Sets the Visible indicating whether this constraint is visible.

## Syntax

RotateRotateConstraint.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
