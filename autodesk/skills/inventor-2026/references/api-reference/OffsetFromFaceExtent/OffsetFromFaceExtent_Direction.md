# OffsetFromFaceExtent.Direction Property

Parent Object: [OffsetFromFaceExtent](../OffsetFromFaceExtent/OffsetFromFaceExtent.md)

## Description

Read-write property that gets and sets the distance direction of the feature. Valid input is kPositiveExtentDirection, kNegativeExtentDirection, or kSymmetricExtentDirection. kPositiveExtentDirection defines the offset direction to be in the same direction as.

## Syntax

OffsetFromFaceExtent.**Direction**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2018
