# SketchArc3D.CurvatureDisplay Property

Parent Object: [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Description

Specifies whether curvature information is displayed for the arc.

## Syntax

SketchArc3D.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
