# WeldBeads.Item Property

Parent Object: [WeldBeads](../WeldBeads/WeldBeads.md)

## Description

Returns the specified WeldBead object from the collection

## Syntax

WeldBeads.**Item**( ***Index*** As Variant ) As [WeldBead](../WeldBead/WeldBead.md)

## Property Value

This is a read only property whose value is a [WeldBead](../WeldBead/WeldBead.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the WeldBead to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the WeldBead name. If an out of range index or a name of a non-existent WeldBead is provided, an error will occur. |

## Version

Introduced in version 8
