# TriangleFanGraphics.ColorMapper Property

Parent Object: [TriangleFanGraphics](../TriangleFanGraphics/TriangleFanGraphics.md)

## Description

Read-write property that gets and sets the GraphicsColorMapper object associated with the set.

## Syntax

TriangleFanGraphics.**ColorMapper**() As [GraphicsColorMapper](../GraphicsColorMapper/GraphicsColorMapper.md)

## Property Value

This is a read/write property whose value is a [GraphicsColorMapper](../GraphicsColorMapper/GraphicsColorMapper.md).

## Version

Introduced in version 2010
