# Faces Object

## Description

The Faces object represents a collection of objects. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../Faces/Faces_Count.md) | Property that returns the number of items in this collection. |
| [Item](../Faces/Faces_Item.md) | Allows integer-indexed access to objects in the collection. |
| [Type](../Faces/Faces_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
