# SelectionPreferences.Item Property

Parent Object: [SelectionPreferences](../SelectionPreferences/SelectionPreferences.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

SelectionPreferences.**Item**( ***Index*** As Long ) As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the ObjectTypeEnum to return. |

## Version

Introduced in version 2016
