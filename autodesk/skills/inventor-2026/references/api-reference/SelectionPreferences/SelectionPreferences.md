# SelectionPreferences Object

## Description

SelectionPreferences object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SelectionPreferences/SelectionPreferences_Add.md) | Adds a type to the selection preferences. |
| [Clear](../SelectionPreferences/SelectionPreferences_Clear.md) | clear all the types in the selection preferences. |
| [Remove](../SelectionPreferences/SelectionPreferences_Remove.md) | Remove a type from the selection preferences. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SelectionPreferences/SelectionPreferences_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../SelectionPreferences/SelectionPreferences_Count.md) | Gets the number of items in this collection. |
| [Item](../SelectionPreferences/SelectionPreferences_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../SelectionPreferences/SelectionPreferences_Type.md) | Gets the constant that indicates the type of this object. |
