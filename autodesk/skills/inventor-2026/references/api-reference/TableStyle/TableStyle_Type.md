# TableStyle.Type Property

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TableStyle.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
