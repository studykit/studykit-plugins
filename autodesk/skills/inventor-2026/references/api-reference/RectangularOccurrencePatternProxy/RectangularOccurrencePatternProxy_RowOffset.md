# RectangularOccurrencePatternProxy.RowOffset Property

Parent Object: [RectangularOccurrencePatternProxy](../RectangularOccurrencePatternProxy/RectangularOccurrencePatternProxy.md)

## Description

Property returning the Parameter object that controls the distance between rows.

## Syntax

RectangularOccurrencePatternProxy.**RowOffset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
