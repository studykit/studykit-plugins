# iAssemblyMember.ParentFactory Property

Parent Object: [iAssemblyMember](../iAssemblyMember/iAssemblyMember.md)

## Description

Property that returns the iAssembly factory that created this iAssemblyMember.

## Syntax

iAssemblyMember.**ParentFactory**() As [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Property Value

This is a read only property whose value is an [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md).

## Version

Introduced in version 11
