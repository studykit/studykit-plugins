# AssemblyDocument.ReferenceKeyManager Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Gets this document's ReferenceKeyManager.

## Syntax

AssemblyDocument.**ReferenceKeyManager**() As [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md)

## Property Value

This is a read only property whose value is a [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md).

## Version

Introduced in version 5
