# AssemblyDocument.TextStyles Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Gets the TextStylesEnumerator object.

## Syntax

AssemblyDocument.**TextStyles**() As [TextStylesEnumerator](../TextStylesEnumerator/TextStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [TextStylesEnumerator](../TextStylesEnumerator/TextStylesEnumerator.md).

## Version

Introduced in version 2017
