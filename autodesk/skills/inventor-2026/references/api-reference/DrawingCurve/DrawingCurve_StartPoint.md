# DrawingCurve.StartPoint Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the start point of the curve in sheet space. This property returns Nothing for circular curves.

## Syntax

DrawingCurve.**StartPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).
