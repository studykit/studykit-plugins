# DWGEntity.ParentEntity Property

Parent Object: [DWGEntity](../DWGEntity/DWGEntity.md)

## Description

Read-only property that returns the parent DWGEntity of the entity.

## Syntax

DWGEntity.**ParentEntity**() As [DWGEntity](../DWGEntity/DWGEntity.md)

## Property Value

This is a read only property whose value is a [DWGEntity](../DWGEntity/DWGEntity.md).

## Version

Introduced in version 2017
