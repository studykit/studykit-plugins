# Document.ClientViews Property

Parent Object: [Document](../Document/Document.md)

## Description

Returns document's client views.

## Syntax

Document.**ClientViews**() As [ClientViews](../ClientViews/ClientViews.md)

## Property Value

This is a read only property whose value is a [ClientViews](../ClientViews/ClientViews.md).

## Version

Introduced in version 2017
