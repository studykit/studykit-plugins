# ModelWeldingSymbolProxy.Name Property

Parent Object: [ModelWeldingSymbolProxy](../ModelWeldingSymbolProxy/ModelWeldingSymbolProxy.md)

## Description

Read-write property that gets and sets the name of the annotation.

## Syntax

ModelWeldingSymbolProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2024
