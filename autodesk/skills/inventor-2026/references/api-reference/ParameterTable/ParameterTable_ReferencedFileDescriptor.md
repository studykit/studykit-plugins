# ParameterTable.ReferencedFileDescriptor Property

Parent Object: [ParameterTable](../ParameterTable/ParameterTable.md)

## Description

Gets the description of the linked file.

## Syntax

ParameterTable.**ReferencedFileDescriptor**() As [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Property Value

This is a read only property whose value is a [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md).

## Version

Introduced in version 2012
