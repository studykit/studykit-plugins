# PresentationMeshFeatureSet.Suppressed Property

Parent Object: [PresentationMeshFeatureSet](../PresentationMeshFeatureSet/PresentationMeshFeatureSet.md)

## Description

Read-only property that gets whether all of the mesh features within this set are suppressed or not.

## Syntax

PresentationMeshFeatureSet.**Suppressed**() As [MeshFeaturesSuppressionEnum](../MeshFeaturesSuppressionEnum.md)

## Property Value

This is a read only property whose value is a [MeshFeaturesSuppressionEnum](../MeshFeaturesSuppressionEnum.md).

## Version

Introduced in version 2018
