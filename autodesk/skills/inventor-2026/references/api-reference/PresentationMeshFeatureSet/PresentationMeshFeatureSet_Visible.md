# PresentationMeshFeatureSet.Visible Property

Parent Object: [PresentationMeshFeatureSet](../PresentationMeshFeatureSet/PresentationMeshFeatureSet.md)

## Description

Read-only property that gets whether all of the mesh features within this set are visible.

## Syntax

PresentationMeshFeatureSet.**Visible**() As [MeshFeaturesVisibilityEnum](../MeshFeaturesVisibilityEnum.md)

## Property Value

This is a read only property whose value is a [MeshFeaturesVisibilityEnum](../MeshFeaturesVisibilityEnum.md).

## Version

Introduced in version 2018
