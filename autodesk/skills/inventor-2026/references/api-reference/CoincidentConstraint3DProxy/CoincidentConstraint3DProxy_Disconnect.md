# CoincidentConstraint3DProxy.Disconnect Method

Parent Object: [CoincidentConstraint3DProxy](../CoincidentConstraint3DProxy/CoincidentConstraint3DProxy.md)

## Description

Method that disconnects the input entity from the coincident constraint. Valid inputs are SketchPoint, Vertex or WorkPoint.

## Syntax

CoincidentConstraint3DProxy.**Disconnect**( ***Entity*** As Object )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Entity | Object | Input object that specifies which entity to disconnect. Valid inputs are , Vertex or WorkPoint. |

## Version

Introduced in version 6
