# HemFeatures.Add Method

Parent Object: [HemFeatures](../HemFeatures/HemFeatures.md)

## Description

Method that creates a new hem feature. The newly created HemFeature object is returned.

## Syntax

HemFeatures.**Add**( ***HemDefinition*** As [HemDefinition](../HemDefinition/HemDefinition.md) ) As [HemFeature](../HemFeature/HemFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| HemDefinition | [HemDefinition](../HemDefinition/HemDefinition.md) | Input HemDefinition object that defines the hem feature you want to create. A HemDefinition object can be created using the HemFeatures.CreateHemDefinition method. It can also be obtained from an existing HemFeature object. |

## Version

Introduced in version 2010
