# AnalysisManager.ActiveAnalysisType Property

Parent Object: [AnalysisManager](../AnalysisManager/AnalysisManager.md)

## Description

Property that returns the enum indicating the type of the currently active analysis.

## Syntax

AnalysisManager.**ActiveAnalysisType**() As [AnalysisTypeEnum](../AnalysisTypeEnum.md)

## Property Value

This is a read only property whose value is an [AnalysisTypeEnum](../AnalysisTypeEnum.md).

## Version

Introduced in version 2009
