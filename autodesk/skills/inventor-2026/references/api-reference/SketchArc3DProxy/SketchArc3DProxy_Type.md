# SketchArc3DProxy.Type Property

Parent Object: [SketchArc3DProxy](../SketchArc3DProxy/SketchArc3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchArc3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
