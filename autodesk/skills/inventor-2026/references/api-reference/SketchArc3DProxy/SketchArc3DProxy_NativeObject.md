# SketchArc3DProxy.NativeObject Property

Parent Object: [SketchArc3DProxy](../SketchArc3DProxy/SketchArc3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchArc3DProxy.**NativeObject**() As [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Property Value

This is a read only property whose value is a [SketchArc3D](../SketchArc3D/SketchArc3D.md).

## Version

Introduced in version 6
