# DirectEditFeatureProxy.DirectEditOperations Property

Parent Object: [DirectEditFeatureProxy](../DirectEditFeatureProxy/DirectEditFeatureProxy.md)

## Description

Gets all the operations of this direct edit feature.

## Syntax

DirectEditFeatureProxy.**DirectEditOperations**() As [DirectEditOperations](../DirectEditOperations/DirectEditOperations.md)

## Property Value

This is a read only property whose value is a [DirectEditOperations](../DirectEditOperations/DirectEditOperations.md).

## Version

Introduced in version 2015
