# RotateTranslateConstraintProxy.NativeObject Property

Parent Object: [RotateTranslateConstraintProxy](../RotateTranslateConstraintProxy/RotateTranslateConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RotateTranslateConstraintProxy.**NativeObject**() As [RotateTranslateConstraint](../RotateTranslateConstraint/RotateTranslateConstraint.md)

## Property Value

This is a read only property whose value is a [RotateTranslateConstraint](../RotateTranslateConstraint/RotateTranslateConstraint.md).

## Version

Introduced in version 11
