# FlangeDefinition.Parent Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Property that returns the parent FlangeFeature of this FlangeDefinition object.

## Syntax

FlangeDefinition.**Parent**() As [FlangeFeature](../FlangeFeature/FlangeFeature.md)

## Property Value

This is a read only property whose value is a [FlangeFeature](../FlangeFeature/FlangeFeature.md).

## Version

Introduced in version 2009
