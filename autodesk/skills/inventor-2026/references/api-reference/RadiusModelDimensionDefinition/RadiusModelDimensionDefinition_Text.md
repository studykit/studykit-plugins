# RadiusModelDimensionDefinition.Text Property

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Read-only property that gets the text of the dimension. Properties on the returned ModelAnnotationText object can be edited to change the displayed text.

## Syntax

RadiusModelDimensionDefinition.**Text**() As [ModelAnnotationText](../ModelAnnotationText/ModelAnnotationText.md)

## Property Value

This is a read only property whose value is a [ModelAnnotationText](../ModelAnnotationText/ModelAnnotationText.md).

## Version

Introduced in version 2018
