# RadiusModelDimensionDefinition.IsLeaderFromCenter Property

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Read-write property that gets and sets whether the leader starts from the center of the arc or the circle.

## Syntax

RadiusModelDimensionDefinition.**IsLeaderFromCenter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
