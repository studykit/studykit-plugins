# ObjectDefaultsStyle.SurfaceTextureStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for surface textures.

## Syntax

ObjectDefaultsStyle.**SurfaceTextureStyle**() As [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Property Value

This is a read/write property whose value is a [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md).

## Version

Introduced in version 2010
