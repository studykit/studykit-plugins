# ObjectDefaultsStyle.OrdinateDimensionSetLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default layer for ordinate dimension sets.

## Syntax

ObjectDefaultsStyle.**OrdinateDimensionSetLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2010
