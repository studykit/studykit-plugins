# MiniToolbarValueEditor.CanValueBeEqualToMaximumValue Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets whether the maximum value is valid value or not.

## Syntax

MiniToolbarValueEditor.**CanValueBeEqualToMaximumValue**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
