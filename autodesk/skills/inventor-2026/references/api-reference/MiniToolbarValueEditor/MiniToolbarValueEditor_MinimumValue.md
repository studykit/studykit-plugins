# MiniToolbarValueEditor.MinimumValue Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the minimum valid value for this control in database units.

## Syntax

MiniToolbarValueEditor.**MinimumValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2012
