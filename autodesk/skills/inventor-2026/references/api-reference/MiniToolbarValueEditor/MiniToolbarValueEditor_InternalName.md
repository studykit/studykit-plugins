# MiniToolbarValueEditor.InternalName Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-only property that returns a string uniquely identifying this control within the toolbar.

## Syntax

MiniToolbarValueEditor.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
