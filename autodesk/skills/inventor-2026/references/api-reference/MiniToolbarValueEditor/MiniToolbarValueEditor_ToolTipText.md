# MiniToolbarValueEditor.ToolTipText Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the tooltip text for the control.

## Syntax

MiniToolbarValueEditor.**ToolTipText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
