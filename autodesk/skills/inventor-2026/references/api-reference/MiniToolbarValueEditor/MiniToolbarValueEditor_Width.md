# MiniToolbarValueEditor.Width Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the width of the control (in pixels).

## Syntax

MiniToolbarValueEditor.**Width**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2012
