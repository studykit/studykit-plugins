# PresentationSnapshotView.SetOpacity Method

Parent Object: [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Description

Method that sets override opacity to PresentationComponent objects.

## Syntax

PresentationSnapshotView.**SetOpacity**( ***PresentationComponents*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md), ***Opacity*** As Double )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PresentationComponents | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection object that includes the PresentationComponent objects. |
| Opacity | Double | Input Double value that specifies the opacity. Valid value is from 0 to 1. |

## Version

Introduced in version 2018
