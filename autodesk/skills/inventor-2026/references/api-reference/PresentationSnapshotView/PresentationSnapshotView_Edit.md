# PresentationSnapshotView.Edit Method

Parent Object: [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Description

Method that enters the presentation snapshot view edit mode. Currently the edit(or even for some query functions) out of edit mode for snapshot view is not allowed.

## Syntax

PresentationSnapshotView.**Edit**()

## Version

Introduced in version 2018
