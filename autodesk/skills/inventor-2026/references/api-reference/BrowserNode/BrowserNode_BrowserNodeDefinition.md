# BrowserNode.BrowserNodeDefinition Property

Parent Object: [BrowserNode](../BrowserNode/BrowserNode.md)

## Description

Property that returns a object which defines the various inputs that were used to create the browser node.

## Syntax

BrowserNode.**BrowserNodeDefinition**() As [BrowserNodeDefinition](../BrowserNodeDefinition/BrowserNodeDefinition.md)

## Property Value

This is a read only property whose value is a [BrowserNodeDefinition](../BrowserNodeDefinition/BrowserNodeDefinition.md).
