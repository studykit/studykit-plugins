# iPartFactory.MaterialColumn Property

Parent Object: [iPartFactory](../iPartFactory/iPartFactory.md)

## Description

Gets and sets the column to use as the material column.

## Syntax

iPartFactory.**MaterialColumn**() As [iPartTableColumn](../iPartTableColumn/iPartTableColumn.md)

## Property Value

This is a read/write property whose value is an [iPartTableColumn](../iPartTableColumn/iPartTableColumn.md).

## Version

Introduced in version 2011
