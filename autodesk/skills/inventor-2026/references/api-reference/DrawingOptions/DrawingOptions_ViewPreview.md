# DrawingOptions.ViewPreview Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the preview type for drawing views.

## Syntax

DrawingOptions.**ViewPreview**() As [ViewPreviewTypeEnum](../ViewPreviewTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ViewPreviewTypeEnum](../ViewPreviewTypeEnum.md).

## Version

Introduced in version 2008
