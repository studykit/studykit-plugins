# DrawingOptions.InventorDWGFileVersion Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the AutoCAD DWG version for Inventor DWG files.

## Syntax

DrawingOptions.**InventorDWGFileVersion**() As [DWGFileVersionEnum](../DWGFileVersionEnum.md)

## Property Value

This is a read/write property whose value is a [DWGFileVersionEnum](../DWGFileVersionEnum.md).

## Version

Introduced in version 2010
