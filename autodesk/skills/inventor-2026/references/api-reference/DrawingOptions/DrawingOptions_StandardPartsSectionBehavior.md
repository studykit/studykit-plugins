# DrawingOptions.StandardPartsSectionBehavior Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the sectioning behavior for standard parts when section views are created in the drawing.

## Syntax

DrawingOptions.**StandardPartsSectionBehavior**() As [StandardPartsSectionBehaviorEnum](../StandardPartsSectionBehaviorEnum.md)

## Property Value

This is a read/write property whose value is a [StandardPartsSectionBehaviorEnum](../StandardPartsSectionBehaviorEnum.md).

## Version

Introduced in version 2008
