# DrawingOptions.ViewBlockInsertionPoint Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets default insertion point for a view block.

## Syntax

DrawingOptions.**ViewBlockInsertionPoint**() As [ViewBlockInsertionPointEnum](../ViewBlockInsertionPointEnum.md)

## Property Value

This is a read/write property whose value is a [ViewBlockInsertionPointEnum](../ViewBlockInsertionPointEnum.md).

## Version

Introduced in version 2010
