# DesignViewRepresentations Object

## Description

The DesignViewRepresentations collection object provides access to all the design view representations in the assembly and provides methods to create new ones.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../DesignViewRepresentations/DesignViewRepresentations_Add.md) | Method that creates a new DesignViewRepresentation. The new created DesignViewRepresentation is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DesignViewRepresentations/DesignViewRepresentations_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DesignViewRepresentations/DesignViewRepresentations_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DesignViewRepresentations/DesignViewRepresentations_Item.md) | Returns the specified DesignViewRepresentation object from the collection. |
| [Type](../DesignViewRepresentations/DesignViewRepresentations_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[RepresentationsManager.DesignViewRepresentations](../RepresentationsManager/RepresentationsManager_DesignViewRepresentations.md)

## Version

Introduced in version 11
