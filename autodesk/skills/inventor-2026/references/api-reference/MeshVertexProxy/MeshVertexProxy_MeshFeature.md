# MeshVertexProxy.MeshFeature Property

Parent Object: [MeshVertexProxy](../MeshVertexProxy/MeshVertexProxy.md)

## Description

Read-only property that returns the MeshFeature this MeshVertex is located on.

## Syntax

MeshVertexProxy.**MeshFeature**() As [MeshFeature](../MeshFeature/MeshFeature.md)

## Property Value

This is a read only property whose value is a [MeshFeature](../MeshFeature/MeshFeature.md).

## Version

Introduced in version 2017
