# CropOperation.DisplayCropCutLine Property

Parent Object: [CropOperation](../CropOperation/CropOperation.md)

## Description

Read-write property that gets and sets whether to display the crop cut line or not.

## Syntax

CropOperation.**DisplayCropCutLine**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
