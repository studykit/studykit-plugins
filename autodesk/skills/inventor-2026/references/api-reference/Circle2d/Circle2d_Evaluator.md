# Circle2d.Evaluator Property

Parent Object: [Circle2d](../Circle2d/Circle2d.md)

## Description

Gets the Curve2dEvaluator for this circle.

## Syntax

Circle2d.**Evaluator**() As [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md)

## Property Value

This is a read only property whose value is a [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md).

## Version

Introduced in version 4
