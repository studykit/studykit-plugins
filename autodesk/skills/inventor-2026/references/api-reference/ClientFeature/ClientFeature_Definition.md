# ClientFeature.Definition Property

Parent Object: [ClientFeature](../ClientFeature/ClientFeature.md)

## Description

Property that returns the ClientFeatureDefinition object that specifies the various inputs used to create the feature.

## Syntax

ClientFeature.**Definition**() As [ClientFeatureDefinition](../ClientFeatureDefinition/ClientFeatureDefinition.md)

## Property Value

This is a read only property whose value is a [ClientFeatureDefinition](../ClientFeatureDefinition/ClientFeatureDefinition.md).

## Version

Introduced in version 2008
