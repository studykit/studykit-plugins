# ClientFeature.HiddenParameters Property

Parent Object: [ClientFeature](../ClientFeature/ClientFeature.md)

## Description

Read-write property that gets and sets the collection of Parameters that are to be hidden from the user.

## Syntax

ClientFeature.**HiddenParameters**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2009
