# FaceProxy.EdgeLoops Property

Parent Object: [FaceProxy](../FaceProxy/FaceProxy.md)

## Description

Gets the EdgeLoops collection referenced by this Face.

## Syntax

FaceProxy.**EdgeLoops**() As [EdgeLoops](../EdgeLoops/EdgeLoops.md)

## Property Value

This is a read only property whose value is an [EdgeLoops](../EdgeLoops/EdgeLoops.md).

## Version

Introduced in version 4
