# FaceProxy.Edges Property

Parent Object: [FaceProxy](../FaceProxy/FaceProxy.md)

## Description

Gets the Edges referenced by this Face.

## Syntax

FaceProxy.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 4
