# SurfaceBodyProxy.GeometryForm Property

Parent Object: [SurfaceBodyProxy](../SurfaceBodyProxy/SurfaceBodyProxy.md)

## Description

Gets the form of the underlying geometry as a combination of one or more CurveGeometryFormEnum values.

## Syntax

SurfaceBodyProxy.**GeometryForm**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
