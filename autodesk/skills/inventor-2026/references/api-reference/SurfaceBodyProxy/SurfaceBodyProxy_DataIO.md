# SurfaceBodyProxy.DataIO Property

Parent Object: [SurfaceBodyProxy](../SurfaceBodyProxy/SurfaceBodyProxy.md)

## Description

Returns the DataIO object.

## Syntax

SurfaceBodyProxy.**DataIO**() As [DataIO](../DataIO/DataIO.md)

## Property Value

This is a read only property whose value is a [DataIO](../DataIO/DataIO.md).

## Version

Introduced in version 9
