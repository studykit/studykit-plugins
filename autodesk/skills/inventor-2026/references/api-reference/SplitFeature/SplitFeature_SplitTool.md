# SplitFeature.SplitTool Property

Parent Object: [SplitFeature](../SplitFeature/SplitFeature.md)

## Description

Gets and sets the split tool used to create the split feature. The object can be a WorkPlane, WorkSurface, SurfaceBody or a Path.

## Syntax

SplitFeature.**SplitTool**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 6
