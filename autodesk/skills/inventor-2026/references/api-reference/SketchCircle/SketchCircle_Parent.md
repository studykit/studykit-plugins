# SketchCircle.Parent Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchCircle.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 5
