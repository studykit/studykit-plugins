# SketchCircle.CenterSketchPoint Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Property that gets the sketch point that defines the position of the center of the circle.

## Syntax

SketchCircle.**CenterSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).
