# GraphicsColorMapper.MappedColor Property

Parent Object: [GraphicsColorMapper](../GraphicsColorMapper/GraphicsColorMapper.md)

## Description

Property that returns the color at a given value in the map. This property is informational only and need not be used when assigning a color map to a primitive.

## Syntax

GraphicsColorMapper.**MappedColor**( ***Value*** As Double ) As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | Double | Double value to retrieve the color for. |

## Version

Introduced in version 2010
