# FrontViewPlaneEnum Enumerator

## Description

Constants indicating the viewing plane to use as front view.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kFrontViewFromModel | 80388 | Front view orientation settings from the model document. |
| kFrontViewXYPlane | 80385 | XY plane. |
| kFrontViewXZPlane | 80387 | XZ plane. |
| kFrontViewYZPlane | 80386 | YZ plane. |

## Version

Introduced in version 2010
