# RadiusDimConstraint.TextPoint Property

Parent Object: [RadiusDimConstraint](../RadiusDimConstraint/RadiusDimConstraint.md)

## Description

Gets and sets the position of the dimension text.

## Syntax

RadiusDimConstraint.**TextPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
