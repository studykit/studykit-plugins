# ImportedComponentProxy.NativeObject Property

Parent Object: [ImportedComponentProxy](../ImportedComponentProxy/ImportedComponentProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ImportedComponentProxy.**NativeObject**() As [ImportedComponent](../ImportedComponent/ImportedComponent.md)

## Property Value

This is a read only property whose value is an [ImportedComponent](../ImportedComponent/ImportedComponent.md).

## Version

Introduced in version 2016
