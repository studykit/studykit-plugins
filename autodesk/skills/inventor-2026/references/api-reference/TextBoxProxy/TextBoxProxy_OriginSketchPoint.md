# TextBoxProxy.OriginSketchPoint Property

Parent Object: [TextBoxProxy](../TextBoxProxy/TextBoxProxy.md)

## Description

Gets the SketchPoint on the origin of this TextBox. This will be Nothing in the case that the boundaries are displayed

## Syntax

TextBoxProxy.**OriginSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 10
