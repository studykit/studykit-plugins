# BalloonTip.ClientData Property

Parent Object: [BalloonTip](../BalloonTip/BalloonTip.md)

## Description

Gets/sets property that contains some additional data that the client would like to store with the BalloonTip.

## Syntax

BalloonTip.**ClientData**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
