# CornerChamferDefinition.Parent Property

Parent Object: [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md)

## Description

Property that returns the parent CornerChamferFeature object of this CornerChamferDefinition object.

## Syntax

CornerChamferDefinition.**Parent**() As [CornerChamferFeature](../CornerChamferFeature/CornerChamferFeature.md)

## Property Value

This is a read only property whose value is a [CornerChamferFeature](../CornerChamferFeature/CornerChamferFeature.md).

## Version

Introduced in version 2009
