# DirectEditMoveOperationProxy.Name Property

Parent Object: [DirectEditMoveOperationProxy](../DirectEditMoveOperationProxy/DirectEditMoveOperationProxy.md)

## Description

Gets the name of this direct edit operation within the scope of this Document.

## Syntax

DirectEditMoveOperationProxy.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2015
