# HorizontalAlignConstraint.PointTwo Property

Parent Object: [HorizontalAlignConstraint](../HorizontalAlignConstraint/HorizontalAlignConstraint.md)

## Description

Property that returns the second sketch point being constrained.

## Syntax

HorizontalAlignConstraint.**PointTwo**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 5
