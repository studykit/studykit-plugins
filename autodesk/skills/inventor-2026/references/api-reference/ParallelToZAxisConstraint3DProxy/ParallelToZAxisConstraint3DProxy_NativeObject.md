# ParallelToZAxisConstraint3DProxy.NativeObject Property

Parent Object: [ParallelToZAxisConstraint3DProxy](../ParallelToZAxisConstraint3DProxy/ParallelToZAxisConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ParallelToZAxisConstraint3DProxy.**NativeObject**() As [ParallelToZAxisConstraint3D](../ParallelToZAxisConstraint3D/ParallelToZAxisConstraint3D.md)

## Property Value

This is a read only property whose value is a [ParallelToZAxisConstraint3D](../ParallelToZAxisConstraint3D/ParallelToZAxisConstraint3D.md).

## Version

Introduced in version 2017
