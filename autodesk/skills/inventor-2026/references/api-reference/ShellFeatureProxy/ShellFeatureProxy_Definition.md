# ShellFeatureProxy.Definition Property

Parent Object: [ShellFeatureProxy](../ShellFeatureProxy/ShellFeatureProxy.md)

## Description

Gets and sets the ShellDefinition object which defines the various input that was used to create the shell feature.

## Syntax

ShellFeatureProxy.**Definition**() As [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Property Value

This is a read/write property whose value is a [ShellDefinition](../ShellDefinition/ShellDefinition.md).

## Version

Introduced in version 9
