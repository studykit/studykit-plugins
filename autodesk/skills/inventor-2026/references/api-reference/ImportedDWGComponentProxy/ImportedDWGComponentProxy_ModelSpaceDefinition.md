# ImportedDWGComponentProxy.ModelSpaceDefinition Property

Parent Object: [ImportedDWGComponentProxy](../ImportedDWGComponentProxy/ImportedDWGComponentProxy.md)

## Description

Read-only property that returns the DWGModelSpaceDefinition object.

## Syntax

ImportedDWGComponentProxy.**ModelSpaceDefinition**() As [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Property Value

This is a read only property whose value is a [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md).

## Version

Introduced in version 2016
