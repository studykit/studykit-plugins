# FlatBendResult.Parent Property

Parent Object: [FlatBendResult](../FlatBendResult/FlatBendResult.md)

## Description

Property that returns the parent FlatPattern object.

## Syntax

FlatBendResult.**Parent**() As [FlatPattern](../FlatPattern/FlatPattern.md)

## Property Value

This is a read only property whose value is a [FlatPattern](../FlatPattern/FlatPattern.md).

## Version

Introduced in version 2008
