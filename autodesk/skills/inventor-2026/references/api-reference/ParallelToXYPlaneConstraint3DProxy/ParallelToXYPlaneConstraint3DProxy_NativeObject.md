# ParallelToXYPlaneConstraint3DProxy.NativeObject Property

Parent Object: [ParallelToXYPlaneConstraint3DProxy](../ParallelToXYPlaneConstraint3DProxy/ParallelToXYPlaneConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ParallelToXYPlaneConstraint3DProxy.**NativeObject**() As [ParallelToXYPlaneConstraint3D](../ParallelToXYPlaneConstraint3D/ParallelToXYPlaneConstraint3D.md)

## Property Value

This is a read only property whose value is a [ParallelToXYPlaneConstraint3D](../ParallelToXYPlaneConstraint3D/ParallelToXYPlaneConstraint3D.md).

## Version

Introduced in version 2017
