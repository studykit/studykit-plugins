# SketchOffsetSplineProxy.LineWeight Property

Parent Object: [SketchOffsetSplineProxy](../SketchOffsetSplineProxy/SketchOffsetSplineProxy.md)

## Description

Gets and sets the LineWeight applied to this sketch spline.

## Syntax

SketchOffsetSplineProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
