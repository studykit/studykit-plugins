# FeatureApproximationTypeEnum Enumerator

## Description

Constants indicating the feature approximation type of a thicken feature.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kMeanApproximation | 60674 | Mean approximation type. |
| kNeverTooThickApproximation | 60675 | Never too thick approximation type. |
| kNeverTooThinApproximation | 60676 | Never too thin approximation type. |
| kNoApproximation | 60673 | No approximation type. |
