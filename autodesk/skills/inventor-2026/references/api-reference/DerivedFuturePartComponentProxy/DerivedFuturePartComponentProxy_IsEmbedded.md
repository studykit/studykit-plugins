# DerivedFuturePartComponentProxy.IsEmbedded Property

Parent Object: [DerivedFuturePartComponentProxy](../DerivedFuturePartComponentProxy/DerivedFuturePartComponentProxy.md)

## Description

Property that returns whether the reference component refers to an embedded document or a linked document.

## Syntax

DerivedFuturePartComponentProxy.**IsEmbedded**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
