# File.AllReferencedFiles Property

Parent Object: [File](../File/File.md)

## Description

Property that returns all the file references of this File along with all of the recursively nested references.

## Syntax

File.**AllReferencedFiles**() As [FilesEnumerator](../FilesEnumerator/FilesEnumerator.md)

## Property Value

This is a read only property whose value is a [FilesEnumerator](../FilesEnumerator/FilesEnumerator.md).

## Version

Introduced in version 11
