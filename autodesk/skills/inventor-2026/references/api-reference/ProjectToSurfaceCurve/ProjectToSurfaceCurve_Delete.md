# ProjectToSurfaceCurve.Delete Method

Parent Object: [ProjectToSurfaceCurve](../ProjectToSurfaceCurve/ProjectToSurfaceCurve.md)

## Description

Method that deletes the project to surface curve. This will delete all of the related sketch entities.

## Syntax

ProjectToSurfaceCurve.**Delete**()

## Version

Introduced in version 2021
