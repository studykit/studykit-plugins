# DerivedPartComponent.SuppressLinkToFile Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Gets and Sets a Boolean flag indicating whether the link to the part file is suppressed.

## Syntax

DerivedPartComponent.**SuppressLinkToFile**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
