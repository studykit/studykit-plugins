# DerivedPartComponent.ReferencedDocumentDescriptor Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Property that returns the DocumentDescriptor object.

## Syntax

DerivedPartComponent.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 11
