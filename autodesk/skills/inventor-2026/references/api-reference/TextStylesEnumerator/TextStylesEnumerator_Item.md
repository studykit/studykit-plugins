# TextStylesEnumerator.Item Property

Parent Object: [TextStylesEnumerator](../TextStylesEnumerator/TextStylesEnumerator.md)

## Description

Returns the specified object in the collection.

## Syntax

TextStylesEnumerator.**Item**( ***Index*** As Variant ) As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read only property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant that specifies the index within the collection of the item to return. |

## Version

Introduced in version 9
