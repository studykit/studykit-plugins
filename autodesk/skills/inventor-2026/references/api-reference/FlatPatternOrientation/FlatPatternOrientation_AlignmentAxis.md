# FlatPatternOrientation.AlignmentAxis Property

Parent Object: [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Description

Read-write property that gets and sets the alignment axis of flat pattern orientation.

## Syntax

FlatPatternOrientation.**AlignmentAxis**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2015
