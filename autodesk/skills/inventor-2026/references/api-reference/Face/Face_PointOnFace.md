# Face.PointOnFace Property

Parent Object: [Face](../Face/Face.md)

## Description

Property that returns a characteristic somewhere on the interior of the Face.

## Syntax

Face.**PointOnFace**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 9
