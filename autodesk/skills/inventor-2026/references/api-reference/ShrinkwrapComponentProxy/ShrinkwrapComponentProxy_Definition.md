# ShrinkwrapComponentProxy.Definition Property

Parent Object: [ShrinkwrapComponentProxy](../ShrinkwrapComponentProxy/ShrinkwrapComponentProxy.md)

## Description

Gets or Sets ShrinkwrapDefinition that defines the current state of this shrinkwrap component.

## Syntax

ShrinkwrapComponentProxy.**Definition**() As [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Property Value

This is a read/write property whose value is a [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md).

## Version

Introduced in version 2018
