# ColorScheme.PrimarySelectColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Color property that returns the primary selection .

## Syntax

ColorScheme.**PrimarySelectColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
