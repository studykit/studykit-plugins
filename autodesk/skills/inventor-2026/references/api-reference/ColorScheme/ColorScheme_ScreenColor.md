# ColorScheme.ScreenColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Property that returns the background screen color.

## Syntax

ColorScheme.**ScreenColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
