# TableFormat.OutsideLineColor Property

Parent Object: [TableFormat](../TableFormat/TableFormat.md)

## Description

Gets and sets the color of the outside lines.

## Syntax

TableFormat.**OutsideLineColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
