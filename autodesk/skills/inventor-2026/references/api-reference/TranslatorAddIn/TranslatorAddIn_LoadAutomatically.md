# TranslatorAddIn.LoadAutomatically Property

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Gets/Sets whether the add-in loads automatically based on the load behavior specified for the add-in. If set to False, the add-in needs to be manually loaded by the user.

## Syntax

TranslatorAddIn.**LoadAutomatically**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
