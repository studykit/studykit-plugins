# DerivedAssemblyComponentProxy.NativeObject Property

Parent Object: [DerivedAssemblyComponentProxy](../DerivedAssemblyComponentProxy/DerivedAssemblyComponentProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DerivedAssemblyComponentProxy.**NativeObject**() As [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Property Value

This is a read only property whose value is a [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md).

## Version

Introduced in version 2008
