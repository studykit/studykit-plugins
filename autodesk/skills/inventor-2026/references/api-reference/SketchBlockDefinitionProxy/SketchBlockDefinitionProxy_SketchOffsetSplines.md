# SketchBlockDefinitionProxy.SketchOffsetSplines Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the collection object. This collection provides access to the existing offset splines in the sketch.

## Syntax

SketchBlockDefinitionProxy.**SketchOffsetSplines**() As [SketchOffsetSplines](../SketchOffsetSplines/SketchOffsetSplines.md)

## Property Value

This is a read only property whose value is a [SketchOffsetSplines](../SketchOffsetSplines/SketchOffsetSplines.md).

## Version

Introduced in version 2010
