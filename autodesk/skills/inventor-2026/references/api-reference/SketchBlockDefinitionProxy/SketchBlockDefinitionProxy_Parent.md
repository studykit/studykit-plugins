# SketchBlockDefinitionProxy.Parent Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that gets the parent object from whom this object can logically be reached.

## Syntax

SketchBlockDefinitionProxy.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2010
