# SketchBlockDefinitionProxy.SketchArcs Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

SketchBlockDefinitionProxy.**SketchArcs**() As [SketchArcs](../SketchArcs/SketchArcs.md)

## Property Value

This is a read only property whose value is a [SketchArcs](../SketchArcs/SketchArcs.md).

## Version

Introduced in version 2010
