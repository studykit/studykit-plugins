# ModelLeader.ArrowheadType Property

Parent Object: [ModelLeader](../ModelLeader/ModelLeader.md)

## Description

Gets and sets the arrowhead type of the leader.

## Syntax

ModelLeader.**ArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2018
