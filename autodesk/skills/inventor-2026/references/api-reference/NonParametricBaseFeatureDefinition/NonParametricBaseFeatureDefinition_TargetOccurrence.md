# NonParametricBaseFeatureDefinition.TargetOccurrence Property

Parent Object: [NonParametricBaseFeatureDefinition](../NonParametricBaseFeatureDefinition/NonParametricBaseFeatureDefinition.md)

## Description

Read-write property that specifies the desired target Occurrence.

## Syntax

NonParametricBaseFeatureDefinition.**TargetOccurrence**() As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Property Value

This is a read/write property whose value is a [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md).
