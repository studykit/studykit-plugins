# Sheet.CustomTables Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the CustomTables collection object for this sheet.

## Syntax

Sheet.**CustomTables**() As [CustomTables](../CustomTables/CustomTables.md)

## Property Value

This is a read only property whose value is a [CustomTables](../CustomTables/CustomTables.md).
