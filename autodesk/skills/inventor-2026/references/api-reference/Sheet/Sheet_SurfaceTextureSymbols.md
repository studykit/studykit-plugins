# Sheet.SurfaceTextureSymbols Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the SurfaceTextureSymbols collection object.

## Syntax

Sheet.**SurfaceTextureSymbols**() As [SurfaceTextureSymbols](../SurfaceTextureSymbols/SurfaceTextureSymbols.md)

## Property Value

This is a read only property whose value is a [SurfaceTextureSymbols](../SurfaceTextureSymbols/SurfaceTextureSymbols.md).
