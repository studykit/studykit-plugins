# Sheet.WeldingSymbols Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Get the DrawingWeldingSymbols collection object.

## Syntax

Sheet.**WeldingSymbols**() As [DrawingWeldingSymbols](../DrawingWeldingSymbols/DrawingWeldingSymbols.md)

## Property Value

This is a read only property whose value is a [DrawingWeldingSymbols](../DrawingWeldingSymbols/DrawingWeldingSymbols.md).

## Version

Introduced in version 2024
