# SketchRectangularPattern.Parent Property

Parent Object: [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SketchRectangularPattern.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 2025.1
