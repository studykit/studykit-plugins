# ReferencedOLEFileDescriptor.OLEDocumentType Property

Parent Object: [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Description

Property that returns the type of OLE reference this represents. Valid values are kOLEDocumentEmbeddingObject, kOLEDocumentLinkObject, or kOLEDocumentUnknownObject.

## Syntax

ReferencedOLEFileDescriptor.**OLEDocumentType**() As [OLEDocumentTypeEnum](../OLEDocumentTypeEnum.md)

## Property Value

This is a read only property whose value is an [OLEDocumentTypeEnum](../OLEDocumentTypeEnum.md).

## Version

Introduced in version 4
