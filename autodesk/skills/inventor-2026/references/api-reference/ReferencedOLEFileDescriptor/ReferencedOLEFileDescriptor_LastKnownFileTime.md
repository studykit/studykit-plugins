# ReferencedOLEFileDescriptor.LastKnownFileTime Property

Parent Object: [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Description

Property that returns the time stamp on the file when it was last reviewed. This property returns Nothing for embeddings.

## Syntax

ReferencedOLEFileDescriptor.**LastKnownFileTime**() As Variant

## Property Value

This is a read only property whose value is a Variant.

## Version

Introduced in version 9
