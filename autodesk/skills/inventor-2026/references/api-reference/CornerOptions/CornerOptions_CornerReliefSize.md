# CornerOptions.CornerReliefSize Property

Parent Object: [CornerOptions](../CornerOptions/CornerOptions.md)

## Description

Provides access to the two bend corner relief size of a sheet metal feature.

## Syntax

CornerOptions.**CornerReliefSize**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2009
