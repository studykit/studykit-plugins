# ModelFeatureControlFrameProxy.Visible Property

Parent Object: [ModelFeatureControlFrameProxy](../ModelFeatureControlFrameProxy/ModelFeatureControlFrameProxy.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelFeatureControlFrameProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
