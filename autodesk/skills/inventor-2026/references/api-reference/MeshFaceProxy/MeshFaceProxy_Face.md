# MeshFaceProxy.Face Property

Parent Object: [MeshFaceProxy](../MeshFaceProxy/MeshFaceProxy.md)

## Description

Read-only property that returns the transient Brep Face for the MeshFace.

## Syntax

MeshFaceProxy.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2017
