# CornerRoundEdgeSet.Radius Property

Parent Object: [CornerRoundEdgeSet](../CornerRoundEdgeSet/CornerRoundEdgeSet.md)

## Description

Gets and sets the radius of a sheet metal corner round feature.

## Syntax

CornerRoundEdgeSet.**Radius**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2009
