# PartsListSortSettings.SortNumerically Property

Parent Object: [PartsListSortSettings](../PartsListSortSettings/PartsListSortSettings.md)

## Description

Read-write property that gets and sets whether sorts by numerical order or not.

## Syntax

PartsListSortSettings.**SortNumerically**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
