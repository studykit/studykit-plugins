# RegionProperties.Perimeter Property

Parent Object: [RegionProperties](../RegionProperties/RegionProperties.md)

## Description

Property that returns the perimeter of the section in database units.

## Syntax

RegionProperties.**Perimeter**() As Double

## Property Value

This is a read only property whose value is a Double.
