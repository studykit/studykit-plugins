# RegionProperties.Accuracy Property

Parent Object: [RegionProperties](../RegionProperties/RegionProperties.md)

## Description

Get and sets the desired accuracy of calculations.

## Syntax

RegionProperties.**Accuracy**() As [AccuracyEnum](../AccuracyEnum.md)

## Property Value

This is a read/write property whose value is an [AccuracyEnum](../AccuracyEnum.md).
