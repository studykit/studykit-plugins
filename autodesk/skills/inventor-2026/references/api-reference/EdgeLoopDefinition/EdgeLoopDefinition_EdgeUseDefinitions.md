# EdgeLoopDefinition.EdgeUseDefinitions Property

Parent Object: [EdgeLoopDefinition](../EdgeLoopDefinition/EdgeLoopDefinition.md)

## Description

Property that returns the collection of EdgeUseDefinition objects associated with this EdgeLoopDefinition object.

## Syntax

EdgeLoopDefinition.**EdgeUseDefinitions**() As [EdgeUseDefinitions](../EdgeUseDefinitions/EdgeUseDefinitions.md)

## Property Value

This is a read only property whose value is an [EdgeUseDefinitions](../EdgeUseDefinitions/EdgeUseDefinitions.md).
