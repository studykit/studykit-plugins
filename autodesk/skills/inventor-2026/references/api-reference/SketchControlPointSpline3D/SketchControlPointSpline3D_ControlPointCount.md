# SketchControlPointSpline3D.ControlPointCount Property

Parent Object: [SketchControlPointSpline3D](../SketchControlPointSpline3D/SketchControlPointSpline3D.md)

## Description

Read-only property that returns the number of control points defining the control point spline.

## Syntax

SketchControlPointSpline3D.**ControlPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2014
