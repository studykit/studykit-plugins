# GeneralDataTypeEnum Enumerator

## Description

Enum for the different family parameter value types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBooleanGeneralData | 52481 | Implies the value is a boolean. |
| kCollectionGeneralData | 52482 | Implies the value is a collection. |
| kDoubleGeneralData | 52483 | Implies the value is a double. |
| kIDispatchGeneralData | 52486 | Implies the value is an IDispatch-based XML-DOM. |
| kIntegerGeneralData | 52484 | Implies the value is an integer. |
| kStringGeneralData | 52485 | Implies the value is a string. |
| kVTableGeneralData | 52487 | Implies the value is a v-table interface-based XML-DOM. |

## Version

Introduced in version 10
