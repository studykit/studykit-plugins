# ComponentOccurrenceProxy.PatternElement Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that returns the pattern element this occurrence represents. In the case where this occurrence is not part of a pattern this property returns Nothing. The IsPatternElement property can be used to check if this occurrence is part of a pattern.

## Syntax

ComponentOccurrenceProxy.**PatternElement**() As [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Property Value

This is a read only property whose value is an [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md).

## Version

Introduced in version 5
