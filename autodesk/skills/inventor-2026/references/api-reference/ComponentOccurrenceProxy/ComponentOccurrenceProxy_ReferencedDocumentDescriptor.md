# ComponentOccurrenceProxy.ReferencedDocumentDescriptor Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that returns an enumeration of descriptors that represent the native document references held by this document.

## Syntax

ComponentOccurrenceProxy.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 11
