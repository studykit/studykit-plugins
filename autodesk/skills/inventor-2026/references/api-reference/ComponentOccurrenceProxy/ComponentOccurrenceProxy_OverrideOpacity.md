# ComponentOccurrenceProxy.OverrideOpacity Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets and sets the opacity of an occurrence.

## Syntax

ComponentOccurrenceProxy.**OverrideOpacity**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
