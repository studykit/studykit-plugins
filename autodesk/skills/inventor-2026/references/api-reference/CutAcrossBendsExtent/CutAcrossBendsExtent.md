# CutAcrossBendsExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The CutAcrossBendsExtent object represents the extent definition for a sheet metal cut feature that cuts across the bends of the part.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CutAcrossBendsExtent/CutAcrossBendsExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Distance](../CutAcrossBendsExtent/CutAcrossBendsExtent_Distance.md) | Property that returns the parameter controlling the depth of the cut. |
| [Parent](../CutAcrossBendsExtent/CutAcrossBendsExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Type](../CutAcrossBendsExtent/CutAcrossBendsExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2009
