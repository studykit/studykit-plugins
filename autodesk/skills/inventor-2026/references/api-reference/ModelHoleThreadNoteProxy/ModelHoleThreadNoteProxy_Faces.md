# ModelHoleThreadNoteProxy.Faces Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Returns the model hole or thread feature faces associated with the hole or thread note.

## Syntax

ModelHoleThreadNoteProxy.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2023
