# FileOptions.DefaultTemplateDrawingStandard Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Gets and sets the standard for the default template.

## Syntax

FileOptions.**DefaultTemplateDrawingStandard**() As [DraftingStandardEnum](../DraftingStandardEnum.md)

## Property Value

This is a read/write property whose value is a [DraftingStandardEnum](../DraftingStandardEnum.md).

## Version

Introduced in version 2013
