# FileOptions.DefaultTemplateUnitsAreInches Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Gets and sets the units for the default template, the options are either inches or millimeters.

## Syntax

FileOptions.**DefaultTemplateUnitsAreInches**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
