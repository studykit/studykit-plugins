# GroundConstraintProxy.Type Property

Parent Object: [GroundConstraintProxy](../GroundConstraintProxy/GroundConstraintProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

GroundConstraintProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
