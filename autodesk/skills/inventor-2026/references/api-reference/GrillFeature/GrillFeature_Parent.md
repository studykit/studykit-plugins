# GrillFeature.Parent Property

Parent Object: [GrillFeature](../GrillFeature/GrillFeature.md)

## Description

Property that returns the parent of the feature.

## Syntax

GrillFeature.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
