# ModelGeneralNoteDefinition.Intent Property

Parent Object: [ModelGeneralNoteDefinition](../ModelGeneralNoteDefinition/ModelGeneralNoteDefinition.md)

## Description

Read-write property that gets and sets the geometric entity the note is attached to.

## Syntax

ModelGeneralNoteDefinition.**Intent**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2018
