# FactoryOptions.MemberNameIndexDigits Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the number of digits to use for the member name index.

## Syntax

FactoryOptions.**MemberNameIndexDigits**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
