# FactoryOptions.PartNumberIndexDigits Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the number of digits to use for the part number index.

## Syntax

FactoryOptions.**PartNumberIndexDigits**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
