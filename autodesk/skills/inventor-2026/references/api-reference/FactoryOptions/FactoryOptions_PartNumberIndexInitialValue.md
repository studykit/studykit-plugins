# FactoryOptions.PartNumberIndexInitialValue Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the starting value of the part number index.

## Syntax

FactoryOptions.**PartNumberIndexInitialValue**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
