# FactoryOptions.MemberNameIndexSeparator Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the separator for the member name index.

## Syntax

FactoryOptions.**MemberNameIndexSeparator**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
