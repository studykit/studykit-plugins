# DeleteFaceFeatures.Type Property

Parent Object: [DeleteFaceFeatures](../DeleteFaceFeatures/DeleteFaceFeatures.md)

## Syntax

DeleteFaceFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
