# DeleteFaceFeatures Object

## Description

The DeleteFaceFeatures object represents a collection of objects on a part.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../DeleteFaceFeatures/DeleteFaceFeatures_Add.md) | Method that creates a new DeleteFaceFeature. The new created DeleteFaceFeature is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DeleteFaceFeatures/DeleteFaceFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DeleteFaceFeatures/DeleteFaceFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../DeleteFaceFeatures/DeleteFaceFeatures_Item.md) | Returns the specified DeleteFaceFeature object from the collection. |
| [Type](../DeleteFaceFeatures/DeleteFaceFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.DeleteFaceFeatures](../PartFeatures/PartFeatures_DeleteFaceFeatures.md), [SheetMetalFeatures.DeleteFaceFeatures](../SheetMetalFeatures/SheetMetalFeatures_DeleteFaceFeatures.md)

## Version

Introduced in version 6
