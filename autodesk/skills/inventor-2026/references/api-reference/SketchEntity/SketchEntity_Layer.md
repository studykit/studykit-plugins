# SketchEntity.Layer Property

Parent Object: [SketchEntity](../SketchEntity/SketchEntity.md)

## Description

Gets and sets the layer applied to this sketch entity.

## Syntax

SketchEntity.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
