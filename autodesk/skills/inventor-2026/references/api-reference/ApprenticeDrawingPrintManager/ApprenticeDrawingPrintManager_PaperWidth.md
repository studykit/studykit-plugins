# ApprenticeDrawingPrintManager.PaperWidth Property

Parent Object: [ApprenticeDrawingPrintManager](../ApprenticeDrawingPrintManager/ApprenticeDrawingPrintManager.md)

## Description

Gets/Sets the width of the paper to be printed. Setting this, sets PaperSize to kPaperSizeCustom.

## Syntax

ApprenticeDrawingPrintManager.**PaperWidth**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
