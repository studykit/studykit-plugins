# ApprenticeDrawingPrintManager.PrintRange Property

Parent Object: [ApprenticeDrawingPrintManager](../ApprenticeDrawingPrintManager/ApprenticeDrawingPrintManager.md)

## Description

Gets/Sets information about which sheets are to be printed. Defaults to kPrintCurrentSheet.

## Syntax

ApprenticeDrawingPrintManager.**PrintRange**() As [PrintRangeEnum](../PrintRangeEnum.md)

## Property Value

This is a read/write property whose value is a [PrintRangeEnum](../PrintRangeEnum.md).

## Version

Introduced in version 9
