# SketchEquationCurve.LineScale Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Gets and sets the LineScale applied to this sketch spline.

## Syntax

SketchEquationCurve.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
