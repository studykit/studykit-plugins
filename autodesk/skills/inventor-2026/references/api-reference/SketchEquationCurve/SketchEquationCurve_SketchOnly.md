# SketchEquationCurve.SketchOnly Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Gets and sets whether this entity is visible only when editing the sketch.

## Syntax

SketchEquationCurve.**SketchOnly**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
