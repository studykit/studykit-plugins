# SketchEquationCurve.DisabledActionTypes Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Gets and sets whether specific user actions are blocked on this sketch geometry.

## Syntax

SketchEquationCurve.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 2014
