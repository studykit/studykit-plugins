# SketchEquationCurve.LineDefinitionSpace Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Gets and sets the LineDefinitionSpace applied to this sketch spline.

## Syntax

SketchEquationCurve.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2014
