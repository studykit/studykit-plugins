# SculptFeature.Parameters Property

Parent Object: [SculptFeature](../SculptFeature/SculptFeature.md)

## Description

Property that returns all the parameters associated with the feature.

## Syntax

SculptFeature.**Parameters**() As [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md)

## Property Value

This is a read only property whose value is a [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md).

## Version

Introduced in version 11
