# PresentationFacesEnumerator.Type Property

Parent Object: [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md)

## Description

## Syntax

PresentationFacesEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
