# MiniToolbarListItem.Text Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-write property that gets and sets the visible text for this item.

## Syntax

MiniToolbarListItem.**Text**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
