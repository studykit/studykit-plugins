# MiniToolbarListItem.IsSeperator Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-only property that indicates if this control is a separator or not.

## Syntax

MiniToolbarListItem.**IsSeperator**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
