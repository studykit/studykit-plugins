# OrientedBox.Contains Method

Parent Object: [OrientedBox](../OrientedBox/OrientedBox.md)

## Description

Determines whether the specified point is contained within this oriented box.

## Syntax

OrientedBox.**Contains**( ***Point*** As [Point](../Point/Point.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input Point object that specifies the coordinate to check. |

## Version

Introduced in version 2016
