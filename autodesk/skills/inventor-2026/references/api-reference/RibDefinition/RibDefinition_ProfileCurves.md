# RibDefinition.ProfileCurves Property

Parent Object: [RibDefinition](../RibDefinition/RibDefinition.md)

## Description

Read-write property that gets and sets the collection of sketch entities used as the profile for feature creation.

## Syntax

RibDefinition.**ProfileCurves**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
