# SurfaceGraphicsVertexList.Clear Method

Parent Object: [SurfaceGraphicsVertexList](../SurfaceGraphicsVertexList/SurfaceGraphicsVertexList.md)

## Description

Method that removes all the vertices from the list. No vertices will be displayed after the method is called.

## Syntax

SurfaceGraphicsVertexList.**Clear**()

## Version

Introduced in version 2013
