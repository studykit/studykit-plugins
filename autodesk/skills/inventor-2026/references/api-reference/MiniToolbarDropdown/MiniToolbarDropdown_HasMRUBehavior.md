# MiniToolbarDropdown.HasMRUBehavior Property

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Read-write property that specifies if this button displays the most recently selected item from the list.

## Syntax

MiniToolbarDropdown.**HasMRUBehavior**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
