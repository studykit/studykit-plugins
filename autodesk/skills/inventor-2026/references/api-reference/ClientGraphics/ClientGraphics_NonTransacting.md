# ClientGraphics.NonTransacting Property

Parent Object: [ClientGraphics](../ClientGraphics/ClientGraphics.md)

## Description

Read-only property that returns whether the creation of this ClientGraphics object and all its associated data is non-transacting.

## Syntax

ClientGraphics.**NonTransacting**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
