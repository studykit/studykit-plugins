# ObjectDefaultsStylesEnumerator.Count Property

Parent Object: [ObjectDefaultsStylesEnumerator](../ObjectDefaultsStylesEnumerator/ObjectDefaultsStylesEnumerator.md)

## Description

Property that returns the number of items in this collection.

## Syntax

ObjectDefaultsStylesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 9
