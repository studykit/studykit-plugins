# ColorSchemes.HighlightHiddenFaces Property

Parent Object: [ColorSchemes](../ColorSchemes/ColorSchemes.md)

## Description

Gets and sets whether enables or disables hidden faces highlighting.

## Syntax

ColorSchemes.**HighlightHiddenFaces**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
