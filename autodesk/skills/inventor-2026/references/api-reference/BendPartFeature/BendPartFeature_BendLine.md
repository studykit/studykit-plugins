# BendPartFeature.BendLine Property

Parent Object: [BendPartFeature](../BendPartFeature/BendPartFeature.md)

## Description

Property that gets and sets the sketch line that represents the bend line.

## Syntax

BendPartFeature.**BendLine**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read/write property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 2010
