# EdgeDefinition.EndVertex Property

Parent Object: [EdgeDefinition](../EdgeDefinition/EdgeDefinition.md)

## Description

Gets and sets the end vertex of the edge. This can be Nothing where no end vertex is specifically defined.

## Syntax

EdgeDefinition.**EndVertex**() As [VertexDefinition](../VertexDefinition/VertexDefinition.md)

## Property Value

This is a read/write property whose value is a [VertexDefinition](../VertexDefinition/VertexDefinition.md).

## Version

Introduced in version 2011
