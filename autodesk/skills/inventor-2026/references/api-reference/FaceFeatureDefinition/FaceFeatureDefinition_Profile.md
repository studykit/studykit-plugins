# FaceFeatureDefinition.Profile Property

Parent Object: [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md)

## Description

Gets and sets the Profile object that defines the shape of the face feature.

## Syntax

FaceFeatureDefinition.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 2009
