# FaceFeatureDefinition.Direction Property

Parent Object: [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md)

## Description

Gets and sets the extent direction which indicates which side of the sketch the projection of the face is in.

## Syntax

FaceFeatureDefinition.**Direction**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2009
