# ChangeDefinition.Parent Property

Parent Object: [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ChangeDefinition.**Parent**() As [ChangeManager](../ChangeManager/ChangeManager.md)

## Property Value

This is a read only property whose value is a [ChangeManager](../ChangeManager/ChangeManager.md).

## Version

Introduced in version 9
