# FileOpenOptions.DefaultDesignViewIsAssociativeInAssembly Property

Parent Object: [FileOpenOptions](../FileOpenOptions/FileOpenOptions.md)

## Description

Gets and sets whether the default design view representation is associative in assembly documents.

## Syntax

FileOpenOptions.**DefaultDesignViewIsAssociativeInAssembly**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
