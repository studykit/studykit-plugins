# ApplicationAddIn.AddInType Property

Parent Object: [ApplicationAddIn](../ApplicationAddIn/ApplicationAddIn.md)

## Description

Gets the constant that indicates the type of this AddIn.

## Syntax

ApplicationAddIn.**AddInType**() As [ApplicationAddInTypeEnum](../ApplicationAddInTypeEnum.md)

## Property Value

This is a read only property whose value is an [ApplicationAddInTypeEnum](../ApplicationAddInTypeEnum.md).

## Version

Introduced in version 4
