# MergedCell.EndCell Property

Parent Object: [MergedCell](../MergedCell/MergedCell.md)

## Description

Returns the end cell that is merged into this MergedCell.

## Syntax

MergedCell.**EndCell**() As [Cell](../Cell/Cell.md)

## Property Value

This is a read only property whose value is a [Cell](../Cell/Cell.md).

## Version

Introduced in version 2025.1
