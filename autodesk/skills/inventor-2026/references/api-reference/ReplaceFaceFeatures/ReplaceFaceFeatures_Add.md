# ReplaceFaceFeatures.Add Method

Parent Object: [ReplaceFaceFeatures](../ReplaceFaceFeatures/ReplaceFaceFeatures.md)

## Description

Method that creates a new ReplaceFaceFeature. The newly created ReplaceFaceFeature is returned.

## Syntax

ReplaceFaceFeatures.**Add**( ***Definition*** As [ReplaceFaceDefinition](../ReplaceFaceDefinition/ReplaceFaceDefinition.md) ) As [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ReplaceFaceDefinition](../ReplaceFaceDefinition/ReplaceFaceDefinition.md) | Input ReplaceFaceDefinition object created using the CreateDefinition method. |

## Version

Introduced in version 2024.1
