# ReferenceComponents.iFeatureTemplateDescriptors Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Property that returns the iFeatureTemplateDescriptors collection object. This collection provides access to existing iFeatureTemplateDescriptor objects. Note: iFeatureTemplateDescriptors was previously known as iFeatureDescriptors and were accessed through the DerivativeDescriptors collection. The DerivativeDescriptors collection has been removed and is now obsolete.

## Syntax

ReferenceComponents.**iFeatureTemplateDescriptors**() As [iFeatureTemplateDescriptors](../iFeatureTemplateDescriptors/iFeatureTemplateDescriptors.md)

## Property Value

This is a read only property whose value is an [iFeatureTemplateDescriptors](../iFeatureTemplateDescriptors/iFeatureTemplateDescriptors.md).

## Version

Introduced in version 5.3
