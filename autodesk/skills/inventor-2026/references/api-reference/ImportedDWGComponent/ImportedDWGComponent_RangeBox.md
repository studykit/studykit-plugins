# ImportedDWGComponent.RangeBox Property

Parent Object: [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Description

Read-only property that returns the bounding box of the DWG graphics.

## Syntax

ImportedDWGComponent.**RangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2018
