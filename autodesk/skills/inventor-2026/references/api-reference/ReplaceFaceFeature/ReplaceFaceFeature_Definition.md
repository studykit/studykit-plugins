# ReplaceFaceFeature.Definition Property

Parent Object: [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md)

## Description

Read-write property that gets and sets the ReplaceFaceDefinition object associated with this replace face feature.

## Syntax

ReplaceFaceFeature.**Definition**() As [ReplaceFaceDefinition](../ReplaceFaceDefinition/ReplaceFaceDefinition.md)

## Property Value

This is a read/write property whose value is a [ReplaceFaceDefinition](../ReplaceFaceDefinition/ReplaceFaceDefinition.md).

## Version

Introduced in version 2024.1
