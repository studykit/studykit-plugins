# LinearModelDimensionDefinition.IsOppositeAngleOfArc Property

Parent Object: [LinearModelDimensionDefinition](../LinearModelDimensionDefinition/LinearModelDimensionDefinition.md)

## Description

Read-write property that gets and sets whether the opposite angle of an arc is dimensioned or not. This is only applicable when an arc is dimensioned and the value of this property should be ignored in other cases.

## Syntax

LinearModelDimensionDefinition.**IsOppositeAngleOfArc**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
