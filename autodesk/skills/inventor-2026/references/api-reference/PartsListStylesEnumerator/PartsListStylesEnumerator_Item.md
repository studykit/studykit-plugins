# PartsListStylesEnumerator.Item Property

Parent Object: [PartsListStylesEnumerator](../PartsListStylesEnumerator/PartsListStylesEnumerator.md)

## Description

Returns the specified PartsListStyle object from the collection.

## Syntax

PartsListStylesEnumerator.**Item**( ***Index*** As Variant ) As [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Property Value

This is a read only property whose value is a [PartsListStyle](../PartsListStyle/PartsListStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the PartsListStyle to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the PartsListStyle name. If an out of range index or a name of a non-existent PartsListStyle is provided, an error will occur. |

## Version

Introduced in version 2011
