# UnfoldFeature.ResultSketches Property

Parent Object: [UnfoldFeature](../UnfoldFeature/UnfoldFeature.md)

## Description

Property that returns the sketches that were created as a \result of this feature. The ReferencedEntity property of the sketches returned will return the input sketch this result is based on.

## Syntax

UnfoldFeature.**ResultSketches**() As [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md).

## Version

Introduced in version 2010
