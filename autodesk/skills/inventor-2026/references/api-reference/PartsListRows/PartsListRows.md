# PartsListRows Object

## Description

The PartsListRows collection object provides access to all existing objects in the parts list.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../PartsListRows/PartsListRows_Add.md) | Method that creates a custom PartsListRow. The newly created PartsListRow is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PartsListRows/PartsListRows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../PartsListRows/PartsListRows_Count.md) | Property that returns the number of items in this collection. |
| [Item](../PartsListRows/PartsListRows_Item.md) | Returns the specified PartsListRows object from the collection. This is the default property of the PartsListRows collection object. |
| [Type](../PartsListRows/PartsListRows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
