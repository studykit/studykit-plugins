# ModelDatumIdentifierDefinition.ArrowheadType Property

Parent Object: [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md)

## Description

Read-write property that gets and sets the type of the arrowhead. This is a style override.

## Syntax

ModelDatumIdentifierDefinition.**ArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2018
