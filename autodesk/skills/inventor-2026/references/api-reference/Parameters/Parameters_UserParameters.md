# Parameters.UserParameters Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Property that returns the UserParameters collection object.

## Syntax

Parameters.**UserParameters**() As [UserParameters](../UserParameters/UserParameters.md)

## Property Value

This is a read only property whose value is a [UserParameters](../UserParameters/UserParameters.md).
