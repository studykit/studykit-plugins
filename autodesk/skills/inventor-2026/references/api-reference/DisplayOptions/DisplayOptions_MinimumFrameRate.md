# DisplayOptions.MinimumFrameRate Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets the number of frames to display for the model during interactive viewing operations like Rotate, Pan and Zoom.

## Syntax

DisplayOptions.**MinimumFrameRate**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
