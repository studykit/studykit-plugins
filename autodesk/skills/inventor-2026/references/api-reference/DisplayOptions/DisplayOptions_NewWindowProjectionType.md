# DisplayOptions.NewWindowProjectionType Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets the default projection type for new windows.

## Syntax

DisplayOptions.**NewWindowProjectionType**() As [ProjectionTypeEnum](../ProjectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ProjectionTypeEnum](../ProjectionTypeEnum.md).

## Version

Introduced in version 2010
