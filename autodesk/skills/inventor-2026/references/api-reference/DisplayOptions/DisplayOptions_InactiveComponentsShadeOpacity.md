# DisplayOptions.InactiveComponentsShadeOpacity Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that gets and sets the opacity level (expressed as a percentage) for the shading of inactive components.

## Syntax

DisplayOptions.**InactiveComponentsShadeOpacity**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2011
