# Sketch.SketchEllipses Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Property that returns the SketchEllipses collection object.

## Syntax

Sketch.**SketchEllipses**() As [SketchEllipses](../SketchEllipses/SketchEllipses.md)

## Property Value

This is a read only property whose value is a [SketchEllipses](../SketchEllipses/SketchEllipses.md).

## Version

Introduced in version 5.3
