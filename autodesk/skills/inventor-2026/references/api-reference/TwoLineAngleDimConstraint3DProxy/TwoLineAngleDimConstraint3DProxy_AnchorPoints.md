# TwoLineAngleDimConstraint3DProxy.AnchorPoints Property

Parent Object: [TwoLineAngleDimConstraint3DProxy](../TwoLineAngleDimConstraint3DProxy/TwoLineAngleDimConstraint3DProxy.md)

## Description

Gets the anchor points of dimension.

## Syntax

TwoLineAngleDimConstraint3DProxy.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
