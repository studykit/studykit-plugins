# LinearHolePlacementDefinition.DistanceTwo Property

Parent Object: [LinearHolePlacementDefinition](../LinearHolePlacementDefinition/LinearHolePlacementDefinition.md)

## Description

Property that returns the parameter controlling the distance of the hole center from the second reference entity.

## Syntax

LinearHolePlacementDefinition.**DistanceTwo**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 10
