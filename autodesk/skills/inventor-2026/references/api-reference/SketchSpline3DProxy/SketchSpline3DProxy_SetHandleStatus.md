# SketchSpline3DProxy.SetHandleStatus Method

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Method that sets whether the handle at a given fit point is enabled or disabled.

## Syntax

SketchSpline3DProxy.**SetHandleStatus**( ***FitPoint*** As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md), ***Value*** As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FitPoint | [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md) | Input SketchPoint3D object that specifies the fit point at which to set the handle status. |
| Value | Boolean | Input Boolean that specifies whether the handle at the input fit point is enabled or disabled. |

## Version

Introduced in version 2008
