# CosmeticWeldDefinition.ExtentType Property

Parent Object: [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md)

## Description

Read only property that gets and set the extent type for the weld.

## Syntax

CosmeticWeldDefinition.**ExtentType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).

## Version

Introduced in version 2017
