# UserInterfaceEvents.OnResetInventorLayout Event

Parent Object: [UserInterfaceEvents](../UserInterfaceEvents/UserInterfaceEvents.md)

## Description

Event that is fired when the Inventor layout is reset.

## Syntax

UserInterfaceEvents.**OnResetInventorLayout**( ***Context*** As [NameValueMap](../NameValueMap/NameValueMap.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Context | [NameValueMap](../NameValueMap/NameValueMap.md) | Input NameValueMap object that contains additional information about the event. No additional information is provided for this event. |

## Version

Introduced in version 2022
