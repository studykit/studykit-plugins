# TextureMap.Transformation Property

Parent Object: [TextureMap](../TextureMap/TextureMap.md)

## Description

A matrix that defines how the coordinates map to the actual face coordinates.

## Syntax

TextureMap.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 10
