# DWGAcadSupportedProxy.Parent Property

Parent Object: [DWGAcadSupportedProxy](../DWGAcadSupportedProxy/DWGAcadSupportedProxy.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DWGAcadSupportedProxy.**Parent**() As [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Property Value

This is a read only property whose value is an [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md).

## Version

Introduced in version 2018
