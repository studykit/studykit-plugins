# Profile3DProxy Object

Derived from: [Profile3D](../Profile3D/Profile3D.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../Profile3DProxy/Profile3DProxy_Delete.md) | Method that deletes this Profile3D object. |
| [GetReferenceKey](../Profile3DProxy/Profile3DProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Profile3DProxy/Profile3DProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../Profile3DProxy/Profile3DProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../Profile3DProxy/Profile3DProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Count](../Profile3DProxy/Profile3DProxy_Count.md) | Property that returns the number of items in the collection. |
| [Item](../Profile3DProxy/Profile3DProxy_Item.md) | Method that returns the specified ProfilePath3D object from the collection. |
| [NativeObject](../Profile3DProxy/Profile3DProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../Profile3DProxy/Profile3DProxy_Parent.md) | Property that returns the 3D sketch that the profile was derived from. |
| [Type](../Profile3DProxy/Profile3DProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Wires](../Profile3DProxy/Profile3DProxy_Wires.md) | Property returning the Wires collection object associated with this Profile3D. |

## Version

Introduced in version 6
