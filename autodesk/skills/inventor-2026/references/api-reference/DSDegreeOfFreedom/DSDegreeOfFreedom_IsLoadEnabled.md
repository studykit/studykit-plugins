# DSDegreeOfFreedom.IsLoadEnabled Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets and sets if the load settings for this joint are enabled.

## Syntax

DSDegreeOfFreedom.**IsLoadEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
