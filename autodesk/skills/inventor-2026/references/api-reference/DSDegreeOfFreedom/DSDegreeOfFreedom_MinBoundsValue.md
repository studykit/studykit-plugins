# DSDegreeOfFreedom.MinBoundsValue Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets the minimum value of the joint.

## Syntax

DSDegreeOfFreedom.**MinBoundsValue**() As [DSValue](../DSValue/DSValue.md)

## Property Value

This is a read only property whose value is a [DSValue](../DSValue/DSValue.md).

## Version

Introduced in version 2013
