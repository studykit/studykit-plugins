# DSDegreeOfFreedom.IsVelocityComputed Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets and sets if the velocity is computed.

## Syntax

DSDegreeOfFreedom.**IsVelocityComputed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
