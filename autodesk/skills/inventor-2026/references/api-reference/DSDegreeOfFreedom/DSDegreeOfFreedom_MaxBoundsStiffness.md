# DSDegreeOfFreedom.MaxBoundsStiffness Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets the maximum stiffness value. The value is in database units, where lengths are is always expressed in centimeters, angles in radians, and time in seconds.

## Syntax

DSDegreeOfFreedom.**MaxBoundsStiffness**() As [DSValue](../DSValue/DSValue.md)

## Property Value

This is a read only property whose value is a [DSValue](../DSValue/DSValue.md).

## Version

Introduced in version 2013
