# DSDegreeOfFreedom.InitialPosition Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets and sets the initial position. This property is writable when the DSDegreeOfFreedom object is transient and not associated with an existing joint.

## Syntax

DSDegreeOfFreedom.**InitialPosition**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2013
