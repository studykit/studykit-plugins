# OrdinateDimensionsEnumerator Object

## Description

Allows iteration of available OrdinateDimension objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../OrdinateDimensionsEnumerator/OrdinateDimensionsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../OrdinateDimensionsEnumerator/OrdinateDimensionsEnumerator_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../OrdinateDimensionsEnumerator/OrdinateDimensionsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[OrdinateDimensionSet.Members](../OrdinateDimensionSet/OrdinateDimensionSet_Members.md)

## Version

Introduced in version 2010
