# SurfaceBodyDefinition.Type Property

Parent Object: [SurfaceBodyDefinition](../SurfaceBodyDefinition/SurfaceBodyDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SurfaceBodyDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
