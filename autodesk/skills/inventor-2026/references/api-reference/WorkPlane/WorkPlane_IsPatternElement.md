# WorkPlane.IsPatternElement Property

Parent Object: [WorkPlane](../WorkPlane/WorkPlane.md)

## Description

Property that gets whether the work plane was created by a pattern. If so, edits and delete are not allowed.

## Syntax

WorkPlane.**IsPatternElement**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
