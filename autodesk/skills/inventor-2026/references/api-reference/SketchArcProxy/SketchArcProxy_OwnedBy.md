# SketchArcProxy.OwnedBy Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Indicates entity or entities that own this object.

## Syntax

SketchArcProxy.**OwnedBy**() As [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md).

## Version

Introduced in version 11
