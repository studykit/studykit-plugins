# FilletConstantRadiusFaceSet.BiasPoint Property

Parent Object: [FilletConstantRadiusFaceSet](../FilletConstantRadiusFaceSet/FilletConstantRadiusFaceSet.md)

## Description

Optional input Point object that specifies a bias point to help choose between solutions if there are multiple possibilities.

## Syntax

FilletConstantRadiusFaceSet.**BiasPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 11
