# PointCloudScan.Visible Property

Parent Object: [PointCloudScan](../PointCloudScan/PointCloudScan.md)

## Description

Read-write property that gets and set the visibility of the point cloud scan object.

## Syntax

PointCloudScan.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
