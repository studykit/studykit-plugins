# SurfaceTextureGBDefinition.LayDirection Property

Parent Object: [SurfaceTextureGBDefinition](../SurfaceTextureGBDefinition/SurfaceTextureGBDefinition.md)

## Description

Gets and sets the direction of lay.

## Syntax

SurfaceTextureGBDefinition.**LayDirection**() As [LayDirectionTypeEnum](../LayDirectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LayDirectionTypeEnum](../LayDirectionTypeEnum.md).

## Version

Introduced in version 2016
