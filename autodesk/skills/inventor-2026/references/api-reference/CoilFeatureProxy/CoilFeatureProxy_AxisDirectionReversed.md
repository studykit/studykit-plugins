# CoilFeatureProxy.AxisDirectionReversed Property

Parent Object: [CoilFeatureProxy](../CoilFeatureProxy/CoilFeatureProxy.md)

## Description

Property that gets and sets whether the axis direction of the coil is reversed.

## Syntax

CoilFeatureProxy.**AxisDirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
