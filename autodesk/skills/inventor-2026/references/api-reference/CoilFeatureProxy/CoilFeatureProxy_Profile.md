# CoilFeatureProxy.Profile Property

Parent Object: [CoilFeatureProxy](../CoilFeatureProxy/CoilFeatureProxy.md)

## Description

Property that gets and sets the profile that defines the shape of the coil.

## Syntax

CoilFeatureProxy.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 9
