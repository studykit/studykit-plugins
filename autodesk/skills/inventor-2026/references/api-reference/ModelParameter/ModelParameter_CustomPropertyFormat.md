# ModelParameter.CustomPropertyFormat Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Property that returns the CustomPropertyFormat object associated with this parameter. The CustomPropertyFormat provides control over how the parameter value is formatted when it is exposed as a custom iProperty.

## Syntax

ModelParameter.**CustomPropertyFormat**() As [CustomPropertyFormat](../CustomPropertyFormat/CustomPropertyFormat.md)

## Property Value

This is a read only property whose value is a [CustomPropertyFormat](../CustomPropertyFormat/CustomPropertyFormat.md).

## Version

Introduced in version 2010
