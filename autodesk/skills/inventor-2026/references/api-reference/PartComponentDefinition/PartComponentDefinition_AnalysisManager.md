# PartComponentDefinition.AnalysisManager Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the AnalysisManager object.

## Syntax

PartComponentDefinition.**AnalysisManager**() As [AnalysisManager](../AnalysisManager/AnalysisManager.md)

## Property Value

This is a read only property whose value is an [AnalysisManager](../AnalysisManager/AnalysisManager.md).
