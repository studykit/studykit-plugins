# BIMComponentPropertySets Object

## Description

BIMComponentPropertySets object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BIMComponentPropertySets/BIMComponentPropertySets_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BIMComponentPropertySets/BIMComponentPropertySets_Count.md) | Read-only property that returns the total number of BIM component property sets this BIM document. |
| [Item](../BIMComponentPropertySets/BIMComponentPropertySets_Item.md) | Returns the specified BIMComponentPropertySet object from the collection. This collection is empty until the component type is defined using the ComponentType property of the BIMComponentDescription object. |
| [Type](../BIMComponentPropertySets/BIMComponentPropertySets_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BIMComponentDescription.ComponentPropertySets](../BIMComponentDescription/BIMComponentDescription_ComponentPropertySets.md)

## Version

Introduced in version 2011
