# PunchNote.LineSpacingType Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets/Sets the LineSpacingType used by the DrawingNote.

## Syntax

PunchNote.**LineSpacingType**() As [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md)

## Property Value

This is a read/write property whose value is a [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md).

## Version

Introduced in version 2010
