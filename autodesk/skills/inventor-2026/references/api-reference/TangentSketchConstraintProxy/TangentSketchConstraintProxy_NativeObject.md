# TangentSketchConstraintProxy.NativeObject Property

Parent Object: [TangentSketchConstraintProxy](../TangentSketchConstraintProxy/TangentSketchConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

TangentSketchConstraintProxy.**NativeObject**() As [TangentSketchConstraint](../TangentSketchConstraint/TangentSketchConstraint.md)

## Property Value

This is a read only property whose value is a [TangentSketchConstraint](../TangentSketchConstraint/TangentSketchConstraint.md).

## Version

Introduced in version 6
