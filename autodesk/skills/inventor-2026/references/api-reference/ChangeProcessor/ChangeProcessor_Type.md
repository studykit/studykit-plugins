# ChangeProcessor.Type Property

Parent Object: [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ChangeProcessor.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
