# ChangeProcessor.MergeWithPrevious Property

Parent Object: [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Description

Gets and sets whether to merge this change process with the previously committed transaction.

## Syntax

ChangeProcessor.**MergeWithPrevious**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
