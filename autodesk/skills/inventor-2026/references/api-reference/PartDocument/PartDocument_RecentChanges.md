# PartDocument.RecentChanges Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets a bit-encoded value where the bits indicate the kind of changes made to the document since it became dirty.

## Syntax

PartDocument.**RecentChanges**() As [CommandTypesEnum](../CommandTypesEnum.md)

## Property Value

This is a read only property whose value is a [CommandTypesEnum](../CommandTypesEnum.md).

## Version

Introduced in version 8
