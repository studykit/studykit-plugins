# PartDocument.DisplayNameOverridden Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Read-write property that gets and sets whether the display name of the document has been overridden.

## Syntax

PartDocument.**DisplayNameOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
