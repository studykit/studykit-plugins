# PartDocument.ReferencedOpaqueFileDescriptors Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the collection of descriptor objects, presenting (editable) information about opaque references held.

## Syntax

PartDocument.**ReferencedOpaqueFileDescriptors**() As [ReferencedOpaqueFileDescriptors](../ReferencedOpaqueFileDescriptors/ReferencedOpaqueFileDescriptors.md)

## Property Value

This is a read only property whose value is a [ReferencedOpaqueFileDescriptors](../ReferencedOpaqueFileDescriptors/ReferencedOpaqueFileDescriptors.md).

## Version

Introduced in version 2017
