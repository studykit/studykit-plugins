# PartDocument.ActiveLightingStyle Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the active lighting style object.

## Syntax

PartDocument.**ActiveLightingStyle**() As [LightingStyle](../LightingStyle/LightingStyle.md)

## Property Value

This is a read/write property whose value is a [LightingStyle](../LightingStyle/LightingStyle.md).

## Version

Introduced in version 10
