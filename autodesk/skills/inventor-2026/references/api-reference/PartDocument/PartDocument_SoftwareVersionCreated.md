# PartDocument.SoftwareVersionCreated Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the object that encapsulates the version of the software with which this document was first created.

## Syntax

PartDocument.**SoftwareVersionCreated**() As [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Property Value

This is a read only property whose value is a [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md).

## Version

Introduced in version 4
