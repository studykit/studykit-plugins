# RevitExportDefinition.Location Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the location of the RevitExport. This cannot be modified after the Revit model has been created.

## Syntax

RevitExportDefinition.**Location**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
