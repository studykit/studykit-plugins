# RevitExportDefinition.ActiveModelState Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the name of the active model state for the RevitExport.

## Syntax

RevitExportDefinition.**ActiveModelState**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
