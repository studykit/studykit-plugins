# RevitExportDefinition.RemovePartsBySize Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets whether remove parts by size or not.

## Syntax

RevitExportDefinition.**RemovePartsBySize**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
