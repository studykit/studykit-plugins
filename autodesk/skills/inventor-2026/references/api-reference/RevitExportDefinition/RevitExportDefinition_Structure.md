# RevitExportDefinition.Structure Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the structure.

## Syntax

RevitExportDefinition.**Structure**() As [RevitExportStructureTypeEnum](../RevitExportStructureTypeEnum.md)

## Property Value

This is a read/write property whose value is a [RevitExportStructureTypeEnum](../RevitExportStructureTypeEnum.md).

## Version

Introduced in version 2022
