# RevolutionAndHeightCoilExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The RevolutionAndHeightExtent object provides access to the information that defines the variables for a coil feature created using revolution and height values.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevolutionAndHeightCoilExtent/RevolutionAndHeightCoilExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Height](../RevolutionAndHeightCoilExtent/RevolutionAndHeightCoilExtent_Height.md) | Property that returns the parameter that controls the height of the coil feature. |
| [Parent](../RevolutionAndHeightCoilExtent/RevolutionAndHeightCoilExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Revolution](../RevolutionAndHeightCoilExtent/RevolutionAndHeightCoilExtent_Revolution.md) | Property that returns the parameter that controls the number of revolutions of the coil feature. |
| [TaperAngle](../RevolutionAndHeightCoilExtent/RevolutionAndHeightCoilExtent_TaperAngle.md) | Property that returns the parameter that controls the taper angle of the coil feature. |
| [Type](../RevolutionAndHeightCoilExtent/RevolutionAndHeightCoilExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 5.3
