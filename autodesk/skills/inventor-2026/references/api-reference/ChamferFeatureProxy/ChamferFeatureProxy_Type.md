# ChamferFeatureProxy.Type Property

Parent Object: [ChamferFeatureProxy](../ChamferFeatureProxy/ChamferFeatureProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ChamferFeatureProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
