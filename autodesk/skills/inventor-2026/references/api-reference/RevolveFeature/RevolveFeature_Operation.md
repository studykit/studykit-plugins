# RevolveFeature.Operation Property

Parent Object: [RevolveFeature](../RevolveFeature/RevolveFeature.md)

## Description

Property that gets and sets the type of operation used to add the feature to the model. Valid input is kJoinOperation, kCutOperation, kIntersectOperation, or kSurfaceOperation.

## Syntax

RevolveFeature.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 5
