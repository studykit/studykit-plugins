# ConcentricConstraint.EntityTwo Property

Parent Object: [ConcentricConstraint](../ConcentricConstraint/ConcentricConstraint.md)

## Description

Gets the sketch entity being constrained.

## Syntax

ConcentricConstraint.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
