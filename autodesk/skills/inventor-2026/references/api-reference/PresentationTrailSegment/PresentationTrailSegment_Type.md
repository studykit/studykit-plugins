# PresentationTrailSegment.Type Property

Parent Object: [PresentationTrailSegment](../PresentationTrailSegment/PresentationTrailSegment.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

PresentationTrailSegment.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
