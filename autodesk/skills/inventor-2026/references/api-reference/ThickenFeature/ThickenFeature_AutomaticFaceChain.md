# ThickenFeature.AutomaticFaceChain Property

Parent Object: [ThickenFeature](../ThickenFeature/ThickenFeature.md)

## Description

Property that gets whether or not to perform chaining of tangent continuous faces.

## Syntax

ThickenFeature.**AutomaticFaceChain**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
