# ThickenFeature.ApproximationOptimized Property

Parent Object: [ThickenFeature](../ThickenFeature/ThickenFeature.md)

## Description

Gets and sets whether the approximation is optimized.

## Syntax

ThickenFeature.**ApproximationOptimized**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
