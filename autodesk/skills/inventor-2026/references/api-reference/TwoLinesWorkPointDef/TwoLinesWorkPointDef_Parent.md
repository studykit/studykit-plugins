# TwoLinesWorkPointDef.Parent Property

Parent Object: [TwoLinesWorkPointDef](../TwoLinesWorkPointDef/TwoLinesWorkPointDef.md)

## Description

Property returning the parent WorkPoint object.

## Syntax

TwoLinesWorkPointDef.**Parent**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 4
