# ModelSurfaceTextureSymbolProxy.NativeObject Property

Parent Object: [ModelSurfaceTextureSymbolProxy](../ModelSurfaceTextureSymbolProxy/ModelSurfaceTextureSymbolProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ModelSurfaceTextureSymbolProxy.**NativeObject**() As [ModelSurfaceTextureSymbol](../ModelSurfaceTextureSymbol/ModelSurfaceTextureSymbol.md)

## Property Value

This is a read only property whose value is a [ModelSurfaceTextureSymbol](../ModelSurfaceTextureSymbol/ModelSurfaceTextureSymbol.md).

## Version

Introduced in version 2018
