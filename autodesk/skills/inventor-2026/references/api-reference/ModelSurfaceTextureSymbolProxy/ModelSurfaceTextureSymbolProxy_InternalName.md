# ModelSurfaceTextureSymbolProxy.InternalName Property

Parent Object: [ModelSurfaceTextureSymbolProxy](../ModelSurfaceTextureSymbolProxy/ModelSurfaceTextureSymbolProxy.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelSurfaceTextureSymbolProxy.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
