# SketchBlockDefinition.CircularPatterns Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets the SketchCircularPatterns collection object.

## Syntax

SketchBlockDefinition.**CircularPatterns**() As [SketchCircularPatterns](../SketchCircularPatterns/SketchCircularPatterns.md)

## Property Value

This is a read only property whose value is a [SketchCircularPatterns](../SketchCircularPatterns/SketchCircularPatterns.md).

## Version

Introduced in version 2025.1
