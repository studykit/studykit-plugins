# SketchBlockDefinition.SketchToModelSpace Method

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Method that takes a 2D coordinate in sketch space, and returns a Point3d containing the coordinates of the point in model space.

## Syntax

SketchBlockDefinition.**SketchToModelSpace**( ***SketchCoordinate*** As [Point2d](../Point2d/Point2d.md) ) As [Point](../Point/Point.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SketchCoordinate | [Point2d](../Point2d/Point2d.md) | Input Point2d that specifies the 2d coordinate in sketch space. |

## Version

Introduced in version 2010
