# SketchPatternElement.Index Property

Parent Object: [SketchPatternElement](../SketchPatternElement/SketchPatternElement.md)

## Description

Returns the index of this element within the pattern.

## Syntax

SketchPatternElement.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2025.1
