# FinishParameters.Type Property

Parent Object: [FinishParameters](../FinishParameters/FinishParameters.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

FinishParameters.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2024
