# ModelStateTable.TableColumns Property

Parent Object: [ModelStateTable](../ModelStateTable/ModelStateTable.md)

## Description

Read-only property that returns the ModelStateTableColumns collection object.

## Syntax

ModelStateTable.**TableColumns**() As [ModelStateTableColumns](../ModelStateTableColumns/ModelStateTableColumns.md)

## Property Value

This is a read only property whose value is a [ModelStateTableColumns](../ModelStateTableColumns/ModelStateTableColumns.md).

## Version

Introduced in version 2022
