# DWGBlockReference.Definition Property

Parent Object: [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md)

## Description

Read-only property that returns the definition object that this DWGBlockReference is an instance of.

## Syntax

DWGBlockReference.**Definition**() As [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Property Value

This is a read only property whose value is a [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md).

## Version

Introduced in version 2016
