# FlangeFeature.Adaptive Property

Parent Object: [FlangeFeature](../FlangeFeature/FlangeFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

FlangeFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
