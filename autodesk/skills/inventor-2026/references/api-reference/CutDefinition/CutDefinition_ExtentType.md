# CutDefinition.ExtentType Property

Parent Object: [CutDefinition](../CutDefinition/CutDefinition.md)

## Description

Property that returns an enum value indicating how the extent of the feature is defined.

## Syntax

CutDefinition.**ExtentType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).

## Version

Introduced in version 2009
