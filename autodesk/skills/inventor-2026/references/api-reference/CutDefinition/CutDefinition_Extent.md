# CutDefinition.Extent Property

Parent Object: [CutDefinition](../CutDefinition/CutDefinition.md)

## Description

Property that returns the PartFeatureExtent object that defines how the extent of the feature is defined.

## Syntax

CutDefinition.**Extent**() As [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md).

## Version

Introduced in version 2009
