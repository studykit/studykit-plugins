# ReferenceFeature.Parameters Property

Parent Object: [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md)

## Description

Property that returns all the parameters associated with the feature.

## Syntax

ReferenceFeature.**Parameters**() As [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md)

## Property Value

This is a read only property whose value is a [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md).

## Version

Introduced in version 11
