# EdgeLoopProxy.NativeObject Property

Parent Object: [EdgeLoopProxy](../EdgeLoopProxy/EdgeLoopProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

EdgeLoopProxy.**NativeObject**() As [EdgeLoop](../EdgeLoop/EdgeLoop.md)

## Property Value

This is a read only property whose value is an [EdgeLoop](../EdgeLoop/EdgeLoop.md).

## Version

Introduced in version 4
