# SurfaceGraphicsEdge.Selectable Property

Parent Object: [SurfaceGraphicsEdge](../SurfaceGraphicsEdge/SurfaceGraphicsEdge.md)

## Description

Read-write property that specifies whether the surface graphics edge can be selected or not.

## Syntax

SurfaceGraphicsEdge.**Selectable**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
