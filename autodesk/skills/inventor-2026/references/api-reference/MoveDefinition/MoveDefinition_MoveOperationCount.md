# MoveDefinition.MoveOperationCount Property

Parent Object: [MoveDefinition](../MoveDefinition/MoveDefinition.md)

## Description

Property that returns the number of move operations associated with this definition. The MoveOperation property can be used to access a specific operation.

## Syntax

MoveDefinition.**MoveOperationCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2013
