# SimplificationRemoveStyleEnum Enumerator

## Description

Constants identifying how embosses/holes/chamfers/fillets/pockets/tunnels are removed.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kSimplificationRemoveAll | 118786 | Remove all. |
| kSimplificationRemoveByRange | 118787 | Remove in range. |
| kSimplificationRemoveNone | 118785 | Do not remove any. |

## Version

Introduced in version 2022
