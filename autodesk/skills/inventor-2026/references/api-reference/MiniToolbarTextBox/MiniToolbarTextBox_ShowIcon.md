# MiniToolbarTextBox.ShowIcon Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Gets and sets whether the icon associated with the currently selected item should be displayed on the face of the control.

## Syntax

MiniToolbarTextBox.**ShowIcon**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
