# MiniToolbarTextBox.SelectedIndex Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Gets and sets the index of the item currently selected.

## Syntax

MiniToolbarTextBox.**SelectedIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2018
