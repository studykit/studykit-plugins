# ModelDatumIdentifier.Definition Property

Parent Object: [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md)

## Description

Read-write property that gets and sets the definition associated with this symbol.

## Syntax

ModelDatumIdentifier.**Definition**() As [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md)

## Property Value

This is a read/write property whose value is a [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md).

## Version

Introduced in version 2018
