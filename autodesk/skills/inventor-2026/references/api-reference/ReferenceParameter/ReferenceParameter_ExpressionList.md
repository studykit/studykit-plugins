# ReferenceParameter.ExpressionList Property

Parent Object: [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Description

Property that returns the expression list associated with this parameter.

## Syntax

ReferenceParameter.**ExpressionList**() As [ExpressionList](../ExpressionList/ExpressionList.md)

## Property Value

This is a read only property whose value is an [ExpressionList](../ExpressionList/ExpressionList.md).

## Version

Introduced in version 2011
