# DerivedFutureAssemblyComponent.SuppressLinkToFile Property

Parent Object: [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md)

## Description

Read-write property that suppresses or unsuppresses the link to the base assembly.

## Syntax

DerivedFutureAssemblyComponent.**SuppressLinkToFile**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
