# Circle Object

## Description

The mathematical Circle object, one of the transient geometry objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Circle/Circle_Copy.md) | Creates a copy of this Circle object. The result is entirely independent and can be edited without affecting the original Circle object. |
| [GetCircleData](../Circle/Circle_GetCircleData.md) | Get the data defining this circle. |
| [PutCircleData](../Circle/Circle_PutCircleData.md) | Method that sets the data defining this circle. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Center](../Circle/Circle_Center.md) | The center Point of the circle. |
| [Evaluator](../Circle/Circle_Evaluator.md) | Gets the CurveEvaluator for this circle. |
| [Normal](../Circle/Circle_Normal.md) | The normal of the circle. |
| [Radius](../Circle/Circle_Radius.md) | The radius of the circle. |
