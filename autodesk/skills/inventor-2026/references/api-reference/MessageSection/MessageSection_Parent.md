# MessageSection.Parent Property

Parent Object: [MessageSection](../MessageSection/MessageSection.md)

## Description

Property that returns the parent ErrorManager object.

## Syntax

MessageSection.**Parent**() As [ErrorManager](../ErrorManager/ErrorManager.md)

## Property Value

This is a read only property whose value is an [ErrorManager](../ErrorManager/ErrorManager.md).

## Version

Introduced in version 2010
