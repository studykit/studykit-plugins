# DWGBlockDefinitionProxy.BlockReferences Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGBlockReferencesEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**BlockReferences**() As [DWGBlockReferencesEnumerator](../DWGBlockReferencesEnumerator/DWGBlockReferencesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGBlockReferencesEnumerator](../DWGBlockReferencesEnumerator/DWGBlockReferencesEnumerator.md).

## Version

Introduced in version 2016
