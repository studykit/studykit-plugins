# DWGBlockDefinitionProxy.NativeObject Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGBlockDefinitionProxy.**NativeObject**() As [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Property Value

This is a read only property whose value is a [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md).

## Version

Introduced in version 2016
