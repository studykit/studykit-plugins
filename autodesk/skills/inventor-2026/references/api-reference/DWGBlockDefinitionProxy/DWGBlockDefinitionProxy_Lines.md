# DWGBlockDefinitionProxy.Lines Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGLinesEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**Lines**() As [DWGLinesEnumerator](../DWGLinesEnumerator/DWGLinesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGLinesEnumerator](../DWGLinesEnumerator/DWGLinesEnumerator.md).

## Version

Introduced in version 2016
