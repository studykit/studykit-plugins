# DWGBlockDefinitionProxy.EllipticalArcs Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGEllipticalArcsEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**EllipticalArcs**() As [DWGEllipticalArcsEnumerator](../DWGEllipticalArcsEnumerator/DWGEllipticalArcsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEllipticalArcsEnumerator](../DWGEllipticalArcsEnumerator/DWGEllipticalArcsEnumerator.md).

## Version

Introduced in version 2016
