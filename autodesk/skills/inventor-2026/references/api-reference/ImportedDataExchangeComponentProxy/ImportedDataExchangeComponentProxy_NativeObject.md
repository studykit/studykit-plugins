# ImportedDataExchangeComponentProxy.NativeObject Property

Parent Object: [ImportedDataExchangeComponentProxy](../ImportedDataExchangeComponentProxy/ImportedDataExchangeComponentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

ImportedDataExchangeComponentProxy.**NativeObject**() As [ImportedDataExchangeComponent](../ImportedDataExchangeComponent/ImportedDataExchangeComponent.md)

## Property Value

This is a read only property whose value is an [ImportedDataExchangeComponent](../ImportedDataExchangeComponent/ImportedDataExchangeComponent.md).

## Version

Introduced in version 2023
