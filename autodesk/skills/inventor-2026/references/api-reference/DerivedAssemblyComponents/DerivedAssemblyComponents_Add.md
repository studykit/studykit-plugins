# DerivedAssemblyComponents.Add Method

Parent Object: [DerivedAssemblyComponents](../DerivedAssemblyComponents/DerivedAssemblyComponents.md)

## Description

Method that creates a new using the information supplied by the input DerivedAssemblyDefinition object. If successful, the new DerivedAssemblyComponent is returned.

## Syntax

DerivedAssemblyComponents.**Add**( ***Definition*** As [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md) ) As [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md) | Input object that defines the derived component. |
