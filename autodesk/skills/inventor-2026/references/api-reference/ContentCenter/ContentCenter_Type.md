# ContentCenter.Type Property

Parent Object: [ContentCenter](../ContentCenter/ContentCenter.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ContentCenter.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
