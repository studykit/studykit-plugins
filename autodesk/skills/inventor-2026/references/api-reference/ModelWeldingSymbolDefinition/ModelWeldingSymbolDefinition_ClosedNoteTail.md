# ModelWeldingSymbolDefinition.ClosedNoteTail Property

Parent Object: [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Description

Gets and sets whether to enclose the note text in a box or not.

## Syntax

ModelWeldingSymbolDefinition.**ClosedNoteTail**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
