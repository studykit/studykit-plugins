# DWGEntityEllipticalArcSegment.Geometry Property

Parent Object: [DWGEntityEllipticalArcSegment](../DWGEntityEllipticalArcSegment/DWGEntityEllipticalArcSegment.md)

## Description

Read-only property that returns the segment geometry.

## Syntax

DWGEntityEllipticalArcSegment.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
