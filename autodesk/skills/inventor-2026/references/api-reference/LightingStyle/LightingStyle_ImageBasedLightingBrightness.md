# LightingStyle.ImageBasedLightingBrightness Property

Parent Object: [LightingStyle](../LightingStyle/LightingStyle.md)

## Description

Gets and sets the brightness of the lighting style.

## Syntax

LightingStyle.**ImageBasedLightingBrightness**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
