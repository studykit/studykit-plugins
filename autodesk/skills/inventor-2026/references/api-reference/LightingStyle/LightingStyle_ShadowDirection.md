# LightingStyle.ShadowDirection Property

Parent Object: [LightingStyle](../LightingStyle/LightingStyle.md)

## Description

Gets and sets the direction of the light source that controls the shadow.

## Syntax

LightingStyle.**ShadowDirection**() As [ShadowDirectionEnum](../ShadowDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [ShadowDirectionEnum](../ShadowDirectionEnum.md).

## Version

Introduced in version 2011
