# LightingStyle.ImageBasedLightingRotation Property

Parent Object: [LightingStyle](../LightingStyle/LightingStyle.md)

## Description

Gets and sets the rotation to be applied to the image.

## Syntax

LightingStyle.**ImageBasedLightingRotation**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
