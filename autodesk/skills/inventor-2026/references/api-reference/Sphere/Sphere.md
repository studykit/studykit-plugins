# Sphere Object

## Description

The Sphere object. The object created is a transient mathematical object and is not displayed graphically.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Sphere/Sphere_Copy.md) | Creates a copy of this Sphere object. The result is entirely independent and can be edited without affecting the original Sphere object. |
| [GetSphereData](../Sphere/Sphere_GetSphereData.md) | Get the data defining this sphere. |
| [PutSphereData](../Sphere/Sphere_PutSphereData.md) | Method that sets the data defining this sphere. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [CenterPoint](../Sphere/Sphere_CenterPoint.md) | Specifies the center of the sphere. |
| [Evaluator](../Sphere/Sphere_Evaluator.md) | Gets the surface evaluator for this sphere. |
| [Radius](../Sphere/Sphere_Radius.md) | Specifies the radius of the sphere. |

## Accessed From

[Sphere.Copy](../Sphere/Sphere_Copy.md), [TransientGeometry.CreateSphere](../TransientGeometry/TransientGeometry_CreateSphere.md)

## Version

Introduced in version 4
