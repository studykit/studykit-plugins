# PresentationMeshFeatureEntity.ColorBinding Property

Parent Object: [PresentationMeshFeatureEntity](../PresentationMeshFeatureEntity/PresentationMeshFeatureEntity.md)

## Description

Read-only property that returns how the colors are defined for the mesh entity.

## Syntax

PresentationMeshFeatureEntity.**ColorBinding**() As [ColorBindingEnum](../ColorBindingEnum.md)

## Property Value

This is a read only property whose value is a [ColorBindingEnum](../ColorBindingEnum.md).

## Version

Introduced in version 2018
