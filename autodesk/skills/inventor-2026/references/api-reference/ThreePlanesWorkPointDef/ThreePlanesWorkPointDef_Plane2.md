# ThreePlanesWorkPointDef.Plane2 Property

Parent Object: [ThreePlanesWorkPointDef](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef.md)

## Description

Property that returns the second plane of a work point that is defined by the intersection of three planes. This object can be a planar Face, WorkPlane, or Sketch object.

## Syntax

ThreePlanesWorkPointDef.**Plane2**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
