# LinearGeneralDimension.Text Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets the DimensionText setting.

## Syntax

LinearGeneralDimension.**Text**() As [DimensionText](../DimensionText/DimensionText.md)

## Property Value

This is a read/write property whose value is a [DimensionText](../DimensionText/DimensionText.md).

## Version

Introduced in version 11
