# LinearGeneralDimension.DimensionType Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Property that returns the dimension type. Possible values are kAlignedDimensionType, kHorizontalDimensionType, kVerticalDimensionType, kArcLengthDimensionType, kSymmetricDimensionType and kDiametricDimensionType.

## Syntax

LinearGeneralDimension.**DimensionType**() As [DimensionTypeEnum](../DimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [DimensionTypeEnum](../DimensionTypeEnum.md).

## Version

Introduced in version 11
