# LinearGeneralDimension.ExtensionLineTwoVisible Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets whether the second extension line is visible.

## Syntax

LinearGeneralDimension.**ExtensionLineTwoVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
