# LinearGeneralDimension.HideSecondArrowhead Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets whether hide the second arrowhead of the dimension.

## Syntax

LinearGeneralDimension.**HideSecondArrowhead**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
