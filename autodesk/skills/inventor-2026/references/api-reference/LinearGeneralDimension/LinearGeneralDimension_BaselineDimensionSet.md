# LinearGeneralDimension.BaselineDimensionSet Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Property that returns the BaselineDimensionSet that owns this dimension.

## Syntax

LinearGeneralDimension.**BaselineDimensionSet**() As [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md)

## Property Value

This is a read only property whose value is a [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md).

## Version

Introduced in version 2010
