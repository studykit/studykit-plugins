# TangentDistanceDimConstraintProxy.AnchorPoints Property

Parent Object: [TangentDistanceDimConstraintProxy](../TangentDistanceDimConstraintProxy/TangentDistanceDimConstraintProxy.md)

## Description

Gets the anchor points of dimension.

## Syntax

TangentDistanceDimConstraintProxy.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
