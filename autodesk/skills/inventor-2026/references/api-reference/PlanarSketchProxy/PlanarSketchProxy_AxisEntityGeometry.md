# PlanarSketchProxy.AxisEntityGeometry Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that gets the geometry that describes the axis entity.

## Syntax

PlanarSketchProxy.**AxisEntityGeometry**() As [Line](../Line/Line.md)

## Property Value

This is a read only property whose value is a [Line](../Line/Line.md).

## Version

Introduced in version 6
