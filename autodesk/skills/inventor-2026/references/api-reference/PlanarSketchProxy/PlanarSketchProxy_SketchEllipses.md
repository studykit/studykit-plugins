# PlanarSketchProxy.SketchEllipses Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that returns the SketchEllipses collection object.

## Syntax

PlanarSketchProxy.**SketchEllipses**() As [SketchEllipses](../SketchEllipses/SketchEllipses.md)

## Property Value

This is a read only property whose value is a [SketchEllipses](../SketchEllipses/SketchEllipses.md).

## Version

Introduced in version 6
