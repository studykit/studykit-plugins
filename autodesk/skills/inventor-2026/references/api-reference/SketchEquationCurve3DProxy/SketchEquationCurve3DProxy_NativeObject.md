# SketchEquationCurve3DProxy.NativeObject Property

Parent Object: [SketchEquationCurve3DProxy](../SketchEquationCurve3DProxy/SketchEquationCurve3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchEquationCurve3DProxy.**NativeObject**() As [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Property Value

This is a read only property whose value is a [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md).

## Version

Introduced in version 2014
