# OffsetDimConstraintProxy.TextPoint Property

Parent Object: [OffsetDimConstraintProxy](../OffsetDimConstraintProxy/OffsetDimConstraintProxy.md)

## Description

Gets and sets the position of the dimension text.

## Syntax

OffsetDimConstraintProxy.**TextPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 6
