# FeatureControlFrameStyle.AlternateLinearPrecision Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the precision for linear alternate units. This property is used only when the AlternateUnitsDisplay property is set to true.

## Syntax

FeatureControlFrameStyle.**AlternateLinearPrecision**() As [LinearPrecisionEnum](../LinearPrecisionEnum.md)

## Property Value

This is a read/write property whose value is a [LinearPrecisionEnum](../LinearPrecisionEnum.md).

## Version

Introduced in version 2009
