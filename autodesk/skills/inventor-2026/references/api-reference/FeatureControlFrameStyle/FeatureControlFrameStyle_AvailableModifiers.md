# FeatureControlFrameStyle.AvailableModifiers Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the material removal modifiers to be made available in the feature control frame creation and edit dialog. A combination of enum values can be provided.

## Syntax

FeatureControlFrameStyle.**AvailableModifiers**() As [MaterialRemovalModifierEnum](../MaterialRemovalModifierEnum.md)

## Property Value

This is a read/write property whose value is a [MaterialRemovalModifierEnum](../MaterialRemovalModifierEnum.md).

## Version

Introduced in version 2009
