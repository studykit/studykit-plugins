# FeatureControlFrameStyle.UpToDate Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets the up-to-date status of the style against the global repository.

## Syntax

FeatureControlFrameStyle.**UpToDate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
