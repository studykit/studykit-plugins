# PointCloudProxy.Visible Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-write property that gets and sets the visibility of this point cloud.

## Syntax

PointCloudProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
