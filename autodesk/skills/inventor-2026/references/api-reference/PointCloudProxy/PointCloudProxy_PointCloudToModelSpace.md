# PointCloudProxy.PointCloudToModelSpace Method

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Method that takes a 3d coordinate in point cloud space and returns a Point object containing the coordinate point in model space.

## Syntax

PointCloudProxy.**PointCloudToModelSpace**( ***PointCloudCoordinate*** As [Point](../Point/Point.md) ) As [Point](../Point/Point.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PointCloudCoordinate | [Point](../Point/Point.md) | Input Point object of a point in the point cloud space. |

## Version

Introduced in version 2016
