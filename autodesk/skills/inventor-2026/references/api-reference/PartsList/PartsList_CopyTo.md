# PartsList.CopyTo Method

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Method that copies the parts list to another sheet.

## Syntax

PartsList.**CopyTo**( ***TargetSheet*** As [Sheet](../Sheet/Sheet.md) ) As [PartsList](../PartsList/PartsList.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TargetSheet | [Sheet](../Sheet/Sheet.md) | Input Sheet object that specifies the sheet to copy the parts list to. |

## Version

Introduced in version 2009
