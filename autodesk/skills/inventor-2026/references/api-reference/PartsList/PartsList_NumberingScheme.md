# PartsList.NumberingScheme Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Property that returns the numbering scheme used for a 'parts only' parts list. The property returns an error if this is not a 'parts only' parts list. Possible return values are kNumericNumbering, kLowercaseAlphaNumbering and kUppercaseAlphaNumbering.

## Syntax

PartsList.**NumberingScheme**() As [NumberingSchemeEnum](../NumberingSchemeEnum.md)

## Property Value

This is a read only property whose value is a [NumberingSchemeEnum](../NumberingSchemeEnum.md).

## Version

Introduced in version 11
