# SketchSplineHandle.Geometry Property

Parent Object: [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md)

## Description

Property that returns a LineSegment2d geometry object. The object returned represents a snapshot view of the current state of the sketch line.

## Syntax

SketchSplineHandle.**Geometry**() As [LineSegment2d](../LineSegment2d/LineSegment2d.md)

## Property Value

This is a read only property whose value is a [LineSegment2d](../LineSegment2d/LineSegment2d.md).

## Version

Introduced in version 2008
