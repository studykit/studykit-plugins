# ExtrudeDefinition.Profile Property

Parent Object: [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Description

Read-write property gets and sets the sketch profile used for the extrude feature.

## Syntax

ExtrudeDefinition.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).
