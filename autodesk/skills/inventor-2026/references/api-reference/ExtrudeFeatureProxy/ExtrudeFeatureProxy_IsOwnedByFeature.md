# ExtrudeFeatureProxy.IsOwnedByFeature Property

Parent Object: [ExtrudeFeatureProxy](../ExtrudeFeatureProxy/ExtrudeFeatureProxy.md)

## Description

Property that returns whether this object is owned by a feature. If True, the OwnedBy property returns the owning feature.

## Syntax

ExtrudeFeatureProxy.**IsOwnedByFeature**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
