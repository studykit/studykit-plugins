# ApprenticeServerComponent.TransientObjects Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the TransientObjects object.

## Syntax

ApprenticeServerComponent.**TransientObjects**() As [TransientObjects](../TransientObjects/TransientObjects.md)

## Property Value

This is a read only property whose value is a [TransientObjects](../TransientObjects/TransientObjects.md).

## Version

Introduced in version 2008
