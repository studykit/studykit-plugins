# FeaturePatternElements.Count Property

Parent Object: [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md)

## Description

Property that returns the number of items in this collection.

## Syntax

FeaturePatternElements.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5
