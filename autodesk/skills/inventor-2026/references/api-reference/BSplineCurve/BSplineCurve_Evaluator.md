# BSplineCurve.Evaluator Property

Parent Object: [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Description

Property that returns an evaluator object for the curve. The evaluator supports general curve evaluation functions.

## Syntax

BSplineCurve.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 4
