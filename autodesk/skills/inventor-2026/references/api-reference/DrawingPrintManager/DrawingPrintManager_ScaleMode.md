# DrawingPrintManager.ScaleMode Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the mode by which the scale of the print is defined. Defaults to kPrintBestFitScale.

## Syntax

DrawingPrintManager.**ScaleMode**() As [PrintScaleModeEnum](../PrintScaleModeEnum.md)

## Property Value

This is a read/write property whose value is a [PrintScaleModeEnum](../PrintScaleModeEnum.md).
