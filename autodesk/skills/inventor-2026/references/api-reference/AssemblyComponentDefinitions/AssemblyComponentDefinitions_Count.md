# AssemblyComponentDefinitions.Count Property

Parent Object: [AssemblyComponentDefinitions](../AssemblyComponentDefinitions/AssemblyComponentDefinitions.md)

## Description

Property that returns the number of items in this collection.

## Syntax

AssemblyComponentDefinitions.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
