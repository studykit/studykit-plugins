# SweepDefinition.SweepType Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the sweep type. When set this value the properties in this SweepDefinition which are not supported with the sweep type will be reset.

## Syntax

SweepDefinition.**SweepType**() As [SweepTypeEnum](../SweepTypeEnum.md)

## Property Value

This is a read/write property whose value is a [SweepTypeEnum](../SweepTypeEnum.md).

## Version

Introduced in version 2015
