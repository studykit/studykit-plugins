# SweepFeature.Suppressed Property

Parent Object: [SweepFeature](../SweepFeature/SweepFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

SweepFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
