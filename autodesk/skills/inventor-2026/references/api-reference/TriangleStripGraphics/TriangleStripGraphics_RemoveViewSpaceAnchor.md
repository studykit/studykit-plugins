# TriangleStripGraphics.RemoveViewSpaceAnchor Method

Parent Object: [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

TriangleStripGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
