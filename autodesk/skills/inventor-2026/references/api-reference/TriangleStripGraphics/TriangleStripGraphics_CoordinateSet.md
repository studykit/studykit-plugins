# TriangleStripGraphics.CoordinateSet Property

Parent Object: [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Description

Gets and sets the GraphicsCoordinateSet associated with the set.

## Syntax

TriangleStripGraphics.**CoordinateSet**() As [GraphicsCoordinateSet](../GraphicsCoordinateSet/GraphicsCoordinateSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsCoordinateSet](../GraphicsCoordinateSet/GraphicsCoordinateSet.md).
