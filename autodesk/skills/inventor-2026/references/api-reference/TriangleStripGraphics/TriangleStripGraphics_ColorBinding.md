# TriangleStripGraphics.ColorBinding Property

Parent Object: [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Description

Gets and sets how the colors are defined for the triangle.

## Syntax

TriangleStripGraphics.**ColorBinding**() As [ColorBindingEnum](../ColorBindingEnum.md)

## Property Value

This is a read/write property whose value is a [ColorBindingEnum](../ColorBindingEnum.md).

## Version

Introduced in version 5
