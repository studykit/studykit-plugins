# Collected documentation references

Local copies of official documentation cited as evidence in this repository,
saved by the guard plugin's Grounded output style. This is guard's default refs
directory (`wiki/ref/`); override it with `refs_dir` in `.claude/guard.local.json`.
Each file records the relevant excerpt of one source and names its origin URL, so a
citation stays inspectable after the upstream page changes. Files are committed
through the normal git workflow.
