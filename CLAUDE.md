# Claude Plugin Marketplace

This is the claude-plugin-marketplace directory - a collection of Claude Code plugins for various use cases.

## Language Requirements

**All documentation must be written in English.** When creating or editing markdown files, README files, CLAUDE.md files, or any other documentation, always use English.

## Available Plugins

- **english-plugin**: English-Korean translation and language learning
- **doc-plugin**: Document format handling (CHM, PDF, etc.)
- **autodesk-plugin**: Autodesk Inventor API development support
