# SketchSplineProxy.SetHandleStatus Method

Parent Object: [SketchSplineProxy](../SketchSplineProxy/SketchSplineProxy.md)

## Description

Method that sets whether the handle at a given fit point is enabled or disabled.

## Syntax

SketchSplineProxy.**SetHandleStatus**( ***FitPoint*** As [SketchPoint](../SketchPoint/SketchPoint.md), ***Value*** As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FitPoint | [SketchPoint](../SketchPoint/SketchPoint.md) | Input SketchPoint object that specifies the fit point at which to set the handle status. |
| Value | Boolean | Input Boolean that specifies whether the handle at the input fit point is enabled or disabled. |

## Version

Introduced in version 2008
