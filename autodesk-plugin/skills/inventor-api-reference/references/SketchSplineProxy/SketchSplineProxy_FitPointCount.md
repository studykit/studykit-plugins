# SketchSplineProxy.FitPointCount Property

Parent Object: [SketchSplineProxy](../SketchSplineProxy/SketchSplineProxy.md)

## Description

Property that returns the number of fit points for the spline.

## Syntax

SketchSplineProxy.**FitPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 6
