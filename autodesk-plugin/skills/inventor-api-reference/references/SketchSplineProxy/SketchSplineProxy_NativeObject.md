# SketchSplineProxy.NativeObject Property

Parent Object: [SketchSplineProxy](../SketchSplineProxy/SketchSplineProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchSplineProxy.**NativeObject**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Property Value

This is a read only property whose value is a [SketchSpline](../SketchSpline/SketchSpline.md).

## Version

Introduced in version 6
