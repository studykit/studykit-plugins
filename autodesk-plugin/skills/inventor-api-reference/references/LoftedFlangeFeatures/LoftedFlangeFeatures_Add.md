# LoftedFlangeFeatures.Add Method

Parent Object: [LoftedFlangeFeatures](../LoftedFlangeFeatures/LoftedFlangeFeatures.md)

## Description

Method that creates a new lofted flange feature. The newly created LoftedFlangeFeature object is returned.

## Syntax

LoftedFlangeFeatures.**Add**( ***LoftedFlangeDefinition*** As [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md) ) As [LoftedFlangeFeature](../LoftedFlangeFeature/LoftedFlangeFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LoftedFlangeDefinition | [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md) | Object that defines the lofted flange feature you want to create. A LoftedFlangeDefinition object can be created using the CreateLoftedFlangeDefinition method on the LoftedFlangeFeatures object. |
