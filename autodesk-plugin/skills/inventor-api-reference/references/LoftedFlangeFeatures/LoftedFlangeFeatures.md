# LoftedFlangeFeatures Object

## Description

The LoftedFlangeFeatures collection object provides access to the LoftedFlangeFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../LoftedFlangeFeatures/LoftedFlangeFeatures_Add.md) | Method that creates a new lofted flange feature. The newly created LoftedFlangeFeature object is returned. |
| [CreateLoftedFlangeDefinition](../LoftedFlangeFeatures/LoftedFlangeFeatures_CreateLoftedFlangeDefinition.md) | Method that creates a new LoftedFlangeDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LoftedFlangeFeatures/LoftedFlangeFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LoftedFlangeFeatures/LoftedFlangeFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../LoftedFlangeFeatures/LoftedFlangeFeatures_Item.md) | Returns the specified LoftedFlangeFeature object from the collection. |
| [Type](../LoftedFlangeFeatures/LoftedFlangeFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
