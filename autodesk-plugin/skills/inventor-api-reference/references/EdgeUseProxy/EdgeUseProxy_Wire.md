# EdgeUseProxy.Wire Property

Parent Object: [EdgeUseProxy](../EdgeUseProxy/EdgeUseProxy.md)

## Description

Get the Wire containing this EdgeUse. Returns Nothing if this EdgeUse does not belong to a Wire.

## Syntax

EdgeUseProxy.**Wire**() As [Wire](../Wire/Wire.md)

## Property Value

This is a read only property whose value is a [Wire](../Wire/Wire.md).

## Version

Introduced in version 2008
