# EdgeUseProxy.IsParamReversed Property

Parent Object: [EdgeUseProxy](../EdgeUseProxy/EdgeUseProxy.md)

## Description

Gets whether the parameterization of the geometry obtained from the Curve property is aligned or opposed to the topological sense of this EdgeUse.

## Syntax

EdgeUseProxy.**IsParamReversed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
