# BIMComponentDescription.RevitFamilyCategory Property

Parent Object: [BIMComponentDescription](../BIMComponentDescription/BIMComponentDescription.md)

## Description

Read-write property that gets and sets the Revit family category Id.

## Remarks

A Revit family category is identified by a category ID. The list of available categories is defined in the folder:
Bin\BIM Content Design Data\Support\AuthoringTemplates\en-US.
