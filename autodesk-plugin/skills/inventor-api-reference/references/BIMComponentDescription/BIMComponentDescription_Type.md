# BIMComponentDescription.Type Property

Parent Object: [BIMComponentDescription](../BIMComponentDescription/BIMComponentDescription.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

BIMComponentDescription.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
