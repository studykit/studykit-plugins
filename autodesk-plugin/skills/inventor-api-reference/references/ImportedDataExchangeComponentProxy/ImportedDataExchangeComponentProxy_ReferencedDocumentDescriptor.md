# ImportedDataExchangeComponentProxy.ReferencedDocumentDescriptor Property

Parent Object: [ImportedDataExchangeComponentProxy](../ImportedDataExchangeComponentProxy/ImportedDataExchangeComponentProxy.md)

## Description

Property that returns the DocumentDescriptor object.

## Syntax

ImportedDataExchangeComponentProxy.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 2023
