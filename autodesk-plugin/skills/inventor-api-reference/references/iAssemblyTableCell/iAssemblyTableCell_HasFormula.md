# iAssemblyTableCell.HasFormula Property

Parent Object: [iAssemblyTableCell](../iAssemblyTableCell/iAssemblyTableCell.md)

## Description

Property that returns whether a formula (equation) was input into this cell via Excel. Such cells show with a red background in the user interface and are not editable.

## Syntax

iAssemblyTableCell.**HasFormula**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
