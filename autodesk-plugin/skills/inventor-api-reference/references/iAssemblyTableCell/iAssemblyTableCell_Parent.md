# iAssemblyTableCell.Parent Property

Parent Object: [iAssemblyTableCell](../iAssemblyTableCell/iAssemblyTableCell.md)

## Description

Property that returns the parent iAssemblyFactory object.

## Syntax

iAssemblyTableCell.**Parent**() As [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Property Value

This is a read only property whose value is an [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md).

## Version

Introduced in version 11
