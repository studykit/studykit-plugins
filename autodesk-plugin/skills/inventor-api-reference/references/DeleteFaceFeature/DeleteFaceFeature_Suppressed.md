# DeleteFaceFeature.Suppressed Property

Parent Object: [DeleteFaceFeature](../DeleteFaceFeature/DeleteFaceFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

DeleteFaceFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
