# DeleteFaceFeature.AppearanceSourceType Property

Parent Object: [DeleteFaceFeature](../DeleteFaceFeature/DeleteFaceFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

DeleteFaceFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
