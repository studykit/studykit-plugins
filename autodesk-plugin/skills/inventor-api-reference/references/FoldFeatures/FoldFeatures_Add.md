# FoldFeatures.Add Method

Parent Object: [FoldFeatures](../FoldFeatures/FoldFeatures.md)

## Description

Method that creates a new fold feature. The newly created CutFeature object is returned.

## Syntax

FoldFeatures.**Add**( ***FoldDefinition*** As [FoldDefinition](../FoldDefinition/FoldDefinition.md) ) As [FoldFeature](../FoldFeature/FoldFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FoldDefinition | [FoldDefinition](../FoldDefinition/FoldDefinition.md) | Input FoldDefinition object that defines the fold feature you want to create. A FoldDefinition object can be created using the FoldFeatures.CreateFoldDefinition method. It can also be obtained from an existing FoldFeature object. |
