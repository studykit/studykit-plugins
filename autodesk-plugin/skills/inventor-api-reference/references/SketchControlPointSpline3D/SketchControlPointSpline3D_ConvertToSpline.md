# SketchControlPointSpline3D.ConvertToSpline Method

Parent Object: [SketchControlPointSpline3D](../SketchControlPointSpline3D/SketchControlPointSpline3D.md)

## Description

Method that inserts a new control point into the existing control point spline.

## Syntax

SketchControlPointSpline3D.**ConvertToSpline**() As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Version

Introduced in version 2014
