# LineStripGraphics.ColorBinding Property

Parent Object: [LineStripGraphics](../LineStripGraphics/LineStripGraphics.md)

## Description

Gets and sets how colors are defined for the line.

## Syntax

LineStripGraphics.**ColorBinding**() As [ColorBindingEnum](../ColorBindingEnum.md)

## Property Value

This is a read/write property whose value is a [ColorBindingEnum](../ColorBindingEnum.md).

## Version

Introduced in version 5
