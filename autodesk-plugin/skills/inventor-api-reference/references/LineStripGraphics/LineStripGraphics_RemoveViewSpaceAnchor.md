# LineStripGraphics.RemoveViewSpaceAnchor Method

Parent Object: [LineStripGraphics](../LineStripGraphics/LineStripGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

LineStripGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
