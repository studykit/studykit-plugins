# LayersEnumerator Object

## Description

The LayersEnumerator object provides access to the existing Layer objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LayersEnumerator/LayersEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LayersEnumerator/LayersEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../LayersEnumerator/LayersEnumerator_Item.md) | Returns the specified Layer object from the collection. |
| [Type](../LayersEnumerator/LayersEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.Layers](../DrawingStylesManager/DrawingStylesManager_Layers.md)

## Version

Introduced in version 10
