# Bend Object

## Description

The Bend object represents a bend in a sheet metal part.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Bend/Bend_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [BackFaces](../Bend/Bend_BackFaces.md) | Property that returns the collection of front faces in the model that represent this bend. |
| [FrontFaces](../Bend/Bend_FrontFaces.md) | Property that returns the collection of faces in the model that represent this bend. |
| [IsFlat](../Bend/Bend_IsFlat.md) | Property that indicates if the bend is in a folded or flat state |
| [Parent](../Bend/Bend_Parent.md) | Property that returns the parent SheetMetalComponentDefinition of this Bend object. |
| [Type](../Bend/Bend_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BendsEnumerator.Item](../BendsEnumerator/BendsEnumerator_Item.md), [FlatBendResult.Bend](../FlatBendResult/FlatBendResult_Bend.md)

## Version
