# Bend.Parent Property

Parent Object: [Bend](../Bend/Bend.md)

## Description

Property that returns the parent SheetMetalComponentDefinition of this Bend object.

## Syntax

Bend.**Parent**() As [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Property Value

This is a read only property whose value is a [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md).

## Version

Introduced in version 2010
