# SweepFeature.IsSolidSweep Property

Parent Object: [SweepFeature](../SweepFeature/SweepFeature.md)

## Description

Read-only property that returns whether the feature is a solid sweep feature.

## Syntax

SweepFeature.**IsSolidSweep**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2020
