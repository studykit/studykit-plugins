# SketchImageProxy.NativeObject Property

Parent Object: [SketchImageProxy](../SketchImageProxy/SketchImageProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchImageProxy.**NativeObject**() As [SketchImage](../SketchImage/SketchImage.md)

## Property Value

This is a read only property whose value is a [SketchImage](../SketchImage/SketchImage.md).

## Version

Introduced in version 11
