# SketchImageProxy.MirrorHorizontal Method

Parent Object: [SketchImageProxy](../SketchImageProxy/SketchImageProxy.md)

## Description

Method that mirrors the image horizontally.

## Syntax

SketchImageProxy.**MirrorHorizontal**()

## Version

Introduced in version 11
