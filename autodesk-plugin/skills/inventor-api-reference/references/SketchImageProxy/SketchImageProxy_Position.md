# SketchImageProxy.Position Property

Parent Object: [SketchImageProxy](../SketchImageProxy/SketchImageProxy.md)

## Description

Gets and sets the position of the image on the sketch.

## Syntax

SketchImageProxy.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 11
