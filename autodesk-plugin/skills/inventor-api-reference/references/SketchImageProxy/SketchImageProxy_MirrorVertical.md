# SketchImageProxy.MirrorVertical Method

Parent Object: [SketchImageProxy](../SketchImageProxy/SketchImageProxy.md)

## Description

Method that mirrors the image vertically.

## Syntax

SketchImageProxy.**MirrorVertical**()

## Version

Introduced in version 11
