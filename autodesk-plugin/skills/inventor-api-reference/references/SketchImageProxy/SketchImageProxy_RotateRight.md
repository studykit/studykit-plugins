# SketchImageProxy.RotateRight Method

Parent Object: [SketchImageProxy](../SketchImageProxy/SketchImageProxy.md)

## Description

Method that rotates the image clockwise by 90 degrees.

## Syntax

SketchImageProxy.**RotateRight**()

## Version

Introduced in version 11
