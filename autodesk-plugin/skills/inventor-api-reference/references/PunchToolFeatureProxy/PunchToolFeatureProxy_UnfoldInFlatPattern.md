# PunchToolFeatureProxy.UnfoldInFlatPattern Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Read-write property that gets and sets whether the punch tool feature is unfoldable in flat pattern.

## Syntax

PunchToolFeatureProxy.**UnfoldInFlatPattern**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
