# PunchToolFeatureProxy.iFeatureTemplateDescriptor Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Property that returns the iFeatureTemplateDescriptor object associated with this iFeature.

## Syntax

PunchToolFeatureProxy.**iFeatureTemplateDescriptor**() As [iFeatureTemplateDescriptor](../iFeatureTemplateDescriptor/iFeatureTemplateDescriptor.md)

## Property Value

This is a read only property whose value is an [iFeatureTemplateDescriptor](../iFeatureTemplateDescriptor/iFeatureTemplateDescriptor.md).

## Version

Introduced in version 2009
