# PunchToolFeatureProxy.Depth Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Property that returns the parameter object that defines the custom punch depth.

## Syntax

PunchToolFeatureProxy.**Depth**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
