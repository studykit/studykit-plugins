# PunchToolFeatureProxy.Angle Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Property that returns the parameter object that controls the angle of rotation of the punch tool.

## Syntax

PunchToolFeatureProxy.**Angle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
