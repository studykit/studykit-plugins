# MoveAlongRayMoveOperation.EntityNaturalDirection Property

Parent Object: [MoveAlongRayMoveOperation](../MoveAlongRayMoveOperation/MoveAlongRayMoveOperation.md)

## Description

Read-write Boolean property that specifies if the movement direction of the bodies is in the natural direction of the direction entity.

## Syntax

MoveAlongRayMoveOperation.**EntityNaturalDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
