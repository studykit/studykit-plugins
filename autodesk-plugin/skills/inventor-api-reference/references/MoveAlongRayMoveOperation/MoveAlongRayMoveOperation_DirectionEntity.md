# MoveAlongRayMoveOperation.DirectionEntity Property

Parent Object: [MoveAlongRayMoveOperation](../MoveAlongRayMoveOperation/MoveAlongRayMoveOperation.md)

## Description

Read-write property that gets and sets the entity used to define the direction of the move.

## Syntax

MoveAlongRayMoveOperation.**DirectionEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2013
