# HighlightSet.Type Property

Parent Object: [HighlightSet](../HighlightSet/HighlightSet.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

HighlightSet.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
