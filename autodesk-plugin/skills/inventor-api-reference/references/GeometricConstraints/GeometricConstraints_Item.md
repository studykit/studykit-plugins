# GeometricConstraints.Item Property

Parent Object: [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md)

## Description

Returns the specified geometric sketch constraint object from the collection.

## Syntax

GeometricConstraints.**Item**( ***Index*** As Long ) As [GeometricConstraint](../GeometricConstraint/GeometricConstraint.md)

## Property Value

This is a read only property whose value is a [GeometricConstraint](../GeometricConstraint/GeometricConstraint.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 5
