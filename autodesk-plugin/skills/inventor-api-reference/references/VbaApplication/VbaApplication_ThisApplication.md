# VbaApplication.ThisApplication Property

Parent Object: [VbaApplication](../VbaApplication/VbaApplication.md)

## Description

Gets the VBA application interface.

## Syntax

VbaApplication.**ThisApplication**() As [Application](../Application/Application.md)

## Property Value

This is a read only property whose value is an [Application](../Application/Application.md).

## Version

Introduced in version 5
