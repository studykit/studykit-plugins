# VbaApplication.ThisServer Property

Parent Object: [VbaApplication](../VbaApplication/VbaApplication.md)

## Description

Gets the VBA InventorServer interface.

## Syntax

VbaApplication.**ThisServer**() As [InventorServerObject](InventorServerObject.md)

## Property Value

This is a read only property whose value is an [InventorServerObject](InventorServerObject.md).

## Version

Introduced in version 2011
