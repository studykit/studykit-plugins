# SelectEvents.Enabled Property

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Gets/Sets the flag indicating whether the selection is enabled.

## Syntax

SelectEvents.**Enabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
