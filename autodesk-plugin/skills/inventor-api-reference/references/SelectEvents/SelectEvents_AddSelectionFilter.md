# SelectEvents.AddSelectionFilter Method

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Adds a new filter (indication of the type of object(s) of interest) to the existing list of filters.

## Syntax

SelectEvents.**AddSelectionFilter**( ***Filter*** As [SelectionFilterEnum](../SelectionFilterEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Filter | [SelectionFilterEnum](../SelectionFilterEnum.md) |  |
