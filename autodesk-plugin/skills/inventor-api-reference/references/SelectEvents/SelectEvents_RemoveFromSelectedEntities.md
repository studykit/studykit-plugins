# SelectEvents.RemoveFromSelectedEntities Method

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Removes entities from the selected entities collection.

## Syntax

SelectEvents.**RemoveFromSelectedEntities**( ***ItemObject*** As Object )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ItemObject | Object | Input object that specifies the object to remove from the selected entities collection. |

## Version

Introduced in version 2009
