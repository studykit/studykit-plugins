# SelectEvents.ClearSelectionFilter Method

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Clears out the filter list.

## Syntax

SelectEvents.**ClearSelectionFilter**()

## Version

Introduced in version 5
