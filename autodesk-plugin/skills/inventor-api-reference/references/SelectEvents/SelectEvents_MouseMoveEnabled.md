# SelectEvents.MouseMoveEnabled Property

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Gets/Sets the flag indicating whether a mouse move should fire an event (OnPreSelectMouseMove). Default is TRUE.

## Syntax

SelectEvents.**MouseMoveEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
