# SelectEvents.ResetSelections Method

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Resets the set of selections being recorded by this object to an empty set.

## Syntax

SelectEvents.**ResetSelections**()

## Version

Introduced in version 5
