# SelectEvents.RestrictSelectionToOccurrence Property

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Read-write property that gets and sets the ComponentOccurrence to which the selection should be restricted.

## Syntax

SelectEvents.**RestrictSelectionToOccurrence**() As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Property Value

This is a read/write property whose value is a [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md).

## Version

Introduced in version 2025.1
