# SelectEvents.ClearWindowSelectionFilter Method

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Clears out the window selection filter list.

## Syntax

SelectEvents.**ClearWindowSelectionFilter**()

## Version

Introduced in version 2009
