# SelectEvents.IgnoreProfileInnerLoops Property

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Gets and sets whether to ignore inner loops during 2d and 3d sketch profile selection. This property defaults to False.

## Syntax

SelectEvents.**IgnoreProfileInnerLoops**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
