# SelectEvents.AddWindowSelectionFilter Method

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Adds a new filter (indication of the type of object(s) of interest) to the list of filters for Window Select (defaults to same filter list specified by AddSelectionFilter).

## Syntax

SelectEvents.**AddWindowSelectionFilter**( ***Filter*** As [SelectionFilterEnum](../SelectionFilterEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Filter | [SelectionFilterEnum](../SelectionFilterEnum.md) |  |

## Version

Introduced in version 2009
