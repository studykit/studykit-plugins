# Machining Object

## Description

Allows editing for environment-specific features.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Edit](../Machining/Machining_Edit.md) | Method that activates this object for editing by the user. |
| [ExitEdit](../Machining/Machining_ExitEdit.md) | Method that causes the current editing environment to be closed and the user interface to return to weldment editing environment. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Machining/Machining_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Type](../Machining/Machining_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[WeldmentComponentDefinition.Machining](../WeldmentComponentDefinition/WeldmentComponentDefinition_Machining.md)

## Version

Introduced in version 2010
