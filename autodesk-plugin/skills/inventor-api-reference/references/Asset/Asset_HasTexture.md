# Asset.HasTexture Property

Parent Object: [Asset](../Asset/Asset.md)

## Description

Gets the flag that indicates if this asset has texture.

## Syntax

Asset.**HasTexture**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.
