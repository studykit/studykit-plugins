# BrowserPanesEvents Object

## Description

Object providing event notification for browser events.

## Events

|  |  |
| --- | --- |
| Name | Description |
| [OnBrowserNodeActivate](../BrowserPanesEvents/BrowserPanesEvents_OnBrowserNodeActivate.md) | Event that is fired whenever a node is activated by the user. |
| [OnBrowserNodeDeleteEntry](../BrowserPanesEvents/BrowserPanesEvents_OnBrowserNodeDeleteEntry.md) | Event that is fired whenever the user requests that a node be deleted. |
| [OnBrowserNodeGetDisplayObjects](../BrowserPanesEvents/BrowserPanesEvents_OnBrowserNodeGetDisplayObjects.md) | Event that is fired when a user requests that the objects represented by a browser node be highlighted. |
| [OnBrowserNodeLabelEdit](../BrowserPanesEvents/BrowserPanesEvents_OnBrowserNodeLabelEdit.md) | Event that is fired whenever a node is renamed by the user. |
| [OnBrowserNodesReorder](../BrowserPanesEvents/BrowserPanesEvents_OnBrowserNodesReorder.md) | Event that fires just before and after browser nodes are reordered. |

## Accessed From

[BrowserPanes.BrowserPanesEvents](../BrowserPanes/BrowserPanes_BrowserPanesEvents.md)

## Version

Introduced in version 9
