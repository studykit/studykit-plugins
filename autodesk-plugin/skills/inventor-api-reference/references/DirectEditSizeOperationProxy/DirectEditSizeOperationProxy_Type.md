# DirectEditSizeOperationProxy.Type Property

Parent Object: [DirectEditSizeOperationProxy](../DirectEditSizeOperationProxy/DirectEditSizeOperationProxy.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

DirectEditSizeOperationProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version
