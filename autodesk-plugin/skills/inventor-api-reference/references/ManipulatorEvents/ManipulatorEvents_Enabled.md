# ManipulatorEvents.Enabled Property

Parent Object: [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md)

## Description

Gets and sets whether the manipulator events is enabled or not.

## Syntax

ManipulatorEvents.**Enabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
