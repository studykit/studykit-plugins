# ManipulatorEvents.SuggestedMiniToolbarPosition Method

Parent Object: [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md)

## Description

Calculates a suggested position to place mini toolbar with input mini toolbar size.

## Syntax

ManipulatorEvents.**SuggestedMiniToolbarPosition**( ***MiniToolbarHeight*** As Long, ***MiniToolbarWidth*** As Long ) As [Point2d](../Point2d/Point2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| MiniToolbarHeight | Long | Input Long indicating the mini toolbar height. |
| MiniToolbarWidth | Long | Input Long indicating the mini toolbar width. |

## Version

Introduced in version 2017
