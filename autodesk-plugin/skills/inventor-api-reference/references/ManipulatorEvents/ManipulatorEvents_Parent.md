# ManipulatorEvents.Parent Property

Parent Object: [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ManipulatorEvents.**Parent**() As [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Property Value

This is a read only property whose value is an [InteractionEvents](../InteractionEvents/InteractionEvents.md).

## Version

Introduced in version 2017
