# OrientedBox.CornerPoint Property

Parent Object: [OrientedBox](../OrientedBox/OrientedBox.md)

## Description

Read-write property that gets and sets the corner point for the OrientedBox object.

## Syntax

OrientedBox.**CornerPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2016
