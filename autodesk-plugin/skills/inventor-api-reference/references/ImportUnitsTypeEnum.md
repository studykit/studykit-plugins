# ImportUnitsTypeEnum Enumerator

## Description

Specifies the import document units type.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kCentimeterUnitsType | 110084 | Indicates document units will be centimeters. |
| kFootUnitsType | 110083 | Indicates document units will be feet. |
| kInchUnitsType | 110082 | Indicates document units will be inches. |
| kMeterUnitsType | 110086 | Indicates document units will be meters. |
| kMicronUnitsType | 110087 | Indicates document units will be microns. |
| kMillimeterUnitsType | 110085 | Indicates document units will be millimeters. |
| kSourceUnitsType | 110081 | Indicates document units will be the same as the source. |

## Version

Introduced in version 2016
