# BIMElectricalConnectorDefinition.ExposeApparentLoadPhase3AsParameter Property

Parent Object: [BIMElectricalConnectorDefinition](../BIMElectricalConnectorDefinition/BIMElectricalConnectorDefinition.md)

## Description

Read-write property that gets and sets if expose the apparent load phase 3 as parameter.

## Syntax

BIMElectricalConnectorDefinition.**ExposeApparentLoadPhase3AsParameter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
