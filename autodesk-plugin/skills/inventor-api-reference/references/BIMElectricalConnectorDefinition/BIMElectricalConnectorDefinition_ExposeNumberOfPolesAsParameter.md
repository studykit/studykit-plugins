# BIMElectricalConnectorDefinition.ExposeNumberOfPolesAsParameter Property

Parent Object: [BIMElectricalConnectorDefinition](../BIMElectricalConnectorDefinition/BIMElectricalConnectorDefinition.md)

## Description

Read-write property that gets and sets if expose the number of poles as parameter.

## Syntax

BIMElectricalConnectorDefinition.**ExposeNumberOfPolesAsParameter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
