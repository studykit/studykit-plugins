# BIMElectricalConnectorDefinition.ExposeApparentLoadPhase1AsParameter Property

Parent Object: [BIMElectricalConnectorDefinition](../BIMElectricalConnectorDefinition/BIMElectricalConnectorDefinition.md)

## Description

Read-write property that gets and sets if expose the apparent load phase 1 as parameter.

## Syntax

BIMElectricalConnectorDefinition.**ExposeApparentLoadPhase1AsParameter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
