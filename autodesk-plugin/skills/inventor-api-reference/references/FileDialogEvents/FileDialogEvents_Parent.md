# FileDialogEvents.Parent Property

Parent Object: [FileDialogEvents](../FileDialogEvents/FileDialogEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

FileDialogEvents.**Parent**() As [FileDialog](../FileDialog/FileDialog.md)

## Property Value

This is a read only property whose value is a [FileDialog](../FileDialog/FileDialog.md).

## Version

Introduced in version 2008
