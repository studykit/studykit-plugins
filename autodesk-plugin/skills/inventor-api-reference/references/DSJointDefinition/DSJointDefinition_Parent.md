# DSJointDefinition.Parent Property

Parent Object: [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md)

## Description

Gets the parent DSJoint object this definition is associated with. This can return Nothing in the case where the definition is not associated with any joint.

## Syntax

DSJointDefinition.**Parent**() As [DSJoint](../DSJoint/DSJoint.md)

## Property Value

This is a read only property whose value is a [DSJoint](../DSJoint/DSJoint.md).

## Version

Introduced in version 2013
