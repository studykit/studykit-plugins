# DSJointDefinition.DegreesOfFreedom Property

Parent Object: [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md)

## Description

Gets the DSDegreesOfFreedom collection object provides access to the degrees of freedom associated with this joint.

## Syntax

DSJointDefinition.**DegreesOfFreedom**() As [DSDegreesOfFreedom](../DSDegreesOfFreedom/DSDegreesOfFreedom.md)

## Property Value

This is a read only property whose value is a [DSDegreesOfFreedom](../DSDegreesOfFreedom/DSDegreesOfFreedom.md).

## Version

Introduced in version 2013
