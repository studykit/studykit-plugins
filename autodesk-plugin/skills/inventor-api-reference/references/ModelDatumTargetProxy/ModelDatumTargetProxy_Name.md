# ModelDatumTargetProxy.Name Property

Parent Object: [ModelDatumTargetProxy](../ModelDatumTargetProxy/ModelDatumTargetProxy.md)

## Description

Read-write property that gets and sets the name of the annotation.

## Syntax

ModelDatumTargetProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2023
