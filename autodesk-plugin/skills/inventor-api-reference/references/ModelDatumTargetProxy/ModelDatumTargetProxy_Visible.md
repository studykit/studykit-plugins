# ModelDatumTargetProxy.Visible Property

Parent Object: [ModelDatumTargetProxy](../ModelDatumTargetProxy/ModelDatumTargetProxy.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelDatumTargetProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
