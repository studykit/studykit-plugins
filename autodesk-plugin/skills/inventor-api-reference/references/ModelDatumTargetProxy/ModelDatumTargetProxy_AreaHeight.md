# ModelDatumTargetProxy.AreaHeight Property

Parent Object: [ModelDatumTargetProxy](../ModelDatumTargetProxy/ModelDatumTargetProxy.md)

## Description

Returns the parameter that controls the datum target area height.

## Syntax

ModelDatumTargetProxy.**AreaHeight**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2023
