# ModelDatumTargetProxy.Index Property

Parent Object: [ModelDatumTargetProxy](../ModelDatumTargetProxy/ModelDatumTargetProxy.md)

## Description

Gets and sets the index of this model datum target object.

## Syntax

ModelDatumTargetProxy.**Index**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2023
