# FamilyParameterValueTypeEnum Enumerator

## Description

Enum for the different family parameter value types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kConstantFamilyParameterValue | 51457 | Implies the value is a constant string. |
| kCustomFamilyParameterValue | 51458 | Implies the value is a custom string. |
| kExpressionFamilyParameterValue | 51459 | Implies the value is an expression string. |

## Version

Introduced in version 10
