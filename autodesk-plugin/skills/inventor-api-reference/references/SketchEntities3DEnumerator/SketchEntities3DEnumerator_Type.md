# SketchEntities3DEnumerator.Type Property

Parent Object: [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchEntities3DEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
