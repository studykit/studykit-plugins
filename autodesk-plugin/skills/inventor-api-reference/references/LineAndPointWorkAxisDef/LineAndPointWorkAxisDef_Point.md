# LineAndPointWorkAxisDef.Point Property

Parent Object: [LineAndPointWorkAxisDef](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef.md)

## Description

Property that returns the point used to define a work axis that is along a line and passes through a point.. This object can be a vertex, WorkPoint, SketchPoint, or SketchPoint3D object.

## Syntax

LineAndPointWorkAxisDef.**Point**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
