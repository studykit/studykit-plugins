# DistanceFromFaceExtent.Distance Property

Parent Object: [DistanceFromFaceExtent](../DistanceFromFaceExtent/DistanceFromFaceExtent.md)

## Description

Read-only property that returns the parameter controlling the distance extent of the feature.

## Syntax

DistanceFromFaceExtent.**Distance**() As Variant

## Property Value

This is a read only property whose value is a Variant.

## Version

Introduced in version 2018
