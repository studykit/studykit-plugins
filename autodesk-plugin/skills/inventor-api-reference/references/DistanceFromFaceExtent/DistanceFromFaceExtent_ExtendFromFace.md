# DistanceFromFaceExtent.ExtendFromFace Property

Parent Object: [DistanceFromFaceExtent](../DistanceFromFaceExtent/DistanceFromFaceExtent.md)

## Description

Property that gets and sets whether the 'from face' should be extended to contain the extents of the feature. This property is not valid for every surface type.

## Syntax

DistanceFromFaceExtent.**ExtendFromFace**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
