# AssemblyWorkPlaneDef.XAxis Property

Parent Object: [AssemblyWorkPlaneDef](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef.md)

## Description

Property that returns the X-axis vector of the work plane.

## Syntax

AssemblyWorkPlaneDef.**XAxis**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 7
