# AssemblyWorkPlaneDef Object

## Description

The AssemblyWorkPlaneDef object allows you to access the information specific to work planes in an assembly.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_GetData.md) | Method that gets all of the data defining an assembly work plane. |
| [PutData](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_PutData.md) | Method that sets all of the data defining an assembly work plane. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Constraints](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_Constraints.md) | Property that returns the collection of constraints that are acting on the work plane. This returns all of the constraints, including those returned by the DefinitionConstraints property |
| [OriginPoint](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_OriginPoint.md) | Property that returns the origin point of the work plane. |
| [Parent](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_Parent.md) | Property that returns the parent WorkPlane object. |
| [Type](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [XAxis](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_XAxis.md) | Property that returns the X-axis vector of the work plane. |
| [YAxis](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef_YAxis.md) | Property that returns the Y-axis vector of the work plane. |

## Version

Introduced in version 6
