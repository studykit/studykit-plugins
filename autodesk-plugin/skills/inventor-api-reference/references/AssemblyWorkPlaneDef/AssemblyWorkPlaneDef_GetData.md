# AssemblyWorkPlaneDef.GetData Method

Parent Object: [AssemblyWorkPlaneDef](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef.md)

## Description

Method that gets all of the data defining an assembly work plane.

## Syntax

AssemblyWorkPlaneDef.**GetData**( ***OriginPoint*** As [Point](../Point/Point.md), ***XAxis*** As [UnitVector](../UnitVector/UnitVector.md), ***YAxis*** As [UnitVector](../UnitVector/UnitVector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| OriginPoint | [Point](../Point/Point.md) | Output object that defines the origin of the work plane. |
| XAxis | [UnitVector](../UnitVector/UnitVector.md) | Output object that defines the X-axis vector of the work plane. |
| YAxis | [UnitVector](../UnitVector/UnitVector.md) | Output object that defines the Y-axis vector of the work plane. |

## Version

Introduced in version 7
