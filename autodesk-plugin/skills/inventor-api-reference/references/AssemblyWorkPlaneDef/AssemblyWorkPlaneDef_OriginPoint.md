# AssemblyWorkPlaneDef.OriginPoint Property

Parent Object: [AssemblyWorkPlaneDef](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef.md)

## Description

Property that returns the origin point of the work plane.

## Syntax

AssemblyWorkPlaneDef.**OriginPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 7
