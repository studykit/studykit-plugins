# ModelFeatureControlFrameRows.Item Property

Parent Object: [ModelFeatureControlFrameRows](../ModelFeatureControlFrameRows/ModelFeatureControlFrameRows.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

ModelFeatureControlFrameRows.**Item**( ***Index*** As Long ) As [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Property Value

This is a read only property whose value is a [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2015
