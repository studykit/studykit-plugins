# MateConstraintProxy.Name Property

Parent Object: [MateConstraintProxy](../MateConstraintProxy/MateConstraintProxy.md)

## Description

Gets/Sets the displayable name of this constraint.

## Syntax

MateConstraintProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
