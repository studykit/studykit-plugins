# MateConstraintProxy.Type Property

Parent Object: [MateConstraintProxy](../MateConstraintProxy/MateConstraintProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

MateConstraintProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
