# MateConstraintProxy.NativeObject Property

Parent Object: [MateConstraintProxy](../MateConstraintProxy/MateConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MateConstraintProxy.**NativeObject**() As [MateConstraint](../MateConstraint/MateConstraint.md)

## Property Value

This is a read only property whose value is a [MateConstraint](../MateConstraint/MateConstraint.md).

## Version

Introduced in version 11
