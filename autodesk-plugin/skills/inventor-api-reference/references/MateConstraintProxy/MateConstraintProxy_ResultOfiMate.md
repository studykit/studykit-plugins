# MateConstraintProxy.ResultOfiMate Property

Parent Object: [MateConstraintProxy](../MateConstraintProxy/MateConstraintProxy.md)

## Description

Property that indicates if this iMateResult represents a composite iMate.

## Syntax

MateConstraintProxy.**ResultOfiMate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
