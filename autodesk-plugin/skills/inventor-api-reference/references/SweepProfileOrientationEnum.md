# SweepProfileOrientationEnum Enumerator

## Description

Enum indicating the orientation of the profile for sweep feature.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAlignToVector | 59651 | Align to vector. |
| kNormalToPath | 59649 | Normal to path. |
| kParallelToOriginalProfile | 59650 | Parallel to original profile. |

## Version

Introduced in version 11
