# MultiLineDimensionTextEnum Enumerator

## Description

Constants indicating the various types of multiline dimension text.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllAboveLandingLine | 101634 | All above landing line. |
| kAllAboveLandingLineWithUnderline | 101635 | All above landing line with underline. |
| kFirstLineAboveLandingLine | 101636 | First line above landing line. |
| kFirstLineCenteredOnLandingLine | 101633 | First line centered on landing line. |
| kJISAlignment | 101637 | JIS alignment. |

## Version

Introduced in version 2015
