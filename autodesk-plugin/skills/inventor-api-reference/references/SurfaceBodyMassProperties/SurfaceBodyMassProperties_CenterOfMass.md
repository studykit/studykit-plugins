# SurfaceBodyMassProperties.CenterOfMass Property

Parent Object: [SurfaceBodyMassProperties](../SurfaceBodyMassProperties/SurfaceBodyMassProperties.md)

## Description

Read-only property that returns the center of mass.

## Syntax

SurfaceBodyMassProperties.**CenterOfMass**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2023
