# SurfaceBodyMassProperties.Area Property

Parent Object: [SurfaceBodyMassProperties](../SurfaceBodyMassProperties/SurfaceBodyMassProperties.md)

## Description

Read-only property that returns the area of the surface body in database units.

## Syntax

SurfaceBodyMassProperties.**Area**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2023
