# PointGraphics.CoordinateIndexSet Property

Parent Object: [PointGraphics](../PointGraphics/PointGraphics.md)

## Description

Gets and sets the GraphicsIndexSet that defines indices within the coordinate set to use for the point of the set.

## Syntax

PointGraphics.**CoordinateIndexSet**() As [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md).

## Version

Introduced in version 5
