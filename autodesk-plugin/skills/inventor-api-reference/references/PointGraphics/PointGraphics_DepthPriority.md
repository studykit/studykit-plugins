# PointGraphics.DepthPriority Property

Parent Object: [PointGraphics](../PointGraphics/PointGraphics.md)

## Description

Read-write property that allows you specify a priority to a set.

## Syntax

PointGraphics.**DepthPriority**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 5
