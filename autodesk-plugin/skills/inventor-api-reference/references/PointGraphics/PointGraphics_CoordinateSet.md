# PointGraphics.CoordinateSet Property

Parent Object: [PointGraphics](../PointGraphics/PointGraphics.md)

## Description

Gets and sets the GraphicsCoordinateSet associate with set.

## Syntax

PointGraphics.**CoordinateSet**() As [GraphicsCoordinateSet](../GraphicsCoordinateSet/GraphicsCoordinateSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsCoordinateSet](../GraphicsCoordinateSet/GraphicsCoordinateSet.md).
