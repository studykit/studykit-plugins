# BalloonValueSet.ReferencedRow Property

Parent Object: [BalloonValueSet](../BalloonValueSet/BalloonValueSet.md)

## Description

Property that returns the referenced DrawingBOMRow object.

## Syntax

BalloonValueSet.**ReferencedRow**() As [DrawingBOMRow](../DrawingBOMRow/DrawingBOMRow.md)

## Property Value

This is a read only property whose value is a [DrawingBOMRow](../DrawingBOMRow/DrawingBOMRow.md).
