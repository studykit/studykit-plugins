# BalloonValueSet.Parent Property

Parent Object: [BalloonValueSet](../BalloonValueSet/BalloonValueSet.md)

## Description

Property returning the parent balloon.

## Syntax

BalloonValueSet.**Parent**() As [Balloon](../Balloon/Balloon.md)

## Property Value

This is a read only property whose value is a [Balloon](../Balloon/Balloon.md).

## Version

Introduced in version 9
