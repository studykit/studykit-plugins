# SketchCircles3D Object

## Description

The SketchCircles3D object provides access to all the objects in a sketch. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddByCenterRadius](../SketchCircles3D/SketchCircles3D_AddByCenterRadius.md) | Method that creates a new sketch circle at a specified location and radius. |
| [AddByThreePoints](../SketchCircles3D/SketchCircles3D_AddByThreePoints.md) | Method that creates a new sketch circle passing through the three input points. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchCircles3D/SketchCircles3D_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchCircles3D/SketchCircles3D_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchCircles3D/SketchCircles3D_Item.md) | Returns the specified SketchCircle3D object from the collection. |
| [Type](../SketchCircles3D/SketchCircles3D_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
