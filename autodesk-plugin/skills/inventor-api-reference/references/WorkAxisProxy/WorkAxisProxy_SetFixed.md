# WorkAxisProxy.SetFixed Method

Parent Object: [WorkAxisProxy](../WorkAxisProxy/WorkAxisProxy.md)

## Description

Method that redefines the work axis so it passes through the input point in the direction of the input vector.

## Syntax

WorkAxisProxy.**SetFixed**( ***Point*** As [Point](../Point/Point.md), ***Axis*** As [UnitVector](../UnitVector/UnitVector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input Point object. |
| Axis | [UnitVector](../UnitVector/UnitVector.md) | Input UnitVector object. |

## Version

Introduced in version 2008
