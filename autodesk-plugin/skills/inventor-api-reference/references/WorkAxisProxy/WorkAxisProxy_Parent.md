# WorkAxisProxy.Parent Property

Parent Object: [WorkAxisProxy](../WorkAxisProxy/WorkAxisProxy.md)

## Description

Property returning the parent  object.

## Syntax

WorkAxisProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version
