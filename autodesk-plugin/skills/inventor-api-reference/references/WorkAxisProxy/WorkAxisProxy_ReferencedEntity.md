# WorkAxisProxy.ReferencedEntity Property

Parent Object: [WorkAxisProxy](../WorkAxisProxy/WorkAxisProxy.md)

## Description

Property indicating the object this entity is dependent on.

## Syntax

WorkAxisProxy.**ReferencedEntity**() As [WorkAxis](../WorkAxis/WorkAxis.md)

## Property Value

This is a read only property whose value is a [WorkAxis](../WorkAxis/WorkAxis.md).

## Version

Introduced in version 5.3
