# FinishDefinition.Parent Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

FinishDefinition.**Parent**() As [FinishFeature](../FinishFeature/FinishFeature.md)

## Property Value

This is a read only property whose value is a [FinishFeature](../FinishFeature/FinishFeature.md).

## Version

Introduced in version 2024
