# FinishDefinition.Thickness Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that provides access to the thickness of the feature.

## Syntax

FinishDefinition.**Thickness**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2024
