# FinishDefinition.FinishType Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that gets and sets the finish type for the finish feature.

## Syntax

FinishDefinition.**FinishType**() As [FinishTypeEnum](../FinishTypeEnum.md)

## Property Value

This is a read/write property whose value is a [FinishTypeEnum](../FinishTypeEnum.md).

## Version

Introduced in version 2024
