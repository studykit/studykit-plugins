# FinishDefinition.ExcludedEntities Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that gets and sets the Face objects and their proxies that will be excluded from the finish feature. Valid objects in the ObjectCollection can be Face objects and the proxies.

## Syntax

FinishDefinition.**ExcludedEntities**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2025.1
