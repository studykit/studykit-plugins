# FlatPatternOrientation.FlipBaseFace Property

Parent Object: [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Description

Read-write property that gets and sets whether to flip the flat pattern by swapping the top and bottom faces or not.

## Syntax

FlatPatternOrientation.**FlipBaseFace**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
