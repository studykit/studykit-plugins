# FlatPatternOrientation.Parent Property

Parent Object: [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

FlatPatternOrientation.**Parent**() As [FlatPattern](../FlatPattern/FlatPattern.md)

## Property Value

This is a read only property whose value is a [FlatPattern](../FlatPattern/FlatPattern.md).

## Version

Introduced in version 2015
