# FlatPatternOrientation.AlignmentType Property

Parent Object: [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Description

Read-write property that gets and sets the alignment type of flat pattern orientation.

## Syntax

FlatPatternOrientation.**AlignmentType**() As [AlignmentTypeEnum](../AlignmentTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AlignmentTypeEnum](../AlignmentTypeEnum.md).

## Version

Introduced in version 2015
