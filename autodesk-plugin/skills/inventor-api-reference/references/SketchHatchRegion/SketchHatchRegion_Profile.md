# SketchHatchRegion.Profile Property

Parent Object: [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md)

## Description

Read-only property that returns the profile associated with the sketch hatch region.

## Syntax

SketchHatchRegion.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read only property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 2021
