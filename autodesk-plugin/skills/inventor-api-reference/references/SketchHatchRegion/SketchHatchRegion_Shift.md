# SketchHatchRegion.Shift Property

Parent Object: [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md)

## Description

Read-write property that specifies the hatch pattern offset from original hatch pattern position.

## Syntax

SketchHatchRegion.**Shift**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2021
