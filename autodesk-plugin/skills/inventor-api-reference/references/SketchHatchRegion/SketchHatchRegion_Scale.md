# SketchHatchRegion.Scale Property

Parent Object: [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md)

## Description

Read-write property that specifies the scale of the distance between lines.

## Syntax

SketchHatchRegion.**Scale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2021
