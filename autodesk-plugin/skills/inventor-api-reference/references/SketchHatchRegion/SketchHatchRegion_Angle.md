# SketchHatchRegion.Angle Property

Parent Object: [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md)

## Description

Read-write property that gets and sets the rotation angle of the pattern in radians.

## Syntax

SketchHatchRegion.**Angle**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2021
