# SketchHatchRegion.LineWeight Property

Parent Object: [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md)

## Description

Read-write property that specifies the line weight of the sketch hatch region object in centimeters. A value of 0 indicates the line weight is By Layer.

## Syntax

SketchHatchRegion.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2021
