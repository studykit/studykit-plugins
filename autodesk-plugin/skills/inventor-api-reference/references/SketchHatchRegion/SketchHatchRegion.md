SketchHatchRegion Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Angle](../SketchHatchRegion/SketchHatchRegion_Angle.md) | Read-write property that gets and sets the rotation angle of the pattern in radians. |
| [Application](../SketchHatchRegion/SketchHatchRegion_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Color](../SketchHatchRegion/SketchHatchRegion_Color.md) | Read-write property that gets and sets the color of the hatch pattern. |
| [DoublePattern](../SketchHatchRegion/SketchHatchRegion_DoublePattern.md) | Read-write property that specifies if a duplicate hatch pattern should be created that is perpendicular to the hatch pattern assigned to the sketch hatch region. |
| [Layer](../SketchHatchRegion/SketchHatchRegion_Layer.md) | Read-write property that gets and sets the layer associated with the hatch region object. |
| [LineWeight](../SketchHatchRegion/SketchHatchRegion_LineWeight.md) | Read-write property that specifies the line weight of the sketch hatch region object in centimeters. A value of 0 indicates the line weight is By Layer. |
| [Parent](../SketchHatchRegion/SketchHatchRegion_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Pattern](../SketchHatchRegion/SketchHatchRegion_Pattern.md) | Read-write property that gets and sets the DrawingHatchPattern of the sketch hatch region. |
| [Profile](../SketchHatchRegion/SketchHatchRegion_Profile.md) | Read-only property that returns the profile associated with the sketch hatch region. |
| [Scale](../SketchHatchRegion/SketchHatchRegion_Scale.md) | Read-write property that specifies the scale of the distance between lines. |
| [Shift](../SketchHatchRegion/SketchHatchRegion_Shift.md) | Read-write property that specifies the hatch pattern offset from original hatch pattern position. |
| [Type](../SketchHatchRegion/SketchHatchRegion_Type.md) | Gets the constant that indicates the type of this object. |
