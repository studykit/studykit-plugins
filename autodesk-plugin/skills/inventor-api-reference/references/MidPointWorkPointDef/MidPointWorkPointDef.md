# MidPointWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point defined to be at the midpoint of an edge.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MidPointWorkPointDef/MidPointWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Edge](../MidPointWorkPointDef/MidPointWorkPointDef_Edge.md) | Property that returns the object of a work point that is defined to be at the center of the edge. Any open edge is valid as input. Inputting a closed edge, (i.e. circle), will cause and error to occur. |
| [Parent](../MidPointWorkPointDef/MidPointWorkPointDef_Parent.md) | Property returning the parent WorkPoint object. |
| [Type](../MidPointWorkPointDef/MidPointWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
