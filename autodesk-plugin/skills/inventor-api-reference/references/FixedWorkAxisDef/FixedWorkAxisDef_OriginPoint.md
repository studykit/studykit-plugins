# FixedWorkAxisDef.OriginPoint Property

Parent Object: [FixedWorkAxisDef](../FixedWorkAxisDef/FixedWorkAxisDef.md)

## Description

Property that indicates the origin point of the work axis.

## Syntax

FixedWorkAxisDef.**OriginPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
