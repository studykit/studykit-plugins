# FixedWorkAxisDef.GetData Method

Parent Object: [FixedWorkAxisDef](../FixedWorkAxisDef/FixedWorkAxisDef.md)

## Description

Method that gets all of the data defining a fixed work axis.

## Syntax

FixedWorkAxisDef.**GetData**( ***OriginPoint*** As [Point](../Point/Point.md), ***Axis*** As [UnitVector](../UnitVector/UnitVector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| OriginPoint | [Point](../Point/Point.md) | Output object that defines the origin of the work axis. |
| Axis | [UnitVector](../UnitVector/UnitVector.md) | Output object that defines the X-axis vector of the work axis. |

## Version

Introduced in version 4
