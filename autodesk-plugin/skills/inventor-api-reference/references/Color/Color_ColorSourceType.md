# Color.ColorSourceType Property

Parent Object: [Color](../Color/Color.md)

## Description

Gets and sets the color source type.

## Syntax

Color.**ColorSourceType**() As [ColorSourceTypeEnum](../ColorSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ColorSourceTypeEnum](../ColorSourceTypeEnum.md).

## Version

Introduced in version 2010
