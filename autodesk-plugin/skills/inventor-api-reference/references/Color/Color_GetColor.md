# Color.GetColor Method

Parent Object: [Color](../Color/Color.md)

## Description

Method that gets the color components of the Color object.

## Syntax

Color.**GetColor**( ***Red*** As Byte, ***Green*** As Byte, ***Blue*** As Byte )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Red | Byte | Output Byte that specifies the red component of the color. This value will be between 0 and 255. |
| Green | Byte | Output Byte that specifies the green component of the color. This value will be between 0 and 255. |
| Blue | Byte | Output Byte that specifies the blue component of the color. This value will be between 0 and 255. |

## Version

Introduced in version 8
