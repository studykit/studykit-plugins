# CurveGraphics.RemoveViewSpaceAnchor Method

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

CurveGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 2008
