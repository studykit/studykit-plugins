# CurveGraphics.LineDefinitionSpace Property

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

Gets and sets the LineDefinitionSpace applied to this curve graphics.

## Syntax

CurveGraphics.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2008
