# CurveGraphics.Color Property

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

Gets and sets color associated with this primitive.

## Syntax

CurveGraphics.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2008
