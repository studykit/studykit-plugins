# CurveGraphics.Curve Property

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

Gets and sets curve associated with this primitive.

## Syntax

CurveGraphics.**Curve**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2008
