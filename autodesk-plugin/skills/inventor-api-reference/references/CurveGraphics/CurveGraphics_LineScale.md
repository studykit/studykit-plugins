# CurveGraphics.LineScale Property

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

Gets and sets the LineScale applied to this curve graphics.

## Syntax

CurveGraphics.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
