# CurveGraphics.Id Property

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

Read-only property that returns the Id of the object.

## Syntax

CurveGraphics.**Id**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2017
