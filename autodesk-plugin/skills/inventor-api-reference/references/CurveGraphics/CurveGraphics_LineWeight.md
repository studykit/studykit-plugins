# CurveGraphics.LineWeight Property

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

Gets and sets the LineWeight applied to this curve graphics.

## Syntax

CurveGraphics.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
