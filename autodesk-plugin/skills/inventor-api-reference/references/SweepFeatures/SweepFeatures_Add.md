# SweepFeatures.Add Method

Parent Object: [SweepFeatures](../SweepFeatures/SweepFeatures.md)

## Description

Method that creates a new SweepFeature object.

## Syntax

SweepFeatures.**Add**( ***Definition*** As [SweepDefinition](../SweepDefinition/SweepDefinition.md) ) As [SweepFeature](../SweepFeature/SweepFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [SweepDefinition](../SweepDefinition/SweepDefinition.md) | Input SweepDefinition object that defines the input for the sweep feature. An SweepDefinition object can be created using the SweepFeatures.CreateSweepDefinition method. It can also be obtained from an existing SweepFeature object. |

## Version

Introduced in version 2015
