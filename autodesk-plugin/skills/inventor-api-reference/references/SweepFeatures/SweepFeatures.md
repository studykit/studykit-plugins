# SweepFeatures Object

## Description

The SweepFeatures collection object provides access to all of the objects in a component definition and provides methods to create additional SweepFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SweepFeatures/SweepFeatures_Add.md) | Method that creates a new SweepFeature object. |
| [AddSolidSweep](../SweepFeatures/SweepFeatures_AddSolidSweep.md) | Method that creates a new SweepFeature object based on a surfacebody. |
| [CreateSolidSweepDefinition](../SweepFeatures/SweepFeatures_CreateSolidSweepDefinition.md) | Method that creates the SolidSweepDefinition object. |
| [CreateSweepDefinition](../SweepFeatures/SweepFeatures_CreateSweepDefinition.md) | Method that creates the SweepDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SweepFeatures/SweepFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SweepFeatures/SweepFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SweepFeatures/SweepFeatures_Item.md) | Method that returns the specified SweepFeature object from the collection. |
| [Type](../SweepFeatures/SweepFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
