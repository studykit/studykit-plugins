# UserCoordinateSystems.Item Property

Parent Object: [UserCoordinateSystems](../UserCoordinateSystems/UserCoordinateSystems.md)

## Description

Returns the specified UserCoordinateSystem object from the collection.

## Syntax

UserCoordinateSystems.**Item**( ***Index*** As Variant ) As [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Property Value

This is a read only property whose value is a [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the UserCoordinateSystem to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the UserCoordinateSystem name. If an out of range index or a name of a non-existent UserCoordinateSystem is provided, an error will occur. |

## Version

Introduced in version 2010
