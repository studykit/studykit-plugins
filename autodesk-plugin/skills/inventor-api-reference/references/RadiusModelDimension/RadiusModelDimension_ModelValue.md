# RadiusModelDimension.ModelValue Property

Parent Object: [RadiusModelDimension](../RadiusModelDimension/RadiusModelDimension.md)

## Description

Read-only property that gets the dimension value as defined in the model.

## Syntax

RadiusModelDimension.**ModelValue**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2018
