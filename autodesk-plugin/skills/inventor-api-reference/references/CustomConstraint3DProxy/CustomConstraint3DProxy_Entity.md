# CustomConstraint3DProxy.Entity Property

Parent Object: [CustomConstraint3DProxy](../CustomConstraint3DProxy/CustomConstraint3DProxy.md)

## Description

Property that returns the entity to which this custom constraint has been applied.

## Syntax

CustomConstraint3DProxy.**Entity**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 11
