# ImportedWorkGeometryTypeEnum Enumerator

## Description

Specifies the work geometry type.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kWorkAxesWorkGeometryType | 4 | Indicates work axes are included. |
| kWorkPlanesWorkGeometryType | 1 | Indicates work planes are included. |
| kWorkPointsWorkGeometryType | 2 | Indicates work points are included. |

## Version

Introduced in version 2016
