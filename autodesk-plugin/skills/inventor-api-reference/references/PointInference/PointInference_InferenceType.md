# PointInference.InferenceType Property

Parent Object: [PointInference](../PointInference/PointInference.md)

## Description

Property that returns the type of inference.

## Syntax

PointInference.**InferenceType**() As [PointInferenceEnum](../PointInferenceEnum.md)

## Property Value

This is a read only property whose value is a [PointInferenceEnum](../PointInferenceEnum.md).

## Version

Introduced in version 5
