# SketchSpline3DProxy.FitPointCount Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Property that returns the number of fit points for the spline.

## Syntax

SketchSpline3DProxy.**FitPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 8
