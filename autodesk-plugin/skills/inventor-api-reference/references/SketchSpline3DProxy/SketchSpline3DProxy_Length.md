# SketchSpline3DProxy.Length Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchSpline3DProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
