# SketchSpline3DProxy.CurvatureDisplay Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Specifies whether curvature information is displayed for the spline.

## Syntax

SketchSpline3DProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 8
