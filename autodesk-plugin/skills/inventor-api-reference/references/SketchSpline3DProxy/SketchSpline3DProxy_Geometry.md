# SketchSpline3DProxy.Geometry Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Gets and sets a BSplineCurve geometry object. The object returned represents a 'snapshot' view of the current state of the sketch spline.

## Syntax

SketchSpline3DProxy.**Geometry**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read/write property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 6
