# SketchSpline3DProxy.StartTangentScale Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Gets and sets the tangent scale parameter at the start point of the spline.

## Syntax

SketchSpline3DProxy.**StartTangentScale**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 8
