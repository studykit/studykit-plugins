# BreakOperations.Item Property

Parent Object: [BreakOperations](../BreakOperations/BreakOperations.md)

## Description

Method that returns the specified break operation object from the collection.

## Syntax

BreakOperations.**Item**( ***Index*** As Long ) As [BreakOperation](../BreakOperation/BreakOperation.md)

## Property Value

This is a read only property whose value is a [BreakOperation](../BreakOperation/BreakOperation.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
