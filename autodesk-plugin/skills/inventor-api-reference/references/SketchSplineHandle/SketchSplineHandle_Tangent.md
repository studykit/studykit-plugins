# SketchSplineHandle.Tangent Property

Parent Object: [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md)

## Description

Gets and sets the tangent vector at the fit point associated with this handle.

## Syntax

SketchSplineHandle.**Tangent**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read/write property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).

## Version

Introduced in version 2008
