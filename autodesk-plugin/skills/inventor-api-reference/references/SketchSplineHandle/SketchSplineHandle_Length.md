# SketchSplineHandle.Length Property

Parent Object: [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchSplineHandle.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
