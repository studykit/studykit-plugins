# SketchSplineHandle.Construction Property

Parent Object: [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md)

## Description

Gets and sets whether the entity behaves as a construction entity or not.

## Syntax

SketchSplineHandle.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
