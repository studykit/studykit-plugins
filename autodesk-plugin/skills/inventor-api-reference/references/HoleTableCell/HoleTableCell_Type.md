# HoleTableCell.Type Property

Parent Object: [HoleTableCell](../HoleTableCell/HoleTableCell.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

HoleTableCell.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
