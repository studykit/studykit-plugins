# HoleTableCell.StackedTextPosition Property

Parent Object: [HoleTableCell](../HoleTableCell/HoleTableCell.md)

## Description

Gets and sets the position (alignment) of the stacked text with respect to regular text.

## Syntax

HoleTableCell.**StackedTextPosition**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 2009
