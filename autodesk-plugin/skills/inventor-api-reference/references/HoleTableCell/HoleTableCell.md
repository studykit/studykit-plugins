# HoleTableCell Object

## Description

The HoleTableCell object represents an individual cell in the hole table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HoleTableCell/HoleTableCell_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [FormattedText](../HoleTableCell/HoleTableCell_FormattedText.md) | Gets/Sets formatted text of the HoleTable cell. |
| [Parent](../HoleTableCell/HoleTableCell_Parent.md) | Property that returns the parent HoleTable object. |
| [StackedTextPosition](../HoleTableCell/HoleTableCell_StackedTextPosition.md) | Gets and sets the position (alignment) of the stacked text with respect to regular text. |
| [Text](../HoleTableCell/HoleTableCell_Text.md) | Gets/Sets the text of the HoleTable cell. |
| [Type](../HoleTableCell/HoleTableCell_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[HoleTableRow.Item](../HoleTableRow/HoleTableRow_Item.md)

## Version

Introduced in version 10
