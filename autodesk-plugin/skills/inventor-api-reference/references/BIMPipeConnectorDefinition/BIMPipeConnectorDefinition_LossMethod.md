# BIMPipeConnectorDefinition.LossMethod Property

Parent Object: [BIMPipeConnectorDefinition](../BIMPipeConnectorDefinition/BIMPipeConnectorDefinition.md)

## Description

Read-write property that gets and sets the loss method used for this connector.

## Syntax

BIMPipeConnectorDefinition.**LossMethod**() As [BIMPipeLossMethodEnum](../BIMPipeLossMethodEnum.md)

## Property Value

This is a read/write property whose value is a [BIMPipeLossMethodEnum](../BIMPipeLossMethodEnum.md).

## Version

Introduced in version 2011
