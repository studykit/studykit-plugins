# BIMPipeConnectorDefinition.ConnectionType Property

Parent Object: [BIMPipeConnectorDefinition](../BIMPipeConnectorDefinition/BIMPipeConnectorDefinition.md)

## Description

Read-write property that gets and sets the connection type for this connector.

## Syntax

BIMPipeConnectorDefinition.**ConnectionType**() As [BIMPipeConnectionTypeEnum](../BIMPipeConnectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BIMPipeConnectionTypeEnum](../BIMPipeConnectionTypeEnum.md).

## Version

Introduced in version 2011
