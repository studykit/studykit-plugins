# WeldSymbol.EnableSecondaryFilletWeld Property

Parent Object: [WeldSymbol](../WeldSymbol/WeldSymbol.md)

## Description

Gets and sets whether to enable the secondary fillet weld or not.

## Syntax

WeldSymbol.**EnableSecondaryFilletWeld**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
