# WeldSymbol.Contour Property

Parent Object: [WeldSymbol](../WeldSymbol/WeldSymbol.md)

## Description

Gets and sets the contour finish type for the weld.

## Syntax

WeldSymbol.**Contour**() As [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md).

## Version

Introduced in version 2024
