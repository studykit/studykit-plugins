# GraphicsDataSet Object

## Description

The GraphicsDataSet object is the base class for the various utility objects used for creating custom graphics.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../GraphicsDataSet/GraphicsDataSet_Delete.md) | Method that deletes the GraphicsDataSet. |
| [Remove](../GraphicsDataSet/GraphicsDataSet_Remove.md) | Method that removes a coordinate from the set. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../GraphicsDataSet/GraphicsDataSet_Count.md) | Property that returns the number of objects in this collection. |
| [Id](../GraphicsDataSet/GraphicsDataSet_Id.md) | Property returning the unique id of this GraphicsDataSet. |
| [Type](../GraphicsDataSet/GraphicsDataSet_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
