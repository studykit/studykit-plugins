# TriangleGraphics.BurnThrough Property

Parent Object: [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Description

Read-write property that specifies whether or not graphics are always visible even if they are blocked by other object.

## Syntax

TriangleGraphics.**BurnThrough**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
