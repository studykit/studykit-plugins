# TriangleGraphics.SetCustomImage Method

Parent Object: [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Description

Method that sets the custom image to use for this TriangleGraphics object.

## Syntax

TriangleGraphics.**SetCustomImage**( ***ImageSet*** As [GraphicsImageSet](../GraphicsImageSet/GraphicsImageSet.md), ***ImageIndex*** As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ImageSet | [GraphicsImageSet](../GraphicsImageSet/GraphicsImageSet.md) | Input GraphicsImageSet to assign to this TriangleGraphics object. |
| ImageIndex | Long | Input Long that indicates which image in the GraphicsImageSet to use for this TriangleGraphics object. |

## Version

Introduced in version 2017
