# TriangleGraphics.HasCustomImage Property

Parent Object: [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Description

Read-write property that gets and sets whether a custom image is set for this TriangleGraphics object.

## Syntax

TriangleGraphics.**HasCustomImage**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
