# ReportLocationEnum Enumerator

## Description

Constants identifying the save location for a translation report.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kEmbeddedReport | 107778 | Translation report saved inside the document. |
| kExternalReport | 107779 | Translation report saved to an external file. |
| kNoReport | 107777 | No translation report generated. |

## Version

Introduced in version 2016
