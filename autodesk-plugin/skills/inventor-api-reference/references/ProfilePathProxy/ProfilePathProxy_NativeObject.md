# ProfilePathProxy.NativeObject Property

Parent Object: [ProfilePathProxy](../ProfilePathProxy/ProfilePathProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ProfilePathProxy.**NativeObject**() As [ProfilePath](../ProfilePath/ProfilePath.md)

## Property Value

This is a read only property whose value is a [ProfilePath](../ProfilePath/ProfilePath.md).

## Version

Introduced in version 6
