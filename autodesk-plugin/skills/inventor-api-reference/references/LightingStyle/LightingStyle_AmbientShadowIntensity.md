# LightingStyle.AmbientShadowIntensity Property

Parent Object: [LightingStyle](../LightingStyle/LightingStyle.md)

## Description

Gets and sets the intensity of the ambient shadow (occlusion).

## Syntax

LightingStyle.**AmbientShadowIntensity**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
