# LightingStyle.ImageBasedLightingShowImage Property

Parent Object: [LightingStyle](../LightingStyle/LightingStyle.md)

## Description

Gets and sets whether to display the image in the graphics scene.

## Syntax

LightingStyle.**ImageBasedLightingShowImage**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
