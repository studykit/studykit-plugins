# LightingStyle.LightingStyleType Property

Parent Object: [LightingStyle](../LightingStyle/LightingStyle.md)

## Description

Gets the lighting style type.

## Syntax

LightingStyle.**LightingStyleType**() As [LightingStyleTypeEnum](../LightingStyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [LightingStyleTypeEnum](../LightingStyleTypeEnum.md).

## Version

Introduced in version 2024
