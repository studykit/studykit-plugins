# Polyline2d.PointAtIndex Property

Parent Object: [Polyline2d](../Polyline2d/Polyline2d.md)

## Description

Gets or sets the x, y position of a specified point.

## Syntax

Polyline2d.**PointAtIndex**( ***Index*** As Long ) As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long |  |
