# Polyline2d.Copy Method

Parent Object: [Polyline2d](../Polyline2d/Polyline2d.md)

## Description

Creates a copy of this Polyline2d object. The result is entirely independent and can be edited without affecting the original Polyline2d object.

## Syntax

Polyline2d.**Copy**() As [Polyline2d](../Polyline2d/Polyline2d.md)

## Version

Introduced in version 2013
