# ShellFeatureProxy.Name Property

Parent Object: [ShellFeatureProxy](../ShellFeatureProxy/ShellFeatureProxy.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

ShellFeatureProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 9
