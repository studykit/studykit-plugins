# MapPointCurves.Item Property

Parent Object: [MapPointCurves](../MapPointCurves/MapPointCurves.md)

## Description

Method that returns the specified MapPointCurve object from the collection.

## Syntax

MapPointCurves.**Item**( ***Index*** As Long ) As [MapPointCurve](../MapPointCurve/MapPointCurve.md)

## Property Value

This is a read only property whose value is a [MapPointCurve](../MapPointCurve/MapPointCurve.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the MapPointCurve to return. |

## Version

Introduced in version 6
