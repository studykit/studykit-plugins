# MapPointCurves.Count Property

Parent Object: [MapPointCurves](../MapPointCurves/MapPointCurves.md)

## Description

Property that returns the number of MapPointCurve objects in the collection.

## Syntax

MapPointCurves.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 6
