# Tolerance.SetToLimits Method

Parent Object: [Tolerance](../Tolerance/Tolerance.md)

## Description

Method that sets the type of this tolerance to be a limits tolerance.

## Syntax

Tolerance.**SetToLimits**( ***LimitsToleranceType*** As [ToleranceTypeEnum](../ToleranceTypeEnum.md), ***UpperTolerance*** As Variant, ***LowerTolerance*** As Variant )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LimitsToleranceType | [ToleranceTypeEnum](../ToleranceTypeEnum.md) | constant Indicates the type of limits tolerance. |
| UpperTolerance | Variant | Input Variant that defines the upper limit of the tolerance. |
| LowerTolerance | Variant | Input Variant that defines the lower limit of the tolerance. |

## Version

Introduced in version 6
