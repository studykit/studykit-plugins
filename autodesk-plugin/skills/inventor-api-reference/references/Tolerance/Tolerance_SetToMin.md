# Tolerance.SetToMin Method

Parent Object: [Tolerance](../Tolerance/Tolerance.md)

## Description

Sets the type of this tolerance to be a MIN type of tolerance.

## Syntax

Tolerance.**SetToMin**()

## Version

Introduced in version 10
