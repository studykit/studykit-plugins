# DrawingSketches Object

## Description

The DrawingSketches collection object provides access to the objects associated with the object the collection was obtained from.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../DrawingSketches/DrawingSketches_Add.md) | Method that creates a new drawing sketch. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingSketches/DrawingSketches_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DrawingSketches/DrawingSketches_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DrawingSketches/DrawingSketches_Item.md) | Returns the specified DrawingSketch object from the collection. |
| [Type](../DrawingSketches/DrawingSketches_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DetailDrawingView.Sketches](../DetailDrawingView/DetailDrawingView_Sketches.md), [DrawingView.Sketches](../DrawingView/DrawingView_Sketches.md), [SectionDrawingView.Sketches](../SectionDrawingView/SectionDrawingView_Sketches.md), [Sheet.Sketches](../Sheet/Sheet_Sketches.md)

## Version

Introduced in version 5.3
