# FilletConstantRadiusEdgeSet.AllRounds Property

Parent Object: [FilletConstantRadiusEdgeSet](../FilletConstantRadiusEdgeSet/FilletConstantRadiusEdgeSet.md)

## Description

Property that returns if the edge set defines all convex edges.

## Syntax

FilletConstantRadiusEdgeSet.**AllRounds**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5.3
