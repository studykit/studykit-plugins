# FilletConstantRadiusEdgeSet.Radius Property

Parent Object: [FilletConstantRadiusEdgeSet](../FilletConstantRadiusEdgeSet/FilletConstantRadiusEdgeSet.md)

## Description

Property that returns the parameter that controls the radius of the fillet. This property will return Nothing if the fillet feature has not been created yet.

## Syntax

FilletConstantRadiusEdgeSet.**Radius**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 5.3
