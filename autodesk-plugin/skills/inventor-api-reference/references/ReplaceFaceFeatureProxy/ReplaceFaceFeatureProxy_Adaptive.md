# ReplaceFaceFeatureProxy.Adaptive Property

Parent Object: [ReplaceFaceFeatureProxy](../ReplaceFaceFeatureProxy/ReplaceFaceFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

ReplaceFaceFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
