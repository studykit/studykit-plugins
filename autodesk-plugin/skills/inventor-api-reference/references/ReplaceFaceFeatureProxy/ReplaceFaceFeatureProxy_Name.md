# ReplaceFaceFeatureProxy.Name Property

Parent Object: [ReplaceFaceFeatureProxy](../ReplaceFaceFeatureProxy/ReplaceFaceFeatureProxy.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

ReplaceFaceFeatureProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 9
