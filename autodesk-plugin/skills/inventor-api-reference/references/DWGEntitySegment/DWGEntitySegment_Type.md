# DWGEntitySegment.Type Property

Parent Object: [DWGEntitySegment](../DWGEntitySegment/DWGEntitySegment.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

DWGEntitySegment.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2018
