# DWGEntitySegment.Parent Property

Parent Object: [DWGEntitySegment](../DWGEntitySegment/DWGEntitySegment.md)

## Description

Read-only property that returns the parent DWGEntity.

## Syntax

DWGEntitySegment.**Parent**() As [DWGEntity](../DWGEntity/DWGEntity.md)

## Property Value

This is a read only property whose value is a [DWGEntity](../DWGEntity/DWGEntity.md).

## Version

Introduced in version 2018
