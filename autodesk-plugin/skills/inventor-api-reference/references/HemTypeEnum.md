# HemTypeEnum Enumerator

## Description

Defines the constants used to specify the type of hems for a hem feature.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDoubleHemType | 75268 | Double hem type. |
| kRolledHemType | 75267 | Rolled hem type. |
| kSingleHemType | 75265 | Single hem type. |
| kTeardropHemType | 75266 | Teardrop hem type. |
