# RuledSurfaceFeatureProxy.NativeObject Property

Parent Object: [RuledSurfaceFeatureProxy](../RuledSurfaceFeatureProxy/RuledSurfaceFeatureProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

RuledSurfaceFeatureProxy.**NativeObject**() As [RuledSurfaceFeature](../RuledSurfaceFeature/RuledSurfaceFeature.md)

## Property Value

This is a read only property whose value is a [RuledSurfaceFeature](../RuledSurfaceFeature/RuledSurfaceFeature.md).

## Version

Introduced in version 2016
