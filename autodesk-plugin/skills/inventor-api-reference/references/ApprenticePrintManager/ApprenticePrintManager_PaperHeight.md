# ApprenticePrintManager.PaperHeight Property

Parent Object: [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Description

Gets/Sets the height of the paper to be printed. Setting this, sets PaperSize to kPaperSizeCustom.

## Syntax

ApprenticePrintManager.**PaperHeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
