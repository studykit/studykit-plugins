# ApprenticePrintManager.PaperSize Property

Parent Object: [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Description

Gets/Sets the size of the paper to be printed. Defaults to the printer's default.

## Syntax

ApprenticePrintManager.**PaperSize**() As [PaperSizeEnum](../PaperSizeEnum.md)

## Property Value

This is a read/write property whose value is a [PaperSizeEnum](../PaperSizeEnum.md).

## Version

Introduced in version 9
