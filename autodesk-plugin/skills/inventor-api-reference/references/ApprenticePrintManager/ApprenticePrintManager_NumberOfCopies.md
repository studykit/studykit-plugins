# ApprenticePrintManager.NumberOfCopies Property

Parent Object: [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Description

Gets/Sets the number of copies to be printed. Defaults to 1.

## Syntax

ApprenticePrintManager.**NumberOfCopies**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 9
