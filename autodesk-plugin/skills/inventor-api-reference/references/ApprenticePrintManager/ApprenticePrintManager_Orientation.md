# ApprenticePrintManager.Orientation Property

Parent Object: [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Description

Gets/Sets the orientation of the printer. Defaults to the printer's default.

## Syntax

ApprenticePrintManager.**Orientation**() As [PrintOrientationEnum](../PrintOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [PrintOrientationEnum](../PrintOrientationEnum.md).

## Version

Introduced in version 9
