# ViewTabsEnumerator.Item Property

Parent Object: [ViewTabsEnumerator](../ViewTabsEnumerator/ViewTabsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

ViewTabsEnumerator.**Item**( ***Index*** As Long ) As [ViewTab](../ViewTab/ViewTab.md)

## Property Value

This is a read only property whose value is a [ViewTab](../ViewTab/ViewTab.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the ViewTab to return. |

## Version

Introduced in version 2022
