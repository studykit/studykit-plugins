# DWGAcadSupportedProxyProxy.HandleID Property

Parent Object: [DWGAcadSupportedProxyProxy](../DWGAcadSupportedProxyProxy/DWGAcadSupportedProxyProxy.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGAcadSupportedProxyProxy.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
