# DWGAcadSupportedProxyProxy.Segments Property

Parent Object: [DWGAcadSupportedProxyProxy](../DWGAcadSupportedProxyProxy/DWGAcadSupportedProxyProxy.md)

## Description

Read-only property that returns the DWGEntitySegmentsEnumerator object.

## Syntax

DWGAcadSupportedProxyProxy.**Segments**() As [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md).

## Version

Introduced in version 2018
