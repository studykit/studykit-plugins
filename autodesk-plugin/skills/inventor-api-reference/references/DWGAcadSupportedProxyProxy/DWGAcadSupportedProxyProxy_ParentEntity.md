# DWGAcadSupportedProxyProxy.ParentEntity Property

Parent Object: [DWGAcadSupportedProxyProxy](../DWGAcadSupportedProxyProxy/DWGAcadSupportedProxyProxy.md)

## Description

Read-only property that returns the parent DWGEntity of the entity.

## Syntax

DWGAcadSupportedProxyProxy.**ParentEntity**() As [DWGEntity](../DWGEntity/DWGEntity.md)

## Property Value

This is a read only property whose value is a [DWGEntity](../DWGEntity/DWGEntity.md).

## Version

Introduced in version 2018
