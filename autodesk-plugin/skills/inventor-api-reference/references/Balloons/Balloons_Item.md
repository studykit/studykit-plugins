# Balloons.Item Property

Parent Object: [Balloons](../Balloons/Balloons.md)

## Description

Returns the specified object from the collection.

## Syntax

Balloons.**Item**( ***Index*** As Long ) As [Balloon](../Balloon/Balloon.md)

## Property Value

This is a read only property whose value is a [Balloon](../Balloon/Balloon.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index number of the Balloon to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 9
