# Balloons Object

## Description

The Balloons collection object provides access to all of the objects on a sheet. This collection is available via the Balloons property of the Sheet object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Balloons/Balloons_Add.md) | Method that creates a new Balloon. The newly created Balloon is returned. The corresponding BOMView in the model is automatically enabled if not already enabled. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Balloons/Balloons_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Balloons/Balloons_Count.md) | Property that returns the number of items in the collection. |
| [Item](../Balloons/Balloons_Item.md) | Returns the specified object from the collection. |
| [Type](../Balloons/Balloons_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Sheet.Balloons](../Sheet/Sheet_Balloons.md)

## Version

Introduced in version 9
