# ModelToleranceFeature.Name Property

Parent Object: [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Description

Gets or sets the name of the tolerance feature.

## Syntax

ModelToleranceFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
