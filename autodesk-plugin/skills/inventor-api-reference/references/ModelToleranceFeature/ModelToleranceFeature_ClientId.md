# ModelToleranceFeature.ClientId Property

Parent Object: [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Description

Returns string that uniquely identifies the client.

## Syntax

ModelToleranceFeature.**ClientId**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
