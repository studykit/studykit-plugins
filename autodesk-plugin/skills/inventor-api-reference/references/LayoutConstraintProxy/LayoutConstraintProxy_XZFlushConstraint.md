# LayoutConstraintProxy.XZFlushConstraint Property

Parent Object: [LayoutConstraintProxy](../LayoutConstraintProxy/LayoutConstraintProxy.md)

## Description

Property that returns the flush constraint that defines the XZ plane of the layout constraint.

## Syntax

LayoutConstraintProxy.**XZFlushConstraint**() As [FlushConstraint](../FlushConstraint/FlushConstraint.md)

## Property Value

This is a read only property whose value is a [FlushConstraint](../FlushConstraint/FlushConstraint.md).

## Version

Introduced in version 2010
