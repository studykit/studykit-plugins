# LayoutConstraintProxy Object

Derived from: [LayoutConstraint](../LayoutConstraint/LayoutConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../LayoutConstraintProxy/LayoutConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../LayoutConstraintProxy/LayoutConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LayoutConstraintProxy/LayoutConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../LayoutConstraintProxy/LayoutConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../LayoutConstraintProxy/LayoutConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [HealthStatus](../LayoutConstraintProxy/LayoutConstraintProxy_HealthStatus.md) | Property that returns an enum indicating the current state of the object. |
| [Name](../LayoutConstraintProxy/LayoutConstraintProxy_Name.md) | Gets/Sets the displayable name of this constraint. |
| [NativeObject](../LayoutConstraintProxy/LayoutConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../LayoutConstraintProxy/LayoutConstraintProxy_Parent.md) | Property that returns the parent of the object. |
| [Type](../LayoutConstraintProxy/LayoutConstraintProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [XYFlushConstraint](../LayoutConstraintProxy/LayoutConstraintProxy_XYFlushConstraint.md) | Property that returns the flush constraint that defines the XY plane of the layout constraint. |
| [XZFlushConstraint](../LayoutConstraintProxy/LayoutConstraintProxy_XZFlushConstraint.md) | Property that returns the flush constraint that defines the XZ plane of the layout constraint. |
| [YZFlushConstraint](../LayoutConstraintProxy/LayoutConstraintProxy_YZFlushConstraint.md) | Property that returns the flush constraint that defines the YZ plane of the layout constraint. |
| [ZAngleConstraint](../LayoutConstraintProxy/LayoutConstraintProxy_ZAngleConstraint.md) | Property that returns the angle constraint that defines the rotation about the Z axis of the layout constraint. |

## Version

Introduced in version 2010
