# RotateTranslateConstraint.Name Property

Parent Object: [RotateTranslateConstraint](../RotateTranslateConstraint/RotateTranslateConstraint.md)

## Description

Gets/Sets the displayable name of this constraint.

## Syntax

RotateTranslateConstraint.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 4
