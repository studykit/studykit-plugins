# RotateTranslateConstraint.Distance Property

Parent Object: [RotateTranslateConstraint](../RotateTranslateConstraint/RotateTranslateConstraint.md)

## Description

Gets the Parameter object that controls the distance of translation for every revolution of the rotated occurrence.

## Syntax

RotateTranslateConstraint.**Distance**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2014
