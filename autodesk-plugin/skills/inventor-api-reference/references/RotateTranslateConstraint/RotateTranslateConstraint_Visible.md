# RotateTranslateConstraint.Visible Property

Parent Object: [RotateTranslateConstraint](../RotateTranslateConstraint/RotateTranslateConstraint.md)

## Description

Gets/Sets the Visible indicating whether this constraint is visible.

## Syntax

RotateTranslateConstraint.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
