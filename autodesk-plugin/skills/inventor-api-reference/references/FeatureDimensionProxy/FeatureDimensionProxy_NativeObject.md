# FeatureDimensionProxy.NativeObject Property

Parent Object: [FeatureDimensionProxy](../FeatureDimensionProxy/FeatureDimensionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FeatureDimensionProxy.**NativeObject**() As [FeatureDimension](../FeatureDimension/FeatureDimension.md)

## Property Value

This is a read only property whose value is a [FeatureDimension](../FeatureDimension/FeatureDimension.md).

## Version

Introduced in version 2008
