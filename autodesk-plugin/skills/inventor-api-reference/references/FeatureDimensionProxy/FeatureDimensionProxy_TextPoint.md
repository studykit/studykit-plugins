# FeatureDimensionProxy.TextPoint Property

Parent Object: [FeatureDimensionProxy](../FeatureDimensionProxy/FeatureDimensionProxy.md)

## Description

Property that returns the position of the dimension text.

## Syntax

FeatureDimensionProxy.**TextPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2008
