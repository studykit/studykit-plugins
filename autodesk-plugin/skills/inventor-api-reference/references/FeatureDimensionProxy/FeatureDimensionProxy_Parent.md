# FeatureDimensionProxy.Parent Property

Parent Object: [FeatureDimensionProxy](../FeatureDimensionProxy/FeatureDimensionProxy.md)

## Description

Property that returns the parent PartFeature object.

## Syntax

FeatureDimensionProxy.**Parent**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 2008
