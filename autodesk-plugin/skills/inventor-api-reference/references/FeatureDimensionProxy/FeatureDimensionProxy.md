# FeatureDimensionProxy Object

Derived from: [FeatureDimension](../FeatureDimension/FeatureDimension.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetReferenceKey](../FeatureDimensionProxy/FeatureDimensionProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AnchorPoints](../FeatureDimensionProxy/FeatureDimensionProxy_AnchorPoints.md) | Gets the anchor points of dimension. |
| [Application](../FeatureDimensionProxy/FeatureDimensionProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../FeatureDimensionProxy/FeatureDimensionProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
