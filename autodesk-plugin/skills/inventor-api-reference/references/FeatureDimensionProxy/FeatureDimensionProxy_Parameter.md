# FeatureDimensionProxy.Parameter Property

Parent Object: [FeatureDimensionProxy](../FeatureDimensionProxy/FeatureDimensionProxy.md)

## Description

Property that returns the parameter associated with the dimension.

## Syntax

FeatureDimensionProxy.**Parameter**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2008
