# PresentationTrailSegments.Item Property

Parent Object: [PresentationTrailSegments](../PresentationTrailSegments/PresentationTrailSegments.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationTrailSegments.**Item**( ***Index*** As Long ) As [PresentationTrailSegment](../PresentationTrailSegment/PresentationTrailSegment.md)

## Property Value

This is a read only property whose value is a [PresentationTrailSegment](../PresentationTrailSegment/PresentationTrailSegment.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of an item to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
