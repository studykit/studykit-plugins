# WireDefinitions Object

## Description

Represents a set of definitions of 3D wireframe geometry that will be used to create an Wire object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../WireDefinitions/WireDefinitions_Add.md) | Creates a new WireDefinition within the associated SurfaceBodyDefinition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../WireDefinitions/WireDefinitions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../WireDefinitions/WireDefinitions_Count.md) | Property that returns the number of items in this collection. |
| [Item](../WireDefinitions/WireDefinitions_Item.md) | Returns the specified WireDefinition object from the collection |
| [Type](../WireDefinitions/WireDefinitions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FaceShellDefinition.WireDefinitions](../FaceShellDefinition/FaceShellDefinition_WireDefinitions.md)

## Version

Introduced in version 2013
