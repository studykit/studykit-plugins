# AssemblySymmetryConstraint.Name Property

Parent Object: [AssemblySymmetryConstraint](../AssemblySymmetryConstraint/AssemblySymmetryConstraint.md)

## Description

Gets/Sets the displayable name of this constraint.

## Syntax

AssemblySymmetryConstraint.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2014
