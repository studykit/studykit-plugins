# ClientFeature.BrowserNode Property

Parent Object: [ClientFeature](../ClientFeature/ClientFeature.md)

## Description

Property that returns the BrowserNode object in the model browser associated with this client feature.

## Syntax

ClientFeature.**BrowserNode**() As [BrowserNode](../BrowserNode/BrowserNode.md)

## Property Value

This is a read only property whose value is a [BrowserNode](../BrowserNode/BrowserNode.md).

## Version

Introduced in version 2008
