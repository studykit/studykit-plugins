# HoleTableStyle.IncludeCounterSinkHoleFeatures Property

Parent Object: [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md)

## Description

Gets and sets whether to include countersunk hole features.

## Syntax

HoleTableStyle.**IncludeCounterSinkHoleFeatures**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
