# HoleTableStyle.SequentialNumbering Property

Parent Object: [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md)

## Description

Gets and sets whether to replace the alphanumeric hole tags with sequential numbering of the holes in the hole table.

## Syntax

HoleTableStyle.**SequentialNumbering**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
