# HoleTableStyle.ReformatOnCustomHoleMatch Property

Parent Object: [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md)

## Description

Gets and sets whether to re-index the hole tags and resort contents when a Match Custom Hole operation is done.

## Syntax

HoleTableStyle.**ReformatOnCustomHoleMatch**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
