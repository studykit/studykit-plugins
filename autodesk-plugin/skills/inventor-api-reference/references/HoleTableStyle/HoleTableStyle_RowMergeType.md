# HoleTableStyle.RowMergeType Property

Parent Object: [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md)

## Description

Gets and sets the row merge option for the hole table.

## Syntax

HoleTableStyle.**RowMergeType**() As [RowMergeTypeEnum](../RowMergeTypeEnum.md)

## Property Value

This is a read/write property whose value is a [RowMergeTypeEnum](../RowMergeTypeEnum.md).

## Version

Introduced in version 2009
