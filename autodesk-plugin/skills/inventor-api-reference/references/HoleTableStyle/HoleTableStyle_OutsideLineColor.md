# HoleTableStyle.OutsideLineColor Property

Parent Object: [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md)

## Description

Gets and sets the color of the outer lines of the table.

## Syntax

HoleTableStyle.**OutsideLineColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2009
