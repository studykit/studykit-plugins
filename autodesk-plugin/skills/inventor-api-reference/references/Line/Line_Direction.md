# Line.Direction Property

Parent Object: [Line](../Line/Line.md)

## Description

Specifies the direction of the line.

## Syntax

Line.**Direction**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).
