# Line.RootPoint Property

Parent Object: [Line](../Line/Line.md)

## Description

Specifies the root point of the line.

## Syntax

Line.**RootPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
