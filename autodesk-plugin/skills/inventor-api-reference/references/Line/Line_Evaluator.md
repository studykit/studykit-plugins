# Line.Evaluator Property

Parent Object: [Line](../Line/Line.md)

## Description

Gets the CurveEvaluator for the line.

## Syntax

Line.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 4
