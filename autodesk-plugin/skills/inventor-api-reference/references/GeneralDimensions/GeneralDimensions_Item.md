# GeneralDimensions.Item Property

Parent Object: [GeneralDimensions](../GeneralDimensions/GeneralDimensions.md)

## Description

Method that returns the specified dimension object from the collection.

## Syntax

GeneralDimensions.**Item**( ***Index*** As Long ) As [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Property Value

This is a read only property whose value is a [GeneralDimension](../GeneralDimension/GeneralDimension.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |
