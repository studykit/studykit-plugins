# SheetSettings.SheetLabel Property

Parent Object: [SheetSettings](../SheetSettings/SheetSettings.md)

## Description

Gets/Sets the default prefix name used for newly created drawing sheets.

## Syntax

SheetSettings.**SheetLabel**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
