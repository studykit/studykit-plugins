# SheetSettings.SheetOutlineColor Property

Parent Object: [SheetSettings](../SheetSettings/SheetSettings.md)

## Description

Gets/Sets the color used for the border of drawing sheets.

## Syntax

SheetSettings.**SheetOutlineColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
