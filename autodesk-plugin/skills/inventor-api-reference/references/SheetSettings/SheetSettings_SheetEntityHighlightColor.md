# SheetSettings.SheetEntityHighlightColor Property

Parent Object: [SheetSettings](../SheetSettings/SheetSettings.md)

## Description

Gets/Sets the color used to highlight elements on the drawing sheet when the cursor passes over them.

## Syntax

SheetSettings.**SheetEntityHighlightColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
