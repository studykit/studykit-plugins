# SheetSettings.SheetColor Property

Parent Object: [SheetSettings](../SheetSettings/SheetSettings.md)

## Description

Gets/Sets the background color used for drawing sheets.

## Syntax

SheetSettings.**SheetColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
