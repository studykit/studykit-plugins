# SheetSettings Object

## Description

The SheetSettings object provides access to properties that provide read and write access of the various sheet related document specific settings. This is somewhat equivalent to the Sheet tab of the Document Settings dialog.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SheetSettings/SheetSettings_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../SheetSettings/SheetSettings_Parent.md) | Property that returns the parent DrawingDocument object. |
| [SheetColor](../SheetSettings/SheetSettings_SheetColor.md) | Gets/Sets the background color used for drawing sheets. |
| [SheetEntityHighlightColor](../SheetSettings/SheetSettings_SheetEntityHighlightColor.md) | Gets/Sets the color used to highlight elements on the drawing sheet when the cursor passes over them. |
| [SheetEntitySelectionColor](../SheetSettings/SheetSettings_SheetEntitySelectionColor.md) | Gets/Sets the color used to illuminate elements on the drawing sheet when they are selected. |
| [SheetLabel](../SheetSettings/SheetSettings_SheetLabel.md) | Gets/Sets the default prefix name used for newly created drawing sheets. |
| [SheetOutlineColor](../SheetSettings/SheetSettings_SheetOutlineColor.md) | Gets/Sets the color used for the border of drawing sheets. |
| [Type](../SheetSettings/SheetSettings_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingDocument.SheetSettings](../DrawingDocument/DrawingDocument_SheetSettings.md)

## Version

Introduced in version 11
