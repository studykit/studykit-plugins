# DerivedAssemblyComponent.SuppressAll Property

Parent Object: [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Description

Property that gets and sets the suppress state for all of the Reference features created by this derived assembly component.

## Syntax

DerivedAssemblyComponent.**SuppressAll**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
