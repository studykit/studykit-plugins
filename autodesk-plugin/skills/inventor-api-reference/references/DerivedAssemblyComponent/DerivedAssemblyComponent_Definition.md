# DerivedAssemblyComponent.Definition Property

Parent Object: [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Description

Gets or sets DerivedAssemblyDefinition that defines the current state of this derived assembly.

## Syntax

DerivedAssemblyComponent.**Definition**() As [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Property Value

This is a read/write property whose value is a [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md).

## Version

Introduced in version 5.3
