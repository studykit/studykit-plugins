# BoundaryPatchLoop Object

## Description

The BoundaryPatchLoop object is used to define a boundary loop for a boundary patch feature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../BoundaryPatchLoop/BoundaryPatchLoop_Delete.md) | Method that deletes this boundary patch loop. |
| [GetBoundaryCondition](../BoundaryPatchLoop/BoundaryPatchLoop_GetBoundaryCondition.md) | Method that gets the boundary condition for the specified boundary path loop entity. |
| [GetBoundaryTangentWeight](../BoundaryPatchLoop/BoundaryPatchLoop_GetBoundaryTangentWeight.md) | Method that gets the tangent weight for the specified boundary path loop entity. |
| [SetBoundaryCondition](../BoundaryPatchLoop/BoundaryPatchLoop_SetBoundaryCondition.md) | Method that sets the boundary condition for the specified boundary path loop entity. |
| [SetBoundaryTangentWeight](../BoundaryPatchLoop/BoundaryPatchLoop_SetBoundaryTangentWeight.md) | Method that sets the tangetn weight for the specified boundary path loop entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BoundaryPatchLoop/BoundaryPatchLoop_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [BoundaryPath](../BoundaryPatchLoop/BoundaryPatchLoop_BoundaryPath.md) | Property that gets and sets the boundary path for a boundary patch loop. |
| [Type](../BoundaryPatchLoop/BoundaryPatchLoop_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BoundaryPatchLoops.Add](../BoundaryPatchLoops/BoundaryPatchLoops_Add.md), [BoundaryPatchLoops.Item](../BoundaryPatchLoops/BoundaryPatchLoops_Item.md)

## Version

Introduced in version 2008
