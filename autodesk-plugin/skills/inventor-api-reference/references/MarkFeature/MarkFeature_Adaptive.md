# MarkFeature.Adaptive Property

Parent Object: [MarkFeature](../MarkFeature/MarkFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

MarkFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
