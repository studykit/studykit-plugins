# MarkFeature.AppearanceSourceType Property

Parent Object: [MarkFeature](../MarkFeature/MarkFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

MarkFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2023
