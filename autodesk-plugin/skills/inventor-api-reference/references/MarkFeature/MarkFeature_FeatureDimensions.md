# MarkFeature.FeatureDimensions Property

Parent Object: [MarkFeature](../MarkFeature/MarkFeature.md)

## Description

Property that returns the FeatureDimensions collection object.

## Syntax

MarkFeature.**FeatureDimensions**() As [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md)

## Property Value

This is a read only property whose value is a [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md).

## Version

Introduced in version 2023
