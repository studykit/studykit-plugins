# PunchNote.HorizontalJustification Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets/Sets horizontal justification of the DrawingNote.

## Syntax

PunchNote.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 2010
