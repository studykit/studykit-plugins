# PunchNote.UnitAttributes Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Get the unit attributes associated with the LeaderNote.

## Syntax

PunchNote.**UnitAttributes**() As [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Property Value

This is a read only property whose value is a [UnitAttributes](../UnitAttributes/UnitAttributes.md).

## Version

Introduced in version 2016
