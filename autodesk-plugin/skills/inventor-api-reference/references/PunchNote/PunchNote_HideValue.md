# PunchNote.HideValue Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets and sets whether to display the punch note value in the note.

## Syntax

PunchNote.**HideValue**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
