# PunchNote.PunchEdge Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets and sets the punch edge associated with the note.

## Syntax

PunchNote.**PunchEdge**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2010
