# PunchNote.FormattedPunchNote Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets and sets the fully formatted string that defines the contents of the punch note.

## Syntax

PunchNote.**FormattedPunchNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
