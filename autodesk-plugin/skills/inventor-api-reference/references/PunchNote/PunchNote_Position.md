# PunchNote.Position Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets/Sets the position of the DrawingNote on the sheet.

## Syntax

PunchNote.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2010
