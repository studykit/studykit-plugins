# LineWeightTypeEnum Enumerator

## Description

Enum indicating the type of line weight.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kRangeLineWeight | 66050 | Lines will be displayed with a particular line weight,which depending on the actual line thickness that falls into one of the defined range of line thickness values. |
| kTrueLineWeight | 66049 | The line weights should correspond to the actual line dimensions. |

## Version

Introduced in version 2008
