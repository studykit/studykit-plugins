# DirectEditFeature.Shared Property

Parent Object: [DirectEditFeature](../DirectEditFeature/DirectEditFeature.md)

## Description

Gets and sets whether the part feature is shared or not, applies only to surface features.

## Syntax

DirectEditFeature.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
