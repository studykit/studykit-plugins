# DirectEditFeature.DirectEditOperations Property

Parent Object: [DirectEditFeature](../DirectEditFeature/DirectEditFeature.md)

## Description

Gets all the operations of this direct edit feature.

## Syntax

DirectEditFeature.**DirectEditOperations**() As [DirectEditOperations](../DirectEditOperations/DirectEditOperations.md)

## Property Value

This is a read only property whose value is a [DirectEditOperations](../DirectEditOperations/DirectEditOperations.md).

## Version

Introduced in version 2015
