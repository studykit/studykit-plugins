# MeasureEvents.ListParameters Method

Parent Object: [MeasureEvents](../MeasureEvents/MeasureEvents.md)

## Description

Method that invokes the 'List Parameters' command, prompting the user to select a parameter from a list.

## Syntax

MeasureEvents.**ListParameters**( [***ScreenPosition***] As Variant, [***Component***] As Variant, [***DisplayAllParameters***] As Boolean, [***HWND***] As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ScreenPosition | Variant |  |
| Component | Variant | This is an optional argument whose default value is null. |
| DisplayAllParameters | Boolean | This is an optional argument whose default value is False. |
| HWND | Long | This is an optional argument whose default value is 0. |

## Version

Introduced in version 2009
