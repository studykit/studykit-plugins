# MeasureEvents.ShowDimensions Method

Parent Object: [MeasureEvents](../MeasureEvents/MeasureEvents.md)

## Description

Method that invokes the 'Show Dimensions' command, prompting the user to select a feature to show the dimensions for.

## Syntax

MeasureEvents.**ShowDimensions**()

## Version

Introduced in version 2009
