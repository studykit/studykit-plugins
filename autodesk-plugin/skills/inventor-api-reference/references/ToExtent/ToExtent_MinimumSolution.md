# ToExtent.MinimumSolution Property

Parent Object: [ToExtent](../ToExtent/ToExtent.md)

## Description

Gets and sets whether the feature terminates on the nearest valid face when there are multiple options for valid termination faces.

## Syntax

ToExtent.**MinimumSolution**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
