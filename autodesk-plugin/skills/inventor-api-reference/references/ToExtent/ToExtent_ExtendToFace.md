# ToExtent.ExtendToFace Property

Parent Object: [ToExtent](../ToExtent/ToExtent.md)

## Description

Property that gets and sets whether the plane defined by the 'to entity' should be extended to contain the extents of the profile.

## Syntax

ToExtent.**ExtendToFace**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
