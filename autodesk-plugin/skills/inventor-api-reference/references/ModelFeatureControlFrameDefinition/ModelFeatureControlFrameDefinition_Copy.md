# ModelFeatureControlFrameDefinition.Copy Method

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Method that creates a copy of this LinearModelDimensionDefinition object. The new LinearModelDimensionDefinition object is independent of any dimension. It can edited and used as input to edit an existing dimension or to create a new dimension.

## Syntax

ModelFeatureControlFrameDefinition.**Copy**() As [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Version

Introduced in version 2018
