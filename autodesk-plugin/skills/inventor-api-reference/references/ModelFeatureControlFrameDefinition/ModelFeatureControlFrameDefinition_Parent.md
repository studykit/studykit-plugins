# ModelFeatureControlFrameDefinition.Parent Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelFeatureControlFrameDefinition.**Parent**() As [ModelFeatureControlFrame](../ModelFeatureControlFrame/ModelFeatureControlFrame.md)

## Property Value

This is a read only property whose value is a [ModelFeatureControlFrame](../ModelFeatureControlFrame/ModelFeatureControlFrame.md).

## Version

Introduced in version 2018
