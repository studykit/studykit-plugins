# ModelFeatureControlFrameDefinition.Leader Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Read-only property that returns the leader associated with the feature control frame.

## Syntax

ModelFeatureControlFrameDefinition.**Leader**() As [ModelLeader](../ModelLeader/ModelLeader.md)

## Property Value

This is a read only property whose value is a [ModelLeader](../ModelLeader/ModelLeader.md).

## Version

Introduced in version 2018
