# ModelFeatureControlFrameDefinition.OverrideMergeSymbol Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Gets and sets the merge symbol override value. Set this property will set the MergeSymbolOverridden to True.

## Syntax

ModelFeatureControlFrameDefinition.**OverrideMergeSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
