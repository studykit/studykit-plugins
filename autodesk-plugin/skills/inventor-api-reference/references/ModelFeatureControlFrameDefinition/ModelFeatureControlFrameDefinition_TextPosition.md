# ModelFeatureControlFrameDefinition.TextPosition Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Gets and sets the position of the frame in the part or assembly. The point is projected onto the orientation plane first and then projected onto the landing line to locate the text.

## Syntax

ModelFeatureControlFrameDefinition.**TextPosition**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
