# DrawingCurve.LineWeight Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Gets and sets the line weight override for the curve.

## Syntax

DrawingCurve.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
