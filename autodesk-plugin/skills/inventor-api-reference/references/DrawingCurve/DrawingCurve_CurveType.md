# DrawingCurve.CurveType Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the type of the underlying curve geometry that this curve is defined by.

## Syntax

DrawingCurve.**CurveType**() As [CurveTypeEnum](../CurveTypeEnum.md)

## Property Value

This is a read only property whose value is a [CurveTypeEnum](../CurveTypeEnum.md).
