# DrawingCurve.CenterPoint Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the center point of the curve in sheet space. This property returns the center for arcs and circles and Nothing if the curve does not have a center.

## Syntax

DrawingCurve.**CenterPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).
