# DrawingCurve.MidPoint Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the mid point of the curve in sheet space. This property returns the mid point for linear curves and arcs.

## Syntax

DrawingCurve.**MidPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).
