# ModelParameter.Comment Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Gets the user comments against this parameter.

## Syntax

ModelParameter.**Comment**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 4
