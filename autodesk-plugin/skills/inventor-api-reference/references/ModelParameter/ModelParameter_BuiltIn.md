# ModelParameter.BuiltIn Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Property that returns whether this is an Inventor-generated parameter or an API-created one. Returns True if it is Inventor-generated.

## Syntax

ModelParameter.**BuiltIn**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 10
