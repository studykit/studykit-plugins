# ModelParameter.InUse Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Property that returns whether whether this parameter is currently in use (by a dimension, feature or by any other parameter).

## Syntax

ModelParameter.**InUse**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
