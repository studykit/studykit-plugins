# ModelParameter.Parent Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Property that returns the parent Document of this parameter.

## Syntax

ModelParameter.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
