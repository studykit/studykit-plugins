# ButtonDefinition.OnExecute Event

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Event that fires when the ButtonDefinition is executed.

## Syntax

ButtonDefinition.**OnExecute**( ***Context*** As [NameValueMap](../NameValueMap/NameValueMap.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Context | [NameValueMap](../NameValueMap/NameValueMap.md) | Input object that can be used to determine the context of why the event fired. |

## Version

Introduced in version 9
