# ButtonDefinition.Type Property

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ButtonDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
