# ButtonDefinition.DefinitionType Property

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Gets the ControlDefinitionType.

## Syntax

ButtonDefinition.**DefinitionType**() As [ControlDefinitionTypeEnum](../ControlDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [ControlDefinitionTypeEnum](../ControlDefinitionTypeEnum.md).

## Version

Introduced in version 9
