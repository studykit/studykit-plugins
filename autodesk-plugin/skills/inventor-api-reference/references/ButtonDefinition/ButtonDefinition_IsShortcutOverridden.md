# ButtonDefinition.IsShortcutOverridden Property

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Gets whether the default (internal) shortcut has been overridden by the user or through the API.

## Syntax

ButtonDefinition.**IsShortcutOverridden**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
