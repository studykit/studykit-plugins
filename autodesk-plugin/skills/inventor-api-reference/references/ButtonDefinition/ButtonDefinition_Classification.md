# ButtonDefinition.Classification Property

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Gets the classification.

## Syntax

ButtonDefinition.**Classification**() As [CommandTypesEnum](../CommandTypesEnum.md)

## Property Value

This is a read only property whose value is a [CommandTypesEnum](../CommandTypesEnum.md).

## Version

Introduced in version 9
