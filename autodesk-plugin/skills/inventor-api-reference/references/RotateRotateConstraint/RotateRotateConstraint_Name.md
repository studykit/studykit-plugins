# RotateRotateConstraint.Name Property

Parent Object: [RotateRotateConstraint](../RotateRotateConstraint/RotateRotateConstraint.md)

## Description

Gets/Sets the displayable name of this constraint.

## Syntax

RotateRotateConstraint.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 4
