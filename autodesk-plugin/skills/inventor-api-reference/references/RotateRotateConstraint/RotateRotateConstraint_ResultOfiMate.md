# RotateRotateConstraint.ResultOfiMate Property

Parent Object: [RotateRotateConstraint](../RotateRotateConstraint/RotateRotateConstraint.md)

## Description

Property that indicates if this iMateResult represents a composite iMate.

## Syntax

RotateRotateConstraint.**ResultOfiMate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
