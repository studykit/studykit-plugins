# SketchedSymbol.Color Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets the color for the symbol.

## Syntax

SketchedSymbol.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2010
