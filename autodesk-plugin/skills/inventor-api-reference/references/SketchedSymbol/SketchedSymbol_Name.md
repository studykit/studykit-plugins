# SketchedSymbol.Name Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets the name of the sketched symbol instance.

## Syntax

SketchedSymbol.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
