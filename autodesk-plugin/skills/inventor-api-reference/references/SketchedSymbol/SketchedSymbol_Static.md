# SketchedSymbol.Static Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets whether to show the scale and rotation grip points on the SketchedSymbol.

## Syntax

SketchedSymbol.**Static**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
