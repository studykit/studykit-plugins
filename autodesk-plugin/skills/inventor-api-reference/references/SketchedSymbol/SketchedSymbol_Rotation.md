# SketchedSymbol.Rotation Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets the rotation angle of the symbol in radians.

## Syntax

SketchedSymbol.**Rotation**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 10
