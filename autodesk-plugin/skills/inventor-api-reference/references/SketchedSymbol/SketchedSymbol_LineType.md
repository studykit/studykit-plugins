# SketchedSymbol.LineType Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets the line type override for the symbol.

## Syntax

SketchedSymbol.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2010
