# ApprenticeServerDrawingDocument.PropertySets Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Gets the object that manages the File Properties.

## Syntax

ApprenticeServerDrawingDocument.**PropertySets**() As [PropertySets](../PropertySets/PropertySets.md)

## Property Value

This is a read only property whose value is a [PropertySets](../PropertySets/PropertySets.md).

## Version

Introduced in version 4
