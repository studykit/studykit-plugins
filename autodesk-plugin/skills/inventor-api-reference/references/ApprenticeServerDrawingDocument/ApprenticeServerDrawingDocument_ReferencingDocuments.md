# ApprenticeServerDrawingDocument.ReferencingDocuments Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Property that returns all the documents in memory that reference this document.

## Syntax

ApprenticeServerDrawingDocument.**ReferencingDocuments**() As [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md)

## Property Value

This is a read only property whose value is an [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md).

## Version

Introduced in version 11
