# ModelDatumProxy.Definition Property

Parent Object: [ModelDatumProxy](../ModelDatumProxy/ModelDatumProxy.md)

## Description

Gets and sets the ModelDatumDefinition associated with this ModelDatum object.

## Syntax

ModelDatumProxy.**Definition**() As [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Property Value

This is a read/write property whose value is a [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md).

## Version

Introduced in version 2023
