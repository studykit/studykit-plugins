# RevisionTableColumn Object

## Description

This object represent a column in the revision table.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../RevisionTableColumn/RevisionTableColumn_Delete.md) | Method that deletes the RevisionTable. |
| [GetFilePropertyId](../RevisionTableColumn/RevisionTableColumn_GetFilePropertyId.md) | Method that returns the id of the property associated with the column. |
| [Reposition](../RevisionTableColumn/RevisionTableColumn_Reposition.md) | Method that repositions the column within the revision table. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevisionTableColumn/RevisionTableColumn_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [CustomPropertyName](../RevisionTableColumn/RevisionTableColumn_CustomPropertyName.md) | Read-only property that returns the name of the custom property associated with this column. |
| [Parent](../RevisionTableColumn/RevisionTableColumn_Parent.md) | Property that returns the parent RevisionTable. |
| [PropertyType](../RevisionTableColumn/RevisionTableColumn_PropertyType.md) | Read-only property that returns the property type associated with this column. |
| [Title](../RevisionTableColumn/RevisionTableColumn_Title.md) | Specifies the title of the column. |
| [TitleHorizontalJustification](../RevisionTableColumn/RevisionTableColumn_TitleHorizontalJustification.md) | Specifies the horizontal justification of the column title. |
| [Type](../RevisionTableColumn/RevisionTableColumn_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [ValueHorizontalJustification](../RevisionTableColumn/RevisionTableColumn_ValueHorizontalJustification.md) | Specifies the horizontal justification of the column values. |
| [Width](../RevisionTableColumn/RevisionTableColumn_Width.md) | Specifies the width of the column. |

## Accessed From

[RevisionTableColumns.Add](../RevisionTableColumns/RevisionTableColumns_Add.md), [RevisionTableColumns.Item](../RevisionTableColumns/RevisionTableColumns_Item.md)

## Version

Introduced in version 10
