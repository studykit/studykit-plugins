# RevisionTableColumn.PropertyType Property

Parent Object: [RevisionTableColumn](../RevisionTableColumn/RevisionTableColumn.md)

## Description

Read-only property that returns the property type associated with this column.

## Syntax

RevisionTableColumn.**PropertyType**() As [RevisionTablePropertyEnum](../RevisionTablePropertyEnum.md)

## Property Value

This is a read only property whose value is a [RevisionTablePropertyEnum](../RevisionTablePropertyEnum.md).

## Version

Introduced in version 2012
