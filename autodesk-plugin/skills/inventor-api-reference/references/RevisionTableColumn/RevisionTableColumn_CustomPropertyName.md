# RevisionTableColumn.CustomPropertyName Property

Parent Object: [RevisionTableColumn](../RevisionTableColumn/RevisionTableColumn.md)

## Description

Read-only property that returns the name of the custom property associated with this column.

## Syntax

RevisionTableColumn.**CustomPropertyName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
