# WebBrowserDialog.WebBrowserDialogEvents Property

Parent Object: [WebBrowserDialog](../WebBrowserDialog/WebBrowserDialog.md)

## Description

Read-only property that gets the WebBrowserDialogEvents object.

## Syntax

WebBrowserDialog.**WebBrowserDialogEvents**() As [WebBrowserDialogEvents](../WebBrowserDialogEvents/WebBrowserDialogEvents.md)

## Property Value

This is a read only property whose value is a [WebBrowserDialogEvents](../WebBrowserDialogEvents/WebBrowserDialogEvents.md).

## Version

Introduced in version 2017
