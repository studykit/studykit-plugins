# ViewTabGroup Object

## Description

ViewTabGroup object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Close](../ViewTabGroup/ViewTabGroup_Close.md) | Method that closes this ViewTabGroup. |
| [MoveToGroup](../ViewTabGroup/ViewTabGroup_MoveToGroup.md) | Method that moves the current ViewTabGroup to another ViewTabGroup. The ViewTabGroup the current ViewTabGroup moves into will be returned. |
| [MoveToNewViewFrame](../ViewTabGroup/ViewTabGroup_MoveToNewViewFrame.md) | Method that moves current ViewTabGroup to a new ViewFrame. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ViewTabGroup/ViewTabGroup_Application.md) | Read-only property that returns the root Application object. |
| [Count](../ViewTabGroup/ViewTabGroup_Count.md) | Read-only property that returns the number of view tabs in this view tab group. |
| [Item](../ViewTabGroup/ViewTabGroup_Item.md) | Read-only property that returns the specified ViewTab object from the collection. |
| [Type](../ViewTabGroup/ViewTabGroup_Type.md) | Gets the constant that indicates the type of this object. |
| [ViewFrame](../ViewTabGroup/ViewTabGroup_ViewFrame.md) | Read-only property that returns the ViewFrame this view tab Group is located in. |
