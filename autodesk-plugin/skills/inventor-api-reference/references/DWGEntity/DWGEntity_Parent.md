# DWGEntity.Parent Property

Parent Object: [DWGEntity](../DWGEntity/DWGEntity.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DWGEntity.**Parent**() As [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Property Value

This is a read only property whose value is an [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md).

## Version

Introduced in version 2016
