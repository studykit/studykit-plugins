# SelectSet.SelectMultiple Method

Parent Object: [SelectSet](../SelectSet/SelectSet.md)

## Description

Adds multiple entities to the select set.

## Syntax

SelectSet.**SelectMultiple**( ***Entities*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Entities | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Object collection of entities to add to the set. |

## Version

Introduced in version 2011
