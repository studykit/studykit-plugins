# DWGACMStandardPartProxy.HandleID Property

Parent Object: [DWGACMStandardPartProxy](../DWGACMStandardPartProxy/DWGACMStandardPartProxy.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGACMStandardPartProxy.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2017
