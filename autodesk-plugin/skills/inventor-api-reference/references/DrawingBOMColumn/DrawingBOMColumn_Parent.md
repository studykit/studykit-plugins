# DrawingBOMColumn.Parent Property

Parent Object: [DrawingBOMColumn](../DrawingBOMColumn/DrawingBOMColumn.md)

## Description

Property that returns the parent DrawingBOM object.

## Syntax

DrawingBOMColumn.**Parent**() As [DrawingBOM](../DrawingBOM/DrawingBOM.md)

## Property Value

This is a read only property whose value is a [DrawingBOM](../DrawingBOM/DrawingBOM.md).

## Version

Introduced in version 2009
