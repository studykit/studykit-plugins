# DrawingBOMColumn.CustomPropertyName Property

Parent Object: [DrawingBOMColumn](../DrawingBOMColumn/DrawingBOMColumn.md)

## Description

Property that returns the name of the custom property associated with this column.

## Syntax

DrawingBOMColumn.**CustomPropertyName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2009
