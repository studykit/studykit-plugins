# DrawingBOMColumn.Type Property

Parent Object: [DrawingBOMColumn](../DrawingBOMColumn/DrawingBOMColumn.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DrawingBOMColumn.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
