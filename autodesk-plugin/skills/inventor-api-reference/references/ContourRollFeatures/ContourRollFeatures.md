# ContourRollFeatures Object

## Description

The ContourRollFeatures collection object provides access to the ContourRollFeature objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ContourRollFeatures/ContourRollFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ContourRollFeatures/ContourRollFeatures_Count.md) | Property that returns the number of ContourRollFeature objects in the collection. |
| [Item](../ContourRollFeatures/ContourRollFeatures_Item.md) | Returns the specified ContourRollFeature object from the collection. |
| [Type](../ContourRollFeatures/ContourRollFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SheetMetalFeatures.ContourRollFeatures](../SheetMetalFeatures/SheetMetalFeatures_ContourRollFeatures.md)

## Version

Introduced in version 2010
