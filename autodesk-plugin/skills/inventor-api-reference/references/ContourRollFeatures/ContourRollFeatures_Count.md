# ContourRollFeatures.Count Property

Parent Object: [ContourRollFeatures](../ContourRollFeatures/ContourRollFeatures.md)

## Description

Property that returns the number of ContourRollFeature objects in the collection.

## Syntax

ContourRollFeatures.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
