# MeshFeatureEntity.ColorSet Property

Parent Object: [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Description

Read-only property that returns the color set of the mesh feature entity.

## Syntax

MeshFeatureEntity.**ColorSet**() As [GraphicsColorSet](../GraphicsColorSet/GraphicsColorSet.md)

## Property Value

This is a read only property whose value is a [GraphicsColorSet](../GraphicsColorSet/GraphicsColorSet.md).

## Version

Introduced in version 2017
