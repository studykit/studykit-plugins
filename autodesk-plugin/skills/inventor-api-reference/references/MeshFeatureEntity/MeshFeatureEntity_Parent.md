# MeshFeatureEntity.Parent Property

Parent Object: [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

MeshFeatureEntity.**Parent**() As [MeshFeature](../MeshFeature/MeshFeature.md)

## Property Value

This is a read only property whose value is a [MeshFeature](../MeshFeature/MeshFeature.md).

## Version

Introduced in version 2017
