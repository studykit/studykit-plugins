# SketchSplineHandle3D.Construction Property

Parent Object: [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchSplineHandle3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
