# MateiMateDefinitionProxy.Entity Property

Parent Object: [MateiMateDefinitionProxy](../MateiMateDefinitionProxy/MateiMateDefinitionProxy.md)

## Description

Property that returns the entity used for the iMate definition.

## Syntax

MateiMateDefinitionProxy.**Entity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
