# MateiMateDefinitionProxy.NativeObject Property

Parent Object: [MateiMateDefinitionProxy](../MateiMateDefinitionProxy/MateiMateDefinitionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MateiMateDefinitionProxy.**NativeObject**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version

Introduced in version 6
