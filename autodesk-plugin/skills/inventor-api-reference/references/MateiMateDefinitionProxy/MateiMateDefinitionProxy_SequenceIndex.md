# MateiMateDefinitionProxy.SequenceIndex Property

Parent Object: [MateiMateDefinitionProxy](../MateiMateDefinitionProxy/MateiMateDefinitionProxy.md)

## Description

Gets the index of this iMateDefinition.

## Syntax

MateiMateDefinitionProxy.**SequenceIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 6
