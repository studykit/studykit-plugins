# InteractionGraphics.GraphicsDataSets Property

Parent Object: [InteractionGraphics](../InteractionGraphics/InteractionGraphics.md)

## Description

The object that can be used with the preview and overlay graphics properties.

## Syntax

InteractionGraphics.**GraphicsDataSets**() As [GraphicsDataSets](../GraphicsDataSets/GraphicsDataSets.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSets](../GraphicsDataSets/GraphicsDataSets.md).
