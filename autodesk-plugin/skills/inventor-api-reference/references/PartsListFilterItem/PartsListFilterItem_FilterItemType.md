# PartsListFilterItem.FilterItemType Property

Parent Object: [PartsListFilterItem](../PartsListFilterItem/PartsListFilterItem.md)

## Description

Read-only property that returns the filter item type.

## Syntax

PartsListFilterItem.**FilterItemType**() As [PartsListFilterItemTypeEnum](../PartsListFilterItemTypeEnum.md)

## Property Value

This is a read only property whose value is a [PartsListFilterItemTypeEnum](../PartsListFilterItemTypeEnum.md).

## Version

Introduced in version 2024.1
