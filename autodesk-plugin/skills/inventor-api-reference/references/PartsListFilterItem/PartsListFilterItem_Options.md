# PartsListFilterItem.Options Property

Parent Object: [PartsListFilterItem](../PartsListFilterItem/PartsListFilterItem.md)

## Description

Read-write property that gets and sets options for the filter item.

## Syntax

PartsListFilterItem.**Options**() As [NameValueMap](../NameValueMap/NameValueMap.md)

## Property Value

This is a read/write property whose value is a [NameValueMap](../NameValueMap/NameValueMap.md).

## Version

Introduced in version 2024.1
