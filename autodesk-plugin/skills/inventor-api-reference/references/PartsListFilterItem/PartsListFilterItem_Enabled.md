# PartsListFilterItem.Enabled Property

Parent Object: [PartsListFilterItem](../PartsListFilterItem/PartsListFilterItem.md)

## Description

Read-write property that gets and sets whether this filter item is enabled or not. This is applicable when the filter item is from a parts list style.

## Syntax

PartsListFilterItem.**Enabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024.1
