# SculptFeature.Operation Property

Parent Object: [SculptFeature](../SculptFeature/SculptFeature.md)

## Description

Gets and sets the type of operation used to add the feature to the model.

## Syntax

SculptFeature.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2011
