# BendDefinition.IsDoubleBendFirstEdgeFixed Property

Parent Object: [BendDefinition](../BendDefinition/BendDefinition.md)

## Description

Specify which face remains fixed and which one is extended or trimmed in the case of double bends.

## Syntax

BendDefinition.**IsDoubleBendFirstEdgeFixed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
