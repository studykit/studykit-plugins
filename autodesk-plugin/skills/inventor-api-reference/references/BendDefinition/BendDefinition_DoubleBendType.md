# BendDefinition.DoubleBendType Property

Parent Object: [BendDefinition](../BendDefinition/BendDefinition.md)

## Description

Specifies if the bend is a double bend or not.

## Syntax

BendDefinition.**DoubleBendType**() As [DoubleBendTypeEnum](../DoubleBendTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DoubleBendTypeEnum](../DoubleBendTypeEnum.md).

## Version

Introduced in version 2009
