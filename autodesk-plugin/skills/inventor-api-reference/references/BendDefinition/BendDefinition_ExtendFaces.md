# BendDefinition.ExtendFaces Property

Parent Object: [BendDefinition](../BendDefinition/BendDefinition.md)

## Description

Specifies if the existing faces will be extended to the bend or if a perpendicular section should be created to connect to the bend.

## Syntax

BendDefinition.**ExtendFaces**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
