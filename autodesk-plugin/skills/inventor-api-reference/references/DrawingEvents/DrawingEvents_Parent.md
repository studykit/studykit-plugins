# DrawingEvents.Parent Property

Parent Object: [DrawingEvents](../DrawingEvents/DrawingEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DrawingEvents.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 9
