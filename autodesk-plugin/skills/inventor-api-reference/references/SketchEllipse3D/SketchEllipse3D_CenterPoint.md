# SketchEllipse3D.CenterPoint Property

Parent Object: [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Description

Property that returns the point that defines the position of the center of the ellipse.

## Syntax

SketchEllipse3D.**CenterPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
