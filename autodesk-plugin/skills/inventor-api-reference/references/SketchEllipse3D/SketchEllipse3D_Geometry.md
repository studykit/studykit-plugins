# SketchEllipse3D.Geometry Property

Parent Object: [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Description

Gets and sets an EllipseFull geometry object. The object returned represents a 'snapshot' view of the current state of the sketch ellipse.

## Syntax

SketchEllipse3D.**Geometry**() As [EllipseFull](../EllipseFull/EllipseFull.md)

## Property Value

This is a read/write property whose value is an [EllipseFull](../EllipseFull/EllipseFull.md).

## Version

Introduced in version 6
