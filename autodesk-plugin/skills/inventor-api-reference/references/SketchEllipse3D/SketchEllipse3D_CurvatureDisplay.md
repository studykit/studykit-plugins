# SketchEllipse3D.CurvatureDisplay Property

Parent Object: [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Description

Specifies whether curvature information is displayed for the ellipse.

## Syntax

SketchEllipse3D.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
