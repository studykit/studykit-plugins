# RevisionCloudDefinition.ControlPoints Property

Parent Object: [RevisionCloudDefinition](../RevisionCloudDefinition/RevisionCloudDefinition.md)

## Description

Returns the RevisionCloudControlPoints object that handles control points of the revision cloud.

## Syntax

RevisionCloudDefinition.**ControlPoints**() As [RevisionCloudControlPoints](../RevisionCloudControlPoints/RevisionCloudControlPoints.md)

## Property Value

This is a read only property whose value is a [RevisionCloudControlPoints](../RevisionCloudControlPoints/RevisionCloudControlPoints.md).

## Version

Introduced in version 2024
