# RevisionCloudDefinition.Parent Property

Parent Object: [RevisionCloudDefinition](../RevisionCloudDefinition/RevisionCloudDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

RevisionCloudDefinition.**Parent**() As [RevisionCloud](../RevisionCloud/RevisionCloud.md)

## Property Value

This is a read only property whose value is a [RevisionCloud](../RevisionCloud/RevisionCloud.md).

## Version

Introduced in version 2024
