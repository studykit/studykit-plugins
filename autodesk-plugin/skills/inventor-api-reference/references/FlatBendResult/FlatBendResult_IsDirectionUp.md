# FlatBendResult.IsDirectionUp Property

Parent Object: [FlatBendResult](../FlatBendResult/FlatBendResult.md)

## Description

Property that returns whether the direction of the bend is up or down.

## Syntax

FlatBendResult.**IsDirectionUp**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
