# FlatBendResult.Edge Property

Parent Object: [FlatBendResult](../FlatBendResult/FlatBendResult.md)

## Description

Property that returns the edge in the flat pattern that belongs to the bend result.

## Syntax

FlatBendResult.**Edge**() As [Edge](../Edge/Edge.md)

## Property Value

This is a read only property whose value is an [Edge](../Edge/Edge.md).

## Version

Introduced in version 2008
