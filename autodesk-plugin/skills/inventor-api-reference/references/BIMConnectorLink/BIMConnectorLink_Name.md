# BIMConnectorLink.Name Property

Parent Object: [BIMConnectorLink](../BIMConnectorLink/BIMConnectorLink.md)

## Description

Read-write property that gets and sets the displayed name of the connectorlink. This is the name that is visible in the browser and is editable by the end-user.

## Syntax

BIMConnectorLink.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
