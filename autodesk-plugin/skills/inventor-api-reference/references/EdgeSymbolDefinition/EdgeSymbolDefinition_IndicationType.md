# EdgeSymbolDefinition.IndicationType Property

Parent Object: [EdgeSymbolDefinition](../EdgeSymbolDefinition/EdgeSymbolDefinition.md)

## Description

Read-write property that gets and sets the indication type of the edge symbol.

## Syntax

EdgeSymbolDefinition.**IndicationType**() As [EdgeSymbolIndicationTypeEnum](../EdgeSymbolIndicationTypeEnum.md)

## Property Value

This is a read/write property whose value is an [EdgeSymbolIndicationTypeEnum](../EdgeSymbolIndicationTypeEnum.md).

## Version

Introduced in version 2024
