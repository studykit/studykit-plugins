# EdgeSymbolDefinition.Layer Property

Parent Object: [EdgeSymbolDefinition](../EdgeSymbolDefinition/EdgeSymbolDefinition.md)

## Description

Read-write property that gets and sets the layer associated with this object.

## Syntax

EdgeSymbolDefinition.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2024
