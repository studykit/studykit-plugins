# ParallelToXZPlaneConstraint3DProxy.NativeObject Property

Parent Object: [ParallelToXZPlaneConstraint3DProxy](../ParallelToXZPlaneConstraint3DProxy/ParallelToXZPlaneConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ParallelToXZPlaneConstraint3DProxy.**NativeObject**() As [ParallelToXZPlaneConstraint3D](../ParallelToXZPlaneConstraint3D/ParallelToXZPlaneConstraint3D.md)

## Property Value

This is a read only property whose value is a [ParallelToXZPlaneConstraint3D](../ParallelToXZPlaneConstraint3D/ParallelToXZPlaneConstraint3D.md).

## Version

Introduced in version 2017
