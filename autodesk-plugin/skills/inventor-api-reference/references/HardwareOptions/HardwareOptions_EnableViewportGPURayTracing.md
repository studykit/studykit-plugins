# HardwareOptions.EnableViewportGPURayTracing Property

Parent Object: [HardwareOptions](../HardwareOptions/HardwareOptions.md)

## Description

Read-write property that specifies whether to use GPU Raytracing.

## Syntax

HardwareOptions.**EnableViewportGPURayTracing**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
