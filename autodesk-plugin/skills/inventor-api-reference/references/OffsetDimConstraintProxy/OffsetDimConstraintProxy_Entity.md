# OffsetDimConstraintProxy.Entity Property

Parent Object: [OffsetDimConstraintProxy](../OffsetDimConstraintProxy/OffsetDimConstraintProxy.md)

## Description

Property that returns the circle or arc being constrained.

## Syntax

OffsetDimConstraintProxy.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
