# OffsetDimConstraintProxy.Line Property

Parent Object: [OffsetDimConstraintProxy](../OffsetDimConstraintProxy/OffsetDimConstraintProxy.md)

## Description

Property that returns the proxy SketchLine being constrained.

## Syntax

OffsetDimConstraintProxy.**Line**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 6
