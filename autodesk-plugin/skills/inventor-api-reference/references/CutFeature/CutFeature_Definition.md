# CutFeature.Definition Property

Parent Object: [CutFeature](../CutFeature/CutFeature.md)

## Description

Property that gets/sets the CutDefinition object associated with this cut feature.

## Syntax

CutFeature.**Definition**() As [CutDefinition](../CutDefinition/CutDefinition.md)

## Property Value

This is a read only property whose value is a [CutDefinition](../CutDefinition/CutDefinition.md).

## Version

Introduced in version 2009
