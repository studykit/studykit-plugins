# Circle.Normal Property

Parent Object: [Circle](../Circle/Circle.md)

## Description

The normal of the circle.

## Syntax

Circle.**Normal**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
