# AngleConstraintProxy.Angle Property

Parent Object: [AngleConstraintProxy](../AngleConstraintProxy/AngleConstraintProxy.md)

## Description

Property that returns the object that controls the angle of the constraint.

## Syntax

AngleConstraintProxy.**Angle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 11
