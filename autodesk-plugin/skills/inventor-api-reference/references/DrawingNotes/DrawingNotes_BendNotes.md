# DrawingNotes.BendNotes Property

Parent Object: [DrawingNotes](../DrawingNotes/DrawingNotes.md)

## Description

Property that returns the collection of bend notes on the sheet.

## Syntax

DrawingNotes.**BendNotes**() As [BendNotes](../BendNotes/BendNotes.md)

## Property Value

This is a read only property whose value is a [BendNotes](../BendNotes/BendNotes.md).
