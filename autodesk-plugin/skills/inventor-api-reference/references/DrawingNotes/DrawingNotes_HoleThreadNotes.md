# DrawingNotes.HoleThreadNotes Property

Parent Object: [DrawingNotes](../DrawingNotes/DrawingNotes.md)

## Description

Property that returns the collection of hole and thread notes on the sheet.

## Syntax

DrawingNotes.**HoleThreadNotes**() As [HoleThreadNotes](../HoleThreadNotes/HoleThreadNotes.md)

## Property Value

This is a read only property whose value is a [HoleThreadNotes](../HoleThreadNotes/HoleThreadNotes.md).
