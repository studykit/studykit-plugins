# DrawingNotes.LeaderNotes Property

Parent Object: [DrawingNotes](../DrawingNotes/DrawingNotes.md)

## Description

Property that returns the collection of leader notes on the sheet.

## Syntax

DrawingNotes.**LeaderNotes**() As [LeaderNotes](../LeaderNotes/LeaderNotes.md)

## Property Value

This is a read only property whose value is a [LeaderNotes](../LeaderNotes/LeaderNotes.md).
