# DrawingNotes.GeneralNotes Property

Parent Object: [DrawingNotes](../DrawingNotes/DrawingNotes.md)

## Description

Property that returns the collection of general notes on the sheet.

## Syntax

DrawingNotes.**GeneralNotes**() As [GeneralNotes](../GeneralNotes/GeneralNotes.md)

## Property Value

This is a read only property whose value is a [GeneralNotes](../GeneralNotes/GeneralNotes.md).
