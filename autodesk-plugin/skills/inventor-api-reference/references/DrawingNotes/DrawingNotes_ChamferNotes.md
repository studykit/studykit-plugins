# DrawingNotes.ChamferNotes Property

Parent Object: [DrawingNotes](../DrawingNotes/DrawingNotes.md)

## Description

Property that returns the collection of chamfer notes on the sheet.

## Syntax

DrawingNotes.**ChamferNotes**() As [ChamferNotes](../ChamferNotes/ChamferNotes.md)

## Property Value

This is a read only property whose value is a [ChamferNotes](../ChamferNotes/ChamferNotes.md).

## Version

Introduced in version 2010
