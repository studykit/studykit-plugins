# DrawingNotes.PunchNotes Property

Parent Object: [DrawingNotes](../DrawingNotes/DrawingNotes.md)

## Description

Property that returns the collection of punch notes on the sheet.

## Syntax

DrawingNotes.**PunchNotes**() As [PunchNotes](../PunchNotes/PunchNotes.md)

## Property Value

This is a read only property whose value is a [PunchNotes](../PunchNotes/PunchNotes.md).
