# DrawingNotes.Type Property

Parent Object: [DrawingNotes](../DrawingNotes/DrawingNotes.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DrawingNotes.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
