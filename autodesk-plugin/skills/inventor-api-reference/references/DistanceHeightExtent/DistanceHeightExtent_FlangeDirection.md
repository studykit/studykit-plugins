# DistanceHeightExtent.FlangeDirection Property

Parent Object: [DistanceHeightExtent](../DistanceHeightExtent/DistanceHeightExtent.md)

## Description

Gets and sets the direction of the flange.

## Syntax

DistanceHeightExtent.**FlangeDirection**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2009
