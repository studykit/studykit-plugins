# DistanceHeightExtent.HeightDatumType Property

Parent Object: [DistanceHeightExtent](../DistanceHeightExtent/DistanceHeightExtent.md)

## Description

Gets and set the how the height of the flange is defined when the height extent is a distance extent.

## Syntax

DistanceHeightExtent.**HeightDatumType**() As [HeightDatumTypeEnum](../HeightDatumTypeEnum.md)

## Property Value

This is a read/write property whose value is a [HeightDatumTypeEnum](../HeightDatumTypeEnum.md).

## Version

Introduced in version 2009
