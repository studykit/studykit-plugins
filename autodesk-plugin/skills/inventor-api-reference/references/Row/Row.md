# Row Object

## Description

The Row object represents a single row in this custom table. See the Custom Tables overview for more information.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../Row/Row_Delete.md) | Method that deletes this Row object. |
| [Reposition](../Row/Row_Reposition.md) | Method that repositions the row within the table. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Row/Row_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Row/Row_Count.md) | Property that returns the number of items in the collection. |
| [Height](../Row/Row_Height.md) | Specifies the height of the row. |
| [Item](../Row/Row_Item.md) | Returns the specified Cell object from the collection. |
| [Parent](../Row/Row_Parent.md) | Property that returns the parent Table. |
| [Type](../Row/Row_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Visible](../Row/Row_Visible.md) | Gets and sets whether the row is visible. |

## Accessed From

[Rows.Add](../Rows/Rows_Add.md), [Rows.Item](../Rows/Rows_Item.md)

## Version

Introduced in version 9
