# ChamferNote.Retrieved Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Indicates whether the drawing note was created as a result of retrieving it from the model.

## Syntax

ChamferNote.**Retrieved**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025
