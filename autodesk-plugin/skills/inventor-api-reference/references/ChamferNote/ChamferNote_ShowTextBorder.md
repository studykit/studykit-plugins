# ChamferNote.ShowTextBorder Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Gets/Sets whether to show the text border or not.

## Syntax

ChamferNote.**ShowTextBorder**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
