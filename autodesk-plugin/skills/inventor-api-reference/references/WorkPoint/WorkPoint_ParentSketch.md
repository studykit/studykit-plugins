# WorkPoint.ParentSketch Property

Parent Object: [WorkPoint](../WorkPoint/WorkPoint.md)

## Description

Property that returns the parent ComponentDefinition object.

## Syntax

WorkPoint.**ParentSketch**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2008
