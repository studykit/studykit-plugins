# WorkPoint.Dependents Property

Parent Object: [WorkPoint](../WorkPoint/WorkPoint.md)

## Description

Property that returns the collection of objects that have a direct dependency on the work point.

## Syntax

WorkPoint.**Dependents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
