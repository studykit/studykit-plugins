# WorkPoint.OwnedBy Property

Parent Object: [WorkPoint](../WorkPoint/WorkPoint.md)

## Description

Read-only property that returns the client feature that owns this object. This property returns Nothing if the IsOwnedByFeature property returns False.

## Syntax

WorkPoint.**OwnedBy**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 2008
