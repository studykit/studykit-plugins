# WorkPoint.IsPatternElement Property

Parent Object: [WorkPoint](../WorkPoint/WorkPoint.md)

## Description

Property that gets whether the work point was created by a pattern. If so, edits and delete are not allowed.

## Syntax

WorkPoint.**IsPatternElement**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
