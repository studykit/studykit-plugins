# ViewList.Remove Method

Parent Object: [ViewList](../ViewList/ViewList.md)

## Description

Method that removes a View object from the collection.

## Syntax

ViewList.**Remove**( ***View*** As [View](../View/View.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| View | [View](../View/View.md) | View object to remove. |

## Version

Introduced in version 9
