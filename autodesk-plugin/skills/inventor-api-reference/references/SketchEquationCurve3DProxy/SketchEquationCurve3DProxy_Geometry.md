# SketchEquationCurve3DProxy.Geometry Property

Parent Object: [SketchEquationCurve3DProxy](../SketchEquationCurve3DProxy/SketchEquationCurve3DProxy.md)

## Description

Read-only property that gets a BSplineCurve2d geometry object. The object returned represents a 'snapshot' view of the current state of the spline.

## Syntax

SketchEquationCurve3DProxy.**Geometry**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read only property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 2014
