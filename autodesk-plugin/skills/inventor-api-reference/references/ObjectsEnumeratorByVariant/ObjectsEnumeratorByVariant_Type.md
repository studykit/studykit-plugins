# ObjectsEnumeratorByVariant.Type Property

Parent Object: [ObjectsEnumeratorByVariant](../ObjectsEnumeratorByVariant/ObjectsEnumeratorByVariant.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ObjectsEnumeratorByVariant.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
