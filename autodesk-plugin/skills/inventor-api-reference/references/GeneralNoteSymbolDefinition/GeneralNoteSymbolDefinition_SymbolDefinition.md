# GeneralNoteSymbolDefinition.SymbolDefinition Property

Parent Object: [GeneralNoteSymbolDefinition](../GeneralNoteSymbolDefinition/GeneralNoteSymbolDefinition.md)

## Description

Get the height of the text within the GeneralNote.

## Syntax

GeneralNoteSymbolDefinition.**SymbolDefinition**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2018
