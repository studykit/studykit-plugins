# ModelCompositeAnnotationDefinition.AddAnnotation Method

Parent Object: [ModelCompositeAnnotationDefinition](../ModelCompositeAnnotationDefinition/ModelCompositeAnnotationDefinition.md)

## Description

Adds a new member to the model composite annotation.

## Syntax

ModelCompositeAnnotationDefinition.**AddAnnotation**( ***ModelAnnotation*** As [ModelAnnotation](../ModelAnnotation/ModelAnnotation.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ModelAnnotation | [ModelAnnotation](../ModelAnnotation/ModelAnnotation.md) | Input ModelAnnotation object that specifies the ModelAnnotation to be added to the ModelCompositeAnnotation. A valid ModelAnnotation to be added to this ModelCompositeAnnotation should be directly or indirectly attached to the base annotation of this ModelCompositeAnnotation. |

## Version

Introduced in version 2018
