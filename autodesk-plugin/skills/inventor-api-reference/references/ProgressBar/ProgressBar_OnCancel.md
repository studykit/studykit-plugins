# ProgressBar.OnCancel Event

Parent Object: [ProgressBar](../ProgressBar/ProgressBar.md)

## Description

Event that is fired when the user chooses the cancel button on the progress bar.

## Syntax

ProgressBar.**OnCancel**()

## Version

Introduced in version 2010
