# TranslatorAddIn.SupportsSaveCopyAsFrom Property

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Property that returns semicolon-separated Inventor file types that this translator supports saving from. Example: ".iam;.ipt."

## Syntax

TranslatorAddIn.**SupportsSaveCopyAsFrom**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2011
