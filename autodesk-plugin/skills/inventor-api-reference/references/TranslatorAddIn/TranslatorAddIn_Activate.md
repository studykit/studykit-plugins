# TranslatorAddIn.Activate Method

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Creates and initializes the AddIn. No effect if AddIn already active.

## Syntax

TranslatorAddIn.**Activate**()
