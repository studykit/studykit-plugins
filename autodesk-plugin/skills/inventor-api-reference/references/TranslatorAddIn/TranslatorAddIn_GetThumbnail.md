# TranslatorAddIn.GetThumbnail Method

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Obtains the thumbnail, if any, for the given data-source. Could be a metafile handle (long) or the interface to a StdPicture object.

## Syntax

TranslatorAddIn.**GetThumbnail**( ***SourceData*** As [DataMedium](../DataMedium/DataMedium.md) ) As Variant

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SourceData | [DataMedium](../DataMedium/DataMedium.md) | Input DataMedium object that specifies the data source. |

## Version

Introduced in version 5
