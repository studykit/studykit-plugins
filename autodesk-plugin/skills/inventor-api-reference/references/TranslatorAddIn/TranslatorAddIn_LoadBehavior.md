# TranslatorAddIn.LoadBehavior Property

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Gets a constant indicating the load behavior (load time) of the add-in. This applies only if the LoadAutomatically property is set to True.

## Syntax

TranslatorAddIn.**LoadBehavior**() As [AddInLoadBehaviorEnum](../AddInLoadBehaviorEnum.md)

## Property Value

This is a read only property whose value is an [AddInLoadBehaviorEnum](../AddInLoadBehaviorEnum.md).

## Version

Introduced in version 2012
