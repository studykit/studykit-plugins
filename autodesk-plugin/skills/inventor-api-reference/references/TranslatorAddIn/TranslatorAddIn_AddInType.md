# TranslatorAddIn.AddInType Property

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Gets the constant that indicates the type of this AddIn.

## Syntax

TranslatorAddIn.**AddInType**() As [ApplicationAddInTypeEnum](../ApplicationAddInTypeEnum.md)

## Property Value

This is a read only property whose value is an [ApplicationAddInTypeEnum](../ApplicationAddInTypeEnum.md).

## Version

Introduced in version 4
