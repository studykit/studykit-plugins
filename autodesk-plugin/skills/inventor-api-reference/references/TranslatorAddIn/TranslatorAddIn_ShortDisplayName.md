# TranslatorAddIn.ShortDisplayName Property

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Property that returns the short display name of the Add-in. Used in places to succinctly identify the AddIn inside Inventor's UI.

## Syntax

TranslatorAddIn.**ShortDisplayName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
