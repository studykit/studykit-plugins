# TranslatorAddIn.FileExtensions Property

Parent Object: [TranslatorAddIn](../TranslatorAddIn/TranslatorAddIn.md)

## Description

Property that gets the semicolon-separated list of the native file extensions that this translator reads from and/or writes to.

## Syntax

TranslatorAddIn.**FileExtensions**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
