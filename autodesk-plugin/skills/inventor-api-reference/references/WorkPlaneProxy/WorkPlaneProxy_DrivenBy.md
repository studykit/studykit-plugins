# WorkPlaneProxy.DrivenBy Property

Parent Object: [WorkPlaneProxy](../WorkPlaneProxy/WorkPlaneProxy.md)

## Description

Property that returns the collection of objects that the work plane is dependent on.

## Syntax

WorkPlaneProxy.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 5
