# WorkPlaneProxy.SetByTorusMidPlane Method

Parent Object: [WorkPlaneProxy](../WorkPlaneProxy/WorkPlaneProxy.md)

## Description

Method that redefines the work plane to be at the mid-plane of the torus specified by the input face.

## Syntax

WorkPlaneProxy.**SetByTorusMidPlane**( ***Face*** As [Face](../Face/Face.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Face | [Face](../Face/Face.md) | Input Face object that represents a torus surface. |

## Version

Introduced in version 2008
