# AngleConstraint.SolutionType Property

Parent Object: [AngleConstraint](../AngleConstraint/AngleConstraint.md)

## Description

Property that returns the solution type.

## Syntax

AngleConstraint.**SolutionType**() As [AngleConstraintSolutionTypeEnum](../AngleConstraintSolutionTypeEnum.md)

## Property Value

This is a read only property whose value is an [AngleConstraintSolutionTypeEnum](../AngleConstraintSolutionTypeEnum.md).

## Version

Introduced in version 2009
