# DSJoint Object

## Description

DSJoint object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DSJoint/DSJoint_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Definition](../DSJoint/DSJoint_Definition.md) | Gets the definition object associated with this joint. |
| [IsSuppressed](../DSJoint/DSJoint_IsSuppressed.md) | Gets whether the joint is suppressed or not. A value of True indicates the joint is suppressed. |
| [JointType](../DSJoint/DSJoint_JointType.md) | Returns the type of simulation joint this object represents |
| [Name](../DSJoint/DSJoint_Name.md) | Gets and sets the displayed name of the joint. This is the name that is visible in the browser and is editable by the end-user. |
| [Parent](../DSJoint/DSJoint_Parent.md) | Gets the parent DynamicSimulation object. |
| [Type](../DSJoint/DSJoint_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DSJointDefinition.Parent](../DSJointDefinition/DSJointDefinition_Parent.md), [DSJoints.Item](../DSJoints/DSJoints_Item.md)

## Version

Introduced in version 2013
