# InventorVBAComponent Object

## Description

The InventorVBAComponent object represents a code, class or form module.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [InventorVBAMembers](../InventorVBAComponent/InventorVBAComponent_InventorVBAMembers.md) | Property that returns the InventorVBAMembers collection. |
| [Name](../InventorVBAComponent/InventorVBAComponent_Name.md) | Gets/Sets the components display name. |
| [Parent](../InventorVBAComponent/InventorVBAComponent_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../InventorVBAComponent/InventorVBAComponent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [VBComponent](../InventorVBAComponent/InventorVBAComponent_VBComponent.md) | Property that returns the VBComponent object corresponding to this InventorVBAComponent. |
