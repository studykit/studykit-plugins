# InventorVBAComponent.Parent Property

Parent Object: [InventorVBAComponent](../InventorVBAComponent/InventorVBAComponent.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

InventorVBAComponent.**Parent**() As [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Property Value

This is a read only property whose value is an [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md).

## Version

Introduced in version 6
