# ThreePointAngleDimConstraintProxy.DimensionCenterPoint Property

Parent Object: [ThreePointAngleDimConstraintProxy](../ThreePointAngleDimConstraintProxy/ThreePointAngleDimConstraintProxy.md)

## Description

Gets the center of the dimension line.

## Syntax

ThreePointAngleDimConstraintProxy.**DimensionCenterPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
