# PresentationComponentsEnumerator.Item Property

Parent Object: [PresentationComponentsEnumerator](../PresentationComponentsEnumerator/PresentationComponentsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationComponentsEnumerator.**Item**( ***Index*** As Long ) As [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Property Value

This is a read only property whose value is a [PresentationComponent](../PresentationComponent/PresentationComponent.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long to specify the index of the object to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
