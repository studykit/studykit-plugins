# ShellFeatures.Item Property

Parent Object: [ShellFeatures](../ShellFeatures/ShellFeatures.md)

## Description

Returns the specified object in the collection. The index can be numeric or the object name.

## Syntax

ShellFeatures.**Item**( ***Index*** As Variant ) As [ShellFeature](../ShellFeature/ShellFeature.md)

## Property Value

This is a read only property whose value is a [ShellFeature](../ShellFeature/ShellFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the ShellFeature to return. |

## Version

Introduced in version 5
