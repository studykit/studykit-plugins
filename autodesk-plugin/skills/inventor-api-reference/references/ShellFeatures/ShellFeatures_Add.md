# ShellFeatures.Add Method

Parent Object: [ShellFeatures](../ShellFeatures/ShellFeatures.md)

## Description

Method that creates a new ShellFeature. The new created ShellFeature is returned.

## Syntax

ShellFeatures.**Add**( ***Definition*** As [ShellDefinition](../ShellDefinition/ShellDefinition.md) ) As [ShellFeature](../ShellFeature/ShellFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ShellDefinition](../ShellDefinition/ShellDefinition.md) | Property that returns the object which defines the various input that was used to create the shell feature. |

## Version

Introduced in version 9
