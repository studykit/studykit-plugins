# PitchAndRevolutionCoilExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The PitchAndRevolutionExtent object provides access to the information that defines the variables for a coil feature created using pitch and revolution values.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PitchAndRevolutionCoilExtent/PitchAndRevolutionCoilExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../PitchAndRevolutionCoilExtent/PitchAndRevolutionCoilExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Pitch](../PitchAndRevolutionCoilExtent/PitchAndRevolutionCoilExtent_Pitch.md) | Property that returns the parameter that controls the pitch of the coil feature. |
| [Revolution](../PitchAndRevolutionCoilExtent/PitchAndRevolutionCoilExtent_Revolution.md) | Property that returns the parameter that controls the number of revolutions of the coil feature. |
| [TaperAngle](../PitchAndRevolutionCoilExtent/PitchAndRevolutionCoilExtent_TaperAngle.md) | Property that returns the parameter that controls the taper angle of the coil feature. |
| [Type](../PitchAndRevolutionCoilExtent/PitchAndRevolutionCoilExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 5.3
