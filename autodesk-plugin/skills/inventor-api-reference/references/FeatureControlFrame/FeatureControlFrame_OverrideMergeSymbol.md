# FeatureControlFrame.OverrideMergeSymbol Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Gets and sets the merge symbol override value. Set this property will set the MergeSymbolOverridden to True.

## Syntax

FeatureControlFrame.**OverrideMergeSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
