# FeatureControlFrame.Style Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Gets and sets the associated FeatureControlFrameStyle object.

## Syntax

FeatureControlFrame.**Style**() As [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Property Value

This is a read/write property whose value is a [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md).

## Version

Introduced in version 2009
