# FeatureControlFrame.DatumIdentifier Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Gets and sets a datum identifier for the feature control frame. Setting this property fails if the associated style does not allow the specification of a datum identifier.

## Syntax

FeatureControlFrame.**DatumIdentifier**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
