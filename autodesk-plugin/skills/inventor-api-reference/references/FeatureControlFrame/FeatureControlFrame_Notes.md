# FeatureControlFrame.Notes Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Gets and sets notes for the feature control frame. The string can contain symbols specified using the StyleOverride tag. For instance, use 'm' to specify (M).

## Syntax

FeatureControlFrame.**Notes**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
