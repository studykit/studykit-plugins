# FeatureControlFrame.Leader Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Property that returns the leader associated with the feature control frame symbol.

## Syntax

FeatureControlFrame.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 2009
