# DocumentDescriptorsEnumerator Object

## Description

The DocumentDescriptorsEnumerator object provides information about native document references held.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator_Item.md) | Returns the specified DocumentDescriptor object from the collection. |
| [Type](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
