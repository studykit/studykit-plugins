# ShrinkwrapComponentProxy.BreakLinkToFile Method

Parent Object: [ShrinkwrapComponentProxy](../ShrinkwrapComponentProxy/ShrinkwrapComponentProxy.md)

## Description

Method that breaks the connection of the derived component with the part or assembly from which it was created. When the link is broken the ReferencedFile property will return Nothing.

## Syntax

ShrinkwrapComponentProxy.**BreakLinkToFile**()

## Version

Introduced in version 2019
