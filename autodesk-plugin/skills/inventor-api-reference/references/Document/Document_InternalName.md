# Document.InternalName Property

Parent Object: [Document](../Document/Document.md)

## Description

Gets the Internal Name (a GUID) for this document.

## Syntax

Document.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
