# Document.VBAProject Property

Parent Object: [Document](../Document/Document.md)

## Description

Property that returns the Autodesk Inventor VBA project for this document.

## Syntax

Document.**VBAProject**() As [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Property Value

This is a read only property whose value is an [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md).

## Version

Introduced in version 6
