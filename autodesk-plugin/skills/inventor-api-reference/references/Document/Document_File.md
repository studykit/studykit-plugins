# Document.File Property

Parent Object: [Document](../Document/Document.md)

## Description

Property that returns the file (storage) on disk that contains this document.

## Syntax

Document.**File**() As [File](../File/File.md)

## Property Value

This is a read only property whose value is a [File](../File/File.md).

## Version

Introduced in version 11
