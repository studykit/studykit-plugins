# Document.ActivatedObject Property

Parent Object: [Document](../Document/Document.md)

## Description

Gets the object that has been in-place activated for edit within the context of this document.

## Syntax

Document.**ActivatedObject**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
