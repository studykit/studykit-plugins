# Document.HasPrivateStream Method

Parent Object: [Document](../Document/Document.md)

## Description

Obtains a Boolean flag indicating if the specified stream exists within this document.

## Syntax

Document.**HasPrivateStream**( ***StreamName*** As String ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| StreamName | String | Input String value that specifies the name of the stream. |

## Version

Introduced in version 4
