# Document.RevertReservedForWriteByMe Method

Parent Object: [Document](../Document/Document.md)

## Description

Method that reverts the file checked out by the caller.

## Syntax

Document.**RevertReservedForWriteByMe**()

## Version

Introduced in version 6
