# DWGEntitySplineSegmentProxy.NativeObject Property

Parent Object: [DWGEntitySplineSegmentProxy](../DWGEntitySplineSegmentProxy/DWGEntitySplineSegmentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGEntitySplineSegmentProxy.**NativeObject**() As [DWGEntitySplineSegment](../DWGEntitySplineSegment/DWGEntitySplineSegment.md)

## Property Value

This is a read only property whose value is a [DWGEntitySplineSegment](../DWGEntitySplineSegment/DWGEntitySplineSegment.md).

## Version

Introduced in version 2018
