# RevitExportDefinition.PreservedFeatures Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the features to preserve from remove. For each preserved feature a Face from the feature is required to indicate it.

## Syntax

RevitExportDefinition.**PreservedFeatures**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2022
