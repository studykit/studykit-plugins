# RevitExportDefinition.RemovePartsSize Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the bounding box diagonal length in centimeters of the parts being removed.

## Syntax

RevitExportDefinition.**RemovePartsSize**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
