# RevitExportDefinition.RemoveEmbossMaxHeightRange Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the max height of embosses in centimeters to remove for the RevitExport. This is only applicable when the RemoveEmbossesStyle is set to kRevitExportRemoveByRange.

## Syntax

RevitExportDefinition.**RemoveEmbossMaxHeightRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
