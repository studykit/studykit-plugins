# RevitExportDefinition.RevitVersion Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the Revit version. Use the FileManager.GetRevitEngineInstallationStatus to get the Revit versions that can be used to create RevitExport objects.

## Syntax

RevitExportDefinition.**RevitVersion**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025
