# RevitExportDefinition.ActivePositionalRepresentation Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the name of the active Positional Representation for the RevitExport.

## Syntax

RevitExportDefinition.**ActivePositionalRepresentation**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
