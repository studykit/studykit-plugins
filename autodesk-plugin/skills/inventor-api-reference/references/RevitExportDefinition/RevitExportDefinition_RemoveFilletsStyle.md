# RevitExportDefinition.RemoveFilletsStyle Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the style to remove fillets for the RevitExport.

## Syntax

RevitExportDefinition.**RemoveFilletsStyle**() As [SimplificationRemoveStyleEnum](../SimplificationRemoveStyleEnum.md)

## Property Value

This is a read/write property whose value is a [SimplificationRemoveStyleEnum](../SimplificationRemoveStyleEnum.md).

## Version

Introduced in version 2022
