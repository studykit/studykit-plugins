# RevitExportDefinition.RemoveFilletsRadiusRange Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the radius of fillets in centimeters to remove for the RevitExport. This is only applicable when the RemoveFilletsStyle is set to kRevitExportRemoveByRange.

## Syntax

RevitExportDefinition.**RemoveFilletsRadiusRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
