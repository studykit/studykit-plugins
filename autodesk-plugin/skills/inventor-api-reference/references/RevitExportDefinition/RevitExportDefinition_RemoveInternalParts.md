# RevitExportDefinition.RemoveInternalParts Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets whether to remove internal parts or not.

## Syntax

RevitExportDefinition.**RemoveInternalParts**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
