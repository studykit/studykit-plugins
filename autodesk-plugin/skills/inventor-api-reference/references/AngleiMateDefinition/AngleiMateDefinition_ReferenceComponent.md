# AngleiMateDefinition.ReferenceComponent Property

Parent Object: [AngleiMateDefinition](../AngleiMateDefinition/AngleiMateDefinition.md)

## Description

Property that returns the ReferenceComponent that resulted in the creation of this feature.

## Syntax

AngleiMateDefinition.**ReferenceComponent**() As [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md)

## Property Value

This is a read only property whose value is a [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md).

## Version

Introduced in version 2009
