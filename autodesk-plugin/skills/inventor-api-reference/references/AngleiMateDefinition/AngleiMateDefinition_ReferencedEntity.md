# AngleiMateDefinition.ReferencedEntity Property

Parent Object: [AngleiMateDefinition](../AngleiMateDefinition/AngleiMateDefinition.md)

## Description

Gets the object this entity is dependent on.

## Syntax

AngleiMateDefinition.**ReferencedEntity**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version
