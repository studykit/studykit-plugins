# AngleiMateDefinition.DirectionOneReversed Property

Parent Object: [AngleiMateDefinition](../AngleiMateDefinition/AngleiMateDefinition.md)

## Description

Property that gets whether the direction is reversed.

## Syntax

AngleiMateDefinition.**DirectionOneReversed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
