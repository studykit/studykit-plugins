# DWGLineProxy.Direction Property

Parent Object: [DWGLineProxy](../DWGLineProxy/DWGLineProxy.md)

## Description

Read-only property that returns the UnitVector indicating the direction of the line.

## Syntax

DWGLineProxy.**Direction**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2016
