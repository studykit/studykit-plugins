# ThreadFeatureProxy.DirectionReversed Property

Parent Object: [ThreadFeatureProxy](../ThreadFeatureProxy/ThreadFeatureProxy.md)

## Description

Property that gets and sets whether the direction of the thread is in the default direction or reversed.

## Syntax

ThreadFeatureProxy.**DirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
