# ThreadFeatureProxy.NativeObject Property

Parent Object: [ThreadFeatureProxy](../ThreadFeatureProxy/ThreadFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ThreadFeatureProxy.**NativeObject**() As [ThreadFeature](../ThreadFeature/ThreadFeature.md)

## Property Value

This is a read only property whose value is a [ThreadFeature](../ThreadFeature/ThreadFeature.md).

## Version

Introduced in version 9
