# HorizontalAlignConstraint.Deletable Property

Parent Object: [HorizontalAlignConstraint](../HorizontalAlignConstraint/HorizontalAlignConstraint.md)

## Description

Indicates whether this object is deletable.

## Syntax

HorizontalAlignConstraint.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
