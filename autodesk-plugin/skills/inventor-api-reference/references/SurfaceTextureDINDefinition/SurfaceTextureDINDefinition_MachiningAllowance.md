# SurfaceTextureDINDefinition.MachiningAllowance Property

Parent Object: [SurfaceTextureDINDefinition](../SurfaceTextureDINDefinition/SurfaceTextureDINDefinition.md)

## Description

Gets and sets the machining allowance.

## Syntax

SurfaceTextureDINDefinition.**MachiningAllowance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
