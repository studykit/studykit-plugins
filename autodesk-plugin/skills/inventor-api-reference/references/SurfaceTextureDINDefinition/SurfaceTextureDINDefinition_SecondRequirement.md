# SurfaceTextureDINDefinition.SecondRequirement Property

Parent Object: [SurfaceTextureDINDefinition](../SurfaceTextureDINDefinition/SurfaceTextureDINDefinition.md)

## Description

Gets and sets the second requirement value for the surface.

## Syntax

SurfaceTextureDINDefinition.**SecondRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
