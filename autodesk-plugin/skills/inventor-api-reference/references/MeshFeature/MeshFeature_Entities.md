# MeshFeature.Entities Property

Parent Object: [MeshFeature](../MeshFeature/MeshFeature.md)

## Description

Read-only property that returns the MeshFeatureEntitiesEnumerator collection object.

## Syntax

MeshFeature.**Entities**() As [MeshFeatureEntitiesEnumerator](../MeshFeatureEntitiesEnumerator/MeshFeatureEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [MeshFeatureEntitiesEnumerator](../MeshFeatureEntitiesEnumerator/MeshFeatureEntitiesEnumerator.md).

## Version

Introduced in version 2017
