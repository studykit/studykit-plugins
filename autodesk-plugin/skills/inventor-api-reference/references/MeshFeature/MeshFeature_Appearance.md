# MeshFeature.Appearance Property

Parent Object: [MeshFeature](../MeshFeature/MeshFeature.md)

## Description

Read-write property that gets and sets the appearance of this object.

## Syntax

MeshFeature.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2017
