# MeshFeature.HealthStatus Property

Parent Object: [MeshFeature](../MeshFeature/MeshFeature.md)

## Description

Read-only property that returns an enum indicating the current state of the object.

## Syntax

MeshFeature.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2017
