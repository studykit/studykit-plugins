# ProjectPaths.Clear Method

Parent Object: [ProjectPaths](../ProjectPaths/ProjectPaths.md)

## Description

Method that deletes all the paths in the collection.

## Syntax

ProjectPaths.**Clear**()

## Version

Introduced in version 2011
