# LoftSectionDimensions Object

## Description

The LoftSectionDimensions object is used to define the placed section dimensions for an area-graph type loft.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddDrivenSection](../LoftSectionDimensions/LoftSectionDimensions_AddDrivenSection.md) | Method that creates a new LoftSectionDimension that represents a driven placed section for an area-graph type loft feature. |
| [AddDrivingSection](../LoftSectionDimensions/LoftSectionDimensions_AddDrivingSection.md) | Method that creates a new LoftSectionDimension that represents a driving placed section for an area-graph type loft feature. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LoftSectionDimensions/LoftSectionDimensions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LoftSectionDimensions/LoftSectionDimensions_Count.md) | Property that specifies the number of LoftSectionDimension objects in the collection. |
| [Item](../LoftSectionDimensions/LoftSectionDimensions_Item.md) | Returns the specified LoftSectionDimension object from the collection |
| [Type](../LoftSectionDimensions/LoftSectionDimensions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[LoftDefinition.LoftSectionDimensions](../LoftDefinition/LoftDefinition_LoftSectionDimensions.md)

## Version

Introduced in version 2008
