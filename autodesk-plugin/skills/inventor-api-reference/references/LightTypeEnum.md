# LightTypeEnum Enumerator

## Description

Coordinate space in which light direction (and position if applicable) is defined.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kGroundPlaneSpaceLight | 52995 | Light defined in Ground Plane (ViewCube) space. |
| kModelSpaceLight | 52993 | Light defined in model (world) space. |
| kViewSpaceLight | 52994 | Light defined in eye (camera) space. |

## Version

Introduced in version 10
