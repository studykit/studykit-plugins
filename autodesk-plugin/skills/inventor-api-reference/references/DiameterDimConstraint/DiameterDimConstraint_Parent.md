# DiameterDimConstraint.Parent Property

Parent Object: [DiameterDimConstraint](../DiameterDimConstraint/DiameterDimConstraint.md)

## Description

Property that returns the parent sketch of the object.

## Syntax

DiameterDimConstraint.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 5
