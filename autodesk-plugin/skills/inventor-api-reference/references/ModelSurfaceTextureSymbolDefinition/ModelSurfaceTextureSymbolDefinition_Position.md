# ModelSurfaceTextureSymbolDefinition.Position Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Gets and sets the position of the symbol on the model.

## Syntax

ModelSurfaceTextureSymbolDefinition.**Position**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
