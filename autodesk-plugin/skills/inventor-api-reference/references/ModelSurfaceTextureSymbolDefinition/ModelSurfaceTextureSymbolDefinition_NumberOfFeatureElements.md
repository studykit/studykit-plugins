# ModelSurfaceTextureSymbolDefinition.NumberOfFeatureElements Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Gets and sets the number of feature elements.

## Syntax

ModelSurfaceTextureSymbolDefinition.**NumberOfFeatureElements**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025
