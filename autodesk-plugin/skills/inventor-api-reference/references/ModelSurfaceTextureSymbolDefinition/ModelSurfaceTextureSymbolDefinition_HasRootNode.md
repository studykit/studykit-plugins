# ModelSurfaceTextureSymbolDefinition.HasRootNode Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-only property that returns if a root node exists for this leader. If False, there are no leader segments associated with this leader and the RootNode property will return Nothing.

## Syntax

ModelSurfaceTextureSymbolDefinition.**HasRootNode**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
