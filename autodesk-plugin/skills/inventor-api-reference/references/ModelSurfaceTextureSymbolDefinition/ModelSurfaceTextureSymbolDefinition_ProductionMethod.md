# ModelSurfaceTextureSymbolDefinition.ProductionMethod Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets the production method, treatment, or coating. If the active drafting standard is based on ANSI, this property can be used to specify a note callout. Denoted by B in the surface texture symbol dialog.

## Syntax

ModelSurfaceTextureSymbolDefinition.**ProductionMethod**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
