# ModelSurfaceTextureSymbolDefinition.IsMajority Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets whether this symbol specifies the standard surface characteristics for the component.

## Syntax

ModelSurfaceTextureSymbolDefinition.**IsMajority**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
