# ModelSurfaceTextureSymbolDefinition.ProfileDirection Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Gets and sets the profile direction relative to primary surface lay direction.

## Syntax

ModelSurfaceTextureSymbolDefinition.**ProfileDirection**() As [ProfileDirectionTypeEnum](../ProfileDirectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ProfileDirectionTypeEnum](../ProfileDirectionTypeEnum.md).

## Version

Introduced in version 2025
