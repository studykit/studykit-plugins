# ModelSurfaceTextureSymbolDefinition.AdditionalRoughness Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets the roughness value other than Ra or the parameter value other than Ra. For ANSI, this property can also be used to specify the waviness height. Denoted by F in the surface texture symbol dialog.

## Syntax

ModelSurfaceTextureSymbolDefinition.**AdditionalRoughness**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
