# ModelSurfaceTextureSymbolDefinition.Leader Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-only property that returns the leader associated with the leader note.

## Syntax

ModelSurfaceTextureSymbolDefinition.**Leader**() As [ModelLeader](../ModelLeader/ModelLeader.md)

## Property Value

This is a read only property whose value is a [ModelLeader](../ModelLeader/ModelLeader.md).

## Version

Introduced in version 2018
