# ModelSurfaceTextureSymbolDefinition.Faces Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Gets or sets the faces the surface texture symbol is associative with.

## Syntax

ModelSurfaceTextureSymbolDefinition.**Faces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2023
