# ModelSurfaceTextureSymbolDefinition.AnnotationPlaneDefinition Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets the annotation plane definition for the symbol.

## Syntax

ModelSurfaceTextureSymbolDefinition.**AnnotationPlaneDefinition**() As [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md).

## Version

Introduced in version 2018
