# Wires.Item Property

Parent Object: [Wires](../Wires/Wires.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

Wires.**Item**( ***Index*** As Long ) As [Wire](../Wire/Wire.md)

## Property Value

This is a read only property whose value is a [Wire](../Wire/Wire.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the Wire object to return. |

## Version

Introduced in version 2009
