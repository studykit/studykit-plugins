# Sheet.HoleTables Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the HoleTables collection object.

## Syntax

Sheet.**HoleTables**() As [HoleTables](../HoleTables/HoleTables.md)

## Property Value

This is a read only property whose value is a [HoleTables](../HoleTables/HoleTables.md).
