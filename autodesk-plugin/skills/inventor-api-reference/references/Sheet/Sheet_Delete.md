# Sheet.Delete Method

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Method that deletes this sheet.

## Syntax

Sheet.**Delete**( [***RetainDependentViews***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| RetainDependentViews | Boolean | Indicates whether to retain any views dependent on this sheet. |

## Version

Introduced in version 9
