# Sheet.GetAvailableDesignViewsToRetrieve3DAnnotations Method

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Gets an array of strings containing names of the design views of model document that the 3D annotations can be retrieved from.

## Syntax

Sheet.**GetAvailableDesignViewsToRetrieve3DAnnotations**( ***View*** As [DrawingView](../DrawingView/DrawingView.md) ) As String()

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| View | [DrawingView](../DrawingView/DrawingView.md) | Input DrawingView object from which to get the names of the model document’s design views that can be used to get the retrievable annotations. |

## Version

Introduced in version 2025
