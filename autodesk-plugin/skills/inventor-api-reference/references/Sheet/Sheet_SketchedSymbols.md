# Sheet.SketchedSymbols Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the SketchedSymbols collection object. This object provides access to all sketched symbols on a sheet.

## Syntax

Sheet.**SketchedSymbols**() As [SketchedSymbols](../SketchedSymbols/SketchedSymbols.md)

## Property Value

This is a read only property whose value is a [SketchedSymbols](../SketchedSymbols/SketchedSymbols.md).
