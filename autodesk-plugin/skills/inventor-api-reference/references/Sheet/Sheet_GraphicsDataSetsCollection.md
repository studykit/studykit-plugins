# Sheet.GraphicsDataSetsCollection Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Gets the GraphicsDataSetsCollection object.

## Syntax

Sheet.**GraphicsDataSetsCollection**() As [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md).

## Version

Introduced in version 2008
