# Sheet.Centermarks Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the Centermarks collection object.

## Syntax

Sheet.**Centermarks**() As [Centermarks](../Centermarks/Centermarks.md)

## Property Value

This is a read only property whose value is a [Centermarks](../Centermarks/Centermarks.md).

## Version

Introduced in version 2008
