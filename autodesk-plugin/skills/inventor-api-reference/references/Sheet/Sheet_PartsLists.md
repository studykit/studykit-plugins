# Sheet.PartsLists Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the PartsLists collection object associated with this sheet. This provides access to the existing parts lists on the sheet.

## Syntax

Sheet.**PartsLists**() As [PartsLists](../PartsLists/PartsLists.md)

## Property Value

This is a read only property whose value is a [PartsLists](../PartsLists/PartsLists.md).
