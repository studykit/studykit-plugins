# Sheet.DrawingDimensions Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the DrawingDimensions collection object. This object provides access to all of the dimensions on the sheet.

## Syntax

Sheet.**DrawingDimensions**() As [DrawingDimensions](../DrawingDimensions/DrawingDimensions.md)

## Property Value

This is a read only property whose value is a [DrawingDimensions](../DrawingDimensions/DrawingDimensions.md).
