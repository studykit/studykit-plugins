# Sheet.TransitionSymbols Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Returns the collection of edge symbols on this sheet.

## Syntax

Sheet.**TransitionSymbols**() As [TransitionSymbols](../TransitionSymbols/TransitionSymbols.md)

## Property Value

This is a read only property whose value is a [TransitionSymbols](../TransitionSymbols/TransitionSymbols.md).

## Version

Introduced in version 2025.1
