# Sheet.RevisionClouds Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Get the RevisionClouds collection object.

## Syntax

Sheet.**RevisionClouds**() As [RevisionClouds](../RevisionClouds/RevisionClouds.md)

## Property Value

This is a read only property whose value is a [RevisionClouds](../RevisionClouds/RevisionClouds.md).
