# Sheet.Balloons Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the collection of on this sheet.

## Syntax

Sheet.**Balloons**() As [Balloons](../Balloons/Balloons.md)

## Property Value

This is a read only property whose value is a [Balloons](../Balloons/Balloons.md).
