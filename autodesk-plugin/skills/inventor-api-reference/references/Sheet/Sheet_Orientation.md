# Sheet.Orientation Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Gets or sets the orientation of the sheet as it is viewed or printed.

## Syntax

Sheet.**Orientation**() As [PageOrientationTypeEnum](../PageOrientationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PageOrientationTypeEnum](../PageOrientationTypeEnum.md).

## Version

Introduced in version 4
