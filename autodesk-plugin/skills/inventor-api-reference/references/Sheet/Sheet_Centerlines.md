# Sheet.Centerlines Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the Centerlines collection object.

## Syntax

Sheet.**Centerlines**() As [Centerlines](../Centerlines/Centerlines.md)

## Property Value

This is a read only property whose value is a [Centerlines](../Centerlines/Centerlines.md).

## Version

Introduced in version 2008
