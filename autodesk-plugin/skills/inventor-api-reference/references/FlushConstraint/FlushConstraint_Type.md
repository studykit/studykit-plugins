# FlushConstraint.Type Property

Parent Object: [FlushConstraint](../FlushConstraint/FlushConstraint.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FlushConstraint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 4
