# DecalFeatures Object

## Description

The DecalFeatures collection object provides access to all of the DecalFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../DecalFeatures/DecalFeatures_Add.md) | Method that creates a new decal feature by applying an image on one or more faces in the model. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DecalFeatures/DecalFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DecalFeatures/DecalFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../DecalFeatures/DecalFeatures_Item.md) | Returns the specified DecalFeature object from the collection. This is the default property of the Decals collection object. |
| [Type](../DecalFeatures/DecalFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.DecalFeatures](../PartFeatures/PartFeatures_DecalFeatures.md), [SheetMetalFeatures.DecalFeatures](../SheetMetalFeatures/SheetMetalFeatures_DecalFeatures.md)
