# BendPartFeatureProxy.BendPartType Property

Parent Object: [BendPartFeatureProxy](../BendPartFeatureProxy/BendPartFeatureProxy.md)

## Description

Property that returns the type of the bend part feature. The valid return values are kArcLengthAndAngleBendPart, kRadiusAndAngleBendPart and kRadiusAndArcLengthBendPart.

## Syntax

BendPartFeatureProxy.**BendPartType**() As [BendPartTypeEnum](../BendPartTypeEnum.md)

## Property Value

This is a read only property whose value is a [BendPartTypeEnum](../BendPartTypeEnum.md).

## Version

Introduced in version 2010
