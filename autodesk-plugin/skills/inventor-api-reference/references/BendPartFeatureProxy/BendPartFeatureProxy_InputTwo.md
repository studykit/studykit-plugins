# BendPartFeatureProxy.InputTwo Property

Parent Object: [BendPartFeatureProxy](../BendPartFeatureProxy/BendPartFeatureProxy.md)

## Description

Input Variant that defines the second input for the bend arc.

## Syntax

BendPartFeatureProxy.**InputTwo**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
