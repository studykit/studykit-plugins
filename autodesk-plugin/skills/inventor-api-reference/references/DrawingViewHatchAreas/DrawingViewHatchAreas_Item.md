# DrawingViewHatchAreas.Item Property

Parent Object: [DrawingViewHatchAreas](../DrawingViewHatchAreas/DrawingViewHatchAreas.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

DrawingViewHatchAreas.**Item**( ***Index*** As Long ) As [DrawingViewHatchArea](../DrawingViewHatchArea/DrawingViewHatchArea.md)

## Property Value

This is a read only property whose value is a [DrawingViewHatchArea](../DrawingViewHatchArea/DrawingViewHatchArea.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the item in the collection. If an out of range index of DrawingViewHatchArea is provided, an error will occur. |

## Version

Introduced in version 2022
