# StylesLibraryAccessEnum Enumerator

## Description

Styles Library Access.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kNoStylesLibraryAccess | 54019 | Styles Library is not accessible. |
| kReadOnlyStylesLibraryAccess | 54017 | Styles Library is accessible, though styles can only be read from the library but not written to. |
| kReadWriteStylesLibraryAccess | 54018 | Styles Library is accessible, and styles can be read from and written into the library. |

## Version

Introduced in version 10
