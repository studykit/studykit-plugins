# SketchBlockProxy.ConstraintStatus Property

Parent Object: [SketchBlockProxy](../SketchBlockProxy/SketchBlockProxy.md)

## Description

Property that returns the constraint status of the sketch block. Possible return values are kFullyConstrainedConstraintStatus, kOverConstrainedConstraintStatus, kUnderConstrainedConstraintStatus and kUnknownConstraintStatus.

## Syntax

SketchBlockProxy.**ConstraintStatus**() As [ConstraintStatusEnum](../ConstraintStatusEnum.md)

## Property Value

This is a read only property whose value is a [ConstraintStatusEnum](../ConstraintStatusEnum.md).

## Version

Introduced in version 2010
