# SketchBlockProxy.Color Property

Parent Object: [SketchBlockProxy](../SketchBlockProxy/SketchBlockProxy.md)

## Description

Gets and sets the color for the sketch block. Setting the property to Nothing restores the sketch block to the default color.

## Syntax

SketchBlockProxy.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2010
