# SketchBlockProxy.HasReferenceComponent Property

Parent Object: [SketchBlockProxy](../SketchBlockProxy/SketchBlockProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchBlockProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
