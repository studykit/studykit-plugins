# ApplicationAddIns.UpdateAddInList Method

Parent Object: [ApplicationAddIns](../ApplicationAddIns/ApplicationAddIns.md)

## Description

This method searches for new Add-ins installed since Inventor started. If a new add-in is found, it is added to the add-in list and is loaded if appropriate.

## Syntax

ApplicationAddIns.**UpdateAddInList**()

## Version

Introduced in version 2013
