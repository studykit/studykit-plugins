# BOMRow.BOMStructure Property

Parent Object: [BOMRow](../BOMRow/BOMRow.md)

## Description

Gets and sets how the BOM item represented by this row is used/viewed relating to design.

## Syntax

BOMRow.**BOMStructure**() As [BOMStructureEnum](../BOMStructureEnum.md)

## Property Value

This is a read/write property whose value is a [BOMStructureEnum](../BOMStructureEnum.md).

## Version

Introduced in version 10
