# BOMRow.ComponentOccurrences Property

Parent Object: [BOMRow](../BOMRow/BOMRow.md)

## Description

Gets the ComponentOccurrences associated with this row in the BOM.

## Syntax

BOMRow.**ComponentOccurrences**() As [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md).

## Version

Introduced in version 2022
