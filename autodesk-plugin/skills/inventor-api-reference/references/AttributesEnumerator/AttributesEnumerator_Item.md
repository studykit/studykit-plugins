# AttributesEnumerator.Item Property

Parent Object: [AttributesEnumerator](../AttributesEnumerator/AttributesEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

AttributesEnumerator.**Item**( ***Index*** As Long ) As [Attribute](../Attribute/Attribute.md)

## Property Value

This is a read only property whose value is an [Attribute](../Attribute/Attribute.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 5
