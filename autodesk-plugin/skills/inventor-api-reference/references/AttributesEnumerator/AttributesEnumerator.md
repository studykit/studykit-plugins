# AttributesEnumerator Object

## Description

The AttributesEnumerator object provides access to a list of Attribute objects. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../AttributesEnumerator/AttributesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../AttributesEnumerator/AttributesEnumerator_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../AttributesEnumerator/AttributesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[AttributeManager.FindAttributes](../AttributeManager/AttributeManager_FindAttributes.md), [AttributeSet.AddAttributes](../AttributeSet/AttributeSet_AddAttributes.md)

## Version

Introduced in version 5
