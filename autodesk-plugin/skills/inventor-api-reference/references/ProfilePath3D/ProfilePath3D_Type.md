# ProfilePath3D.Type Property

Parent Object: [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ProfilePath3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
