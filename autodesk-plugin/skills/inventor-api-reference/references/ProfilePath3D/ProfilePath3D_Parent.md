# ProfilePath3D.Parent Property

Parent Object: [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Description

Property that returns the parent Profile3D.

## Syntax

ProfilePath3D.**Parent**() As [Profile3D](../Profile3D/Profile3D.md)

## Property Value

This is a read only property whose value is a [Profile3D](../Profile3D/Profile3D.md).

## Version

Introduced in version 6
