# ProfilePath3D.Closed Property

Parent Object: [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Description

Property that returns a Boolean indicating if the path is closed or not. Returns True in the case of a closed path.

## Syntax

ProfilePath3D.**Closed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
