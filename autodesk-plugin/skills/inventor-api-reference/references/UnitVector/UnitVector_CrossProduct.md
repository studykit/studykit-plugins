# UnitVector.CrossProduct Method

Parent Object: [UnitVector](../UnitVector/UnitVector.md)

## Description

Determine the cross product between this vector and the specified vector.

## Syntax

UnitVector.**CrossProduct**( ***Argument*** As [UnitVector](../UnitVector/UnitVector.md) ) As [UnitVector](../UnitVector/UnitVector.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [UnitVector](../UnitVector/UnitVector.md) | Input  used for comparison. |
