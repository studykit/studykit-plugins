# UnitVector.Copy Method

Parent Object: [UnitVector](../UnitVector/UnitVector.md)

## Description

Creates a copy of this UnitVector object. The result is entirely independent and can be edited without affecting the original UnitVector object.

## Syntax

UnitVector.**Copy**() As [UnitVector](../UnitVector/UnitVector.md)

## Version

Introduced in version 2013
