# UnitVector.AngleTo Method

Parent Object: [UnitVector](../UnitVector/UnitVector.md)

## Description

Determines the angle between this vector and the specified vector.

## Remarks

When one of the input vectors is null (0,0,0) this method will return -NaN.

## Syntax

UnitVector.**AngleTo**( ***Argument*** As [UnitVector](../UnitVector/UnitVector.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [UnitVector](../UnitVector/UnitVector.md) | Input  used for comparison. |

## Version

Introduced in version 4
