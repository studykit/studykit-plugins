# UnitVector.TransformBy Method

Parent Object: [UnitVector](../UnitVector/UnitVector.md)

## Description

Transform this vector by the specified matrix.

## Syntax

UnitVector.**TransformBy**( ***Matrix*** As [Matrix](../Matrix/Matrix.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix | [Matrix](../Matrix/Matrix.md) | Input Matrix object that specifies the matrix to transform the vector by. |

## Version

Introduced in version 4
