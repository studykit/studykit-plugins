# ASideDefinitions.Add Method

Parent Object: [ASideDefinitions](../ASideDefinitions/ASideDefinitions.md)

## Description

Method that creates a new ASideFaceDefinition object with the input face.

## Syntax

ASideDefinitions.**Add**( ***pInput*** As [Face](../Face/Face.md) ) As [ASideDefinition](../ASideDefinition/ASideDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| pInput | [Face](../Face/Face.md) | Input Face object to specify a face to create an A-side face definition on it. |

## Version

Introduced in version 2016
