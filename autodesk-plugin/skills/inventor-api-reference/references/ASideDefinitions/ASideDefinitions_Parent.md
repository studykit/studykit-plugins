# ASideDefinitions.Parent Property

Parent Object: [ASideDefinitions](../ASideDefinitions/ASideDefinitions.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ASideDefinitions.**Parent**() As [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Property Value

This is a read only property whose value is a [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md).

## Version

Introduced in version 2016
