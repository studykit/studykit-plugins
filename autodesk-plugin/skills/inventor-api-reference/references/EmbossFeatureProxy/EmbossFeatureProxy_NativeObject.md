# EmbossFeatureProxy.NativeObject Property

Parent Object: [EmbossFeatureProxy](../EmbossFeatureProxy/EmbossFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

EmbossFeatureProxy.**NativeObject**() As [EmbossFeature](../EmbossFeature/EmbossFeature.md)

## Property Value

This is a read only property whose value is an [EmbossFeature](../EmbossFeature/EmbossFeature.md).

## Version

Introduced in version 9
