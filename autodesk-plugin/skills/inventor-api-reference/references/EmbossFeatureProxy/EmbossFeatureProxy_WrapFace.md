# EmbossFeatureProxy.WrapFace Property

Parent Object: [EmbossFeatureProxy](../EmbossFeatureProxy/EmbossFeatureProxy.md)

## Description

Gets and sets the face around which the emboss feature is wrapped.

## Syntax

EmbossFeatureProxy.**WrapFace**() As [Face](../Face/Face.md)

## Property Value

This is a read/write property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2008
