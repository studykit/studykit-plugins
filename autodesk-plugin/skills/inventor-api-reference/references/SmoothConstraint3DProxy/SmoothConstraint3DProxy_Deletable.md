# SmoothConstraint3DProxy.Deletable Property

Parent Object: [SmoothConstraint3DProxy](../SmoothConstraint3DProxy/SmoothConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

SmoothConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
