# SmoothConstraint3DProxy.NativeObject Property

Parent Object: [SmoothConstraint3DProxy](../SmoothConstraint3DProxy/SmoothConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SmoothConstraint3DProxy.**NativeObject**() As [SmoothConstraint3D](../SmoothConstraint3D/SmoothConstraint3D.md)

## Property Value

This is a read only property whose value is a [SmoothConstraint3D](../SmoothConstraint3D/SmoothConstraint3D.md).

## Version

Introduced in version 11
