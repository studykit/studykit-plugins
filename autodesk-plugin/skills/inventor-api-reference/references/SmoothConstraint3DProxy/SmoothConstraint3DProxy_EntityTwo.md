# SmoothConstraint3DProxy.EntityTwo Property

Parent Object: [SmoothConstraint3DProxy](../SmoothConstraint3DProxy/SmoothConstraint3DProxy.md)

## Description

Property that returns the sketch entity that is constrained to be smooth (G2-continuous) to entity one.

## Syntax

SmoothConstraint3DProxy.**EntityTwo**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 11
