# BSplineCurve2d.Copy Method

Parent Object: [BSplineCurve2d](../BSplineCurve2d/BSplineCurve2d.md)

## Description

Creates a copy of this BSplineCurve2d object. The result is entirely independent and can be edited without affecting the original BSplineCurve2d object.

## Syntax

BSplineCurve2d.**Copy**() As [BSplineCurve2d](../BSplineCurve2d/BSplineCurve2d.md)

## Version

Introduced in version 2013
