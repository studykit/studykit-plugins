# FeatureControlFrameRow.DatumOne Property

Parent Object: [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md)

## Description

Gets and sets the first datum that affects the tolerance. Returns a null string if the datum is not specified. The string can contain symbols specified using the StyleOverride tag. E.g. use 'm' to specify (M).

## Syntax

FeatureControlFrameRow.**DatumOne**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
