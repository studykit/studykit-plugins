# FeatureControlFrameRow.LowerTolerance Property

Parent Object: [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md)

## Description

Gets and sets the lower tolerance associated with this geometric characteristic. Applies to ANSI standard only. The string can contain symbols specified using the StyleOverride tag. E.g. use 'm' to specify (M).

## Syntax

FeatureControlFrameRow.**LowerTolerance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
