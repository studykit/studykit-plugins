# FeatureControlFrameRow.InlineNote Property

Parent Object: [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md)

## Description

Gets and sets the inline note. This is applicable only when the FeatureControlFrameStyle.EnableInlineNotes is set to True when the style is applied to this FeatureControlFrame object.

## Syntax

FeatureControlFrameRow.**InlineNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2023
