# FeatureControlFrameRow.Tolerance Property

Parent Object: [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md)

## Description

Gets and sets the tolerance associated with this geometric characteristic. The string can contain symbols specified using the StyleOverride tag. E.g. use 'm' to specify (M).

## Syntax

FeatureControlFrameRow.**Tolerance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
