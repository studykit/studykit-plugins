# FeatureControlFrameRow.Parent Property

Parent Object: [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md)

## Description

Property that returns the parent FeatureControlFrame object.

## Syntax

FeatureControlFrameRow.**Parent**() As [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Property Value

This is a read only property whose value is a [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md).

## Version

Introduced in version 2009
