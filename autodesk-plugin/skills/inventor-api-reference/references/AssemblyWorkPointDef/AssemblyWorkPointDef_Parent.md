# AssemblyWorkPointDef.Parent Property

Parent Object: [AssemblyWorkPointDef](../AssemblyWorkPointDef/AssemblyWorkPointDef.md)

## Description

Property that returns the parent WorkPoint object.

## Syntax

AssemblyWorkPointDef.**Parent**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 6
