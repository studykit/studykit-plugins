# AssemblyWorkPointDef Object

## Description

The AssemblyWorkPointDef object allows you to get and set the information that specifies a fixed work point.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../AssemblyWorkPointDef/AssemblyWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Constraints](../AssemblyWorkPointDef/AssemblyWorkPointDef_Constraints.md) | Property that returns the collection of constraints that are acting on the work point. This returns all of the constraints, including those returned by the DefinitionConstraints property. |
| [Parent](../AssemblyWorkPointDef/AssemblyWorkPointDef_Parent.md) | Property that returns the parent WorkPoint object. |
| [Point](../AssemblyWorkPointDef/AssemblyWorkPointDef_Point.md) | Specifies the origin point of the work point. |
| [Type](../AssemblyWorkPointDef/AssemblyWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 6
