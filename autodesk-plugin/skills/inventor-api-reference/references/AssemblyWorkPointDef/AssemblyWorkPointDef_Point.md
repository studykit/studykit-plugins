# AssemblyWorkPointDef.Point Property

Parent Object: [AssemblyWorkPointDef](../AssemblyWorkPointDef/AssemblyWorkPointDef.md)

## Description

Specifies the origin point of the work point.

## Syntax

AssemblyWorkPointDef.**Point**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 7
