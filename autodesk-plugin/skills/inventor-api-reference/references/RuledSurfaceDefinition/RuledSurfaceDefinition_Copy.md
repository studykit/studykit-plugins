# RuledSurfaceDefinition.Copy Method

Parent Object: [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md)

## Description

Method that creates a copy of this RuledSurfaceDefinition Object. The new RuledSurfaceDefinition Object is independent of any RuledSurfaceFeature Object.

## Syntax

RuledSurfaceDefinition.**Copy**() As [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md)

## Version

Introduced in version 2016
