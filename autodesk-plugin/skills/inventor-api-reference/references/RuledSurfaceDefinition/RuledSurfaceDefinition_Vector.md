# RuledSurfaceDefinition.Vector Property

Parent Object: [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md)

## Description

Read-write property that gets and sets the extend direction of the ruled surface.

## Syntax

RuledSurfaceDefinition.**Vector**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2016
