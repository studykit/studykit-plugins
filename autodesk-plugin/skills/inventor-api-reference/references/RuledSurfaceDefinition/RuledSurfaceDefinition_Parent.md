# RuledSurfaceDefinition.Parent Property

Parent Object: [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

RuledSurfaceDefinition.**Parent**() As [RuledSurfaceFeature](../RuledSurfaceFeature/RuledSurfaceFeature.md)

## Property Value

This is a read only property whose value is a [RuledSurfaceFeature](../RuledSurfaceFeature/RuledSurfaceFeature.md).

## Version

Introduced in version 2016
