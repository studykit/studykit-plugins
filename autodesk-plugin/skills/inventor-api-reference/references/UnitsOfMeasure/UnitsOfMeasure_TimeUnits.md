# UnitsOfMeasure.TimeUnits Property

Parent Object: [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md)

## Description

Gets/Sets the default unit of time for this Document.

## Syntax

UnitsOfMeasure.**TimeUnits**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).

## Version

Introduced in version 4
