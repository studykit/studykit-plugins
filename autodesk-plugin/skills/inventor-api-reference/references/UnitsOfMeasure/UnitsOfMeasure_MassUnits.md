# UnitsOfMeasure.MassUnits Property

Parent Object: [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md)

## Description

Gets/Sets the default unit of mass for this Document.

## Syntax

UnitsOfMeasure.**MassUnits**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).

## Version

Introduced in version 4
