# CustomPropertyFormat.ShowTrailingZeros Property

Parent Object: [CustomPropertyFormat](../CustomPropertyFormat/CustomPropertyFormat.md)

## Description

Gets/Sets if trailing zeros should be shown in the formatted iProperty value.

## Syntax

CustomPropertyFormat.**ShowTrailingZeros**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
