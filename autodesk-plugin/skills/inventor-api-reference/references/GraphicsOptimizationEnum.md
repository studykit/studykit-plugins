# GraphicsOptimizationEnum Enumerator

## Description

Constants specifying the graphics card optimization setting.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kConservativeGraphicsOptimization | 55299 | Using conservative optimization setting. |
| kDriverGraphicsOptimization | 55300 | Using Microsoft GDI generic OpenGL driver or Direct3D Software Renderer. |
| kFullGraphicsOptimization | 55298 | Using full optimization setting. |
| kRecommendedGraphicsOptimization | 55297 | Using recommended optimization setting. |

## Version

Introduced in version 11
