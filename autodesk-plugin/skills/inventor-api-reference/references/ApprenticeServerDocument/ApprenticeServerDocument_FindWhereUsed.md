# ApprenticeServerDocument.FindWhereUsed Method

Parent Object: [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md)

## Description

Method that obtains the set of documents that reference the given file within this document.

## Syntax

ApprenticeServerDocument.**FindWhereUsed**( ***FullFileName*** As String ) As [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input String that specifies the full file name of the file to search for references to. to. |

## Version

Introduced in version 5
