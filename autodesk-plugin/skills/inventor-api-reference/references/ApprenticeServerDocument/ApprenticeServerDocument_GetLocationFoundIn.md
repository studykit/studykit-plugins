# ApprenticeServerDocument.GetLocationFoundIn Method

Parent Object: [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md)

## Description

Obtains the name of the location this file was found in.

## Syntax

ApprenticeServerDocument.**GetLocationFoundIn**( ***LocationName*** As String, ***Type*** As [LocationTypeEnum](../LocationTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LocationName | String | Output String that specifies the name of the location. |
| Type | [LocationTypeEnum](../LocationTypeEnum.md) | Output that specifies the location type. |

## Version

Introduced in version 4
