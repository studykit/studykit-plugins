# OrdinateDimensionSets.Item Property

Parent Object: [OrdinateDimensionSets](../OrdinateDimensionSets/OrdinateDimensionSets.md)

## Description

Method that returns the specified dimension set object from the collection.

## Syntax

OrdinateDimensionSets.**Item**( ***Index*** As Long ) As [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Property Value

This is a read only property whose value is an [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
