# OrdinateDimensionSets Object

## Description

The OrdinateDimensionSets collection object provides access to all of the ordinate dimension sets on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../OrdinateDimensionSets/OrdinateDimensionSets_Add.md) | Method that creates an ordinate dimension set and returns the newly created OrdinateDimensionSet object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../OrdinateDimensionSets/OrdinateDimensionSets_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../OrdinateDimensionSets/OrdinateDimensionSets_Count.md) | Property that returns the number of items in the collection. |
| [Item](../OrdinateDimensionSets/OrdinateDimensionSets_Item.md) | Method that returns the specified dimension set object from the collection. |
| [Type](../OrdinateDimensionSets/OrdinateDimensionSets_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingDimensions.OrdinateDimensionSets](../DrawingDimensions/DrawingDimensions_OrdinateDimensionSets.md)

## Version

Introduced in version 2010
