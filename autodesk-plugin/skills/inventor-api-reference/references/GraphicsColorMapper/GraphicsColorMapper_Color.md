# GraphicsColorMapper.Color Property

Parent Object: [GraphicsColorMapper](../GraphicsColorMapper/GraphicsColorMapper.md)

## Description

Read-write property that gets and sets the color at a given index in the set.

## Syntax

GraphicsColorMapper.**Color**( ***Index*** As Long ) As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).
