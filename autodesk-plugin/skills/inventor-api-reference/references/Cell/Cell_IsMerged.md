# Cell.IsMerged Property

Parent Object: [Cell](../Cell/Cell.md)

## Description

Returns whether this cell is merged or not.

## Syntax

Cell.**IsMerged**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025.1
