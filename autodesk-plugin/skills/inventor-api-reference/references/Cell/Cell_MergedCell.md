# Cell.MergedCell Property

Parent Object: [Cell](../Cell/Cell.md)

## Description

Returns the merged cell that this cell is merged in.

## Syntax

Cell.**MergedCell**() As [MergedCell](../MergedCell/MergedCell.md)

## Property Value

This is a read only property whose value is a [MergedCell](../MergedCell/MergedCell.md).

## Version

Introduced in version 2025.1
