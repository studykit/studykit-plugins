# Cell.Parent Property

Parent Object: [Cell](../Cell/Cell.md)

## Description

Property that returns the parent Table object.

## Syntax

Cell.**Parent**() As [CustomTable](../CustomTable/CustomTable.md)

## Property Value

This is a read only property whose value is a [CustomTable](../CustomTable/CustomTable.md).

## Version

Introduced in version 9
