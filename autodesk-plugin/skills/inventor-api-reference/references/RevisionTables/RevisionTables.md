# RevisionTables Object

## Description

The RevisionTables collection object represents all the revision history tables on a sheet and provides methods to create additional tables.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../RevisionTables/RevisionTables_Add.md) | Method that creates a new RevisionTable. The new created RevisionTable is returned. |
| [Add2](../RevisionTables/RevisionTables_Add2.md) | Creates a new RevisionTable object on the sheet. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevisionTables/RevisionTables_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../RevisionTables/RevisionTables_Count.md) | Property that returns the number of items in the collection. |
| [Item](../RevisionTables/RevisionTables_Item.md) | Returns the specified RevisionTable object from the collection. |
| [Type](../RevisionTables/RevisionTables_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Sheet.RevisionTables](../Sheet/Sheet_RevisionTables.md)

## Version

Introduced in version 10
