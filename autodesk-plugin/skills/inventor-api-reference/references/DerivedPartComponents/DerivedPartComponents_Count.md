# DerivedPartComponents.Count Property

Parent Object: [DerivedPartComponents](../DerivedPartComponents/DerivedPartComponents.md)

## Description

Property that returns the number of items in the collection.

## Syntax

DerivedPartComponents.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5.3
