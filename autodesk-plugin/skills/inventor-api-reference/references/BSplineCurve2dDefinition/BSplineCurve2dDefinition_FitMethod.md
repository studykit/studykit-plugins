# BSplineCurve2dDefinition.FitMethod Property

Parent Object: [BSplineCurve2dDefinition](../BSplineCurve2dDefinition/BSplineCurve2dDefinition.md)

## Description

Gets and sets the FitMethod setting.

## Syntax

BSplineCurve2dDefinition.**FitMethod**() As [SplineFitMethodEnum](../SplineFitMethodEnum.md)

## Property Value

This is a read/write property whose value is a [SplineFitMethodEnum](../SplineFitMethodEnum.md).

## Version

Introduced in version 9
