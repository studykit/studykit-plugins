# BSplineCurve2dDefinition.Type Property

Parent Object: [BSplineCurve2dDefinition](../BSplineCurve2dDefinition/BSplineCurve2dDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

BSplineCurve2dDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
