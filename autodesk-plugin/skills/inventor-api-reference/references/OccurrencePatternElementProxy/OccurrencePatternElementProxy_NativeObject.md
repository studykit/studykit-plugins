# OccurrencePatternElementProxy.NativeObject Property

Parent Object: [OccurrencePatternElementProxy](../OccurrencePatternElementProxy/OccurrencePatternElementProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

OccurrencePatternElementProxy.**NativeObject**() As [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Property Value

This is a read only property whose value is an [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md).

## Version

Introduced in version 2010
