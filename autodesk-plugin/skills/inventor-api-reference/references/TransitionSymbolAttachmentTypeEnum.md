# TransitionSymbolAttachmentTypeEnum Enumerator

## Description

Constants that define the drawing transition symbol attachment types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDrawingViewAttachmentType | 129283 | Transition symbol attaches to a drawing view. |
| kGeometryAttachmentType | 129282 | Transition symbol attaches to a geometry( DrawingCurve, drawing annotations etc.). |
| kNoAttachmentType | 129281 | Transition symbol. |

## Version

Introduced in version 2025.1
