# EllipseFull.MajorAxisVector Property

Parent Object: [EllipseFull](../EllipseFull/EllipseFull.md)

## Description

Specifies the direction of the major axis of the ellipse.

## Syntax

EllipseFull.**MajorAxisVector**() As [Vector](../Vector/Vector.md)

## Property Value

This is a read/write property whose value is a [Vector](../Vector/Vector.md).

## Version

Introduced in version 4
