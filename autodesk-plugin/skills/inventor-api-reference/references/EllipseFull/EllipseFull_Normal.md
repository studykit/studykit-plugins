# EllipseFull.Normal Property

Parent Object: [EllipseFull](../EllipseFull/EllipseFull.md)

## Description

Specifies the axis of the ellipse. The axis extends out of the ellipse.

## Syntax

EllipseFull.**Normal**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
