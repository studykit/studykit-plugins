# EllipseFull Object

## Description

The EllipseFull object. The object created is a transient mathematical object and is not displayed graphically

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../EllipseFull/EllipseFull_Copy.md) | Creates a copy of this EllipseFull object. The result is entirely independent and can be edited without affecting the original EllipseFull object. |
| [GetEllipseFullData](../EllipseFull/EllipseFull_GetEllipseFullData.md) | Get the data defining this ellipse. |
| [PutEllipseFullData](../EllipseFull/EllipseFull_PutEllipseFullData.md) | Method that sets the data defining this ellipse. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Center](../EllipseFull/EllipseFull_Center.md) | Specifies the center of the ellipse. |
| [Evaluator](../EllipseFull/EllipseFull_Evaluator.md) | Gets the CurveEvaluator for this ellipse. |
| [MajorAxisVector](../EllipseFull/EllipseFull_MajorAxisVector.md) | Specifies the direction of the major axis of the ellipse. |
| [MinorMajorRatio](../EllipseFull/EllipseFull_MinorMajorRatio.md) | Specifies the ratio of the minor and major axes of the ellipse. |
| [Normal](../EllipseFull/EllipseFull_Normal.md) | Specifies the axis of the ellipse. The axis extends out of the ellipse. |
