# TwoPointDistanceDimConstraint3DProxy.NativeObject Property

Parent Object: [TwoPointDistanceDimConstraint3DProxy](../TwoPointDistanceDimConstraint3DProxy/TwoPointDistanceDimConstraint3DProxy.md)

## Description

## Syntax

TwoPointDistanceDimConstraint3DProxy.**NativeObject**() As [TwoPointDistanceDimConstraint3D](../TwoPointDistanceDimConstraint3D/TwoPointDistanceDimConstraint3D.md)

## Property Value

This is a read only property whose value is a [TwoPointDistanceDimConstraint3D](../TwoPointDistanceDimConstraint3D/TwoPointDistanceDimConstraint3D.md).

## Version
