# TangentDistanceDimConstraintProxy.ProximityPointTwo Property

Parent Object: [TangentDistanceDimConstraintProxy](../TangentDistanceDimConstraintProxy/TangentDistanceDimConstraintProxy.md)

## Description

The point supplied by this argument is used when EntityTwo is a circle or arc. This point specifies which of the possible two tangent cases is used when creating the constraint. The closest tangent to the input point is used. If EntityTwo is a line, this argument is ignored.

## Syntax

TangentDistanceDimConstraintProxy.**ProximityPointTwo**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 8
