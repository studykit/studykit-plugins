# ApprenticeDrawingPrintManager.PrintExcludedSheets Property

Parent Object: [ApprenticeDrawingPrintManager](../ApprenticeDrawingPrintManager/ApprenticeDrawingPrintManager.md)

## Description

Gets/Sets the Boolean flag indicating whether excluded sheets (when PrintRange is set to kPrintAllSheets or kPrintSheetRange) should be printed. Defaults to FALSE.

## Syntax

ApprenticeDrawingPrintManager.**PrintExcludedSheets**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
