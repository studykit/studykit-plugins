# ApprenticeDrawingPrintManager.SetCurrentView Method

Parent Object: [ApprenticeDrawingPrintManager](../ApprenticeDrawingPrintManager/ApprenticeDrawingPrintManager.md)

## Description

Property to set the current view, to be used for printing.

## Syntax

ApprenticeDrawingPrintManager.**SetCurrentView**( ***CurrentView*** As [ClientView](../ClientView/ClientView.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CurrentView | [ClientView](../ClientView/ClientView.md) | Input ClientView that defines the area to print. |

## Version

Introduced in version 9
