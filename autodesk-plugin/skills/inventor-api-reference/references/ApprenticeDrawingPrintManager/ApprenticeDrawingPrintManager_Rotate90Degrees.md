# ApprenticeDrawingPrintManager.Rotate90Degrees Property

Parent Object: [ApprenticeDrawingPrintManager](../ApprenticeDrawingPrintManager/ApprenticeDrawingPrintManager.md)

## Description

Gets/Sets the Boolean flag indicating whether the Drawing should be rotated 90-deg in the print.

## Syntax

ApprenticeDrawingPrintManager.**Rotate90Degrees**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
