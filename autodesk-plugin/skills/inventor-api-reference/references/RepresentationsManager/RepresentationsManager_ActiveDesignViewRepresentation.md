# RepresentationsManager.ActiveDesignViewRepresentation Property

Parent Object: [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md)

## Description

Property that returns the active DesignViewRepresentation object.

## Syntax

RepresentationsManager.**ActiveDesignViewRepresentation**() As [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Property Value

This is a read only property whose value is a [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md).

## Version

Introduced in version 11
