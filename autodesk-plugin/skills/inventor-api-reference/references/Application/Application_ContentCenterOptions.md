# Application.ContentCenterOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Returns the ContentCenterOptions object.

## Syntax

Application.**ContentCenterOptions**() As [ContentCenterOptions](../ContentCenterOptions/ContentCenterOptions.md)

## Property Value

This is a read only property whose value is a [ContentCenterOptions](../ContentCenterOptions/ContentCenterOptions.md).

## Version

Introduced in version 2010
