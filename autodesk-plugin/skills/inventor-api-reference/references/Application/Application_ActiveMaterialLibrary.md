# Application.ActiveMaterialLibrary Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets and sets the library whose contents is listed in the materials drop-down list in the QAT(Quick Access Toolbar) at the top of the Inventor window.

## Syntax

Application.**ActiveMaterialLibrary**() As [AssetLibrary](../AssetLibrary/AssetLibrary.md)

## Property Value

This is a read/write property whose value is an [AssetLibrary](../AssetLibrary/AssetLibrary.md).

## Version

Introduced in version 2014
