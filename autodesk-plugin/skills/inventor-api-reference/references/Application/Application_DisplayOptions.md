# Application.DisplayOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the Display Options.

## Syntax

Application.**DisplayOptions**() As [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Property Value

This is a read only property whose value is a [DisplayOptions](../DisplayOptions/DisplayOptions.md).

## Version

Introduced in version 11
