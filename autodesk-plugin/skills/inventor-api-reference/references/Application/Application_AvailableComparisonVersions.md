# Application.AvailableComparisonVersions Property

Parent Object: [Application](../Application/Application.md)

## Description

Read-only property that returns an array of strings indicating the versions can be used to compare the commands with selected version.

## Syntax

Application.**AvailableComparisonVersions**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2020
