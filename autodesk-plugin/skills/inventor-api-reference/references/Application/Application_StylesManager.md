# Application.StylesManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the StylesManager object.

## Syntax

Application.**StylesManager**() As [StylesManager](../StylesManager/StylesManager.md)

## Property Value

This is a read only property whose value is a [StylesManager](../StylesManager/StylesManager.md).

## Version

Introduced in version 2010
