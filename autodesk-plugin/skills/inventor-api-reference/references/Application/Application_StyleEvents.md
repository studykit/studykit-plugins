# Application.StyleEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the StyleEvents object. This object supports a set of events that relate to styles.

## Syntax

Application.**StyleEvents**() As [StyleEvents](../StyleEvents/StyleEvents.md)

## Property Value

This is a read only property whose value is a [StyleEvents](../StyleEvents/StyleEvents.md).

## Version

Introduced in version 11
