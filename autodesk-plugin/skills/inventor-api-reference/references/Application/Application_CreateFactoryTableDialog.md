# Application.CreateFactoryTableDialog Method

Parent Object: [Application](../Application/Application.md)

## Description

Creates a Factory Table dialog.

## Syntax

Application.**CreateFactoryTableDialog**( ***FullFileName*** As String ) As [FactoryTableDialog](../FactoryTableDialog/FactoryTableDialog.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input String that indicates the full filename of an ipart/iassembly document. |

## Version

Introduced in version 2017
