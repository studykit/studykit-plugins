# Application.FileAccessEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the FileAccessEvents object. This object supports a set of events that are fired as a result of a file being accessed.

## Syntax

Application.**FileAccessEvents**() As [FileAccessEvents](../FileAccessEvents/FileAccessEvents.md)

## Property Value

This is a read only property whose value is a [FileAccessEvents](../FileAccessEvents/FileAccessEvents.md).

## Version

Introduced in version 4
