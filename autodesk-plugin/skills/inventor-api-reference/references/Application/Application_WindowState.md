# Application.WindowState Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets/Sets the frame windows state.

## Syntax

Application.**WindowState**() As [WindowsSizeEnum](../WindowsSizeEnum.md)

## Property Value

This is a read/write property whose value is a [WindowsSizeEnum](../WindowsSizeEnum.md).

## Version

Introduced in version 6
