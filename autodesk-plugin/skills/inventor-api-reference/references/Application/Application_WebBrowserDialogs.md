# Application.WebBrowserDialogs Property

Parent Object: [Application](../Application/Application.md)

## Description

Read-only property that returns the WebBrowserDialogs collection object.

## Syntax

Application.**WebBrowserDialogs**() As [WebBrowserDialogs](../WebBrowserDialogs/WebBrowserDialogs.md)

## Property Value

This is a read only property whose value is a [WebBrowserDialogs](../WebBrowserDialogs/WebBrowserDialogs.md).

## Version

Introduced in version 2017
