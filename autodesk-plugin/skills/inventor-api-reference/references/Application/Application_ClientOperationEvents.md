# Application.ClientOperationEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Read-only property that returns the ClientOperationEvents object.

## Syntax

Application.**ClientOperationEvents**() As [ClientOperationEvents](../ClientOperationEvents/ClientOperationEvents.md)

## Property Value

This is a read only property whose value is a [ClientOperationEvents](../ClientOperationEvents/ClientOperationEvents.md).

## Version

Introduced in version 2025.1
