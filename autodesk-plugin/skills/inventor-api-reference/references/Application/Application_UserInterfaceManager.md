# Application.UserInterfaceManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the UserInterfaceManager object. The UserInterfaceManager object is the starting point for UI and environment control, providing access to available environments, command bars, browsers, panel bars, toolbars etc.

## Syntax

Application.**UserInterfaceManager**() As [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Property Value

This is a read only property whose value is a [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md).
