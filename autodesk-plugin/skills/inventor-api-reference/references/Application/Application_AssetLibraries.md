# Application.AssetLibraries Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets AssetLibraries collection object. This collection provides access to the all of the appearance and material libraries.

## Syntax

Application.**AssetLibraries**() As [AssetLibraries](../AssetLibraries/AssetLibraries.md)

## Property Value

This is a read only property whose value is an [AssetLibraries](../AssetLibraries/AssetLibraries.md).
