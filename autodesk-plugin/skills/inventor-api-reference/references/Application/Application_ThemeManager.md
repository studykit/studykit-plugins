# Application.ThemeManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the ThemeManager object.

## Syntax

Application.**ThemeManager**() As [ThemeManager](../ThemeManager/ThemeManager.md)

## Property Value

This is a read only property whose value is a [ThemeManager](../ThemeManager/ThemeManager.md).

## Version

Introduced in version 2020
