# Application.ColorSchemes Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the ColorSchemes object. The ColorSchemes object provides access to the color schemes. This is somewhat equivalent to the Colors tab of the Application Options dialog.

## Syntax

Application.**ColorSchemes**() As [ColorSchemes](../ColorSchemes/ColorSchemes.md)

## Property Value

This is a read only property whose value is a [ColorSchemes](../ColorSchemes/ColorSchemes.md).

## Version

Introduced in version 9
