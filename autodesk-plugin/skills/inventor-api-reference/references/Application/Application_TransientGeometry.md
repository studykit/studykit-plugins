# Application.TransientGeometry Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the object through which all transient geometry objects can be constructed.

## Syntax

Application.**TransientGeometry**() As [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Property Value

This is a read only property whose value is a [TransientGeometry](../TransientGeometry/TransientGeometry.md).
