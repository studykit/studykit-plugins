# Application.SketchEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the SketchEvents object. This object supports a set of 2d and 3d sketch-related events that are common to parts, assemblies and drawings.

## Syntax

Application.**SketchEvents**() As [SketchEvents](../SketchEvents/SketchEvents.md)

## Property Value

This is a read only property whose value is a [SketchEvents](../SketchEvents/SketchEvents.md).

## Version

Introduced in version 11
