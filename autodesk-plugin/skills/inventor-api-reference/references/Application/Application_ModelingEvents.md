# Application.ModelingEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the ModelingEvents object. This object supports a set of events that are common to parts and assemblies.

## Syntax

Application.**ModelingEvents**() As [ModelingEvents](../ModelingEvents/ModelingEvents.md)

## Property Value

This is a read only property whose value is a [ModelingEvents](../ModelingEvents/ModelingEvents.md).

## Version

Introduced in version 11
