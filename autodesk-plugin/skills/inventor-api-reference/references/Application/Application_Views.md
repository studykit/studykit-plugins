# Application.Views Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets all the open Views in a collection.

## Syntax

Application.**Views**() As [ViewsEnumerator](../ViewsEnumerator/ViewsEnumerator.md)

## Property Value

This is a read only property whose value is a [ViewsEnumerator](../ViewsEnumerator/ViewsEnumerator.md).

## Version

Introduced in version 2010
