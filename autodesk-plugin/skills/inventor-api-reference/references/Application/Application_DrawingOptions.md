# Application.DrawingOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the DrawingOptions object. The DrawingOptions object provides access to various drawing related application level options. This is somewhat equivalent to the Drawing tab of the Application Options dialog.

## Syntax

Application.**DrawingOptions**() As [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Property Value

This is a read only property whose value is a [DrawingOptions](../DrawingOptions/DrawingOptions.md).

## Version

Introduced in version 2008
