# Application.ConstructInternalNameAndRevisionId Method

Parent Object: [Application](../Application/Application.md)

## Description

Constructs and returns the Internal Name and Revision Identifier

## Syntax

Application.**ConstructInternalNameAndRevisionId**( ***InternalNameToken*** As String, ***RevisionIdToken*** As String, ***InternalName*** As String, ***RevisionId*** As String )

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| InternalNameToken | String | \_nt\_ |
| RevisionIdToken | String | \_nt\_ |
| InternalName | String | \_nt\_ |
| RevisionId | String | \_nt\_ |

## Version
