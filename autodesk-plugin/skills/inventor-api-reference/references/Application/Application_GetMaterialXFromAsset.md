# Application.GetMaterialXFromAsset Method

Parent Object: [Application](../Application/Application.md)

## Description

Method that gets the MaterialX string from an Asset.

## Syntax

Application.**GetMaterialXFromAsset**( ***Asset*** As [Asset](../Asset/Asset.md), [***converterOptions***] As Variant ) As String

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Asset | [Asset](../Asset/Asset.md) | Input Asset object from which to generate the MaterialX string. |
| converterOptions | Variant | Optional input NameValueMap that specifies more info to determine how to get the materialx string. Currently this is ignored. |

## Version

Introduced in version 2023
