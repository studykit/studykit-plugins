# Application.CameraEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the object that fires the Camera related events.

## Syntax

Application.**CameraEvents**() As [CameraEvents](../CameraEvents/CameraEvents.md)

## Property Value

This is a read only property whose value is a [CameraEvents](../CameraEvents/CameraEvents.md).

## Version

Introduced in version 2011
