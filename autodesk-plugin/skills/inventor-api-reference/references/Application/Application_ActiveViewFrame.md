# Application.ActiveViewFrame Property

Parent Object: [Application](../Application/Application.md)

## Description

Read-only property that returns the active ViewFrame object.

## Syntax

Application.**ActiveViewFrame**() As [ViewFrame](../ViewFrame/ViewFrame.md)

## Property Value

This is a read only property whose value is a [ViewFrame](../ViewFrame/ViewFrame.md).

## Version

Introduced in version 2021
