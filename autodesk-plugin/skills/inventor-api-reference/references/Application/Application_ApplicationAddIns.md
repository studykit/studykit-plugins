# Application.ApplicationAddIns Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the ApplicationAddIns object. This object provides access to the Add-Ins currently installed.

## Syntax

Application.**ApplicationAddIns**() As [ApplicationAddIns](../ApplicationAddIns/ApplicationAddIns.md)

## Property Value

This is a read only property whose value is an [ApplicationAddIns](../ApplicationAddIns/ApplicationAddIns.md).
