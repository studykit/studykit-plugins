# Application.DesignProjectManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the design project manager object.

## Syntax

Application.**DesignProjectManager**() As [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md)

## Property Value

This is a read only property whose value is a [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md).
