# SurfaceBodyProxy.Edges Property

Parent Object: [SurfaceBodyProxy](../SurfaceBodyProxy/SurfaceBodyProxy.md)

## Description

Gets the referenced by this SurfaceBody.

## Syntax

SurfaceBodyProxy.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 4
