# SurfaceBodyProxy.BindTransientKeyToObject Method

Parent Object: [SurfaceBodyProxy](../SurfaceBodyProxy/SurfaceBodyProxy.md)

## Description

Bind the transient key of a subentity on this body to a live object.

## Syntax

SurfaceBodyProxy.**BindTransientKeyToObject**( ***TransientKey*** As Long ) As Object

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TransientKey | Long | Transient key to bind. |

## Version

Introduced in version 9
