# PathEntityProxy.Type Property

Parent Object: [PathEntityProxy](../PathEntityProxy/PathEntityProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

PathEntityProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
