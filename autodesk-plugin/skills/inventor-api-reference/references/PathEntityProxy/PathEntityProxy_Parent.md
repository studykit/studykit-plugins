# PathEntityProxy.Parent Property

Parent Object: [PathEntityProxy](../PathEntityProxy/PathEntityProxy.md)

## Description

Property that returns the parent Path of the entity.

## Syntax

PathEntityProxy.**Parent**() As [Path](../Path/Path.md)

## Property Value

This is a read only property whose value is a [Path](../Path/Path.md).

## Version

Introduced in version 9
