# PathEntityProxy.SketchEntity Property

Parent Object: [PathEntityProxy](../PathEntityProxy/PathEntityProxy.md)

## Description

Property that gets the sketch entity (2D or 3D) this entity was derived from.

## Syntax

PathEntityProxy.**SketchEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 9
