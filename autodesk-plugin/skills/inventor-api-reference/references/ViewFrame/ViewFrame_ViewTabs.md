# ViewFrame.ViewTabs Property

Parent Object: [ViewFrame](../ViewFrame/ViewFrame.md)

## Description

Read-only property that returns the view tabs that are located in this view frame.

## Syntax

ViewFrame.**ViewTabs**() As [ViewTabsEnumerator](../ViewTabsEnumerator/ViewTabsEnumerator.md)

## Property Value

This is a read only property whose value is a [ViewTabsEnumerator](../ViewTabsEnumerator/ViewTabsEnumerator.md).

## Version

Introduced in version 2022
