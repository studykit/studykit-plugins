# ViewFrame.Type Property

Parent Object: [ViewFrame](../ViewFrame/ViewFrame.md)

## Description

Read-only property returning kViewFrameObject indicating this object’s type.

## Syntax

ViewFrame.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2021
