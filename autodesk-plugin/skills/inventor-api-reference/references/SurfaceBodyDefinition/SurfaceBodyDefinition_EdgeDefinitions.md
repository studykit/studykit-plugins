# SurfaceBodyDefinition.EdgeDefinitions Property

Parent Object: [SurfaceBodyDefinition](../SurfaceBodyDefinition/SurfaceBodyDefinition.md)

## Description

Property that returns the collection of EdgeDefinition objects associated with this SurfaceBodyDefinition object.

## Syntax

SurfaceBodyDefinition.**EdgeDefinitions**() As [EdgeDefinitions](../EdgeDefinitions/EdgeDefinitions.md)

## Property Value

This is a read only property whose value is an [EdgeDefinitions](../EdgeDefinitions/EdgeDefinitions.md).
