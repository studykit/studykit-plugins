# ThickenFeatures Object

## Description

The ThickenFeatures collection object represents a collection of objects on a part.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ThickenFeatures/ThickenFeatures_Add.md) | Method that creates a new ThickenFeature. The new created ThickenFeature is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ThickenFeatures/ThickenFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ThickenFeatures/ThickenFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ThickenFeatures/ThickenFeatures_Item.md) | Returns the specified ThickenFeature object from the collection. |
| [Type](../ThickenFeatures/ThickenFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.ThickenFeatures](../PartFeatures/PartFeatures_ThickenFeatures.md), [SheetMetalFeatures.ThickenFeatures](../SheetMetalFeatures/SheetMetalFeatures_ThickenFeatures.md)

## Version

Introduced in version 6
