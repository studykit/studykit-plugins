# ThickenFeatures.Type Property

Parent Object: [ThickenFeatures](../ThickenFeatures/ThickenFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ThickenFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
