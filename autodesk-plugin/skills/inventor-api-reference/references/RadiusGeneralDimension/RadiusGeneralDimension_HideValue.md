# RadiusGeneralDimension.HideValue Property

Parent Object: [RadiusGeneralDimension](../RadiusGeneralDimension/RadiusGeneralDimension.md)

## Description

Gets and sets the HideValue setting.

## Syntax

RadiusGeneralDimension.**HideValue**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
