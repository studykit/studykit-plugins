# RadiusGeneralDimension.Jogged Property

Parent Object: [RadiusGeneralDimension](../RadiusGeneralDimension/RadiusGeneralDimension.md)

## Description

Gets and sets whether the dimension is jogged.

## Syntax

RadiusGeneralDimension.**Jogged**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
