# DimensionText.Origin Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Gets and sets the origin position of the text.

## Syntax

DimensionText.**Origin**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).
