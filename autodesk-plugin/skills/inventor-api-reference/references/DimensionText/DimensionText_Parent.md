# DimensionText.Parent Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Property that returns the parent of the object.

## Syntax

DimensionText.**Parent**() As [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Property Value

This is a read only property whose value is a [DrawingDimension](../DrawingDimension/DrawingDimension.md).

## Version

Introduced in version 9
