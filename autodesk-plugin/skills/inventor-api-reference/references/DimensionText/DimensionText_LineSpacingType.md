# DimensionText.LineSpacingType Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Gets and sets the line spacing method of the text.

## Syntax

DimensionText.**LineSpacingType**() As [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md)

## Property Value

This is a read/write property whose value is a [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md).

## Version

Introduced in version 9
