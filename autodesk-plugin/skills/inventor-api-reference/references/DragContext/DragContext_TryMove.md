# DragContext.TryMove Method

Parent Object: [DragContext](../DragContext/DragContext.md)

## Description

Method that tries to position the object at the exact location given. Returns false if object could not be placed at the exact transform given.

## Syntax

DragContext.**TryMove**( ***NewPosition*** As [Matrix](../Matrix/Matrix.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| NewPosition | [Matrix](../Matrix/Matrix.md) |  |

## Version

Introduced in version 2011
