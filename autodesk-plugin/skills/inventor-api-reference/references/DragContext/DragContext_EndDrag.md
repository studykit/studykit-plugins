# DragContext.EndDrag Method

Parent Object: [DragContext](../DragContext/DragContext.md)

## Description

Method called when the drag is finished. Use this call to commit the drag changes.

## Syntax

DragContext.**EndDrag**()

## Version

Introduced in version 2011
