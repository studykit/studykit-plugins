# FilletConstantRadiusFaceSet.Radius Property

Parent Object: [FilletConstantRadiusFaceSet](../FilletConstantRadiusFaceSet/FilletConstantRadiusFaceSet.md)

## Description

Property that returns the parameter that controls the radius of the fillet. This property will return Nothing if the fillet feature has not been created yet.

## Syntax

FilletConstantRadiusFaceSet.**Radius**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 11
