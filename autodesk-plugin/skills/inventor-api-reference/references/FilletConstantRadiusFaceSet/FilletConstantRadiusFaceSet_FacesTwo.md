# FilletConstantRadiusFaceSet.FacesTwo Property

Parent Object: [FilletConstantRadiusFaceSet](../FilletConstantRadiusFaceSet/FilletConstantRadiusFaceSet.md)

## Description

FaceCollection object that contains the second set of faces for the face fillet feature. This can contain more than one face only if the faces are tangentially connected. The Face.TangentiallyConnectedFaces property can be used to get a collection of faces that are tangent to a given face.

## Syntax

FilletConstantRadiusFaceSet.**FacesTwo**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read only property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 11
