# ChamferNotes.Item Property

Parent Object: [ChamferNotes](../ChamferNotes/ChamferNotes.md)

## Description

Returns the specified ChamferNote object from the collection.

## Syntax

ChamferNotes.**Item**( ***Index*** As Long ) As [ChamferNote](../ChamferNote/ChamferNote.md)

## Property Value

This is a read only property whose value is a [ChamferNote](../ChamferNote/ChamferNote.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
