# ChamferNotes Object

## Description

The ChamferNotes collection object provides access to all of the chamfer notes on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ChamferNotes/ChamferNotes_Add.md) | Method that creates a chamfer note positioned at the specified point on the sheet. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ChamferNotes/ChamferNotes_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ChamferNotes/ChamferNotes_Count.md) | Property that returns the number of items in the collection. |
| [Item](../ChamferNotes/ChamferNotes_Item.md) | Returns the specified ChamferNote object from the collection. |
| [Type](../ChamferNotes/ChamferNotes_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingNotes.ChamferNotes](../DrawingNotes/DrawingNotes_ChamferNotes.md)

## Version

Introduced in version 2010
