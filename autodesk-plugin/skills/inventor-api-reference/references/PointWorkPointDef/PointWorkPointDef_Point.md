# PointWorkPointDef.Point Property

Parent Object: [PointWorkPointDef](../PointWorkPointDef/PointWorkPointDef.md)

## Description

Property that returns the point of a work point that is defined to be coincident to the point. This object can be a Vertex or SketchPoint object.

## Syntax

PointWorkPointDef.**Point**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
