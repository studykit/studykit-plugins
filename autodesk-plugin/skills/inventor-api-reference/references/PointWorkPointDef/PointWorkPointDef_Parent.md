# PointWorkPointDef.Parent Property

Parent Object: [PointWorkPointDef](../PointWorkPointDef/PointWorkPointDef.md)

## Description

Property returning the parent WorkPoint object.

## Syntax

PointWorkPointDef.**Parent**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 4
