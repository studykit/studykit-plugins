# PointWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point defined to be coincident to a point.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointWorkPointDef/PointWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../PointWorkPointDef/PointWorkPointDef_Parent.md) | Property returning the parent WorkPoint object. |
| [Point](../PointWorkPointDef/PointWorkPointDef_Point.md) | Property that returns the point of a work point that is defined to be coincident to the point. This object can be a Vertex or SketchPoint object. |
| [Type](../PointWorkPointDef/PointWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
