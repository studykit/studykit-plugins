# SurfaceTextureGBDefinition.SurfaceTextureType Property

Parent Object: [SurfaceTextureGBDefinition](../SurfaceTextureGBDefinition/SurfaceTextureGBDefinition.md)

## Description

Gets and sets the surface type for the symbol.

## Syntax

SurfaceTextureGBDefinition.**SurfaceTextureType**() As [SurfaceTextureTypeEnum](../SurfaceTextureTypeEnum.md)

## Property Value

This is a read/write property whose value is a [SurfaceTextureTypeEnum](../SurfaceTextureTypeEnum.md).

## Version

Introduced in version 2016
