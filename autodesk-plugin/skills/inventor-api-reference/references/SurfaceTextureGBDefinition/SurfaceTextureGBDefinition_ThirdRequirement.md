# SurfaceTextureGBDefinition.ThirdRequirement Property

Parent Object: [SurfaceTextureGBDefinition](../SurfaceTextureGBDefinition/SurfaceTextureGBDefinition.md)

## Description

Gets and sets the third requirement value for the surface.

## Syntax

SurfaceTextureGBDefinition.**ThirdRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
