# ProfileEntityProxy.NativeObject Property

Parent Object: [ProfileEntityProxy](../ProfileEntityProxy/ProfileEntityProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ProfileEntityProxy.**NativeObject**() As [ProfileEntity](../ProfileEntity/ProfileEntity.md)

## Property Value

This is a read only property whose value is a [ProfileEntity](../ProfileEntity/ProfileEntity.md).

## Version

Introduced in version 6
