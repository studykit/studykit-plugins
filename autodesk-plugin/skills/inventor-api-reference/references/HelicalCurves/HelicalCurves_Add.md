# HelicalCurves.Add Method

Parent Object: [HelicalCurves](../HelicalCurves/HelicalCurves.md)

## Description

Creates a helical curve.

## Syntax

HelicalCurves.**Add**( ***HelicalCurveDefinition*** As [HelicalCurveDefinition](../HelicalCurveDefinition/HelicalCurveDefinition.md) ) As [HelicalCurve](../HelicalCurve/HelicalCurve.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| HelicalCurveDefinition | [HelicalCurveDefinition](../HelicalCurveDefinition/HelicalCurveDefinition.md) | Input HelicalCurveDefinition object that defines the inputs of the helical curve. |

## Version

Introduced in version 2019
