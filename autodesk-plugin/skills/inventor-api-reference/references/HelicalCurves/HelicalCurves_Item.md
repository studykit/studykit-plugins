# HelicalCurves.Item Property

Parent Object: [HelicalCurves](../HelicalCurves/HelicalCurves.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

HelicalCurves.**Item**( ***Index*** As Variant ) As [HelicalCurve](../HelicalCurve/HelicalCurve.md)

## Property Value

This is a read only property whose value is a [HelicalCurve](../HelicalCurve/HelicalCurve.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the helical curve to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the helical curve name. If an out of range index or a name of a non-existent helical curve is provided, an error occurs. |

## Version

Introduced in version 2019
