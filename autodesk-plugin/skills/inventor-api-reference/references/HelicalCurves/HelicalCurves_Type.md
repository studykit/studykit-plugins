# HelicalCurves.Type Property

Parent Object: [HelicalCurves](../HelicalCurves/HelicalCurves.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

HelicalCurves.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2019
