# FlushiMateDefinition.MatchList Property

Parent Object: [FlushiMateDefinition](../FlushiMateDefinition/FlushiMateDefinition.md)

## Description

Gets and sets an array of strings that specifies the priority order for iMate definition matches.

## Syntax

FlushiMateDefinition.**MatchList**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 11
