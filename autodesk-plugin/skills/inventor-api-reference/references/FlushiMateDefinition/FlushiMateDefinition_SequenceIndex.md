# FlushiMateDefinition.SequenceIndex Property

Parent Object: [FlushiMateDefinition](../FlushiMateDefinition/FlushiMateDefinition.md)

## Description

Gets the index of this iMateDefinition.

## Syntax

FlushiMateDefinition.**SequenceIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 6
