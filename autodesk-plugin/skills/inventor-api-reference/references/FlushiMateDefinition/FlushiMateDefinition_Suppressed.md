# FlushiMateDefinition.Suppressed Property

Parent Object: [FlushiMateDefinition](../FlushiMateDefinition/FlushiMateDefinition.md)

## Description

Gets the boolean flag that indicates whether this iMateDefinition is suppressed or not.

## Syntax

FlushiMateDefinition.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
