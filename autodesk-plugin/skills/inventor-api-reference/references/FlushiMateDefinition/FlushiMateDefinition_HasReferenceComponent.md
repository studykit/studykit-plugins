# FlushiMateDefinition.HasReferenceComponent Property

Parent Object: [FlushiMateDefinition](../FlushiMateDefinition/FlushiMateDefinition.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

FlushiMateDefinition.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
