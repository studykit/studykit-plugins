# FlushiMateDefinition.Offset Property

Parent Object: [FlushiMateDefinition](../FlushiMateDefinition/FlushiMateDefinition.md)

## Description

Property that returns the Parameter object that controls the offset of the flush iMate definition.

## Syntax

FlushiMateDefinition.**Offset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
