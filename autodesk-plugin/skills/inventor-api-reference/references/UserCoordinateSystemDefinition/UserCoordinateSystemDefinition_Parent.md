# UserCoordinateSystemDefinition.Parent Property

Parent Object: [UserCoordinateSystemDefinition](../UserCoordinateSystemDefinition/UserCoordinateSystemDefinition.md)

## Description

Property that returns the parent UserCoordinateSystem of this definition. In the case where this is a newly created UserCoordinateSystemDefinition object, this property will return Nothing because there is not a logical parent for the object.

## Syntax

UserCoordinateSystemDefinition.**Parent**() As [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Property Value

This is a read only property whose value is a [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md).

## Version

Introduced in version 2011
