# FinishParameter.DisplayFormat Property

Parent Object: [FinishParameter](../FinishParameter/FinishParameter.md)

## Description

Gets and sets the display format for the parameter.

## Syntax

FinishParameter.**DisplayFormat**() As [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md)

## Property Value

This is a read/write property whose value is a [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md).

## Version

Introduced in version 2024
