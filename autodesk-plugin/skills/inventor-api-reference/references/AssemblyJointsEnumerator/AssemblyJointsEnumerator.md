# AssemblyJointsEnumerator Object

## Description

Provides access to the joints acting directly on a particular occurrence. This object may also be used wherever a snapshot of a set of assembly joints needs to be returned (this is a snapshot taken at the time this object is retrieved).

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../AssemblyJointsEnumerator/AssemblyJointsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../AssemblyJointsEnumerator/AssemblyJointsEnumerator_Item.md) | Method that returns the specified AssemblyJoint object from the collection. This is the default method of the AssemblyJointsEnumerator collection object. |
| [Type](../AssemblyJointsEnumerator/AssemblyJointsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ComponentOccurrence.Joints](../ComponentOccurrence/ComponentOccurrence_Joints.md), [ComponentOccurrenceProxy.Joints](../ComponentOccurrenceProxy/ComponentOccurrenceProxy_Joints.md), [RigidBodyJoint.Joints](RigidBodyJoint_Joints.md)
