# ClientResourceMaps.Item Property

Parent Object: [ClientResourceMaps](../ClientResourceMaps/ClientResourceMaps.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

ClientResourceMaps.**Item**( ***Index*** As Long ) As [ClientResourceMap](../ClientResourceMap/ClientResourceMap.md)

## Property Value

This is a read only property whose value is a [ClientResourceMap](../ClientResourceMap/ClientResourceMap.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the ClientNodeIconResourceMap to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2022
