# CosmeticBendFeature.Adaptive Property

Parent Object: [CosmeticBendFeature](../CosmeticBendFeature/CosmeticBendFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

CosmeticBendFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
