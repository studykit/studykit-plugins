# CornerRoundFeatures.GetEdgesFromFeature Method

Parent Object: [CornerRoundFeatures](../CornerRoundFeatures/CornerRoundFeatures.md)

## Description

Method that gets the edges from a part feature that can be used to create corner round feature.

## Syntax

CornerRoundFeatures.**GetEdgesFromFeature**( ***Feature*** As [PartFeature](../PartFeature/PartFeature.md) ) As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Feature | [PartFeature](../PartFeature/PartFeature.md) | Input PartFeature object that specifies a part feature to get its edges for creating corner round feature. |

## Version

Introduced in version 2025
