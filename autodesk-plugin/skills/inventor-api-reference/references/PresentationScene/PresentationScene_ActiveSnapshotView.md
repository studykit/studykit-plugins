# PresentationScene.ActiveSnapshotView Property

Parent Object: [PresentationScene](../PresentationScene/PresentationScene.md)

## Description

Read-only property that returns active presentation snapshot view.

## Syntax

PresentationScene.**ActiveSnapshotView**() As [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Property Value

This is a read only property whose value is a [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md).

## Version

Introduced in version 2018
