# PresentationScene.GraphicsDataSetsCollection Property

Parent Object: [PresentationScene](../PresentationScene/PresentationScene.md)

## Description

Read-only property that gets the GraphicsDataSetsCollection.

## Syntax

PresentationScene.**GraphicsDataSetsCollection**() As [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md).

## Version

Introduced in version 2018
