# ConcentricConstraintProxy Object

Derived from: [ConcentricConstraint](../ConcentricConstraint/ConcentricConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../ConcentricConstraintProxy/ConcentricConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../ConcentricConstraintProxy/ConcentricConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ConcentricConstraintProxy/ConcentricConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../ConcentricConstraintProxy/ConcentricConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [CenterPointConstraint](../ConcentricConstraintProxy/ConcentricConstraintProxy_CenterPointConstraint.md) | Gets a Boolean indicating if the constraint is between the arc and its centerpoint or not. |
| [ContainingOccurrence](../ConcentricConstraintProxy/ConcentricConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../ConcentricConstraintProxy/ConcentricConstraintProxy_Deletable.md) | Indicates whether this object is deletable. |
| [EntityOne](../ConcentricConstraintProxy/ConcentricConstraintProxy_EntityOne.md) | Gets the sketch entity being constrained. |
| [EntityTwo](../ConcentricConstraintProxy/ConcentricConstraintProxy_EntityTwo.md) | Gets the sketch entity being constrained. |
| [NativeObject](../ConcentricConstraintProxy/ConcentricConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../ConcentricConstraintProxy/ConcentricConstraintProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Type](../ConcentricConstraintProxy/ConcentricConstraintProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 6
