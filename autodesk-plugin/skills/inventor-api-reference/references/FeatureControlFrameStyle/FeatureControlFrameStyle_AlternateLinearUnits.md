# FeatureControlFrameStyle.AlternateLinearUnits Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the units of length measure used for alternate units. This property is used only when the AlternateUnitsDisplay property is set to true.

## Syntax

FeatureControlFrameStyle.**AlternateLinearUnits**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).

## Version

Introduced in version 2009
