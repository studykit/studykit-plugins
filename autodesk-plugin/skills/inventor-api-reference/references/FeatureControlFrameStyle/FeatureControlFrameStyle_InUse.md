# FeatureControlFrameStyle.InUse Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that indicates if this style is in use.

## Syntax

FeatureControlFrameStyle.**InUse**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
