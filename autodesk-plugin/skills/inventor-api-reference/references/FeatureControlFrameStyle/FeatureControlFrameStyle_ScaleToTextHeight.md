# FeatureControlFrameStyle.ScaleToTextHeight Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to define the symbol size by text height.

## Syntax

FeatureControlFrameStyle.**ScaleToTextHeight**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
