# FeatureControlFrameStyle.AvailableGeometricCharacteristics Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the geometric characteristics to be made available in the feature control frame creation and edit dialog. A combination of enum values can be provided.

## Syntax

FeatureControlFrameStyle.**AvailableGeometricCharacteristics**() As [GeometricCharacteristicEnum](../GeometricCharacteristicEnum.md)

## Property Value

This is a read/write property whose value is a [GeometricCharacteristicEnum](../GeometricCharacteristicEnum.md).

## Version

Introduced in version 2009
