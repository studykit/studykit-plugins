# FeatureControlFrameStyle.SaveToGlobal Method

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

FeatureControlFrameStyle.**SaveToGlobal**()

## Version

Introduced in version 2009
