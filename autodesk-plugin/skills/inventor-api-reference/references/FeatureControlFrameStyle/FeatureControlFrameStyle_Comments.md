# FeatureControlFrameStyle.Comments Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Gets and sets comments associated with the style.

## Syntax

FeatureControlFrameStyle.**Comments**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
