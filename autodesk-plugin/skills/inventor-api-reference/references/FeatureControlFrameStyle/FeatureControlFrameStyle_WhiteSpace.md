# FeatureControlFrameStyle.WhiteSpace Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the space before and after text in the tolerance and datum cells.

## Syntax

FeatureControlFrameStyle.**WhiteSpace**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
