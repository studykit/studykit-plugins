# FeatureControlFrameStyle.AlignCellsVertically Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to add spaces to short cells so that pairs of cells align vertically.

## Syntax

FeatureControlFrameStyle.**AlignCellsVertically**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
