# FeatureControlFrameStyle.Size Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the size of the feature control frame in linear units. This value is used only if the ScaleToTextHeight property is set to False.

## Syntax

FeatureControlFrameStyle.**Size**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
