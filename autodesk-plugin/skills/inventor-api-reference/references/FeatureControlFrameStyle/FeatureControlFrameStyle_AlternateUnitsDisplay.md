# FeatureControlFrameStyle.AlternateUnitsDisplay Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to display alternate units.

## Syntax

FeatureControlFrameStyle.**AlternateUnitsDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
