# FeatureControlFrameStyle.AlternateTrailingZeroDisplay Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to display the trailing zeroes for alternate units. This property is used only when the AlternateUnitsDisplay property is set to true.

## Syntax

FeatureControlFrameStyle.**AlternateTrailingZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
