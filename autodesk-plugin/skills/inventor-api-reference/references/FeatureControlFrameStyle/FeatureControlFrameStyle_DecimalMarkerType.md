# FeatureControlFrameStyle.DecimalMarkerType Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the character to use as a decimal marker for primary units. Valid values are kPeriodDecimalMarker and kCommaDecimalMarker.

## Syntax

FeatureControlFrameStyle.**DecimalMarkerType**() As [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md).

## Version

Introduced in version 2009
