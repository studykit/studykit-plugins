# FeatureControlFrameStyle.AttachLeaderToCorner Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether the leader terminates on the corner of the frame or at the midpoint of a vertical leg of the frame.

## Syntax

FeatureControlFrameStyle.**AttachLeaderToCorner**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
