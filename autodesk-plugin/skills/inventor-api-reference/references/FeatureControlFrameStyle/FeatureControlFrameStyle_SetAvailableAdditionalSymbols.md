# FeatureControlFrameStyle.SetAvailableAdditionalSymbols Method

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Sets the array of FeatureControlFrameAdditionalSymbolsEnum to specify additional symbols to be made available in the feature control frame creation and edit dialog.

## Syntax

FeatureControlFrameStyle.**SetAvailableAdditionalSymbols**( ***AvailaleAdditionalSymbols***() As [FeatureControlFrameAdditionalSymbolsEnum](../FeatureControlFrameAdditionalSymbolsEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| AvailaleAdditionalSymbols | [FeatureControlFrameAdditionalSymbolsEnum](../FeatureControlFrameAdditionalSymbolsEnum.md) | Input array of FeatureControlFrameAdditionalSymbolsEnum to specify additional symbols to be made available in the feature control frame creation and edit dialog. |

## Version

Introduced in version 2016
