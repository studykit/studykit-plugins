# FeatureControlFrameStyle.LeaderStyle Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the leader style used for a feature control frame symbol.

## Syntax

FeatureControlFrameStyle.**LeaderStyle**() As [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Property Value

This is a read/write property whose value is a [LeaderStyle](../LeaderStyle/LeaderStyle.md).

## Version

Introduced in version 2009
