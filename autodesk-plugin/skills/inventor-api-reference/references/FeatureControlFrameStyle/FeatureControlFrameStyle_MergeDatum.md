# FeatureControlFrameStyle.MergeDatum Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to merge datum cells when the data is the same.

## Syntax

FeatureControlFrameStyle.**MergeDatum**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
