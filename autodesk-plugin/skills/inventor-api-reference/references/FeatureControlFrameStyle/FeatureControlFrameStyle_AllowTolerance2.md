# FeatureControlFrameStyle.AllowTolerance2 Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to provide the ability to add a second tolerance in the feature control frame dialog.

## Syntax

FeatureControlFrameStyle.**AllowTolerance2**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
