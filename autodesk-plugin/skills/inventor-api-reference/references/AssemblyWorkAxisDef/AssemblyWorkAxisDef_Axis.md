# AssemblyWorkAxisDef.Axis Property

Parent Object: [AssemblyWorkAxisDef](../AssemblyWorkAxisDef/AssemblyWorkAxisDef.md)

## Description

Property that returns the X-axis vector of the work axis.

## Syntax

AssemblyWorkAxisDef.**Axis**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 7
