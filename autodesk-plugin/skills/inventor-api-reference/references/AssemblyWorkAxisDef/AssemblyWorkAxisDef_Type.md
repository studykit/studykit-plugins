# AssemblyWorkAxisDef.Type Property

Parent Object: [AssemblyWorkAxisDef](../AssemblyWorkAxisDef/AssemblyWorkAxisDef.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

AssemblyWorkAxisDef.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
