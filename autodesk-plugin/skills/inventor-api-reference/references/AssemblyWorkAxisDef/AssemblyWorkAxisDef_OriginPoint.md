# AssemblyWorkAxisDef.OriginPoint Property

Parent Object: [AssemblyWorkAxisDef](../AssemblyWorkAxisDef/AssemblyWorkAxisDef.md)

## Description

Property that returns the origin point of the work axis.

## Syntax

AssemblyWorkAxisDef.**OriginPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 7
