# AssemblyWorkAxisDef.Constraints Property

Parent Object: [AssemblyWorkAxisDef](../AssemblyWorkAxisDef/AssemblyWorkAxisDef.md)

## Description

Property that returns the collection of constraints that are acting on the work axis. This returns all of the constraints, including those returned by the DefinitionConstraints property.

## Syntax

AssemblyWorkAxisDef.**Constraints**() As [AssemblyConstraintsEnumerator](../AssemblyConstraintsEnumerator/AssemblyConstraintsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssemblyConstraintsEnumerator](../AssemblyConstraintsEnumerator/AssemblyConstraintsEnumerator.md).

## Version

Introduced in version 6
