# FlangeEdgeSet Object

## Description

Flange edge set Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../FlangeEdgeSet/FlangeEdgeSet_Delete.md) | Method that deletes the flange edge set. |
| [SetCenteredWidthExtent](../FlangeEdgeSet/FlangeEdgeSet_SetCenteredWidthExtent.md) | Method that sets the width extent of the specified physical flange to define a flange that’s centered along the edge and has a defined width. |
| [SetEdgeWidthExtent](../FlangeEdgeSet/FlangeEdgeSet_SetEdgeWidthExtent.md) | Method that changes the flange edge set object to define a flange whose width is defined by the length of the input edge(s). |
| [SetFromToWidthExtent](../FlangeEdgeSet/FlangeEdgeSet_SetFromToWidthExtent.md) | Method that sets the width extent of the specified physical flange to define a flange whose width is defined as being between two entities. |
| [SetOffsetdWidthExtent](../FlangeEdgeSet/FlangeEdgeSet_SetOffsetdWidthExtent.md) | Method that sets the width extent of the specified physical flange to define a flange whose width is defined with respect to two entities. |
| [SetWidthOffsetWidthExtent](../FlangeEdgeSet/FlangeEdgeSet_SetWidthOffsetWidthExtent.md) | Method that sets the width extent of the specified physical flange to define a flange whose width is defined relative to another entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FlangeEdgeSet/FlangeEdgeSet_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Edges](../FlangeEdgeSet/FlangeEdgeSet_Edges.md) | Read-write property that gets and sets the collection of edges in the edge set. |
| [Type](../FlangeEdgeSet/FlangeEdgeSet_Type.md) | Gets the constant that indicates the type of this object. |
| [WidthExtent](../FlangeEdgeSet/FlangeEdgeSet_WidthExtent.md) | Read-only property that returns width extent of this FlangeEdgeSet. |
| [WidthExtentType](../FlangeEdgeSet/FlangeEdgeSet_WidthExtentType.md) | Read-only property that returns width extent type of this FlangeEdgeSet. |
