# FlangeEdgeSet.WidthExtent Property

Parent Object: [FlangeEdgeSet](../FlangeEdgeSet/FlangeEdgeSet.md)

## Description

Read-only property that returns width extent of this FlangeEdgeSet.

## Syntax

FlangeEdgeSet.**WidthExtent**() As [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md).

## Version

Introduced in version 2025
