# iMateDefinition.Type Property

Parent Object: [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iMateDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
