# GeneralNote.VerticalJustification Property

Parent Object: [GeneralNote](../GeneralNote/GeneralNote.md)

## Description

Gets/Sets vertical justification of the DrawingNote.

## Syntax

GeneralNote.**VerticalJustification**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 10
