# GeneralNote.BackgroundColor Property

Parent Object: [GeneralNote](../GeneralNote/GeneralNote.md)

## Description

Gets/Sets the BackgroundColor associated with the GeneralNote.

## Syntax

GeneralNote.**BackgroundColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2017
