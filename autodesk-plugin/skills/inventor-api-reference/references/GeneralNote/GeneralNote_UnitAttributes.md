# GeneralNote.UnitAttributes Property

Parent Object: [GeneralNote](../GeneralNote/GeneralNote.md)

## Description

Get the unit attributes associated with the GeneralNote.

## Syntax

GeneralNote.**UnitAttributes**() As [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Property Value

This is a read only property whose value is a [UnitAttributes](../UnitAttributes/UnitAttributes.md).

## Version

Introduced in version 2016
