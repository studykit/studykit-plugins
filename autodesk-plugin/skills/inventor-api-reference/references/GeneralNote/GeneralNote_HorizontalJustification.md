# GeneralNote.HorizontalJustification Property

Parent Object: [GeneralNote](../GeneralNote/GeneralNote.md)

## Description

Gets/Sets horizontal justification of the DrawingNote.

## Syntax

GeneralNote.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 10
