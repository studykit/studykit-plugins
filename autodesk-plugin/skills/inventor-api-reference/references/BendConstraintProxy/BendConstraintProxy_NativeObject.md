# BendConstraintProxy.NativeObject Property

Parent Object: [BendConstraintProxy](../BendConstraintProxy/BendConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

BendConstraintProxy.**NativeObject**() As [BendConstraint](../BendConstraint/BendConstraint.md)

## Property Value

This is a read only property whose value is a [BendConstraint](../BendConstraint/BendConstraint.md).

## Version

Introduced in version 6
