# Layer.Visible Property

Parent Object: [Layer](../Layer/Layer.md)

## Description

Property that gets and sets whether the layer is turned on or off.

## Syntax

Layer.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
