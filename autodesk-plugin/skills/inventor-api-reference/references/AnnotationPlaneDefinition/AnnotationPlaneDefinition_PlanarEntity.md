# AnnotationPlaneDefinition.PlanarEntity Property

Parent Object: [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Description

Read-only property that returns the planar entity the annotation plane is associated with. This property can return Nothing in the case it’s not associated with a planar entity.

## Syntax

AnnotationPlaneDefinition.**PlanarEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
