# AnnotationPlaneDefinition.XAxis Property

Parent Object: [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Description

Read-only property that returns the orientation of the x axis of the annotation plane in model space.

## Syntax

AnnotationPlaneDefinition.**XAxis**() As [Line](../Line/Line.md)

## Property Value

This is a read only property whose value is a [Line](../Line/Line.md).

## Version

Introduced in version 2018
