# ExpressionList.Parent Property

Parent Object: [ExpressionList](../ExpressionList/ExpressionList.md)

## Description

Property that returns the parent Parameter object.

## Syntax

ExpressionList.**Parent**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2011
