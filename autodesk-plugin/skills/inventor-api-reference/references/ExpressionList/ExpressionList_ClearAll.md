# ExpressionList.ClearAll Method

Parent Object: [ExpressionList](../ExpressionList/ExpressionList.md)

## Description

Method that removes all items from the list.

## Syntax

ExpressionList.**ClearAll**()

## Version

Introduced in version 2011
