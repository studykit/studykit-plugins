# iFeatureWorkPlaneInput.Type Property

Parent Object: [iFeatureWorkPlaneInput](../iFeatureWorkPlaneInput/iFeatureWorkPlaneInput.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iFeatureWorkPlaneInput.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version
