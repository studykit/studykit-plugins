# iFeatureWorkPlaneInput.FlipOption Property

Parent Object: [iFeatureWorkPlaneInput](../iFeatureWorkPlaneInput/iFeatureWorkPlaneInput.md)

## Description

Gets and sets whether direction of the selection needs to be flipped.

## Syntax

iFeatureWorkPlaneInput.**FlipOption**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
