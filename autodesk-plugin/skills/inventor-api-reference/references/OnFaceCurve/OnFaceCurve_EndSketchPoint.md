# OnFaceCurve.EndSketchPoint Property

Parent Object: [OnFaceCurve](../OnFaceCurve/OnFaceCurve.md)

## Description

Read-only property that returns the SketchPoint3D that defines the position of the end of the curve.

## Syntax

OnFaceCurve.**EndSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 2017
