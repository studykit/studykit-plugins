# OnFaceCurve.Faces Property

Parent Object: [OnFaceCurve](../OnFaceCurve/OnFaceCurve.md)

## Description

Read-only property that returns the NameValueMap object that contains the faces in sequence that the sketch splines on.

## Syntax

OnFaceCurve.**Faces**() As [NameValueMap](../NameValueMap/NameValueMap.md)

## Property Value

This is a read only property whose value is a [NameValueMap](../NameValueMap/NameValueMap.md).

## Version

Introduced in version 2017
