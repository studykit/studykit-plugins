# AssemblyOptions.ZoomTargetComponentWithiMate Property

Parent Object: [AssemblyOptions](../AssemblyOptions/AssemblyOptions.md)

## Description

Gets/Sets the default zoom behavior for the graphics window when placing components with iMates.

## Syntax

AssemblyOptions.**ZoomTargetComponentWithiMate**() As [ZoomTargetComponentWithiMateEnum](../ZoomTargetComponentWithiMateEnum.md)

## Property Value

This is a read/write property whose value is a [ZoomTargetComponentWithiMateEnum](../ZoomTargetComponentWithiMateEnum.md).

## Version

Introduced in version 11
