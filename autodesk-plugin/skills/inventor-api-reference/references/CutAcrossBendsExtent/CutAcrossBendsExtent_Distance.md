# CutAcrossBendsExtent.Distance Property

Parent Object: [CutAcrossBendsExtent](../CutAcrossBendsExtent/CutAcrossBendsExtent.md)

## Description

Property that returns the parameter controlling the depth of the cut.

## Syntax

CutAcrossBendsExtent.**Distance**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
