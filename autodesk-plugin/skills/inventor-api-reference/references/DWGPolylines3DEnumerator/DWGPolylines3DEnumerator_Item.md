# DWGPolylines3DEnumerator.Item Property

Parent Object: [DWGPolylines3DEnumerator](../DWGPolylines3DEnumerator/DWGPolylines3DEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGPolylines3DEnumerator.**Item**( ***Index*** As Variant ) As [DWGPolyline3D](../DWGPolyline3D/DWGPolyline3D.md)

## Property Value

This is a read only property whose value is a [DWGPolyline3D](../DWGPolyline3D/DWGPolyline3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
