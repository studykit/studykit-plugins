# DWGPolylines3DEnumerator Object

## Description

DWGPolylines3DEnumerator Collection Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DWGPolylines3DEnumerator/DWGPolylines3DEnumerator_Application.md) | Gets the root Application object. | [Count](../DWGPolylines3DEnumerator/DWGPolylines3DEnumerator_Count.md) | Gets the number of items in this collection. |
| [Item](../DWGPolylines3DEnumerator/DWGPolylines3DEnumerator_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
