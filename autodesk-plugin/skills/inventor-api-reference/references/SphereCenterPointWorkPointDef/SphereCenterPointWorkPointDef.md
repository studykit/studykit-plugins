# SphereCenterPointWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point defined to be at the center of a sphere.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SphereCenterPointWorkPointDef/SphereCenterPointWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Face](../SphereCenterPointWorkPointDef/SphereCenterPointWorkPointDef_Face.md) | Gets the face object whose geometry is a Sphere surface, the center point of which defines the position of the work point. |
| [Parent](../SphereCenterPointWorkPointDef/SphereCenterPointWorkPointDef_Parent.md) | Read-only property that returns the parent WorkPoint object. |
| [Type](../SphereCenterPointWorkPointDef/SphereCenterPointWorkPointDef_Type.md) | Read-only property that returns kSphereCenterPointWorkPointDefObject indicating the type of object. |

## Version

Introduced in version 2014
