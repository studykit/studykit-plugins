# Sketches3DEnumerator.Type Property

Parent Object: [Sketches3DEnumerator](../Sketches3DEnumerator/Sketches3DEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

Sketches3DEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
