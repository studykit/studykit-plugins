# Sketches3DEnumerator Object

## Description

The Sketches3DEnumerator allows iteration of available Sketch3D objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../Sketches3DEnumerator/Sketches3DEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../Sketches3DEnumerator/Sketches3DEnumerator_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../Sketches3DEnumerator/Sketches3DEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DerivedPartComponent.Sketches3D](../DerivedPartComponent/DerivedPartComponent_Sketches3D.md), [DerivedPartComponentProxy.Sketches3D](../DerivedPartComponentProxy/DerivedPartComponentProxy_Sketches3D.md)

## Version

Introduced in version 2009
