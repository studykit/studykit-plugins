# Sketches3DEnumerator.Item Property

Parent Object: [Sketches3DEnumerator](../Sketches3DEnumerator/Sketches3DEnumerator.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

Sketches3DEnumerator.**Item**( ***Index*** As Long ) As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long |  |

## Version

Introduced in version 2009
