# ComponentOccurrence.DefinitionDocumentType Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Gets the type of the Document whose occurrence this is.

## Syntax

ComponentOccurrence.**DefinitionDocumentType**() As [DocumentTypeEnum](../DocumentTypeEnum.md)

## Property Value

This is a read only property whose value is a [DocumentTypeEnum](../DocumentTypeEnum.md).
