# ComponentOccurrence.IsSubstituteOccurrence Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Property that returns whether this occurrence references a substitute part in the context of a substitute model state.

## Syntax

ComponentOccurrence.**IsSubstituteOccurrence**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
