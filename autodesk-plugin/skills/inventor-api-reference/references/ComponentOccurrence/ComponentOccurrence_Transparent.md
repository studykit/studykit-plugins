# ComponentOccurrence.Transparent Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

'Put' is Inventor Only: Gets/Sets the Boolean flag that specifies whether this Occurrence is transparent or not.

## Syntax

ComponentOccurrence.**Transparent**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
