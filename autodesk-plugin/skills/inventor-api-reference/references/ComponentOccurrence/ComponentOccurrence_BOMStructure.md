# ComponentOccurrence.BOMStructure Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Gets and sets the BOM structure override.

## Syntax

ComponentOccurrence.**BOMStructure**() As [BOMStructureEnum](../BOMStructureEnum.md)

## Property Value

This is a read/write property whose value is a [BOMStructureEnum](../BOMStructureEnum.md).

## Version

Introduced in version 10
