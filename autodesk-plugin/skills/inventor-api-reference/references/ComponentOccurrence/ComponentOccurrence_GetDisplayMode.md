# ComponentOccurrence.GetDisplayMode Method

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Gets the display mode type - default or override.

## Syntax

ComponentOccurrence.**GetDisplayMode**( ***DisplayModeSourceType*** As [DisplayModeSourceTypeEnum](../DisplayModeSourceTypeEnum.md) ) As [DisplayModeEnum](../DisplayModeEnum.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DisplayModeSourceType | [DisplayModeSourceTypeEnum](../DisplayModeSourceTypeEnum.md) | Enum indicating the display mode. |

## Version

Introduced in version 10
