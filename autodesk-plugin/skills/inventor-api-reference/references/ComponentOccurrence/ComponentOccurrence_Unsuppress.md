# ComponentOccurrence.Unsuppress Method

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Method that unsuppresses the occurrence.

## Syntax

ComponentOccurrence.**Unsuppress**()

## Version

Introduced in version 11
