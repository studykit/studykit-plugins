# ComponentOccurrence.OverrideOpacity Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Gets and sets the opacity of an occurrence.

## Syntax

ComponentOccurrence.**OverrideOpacity**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
