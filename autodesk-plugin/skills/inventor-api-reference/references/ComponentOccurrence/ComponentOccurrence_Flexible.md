# ComponentOccurrence.Flexible Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Only works for assembly occurrences. Gets and sets the Boolean flag that specifies whether the assembly is flexible.

## Syntax

ComponentOccurrence.**Flexible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
