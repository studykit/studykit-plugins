# CircularOccurrencePattern.Visible Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Gets/Sets the Visible property for the pattern.

## Syntax

CircularOccurrencePattern.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
