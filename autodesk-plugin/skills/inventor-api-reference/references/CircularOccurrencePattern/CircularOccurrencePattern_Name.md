# CircularOccurrencePattern.Name Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Gets/Sets the name for the pattern.

## Syntax

CircularOccurrencePattern.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 6
