# CircularOccurrencePattern.AngleOffset Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Property returning the Parameter object that controls the angle offset between pattern instances.

## Syntax

CircularOccurrencePattern.**AngleOffset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
