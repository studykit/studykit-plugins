# CircularOccurrencePattern.PatternElement Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Property that returns the pattern element this occurrence pattern represents. In the case where this occurrence pattern is not part of a parent pattern this property returns Nothing. The IsPatternElement property can be used to check if this occurrence pattern is part of a parent pattern.

## Syntax

CircularOccurrencePattern.**PatternElement**() As [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Property Value

This is a read only property whose value is an [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md).

## Version

Introduced in version 2010
