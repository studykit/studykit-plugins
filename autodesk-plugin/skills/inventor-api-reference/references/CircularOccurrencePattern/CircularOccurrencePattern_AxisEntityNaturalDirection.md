# CircularOccurrencePattern.AxisEntityNaturalDirection Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Gets/Sets whether the axis direction is in the natural direction of the axis entity or not.

## Syntax

CircularOccurrencePattern.**AxisEntityNaturalDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
