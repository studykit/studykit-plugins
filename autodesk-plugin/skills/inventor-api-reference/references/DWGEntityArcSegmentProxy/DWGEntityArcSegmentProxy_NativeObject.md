# DWGEntityArcSegmentProxy.NativeObject Property

Parent Object: [DWGEntityArcSegmentProxy](../DWGEntityArcSegmentProxy/DWGEntityArcSegmentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGEntityArcSegmentProxy.**NativeObject**() As [DWGEntityArcSegment](../DWGEntityArcSegment/DWGEntityArcSegment.md)

## Property Value

This is a read only property whose value is a [DWGEntityArcSegment](../DWGEntityArcSegment/DWGEntityArcSegment.md).

## Version

Introduced in version 2018
