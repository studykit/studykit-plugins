# SheetFormats.Type Property

Parent Object: [SheetFormats](../SheetFormats/SheetFormats.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SheetFormats.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
