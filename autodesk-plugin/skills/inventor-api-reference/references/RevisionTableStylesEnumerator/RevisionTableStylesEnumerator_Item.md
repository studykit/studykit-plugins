# RevisionTableStylesEnumerator.Item Property

Parent Object: [RevisionTableStylesEnumerator](../RevisionTableStylesEnumerator/RevisionTableStylesEnumerator.md)

## Description

Returns the specified RevisionTableStyle object from the collection.

## Syntax

RevisionTableStylesEnumerator.**Item**( ***Index*** As Variant ) As [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Property Value

This is a read only property whose value is a [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the RevisionTableStyle to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the RevisionTableStyle name. If an out of range index or a name of a non\-existent RevisionTableStyle is provided, an error will occur. |

## Version

Introduced in version 2011
