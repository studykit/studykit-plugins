# SketchSplines3D Object

## Description

The SketchSplines3D object provides access to all the objects in a sketch and provides methods to create additional sketch splines. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchSplines3D/SketchSplines3D_Add.md) | Method that creates a sketch spline through the set of input points. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchSplines3D/SketchSplines3D_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchSplines3D/SketchSplines3D_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchSplines3D/SketchSplines3D_Item.md) | Returns the specified SketchSpline3D object from the collection. |
| [Type](../SketchSplines3D/SketchSplines3D_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Sketch3D.SketchSplines3D](../Sketch3D/Sketch3D_SketchSplines3D.md), [Sketch3DProxy.SketchSplines3D](../Sketch3DProxy/Sketch3DProxy_SketchSplines3D.md)

## Version

Introduced in version 6
