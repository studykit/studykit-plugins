# GeneralDimensionTypeEnum Enumerator

## Description

Enum indicating whether the dimension is a true dimension or a projected dimension.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kProjectedGeneralDimension | 72194 | General dimensions projected to the sheet plane. |
| kTrueGeneralDimension | 72193 | General dimensions drawn on model planes. |

## Version

Introduced in version 2008
