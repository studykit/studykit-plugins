# LoftSectionDimension.IsDrivingSection Property

Parent Object: [LoftSectionDimension](../LoftSectionDimension/LoftSectionDimension.md)

## Description

Read-write property that specifies whether the placed section for an area-graph type loft is driving or driven.

## Syntax

LoftSectionDimension.**IsDrivingSection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
