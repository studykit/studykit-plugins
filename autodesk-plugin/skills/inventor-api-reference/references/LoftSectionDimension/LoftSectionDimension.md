# LoftSectionDimension Object

## Description

The LoftSectionDimension object is used to define the position and area for a placed section of an area-graph type loft.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../LoftSectionDimension/LoftSectionDimension_Delete.md) | Method that deletes this placed section of an area-graph type loft. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LoftSectionDimension/LoftSectionDimension_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [IsDrivingSection](../LoftSectionDimension/LoftSectionDimension_IsDrivingSection.md) | Read-write property that specifies whether the placed section for an area-graph type loft is driving or driven. |
| [Position](../LoftSectionDimension/LoftSectionDimension_Position.md) | Read-write property that gets and sets the positional distance (from the starting section) of the placed section for an area-graph type loft. |
| [PositionAsAbsoluteDistance](../LoftSectionDimension/LoftSectionDimension_PositionAsAbsoluteDistance.md) | Read-write Boolean property that gets and sets the whether the positional distance of the placed section for an area-graph type loft is specified as an absolute distance value. |
| [Size](../LoftSectionDimension/LoftSectionDimension_Size.md) | Read-write property that gets and sets the size of the placed section for an area-graph type loft. |
| [SizeAsScaleFactor](../LoftSectionDimension/LoftSectionDimension_SizeAsScaleFactor.md) | Read-write property that gets and sets whether the size of the placed section for an area-graph type loft is specified as a scale factor with respect to its default size. |
| [Type](../LoftSectionDimension/LoftSectionDimension_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
