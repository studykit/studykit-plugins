# PresentationFace.Evaluator Property

Parent Object: [PresentationFace](../PresentationFace/PresentationFace.md)

## Description

Read-only property that returns the SurfaceEvaluator for this presentation face.

## Syntax

PresentationFace.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 2018
