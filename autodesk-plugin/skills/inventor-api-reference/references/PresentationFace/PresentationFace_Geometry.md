# PresentationFace.Geometry Property

Parent Object: [PresentationFace](../PresentationFace/PresentationFace.md)

## Description

Read-only property that returns the underlying geometry of the face.

## Syntax

PresentationFace.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
