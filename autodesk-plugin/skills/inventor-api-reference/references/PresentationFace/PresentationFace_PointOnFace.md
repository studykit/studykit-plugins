# PresentationFace.PointOnFace Property

Parent Object: [PresentationFace](../PresentationFace/PresentationFace.md)

## Description

Read-only property that returns a characteristic somewhere on the interior of the face.

## Syntax

PresentationFace.**PointOnFace**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
