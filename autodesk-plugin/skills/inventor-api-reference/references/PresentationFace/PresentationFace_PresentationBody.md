# PresentationFace.PresentationBody Property

Parent Object: [PresentationFace](../PresentationFace/PresentationFace.md)

## Description

Read-only property that returns the presentation body that this face is associative with.

## Syntax

PresentationFace.**PresentationBody**() As [PresentationBody](../PresentationBody/PresentationBody.md)

## Property Value

This is a read only property whose value is a [PresentationBody](../PresentationBody/PresentationBody.md).

## Version

Introduced in version 2018
