# HoleThreadNote.SingleDimensionLine Property

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Gets and sets whether to use a single dimension line.

## Syntax

HoleThreadNote.**SingleDimensionLine**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
