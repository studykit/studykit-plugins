# HoleThreadNote.PromoteToSketch Method

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Method that copies the dimension to the underlying sketch. This method only works for dimensions associated with a draft view.

## Syntax

HoleThreadNote.**PromoteToSketch**() As [DimensionConstraint](../DimensionConstraint/DimensionConstraint.md)

## Version

Introduced in version 2010
