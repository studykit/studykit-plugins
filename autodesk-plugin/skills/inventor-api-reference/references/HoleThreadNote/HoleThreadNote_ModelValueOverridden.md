# HoleThreadNote.ModelValueOverridden Property

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Read-write property that gets and sets whether the model value is overridden for the dimension.

## Syntax

HoleThreadNote.**ModelValueOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
