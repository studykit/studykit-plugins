# HoleThreadNote.IsInspectionDimension Property

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Gets and sets whether this is an inspection dimension.

## Syntax

HoleThreadNote.**IsInspectionDimension**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
