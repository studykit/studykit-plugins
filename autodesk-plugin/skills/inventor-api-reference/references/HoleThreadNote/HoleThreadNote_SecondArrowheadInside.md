# HoleThreadNote.SecondArrowheadInside Property

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Read-write property that gets and sets whether the second arrowhead is inside or outside.

## Syntax

HoleThreadNote.**SecondArrowheadInside**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
