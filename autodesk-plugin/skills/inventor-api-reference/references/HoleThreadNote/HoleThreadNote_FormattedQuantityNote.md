# HoleThreadNote.FormattedQuantityNote Property

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Gets and sets the fully formatted string that defines the quantity note.

## Syntax

HoleThreadNote.**FormattedQuantityNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
