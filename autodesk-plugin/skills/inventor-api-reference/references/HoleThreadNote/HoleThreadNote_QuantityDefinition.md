# HoleThreadNote.QuantityDefinition Property

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Gets and sets how the quantity value is set for the note.

## Syntax

HoleThreadNote.**QuantityDefinition**() As [HoleNoteQuantityEnum](../HoleNoteQuantityEnum.md)

## Property Value

This is a read/write property whose value is a [HoleNoteQuantityEnum](../HoleNoteQuantityEnum.md).

## Version

Introduced in version 2010
