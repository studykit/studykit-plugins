# HelpManager Object

## Description

The HelpManager object encapsulates all of the help functionality.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [HelpEvents](../HelpManager/HelpManager_HelpEvents.md) | Gets the object that fires Help related events. |
| [HelpPath](../HelpManager/HelpManager_HelpPath.md) | Gets the full path location of the help directory. |
| [Parent](../HelpManager/HelpManager_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../HelpManager/HelpManager_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Application.HelpManager](../Application/Application_HelpManager.md), [ApprenticeServer.HelpManager](../ApprenticeServer/ApprenticeServer_HelpManager.md), [ApprenticeServerComponent.HelpManager](../ApprenticeServerComponent/ApprenticeServerComponent_HelpManager.md)
