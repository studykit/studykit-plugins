# DWGPolyline3DProxy.Segments Property

Parent Object: [DWGPolyline3DProxy](../DWGPolyline3DProxy/DWGPolyline3DProxy.md)

## Description

Read-only property that returns the DWGEntitySegmentsEnumerator object.

## Syntax

DWGPolyline3DProxy.**Segments**() As [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md).

## Version

Introduced in version 2018
