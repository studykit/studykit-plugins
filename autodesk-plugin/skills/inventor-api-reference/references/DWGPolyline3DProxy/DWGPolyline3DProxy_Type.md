# DWGPolyline3DProxy.Type Property

Parent Object: [DWGPolyline3DProxy](../DWGPolyline3DProxy/DWGPolyline3DProxy.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

DWGPolyline3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2016
