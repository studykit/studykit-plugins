# DWGPolyline3DProxy.NativeObject Property

Parent Object: [DWGPolyline3DProxy](../DWGPolyline3DProxy/DWGPolyline3DProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGPolyline3DProxy.**NativeObject**() As [DWGPolyline3D](../DWGPolyline3D/DWGPolyline3D.md)

## Property Value

This is a read only property whose value is a [DWGPolyline3D](../DWGPolyline3D/DWGPolyline3D.md).

## Version

Introduced in version 2016
