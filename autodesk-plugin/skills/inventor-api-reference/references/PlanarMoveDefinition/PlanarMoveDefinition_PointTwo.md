# PlanarMoveDefinition.PointTwo Property

Parent Object: [PlanarMoveDefinition](../PlanarMoveDefinition/PlanarMoveDefinition.md)

## Description

Gets and sets the second point for the planar move.

## Syntax

PlanarMoveDefinition.**PointTwo**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2011
