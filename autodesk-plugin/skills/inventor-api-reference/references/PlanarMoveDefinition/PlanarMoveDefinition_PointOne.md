# PlanarMoveDefinition.PointOne Property

Parent Object: [PlanarMoveDefinition](../PlanarMoveDefinition/PlanarMoveDefinition.md)

## Description

Gets and sets the first point for the planar move.

## Syntax

PlanarMoveDefinition.**PointOne**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2011
