# DockableWindows.Events Property

Parent Object: [DockableWindows](../DockableWindows/DockableWindows.md)

## Description

Property that returns the DockableWindowsEvents object that hosts events related to dockable windows.

## Syntax

DockableWindows.**Events**() As [DockableWindowsEvents](../DockableWindowsEvents/DockableWindowsEvents.md)

## Property Value

This is a read only property whose value is a [DockableWindowsEvents](../DockableWindowsEvents/DockableWindowsEvents.md).

## Version

Introduced in version 2012
