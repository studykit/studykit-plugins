# ModelStateTableCell.Parent Property

Parent Object: [ModelStateTableCell](../ModelStateTableCell/ModelStateTableCell.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelStateTableCell.**Parent**() As [ModelStateTable](../ModelStateTable/ModelStateTable.md)

## Property Value

This is a read only property whose value is a [ModelStateTable](../ModelStateTable/ModelStateTable.md).

## Version

Introduced in version 2022
