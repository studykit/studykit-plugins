# ModelStateTableCell.Row Property

Parent Object: [ModelStateTableCell](../ModelStateTableCell/ModelStateTableCell.md)

## Description

Read-only property that returns the index of this cell within the parent ModelStatesTableRow.

## Syntax

ModelStateTableCell.**Row**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2022
