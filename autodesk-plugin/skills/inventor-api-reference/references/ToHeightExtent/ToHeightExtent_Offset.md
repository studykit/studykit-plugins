# ToHeightExtent.Offset Property

Parent Object: [ToHeightExtent](../ToHeightExtent/ToHeightExtent.md)

## Description

Property that returns the parameter controlling the offset distance of the flange relative to the 'To' point.

## Syntax

ToHeightExtent.**Offset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
