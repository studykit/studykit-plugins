# SurfaceGraphicsFace.Parent Property

Parent Object: [SurfaceGraphicsFace](../SurfaceGraphicsFace/SurfaceGraphicsFace.md)

## Description

Property that returns the parent SurfaceGraphics object.

## Syntax

SurfaceGraphicsFace.**Parent**() As [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Property Value

This is a read only property whose value is a [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md).

## Version

Introduced in version 2009
