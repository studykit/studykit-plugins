# SurfaceGraphicsFace Object

## Description

The SurfaceGraphicsFace represents an individual face displayed by the SurfaceGraphics object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Appearance](../SurfaceGraphicsFace/SurfaceGraphicsFace_Appearance.md) | Gets and sets appearance associated with the surface graphics face. |
| [Application](../SurfaceGraphicsFace/SurfaceGraphicsFace_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Color](../SurfaceGraphicsFace/SurfaceGraphicsFace_Color.md) | Gets and sets color associated with this primitive. |
| [Face](../SurfaceGraphicsFace/SurfaceGraphicsFace_Face.md) | Property that returns the Face object associated with the SurfaceGraphicsFace. |
| [Index](../SurfaceGraphicsFace/SurfaceGraphicsFace_Index.md) | Property that returns the index of the SurfaceGraphicsFace object within the SurfaceGraphicsFaceList. |
| [Parent](../SurfaceGraphicsFace/SurfaceGraphicsFace_Parent.md) | Property that returns the parent SurfaceGraphics object. |
| [Selectable](../SurfaceGraphicsFace/SurfaceGraphicsFace_Selectable.md) | Read-write property that specifies whether the surface graphics face can be selected or not. |
| [Type](../SurfaceGraphicsFace/SurfaceGraphicsFace_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SurfaceGraphicsFaceList.Item](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Item.md)

## Version

Introduced in version 2009
