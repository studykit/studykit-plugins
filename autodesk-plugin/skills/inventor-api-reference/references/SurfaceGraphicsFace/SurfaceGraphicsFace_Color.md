# SurfaceGraphicsFace.Color Property

Parent Object: [SurfaceGraphicsFace](../SurfaceGraphicsFace/SurfaceGraphicsFace.md)

## Description

Gets and sets color associated with this primitive.

## Syntax

SurfaceGraphicsFace.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2009
