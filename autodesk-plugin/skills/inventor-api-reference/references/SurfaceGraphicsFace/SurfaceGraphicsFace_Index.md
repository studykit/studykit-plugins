# SurfaceGraphicsFace.Index Property

Parent Object: [SurfaceGraphicsFace](../SurfaceGraphicsFace/SurfaceGraphicsFace.md)

## Description

Property that returns the index of the SurfaceGraphicsFace object within the SurfaceGraphicsFaceList.

## Syntax

SurfaceGraphicsFace.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
