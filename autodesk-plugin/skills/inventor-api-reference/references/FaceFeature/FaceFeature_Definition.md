# FaceFeature.Definition Property

Parent Object: [FaceFeature](../FaceFeature/FaceFeature.md)

## Description

Property that gets the FaceFeatureDefinition object associated with this face feature.

## Syntax

FaceFeature.**Definition**() As [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md)

## Property Value

This is a read only property whose value is a [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md).

## Version

Introduced in version 2009
