# FaceFeature.Suppressed Property

Parent Object: [FaceFeature](../FaceFeature/FaceFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

FaceFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
