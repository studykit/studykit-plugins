# GroundConstraint.Entity Property

Parent Object: [GroundConstraint](../GroundConstraint/GroundConstraint.md)

## Description

Property that returns entity being grounded.

## Syntax

GroundConstraint.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
