# CircularPatternFeatures Object

## Description

The CircularPatternFeatures collection object provides access to all of the objects in a component definition and provides methods to create additional CircularPatternFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddByDefinition](../CircularPatternFeatures/CircularPatternFeatures_AddByDefinition.md) | Method that creates a new CircularPattern feature. |
| [CreateDefinition](../CircularPatternFeatures/CircularPatternFeatures_CreateDefinition.md) | Method that creates a new CircularPatternFeatureDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CircularPatternFeatures/CircularPatternFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CircularPatternFeatures/CircularPatternFeatures_Count.md) | Property that returns the number of items in the collection. |
| [Item](../CircularPatternFeatures/CircularPatternFeatures_Item.md) | Returns the specified CircularPatternFeature object from the collection. This is the default property of the CircularPatternFeatures collection object. |
| [Type](../CircularPatternFeatures/CircularPatternFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Features.CircularPatternFeatures](../Features/Features_CircularPatternFeatures.md), [FlatPatternFeatures.CircularPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures_CircularPatternFeatures.md), [PartFeatures.CircularPatternFeatures](../PartFeatures/PartFeatures_CircularPatternFeatures.md), [SheetMetalFeatures.CircularPatternFeatures](../SheetMetalFeatures/SheetMetalFeatures_CircularPatternFeatures.md)

## Version

Introduced in version 5
