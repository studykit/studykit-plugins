# SketchConstraints3DEnumerator Object

## Description

The SketchConstraints3DEnumerator object provides access to a list of SketchConstraint objects in a specific 3D sketch. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator_Item.md) | Returns the specified SketchConstraint object from the collection. |
| [Type](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
