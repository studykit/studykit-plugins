# AngularModelDimensionDefinition.FirstArrowheadType Property

Parent Object: [AngularModelDimensionDefinition](../AngularModelDimensionDefinition/AngularModelDimensionDefinition.md)

## Description

Read-write property that gets and sets the type of the first arrowhead. This is a style override.

## Syntax

AngularModelDimensionDefinition.**FirstArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2018
