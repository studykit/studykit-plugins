# AngularModelDimensionDefinition.OverrideModelValue Property

Parent Object: [AngularModelDimensionDefinition](../AngularModelDimensionDefinition/AngularModelDimensionDefinition.md)

## Description

Read-write property that gets and sets the display value for the dimension.

## Syntax

AngularModelDimensionDefinition.**OverrideModelValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
