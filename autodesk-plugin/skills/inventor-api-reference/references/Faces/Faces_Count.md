# Faces.Count Property

Parent Object: [Faces](../Faces/Faces.md)

## Description

Property that returns the number of items in this collection.

## Syntax

Faces.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
