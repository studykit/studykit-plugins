# MoveFeatureProxy.Suppressed Property

Parent Object: [MoveFeatureProxy](../MoveFeatureProxy/MoveFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

MoveFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
