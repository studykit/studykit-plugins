# MoveFeatureProxy.NativeObject Property

Parent Object: [MoveFeatureProxy](../MoveFeatureProxy/MoveFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MoveFeatureProxy.**NativeObject**() As [MoveFeature](../MoveFeature/MoveFeature.md)

## Property Value

This is a read only property whose value is a [MoveFeature](../MoveFeature/MoveFeature.md).

## Version

Introduced in version 2010
