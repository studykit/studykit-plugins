# EllipticalCone.AxisVector Property

Parent Object: [EllipticalCone](../EllipticalCone/EllipticalCone.md)

## Description

Specifies the direction of the axis of the elliptical cone.

## Syntax

EllipticalCone.**AxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
