# EllipticalCone Object

## Description

The EllipticalCone object. The object created is a transient mathematical object and is not displayed graphically.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../EllipticalCone/EllipticalCone_Copy.md) | Creates a copy of this EllipticalCone object. The result is entirely independent and can be edited without affecting the original EllipticalCone object. |
| [GetEllipticalConeData](../EllipticalCone/EllipticalCone_GetEllipticalConeData.md) | Get the data defining this elliptical cone. |
| [PutEllipticalConeData](../EllipticalCone/EllipticalCone_PutEllipticalConeData.md) | Method that sets the data defining this elliptical cone. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AxisVector](../EllipticalCone/EllipticalCone_AxisVector.md) | Specifies the direction of the axis of the elliptical cone. |
| [BasePoint](../EllipticalCone/EllipticalCone_BasePoint.md) | Specifies the base of the elliptical cone. |
| [Evaluator](../EllipticalCone/EllipticalCone_Evaluator.md) | Gets the surface evaluator for this elliptical cone. |
| [HalfAngle](../EllipticalCone/EllipticalCone_HalfAngle.md) | Specifies the taper angle of the elliptical cone. This is the angle measured from the axis to the side of the elliptical cone. |
| [IsExpanding](../EllipticalCone/EllipticalCone_IsExpanding.md) | Specifies whether the taper of the elliptical cone is expanding outwards or inwards in the direction of the axis vector. If True and the HalfAngle is positive, then the elliptical cone is expanding outward along the axis vector. |
| [MajorAxisVector](../EllipticalCone/EllipticalCone_MajorAxisVector.md) | Specifies the direction of the major axis of the elliptical cone. |
| [MinorMajorRatio](../EllipticalCone/EllipticalCone_MinorMajorRatio.md) | Specifies the ratio of the minor and major axes of the elliptical cone. |
