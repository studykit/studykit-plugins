# DWGPointsEnumerator.Count Property

Parent Object: [DWGPointsEnumerator](../DWGPointsEnumerator/DWGPointsEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGPointsEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
