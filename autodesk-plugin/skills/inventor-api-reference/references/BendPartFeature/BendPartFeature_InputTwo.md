# BendPartFeature.InputTwo Property

Parent Object: [BendPartFeature](../BendPartFeature/BendPartFeature.md)

## Description

Input Variant that defines the second input for the bend arc.

## Syntax

BendPartFeature.**InputTwo**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
