# BendPartFeature.InputOne Property

Parent Object: [BendPartFeature](../BendPartFeature/BendPartFeature.md)

## Description

Input Variant that defines the first input for the bend arc.

## Syntax

BendPartFeature.**InputOne**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
