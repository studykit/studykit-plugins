# ModelGeneralNoteProxy.Visible Property

Parent Object: [ModelGeneralNoteProxy](../ModelGeneralNoteProxy/ModelGeneralNoteProxy.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelGeneralNoteProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
