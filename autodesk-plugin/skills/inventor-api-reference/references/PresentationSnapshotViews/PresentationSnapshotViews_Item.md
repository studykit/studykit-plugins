# PresentationSnapshotViews.Item Property

Parent Object: [PresentationSnapshotViews](../PresentationSnapshotViews/PresentationSnapshotViews.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationSnapshotViews.**Item**( ***Index*** As Variant ) As [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Property Value

This is a read only property whose value is a [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of an item to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
