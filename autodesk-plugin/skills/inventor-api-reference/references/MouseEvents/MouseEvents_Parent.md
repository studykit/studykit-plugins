# MouseEvents.Parent Property

Parent Object: [MouseEvents](../MouseEvents/MouseEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

MouseEvents.**Parent**() As [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Property Value

This is a read only property whose value is an [InteractionEvents](../InteractionEvents/InteractionEvents.md).

## Version

Introduced in version 5
