# RectangularOccurrencePatternProxy.OccurrencePatternElements Property

Parent Object: [RectangularOccurrencePatternProxy](../RectangularOccurrencePatternProxy/RectangularOccurrencePatternProxy.md)

## Description

Property returning an OccurrencePatternElements collection object. The first element within this collection will contain the components that were provided as input for the pattern.

## Syntax

RectangularOccurrencePatternProxy.**OccurrencePatternElements**() As [OccurrencePatternElements](../OccurrencePatternElements/OccurrencePatternElements.md)

## Property Value

This is a read only property whose value is an [OccurrencePatternElements](../OccurrencePatternElements/OccurrencePatternElements.md).

## Version

Introduced in version 2010
