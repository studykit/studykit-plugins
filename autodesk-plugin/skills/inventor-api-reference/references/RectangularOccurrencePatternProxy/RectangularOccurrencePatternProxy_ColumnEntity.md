# RectangularOccurrencePatternProxy.ColumnEntity Property

Parent Object: [RectangularOccurrencePatternProxy](../RectangularOccurrencePatternProxy/RectangularOccurrencePatternProxy.md)

## Description

Gets/Sets the entity used to define the column (x) direction of the pattern.

## Syntax

RectangularOccurrencePatternProxy.**ColumnEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2010
