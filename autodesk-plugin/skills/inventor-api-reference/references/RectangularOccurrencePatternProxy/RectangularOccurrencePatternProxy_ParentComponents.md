# RectangularOccurrencePatternProxy.ParentComponents Property

Parent Object: [RectangularOccurrencePatternProxy](../RectangularOccurrencePatternProxy/RectangularOccurrencePatternProxy.md)

## Description

Property that gets and sets the components used as input for the pattern.

## Syntax

RectangularOccurrencePatternProxy.**ParentComponents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2010
