# PartFeatureOperationEnum Enumerator

## Description

Constants identifying the type of Boolean operation to be performed during this feature placement.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kCutOperation | 20482 | Cut operation. |
| kIntersectOperation | 20483 | Intersect operation. |
| kJoinOperation | 20481 | Join operation. |
| kNewBodyOperation | 20485 | New Body operation. |
| kSurfaceOperation | 20484 | Surface operation. |

## Version

Introduced in version 5
