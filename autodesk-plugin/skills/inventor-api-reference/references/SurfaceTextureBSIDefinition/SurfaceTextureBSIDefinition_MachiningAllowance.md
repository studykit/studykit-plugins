# SurfaceTextureBSIDefinition.MachiningAllowance Property

Parent Object: [SurfaceTextureBSIDefinition](../SurfaceTextureBSIDefinition/SurfaceTextureBSIDefinition.md)

## Description

Gets and sets the machining allowance.

## Syntax

SurfaceTextureBSIDefinition.**MachiningAllowance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
