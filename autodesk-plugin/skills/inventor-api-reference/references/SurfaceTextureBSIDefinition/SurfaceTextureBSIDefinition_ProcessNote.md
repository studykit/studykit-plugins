# SurfaceTextureBSIDefinition.ProcessNote Property

Parent Object: [SurfaceTextureBSIDefinition](../SurfaceTextureBSIDefinition/SurfaceTextureBSIDefinition.md)

## Description

Gets and sets the process requirements for the surface.

## Syntax

SurfaceTextureBSIDefinition.**ProcessNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
