# SurfaceTextureBSIDefinition.FirstRequirement Property

Parent Object: [SurfaceTextureBSIDefinition](../SurfaceTextureBSIDefinition/SurfaceTextureBSIDefinition.md)

## Description

Gets and sets the first requirement value for the surface.

## Syntax

SurfaceTextureBSIDefinition.**FirstRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
