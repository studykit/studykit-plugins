# PartsList.Position Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the origin position of the table.

## Syntax

PartsList.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 9
