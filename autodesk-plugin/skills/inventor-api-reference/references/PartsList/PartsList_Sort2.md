# PartsList.Sort2 Method

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Changes the sort order of items in the parts list.

## Syntax

PartsList.**Sort2**( ***PrimaryColumnTitle*** As String, [***PrimaryColumnAscending***] As Boolean, [***SecondaryColumnTitle***] As String, [***SecondaryColumnAscending***] As Boolean, [***TertiaryColumnTitle***] As String, [***TertiaryColumnAscending***] As Boolean, [***SortByString***] As Boolean, [***AutoSortOnUpdate***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PrimaryColumnTitle | String |  |
| PrimaryColumnAscending | Boolean |  |
| SecondaryColumnTitle | String | This is an optional argument whose default value is "". |
| SecondaryColumnAscending | Boolean | This is an optional argument whose default value is True. |
| TertiaryColumnTitle | String | This is an optional argument whose default value is "". |
| TertiaryColumnAscending | Boolean | This is an optional argument whose default value is True. |
| SortByString | Boolean | This is an optional argument whose default value is True. |
| AutoSortOnUpdate | Boolean | This is an optional argument whose default value is False. |

## Version

Introduced in version 2018
