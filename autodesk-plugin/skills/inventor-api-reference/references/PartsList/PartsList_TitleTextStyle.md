# PartsList.TitleTextStyle Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the text style used for the title of the parts list.

## Syntax

PartsList.**TitleTextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2009
