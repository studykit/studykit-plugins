# PartsList.PartsListColumns Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets the PartsListColumns collection object for this parts list table.

## Syntax

PartsList.**PartsListColumns**() As [PartsListColumns](../PartsListColumns/PartsListColumns.md)

## Property Value

This is a read only property whose value is a [PartsListColumns](../PartsListColumns/PartsListColumns.md).
