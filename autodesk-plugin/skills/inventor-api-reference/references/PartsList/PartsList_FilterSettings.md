# PartsList.FilterSettings Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Read-only property that returns the filter settings for the parts list.

## Syntax

PartsList.**FilterSettings**() As [PartsListFilterSettings](../PartsListFilterSettings/PartsListFilterSettings.md)

## Property Value

This is a read only property whose value is a [PartsListFilterSettings](../PartsListFilterSettings/PartsListFilterSettings.md).

## Version

Introduced in version 2024.1
