# PartsList.Style Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the associated PartsListStyle object.

## Syntax

PartsList.**Style**() As [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Property Value

This is a read/write property whose value is a [PartsListStyle](../PartsListStyle/PartsListStyle.md).

## Version

Introduced in version 2011
