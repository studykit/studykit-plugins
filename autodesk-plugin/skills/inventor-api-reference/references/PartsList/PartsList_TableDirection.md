# PartsList.TableDirection Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the vertical direction of the table.

## Syntax

PartsList.**TableDirection**() As [TableDirectionEnum](../TableDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [TableDirectionEnum](../TableDirectionEnum.md).

## Version

Introduced in version 9
