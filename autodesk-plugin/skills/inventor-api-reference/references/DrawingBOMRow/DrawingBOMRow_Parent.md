# DrawingBOMRow.Parent Property

Parent Object: [DrawingBOMRow](../DrawingBOMRow/DrawingBOMRow.md)

## Description

Property that returns the parent DrawingBOM object.

## Syntax

DrawingBOMRow.**Parent**() As [DrawingBOM](../DrawingBOM/DrawingBOM.md)

## Property Value

This is a read only property whose value is a [DrawingBOM](../DrawingBOM/DrawingBOM.md).

## Version

Introduced in version 2009
