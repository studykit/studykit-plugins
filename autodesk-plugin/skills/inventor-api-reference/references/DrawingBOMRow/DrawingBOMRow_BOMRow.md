# DrawingBOMRow.BOMRow Property

Parent Object: [DrawingBOMRow](../DrawingBOMRow/DrawingBOMRow.md)

## Description

Property that returns the corresponding BOMRow from the model.

## Syntax

DrawingBOMRow.**BOMRow**() As [BOMRow](../BOMRow/BOMRow.md)

## Property Value

This is a read only property whose value is a [BOMRow](../BOMRow/BOMRow.md).
