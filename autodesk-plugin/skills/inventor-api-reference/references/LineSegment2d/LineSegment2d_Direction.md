# LineSegment2d.Direction Property

Parent Object: [LineSegment2d](../LineSegment2d/LineSegment2d.md)

## Description

Gets the direction of the line segment.

## Syntax

LineSegment2d.**Direction**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read only property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).

## Version

Introduced in version 5
