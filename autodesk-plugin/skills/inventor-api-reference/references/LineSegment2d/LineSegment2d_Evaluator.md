# LineSegment2d.Evaluator Property

Parent Object: [LineSegment2d](../LineSegment2d/LineSegment2d.md)

## Description

Gets the Curve2dEvaluator for this line segment.

## Syntax

LineSegment2d.**Evaluator**() As [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md)

## Property Value

This is a read only property whose value is a [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md).

## Version

Introduced in version 5
