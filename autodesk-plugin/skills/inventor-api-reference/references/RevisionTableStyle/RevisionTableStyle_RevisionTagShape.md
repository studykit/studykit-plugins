# RevisionTableStyle.RevisionTagShape Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Gets and sets the shape for revision tags.

## Syntax

RevisionTableStyle.**RevisionTagShape**() As [RevisionTagShapeEnum](../RevisionTagShapeEnum.md)

## Property Value

This is a read/write property whose value is a [RevisionTagShapeEnum](../RevisionTagShapeEnum.md).

## Version

Introduced in version 2011
