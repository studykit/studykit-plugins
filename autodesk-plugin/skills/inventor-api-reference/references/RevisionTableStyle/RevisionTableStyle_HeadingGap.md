# RevisionTableStyle.HeadingGap Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Gets and sets the heading gap in centimeters.

## Syntax

RevisionTableStyle.**HeadingGap**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
