# RevisionTableStyle.RevisionIndexDefaults Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Gets the RevisionIndexDefaults object.

## Syntax

RevisionTableStyle.**RevisionIndexDefaults**() As [RevisionIndexDefaults](../RevisionIndexDefaults/RevisionIndexDefaults.md)

## Property Value

This is a read only property whose value is a [RevisionIndexDefaults](../RevisionIndexDefaults/RevisionIndexDefaults.md).

## Version

Introduced in version 2025
