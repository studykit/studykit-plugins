# RevisionTableStyle.InsideLineWeight Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Gets and sets the line weight of the inner lines of the table.

## Syntax

RevisionTableStyle.**InsideLineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
