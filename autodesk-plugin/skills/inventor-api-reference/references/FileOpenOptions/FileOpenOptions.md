# FileOpenOptions Object

## Description

The FileOpenOptions object provides access to options affecting the opening of files.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [DefaultDesignViewInAssembly](../FileOpenOptions/FileOpenOptions_DefaultDesignViewInAssembly.md) | Gets and sets the default design view representation to use when opening assembly documents. |
| [DefaultDesignViewInPart](../FileOpenOptions/FileOpenOptions_DefaultDesignViewInPart.md) | Gets and sets the default design view representation to use when opening part documents. |
| [DefaultDesignViewIsAssociativeInAssembly](../FileOpenOptions/FileOpenOptions_DefaultDesignViewIsAssociativeInAssembly.md) | Gets and sets whether the default design view representation is associative in assembly documents. |
| [DefaultDesignViewIsAssociativeInPart](../FileOpenOptions/FileOpenOptions_DefaultDesignViewIsAssociativeInPart.md) | Gets and sets whether the default design view representation is associative in part documents. |
| [DefaultModelStateInAssembly](../FileOpenOptions/FileOpenOptions_DefaultModelStateInAssembly.md) | Gets and sets the default model state to use when opening assembly documents. |
| [DefaultModelStateInPart](../FileOpenOptions/FileOpenOptions_DefaultModelStateInPart.md) | Gets and sets the default model state to use when opening part documents. |
| [SkipAllUnresolvedFilesInAssembly](../FileOpenOptions/FileOpenOptions_SkipAllUnresolvedFilesInAssembly.md) | Enables/disables skip all unresolved files during openning in assembly. |
| [SkipAllUnresolvedFilesInDrawing](../FileOpenOptions/FileOpenOptions_SkipAllUnresolvedFilesInDrawing.md) | Enables/disables skip all unresolved files during openning in drawing. |
| [SkipAllUnresolvedFilesInPart](../FileOpenOptions/FileOpenOptions_SkipAllUnresolvedFilesInPart.md) | Enables/disables skip all unresolved files during openning in part. |

## Accessed From

[FileOptions.FileOpenOptions](../FileOptions/FileOptions_FileOpenOptions.md)

## Version

Introduced in version 2011
