# FileOpenOptions.DefaultModelStateInAssembly Property

Parent Object: [FileOpenOptions](../FileOpenOptions/FileOpenOptions.md)

## Description

Gets and sets the default model state to use when opening assembly documents.

## Syntax

FileOpenOptions.**DefaultModelStateInAssembly**() As [ModelStateTypeEnum](../ModelStateTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ModelStateTypeEnum](../ModelStateTypeEnum.md).

## Version

Introduced in version 2022
