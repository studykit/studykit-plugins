# TransitionSymbolStyle.TextStyle Property

Parent Object: [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Description

Gets and sets the text style used for the transition symbol style.

## Syntax

TransitionSymbolStyle.**TextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2025.1
