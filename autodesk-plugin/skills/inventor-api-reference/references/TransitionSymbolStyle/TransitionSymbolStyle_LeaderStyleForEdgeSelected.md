# TransitionSymbolStyle.LeaderStyleForEdgeSelected Property

Parent Object: [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Description

Gets and sets the leader style used for the transition symbol style when attaches to geometry.

## Syntax

TransitionSymbolStyle.**LeaderStyleForEdgeSelected**() As [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Property Value

This is a read/write property whose value is a [LeaderStyle](../LeaderStyle/LeaderStyle.md).

## Version

Introduced in version 2025.1
