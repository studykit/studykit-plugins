# CustomTable.Rows Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Property that returns the Rows collection object.

## Syntax

CustomTable.**Rows**() As [Rows](../Rows/Rows.md)

## Property Value

This is a read only property whose value is a [Rows](../Rows/Rows.md).

## Version

Introduced in version 9
