# CustomTable.MaximumRows Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the maximum number of rows per section.

## Syntax

CustomTable.**MaximumRows**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2009
