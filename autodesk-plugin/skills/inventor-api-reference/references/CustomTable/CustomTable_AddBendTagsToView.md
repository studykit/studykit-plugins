# CustomTable.AddBendTagsToView Method

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Method that adds bend tags to drawing view.

## Syntax

CustomTable.**AddBendTagsToView**( ***DrawingView*** As [DrawingView](../DrawingView/DrawingView.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DrawingView | [DrawingView](../DrawingView/DrawingView.md) | Input DrawingView object that specifies the DrawingView to add bend tags to. |

## Version

Introduced in version 2025
