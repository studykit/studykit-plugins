# CustomTable.CopyTo Method

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Method that copies the table to another sheet, either within the same document or in another document.

## Syntax

CustomTable.**CopyTo**( ***TargetSheet*** As [Sheet](../Sheet/Sheet.md) ) As [CustomTable](../CustomTable/CustomTable.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TargetSheet | [Sheet](../Sheet/Sheet.md) | Input Sheet object that specifies the sheet to copy the custom table to. |

## Version

Introduced in version 2011
