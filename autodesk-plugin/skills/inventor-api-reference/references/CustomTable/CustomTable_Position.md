# CustomTable.Position Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Specifies the position of the table.

## Syntax

CustomTable.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 9
