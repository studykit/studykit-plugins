# CustomTable.Layer Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the layer used by the table.

## Syntax

CustomTable.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
