# CustomTable.MergeCells Method

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Merges the table cells.

## Syntax

CustomTable.**MergeCells**( ***StartCell*** As [Cell](../Cell/Cell.md), ***EndCell*** As [Cell](../Cell/Cell.md) ) As [MergedCell](../MergedCell/MergedCell.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| StartCell | [Cell](../Cell/Cell.md) | Input Cell object that indicates the start cell being merged. |
| EndCell | [Cell](../Cell/Cell.md) | Input Cell object that indicates the end cell being merged. |

## Version

Introduced in version 2025.1
