# IntentTypeEnum Enumerator

## Description

Constants indicating the type of value that the GeometryIntent's Intent property will return.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kGeometryIntent | 58116 | Intent is a DrawingCurve or a SketchEntity. |
| kNoPointIntent | 58117 | The GeometryIntent is not a point and the Intent property will not return a meaningful value. |
| kParameterIntent | 58115 | Intent is a double value. |
| kPoint2dIntent | 58114 | Intent is a Point2d object. |
| kPointEnumIntent | 58113 | Intent is a PointIntentEnum. |
| kPointIntent | 58118 | Intent is a Point object. |

## Version

Introduced in version 11
