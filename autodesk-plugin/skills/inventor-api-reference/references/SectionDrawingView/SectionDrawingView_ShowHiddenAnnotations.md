# SectionDrawingView.ShowHiddenAnnotations Method

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Method that displays all the annotations hidden by the user.

## Syntax

SectionDrawingView.**ShowHiddenAnnotations**()

## Version

Introduced in version 11
