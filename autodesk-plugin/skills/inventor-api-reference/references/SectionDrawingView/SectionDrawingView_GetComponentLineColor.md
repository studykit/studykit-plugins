# SectionDrawingView.GetComponentLineColor Method

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Method that gets color of a component in drawing view.

## Syntax

SectionDrawingView.**GetComponentLineColor**( ***Component*** As Object ) As [Color](../Color/Color.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Component | Object | Input Document, ComponentOccurrence, ComponentOccurrenceProxy, SurfaceBody, SurfaceBodyProxy, PartFeature or PartFeatureProxy object to indicate a component in the drawing view. |

## Version

Introduced in version 2025.1
