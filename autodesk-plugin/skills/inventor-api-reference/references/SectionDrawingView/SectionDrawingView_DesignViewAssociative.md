# SectionDrawingView.DesignViewAssociative Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Read-only property that gets whether the DesignViewRepresentation for a drawing view of a model is associative or not.

## Syntax

SectionDrawingView.**DesignViewAssociative**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2021
