# SectionDrawingView.DisplayTangentEdges Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Read-write property that gets and sets whether to display the tangent edges on the drawing view.

## Syntax

SectionDrawingView.**DisplayTangentEdges**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
