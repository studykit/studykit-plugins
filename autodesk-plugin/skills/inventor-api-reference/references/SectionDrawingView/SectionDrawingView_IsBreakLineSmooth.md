# SectionDrawingView.IsBreakLineSmooth Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Read-write property that gets and sets whether the cut edges are smooth or jogged.

## Syntax

SectionDrawingView.**IsBreakLineSmooth**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
