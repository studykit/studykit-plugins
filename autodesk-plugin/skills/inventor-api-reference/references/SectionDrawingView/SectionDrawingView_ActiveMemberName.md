# SectionDrawingView.ActiveMemberName Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets the name of the active member for a drawing view of an iPart or an iAssembly.

## Syntax

SectionDrawingView.**ActiveMemberName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
