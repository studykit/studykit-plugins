# SectionDrawingView.PresentationViewAssociative Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether to associate the drawing view to the referenced presentation view.

## Syntax

SectionDrawingView.**PresentationViewAssociative**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
