# SectionDrawingView.UpToDate Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that returns whether the drawing view is up to date with respect to the model.

## Syntax

SectionDrawingView.**UpToDate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
