# SectionDrawingView.IncludeSlice Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that gets and sets whether some components are sliced and some components are sectioned, depending on their individual settings.

## Syntax

SectionDrawingView.**IncludeSlice**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
