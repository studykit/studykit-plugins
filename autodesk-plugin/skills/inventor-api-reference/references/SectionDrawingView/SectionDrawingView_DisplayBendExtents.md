# SectionDrawingView.DisplayBendExtents Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether to display sheet metal bend extents on the drawing view.

## Syntax

SectionDrawingView.**DisplayBendExtents**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
