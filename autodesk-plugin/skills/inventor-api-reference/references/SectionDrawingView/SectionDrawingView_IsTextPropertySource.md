# SectionDrawingView.IsTextPropertySource Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Read-write property that gets and sets whether the drawing view is specified as the text property source.This is applicable only when the drawing view is a base view.

## Syntax

SectionDrawingView.**IsTextPropertySource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
