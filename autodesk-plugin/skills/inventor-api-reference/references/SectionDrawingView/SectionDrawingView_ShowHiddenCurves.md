# SectionDrawingView.ShowHiddenCurves Method

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Method that displays all the curves explicitly hidden by the user.

## Syntax

SectionDrawingView.**ShowHiddenCurves**()

## Version

Introduced in version 11
