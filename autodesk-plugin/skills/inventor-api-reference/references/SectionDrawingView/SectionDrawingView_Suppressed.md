# SectionDrawingView.Suppressed Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether this drawing view is suppressed or not.

## Syntax

SectionDrawingView.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
