# SectionDrawingView.SectionLineSketch Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets the object that contains the profile used to define the section line of the view.

## Syntax

SectionDrawingView.**SectionLineSketch**() As [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Property Value

This is a read only property whose value is a [DrawingSketch](../DrawingSketch/DrawingSketch.md).

## Version

Introduced in version 9
