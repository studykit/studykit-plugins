# SectionDrawingView.DisplayThreadFeatures Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether to display thread features on the drawing view.

## Syntax

SectionDrawingView.**DisplayThreadFeatures**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
