# SectionDrawingView.ShowLabel Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether to show the label of the drawing view on the sheet.

## Syntax

SectionDrawingView.**ShowLabel**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
