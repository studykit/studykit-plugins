# SectionDrawingView.Top Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that specifies the position of the top edge of the drawing view.

## Syntax

SectionDrawingView.**Top**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 9
