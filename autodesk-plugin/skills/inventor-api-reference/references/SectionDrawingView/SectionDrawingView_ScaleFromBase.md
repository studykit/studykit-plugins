# SectionDrawingView.ScaleFromBase Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether the scale of the view derives from the parent view or not.

## Syntax

SectionDrawingView.**ScaleFromBase**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
