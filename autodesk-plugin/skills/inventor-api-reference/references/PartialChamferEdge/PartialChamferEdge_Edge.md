# PartialChamferEdge.Edge Property

Parent Object: [PartialChamferEdge](../PartialChamferEdge/PartialChamferEdge.md)

## Description

Returns the Edge or EdgeCollection contains the Edge objects of being partially chamfered.

## Syntax

PartialChamferEdge.**Edge**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read only property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2019
