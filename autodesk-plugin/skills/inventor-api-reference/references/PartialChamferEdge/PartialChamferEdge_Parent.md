# PartialChamferEdge.Parent Property

Parent Object: [PartialChamferEdge](../PartialChamferEdge/PartialChamferEdge.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PartialChamferEdge.**Parent**() As [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Property Value

This is a read only property whose value is a [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md).

## Version

Introduced in version 2018
