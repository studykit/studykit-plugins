# InteractionEvents.MouseEvents Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the Mouse Events object.

## Syntax

InteractionEvents.**MouseEvents**() As [MouseEvents](../MouseEvents/MouseEvents.md)

## Property Value

This is a read only property whose value is a [MouseEvents](../MouseEvents/MouseEvents.md).
