# InteractionEvents.Stop Method

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Stops this object inside Inventor. This will cause the OnTerminate to fire halting all activity linked with this object.

## Syntax

InteractionEvents.**Stop**()
