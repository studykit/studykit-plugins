# InteractionEvents.Parent Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

InteractionEvents.**Parent**() As [CommandManager](../CommandManager/CommandManager.md)

## Property Value

This is a read only property whose value is a [CommandManager](../CommandManager/CommandManager.md).

## Version

Introduced in version 5
