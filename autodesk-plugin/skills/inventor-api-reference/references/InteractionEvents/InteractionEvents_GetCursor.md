# InteractionEvents.GetCursor Method

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the cursor for the command in which this interaction takes place.

## Syntax

InteractionEvents.**GetCursor**( ***CursorType*** As [CursorTypeEnum](../CursorTypeEnum.md), ***Cursor*** As Variant, ***CursorHotSpot*** As [CursorHotSpotEnum](../CursorHotSpotEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CursorType | [CursorTypeEnum](../CursorTypeEnum.md) | Returns the CursorTypeEnum indicating the cursor type. |
| Cursor | Variant | Returns the cursor. |
| CursorHotSpot | [CursorHotSpotEnum](../CursorHotSpotEnum.md) | Returns the CursorHotSpotEnum indicating the hotspot location relative to the cursor. |

## Version

Introduced in version 9
