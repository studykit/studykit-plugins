# InteractionEvents.TargetDocument Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the Document which is the context in which this interaction takes place.

## Syntax

InteractionEvents.**TargetDocument**() As [Document](../Document/Document.md)

## Property Value

This is a read only property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 5
