# InteractionEvents.Type Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

InteractionEvents.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
