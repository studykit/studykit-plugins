# InteractionEvents.MeasureEvents Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the Measure Events object.

## Syntax

InteractionEvents.**MeasureEvents**() As [MeasureEvents](../MeasureEvents/MeasureEvents.md)

## Property Value

This is a read only property whose value is a [MeasureEvents](../MeasureEvents/MeasureEvents.md).
