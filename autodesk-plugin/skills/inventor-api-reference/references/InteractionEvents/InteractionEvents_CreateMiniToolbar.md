# InteractionEvents.CreateMiniToolbar Method

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Method that creates a MiniToolbar object associated with the parent InteractionEvents.

## Syntax

InteractionEvents.**CreateMiniToolbar**() As [MiniToolbar](../MiniToolbar/MiniToolbar.md)
