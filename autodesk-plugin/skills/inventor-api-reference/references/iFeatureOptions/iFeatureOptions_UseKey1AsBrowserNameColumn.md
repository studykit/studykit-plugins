# iFeatureOptions.UseKey1AsBrowserNameColumn Property

Parent Object: [iFeatureOptions](../iFeatureOptions/iFeatureOptions.md)

## Description

Gets/Sets whether to use the Key 1 column of a table-driven iFeature as the Browser Name column.

## Syntax

iFeatureOptions.**UseKey1AsBrowserNameColumn**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
