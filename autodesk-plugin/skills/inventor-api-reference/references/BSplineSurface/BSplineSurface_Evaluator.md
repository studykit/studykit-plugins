# BSplineSurface.Evaluator Property

Parent Object: [BSplineSurface](../BSplineSurface/BSplineSurface.md)

## Description

Gets the surface evaluator for this surface.

## Syntax

BSplineSurface.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
