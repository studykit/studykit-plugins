# SketchLine.EndSketchPoint Property

Parent Object: [SketchLine](../SketchLine/SketchLine.md)

## Description

Property that returns the that defines the position of the end of the line.

## Syntax

SketchLine.**EndSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).
