# SketchLine.LineDefinitionSpace Property

Parent Object: [SketchLine](../SketchLine/SketchLine.md)

## Description

Gets and sets the LineDefinitionSpace applied to this sketch line.

## Syntax

SketchLine.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 9
