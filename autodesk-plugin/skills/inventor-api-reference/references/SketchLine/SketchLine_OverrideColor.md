# SketchLine.OverrideColor Property

Parent Object: [SketchLine](../SketchLine/SketchLine.md)

## Description

Gets and sets the color applied to this sketch line.

## Syntax

SketchLine.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).
