# FilletVariableRadiusEdgeSet.StartRadius Property

Parent Object: [FilletVariableRadiusEdgeSet](../FilletVariableRadiusEdgeSet/FilletVariableRadiusEdgeSet.md)

## Description

Property that returns the parameter that controls the start radius of the fillet.

## Syntax

FilletVariableRadiusEdgeSet.**StartRadius**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 5.3
