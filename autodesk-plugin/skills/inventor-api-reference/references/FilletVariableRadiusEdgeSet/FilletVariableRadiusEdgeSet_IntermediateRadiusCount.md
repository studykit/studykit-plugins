# FilletVariableRadiusEdgeSet.IntermediateRadiusCount Property

Parent Object: [FilletVariableRadiusEdgeSet](../FilletVariableRadiusEdgeSet/FilletVariableRadiusEdgeSet.md)

## Description

Property that returns the number of intermediate points defined in the edge set.

## Syntax

FilletVariableRadiusEdgeSet.**IntermediateRadiusCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5.3
