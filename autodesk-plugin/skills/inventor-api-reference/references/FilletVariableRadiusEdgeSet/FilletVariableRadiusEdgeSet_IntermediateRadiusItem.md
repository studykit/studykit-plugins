# FilletVariableRadiusEdgeSet.IntermediateRadiusItem Property

Parent Object: [FilletVariableRadiusEdgeSet](../FilletVariableRadiusEdgeSet/FilletVariableRadiusEdgeSet.md)

## Description

Returns the specified FilletIntermediateRadius object.

## Syntax

FilletVariableRadiusEdgeSet.**IntermediateRadiusItem**( ***Index*** As Long ) As [FilletIntermediateRadius](../FilletIntermediateRadius/FilletIntermediateRadius.md)

## Property Value

This is a read only property whose value is a [FilletIntermediateRadius](../FilletIntermediateRadius/FilletIntermediateRadius.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |
