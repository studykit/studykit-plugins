# Point2d Object

## Description

The Point2d object. The object created is a transient mathematical object and is not displayed graphically. Also, for more information see the Transient Geometry article in the overview section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Point2d/Point2d_Copy.md) | Creates a copy of this Point2d object. The result is entirely independent and can be edited without affecting the original Point2d object. |
| [DistanceTo](../Point2d/Point2d_DistanceTo.md) | Method that returns the distance between this point and the specified point. |
| [GetPointData](../Point2d/Point2d_GetPointData.md) | Get the data defining this point. |
| [IsEqualTo](../Point2d/Point2d_IsEqualTo.md) | Compares this point for equality with the specified point. |
| [PutPointData](../Point2d/Point2d_PutPointData.md) | Method that sets the data defining this point. |
| [TransformBy](../Point2d/Point2d_TransformBy.md) | Transform this point by the specified matrix. |
| [TranslateBy](../Point2d/Point2d_TranslateBy.md) | Translate this point by the specified vector. |
| [VectorTo](../Point2d/Point2d_VectorTo.md) | Gets the vector of translation between this point and the specified point. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [X](../Point2d/Point2d_X.md) | Specifies the X coordinate of the point. If not supplied, the X value will default to 0. |
| [Y](../Point2d/Point2d_Y.md) | Specifies the Y coordinate of the point. If not supplied, the Y value will default to 0. |
