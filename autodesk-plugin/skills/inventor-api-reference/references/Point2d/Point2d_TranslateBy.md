# Point2d.TranslateBy Method

Parent Object: [Point2d](../Point2d/Point2d.md)

## Description

Translate this point by the specified vector.

## Syntax

Point2d.**TranslateBy**( ***Vector*** As [Vector2d](../Vector2d/Vector2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [Vector2d](../Vector2d/Vector2d.md) | Input Vector2d object that specifies the vector to translate by. |

## Version

Introduced in version 2008
