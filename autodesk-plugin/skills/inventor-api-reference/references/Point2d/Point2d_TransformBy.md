# Point2d.TransformBy Method

Parent Object: [Point2d](../Point2d/Point2d.md)

## Description

Transform this point by the specified matrix.

## Syntax

Point2d.**TransformBy**( ***Matrix*** As [Matrix2d](../Matrix2d/Matrix2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix | [Matrix2d](../Matrix2d/Matrix2d.md) | Input Matrix2d object that specifies the matrix to transform by. |

## Version

Introduced in version 2008
