# Point2d.X Property

Parent Object: [Point2d](../Point2d/Point2d.md)

## Description

Specifies the X coordinate of the point. If not supplied, the X value will default to 0.

## Syntax

Point2d.**X**() As Double

## Property Value

This is a read/write property whose value is a Double.
