# Point2d.DistanceTo Method

Parent Object: [Point2d](../Point2d/Point2d.md)

## Description

Method that returns the distance between this point and the specified point.

## Syntax

Point2d.**DistanceTo**( ***Point*** As [Point2d](../Point2d/Point2d.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point2d](../Point2d/Point2d.md) | Point2d object that specifies the point to which the distance is to be returned. |
