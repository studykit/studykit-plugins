# ParallelToZAxisConstraint3D.Delete Method

Parent Object: [ParallelToZAxisConstraint3D](../ParallelToZAxisConstraint3D/ParallelToZAxisConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ParallelToZAxisConstraint3D.**Delete**()

## Version

Introduced in version 2017
