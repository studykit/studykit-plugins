# FoldDefinition.BendLine Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets the SketchLine object that defines the location of the bend line.

## Syntax

FoldDefinition.**BendLine**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read/write property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 2009
