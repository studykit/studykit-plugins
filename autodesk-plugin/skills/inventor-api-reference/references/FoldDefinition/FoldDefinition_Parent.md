# FoldDefinition.Parent Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Property that returns the parent FoldFeature object of this FoldDefinition object.

## Syntax

FoldDefinition.**Parent**() As [FoldFeature](../FoldFeature/FoldFeature.md)

## Property Value

This is a read only property whose value is a [FoldFeature](../FoldFeature/FoldFeature.md).

## Version

Introduced in version 2009
