# FoldDefinition.IsPositiveBendSide Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets whether the flip direction is positive or negative.

## Syntax

FoldDefinition.**IsPositiveBendSide**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
