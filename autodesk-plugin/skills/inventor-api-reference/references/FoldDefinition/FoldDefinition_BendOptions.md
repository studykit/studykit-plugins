# FoldDefinition.BendOptions Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets the BendOptions object that defines the bend options for this bend feature.

## Syntax

FoldDefinition.**BendOptions**() As [BendOptions](../BendOptions/BendOptions.md)

## Property Value

This is a read/write property whose value is a [BendOptions](../BendOptions/BendOptions.md).

## Version

Introduced in version 2009
