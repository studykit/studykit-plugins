# FoldDefinition.BendAngle Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets the bend angle of a sheet metal fold feature.

## Syntax

FoldDefinition.**BendAngle**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2009
