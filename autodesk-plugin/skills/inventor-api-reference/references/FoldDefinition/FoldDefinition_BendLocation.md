# FoldDefinition.BendLocation Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets the position of the bend with respect to the bend line.

## Syntax

FoldDefinition.**BendLocation**() As [BendLocationEnum](../BendLocationEnum.md)

## Property Value

This is a read/write property whose value is a [BendLocationEnum](../BendLocationEnum.md).

## Version

Introduced in version 2009
