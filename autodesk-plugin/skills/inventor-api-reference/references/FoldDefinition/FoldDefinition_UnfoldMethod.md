# FoldDefinition.UnfoldMethod Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets the UnfoldMethod object that defines how this bend is unfolded.

## Syntax

FoldDefinition.**UnfoldMethod**() As [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Property Value

This is a read/write property whose value is a [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md).

## Version

Introduced in version 2009
