# DistanceExtent.Type Property

Parent Object: [DistanceExtent](../DistanceExtent/DistanceExtent.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DistanceExtent.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
