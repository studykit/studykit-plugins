# ModelDatumDefinition.DatumTargetTextPositions Property

Parent Object: [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Description

Gets or sets the datum target text positions.

## Syntax

ModelDatumDefinition.**DatumTargetTextPositions**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2023
