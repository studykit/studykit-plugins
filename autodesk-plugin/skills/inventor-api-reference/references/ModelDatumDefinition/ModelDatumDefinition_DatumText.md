# ModelDatumDefinition.DatumText Property

Parent Object: [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Description

Gets and sets the datum text.

## Syntax

ModelDatumDefinition.**DatumText**() As String

## Property Value

This is a read/write property whose value is a String.
