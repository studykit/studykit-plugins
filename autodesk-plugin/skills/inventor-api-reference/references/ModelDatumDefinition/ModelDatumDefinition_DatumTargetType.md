# ModelDatumDefinition.DatumTargetType Property

Parent Object: [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Description

Gets and sets the datum target type.

## Syntax

ModelDatumDefinition.**DatumTargetType**() As [ModelDatumTargetTypeEnum](../ModelDatumTargetTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ModelDatumTargetTypeEnum](../ModelDatumTargetTypeEnum.md).

## Version

Introduced in version 2023
