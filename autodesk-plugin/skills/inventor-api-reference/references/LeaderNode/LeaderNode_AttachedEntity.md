# LeaderNode.AttachedEntity Property

Parent Object: [LeaderNode](../LeaderNode/LeaderNode.md)

## Description

Gets and sets the attached entity of the leaf leader node.

## Syntax

LeaderNode.**AttachedEntity**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 11
