# LeaderNode.Position Property

Parent Object: [LeaderNode](../LeaderNode/LeaderNode.md)

## Description

Gets and sets the leader node location on the sheet.

## Syntax

LeaderNode.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 11
