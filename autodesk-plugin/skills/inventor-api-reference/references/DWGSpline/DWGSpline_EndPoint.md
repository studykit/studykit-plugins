# DWGSpline.EndPoint Property

Parent Object: [DWGSpline](../DWGSpline/DWGSpline.md)

## Description

Read-only property that returns the end point geometry.

## Syntax

DWGSpline.**EndPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
