# DWGSpline.StartPoint Property

Parent Object: [DWGSpline](../DWGSpline/DWGSpline.md)

## Description

Read-only property that returns the start point geometry.

## Syntax

DWGSpline.**StartPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
