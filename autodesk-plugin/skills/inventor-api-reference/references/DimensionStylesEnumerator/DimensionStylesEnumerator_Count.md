# DimensionStylesEnumerator.Count Property

Parent Object: [DimensionStylesEnumerator](../DimensionStylesEnumerator/DimensionStylesEnumerator.md)

## Description

Property that returns the number of items in the collection.

## Syntax

DimensionStylesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 9
