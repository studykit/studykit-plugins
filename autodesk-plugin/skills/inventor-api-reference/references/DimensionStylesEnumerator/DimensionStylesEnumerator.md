# DimensionStylesEnumerator Object

## Description

The DimensionStylesEnumerator collection object provides access to the existing objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DimensionStylesEnumerator/DimensionStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DimensionStylesEnumerator/DimensionStylesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DimensionStylesEnumerator/DimensionStylesEnumerator_Item.md) | Returns the specified object from the collection. |
| [Type](../DimensionStylesEnumerator/DimensionStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.DimensionStyles](../DrawingStylesManager/DrawingStylesManager_DimensionStyles.md)

## Version

Introduced in version 9
