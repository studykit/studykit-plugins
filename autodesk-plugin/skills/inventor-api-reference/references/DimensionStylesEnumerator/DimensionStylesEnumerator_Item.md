# DimensionStylesEnumerator.Item Property

Parent Object: [DimensionStylesEnumerator](../DimensionStylesEnumerator/DimensionStylesEnumerator.md)

## Description

Returns the specified object from the collection.

## Syntax

DimensionStylesEnumerator.**Item**( ***Index*** As Variant ) As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read only property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the DimensionStyle to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the DimensionStyle name. If an out of range index or a name of a non-existent DimensionStyle is provided, an error will occur. |

## Version

Introduced in version 9
