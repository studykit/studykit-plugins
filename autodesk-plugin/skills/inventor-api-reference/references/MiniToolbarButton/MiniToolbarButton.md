# MiniToolbarButton Object

## Description

The MiniToolbarButton object represents a button control within a MiniToolbar.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../MiniToolbarButton/MiniToolbarButton_Delete.md) | Method that deletes the control. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MiniToolbarButton/MiniToolbarButton_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AutoHide](../MiniToolbarButton/MiniToolbarButton_AutoHide.md) | Read-write property that gets and sets whether the control should be automatically hidden when the user moves the cursor away from the MiniToolbar. |
| [ControlType](../MiniToolbarButton/MiniToolbarButton_ControlType.md) | Read-only property that returns the control type. |
| [DisplayName](../MiniToolbarButton/MiniToolbarButton_DisplayName.md) | Read-write property that gets and sets the display name for the control. |
| [Enabled](../MiniToolbarButton/MiniToolbarButton_Enabled.md) | Read-write property that gets and sets whether the control is enabled. |
| [Index](../MiniToolbarButton/MiniToolbarButton_Index.md) | Read-only property that returns the index this control is currently positioned at within the mini toolbar. |
| [InternalName](../MiniToolbarButton/MiniToolbarButton_InternalName.md) | Read-only property that returns a string uniquely identifying this control within the toolbar. |
| [LargeIcon](../MiniToolbarButton/MiniToolbarButton_LargeIcon.md) | Read-write property that gets and sets the large icon for the control. |
| [Parent](../MiniToolbarButton/MiniToolbarButton_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Pressed](../MiniToolbarButton/MiniToolbarButton_Pressed.md) | Read-write property that gets and sets whether the button is currently pressed (active). |
| [StandardIcon](../MiniToolbarButton/MiniToolbarButton_StandardIcon.md) | Read-write property that gets and sets the standard (small) icon for the control. |
| [ToolTipText](../MiniToolbarButton/MiniToolbarButton_ToolTipText.md) | Read-write property that gets and sets the tooltip text for the control. |
| [Type](../MiniToolbarButton/MiniToolbarButton_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Visible](../MiniToolbarButton/MiniToolbarButton_Visible.md) | Read-write property that gets and sets whether the control is visible. |

## Events

|  |  |
| --- | --- |
