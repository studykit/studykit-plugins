# MiniToolbarButton.OnClick Event

Parent Object: [MiniToolbarButton](../MiniToolbarButton/MiniToolbarButton.md)

## Description

Event that is fired when the button is clicked by the user.

## Syntax

MiniToolbarButton.**OnClick**()

## Version

Introduced in version 2012
