# MiniToolbarButton.DisplayName Property

Parent Object: [MiniToolbarButton](../MiniToolbarButton/MiniToolbarButton.md)

## Description

Read-write property that gets and sets the display name for the control.

## Syntax

MiniToolbarButton.**DisplayName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
