# MiniToolbarButton.LargeIcon Property

Parent Object: [MiniToolbarButton](../MiniToolbarButton/MiniToolbarButton.md)

## Description

Read-write property that gets and sets the large icon for the control.

## Syntax

MiniToolbarButton.**LargeIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2012
