# HeadsUpDisplayOptions.Parent Property

Parent Object: [HeadsUpDisplayOptions](../HeadsUpDisplayOptions/HeadsUpDisplayOptions.md)

## Description

Property that returns the parent SketchOptions object.

## Syntax

HeadsUpDisplayOptions.**Parent**() As [SketchOptions](../SketchOptions/SketchOptions.md)

## Property Value

This is a read only property whose value is a [SketchOptions](../SketchOptions/SketchOptions.md).

## Version

Introduced in version 2011
