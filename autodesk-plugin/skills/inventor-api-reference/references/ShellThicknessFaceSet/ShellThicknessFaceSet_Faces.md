# ShellThicknessFaceSet.Faces Property

Parent Object: [ShellThicknessFaceSet](../ShellThicknessFaceSet/ShellThicknessFaceSet.md)

## Description

Gets and sets the shell boundary relative to part face.

## Syntax

ShellThicknessFaceSet.**Faces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 9
