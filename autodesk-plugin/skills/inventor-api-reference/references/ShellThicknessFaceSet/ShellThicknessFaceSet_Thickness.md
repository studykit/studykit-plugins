# ShellThicknessFaceSet.Thickness Property

Parent Object: [ShellThicknessFaceSet](../ShellThicknessFaceSet/ShellThicknessFaceSet.md)

## Description

Property that returns the parameter that controls the thickness of the faces in the face set. This property will return Nothing if the shell feature has not been created yet.

## Syntax

ShellThicknessFaceSet.**Thickness**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 9
