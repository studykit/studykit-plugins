# RefoldFeatureProxy.ConsumeInputs Property

Parent Object: [RefoldFeatureProxy](../RefoldFeatureProxy/RefoldFeatureProxy.md)

## Description

Gets and sets whether the inputs to this feature should be nested under this feature in the browser.

## Syntax

RefoldFeatureProxy.**ConsumeInputs**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
