# RefoldFeatureProxy.Bends Property

Parent Object: [RefoldFeatureProxy](../RefoldFeatureProxy/RefoldFeatureProxy.md)

## Description

Property that returns the collection of Bend objects refolded by this feature.

## Syntax

RefoldFeatureProxy.**Bends**() As [BendsEnumerator](../BendsEnumerator/BendsEnumerator.md)

## Property Value

This is a read only property whose value is a [BendsEnumerator](../BendsEnumerator/BendsEnumerator.md).

## Version

Introduced in version 2010
