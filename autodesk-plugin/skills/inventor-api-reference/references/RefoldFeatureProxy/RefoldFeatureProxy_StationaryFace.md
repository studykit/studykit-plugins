# RefoldFeatureProxy.StationaryFace Property

Parent Object: [RefoldFeatureProxy](../RefoldFeatureProxy/RefoldFeatureProxy.md)

## Description

Property that returns the Face object that was specified as the stationary face during creation.

## Syntax

RefoldFeatureProxy.**StationaryFace**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2010
