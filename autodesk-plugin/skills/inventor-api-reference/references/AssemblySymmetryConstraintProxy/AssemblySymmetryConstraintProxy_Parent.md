# AssemblySymmetryConstraintProxy.Parent Property

Parent Object: [AssemblySymmetryConstraintProxy](../AssemblySymmetryConstraintProxy/AssemblySymmetryConstraintProxy.md)

## Description

Property that returns the parent of the object.

## Syntax

AssemblySymmetryConstraintProxy.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).

## Version

Introduced in version 2014
