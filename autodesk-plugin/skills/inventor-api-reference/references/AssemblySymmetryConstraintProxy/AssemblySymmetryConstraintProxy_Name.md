# AssemblySymmetryConstraintProxy.Name Property

Parent Object: [AssemblySymmetryConstraintProxy](../AssemblySymmetryConstraintProxy/AssemblySymmetryConstraintProxy.md)

## Description

Gets/Sets the displayable name of this constraint.

## Syntax

AssemblySymmetryConstraintProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2014
