# ColorAssetValue.Values Property

Parent Object: [ColorAssetValue](../ColorAssetValue/ColorAssetValue.md)

## Description

Gets and sets the values associated with this asset value. HasMultipleValues property indicates if this property will be returning more than one value.

## Syntax

ColorAssetValue.**Values**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).
