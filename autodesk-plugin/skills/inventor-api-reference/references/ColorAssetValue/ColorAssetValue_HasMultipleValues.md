# ColorAssetValue.HasMultipleValues Property

Parent Object: [ColorAssetValue](../ColorAssetValue/ColorAssetValue.md)

## Description

Gets the boolean flag that indicates if this value has multiple values or not.

## Syntax

ColorAssetValue.**HasMultipleValues**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.
