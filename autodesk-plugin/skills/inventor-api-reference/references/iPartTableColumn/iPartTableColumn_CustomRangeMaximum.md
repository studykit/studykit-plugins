# iPartTableColumn.CustomRangeMaximum Property

Parent Object: [iPartTableColumn](../iPartTableColumn/iPartTableColumn.md)

## Description

Gets and sets the maximum value for a custom parameter.

## Syntax

iPartTableColumn.**CustomRangeMaximum**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
