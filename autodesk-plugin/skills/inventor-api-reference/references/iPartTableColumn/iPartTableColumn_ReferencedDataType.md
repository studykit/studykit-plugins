# iPartTableColumn.ReferencedDataType Property

Parent Object: [iPartTableColumn](../iPartTableColumn/iPartTableColumn.md)

## Description

Property that returns the data type contained in the column.

## Syntax

iPartTableColumn.**ReferencedDataType**() As [iComponentColumnTypeEnum](../iComponentColumnTypeEnum.md)

## Property Value

This is a read only property whose value is an [iComponentColumnTypeEnum](../iComponentColumnTypeEnum.md).

## Version

Introduced in version 2011
