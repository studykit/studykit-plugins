# iPartTableColumn.CustomRangeMinimum Property

Parent Object: [iPartTableColumn](../iPartTableColumn/iPartTableColumn.md)

## Description

Gets and sets the minimum value for a custom parameter.

## Syntax

iPartTableColumn.**CustomRangeMinimum**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
