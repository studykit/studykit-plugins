# DrawingWeldingSymbolDefinition.Parent Property

Parent Object: [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DrawingWeldingSymbolDefinition.**Parent**() As [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Property Value

This is a read only property whose value is a [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md).

## Version

Introduced in version 2024
