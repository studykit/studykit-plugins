# DrawingWeldingSymbolDefinition.ClosedNoteTail Property

Parent Object: [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md)

## Description

Gets and sets whether to enclose the note text in a box or not.

## Syntax

DrawingWeldingSymbolDefinition.**ClosedNoteTail**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
