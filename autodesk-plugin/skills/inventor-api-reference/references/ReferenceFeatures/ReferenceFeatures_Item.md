# ReferenceFeatures.Item Property

Parent Object: [ReferenceFeatures](../ReferenceFeatures/ReferenceFeatures.md)

## Description

Returns the specified ReferenceFeature object from the collection.

## Syntax

ReferenceFeatures.**Item**( ***Index*** As Variant ) As [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md)

## Property Value

This is a read only property whose value is a [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the ReferenceFeature to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the ReferenceFeature name. If an out of range index or a name of a non-existent ReferenceFeature is provided, an error occurs. |

## Version

Introduced in version 5.3
