# ReferenceFeatures.Count Property

Parent Object: [ReferenceFeatures](../ReferenceFeatures/ReferenceFeatures.md)

## Description

Property that returns the number of items in the collection.

## Syntax

ReferenceFeatures.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5.3
