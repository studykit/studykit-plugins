# DirectEditScaleOperationProxy.Faces Property

Parent Object: [DirectEditScaleOperationProxy](../DirectEditScaleOperationProxy/DirectEditScaleOperationProxy.md)

## Description

Gets the faces that are operated in the direct edit operation.

## Syntax

DirectEditScaleOperationProxy.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2016
