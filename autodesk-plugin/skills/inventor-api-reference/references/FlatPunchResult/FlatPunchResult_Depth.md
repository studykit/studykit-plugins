# FlatPunchResult.Depth Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns the depth of the punch.

## Syntax

FlatPunchResult.**Depth**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
