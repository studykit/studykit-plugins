# FlatPunchResult.IsDirectionUp Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that whether the direction of the punch is up or down.

## Syntax

FlatPunchResult.**IsDirectionUp**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
