# FlatPunchResult.FoldedPosition Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns a Point indicating the position of the punch instance in the folded model.

## Syntax

FlatPunchResult.**FoldedPosition**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2008
