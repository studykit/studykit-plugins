# FlatPunchResult.FoldedAngle Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns an angle value indicating the punch orientation in the folded model.

## Syntax

FlatPunchResult.**FoldedAngle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
