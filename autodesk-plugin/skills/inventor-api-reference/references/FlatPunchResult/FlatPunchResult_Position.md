# FlatPunchResult.Position Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns a Point indicating the position of the punch instance in the flat pattern. This property may return Nothing if this information is not available.

## Syntax

FlatPunchResult.**Position**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2008
