# FlatPunchResult.HasDepth Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Gets whether there is a depth value associated with this punch result.

## Syntax

FlatPunchResult.**HasDepth**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
