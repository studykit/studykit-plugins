# FlatPunchResult.InternalName Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns a string that uniquely identifies a punch instance within the flat pattern.

## Syntax

FlatPunchResult.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2008
