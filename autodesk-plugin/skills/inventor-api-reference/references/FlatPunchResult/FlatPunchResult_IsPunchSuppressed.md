# FlatPunchResult.IsPunchSuppressed Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns whether the punch feature that resulted in this punch result is currently suppressed.

## Syntax

FlatPunchResult.**IsPunchSuppressed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
