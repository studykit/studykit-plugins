# GroundPlaneSettings.Plane Property

Parent Object: [GroundPlaneSettings](../GroundPlaneSettings/GroundPlaneSettings.md)

## Description

Property that returns the geometry of the ground plane. The returned Plane object provides information about the position and normal of the work plane.

## Syntax

GroundPlaneSettings.**Plane**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).

## Version

Introduced in version 2011
