# GroundPlaneSettings.FrontDirection Property

Parent Object: [GroundPlaneSettings](../GroundPlaneSettings/GroundPlaneSettings.md)

## Description

Gets front directon of the ground plane.

## Syntax

GroundPlaneSettings.**FrontDirection**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2012
