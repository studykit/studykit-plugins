# GroundPlaneSettings.MinorGridLineSpacing Property

Parent Object: [GroundPlaneSettings](../GroundPlaneSettings/GroundPlaneSettings.md)

## Description

Gets and sets the spacing between minor grid lines in centimeters.

## Syntax

GroundPlaneSettings.**MinorGridLineSpacing**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
