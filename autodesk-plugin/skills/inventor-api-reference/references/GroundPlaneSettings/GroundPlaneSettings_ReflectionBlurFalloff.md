# GroundPlaneSettings.ReflectionBlurFalloff Property

Parent Object: [GroundPlaneSettings](../GroundPlaneSettings/GroundPlaneSettings.md)

## Description

Gets and sets where the model reflection seems to fade away.

## Syntax

GroundPlaneSettings.**ReflectionBlurFalloff**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
