# ReferencedOLEFileDescriptor.ReferenceStatus Property

Parent Object: [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Description

Property that returns the current status of the file reference. This property returns kUpToDateReference for embeddings.

## Syntax

ReferencedOLEFileDescriptor.**ReferenceStatus**() As [ReferenceStatusEnum](../ReferenceStatusEnum.md)

## Property Value

This is a read only property whose value is a [ReferenceStatusEnum](../ReferenceStatusEnum.md).

## Version

Introduced in version 9
