# ReferencedOLEFileDescriptor.Parent Property

Parent Object: [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Description

Property that returns the parent Document (in Inventor) or ApprenticeServerDocument (in Apprentice).

## Syntax

ReferencedOLEFileDescriptor.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2011
