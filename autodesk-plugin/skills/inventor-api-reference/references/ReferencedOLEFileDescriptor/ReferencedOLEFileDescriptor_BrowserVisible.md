# ReferencedOLEFileDescriptor.BrowserVisible Property

Parent Object: [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Description

Gets/Sets visible of the file reference in 3rdPartyFolder.

## Syntax

ReferencedOLEFileDescriptor.**BrowserVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
