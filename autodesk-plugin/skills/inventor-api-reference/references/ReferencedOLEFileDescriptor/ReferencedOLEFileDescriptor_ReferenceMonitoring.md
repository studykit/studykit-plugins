# ReferencedOLEFileDescriptor.ReferenceMonitoring Property

Parent Object: [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Description

Gets/Sets whether the referenced file is monitored by Inventor for changes.

## Syntax

ReferencedOLEFileDescriptor.**ReferenceMonitoring**() As [ReferenceMonitoringEnum](../ReferenceMonitoringEnum.md)

## Property Value

This is a read/write property whose value is a [ReferenceMonitoringEnum](../ReferenceMonitoringEnum.md).

## Version

Introduced in version 2011
