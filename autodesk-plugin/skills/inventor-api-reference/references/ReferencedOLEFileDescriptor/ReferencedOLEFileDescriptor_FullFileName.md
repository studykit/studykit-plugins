# ReferencedOLEFileDescriptor.FullFileName Property

Parent Object: [ReferencedOLEFileDescriptor](../ReferencedOLEFileDescriptor/ReferencedOLEFileDescriptor.md)

## Description

Property that returns the full path name of the file as currently found (or the last known full file name, if reference is not found). This property returns a null string for an embedding descriptor.

## Syntax

ReferencedOLEFileDescriptor.**FullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
