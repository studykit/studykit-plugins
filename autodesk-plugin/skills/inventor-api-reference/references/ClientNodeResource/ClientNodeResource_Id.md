# ClientNodeResource.Id Property

Parent Object: [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Description

Property returning the integer identifier that was supplied during the creation of this resource.

## Syntax

ClientNodeResource.**Id**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 8
