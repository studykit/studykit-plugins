# ClientNodeResource.ExpandedSuppressedIconName Property

Parent Object: [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Description

Gets and sets the expanded icon name to be used when the associated data model object becomes suppressed.

## Syntax

ClientNodeResource.**ExpandedSuppressedIconName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
