# ClientNodeResource.IconName Property

Parent Object: [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Description

Gets and sets picture name associated with a ClientNodeResource object.

## Syntax

ClientNodeResource.**IconName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
