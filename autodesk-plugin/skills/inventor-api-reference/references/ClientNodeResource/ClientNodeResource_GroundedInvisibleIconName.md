# ClientNodeResource.GroundedInvisibleIconName Property

Parent Object: [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Description

Gets and sets the icon name to be used when the associated data model object becomes grounded and invisible.

## Syntax

ClientNodeResource.**GroundedInvisibleIconName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
