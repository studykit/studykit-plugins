# ClientNodeResource.Type Property

Parent Object: [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ClientNodeResource.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 8
