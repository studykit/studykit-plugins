# SymmetryConstraintProxy.Deletable Property

Parent Object: [SymmetryConstraintProxy](../SymmetryConstraintProxy/SymmetryConstraintProxy.md)

## Description

Indicates whether this object is deletable.

## Syntax

SymmetryConstraintProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
