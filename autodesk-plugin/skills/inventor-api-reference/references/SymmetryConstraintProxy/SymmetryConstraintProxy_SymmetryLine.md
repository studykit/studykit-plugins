# SymmetryConstraintProxy.SymmetryLine Property

Parent Object: [SymmetryConstraintProxy](../SymmetryConstraintProxy/SymmetryConstraintProxy.md)

## Description

Property that returns the sketch entity that defines the axis of symmetry.

## Syntax

SymmetryConstraintProxy.**SymmetryLine**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 6
