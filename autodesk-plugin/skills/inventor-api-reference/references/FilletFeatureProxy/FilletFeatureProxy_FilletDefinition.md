# FilletFeatureProxy.FilletDefinition Property

Parent Object: [FilletFeatureProxy](../FilletFeatureProxy/FilletFeatureProxy.md)

## Description

Property that returns the object which defines the various input that was used to create the fillet feature.

## Syntax

FilletFeatureProxy.**FilletDefinition**() As [FilletDefinition](../FilletDefinition/FilletDefinition.md)

## Property Value

This is a read only property whose value is a [FilletDefinition](../FilletDefinition/FilletDefinition.md).

## Version

Introduced in version 9
