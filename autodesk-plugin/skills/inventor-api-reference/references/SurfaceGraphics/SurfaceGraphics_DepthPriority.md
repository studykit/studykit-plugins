# SurfaceGraphics.DepthPriority Property

Parent Object: [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Description

Read-write property that allows you to specify a display priority to the surface graphics.

## Syntax

SurfaceGraphics.**DepthPriority**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2009
