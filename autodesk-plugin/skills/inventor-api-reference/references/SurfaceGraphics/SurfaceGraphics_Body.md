# SurfaceGraphics.Body Property

Parent Object: [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Description

Property that returns the SurfaceBody object associated with the SurfaceGraphics.

## Syntax

SurfaceGraphics.**Body**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2009
