# SurfaceGraphics.LineWeight Property

Parent Object: [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Description

Gets and sets the LineWeight applied to this surface graphics.

## Syntax

SurfaceGraphics.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
