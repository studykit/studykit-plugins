# SurfaceGraphics.RemoveViewSpaceAnchor Method

Parent Object: [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

SurfaceGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 2009
