# SurfaceGraphics.LineScale Property

Parent Object: [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Description

Gets and sets the LineScale applied to this surface graphics.

## Syntax

SurfaceGraphics.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
