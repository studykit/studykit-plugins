# ModelCompositeAnnotations Object

## Description

The ModelCompositeAnnotations collection object provides access to all of the annotation groups in a part or assembly.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ModelCompositeAnnotations/ModelCompositeAnnotations_Add.md) | Adds a new ModelCompositeAnnotation object. |
| [CreateDefinition](../ModelCompositeAnnotations/ModelCompositeAnnotations_CreateDefinition.md) | Creates a model composite annotation definition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ModelCompositeAnnotations/ModelCompositeAnnotations_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../ModelCompositeAnnotations/ModelCompositeAnnotations_Count.md) | Gets the number of items in this collection. |
| [Item](../ModelCompositeAnnotations/ModelCompositeAnnotations_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../ModelCompositeAnnotations/ModelCompositeAnnotations_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[ModelAnnotations.ModelCompositeAnnotations](../ModelAnnotations/ModelAnnotations_ModelCompositeAnnotations.md)
