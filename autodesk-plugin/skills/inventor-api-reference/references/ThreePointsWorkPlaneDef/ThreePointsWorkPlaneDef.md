# ThreePointsWorkPlaneDef Object

## Description

Object that allows you to get and set the information that specifies a work plane defined by three points.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef_GetData.md) | Method that gets all of the data defining a work planed defined by three points. Point1 to Point2 defines the positive X axis and Point3 defines the positive Y direction. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef_Parent.md) | Property returning the parent WorkPlane object. |
| [Point1](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef_Point1.md) | Property that returns the first point of a three point defined work plane. The first point defines the origin of the work plane. This object can be a WorkPoint, Vertex, or SketchPoint object. |
| [Point2](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef_Point2.md) | Property that returns the second point of a three point defined work plane. The second point defines the X-axis in relation to the first point. This object can be a WorkPoint, Vertex, or SketchPoint object. |
| [Point3](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef_Point3.md) | Property that returns the third point of a three point defined work plane. The third point defines the positive y-axis direction of the work plane. This object can be a WorkPoint, Vertex, or SketchPoint object. |
| [Type](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
