# LoftDefinition.LastSectionCondition Property

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Property that specifies the condition of the loft at the ending section.

## Syntax

LoftDefinition.**LastSectionCondition**() As [LoftConditionEnum](../LoftConditionEnum.md)

## Property Value

This is a read/write property whose value is a [LoftConditionEnum](../LoftConditionEnum.md).

## Version

Introduced in version 2008
