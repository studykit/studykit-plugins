# LoftDefinition.Sections Property

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Property that specifies the sections used for a loft.

## Syntax

LoftDefinition.**Sections**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2008
