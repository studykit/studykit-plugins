# TransactionsEnumerator.Item Property

Parent Object: [TransactionsEnumerator](../TransactionsEnumerator/TransactionsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

TransactionsEnumerator.**Item**( ***Index*** As Long ) As [Transaction](../Transaction/Transaction.md)

## Property Value

This is a read only property whose value is a [Transaction](../Transaction/Transaction.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the index of the Transaction to return. |

## Version

Introduced in version 5
