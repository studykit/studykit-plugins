# HelicalCurve.Parent Property

Parent Object: [HelicalCurve](../HelicalCurve/HelicalCurve.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

HelicalCurve.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2019
