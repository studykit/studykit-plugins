# BoundaryPatchFeatureProxy.NativeObject Property

Parent Object: [BoundaryPatchFeatureProxy](../BoundaryPatchFeatureProxy/BoundaryPatchFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

BoundaryPatchFeatureProxy.**NativeObject**() As [BoundaryPatchFeature](../BoundaryPatchFeature/BoundaryPatchFeature.md)

## Property Value

This is a read only property whose value is a [BoundaryPatchFeature](../BoundaryPatchFeature/BoundaryPatchFeature.md).

## Version

Introduced in version 9
