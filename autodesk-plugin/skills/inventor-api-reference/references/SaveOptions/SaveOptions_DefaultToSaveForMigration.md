# SaveOptions.DefaultToSaveForMigration Property

Parent Object: [SaveOptions](../SaveOptions/SaveOptions.md)

## Description

Gets/Sets the default save condition for a document that has been migrated from a previous version of Autodesk Inventor is closed without being explicitly saved.

## Syntax

SaveOptions.**DefaultToSaveForMigration**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2021
