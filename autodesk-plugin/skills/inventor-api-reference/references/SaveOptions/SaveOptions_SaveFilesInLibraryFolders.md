# SaveOptions.SaveFilesInLibraryFolders Property

Parent Object: [SaveOptions](../SaveOptions/SaveOptions.md)

## Description

Gets/Sets whether to save the files in library folders or not.

## Syntax

SaveOptions.**SaveFilesInLibraryFolders**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2021
