# SaveOptions.DefaultToSaveForManualUpdates Property

Parent Object: [SaveOptions](../SaveOptions/SaveOptions.md)

## Description

Gets/Sets the default save condition for a document that has manual updates.

## Syntax

SaveOptions.**DefaultToSaveForManualUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2021
