# SaveOptions.SaveReminderTimer Property

Parent Object: [SaveOptions](../SaveOptions/SaveOptions.md)

## Description

Gets/Sets the save reminder timer (in minutes).

## Syntax

SaveOptions.**SaveReminderTimer**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
