# CircularOccurrencePatternProxy.AxisEntityNaturalDirection Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Gets/Sets whether the axis direction is in the natural direction of the axis entity or not.

## Syntax

CircularOccurrencePatternProxy.**AxisEntityNaturalDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
