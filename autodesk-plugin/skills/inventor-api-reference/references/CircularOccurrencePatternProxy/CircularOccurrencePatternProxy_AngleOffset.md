# CircularOccurrencePatternProxy.AngleOffset Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Property returning the Parameter object that controls the angle offset between pattern instances.

## Syntax

CircularOccurrencePatternProxy.**AngleOffset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
