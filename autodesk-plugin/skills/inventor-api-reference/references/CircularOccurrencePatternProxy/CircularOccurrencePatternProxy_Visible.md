# CircularOccurrencePatternProxy.Visible Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Gets/Sets the Visible property for the pattern.

## Syntax

CircularOccurrencePatternProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
