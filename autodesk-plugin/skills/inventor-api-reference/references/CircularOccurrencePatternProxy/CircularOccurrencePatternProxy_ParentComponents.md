# CircularOccurrencePatternProxy.ParentComponents Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Property that gets and sets the components used as input for the pattern.

## Syntax

CircularOccurrencePatternProxy.**ParentComponents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2010
