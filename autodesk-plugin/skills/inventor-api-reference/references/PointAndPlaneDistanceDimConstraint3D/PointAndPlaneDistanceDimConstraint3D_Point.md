# PointAndPlaneDistanceDimConstraint3D.Point Property

Parent Object: [PointAndPlaneDistanceDimConstraint3D](../PointAndPlaneDistanceDimConstraint3D/PointAndPlaneDistanceDimConstraint3D.md)

## Description

Property that returns the 3D sketch point being constrained.

## Syntax

PointAndPlaneDistanceDimConstraint3D.**Point**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 11
