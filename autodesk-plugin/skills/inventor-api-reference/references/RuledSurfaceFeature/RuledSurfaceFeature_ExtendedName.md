# RuledSurfaceFeature.ExtendedName Property

Parent Object: [RuledSurfaceFeature](../RuledSurfaceFeature/RuledSurfaceFeature.md)

## Description

Read-only property that returns the full feature name including any extended information.

## Syntax

RuledSurfaceFeature.**ExtendedName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
