# CornerDefinition.AlignedCornerExtension Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Gets and sets how the corner is extened.

## Syntax

CornerDefinition.**AlignedCornerExtension**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
