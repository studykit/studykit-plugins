# CornerDefinition Object

## Description

The CornerDefinition object represents all of the information that defines a corner feature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../CornerDefinition/CornerDefinition_Copy.md) | Function that creates a copy of this CornerDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AlignedCornerExtension](../CornerDefinition/CornerDefinition_AlignedCornerExtension.md) | Gets and sets how the corner is extened. |
| [Application](../CornerDefinition/CornerDefinition_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [BendOptions](../CornerDefinition/CornerDefinition_BendOptions.md) | Gets and sets the BendOptions object that defines the bend options for this bend feature. |
| [CornerDefinitionType](../CornerDefinition/CornerDefinition_CornerDefinitionType.md) | Property that specifies the geometry that is used to measure the corner seam or miter gap. |
| [CornerOptions](../CornerDefinition/CornerDefinition_CornerOptions.md) | Gets and sets the CornerDefinition object that defines how the corners are modeled. |
| [CornerType](../CornerDefinition/CornerDefinition_CornerType.md) | Gets how the corner seam or miter is defined. |
| [Edges](../CornerDefinition/CornerDefinition_Edges.md) | Gets the EdgeCollection object that specifies either the two edges that represent the two faces to create the seam between or the set of the create a ripped edge seam on. |
| [GapWidth](../CornerDefinition/CornerDefinition_GapWidth.md) | Gets the width of the gap of a sheet metal corner feature. |
| [IsRippedCorner](../CornerDefinition/CornerDefinition_IsRippedCorner.md) | Gets if this corner is a seam or ripped corner. |
| [Parent](../CornerDefinition/CornerDefinition_Parent.md) | Property that returns the parent CornerFeature object of this CornerDefinition object. |
| [PercentOverlap](../CornerDefinition/CornerDefinition_PercentOverlap.md) | Property that specifies the overlap as a percentage of the flange thickness. |
| [Type](../CornerDefinition/CornerDefinition_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
