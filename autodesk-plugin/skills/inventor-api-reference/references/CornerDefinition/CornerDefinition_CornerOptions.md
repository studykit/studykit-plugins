# CornerDefinition.CornerOptions Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Gets and sets the CornerDefinition object that defines how the corners are modeled.

## Syntax

CornerDefinition.**CornerOptions**() As [CornerOptions](../CornerOptions/CornerOptions.md)

## Property Value

This is a read/write property whose value is a [CornerOptions](../CornerOptions/CornerOptions.md).

## Version

Introduced in version 2009
