# CornerDefinition.IsRippedCorner Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Gets if this corner is a seam or ripped corner.

## Syntax

CornerDefinition.**IsRippedCorner**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
