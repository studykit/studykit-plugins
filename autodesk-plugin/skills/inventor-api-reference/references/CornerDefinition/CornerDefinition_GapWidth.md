# CornerDefinition.GapWidth Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Gets the width of the gap of a sheet metal corner feature.

## Syntax

CornerDefinition.**GapWidth**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2009
