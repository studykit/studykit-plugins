# FreeformFeatureProxy.NativeObject Property

Parent Object: [FreeformFeatureProxy](../FreeformFeatureProxy/FreeformFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FreeformFeatureProxy.**NativeObject**() As [FreeformFeature](../FreeformFeature/FreeformFeature.md)

## Property Value

This is a read only property whose value is a [FreeformFeature](../FreeformFeature/FreeformFeature.md).

## Version

Introduced in version 2015
