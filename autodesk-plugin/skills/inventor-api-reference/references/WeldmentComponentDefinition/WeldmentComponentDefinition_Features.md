# WeldmentComponentDefinition.Features Property

Parent Object: [WeldmentComponentDefinition](../WeldmentComponentDefinition/WeldmentComponentDefinition.md)

## Description

Gets the Features collection for this assembly (assembly features).

## Syntax

WeldmentComponentDefinition.**Features**() As [Features](../Features/Features.md)

## Property Value

This is a read only property whose value is a [Features](../Features/Features.md).

## Version

Introduced in version 10
