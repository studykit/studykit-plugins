# WeldmentComponentDefinition.Constraints Property

Parent Object: [WeldmentComponentDefinition](../WeldmentComponentDefinition/WeldmentComponentDefinition.md)

## Description

Property that returns the AssemblyConstraints collection object.

## Syntax

WeldmentComponentDefinition.**Constraints**() As [AssemblyConstraints](../AssemblyConstraints/AssemblyConstraints.md)

## Property Value

This is a read only property whose value is an [AssemblyConstraints](../AssemblyConstraints/AssemblyConstraints.md).

## Version

Introduced in version 8
