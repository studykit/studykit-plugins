# SurfaceGraphicsEdge.LineDefinitionSpace Property

Parent Object: [SurfaceGraphicsEdge](../SurfaceGraphicsEdge/SurfaceGraphicsEdge.md)

## Description

Gets and sets the LineDefinitionSpace applied to this surface graphics edge.

## Syntax

SurfaceGraphicsEdge.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2009
