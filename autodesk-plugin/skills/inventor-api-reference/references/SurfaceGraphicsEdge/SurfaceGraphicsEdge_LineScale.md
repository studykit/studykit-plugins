# SurfaceGraphicsEdge.LineScale Property

Parent Object: [SurfaceGraphicsEdge](../SurfaceGraphicsEdge/SurfaceGraphicsEdge.md)

## Description

Gets and sets the LineScale applied to this surface graphics edge.

## Syntax

SurfaceGraphicsEdge.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
