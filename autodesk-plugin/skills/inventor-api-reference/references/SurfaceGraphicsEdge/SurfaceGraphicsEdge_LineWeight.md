# SurfaceGraphicsEdge.LineWeight Property

Parent Object: [SurfaceGraphicsEdge](../SurfaceGraphicsEdge/SurfaceGraphicsEdge.md)

## Description

Gets and sets the LineWeight applied to this surface graphice edge.

## Syntax

SurfaceGraphicsEdge.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
