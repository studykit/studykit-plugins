# SurfaceGraphicsEdge.Parent Property

Parent Object: [SurfaceGraphicsEdge](../SurfaceGraphicsEdge/SurfaceGraphicsEdge.md)

## Description

Property that returns the parent SurfaceGraphics object.

## Syntax

SurfaceGraphicsEdge.**Parent**() As [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md)

## Property Value

This is a read only property whose value is a [SurfaceGraphics](../SurfaceGraphics/SurfaceGraphics.md).

## Version

Introduced in version 2009
