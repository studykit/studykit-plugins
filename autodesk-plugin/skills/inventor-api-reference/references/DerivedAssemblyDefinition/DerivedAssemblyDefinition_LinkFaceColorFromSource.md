# DerivedAssemblyDefinition.LinkFaceColorFromSource Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Read-write property that gets and sets whether color override should be removed from the resulting derived part.

## Syntax

DerivedAssemblyDefinition.**LinkFaceColorFromSource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
