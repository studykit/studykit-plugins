# DerivedAssemblyDefinition.IncludeAllParameters Method

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Method that includes all parameters from the source assembly. This includes top level parameters in the source assembly as well as parameters in sub-assemblies and parts.

## Syntax

DerivedAssemblyDefinition.**IncludeAllParameters**()

## Version

Introduced in version 2017
