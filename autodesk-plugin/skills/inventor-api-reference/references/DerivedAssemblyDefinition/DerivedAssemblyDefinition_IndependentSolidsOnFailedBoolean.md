# DerivedAssemblyDefinition.IndependentSolidsOnFailedBoolean Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Read-write property that gets and sets whether to create independent solids in the derived part when a Boolean operation fails.

## Syntax

DerivedAssemblyDefinition.**IndependentSolidsOnFailedBoolean**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
