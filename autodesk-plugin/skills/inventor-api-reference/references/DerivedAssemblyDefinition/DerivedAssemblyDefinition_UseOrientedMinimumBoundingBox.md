# DerivedAssemblyDefinition.UseOrientedMinimumBoundingBox Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Read-write Boolean property that specifies whether to use the oriented minimum bounding box or orthogonal bounding box for the components that are included as bounding box.

## Syntax

DerivedAssemblyDefinition.**UseOrientedMinimumBoundingBox**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
