# DerivedAssemblyDefinition.MirrorPlane Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Gets and sets the plane about which the base assembly will be mirrored to create the derived assembly.

## Syntax

DerivedAssemblyDefinition.**MirrorPlane**() As [DerivedPartMirrorPlaneEnum](../DerivedPartMirrorPlaneEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedPartMirrorPlaneEnum](../DerivedPartMirrorPlaneEnum.md).

## Version

Introduced in version 2009
