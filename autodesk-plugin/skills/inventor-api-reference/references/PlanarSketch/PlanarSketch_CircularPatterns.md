# PlanarSketch.CircularPatterns Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets the SketchCircularPatterns collection object.

## Syntax

PlanarSketch.**CircularPatterns**() As [SketchCircularPatterns](../SketchCircularPatterns/SketchCircularPatterns.md)

## Property Value

This is a read only property whose value is a [SketchCircularPatterns](../SketchCircularPatterns/SketchCircularPatterns.md).

## Version

Introduced in version 2025.1
