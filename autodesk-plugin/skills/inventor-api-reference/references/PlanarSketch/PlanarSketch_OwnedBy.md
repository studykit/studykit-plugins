# PlanarSketch.OwnedBy Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the PartFeature object. This property should return the UnfoldFeature or RefoldFeature object that created the sketch.

## Syntax

PlanarSketch.**OwnedBy**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 2008
