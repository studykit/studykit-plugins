# PlanarSketch.LineType Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets and Sets the line type override for the sketch.

## Syntax

PlanarSketch.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2010
