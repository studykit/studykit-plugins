# PlanarSketch.PlanarEntityGeometry Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the geometry that describes the plane the sketch is based on.

## Syntax

PlanarSketch.**PlanarEntityGeometry**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).
