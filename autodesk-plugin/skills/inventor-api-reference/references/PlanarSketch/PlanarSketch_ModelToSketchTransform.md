# PlanarSketch.ModelToSketchTransform Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the transformation from model space to the 2d sketch coordinate space.

## Syntax

PlanarSketch.**ModelToSketchTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 6
