# PlanarSketch.SketchBlocks Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the SketchBlocks collection object. Only the first level sketch blocks in the sketch are returned. Use SketchBlock.ChildBlocks property recursively to get sketch blocks at all levels.

## Syntax

PlanarSketch.**SketchBlocks**() As [SketchBlocks](../SketchBlocks/SketchBlocks.md)

## Property Value

This is a read only property whose value is a [SketchBlocks](../SketchBlocks/SketchBlocks.md).
