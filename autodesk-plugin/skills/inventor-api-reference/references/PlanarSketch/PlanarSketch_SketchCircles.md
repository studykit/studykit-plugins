# PlanarSketch.SketchCircles Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

PlanarSketch.**SketchCircles**() As [SketchCircles](../SketchCircles/SketchCircles.md)

## Property Value

This is a read only property whose value is a [SketchCircles](../SketchCircles/SketchCircles.md).

## Version

Introduced in version 5
