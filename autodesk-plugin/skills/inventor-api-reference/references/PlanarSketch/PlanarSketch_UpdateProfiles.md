# PlanarSketch.UpdateProfiles Method

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Method that updates all the profiles within the sketch.

## Syntax

PlanarSketch.**UpdateProfiles**()

## Version
