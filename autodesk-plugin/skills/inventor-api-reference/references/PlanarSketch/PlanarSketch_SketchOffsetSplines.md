# PlanarSketch.SketchOffsetSplines Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the collection object. This collection provides access to the existing offset splines in the sketch.

## Syntax

PlanarSketch.**SketchOffsetSplines**() As [SketchOffsetSplines](../SketchOffsetSplines/SketchOffsetSplines.md)

## Property Value

This is a read only property whose value is a [SketchOffsetSplines](../SketchOffsetSplines/SketchOffsetSplines.md).

## Version

Introduced in version 9
