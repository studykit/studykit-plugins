# PlanarSketch.GeometricConstraints Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the collection of all geometric constraints on the sketch.

## Syntax

PlanarSketch.**GeometricConstraints**() As [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md)

## Property Value

This is a read only property whose value is a [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md).

## Version

Introduced in version 5
