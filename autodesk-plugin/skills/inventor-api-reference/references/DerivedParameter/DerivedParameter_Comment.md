# DerivedParameter.Comment Property

Parent Object: [DerivedParameter](../DerivedParameter/DerivedParameter.md)

## Description

Gets the user comments against this parameter.

## Syntax

DerivedParameter.**Comment**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
