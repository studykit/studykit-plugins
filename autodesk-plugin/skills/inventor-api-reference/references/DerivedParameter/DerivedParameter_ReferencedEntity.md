# DerivedParameter.ReferencedEntity Property

Parent Object: [DerivedParameter](../DerivedParameter/DerivedParameter.md)

## Description

Property that returns the Parameter that this parameter is dependent on.

## Syntax

DerivedParameter.**ReferencedEntity**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 11
