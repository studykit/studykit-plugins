# DerivedParameter.Parent Property

Parent Object: [DerivedParameter](../DerivedParameter/DerivedParameter.md)

## Description

Property that returns the parent Document of this parameter.

## Syntax

DerivedParameter.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
