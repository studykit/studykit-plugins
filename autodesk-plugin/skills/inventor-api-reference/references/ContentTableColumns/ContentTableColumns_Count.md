# ContentTableColumns.Count Property

Parent Object: [ContentTableColumns](../ContentTableColumns/ContentTableColumns.md)

## Description

Property that returns the number of ContentTableColumn objects in the collection.

## Syntax

ContentTableColumns.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
