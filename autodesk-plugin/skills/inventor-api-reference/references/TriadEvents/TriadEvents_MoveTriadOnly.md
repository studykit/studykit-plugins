# TriadEvents.MoveTriadOnly Property

Parent Object: [TriadEvents](../TriadEvents/TriadEvents.md)

## Description

Gets/sets a flag indicating whether to move the triad independent of the object that the triad is being used to move.

## Syntax

TriadEvents.**MoveTriadOnly**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
