# TriadEvents.GlobalTransform Property

Parent Object: [TriadEvents](../TriadEvents/TriadEvents.md)

## Description

Gets/Sets the current position of the Triad.

## Syntax

TriadEvents.**GlobalTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 9
