# TriadEvents.Repeat Property

Parent Object: [TriadEvents](../TriadEvents/TriadEvents.md)

## Description

Gets and sets a flag indicating whether the triad should be terminated after one sequence of moves.

## Syntax

TriadEvents.**Repeat**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
