# ClientFeatureDefinition.FeatureInputs Property

Parent Object: [ClientFeatureDefinition](../ClientFeatureDefinition/ClientFeatureDefinition.md)

## Description

Read-write property that gets and sets the objects that are inputs to the client feature.

## Syntax

ClientFeatureDefinition.**FeatureInputs**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2008
