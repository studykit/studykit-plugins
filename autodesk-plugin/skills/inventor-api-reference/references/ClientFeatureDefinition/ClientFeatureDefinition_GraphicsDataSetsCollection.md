# ClientFeatureDefinition.GraphicsDataSetsCollection Property

Parent Object: [ClientFeatureDefinition](../ClientFeatureDefinition/ClientFeatureDefinition.md)

## Description

Read-only property that returns the GraphicsDataSetsCollection object.

## Syntax

ClientFeatureDefinition.**GraphicsDataSetsCollection**() As [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md).

## Version

Introduced in version 2012
