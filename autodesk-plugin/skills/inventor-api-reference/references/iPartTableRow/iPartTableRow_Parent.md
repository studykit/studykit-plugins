# iPartTableRow.Parent Property

Parent Object: [iPartTableRow](../iPartTableRow/iPartTableRow.md)

## Description

Property that returns the parent iPartFactory object.

## Syntax

iPartTableRow.**Parent**() As [iPartFactory](../iPartFactory/iPartFactory.md)

## Property Value

This is a read only property whose value is an [iPartFactory](../iPartFactory/iPartFactory.md).

## Version

Introduced in version 6
