# ParameterTypeEnum Enumerator

## Description

Parameter Types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDerivedParameter | 11525 | Parameters automatically created from derived component. |
| kFinishParameter | 11526 | Parameters automatically created from finish feature. |
| kModelParameter | 11521 | Parameter created automatically by Inventor when a command requires it. |
| kReferenceParameter | 11522 | Parameter created automatically by Inventor while creating a driven dimension, for example. |
| kTableParameter | 11523 | Parameters created from a table; like a spreadsheet. |
| kUserParameter | 11524 | Parameters explicitly created by the user. |

## Version

Introduced in version 4
