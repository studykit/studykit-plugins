# CommandControl.UseLargeIcon Property

Parent Object: [CommandControl](../CommandControl/CommandControl.md)

## Description

Gets and sets whether to display the control in the larger size. The 'Ribbon Appearance' user choice overrides this setting.

## Syntax

CommandControl.**UseLargeIcon**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
