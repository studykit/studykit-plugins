# CommandControl.ChildControls Property

Parent Object: [CommandControl](../CommandControl/CommandControl.md)

## Description

Property that returns a CommandControls collection object that provides access to the child controls of this control.

## Syntax

CommandControl.**ChildControls**() As [CommandControls](../CommandControls/CommandControls.md)

## Property Value

This is a read only property whose value is a [CommandControls](../CommandControls/CommandControls.md).
