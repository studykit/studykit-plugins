# MoveFaceFeatureProxy.Definition Property

Parent Object: [MoveFaceFeatureProxy](../MoveFaceFeatureProxy/MoveFaceFeatureProxy.md)

## Description

Gets and sets the MoveFaceDefinition object associated with this feature.

## Syntax

MoveFaceFeatureProxy.**Definition**() As [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md)

## Property Value

This is a read/write property whose value is a [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md).

## Version

Introduced in version 2011
