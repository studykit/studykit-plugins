# DSLoad.LoadType Property

Parent Object: [DSLoad](../DSLoad/DSLoad.md)

## Description

Gets the type of load this load object represents.

## Syntax

DSLoad.**LoadType**() As [DSLoadTypeEnum](../DSLoadTypeEnum.md)

## Property Value

This is a read only property whose value is a [DSLoadTypeEnum](../DSLoadTypeEnum.md).

## Version

Introduced in version 2013
