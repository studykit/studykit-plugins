# DSLoad Object

## Description

DSLoad Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DSLoad/DSLoad_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Definition](../DSLoad/DSLoad_Definition.md) | Gets and sets the load definition object associated with this load. |
| [IsSuppressed](../DSLoad/DSLoad_IsSuppressed.md) | Gets whether the load is suppressed or not. A value of True indicates the load is suppressed. |
| [LoadType](../DSLoad/DSLoad_LoadType.md) | Gets the type of load this load object represents. |
| [Name](../DSLoad/DSLoad_Name.md) | Gets and sets the name of this load as seen in the browser. |
| [Type](../DSLoad/DSLoad_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DSLoadDefinition.Parent](../DSLoadDefinition/DSLoadDefinition_Parent.md), [DSLoads.Item](../DSLoads/DSLoads_Item.md)

## Version

Introduced in version 2013
