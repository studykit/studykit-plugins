# SketchCircle.LineWeight Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Gets and sets the LineWeight applied to this sketch circle.

## Syntax

SketchCircle.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
