# SketchCircle.Layer Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Gets and sets the layer applied to this sketch entity.

## Syntax

SketchCircle.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).
