# SketchCircle.Geometry3d Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Read-only property that returns circle geometry that represents this circle in model space.

## Syntax

SketchCircle.**Geometry3d**() As [Circle](../Circle/Circle.md)

## Property Value

This is a read only property whose value is a [Circle](../Circle/Circle.md).

## Version

Introduced in version 2013
