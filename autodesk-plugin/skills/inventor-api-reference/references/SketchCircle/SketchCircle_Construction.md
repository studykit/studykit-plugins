# SketchCircle.Construction Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Gets and sets whether the entity behaves as a construction entity or not.

## Syntax

SketchCircle.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
