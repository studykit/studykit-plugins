# ModelHoleThreadNotes.Add Method

Parent Object: [ModelHoleThreadNotes](../ModelHoleThreadNotes/ModelHoleThreadNotes.md)

## Description

Method that creates a thread or hole note.

## Syntax

ModelHoleThreadNotes.**Add**( ***Definition*** As [ModelHoleThreadNoteDefinition](../ModelHoleThreadNoteDefinition/ModelHoleThreadNoteDefinition.md) ) As [ModelHoleThreadNote](../ModelHoleThreadNote/ModelHoleThreadNote.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ModelHoleThreadNoteDefinition](../ModelHoleThreadNoteDefinition/ModelHoleThreadNoteDefinition.md) | Input ModelHoleThreadNoteDefinition object that defines the hole or thread note to be created. |

## Version

Introduced in version 2018
