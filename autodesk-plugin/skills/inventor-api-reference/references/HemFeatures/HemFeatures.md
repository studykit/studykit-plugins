# HemFeatures Object

## Description

The HemFeatures collection object provides access to all of the objects in a sheet metal component definition.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../HemFeatures/HemFeatures_Add.md) | Method that creates a new hem feature. The newly created HemFeature object is returned. |
| [CreateHemDefinition](../HemFeatures/HemFeatures_CreateHemDefinition.md) | Method that creates a new HemDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HemFeatures/HemFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../HemFeatures/HemFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../HemFeatures/HemFeatures_Item.md) | Returns the specified HemFeature object from the collection. This is the default property of the HemFeatures collection object. |
| [Type](../HemFeatures/HemFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SheetMetalFeatures.HemFeatures](../SheetMetalFeatures/SheetMetalFeatures_HemFeatures.md)

## Version

Introduced in version 5.3
