# HemFeatures.Item Property

Parent Object: [HemFeatures](../HemFeatures/HemFeatures.md)

## Description

Returns the specified HemFeature object from the collection. This is the default property of the HemFeatures collection object.

## Syntax

HemFeatures.**Item**( ***Index*** As Variant ) As [HemFeature](../HemFeature/HemFeature.md)

## Property Value

This is a read only property whose value is a [HemFeature](../HemFeature/HemFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the to return. |

## Version

Introduced in version 5.3
