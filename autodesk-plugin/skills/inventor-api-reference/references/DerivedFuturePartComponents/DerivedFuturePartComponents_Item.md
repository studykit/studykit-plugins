# DerivedFuturePartComponents.Item Property

Parent Object: [DerivedFuturePartComponents](../DerivedFuturePartComponents/DerivedFuturePartComponents.md)

## Description

Method that returns the specified DerivedFuturePartComponent object from the collection.

## Syntax

DerivedFuturePartComponents.**Item**( ***Value*** As Long ) As [DerivedFuturePartComponent](../DerivedFuturePartComponent/DerivedFuturePartComponent.md)

## Property Value

This is a read only property whose value is a [DerivedFuturePartComponent](../DerivedFuturePartComponent/DerivedFuturePartComponent.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | Long | Input Variant value that specifies the DerivedFuturePartComponent to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the derived component’s name. If an out of range index or a name of a non-existent DerivedFuturePartComponent is provided, an error occurs. |

## Version

Introduced in version 2018
