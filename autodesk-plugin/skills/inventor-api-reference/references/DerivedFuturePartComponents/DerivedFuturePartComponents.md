# DerivedFuturePartComponents Object

## Description

The DerivedFuturePartComponents collection object provides access to all of the existing DerivedFuturePartComponent objects in a part and provides methods to create additional derived components.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../DerivedFuturePartComponents/DerivedFuturePartComponents_Add.md) | Method that creates a new DerivedFuturePartComponent using the information supplied by the input DerivedFuturePartDefinition object. If successful, the new DerivedFuturePartComponent is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DerivedFuturePartComponents/DerivedFuturePartComponents_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DerivedFuturePartComponents/DerivedFuturePartComponents_Count.md) | Gets the number of items in this collection. |
| [Item](../DerivedFuturePartComponents/DerivedFuturePartComponents_Item.md) | Method that returns the specified DerivedFuturePartComponent object from the collection. |
| [Parent](../DerivedFuturePartComponents/DerivedFuturePartComponents_Parent.md) | Read-only property that returns the ComponentDefinition object. |
| [Type](../DerivedFuturePartComponents/DerivedFuturePartComponents_Type.md) | Read-only property returning kDerivedFuturePartComponentsObject indicating the type of object. |

## Accessed From

[ReferenceComponents.DerivedFuturePartComponents](../ReferenceComponents/ReferenceComponents_DerivedFuturePartComponents.md)

## Version

Introduced in version 2018
