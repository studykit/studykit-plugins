# Rows.Count Property

Parent Object: [Rows](../Rows/Rows.md)

## Description

Property that returns the number of items in the collection.

## Syntax

Rows.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 9
