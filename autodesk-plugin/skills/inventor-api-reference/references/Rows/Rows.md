# Rows Object

## Description

The Rows collection object represents all the objects contained by this custom table. See the Custom Tables overview for more information.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Rows/Rows_Add.md) | Method that creates a new Row. The new created Row is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Rows/Rows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Rows/Rows_Count.md) | Property that returns the number of items in the collection. |
| [Item](../Rows/Rows_Item.md) | Returns the specified Row object from the collection. |
| [Type](../Rows/Rows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[CustomTable.Rows](../CustomTable/CustomTable_Rows.md)

## Version

Introduced in version 9
