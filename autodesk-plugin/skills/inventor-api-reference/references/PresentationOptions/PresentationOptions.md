# PresentationOptions Object

## Description

The PresentationOptions object provides access to properties that provide read and write access of the presentation related application options.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PresentationOptions/PresentationOptions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [SkipAllUnresolvedFiles](../PresentationOptions/PresentationOptions_SkipAllUnresolvedFiles.md) | Enables/disables skip all unresolved files during openning. |
| [Type](../PresentationOptions/PresentationOptions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Application.PresentationOptions](../Application/Application_PresentationOptions.md), [InventorServer.PresentationOptions](InventorServer_PresentationOptions.md), [InventorServerObject.PresentationOptions](InventorServerObject_PresentationOptions.md)

## Version

Introduced in version 2011
