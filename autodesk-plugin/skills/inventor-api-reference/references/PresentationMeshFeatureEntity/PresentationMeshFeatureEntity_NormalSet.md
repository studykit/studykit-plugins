# PresentationMeshFeatureEntity.NormalSet Property

Parent Object: [PresentationMeshFeatureEntity](../PresentationMeshFeatureEntity/PresentationMeshFeatureEntity.md)

## Description

Read-only property that returns the normal set of the mesh feature entity.

## Syntax

PresentationMeshFeatureEntity.**NormalSet**() As [GraphicsNormalSet](../GraphicsNormalSet/GraphicsNormalSet.md)

## Property Value

This is a read only property whose value is a [GraphicsNormalSet](../GraphicsNormalSet/GraphicsNormalSet.md).

## Version

Introduced in version 2018
