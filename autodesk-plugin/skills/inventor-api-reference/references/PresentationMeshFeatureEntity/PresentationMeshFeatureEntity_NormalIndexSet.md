# PresentationMeshFeatureEntity.NormalIndexSet Property

Parent Object: [PresentationMeshFeatureEntity](../PresentationMeshFeatureEntity/PresentationMeshFeatureEntity.md)

## Description

Read-only property that returns the normal index set of the mesh feature entity.

## Syntax

PresentationMeshFeatureEntity.**NormalIndexSet**() As [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md)

## Property Value

This is a read only property whose value is a [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md).

## Version

Introduced in version 2018
