# PresentationMeshFeatureEntity.Parent Property

Parent Object: [PresentationMeshFeatureEntity](../PresentationMeshFeatureEntity/PresentationMeshFeatureEntity.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PresentationMeshFeatureEntity.**Parent**() As [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Property Value

This is a read only property whose value is a [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md).

## Version

Introduced in version 2018
