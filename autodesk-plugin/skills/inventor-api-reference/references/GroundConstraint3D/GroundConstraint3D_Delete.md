# GroundConstraint3D.Delete Method

Parent Object: [GroundConstraint3D](../GroundConstraint3D/GroundConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

GroundConstraint3D.**Delete**()

## Version

Introduced in version 10
