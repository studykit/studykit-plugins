# ModelWeldingSymbolProxy.OwnedByToleranceFeature Property

Parent Object: [ModelWeldingSymbolProxy](../ModelWeldingSymbolProxy/ModelWeldingSymbolProxy.md)

## Description

Returns the owning ModelToleranceFeature object.

## Syntax

ModelWeldingSymbolProxy.**OwnedByToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2024
