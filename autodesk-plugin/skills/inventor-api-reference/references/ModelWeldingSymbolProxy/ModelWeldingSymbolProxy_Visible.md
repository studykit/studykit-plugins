# ModelWeldingSymbolProxy.Visible Property

Parent Object: [ModelWeldingSymbolProxy](../ModelWeldingSymbolProxy/ModelWeldingSymbolProxy.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelWeldingSymbolProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
