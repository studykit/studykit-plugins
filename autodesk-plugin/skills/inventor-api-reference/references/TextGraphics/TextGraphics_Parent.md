# TextGraphics.Parent Property

Parent Object: [TextGraphics](../TextGraphics/TextGraphics.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

TextGraphics.**Parent**() As [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Property Value

This is a read only property whose value is a [GraphicsNode](../GraphicsNode/GraphicsNode.md).

## Version

Introduced in version 5
