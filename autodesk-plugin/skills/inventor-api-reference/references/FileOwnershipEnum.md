# FileOwnershipEnum Enumerator

## Description

Enum indicating the ownership over other referenced file.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kExclusiveOwnership | 68611 | Indicating that this file is exclusively owned and cannot be referenced by any other file. |
| kNoOwnership | 68609 | No ownership over other referenced file. |
| kSaveOwnership | 68610 | Indicating that this file cannot be saved on its own and must be saved with the referencing file. |

## Version

Introduced in version 2008
