# MaterialAsset.AssetType Property

Parent Object: [MaterialAsset](../MaterialAsset/MaterialAsset.md)

## Description

Gets the data type that returns whether this is an appearance, physical properties, or material asset.

## Syntax

MaterialAsset.**AssetType**() As [AssetTypeEnum](../AssetTypeEnum.md)

## Property Value

This is a read only property whose value is an [AssetTypeEnum](../AssetTypeEnum.md).

## Version

Introduced in version 2014
