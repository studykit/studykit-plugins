# MaterialAsset.LocalType Property

Parent Object: [MaterialAsset](../MaterialAsset/MaterialAsset.md)

## Description

Returns the local type of this asset. This is applicable to appearance asset.

## Syntax

MaterialAsset.**LocalType**() As [AssetLocalTypeEnum](../AssetLocalTypeEnum.md)

## Property Value

This is a read only property whose value is an [AssetLocalTypeEnum](../AssetLocalTypeEnum.md).

## Version

Introduced in version 2023
