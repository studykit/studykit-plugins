# MaterialAsset.CategoryName Property

Parent Object: [MaterialAsset](../MaterialAsset/MaterialAsset.md)

## Description

Gets the name of the category this Asset is designated to be in. This can include the the category and subcategories which are delimited by a colon.

## Syntax

MaterialAsset.**CategoryName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2014
