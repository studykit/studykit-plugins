# DWGArcProxy.NativeObject Property

Parent Object: [DWGArcProxy](../DWGArcProxy/DWGArcProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGArcProxy.**NativeObject**() As [DWGArc](../DWGArc/DWGArc.md)

## Property Value

This is a read only property whose value is a [DWGArc](../DWGArc/DWGArc.md).

## Version

Introduced in version 2016
