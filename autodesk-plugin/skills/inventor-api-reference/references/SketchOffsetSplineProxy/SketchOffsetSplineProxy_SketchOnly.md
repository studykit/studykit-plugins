# SketchOffsetSplineProxy.SketchOnly Property

Parent Object: [SketchOffsetSplineProxy](../SketchOffsetSplineProxy/SketchOffsetSplineProxy.md)

## Description

Gets and sets whether this entity is visible only when editing the sketch.

## Syntax

SketchOffsetSplineProxy.**SketchOnly**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
