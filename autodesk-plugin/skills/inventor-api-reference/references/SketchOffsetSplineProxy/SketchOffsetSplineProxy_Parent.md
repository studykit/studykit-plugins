# SketchOffsetSplineProxy.Parent Property

Parent Object: [SketchOffsetSplineProxy](../SketchOffsetSplineProxy/SketchOffsetSplineProxy.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchOffsetSplineProxy.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version
