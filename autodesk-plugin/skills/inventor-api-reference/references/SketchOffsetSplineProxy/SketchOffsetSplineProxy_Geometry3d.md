# SketchOffsetSplineProxy.Geometry3d Property

Parent Object: [SketchOffsetSplineProxy](../SketchOffsetSplineProxy/SketchOffsetSplineProxy.md)

## Description

Read-only property that returns b-spline geometry that represents this spline in model space.

## Syntax

SketchOffsetSplineProxy.**Geometry3d**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read only property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 2013
