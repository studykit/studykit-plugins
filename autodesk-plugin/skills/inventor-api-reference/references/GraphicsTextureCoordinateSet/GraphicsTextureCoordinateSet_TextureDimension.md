# GraphicsTextureCoordinateSet.TextureDimension Property

Parent Object: [GraphicsTextureCoordinateSet](../GraphicsTextureCoordinateSet/GraphicsTextureCoordinateSet.md)

## Description

Read-write property that defines the dimension of the texture coordinate array.

## Syntax

GraphicsTextureCoordinateSet.**TextureDimension**() As [TextureDimensionEnum](../TextureDimensionEnum.md)

## Property Value

This is a read/write property whose value is a [TextureDimensionEnum](../TextureDimensionEnum.md).

## Version

Introduced in version 2012
