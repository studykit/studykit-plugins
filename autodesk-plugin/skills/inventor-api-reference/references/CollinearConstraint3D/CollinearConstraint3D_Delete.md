# CollinearConstraint3D.Delete Method

Parent Object: [CollinearConstraint3D](../CollinearConstraint3D/CollinearConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

CollinearConstraint3D.**Delete**()

## Version

Introduced in version 10
