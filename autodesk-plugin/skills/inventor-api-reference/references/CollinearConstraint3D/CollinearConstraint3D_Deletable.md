# CollinearConstraint3D.Deletable Property

Parent Object: [CollinearConstraint3D](../CollinearConstraint3D/CollinearConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

CollinearConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
