# RevitExport.Definition Property

Parent Object: [RevitExport](../RevitExport/RevitExport.md)

## Description

Read-write property that returns the RevitExport definition that defines the current state of the RevitExport.

## Syntax

RevitExport.**Definition**() As [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Property Value

This is a read/write property whose value is a [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md).

## Version

Introduced in version 2022
