# RevitExport.Name Property

Parent Object: [RevitExport](../RevitExport/RevitExport.md)

## Description

Read-only property that gets browser node name of RevitExport..

## Syntax

RevitExport.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
