# MoveDefinition.SurfaceBodies Property

Parent Object: [MoveDefinition](../MoveDefinition/MoveDefinition.md)

## Description

Read-write property that gets and sets the set of bodies that are affected by this feature.

## Syntax

MoveDefinition.**SurfaceBodies**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2013
