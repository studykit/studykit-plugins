# UnfoldFeatures Object

## Description

Object allowing iteration of the collection of UnfoldFeature objects, and allows creation of UnfoldFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../UnfoldFeatures/UnfoldFeatures_Add.md) | Method that creates a new unfold feature. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../UnfoldFeatures/UnfoldFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../UnfoldFeatures/UnfoldFeatures_Count.md) | Property that returns the number of UnfoldFeature objects in the collection. |
| [Item](../UnfoldFeatures/UnfoldFeatures_Item.md) | Returns the specified UnfoldFeature object from the collection. |
| [Type](../UnfoldFeatures/UnfoldFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SheetMetalFeatures.UnfoldFeatures](../SheetMetalFeatures/SheetMetalFeatures_UnfoldFeatures.md)

## Version

Introduced in version 2010
