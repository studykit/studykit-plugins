# ObjectDefaultsStyle.CropCutLineStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Read-write property that gets and sets the default style for crop cut line.

## Syntax

ObjectDefaultsStyle.**CropCutLineStyle**() As [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Property Value

This is a read/write property whose value is a [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md).

## Version

Introduced in version 2025.1
