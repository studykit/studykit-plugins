# ObjectDefaultsStyle.FeatureControlFrameStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for feature control frames.

## Syntax

ObjectDefaultsStyle.**FeatureControlFrameStyle**() As [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Property Value

This is a read/write property whose value is a [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md).

## Version

Introduced in version 2010
