# ObjectDefaultsStyle.ChainDimensionLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Read-write property that gets and sets the default layer for chain dimension sets.

## Syntax

ObjectDefaultsStyle.**ChainDimensionLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2011
