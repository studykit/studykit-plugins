# ObjectDefaultsStyle.WeldSymbolStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for weld symbols.

## Syntax

ObjectDefaultsStyle.**WeldSymbolStyle**() As [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Property Value

This is a read/write property whose value is a [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md).

## Version

Introduced in version 2024
