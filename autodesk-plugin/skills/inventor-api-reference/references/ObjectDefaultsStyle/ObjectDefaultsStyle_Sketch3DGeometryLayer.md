# ObjectDefaultsStyle.Sketch3DGeometryLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default layer for 3D sketch geometries.

## Syntax

ObjectDefaultsStyle.**Sketch3DGeometryLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2010
