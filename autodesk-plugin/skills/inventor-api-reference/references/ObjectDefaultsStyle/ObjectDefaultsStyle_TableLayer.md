# ObjectDefaultsStyle.TableLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default layer for general (custom) tables.

## Syntax

ObjectDefaultsStyle.**TableLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2011
