# ObjectDefaultsStyle.HoleTableStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for hole tables.

## Syntax

ObjectDefaultsStyle.**HoleTableStyle**() As [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md)

## Property Value

This is a read/write property whose value is a [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md).

## Version

Introduced in version 2010
