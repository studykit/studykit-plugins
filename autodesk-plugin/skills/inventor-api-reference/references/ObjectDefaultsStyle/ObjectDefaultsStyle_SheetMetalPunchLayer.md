# ObjectDefaultsStyle.SheetMetalPunchLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default layer for a center mark identifying a sheet metal punch origin.

## Syntax

ObjectDefaultsStyle.**SheetMetalPunchLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2010
