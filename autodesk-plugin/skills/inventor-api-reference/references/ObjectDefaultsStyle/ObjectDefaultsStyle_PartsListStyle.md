# ObjectDefaultsStyle.PartsListStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for parts lists.

## Syntax

ObjectDefaultsStyle.**PartsListStyle**() As [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Property Value

This is a read/write property whose value is a [PartsListStyle](../PartsListStyle/PartsListStyle.md).

## Version

Introduced in version 2011
