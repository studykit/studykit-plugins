# ObjectDefaultsStyle.CenterOfGravityStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for center mark identifying the center of gravity.

## Syntax

ObjectDefaultsStyle.**CenterOfGravityStyle**() As [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Property Value

This is a read/write property whose value is a [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md).

## Version

Introduced in version 2010
