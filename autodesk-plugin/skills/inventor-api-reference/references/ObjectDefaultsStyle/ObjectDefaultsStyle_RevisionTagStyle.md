# ObjectDefaultsStyle.RevisionTagStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for revision tags.

## Syntax

ObjectDefaultsStyle.**RevisionTagStyle**() As [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Property Value

This is a read/write property whose value is a [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md).

## Version

Introduced in version 2011
