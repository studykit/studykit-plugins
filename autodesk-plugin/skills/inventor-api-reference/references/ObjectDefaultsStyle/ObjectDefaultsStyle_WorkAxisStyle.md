# ObjectDefaultsStyle.WorkAxisStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for a centerline or center mark identifying a work axis.

## Syntax

ObjectDefaultsStyle.**WorkAxisStyle**() As [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Property Value

This is a read/write property whose value is a [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md).

## Version

Introduced in version 2010
