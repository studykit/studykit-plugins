# ObjectDefaultsStyle.OrdinateSetDimensionStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default dimension style for ordinate set dimensions.

## Syntax

ObjectDefaultsStyle.**OrdinateSetDimensionStyle**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 9
