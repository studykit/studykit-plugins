# ObjectDefaultsStyle.SketchedSymbolLeaderStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default leader style for user defined symbols.

## Syntax

ObjectDefaultsStyle.**SketchedSymbolLeaderStyle**() As [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Property Value

This is a read/write property whose value is a [LeaderStyle](../LeaderStyle/LeaderStyle.md).

## Version

Introduced in version 2010
