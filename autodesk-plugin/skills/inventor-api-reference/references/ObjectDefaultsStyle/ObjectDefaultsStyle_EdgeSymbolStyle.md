# ObjectDefaultsStyle.EdgeSymbolStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Read-write property that gets and sets the default style for edge symbols.

## Syntax

ObjectDefaultsStyle.**EdgeSymbolStyle**() As [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md)

## Property Value

This is a read/write property whose value is an [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md).

## Version

Introduced in version 2024
