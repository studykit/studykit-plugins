# ObjectDefaultsStyle.ChamferNoteStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default dimension style for chamfer notes.

## Syntax

ObjectDefaultsStyle.**ChamferNoteStyle**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 2010
