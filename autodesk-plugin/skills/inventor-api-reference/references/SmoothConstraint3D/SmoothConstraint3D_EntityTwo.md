# SmoothConstraint3D.EntityTwo Property

Parent Object: [SmoothConstraint3D](../SmoothConstraint3D/SmoothConstraint3D.md)

## Description

Property that returns the sketch entity that is constrained to be smooth (G2-continuous) to entity one.

## Syntax

SmoothConstraint3D.**EntityTwo**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 11
