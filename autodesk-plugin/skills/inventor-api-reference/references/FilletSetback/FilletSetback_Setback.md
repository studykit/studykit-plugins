# FilletSetback.Setback Property

Parent Object: [FilletSetback](../FilletSetback/FilletSetback.md)

## Description

Gets and sets the setback value.

## Syntax

FilletSetback.**Setback**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 5.3
