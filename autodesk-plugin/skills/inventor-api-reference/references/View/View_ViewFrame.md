# View.ViewFrame Property

Parent Object: [View](../View/View.md)

## Description

Read-only property that returns the ViewFrame this view is located in.

## Syntax

View.**ViewFrame**() As [ViewFrame](../ViewFrame/ViewFrame.md)

## Property Value

This is a read only property whose value is a [ViewFrame](../ViewFrame/ViewFrame.md).

## Version

Introduced in version 2021
