# View.ViewTab Property

Parent Object: [View](../View/View.md)

## Description

Read-only property that returns the ViewTab of this view.

## Syntax

View.**ViewTab**() As [ViewTab](../ViewTab/ViewTab.md)

## Property Value

This is a read only property whose value is a [ViewTab](../ViewTab/ViewTab.md).
