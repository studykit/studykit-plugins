# FromToWidthExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The FromToWidthExtent is used by sheet metal features when the width of the feature is defined from one entity to another entity.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FromToWidthExtent/FromToWidthExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [FromEntity](../FromToWidthExtent/FromToWidthExtent_FromEntity.md) | Property that gets and set the entity that defines the 'from' extents of the feature. |
| [Parent](../FromToWidthExtent/FromToWidthExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [ToEntity](../FromToWidthExtent/FromToWidthExtent_ToEntity.md) | Property that gets and set the entity that defines the 'to' extents of the feature. |
| [Type](../FromToWidthExtent/FromToWidthExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2009
