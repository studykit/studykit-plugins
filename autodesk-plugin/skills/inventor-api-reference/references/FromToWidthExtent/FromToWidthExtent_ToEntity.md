# FromToWidthExtent.ToEntity Property

Parent Object: [FromToWidthExtent](../FromToWidthExtent/FromToWidthExtent.md)

## Description

Property that gets and set the entity that defines the 'to' extents of the feature.

## Syntax

FromToWidthExtent.**ToEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2009
