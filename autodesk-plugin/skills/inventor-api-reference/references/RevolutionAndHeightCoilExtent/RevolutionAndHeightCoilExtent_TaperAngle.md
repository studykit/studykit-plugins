# RevolutionAndHeightCoilExtent.TaperAngle Property

Parent Object: [RevolutionAndHeightCoilExtent](../RevolutionAndHeightCoilExtent/RevolutionAndHeightCoilExtent.md)

## Description

Property that returns the parameter that controls the taper angle of the coil feature.

## Syntax

RevolutionAndHeightCoilExtent.**TaperAngle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 5.3
