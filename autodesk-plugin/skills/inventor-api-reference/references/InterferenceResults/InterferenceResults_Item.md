# InterferenceResults.Item Property

Parent Object: [InterferenceResults](../InterferenceResults/InterferenceResults.md)

## Description

Returns the specified InterferenceResult object from the collection.

## Syntax

InterferenceResults.**Item**( ***Index*** As Long ) As [InterferenceResult](../InterferenceResult/InterferenceResult.md)

## Property Value

This is a read only property whose value is an [InterferenceResult](../InterferenceResult/InterferenceResult.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input value that specifies the InterferenceResult object to return. If an out-of-range index is provided, an error will occur. |

## Version

Introduced in version 6
