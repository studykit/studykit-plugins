# InterferenceResults.Type Property

Parent Object: [InterferenceResults](../InterferenceResults/InterferenceResults.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

InterferenceResults.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
