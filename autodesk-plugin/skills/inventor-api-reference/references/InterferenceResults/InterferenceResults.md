# InterferenceResults Object

## Description

In an assembly, two or more components cannot occupy the same space at the same time. To detect such errors, you can analyze assemblies for interference.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../InterferenceResults/InterferenceResults_Count.md) | Property that returns the number of items in the collection. |
| [Item](../InterferenceResults/InterferenceResults_Item.md) | Returns the specified InterferenceResult object from the collection. |
| [Type](../InterferenceResults/InterferenceResults_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
