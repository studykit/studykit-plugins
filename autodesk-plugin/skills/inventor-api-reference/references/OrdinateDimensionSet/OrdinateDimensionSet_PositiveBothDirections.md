# OrdinateDimensionSet.PositiveBothDirections Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets whether dimension values should be positive in both directions of the origin.

## Syntax

OrdinateDimensionSet.**PositiveBothDirections**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
