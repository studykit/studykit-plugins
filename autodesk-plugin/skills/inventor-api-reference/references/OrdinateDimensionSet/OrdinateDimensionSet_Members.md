# OrdinateDimensionSet.Members Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Property that returns all the member OrdinateDimension objects.

## Syntax

OrdinateDimensionSet.**Members**() As [OrdinateDimensionsEnumerator](../OrdinateDimensionsEnumerator/OrdinateDimensionsEnumerator.md)

## Property Value

This is a read only property whose value is an [OrdinateDimensionsEnumerator](../OrdinateDimensionsEnumerator/OrdinateDimensionsEnumerator.md).

## Version

Introduced in version 2010
