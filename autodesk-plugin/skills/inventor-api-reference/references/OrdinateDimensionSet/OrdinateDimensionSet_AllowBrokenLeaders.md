# OrdinateDimensionSet.AllowBrokenLeaders Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets whether to allow jog points (movable vertices) on the dimensions.

## Syntax

OrdinateDimensionSet.**AllowBrokenLeaders**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
