# OrdinateDimensionSet.Precision Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets the number of decimal places displayed for all the members in this set.

## Syntax

OrdinateDimensionSet.**Precision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
