# OrdinateDimensionSet.AlignMembers Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets whether to align the text and the jog points of all the members.

## Syntax

OrdinateDimensionSet.**AlignMembers**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
