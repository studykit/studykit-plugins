# OrdinateDimensionSet.DirectionReversed Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets whether to reverse the direction of positive numbers.

## Syntax

OrdinateDimensionSet.**DirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
