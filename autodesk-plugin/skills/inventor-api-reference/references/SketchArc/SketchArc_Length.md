# SketchArc.Length Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Property that returns the length of the entity in centimeters.

## Syntax

SketchArc.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 5
