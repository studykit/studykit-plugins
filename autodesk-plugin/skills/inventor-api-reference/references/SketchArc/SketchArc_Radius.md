# SketchArc.Radius Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Property that returns the radius of the arc.

## Syntax

SketchArc.**Radius**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 5
