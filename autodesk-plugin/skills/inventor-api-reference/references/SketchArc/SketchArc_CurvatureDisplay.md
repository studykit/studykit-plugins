# SketchArc.CurvatureDisplay Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Specifies whether curvature information is displayed for the arc.

## Syntax

SketchArc.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
