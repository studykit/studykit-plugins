# DriveConstraintSettings.StartValue Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Gets and sets the expression for the start value.

## Syntax

DriveConstraintSettings.**StartValue**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
