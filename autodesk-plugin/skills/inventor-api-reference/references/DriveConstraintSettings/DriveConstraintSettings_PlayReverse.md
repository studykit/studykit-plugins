# DriveConstraintSettings.PlayReverse Method

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Drives the constraint in reverse.

## Syntax

DriveConstraintSettings.**PlayReverse**()

## Version

Introduced in version 2012
