# DriveConstraintSettings.DriveAdaptivity Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Gets and sets whether to adapt components while maintaining constraint relationships.

## Syntax

DriveConstraintSettings.**DriveAdaptivity**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
