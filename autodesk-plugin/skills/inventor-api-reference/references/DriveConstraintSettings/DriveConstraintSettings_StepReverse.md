# DriveConstraintSettings.StepReverse Method

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Reverses the constraint by one step.

## Syntax

DriveConstraintSettings.**StepReverse**()

## Version

Introduced in version 2012
