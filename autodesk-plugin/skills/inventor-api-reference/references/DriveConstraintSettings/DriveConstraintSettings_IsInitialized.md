# DriveConstraintSettings.IsInitialized Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Returns whether the drive constraint parameters have been set for this constraint.

## Syntax

DriveConstraintSettings.**IsInitialized**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
