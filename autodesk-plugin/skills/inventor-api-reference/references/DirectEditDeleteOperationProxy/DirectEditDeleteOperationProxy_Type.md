# DirectEditDeleteOperationProxy.Type Property

Parent Object: [DirectEditDeleteOperationProxy](../DirectEditDeleteOperationProxy/DirectEditDeleteOperationProxy.md)

## Description

## Syntax

DirectEditDeleteOperationProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
