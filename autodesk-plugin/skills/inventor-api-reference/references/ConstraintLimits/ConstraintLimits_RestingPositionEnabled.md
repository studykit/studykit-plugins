# ConstraintLimits.RestingPositionEnabled Property

Parent Object: [ConstraintLimits](../ConstraintLimits/ConstraintLimits.md)

## Description

Read-write property that gets and sets whether the resting position value should be enabled. If enabled, the constraint snaps back to the nominal value after a drag or a solve.

## Syntax

ConstraintLimits.**RestingPositionEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
