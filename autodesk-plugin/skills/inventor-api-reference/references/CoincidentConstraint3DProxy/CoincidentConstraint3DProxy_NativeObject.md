# CoincidentConstraint3DProxy.NativeObject Property

Parent Object: [CoincidentConstraint3DProxy](../CoincidentConstraint3DProxy/CoincidentConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

CoincidentConstraint3DProxy.**NativeObject**() As [CoincidentConstraint3D](../CoincidentConstraint3D/CoincidentConstraint3D.md)

## Property Value

This is a read only property whose value is a [CoincidentConstraint3D](../CoincidentConstraint3D/CoincidentConstraint3D.md).

## Version

Introduced in version 6
