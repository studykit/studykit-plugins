# CoincidentConstraint3DProxy.Deletable Property

Parent Object: [CoincidentConstraint3DProxy](../CoincidentConstraint3DProxy/CoincidentConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

CoincidentConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
