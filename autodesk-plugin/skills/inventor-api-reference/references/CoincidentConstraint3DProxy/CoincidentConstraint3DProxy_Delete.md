# CoincidentConstraint3DProxy.Delete Method

Parent Object: [CoincidentConstraint3DProxy](../CoincidentConstraint3DProxy/CoincidentConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

CoincidentConstraint3DProxy.**Delete**()

## Version

Introduced in version 6
