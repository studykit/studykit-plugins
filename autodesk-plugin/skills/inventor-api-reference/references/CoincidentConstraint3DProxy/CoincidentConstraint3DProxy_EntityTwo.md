# CoincidentConstraint3DProxy.EntityTwo Property

Parent Object: [CoincidentConstraint3DProxy](../CoincidentConstraint3DProxy/CoincidentConstraint3DProxy.md)

## Description

Property that returns the second entity being constrained. This can be a SketchPoint3D, SketchLine3D, WorkPlane or a Face.

## Syntax

CoincidentConstraint3DProxy.**EntityTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
