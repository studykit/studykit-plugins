# iAssemblyTableColumn.FormattedHeading Property

Parent Object: [iAssemblyTableColumn](../iAssemblyTableColumn/iAssemblyTableColumn.md)

## Description

Property that returns the heading of the column in XML format.

## Syntax

iAssemblyTableColumn.**FormattedHeading**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 11
