# iAssemblyTableColumn.ReferencedDataType Property

Parent Object: [iAssemblyTableColumn](../iAssemblyTableColumn/iAssemblyTableColumn.md)

## Description

Property that returns the data type contained in the column.

## Syntax

iAssemblyTableColumn.**ReferencedDataType**() As [iComponentColumnTypeEnum](../iComponentColumnTypeEnum.md)

## Property Value

This is a read only property whose value is an [iComponentColumnTypeEnum](../iComponentColumnTypeEnum.md).

## Version

Introduced in version 2011
