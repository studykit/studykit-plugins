# ExtrudeDefinition.ExtentType Property

Parent Object: [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Description

Read-only property that returns an enum indicating how the extent of the feature is defined. This indicates the type of object returned by the Extent.

## Syntax

ExtrudeDefinition.**ExtentType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).
