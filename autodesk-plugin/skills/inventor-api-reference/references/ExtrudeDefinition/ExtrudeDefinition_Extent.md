# ExtrudeDefinition.Extent Property

Parent Object: [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Description

Read-only property that returns the PartFeatureExtent object that defines how the extent of the feature is defined. The type of extent object returne.

## Syntax

ExtrudeDefinition.**Extent**() As [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md).
