# ExtrudeDefinition.TaperAngle Property

Parent Object: [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Description

Read-write property that provides access to the taper angle of an extrude feature. If not supplied, the feature will be created with a taper angle of.

## Syntax

ExtrudeDefinition.**TaperAngle**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2012
