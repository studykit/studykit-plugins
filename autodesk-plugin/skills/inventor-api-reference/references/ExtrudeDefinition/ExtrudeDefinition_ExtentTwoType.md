# ExtrudeDefinition.ExtentTwoType Property

Parent Object: [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Description

Read-only property that returns an enum indicating how the second direction extent of the feature is defined. This indicates the type of object retur.

## Syntax

ExtrudeDefinition.**ExtentTwoType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).

## Version

Introduced in version 2012
