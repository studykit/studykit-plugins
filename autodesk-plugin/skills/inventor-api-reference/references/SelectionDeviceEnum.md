# SelectionDeviceEnum Enumerator

## Description

Enumeration of the kind of UI used to select object.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBrowserSelection | 15362 | Selection made by pck in a browser. |
| kGraphicsWindowSelection | 15361 | Selection made by pick in graphics window. |
| kNameSelection | 15363 | Selection made by providing the name of the object. |
| kUnknownSelection | 15364 | Selection made by some unknown means. |

## Version

Introduced in version 5
