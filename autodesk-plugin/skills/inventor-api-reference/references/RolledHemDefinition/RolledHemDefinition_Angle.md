# RolledHemDefinition.Angle Property

Parent Object: [RolledHemDefinition](../RolledHemDefinition/RolledHemDefinition.md)

## Description

Property that returns the parameter controlling the bend angle of a hem feature defined by a rolled style of bend.

## Syntax

RolledHemDefinition.**Angle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
