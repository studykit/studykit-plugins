# FilletSetbackVertex.MinimalSetback Property

Parent Object: [FilletSetbackVertex](../FilletSetbackVertex/FilletSetbackVertex.md)

## Description

Gets and sets whether the minimum allowable setback should be used for this vertex.

## Syntax

FilletSetbackVertex.**MinimalSetback**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
