# FilletSetbackVertex.UseSetbacks Property

Parent Object: [FilletSetbackVertex](../FilletSetbackVertex/FilletSetbackVertex.md)

## Description

Gets and sets whether or not to use setback values.

## Syntax

FilletSetbackVertex.**UseSetbacks**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
