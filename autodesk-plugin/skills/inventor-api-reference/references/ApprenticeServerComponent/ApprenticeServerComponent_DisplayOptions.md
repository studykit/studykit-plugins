# ApprenticeServerComponent.DisplayOptions Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the Display Options object.

## Syntax

ApprenticeServerComponent.**DisplayOptions**() As [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Property Value

This is a read only property whose value is a [DisplayOptions](../DisplayOptions/DisplayOptions.md).

## Version

Introduced in version 11
