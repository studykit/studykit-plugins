# ApprenticeServerComponent.DesignProjectManager Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the design project manager object.

## Syntax

ApprenticeServerComponent.**DesignProjectManager**() As [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md)

## Property Value

This is a read only property whose value is a [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md).

## Version

Introduced in version 2011
