# ApprenticeServerComponent.FileOptions Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the File Options object.

## Syntax

ApprenticeServerComponent.**FileOptions**() As [FileOptions](../FileOptions/FileOptions.md)

## Property Value

This is a read only property whose value is a [FileOptions](../FileOptions/FileOptions.md).

## Version

Introduced in version 9
