# ApprenticeServerComponent.Close2 Method

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Closes the current file. After this call there is no top-level Document. Optionally forces any child Documents to close.

## Syntax

ApprenticeServerComponent.**Close2**( [***ForceDependentsToClose***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ForceDependentsToClose | Boolean | Optional input Boolean value that specifies whether force to close dependent documents or not. |

## Version

Introduced in version 2022
