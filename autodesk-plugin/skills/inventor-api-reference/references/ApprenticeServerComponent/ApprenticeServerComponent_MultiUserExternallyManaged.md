# ApprenticeServerComponent.MultiUserExternallyManaged Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets/Sets the Boolean flag indicating whether Multi-User facility is managed by Inventor or not.

## Syntax

ApprenticeServerComponent.**MultiUserExternallyManaged**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
