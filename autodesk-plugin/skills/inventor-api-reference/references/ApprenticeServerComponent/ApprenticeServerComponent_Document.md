# ApprenticeServerComponent.Document Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the top-level Apprentice Server Document.

## Syntax

ApprenticeServerComponent.**Document**() As [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md)

## Property Value

This is a read only property whose value is an [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md).

## Version

Introduced in version 4
