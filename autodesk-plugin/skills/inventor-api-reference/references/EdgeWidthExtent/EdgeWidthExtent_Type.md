# EdgeWidthExtent.Type Property

Parent Object: [EdgeWidthExtent](../EdgeWidthExtent/EdgeWidthExtent.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

EdgeWidthExtent.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
