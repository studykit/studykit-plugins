# AngularModelDimension.ExtensionLineTwo Property

Parent Object: [AngularModelDimension](../AngularModelDimension/AngularModelDimension.md)

## Description

Read-only property that returns the second extension line of the dimension. This property may return Nothing in cases where there are no extension lines.

## Syntax

AngularModelDimension.**ExtensionLineTwo**() As [LineSegment](../LineSegment/LineSegment.md)

## Property Value

This is a read only property whose value is a [LineSegment](../LineSegment/LineSegment.md).

## Version

Introduced in version 2018
