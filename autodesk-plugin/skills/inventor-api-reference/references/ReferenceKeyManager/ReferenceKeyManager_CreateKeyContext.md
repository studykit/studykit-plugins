# ReferenceKeyManager.CreateKeyContext Method

Parent Object: [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md)

## Description

Method to create a key context to use in the creation of reference keys.

## Syntax

ReferenceKeyManager.**CreateKeyContext**() As Long

## Version

Introduced in version 5
