# AliasFreeformFeatures Object

## Description

The AliasFreeFormFeatures collection object provides access to existing AliasFreeFormFeature objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../AliasFreeformFeatures/AliasFreeformFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../AliasFreeformFeatures/AliasFreeformFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../AliasFreeformFeatures/AliasFreeformFeatures_Item.md) | Returns the specified object from the collection. |
| [Type](../AliasFreeformFeatures/AliasFreeformFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.AliasFreeformFeatures](../PartFeatures/PartFeatures_AliasFreeformFeatures.md), [SheetMetalFeatures.AliasFreeformFeatures](../SheetMetalFeatures/SheetMetalFeatures_AliasFreeformFeatures.md)

## Version

Introduced in version 2011
