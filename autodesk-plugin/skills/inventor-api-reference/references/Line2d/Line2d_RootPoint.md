# Line2d.RootPoint Property

Parent Object: [Line2d](../Line2d/Line2d.md)

## Description

Specifies the root point of the line.

## Syntax

Line2d.**RootPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 4
