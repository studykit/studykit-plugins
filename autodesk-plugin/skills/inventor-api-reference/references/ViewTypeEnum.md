# ViewTypeEnum Enumerator

## Description

View Types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBrowserViewType | 9219 | View showing the tree representation of the contents in the Document. |
| kGraphicsViewType | 9218 | View showing the pictorial representation of the Document. |
| kNoteBookViewType | 9220 | View showing the Engineers Notebook aspect of the Document. |
| kUnknownViewType | 9217 | Unknown View type. |

## Version

Introduced in version 4
