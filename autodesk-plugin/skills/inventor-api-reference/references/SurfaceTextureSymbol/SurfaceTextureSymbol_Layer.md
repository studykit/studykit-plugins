# SurfaceTextureSymbol.Layer Property

Parent Object: [SurfaceTextureSymbol](../SurfaceTextureSymbol/SurfaceTextureSymbol.md)

## Description

Gets and sets the layer associated with this object.

## Syntax

SurfaceTextureSymbol.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2009
