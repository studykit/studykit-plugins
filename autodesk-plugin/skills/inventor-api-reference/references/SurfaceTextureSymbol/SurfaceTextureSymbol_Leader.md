# SurfaceTextureSymbol.Leader Property

Parent Object: [SurfaceTextureSymbol](../SurfaceTextureSymbol/SurfaceTextureSymbol.md)

## Description

Property that returns the leader associated with the surface texture symbol.

## Syntax

SurfaceTextureSymbol.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 2009
