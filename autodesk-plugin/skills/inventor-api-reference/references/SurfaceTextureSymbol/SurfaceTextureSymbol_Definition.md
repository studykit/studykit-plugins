# SurfaceTextureSymbol.Definition Property

Parent Object: [SurfaceTextureSymbol](../SurfaceTextureSymbol/SurfaceTextureSymbol.md)

## Description

Gets and sets the SurfaceTextureSymbolDefinition object associated with this symbol.

## Syntax

SurfaceTextureSymbol.**Definition**() As [SurfaceTextureSymbolDefinition](../SurfaceTextureSymbolDefinition/SurfaceTextureSymbolDefinition.md)

## Property Value

This is a read/write property whose value is a [SurfaceTextureSymbolDefinition](../SurfaceTextureSymbolDefinition/SurfaceTextureSymbolDefinition.md).

## Version

Introduced in version 2016
