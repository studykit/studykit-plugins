# DWGBlockDefinitionProxy.Polylines3D Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGPolylines3DEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**Polylines3D**() As [DWGPolylines3DEnumerator](../DWGPolylines3DEnumerator/DWGPolylines3DEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGPolylines3DEnumerator](../DWGPolylines3DEnumerator/DWGPolylines3DEnumerator.md).

## Version

Introduced in version 2016
