# DWGBlockDefinitionProxy.Points Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGPointsEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**Points**() As [DWGPointsEnumerator](../DWGPointsEnumerator/DWGPointsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGPointsEnumerator](../DWGPointsEnumerator/DWGPointsEnumerator.md).

## Version

Introduced in version 2016
