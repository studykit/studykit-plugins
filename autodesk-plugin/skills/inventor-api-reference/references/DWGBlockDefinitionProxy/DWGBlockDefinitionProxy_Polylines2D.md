# DWGBlockDefinitionProxy.Polylines2D Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGPolylines2DEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**Polylines2D**() As [DWGPolylines2DEnumerator](../DWGPolylines2DEnumerator/DWGPolylines2DEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGPolylines2DEnumerator](../DWGPolylines2DEnumerator/DWGPolylines2DEnumerator.md).

## Version

Introduced in version 2016
