# DWGBlockDefinitionProxy.Entities Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGEntitiesEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**Entities**() As [DWGEntitiesEnumerator](../DWGEntitiesEnumerator/DWGEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEntitiesEnumerator](../DWGEntitiesEnumerator/DWGEntitiesEnumerator.md).

## Version

Introduced in version 2016
