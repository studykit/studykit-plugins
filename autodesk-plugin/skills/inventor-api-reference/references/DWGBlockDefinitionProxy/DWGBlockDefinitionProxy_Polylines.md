# DWGBlockDefinitionProxy.Polylines Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGPolylinesEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**Polylines**() As [DWGPolylinesEnumerator](../DWGPolylinesEnumerator/DWGPolylinesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGPolylinesEnumerator](../DWGPolylinesEnumerator/DWGPolylinesEnumerator.md).

## Version

Introduced in version 2016
