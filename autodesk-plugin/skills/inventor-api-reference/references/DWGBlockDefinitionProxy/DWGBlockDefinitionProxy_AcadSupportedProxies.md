# DWGBlockDefinitionProxy.AcadSupportedProxies Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGAcadSupportedProxiesEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**AcadSupportedProxies**() As [DWGAcadSupportedProxiesEnumerator](../DWGAcadSupportedProxiesEnumerator/DWGAcadSupportedProxiesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGAcadSupportedProxiesEnumerator](../DWGAcadSupportedProxiesEnumerator/DWGAcadSupportedProxiesEnumerator.md).

## Version

Introduced in version 2018
