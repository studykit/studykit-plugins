# DWGBlockDefinitionProxy.Arcs Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns the DWGArcsEnumerator collection object.

## Syntax

DWGBlockDefinitionProxy.**Arcs**() As [DWGArcsEnumerator](../DWGArcsEnumerator/DWGArcsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGArcsEnumerator](../DWGArcsEnumerator/DWGArcsEnumerator.md).

## Version

Introduced in version 2016
