# SheetMetalFeatures Object

Derived from: [PartFeatures](../PartFeatures/PartFeatures.md) Object

## Description

The SheetMetalFeaturesFeatures collection object provides access to all of the SheetMetalFeature objects in a sheet metal component definition.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [CreatePath](../SheetMetalFeatures/SheetMetalFeatures_CreatePath.md) | Method that creates a path used to define the shape of several part features such as sweep, rectangular pattern, split, etc. All other 2D and 3D curves that are connected to the input curve are obtained and used to create a Path object. The new Path is returned. |
| [CreateSpecifiedPath](../SheetMetalFeatures/SheetMetalFeatures_CreateSpecifiedPath.md) | Method that creates a path used to define the shape of several part features such as sweep, rectangular pattern, split, etc. The new path is returned. This method will fail if the input curves are not connected end-to-end. |
