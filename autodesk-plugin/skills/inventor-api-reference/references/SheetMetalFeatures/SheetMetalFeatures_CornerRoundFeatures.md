# SheetMetalFeatures.CornerRoundFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the CornerRoundFeatures collection object.

## Syntax

SheetMetalFeatures.**CornerRoundFeatures**() As [CornerRoundFeatures](../CornerRoundFeatures/CornerRoundFeatures.md)

## Property Value

This is a read only property whose value is a [CornerRoundFeatures](../CornerRoundFeatures/CornerRoundFeatures.md).

## Version

Introduced in version 5.3
