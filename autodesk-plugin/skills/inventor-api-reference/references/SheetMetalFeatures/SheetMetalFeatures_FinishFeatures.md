# SheetMetalFeatures.FinishFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are FinishFeatures.

## Syntax

SheetMetalFeatures.**FinishFeatures**() As [FinishFeatures](../FinishFeatures/FinishFeatures.md)

## Property Value

This is a read only property whose value is a [FinishFeatures](../FinishFeatures/FinishFeatures.md).

## Version

Introduced in version 2024
