# SheetMetalFeatures.RipFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the RipFeatures collection object.

## Syntax

SheetMetalFeatures.**RipFeatures**() As [RipFeatures](../RipFeatures/RipFeatures.md)

## Property Value

This is a read only property whose value is a [RipFeatures](../RipFeatures/RipFeatures.md).
