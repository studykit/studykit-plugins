# SheetMetalFeatures.RestFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are RestFeatures.

## Syntax

SheetMetalFeatures.**RestFeatures**() As [RestFeatures](../RestFeatures/RestFeatures.md)

## Property Value

This is a read only property whose value is a [RestFeatures](../RestFeatures/RestFeatures.md).

## Version

Introduced in version 2010
