# SheetMetalFeatures.CornerFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the CornerFeatures collection object.

## Syntax

SheetMetalFeatures.**CornerFeatures**() As [CornerFeatures](../CornerFeatures/CornerFeatures.md)

## Property Value

This is a read only property whose value is a [CornerFeatures](../CornerFeatures/CornerFeatures.md).

## Version

Introduced in version 5.3
