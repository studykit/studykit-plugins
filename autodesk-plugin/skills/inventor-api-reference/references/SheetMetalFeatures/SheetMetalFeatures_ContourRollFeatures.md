# SheetMetalFeatures.ContourRollFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the ContourRollFeatures Object.

## Syntax

SheetMetalFeatures.**ContourRollFeatures**() As [ContourRollFeatures](../ContourRollFeatures/ContourRollFeatures.md)

## Property Value

This is a read only property whose value is a [ContourRollFeatures](../ContourRollFeatures/ContourRollFeatures.md).

## Version

Introduced in version 2010
