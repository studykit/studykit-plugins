# SheetMetalFeatures.DirectEditFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that besides listing out the subset of part features that are DirectEdit features.

## Syntax

SheetMetalFeatures.**DirectEditFeatures**() As [DirectEditFeatures](../DirectEditFeatures/DirectEditFeatures.md)

## Property Value

This is a read only property whose value is a [DirectEditFeatures](../DirectEditFeatures/DirectEditFeatures.md).

## Version

Introduced in version 2015
