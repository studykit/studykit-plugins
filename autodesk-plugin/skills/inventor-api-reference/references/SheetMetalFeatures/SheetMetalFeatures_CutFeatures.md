# SheetMetalFeatures.CutFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the CutFeatures object.

## Syntax

SheetMetalFeatures.**CutFeatures**() As [CutFeatures](../CutFeatures/CutFeatures.md)

## Property Value

This is a read only property whose value is a [CutFeatures](../CutFeatures/CutFeatures.md).
