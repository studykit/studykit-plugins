# SheetMetalFeatures.KnitFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the KnitFeatures collection object. This collection provides access to existing KnitFeature objects.

## Syntax

SheetMetalFeatures.**KnitFeatures**() As [KnitFeatures](../KnitFeatures/KnitFeatures.md)

## Property Value

This is a read only property whose value is a [KnitFeatures](../KnitFeatures/KnitFeatures.md).

## Version

Introduced in version 6
