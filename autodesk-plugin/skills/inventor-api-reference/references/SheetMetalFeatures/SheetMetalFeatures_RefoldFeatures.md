# SheetMetalFeatures.RefoldFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the RefoldFeatures collection object. This collection provides access to existing RefoldFeature objects and provides functionality to create new RefoldFeature objects.

## Syntax

SheetMetalFeatures.**RefoldFeatures**() As [RefoldFeatures](../RefoldFeatures/RefoldFeatures.md)

## Property Value

This is a read only property whose value is a [RefoldFeatures](../RefoldFeatures/RefoldFeatures.md).

## Version

Introduced in version 2010
