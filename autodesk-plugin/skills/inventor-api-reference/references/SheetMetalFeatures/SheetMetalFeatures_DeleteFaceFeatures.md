# SheetMetalFeatures.DeleteFaceFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the DeleteFaceFeatures collection object. This collection provides access to existing DeleteFaceFeature objects.

## Syntax

SheetMetalFeatures.**DeleteFaceFeatures**() As [DeleteFaceFeatures](../DeleteFaceFeatures/DeleteFaceFeatures.md)

## Property Value

This is a read only property whose value is a [DeleteFaceFeatures](../DeleteFaceFeatures/DeleteFaceFeatures.md).

## Version

Introduced in version 6
