# SheetMetalFeatures.FoldFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the FoldFeatures collection object. This collection provides access to existing FoldFeature objects.

## Syntax

SheetMetalFeatures.**FoldFeatures**() As [FoldFeatures](../FoldFeatures/FoldFeatures.md)

## Property Value

This is a read only property whose value is a [FoldFeatures](../FoldFeatures/FoldFeatures.md).
