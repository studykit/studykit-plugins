# SheetMetalFeatures.LoftedFlangeFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the LoftedFlangeFeatures collection object.

## Syntax

SheetMetalFeatures.**LoftedFlangeFeatures**() As [LoftedFlangeFeatures](../LoftedFlangeFeatures/LoftedFlangeFeatures.md)

## Property Value

This is a read only property whose value is a [LoftedFlangeFeatures](../LoftedFlangeFeatures/LoftedFlangeFeatures.md).
