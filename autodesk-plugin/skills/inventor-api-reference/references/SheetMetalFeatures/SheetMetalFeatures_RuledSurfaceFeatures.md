# SheetMetalFeatures.RuledSurfaceFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are RuledSurfaceFeatures.

## Syntax

SheetMetalFeatures.**RuledSurfaceFeatures**() As [RuledSurfaceFeatures](../RuledSurfaceFeatures/RuledSurfaceFeatures.md)

## Property Value

This is a read only property whose value is a [RuledSurfaceFeatures](../RuledSurfaceFeatures/RuledSurfaceFeatures.md).

## Version

Introduced in version 2016
