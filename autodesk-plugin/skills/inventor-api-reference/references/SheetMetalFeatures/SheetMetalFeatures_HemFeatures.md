# SheetMetalFeatures.HemFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the HemFeatures collection object. This collection provides access to existing HemFeature objects.

## Syntax

SheetMetalFeatures.**HemFeatures**() As [HemFeatures](../HemFeatures/HemFeatures.md)

## Property Value

This is a read only property whose value is a [HemFeatures](../HemFeatures/HemFeatures.md).

## Version

Introduced in version 5.3
