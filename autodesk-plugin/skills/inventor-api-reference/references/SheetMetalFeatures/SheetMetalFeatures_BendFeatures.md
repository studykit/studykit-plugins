# SheetMetalFeatures.BendFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the BendFeatures collection object. This collection provides access to existing BendFeature objects.

## Syntax

SheetMetalFeatures.**BendFeatures**() As [BendFeatures](../BendFeatures/BendFeatures.md)

## Property Value

This is a read only property whose value is a [BendFeatures](../BendFeatures/BendFeatures.md).
