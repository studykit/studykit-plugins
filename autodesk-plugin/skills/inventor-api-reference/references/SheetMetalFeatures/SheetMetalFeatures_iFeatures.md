# SheetMetalFeatures.iFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the iFeatures collection object.

## Syntax

SheetMetalFeatures.**iFeatures**() As [iFeatures](../iFeatures/iFeatures.md)

## Property Value

This is a read only property whose value is an [iFeatures](../iFeatures/iFeatures.md).

## Version

Introduced in version 2009
