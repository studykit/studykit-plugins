# BoundaryPatchLoops.Type Property

Parent Object: [BoundaryPatchLoops](../BoundaryPatchLoops/BoundaryPatchLoops.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

BoundaryPatchLoops.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2008
