# BoundaryPatchLoops Object

## Description

The BoundaryPatchLoops object is used to define the boundary loops for a boundary patch feature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../BoundaryPatchLoops/BoundaryPatchLoops_Add.md) | Method that creates a new BoundaryPatchLoop object using the specified boundary path. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BoundaryPatchLoops/BoundaryPatchLoops_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BoundaryPatchLoops/BoundaryPatchLoops_Count.md) | Property that specifies the number of BoundaryPatchLoop objects in the collection. |
| [Item](../BoundaryPatchLoops/BoundaryPatchLoops_Item.md) | Method that returns the specified BoundaryPatchLoop object from the collection. |
| [Type](../BoundaryPatchLoops/BoundaryPatchLoops_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BoundaryPatchDefinition.BoundaryPatchLoops](../BoundaryPatchDefinition/BoundaryPatchDefinition_BoundaryPatchLoops.md)

## Version

Introduced in version 2008
