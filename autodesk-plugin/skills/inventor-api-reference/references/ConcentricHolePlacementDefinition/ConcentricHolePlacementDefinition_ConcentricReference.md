# ConcentricHolePlacementDefinition.ConcentricReference Property

Parent Object: [ConcentricHolePlacementDefinition](../ConcentricHolePlacementDefinition/ConcentricHolePlacementDefinition.md)

## Description

Property that indicates the circular Edge or the cylindrical Face which forms the reference for the hole placement.

## Syntax

ConcentricHolePlacementDefinition.**ConcentricReference**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
