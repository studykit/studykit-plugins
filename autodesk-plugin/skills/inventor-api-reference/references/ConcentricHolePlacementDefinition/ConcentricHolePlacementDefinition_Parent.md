# ConcentricHolePlacementDefinition.Parent Property

Parent Object: [ConcentricHolePlacementDefinition](../ConcentricHolePlacementDefinition/ConcentricHolePlacementDefinition.md)

## Description

Property that returns the parent PartFeature of the definition. This property returns Nothing in the case where the definition object is created using one of the Create methods on the HoleFeatures object.

## Syntax

ConcentricHolePlacementDefinition.**Parent**() As [HoleFeature](../HoleFeature/HoleFeature.md)

## Property Value

This is a read only property whose value is a [HoleFeature](../HoleFeature/HoleFeature.md).

## Version

Introduced in version 10
