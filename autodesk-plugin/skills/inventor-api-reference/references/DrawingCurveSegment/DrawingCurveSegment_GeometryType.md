# DrawingCurveSegment.GeometryType Property

Parent Object: [DrawingCurveSegment](../DrawingCurveSegment/DrawingCurveSegment.md)

## Description

Read-only property that returns the type of the geometry object that will be returned by the Geometry property.

## Syntax

DrawingCurveSegment.**GeometryType**() As [Curve2dTypeEnum](../Curve2dTypeEnum.md)

## Property Value

This is a read only property whose value is a [Curve2dTypeEnum](../Curve2dTypeEnum.md).

## Version

Introduced in version 2013
