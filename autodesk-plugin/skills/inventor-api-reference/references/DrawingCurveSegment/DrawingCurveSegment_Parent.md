# DrawingCurveSegment.Parent Property

Parent Object: [DrawingCurveSegment](../DrawingCurveSegment/DrawingCurveSegment.md)

## Description

Property that returns the parent drawing curve.

## Syntax

DrawingCurveSegment.**Parent**() As [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Property Value

This is a read only property whose value is a [DrawingCurve](../DrawingCurve/DrawingCurve.md).
