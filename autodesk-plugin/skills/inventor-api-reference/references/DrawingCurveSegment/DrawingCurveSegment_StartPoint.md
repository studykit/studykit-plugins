# DrawingCurveSegment.StartPoint Property

Parent Object: [DrawingCurveSegment](../DrawingCurveSegment/DrawingCurveSegment.md)

## Description

Property that returns the start point of the curve segment in sheet space. This property returns Nothing for circular segments.

## Syntax

DrawingCurveSegment.**StartPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 11
