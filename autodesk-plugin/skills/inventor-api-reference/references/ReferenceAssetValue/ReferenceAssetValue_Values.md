# ReferenceAssetValue.Values Property

Parent Object: [ReferenceAssetValue](../ReferenceAssetValue/ReferenceAssetValue.md)

## Description

Gets and sets the values associated with this asset value. HasMultipleValues property indicates if this property will be returning more than one value.

## Syntax

ReferenceAssetValue.**Values**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
