# LineLengthDimConstraint3D.DimensionPlane Property

Parent Object: [LineLengthDimConstraint3D](../LineLengthDimConstraint3D/LineLengthDimConstraint3D.md)

## Description

Property that returns the transient dimension plane used to place and position the dimension text for this dimension constraint.

## Syntax

LineLengthDimConstraint3D.**DimensionPlane**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).

## Version

Introduced in version 11
