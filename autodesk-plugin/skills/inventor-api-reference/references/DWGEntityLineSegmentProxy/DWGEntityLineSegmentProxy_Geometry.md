# DWGEntityLineSegmentProxy.Geometry Property

Parent Object: [DWGEntityLineSegmentProxy](../DWGEntityLineSegmentProxy/DWGEntityLineSegmentProxy.md)

## Description

Read-only property that returns the segment geometry.

## Syntax

DWGEntityLineSegmentProxy.**Geometry**() As [LineSegment](../LineSegment/LineSegment.md)

## Property Value

This is a read only property whose value is a [LineSegment](../LineSegment/LineSegment.md).

## Version

Introduced in version 2018
