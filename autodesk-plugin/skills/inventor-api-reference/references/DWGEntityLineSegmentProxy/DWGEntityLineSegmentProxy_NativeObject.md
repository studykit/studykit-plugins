# DWGEntityLineSegmentProxy.NativeObject Property

Parent Object: [DWGEntityLineSegmentProxy](../DWGEntityLineSegmentProxy/DWGEntityLineSegmentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGEntityLineSegmentProxy.**NativeObject**() As [DWGEntityLineSegment](../DWGEntityLineSegment/DWGEntityLineSegment.md)

## Property Value

This is a read only property whose value is a [DWGEntityLineSegment](../DWGEntityLineSegment/DWGEntityLineSegment.md).

## Version

Introduced in version 2018
