# DerivedFutureAssemblyDefinition.ActivePositionalRepresentation Property

Parent Object: [DerivedFutureAssemblyDefinition](../DerivedFutureAssemblyDefinition/DerivedFutureAssemblyDefinition.md)

## Description

Read-write property that gets and sets the name of the active Positional Representation for the derived future assembly.

## Syntax

DerivedFutureAssemblyDefinition.**ActivePositionalRepresentation**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
