# DerivedFutureAssemblyDefinition.Type Property

Parent Object: [DerivedFutureAssemblyDefinition](../DerivedFutureAssemblyDefinition/DerivedFutureAssemblyDefinition.md)

## Description

Read-only property returning kDerivedFutureAssemblyDefinitionObject indicating the type of object.

## Syntax

DerivedFutureAssemblyDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2018
