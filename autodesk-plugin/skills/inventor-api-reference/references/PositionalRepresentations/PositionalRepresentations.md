# PositionalRepresentations Object

## Description

The PositionalRepresentations collection object provides access to all the positional representations in the assembly and provides methods to create new ones.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../PositionalRepresentations/PositionalRepresentations_Add.md) | Method that creates a new PositionalRepresentation. The new created PositionalRepresentation is returned. |
| [Export](../PositionalRepresentations/PositionalRepresentations_Export.md) | Exports the PositionalRepresentations to an external file. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PositionalRepresentations/PositionalRepresentations_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../PositionalRepresentations/PositionalRepresentations_Count.md) | Property that returns the number of items in the collection. |
| [ExcelWorkSheet](../PositionalRepresentations/PositionalRepresentations_ExcelWorkSheet.md) | Property that returns the Excel spreadsheet that contains the positional representations data. |
| [Item](../PositionalRepresentations/PositionalRepresentations_Item.md) | Returns the specified PositionalRepresentation object from the collection. |
| [Type](../PositionalRepresentations/PositionalRepresentations_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[RepresentationsManager.PositionalRepresentations](../RepresentationsManager/RepresentationsManager_PositionalRepresentations.md)

## Version

Introduced in version 11
