# PositionalRepresentations.Item Property

Parent Object: [PositionalRepresentations](../PositionalRepresentations/PositionalRepresentations.md)

## Description

Returns the specified PositionalRepresentation object from the collection.

## Syntax

PositionalRepresentations.**Item**( ***Index*** As Variant ) As [PositionalRepresentation](../PositionalRepresentation/PositionalRepresentation.md)

## Property Value

This is a read only property whose value is a [PositionalRepresentation](../PositionalRepresentation/PositionalRepresentation.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the PositionalRepresentation to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the PositionalRepresentation name. If an out of range index or a name of a non-existent PositionalRepresentation is provided, an error will occur . |

## Version

Introduced in version 11
