# PositionalRepresentations.Count Property

Parent Object: [PositionalRepresentations](../PositionalRepresentations/PositionalRepresentations.md)

## Description

Property that returns the number of items in the collection.

## Syntax

PositionalRepresentations.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 11
