# GraphicsSettingTypeEnum Enumerator

## Description

Constants specifying the graphics card optimization setting.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kCompatibilityGraphicsSetting | 91651 | Use this setting for graphics compatible with previous Inventor releases. |
| kConservativeGraphicsSetting | 91652 | Use this setting for conservative graphics hardware use with Inventor. |
| kPerformanceGraphicsSetting | 91650 | Use this setting when performance is desired over realistic visualization. |
| kQualityGraphicsSetting | 91649 | Use this setting for the highest quality realistic visualization. |

## Version

Introduced in version 2011
