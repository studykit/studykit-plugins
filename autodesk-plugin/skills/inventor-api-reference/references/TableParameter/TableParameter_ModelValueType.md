# TableParameter.ModelValueType Property

Parent Object: [TableParameter](../TableParameter/TableParameter.md)

## Description

Gets/(Sets) the setting which is used for determining the method used to compute the model value.

## Syntax

TableParameter.**ModelValueType**() As [ModelValueTypeEnum](../ModelValueTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ModelValueTypeEnum](../ModelValueTypeEnum.md).

## Version

Introduced in version 6
