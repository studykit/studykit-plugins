# TableParameter.ExpressionList Property

Parent Object: [TableParameter](../TableParameter/TableParameter.md)

## Description

Property that returns the expression list associated with this parameter.

## Syntax

TableParameter.**ExpressionList**() As [ExpressionList](../ExpressionList/ExpressionList.md)

## Property Value

This is a read only property whose value is an [ExpressionList](../ExpressionList/ExpressionList.md).

## Version

Introduced in version 2011
