# TableParameter.Type Property

Parent Object: [TableParameter](../TableParameter/TableParameter.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TableParameter.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 4
