# TableParameter.ConvertToUserParameter Method

Parent Object: [TableParameter](../TableParameter/TableParameter.md)

## Description

Method that causes the parameter to be converted to a user parameter. The new UserParameter object is returned.

## Syntax

TableParameter.**ConvertToUserParameter**() As [UserParameter](../UserParameter/UserParameter.md)

## Version

Introduced in version 11
