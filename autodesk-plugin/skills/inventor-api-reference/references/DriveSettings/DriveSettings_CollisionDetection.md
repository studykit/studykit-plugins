# DriveSettings.CollisionDetection Property

Parent Object: [DriveSettings](../DriveSettings/DriveSettings.md)

## Description

Read-write property that gets and sets whether to drive the constrained assembly until a collision is detected.

## Syntax

DriveSettings.**CollisionDetection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
