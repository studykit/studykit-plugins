# DriveSettings.DriveAdaptivity Property

Parent Object: [DriveSettings](../DriveSettings/DriveSettings.md)

## Description

Read-write property that gets and sets whether to adapt components while maintaining constraint or connection relationships.

## Syntax

DriveSettings.**DriveAdaptivity**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
