# DriveSettings.RepetitionStartEndStart Property

Parent Object: [DriveSettings](../DriveSettings/DriveSettings.md)

## Description

Read-write property that gets and sets whether to drive the relationship from the start value to the end value and then in reverse to the start value.

## Syntax

DriveSettings.**RepetitionStartEndStart**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
