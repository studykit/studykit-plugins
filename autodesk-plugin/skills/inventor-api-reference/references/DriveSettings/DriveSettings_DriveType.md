# DriveSettings.DriveType Property

Parent Object: [DriveSettings](../DriveSettings/DriveSettings.md)

## Description

Read-write property that gets and sets the drive type to specify which value to drive.

## Syntax

DriveSettings.**DriveType**() As [DriveTypeEnum](../DriveTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DriveTypeEnum](../DriveTypeEnum.md).

## Version

Introduced in version 2014
