# File.ReferencedFileDescriptors Property

Parent Object: [File](../File/File.md)

## Description

Property that returns an enumeration of descriptors that represent the direct file references held by this file.

## Syntax

File.**ReferencedFileDescriptors**() As [FileDescriptorsEnumerator](../FileDescriptorsEnumerator/FileDescriptorsEnumerator.md)

## Property Value

This is a read only property whose value is a [FileDescriptorsEnumerator](../FileDescriptorsEnumerator/FileDescriptorsEnumerator.md).

## Version

Introduced in version 11
