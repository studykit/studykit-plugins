# File.SoftwareVersionCreated Property

Parent Object: [File](../File/File.md)

## Description

Read-only property that returns object that encapsulates version of the software with which this file was created.

## Syntax

File.**SoftwareVersionCreated**() As [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Property Value

This is a read only property whose value is a [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md).

## Version

Introduced in version 2023
