# Centermarks.Item Property

Parent Object: [Centermarks](../Centermarks/Centermarks.md)

## Description

Method that returns the specified Centermark object from the collection.

## Syntax

Centermarks.**Item**( ***Index*** As Long ) As [Centermark](../Centermark/Centermark.md)

## Property Value

This is a read only property whose value is a [Centermark](../Centermark/Centermark.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2008
