# ParallelToXYPlaneConstraint3D.Deletable Property

Parent Object: [ParallelToXYPlaneConstraint3D](../ParallelToXYPlaneConstraint3D/ParallelToXYPlaneConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

ParallelToXYPlaneConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.
