# ParallelToXYPlaneConstraint3D.Delete Method

Parent Object: [ParallelToXYPlaneConstraint3D](../ParallelToXYPlaneConstraint3D/ParallelToXYPlaneConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ParallelToXYPlaneConstraint3D.**Delete**()

## Version

Introduced in version 2017
