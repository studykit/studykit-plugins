# DrawingViewLabel.WidthScale Property

Parent Object: [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Description

Read-write property that gets and sets the width scale factor for the drawing view label.

## Syntax

DrawingViewLabel.**WidthScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
