# DrawingViewLabel.FormattedText Property

Parent Object: [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Description

Read-write property that gets and sets the fully formatted string that defines the contents of the drawing view label.

## Syntax

DrawingViewLabel.**FormattedText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
