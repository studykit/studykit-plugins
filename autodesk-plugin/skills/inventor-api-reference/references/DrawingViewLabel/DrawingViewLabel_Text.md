# DrawingViewLabel.Text Property

Parent Object: [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Description

Property that gets the string that defines the drawing view label.

## Syntax

DrawingViewLabel.**Text**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2009
