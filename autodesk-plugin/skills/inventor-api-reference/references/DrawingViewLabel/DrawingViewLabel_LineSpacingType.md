# DrawingViewLabel.LineSpacingType Property

Parent Object: [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Description

Read-write property that gets and sets the method used to define the line spacing.

## Syntax

DrawingViewLabel.**LineSpacingType**() As [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md)

## Property Value

This is a read/write property whose value is a [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md).

## Version

Introduced in version 2009
