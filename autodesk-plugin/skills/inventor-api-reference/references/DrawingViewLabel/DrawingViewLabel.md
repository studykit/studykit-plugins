# DrawingViewLabel Object

## Description

The DrawingViewLabel object represents the text in a drawing view label.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [RestoreViewLabel](../DrawingViewLabel/DrawingViewLabel_RestoreViewLabel.md) | Method that restores the drawing view label to default position. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingViewLabel/DrawingViewLabel_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Color](../DrawingViewLabel/DrawingViewLabel_Color.md) | Read-write property that gets and sets the color of the text. |
| [ConstrainToBorder](../DrawingViewLabel/DrawingViewLabel_ConstrainToBorder.md) | Gets and sets whether to constrain the view label to the view. |
| [FormattedText](../DrawingViewLabel/DrawingViewLabel_FormattedText.md) | Read-write property that gets and sets the fully formatted string that defines the contents of the drawing view label. |
| [HorizontalJustification](../DrawingViewLabel/DrawingViewLabel_HorizontalJustification.md) | Read-write property that gets and sets the horizontal justification override of the drawing view label. |
| [Layer](../DrawingViewLabel/DrawingViewLabel_Layer.md) | Gets and sets the layer associated with the view label. |
| [LineSpacing](../DrawingViewLabel/DrawingViewLabel_LineSpacing.md) | Read-write property that gets and sets the line spacing of the drawing view label. |
| [LineSpacingType](../DrawingViewLabel/DrawingViewLabel_LineSpacingType.md) | Read-write property that gets and sets the method used to define the line spacing. |
| [Parent](../DrawingViewLabel/DrawingViewLabel_Parent.md) | Property that returns the parent of the object. |
| [Position](../DrawingViewLabel/DrawingViewLabel_Position.md) | Read-write property that gets and sets the position of the label. |
| [RangeBox](../DrawingViewLabel/DrawingViewLabel_RangeBox.md) | Property that returns a Box2D object which contains the lower-left and upper-right corners of a rectangle that is guaranteed to enclose this object. |
| [StackedTextPosition](../DrawingViewLabel/DrawingViewLabel_StackedTextPosition.md) | Gets and sets the position (alignment) of the stacked text with respect to regular text. |
| [Text](../DrawingViewLabel/DrawingViewLabel_Text.md) | Property that gets the string that defines the drawing view label. |
| [TextStyle](../DrawingViewLabel/DrawingViewLabel_TextStyle.md) | Gets and sets the text style associated with the view label. |
| [Type](../DrawingViewLabel/DrawingViewLabel_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [VerticalJustification](../DrawingViewLabel/DrawingViewLabel_VerticalJustification.md) | Read-write property that gets and sets the vertical justification override of the drawing view label. |
| [WidthScale](../DrawingViewLabel/DrawingViewLabel_WidthScale.md) | Read-write property that gets and sets the width scale factor for the drawing view label. |
