# PublisherServer.GetActivePresentationScene Method

Parent Object: [PublisherServer](../PublisherServer/PublisherServer.md)

## Description

Method that gets the active PresentationScene object.

## Syntax

PublisherServer.**GetActivePresentationScene**( ***Document*** As [PresentationDocument](../PresentationDocument/PresentationDocument.md) ) As [PresentationScene](../PresentationScene/PresentationScene.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Document | [PresentationDocument](../PresentationDocument/PresentationDocument.md) | Inputs PresentationDocument object that indicates a presentation document. |

## Version

Introduced in version 2018
