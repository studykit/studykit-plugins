# BreakOperation.Layer Property

Parent Object: [BreakOperation](../BreakOperation/BreakOperation.md)

## Description

Gets and sets the layer associated with this object.

## Syntax

BreakOperation.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2010
