# BreakOperation.BreakStyle Property

Parent Object: [BreakOperation](../BreakOperation/BreakOperation.md)

## Description

Read-write property that gets and sets the break style.

## Syntax

BreakOperation.**BreakStyle**() As [BreakStyleEnum](../BreakStyleEnum.md)

## Property Value

This is a read/write property whose value is a [BreakStyleEnum](../BreakStyleEnum.md).

## Version

Introduced in version 2010
