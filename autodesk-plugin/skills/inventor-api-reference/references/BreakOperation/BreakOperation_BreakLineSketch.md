# BreakOperation.BreakLineSketch Property

Parent Object: [BreakOperation](../BreakOperation/BreakOperation.md)

## Description

Read-only property that returns the DrawingSketch object containing the break lines.

## Syntax

BreakOperation.**BreakLineSketch**() As [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Property Value

This is a read only property whose value is a [DrawingSketch](../DrawingSketch/DrawingSketch.md).

## Version

Introduced in version 2026
