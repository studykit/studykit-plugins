# CollinearConstraintProxy.NativeObject Property

Parent Object: [CollinearConstraintProxy](../CollinearConstraintProxy/CollinearConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

CollinearConstraintProxy.**NativeObject**() As [CollinearConstraint](../CollinearConstraint/CollinearConstraint.md)

## Property Value

This is a read only property whose value is a [CollinearConstraint](../CollinearConstraint/CollinearConstraint.md).

## Version

Introduced in version 6
