# ProjectedCutProxy.NativeObject Property

Parent Object: [ProjectedCutProxy](../ProjectedCutProxy/ProjectedCutProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ProjectedCutProxy.**NativeObject**() As [ProjectedCut](../ProjectedCut/ProjectedCut.md)

## Property Value

This is a read only property whose value is a [ProjectedCut](../ProjectedCut/ProjectedCut.md).

## Version

Introduced in version 2010
