# ProjectedCutProxy.BreakLink Method

Parent Object: [ProjectedCutProxy](../ProjectedCutProxy/ProjectedCutProxy.md)

## Description

Method that breaks the link between the projected cut edges and the model.

## Syntax

ProjectedCutProxy.**BreakLink**()

## Version

Introduced in version 2010
