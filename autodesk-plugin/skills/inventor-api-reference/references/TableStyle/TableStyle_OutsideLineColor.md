# TableStyle.OutsideLineColor Property

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Gets and sets the color of the outer lines of the table.

## Syntax

TableStyle.**OutsideLineColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2011
