# TableStyle.TableDirection Property

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Gets and sets the direction of the table.

## Syntax

TableStyle.**TableDirection**() As [TableDirectionEnum](../TableDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [TableDirectionEnum](../TableDirectionEnum.md).

## Version

Introduced in version 2011
