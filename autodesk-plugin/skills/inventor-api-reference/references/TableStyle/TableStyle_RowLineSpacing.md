# TableStyle.RowLineSpacing Property

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Gets and sets the spacing between the rows.

## Syntax

TableStyle.**RowLineSpacing**() As [LineSpacingEnum](../LineSpacingEnum.md)

## Property Value

This is a read/write property whose value is a [LineSpacingEnum](../LineSpacingEnum.md).

## Version

Introduced in version 2011
