# TableStyle.StyleType Property

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Gets the type of the style.

## Syntax

TableStyle.**StyleType**() As [StyleTypeEnum](../StyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [StyleTypeEnum](../StyleTypeEnum.md).

## Version

Introduced in version 2011
