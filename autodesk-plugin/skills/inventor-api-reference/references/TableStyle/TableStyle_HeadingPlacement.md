# TableStyle.HeadingPlacement Property

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Gets and sets the location of the heading (title).

## Syntax

TableStyle.**HeadingPlacement**() As [HeadingPlacementEnum](../HeadingPlacementEnum.md)

## Property Value

This is a read/write property whose value is a [HeadingPlacementEnum](../HeadingPlacementEnum.md).

## Version

Introduced in version 2011
