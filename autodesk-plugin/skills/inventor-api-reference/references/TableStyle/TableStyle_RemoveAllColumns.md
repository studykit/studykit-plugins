# TableStyle.RemoveAllColumns Method

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Method that removes all columns from the style.

## Syntax

TableStyle.**RemoveAllColumns**()

## Version

Introduced in version 2011
