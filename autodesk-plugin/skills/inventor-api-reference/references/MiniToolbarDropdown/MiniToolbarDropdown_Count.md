# MiniToolbarDropdown.Count Property

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Read-only property that returns the number of items in the dropdown.

## Syntax

MiniToolbarDropdown.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2012
