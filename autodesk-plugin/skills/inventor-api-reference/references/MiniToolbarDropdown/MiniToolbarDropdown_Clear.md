# MiniToolbarDropdown.Clear Method

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Method that removes all the items from the control.

## Syntax

MiniToolbarDropdown.**Clear**()

## Version

Introduced in version 2012
