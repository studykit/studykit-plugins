# MiniToolbarDropdown.ShowText Property

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Read-write property that gets and sets whether the display name associated with the currently selected item should be displayed on the face of the control.

## Syntax

MiniToolbarDropdown.**ShowText**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
