# MiniToolbarDropdown.SelectedIndex Property

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Read-write property that gets and sets the index of the item currently selected.

## Syntax

MiniToolbarDropdown.**SelectedIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2012
