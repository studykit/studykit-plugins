# BossFeature.Parent Property

Parent Object: [BossFeature](../BossFeature/BossFeature.md)

## Description

Property that returns the parent of the feature.

## Syntax

BossFeature.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
