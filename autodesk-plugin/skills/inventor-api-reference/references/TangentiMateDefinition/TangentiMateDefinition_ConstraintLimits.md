# TangentiMateDefinition.ConstraintLimits Property

Parent Object: [TangentiMateDefinition](../TangentiMateDefinition/TangentiMateDefinition.md)

## Description

Property that returns the ConstraintLimits object that provides access to various limits related settings for the iMate definition.

## Syntax

TangentiMateDefinition.**ConstraintLimits**() As [ConstraintLimits](../ConstraintLimits/ConstraintLimits.md)

## Property Value

This is a read only property whose value is a [ConstraintLimits](../ConstraintLimits/ConstraintLimits.md).

## Version

Introduced in version 2011
