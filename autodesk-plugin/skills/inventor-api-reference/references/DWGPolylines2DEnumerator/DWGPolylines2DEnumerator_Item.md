# DWGPolylines2DEnumerator.Item Property

Parent Object: [DWGPolylines2DEnumerator](../DWGPolylines2DEnumerator/DWGPolylines2DEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGPolylines2DEnumerator.**Item**( ***Index*** As Variant ) As [DWGPolyline2D](../DWGPolyline2D/DWGPolyline2D.md)

## Property Value

This is a read only property whose value is a [DWGPolyline2D](../DWGPolyline2D/DWGPolyline2D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
