# CornerRoundFeature.Suppressed Property

Parent Object: [CornerRoundFeature](../CornerRoundFeature/CornerRoundFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

CornerRoundFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
