# DWGLinesEnumerator.Item Property

Parent Object: [DWGLinesEnumerator](../DWGLinesEnumerator/DWGLinesEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGLinesEnumerator.**Item**( ***Index*** As Variant ) As [DWGLine](../DWGLine/DWGLine.md)

## Property Value

This is a read only property whose value is a [DWGLine](../DWGLine/DWGLine.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
