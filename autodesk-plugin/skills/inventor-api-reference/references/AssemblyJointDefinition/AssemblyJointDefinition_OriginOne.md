# AssemblyJointDefinition.OriginOne Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-write property that gets and sets the first origin associated with the assembly joint object.

## Syntax

AssemblyJointDefinition.**OriginOne**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2014
