# AssemblyJointDefinition.LinearPosition Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-write property that gets and sets the linear position of the first occurrence relative to the second occurrence that the assembly joint object.

## Syntax

AssemblyJointDefinition.**LinearPosition**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2014
