# AssemblyJointDefinition.AngularPosition Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-write property that gets and sets the angular position around the origin direction of the first occurrence the assembly joint object is between with.

## Syntax

AssemblyJointDefinition.**AngularPosition**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2014
