# TriangleStripGraphics.NormalBinding Property

Parent Object: [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Description

Gets and sets how the normals are defined for the triangles.

## Syntax

TriangleStripGraphics.**NormalBinding**() As [NormalBindingEnum](../NormalBindingEnum.md)

## Property Value

This is a read/write property whose value is a [NormalBindingEnum](../NormalBindingEnum.md).

## Version

Introduced in version 5
