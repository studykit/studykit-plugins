# SketchEllipse3DProxy.CurvatureDisplay Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Specifies whether curvature information is displayed for the ellipse.

## Syntax

SketchEllipse3DProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
