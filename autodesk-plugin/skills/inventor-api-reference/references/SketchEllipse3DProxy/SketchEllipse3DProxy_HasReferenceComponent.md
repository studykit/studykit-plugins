# SketchEllipse3DProxy.HasReferenceComponent Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchEllipse3DProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
