# SketchEllipse3DProxy.NativeObject Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchEllipse3DProxy.**NativeObject**() As [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Property Value

This is a read only property whose value is a [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md).

## Version

Introduced in version 6
