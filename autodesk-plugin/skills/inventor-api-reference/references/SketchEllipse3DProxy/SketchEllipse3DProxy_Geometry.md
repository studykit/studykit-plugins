# SketchEllipse3DProxy.Geometry Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Gets and sets an EllipseFull geometry object. The object returned represents a 'snapshot' view of the current state of the sketch ellipse.

## Syntax

SketchEllipse3DProxy.**Geometry**() As [EllipseFull](../EllipseFull/EllipseFull.md)

## Property Value

This is a read/write property whose value is an [EllipseFull](../EllipseFull/EllipseFull.md).

## Version

Introduced in version 6
