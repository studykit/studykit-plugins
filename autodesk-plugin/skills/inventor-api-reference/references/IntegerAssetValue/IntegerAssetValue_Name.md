# IntegerAssetValue.Name Property

Parent Object: [IntegerAssetValue](../IntegerAssetValue/IntegerAssetValue.md)

## Description

Gets the key name of the value. This name will remain constant for all languages and is the name used as input to the Item property or the Asset object.

## Syntax

IntegerAssetValue.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2014
