# EqualRadiusConstraint.EntityTwo Property

Parent Object: [EqualRadiusConstraint](../EqualRadiusConstraint/EqualRadiusConstraint.md)

## Description

Property that returns the sketch circle or arc being constrained.

## Syntax

EqualRadiusConstraint.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
