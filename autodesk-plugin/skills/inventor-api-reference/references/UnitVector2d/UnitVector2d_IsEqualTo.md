# UnitVector2d.IsEqualTo Method

Parent Object: [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Description

Compare this unit vector2d for equality to the specified unit vector2d.

## Syntax

UnitVector2d.**IsEqualTo**( ***Vector*** As [UnitVector2d](../UnitVector2d/UnitVector2d.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [UnitVector2d](../UnitVector2d/UnitVector2d.md) | Vector to compare to. |
| Tolerance | Double | Tolerance for comparison. |

## Version

Introduced in version 11
