# UnitVector2d.AngleTo Method

Parent Object: [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Description

Determines the angle between this unit vector2d and the specified unit vector2d.

## Syntax

UnitVector2d.**AngleTo**( ***Vector*** As [UnitVector2d](../UnitVector2d/UnitVector2d.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [UnitVector2d](../UnitVector2d/UnitVector2d.md) | 2nd vector to measure to. |

## Version

Introduced in version 11
