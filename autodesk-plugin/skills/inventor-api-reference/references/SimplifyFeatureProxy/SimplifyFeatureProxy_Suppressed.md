# SimplifyFeatureProxy.Suppressed Property

Parent Object: [SimplifyFeatureProxy](../SimplifyFeatureProxy/SimplifyFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

SimplifyFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2026
