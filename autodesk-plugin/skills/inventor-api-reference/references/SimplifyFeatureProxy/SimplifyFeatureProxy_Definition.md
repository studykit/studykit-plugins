# SimplifyFeatureProxy.Definition Property

Parent Object: [SimplifyFeatureProxy](../SimplifyFeatureProxy/SimplifyFeatureProxy.md)

## Description

Property that gets and sets the SimplifyDefinition object associated with this simplify feature.

## Syntax

SimplifyFeatureProxy.**Definition**() As [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md)

## Property Value

This is a read/write property whose value is a [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md).

## Version

Introduced in version 2026
