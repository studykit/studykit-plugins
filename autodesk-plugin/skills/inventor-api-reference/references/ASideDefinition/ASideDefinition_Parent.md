# ASideDefinition.Parent Property

Parent Object: [ASideDefinition](../ASideDefinition/ASideDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ASideDefinition.**Parent**() As [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Property Value

This is a read only property whose value is a [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md).

## Version

Introduced in version 2016
