# ASideDefinition.Status Property

Parent Object: [ASideDefinition](../ASideDefinition/ASideDefinition.md)

## Description

Read-only property that returns the status of the A-side face.

## Syntax

ASideDefinition.**Status**() As [ASideFaceStatusEnum](../ASideFaceStatusEnum.md)

## Property Value

This is a read only property whose value is an [ASideFaceStatusEnum](../ASideFaceStatusEnum.md).

## Version

Introduced in version 2016
