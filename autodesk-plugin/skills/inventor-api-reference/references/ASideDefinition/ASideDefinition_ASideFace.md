# ASideDefinition.ASideFace Property

Parent Object: [ASideDefinition](../ASideDefinition/ASideDefinition.md)

## Description

Read-only property that returns the face object that is used to define the A-side face definition.

## Syntax

ASideDefinition.**ASideFace**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2016
