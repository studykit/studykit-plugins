# PathProxy.NativeObject Property

Parent Object: [PathProxy](../PathProxy/PathProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

PathProxy.**NativeObject**() As [Path](../Path/Path.md)

## Property Value

This is a read only property whose value is a [Path](../Path/Path.md).

## Version

Introduced in version 9
