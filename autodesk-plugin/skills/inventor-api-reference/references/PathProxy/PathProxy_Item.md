# PathProxy.Item Property

Parent Object: [PathProxy](../PathProxy/PathProxy.md)

## Description

Returns the specified PathEntity object from the collection.

## Syntax

PathProxy.**Item**( ***Index*** As Long ) As [PathEntity](../PathEntity/PathEntity.md)

## Property Value

This is a read only property whose value is a [PathEntity](../PathEntity/PathEntity.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 9
