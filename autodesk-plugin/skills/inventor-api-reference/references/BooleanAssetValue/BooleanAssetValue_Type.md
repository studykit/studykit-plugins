# BooleanAssetValue.Type Property

Parent Object: [BooleanAssetValue](../BooleanAssetValue/BooleanAssetValue.md)

## Description

Read-only property returning kAssetValueObject indicating this object’s type.

## Syntax

BooleanAssetValue.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
