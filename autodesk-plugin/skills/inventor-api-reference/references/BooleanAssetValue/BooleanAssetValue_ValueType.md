# BooleanAssetValue.ValueType Property

Parent Object: [BooleanAssetValue](../BooleanAssetValue/BooleanAssetValue.md)

## Description

Read-only property that returns the data type that the Value property for this AssetValue object will return.

## Syntax

BooleanAssetValue.**ValueType**() As [AssetValueTypeEnum](../AssetValueTypeEnum.md)

## Property Value

This is a read only property whose value is an [AssetValueTypeEnum](../AssetValueTypeEnum.md).

## Version

Introduced in version 2014
