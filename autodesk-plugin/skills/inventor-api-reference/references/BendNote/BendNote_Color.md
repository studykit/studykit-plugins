# BendNote.Color Property

Parent Object: [BendNote](../BendNote/BendNote.md)

## Description

Gets/Sets color of the DrawingNote.

## Syntax

BendNote.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2010
