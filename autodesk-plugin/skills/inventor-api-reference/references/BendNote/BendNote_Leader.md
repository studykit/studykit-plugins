# BendNote.Leader Property

Parent Object: [BendNote](../BendNote/BendNote.md)

## Description

Property that returns the Leader object.

## Syntax

BendNote.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 2010
