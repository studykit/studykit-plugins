# AnnotationPlanes.Item Property

Parent Object: [AnnotationPlanes](../AnnotationPlanes/AnnotationPlanes.md)

## Description

Returns the specified object from the collection.

## Syntax

AnnotationPlanes.**Item**( ***Index*** As Variant ) As [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Property Value

This is a read only property whose value is an [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | The index of the item within the collection to return. |

## Version

Introduced in version 2018
