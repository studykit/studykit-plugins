# LineSegment.MidPoint Property

Parent Object: [LineSegment](../LineSegment/LineSegment.md)

## Description

Property that returns the midpoint of the line segment.

## Syntax

LineSegment.**MidPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).
