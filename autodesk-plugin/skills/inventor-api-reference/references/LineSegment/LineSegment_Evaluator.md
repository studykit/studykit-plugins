# LineSegment.Evaluator Property

Parent Object: [LineSegment](../LineSegment/LineSegment.md)

## Description

Property that returns the CurveEvaluator for this line segment.

## Syntax

LineSegment.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 6
