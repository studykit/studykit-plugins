# HoleFeatures.CreateSketchPlacementDefinition Method

Parent Object: [HoleFeatures](../HoleFeatures/HoleFeatures.md)

## Description

Method that creates a new SketchHolePlacementDefinition object that can be used for defining the placement of Hole features using sketch points.

## Syntax

HoleFeatures.**CreateSketchPlacementDefinition**( ***HoleCenterPoints*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) ) As [SketchHolePlacementDefinition](../SketchHolePlacementDefinition/SketchHolePlacementDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| HoleCenterPoints | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | ObjectCollection object that contains the SketchPoint objects that will be used to define the centers of the holes. All of the SketchPoint objects used as input must be within the same sketch. |

## Version

Introduced in version 11
