# SketchBlock.Edit Method

Parent Object: [SketchBlock](../SketchBlock/SketchBlock.md)

## Description

Method that activates the sketch block for editing by the user.

## Syntax

SketchBlock.**Edit**()

## Version

Introduced in version 2010
