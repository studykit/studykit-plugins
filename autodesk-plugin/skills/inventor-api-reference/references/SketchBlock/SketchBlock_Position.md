# SketchBlock.Position Property

Parent Object: [SketchBlock](../SketchBlock/SketchBlock.md)

## Description

Gets and sets the position of the sketch block.

## Syntax

SketchBlock.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2010
