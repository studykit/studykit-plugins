# SketchEllipticalArcs Object

## Description

The SketchEllipticalArcs object provides access to all the objects in a sketch. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchEllipticalArcs/SketchEllipticalArcs_Add.md) | Method that creates a new sketch elliptical arc. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchEllipticalArcs/SketchEllipticalArcs_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchEllipticalArcs/SketchEllipticalArcs_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchEllipticalArcs/SketchEllipticalArcs_Item.md) | Returns the specified SketchEllipticalArc object from the collection. |
| [Type](../SketchEllipticalArcs/SketchEllipticalArcs_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
