# AssemblyJointProxy.Protected Property

Parent Object: [AssemblyJointProxy](../AssemblyJointProxy/AssemblyJointProxy.md)

## Description

Read-write property that gets and sets whether the degree of freedom of the assembly joint is protected or not.

## Syntax

AssemblyJointProxy.**Protected**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
