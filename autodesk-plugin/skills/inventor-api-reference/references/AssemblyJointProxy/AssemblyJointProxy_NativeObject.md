# AssemblyJointProxy.NativeObject Property

Parent Object: [AssemblyJointProxy](../AssemblyJointProxy/AssemblyJointProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

AssemblyJointProxy.**NativeObject**() As [AssemblyJoint](../AssemblyJoint/AssemblyJoint.md)

## Property Value

This is a read only property whose value is an [AssemblyJoint](../AssemblyJoint/AssemblyJoint.md).

## Version

Introduced in version 2014
