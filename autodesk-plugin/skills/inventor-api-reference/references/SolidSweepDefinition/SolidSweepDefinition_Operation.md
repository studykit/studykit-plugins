# SolidSweepDefinition.Operation Property

Parent Object: [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md)

## Description

Read-write property that gets and sets the type of operation used to add the feature to the model.

## Syntax

SolidSweepDefinition.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2019
