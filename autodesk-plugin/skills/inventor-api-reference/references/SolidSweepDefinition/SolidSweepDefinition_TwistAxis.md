# SolidSweepDefinition.TwistAxis Property

Parent Object: [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md)

## Description

Read-write property that gets and sets the twist axis.

## Syntax

SolidSweepDefinition.**TwistAxis**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2020
