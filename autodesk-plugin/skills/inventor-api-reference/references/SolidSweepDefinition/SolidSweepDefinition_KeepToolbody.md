# SolidSweepDefinition.KeepToolbody Property

Parent Object: [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md)

## Description

Read-write property that gets and sets whether to keep the toolbody.

## Syntax

SolidSweepDefinition.**KeepToolbody**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
