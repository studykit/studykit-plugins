# ContentFamily.ActiveCell Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that returns a floating cell object that caller can use for iterating the table using indices. Set the Row and Column properties on the returned object to iterate through the table.

## Syntax

ContentFamily.**ActiveCell**() As [ContentTableCell](../ContentTableCell/ContentTableCell.md)

## Property Value

This is a read only property whose value is a [ContentTableCell](../ContentTableCell/ContentTableCell.md).

## Version

Introduced in version 2011
