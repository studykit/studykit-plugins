# ContentFamily.DisplayName Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Gets/Sets the name of this family.

## Syntax

ContentFamily.**DisplayName**() As String

## Property Value

This is a read/write property whose value is a String.
