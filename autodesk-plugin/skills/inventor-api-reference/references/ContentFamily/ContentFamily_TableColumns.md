# ContentFamily.TableColumns Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that returns the collection of columns for the table associated with this family.

## Syntax

ContentFamily.**TableColumns**() As [ContentTableColumns](../ContentTableColumns/ContentTableColumns.md)

## Property Value

This is a read only property whose value is a [ContentTableColumns](../ContentTableColumns/ContentTableColumns.md).

## Version

Introduced in version 2010
