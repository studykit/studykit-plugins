# EdgeLoopProxy.Edges Property

Parent Object: [EdgeLoopProxy](../EdgeLoopProxy/EdgeLoopProxy.md)

## Description

Gets the referenced by this EdgeLoop.

## Syntax

EdgeLoopProxy.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 4
