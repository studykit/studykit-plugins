# ChangeManager.Parent Property

Parent Object: [ChangeManager](../ChangeManager/ChangeManager.md)

## Description

Property that returns the Application object, the parent of the ChangeManager object.

## Syntax

ChangeManager.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 9
