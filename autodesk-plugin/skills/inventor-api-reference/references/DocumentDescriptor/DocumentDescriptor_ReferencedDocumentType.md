# DocumentDescriptor.ReferencedDocumentType Property

Parent Object: [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Description

Property that returns type of the referenced document.

## Syntax

DocumentDescriptor.**ReferencedDocumentType**() As [DocumentTypeEnum](../DocumentTypeEnum.md)

## Property Value

This is a read only property whose value is a [DocumentTypeEnum](../DocumentTypeEnum.md).

## Version

Introduced in version 11
