# DocumentDescriptor.ReferencedDocument Property

Parent Object: [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Description

Property that returns the Document object being referenced.

## Syntax

DocumentDescriptor.**ReferencedDocument**() As Object

## Property Value

This is a read only property whose value is an Object.
