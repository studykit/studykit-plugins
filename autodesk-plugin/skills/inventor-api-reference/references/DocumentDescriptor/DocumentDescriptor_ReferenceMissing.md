# DocumentDescriptor.ReferenceMissing Property

Parent Object: [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Description

Property that returns whether the referenced document is missing (unresolved or suppressed).

## Syntax

DocumentDescriptor.**ReferenceMissing**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
