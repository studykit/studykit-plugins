# DocumentDescriptor.Parent Property

Parent Object: [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Description

Property that returns the parent of this object.

## Syntax

DocumentDescriptor.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
