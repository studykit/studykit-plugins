# Edge.PointOnEdge Property

Parent Object: [Edge](../Edge/Edge.md)

## Description

Property that returns a characteristic somewhere in the middle of the Edge.

## Syntax

Edge.**PointOnEdge**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 9
