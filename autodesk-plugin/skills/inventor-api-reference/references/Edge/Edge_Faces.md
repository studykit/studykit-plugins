# Edge.Faces Property

Parent Object: [Edge](../Edge/Edge.md)

## Description

Property that returns the that this Edge is referenced from.

## Syntax

Edge.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 4
