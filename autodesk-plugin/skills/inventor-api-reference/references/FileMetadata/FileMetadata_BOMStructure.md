# FileMetadata.BOMStructure Property

Parent Object: [FileMetadata](../FileMetadata/FileMetadata.md)

## Description

Gets and sets the default BOM structure.

## Syntax

FileMetadata.**BOMStructure**() As [BOMStructureEnum](../BOMStructureEnum.md)

## Property Value

This is a read/write property whose value is a [BOMStructureEnum](../BOMStructureEnum.md).

## Version

Introduced in version 2019
