# FileMetadata.Type Property

Parent Object: [FileMetadata](../FileMetadata/FileMetadata.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FileMetadata.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
