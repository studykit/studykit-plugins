# FileMetadata.DisplayName Property

Parent Object: [FileMetadata](../FileMetadata/FileMetadata.md)

## Description

Gets and sets name of the file.

## Syntax

FileMetadata.**DisplayName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
