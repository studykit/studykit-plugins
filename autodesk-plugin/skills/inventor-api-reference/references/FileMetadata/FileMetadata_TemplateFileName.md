# FileMetadata.TemplateFileName Property

Parent Object: [FileMetadata](../FileMetadata/FileMetadata.md)

## Description

Gets and sets the full filename of the file used as the template when create a new file.

## Syntax

FileMetadata.**TemplateFileName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2019
