# TwoLineAngleDimConstraint.LineOne Property

Parent Object: [TwoLineAngleDimConstraint](../TwoLineAngleDimConstraint/TwoLineAngleDimConstraint.md)

## Description

Property that returns the first sketch line being constrained.

## Syntax

TwoLineAngleDimConstraint.**LineOne**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 5
