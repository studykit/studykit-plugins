# ShellDefinition.ApproximationTolerance Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Gets and sets the approximation tolerance.

## Syntax

ShellDefinition.**ApproximationTolerance**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
