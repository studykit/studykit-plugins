# ShellDefinition.FaceSetItem Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Method that returns the specified ShellThicknessFaceSet object from the collection.

## Syntax

ShellDefinition.**FaceSetItem**( ***Index*** As Long ) As [ShellThicknessFaceSet](../ShellThicknessFaceSet/ShellThicknessFaceSet.md)

## Property Value

This is a read only property whose value is a [ShellThicknessFaceSet](../ShellThicknessFaceSet/ShellThicknessFaceSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index within the collection of the item to return. |

## Version

Introduced in version 9
