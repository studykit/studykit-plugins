# ShellDefinition.Method Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Read-write property that gets and sets the shell method. This defaults to kSharpShellMethod when the definition is just created.

## Syntax

ShellDefinition.**Method**() As [ShellMethodEnum](../ShellMethodEnum.md)

## Property Value

This is a read/write property whose value is a [ShellMethodEnum](../ShellMethodEnum.md).

## Version

Introduced in version 2026
