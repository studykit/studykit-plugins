# ShellDefinition.Direction Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Gets and sets the shell boundary relative to part face.

## Syntax

ShellDefinition.**Direction**() As [ShellDirectionEnum](../ShellDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [ShellDirectionEnum](../ShellDirectionEnum.md).

## Version

Introduced in version 9
