# ShellDefinition.ApproximationType Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Gets and sets the approximation method used for the feature.

## Syntax

ShellDefinition.**ApproximationType**() As [FeatureApproximationTypeEnum](../FeatureApproximationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [FeatureApproximationTypeEnum](../FeatureApproximationTypeEnum.md).

## Version

Introduced in version 11
