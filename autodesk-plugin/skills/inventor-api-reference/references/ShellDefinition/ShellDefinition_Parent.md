# ShellDefinition.Parent Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

ShellDefinition.**Parent**() As [ShellFeature](../ShellFeature/ShellFeature.md)

## Property Value

This is a read only property whose value is a [ShellFeature](../ShellFeature/ShellFeature.md).

## Version

Introduced in version 9
