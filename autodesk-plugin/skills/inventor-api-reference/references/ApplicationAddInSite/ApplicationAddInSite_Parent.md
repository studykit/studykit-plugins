# ApplicationAddInSite.Parent Property

Parent Object: [ApplicationAddInSite](../ApplicationAddInSite/ApplicationAddInSite.md)

## Description

Property that returns the parent ApplicationAddIn object.

## Syntax

ApplicationAddInSite.**Parent**() As [ApplicationAddIn](../ApplicationAddIn/ApplicationAddIn.md)

## Property Value

This is a read only property whose value is an [ApplicationAddIn](../ApplicationAddIn/ApplicationAddIn.md).

## Version

Introduced in version 7
