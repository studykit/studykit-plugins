# ApplicationAddInServer.Deactivate Method

Parent Object: [ApplicationAddInServer](../ApplicationAddInServer/ApplicationAddInServer.md)

## Description

Invoked by Autodesk Inventor to shut down the AddIn. AddIn should complete shutdown within this call.

## Syntax

ApplicationAddInServer.**Deactivate**()

## Version

Introduced in version 4
