# Views Object

Derived from: [ViewsEnumerator](../ViewsEnumerator/ViewsEnumerator.md) Object

## Description

The Views collection object provides access to all of the graphic  objects associated with a particular document. It also provides functionality to create new views.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Views/Views_Add.md) | Create a new view of the document. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../Views/Views_Count.md) | Property that returns the number of items in this collection. |
| [Item](../Views/Views_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../Views/Views_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
