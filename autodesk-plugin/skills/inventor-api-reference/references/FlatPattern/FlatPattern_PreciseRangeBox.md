# FlatPattern.PreciseRangeBox Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Gets a bounding box that tightly encloses all the solid and surface bodies under the ComponentDefinition.

## Syntax

FlatPattern.**PreciseRangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2023
