# FlatPattern.WorkAxes Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns that collection object that provides access to all of the existing work axis objects in the flat pattern and provides the functionality to create new work axis objects.

## Syntax

FlatPattern.**WorkAxes**() As [WorkAxes](../WorkAxes/WorkAxes.md)

## Property Value

This is a read only property whose value is a [WorkAxes](../WorkAxes/WorkAxes.md).

## Version

Introduced in version 2010
