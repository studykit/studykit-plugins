# FlatPattern.PartEvents Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the PartEvents object associated with the flat pattern. This object will fire events for changes that occur to the flat pattern.

## Syntax

FlatPattern.**PartEvents**() As [PartEvents](../PartEvents/PartEvents.md)

## Property Value

This is a read only property whose value is a [PartEvents](../PartEvents/PartEvents.md).

## Version

Introduced in version 2010
