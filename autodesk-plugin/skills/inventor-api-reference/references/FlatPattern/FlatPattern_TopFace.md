# FlatPattern.TopFace Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns top face of the flat pattern.

## Syntax

FlatPattern.**TopFace**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2008
