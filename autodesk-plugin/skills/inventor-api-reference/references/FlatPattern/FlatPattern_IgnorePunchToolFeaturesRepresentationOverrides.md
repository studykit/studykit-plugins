# FlatPattern.IgnorePunchToolFeaturesRepresentationOverrides Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Gets and sets whether to ignore the punch tool features representation overrides in flat pattern.

## Syntax

FlatPattern.**IgnorePunchToolFeaturesRepresentationOverrides**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
