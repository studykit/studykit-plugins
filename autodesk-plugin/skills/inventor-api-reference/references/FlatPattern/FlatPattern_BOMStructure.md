# FlatPattern.BOMStructure Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Gets and sets how the component is used/viewed in a BOM.

## Syntax

FlatPattern.**BOMStructure**() As [BOMStructureEnum](../BOMStructureEnum.md)

## Property Value

This is a read/write property whose value is a [BOMStructureEnum](../BOMStructureEnum.md).

## Version

Introduced in version 2010
