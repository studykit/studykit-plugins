# FlatPattern.BaseFace Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the base face from the folded model that was used to create the flat pattern.

## Syntax

FlatPattern.**BaseFace**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2008
