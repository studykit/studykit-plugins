# FlatPattern.RolledBackForEdit Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that gets whether the flat pattern model is currently rolled back to a previous point in the feature history.

## Syntax

FlatPattern.**RolledBackForEdit**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
