# FlatPattern.Parameters Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the set of parameters for the flat pattern.

## Syntax

FlatPattern.**Parameters**() As [Parameters](../Parameters/Parameters.md)

## Property Value

This is a read only property whose value is a [Parameters](../Parameters/Parameters.md).

## Version

Introduced in version 2010
