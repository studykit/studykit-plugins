# FlatPattern.ASideFace Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Read-only property that returns the ASideFaceDefinition from the folded model that was used to create the flat pattern.

## Syntax

FlatPattern.**ASideFace**() As [ASideDefinition](../ASideDefinition/ASideDefinition.md)

## Property Value

This is a read only property whose value is an [ASideDefinition](../ASideDefinition/ASideDefinition.md).

## Version

Introduced in version 2016
