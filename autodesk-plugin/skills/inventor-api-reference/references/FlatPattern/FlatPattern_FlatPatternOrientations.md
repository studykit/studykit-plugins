# FlatPattern.FlatPatternOrientations Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Gets the FlatPatternOrientations collection object.

## Syntax

FlatPattern.**FlatPatternOrientations**() As [FlatPatternOrientations](../FlatPatternOrientations/FlatPatternOrientations.md)

## Property Value

This is a read only property whose value is a [FlatPatternOrientations](../FlatPatternOrientations/FlatPatternOrientations.md).

## Version

Introduced in version 2015
