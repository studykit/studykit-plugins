# FlatPattern.UserCoordinateSystems Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the UserCoordinateSystems collection object.

## Syntax

FlatPattern.**UserCoordinateSystems**() As [UserCoordinateSystems](../UserCoordinateSystems/UserCoordinateSystems.md)

## Property Value

This is a read only property whose value is a [UserCoordinateSystems](../UserCoordinateSystems/UserCoordinateSystems.md).

## Version

Introduced in version 2010
