# FlatPattern.BendAngleReportType Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Gets and sets which angle is reported in the flat pattern.

## Syntax

FlatPattern.**BendAngleReportType**() As [BendAngleTypeEnum](../BendAngleTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BendAngleTypeEnum](../BendAngleTypeEnum.md).

## Version

Introduced in version 2010
