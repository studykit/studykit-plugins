# FlatPattern.MassProperties Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the MassProperties object for the flat pattern.

## Syntax

FlatPattern.**MassProperties**() As [MassProperties](../MassProperties/MassProperties.md)

## Property Value

This is a read only property whose value is a [MassProperties](../MassProperties/MassProperties.md).

## Version

Introduced in version 11
