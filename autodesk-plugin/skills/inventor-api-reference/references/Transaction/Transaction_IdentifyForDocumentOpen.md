# Transaction.IdentifyForDocumentOpen Method

Parent Object: [Transaction](../Transaction/Transaction.md)

## Description

Identify the document open transaction. This action should precede the end of document open transaction.

## Syntax

Transaction.**IdentifyForDocumentOpen**( ***Document*** As [Document](../Document/Document.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Document | [Document](../Document/Document.md) | The associated with this transaction. |

## Version

Introduced in version 9
