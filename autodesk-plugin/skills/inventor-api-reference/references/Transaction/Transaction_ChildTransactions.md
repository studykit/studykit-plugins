# Transaction.ChildTransactions Property

Parent Object: [Transaction](../Transaction/Transaction.md)

## Description

Gets the enumeration of all of the child transactions that this transaction may contain.

## Syntax

Transaction.**ChildTransactions**() As [TransactionsEnumerator](../TransactionsEnumerator/TransactionsEnumerator.md)

## Property Value

This is a read only property whose value is a [TransactionsEnumerator](../TransactionsEnumerator/TransactionsEnumerator.md).

## Version

Introduced in version 5
