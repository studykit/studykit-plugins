# Transaction.CheckPoints Property

Parent Object: [Transaction](../Transaction/Transaction.md)

## Description

Gets the enumeration of all of the check points that may be present in this transaction.

## Syntax

Transaction.**CheckPoints**() As [CheckPointsEnumerator](../CheckPointsEnumerator/CheckPointsEnumerator.md)

## Property Value

This is a read only property whose value is a [CheckPointsEnumerator](../CheckPointsEnumerator/CheckPointsEnumerator.md).

## Version

Introduced in version 5
