# Features.MoveFaceFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the MoveFaceFeatures collection object. This collection provides access to existing MoveFaceFeatures objects.

## Syntax

Features.**MoveFaceFeatures**() As [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md)

## Property Value

This is a read only property whose value is a [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md).

## Version

Introduced in version 10
