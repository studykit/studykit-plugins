# Features.FilletFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the FilletFeatures collection object. This collection provides access to existing FilletFeature objects.

## Syntax

Features.**FilletFeatures**() As [FilletFeatures](../FilletFeatures/FilletFeatures.md)

## Property Value

This is a read only property whose value is a [FilletFeatures](../FilletFeatures/FilletFeatures.md).

## Version

Introduced in version 10
