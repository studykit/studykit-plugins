# DerivedFutureAssemblyOccurrence.SubOccurrences Property

Parent Object: [DerivedFutureAssemblyOccurrence](../DerivedFutureAssemblyOccurrence/DerivedFutureAssemblyOccurrence.md)

## Description

Read-only property that returns the occurrences in the top-level of the assembly this DerivedFutureAssemblyOccurrence object represents. This property is only valid in the case where the IsAssembly property is True.
.

## Syntax

DerivedFutureAssemblyOccurrence.**SubOccurrences**() As [DerivedFutureAssemblyOccurrences](../DerivedFutureAssemblyOccurrences/DerivedFutureAssemblyOccurrences.md)

## Property Value

This is a read only property whose value is a [DerivedFutureAssemblyOccurrences](../DerivedFutureAssemblyOccurrences/DerivedFutureAssemblyOccurrences.md).

## Version

Introduced in version 2018
