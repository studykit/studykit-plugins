# ThemeManager.Themes Property

Parent Object: [ThemeManager](../ThemeManager/ThemeManager.md)

## Description

Read-only property that returns the themes collection object.

## Syntax

ThemeManager.**Themes**() As [ThemesEnumerator](../ThemesEnumerator/ThemesEnumerator.md)

## Property Value

This is a read only property whose value is a [ThemesEnumerator](../ThemesEnumerator/ThemesEnumerator.md).
