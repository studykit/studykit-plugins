# SurfaceGraphicsFaceList.Count Property

Parent Object: [SurfaceGraphicsFaceList](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList.md)

## Description

Property that returns the number of SurfaceGraphicsFace objects in the list.

## Syntax

SurfaceGraphicsFaceList.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2009
