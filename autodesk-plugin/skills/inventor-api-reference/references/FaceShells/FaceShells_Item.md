# FaceShells.Item Property

Parent Object: [FaceShells](../FaceShells/FaceShells.md)

## Description

Allows integer-indexed access to objects in the collection.

## Syntax

FaceShells.**Item**( ***Index*** As Long ) As [FaceShell](../FaceShell/FaceShell.md)

## Property Value

This is a read only property whose value is a [FaceShell](../FaceShell/FaceShell.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 4
