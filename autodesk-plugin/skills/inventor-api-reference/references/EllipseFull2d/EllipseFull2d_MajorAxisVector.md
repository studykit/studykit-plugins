# EllipseFull2d.MajorAxisVector Property

Parent Object: [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md)

## Description

Specifies the direction of the major axis of the ellipse.

## Syntax

EllipseFull2d.**MajorAxisVector**() As [Vector2d](../Vector2d/Vector2d.md)

## Property Value

This is a read/write property whose value is a [Vector2d](../Vector2d/Vector2d.md).

## Version

Introduced in version 4
