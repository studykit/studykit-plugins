# EllipseFull2d Object

## Description

The EllipseFull2d object. The object is a transient mathematical object and is not displayed graphically.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../EllipseFull2d/EllipseFull2d_Copy.md) | Creates a copy of this EllipseFull2d object. The result is entirely independent and can be edited without affecting the original EllipseFull2d object. |
| [GetEllipseFullData](../EllipseFull2d/EllipseFull2d_GetEllipseFullData.md) | Get the data defining this ellipse. |
| [PutEllipseFullData](../EllipseFull2d/EllipseFull2d_PutEllipseFullData.md) | Method that sets the data defining this ellipse. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Center](../EllipseFull2d/EllipseFull2d_Center.md) | Specifies the center of the ellipse. |
| [Evaluator](../EllipseFull2d/EllipseFull2d_Evaluator.md) | Gets the Curve2dEvaluator for this ellipse. |
| [MajorAxisVector](../EllipseFull2d/EllipseFull2d_MajorAxisVector.md) | Specifies the direction of the major axis of the ellipse. |
| [MinorMajorRatio](../EllipseFull2d/EllipseFull2d_MinorMajorRatio.md) | Specifies the ratio of the minor and major axes of the ellipse. |
