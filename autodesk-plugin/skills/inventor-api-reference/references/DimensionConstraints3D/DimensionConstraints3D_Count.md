# DimensionConstraints3D.Count Property

Parent Object: [DimensionConstraints3D](../DimensionConstraints3D/DimensionConstraints3D.md)

## Description

Property that specifies the number of items in the collection.

## Syntax

DimensionConstraints3D.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 11
