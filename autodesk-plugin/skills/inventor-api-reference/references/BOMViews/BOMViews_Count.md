# BOMViews.Count Property

Parent Object: [BOMViews](../BOMViews/BOMViews.md)

## Description

Property that returns the number of items in the collection.

## Syntax

BOMViews.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.
