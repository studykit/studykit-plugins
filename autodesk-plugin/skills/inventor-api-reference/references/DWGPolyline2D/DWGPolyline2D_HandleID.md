# DWGPolyline2D.HandleID Property

Parent Object: [DWGPolyline2D](../DWGPolyline2D/DWGPolyline2D.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGPolyline2D.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
