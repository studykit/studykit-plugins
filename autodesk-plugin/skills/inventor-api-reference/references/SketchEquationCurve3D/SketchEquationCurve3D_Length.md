# SketchEquationCurve3D.Length Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Gets the length of the entity in centimeters.

## Syntax

SketchEquationCurve3D.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2014
