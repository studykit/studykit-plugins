# SketchEquationCurve3D.HasReferenceComponent Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchEquationCurve3D.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
