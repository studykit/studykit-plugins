# SketchEquationCurve3D.Reference Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Gets and sets whether this entity is a reference entity or not.

## Syntax

SketchEquationCurve3D.**Reference**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
