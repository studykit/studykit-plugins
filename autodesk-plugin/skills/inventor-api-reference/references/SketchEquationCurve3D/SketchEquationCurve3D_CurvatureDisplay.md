# SketchEquationCurve3D.CurvatureDisplay Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Specifies whether curvature information is displayed for the curve.

## Syntax

SketchEquationCurve3D.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
