# SketchEquationCurve3D.StartSketchPoint Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Read-only property that gets the sketch point that defines the position of the start of the sketch spline.

## Syntax

SketchEquationCurve3D.**StartSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 2014
