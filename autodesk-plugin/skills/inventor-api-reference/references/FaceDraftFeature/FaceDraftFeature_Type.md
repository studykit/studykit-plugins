# FaceDraftFeature.Type Property

Parent Object: [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FaceDraftFeature.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
