# OrdinateDimensionsEnumerator.Item Property

Parent Object: [OrdinateDimensionsEnumerator](../OrdinateDimensionsEnumerator/OrdinateDimensionsEnumerator.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

OrdinateDimensionsEnumerator.**Item**( ***Index*** As Long ) As [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Property Value

This is a read only property whose value is an [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md).

| Name | Type | Description |
| Index | Long |  |
