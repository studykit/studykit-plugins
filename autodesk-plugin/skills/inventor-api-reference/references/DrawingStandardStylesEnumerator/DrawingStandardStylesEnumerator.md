# DrawingStandardStylesEnumerator Object

## Description

The DrawingStandardStylesEnumerator object provides access to the existing objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingStandardStylesEnumerator/DrawingStandardStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DrawingStandardStylesEnumerator/DrawingStandardStylesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../DrawingStandardStylesEnumerator/DrawingStandardStylesEnumerator_Item.md) | Returns the specified object in the collection. |
| [Type](../DrawingStandardStylesEnumerator/DrawingStandardStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.StandardStyles](../DrawingStylesManager/DrawingStylesManager_StandardStyles.md)

## Version

Introduced in version 9
