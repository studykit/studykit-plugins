# SearchBox.SearchText Property

Parent Object: [SearchBox](../SearchBox/SearchBox.md)

## Description

Read-only property that returns the search text.

## Syntax

SearchBox.**SearchText**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
