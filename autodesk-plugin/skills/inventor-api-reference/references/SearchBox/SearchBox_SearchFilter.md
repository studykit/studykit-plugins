# SearchBox.SearchFilter Property

Parent Object: [SearchBox](../SearchBox/SearchBox.md)

## Description

Read-only property that returns the search box filter. This could be a bit-mask defining various of filters. Set this to 0 will clear the search filter.

## Syntax

SearchBox.**SearchFilter**() As [SearchBoxFilterEnum](../SearchBoxFilterEnum.md)

## Property Value

This is a read only property whose value is a [SearchBoxFilterEnum](../SearchBoxFilterEnum.md).

## Version

Introduced in version 2018
