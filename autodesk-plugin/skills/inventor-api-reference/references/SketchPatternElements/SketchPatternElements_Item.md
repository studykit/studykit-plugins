# SketchPatternElements.Item Property

Parent Object: [SketchPatternElements](../SketchPatternElements/SketchPatternElements.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

SketchPatternElements.**Item**( ***Index*** As Long ) As [SketchPatternElement](../SketchPatternElement/SketchPatternElement.md)

## Property Value

This is a read only property whose value is a [SketchPatternElement](../SketchPatternElement/SketchPatternElement.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the pattern element to return. If an out of range index is provided, an error occurs |

## Version

Introduced in version 2025.1
