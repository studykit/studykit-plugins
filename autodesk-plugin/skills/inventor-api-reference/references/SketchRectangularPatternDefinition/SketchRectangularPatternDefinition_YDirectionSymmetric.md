# SketchRectangularPatternDefinition.YDirectionSymmetric Property

Parent Object: [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Description

Gets and sets if the occurrences are distributed on both sides of the original geometry.

## Syntax

SketchRectangularPatternDefinition.**YDirectionSymmetric**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
