# SketchRectangularPatternDefinition Object

## Description

SketchRectangularPatternDefinition Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_Copy.md) | Creates a copy of this SketchRectangularPatternDefinition object. The new SketchRectangularPatternDefinition object is independent of any pattern. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Associative](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_Associative.md) | Gets and sets whether the pattern will update when changes are made to the part. If set this to False then the pattern constraints will be removed and the pattern will not update when change an element. |
| [BoundarySetting](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_BoundarySetting.md) | Gets and set the PatternBoundarySetting object. When set this property a PatternBoundarySetting can be copied using PatternBoundarySetting. |
| [Fitted](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_Fitted.md) | Gets and sets whether pattern elements are equally fitted within the specified distances. |
| [Geometries](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_Geometries.md) | Gets and sets the ObjectCollection object that contains the sketch geometries to be patterned. |
| [HasBoundarySettingConfigured](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_HasBoundarySettingConfigured.md) | Gets and sets whether this pattern has boundary setting configured or not. |
| [NaturalXDirection](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_NaturalXDirection.md) | Gets and sets the flag that indicates if the direction of the pattern is in the natural direction of the XDirectionEntity or reversed. |
| [NaturalYDirection](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_NaturalYDirection.md) | Gets and sets the flag that indicates if the direction of the pattern is in the natural direction of the YDirectionEntity or reversed. |
| [SuppressedElements](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_SuppressedElements.md) | Gets and sets the suppressed elements. |
| [Type](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_Type.md) | Gets the constant that indicates the type of this object. |
| [XCount](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_XCount.md) | Gets and sets the number of instances in the X direction. |
| [XDirectionEntity](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_XDirectionEntity.md) | Gets and sets the X direction entity. |
| [XDirectionSymmetric](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_XDirectionSymmetric.md) | Gets and sets if the occurrences are distributed on both sides of the original geometry. |
| [XSpacing](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_XSpacing.md) | Gets and sets the spacing between instances in the X direction. |
| [YCount](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_YCount.md) | Gets and sets the number of instances in the Y direction. |
| [YDirectionEntity](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_YDirectionEntity.md) | Gets and sets the Y direction entity. |
| [YDirectionSymmetric](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_YDirectionSymmetric.md) | Gets and sets if the occurrences are distributed on both sides of the original geometry. |
| [YSpacing](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition_YSpacing.md) | Gets and sets the spacing between instances in the Y direction. |
