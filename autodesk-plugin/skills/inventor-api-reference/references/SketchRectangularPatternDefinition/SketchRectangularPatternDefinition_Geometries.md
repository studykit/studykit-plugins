# SketchRectangularPatternDefinition.Geometries Property

Parent Object: [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Description

Gets and sets the ObjectCollection object that contains the sketch geometries to be patterned.

## Syntax

SketchRectangularPatternDefinition.**Geometries**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2025.1
