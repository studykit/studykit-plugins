# SketchRectangularPatternDefinition.XDirectionEntity Property

Parent Object: [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Description

Gets and sets the X direction entity.

## Syntax

SketchRectangularPatternDefinition.**XDirectionEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2025.1
