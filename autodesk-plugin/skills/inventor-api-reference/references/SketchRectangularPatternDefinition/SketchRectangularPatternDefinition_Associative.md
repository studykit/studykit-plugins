# SketchRectangularPatternDefinition.Associative Property

Parent Object: [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Description

Gets and sets whether the pattern will update when changes are made to the part. If set this to False then the pattern constraints will be removed and the pattern will not update when change an element.

## Syntax

SketchRectangularPatternDefinition.**Associative**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
