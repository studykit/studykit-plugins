# InventorVBAProject.Close Method

Parent Object: [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Description

Method that closes this project. Please note that this method works only for user-defined VBA projects. It will not work with Application or Document VBA projects, which are built in.

## Syntax

InventorVBAProject.**Close**()

## Version

Introduced in version 6
