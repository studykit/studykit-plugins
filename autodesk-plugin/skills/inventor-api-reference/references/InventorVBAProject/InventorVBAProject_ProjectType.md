# InventorVBAProject.ProjectType Property

Parent Object: [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Description

Property that returns a constant specifying the type of VBA project this project is.

## Syntax

InventorVBAProject.**ProjectType**() As [VBAProjectTypeEnum](../VBAProjectTypeEnum.md)

## Property Value

This is a read only property whose value is a [VBAProjectTypeEnum](../VBAProjectTypeEnum.md).

## Version

Introduced in version 6
