# ThreePointAngleDimConstraint.PointTwo Property

Parent Object: [ThreePointAngleDimConstraint](../ThreePointAngleDimConstraint/ThreePointAngleDimConstraint.md)

## Description

Property that returns the second sketch point being constrained.

## Syntax

ThreePointAngleDimConstraint.**PointTwo**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 5
