# MeshEdge.MeshFeatureEntity Property

Parent Object: [MeshEdge](../MeshEdge/MeshEdge.md)

## Description

Read-only property that returns the MeshFeatureEntity this MeshEdge is located on.

## Syntax

MeshEdge.**MeshFeatureEntity**() As [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Property Value

This is a read only property whose value is a [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md).

## Version

Introduced in version 2017
