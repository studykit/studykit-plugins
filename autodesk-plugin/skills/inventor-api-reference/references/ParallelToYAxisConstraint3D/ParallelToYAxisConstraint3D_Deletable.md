# ParallelToYAxisConstraint3D.Deletable Property

Parent Object: [ParallelToYAxisConstraint3D](../ParallelToYAxisConstraint3D/ParallelToYAxisConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

ParallelToYAxisConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2017
