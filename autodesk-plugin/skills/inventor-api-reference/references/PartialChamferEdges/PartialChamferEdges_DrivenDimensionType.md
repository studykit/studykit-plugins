# PartialChamferEdges.DrivenDimensionType Property

Parent Object: [PartialChamferEdges](../PartialChamferEdges/PartialChamferEdges.md)

## Description

Gets and sets which partial chamfer dimension is driven dimension.

## Syntax

PartialChamferEdges.**DrivenDimensionType**() As [PartialChamferDrivenDimensionTypeEnum](../PartialChamferDrivenDimensionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PartialChamferDrivenDimensionTypeEnum](../PartialChamferDrivenDimensionTypeEnum.md).

## Version

Introduced in version 2018
