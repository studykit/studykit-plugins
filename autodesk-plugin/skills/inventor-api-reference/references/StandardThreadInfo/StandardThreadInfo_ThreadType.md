# StandardThreadInfo.ThreadType Property

Parent Object: [StandardThreadInfo](../StandardThreadInfo/StandardThreadInfo.md)

## Description

Gets and sets the thread type.

## Syntax

StandardThreadInfo.**ThreadType**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5
