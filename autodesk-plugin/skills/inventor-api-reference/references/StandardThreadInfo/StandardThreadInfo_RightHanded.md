# StandardThreadInfo.RightHanded Property

Parent Object: [StandardThreadInfo](../StandardThreadInfo/StandardThreadInfo.md)

## Description

Gets and sets whether this is a right handed thread or left handed (False).

## Syntax

StandardThreadInfo.**RightHanded**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
