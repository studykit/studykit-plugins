# StandardThreadInfo.NominalSize Property

Parent Object: [StandardThreadInfo](../StandardThreadInfo/StandardThreadInfo.md)

## Description

Property that returns the description of the nominal size. Any string is valid but the following examples are typical of the strings used, '1.6', 'M4', and '9/16'.

## Syntax

StandardThreadInfo.**NominalSize**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 9
