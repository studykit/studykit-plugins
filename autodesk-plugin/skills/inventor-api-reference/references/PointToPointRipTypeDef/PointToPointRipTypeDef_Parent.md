# PointToPointRipTypeDef.Parent Property

Parent Object: [PointToPointRipTypeDef](../PointToPointRipTypeDef/PointToPointRipTypeDef.md)

## Description

Property that returns the parent rip definition feature.

## Syntax

PointToPointRipTypeDef.**Parent**() As [RipDefinition](../RipDefinition/RipDefinition.md)

## Property Value

This is a read only property whose value is a [RipDefinition](../RipDefinition/RipDefinition.md).

## Version

Introduced in version 2011
