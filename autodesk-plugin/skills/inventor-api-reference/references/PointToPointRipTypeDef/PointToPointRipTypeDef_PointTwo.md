# PointToPointRipTypeDef.PointTwo Property

Parent Object: [PointToPointRipTypeDef](../PointToPointRipTypeDef/PointToPointRipTypeDef.md)

## Description

Gets and sets the point that defines the location of the rip.

## Syntax

PointToPointRipTypeDef.**PointTwo**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2011
