# DrawingDocument.DrawingEvents Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that gets the DrawingEvents object. This provides drawing event notification such as onRetrieveDimensions.

## Syntax

DrawingDocument.**DrawingEvents**() As [DrawingEvents](../DrawingEvents/DrawingEvents.md)

## Property Value

This is a read only property whose value is a [DrawingEvents](../DrawingEvents/DrawingEvents.md).

## Version

Introduced in version 9
