# DrawingDocument.ActiveSheet Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the currently active sheet.

## Syntax

DrawingDocument.**ActiveSheet**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).
