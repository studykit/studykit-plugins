# DrawingDocument.DrawingHatchPatternsManager Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Read-only property that returns the DrawingHatchPatternsManager object.

## Syntax

DrawingDocument.**DrawingHatchPatternsManager**() As [DrawingHatchPatternsManager](../DrawingHatchPatternsManager/DrawingHatchPatternsManager.md)

## Property Value

This is a read only property whose value is a [DrawingHatchPatternsManager](../DrawingHatchPatternsManager/DrawingHatchPatternsManager.md).

## Version

Introduced in version 2021
