# DrawingDocument.NonTransactingGraphicsDataSetsCollection Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Read-only property that returns the non-transacting GraphicsDataSetsCollection object.

## Syntax

DrawingDocument.**NonTransactingGraphicsDataSetsCollection**() As [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md).

## Version

Introduced in version 2017
