# DrawingDocument.DefaultCommand Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Gets the default command for this document.

## Syntax

DrawingDocument.**DefaultCommand**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
