# DrawingDocument.FileSaveCounter Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Gets the number that indicates the number of times the file has been saved.

## Syntax

DrawingDocument.**FileSaveCounter**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
