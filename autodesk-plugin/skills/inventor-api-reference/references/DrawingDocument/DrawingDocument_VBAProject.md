# DrawingDocument.VBAProject Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the Autodesk Inventor VBA project for this document.

## Syntax

DrawingDocument.**VBAProject**() As [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Property Value

This is a read only property whose value is an [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md).

## Version

Introduced in version 6
