# DrawingDocument.FilePropertySets Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Read-only property that returns the file PropertySets. This is equivalent to the iProperties that got from Shell Extension.

## Syntax

DrawingDocument.**FilePropertySets**() As [PropertySets](../PropertySets/PropertySets.md)

## Property Value

This is a read only property whose value is a [PropertySets](../PropertySets/PropertySets.md).

## Version

Introduced in version 2022
