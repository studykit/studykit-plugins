# DrawingDocument.HasPrivateStream Method

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Obtains a Boolean flag indicating if the specified stream exists within this document.

## Syntax

DrawingDocument.**HasPrivateStream**( ***StreamName*** As String ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| StreamName | String | Input String value that specifies the name of the stream. |

## Version

Introduced in version 4
