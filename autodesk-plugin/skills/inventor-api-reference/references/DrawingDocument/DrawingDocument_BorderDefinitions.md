# DrawingDocument.BorderDefinitions Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the BorderDefinitions collection object. This object provides access to all of the border definitions in the document and provides functionality to create new border definitions.

## Syntax

DrawingDocument.**BorderDefinitions**() As [BorderDefinitions](../BorderDefinitions/BorderDefinitions.md)

## Property Value

This is a read only property whose value is a [BorderDefinitions](../BorderDefinitions/BorderDefinitions.md).
