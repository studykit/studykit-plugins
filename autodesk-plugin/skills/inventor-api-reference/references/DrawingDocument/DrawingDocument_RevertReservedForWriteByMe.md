# DrawingDocument.RevertReservedForWriteByMe Method

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Method that reverts the file checked out by the caller.

## Syntax

DrawingDocument.**RevertReservedForWriteByMe**()

## Version

Introduced in version 6
