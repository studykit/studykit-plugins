# DrawingDocument.Views Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Gets all the open views of this document in a collection.

## Syntax

DrawingDocument.**Views**() As [Views](../Views/Views.md)

## Property Value

This is a read only property whose value is a [Views](../Views/Views.md).

## Version

Introduced in version 4
