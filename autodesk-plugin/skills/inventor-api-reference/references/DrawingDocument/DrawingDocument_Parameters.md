# DrawingDocument.Parameters Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the set of parameters stored in this drawing document.

## Syntax

DrawingDocument.**Parameters**() As [Parameters](../Parameters/Parameters.md)

## Property Value

This is a read only property whose value is a [Parameters](../Parameters/Parameters.md).

## Version

Introduced in version 2011
