# AuxiliaryFeatureIndicator.Type Property

Parent Object: [AuxiliaryFeatureIndicator](../AuxiliaryFeatureIndicator/AuxiliaryFeatureIndicator.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

AuxiliaryFeatureIndicator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2023
