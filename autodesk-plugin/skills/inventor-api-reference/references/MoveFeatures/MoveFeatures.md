# MoveFeatures Object

## Description

The MoveFeatures object provides access to all of the MoveFeature objects in a part.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../MoveFeatures/MoveFeatures_Add.md) | Creates a new move feature. |
| [CreateMoveDefinition](../MoveFeatures/MoveFeatures_CreateMoveDefinition.md) | Creates a MoveDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MoveFeatures/MoveFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../MoveFeatures/MoveFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../MoveFeatures/MoveFeatures_Item.md) | Property that returns an item from the collection. You can provide either the index of the item in the collection, where the first item is index 1, or you can provide the name of the object. |
| [Type](../MoveFeatures/MoveFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
