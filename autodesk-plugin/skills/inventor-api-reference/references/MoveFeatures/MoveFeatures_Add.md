# MoveFeatures.Add Method

Parent Object: [MoveFeatures](../MoveFeatures/MoveFeatures.md)

## Description

Creates a new move feature.

## Syntax

MoveFeatures.**Add**( ***MoveDefinition*** As [MoveDefinition](../MoveDefinition/MoveDefinition.md) ) As [MoveFeature](../MoveFeature/MoveFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| MoveDefinition | [MoveDefinition](../MoveDefinition/MoveDefinition.md) | Input MoveDefinition object that defines the input for the move feature. |
