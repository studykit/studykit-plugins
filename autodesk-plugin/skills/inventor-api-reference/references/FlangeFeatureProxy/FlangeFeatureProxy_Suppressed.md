# FlangeFeatureProxy.Suppressed Property

Parent Object: [FlangeFeatureProxy](../FlangeFeatureProxy/FlangeFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

FlangeFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
