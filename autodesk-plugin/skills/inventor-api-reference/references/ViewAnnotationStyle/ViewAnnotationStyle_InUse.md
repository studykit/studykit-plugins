# ViewAnnotationStyle.InUse Property

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Property that indicates if this style is in use.

## Syntax

ViewAnnotationStyle.**InUse**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025.1
