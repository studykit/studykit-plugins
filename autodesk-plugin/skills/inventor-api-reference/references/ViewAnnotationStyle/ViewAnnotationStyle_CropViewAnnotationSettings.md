# ViewAnnotationStyle.CropViewAnnotationSettings Property

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Read-only property that returns the CropViewAnnotationStyleSettings object.

## Syntax

ViewAnnotationStyle.**CropViewAnnotationSettings**() As [CropViewAnnotationStyleSettings](../CropViewAnnotationStyleSettings/CropViewAnnotationStyleSettings.md)

## Property Value

This is a read only property whose value is a [CropViewAnnotationStyleSettings](../CropViewAnnotationStyleSettings/CropViewAnnotationStyleSettings.md).

## Version

Introduced in version 2025.1
