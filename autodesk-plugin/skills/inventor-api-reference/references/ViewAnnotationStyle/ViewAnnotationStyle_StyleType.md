# ViewAnnotationStyle.StyleType Property

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Gets the type of the style.

## Syntax

ViewAnnotationStyle.**StyleType**() As [StyleTypeEnum](../StyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [StyleTypeEnum](../StyleTypeEnum.md).

## Version

Introduced in version 2025.1
