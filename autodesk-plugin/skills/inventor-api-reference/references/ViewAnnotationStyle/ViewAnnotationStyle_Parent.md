# ViewAnnotationStyle.Parent Property

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Property returning the parent of this object.

## Syntax

ViewAnnotationStyle.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2025.1
