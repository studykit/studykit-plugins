# ViewAnnotationStyle.UpToDate Property

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Property that gets the up-to-date status of the style against the global repository.

## Syntax

ViewAnnotationStyle.**UpToDate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025.1
