# ViewAnnotationStyle.Type Property

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ViewAnnotationStyle.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2025.1
