# MergedCell.Unmerge Method

Parent Object: [MergedCell](../MergedCell/MergedCell.md)

## Description

Splits this MergedCell object into individual cells.

## Syntax

MergedCell.**Unmerge**()

## Version

Introduced in version 2025.1
