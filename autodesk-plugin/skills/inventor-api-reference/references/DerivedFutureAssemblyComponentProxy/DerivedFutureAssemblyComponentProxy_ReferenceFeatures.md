# DerivedFutureAssemblyComponentProxy.ReferenceFeatures Property

Parent Object: [DerivedFutureAssemblyComponentProxy](../DerivedFutureAssemblyComponentProxy/DerivedFutureAssemblyComponentProxy.md)

## Description

Read-only property that returns the collection containing the ReferenceFeatures that were created as a result of the derived component.

## Syntax

DerivedFutureAssemblyComponentProxy.**ReferenceFeatures**() As [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md)

## Property Value

This is a read only property whose value is a [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md).

## Version

Introduced in version 2018
