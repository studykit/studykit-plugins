# DerivedFutureAssemblyComponentProxy.SuppressLinkToFile Property

Parent Object: [DerivedFutureAssemblyComponentProxy](../DerivedFutureAssemblyComponentProxy/DerivedFutureAssemblyComponentProxy.md)

## Description

Read-write property that suppresses or unsuppresses the link to the base assembly.

## Syntax

DerivedFutureAssemblyComponentProxy.**SuppressLinkToFile**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
