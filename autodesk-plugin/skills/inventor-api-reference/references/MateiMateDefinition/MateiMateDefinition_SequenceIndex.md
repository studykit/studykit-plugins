# MateiMateDefinition.SequenceIndex Property

Parent Object: [MateiMateDefinition](../MateiMateDefinition/MateiMateDefinition.md)

## Description

Gets the index of this iMateDefinition.

## Syntax

MateiMateDefinition.**SequenceIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 6
