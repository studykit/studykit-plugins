# MateiMateDefinition.EntityInferredType Property

Parent Object: [MateiMateDefinition](../MateiMateDefinition/MateiMateDefinition.md)

## Description

Property that returns an enum indicating how the geometry of the entity is interpreted.

## Syntax

MateiMateDefinition.**EntityInferredType**() As [InferredTypeEnum](../InferredTypeEnum.md)

## Property Value

This is a read only property whose value is an [InferredTypeEnum](../InferredTypeEnum.md).

## Version

Introduced in version 6
