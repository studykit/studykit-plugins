# Profile3DProxy.Item Property

Parent Object: [Profile3DProxy](../Profile3DProxy/Profile3DProxy.md)

## Description

Method that returns the specified ProfilePath3D object from the collection.

## Syntax

Profile3DProxy.**Item**( ***Index*** As Long ) As [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Property Value

This is a read only property whose value is a [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
