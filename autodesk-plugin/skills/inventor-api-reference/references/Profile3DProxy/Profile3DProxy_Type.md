# Profile3DProxy.Type Property

Parent Object: [Profile3DProxy](../Profile3DProxy/Profile3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

Profile3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
