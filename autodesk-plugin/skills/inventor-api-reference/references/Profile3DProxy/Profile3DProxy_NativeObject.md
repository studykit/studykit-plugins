# Profile3DProxy.NativeObject Property

Parent Object: [Profile3DProxy](../Profile3DProxy/Profile3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

Profile3DProxy.**NativeObject**() As [Profile3D](../Profile3D/Profile3D.md)

## Property Value

This is a read only property whose value is a [Profile3D](../Profile3D/Profile3D.md).

## Version

Introduced in version 6
