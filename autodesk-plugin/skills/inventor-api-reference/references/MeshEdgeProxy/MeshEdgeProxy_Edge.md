# MeshEdgeProxy.Edge Property

Parent Object: [MeshEdgeProxy](../MeshEdgeProxy/MeshEdgeProxy.md)

## Description

Read-only property that returns the transient Brep Edge for the MeshEdge.

## Syntax

MeshEdgeProxy.**Edge**() As [Edge](../Edge/Edge.md)

## Property Value

This is a read only property whose value is an [Edge](../Edge/Edge.md).

## Version

Introduced in version 2017
