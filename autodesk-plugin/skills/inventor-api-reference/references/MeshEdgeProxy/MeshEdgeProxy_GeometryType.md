# MeshEdgeProxy.GeometryType Property

Parent Object: [MeshEdgeProxy](../MeshEdgeProxy/MeshEdgeProxy.md)

## Description

Read-only property that returns a curve type of the curve that will be returned from the Geometry property.

## Syntax

MeshEdgeProxy.**GeometryType**() As [CurveTypeEnum](../CurveTypeEnum.md)

## Property Value

This is a read only property whose value is a [CurveTypeEnum](../CurveTypeEnum.md).

## Version

Introduced in version 2017
