# MeshEdgeProxy.NativeObject Property

Parent Object: [MeshEdgeProxy](../MeshEdgeProxy/MeshEdgeProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MeshEdgeProxy.**NativeObject**() As [MeshEdge](../MeshEdge/MeshEdge.md)

## Property Value

This is a read only property whose value is a [MeshEdge](../MeshEdge/MeshEdge.md).

## Version

Introduced in version 2017
