# MeshEdgeProxy.MeshFeatureEntity Property

Parent Object: [MeshEdgeProxy](../MeshEdgeProxy/MeshEdgeProxy.md)

## Description

Read-only property that returns the MeshFeatureEntity this MeshEdge is located on.

## Syntax

MeshEdgeProxy.**MeshFeatureEntity**() As [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Property Value

This is a read only property whose value is a [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md).

## Version

Introduced in version 2017
