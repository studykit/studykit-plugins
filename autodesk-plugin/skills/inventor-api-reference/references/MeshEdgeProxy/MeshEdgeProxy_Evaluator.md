# MeshEdgeProxy.Evaluator Property

Parent Object: [MeshEdgeProxy](../MeshEdgeProxy/MeshEdgeProxy.md)

## Description

Read-only property that returns the curve evaluator for this MeshEdge. This evaluator differs from the one that could be obtained from the surface geometry definition in that this evaluator accounts for the topological orientation.

## Syntax

MeshEdgeProxy.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 2017
