# FactoryOptions.PartNumberIndexStepValue Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the step value for the part number index.

## Syntax

FactoryOptions.**PartNumberIndexStepValue**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
