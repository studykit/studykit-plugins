# FactoryOptions.PartNumberIndexSeparator Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the separator for the part number index.

## Syntax

FactoryOptions.**PartNumberIndexSeparator**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
