# FactoryOptions.PartNumberType Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the type of value to use for the part number.

## Syntax

FactoryOptions.**PartNumberType**() As [FactoryOptionsPartNumberTypeEnum](../FactoryOptionsPartNumberTypeEnum.md)

## Property Value

This is a read/write property whose value is a [FactoryOptionsPartNumberTypeEnum](../FactoryOptionsPartNumberTypeEnum.md).

## Version

Introduced in version 2010
