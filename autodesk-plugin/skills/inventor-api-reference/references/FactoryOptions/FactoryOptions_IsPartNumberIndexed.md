# FactoryOptions.IsPartNumberIndexed Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets whether the part number displayed in the iPart factory table is indexed.

## Syntax

FactoryOptions.**IsPartNumberIndexed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
