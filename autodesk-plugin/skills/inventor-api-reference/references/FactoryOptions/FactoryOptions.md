# FactoryOptions Object

## Description

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [ApplyToAllMembers](../FactoryOptions/FactoryOptions_ApplyToAllMembers.md) | Method that applies the factory options to all members of the factory including existing ones. By default, any factory options specified using the FactoryOptions object will only apply to newly created members. If the options need to be applied to existing members also, this method needs to be called. |
