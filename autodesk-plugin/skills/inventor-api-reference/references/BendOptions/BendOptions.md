# BendOptions Object

## Description

The BendOptions object defines the settings that influence the creation of a bend in a sheet metal part.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../BendOptions/BendOptions_Copy.md) | Function that creates a copy of this BendOptions object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [BendReliefDepth](../BendOptions/BendOptions_BendReliefDepth.md) | Provides access to the bend relief depth of a sheet metal feature. |
| [BendReliefShape](../BendOptions/BendOptions_BendReliefShape.md) | Defines the type of bend relief shape to use for the associated sheet metal feature. |
| [BendReliefWidth](../BendOptions/BendOptions_BendReliefWidth.md) | Provides access to the bend relief width of a sheet metal feature. |
