# DWGSplinesEnumerator.Item Property

Parent Object: [DWGSplinesEnumerator](../DWGSplinesEnumerator/DWGSplinesEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGSplinesEnumerator.**Item**( ***Index*** As Variant ) As [DWGSpline](../DWGSpline/DWGSpline.md)

## Property Value

This is a read only property whose value is a [DWGSpline](../DWGSpline/DWGSpline.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
