# FileUIEvents.Type Property

Parent Object: [FileUIEvents](../FileUIEvents/FileUIEvents.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FileUIEvents.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 7
