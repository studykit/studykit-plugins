# FileUIEvents.Parent Property

Parent Object: [FileUIEvents](../FileUIEvents/FileUIEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

FileUIEvents.**Parent**() As [Application](../Application/Application.md)

## Property Value

This is a read only property whose value is an [Application](../Application/Application.md).

## Version

Introduced in version 7
