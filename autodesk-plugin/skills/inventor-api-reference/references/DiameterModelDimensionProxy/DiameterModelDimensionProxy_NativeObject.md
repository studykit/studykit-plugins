# DiameterModelDimensionProxy.NativeObject Property

Parent Object: [DiameterModelDimensionProxy](../DiameterModelDimensionProxy/DiameterModelDimensionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DiameterModelDimensionProxy.**NativeObject**() As [DiameterModelDimension](../DiameterModelDimension/DiameterModelDimension.md)

## Property Value

This is a read only property whose value is a [DiameterModelDimension](../DiameterModelDimension/DiameterModelDimension.md).

## Version

Introduced in version 2018
