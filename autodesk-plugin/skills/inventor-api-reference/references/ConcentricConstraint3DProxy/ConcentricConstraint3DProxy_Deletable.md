# ConcentricConstraint3DProxy.Deletable Property

Parent Object: [ConcentricConstraint3DProxy](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

ConcentricConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
