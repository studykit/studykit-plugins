# PointCloudScans Object

## Description

PointCloudScans Collection Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointCloudScans/PointCloudScans_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../PointCloudScans/PointCloudScans_Count.md) | Gets the number of items in this collection. |
| [Item](../PointCloudScans/PointCloudScans_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../PointCloudScans/PointCloudScans_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PointCloud.Scans](../PointCloud/PointCloud_Scans.md), [PointCloudProxy.Scans](../PointCloudProxy/PointCloudProxy_Scans.md)

## Version

Introduced in version 2016
