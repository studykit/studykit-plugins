# LeaderNote.HorizontalJustification Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Gets/Sets horizontal justification of the DrawingNote.

## Syntax

LeaderNote.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 10
