# LeaderNote.BackgroundColor Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Gets/Sets the BackgroundColor associated with the LeaderNote.

## Syntax

LeaderNote.**BackgroundColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2017
