# LeaderNote.Parent Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Property that returns the parent sheet of the object.

## Syntax

LeaderNote.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 10
