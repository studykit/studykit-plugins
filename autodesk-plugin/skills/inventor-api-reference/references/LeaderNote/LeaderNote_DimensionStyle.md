# LeaderNote.DimensionStyle Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Gets/Sets the DimensionStyle associated with the LeaderNote.

## Syntax

LeaderNote.**DimensionStyle**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 10
