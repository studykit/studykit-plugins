# LeaderNote.RetrievedFrom Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Returns the model annotation this drawing note was retrieved from.

## Syntax

LeaderNote.**RetrievedFrom**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2025
