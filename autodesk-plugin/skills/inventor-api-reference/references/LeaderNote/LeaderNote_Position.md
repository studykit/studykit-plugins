# LeaderNote.Position Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Gets/Sets the position of the DrawingNote on the sheet.

## Syntax

LeaderNote.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
