# LeaderNote.Leader Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Property that returns the Leader object.

## Syntax

LeaderNote.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).
