# FilenameAssetValue.HasMultipleValues Property

Parent Object: [FilenameAssetValue](../FilenameAssetValue/FilenameAssetValue.md)

## Description

Gets the boolean flag that indicates if this value has multiple values or not.

## Syntax

FilenameAssetValue.**HasMultipleValues**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
