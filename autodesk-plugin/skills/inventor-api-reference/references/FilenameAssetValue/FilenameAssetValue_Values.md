# FilenameAssetValue.Values Property

Parent Object: [FilenameAssetValue](../FilenameAssetValue/FilenameAssetValue.md)

## Description

Gets and sets the values associated with this asset value. HasMultipleValues property indicates if this property will be returning more than one value.

## Syntax

FilenameAssetValue.**Values**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2014
