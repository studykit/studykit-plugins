# MeshVertexProxy.NativeObject Property

Parent Object: [MeshVertexProxy](../MeshVertexProxy/MeshVertexProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MeshVertexProxy.**NativeObject**() As [MeshVertex](../MeshVertex/MeshVertex.md)

## Property Value

This is a read only property whose value is a [MeshVertex](../MeshVertex/MeshVertex.md).

## Version

Introduced in version 2017
