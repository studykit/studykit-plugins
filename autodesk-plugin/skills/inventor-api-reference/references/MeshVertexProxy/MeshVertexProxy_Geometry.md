# MeshVertexProxy.Geometry Property

Parent Object: [MeshVertexProxy](../MeshVertexProxy/MeshVertexProxy.md)

## Description

Read-only property that returns the underlying geometry of the MeshEdge.

## Syntax

MeshVertexProxy.**Geometry**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2017
