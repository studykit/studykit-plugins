# DWGPolylinesEnumerator.Item Property

Parent Object: [DWGPolylinesEnumerator](../DWGPolylinesEnumerator/DWGPolylinesEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGPolylinesEnumerator.**Item**( ***Index*** As Variant ) As [DWGPolyline](../DWGPolyline/DWGPolyline.md)

## Property Value

This is a read only property whose value is a [DWGPolyline](../DWGPolyline/DWGPolyline.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
