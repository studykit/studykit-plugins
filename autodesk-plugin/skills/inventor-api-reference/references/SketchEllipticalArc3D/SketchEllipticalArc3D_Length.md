# SketchEllipticalArc3D.Length Property

Parent Object: [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchEllipticalArc3D.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
