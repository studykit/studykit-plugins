# SketchEllipticalArc3D.StartAngle Property

Parent Object: [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md)

## Description

Gets the StartAngle of the elliptical arc.

## Syntax

SketchEllipticalArc3D.**StartAngle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
