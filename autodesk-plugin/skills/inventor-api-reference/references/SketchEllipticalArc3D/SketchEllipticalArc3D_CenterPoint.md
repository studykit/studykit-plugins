# SketchEllipticalArc3D.CenterPoint Property

Parent Object: [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md)

## Description

Property that returns the center point of the elliptical arc.

## Syntax

SketchEllipticalArc3D.**CenterPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
