# SketchEllipticalArc3D.MajorAxisVector Property

Parent Object: [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md)

## Description

Gets the MajorAxisVector of the elliptical arc.

## Syntax

SketchEllipticalArc3D.**MajorAxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2010
