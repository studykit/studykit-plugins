# DynamicSimulation.CurrentTimeStep Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Gets and sets the current time step of the animation. Valid values are 1 to the LastComputedTimeStep.

## Syntax

DynamicSimulation.**CurrentTimeStep**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2013
