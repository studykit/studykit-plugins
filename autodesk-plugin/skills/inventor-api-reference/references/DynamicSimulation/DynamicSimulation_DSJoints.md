# DynamicSimulation.DSJoints Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Gets the DSJoints object which provides access to all of the joints associated with this simulation.

## Syntax

DynamicSimulation.**DSJoints**() As [DSJoints](../DSJoints/DSJoints.md)

## Property Value

This is a read only property whose value is a [DSJoints](../DSJoints/DSJoints.md).

## Version

Introduced in version 2013
