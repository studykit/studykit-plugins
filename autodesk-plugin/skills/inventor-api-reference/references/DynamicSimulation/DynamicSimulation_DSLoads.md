# DynamicSimulation.DSLoads Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Gets the DSLoads object which provides access to all of the loads associated with this simulation.

## Syntax

DynamicSimulation.**DSLoads**() As [DSLoads](../DSLoads/DSLoads.md)

## Property Value

This is a read only property whose value is a [DSLoads](../DSLoads/DSLoads.md).

## Version

Introduced in version 2013
