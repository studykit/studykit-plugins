# TwoPointDistanceDimConstraint.Orientation Property

Parent Object: [TwoPointDistanceDimConstraint](../TwoPointDistanceDimConstraint/TwoPointDistanceDimConstraint.md)

## Description

Property that indicates an enum indicating if the constraint is controlling the horizontal, vertical, or true distance between the points.

## Syntax

TwoPointDistanceDimConstraint.**Orientation**() As [DimensionOrientationEnum](../DimensionOrientationEnum.md)

## Property Value

This is a read only property whose value is a [DimensionOrientationEnum](../DimensionOrientationEnum.md).

## Version

Introduced in version 5
