# DockableWindow.ShowVisibilityCheckBox Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Gets and sets whether to display the visibility check box in the User Interface dropdown in the View tab | Windows panel.

## Syntax

DockableWindow.**ShowVisibilityCheckBox**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
