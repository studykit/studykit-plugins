# DockableWindow.VisibilityControl Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Property that returns the button control automatically created by Inventor to control the visibility of the dockable window.

## Syntax

DockableWindow.**VisibilityControl**() As [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Property Value

This is a read only property whose value is a [ControlDefinition](../ControlDefinition/ControlDefinition.md).

## Version

Introduced in version 2011
