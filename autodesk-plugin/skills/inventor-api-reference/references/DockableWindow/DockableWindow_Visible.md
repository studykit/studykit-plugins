# DockableWindow.Visible Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Read-write property that gets and sets whether the window is currently visible.

## Syntax

DockableWindow.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
