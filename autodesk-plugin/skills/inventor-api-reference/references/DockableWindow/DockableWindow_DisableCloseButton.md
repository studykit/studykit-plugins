# DockableWindow.DisableCloseButton Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Gets and sets whether to disable close button.

## Syntax

DockableWindow.**DisableCloseButton**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
