# DockableWindow.Width Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Gets and sets the current width of the window.

## Syntax

DockableWindow.**Width**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2011
