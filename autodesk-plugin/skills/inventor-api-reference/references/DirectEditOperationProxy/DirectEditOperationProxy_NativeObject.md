# DirectEditOperationProxy.NativeObject Property

Parent Object: [DirectEditOperationProxy](../DirectEditOperationProxy/DirectEditOperationProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

DirectEditOperationProxy.**NativeObject**() As [DirectEditOperation](../DirectEditOperation/DirectEditOperation.md)

## Property Value

This is a read only property whose value is a [DirectEditOperation](../DirectEditOperation/DirectEditOperation.md).

## Version

Introduced in version 2015
