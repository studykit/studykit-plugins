# SketchImage.BoundaryGeometry Property

Parent Object: [SketchImage](../SketchImage/SketchImage.md)

## Description

Property that returns the four construction sketch lines that form the boundary of the image.

## Syntax

SketchImage.**BoundaryGeometry**() As [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md).

## Version

Introduced in version 11
