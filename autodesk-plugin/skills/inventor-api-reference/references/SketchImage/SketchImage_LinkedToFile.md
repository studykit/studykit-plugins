# SketchImage.LinkedToFile Property

Parent Object: [SketchImage](../SketchImage/SketchImage.md)

## Description

Property that returns whether this image is linked to the picture file that was used to create this image.

## Syntax

SketchImage.**LinkedToFile**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
