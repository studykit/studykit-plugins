# TransitionSymbolIndicationTypeEnum Enumerator

## Description

Constants that define the drawing transition symbol attachment types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAdditionalIndicationsPresentedSymbolIndication | 129543 | Specify additional specific transition indications presented in the drawing. |
| kAdditionalIndicationsWithinParenthesesSymbolIndication | 129544 | Specify additional specific transition indications within parentheses. |
| kForAllTransitionsSymbolIndication | 129540 | Symbol defined for all transitions. |
| kForExternalTransitionsSymbolIndication | 129541 | Symbol defined for external transitions. |
| kForInternalTransitionsSymbolIndication | 129542 | Symbol defined for internal transitions. |
| kHorizontallyAdjacentSymbolIndication | 129538 | The symbol is horizontally adjacent. |
| kNoSymbolIndication | 129537 | No symbol indication. |
| kVerticallyAdjacentSymbolIndication | 129539 | The symbol is vertically adjacent. |

## Version

Introduced in version 2025.1
