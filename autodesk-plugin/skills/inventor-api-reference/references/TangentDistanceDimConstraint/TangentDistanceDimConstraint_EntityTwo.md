# TangentDistanceDimConstraint.EntityTwo Property

Parent Object: [TangentDistanceDimConstraint](../TangentDistanceDimConstraint/TangentDistanceDimConstraint.md)

## Description

Property that returns the second circle or line being constrained.

## Syntax

TangentDistanceDimConstraint.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
