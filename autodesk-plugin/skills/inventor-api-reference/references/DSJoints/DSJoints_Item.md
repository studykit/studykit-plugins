# DSJoints.Item Property

Parent Object: [DSJoints](../DSJoints/DSJoints.md)

## Description

Returns the specified DSJoint object from the collection.

## Syntax

DSJoints.**Item**( ***Index*** As Variant ) As [DSJoint](../DSJoint/DSJoint.md)

## Property Value

This is a read only property whose value is a [DSJoint](../DSJoint/DSJoint.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the DSJoint to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the DSJoint name. If an out of range index or a name of a non-existent DSJoint is provided, an error will occur. |

## Version

Introduced in version 2013
