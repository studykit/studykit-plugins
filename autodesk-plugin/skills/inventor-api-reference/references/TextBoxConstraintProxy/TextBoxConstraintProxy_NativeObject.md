# TextBoxConstraintProxy.NativeObject Property

Parent Object: [TextBoxConstraintProxy](../TextBoxConstraintProxy/TextBoxConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

TextBoxConstraintProxy.**NativeObject**() As [TextBoxConstraint](../TextBoxConstraint/TextBoxConstraint.md)

## Property Value

This is a read only property whose value is a [TextBoxConstraint](../TextBoxConstraint/TextBoxConstraint.md).

## Version

Introduced in version 11
