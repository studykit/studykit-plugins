# RevitExportStructureTypeEnum Enumerator

## Description

Constants identifying what Revit export structure type is used.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllInOneElementStructure | 119041 | Everything structured as a single Revit element. |
| kEachTopLevelComponentStructure | 119042 | Top level components as Revit elements. |
| kPartsOnlyStructure | 119043 | Only part components as Revit elements. |

## Version

Introduced in version 2022
