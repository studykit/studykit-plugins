# MouseDragStateEnum Enumerator

## Description

Enumeration of the conditions of the mouse drag.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDragEnter | 15617 | Drag occurring while entering this window. |
| kDragLeave | 15618 | Drag occurring while leaving this window. |
| kDragOver | 15619 | Drag occurring while within this window. |
| kDragUnknown | 15620 | Drag occurring in an unknown state. |

## Version

Introduced in version 5
