# SketchCircles.AddByThreePoints Method

Parent Object: [SketchCircles](../SketchCircles/SketchCircles.md)

## Description

Method that creates a new sketch circle passing through the three input points. The three points are any three points along the perimeter of the desired circle.

## Syntax

SketchCircles.**AddByThreePoints**( ***PointOne*** As [Point2d](../Point2d/Point2d.md), ***PointTwo*** As [Point2d](../Point2d/Point2d.md), ***PointThree*** As [Point2d](../Point2d/Point2d.md) ) As [SketchCircle](../SketchCircle/SketchCircle.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PointOne | [Point2d](../Point2d/Point2d.md) | Input that defines the first point along the circle. |
| PointTwo | [Point2d](../Point2d/Point2d.md) | Input that defines the second point along the circle. |
| PointThree | [Point2d](../Point2d/Point2d.md) | Input that defines the third point along the circle. |

## Version

Introduced in version 5
