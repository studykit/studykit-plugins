# DWGEntitySegmentsEnumerator.Item Property

Parent Object: [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGEntitySegmentsEnumerator.**Item**( ***Index*** As Variant ) As [DWGEntitySegment](../DWGEntitySegment/DWGEntitySegment.md)

## Property Value

This is a read only property whose value is a [DWGEntitySegment](../DWGEntitySegment/DWGEntitySegment.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the segment to return. |

## Version

Introduced in version 2018
