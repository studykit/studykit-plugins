# ContentTableRow.Item Property

Parent Object: [ContentTableRow](../ContentTableRow/ContentTableRow.md)

## Description

Returns the specified ContentTableCell object from the collection.

## Syntax

ContentTableRow.**Item**( ***Index*** As Variant ) As [ContentTableCell](../ContentTableCell/ContentTableCell.md)

## Property Value

This is a read only property whose value is a [ContentTableCell](../ContentTableCell/ContentTableCell.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Value that specifies the ContentTableCell object within the collection to return. Valid index values are 1 to the value of Count or a ContentTableColumn object. |

## Version

Introduced in version 2010
