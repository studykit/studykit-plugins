# ContentTableRow.IsSuppressed Property

Parent Object: [ContentTableRow](../ContentTableRow/ContentTableRow.md)

## Description

Gets/sets the suppressed state of this row.

## Syntax

ContentTableRow.**IsSuppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
