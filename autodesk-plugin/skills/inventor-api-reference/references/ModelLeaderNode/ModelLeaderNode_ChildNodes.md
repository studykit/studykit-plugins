# ModelLeaderNode.ChildNodes Property

Parent Object: [ModelLeaderNode](../ModelLeaderNode/ModelLeaderNode.md)

## Description

Read-only that returns the child nodes under this node. The count will return 0 if the node is a leaf node.

## Syntax

ModelLeaderNode.**ChildNodes**() As [ModelLeaderNodesEnumerator](../ModelLeaderNodesEnumerator/ModelLeaderNodesEnumerator.md)

## Property Value

This is a read only property whose value is a [ModelLeaderNodesEnumerator](../ModelLeaderNodesEnumerator/ModelLeaderNodesEnumerator.md).

## Version

Introduced in version 2018
