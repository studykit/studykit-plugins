# ModelWeldingSymbolDefinitions.Leader Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Read-only property that returns the leader associated with the model welding symbol.

## Syntax

ModelWeldingSymbolDefinitions.**Leader**() As [ModelLeader](../ModelLeader/ModelLeader.md)

## Property Value

This is a read only property whose value is a [ModelLeader](../ModelLeader/ModelLeader.md).

## Version

Introduced in version 2024
