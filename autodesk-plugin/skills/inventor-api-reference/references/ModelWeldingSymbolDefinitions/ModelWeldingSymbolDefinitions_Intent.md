# ModelWeldingSymbolDefinitions.Intent Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Gets and sets the geometric entity the welding symbol is attached to.

## Syntax

ModelWeldingSymbolDefinitions.**Intent**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2024
