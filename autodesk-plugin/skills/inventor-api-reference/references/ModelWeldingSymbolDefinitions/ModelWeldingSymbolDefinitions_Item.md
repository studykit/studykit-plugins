# ModelWeldingSymbolDefinitions.Item Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

ModelWeldingSymbolDefinitions.**Item**( ***Index*** As Long ) As [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Property Value

This is a read only property whose value is a [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of ModelWeldingSymbolDefinition to return. |

## Version

Introduced in version 2024
