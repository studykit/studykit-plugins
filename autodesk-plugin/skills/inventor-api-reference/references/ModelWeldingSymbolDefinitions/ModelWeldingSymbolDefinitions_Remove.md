# ModelWeldingSymbolDefinitions.Remove Method

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Method that removes the specified ModelWeldingSymbolDefinition from the ModelWeldingSymbolDefinitions collection.

## Syntax

ModelWeldingSymbolDefinitions.**Remove**( ***Index*** As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value indicating the index of the ModelWeldingSymbolDefinition to remove. |

## Version

Introduced in version 2024
