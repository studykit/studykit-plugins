# ModelWeldingSymbolDefinitions.Position Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Gets and sets the position of the welding symbol in the part or assembly. The point is projected onto the orientation plane.

## Syntax

ModelWeldingSymbolDefinitions.**Position**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2024
