# ModelWeldingSymbolDefinitions.Count Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Gets the number of items in this collection.

## Syntax

ModelWeldingSymbolDefinitions.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.
