# SketchFillRegions.Add Method

Parent Object: [SketchFillRegions](../SketchFillRegions/SketchFillRegions.md)

## Description

Method to add a SketchFillRegion object to the collection.

## Syntax

SketchFillRegions.**Add**( ***Profile*** As [Profile](../Profile/Profile.md), [***Color***] As Variant ) As [SketchFillRegion](../SketchFillRegion/SketchFillRegion.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Profile | [Profile](../Profile/Profile.md) | The Profile object to use for this region. |
| Color | Variant | The color for this SketchFillRegion. |
