# SketchFillRegions Object

## Description

A collection of SketchFillRegion objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchFillRegions/SketchFillRegions_Add.md) | Method to add a SketchFillRegion object to the collection. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchFillRegions/SketchFillRegions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchFillRegions/SketchFillRegions_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchFillRegions/SketchFillRegions_Item.md) | Method that returns the specified object from the collection. |
