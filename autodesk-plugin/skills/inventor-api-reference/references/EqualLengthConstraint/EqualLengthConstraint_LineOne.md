# EqualLengthConstraint.LineOne Property

Parent Object: [EqualLengthConstraint](../EqualLengthConstraint/EqualLengthConstraint.md)

## Description

Property that returns the sketch line being constrained.

## Syntax

EqualLengthConstraint.**LineOne**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 5
