# ModelDatum.ModelDatumTargets Property

Parent Object: [ModelDatum](../ModelDatum/ModelDatum.md)

## Description

Returns a ModelDatumTargets collection object providing access to the existing model datum targets in this model datum.

## Syntax

ModelDatum.**ModelDatumTargets**() As [ModelDatumTargets](../ModelDatumTargets/ModelDatumTargets.md)

## Property Value

This is a read only property whose value is a [ModelDatumTargets](../ModelDatumTargets/ModelDatumTargets.md).

## Version

Introduced in version 2023
