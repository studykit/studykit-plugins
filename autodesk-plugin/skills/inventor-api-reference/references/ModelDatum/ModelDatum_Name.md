# ModelDatum.Name Property

Parent Object: [ModelDatum](../ModelDatum/ModelDatum.md)

## Description

Gets or sets the name of this object.

## Syntax

ModelDatum.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2023
