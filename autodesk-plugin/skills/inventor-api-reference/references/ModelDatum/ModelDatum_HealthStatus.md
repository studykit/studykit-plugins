# ModelDatum.HealthStatus Property

Parent Object: [ModelDatum](../ModelDatum/ModelDatum.md)

## Description

Returns the current health status of the model datum.

## Syntax

ModelDatum.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2023
