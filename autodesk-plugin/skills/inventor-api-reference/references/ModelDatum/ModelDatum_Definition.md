# ModelDatum.Definition Property

Parent Object: [ModelDatum](../ModelDatum/ModelDatum.md)

## Description

Gets and sets the ModelDatumDefinition associated with this ModelDatum object.

## Syntax

ModelDatum.**Definition**() As [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Property Value

This is a read/write property whose value is a [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md).

## Version

Introduced in version 2023
