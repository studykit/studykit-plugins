# FaceShellDefinitions Object

## Description

The FaceShellDefinitions object represents a collection of FaceShellDefinition objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../FaceShellDefinitions/FaceShellDefinitions_Add.md) | Method that creates a new FaceShellDefinition within the associated SurfaceBodyDefinition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FaceShellDefinitions/FaceShellDefinitions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FaceShellDefinitions/FaceShellDefinitions_Count.md) | Property that returns the number of items in this collection. |
| [Item](../FaceShellDefinitions/FaceShellDefinitions_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../FaceShellDefinitions/FaceShellDefinitions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
