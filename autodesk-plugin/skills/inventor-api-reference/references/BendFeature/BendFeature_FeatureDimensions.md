# BendFeature.FeatureDimensions Property

Parent Object: [BendFeature](../BendFeature/BendFeature.md)

## Description

Property that returns the FeatureDimensions collection object.

## Syntax

BendFeature.**FeatureDimensions**() As [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md)

## Property Value

This is a read only property whose value is a [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md).

## Version

Introduced in version 2008
