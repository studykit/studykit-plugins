# MassProperties.AchievedAccuracy Method

Parent Object: [MassProperties](../MassProperties/MassProperties.md)

## Description

Property that gets the actual accuracy achieved while computing the mass property calculations.

## Syntax

MassProperties.**AchievedAccuracy**( ***AchievedAccuracyArea*** As Double, ***AchievedAccuracyVolume*** As Double )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| AchievedAccuracyArea | Double | Gets the actual accuracy achieved while computing the mass property calculations for the area. |
| AchievedAccuracyVolume | Double | Gets the actual accuracy achieved while computing the mass property calculations for the vol/com/centroid. |

## Version

Introduced in version 9
