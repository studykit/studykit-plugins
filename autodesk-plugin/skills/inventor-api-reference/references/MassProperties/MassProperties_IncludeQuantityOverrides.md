# MassProperties.IncludeQuantityOverrides Property

Parent Object: [MassProperties](../MassProperties/MassProperties.md)

## Description

Gets and sets whether BOM quantity overrides should be used for assembly components to perform mass property calculations.

## Syntax

MassProperties.**IncludeQuantityOverrides**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
