# MassProperties.Mass Property

Parent Object: [MassProperties](../MassProperties/MassProperties.md)

## Description

Gets and sets the mass of the model in database units.

## Syntax

MassProperties.**Mass**() As Double

## Property Value

This is a read/write property whose value is a Double.
