# SketchPoint3D.AttachedEntities Property

Parent Object: [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Description

Property that returns the collection of objects that are connected to this point.

## Syntax

SketchPoint3D.**AttachedEntities**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 6
