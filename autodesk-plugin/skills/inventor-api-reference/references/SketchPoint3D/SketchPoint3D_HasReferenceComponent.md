# SketchPoint3D.HasReferenceComponent Property

Parent Object: [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchPoint3D.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
