# SketchPoint3D.Type Property

Parent Object: [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchPoint3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
