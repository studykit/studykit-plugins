# SketchEllipticalArc3DProxy.Parent Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchEllipticalArc3DProxy.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 6
