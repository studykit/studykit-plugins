# SketchEllipticalArc3DProxy.MajorAxisVector Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Gets the MajorAxisVector of the elliptical arc.

## Syntax

SketchEllipticalArc3DProxy.**MajorAxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2010
