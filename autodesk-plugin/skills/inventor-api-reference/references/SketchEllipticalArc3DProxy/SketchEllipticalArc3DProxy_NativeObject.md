# SketchEllipticalArc3DProxy.NativeObject Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchEllipticalArc3DProxy.**NativeObject**() As [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md)

## Property Value

This is a read only property whose value is a [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md).

## Version

Introduced in version 6
