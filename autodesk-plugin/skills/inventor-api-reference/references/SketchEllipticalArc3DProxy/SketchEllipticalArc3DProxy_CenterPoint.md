# SketchEllipticalArc3DProxy.CenterPoint Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Property that returns the center point of the elliptical arc.

## Syntax

SketchEllipticalArc3DProxy.**CenterPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
