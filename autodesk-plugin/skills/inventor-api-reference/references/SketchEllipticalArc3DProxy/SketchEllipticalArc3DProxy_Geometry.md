# SketchEllipticalArc3DProxy.Geometry Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Gets and sets an EllipticalArc geometry object. The object returned represents a 'snapshot' view of the current state of the sketch ellipse.

## Syntax

SketchEllipticalArc3DProxy.**Geometry**() As [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Property Value

This is a read/write property whose value is an [EllipticalArc](../EllipticalArc/EllipticalArc.md).

## Version

Introduced in version 6
