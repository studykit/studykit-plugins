# SketchEllipticalArc3DProxy.Type Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchEllipticalArc3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
