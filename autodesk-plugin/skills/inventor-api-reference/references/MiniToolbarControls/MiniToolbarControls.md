# MiniToolbarControls Object

## Description

The MiniToolbarControls object contains the controls within a minitoolbar and methods to create new controls.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddButton](../MiniToolbarControls/MiniToolbarControls_AddButton.md) | Method that creates a new button control within the MiniToolbar. |
| [AddCheckBox](../MiniToolbarControls/MiniToolbarControls_AddCheckBox.md) | Method that creates a new check box control within the MiniToolbar. |
| [AddComboBox](../MiniToolbarControls/MiniToolbarControls_AddComboBox.md) | Method that creates a new combobox control within the MiniToolbar. |
| [AddDropdown](../MiniToolbarControls/MiniToolbarControls_AddDropdown.md) | Method that creates a new dropdown control within the MiniToolbar. |
| [AddLabel](../MiniToolbarControls/MiniToolbarControls_AddLabel.md) | Method that creates a new label control within the MiniToolbar. |
| [AddNewLine](../MiniToolbarControls/MiniToolbarControls_AddNewLine.md) | Method that creates a new line within the MiniToolbar. |
| [AddSeparator](../MiniToolbarControls/MiniToolbarControls_AddSeparator.md) | Method that creates a new separator within the MiniToolbar. |
| [AddSlider](../MiniToolbarControls/MiniToolbarControls_AddSlider.md) | Method that creates a new slider control within the MiniToolbar. |
| [AddValueEditor](../MiniToolbarControls/MiniToolbarControls_AddValueEditor.md) | Method that creates a new value edit control within the MiniToolbar. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MiniToolbarControls/MiniToolbarControls_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../MiniToolbarControls/MiniToolbarControls_Count.md) | Gets the number of items in this collection. |
| [Item](../MiniToolbarControls/MiniToolbarControls_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../MiniToolbarControls/MiniToolbarControls_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[MiniToolbar.Controls](../MiniToolbar/MiniToolbar_Controls.md)

## Version

Introduced in version 2012
