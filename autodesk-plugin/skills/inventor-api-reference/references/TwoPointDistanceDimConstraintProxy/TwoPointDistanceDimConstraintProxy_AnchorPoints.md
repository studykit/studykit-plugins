# TwoPointDistanceDimConstraintProxy.AnchorPoints Property

Parent Object: [TwoPointDistanceDimConstraintProxy](../TwoPointDistanceDimConstraintProxy/TwoPointDistanceDimConstraintProxy.md)

## Description

Gets the anchor points of dimension.

## Syntax

TwoPointDistanceDimConstraintProxy.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version
