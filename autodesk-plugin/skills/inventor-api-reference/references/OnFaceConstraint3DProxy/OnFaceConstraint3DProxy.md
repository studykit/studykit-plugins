# OnFaceConstraint3DProxy Object

Derived from: [OnFaceConstraint3D](../OnFaceConstraint3D/OnFaceConstraint3D.md) Object

## Description

OnFaceConstraint3DProxy Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_Delete.md) | Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted. |
| [GetReferenceKey](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_Deletable.md) | Indicates whether this object can be deleted. |
| [Entity](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_Entity.md) | Read-only property that returns the 3D sketch entity being constrained. |
| [Face](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_Face.md) | Read-only property that returns the Face which the 3D sketch entity is constrained to. |
| [NativeObject](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Type](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
