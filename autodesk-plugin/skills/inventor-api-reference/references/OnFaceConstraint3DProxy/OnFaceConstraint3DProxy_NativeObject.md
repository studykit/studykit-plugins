# OnFaceConstraint3DProxy.NativeObject Property

Parent Object: [OnFaceConstraint3DProxy](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

OnFaceConstraint3DProxy.**NativeObject**() As [OnFaceConstraint3D](../OnFaceConstraint3D/OnFaceConstraint3D.md)

## Property Value

This is a read only property whose value is an [OnFaceConstraint3D](../OnFaceConstraint3D/OnFaceConstraint3D.md).

## Version

Introduced in version 2017
