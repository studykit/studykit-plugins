# HelicalCurveProxy.Definition Property

Parent Object: [HelicalCurveProxy](../HelicalCurveProxy/HelicalCurveProxy.md)

## Description

Gets and sets the shape definition of the helical curve.

## Syntax

HelicalCurveProxy.**Definition**() As [HelicalCurveDefinition](../HelicalCurveDefinition/HelicalCurveDefinition.md)

## Property Value

This is a read/write property whose value is a [HelicalCurveDefinition](../HelicalCurveDefinition/HelicalCurveDefinition.md).

## Version

Introduced in version 2019
