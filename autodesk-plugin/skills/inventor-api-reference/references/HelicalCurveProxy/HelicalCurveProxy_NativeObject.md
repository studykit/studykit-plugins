# HelicalCurveProxy.NativeObject Property

Parent Object: [HelicalCurveProxy](../HelicalCurveProxy/HelicalCurveProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

HelicalCurveProxy.**NativeObject**() As [HelicalCurve](../HelicalCurve/HelicalCurve.md)

## Property Value

This is a read only property whose value is a [HelicalCurve](../HelicalCurve/HelicalCurve.md).

## Version

Introduced in version 2019
