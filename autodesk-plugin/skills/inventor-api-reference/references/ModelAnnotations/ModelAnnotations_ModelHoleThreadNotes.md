# ModelAnnotations.ModelHoleThreadNotes Property

Parent Object: [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md)

## Description

Read-only property that returns the ModelHoleThreadNotes collection object. This object provides access to all of the model dimensions in the part or assembly.

## Syntax

ModelAnnotations.**ModelHoleThreadNotes**() As [ModelHoleThreadNotes](../ModelHoleThreadNotes/ModelHoleThreadNotes.md)

## Property Value

This is a read only property whose value is a [ModelHoleThreadNotes](../ModelHoleThreadNotes/ModelHoleThreadNotes.md).

## Version

Introduced in version 2018
