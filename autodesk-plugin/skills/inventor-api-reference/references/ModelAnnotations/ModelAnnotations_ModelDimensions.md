# ModelAnnotations.ModelDimensions Property

Parent Object: [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md)

## Description

Read-only property that returns the ModelDimensions collection object. This object provides access to all of the model dimensions in the part or assembly.

## Syntax

ModelAnnotations.**ModelDimensions**() As [ModelDimensions](../ModelDimensions/ModelDimensions.md)

## Property Value

This is a read only property whose value is a [ModelDimensions](../ModelDimensions/ModelDimensions.md).

## Version

Introduced in version 2018
