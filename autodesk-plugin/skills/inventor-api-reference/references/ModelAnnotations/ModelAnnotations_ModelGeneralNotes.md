# ModelAnnotations.ModelGeneralNotes Property

Parent Object: [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md)

## Description

Returns ModelGeneralNotes collection object.

## Syntax

ModelAnnotations.**ModelGeneralNotes**() As [ModelGeneralNotes](../ModelGeneralNotes/ModelGeneralNotes.md)

## Property Value

This is a read only property whose value is a [ModelGeneralNotes](../ModelGeneralNotes/ModelGeneralNotes.md).

## Version
