# Profiles3D.AddClosed Method

Parent Object: [Profiles3D](../Profiles3D/Profiles3D.md)

## Description

Method that creates a new profile by examining the contents of the sketch and creating as many closed paths as possible. The resulting Profile3D is returned.

## Syntax

Profiles3D.**AddClosed**() As [Profile3D](../Profile3D/Profile3D.md)

## Version

Introduced in version 6
