# RadiusModelDimensionDefinition.Copy Method

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Method that creates a copy of this RadiusModelDimensionDefinition object. The new RadiusModelDimensionDefinition object is independent of any dimension. It can edited and used as input to edit an existing dimension or to create a new dimension.

## Syntax

RadiusModelDimensionDefinition.**Copy**() As [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Version

Introduced in version 2018
