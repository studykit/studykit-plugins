# RadiusModelDimensionDefinition.AnnotationPlane Property

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Read-write property that gets and sets the annotation plane for the dimension. Will return nothing in the case where the ModelDimensionDefinition object is transient and not associated with a dimension.

## Syntax

RadiusModelDimensionDefinition.**AnnotationPlane**() As [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md).

## Version

Introduced in version 2018
