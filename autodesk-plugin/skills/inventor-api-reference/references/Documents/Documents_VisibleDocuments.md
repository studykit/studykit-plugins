# Documents.VisibleDocuments Property

Parent Object: [Documents](../Documents/Documents.md)

## Description

Gets the collection of visible documents.

## Syntax

Documents.**VisibleDocuments**() As [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md).

## Version

Introduced in version 2009
