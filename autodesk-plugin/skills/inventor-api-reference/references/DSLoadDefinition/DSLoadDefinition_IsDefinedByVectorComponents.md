# DSLoadDefinition.IsDefinedByVectorComponents Property

Parent Object: [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Description

Gets and sets if the magnitude and direction of this load is defined by vector components or a specified magnitude and direction entity.

## Syntax

DSLoadDefinition.**IsDefinedByVectorComponents**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
