# DSLoadDefinition.VectorXComponent Property

Parent Object: [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Description

Gets the DSValue object that defines the X component of the vector. This property returns Nothing in the case where the IsDefinedByVectorComponents is False. To change the definition of the load to be defined by a vector use the SetByVector method.

## Syntax

DSLoadDefinition.**VectorXComponent**() As [DSValue](../DSValue/DSValue.md)

## Property Value

This is a read only property whose value is a [DSValue](../DSValue/DSValue.md).

## Version

Introduced in version 2013
