# DSLoadDefinition.Vector Property

Parent Object: [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Description

Gets a Vector indicating the vector of the load. This can be used to get the load vector regardless of whether it has been defined using geometry or the x, y, z, components.

## Syntax

DSLoadDefinition.**Vector**() As [Vector](../Vector/Vector.md)

## Property Value

This is a read only property whose value is a [Vector](../Vector/Vector.md).

## Version

Introduced in version 2013
