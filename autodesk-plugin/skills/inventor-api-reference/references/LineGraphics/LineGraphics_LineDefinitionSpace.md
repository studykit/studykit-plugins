# LineGraphics.LineDefinitionSpace Property

Parent Object: [LineGraphics](../LineGraphics/LineGraphics.md)

## Description

Gets and sets the LineDefinitionSpace applied to this line graphics.

## Syntax

LineGraphics.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2008
