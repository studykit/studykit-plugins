# LineGraphics.ColorIndexSet Property

Parent Object: [LineGraphics](../LineGraphics/LineGraphics.md)

## Description

Gets and sets the GraphicsIndexSet that defines the indices within the GraphicsColorSet to use.

## Syntax

LineGraphics.**ColorIndexSet**() As [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md).

## Version

Introduced in version 5
