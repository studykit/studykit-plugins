# LineGraphics.ColorSet Property

Parent Object: [LineGraphics](../LineGraphics/LineGraphics.md)

## Description

Gets and sets the GraphicsColorSet associated with the set.

## Syntax

LineGraphics.**ColorSet**() As [GraphicsColorSet](../GraphicsColorSet/GraphicsColorSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsColorSet](../GraphicsColorSet/GraphicsColorSet.md).

## Version

Introduced in version 5
