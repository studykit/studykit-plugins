# SplineLengthDimConstraint3D.Spline Property

Parent Object: [SplineLengthDimConstraint3D](../SplineLengthDimConstraint3D/SplineLengthDimConstraint3D.md)

## Description

Returns the 3D sketch spline being constrained.

## Syntax

SplineLengthDimConstraint3D.**Spline**() As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md).

## Version

Introduced in version 2019
