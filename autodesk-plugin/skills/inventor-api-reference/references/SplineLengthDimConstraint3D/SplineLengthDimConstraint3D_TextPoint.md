# SplineLengthDimConstraint3D.TextPoint Property

Parent Object: [SplineLengthDimConstraint3D](../SplineLengthDimConstraint3D/SplineLengthDimConstraint3D.md)

## Description

Gets and sets the position of the dimension text.

## Syntax

SplineLengthDimConstraint3D.**TextPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2019
