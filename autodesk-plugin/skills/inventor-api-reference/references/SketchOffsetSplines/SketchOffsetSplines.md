# SketchOffsetSplines Object

## Description

The SketchOffsetSplines object provides access to all the objects in a sketch. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchOffsetSplines/SketchOffsetSplines_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchOffsetSplines/SketchOffsetSplines_Count.md) | Property that specifies the number of items in the collection. |
| [Item](../SketchOffsetSplines/SketchOffsetSplines_Item.md) | Method that returns the specified SketchOffsetSpline object from the collection. |
| [Type](../SketchOffsetSplines/SketchOffsetSplines_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
