# PresentationMeshFeatureSetsEnumerator.Item Property

Parent Object: [PresentationMeshFeatureSetsEnumerator](../PresentationMeshFeatureSetsEnumerator/PresentationMeshFeatureSetsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationMeshFeatureSetsEnumerator.**Item**( ***Index*** As Long ) As [PresentationMeshFeatureSet](../PresentationMeshFeatureSet/PresentationMeshFeatureSet.md)

## Property Value

This is a read only property whose value is a [PresentationMeshFeatureSet](../PresentationMeshFeatureSet/PresentationMeshFeatureSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Specifies the Long value indicating the index of the object to return. |

## Version

Introduced in version 2018
