# TextureCoordinateSet.GetCoordinates Method

Parent Object: [TextureCoordinateSet](../TextureCoordinateSet/TextureCoordinateSet.md)

## Description

Method to obtain the coordinates to this set.

## Syntax

TextureCoordinateSet.**GetCoordinates**( ***Coordinates***() As Double )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Coordinates | Double | The coordinates to obtain. |

## Version

Introduced in version 10
