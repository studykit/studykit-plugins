# PresentationVertex.Faces Property

Parent Object: [PresentationVertex](../PresentationVertex/PresentationVertex.md)

## Description

Read-only property that returns the PresentationFace objects that are adjacent to this vertex.

## Syntax

PresentationVertex.**Faces**() As [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md).

## Version

Introduced in version 2018
