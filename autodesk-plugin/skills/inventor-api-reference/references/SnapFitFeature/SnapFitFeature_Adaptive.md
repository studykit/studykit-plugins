# SnapFitFeature.Adaptive Property

Parent Object: [SnapFitFeature](../SnapFitFeature/SnapFitFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

SnapFitFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
