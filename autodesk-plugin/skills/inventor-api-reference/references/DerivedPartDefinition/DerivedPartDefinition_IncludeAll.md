# DerivedPartDefinition.IncludeAll Method

Parent Object: [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Description

Method that causes all entities within the derived part component to be set to kDerivedIncludeAll.

## Syntax

DerivedPartDefinition.**IncludeAll**()

## Version

Introduced in version 5.3
