# DerivedPartDefinition.iMateDefinitions Property

Parent Object: [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Description

Property that gets the collection of iMates available in the selected part along with the inclusion option for each iMate.

## Syntax

DerivedPartDefinition.**iMateDefinitions**() As [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md)

## Property Value

This is a read only property whose value is a [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md).

## Version

Introduced in version 2009
