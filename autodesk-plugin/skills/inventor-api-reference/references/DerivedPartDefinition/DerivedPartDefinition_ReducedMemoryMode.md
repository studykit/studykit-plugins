# DerivedPartDefinition.ReducedMemoryMode Property

Parent Object: [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Description

Read-write property that gets and sets the Reduced Memory Mode.

## Syntax

DerivedPartDefinition.**ReducedMemoryMode**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
