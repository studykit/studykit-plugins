# PointCloudScan.SourceFullFileName Property

Parent Object: [PointCloudScan](../PointCloudScan/PointCloudScan.md)

## Description

Read-only property that returns the point cloud scan full file name.

## Syntax

PointCloudScan.**SourceFullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
