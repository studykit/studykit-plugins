# PointCloudScan Object

## Description

The PointCloudScan object represents a scan within a point cloud.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointCloudScan/PointCloudScan_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [DisplayName](../PointCloudScan/PointCloudScan_DisplayName.md) | Read-only property that returns the display name of the point cloud scan. |
| [InternalName](../PointCloudScan/PointCloudScan_InternalName.md) | Read-only property that returns internal name of the point cloud scan. |
| [Origin](../PointCloudScan/PointCloudScan_Origin.md) | Read-only property that returns a point indicating the origin of the PointCloudScan. |
| [Parent](../PointCloudScan/PointCloudScan_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [SourceFullFileName](../PointCloudScan/PointCloudScan_SourceFullFileName.md) | Read-only property that returns the point cloud scan full file name. |
| [Type](../PointCloudScan/PointCloudScan_Type.md) | Gets the constant that indicates the type of this object. |
| [Visible](../PointCloudScan/PointCloudScan_Visible.md) | Read-write property that gets and set the visibility of the point cloud scan object. |

## Accessed From

[PointCloudScans.Item](../PointCloudScans/PointCloudScans_Item.md)

## Version

Introduced in version 2016
