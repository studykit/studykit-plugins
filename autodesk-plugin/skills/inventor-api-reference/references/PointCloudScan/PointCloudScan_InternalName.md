# PointCloudScan.InternalName Property

Parent Object: [PointCloudScan](../PointCloudScan/PointCloudScan.md)

## Description

Read-only property that returns internal name of the point cloud scan.

## Syntax

PointCloudScan.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
