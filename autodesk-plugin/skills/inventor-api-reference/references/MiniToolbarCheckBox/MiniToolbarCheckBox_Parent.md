# MiniToolbarCheckBox.Parent Property

Parent Object: [MiniToolbarCheckBox](../MiniToolbarCheckBox/MiniToolbarCheckBox.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

MiniToolbarCheckBox.**Parent**() As [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Property Value

This is a read only property whose value is a [MiniToolbar](../MiniToolbar/MiniToolbar.md).

## Version

Introduced in version 2012
