# MiniToolbarCheckBox.LargeIcon Property

Parent Object: [MiniToolbarCheckBox](../MiniToolbarCheckBox/MiniToolbarCheckBox.md)

## Description

Read-write property that gets and sets the large icon for the control.

## Syntax

MiniToolbarCheckBox.**LargeIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2012
