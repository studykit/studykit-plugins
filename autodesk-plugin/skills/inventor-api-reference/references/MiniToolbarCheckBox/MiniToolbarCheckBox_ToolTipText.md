# MiniToolbarCheckBox.ToolTipText Property

Parent Object: [MiniToolbarCheckBox](../MiniToolbarCheckBox/MiniToolbarCheckBox.md)

## Description

Read-write property that gets and sets the tooltip text for the control.

## Syntax

MiniToolbarCheckBox.**ToolTipText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
