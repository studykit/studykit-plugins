# MiniToolbarCheckBox.OnClick Event

Parent Object: [MiniToolbarCheckBox](../MiniToolbarCheckBox/MiniToolbarCheckBox.md)

## Description

Event that is fired when the check box is toggled by the user.

## Syntax

MiniToolbarCheckBox.**OnClick**()

## Version

Introduced in version 2012
