# MiniToolbarCheckBox.Checked Property

Parent Object: [MiniToolbarCheckBox](../MiniToolbarCheckBox/MiniToolbarCheckBox.md)

## Description

Read-write property that gets and sets whether the check box is currently checked.

## Syntax

MiniToolbarCheckBox.**Checked**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
