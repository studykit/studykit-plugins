# iFeatureTable.iFeatureTableColumns Property

Parent Object: [iFeatureTable](../iFeatureTable/iFeatureTable.md)

## Description

Property returning the collection of columns in the table.

## Syntax

iFeatureTable.**iFeatureTableColumns**() As [iFeatureTableColumns](../iFeatureTableColumns/iFeatureTableColumns.md)

## Property Value

This is a read only property whose value is an [iFeatureTableColumns](../iFeatureTableColumns/iFeatureTableColumns.md).
