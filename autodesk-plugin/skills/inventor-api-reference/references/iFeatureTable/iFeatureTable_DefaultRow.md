# iFeatureTable.DefaultRow Property

Parent Object: [iFeatureTable](../iFeatureTable/iFeatureTable.md)

## Description

Gets and sets the default row.

## Syntax

iFeatureTable.**DefaultRow**() As [iFeatureTableRow](../iFeatureTableRow/iFeatureTableRow.md)

## Property Value

This is a read/write property whose value is an [iFeatureTableRow](../iFeatureTableRow/iFeatureTableRow.md).

## Version

Introduced in version 2009
