# iFeatureTable.iFeatureTableRows Property

Parent Object: [iFeatureTable](../iFeatureTable/iFeatureTable.md)

## Description

Property returning the collection of rows in the table.

## Syntax

iFeatureTable.**iFeatureTableRows**() As [iFeatureTableRows](../iFeatureTableRows/iFeatureTableRows.md)

## Property Value

This is a read only property whose value is an [iFeatureTableRows](../iFeatureTableRows/iFeatureTableRows.md).

## Version

Introduced in version 2009
