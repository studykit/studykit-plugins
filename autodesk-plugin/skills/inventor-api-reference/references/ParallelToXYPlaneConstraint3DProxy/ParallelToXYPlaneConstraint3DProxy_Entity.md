# ParallelToXYPlaneConstraint3DProxy.Entity Property

Parent Object: [ParallelToXYPlaneConstraint3DProxy](../ParallelToXYPlaneConstraint3DProxy/ParallelToXYPlaneConstraint3DProxy.md)

## Description

Read-only property that returns the 3D sketch entity being constrained.

## Syntax

ParallelToXYPlaneConstraint3DProxy.**Entity**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 2017
