# PointCloudRegion Object

## Description

The PointCloudRegion object represents a region within a point cloud.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointCloudRegion/PointCloudRegion_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Color](../PointCloudRegion/PointCloudRegion_Color.md) | Read-only property that returns internal name of the point cloud region. |
| [DisplayName](../PointCloudRegion/PointCloudRegion_DisplayName.md) | Read-only property that returns the display name of the point cloud region. |
| [InternalName](../PointCloudRegion/PointCloudRegion_InternalName.md) | Read-only property that returns internal name of the point cloud region. |
| [Parent](../PointCloudRegion/PointCloudRegion_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Type](../PointCloudRegion/PointCloudRegion_Type.md) | Gets the constant that indicates the type of this object. |
| [Visible](../PointCloudRegion/PointCloudRegion_Visible.md) | Read-write property that gets and sets the visibility of the point cloud region object. |

## Accessed From

[PointCloudRegions.Item](../PointCloudRegions/PointCloudRegions_Item.md)

## Version

Introduced in version 2016
