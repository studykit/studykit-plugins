# PointCloudRegion.Color Property

Parent Object: [PointCloudRegion](../PointCloudRegion/PointCloudRegion.md)

## Description

Read-only property that returns internal name of the point cloud region.

## Syntax

PointCloudRegion.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2016
