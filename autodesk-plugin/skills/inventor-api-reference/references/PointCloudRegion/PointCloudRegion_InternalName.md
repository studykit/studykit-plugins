# PointCloudRegion.InternalName Property

Parent Object: [PointCloudRegion](../PointCloudRegion/PointCloudRegion.md)

## Description

Read-only property that returns internal name of the point cloud region.

## Syntax

PointCloudRegion.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
