# EndOfFeatures.Type Property

Parent Object: [EndOfFeatures](../EndOfFeatures/EndOfFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

EndOfFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
