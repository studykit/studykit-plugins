# EndOfFeatures Object

## Description

This object represents the End of Part marker in the browser. When traversing through the browser hierarchy, this is one of the nodes that is encountered, indicating the position of the marker in the browser.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../EndOfFeatures/EndOfFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../EndOfFeatures/EndOfFeatures_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../EndOfFeatures/EndOfFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 10
