# FixedWorkPlaneDef.Parent Property

Parent Object: [FixedWorkPlaneDef](../FixedWorkPlaneDef/FixedWorkPlaneDef.md)

## Description

Property that returns the parent WorkPlane object.

## Syntax

FixedWorkPlaneDef.**Parent**() As [WorkPlane](../WorkPlane/WorkPlane.md)

## Property Value

This is a read only property whose value is a [WorkPlane](../WorkPlane/WorkPlane.md).

## Version

Introduced in version 4
