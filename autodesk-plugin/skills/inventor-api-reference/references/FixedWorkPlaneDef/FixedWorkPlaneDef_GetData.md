# FixedWorkPlaneDef.GetData Method

Parent Object: [FixedWorkPlaneDef](../FixedWorkPlaneDef/FixedWorkPlaneDef.md)

## Description

Method that gets all of the data defining a fixed work plane.

## Syntax

FixedWorkPlaneDef.**GetData**( ***OriginPoint*** As [Point](../Point/Point.md), ***XAxis*** As [UnitVector](../UnitVector/UnitVector.md), ***YAxis*** As [UnitVector](../UnitVector/UnitVector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| OriginPoint | [Point](../Point/Point.md) | Output object that defines the origin of the work plane. |
| XAxis | [UnitVector](../UnitVector/UnitVector.md) | Output object that defines the X-axis vector of the work plane. |
| YAxis | [UnitVector](../UnitVector/UnitVector.md) | Output object that defines the Y-axis vector of the work plane. |

## Version

Introduced in version 4
