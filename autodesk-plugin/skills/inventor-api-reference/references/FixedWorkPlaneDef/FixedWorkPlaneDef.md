# FixedWorkPlaneDef Object

## Description

Object that allows you to get and set the information that specifies a fixed work plane.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../FixedWorkPlaneDef/FixedWorkPlaneDef_GetData.md) | Method that gets all of the data defining a fixed work plane. |
| [PutData](../FixedWorkPlaneDef/FixedWorkPlaneDef_PutData.md) | Method that redefines the origin point, x, and y axes of the work plane. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FixedWorkPlaneDef/FixedWorkPlaneDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [OriginPoint](../FixedWorkPlaneDef/FixedWorkPlaneDef_OriginPoint.md) | Property that returns the origin point of the work plane. |
| [Parent](../FixedWorkPlaneDef/FixedWorkPlaneDef_Parent.md) | Property that returns the parent WorkPlane object. |
| [Type](../FixedWorkPlaneDef/FixedWorkPlaneDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [XAxis](../FixedWorkPlaneDef/FixedWorkPlaneDef_XAxis.md) | Property that returns the X-axis vector of the work plane. |
| [YAxis](../FixedWorkPlaneDef/FixedWorkPlaneDef_YAxis.md) | Property that returns the Y-axis vector of the work plane. |

## Version

Introduced in version 4
