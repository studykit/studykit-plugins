# FaceProxy.Evaluator Property

Parent Object: [FaceProxy](../FaceProxy/FaceProxy.md)

## Description

Gets the surface evaluator for this face.

## Syntax

FaceProxy.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
