# FaceProxy.TextureMaps Property

Parent Object: [FaceProxy](../FaceProxy/FaceProxy.md)

## Description

Gets the texture maps associated with this face.

## Syntax

FaceProxy.**TextureMaps**() As [TextureMaps](../TextureMaps/TextureMaps.md)

## Property Value

This is a read only property whose value is a [TextureMaps](../TextureMaps/TextureMaps.md).

## Version

Introduced in version 10
