# FaceProxy.FaceShell Property

Parent Object: [FaceProxy](../FaceProxy/FaceProxy.md)

## Description

Property that returns the FaceShell object.

## Syntax

FaceProxy.**FaceShell**() As [FaceShell](../FaceShell/FaceShell.md)

## Property Value

This is a read only property whose value is a [FaceShell](../FaceShell/FaceShell.md).

## Version

Introduced in version 4
