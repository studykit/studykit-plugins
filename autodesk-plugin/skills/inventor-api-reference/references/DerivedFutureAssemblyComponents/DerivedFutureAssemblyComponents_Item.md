# DerivedFutureAssemblyComponents.Item Property

Parent Object: [DerivedFutureAssemblyComponents](../DerivedFutureAssemblyComponents/DerivedFutureAssemblyComponents.md)

## Description

Method that returns the specified DerivedFutureAssemblyComponent object from the collection.

## Syntax

DerivedFutureAssemblyComponents.**Item**( ***Value*** As Long ) As [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md)

## Property Value

This is a read only property whose value is a [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | Long | Input Variant value that specifies the DerivedFutureAssemblyComponent to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the derived component’s name. If an out of range index or a name of a non-existent DerivedFutureAssemblyComponent is provided, an error occurs. |

## Version

Introduced in version 2018
