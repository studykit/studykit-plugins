# DerivedFutureAssemblyComponents.Add Method

Parent Object: [DerivedFutureAssemblyComponents](../DerivedFutureAssemblyComponents/DerivedFutureAssemblyComponents.md)

## Description

Method that creates a new DerivedFutureAssemblyComponent using the information supplied by the input DerivedFutureAssemblyDefinition object. If successful, the new DerivedFutureAssemblyComponent is returned.

## Syntax

DerivedFutureAssemblyComponents.**Add**( ***Definition*** As [DerivedFutureAssemblyDefinition](../DerivedFutureAssemblyDefinition/DerivedFutureAssemblyDefinition.md) ) As [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [DerivedFutureAssemblyDefinition](../DerivedFutureAssemblyDefinition/DerivedFutureAssemblyDefinition.md) | Input DerivedFutureAssemblyDefinition object that defines the derived component. |

## Version

Introduced in version 2018
