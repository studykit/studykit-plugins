# SketchOptions.ImageInsertionLINKOptionCheckedAsDefault Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Gets/sets the default option for LINK when attaching image.

## Syntax

SketchOptions.**ImageInsertionLINKOptionCheckedAsDefault**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
