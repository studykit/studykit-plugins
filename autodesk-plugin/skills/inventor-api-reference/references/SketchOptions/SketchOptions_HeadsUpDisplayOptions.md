# SketchOptions.HeadsUpDisplayOptions Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Property that returns the HeadsUpDisplayOptions object. The HeadsUpDisplayOptions object provides access to various heads-up display related application level options for sketches.

## Syntax

SketchOptions.**HeadsUpDisplayOptions**() As [HeadsUpDisplayOptions](../HeadsUpDisplayOptions/HeadsUpDisplayOptions.md)

## Property Value

This is a read only property whose value is a [HeadsUpDisplayOptions](../HeadsUpDisplayOptions/HeadsUpDisplayOptions.md).

## Version

Introduced in version 2011
