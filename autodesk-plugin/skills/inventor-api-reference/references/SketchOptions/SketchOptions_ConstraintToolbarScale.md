# SketchOptions.ConstraintToolbarScale Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Gets/sets the scale for the size of the constraint toolbars with respect to the graphics window.

## Syntax

SketchOptions.**ConstraintToolbarScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
