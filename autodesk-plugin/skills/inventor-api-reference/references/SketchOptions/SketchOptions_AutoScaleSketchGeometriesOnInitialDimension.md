# SketchOptions.AutoScaleSketchGeometriesOnInitialDimension Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Enables/disables whether to auto scale sketch geometries on initial dimension.

## Syntax

SketchOptions.**AutoScaleSketchGeometriesOnInitialDimension**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
