# SketchOptions.AutomaticReferenceEdges Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Gets/sets the capability to automatically project edges of the selected face onto the sketch plane as reference geometry, when creating a new sketch.

## Syntax

SketchOptions.**AutomaticReferenceEdges**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
