# SketchOptions.SplineFitMethod Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Gets/sets the default fit method for splines in 2d sketches.

## Syntax

SketchOptions.**SplineFitMethod**() As [SplineFitMethodEnum](../SplineFitMethodEnum.md)

## Property Value

This is a read/write property whose value is a [SplineFitMethodEnum](../SplineFitMethodEnum.md).

## Version

Introduced in version 2010
