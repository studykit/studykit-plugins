# SketchOptions.DisplayMinorGridLines Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Enables/disables the display of minor sketch plane grid lines.

## Syntax

SketchOptions.**DisplayMinorGridLines**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
