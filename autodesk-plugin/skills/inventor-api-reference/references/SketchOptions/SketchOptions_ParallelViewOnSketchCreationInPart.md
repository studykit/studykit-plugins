# SketchOptions.ParallelViewOnSketchCreationInPart Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Enables/disables whether the view is automatically reoriented to be planar to the screen in part environment.

## Syntax

SketchOptions.**ParallelViewOnSketchCreationInPart**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
