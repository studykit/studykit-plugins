# ModelingSettings.MaintainEnhancedGraphicsDetail Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets whether to save enhanced graphics mesh data with the document.

## Syntax

ModelingSettings.**MaintainEnhancedGraphicsDetail**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
