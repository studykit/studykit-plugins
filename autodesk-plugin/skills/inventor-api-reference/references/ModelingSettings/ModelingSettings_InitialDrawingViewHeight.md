# ModelingSettings.InitialDrawingViewHeight Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets the initial view height (in centimeters) to use when creating a drawing view from this model.

## Syntax

ModelingSettings.**InitialDrawingViewHeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
