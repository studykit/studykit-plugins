# ModelingSettings.UserCoordinateSystemSettings Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Property that returns the UserCoordinateSystemSettings object. The UserCoordinateSystemSettings object provides access to various UCS related document level settings.

## Syntax

ModelingSettings.**UserCoordinateSystemSettings**() As [UserCoordinateSystemSettings](../UserCoordinateSystemSettings/UserCoordinateSystemSettings.md)

## Property Value

This is a read only property whose value is a [UserCoordinateSystemSettings](../UserCoordinateSystemSettings/UserCoordinateSystemSettings.md).

## Version

Introduced in version 2010
