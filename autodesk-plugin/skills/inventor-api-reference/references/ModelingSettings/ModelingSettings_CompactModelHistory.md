# ModelingSettings.CompactModelHistory Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets whether the rollback history should be purged when a document is saved.

## Syntax

ModelingSettings.**CompactModelHistory**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 8
