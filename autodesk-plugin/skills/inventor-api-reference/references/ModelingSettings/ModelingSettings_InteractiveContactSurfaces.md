# ModelingSettings.InteractiveContactSurfaces Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets the type of surfaces to use for interactive contact analysis.

## Syntax

ModelingSettings.**InteractiveContactSurfaces**() As [InteractiveContactSurfacesEnum](../InteractiveContactSurfacesEnum.md)

## Property Value

This is a read/write property whose value is an [InteractiveContactSurfacesEnum](../InteractiveContactSurfacesEnum.md).

## Version

Introduced in version 2008
