# ModelingSettings.AutoFindErrorsAfterManualRepair Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets whether to find errors automatically after manual repair.

## Syntax

ModelingSettings.**AutoFindErrorsAfterManualRepair**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2021
