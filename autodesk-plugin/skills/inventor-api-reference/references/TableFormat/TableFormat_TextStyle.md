# TableFormat.TextStyle Property

Parent Object: [TableFormat](../TableFormat/TableFormat.md)

## Description

Gets and sets the text style.

## Syntax

TableFormat.**TextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 9
