# TableFormat Object

## Description

The TableFormat object contains values for the various overrides of the table's appearance.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TableFormat/TableFormat_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [InsideLineColor](../TableFormat/TableFormat_InsideLineColor.md) | Gets and sets the color of the inside lines. |
| [InsideLineWeight](../TableFormat/TableFormat_InsideLineWeight.md) | Gets and sets the weight of the inside lines. |
| [OutsideLineColor](../TableFormat/TableFormat_OutsideLineColor.md) | Gets and sets the color of the outside lines. |
| [OutsideLineWeight](../TableFormat/TableFormat_OutsideLineWeight.md) | Gets and sets the weight of the outside lines. |
| [TextStyle](../TableFormat/TableFormat_TextStyle.md) | Gets and sets the text style. |
| [Type](../TableFormat/TableFormat_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
