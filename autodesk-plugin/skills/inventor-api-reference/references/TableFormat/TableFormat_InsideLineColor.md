# TableFormat.InsideLineColor Property

Parent Object: [TableFormat](../TableFormat/TableFormat.md)

## Description

Gets and sets the color of the inside lines.

## Syntax

TableFormat.**InsideLineColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).
