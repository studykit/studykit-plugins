# BSplineCurveDefinition.Type Property

Parent Object: [BSplineCurveDefinition](../BSplineCurveDefinition/BSplineCurveDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

BSplineCurveDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
