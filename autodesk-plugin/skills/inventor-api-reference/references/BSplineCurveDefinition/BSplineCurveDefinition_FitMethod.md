# BSplineCurveDefinition.FitMethod Property

Parent Object: [BSplineCurveDefinition](../BSplineCurveDefinition/BSplineCurveDefinition.md)

## Description

Gets and sets the FitMethod setting.

## Syntax

BSplineCurveDefinition.**FitMethod**() As [SplineFitMethodEnum](../SplineFitMethodEnum.md)

## Property Value

This is a read/write property whose value is a [SplineFitMethodEnum](../SplineFitMethodEnum.md).

## Version

Introduced in version 9
