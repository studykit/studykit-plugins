# BSplineCurveDefinition.FixedLength Property

Parent Object: [BSplineCurveDefinition](../BSplineCurveDefinition/BSplineCurveDefinition.md)

## Description

Gets and sets whether the curve should have a fixed length.

## Syntax

BSplineCurveDefinition.**FixedLength**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
