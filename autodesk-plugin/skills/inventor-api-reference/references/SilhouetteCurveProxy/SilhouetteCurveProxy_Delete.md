# SilhouetteCurveProxy.Delete Method

Parent Object: [SilhouetteCurveProxy](../SilhouetteCurveProxy/SilhouetteCurveProxy.md)

## Description

Method that deletes the silhouette curve. This will delete all of the related sketch entities.

## Syntax

SilhouetteCurveProxy.**Delete**()

## Version

Introduced in version 2012
