# SilhouetteCurveProxy.SketchEntities Property

Parent Object: [SilhouetteCurveProxy](../SilhouetteCurveProxy/SilhouetteCurveProxy.md)

## Description

Read-only property that returns a collection of sketch entities that belong to silhouette curve.

## Syntax

SilhouetteCurveProxy.**SketchEntities**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 2012
