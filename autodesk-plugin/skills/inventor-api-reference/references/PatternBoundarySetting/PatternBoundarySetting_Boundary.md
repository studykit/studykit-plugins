# PatternBoundarySetting.Boundary Property

Parent Object: [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Description

Read-write property that gets and sets the boundary object for the pattern. This can be a FaceCollection object containing planar Face objects or a Profile object.

## Syntax

PatternBoundarySetting.**Boundary**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2025
