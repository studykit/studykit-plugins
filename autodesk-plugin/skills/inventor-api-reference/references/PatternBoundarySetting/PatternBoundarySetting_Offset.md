# PatternBoundarySetting.Offset Property

Parent Object: [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Description

Read-write property that gets and sets the offset value for the pattern.

## Syntax

PatternBoundarySetting.**Offset**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2025
