# PatternBoundarySetting.OccurrenceBasePoint Property

Parent Object: [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Description

Read-write property that gets and sets the occurrence base point. This can be a WorkPoint, Vertex or GeometryIntent indicating the mid point of an edge or center of circular/elliptical edge.

## Syntax

PatternBoundarySetting.**OccurrenceBasePoint**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2025
