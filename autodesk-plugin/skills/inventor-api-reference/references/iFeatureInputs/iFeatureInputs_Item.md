# iFeatureInputs.Item Property

Parent Object: [iFeatureInputs](../iFeatureInputs/iFeatureInputs.md)

## Description

Returns the specified object in the collection.

## Syntax

iFeatureInputs.**Item**( ***Index*** As Long ) As [iFeatureInput](../iFeatureInput/iFeatureInput.md)

## Property Value

This is a read only property whose value is an [iFeatureInput](../iFeatureInput/iFeatureInput.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 6
