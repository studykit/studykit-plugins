# DrawingOptions.CenterDimensionText Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the dimension text is to be centered when adding dimensions to drawing views.

## Syntax

DrawingOptions.**CenterDimensionText**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
