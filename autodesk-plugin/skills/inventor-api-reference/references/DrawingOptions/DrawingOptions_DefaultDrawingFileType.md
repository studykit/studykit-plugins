# DrawingOptions.DefaultDrawingFileType Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the default file type (.idw or .dwg) to use when creating new drawings.

## Syntax

DrawingOptions.**DefaultDrawingFileType**() As [DefaultDrawingFileTypeEnum](../DefaultDrawingFileTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DefaultDrawingFileTypeEnum](../DefaultDrawingFileTypeEnum.md).

## Version

Introduced in version 2008
