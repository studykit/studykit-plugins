# DrawingOptions.DefaultLayerStyle Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the default style to use for drawing layers.

## Syntax

DrawingOptions.**DefaultLayerStyle**() As [DefaultLayerStyleEnum](../DefaultLayerStyleEnum.md)

## Property Value

This is a read/write property whose value is a [DefaultLayerStyleEnum](../DefaultLayerStyleEnum.md).

## Version

Introduced in version 2008
