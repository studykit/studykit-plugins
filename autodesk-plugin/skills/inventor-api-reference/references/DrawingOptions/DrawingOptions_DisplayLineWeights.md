# DrawingOptions.DisplayLineWeights Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the display of unique line weights in drawings.

## Syntax

DrawingOptions.**DisplayLineWeights**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
