# DrawingOptions.CircleDimensionType Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the type of dimensioning to use while placing circle dimensions on drawing sheets.

## Syntax

DrawingOptions.**CircleDimensionType**() As [CircleDimensionTypeEnum](../CircleDimensionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [CircleDimensionTypeEnum](../CircleDimensionTypeEnum.md).

## Version

Introduced in version 2008
