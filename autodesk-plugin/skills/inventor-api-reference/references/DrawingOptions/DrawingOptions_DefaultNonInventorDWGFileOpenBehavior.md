# DrawingOptions.DefaultNonInventorDWGFileOpenBehavior Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the default file open behavior for non-Inventor DWG files.

## Syntax

DrawingOptions.**DefaultNonInventorDWGFileOpenBehavior**() As [DefaultNonInventorDWGFileOpenBehaviorEnum](../DefaultNonInventorDWGFileOpenBehaviorEnum.md)

## Property Value

This is a read/write property whose value is a [DefaultNonInventorDWGFileOpenBehaviorEnum](../DefaultNonInventorDWGFileOpenBehaviorEnum.md).

## Version

Introduced in version 2008
