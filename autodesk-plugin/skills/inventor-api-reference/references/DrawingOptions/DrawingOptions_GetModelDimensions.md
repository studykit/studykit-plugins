# DrawingOptions.GetModelDimensions Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the default for the dimensions option in the create view dialog boxes.

## Syntax

DrawingOptions.**GetModelDimensions**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
