# DrawingOptions.LinearDimensionType Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the type of dimensioning to use while placing linear dimensions on drawing sheets.

## Syntax

DrawingOptions.**LinearDimensionType**() As [LinearDimensionTypeEnum](../LinearDimensionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LinearDimensionTypeEnum](../LinearDimensionTypeEnum.md).

## Version

Introduced in version 2008
