# DrawingOptions.DefaultObjectStyle Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the default style to use for drawing objects.

## Syntax

DrawingOptions.**DefaultObjectStyle**() As [DefaultObjectStyleEnum](../DefaultObjectStyleEnum.md)

## Property Value

This is a read/write property whose value is a [DefaultObjectStyleEnum](../DefaultObjectStyleEnum.md).

## Version

Introduced in version 2008
