# SmoothConstraintProxy.EntityOne Property

Parent Object: [SmoothConstraintProxy](../SmoothConstraintProxy/SmoothConstraintProxy.md)

## Description

Property that returns the sketch entity that is constrained to be smooth (G2-continuous) to entity two.

## Syntax

SmoothConstraintProxy.**EntityOne**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 11
