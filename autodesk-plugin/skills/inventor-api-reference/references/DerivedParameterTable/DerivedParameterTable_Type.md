# DerivedParameterTable.Type Property

Parent Object: [DerivedParameterTable](../DerivedParameterTable/DerivedParameterTable.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DerivedParameterTable.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
