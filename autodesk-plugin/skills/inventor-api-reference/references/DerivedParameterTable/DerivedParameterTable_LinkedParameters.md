# DerivedParameterTable.LinkedParameters Property

Parent Object: [DerivedParameterTable](../DerivedParameterTable/DerivedParameterTable.md)

## Description

Gets and sets the collection of parameters that have been linked from the referenced document.

## Syntax

DerivedParameterTable.**LinkedParameters**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).
