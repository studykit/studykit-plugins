# BossFeatureProxy.NativeObject Property

Parent Object: [BossFeatureProxy](../BossFeatureProxy/BossFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

BossFeatureProxy.**NativeObject**() As [BossFeature](../BossFeature/BossFeature.md)

## Property Value

This is a read only property whose value is a [BossFeature](../BossFeature/BossFeature.md).

## Version

Introduced in version 2010
