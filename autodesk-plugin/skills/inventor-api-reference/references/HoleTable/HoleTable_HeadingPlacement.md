# HoleTable.HeadingPlacement Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets the placement position of the HoleTable heading.

## Syntax

HoleTable.**HeadingPlacement**() As [HeadingPlacementEnum](../HeadingPlacementEnum.md)

## Property Value

This is a read/write property whose value is a [HeadingPlacementEnum](../HeadingPlacementEnum.md).

## Version

Introduced in version 10
