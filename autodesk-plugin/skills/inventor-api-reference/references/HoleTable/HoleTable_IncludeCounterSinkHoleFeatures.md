# HoleTable.IncludeCounterSinkHoleFeatures Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets whether to include countersunk hole features.

## Syntax

HoleTable.**IncludeCounterSinkHoleFeatures**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
