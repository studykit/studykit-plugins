# HoleTable.SequentialNumbering Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets whether to replace the alphanumeric hole tags with sequential numbering of the holes in the hole table.

## Syntax

HoleTable.**SequentialNumbering**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
