# HoleTable.SecondaryTagModifierOnRollup Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets whether to include a secondary numeric tag when the row merge type is kRollupRowMerge.

## Syntax

HoleTable.**SecondaryTagModifierOnRollup**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
