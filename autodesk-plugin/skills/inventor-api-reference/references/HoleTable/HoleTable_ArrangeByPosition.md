# HoleTable.ArrangeByPosition Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets whether to tags holes relative to their position on the selected view or by their relative size.

## Syntax

HoleTable.**ArrangeByPosition**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
