# HoleTable.Position Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets the position of the HoleTable on the sheet.

## Syntax

HoleTable.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
