# HoleTable.Style Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets the associated HoleTableStyle object.

## Syntax

HoleTable.**Style**() As [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md)

## Property Value

This is a read/write property whose value is a [HoleTableStyle](../HoleTableStyle/HoleTableStyle.md).

## Version

Introduced in version 2009
