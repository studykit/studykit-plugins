# HoleTable.ReformatOnCustomHoleMatch Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets whether to re-index the hole tags and resort contents when a Match Custom Hole operation is done.

## Syntax

HoleTable.**ReformatOnCustomHoleMatch**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
