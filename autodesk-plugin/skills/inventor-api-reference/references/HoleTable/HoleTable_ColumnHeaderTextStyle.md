# HoleTable.ColumnHeaderTextStyle Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets the text style used for the column titles (headers).

## Syntax

HoleTable.**ColumnHeaderTextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2009
