# OriginIndicator.InUse Property

Parent Object: [OriginIndicator](../OriginIndicator/OriginIndicator.md)

## Description

Read-only property that returns whether the origin indicator is used by any dimensions or tables.

## Syntax

OriginIndicator.**InUse**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2024.1
