# OriginIndicator.Layer Property

Parent Object: [OriginIndicator](../OriginIndicator/OriginIndicator.md)

## Description

Gets and sets the layer associated with the origin indicator.

## Syntax

OriginIndicator.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2009
