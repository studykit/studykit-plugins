# ContourFlangeFeature.Definition Property

Parent Object: [ContourFlangeFeature](../ContourFlangeFeature/ContourFlangeFeature.md)

## Description

Read-write property that gets and sets the ContourFlangeDefinition object associated with this contour flange feature.

## Syntax

ContourFlangeFeature.**Definition**() As [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Property Value

This is a read/write property whose value is a [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md).

## Version

Introduced in version 2009
