# ContourFlangeFeature.BendFeature Property

Parent Object: [ContourFlangeFeature](../ContourFlangeFeature/ContourFlangeFeature.md)

## Description

Property that returns the associated bend feature, if there is one.

## Syntax

ContourFlangeFeature.**BendFeature**() As [BendFeature](../BendFeature/BendFeature.md)

## Property Value

This is a read only property whose value is a [BendFeature](../BendFeature/BendFeature.md).

## Version

Introduced in version 2009
