# ContourFlangeFeature.Suppressed Property

Parent Object: [ContourFlangeFeature](../ContourFlangeFeature/ContourFlangeFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

ContourFlangeFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
