# DerivedAssemblyComponents.Parent Property

Parent Object: [DerivedAssemblyComponents](../DerivedAssemblyComponents/DerivedAssemblyComponents.md)

## Description

Property that returns the parent ComponentDefinition object.

## Syntax

DerivedAssemblyComponents.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 5.3
