# BIMComponent.ConnectorLinks Property

Parent Object: [BIMComponent](../BIMComponent/BIMComponent.md)

## Description

Read-only property that returns the collection of connector links for this document. Through the returned object you can access all existing links between connectors and create new links between connectors.

## Syntax

BIMComponent.**ConnectorLinks**() As [BIMConnectorLinks](../BIMConnectorLinks/BIMConnectorLinks.md)

## Property Value

This is a read only property whose value is a [BIMConnectorLinks](../BIMConnectorLinks/BIMConnectorLinks.md).

## Version

Introduced in version 2011
