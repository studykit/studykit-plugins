# BIMComponent.Connectors Property

Parent Object: [BIMComponent](../BIMComponent/BIMComponent.md)

## Description

Read-only property that returns the collection of connectors for this document. Through the returned object you can access all existing connectors and create new connectors.

## Syntax

BIMComponent.**Connectors**() As [BIMConnectors](../BIMConnectors/BIMConnectors.md)

## Property Value

This is a read only property whose value is a [BIMConnectors](../BIMConnectors/BIMConnectors.md).

## Version

Introduced in version 2011
