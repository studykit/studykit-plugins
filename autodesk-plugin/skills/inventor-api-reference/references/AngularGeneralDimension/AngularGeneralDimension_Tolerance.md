# AngularGeneralDimension.Tolerance Property

Parent Object: [AngularGeneralDimension](../AngularGeneralDimension/AngularGeneralDimension.md)

## Description

Property that returns the Tolerance object associated with this dimension.

## Syntax

AngularGeneralDimension.**Tolerance**() As [Tolerance](../Tolerance/Tolerance.md)

## Property Value

This is a read only property whose value is a [Tolerance](../Tolerance/Tolerance.md).

## Version

Introduced in version 11
