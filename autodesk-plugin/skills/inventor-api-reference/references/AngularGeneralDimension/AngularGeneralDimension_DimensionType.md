# AngularGeneralDimension.DimensionType Property

Parent Object: [AngularGeneralDimension](../AngularGeneralDimension/AngularGeneralDimension.md)

## Description

Gets the dimension type.

## Syntax

AngularGeneralDimension.**DimensionType**() As [DimensionTypeEnum](../DimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [DimensionTypeEnum](../DimensionTypeEnum.md).

## Version

Introduced in version 2015
