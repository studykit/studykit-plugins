# AngularGeneralDimension.ExtensionLineOneVisible Property

Parent Object: [AngularGeneralDimension](../AngularGeneralDimension/AngularGeneralDimension.md)

## Description

Gets and sets whether the first extension line is visible.

## Syntax

AngularGeneralDimension.**ExtensionLineOneVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
