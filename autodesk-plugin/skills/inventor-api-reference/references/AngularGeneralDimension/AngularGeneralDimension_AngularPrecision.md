# AngularGeneralDimension.AngularPrecision Property

Parent Object: [AngularGeneralDimension](../AngularGeneralDimension/AngularGeneralDimension.md)

## Description

Gets and sets the angular precision of the dimension value.

## Syntax

AngularGeneralDimension.**AngularPrecision**() As [AngularPrecisionEnum](../AngularPrecisionEnum.md)

## Property Value

This is a read/write property whose value is an [AngularPrecisionEnum](../AngularPrecisionEnum.md).

## Version

Introduced in version 2008
