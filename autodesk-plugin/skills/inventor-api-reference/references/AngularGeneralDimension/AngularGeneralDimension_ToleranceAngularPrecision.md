# AngularGeneralDimension.ToleranceAngularPrecision Property

Parent Object: [AngularGeneralDimension](../AngularGeneralDimension/AngularGeneralDimension.md)

## Description

Gets and sets the angular precision of the tolerance text for the dimension.

## Syntax

AngularGeneralDimension.**ToleranceAngularPrecision**() As [AngularPrecisionEnum](../AngularPrecisionEnum.md)

## Property Value

This is a read/write property whose value is an [AngularPrecisionEnum](../AngularPrecisionEnum.md).

## Version

Introduced in version 2008
