# SketchDrivenPatternFeature.AppearanceSourceType Property

Parent Object: [SketchDrivenPatternFeature](../SketchDrivenPatternFeature/SketchDrivenPatternFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

SketchDrivenPatternFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2017
