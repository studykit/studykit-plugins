# SketchDrivenPatternFeature.Adaptive Property

Parent Object: [SketchDrivenPatternFeature](../SketchDrivenPatternFeature/SketchDrivenPatternFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

SketchDrivenPatternFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
