# SketchDrivenPatternFeature.SurfacesVisible Property

Parent Object: [SketchDrivenPatternFeature](../SketchDrivenPatternFeature/SketchDrivenPatternFeature.md)

## Description

Gets and sets whether all the resulting surfaces are visible or not.

## Syntax

SketchDrivenPatternFeature.**SurfacesVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
