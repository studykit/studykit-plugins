# SketchDrivenPatternFeature.SurfacesOpaque Property

Parent Object: [SketchDrivenPatternFeature](../SketchDrivenPatternFeature/SketchDrivenPatternFeature.md)

## Description

Gets and sets whether all the resulting surfaces are opaque or transparent.

## Syntax

SketchDrivenPatternFeature.**SurfacesOpaque**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
