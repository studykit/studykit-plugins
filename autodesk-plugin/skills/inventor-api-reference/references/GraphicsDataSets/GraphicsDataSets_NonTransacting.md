# GraphicsDataSets.NonTransacting Property

Parent Object: [GraphicsDataSets](../GraphicsDataSets/GraphicsDataSets.md)

## Description

Read-only property that returns whether the creation of this GraphicsDataSets object and all its associated data is non-transacting.

## Syntax

GraphicsDataSets.**NonTransacting**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
