# GraphicsDataSets.SaveWithDocument Property

Parent Object: [GraphicsDataSets](../GraphicsDataSets/GraphicsDataSets.md)

## Description

Property that returns whether to save the GraphicsDataSets with the document.

## Syntax

GraphicsDataSets.**SaveWithDocument**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
