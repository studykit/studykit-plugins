# GraphicsDataSets.ItemById Property

Parent Object: [GraphicsDataSets](../GraphicsDataSets/GraphicsDataSets.md)

## Description

Returns the specified object from the collection.

## Syntax

GraphicsDataSets.**ItemById**( ***DataSetId*** As Long ) As [GraphicsDataSet](../GraphicsDataSet/GraphicsDataSet.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSet](../GraphicsDataSet/GraphicsDataSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DataSetId | Long | Input Long that specifies the ID of the object within the collection. |
