# DisplaySettings.NewWindowProjectionType Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets the default projection type for new windows.

## Syntax

DisplaySettings.**NewWindowProjectionType**() As [ProjectionTypeEnum](../ProjectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ProjectionTypeEnum](../ProjectionTypeEnum.md).

## Version

Introduced in version 2011
