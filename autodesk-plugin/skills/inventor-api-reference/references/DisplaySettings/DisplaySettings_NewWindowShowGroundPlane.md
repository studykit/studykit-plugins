# DisplaySettings.NewWindowShowGroundPlane Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets whether to display ground planes for new windows.

## Syntax

DisplaySettings.**NewWindowShowGroundPlane**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
