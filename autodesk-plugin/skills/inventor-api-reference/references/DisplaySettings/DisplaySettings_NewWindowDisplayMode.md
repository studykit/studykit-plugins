# DisplaySettings.NewWindowDisplayMode Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets the default display mode for new windows.

## Syntax

DisplaySettings.**NewWindowDisplayMode**() As [DisplayModeEnum](../DisplayModeEnum.md)

## Property Value

This is a read/write property whose value is a [DisplayModeEnum](../DisplayModeEnum.md).

## Version

Introduced in version 2011
