# DisplaySettings.Parent Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Property that returns the parent Document object.

## Syntax

DisplaySettings.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2011
