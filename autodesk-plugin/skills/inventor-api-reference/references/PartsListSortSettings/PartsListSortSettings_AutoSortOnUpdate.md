# PartsListSortSettings.AutoSortOnUpdate Property

Parent Object: [PartsListSortSettings](../PartsListSortSettings/PartsListSortSettings.md)

## Description

Read-write property that gets and sets whether to sort parts list automatically when update.

## Syntax

PartsListSortSettings.**AutoSortOnUpdate**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
