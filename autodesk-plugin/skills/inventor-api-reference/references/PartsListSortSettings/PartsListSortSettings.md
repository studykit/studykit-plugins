# PartsListSortSettings Object

## Description

PartsListSortSetting Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PartsListSortSettings/PartsListSortSettings_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [AutoSortOnUpdate](../PartsListSortSettings/PartsListSortSettings_AutoSortOnUpdate.md) | Read-write property that gets and sets whether to sort parts list automatically when update. |
| [Parent](../PartsListSortSettings/PartsListSortSettings_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [PrimaryColumnAscending](../PartsListSortSettings/PartsListSortSettings_PrimaryColumnAscending.md) | Read-write property that gets and sets whether to sort from the lowest value to the highest or vice versa. The default value is True indicating that the sorting will be from the lowest value to the highest. |
| [PrimaryColumnTitle](../PartsListSortSettings/PartsListSortSettings_PrimaryColumnTitle.md) | Read-write property that gets and sets the primary column to use for the sorting. If the title of a non-existing column is provided, an error occurs. |
| [SecondaryColumnAscending](../PartsListSortSettings/PartsListSortSettings_SecondaryColumnAscending.md) | Read-write property that gets and sets whether to sort from the lowest value to the highest or vice versa. The default value is True indicating that the sorting will be from the lowest value to the highest. |
| [SecondaryColumnTitle](../PartsListSortSettings/PartsListSortSettings_SecondaryColumnTitle.md) | Read-write property that gets and sets the secondary column to use for the sorting. If the title of a non-existing column is provided, an error occurs. |
| [SortNumerically](../PartsListSortSettings/PartsListSortSettings_SortNumerically.md) | Read-write property that gets and sets whether sorts by numerical order or not. |
| [TertiaryColumnAscending](../PartsListSortSettings/PartsListSortSettings_TertiaryColumnAscending.md) | Read-write property that gets and sets whether to sort from the lowest value to the highest or vice versa. The default value is True indicating that the sorting will be from the lowest value to the highest. |
| [TertiaryColumnTitle](../PartsListSortSettings/PartsListSortSettings_TertiaryColumnTitle.md) | Read-write property that gets and sets the tertiary column to use for the sorting. If the title of a non-existing column is provided, an error occurs. |
| [Type](../PartsListSortSettings/PartsListSortSettings_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PartsListStyle.SortSettings](../PartsListStyle/PartsListStyle_SortSettings.md)

## Version

Introduced in version 2025
