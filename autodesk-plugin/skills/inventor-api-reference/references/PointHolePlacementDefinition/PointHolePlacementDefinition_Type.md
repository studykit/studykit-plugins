# PointHolePlacementDefinition.Type Property

Parent Object: [PointHolePlacementDefinition](../PointHolePlacementDefinition/PointHolePlacementDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

PointHolePlacementDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
