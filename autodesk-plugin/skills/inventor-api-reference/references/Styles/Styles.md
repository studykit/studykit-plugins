# Styles Object

## Description

The Styles collection object provides access to all the Style objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Styles/Styles_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Styles/Styles_Count.md) | Property that returns the number of items in this collection. |
| [Item](../Styles/Styles_Item.md) | Returns the specified Style object from the collection. |
| [Type](../Styles/Styles_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.Styles](../DrawingStylesManager/DrawingStylesManager_Styles.md)

## Version

Introduced in version 2008
