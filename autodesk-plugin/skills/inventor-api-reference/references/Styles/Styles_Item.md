# Styles.Item Property

Parent Object: [Styles](../Styles/Styles.md)

## Description

Returns the specified Style object from the collection.

## Syntax

Styles.**Item**( ***Index*** As Variant ) As [Style](../Style/Style.md)

## Property Value

This is a read only property whose value is a [Style](../Style/Style.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the Style to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the internal name of a Style. If an out of range index or a name of a non-existent Style is provided, an error will occur. |

## Version

Introduced in version 2008
