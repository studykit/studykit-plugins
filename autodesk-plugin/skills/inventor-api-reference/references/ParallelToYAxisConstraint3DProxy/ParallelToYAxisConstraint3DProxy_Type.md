# ParallelToYAxisConstraint3DProxy.Type Property

Parent Object: [ParallelToYAxisConstraint3DProxy](../ParallelToYAxisConstraint3DProxy/ParallelToYAxisConstraint3DProxy.md)

## Description

## Syntax

ParallelToYAxisConstraint3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
