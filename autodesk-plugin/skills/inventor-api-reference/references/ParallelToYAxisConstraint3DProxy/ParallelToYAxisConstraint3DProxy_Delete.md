# ParallelToYAxisConstraint3DProxy.Delete Method

Parent Object: [ParallelToYAxisConstraint3DProxy](../ParallelToYAxisConstraint3DProxy/ParallelToYAxisConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ParallelToYAxisConstraint3DProxy.**Delete**()

## Version

Introduced in version 2017
