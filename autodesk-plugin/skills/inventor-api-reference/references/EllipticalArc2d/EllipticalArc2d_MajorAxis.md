# EllipticalArc2d.MajorAxis Property

Parent Object: [EllipticalArc2d](../EllipticalArc2d/EllipticalArc2d.md)

## Description

Specifies the major axis for the ellipse.

## Syntax

EllipticalArc2d.**MajorAxis**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read/write property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).

## Version

Introduced in version 5
