# EllipticalArc2d.Center Property

Parent Object: [EllipticalArc2d](../EllipticalArc2d/EllipticalArc2d.md)

## Description

Specifies the center point for the arc.

## Syntax

EllipticalArc2d.**Center**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
