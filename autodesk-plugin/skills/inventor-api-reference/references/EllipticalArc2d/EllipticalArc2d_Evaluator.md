# EllipticalArc2d.Evaluator Property

Parent Object: [EllipticalArc2d](../EllipticalArc2d/EllipticalArc2d.md)

## Description

Gets the Curve2DEvaluator for the arc.

## Syntax

EllipticalArc2d.**Evaluator**() As [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md)

## Property Value

This is a read only property whose value is a [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md).

## Version

Introduced in version 5
