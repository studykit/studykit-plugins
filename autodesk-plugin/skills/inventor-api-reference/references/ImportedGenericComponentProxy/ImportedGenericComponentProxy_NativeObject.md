# ImportedGenericComponentProxy.NativeObject Property

Parent Object: [ImportedGenericComponentProxy](../ImportedGenericComponentProxy/ImportedGenericComponentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

ImportedGenericComponentProxy.**NativeObject**() As [ImportedGenericComponent](../ImportedGenericComponent/ImportedGenericComponent.md)

## Property Value

This is a read only property whose value is an [ImportedGenericComponent](../ImportedGenericComponent/ImportedGenericComponent.md).

## Version

Introduced in version 2016
