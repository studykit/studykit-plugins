# ModelDatumIdentifiers.NextAvailableDatumText Property

Parent Object: [ModelDatumIdentifiers](../ModelDatumIdentifiers/ModelDatumIdentifiers.md)

## Description

Read-only property that returns a string value that indicates an unused datum text.

## Syntax

ModelDatumIdentifiers.**NextAvailableDatumText**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
