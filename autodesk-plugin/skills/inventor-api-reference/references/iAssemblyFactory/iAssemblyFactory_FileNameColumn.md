# iAssemblyFactory.FileNameColumn Property

Parent Object: [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Description

Read-write property that gets and sets the column to use as the file name column.

## Syntax

iAssemblyFactory.**FileNameColumn**() As [iAssemblyTableColumn](../iAssemblyTableColumn/iAssemblyTableColumn.md)

## Property Value

This is a read/write property whose value is an [iAssemblyTableColumn](../iAssemblyTableColumn/iAssemblyTableColumn.md).

## Version

Introduced in version 2011
