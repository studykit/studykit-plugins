# iAssemblyFactory.MemberCacheDir Property

Parent Object: [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Description

Property that returns the directory location for the newly created iAssembly members.

## Syntax

iAssemblyFactory.**MemberCacheDir**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 11
