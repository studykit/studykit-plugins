# iAssemblyFactory.TableRows Property

Parent Object: [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Description

Property that returns the iAssemblyTableRows collection object.

## Syntax

iAssemblyFactory.**TableRows**() As [iAssemblyTableRows](../iAssemblyTableRows/iAssemblyTableRows.md)

## Property Value

This is a read only property whose value is an [iAssemblyTableRows](../iAssemblyTableRows/iAssemblyTableRows.md).
