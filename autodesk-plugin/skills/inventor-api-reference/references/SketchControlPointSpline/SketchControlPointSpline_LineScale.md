# SketchControlPointSpline.LineScale Property

Parent Object: [SketchControlPointSpline](../SketchControlPointSpline/SketchControlPointSpline.md)

## Description

Gets and sets the LineScale applied to this sketch spline.

## Syntax

SketchControlPointSpline.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
