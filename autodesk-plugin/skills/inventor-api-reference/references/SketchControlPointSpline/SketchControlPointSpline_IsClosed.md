# SketchControlPointSpline.IsClosed Property

Parent Object: [SketchControlPointSpline](../SketchControlPointSpline/SketchControlPointSpline.md)

## Description

Read-only property that returns whether the curve is closed.

## Syntax

SketchControlPointSpline.**IsClosed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
