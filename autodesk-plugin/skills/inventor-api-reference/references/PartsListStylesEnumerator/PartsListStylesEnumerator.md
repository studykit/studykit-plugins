# PartsListStylesEnumerator Object

## Description

The PartsListStylesEnumerator object provides access to the existing PartsListStyle objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PartsListStylesEnumerator/PartsListStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../PartsListStylesEnumerator/PartsListStylesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../PartsListStylesEnumerator/PartsListStylesEnumerator_Item.md) | Returns the specified PartsListStyle object from the collection. |
| [Type](../PartsListStylesEnumerator/PartsListStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.PartsListStyles](../DrawingStylesManager/DrawingStylesManager_PartsListStyles.md)

## Version

Introduced in version 2011
