# ClientFeatures.Item Property

Parent Object: [ClientFeatures](../ClientFeatures/ClientFeatures.md)

## Description

Returns the specified ClientFeature object from the collection.

## Syntax

ClientFeatures.**Item**( ***Index*** As Variant ) As [ClientFeature](../ClientFeature/ClientFeature.md)

## Property Value

This is a read only property whose value is a [ClientFeature](../ClientFeature/ClientFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the ClientFeature to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the ClientFeature name. If an out of range index or a name of a non-existent ClientFeature is provided, an error will occur. |

## Version

Introduced in version 2008
