# ClientFeatures Object

## Description

The ClientFeatures collection object provides access to client features and provides methods to create additional client features.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ClientFeatures/ClientFeatures_Add.md) | Method that creates a new ClientFeature. The newly created ClientFeature is returned. |
| [CreateDefinition](../ClientFeatures/ClientFeatures_CreateDefinition.md) | Method that creates a new ClientFeatureDefinition. The newly created ClientFeatureDefinition is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ClientFeatures/ClientFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ClientFeatures/ClientFeatures_Count.md) | Property that returns the number of items in the collection. |
| [Item](../ClientFeatures/ClientFeatures_Item.md) | Returns the specified ClientFeature object from the collection. |
| [Type](../ClientFeatures/ClientFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Features.ClientFeatures](../Features/Features_ClientFeatures.md), [FlatPatternFeatures.ClientFeatures](../FlatPatternFeatures/FlatPatternFeatures_ClientFeatures.md), [PartFeatures.ClientFeatures](../PartFeatures/PartFeatures_ClientFeatures.md), [SheetMetalFeatures.ClientFeatures](../SheetMetalFeatures/SheetMetalFeatures_ClientFeatures.md)

## Version

Introduced in version 2008
