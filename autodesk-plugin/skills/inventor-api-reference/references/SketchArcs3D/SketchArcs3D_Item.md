# SketchArcs3D.Item Property

Parent Object: [SketchArcs3D](../SketchArcs3D/SketchArcs3D.md)

## Description

Returns the specified SketchArc3D object from the collection.

## Syntax

SketchArcs3D.**Item**( ***Index*** As Long ) As [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Property Value

This is a read only property whose value is a [SketchArc3D](../SketchArc3D/SketchArc3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
