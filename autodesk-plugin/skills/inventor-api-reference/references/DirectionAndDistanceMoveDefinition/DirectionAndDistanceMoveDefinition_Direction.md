# DirectionAndDistanceMoveDefinition.Direction Property

Parent Object: [DirectionAndDistanceMoveDefinition](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition.md)

## Description

Gets and sets the entity defining the direction for the move.

## Syntax

DirectionAndDistanceMoveDefinition.**Direction**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2011
