# DirectionAndDistanceMoveDefinition Object

## Description

The DirectionAndDistanceMoveDefinition object defines all of the inputs that are unique to a move face feature defined using a direction vector and a distance.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Direction](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition_Direction.md) | Gets and sets the entity defining the direction for the move. |
| [DirectionReversed](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition_DirectionReversed.md) | Gets and sets whether the direction defined by the entity in the Direction property should be reversed. |
| [Distance](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition_Distance.md) | Property that returns the distance Parameter object associated with the move. |
| [Parent](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition_Parent.md) | Property that returns the parent MoveFaceDefinition object. |
| [Type](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2011
