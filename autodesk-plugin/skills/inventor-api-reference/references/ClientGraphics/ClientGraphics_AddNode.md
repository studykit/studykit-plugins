# ClientGraphics.AddNode Method

Parent Object: [ClientGraphics](../ClientGraphics/ClientGraphics.md)

## Description

Method that creates a new GraphicsNode object.

## Syntax

ClientGraphics.**AddNode**( ***NodeId*** As Long ) As [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| NodeId | Long | Input Long that specifies the identifier for the newly created entity. This id needs to be unique with respect to all other objects in this ClientGraphics object. |
