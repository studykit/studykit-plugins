# Sketch3DProxy.ReferencedEntity Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that returns the referenced 3D sketch, if there is one. Currently this property always returns 'Nothing'; it was implemented to support future functionality.

## Syntax

Sketch3DProxy.**ReferencedEntity**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 6
