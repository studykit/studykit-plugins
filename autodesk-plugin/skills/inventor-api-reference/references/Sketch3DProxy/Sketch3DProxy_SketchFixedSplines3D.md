# Sketch3DProxy.SketchFixedSplines3D Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that gets the collection object.

## Syntax

Sketch3DProxy.**SketchFixedSplines3D**() As [SketchFixedSplines3D](../SketchFixedSplines3D/SketchFixedSplines3D.md)

## Property Value

This is a read only property whose value is a [SketchFixedSplines3D](../SketchFixedSplines3D/SketchFixedSplines3D.md).

## Version

Introduced in version 9
