# Sketch3DProxy.Profiles3D Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that returns the Profiles3D collection object.

## Syntax

Sketch3DProxy.**Profiles3D**() As [Profiles3D](../Profiles3D/Profiles3D.md)

## Property Value

This is a read only property whose value is a [Profiles3D](../Profiles3D/Profiles3D.md).

## Version

Introduced in version 6
