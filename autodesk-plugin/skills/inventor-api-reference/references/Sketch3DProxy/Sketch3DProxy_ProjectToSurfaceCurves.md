# Sketch3DProxy.ProjectToSurfaceCurves Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Read-only property that returns the ProjectToSurfaceCurves collection object. This object provides access to all the existing ProjectToSurfaceCurve objects in the sketch and also provides methods to create new ProjectToSurfaceCurve objects.

## Syntax

Sketch3DProxy.**ProjectToSurfaceCurves**() As [ProjectToSurfaceCurves](../ProjectToSurfaceCurves/ProjectToSurfaceCurves.md)

## Property Value

This is a read only property whose value is a [ProjectToSurfaceCurves](../ProjectToSurfaceCurves/ProjectToSurfaceCurves.md).

## Version

Introduced in version 2021
