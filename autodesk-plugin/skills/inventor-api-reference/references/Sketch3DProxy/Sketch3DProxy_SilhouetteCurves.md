# Sketch3DProxy.SilhouetteCurves Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Gets the SilhouetteCurve collection Object.

## Syntax

Sketch3DProxy.**SilhouetteCurves**() As [SilhouetteCurves](../SilhouetteCurves/SilhouetteCurves.md)

## Property Value

This is a read only property whose value is a [SilhouetteCurves](../SilhouetteCurves/SilhouetteCurves.md).

## Version

Introduced in version 2012
