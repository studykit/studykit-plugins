# Sketch3DProxy.DimensionConstraints3D Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that returns a collection of all dimension constraints on the 3D sketch.

## Syntax

Sketch3DProxy.**DimensionConstraints3D**() As [DimensionConstraints3D](../DimensionConstraints3D/DimensionConstraints3D.md)

## Property Value

This is a read only property whose value is a [DimensionConstraints3D](../DimensionConstraints3D/DimensionConstraints3D.md).

## Version

Introduced in version 11
