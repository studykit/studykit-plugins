# HelicalCurveVariableShapeDefinition.ShapeDefinitionType Property

Parent Object: [HelicalCurveVariableShapeDefinition](../HelicalCurveVariableShapeDefinition/HelicalCurveVariableShapeDefinition.md)

## Description

Gets and sets the shape definition type of the helical curve.

## Syntax

HelicalCurveVariableShapeDefinition.**ShapeDefinitionType**() As [HelicalShapeDefinitionTypeEnum](../HelicalShapeDefinitionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [HelicalShapeDefinitionTypeEnum](../HelicalShapeDefinitionTypeEnum.md).

## Version

Introduced in version 2019
