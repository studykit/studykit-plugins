# ModelAnnotationText.LineSpacingType Property

Parent Object: [ModelAnnotationText](../ModelAnnotationText/ModelAnnotationText.md)

## Description

Property that gets and sets the method used to define the line spacing.

## Syntax

ModelAnnotationText.**LineSpacingType**() As [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md)

## Property Value

This is a read/write property whose value is a [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md).

## Version

Introduced in version 2018
