# ModelAnnotationText.HorizontalJustification Property

Parent Object: [ModelAnnotationText](../ModelAnnotationText/ModelAnnotationText.md)

## Description

Property that gets and sets the horizontal justification override of the annotation text.

## Syntax

ModelAnnotationText.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 2018
