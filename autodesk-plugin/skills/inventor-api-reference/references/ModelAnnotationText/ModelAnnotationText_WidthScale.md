# ModelAnnotationText.WidthScale Property

Parent Object: [ModelAnnotationText](../ModelAnnotationText/ModelAnnotationText.md)

## Description

Property that gets and sets the width scale factor for the dimension text.

## Syntax

ModelAnnotationText.**WidthScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
