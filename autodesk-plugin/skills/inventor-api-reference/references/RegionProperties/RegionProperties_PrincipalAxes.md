# RegionProperties.PrincipalAxes Method

Parent Object: [RegionProperties](../RegionProperties/RegionProperties.md)

## Description

Method that returns the principal axes.

## Syntax

RegionProperties.**PrincipalAxes**( ***XAxis*** As [Vector](../Vector/Vector.md), ***YAxis*** As [Vector](../Vector/Vector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| XAxis | [Vector](../Vector/Vector.md) | Output Vector object that indicates the direction of the X axis. |
| YAxis | [Vector](../Vector/Vector.md) | Output Vector object that indicates the direction of the Y axis. |

## Version

Introduced in version 2008
