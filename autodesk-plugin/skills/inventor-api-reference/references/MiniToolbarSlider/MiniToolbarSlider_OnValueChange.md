# MiniToolbarSlider.OnValueChange Event

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Event that is fired when the value of the slider has changed. This event is fired after the user has finished any scrolling and released the left mouse button. Use the Value property to get the current value of the slider.

## Syntax

MiniToolbarSlider.**OnValueChange**()
