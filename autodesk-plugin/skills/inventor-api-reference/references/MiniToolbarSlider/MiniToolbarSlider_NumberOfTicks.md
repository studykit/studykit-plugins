# MiniToolbarSlider.NumberOfTicks Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets the number of tick marks displayed. This does not include the start and end marks.

## Syntax

MiniToolbarSlider.**NumberOfTicks**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2013
