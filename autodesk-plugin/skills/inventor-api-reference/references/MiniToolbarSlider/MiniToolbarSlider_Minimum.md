# MiniToolbarSlider.Minimum Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets the minimum value of the slider. If the ValueType property is kIntegerType then this value will be rounded to the nearest integer.

## Syntax

MiniToolbarSlider.**Minimum**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2013
