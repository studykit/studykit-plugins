# MiniToolbarSlider.ToolTipText Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets the tooltip text for the control.

## Syntax

MiniToolbarSlider.**ToolTipText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2013
