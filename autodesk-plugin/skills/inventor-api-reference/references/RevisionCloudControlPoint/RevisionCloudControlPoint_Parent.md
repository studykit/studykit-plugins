# RevisionCloudControlPoint.Parent Property

Parent Object: [RevisionCloudControlPoint](../RevisionCloudControlPoint/RevisionCloudControlPoint.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

RevisionCloudControlPoint.**Parent**() As [RevisionCloudDefinition](../RevisionCloudDefinition/RevisionCloudDefinition.md)

## Property Value

This is a read only property whose value is a [RevisionCloudDefinition](../RevisionCloudDefinition/RevisionCloudDefinition.md).

## Version

Introduced in version 2024
