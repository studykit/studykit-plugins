# RevisionCloudControlPoint.Position Property

Parent Object: [RevisionCloudControlPoint](../RevisionCloudControlPoint/RevisionCloudControlPoint.md)

## Description

Gets and sets the position of the control point.

## Syntax

RevisionCloudControlPoint.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2024
