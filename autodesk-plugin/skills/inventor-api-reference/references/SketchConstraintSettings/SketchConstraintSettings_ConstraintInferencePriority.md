# SketchConstraintSettings.ConstraintInferencePriority Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets the priority of inferred constraints when placing.

## Syntax

SketchConstraintSettings.**ConstraintInferencePriority**() As [ConstraintInferencePriorityEnum](../ConstraintInferencePriorityEnum.md)

## Property Value

This is a read/write property whose value is a [ConstraintInferencePriorityEnum](../ConstraintInferencePriorityEnum.md).

## Version

Introduced in version 2015
