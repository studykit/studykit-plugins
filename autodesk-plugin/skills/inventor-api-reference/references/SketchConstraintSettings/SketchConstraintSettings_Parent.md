# SketchConstraintSettings.Parent Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SketchConstraintSettings.**Parent**() As [SketchOptions](../SketchOptions/SketchOptions.md)

## Property Value

This is a read only property whose value is a [SketchOptions](../SketchOptions/SketchOptions.md).

## Version

Introduced in version 2015
