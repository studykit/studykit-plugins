# SketchConstraintSettings.SelectionForGeometricConstraintsInference Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets which geometric constraints have their inference indicators shown to assist creating sketch entities.

## Syntax

SketchConstraintSettings.**SelectionForGeometricConstraintsInference**() As [SketchConstraintInferenceTypeEnum](../SketchConstraintInferenceTypeEnum.md)

## Property Value

This is a read/write property whose value is a [SketchConstraintInferenceTypeEnum](../SketchConstraintInferenceTypeEnum.md).

## Version

Introduced in version 2015
