# SketchConstraintSettings.OverConstrainedDimensionBehavior Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets the behavior when create an over-constrained dimension.

## Syntax

SketchConstraintSettings.**OverConstrainedDimensionBehavior**() As [OverConstrainedDimensionBehaviorEnum](../OverConstrainedDimensionBehaviorEnum.md)

## Property Value

This is a read/write property whose value is an [OverConstrainedDimensionBehaviorEnum](../OverConstrainedDimensionBehaviorEnum.md).

## Version

Introduced in version 2015
