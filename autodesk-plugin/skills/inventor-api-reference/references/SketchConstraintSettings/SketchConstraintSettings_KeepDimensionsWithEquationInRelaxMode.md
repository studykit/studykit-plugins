# SketchConstraintSettings.KeepDimensionsWithEquationInRelaxMode Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets whether to keep the dimensions with an equation in relax dragging and over-constraint solving.

## Syntax

SketchConstraintSettings.**KeepDimensionsWithEquationInRelaxMode**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
