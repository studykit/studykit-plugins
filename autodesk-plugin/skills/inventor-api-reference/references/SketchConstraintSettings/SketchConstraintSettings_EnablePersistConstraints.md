# SketchConstraintSettings.EnablePersistConstraints Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets whether to persist inferred constraints.

## Syntax

SketchConstraintSettings.**EnablePersistConstraints**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
