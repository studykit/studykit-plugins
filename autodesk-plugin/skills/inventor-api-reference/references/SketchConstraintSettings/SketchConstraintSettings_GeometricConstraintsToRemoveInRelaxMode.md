# SketchConstraintSettings.GeometricConstraintsToRemoveInRelaxMode Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets which geometric constraints can be removed in relax mode when dragging sketch entitiess.

## Syntax

SketchConstraintSettings.**GeometricConstraintsToRemoveInRelaxMode**() As [SketchGeometricConstraintTypeEnum](../SketchGeometricConstraintTypeEnum.md)

## Property Value

This is a read/write property whose value is a [SketchGeometricConstraintTypeEnum](../SketchGeometricConstraintTypeEnum.md).

## Version

Introduced in version 2015
