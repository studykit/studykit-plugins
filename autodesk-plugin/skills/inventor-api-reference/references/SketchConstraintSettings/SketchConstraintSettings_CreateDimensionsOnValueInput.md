# SketchConstraintSettings.CreateDimensionsOnValueInput Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets whether to create dimensions when inputting values into heads-up display fields.

## Syntax

SketchConstraintSettings.**CreateDimensionsOnValueInput**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
