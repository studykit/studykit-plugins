# LeaderNodesEnumerator Object

## Description

The LeaderNodesEnumerator object provides an enumeration of LeaderNode objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LeaderNodesEnumerator/LeaderNodesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LeaderNodesEnumerator/LeaderNodesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../LeaderNodesEnumerator/LeaderNodesEnumerator_Item.md) | Method that returns the specified LeaderNode object from the collection. |
| [Type](../LeaderNodesEnumerator/LeaderNodesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Leader.AllLeafNodes](../Leader/Leader_AllLeafNodes.md), [Leader.AllNodes](../Leader/Leader_AllNodes.md), [LeaderNode.ChildNodes](../LeaderNode/LeaderNode_ChildNodes.md)

## Version

Introduced in version 11
