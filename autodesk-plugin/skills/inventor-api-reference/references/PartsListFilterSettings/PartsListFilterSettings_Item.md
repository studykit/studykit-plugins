# PartsListFilterSettings.Item Property

Parent Object: [PartsListFilterSettings](../PartsListFilterSettings/PartsListFilterSettings.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PartsListFilterSettings.**Item**( ***Index*** As Long ) As [PartsListFilterItem](../PartsListFilterItem/PartsListFilterItem.md)

## Property Value

This is a read only property whose value is a [PartsListFilterItem](../PartsListFilterItem/PartsListFilterItem.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specify the PartsListFilterItem to return. If an out of range index is input, an error occurs. |

## Version

Introduced in version 2024.1
