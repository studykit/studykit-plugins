# PartsListFilterSettings.Enabled Property

Parent Object: [PartsListFilterSettings](../PartsListFilterSettings/PartsListFilterSettings.md)

## Description

Read-write property that gets and sets whether the filter settings is enabled or not.

## Syntax

PartsListFilterSettings.**Enabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024.1
