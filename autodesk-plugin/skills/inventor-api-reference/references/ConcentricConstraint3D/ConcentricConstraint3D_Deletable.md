# ConcentricConstraint3D.Deletable Property

Parent Object: [ConcentricConstraint3D](../ConcentricConstraint3D/ConcentricConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

ConcentricConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
