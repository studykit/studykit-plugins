# SketchBlockDefinitionProxy.IsOwnedByFeature Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns whether this object is owned by a feature. This property should return True for sketches that are created as a result of an unfold or refold feature.

## Syntax

SketchBlockDefinitionProxy.**IsOwnedByFeature**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
