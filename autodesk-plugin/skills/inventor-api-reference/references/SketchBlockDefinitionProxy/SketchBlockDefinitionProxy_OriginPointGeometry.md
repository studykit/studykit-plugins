# SketchBlockDefinitionProxy.OriginPointGeometry Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that gets the geometry that describes the origin point.

## Syntax

SketchBlockDefinitionProxy.**OriginPointGeometry**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2010
