# SketchBlockDefinitionProxy.PlanarEntityGeometry Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the geometry that describes the plane the sketch is based on.

## Syntax

SketchBlockDefinitionProxy.**PlanarEntityGeometry**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).

## Version

Introduced in version 2010
