# SketchBlockDefinitionProxy.IsInsertionPointVisible Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and sets whether the insertion point is visible.

## Syntax

SketchBlockDefinitionProxy.**IsInsertionPointVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
