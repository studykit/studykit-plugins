# SketchBlockDefinitionProxy.DeferUpdates Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and Sets whether to defer the solving of the sketch or not.

## Syntax

SketchBlockDefinitionProxy.**DeferUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
