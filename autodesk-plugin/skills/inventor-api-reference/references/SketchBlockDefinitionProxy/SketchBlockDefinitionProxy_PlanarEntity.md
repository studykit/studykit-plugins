# SketchBlockDefinitionProxy.PlanarEntity Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and sets the planar object that defines the planar object the sketch is to be built on.

## Syntax

SketchBlockDefinitionProxy.**PlanarEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2010
