# SketchBlockDefinitionProxy.TextBoxes Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets the TextBoxes collection associated with this Sketch.

## Syntax

SketchBlockDefinitionProxy.**TextBoxes**() As [TextBoxes](../TextBoxes/TextBoxes.md)

## Property Value

This is a read only property whose value is a [TextBoxes](../TextBoxes/TextBoxes.md).

## Version

Introduced in version 2010
