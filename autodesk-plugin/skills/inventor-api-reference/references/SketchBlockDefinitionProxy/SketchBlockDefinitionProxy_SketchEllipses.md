# SketchBlockDefinitionProxy.SketchEllipses Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the SketchEllipses collection object.

## Syntax

SketchBlockDefinitionProxy.**SketchEllipses**() As [SketchEllipses](../SketchEllipses/SketchEllipses.md)

## Property Value

This is a read only property whose value is a [SketchEllipses](../SketchEllipses/SketchEllipses.md).

## Version

Introduced in version 2010
