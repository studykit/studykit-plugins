# SketchBlockDefinitionProxy.AxisIsX Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and sets if the axis entity defines the X or Y axis. True indicates the axis defines the X-axis.

## Syntax

SketchBlockDefinitionProxy.**AxisIsX**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
