# SketchBlockDefinitionProxy.LineType Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and Sets the line type override for the sketch.

## Syntax

SketchBlockDefinitionProxy.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2010
