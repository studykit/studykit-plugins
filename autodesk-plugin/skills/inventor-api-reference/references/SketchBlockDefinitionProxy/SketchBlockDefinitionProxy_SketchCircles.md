# SketchBlockDefinitionProxy.SketchCircles Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

SketchBlockDefinitionProxy.**SketchCircles**() As [SketchCircles](../SketchCircles/SketchCircles.md)

## Property Value

This is a read only property whose value is a [SketchCircles](../SketchCircles/SketchCircles.md).

## Version

Introduced in version 2010
