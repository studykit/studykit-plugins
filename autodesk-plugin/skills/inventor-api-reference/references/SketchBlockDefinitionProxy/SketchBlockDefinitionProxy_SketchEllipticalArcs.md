# SketchBlockDefinitionProxy.SketchEllipticalArcs Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the SketchEllipticalArcs collection object.

## Syntax

SketchBlockDefinitionProxy.**SketchEllipticalArcs**() As [SketchEllipticalArcs](../SketchEllipticalArcs/SketchEllipticalArcs.md)

## Property Value

This is a read only property whose value is a [SketchEllipticalArcs](../SketchEllipticalArcs/SketchEllipticalArcs.md).

## Version

Introduced in version 2010
