# SketchBlockDefinitionProxy.ModelToSketchTransform Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the transformation from model space to the 2d sketch coordinate space.

## Syntax

SketchBlockDefinitionProxy.**ModelToSketchTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2010
