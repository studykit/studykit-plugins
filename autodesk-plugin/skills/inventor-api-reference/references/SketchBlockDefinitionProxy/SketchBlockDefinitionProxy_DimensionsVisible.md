# SketchBlockDefinitionProxy.DimensionsVisible Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and sets whether the dimensions on the sketch are visible.

## Syntax

SketchBlockDefinitionProxy.**DimensionsVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
