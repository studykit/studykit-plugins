# SketchBlockDefinitionProxy.SketchSplines Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the SketchSplines collection object.

## Syntax

SketchBlockDefinitionProxy.**SketchSplines**() As [SketchSplines](../SketchSplines/SketchSplines.md)

## Property Value

This is a read only property whose value is a [SketchSplines](../SketchSplines/SketchSplines.md).

## Version

Introduced in version 2010
