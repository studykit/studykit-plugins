# ClientFeatureElement.UserEditable Property

Parent Object: [ClientFeatureElement](../ClientFeatureElement/ClientFeatureElement.md)

## Description

Read-write property that gets and sets whether this element can be edited by the user.

## Syntax

ClientFeatureElement.**UserEditable**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
