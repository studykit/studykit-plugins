# ClientFeatureElement Object

## Description

Represents a composited Inventor object within a ClientFeature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../ClientFeatureElement/ClientFeatureElement_Delete.md) | Method that deletes this element of the client feature |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ClientFeatureElement/ClientFeatureElement_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [BrowserVisible](../ClientFeatureElement/ClientFeatureElement_BrowserVisible.md) | Read-write property that gets and sets whether this element is visible in the browser under the ClientFeature node. |
| [Element](../ClientFeatureElement/ClientFeatureElement_Element.md) |  |
| [HighlightWithFeature](../ClientFeatureElement/ClientFeatureElement_HighlightWithFeature.md) | Read-write property that gets and sets whether this element should be pre-highlight, highlighted or selected with the client feature. |
| [Parent](../ClientFeatureElement/ClientFeatureElement_Parent.md) |  |
| [Type](../ClientFeatureElement/ClientFeatureElement_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [UserEditable](../ClientFeatureElement/ClientFeatureElement_UserEditable.md) | Read-write property that gets and sets whether this element can be edited by the user. |
