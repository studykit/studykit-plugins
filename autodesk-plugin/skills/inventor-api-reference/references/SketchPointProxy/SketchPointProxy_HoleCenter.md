# SketchPointProxy.HoleCenter Property

Parent Object: [SketchPointProxy](../SketchPointProxy/SketchPointProxy.md)

## Description

Gets and sets whether the point behaves as a hole center or not.

## Syntax

SketchPointProxy.**HoleCenter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
