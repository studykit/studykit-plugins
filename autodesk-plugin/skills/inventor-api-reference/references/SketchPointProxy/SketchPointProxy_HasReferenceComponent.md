# SketchPointProxy.HasReferenceComponent Property

Parent Object: [SketchPointProxy](../SketchPointProxy/SketchPointProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchPointProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
