# SketchPointProxy.Merge Method

Parent Object: [SketchPointProxy](../SketchPointProxy/SketchPointProxy.md)

## Description

Method that merges this sketch point with the input sketch point. Any objects dependent on this sketch point will change their dependency to the new sketch point.

## Syntax

SketchPointProxy.**Merge**( ***SketchPoint*** As [SketchPoint](../SketchPoint/SketchPoint.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SketchPoint | [SketchPoint](../SketchPoint/SketchPoint.md) | Input SketchPoint object. |

## Version

Introduced in version 6
