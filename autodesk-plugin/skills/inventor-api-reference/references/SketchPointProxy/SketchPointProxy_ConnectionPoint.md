# SketchPointProxy.ConnectionPoint Property

Parent Object: [SketchPointProxy](../SketchPointProxy/SketchPointProxy.md)

## Description

Gets and sets whether the point behaves as a connection point or not.

## Syntax

SketchPointProxy.**ConnectionPoint**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
