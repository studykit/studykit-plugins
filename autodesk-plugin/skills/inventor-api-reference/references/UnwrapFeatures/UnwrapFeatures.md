# UnwrapFeatures Object

## Description

Part Unwrap Features Collection Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../UnwrapFeatures/UnwrapFeatures_Add.md) | Method that creates a new Unwrap feature. The newly created UnwrapFeature object is returned. |
| [CreateDefinition](../UnwrapFeatures/UnwrapFeatures_CreateDefinition.md) | Method that creates a new UnwrapDefinition object. The object created does not represent a Unwrap feature but instead is a representation of the information that defines a Unwrap feature. You can use this object as input to the UnwrapFeatures.Add method to create the actual feature. |
| [CreateDefinitionWithOptions](../UnwrapFeatures/UnwrapFeatures_CreateDefinitionWithOptions.md) | Method that creates the UnwrapDefinition. The object created does not represent a Unwrap feature but instead is a representation of the information that defines a Unwrap feature. You can use this object as input to the UnwrapFeatures.Add method to create the. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../UnwrapFeatures/UnwrapFeatures_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../UnwrapFeatures/UnwrapFeatures_Count.md) | Gets the number of items in this collection. |
| [Item](../UnwrapFeatures/UnwrapFeatures_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../UnwrapFeatures/UnwrapFeatures_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PartFeatures.UnwrapFeatures](../PartFeatures/PartFeatures_UnwrapFeatures.md), [SheetMetalFeatures.UnwrapFeatures](../SheetMetalFeatures/SheetMetalFeatures_UnwrapFeatures.md)

## Version

Introduced in version 2020
