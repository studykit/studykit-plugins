# SheetMetalComponentDefinition.iMateDefinitions Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the iMateDefinitions collection object associated with this part.

## Syntax

SheetMetalComponentDefinition.**iMateDefinitions**() As [iMateDefinitions](../iMateDefinitions/iMateDefinitions.md)

## Property Value

This is a read only property whose value is an [iMateDefinitions](../iMateDefinitions/iMateDefinitions.md).

## Version

Introduced in version 6
