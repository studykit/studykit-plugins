# SheetMetalComponentDefinition.Unfold2 Method

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Method that generates the flat pattern with the input face as the base face.

## Syntax

SheetMetalComponentDefinition.**Unfold2**( ***BaseFace*** As [Face](../Face/Face.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| BaseFace | [Face](../Face/Face.md) | Input Face object from the folded model to be used as the base face when generating the flat pattern. |

## Version

Introduced in version 2009
