# SheetMetalComponentDefinition.Occurrences Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the collection object.

## Syntax

SheetMetalComponentDefinition.**Occurrences**() As [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md).

## Version

Introduced in version 5.3
