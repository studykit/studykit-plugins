# SheetMetalComponentDefinition.ModelAnnotations Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-only property that returns the ModelAnnotations collection object. This object provides access to all of the model annotations in the part.

## Syntax

SheetMetalComponentDefinition.**ModelAnnotations**() As [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md)

## Property Value

This is a read only property whose value is a [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md).

## Version

Introduced in version 2018
