# SheetMetalComponentDefinition.ASideDefinitions Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-only property that returns the ASideFaceDefinitions collection object.

## Syntax

SheetMetalComponentDefinition.**ASideDefinitions**() As [ASideDefinitions](../ASideDefinitions/ASideDefinitions.md)

## Property Value

This is a read only property whose value is an [ASideDefinitions](../ASideDefinitions/ASideDefinitions.md).

## Version

Introduced in version 2016
