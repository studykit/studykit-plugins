# SheetMetalComponentDefinition.CompactModelHistoryOnNextSave Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Gets/Sets the flag that decides whether the BRep's modifications through its history are saved explicitly in the next save or not (independent of CompactModelHistory setting).

## Syntax

SheetMetalComponentDefinition.**CompactModelHistoryOnNextSave**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
