# SheetMetalComponentDefinition.MinimumRemnant Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the parameter that controls the minimum remnant of the part.

## Syntax

SheetMetalComponentDefinition.**MinimumRemnant**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
