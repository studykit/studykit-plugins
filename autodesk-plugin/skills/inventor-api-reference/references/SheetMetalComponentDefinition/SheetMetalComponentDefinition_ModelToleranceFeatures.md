# SheetMetalComponentDefinition.ModelToleranceFeatures Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Returns the ModelToleranceFeatures collection object. This object provides access to all of the model annotations in the part.

## Syntax

SheetMetalComponentDefinition.**ModelToleranceFeatures**() As [ModelToleranceFeatures](../ModelToleranceFeatures/ModelToleranceFeatures.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeatures](../ModelToleranceFeatures/ModelToleranceFeatures.md).

## Version

Introduced in version 2019
