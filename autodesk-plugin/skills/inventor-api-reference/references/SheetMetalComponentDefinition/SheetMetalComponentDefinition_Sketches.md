# SheetMetalComponentDefinition.Sketches Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Gets the PlanarSketches collection object that encapsulates all of the planar sketches defined in this ComponentDefinition.

## Syntax

SheetMetalComponentDefinition.**Sketches**() As [PlanarSketches](../PlanarSketches/PlanarSketches.md)

## Property Value

This is a read only property whose value is a [PlanarSketches](../PlanarSketches/PlanarSketches.md).

## Version

Introduced in version 5.3
