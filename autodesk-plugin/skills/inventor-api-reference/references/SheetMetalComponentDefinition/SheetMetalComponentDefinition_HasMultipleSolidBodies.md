# SheetMetalComponentDefinition.HasMultipleSolidBodies Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Determines whether the part contains multiple solid bodies or not.

## Syntax

SheetMetalComponentDefinition.**HasMultipleSolidBodies**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
