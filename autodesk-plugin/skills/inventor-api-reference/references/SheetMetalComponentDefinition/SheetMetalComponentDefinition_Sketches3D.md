# SheetMetalComponentDefinition.Sketches3D Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the Sketches3D collection object that encapsulates all of the 3D sketches defined in this ComponentDefinition.

## Syntax

SheetMetalComponentDefinition.**Sketches3D**() As [Sketches3D](../Sketches3D/Sketches3D.md)

## Property Value

This is a read only property whose value is a [Sketches3D](../Sketches3D/Sketches3D.md).

## Version

Introduced in version 6
