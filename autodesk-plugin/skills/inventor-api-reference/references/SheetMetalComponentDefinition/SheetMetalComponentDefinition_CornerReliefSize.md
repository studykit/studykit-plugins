# SheetMetalComponentDefinition.CornerReliefSize Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the parameter that controls the corner relief size of the part.

## Syntax

SheetMetalComponentDefinition.**CornerReliefSize**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
