# SheetMetalComponentDefinition.UnsuppressFeatures Method

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Method that unsuppresses the specified features.

## Syntax

SheetMetalComponentDefinition.**UnsuppressFeatures**( ***Features*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Features | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Collection of features to unsuppress. |

## Version

Introduced in version 2009
