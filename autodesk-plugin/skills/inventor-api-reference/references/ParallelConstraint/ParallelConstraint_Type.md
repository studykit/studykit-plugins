# ParallelConstraint.Type Property

Parent Object: [ParallelConstraint](../ParallelConstraint/ParallelConstraint.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ParallelConstraint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
