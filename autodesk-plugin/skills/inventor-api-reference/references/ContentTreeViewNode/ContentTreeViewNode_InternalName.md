# ContentTreeViewNode.InternalName Property

Parent Object: [ContentTreeViewNode](../ContentTreeViewNode/ContentTreeViewNode.md)

## Description

Property that returns the internal name of this node. This will be consistent regardless of the language version of Inventor.

## Syntax

ContentTreeViewNode.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2010
