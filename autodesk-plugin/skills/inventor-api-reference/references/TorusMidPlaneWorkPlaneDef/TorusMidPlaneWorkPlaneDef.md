# TorusMidPlaneWorkPlaneDef Object

## Description

Object that allows you to get and set the information that specifies a work plane defined to be at the mid-plane of a torus.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TorusMidPlaneWorkPlaneDef/TorusMidPlaneWorkPlaneDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Face](../TorusMidPlaneWorkPlaneDef/TorusMidPlaneWorkPlaneDef_Face.md) | Property that indicates the Face object whose geometry is a torus surface, the mid-plane of which defines the position of the work plane. |
| [Parent](../TorusMidPlaneWorkPlaneDef/TorusMidPlaneWorkPlaneDef_Parent.md) | Property that returns the parent WorkPlane object. |
| [Type](../TorusMidPlaneWorkPlaneDef/TorusMidPlaneWorkPlaneDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2008
