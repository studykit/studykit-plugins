# DWGEntityArcSegment.Geometry Property

Parent Object: [DWGEntityArcSegment](../DWGEntityArcSegment/DWGEntityArcSegment.md)

## Description

Read-only property that returns the segment geometry.

## Syntax

DWGEntityArcSegment.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
