# ChainDimensionSet.Style Property

Parent Object: [ChainDimensionSet](../ChainDimensionSet/ChainDimensionSet.md)

## Description

Read-write property that gets and sets the dimension style used for this dimension set.

## Syntax

ChainDimensionSet.**Style**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 2011
