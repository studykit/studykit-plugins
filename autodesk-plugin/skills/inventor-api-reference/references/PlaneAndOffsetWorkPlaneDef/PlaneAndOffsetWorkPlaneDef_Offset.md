# PlaneAndOffsetWorkPlaneDef.Offset Property

Parent Object: [PlaneAndOffsetWorkPlaneDef](../PlaneAndOffsetWorkPlaneDef/PlaneAndOffsetWorkPlaneDef.md)

## Description

Property that returns the parameter controlling the offset of the work plane. The offset of the plane can be read and modified by accessing the returned Parameter object.

## Syntax

PlaneAndOffsetWorkPlaneDef.**Offset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 4
