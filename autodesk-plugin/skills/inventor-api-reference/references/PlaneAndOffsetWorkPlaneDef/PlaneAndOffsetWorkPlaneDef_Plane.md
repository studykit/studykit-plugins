# PlaneAndOffsetWorkPlaneDef.Plane Property

Parent Object: [PlaneAndOffsetWorkPlaneDef](../PlaneAndOffsetWorkPlaneDef/PlaneAndOffsetWorkPlaneDef.md)

## Description

Property that returns the plane of an offset work plane. This object can be a planar Face, WorkPlane, or Sketch object. The work plane will be parallel to this plane.

## Syntax

PlaneAndOffsetWorkPlaneDef.**Plane**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
