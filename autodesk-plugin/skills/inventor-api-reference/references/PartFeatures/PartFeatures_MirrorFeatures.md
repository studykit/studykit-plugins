# PartFeatures.MirrorFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the MirrorFeatures collection object. This collection provides access to existing MirrorFeature objects and provides functionality to create new MirrorFeature objects.

## Syntax

PartFeatures.**MirrorFeatures**() As [MirrorFeatures](../MirrorFeatures/MirrorFeatures.md)

## Property Value

This is a read only property whose value is a [MirrorFeatures](../MirrorFeatures/MirrorFeatures.md).

## Version

Introduced in version 5
