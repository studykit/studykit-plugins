# PartFeatures.FaceDraftFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the FaceDraftFeatures collection object. This collection provides access to existing FaceDraftFeature objects.

## Syntax

PartFeatures.**FaceDraftFeatures**() As [FaceDraftFeatures](../FaceDraftFeatures/FaceDraftFeatures.md)

## Property Value

This is a read only property whose value is a [FaceDraftFeatures](../FaceDraftFeatures/FaceDraftFeatures.md).

## Version

Introduced in version 5
