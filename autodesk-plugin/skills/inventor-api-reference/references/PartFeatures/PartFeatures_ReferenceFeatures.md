# PartFeatures.ReferenceFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the ReferenceFeatures collection object.

## Syntax

PartFeatures.**ReferenceFeatures**() As [ReferenceFeatures](../ReferenceFeatures/ReferenceFeatures.md)

## Property Value

This is a read only property whose value is a [ReferenceFeatures](../ReferenceFeatures/ReferenceFeatures.md).

## Version

Introduced in version 5.3
