# PartFeatures.UnwrapFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are UnwrapFeatures.

## Syntax

PartFeatures.**UnwrapFeatures**() As [UnwrapFeatures](../UnwrapFeatures/UnwrapFeatures.md)

## Property Value

This is a read only property whose value is a [UnwrapFeatures](../UnwrapFeatures/UnwrapFeatures.md).

## Version

Introduced in version 2020
