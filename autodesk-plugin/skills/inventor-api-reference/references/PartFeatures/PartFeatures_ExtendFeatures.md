# PartFeatures.ExtendFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the ExtendFeatures collection object. This collection provides access to existing ExtendFeature objects and provides functionality to create new ExtendFeature objects.

## Syntax

PartFeatures.**ExtendFeatures**() As [ExtendFeatures](../ExtendFeatures/ExtendFeatures.md)

## Property Value

This is a read only property whose value is an [ExtendFeatures](../ExtendFeatures/ExtendFeatures.md).

## Version

Introduced in version 11
