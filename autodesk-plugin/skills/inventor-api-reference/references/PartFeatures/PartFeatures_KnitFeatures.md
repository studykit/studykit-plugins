# PartFeatures.KnitFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the KnitFeatures collection object. This collection provides access to existing KnitFeature objects.

## Syntax

PartFeatures.**KnitFeatures**() As [KnitFeatures](../KnitFeatures/KnitFeatures.md)

## Property Value

This is a read only property whose value is a [KnitFeatures](../KnitFeatures/KnitFeatures.md).
