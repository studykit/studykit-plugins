# PartFeatures.SnapFitFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are SnapFitFeatures.

## Syntax

PartFeatures.**SnapFitFeatures**() As [SnapFitFeatures](../SnapFitFeatures/SnapFitFeatures.md)

## Property Value

This is a read only property whose value is a [SnapFitFeatures](../SnapFitFeatures/SnapFitFeatures.md).

## Version

Introduced in version 2010
