# PartFeatures.BossFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are BossFeatures.

## Syntax

PartFeatures.**BossFeatures**() As [BossFeatures](../BossFeatures/BossFeatures.md)

## Property Value

This is a read only property whose value is a [BossFeatures](../BossFeatures/BossFeatures.md).

## Version

Introduced in version 2010
