# PartFeatures.iFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the iFeatures collection object.

## Syntax

PartFeatures.**iFeatures**() As [iFeatures](../iFeatures/iFeatures.md)

## Property Value

This is a read only property whose value is an [iFeatures](../iFeatures/iFeatures.md).
