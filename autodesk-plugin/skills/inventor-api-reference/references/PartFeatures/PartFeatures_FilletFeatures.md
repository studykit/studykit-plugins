# PartFeatures.FilletFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the FilletFeatures collection object. This collection provides access to existing FilletFeature objects and provides functionality to create new FilletFeature objects.

## Syntax

PartFeatures.**FilletFeatures**() As [FilletFeatures](../FilletFeatures/FilletFeatures.md)

## Property Value

This is a read only property whose value is a [FilletFeatures](../FilletFeatures/FilletFeatures.md).
