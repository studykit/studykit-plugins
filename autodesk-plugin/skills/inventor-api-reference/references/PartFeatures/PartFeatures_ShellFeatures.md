# PartFeatures.ShellFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the ShellFeatures collection object. This collection provides access to existing ShellFeature objects.

## Syntax

PartFeatures.**ShellFeatures**() As [ShellFeatures](../ShellFeatures/ShellFeatures.md)

## Property Value

This is a read only property whose value is a [ShellFeatures](../ShellFeatures/ShellFeatures.md).

## Version

Introduced in version 5
