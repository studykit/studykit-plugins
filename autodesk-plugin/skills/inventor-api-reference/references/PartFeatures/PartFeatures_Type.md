# PartFeatures.Type Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

PartFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 4
