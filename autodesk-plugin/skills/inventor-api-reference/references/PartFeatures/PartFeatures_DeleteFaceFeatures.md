# PartFeatures.DeleteFaceFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the DeleteFaceFeatures collection object. This collection provides access to existing DeleteFaceFeature objects.

## Syntax

PartFeatures.**DeleteFaceFeatures**() As [DeleteFaceFeatures](../DeleteFaceFeatures/DeleteFaceFeatures.md)

## Property Value

This is a read only property whose value is a [DeleteFaceFeatures](../DeleteFaceFeatures/DeleteFaceFeatures.md).
