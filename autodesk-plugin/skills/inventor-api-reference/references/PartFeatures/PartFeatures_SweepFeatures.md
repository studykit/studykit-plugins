# PartFeatures.SweepFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the SweepFeatures collection object. This collection provides access to existing SweepFeature objects and provides functionality to create new SweepFeature objects.

## Syntax

PartFeatures.**SweepFeatures**() As [SweepFeatures](../SweepFeatures/SweepFeatures.md)

## Property Value

This is a read only property whose value is a [SweepFeatures](../SweepFeatures/SweepFeatures.md).

## Version

Introduced in version 5
