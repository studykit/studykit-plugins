# PartFeatures.SculptFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the SculptFeatures collection object. This collection provides access to existing SculptFeature objects and provides functionality to create new SculptFeature objects.

## Syntax

PartFeatures.**SculptFeatures**() As [SculptFeatures](../SculptFeatures/SculptFeatures.md)

## Property Value

This is a read only property whose value is a [SculptFeatures](../SculptFeatures/SculptFeatures.md).

## Version

Introduced in version 11
