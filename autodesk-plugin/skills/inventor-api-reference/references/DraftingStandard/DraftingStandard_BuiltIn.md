# DraftingStandard.BuiltIn Property

Parent Object: [DraftingStandard](../DraftingStandard/DraftingStandard.md)

## Description

Gets the Boolean indicating if this object represents a built-in drafting standard. If FALSE, then this object represents a standard that has been customized.

## Syntax

DraftingStandard.**BuiltIn**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
