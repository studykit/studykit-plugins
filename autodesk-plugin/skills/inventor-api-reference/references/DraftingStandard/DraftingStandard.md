# DraftingStandard Object

## Description

The DraftingStandard object represent the currently applied drafting standard, and indicates whether the standard has been customized.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [BaseStandard](../DraftingStandard/DraftingStandard_BaseStandard.md) | Gets the base drafting standard associated with this particular object. |
| [BuiltIn](../DraftingStandard/DraftingStandard_BuiltIn.md) | Gets the Boolean indicating if this object represents a built-in drafting standard. If FALSE, then this object represents a standard that has been customized. |
| [Name](../DraftingStandard/DraftingStandard_Name.md) | Property that indicates the displayable name of this standard. No 'Set' allowed in Apprentice. |
| [Type](../DraftingStandard/DraftingStandard_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ApprenticeServerDrawingDocument.ActiveStandard](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument_ActiveStandard.md)

## Version

Introduced in version 4
