# DraftingStandard.Name Property

Parent Object: [DraftingStandard](../DraftingStandard/DraftingStandard.md)

## Description

Property that indicates the displayable name of this standard. No 'Set' allowed in Apprentice.

## Syntax

DraftingStandard.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
