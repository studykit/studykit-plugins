# AssetCategory.Type Property

Parent Object: [AssetCategory](../AssetCategory/AssetCategory.md)

## Description

Read-only property returning kAssetCategoryObject indicating this object’s type.

## Syntax

AssetCategory.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
