# AssetCategory.Parent Property

Parent Object: [AssetCategory](../AssetCategory/AssetCategory.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

AssetCategory.**Parent**() As [AssetLibrary](../AssetLibrary/AssetLibrary.md)

## Property Value

This is a read only property whose value is an [AssetLibrary](../AssetLibrary/AssetLibrary.md).

## Version

Introduced in version 2014
