# SketchEllipse.LineType Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Gets and sets the LineType applied to this sketch line.

## Syntax

SketchEllipse.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 9
