# SketchEllipse.Area Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Double property that returns the area of the entity in square centimeters.

## Syntax

SketchEllipse.**Area**() As Double

## Property Value

This is a read only property whose value is a Double.
