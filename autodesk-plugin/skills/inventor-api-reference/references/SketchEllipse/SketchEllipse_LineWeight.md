# SketchEllipse.LineWeight Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Gets and sets the LineWeight applied to this sketch line.

## Syntax

SketchEllipse.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
