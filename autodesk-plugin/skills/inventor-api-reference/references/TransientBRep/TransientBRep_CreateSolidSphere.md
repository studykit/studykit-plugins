# TransientBRep.CreateSolidSphere Method

Parent Object: [TransientBRep](../TransientBRep/TransientBRep.md)

## Description

Method that create a solid sphere.

## Remarks

The SurfaceBody containing the sphere is returned.

## Syntax

TransientBRep.**CreateSolidSphere**( ***Center*** As [Point](../Point/Point.md), ***Radius*** As Double ) As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Center | [Point](../Point/Point.md) | Input Point that defines the center of the sphere. |
| Radius | Double | Input Point Double that defines the radius of the sphere in centimeters |

## Version

Introduced in version 2009
