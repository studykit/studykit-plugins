# EdgeUseDefinitions.Add Method

Parent Object: [EdgeUseDefinitions](../EdgeUseDefinitions/EdgeUseDefinitions.md)

## Description

Method that creates a new EdgeUseDefinition within the associated EdgeLoopDefinition.

## Syntax

EdgeUseDefinitions.**Add**( ***EdgeDefinition*** As [EdgeDefinition](../EdgeDefinition/EdgeDefinition.md), ***IsOpposedToEdge*** As Boolean ) As [EdgeUseDefinition](../EdgeUseDefinition/EdgeUseDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| EdgeDefinition | [EdgeDefinition](../EdgeDefinition/EdgeDefinition.md) | Input EdgeDefinition that defines the geometry for this EdgeUseDefinition. |
| IsOpposedToEdge | Boolean | Input Boolean that indicates if the parameterization of this EdgeUseDefinition is reversed with respect to the associated EdgeDefinition object. |

## Version

Introduced in version 2011
