# EdgeUseDefinitions Object

## Description

The EdgeUseDefinitions collection object provides access to existing EdgeUseDefinition objects, and allows creation of new EdgeUseDefinition objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../EdgeUseDefinitions/EdgeUseDefinitions_Add.md) | Method that creates a new EdgeUseDefinition within the associated EdgeLoopDefinition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../EdgeUseDefinitions/EdgeUseDefinitions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../EdgeUseDefinitions/EdgeUseDefinitions_Count.md) | Property that returns the number of items in this collection. |
| [Item](../EdgeUseDefinitions/EdgeUseDefinitions_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../EdgeUseDefinitions/EdgeUseDefinitions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[EdgeLoopDefinition.EdgeUseDefinitions](../EdgeLoopDefinition/EdgeLoopDefinition_EdgeUseDefinitions.md)

## Version

Introduced in version 2011
