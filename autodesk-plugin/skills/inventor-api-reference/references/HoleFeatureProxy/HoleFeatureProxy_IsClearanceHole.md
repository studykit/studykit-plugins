# HoleFeatureProxy.IsClearanceHole Property

Parent Object: [HoleFeatureProxy](../HoleFeatureProxy/HoleFeatureProxy.md)

## Description

Read – only property that returns the Boolean flag indicating whether the hole is clearance hole or not.

## Syntax

HoleFeatureProxy.**IsClearanceHole**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
