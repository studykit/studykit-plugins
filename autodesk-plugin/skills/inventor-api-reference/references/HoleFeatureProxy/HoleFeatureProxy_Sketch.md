# HoleFeatureProxy.Sketch Property

Parent Object: [HoleFeatureProxy](../HoleFeatureProxy/HoleFeatureProxy.md)

## Description

Property that returns the Sketch object that contains the points the hole is based on.

## Syntax

HoleFeatureProxy.**Sketch**() As [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Property Value

This is a read only property whose value is a [PlanarSketch](../PlanarSketch/PlanarSketch.md).

## Version

Introduced in version 9
