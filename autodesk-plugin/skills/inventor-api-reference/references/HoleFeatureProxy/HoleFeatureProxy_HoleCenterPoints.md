# HoleFeatureProxy.HoleCenterPoints Property

Parent Object: [HoleFeatureProxy](../HoleFeatureProxy/HoleFeatureProxy.md)

## Description

Property that returns the set of points that define the centers of the holes of this feature.

## Syntax

HoleFeatureProxy.**HoleCenterPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 9
