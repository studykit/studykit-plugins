# HoleFeatureProxy.DrillPointType Property

Parent Object: [HoleFeatureProxy](../HoleFeatureProxy/HoleFeatureProxy.md)

## Description

Gets and sets hole drill point type.

## Syntax

HoleFeatureProxy.**DrillPointType**() As [HoleDrillPointTypeEnum](../HoleDrillPointTypeEnum.md)

## Property Value

This is a read/write property whose value is a [HoleDrillPointTypeEnum](../HoleDrillPointTypeEnum.md).

## Version

Introduced in version 2025
