# OffsetSplineDimConstraint.Spline Property

Parent Object: [OffsetSplineDimConstraint](../OffsetSplineDimConstraint/OffsetSplineDimConstraint.md)

## Description

Gets the SketchSpline being constrained.

## Syntax

OffsetSplineDimConstraint.**Spline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Property Value

This is a read only property whose value is a [SketchSpline](../SketchSpline/SketchSpline.md).

## Version

Introduced in version 11
