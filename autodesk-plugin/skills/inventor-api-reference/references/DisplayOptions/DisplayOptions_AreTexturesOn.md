# DisplayOptions.AreTexturesOn Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets and sets whether to turn textures on.

## Syntax

DisplayOptions.**AreTexturesOn**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
