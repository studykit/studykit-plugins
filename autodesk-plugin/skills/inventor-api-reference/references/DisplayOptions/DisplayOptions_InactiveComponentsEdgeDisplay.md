# DisplayOptions.InactiveComponentsEdgeDisplay Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that specifies whether edges should be displayed for inactive components.

## Syntax

DisplayOptions.**InactiveComponentsEdgeDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
