# DisplayOptions.UseDocumentDisplaySettings Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that gets and sets whether to use the display appearance settings defined in the document. If set to False, the application level display settings are used.

## Syntax

DisplayOptions.**UseDocumentDisplaySettings**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
