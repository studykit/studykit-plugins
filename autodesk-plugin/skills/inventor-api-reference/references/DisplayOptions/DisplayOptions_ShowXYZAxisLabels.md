# DisplayOptions.ShowXYZAxisLabels Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets and sets whether to show axis labels for the 3D indicator (triad).

## Syntax

DisplayOptions.**ShowXYZAxisLabels**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
