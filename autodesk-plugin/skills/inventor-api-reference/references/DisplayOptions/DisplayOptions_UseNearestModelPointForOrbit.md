# DisplayOptions.UseNearestModelPointForOrbit Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets whether to use the nearest model point to the mouse position as orbit pivot.

## Syntax

DisplayOptions.**UseNearestModelPointForOrbit**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
