# DisplayOptions.SolidLinesForHiddenEdges Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets whether the hidden model edges should be displayed as solid lines.

## Syntax

DisplayOptions.**SolidLinesForHiddenEdges**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
