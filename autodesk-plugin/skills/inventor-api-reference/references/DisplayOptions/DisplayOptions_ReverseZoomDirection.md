# DisplayOptions.ReverseZoomDirection Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets the zoom direction with respect to the movement of the mouse cursor or mouse wheel.

## Syntax

DisplayOptions.**ReverseZoomDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
