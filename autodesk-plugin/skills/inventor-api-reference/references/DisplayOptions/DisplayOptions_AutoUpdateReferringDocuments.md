# DisplayOptions.AutoUpdateReferringDocuments Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Enable/Disable automatically update referring documents.

## Syntax

DisplayOptions.**AutoUpdateReferringDocuments**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
