# DisplayOptions.Type Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DisplayOptions.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
