# Box2d.IsDisjoint Method

Parent Object: [Box2d](../Box2d/Box2d.md)

## Description

Determines whether this Box intersects the specified Box. A return value of True indicates that the box do not intersect.

## Syntax

Box2d.**IsDisjoint**( ***Box*** As [Box2d](../Box2d/Box2d.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Box | [Box2d](../Box2d/Box2d.md) | Input Box2d object to compare. |

## Version

Introduced in version 2008
