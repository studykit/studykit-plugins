# Box2d.Contains Method

Parent Object: [Box2d](../Box2d/Box2d.md)

## Description

Determines whether the specified point is contained within this Box.

## Syntax

Box2d.**Contains**( ***Point*** As [Point2d](../Point2d/Point2d.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point2d](../Point2d/Point2d.md) | Input Point2d object that specifies the coordinate to check. |

## Version

Introduced in version 2008
