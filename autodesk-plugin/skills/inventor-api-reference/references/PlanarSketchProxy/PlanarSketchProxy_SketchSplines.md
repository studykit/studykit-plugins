# PlanarSketchProxy.SketchSplines Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that returns the SketchSplines collection object.

## Syntax

PlanarSketchProxy.**SketchSplines**() As [SketchSplines](../SketchSplines/SketchSplines.md)

## Property Value

This is a read only property whose value is a [SketchSplines](../SketchSplines/SketchSplines.md).

## Version

Introduced in version 6
