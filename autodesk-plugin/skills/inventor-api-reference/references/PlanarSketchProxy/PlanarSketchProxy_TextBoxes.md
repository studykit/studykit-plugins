# PlanarSketchProxy.TextBoxes Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Gets the TextBoxes collection associated with this Sketch.

## Syntax

PlanarSketchProxy.**TextBoxes**() As [TextBoxes](../TextBoxes/TextBoxes.md)

## Property Value

This is a read only property whose value is a [TextBoxes](../TextBoxes/TextBoxes.md).

## Version

Introduced in version 10
