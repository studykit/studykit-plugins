# PlanarSketchProxy.Profiles Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that returns the Profiles collection object.

## Syntax

PlanarSketchProxy.**Profiles**() As [Profiles](../Profiles/Profiles.md)

## Property Value

This is a read only property whose value is a [Profiles](../Profiles/Profiles.md).

## Version

Introduced in version 6
