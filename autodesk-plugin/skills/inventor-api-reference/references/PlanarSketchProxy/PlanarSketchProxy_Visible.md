# PlanarSketchProxy.Visible Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Gets and sets the visibility of the sketch.

## Syntax

PlanarSketchProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
