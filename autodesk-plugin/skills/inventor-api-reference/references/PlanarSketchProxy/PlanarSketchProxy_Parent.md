# PlanarSketchProxy.Parent Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that gets the parent object from whom this object can logically be reached.

## Syntax

PlanarSketchProxy.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
