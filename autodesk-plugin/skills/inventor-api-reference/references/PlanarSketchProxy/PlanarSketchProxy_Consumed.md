# PlanarSketchProxy.Consumed Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Gets whether the sketch is consumed or not.

## Syntax

PlanarSketchProxy.**Consumed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 8
