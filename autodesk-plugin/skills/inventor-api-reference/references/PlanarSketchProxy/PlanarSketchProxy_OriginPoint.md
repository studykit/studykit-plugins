# PlanarSketchProxy.OriginPoint Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Gets and sets origin of the sketch. When set this property, the valid object can be a WorkPoint, Vertex or SketchPoint.

## Syntax

PlanarSketchProxy.**OriginPoint**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 6
