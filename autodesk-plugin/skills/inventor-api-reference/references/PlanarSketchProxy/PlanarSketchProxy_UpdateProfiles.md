# PlanarSketchProxy.UpdateProfiles Method

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Method that updates all the profiles within the sketch.

## Syntax

PlanarSketchProxy.**UpdateProfiles**()
