# PlanarSketchProxy.DisabledActionTypes Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Gets and sets the action types valid for this sketch.

## Syntax

PlanarSketchProxy.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 9
