# PlanarSketchProxy.ModelToSketchTransform Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that returns the transformation from model space to the 2d sketch coordinate space.

## Syntax

PlanarSketchProxy.**ModelToSketchTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 6
