# PlanarSketchProxy.AxisIsX Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Gets and sets if the axis entity defines the X or Y axis. True indicates the axis defines the X-axis.

## Syntax

PlanarSketchProxy.**AxisIsX**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
