# PlanarSketchProxy.OriginPointGeometry Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that gets the geometry that describes the origin point.

## Syntax

PlanarSketchProxy.**OriginPointGeometry**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
