# DimensionConstraint3D.AnchorPoints Property

Parent Object: [DimensionConstraint3D](../DimensionConstraint3D/DimensionConstraint3D.md)

## Description

Gets the anchor points of dimension.

## Syntax

DimensionConstraint3D.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
