# DimensionConstraint3D.Type Property

Parent Object: [DimensionConstraint3D](../DimensionConstraint3D/DimensionConstraint3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DimensionConstraint3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
