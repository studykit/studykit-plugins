# GraphicsNodeProxy.AddTriangleStripGraphics Method

Parent Object: [GraphicsNodeProxy](../GraphicsNodeProxy/GraphicsNodeProxy.md)

## Description

Method that creates a new TriangleStripGraphics graphic primitive.

## Syntax

GraphicsNodeProxy.**AddTriangleStripGraphics**() As [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Version
