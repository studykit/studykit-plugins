# GraphicsNodeProxy.Transformation Property

Parent Object: [GraphicsNodeProxy](../GraphicsNodeProxy/GraphicsNodeProxy.md)

## Description

Property that gets and sets the transformation of the GraphicsNode.

## Syntax

GraphicsNodeProxy.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2008
