# GraphicsNodeProxy.Selectable Property

Parent Object: [GraphicsNodeProxy](../GraphicsNodeProxy/GraphicsNodeProxy.md)

## Description

Property that specifies whether the GraphicsNode object can be selected when the Select command is running in Inventor.

## Syntax

GraphicsNodeProxy.**Selectable**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
