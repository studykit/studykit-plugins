# SectionViewTypeEnum Enumerator

## Description

Defines the types of the section views in part or assembly.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kHalfSectionViewType | 110850 | Half section view type. |
| kNoSectionViewType | 110852 | No section view type. |
| kQuarterSectionViewType | 110849 | Quarter section view type. |
| kThreeQuarterSectionViewType | 110851 | Three quarter section view type. |
