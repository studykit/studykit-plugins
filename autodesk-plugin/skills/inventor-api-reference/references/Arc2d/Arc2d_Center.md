# Arc2d.Center Property

Parent Object: [Arc2d](../Arc2d/Arc2d.md)

## Description

Specifies the center point for this arc.

## Syntax

Arc2d.**Center**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
