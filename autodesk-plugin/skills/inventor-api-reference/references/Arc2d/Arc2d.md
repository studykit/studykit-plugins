# Arc2d Object

## Description

The Arc2d object is a mathematical representation of a 2d arc.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Arc2d/Arc2d_Copy.md) | Creates a copy of this Arc2d object. The result is entirely independent and can be edited without affecting the original Arc2d object. |
| [GetArcData](../Arc2d/Arc2d_GetArcData.md) | Get the data defining this arc. |
| [PutArcData](../Arc2d/Arc2d_PutArcData.md) | Method that sets the data defining this arc. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Center](../Arc2d/Arc2d_Center.md) | Specifies the center point for this arc. |
| [EndPoint](../Arc2d/Arc2d_EndPoint.md) | Property that returns the end point of the arc. |
| [Evaluator](../Arc2d/Arc2d_Evaluator.md) | Property that returns the Curve2dEvaluator for this arc. |
| [MidPoint](../Arc2d/Arc2d_MidPoint.md) | Read-only property that returns the mid point of the arc. |
| [Radius](../Arc2d/Arc2d_Radius.md) | Specifies the radius for this arc. |
| [StartAngle](../Arc2d/Arc2d_StartAngle.md) | Specifies the starting angle for this arc. |
| [StartPoint](../Arc2d/Arc2d_StartPoint.md) | Property that gets the start point of the arc. |
| [SweepAngle](../Arc2d/Arc2d_SweepAngle.md) | Specifies the sweep angle from the start angle for this arc. |
