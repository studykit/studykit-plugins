# ModelDatumIdentifierDefinition.Parent Property

Parent Object: [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelDatumIdentifierDefinition.**Parent**() As [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md)

## Property Value

This is a read only property whose value is a [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md).

## Version

Introduced in version 2018
