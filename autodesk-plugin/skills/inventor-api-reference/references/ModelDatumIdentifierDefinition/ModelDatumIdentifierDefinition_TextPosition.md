# ModelDatumIdentifierDefinition.TextPosition Property

Parent Object: [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md)

## Description

Read-write property that gets and sets the text position of the symbol in the part or assembly. The point is projected onto the orientation plane.

## Syntax

ModelDatumIdentifierDefinition.**TextPosition**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
