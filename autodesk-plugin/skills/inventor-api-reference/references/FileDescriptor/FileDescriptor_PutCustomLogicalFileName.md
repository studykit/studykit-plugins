# FileDescriptor.PutCustomLogicalFileName Method

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

This method does the posting of the Custom Logical Filename referred to in GetCustomLogicalFileName.

## Remarks

Typically, this method is rarely used, since the initialization takes place during the insert of a file into container documents such as an assembly or drawing. Not supported when this description is of an OLE attachment.

## Syntax

FileDescriptor.**PutCustomLogicalFileName**( ***Value***() As Byte )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | Byte |  |

## Version

Introduced in version 11
