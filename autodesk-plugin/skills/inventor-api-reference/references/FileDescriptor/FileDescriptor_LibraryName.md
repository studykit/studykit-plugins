# FileDescriptor.LibraryName Property

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

Gets the name of the library this reference was resolved in. This property is only valid when ReferenceMissing is false.

## Syntax

FileDescriptor.**LibraryName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2008
