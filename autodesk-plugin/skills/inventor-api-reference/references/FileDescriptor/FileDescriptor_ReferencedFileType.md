# FileDescriptor.ReferencedFileType Property

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

Property that returns type of the referenced file.

## Syntax

FileDescriptor.**ReferencedFileType**() As [FileTypeEnum](../FileTypeEnum.md)

## Property Value

This is a read only property whose value is a [FileTypeEnum](../FileTypeEnum.md).

## Version

Introduced in version 11
