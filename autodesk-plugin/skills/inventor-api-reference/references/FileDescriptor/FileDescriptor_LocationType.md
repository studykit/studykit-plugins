# FileDescriptor.LocationType Property

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

Gets the type of location this reference was resolved in. This property is only valid when ReferenceMissing is false.

## Syntax

FileDescriptor.**LocationType**() As [LocationTypeEnum](../LocationTypeEnum.md)

## Property Value

This is a read only property whose value is a [LocationTypeEnum](../LocationTypeEnum.md).

## Version

Introduced in version 2008
