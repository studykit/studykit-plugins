# SurfaceBody.AffectedByFeatures Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Property that returns the features that affected this body. The returned enumerator includes the feature that created this body. This property returns Nothing for transient bodies

## Syntax

SurfaceBody.**AffectedByFeatures**() As [PartFeaturesEnumerator](../PartFeaturesEnumerator/PartFeaturesEnumerator.md)

## Property Value

This is a read only property whose value is a [PartFeaturesEnumerator](../PartFeaturesEnumerator/PartFeaturesEnumerator.md).

## Version

Introduced in version 2010
