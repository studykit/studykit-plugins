# SurfaceBody.IsTransient Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Property that specified if this SurfaceBody is transient or not.

## Syntax

SurfaceBody.**IsTransient**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
