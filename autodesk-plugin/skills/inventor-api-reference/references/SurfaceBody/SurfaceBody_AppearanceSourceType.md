# SurfaceBody.AppearanceSourceType Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Gets and sets the source of the appearance for the body.

## Syntax

SurfaceBody.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).
