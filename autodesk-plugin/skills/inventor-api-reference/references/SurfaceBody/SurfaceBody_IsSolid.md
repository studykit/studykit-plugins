# SurfaceBody.IsSolid Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Determine if this SurfaceBody is solid.

## Syntax

SurfaceBody.**IsSolid**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
