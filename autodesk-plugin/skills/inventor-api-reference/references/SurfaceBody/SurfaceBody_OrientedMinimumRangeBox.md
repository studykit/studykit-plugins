# SurfaceBody.OrientedMinimumRangeBox Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Retrieves the oriented minimum range box for this object.

## Syntax

SurfaceBody.**OrientedMinimumRangeBox**() As [OrientedBox](../OrientedBox/OrientedBox.md)

## Property Value

This is a read only property whose value is an [OrientedBox](../OrientedBox/OrientedBox.md).

## Version

Introduced in version 2021
