# SurfaceBody.MassProperties Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Read-only property that returns mass properties for this body.

## Syntax

SurfaceBody.**MassProperties**() As [SurfaceBodyMassProperties](../SurfaceBodyMassProperties/SurfaceBodyMassProperties.md)

## Property Value

This is a read only property whose value is a [SurfaceBodyMassProperties](../SurfaceBodyMassProperties/SurfaceBodyMassProperties.md).

## Version

Introduced in version 2023
