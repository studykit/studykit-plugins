# DerivedPartUniformScaleDef.Surfaces Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Property that gets the collection of surfaces available in the selected part along with the inclusion option for each surface.

## Syntax

DerivedPartUniformScaleDef.**Surfaces**() As [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md)

## Property Value

This is a read only property whose value is a [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md).

## Version

Introduced in version 6
