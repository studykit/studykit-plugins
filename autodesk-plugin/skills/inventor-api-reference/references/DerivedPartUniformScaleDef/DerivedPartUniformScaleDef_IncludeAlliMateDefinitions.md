# DerivedPartUniformScaleDef.IncludeAlliMateDefinitions Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Gets and sets whether all iMates are included in the derived part.

## Syntax

DerivedPartUniformScaleDef.**IncludeAlliMateDefinitions**() As [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md).

## Version

Introduced in version 2010
