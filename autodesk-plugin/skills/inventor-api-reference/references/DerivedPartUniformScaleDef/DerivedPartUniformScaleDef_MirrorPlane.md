# DerivedPartUniformScaleDef.MirrorPlane Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Property that gets and sets the plane about which the base part will be mirrored to create the derived part. This property is initialized to kDerivedPartNoMirrorPlane when the DerivedPartDefinition object is created.

## Syntax

DerivedPartUniformScaleDef.**MirrorPlane**() As [DerivedPartMirrorPlaneEnum](../DerivedPartMirrorPlaneEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedPartMirrorPlaneEnum](../DerivedPartMirrorPlaneEnum.md).

## Version

Introduced in version 6
