# DerivedPartUniformScaleDef.WorkFeatures Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Property that returns the collection of work features available in the selected part along with the inclusion option for each work feature.

## Syntax

DerivedPartUniformScaleDef.**WorkFeatures**() As [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md)

## Property Value

This is a read only property whose value is a [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md).

## Version

Introduced in version 6
