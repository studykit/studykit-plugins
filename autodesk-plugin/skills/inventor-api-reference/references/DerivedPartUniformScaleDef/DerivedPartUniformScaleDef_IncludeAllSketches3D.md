# DerivedPartUniformScaleDef.IncludeAllSketches3D Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Gets or sets whether all sketches3D are included in the derived part.

## Syntax

DerivedPartUniformScaleDef.**IncludeAllSketches3D**() As [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md).

## Version

Introduced in version 11
