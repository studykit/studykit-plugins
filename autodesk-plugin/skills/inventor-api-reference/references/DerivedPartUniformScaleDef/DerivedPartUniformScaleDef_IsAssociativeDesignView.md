# DerivedPartUniformScaleDef.IsAssociativeDesignView Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Read-write property that gets and sets if there is an associative link to the specified design view.

## Syntax

DerivedPartUniformScaleDef.**IsAssociativeDesignView**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
