# DerivedPartUniformScaleDef.IncludeAllParameters Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Property that defines whether exported parameters are included in the derived part. This property is initialized to True when the DerivedPartDefinition object is created.

## Syntax

DerivedPartUniformScaleDef.**IncludeAllParameters**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
