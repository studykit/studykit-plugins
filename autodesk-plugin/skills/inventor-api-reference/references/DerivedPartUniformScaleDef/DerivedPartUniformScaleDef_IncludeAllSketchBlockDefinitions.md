# DerivedPartUniformScaleDef.IncludeAllSketchBlockDefinitions Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Gets or sets whether all sketch block definitions are included in the derived part.

## Syntax

DerivedPartUniformScaleDef.**IncludeAllSketchBlockDefinitions**() As [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md).

## Version

Introduced in version 2010
