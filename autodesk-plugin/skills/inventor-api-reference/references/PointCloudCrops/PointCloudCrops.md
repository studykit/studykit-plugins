# PointCloudCrops Object

## Description

PointCloudCrops Collection Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../PointCloudCrops/PointCloudCrops_Add.md) | Method that adds a crop to the point cloud and the result PointCloudCrop is returned. |
| [Clear](../PointCloudCrops/PointCloudCrops_Clear.md) | Method that clears all the point cloud crops in this collection. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointCloudCrops/PointCloudCrops_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../PointCloudCrops/PointCloudCrops_Count.md) | Gets the number of items in this collection. |
| [Item](../PointCloudCrops/PointCloudCrops_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../PointCloudCrops/PointCloudCrops_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PointCloud.Crops](../PointCloud/PointCloud_Crops.md), [PointCloudProxy.Crops](../PointCloudProxy/PointCloudProxy_Crops.md)

## Version

Introduced in version 2016
