# PointCloudCrops.Clear Method

Parent Object: [PointCloudCrops](../PointCloudCrops/PointCloudCrops.md)

## Description

Method that clears all the point cloud crops in this collection.

## Syntax

PointCloudCrops.**Clear**()

## Version

Introduced in version 2016
