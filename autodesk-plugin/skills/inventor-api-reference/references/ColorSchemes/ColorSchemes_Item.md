# ColorSchemes.Item Property

Parent Object: [ColorSchemes](../ColorSchemes/ColorSchemes.md)

## Description

Returns the specified ColorScheme object from the collection.

## Syntax

ColorSchemes.**Item**( ***Index*** As Variant ) As [ColorScheme](../ColorScheme/ColorScheme.md)

## Property Value

This is a read only property whose value is a [ColorScheme](../ColorScheme/ColorScheme.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the ColorScheme to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the ColorScheme name. If an out of range index or a name of a non-existent ColorScheme is provided, an error will occur. |

## Version

Introduced in version 9
