# ColorSchemes.IconsColor Property

Parent Object: [ColorSchemes](../ColorSchemes/ColorSchemes.md)

## Description

Gets and sets the color theme for the icons in the ribbon interface.

## Syntax

ColorSchemes.**IconsColor**() As [IconsColorTypeEnum](../IconsColorTypeEnum.md)

## Property Value

This is a read/write property whose value is an [IconsColorTypeEnum](../IconsColorTypeEnum.md).

## Version

Introduced in version 2010
