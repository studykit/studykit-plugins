# ColorSchemes.EnableEnhancedHighlighting Property

Parent Object: [ColorSchemes](../ColorSchemes/ColorSchemes.md)

## Description

Gets and sets whether the enhanced highlighting option is enabled.

## Syntax

ColorSchemes.**EnableEnhancedHighlighting**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
