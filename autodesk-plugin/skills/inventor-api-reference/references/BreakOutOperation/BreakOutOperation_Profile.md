# BreakOutOperation.Profile Property

Parent Object: [BreakOutOperation](../BreakOutOperation/BreakOutOperation.md)

## Description

Read-write property that gets and sets the sketch profile used for the break out.

## Syntax

BreakOutOperation.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 2010
