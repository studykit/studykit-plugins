# BreakOutOperation.DisplayComponentsWithinTheSectionCut Property

Parent Object: [BreakOutOperation](../BreakOutOperation/BreakOutOperation.md)

## Description

Read-write property that gets and sets whether or not to display the components within the section cut.

## Syntax

BreakOutOperation.**DisplayComponentsWithinTheSectionCut**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
