# BreakOutOperation.Delete Method

Parent Object: [BreakOutOperation](../BreakOutOperation/BreakOutOperation.md)

## Description

Method that deletes the break out operation from the drawing view. The break out is deleted from all other affected drawing views as well.

## Syntax

BreakOutOperation.**Delete**()

## Version

Introduced in version 2010
