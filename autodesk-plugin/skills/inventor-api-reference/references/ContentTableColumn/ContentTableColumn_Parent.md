# ContentTableColumn.Parent Property

Parent Object: [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md)

## Description

Property that returns the ContentFamily object this column is associated with.

## Syntax

ContentTableColumn.**Parent**() As [ContentFamily](../ContentFamily/ContentFamily.md)

## Property Value

This is a read only property whose value is a [ContentFamily](../ContentFamily/ContentFamily.md).

## Version

Introduced in version 2010
