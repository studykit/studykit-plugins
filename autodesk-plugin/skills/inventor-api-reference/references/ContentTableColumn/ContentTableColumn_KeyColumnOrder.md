# ContentTableColumn.KeyColumnOrder Property

Parent Object: [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md)

## Description

Read-write Property that allows you to get and set the KeyWeight of the column.

## Syntax

ContentTableColumn.**KeyColumnOrder**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2011
