# ContentTableColumn.IsCustom Property

Parent Object: [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md)

## Description

Read-only property that returns True if the column is a custom column.

## Syntax

ContentTableColumn.**IsCustom**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2024
