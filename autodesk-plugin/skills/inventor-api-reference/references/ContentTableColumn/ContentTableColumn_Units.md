# ContentTableColumn.Units Property

Parent Object: [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md)

## Description

Read-write Property that allows you to get and set the units of the column.

## Syntax

ContentTableColumn.**Units**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2010
