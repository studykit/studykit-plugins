# FilletFeature.FilletDefinition Property

Parent Object: [FilletFeature](../FilletFeature/FilletFeature.md)

## Description

Property that returns the object which defines the various input that was used to create the fillet feature.

## Syntax

FilletFeature.**FilletDefinition**() As [FilletDefinition](../FilletDefinition/FilletDefinition.md)

## Property Value

This is a read only property whose value is a [FilletDefinition](../FilletDefinition/FilletDefinition.md).

## Version

Introduced in version 5.3
