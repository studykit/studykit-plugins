# SketchBlockDefinition.SketchPoints Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the SketchPoints collection object.

## Syntax

SketchBlockDefinition.**SketchPoints**() As [SketchPoints](../SketchPoints/SketchPoints.md)

## Property Value

This is a read only property whose value is a [SketchPoints](../SketchPoints/SketchPoints.md).

## Version

Introduced in version 2010
