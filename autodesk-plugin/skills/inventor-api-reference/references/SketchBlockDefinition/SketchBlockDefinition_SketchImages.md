# SketchBlockDefinition.SketchImages Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns a collection of all images on the sketch.

## Syntax

SketchBlockDefinition.**SketchImages**() As [SketchImages](../SketchImages/SketchImages.md)

## Property Value

This is a read only property whose value is a [SketchImages](../SketchImages/SketchImages.md).

## Version

Introduced in version 2010
