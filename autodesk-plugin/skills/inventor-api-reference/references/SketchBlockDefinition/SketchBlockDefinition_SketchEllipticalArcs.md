# SketchBlockDefinition.SketchEllipticalArcs Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the SketchEllipticalArcs collection object.

## Syntax

SketchBlockDefinition.**SketchEllipticalArcs**() As [SketchEllipticalArcs](../SketchEllipticalArcs/SketchEllipticalArcs.md)

## Property Value

This is a read only property whose value is a [SketchEllipticalArcs](../SketchEllipticalArcs/SketchEllipticalArcs.md).

## Version

Introduced in version 2010
