# SketchBlockDefinition.Adaptive Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and sets whether the sketch is adaptive or not.

## Syntax

SketchBlockDefinition.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
