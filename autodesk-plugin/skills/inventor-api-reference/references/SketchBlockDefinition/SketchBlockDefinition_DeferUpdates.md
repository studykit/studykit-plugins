# SketchBlockDefinition.DeferUpdates Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and Sets whether to defer the solving of the sketch or not.

## Syntax

SketchBlockDefinition.**DeferUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
