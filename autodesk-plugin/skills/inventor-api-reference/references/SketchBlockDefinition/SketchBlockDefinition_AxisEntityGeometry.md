# SketchBlockDefinition.AxisEntityGeometry Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that gets the geometry that describes the axis entity.

## Syntax

SketchBlockDefinition.**AxisEntityGeometry**() As [Line](../Line/Line.md)

## Property Value

This is a read only property whose value is a [Line](../Line/Line.md).

## Version

Introduced in version 2010
