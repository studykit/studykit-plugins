# SketchBlockDefinition.Shared Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and sets whether the profile is shared or not.

## Syntax

SketchBlockDefinition.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
