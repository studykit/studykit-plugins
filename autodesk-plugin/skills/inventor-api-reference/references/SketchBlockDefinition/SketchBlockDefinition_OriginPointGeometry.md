# SketchBlockDefinition.OriginPointGeometry Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that gets the geometry that describes the origin point.

## Syntax

SketchBlockDefinition.**OriginPointGeometry**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2010
