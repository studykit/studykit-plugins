# SketchBlockDefinition.LineWeight Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and Sets the line weight override for the sketch.

## Syntax

SketchBlockDefinition.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
