# SketchBlockDefinition.Profiles Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the Profiles collection object.

## Syntax

SketchBlockDefinition.**Profiles**() As [Profiles](../Profiles/Profiles.md)

## Property Value

This is a read only property whose value is a [Profiles](../Profiles/Profiles.md).

## Version

Introduced in version 2010
