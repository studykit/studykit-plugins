# SketchBlockDefinition.SketchEllipses Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the SketchEllipses collection object.

## Syntax

SketchBlockDefinition.**SketchEllipses**() As [SketchEllipses](../SketchEllipses/SketchEllipses.md)

## Property Value

This is a read only property whose value is a [SketchEllipses](../SketchEllipses/SketchEllipses.md).

## Version

Introduced in version 2010
