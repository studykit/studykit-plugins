# SketchBlockDefinition.Delete Method

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Method that deletes the sketch. This method is only valid for sketches that are not used by a feature.

## Syntax

SketchBlockDefinition.**Delete**()

## Version

Introduced in version 2010
