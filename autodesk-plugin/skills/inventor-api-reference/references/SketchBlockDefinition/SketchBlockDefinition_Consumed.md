# SketchBlockDefinition.Consumed Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets whether the sketch is consumed or not.

## Syntax

SketchBlockDefinition.**Consumed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
