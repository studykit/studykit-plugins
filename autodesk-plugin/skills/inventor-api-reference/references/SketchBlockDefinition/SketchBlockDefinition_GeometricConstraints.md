# SketchBlockDefinition.GeometricConstraints Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the collection of all geometric constraints on the sketch.

## Syntax

SketchBlockDefinition.**GeometricConstraints**() As [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md)

## Property Value

This is a read only property whose value is a [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md).

## Version

Introduced in version 2010
