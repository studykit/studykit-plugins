# SketchBlockDefinition.Description Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and sets the description for the sketch block definition.

## Syntax

SketchBlockDefinition.**Description**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2021
