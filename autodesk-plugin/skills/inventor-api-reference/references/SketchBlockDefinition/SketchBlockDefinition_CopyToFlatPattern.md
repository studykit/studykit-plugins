# SketchBlockDefinition.CopyToFlatPattern Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and sets whether a sheet metal folded model sketch should be copied over (transposed) to the flat pattern.

## Syntax

SketchBlockDefinition.**CopyToFlatPattern**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
