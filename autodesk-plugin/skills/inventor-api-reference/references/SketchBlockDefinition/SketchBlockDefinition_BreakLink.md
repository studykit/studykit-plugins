# SketchBlockDefinition.BreakLink Method

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Method that breaks the link to the source sketch.

## Syntax

SketchBlockDefinition.**BreakLink**()

## Version

Introduced in version 2010
