# SketchBlockDefinition.SketchArcs Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

SketchBlockDefinition.**SketchArcs**() As [SketchArcs](../SketchArcs/SketchArcs.md)

## Property Value

This is a read only property whose value is a [SketchArcs](../SketchArcs/SketchArcs.md).

## Version

Introduced in version 2010
