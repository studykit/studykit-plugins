# GeneralOptions.TooltipDelay Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Gets and sets the length of the delay time (in seconds), while the cursor is hovering over a command in the ribbon, before the tooltip is displayed.

## Syntax

GeneralOptions.**TooltipDelay**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
