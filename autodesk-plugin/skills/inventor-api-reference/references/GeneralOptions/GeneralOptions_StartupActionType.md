# GeneralOptions.StartupActionType Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Gets and sets the type of startup action to perform each time Autodesk Inventor is opened.

## Syntax

GeneralOptions.**StartupActionType**() As [StartupActionTypeEnum](../StartupActionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [StartupActionTypeEnum](../StartupActionTypeEnum.md).

## Version

Introduced in version 2008
