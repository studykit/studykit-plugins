# GeneralOptions.ShowAutocompleteForCommandAlias Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Gets and sets whether the autocomplete dropdown box should be displayed for command aliases.

## Syntax

GeneralOptions.**ShowAutocompleteForCommandAlias**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
