# GeneralOptions.SpellCheckOptions Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Read-only property that returns the SpellCheckOptions object. The SpellCheckOptions object provides access to various spell check related application level options.

## Syntax

GeneralOptions.**SpellCheckOptions**() As [SpellCheckOptions](../SpellCheckOptions/SpellCheckOptions.md)

## Property Value

This is a read only property whose value is a [SpellCheckOptions](../SpellCheckOptions/SpellCheckOptions.md).

## Version

Introduced in version 2020
