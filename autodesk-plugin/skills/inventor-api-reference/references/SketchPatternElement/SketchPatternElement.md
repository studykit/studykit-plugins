# SketchPatternElement Object

## Description

SketchPatternElement Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchPatternElement/SketchPatternElement_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Index](../SketchPatternElement/SketchPatternElement_Index.md) | Returns the index of this element within the pattern. |
| [Parent](../SketchPatternElement/SketchPatternElement_Parent.md) | Returns the parent pattern this element is a member of. |
| [Suppressed](../SketchPatternElement/SketchPatternElement_Suppressed.md) | Gets and sets whether this pattern element is suppressed or not. |
| [Type](../SketchPatternElement/SketchPatternElement_Type.md) | Gets the constant that indicates the type of this object. |
