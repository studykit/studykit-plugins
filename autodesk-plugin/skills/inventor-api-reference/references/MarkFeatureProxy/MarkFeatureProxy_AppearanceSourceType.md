# MarkFeatureProxy.AppearanceSourceType Property

Parent Object: [MarkFeatureProxy](../MarkFeatureProxy/MarkFeatureProxy.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

MarkFeatureProxy.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2023
