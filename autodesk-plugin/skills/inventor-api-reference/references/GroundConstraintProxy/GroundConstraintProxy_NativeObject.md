# GroundConstraintProxy.NativeObject Property

Parent Object: [GroundConstraintProxy](../GroundConstraintProxy/GroundConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

GroundConstraintProxy.**NativeObject**() As [GroundConstraint](../GroundConstraint/GroundConstraint.md)

## Property Value

This is a read only property whose value is a [GroundConstraint](../GroundConstraint/GroundConstraint.md).

## Version

Introduced in version 6
