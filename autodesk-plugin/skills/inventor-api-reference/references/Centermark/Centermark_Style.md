# Centermark.Style Property

Parent Object: [Centermark](../Centermark/Centermark.md)

## Description

Gets and sets the style associated with this center mark.

## Syntax

Centermark.**Style**() As [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Property Value

This is a read/write property whose value is a [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md).

## Version

Introduced in version 2010
