# Centermark.ExtensionLinesVisible Property

Parent Object: [Centermark](../Centermark/Centermark.md)

## Description

Read-write property that gets and sets whether the extension lines associated with this center mark are visible or not.

## Syntax

Centermark.**ExtensionLinesVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
