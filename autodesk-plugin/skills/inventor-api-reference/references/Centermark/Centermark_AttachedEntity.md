# Centermark.AttachedEntity Property

Parent Object: [Centermark](../Centermark/Centermark.md)

## Description

Property that returns the entity the center mark is associated with.

## Syntax

Centermark.**AttachedEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2010
