# Centermark.Position Property

Parent Object: [Centermark](../Centermark/Centermark.md)

## Description

Property that returns the position of the center of the center mark on the sheet.

## Syntax

Centermark.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2009
