# Centermark.Centerlines Property

Parent Object: [Centermark](../Centermark/Centermark.md)

## Description

Property that returns the a collection of associated Centerline objects.

## Syntax

Centermark.**Centerlines**() As [ObjectsEnumerator](../ObjectsEnumerator/ObjectsEnumerator.md)

## Property Value

This is a read only property whose value is an [ObjectsEnumerator](../ObjectsEnumerator/ObjectsEnumerator.md).

## Version

Introduced in version 2009
