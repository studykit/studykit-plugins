# OffsetWidthExtent.OffsetDistanceOne Property

Parent Object: [OffsetWidthExtent](../OffsetWidthExtent/OffsetWidthExtent.md)

## Description

Property that returns the parameter controlling the distance of the feature from this offset entity.

## Syntax

OffsetWidthExtent.**OffsetDistanceOne**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
