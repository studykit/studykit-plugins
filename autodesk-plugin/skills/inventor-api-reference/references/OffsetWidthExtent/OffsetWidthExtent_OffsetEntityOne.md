# OffsetWidthExtent.OffsetEntityOne Property

Parent Object: [OffsetWidthExtent](../OffsetWidthExtent/OffsetWidthExtent.md)

## Description

Property that gets and set the entity the width is measured relative to.

## Syntax

OffsetWidthExtent.**OffsetEntityOne**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2009
