# ChangeDefinition.CreateChangeProcessor Method

Parent Object: [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md)

## Description

Creates a new ChangeProcessor object based on this definition.

## Syntax

ChangeDefinition.**CreateChangeProcessor**() As [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Version
