# ChangeDefinition.ChangeType Property

Parent Object: [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md)

## Description

Gets and sets the type of change (shape edit, query only, etc.) that this ChangeDefinition represents. Defaults kShapeEditCmdType.

## Syntax

ChangeDefinition.**ChangeType**() As [CommandTypesEnum](../CommandTypesEnum.md)

## Property Value

This is a read/write property whose value is a [CommandTypesEnum](../CommandTypesEnum.md).

## Version

Introduced in version 2008
