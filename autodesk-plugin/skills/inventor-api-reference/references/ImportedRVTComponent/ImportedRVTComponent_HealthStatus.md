# ImportedRVTComponent.HealthStatus Property

Parent Object: [ImportedRVTComponent](../ImportedRVTComponent/ImportedRVTComponent.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

ImportedRVTComponent.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2021
