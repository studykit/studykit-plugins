# BaselineDimensionSet.Style Property

Parent Object: [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md)

## Description

Gets and sets the dimension style used for this dimension set.

## Syntax

BaselineDimensionSet.**Style**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 2010
