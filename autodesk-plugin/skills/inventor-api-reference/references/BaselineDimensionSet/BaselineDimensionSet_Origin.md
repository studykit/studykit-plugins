# BaselineDimensionSet.Origin Property

Parent Object: [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md)

## Description

Gets and sets the origin for the baseline dimension set.

## Syntax

BaselineDimensionSet.**Origin**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2010
