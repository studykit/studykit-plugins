# BaselineDimensionSet.DetachMember Method

Parent Object: [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md)

## Description

Method that detaches the member from the set. The member is not deleted, it is merely converted into a vanilla linear general dimension.

## Syntax

BaselineDimensionSet.**DetachMember**( ***Member*** As [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Member | [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md) |  |

## Version

Introduced in version 2010
