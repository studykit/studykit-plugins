# BaselineDimensionSet.Members Property

Parent Object: [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md)

## Description

Property that returns all the member LinearGeneralDimension objects.

## Syntax

BaselineDimensionSet.**Members**() As [GeneralDimensionsEnumerator](../GeneralDimensionsEnumerator/GeneralDimensionsEnumerator.md)

## Property Value

This is a read only property whose value is a [GeneralDimensionsEnumerator](../GeneralDimensionsEnumerator/GeneralDimensionsEnumerator.md).

## Version

Introduced in version 2010
