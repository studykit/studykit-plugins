# FeaturePatternElements.Item Property

Parent Object: [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md)

## Description

Returns the specified object from the collection. This is the default property of the FeaturePatternElements collection object.

## Syntax

FeaturePatternElements.**Item**( ***Index*** As Long ) As [FeaturePatternElement](../FeaturePatternElement/FeaturePatternElement.md)

## Property Value

This is a read only property whose value is a [FeaturePatternElement](../FeaturePatternElement/FeaturePatternElement.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 5
