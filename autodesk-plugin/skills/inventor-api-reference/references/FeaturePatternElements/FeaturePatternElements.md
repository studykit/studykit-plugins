# FeaturePatternElements Object

## Description

Provides access to the instances ( objects) that result from the creation of a feature pattern.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FeaturePatternElements/FeaturePatternElements_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FeaturePatternElements/FeaturePatternElements_Count.md) | Property that returns the number of items in this collection. |
| [Item](../FeaturePatternElements/FeaturePatternElements_Item.md) | Returns the specified object from the collection. This is the default property of the FeaturePatternElements collection object. |
| [Type](../FeaturePatternElements/FeaturePatternElements_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
