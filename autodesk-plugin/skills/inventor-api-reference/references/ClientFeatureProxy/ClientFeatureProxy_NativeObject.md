# ClientFeatureProxy.NativeObject Property

Parent Object: [ClientFeatureProxy](../ClientFeatureProxy/ClientFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ClientFeatureProxy.**NativeObject**() As [ClientFeature](../ClientFeature/ClientFeature.md)

## Property Value

This is a read only property whose value is a [ClientFeature](../ClientFeature/ClientFeature.md).

## Version

Introduced in version 2008
