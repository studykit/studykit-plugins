# PromptMessage.Buttons Property

Parent Object: [PromptMessage](../PromptMessage/PromptMessage.md)

## Description

Read-only property that returns the buttons that are used for the prompt message.

## Syntax

PromptMessage.**Buttons**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2025.1
