# PromptMessage.Title Property

Parent Object: [PromptMessage](../PromptMessage/PromptMessage.md)

## Description

Read-only property that returns the title of the prompt message.

## Syntax

PromptMessage.**Title**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2025.1
