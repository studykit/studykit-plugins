# PromptMessage.Delete Method

Parent Object: [PromptMessage](../PromptMessage/PromptMessage.md)

## Description

Method that deletes this prompt message. This is applicable for custom prompt messages.

## Syntax

PromptMessage.**Delete**()

## Version

Introduced in version 2025.1
