# PromptMessage.DefaultAnswer Property

Parent Object: [PromptMessage](../PromptMessage/PromptMessage.md)

## Description

Read-write property that gets and sets the default response for the prompt message.

## Syntax

PromptMessage.**DefaultAnswer**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2025.1
