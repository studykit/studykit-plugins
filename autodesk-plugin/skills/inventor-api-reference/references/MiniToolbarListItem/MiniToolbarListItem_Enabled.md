# MiniToolbarListItem.Enabled Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-write property that gets and sets whether the control is enabled. This property defaults to True when the control is created.

## Syntax

MiniToolbarListItem.**Enabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
