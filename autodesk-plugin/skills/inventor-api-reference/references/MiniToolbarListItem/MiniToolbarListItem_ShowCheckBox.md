# MiniToolbarListItem.ShowCheckBox Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-write property that gets and sets whether the a check box is displayed beside this item.

## Syntax

MiniToolbarListItem.**ShowCheckBox**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
