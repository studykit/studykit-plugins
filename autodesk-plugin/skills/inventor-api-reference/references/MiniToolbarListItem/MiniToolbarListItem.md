# MiniToolbarListItem Object

## Description

The MiniToolbarListItem object represents an item in either a MiniToolBarComboBox or a MiniToolBarDropdown object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AllowDelete](../MiniToolbarListItem/MiniToolbarListItem_AllowDelete.md) | Read-write property that specifies if this item can be deleted. This will always be false for a separator. |
| [Enabled](../MiniToolbarListItem/MiniToolbarListItem_Enabled.md) | Read-write property that gets and sets whether the control is enabled. This property defaults to True when the control is created. |
| [Index](../MiniToolbarListItem/MiniToolbarListItem_Index.md) | Read-only property that specifies the position of this item within the list. |
| [InternalName](../MiniToolbarListItem/MiniToolbarListItem_InternalName.md) | Read-only property that gets the internal name of this item. |
| [IsChecked](../MiniToolbarListItem/MiniToolbarListItem_IsChecked.md) | Read-write property that specifies if the check box beside this item is checked or not. |
| [IsSeperator](../MiniToolbarListItem/MiniToolbarListItem_IsSeperator.md) | Read-only property that indicates if this control is a separator or not. |
| [LargeIcon](../MiniToolbarListItem/MiniToolbarListItem_LargeIcon.md) | Read-write property that gets and sets the standard size icon for this item. |
| [ShowCheckBox](../MiniToolbarListItem/MiniToolbarListItem_ShowCheckBox.md) | Read-write property that gets and sets whether the a check box is displayed beside this item. |
| [StandardIcon](../MiniToolbarListItem/MiniToolbarListItem_StandardIcon.md) | Read-write property that gets and sets the standard size icon for this item. |
| [Text](../MiniToolbarListItem/MiniToolbarListItem_Text.md) | Read-write property that gets and sets the visible text for this item. |
| [ToolTipText](../MiniToolbarListItem/MiniToolbarListItem_ToolTipText.md) | Read-write property that gets and sets the tool tip text for this item. |
