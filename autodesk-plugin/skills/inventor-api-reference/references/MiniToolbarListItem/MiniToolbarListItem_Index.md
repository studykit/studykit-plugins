# MiniToolbarListItem.Index Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-only property that specifies the position of this item within the list.

## Syntax

MiniToolbarListItem.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2012
