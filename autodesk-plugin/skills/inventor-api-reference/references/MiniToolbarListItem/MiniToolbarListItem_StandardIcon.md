# MiniToolbarListItem.StandardIcon Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-write property that gets and sets the standard size icon for this item.

## Syntax

MiniToolbarListItem.**StandardIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2012
