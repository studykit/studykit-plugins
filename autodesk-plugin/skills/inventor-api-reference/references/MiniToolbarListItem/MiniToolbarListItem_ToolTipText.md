# MiniToolbarListItem.ToolTipText Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-write property that gets and sets the tool tip text for this item.

## Syntax

MiniToolbarListItem.**ToolTipText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
