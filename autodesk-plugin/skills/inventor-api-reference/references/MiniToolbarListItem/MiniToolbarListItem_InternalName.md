# MiniToolbarListItem.InternalName Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-only property that gets the internal name of this item.

## Syntax

MiniToolbarListItem.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
