# DockableWindowsEvents.Parent Property

Parent Object: [DockableWindowsEvents](../DockableWindowsEvents/DockableWindowsEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DockableWindowsEvents.**Parent**() As [DockableWindows](../DockableWindows/DockableWindows.md)

## Property Value

This is a read only property whose value is a [DockableWindows](../DockableWindows/DockableWindows.md).

## Version

Introduced in version 2012
