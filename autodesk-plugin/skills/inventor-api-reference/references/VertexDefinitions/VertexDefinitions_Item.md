# VertexDefinitions.Item Property

Parent Object: [VertexDefinitions](../VertexDefinitions/VertexDefinitions.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

VertexDefinitions.**Item**( ***Index*** As Long ) As [VertexDefinition](../VertexDefinition/VertexDefinition.md)

## Property Value

This is a read only property whose value is a [VertexDefinition](../VertexDefinition/VertexDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the VertexDefinition object within the collection to return. |

## Version

Introduced in version 2011
