# SurfaceTextureISODefinition.SecondRequirement Property

Parent Object: [SurfaceTextureISODefinition](../SurfaceTextureISODefinition/SurfaceTextureISODefinition.md)

## Description

Gets and sets the second requirement value for the surface.

## Syntax

SurfaceTextureISODefinition.**SecondRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
