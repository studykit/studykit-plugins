# SurfaceTextureISODefinition.IsForceTailShown Property

Parent Object: [SurfaceTextureISODefinition](../SurfaceTextureISODefinition/SurfaceTextureISODefinition.md)

## Description

Gets and sets whether to add a tail to the symbol.

## Syntax

SurfaceTextureISODefinition.**IsForceTailShown**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
