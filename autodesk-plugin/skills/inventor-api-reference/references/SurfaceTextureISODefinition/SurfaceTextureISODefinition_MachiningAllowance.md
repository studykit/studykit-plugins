# SurfaceTextureISODefinition.MachiningAllowance Property

Parent Object: [SurfaceTextureISODefinition](../SurfaceTextureISODefinition/SurfaceTextureISODefinition.md)

## Description

Gets and sets the machining allowance.

## Syntax

SurfaceTextureISODefinition.**MachiningAllowance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
