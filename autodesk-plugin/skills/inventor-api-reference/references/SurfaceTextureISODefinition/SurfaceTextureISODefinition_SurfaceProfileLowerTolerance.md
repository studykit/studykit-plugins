# SurfaceTextureISODefinition.SurfaceProfileLowerTolerance Property

Parent Object: [SurfaceTextureISODefinition](../SurfaceTextureISODefinition/SurfaceTextureISODefinition.md)

## Description

Gets and sets the surface profile lower tolerance.

## Syntax

SurfaceTextureISODefinition.**SurfaceProfileLowerTolerance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025
