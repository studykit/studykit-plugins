# SurfaceTextureISODefinition.NumberOfFeatureElements Property

Parent Object: [SurfaceTextureISODefinition](../SurfaceTextureISODefinition/SurfaceTextureISODefinition.md)

## Description

Gets and sets the number of feature elements.

## Syntax

SurfaceTextureISODefinition.**NumberOfFeatureElements**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025
