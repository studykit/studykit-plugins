# SketchSettings.UpperLimitForSecondRangeOfLineWeights Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Gets and sets the upper limit for the second range of line weights.

## Syntax

SketchSettings.**UpperLimitForSecondRangeOfLineWeights**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
