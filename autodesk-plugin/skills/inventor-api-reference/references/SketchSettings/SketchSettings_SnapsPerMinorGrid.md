# SketchSettings.SnapsPerMinorGrid Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Gets and sets the distance between minor grid lines relative to the specified snap distance.

## Syntax

SketchSettings.**SnapsPerMinorGrid**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 8
