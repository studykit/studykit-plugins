# SketchSettings.XSnapSpacing Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Gets and sets the spacing between snap points for the X axis.

## Syntax

SketchSettings.**XSnapSpacing**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 8
