# SketchSettings Object

## Description

The SketchSettings object provides access to properties that provide read and write access to the various sketch related document specific settings. This is somewhat equivalent to the Sketch tab of the Document Settings dialog.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchSettings/SketchSettings_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [DisplayLineWeights](../SketchSettings/SketchSettings_DisplayLineWeights.md) | Gets and sets the display of unique line weights. |
| [LineWeightType](../SketchSettings/SketchSettings_LineWeightType.md) | Gets and sets the type of line weight to apply when displaying lines. |
| [MinorLinesPerMajorGridLine](../SketchSettings/SketchSettings_MinorLinesPerMajorGridLine.md) | Gets and sets the number of minor lines to appear between major lines. |
| [Parent](../SketchSettings/SketchSettings_Parent.md) | Property that returns the parent Document object. |
| [SnapsPerMinorGrid](../SketchSettings/SketchSettings_SnapsPerMinorGrid.md) | Gets and sets the distance between minor grid lines relative to the specified snap distance. |
| [Type](../SketchSettings/SketchSettings_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [UpperLimitForFirstRangeOfLineWeights](../SketchSettings/SketchSettings_UpperLimitForFirstRangeOfLineWeights.md) | Gets and sets the upper limit for the first range of line weights. |
| [UpperLimitForSecondRangeOfLineWeights](../SketchSettings/SketchSettings_UpperLimitForSecondRangeOfLineWeights.md) | Gets and sets the upper limit for the second range of line weights. |
| [UpperLimitForThirdRangeOfLineWeights](../SketchSettings/SketchSettings_UpperLimitForThirdRangeOfLineWeights.md) | Gets and sets the upper limit for the third range of line weights. |
| [XSnapSpacing](../SketchSettings/SketchSettings_XSnapSpacing.md) | Gets and sets the spacing between snap points for the X axis. |
| [YSnapSpacing](../SketchSettings/SketchSettings_YSnapSpacing.md) | Gets and sets the spacing between snap points for the Y axis. |

## Accessed From

[AssemblyDocument.SketchSettings](../AssemblyDocument/AssemblyDocument_SketchSettings.md), [DrawingDocument.SketchSettings](../DrawingDocument/DrawingDocument_SketchSettings.md), [PartDocument.SketchSettings](../PartDocument/PartDocument_SketchSettings.md)

## Version

Introduced in version 8
