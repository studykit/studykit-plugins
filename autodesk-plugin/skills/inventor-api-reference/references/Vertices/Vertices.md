# Vertices Object

## Description

The Vertices object represents a collection of  objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../Vertices/Vertices_Count.md) | Property that returns the number of items in this collection. |
| [Item](../Vertices/Vertices_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../Vertices/Vertices_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
