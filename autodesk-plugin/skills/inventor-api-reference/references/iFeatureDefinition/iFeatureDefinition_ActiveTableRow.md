# iFeatureDefinition.ActiveTableRow Property

Parent Object: [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md)

## Description

Gets and sets the row from the table that should be used to specify the sizes for the iFeature instances.

## Syntax

iFeatureDefinition.**ActiveTableRow**() As [iFeatureTableRow](../iFeatureTableRow/iFeatureTableRow.md)

## Property Value

This is a read/write property whose value is an [iFeatureTableRow](../iFeatureTableRow/iFeatureTableRow.md).
