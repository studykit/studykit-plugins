# iFeatureDefinition.iFeatureTable Property

Parent Object: [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md)

## Description

Property that returns the table information associated with this iFeature.

## Syntax

iFeatureDefinition.**iFeatureTable**() As [iFeatureTable](../iFeatureTable/iFeatureTable.md)

## Property Value

This is a read only property whose value is an [iFeatureTable](../iFeatureTable/iFeatureTable.md).
