# DWGEllipticalArcsEnumerator.Item Property

Parent Object: [DWGEllipticalArcsEnumerator](../DWGEllipticalArcsEnumerator/DWGEllipticalArcsEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGEllipticalArcsEnumerator.**Item**( ***Index*** As Variant ) As [DWGEllipticalArc](../DWGEllipticalArc/DWGEllipticalArc.md)

## Property Value

This is a read only property whose value is a [DWGEllipticalArc](../DWGEllipticalArc/DWGEllipticalArc.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
