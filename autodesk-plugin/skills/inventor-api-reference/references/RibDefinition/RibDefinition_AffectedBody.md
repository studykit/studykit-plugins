# RibDefinition.AffectedBody Property

Parent Object: [RibDefinition](../RibDefinition/RibDefinition.md)

## Description

Read-write property that gets and sets the solid body affected by the creation of this rib feature. For parts containing multiple solid bodies, this property defaults to the primary solid body of the part, and can be set to any of the other solid bodies.

## Syntax

RibDefinition.**AffectedBody**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read/write property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2012
