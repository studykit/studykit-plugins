# RibDefinition.BossSets Property

Parent Object: [RibDefinition](../RibDefinition/RibDefinition.md)

## Description

Read-only property that returns the BossSets collection object containing boss definitions of the rib feature.

## Syntax

RibDefinition.**BossSets**() As [BossSets](../BossSets/BossSets.md)

## Property Value

This is a read only property whose value is a [BossSets](../BossSets/BossSets.md).

## Version

Introduced in version 2012
