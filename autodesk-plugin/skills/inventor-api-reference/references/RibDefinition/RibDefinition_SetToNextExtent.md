# RibDefinition.SetToNextExtent Method

Parent Object: [RibDefinition](../RibDefinition/RibDefinition.md)

## Description

Method that sets the extent type to be ‘to next’ (i.e. it terminates on the next face).

## Syntax

RibDefinition.**SetToNextExtent**()

## Version

Introduced in version 2012
