# RibDefinition.ThicknessDirection Property

Parent Object: [RibDefinition](../RibDefinition/RibDefinition.md)

## Description

Read-write property that gets and sets the side of the profile on which to apply the thickness. This property defaults to kSymmetricExtentDirection for a newly created RibDefinition object.

## Syntax

RibDefinition.**ThicknessDirection**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2012
