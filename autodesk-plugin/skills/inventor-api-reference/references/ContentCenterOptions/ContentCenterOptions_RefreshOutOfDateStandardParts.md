# ContentCenterOptions.RefreshOutOfDateStandardParts Property

Parent Object: [ContentCenterOptions](../ContentCenterOptions/ContentCenterOptions.md)

## Description

Gets/sets whether to automatically refresh out of date standard parts during placement.

## Syntax

ContentCenterOptions.**RefreshOutOfDateStandardParts**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
