# ContentCenterOptions.CheckFamiliesUpdates Property

Parent Object: [ContentCenterOptions](../ContentCenterOptions/ContentCenterOptions.md)

## Description

Gets/sets how to handle linked families - check/do not check for parent version.

## Syntax

ContentCenterOptions.**CheckFamiliesUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
