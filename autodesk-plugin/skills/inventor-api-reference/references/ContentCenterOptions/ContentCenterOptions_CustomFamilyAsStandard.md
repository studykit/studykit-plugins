# ContentCenterOptions.CustomFamilyAsStandard Property

Parent Object: [ContentCenterOptions](../ContentCenterOptions/ContentCenterOptions.md)

## Description

Gets/sets whether to place Family with custom's columns As Standard or As Custom.

## Syntax

ContentCenterOptions.**CustomFamilyAsStandard**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
