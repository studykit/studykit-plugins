# GeneralNoteSymbolDefinitions.Item Property

Parent Object: [GeneralNoteSymbolDefinitions](../GeneralNoteSymbolDefinitions/GeneralNoteSymbolDefinitions.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

GeneralNoteSymbolDefinitions.**Item**( ***Index*** As Long ) As [GeneralNoteSymbolDefinition](../GeneralNoteSymbolDefinition/GeneralNoteSymbolDefinition.md)

## Property Value

This is a read only property whose value is a [GeneralNoteSymbolDefinition](../GeneralNoteSymbolDefinition/GeneralNoteSymbolDefinition.md).
