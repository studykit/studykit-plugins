# GeneralNoteSymbolDefinitions.Count Property

Parent Object: [GeneralNoteSymbolDefinitions](../GeneralNoteSymbolDefinitions/GeneralNoteSymbolDefinitions.md)

## Description

Gets the number of items in this collection.

## Syntax

GeneralNoteSymbolDefinitions.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2018
