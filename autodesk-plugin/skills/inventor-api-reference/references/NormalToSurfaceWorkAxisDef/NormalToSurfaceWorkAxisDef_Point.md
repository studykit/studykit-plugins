# NormalToSurfaceWorkAxisDef.Point Property

Parent Object: [NormalToSurfaceWorkAxisDef](../NormalToSurfaceWorkAxisDef/NormalToSurfaceWorkAxisDef.md)

## Description

Property that returns the point of a normal to surface defined work axis. The point can be a WorkPoint, Vertex, SketchPoint3D or SketchPoint object. The work axis passes through the point.

## Syntax

NormalToSurfaceWorkAxisDef.**Point**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
