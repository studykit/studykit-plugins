# PresentationBody.Name Property

Parent Object: [PresentationBody](../PresentationBody/PresentationBody.md)

## Description

Read-only property that gets the presentation body name.

## Syntax

PresentationBody.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
