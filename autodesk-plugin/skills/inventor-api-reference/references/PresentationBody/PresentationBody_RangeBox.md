# PresentationBody.RangeBox Property

Parent Object: [PresentationBody](../PresentationBody/PresentationBody.md)

## Description

Read-only property that returns the range box that represents the bounds of the presentation body.

## Syntax

PresentationBody.**RangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2018
