# PresentationBody.Vertices Property

Parent Object: [PresentationBody](../PresentationBody/PresentationBody.md)

## Description

Read-only property that returns all the PresentationVertex objects contained within the presentation body.

## Syntax

PresentationBody.**Vertices**() As [PresentationVerticesEnumerator](../PresentationVerticesEnumerator/PresentationVerticesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationVerticesEnumerator](../PresentationVerticesEnumerator/PresentationVerticesEnumerator.md).

## Version

Introduced in version 2018
