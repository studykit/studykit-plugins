# PresentationBody.Faces Property

Parent Object: [PresentationBody](../PresentationBody/PresentationBody.md)

## Description

Read-only property that returns all the PresentationFace objects contained within the presentation body.

## Syntax

PresentationBody.**Faces**() As [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md).

## Version

Introduced in version 2018
