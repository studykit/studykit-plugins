# PresentationBody.Parent Property

Parent Object: [PresentationBody](../PresentationBody/PresentationBody.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PresentationBody.**Parent**() As [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Property Value

This is a read only property whose value is a [PresentationComponent](../PresentationComponent/PresentationComponent.md).

## Version

Introduced in version 2018
