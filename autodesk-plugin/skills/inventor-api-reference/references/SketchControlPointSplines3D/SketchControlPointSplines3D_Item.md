# SketchControlPointSplines3D.Item Property

Parent Object: [SketchControlPointSplines3D](../SketchControlPointSplines3D/SketchControlPointSplines3D.md)

## Description

Method that returns the specified SketchControlPointSpline3D object from the collection.

## Syntax

SketchControlPointSplines3D.**Item**( ***Index*** As Long ) As [SketchControlPointSpline3D](../SketchControlPointSpline3D/SketchControlPointSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchControlPointSpline3D](../SketchControlPointSpline3D/SketchControlPointSpline3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. The first spline in the collection as an index of 1. |

## Version

Introduced in version 2014
