# SketchControlPointSplines3D.Count Property

Parent Object: [SketchControlPointSplines3D](../SketchControlPointSplines3D/SketchControlPointSplines3D.md)

## Description

Gets the number of items in this collection.

## Syntax

SketchControlPointSplines3D.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2014
