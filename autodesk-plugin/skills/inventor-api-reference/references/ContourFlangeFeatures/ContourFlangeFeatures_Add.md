# ContourFlangeFeatures.Add Method

Parent Object: [ContourFlangeFeatures](../ContourFlangeFeatures/ContourFlangeFeatures.md)

## Description

Method that creates a new contour flange feature. The newly created ContourFlangeFeature object is returned.

## Syntax

ContourFlangeFeatures.**Add**( ***ContourFlangeDefinition*** As [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md) ) As [ContourFlangeFeature](../ContourFlangeFeature/ContourFlangeFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ContourFlangeDefinition | [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md) | Input ContourFlangeDefinition object that defines the contour flange feature you want to create. A ContourFlangeDefinition object can be created using the ContourFlangeFeatures.CreateContourFlangeDefinition method. It can also be obtained from an existing ContourFlangeFeature object. |

## Version

Introduced in version 2010
