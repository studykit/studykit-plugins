# ExtrudeFeature.Definition Property

Parent Object: [ExtrudeFeature](../ExtrudeFeature/ExtrudeFeature.md)

## Description

Read-write property that gets and sets the ExtrudeDefinition object associated with this feature.

## Syntax

ExtrudeFeature.**Definition**() As [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Property Value

This is a read/write property whose value is an [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md).
