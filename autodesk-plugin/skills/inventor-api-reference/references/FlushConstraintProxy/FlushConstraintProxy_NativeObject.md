# FlushConstraintProxy.NativeObject Property

Parent Object: [FlushConstraintProxy](../FlushConstraintProxy/FlushConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FlushConstraintProxy.**NativeObject**() As [FlushConstraint](../FlushConstraint/FlushConstraint.md)

## Property Value

This is a read only property whose value is a [FlushConstraint](../FlushConstraint/FlushConstraint.md).

## Version

Introduced in version 11
