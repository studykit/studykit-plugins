# FlushConstraintProxy.Suppressed Property

Parent Object: [FlushConstraintProxy](../FlushConstraintProxy/FlushConstraintProxy.md)

## Description

Gets/Sets the Boolean flag indicating whether this constraint is suppressed.

## Syntax

FlushConstraintProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
