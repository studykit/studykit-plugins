# PartsListRow.Height Property

Parent Object: [PartsListRow](../PartsListRow/PartsListRow.md)

## Description

Gets and sets the height of the parts list row.

## Syntax

PartsListRow.**Height**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
