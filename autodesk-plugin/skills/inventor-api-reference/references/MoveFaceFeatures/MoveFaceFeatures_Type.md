# MoveFaceFeatures.Type Property

Parent Object: [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

MoveFaceFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
