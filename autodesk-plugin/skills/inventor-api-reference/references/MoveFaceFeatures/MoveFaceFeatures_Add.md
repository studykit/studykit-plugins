# MoveFaceFeatures.Add Method

Parent Object: [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md)

## Description

Method that creates a new Move Face feature. The newly created MoveFaceFeature object is returned.

## Syntax

MoveFaceFeatures.**Add**( ***Definition*** As [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md) ) As [MoveFaceFeature](../MoveFaceFeature/MoveFaceFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md) | MoveFaceDefinition object that defines the move face feature you want to create. A MoveFaceDefinition object can be created using the MoveFaceFeatures.CreateDefinition method. It can also be obtained from an existing MoveFaceFeature object. |

## Version

Introduced in version 2011
