# MoveFaceFeatures Object

## Description

The MoveFaceFeatures object provides access to all of the objects in a component definition and provides methods to create additional MoveFaceFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../MoveFaceFeatures/MoveFaceFeatures_Add.md) | Method that creates a new Move Face feature. The newly created MoveFaceFeature object is returned. |
| [CreateDefinition](../MoveFaceFeatures/MoveFaceFeatures_CreateDefinition.md) | Method that creates a new MoveFaceDefinition object |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MoveFaceFeatures/MoveFaceFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../MoveFaceFeatures/MoveFaceFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../MoveFaceFeatures/MoveFaceFeatures_Item.md) | Returns the specified feature object from the collection. This is the default property of this collection object. |
| [Type](../MoveFaceFeatures/MoveFaceFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Features.MoveFaceFeatures](../Features/Features_MoveFaceFeatures.md), [PartFeatures.MoveFaceFeatures](../PartFeatures/PartFeatures_MoveFaceFeatures.md), [SheetMetalFeatures.MoveFaceFeatures](../SheetMetalFeatures/SheetMetalFeatures_MoveFaceFeatures.md)

## Version

Introduced in version 10
