# DrawingDimensions.Item Property

Parent Object: [DrawingDimensions](../DrawingDimensions/DrawingDimensions.md)

## Description

Method that returns the specified dimension object from the collection.

## Syntax

DrawingDimensions.**Item**( ***Index*** As Long ) As [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Property Value

This is a read only property whose value is a [DrawingDimension](../DrawingDimension/DrawingDimension.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 9
