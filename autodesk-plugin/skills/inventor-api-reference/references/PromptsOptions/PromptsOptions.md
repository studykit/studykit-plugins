# PromptsOptions Object

## Description

PromptsOptions object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PromptsOptions/PromptsOptions_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [DoNotShowMessagesForPreexistingProblems](../PromptsOptions/PromptsOptions_DoNotShowMessagesForPreexistingProblems.md) | Read-write Boolean property that gets and sets whether or not to show messages for pre-existing problems. |
| [PromptMessages](../PromptsOptions/PromptsOptions_PromptMessages.md) | Read-only property that returns the collection of prompt messages. |
| [Type](../PromptsOptions/PromptsOptions_Type.md) | Gets the constant that indicates the type of this object. |
