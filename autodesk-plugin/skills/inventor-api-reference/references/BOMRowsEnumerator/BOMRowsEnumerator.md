# BOMRowsEnumerator Object

## Description

The BOMRowsEnumerator object provides access to BOMRow objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BOMRowsEnumerator/BOMRowsEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BOMRowsEnumerator/BOMRowsEnumerator_Count.md) | Property that specifies the number of items in the collection. |
| [Item](../BOMRowsEnumerator/BOMRowsEnumerator_Item.md) | Returns an item in the collection. |
| [Type](../BOMRowsEnumerator/BOMRowsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BOMRow.ChildRows](../BOMRow/BOMRow_ChildRows.md), [BOMView.BOMRows](../BOMView/BOMView_BOMRows.md)

## Version

Introduced in version 11
