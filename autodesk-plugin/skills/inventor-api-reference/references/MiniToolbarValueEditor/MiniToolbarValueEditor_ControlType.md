# MiniToolbarValueEditor.ControlType Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-only property that returns the control type.

## Syntax

MiniToolbarValueEditor.**ControlType**() As [MiniToolbarControlTypeEnum](../MiniToolbarControlTypeEnum.md)

## Property Value

This is a read only property whose value is a [MiniToolbarControlTypeEnum](../MiniToolbarControlTypeEnum.md).

## Version

Introduced in version 2012
