# MiniToolbarValueEditor.OnChange Event

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Event that is fired when the content of the value editor is changed by the user.

## Syntax

MiniToolbarValueEditor.**OnChange**()

## Version

Introduced in version 2012
