# MiniToolbarValueEditor.Expression Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the current, unevaluated content of the value editor.

## Syntax

MiniToolbarValueEditor.**Expression**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
