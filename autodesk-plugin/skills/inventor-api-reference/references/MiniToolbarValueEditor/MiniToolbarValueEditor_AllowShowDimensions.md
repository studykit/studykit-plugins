# MiniToolbarValueEditor.AllowShowDimensions Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets whether to provide the ‘Show Dimensions’ command in the fly-out and right click context menus of the value editor.

## Syntax

MiniToolbarValueEditor.**AllowShowDimensions**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
