# MiniToolbarValueEditor.MostRecentValues Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets an array of strings representing the most recent values associated with this control.

## Syntax

MiniToolbarValueEditor.**MostRecentValues**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2012
