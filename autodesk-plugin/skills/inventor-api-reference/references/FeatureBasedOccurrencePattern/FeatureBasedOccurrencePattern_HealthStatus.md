# FeatureBasedOccurrencePattern.HealthStatus Property

Parent Object: [FeatureBasedOccurrencePattern](../FeatureBasedOccurrencePattern/FeatureBasedOccurrencePattern.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

FeatureBasedOccurrencePattern.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 6
