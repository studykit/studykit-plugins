# ReferenceComponent Object

## Description

The ReferenceComponent object is the base class for the DerivedPartComponent and DerivedAssemblyComponent objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [BreakLinkToFile](../ReferenceComponent/ReferenceComponent_BreakLinkToFile.md) | Method that breaks the connection of the derived component with the part or assembly from which it was created. When the link is broken the ReferencedFile property will return Nothing. |
| [Delete](../ReferenceComponent/ReferenceComponent_Delete.md) | Method that deletes the ReferenceComponent. |
| [GetReferenceKey](../ReferenceComponent/ReferenceComponent_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |
| [SetEndOfPart](../ReferenceComponent/ReferenceComponent_SetEndOfPart.md) | Method that repositions the end-of-part marker relative to the object this method is called from. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ReferenceComponent/ReferenceComponent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../ReferenceComponent/ReferenceComponent_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [HealthStatus](../ReferenceComponent/ReferenceComponent_HealthStatus.md) | Property that returns an enum indicating the current state of the object. |
| [IsEmbedded](../ReferenceComponent/ReferenceComponent_IsEmbedded.md) | Property that returns whether the reference component refers to an embedded document or a linked document. |
| [LinkedToFile](../ReferenceComponent/ReferenceComponent_LinkedToFile.md) | Property that returns whether the derived component is still linked to the base part or assembly document. If True, the link still exists. If False, the link has been broken and the ReferencedFile property will return Nothing. |
| [Name](../ReferenceComponent/ReferenceComponent_Name.md) | Property that returns the component's name. |
| [Parent](../ReferenceComponent/ReferenceComponent_Parent.md) | Property that returns the parent object. |
| [ReferencedDocumentDescriptor](../ReferenceComponent/ReferenceComponent_ReferencedDocumentDescriptor.md) | Property that returns the DocumentDescriptor object. |
| [Type](../ReferenceComponent/ReferenceComponent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
