# DWGEntitySplineSegment.StartPoint Property

Parent Object: [DWGEntitySplineSegment](../DWGEntitySplineSegment/DWGEntitySplineSegment.md)

## Description

Read-only property that returns the start point geometry.

## Syntax

DWGEntitySplineSegment.**StartPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
