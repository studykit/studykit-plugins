# Face.SurfaceType Property

Parent Object: [Face](../Face/Face.md)

## Description

Property that returns a SurfaceTypeEnum that specifies the surface type for this Face.

## Syntax

Face.**SurfaceType**() As [SurfaceTypeEnum](../SurfaceTypeEnum.md)

## Property Value

This is a read only property whose value is a [SurfaceTypeEnum](../SurfaceTypeEnum.md).
