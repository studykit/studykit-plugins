# Face.TextureMaps Property

Parent Object: [Face](../Face/Face.md)

## Description

Gets the texture maps associated with this face.

## Syntax

Face.**TextureMaps**() As [TextureMaps](../TextureMaps/TextureMaps.md)

## Property Value

This is a read only property whose value is a [TextureMaps](../TextureMaps/TextureMaps.md).

## Version

Introduced in version 10
