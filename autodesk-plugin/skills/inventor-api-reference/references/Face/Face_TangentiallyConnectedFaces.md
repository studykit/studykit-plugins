# Face.TangentiallyConnectedFaces Property

Parent Object: [Face](../Face/Face.md)

## Description

Property that returns a FaceCollection that contains the input face and all tangentially connected faces. The CollectionType of the output FaceCollection is set to kFaceTangentiallyConnected.

## Syntax

Face.**TangentiallyConnectedFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read only property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 11
