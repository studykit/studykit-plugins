# Face.Vertices Property

Parent Object: [Face](../Face/Face.md)

## Description

Gets the vertices for this Face.

## Syntax

Face.**Vertices**() As [Vertices](../Vertices/Vertices.md)

## Property Value

This is a read only property whose value is a [Vertices](../Vertices/Vertices.md).

## Version

Introduced in version 4
