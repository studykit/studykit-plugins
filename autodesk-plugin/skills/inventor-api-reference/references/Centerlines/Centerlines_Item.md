# Centerlines.Item Property

Parent Object: [Centerlines](../Centerlines/Centerlines.md)

## Description

Method that returns the specified Centerline object from the collection.

## Syntax

Centerlines.**Item**( ***Index*** As Long ) As [Centerline](../Centerline/Centerline.md)

## Property Value

This is a read only property whose value is a [Centerline](../Centerline/Centerline.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2008
