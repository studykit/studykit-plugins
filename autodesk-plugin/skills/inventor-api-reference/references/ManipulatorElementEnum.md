# ManipulatorElementEnum Enumerator

## Description

Manipulator element type enum.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllManipulatorElements | 0 | Toggle All manipulator elements. |
| kManipulatorOriginScaleElement | 1024 | Toggle global scale element. |
| kManipulatorOriginTranslationElement | 1 | Toggle global translation element. |
| kManipulatorXRotationElement | 128 | Toggle X axis rotation element. |
| kManipulatorXScaleElement | 2048 | Toggle X axis scale element. |
| kManipulatorXTranslationElement | 2 | Toggle X axis translation element. |
| kManipulatorXYScaleElement | 16384 | Toggle XY plane scale element. |
| kManipulatorXYTranslationElement | 16 | Toggle XY plane translation element. |
| kManipulatorXZScaleElement | 32768 | Toggle XZ plane scale element. |
| kManipulatorXZTranslationElement | 32 | Toggle XZ plane translation element. |
| kManipulatorYRotationElement | 256 | Toggle Y axis rotation element. |
| kManipulatorYScaleElement | 4096 | Toggle Y axis scale element. |
| kManipulatorYTranslationElement | 4 | Toggle Y axis translation element. |
| kManipulatorYZScaleElement | 65536 | Toggle YZ plane scale element. |
| kManipulatorYZTranslationElement | 64 | Toggle YZ plane translation element. |
| kManipulatorZRotationElement | 512 | Toggle Z axis rotation element. |
| kManipulatorZScaleElement | 8192 | Toggle Z axis scale element. |
| kManipulatorZTranslationElement | 8 | Toggle Z axis translation element. |

## Version

Introduced in version 2017
