# SketchCircularPatterns Object

## Description

SketchCircularPatterns Collection Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchCircularPatterns/SketchCircularPatterns_Add.md) | Creates a new sketch circular pattern. The new SketchCircularPattern object is returned. |
