# SketchCircularPatterns.Item Property

Parent Object: [SketchCircularPatterns](../SketchCircularPatterns/SketchCircularPatterns.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

SketchCircularPatterns.**Item**( ***Index*** As Long ) As [SketchCircularPattern](../SketchCircularPattern/SketchCircularPattern.md)

## Property Value

This is a read only property whose value is a [SketchCircularPattern](../SketchCircularPattern/SketchCircularPattern.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the pattern to return. If an out of range index is provided, an error occurs |

## Version

Introduced in version 2025
