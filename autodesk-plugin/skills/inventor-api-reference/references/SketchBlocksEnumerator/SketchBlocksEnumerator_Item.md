# SketchBlocksEnumerator.Item Property

Parent Object: [SketchBlocksEnumerator](../SketchBlocksEnumerator/SketchBlocksEnumerator.md)

## Description

Property that returns an item from the collection. You can provide either the index of the item in the collection, where the first item is index 1, or you can provide the name of the object.

## Syntax

SketchBlocksEnumerator.**Item**( ***Index*** As Variant ) As [SketchBlock](../SketchBlock/SketchBlock.md)

## Property Value

This is a read only property whose value is a [SketchBlock](../SketchBlock/SketchBlock.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant |  |

## Version

Introduced in version 2010
