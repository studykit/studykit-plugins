# UserCoordinateSystemSettings.ShowTriad Property

Parent Object: [UserCoordinateSystemSettings](../UserCoordinateSystemSettings/UserCoordinateSystemSettings.md)

## Description

Gets and sets whether whether or not to display the UCS triad.

## Syntax

UserCoordinateSystemSettings.**ShowTriad**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
