# UserCoordinateSystemSettings.ShowZAxis Property

Parent Object: [UserCoordinateSystemSettings](../UserCoordinateSystemSettings/UserCoordinateSystemSettings.md)

## Description

Gets and sets whether whether or not to make the Z axis visible.

## Syntax

UserCoordinateSystemSettings.**ShowZAxis**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
