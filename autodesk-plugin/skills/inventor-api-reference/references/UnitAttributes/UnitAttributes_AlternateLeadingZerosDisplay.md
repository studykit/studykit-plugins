# UnitAttributes.AlternateLeadingZerosDisplay Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets whether display the leading zeros for alternate values.

## Syntax

UnitAttributes.**AlternateLeadingZerosDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
