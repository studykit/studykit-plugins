# UnitAttributes.AlternateLinearUnits Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets the alternate units of length measurement.

## Syntax

UnitAttributes.**AlternateLinearUnits**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).

## Version

Introduced in version 2016
