# UnitAttributes.AlternateDecimalMarkerType Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets the character to be used as decimal marker for alternate values.

## Syntax

UnitAttributes.**AlternateDecimalMarkerType**() As [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md).

## Version

Introduced in version 2016
