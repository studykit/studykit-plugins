# UnitAttributes.DisplayStyleProp Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets the display style for alternate units.

## Syntax

UnitAttributes.**DisplayStyleProp**() As [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md).

## Version

Introduced in version 2016
