# UnitAttributes.AlternateLinearPrecision Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets the linear precision for alternate units.

## Syntax

UnitAttributes.**AlternateLinearPrecision**() As [LinearPrecisionEnum](../LinearPrecisionEnum.md)

## Property Value

This is a read/write property whose value is a [LinearPrecisionEnum](../LinearPrecisionEnum.md).

## Version

Introduced in version 2016
