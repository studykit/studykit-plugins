# UnitAttributes.PrimaryLinearUnits Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets the primary units of length measurement.

## Syntax

UnitAttributes.**PrimaryLinearUnits**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).

## Version

Introduced in version 2016
