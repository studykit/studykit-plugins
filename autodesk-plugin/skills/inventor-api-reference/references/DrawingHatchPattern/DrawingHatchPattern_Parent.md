# DrawingHatchPattern.Parent Property

Parent Object: [DrawingHatchPattern](../DrawingHatchPattern/DrawingHatchPattern.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DrawingHatchPattern.**Parent**() As [DrawingHatchPatternsManager](../DrawingHatchPatternsManager/DrawingHatchPatternsManager.md)

## Property Value

This is a read only property whose value is a [DrawingHatchPatternsManager](../DrawingHatchPatternsManager/DrawingHatchPatternsManager.md).

## Version

Introduced in version 2021
