# DWGEntityProxy.NativeObject Property

Parent Object: [DWGEntityProxy](../DWGEntityProxy/DWGEntityProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGEntityProxy.**NativeObject**() As [DWGEntity](../DWGEntity/DWGEntity.md)

## Property Value

This is a read only property whose value is a [DWGEntity](../DWGEntity/DWGEntity.md).

## Version

Introduced in version 2016
