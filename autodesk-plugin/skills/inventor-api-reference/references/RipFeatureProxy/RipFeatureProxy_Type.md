# RipFeatureProxy.Type Property

Parent Object: [RipFeatureProxy](../RipFeatureProxy/RipFeatureProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RipFeatureProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
