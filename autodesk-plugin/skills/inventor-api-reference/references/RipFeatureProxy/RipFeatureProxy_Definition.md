# RipFeatureProxy.Definition Property

Parent Object: [RipFeatureProxy](../RipFeatureProxy/RipFeatureProxy.md)

## Description

Gets and sets the RipDefinition object associated with this rip feature. Modifying the RipDefinition object will edit the associated RipFeature.

## Syntax

RipFeatureProxy.**Definition**() As [RipDefinition](../RipDefinition/RipDefinition.md)

## Property Value

This is a read/write property whose value is a [RipDefinition](../RipDefinition/RipDefinition.md).

## Version

Introduced in version 2011
