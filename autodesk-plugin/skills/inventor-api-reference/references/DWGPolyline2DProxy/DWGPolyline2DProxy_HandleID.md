# DWGPolyline2DProxy.HandleID Property

Parent Object: [DWGPolyline2DProxy](../DWGPolyline2DProxy/DWGPolyline2DProxy.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGPolyline2DProxy.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
