# DWGPolyline2DProxy.NativeObject Property

Parent Object: [DWGPolyline2DProxy](../DWGPolyline2DProxy/DWGPolyline2DProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGPolyline2DProxy.**NativeObject**() As [DWGPolyline2D](../DWGPolyline2D/DWGPolyline2D.md)

## Property Value

This is a read only property whose value is a [DWGPolyline2D](../DWGPolyline2D/DWGPolyline2D.md).

## Version

Introduced in version 2016
