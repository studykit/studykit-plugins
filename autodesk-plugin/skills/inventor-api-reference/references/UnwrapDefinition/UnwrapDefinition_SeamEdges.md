# UnwrapDefinition.SeamEdges Property

Parent Object: [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md)

## Description

Read-write property that gets and sets an EdgeCollection that specifies the seam edges.

## Syntax

UnwrapDefinition.**SeamEdges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2025
