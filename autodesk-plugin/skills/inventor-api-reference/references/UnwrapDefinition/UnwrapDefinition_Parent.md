# UnwrapDefinition.Parent Property

Parent Object: [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

UnwrapDefinition.**Parent**() As [UnwrapFeature](../UnwrapFeature/UnwrapFeature.md)

## Property Value

This is a read only property whose value is a [UnwrapFeature](../UnwrapFeature/UnwrapFeature.md).

## Version

Introduced in version 2020
