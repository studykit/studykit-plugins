# DraftAnalyses.Item Property

Parent Object: [DraftAnalyses](../DraftAnalyses/DraftAnalyses.md)

## Description

Method that returns the specified DraftAnalysis object from the collection.

## Syntax

DraftAnalyses.**Item**( ***Index*** As Variant ) As [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Property Value

This is a read only property whose value is a [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the object to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the DraftAnalysis name. If an out of range index or a name of a non-existent draft analysis is provided, an error occurs. |

## Version

Introduced in version 2009
