# DraftAnalyses Object

## Description

The DraftAnalyses collection object provides access to existing draft analyses and supports the ability to create new draft analysis.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../DraftAnalyses/DraftAnalyses_Add.md) | Method that creates a new draft analysis. If the draft analysis is created successfully, a DraftAnalysis object corresponding to the newly created draft analysis is returned from this method. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DraftAnalyses/DraftAnalyses_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DraftAnalyses/DraftAnalyses_Count.md) | Property that specifies the number of items in the collection. |
| [Item](../DraftAnalyses/DraftAnalyses_Item.md) | Method that returns the specified DraftAnalysis object from the collection. |
| [Type](../DraftAnalyses/DraftAnalyses_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[AnalysisManager.DraftAnalyses](../AnalysisManager/AnalysisManager_DraftAnalyses.md)

## Version

Introduced in version 2009
