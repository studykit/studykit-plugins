# LibraryFolder Object

## Description

LibraryFolder Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LibraryFolder/LibraryFolder_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [ChildFolders](../LibraryFolder/LibraryFolder_ChildFolders.md) | Read-only property that returns a collection of all library folders contained directly under this folder in the library. |
| [Name](../LibraryFolder/LibraryFolder_Name.md) | Read-only property that returns the name of the library folder object. |
| [Parent](../LibraryFolder/LibraryFolder_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ParentFolder](../LibraryFolder/LibraryFolder_ParentFolder.md) | Read-only property that returns the parent LibraryFolder for this library folder object. |
| [SketchedSymbolDefinitions](../LibraryFolder/LibraryFolder_SketchedSymbolDefinitions.md) | Read-only property that returns the collection of library sketched symbol definitions under this library folder object. |
| [Type](../LibraryFolder/LibraryFolder_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[LibraryFolder.ParentFolder](../LibraryFolder/LibraryFolder_ParentFolder.md), [LibraryFolders.Add](../LibraryFolders/LibraryFolders_Add.md), [LibraryFolders.Item](../LibraryFolders/LibraryFolders_Item.md), [LibrarySketchedSymbolDefinition.ParentFolder](../LibrarySketchedSymbolDefinition/LibrarySketchedSymbolDefinition_ParentFolder.md)
