# LibraryFolder.SketchedSymbolDefinitions Property

Parent Object: [LibraryFolder](../LibraryFolder/LibraryFolder.md)

## Description

Read-only property that returns the collection of library sketched symbol definitions under this library folder object.

## Syntax

LibraryFolder.**SketchedSymbolDefinitions**() As [LibrarySketchedSymbolDefinitions](../LibrarySketchedSymbolDefinitions/LibrarySketchedSymbolDefinitions.md)

## Property Value

This is a read only property whose value is a [LibrarySketchedSymbolDefinitions](../LibrarySketchedSymbolDefinitions/LibrarySketchedSymbolDefinitions.md).

## Version

Introduced in version 2016
