# LibraryFolder.ChildFolders Property

Parent Object: [LibraryFolder](../LibraryFolder/LibraryFolder.md)

## Description

Read-only property that returns a collection of all library folders contained directly under this folder in the library.

## Syntax

LibraryFolder.**ChildFolders**() As [LibraryFolders](../LibraryFolders/LibraryFolders.md)

## Property Value

This is a read only property whose value is a [LibraryFolders](../LibraryFolders/LibraryFolders.md).

## Version

Introduced in version 2016
