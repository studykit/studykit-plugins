# SketchFillRegion Object

## Description

An object representing a color-filled region in a drawing sketch.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../SketchFillRegion/SketchFillRegion_Delete.md) | Method to delete this SketchFillRegion object. |
| [GetReferenceKey](../SketchFillRegion/SketchFillRegion_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchFillRegion/SketchFillRegion_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../SketchFillRegion/SketchFillRegion_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [Color](../SketchFillRegion/SketchFillRegion_Color.md) | Gets and sets the color of the sketch fill region. |
| [Parent](../SketchFillRegion/SketchFillRegion_Parent.md) | Property that gets the parent object from whom this object can logically be reached. |
| [Profile](../SketchFillRegion/SketchFillRegion_Profile.md) | Property to gets the profile object associated with this sketch fill region. |
| [Type](../SketchFillRegion/SketchFillRegion_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SketchFillRegions.Add](../SketchFillRegions/SketchFillRegions_Add.md), [SketchFillRegions.Item](../SketchFillRegions/SketchFillRegions_Item.md)
