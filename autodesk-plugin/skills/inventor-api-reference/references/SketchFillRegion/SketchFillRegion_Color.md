# SketchFillRegion.Color Property

Parent Object: [SketchFillRegion](../SketchFillRegion/SketchFillRegion.md)

## Description

Gets and sets the color of the sketch fill region.

## Syntax

SketchFillRegion.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
