# SketchFillRegion.Type Property

Parent Object: [SketchFillRegion](../SketchFillRegion/SketchFillRegion.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchFillRegion.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
