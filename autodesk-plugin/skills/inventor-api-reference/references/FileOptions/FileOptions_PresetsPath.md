# FileOptions.PresetsPath Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Gets and sets the location of presets files.

## Syntax

FileOptions.**PresetsPath**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2020
