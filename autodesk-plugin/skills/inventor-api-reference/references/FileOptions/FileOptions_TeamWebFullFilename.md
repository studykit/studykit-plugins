# FileOptions.TeamWebFullFilename Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Gets and sets the full filename of the html file.

## Syntax

FileOptions.**TeamWebFullFilename**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 9
