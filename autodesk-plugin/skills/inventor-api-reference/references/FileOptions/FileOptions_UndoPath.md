# FileOptions.UndoPath Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Gets and sets the location of the temporary file used for transactions.

## Syntax

FileOptions.**UndoPath**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 9
