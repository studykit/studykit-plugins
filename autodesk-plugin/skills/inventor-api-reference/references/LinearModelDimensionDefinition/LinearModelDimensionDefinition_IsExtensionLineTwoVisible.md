# LinearModelDimensionDefinition.IsExtensionLineTwoVisible Property

Parent Object: [LinearModelDimensionDefinition](../LinearModelDimensionDefinition/LinearModelDimensionDefinition.md)

## Description

Read-write property that gets and sets whether the second extension line is visible. This property returns False and cannot be set to True for linear symmetric dimensions. For linear diametric dimensions, this property returns False and cannot be set to True.

## Syntax

LinearModelDimensionDefinition.**IsExtensionLineTwoVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
