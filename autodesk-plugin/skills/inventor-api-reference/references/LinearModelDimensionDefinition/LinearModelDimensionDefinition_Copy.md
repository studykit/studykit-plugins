# LinearModelDimensionDefinition.Copy Method

Parent Object: [LinearModelDimensionDefinition](../LinearModelDimensionDefinition/LinearModelDimensionDefinition.md)

## Description

Method that creates a copy of this LinearModelDimensionDefinition object. The new LinearModelDimensionDefinition object is independent of any dimension. It can edited and used as input to edit an existing dimension or to create a new dimension.

## Syntax

LinearModelDimensionDefinition.**Copy**() As [LinearModelDimensionDefinition](../LinearModelDimensionDefinition/LinearModelDimensionDefinition.md)

## Version

Introduced in version 2018
