# iFeatureTableRow.Index Property

Parent Object: [iFeatureTableRow](../iFeatureTableRow/iFeatureTableRow.md)

## Description

Property that returns the index of this row within the iFeatureTableRows collection where the first row has an index of 1.

## Syntax

iFeatureTableRow.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
