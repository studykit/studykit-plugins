# GraphicsNormalSet.Type Property

Parent Object: [GraphicsNormalSet](../GraphicsNormalSet/GraphicsNormalSet.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

GraphicsNormalSet.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
