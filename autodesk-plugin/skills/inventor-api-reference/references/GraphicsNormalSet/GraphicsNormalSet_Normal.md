# GraphicsNormalSet.Normal Property

Parent Object: [GraphicsNormalSet](../GraphicsNormalSet/GraphicsNormalSet.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

GraphicsNormalSet.**Normal**( ***Index*** As Long ) As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).
