# DrawingStylesManager.HoleTableStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the HoleTableStylesEnumerator object.

## Syntax

DrawingStylesManager.**HoleTableStyles**() As [HoleTableStylesEnumerator](../HoleTableStylesEnumerator/HoleTableStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [HoleTableStylesEnumerator](../HoleTableStylesEnumerator/HoleTableStylesEnumerator.md).

## Version

Introduced in version 2009
