# DrawingStylesManager.Layers Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the LayersEnumerator object. This object provides access to all the layers.

## Syntax

DrawingStylesManager.**Layers**() As [LayersEnumerator](../LayersEnumerator/LayersEnumerator.md)

## Property Value

This is a read only property whose value is a [LayersEnumerator](../LayersEnumerator/LayersEnumerator.md).
