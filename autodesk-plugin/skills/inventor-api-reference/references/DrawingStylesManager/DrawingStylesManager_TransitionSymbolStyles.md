# DrawingStylesManager.TransitionSymbolStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Read-only property that returns the TransitionSymbolStylesEnumerator object.

## Syntax

DrawingStylesManager.**TransitionSymbolStyles**() As [TransitionSymbolStylesEnumerator](../TransitionSymbolStylesEnumerator/TransitionSymbolStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [TransitionSymbolStylesEnumerator](../TransitionSymbolStylesEnumerator/TransitionSymbolStylesEnumerator.md).

## Version

Introduced in version 2025.1
