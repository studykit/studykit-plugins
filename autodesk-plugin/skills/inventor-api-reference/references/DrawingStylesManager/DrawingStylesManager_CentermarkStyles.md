# DrawingStylesManager.CentermarkStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the CentermarkStylesEnumerator object. This object provides access to all the center mark styles.

## Syntax

DrawingStylesManager.**CentermarkStyles**() As [CentermarkStylesEnumerator](../CentermarkStylesEnumerator/CentermarkStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [CentermarkStylesEnumerator](../CentermarkStylesEnumerator/CentermarkStylesEnumerator.md).

## Version

Introduced in version 2010
