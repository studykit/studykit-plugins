# DrawingStylesManager.SurfaceTextureStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the SurfaceTextureStylesEnumerator object.

## Syntax

DrawingStylesManager.**SurfaceTextureStyles**() As [SurfaceTextureStylesEnumerator](../SurfaceTextureStylesEnumerator/SurfaceTextureStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [SurfaceTextureStylesEnumerator](../SurfaceTextureStylesEnumerator/SurfaceTextureStylesEnumerator.md).

## Version

Introduced in version 2009
