# DrawingStylesManager.StandardStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the DrawingStandardStylesEnumerator object. This object provides access to all the standard styles.

## Syntax

DrawingStylesManager.**StandardStyles**() As [DrawingStandardStylesEnumerator](../DrawingStandardStylesEnumerator/DrawingStandardStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [DrawingStandardStylesEnumerator](../DrawingStandardStylesEnumerator/DrawingStandardStylesEnumerator.md).

## Version

Introduced in version 9
