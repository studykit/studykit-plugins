# DrawingStylesManager.DimensionStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the DimensionStylesEnumerator object. This object provides access to all the dimension styles.

## Syntax

DrawingStylesManager.**DimensionStyles**() As [DimensionStylesEnumerator](../DimensionStylesEnumerator/DimensionStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [DimensionStylesEnumerator](../DimensionStylesEnumerator/DimensionStylesEnumerator.md).

## Version

Introduced in version 9
