# DrawingStylesManager.Styles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Read-only property that returns the Styles object.

## Syntax

DrawingStylesManager.**Styles**() As [Styles](../Styles/Styles.md)

## Property Value

This is a read only property whose value is a [Styles](../Styles/Styles.md).

## Version

Introduced in version 2008
