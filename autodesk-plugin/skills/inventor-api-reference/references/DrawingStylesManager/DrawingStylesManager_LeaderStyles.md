# DrawingStylesManager.LeaderStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the LeaderStylesEnumerator object.

## Syntax

DrawingStylesManager.**LeaderStyles**() As [LeaderStylesEnumerator](../LeaderStylesEnumerator/LeaderStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [LeaderStylesEnumerator](../LeaderStylesEnumerator/LeaderStylesEnumerator.md).

## Version

Introduced in version 2009
