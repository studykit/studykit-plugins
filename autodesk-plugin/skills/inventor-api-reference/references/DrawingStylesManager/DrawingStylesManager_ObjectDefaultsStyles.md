# DrawingStylesManager.ObjectDefaultsStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that gets the ObjectDefaultsStylesEnumerator collection object.

## Syntax

DrawingStylesManager.**ObjectDefaultsStyles**() As [ObjectDefaultsStylesEnumerator](../ObjectDefaultsStylesEnumerator/ObjectDefaultsStylesEnumerator.md)

## Property Value

This is a read only property whose value is an [ObjectDefaultsStylesEnumerator](../ObjectDefaultsStylesEnumerator/ObjectDefaultsStylesEnumerator.md).

## Version

Introduced in version 9
