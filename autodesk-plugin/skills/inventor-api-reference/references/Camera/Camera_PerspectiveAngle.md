# Camera.PerspectiveAngle Property

Parent Object: [Camera](../Camera/Camera.md)

## Description

Gets/Sets the Perspective Angle.

## Syntax

Camera.**PerspectiveAngle**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 5.3
