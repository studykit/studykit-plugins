# Camera.ModelToViewTransformation Property

Parent Object: [Camera](../Camera/Camera.md)

## Description

Read-only property that returns the transformation matrix from model space to view space.

## Syntax

Camera.**ModelToViewTransformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2017
