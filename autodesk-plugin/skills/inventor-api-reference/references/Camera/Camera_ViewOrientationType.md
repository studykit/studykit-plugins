# Camera.ViewOrientationType Property

Parent Object: [Camera](../Camera/Camera.md)

## Description

Gets/Sets the type of orientation of the camera (Top, Iso, etc. or arbitrary).

## Syntax

Camera.**ViewOrientationType**() As [ViewOrientationTypeEnum](../ViewOrientationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ViewOrientationTypeEnum](../ViewOrientationTypeEnum.md).

## Version

Introduced in version 4
