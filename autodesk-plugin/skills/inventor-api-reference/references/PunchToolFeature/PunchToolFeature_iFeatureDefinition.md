# PunchToolFeature.iFeatureDefinition Property

Parent Object: [PunchToolFeature](../PunchToolFeature/PunchToolFeature.md)

## Description

Gets and sets the associated iFeatureDefinition.

## Syntax

PunchToolFeature.**iFeatureDefinition**() As [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md)

## Property Value

This is a read/write property whose value is an [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md).

## Version

Introduced in version 2009
