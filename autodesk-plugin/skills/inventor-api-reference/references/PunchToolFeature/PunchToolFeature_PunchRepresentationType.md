# PunchToolFeature.PunchRepresentationType Property

Parent Object: [PunchToolFeature](../PunchToolFeature/PunchToolFeature.md)

## Description

Gets and sets punch tool feature representation type.

## Syntax

PunchToolFeature.**PunchRepresentationType**() As [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md).

## Version

Introduced in version 2017
