# PunchToolFeature.Shared Property

Parent Object: [PunchToolFeature](../PunchToolFeature/PunchToolFeature.md)

## Description

Gets and sets whether the part feature is shared or not, applies only to surface features.

## Syntax

PunchToolFeature.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
