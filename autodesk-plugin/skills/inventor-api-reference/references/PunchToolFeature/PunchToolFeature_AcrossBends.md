# PunchToolFeature.AcrossBends Property

Parent Object: [PunchToolFeature](../PunchToolFeature/PunchToolFeature.md)

## Description

Gets and sets the property that defines the across bends.

## Syntax

PunchToolFeature.**AcrossBends**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
