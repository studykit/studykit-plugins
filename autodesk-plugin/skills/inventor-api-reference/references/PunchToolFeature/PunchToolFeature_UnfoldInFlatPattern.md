# PunchToolFeature.UnfoldInFlatPattern Property

Parent Object: [PunchToolFeature](../PunchToolFeature/PunchToolFeature.md)

## Description

Read-write property that gets and sets whether the punch tool feature is unfoldable in flat pattern.

## Syntax

PunchToolFeature.**UnfoldInFlatPattern**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
