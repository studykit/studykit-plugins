# InventorVBAArgument.Parent Property

Parent Object: [InventorVBAArgument](../InventorVBAArgument/InventorVBAArgument.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

InventorVBAArgument.**Parent**() As [InventorVBAMember](../InventorVBAMember/InventorVBAMember.md)

## Property Value

This is a read only property whose value is an [InventorVBAMember](../InventorVBAMember/InventorVBAMember.md).

## Version

Introduced in version 6
