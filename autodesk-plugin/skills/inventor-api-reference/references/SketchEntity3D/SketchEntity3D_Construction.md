# SketchEntity3D.Construction Property

Parent Object: [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchEntity3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
