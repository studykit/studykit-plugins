# SketchEquationCurveProxy.LineDefinitionSpace Property

Parent Object: [SketchEquationCurveProxy](../SketchEquationCurveProxy/SketchEquationCurveProxy.md)

## Description

Gets and sets the LineDefinitionSpace applied to this sketch spline.

## Syntax

SketchEquationCurveProxy.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2014
