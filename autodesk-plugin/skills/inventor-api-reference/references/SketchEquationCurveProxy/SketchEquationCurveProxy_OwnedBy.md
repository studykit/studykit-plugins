# SketchEquationCurveProxy.OwnedBy Property

Parent Object: [SketchEquationCurveProxy](../SketchEquationCurveProxy/SketchEquationCurveProxy.md)

## Description

Indicates entity or entities that own this object.

## Syntax

SketchEquationCurveProxy.**OwnedBy**() As [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md).

## Version

Introduced in version 2014
