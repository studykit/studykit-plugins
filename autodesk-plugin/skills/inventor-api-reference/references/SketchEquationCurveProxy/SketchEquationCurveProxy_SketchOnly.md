# SketchEquationCurveProxy.SketchOnly Property

Parent Object: [SketchEquationCurveProxy](../SketchEquationCurveProxy/SketchEquationCurveProxy.md)

## Description

Gets and sets whether this entity is visible only when editing the sketch.

## Syntax

SketchEquationCurveProxy.**SketchOnly**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
