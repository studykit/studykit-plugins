# FlangeDefinition.Type Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FlangeDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
