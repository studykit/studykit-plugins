# FlangeDefinition.EdgeSetCount Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Read-only property that returns the flange edge set count currently defined in this flange definition.

## Syntax

FlangeDefinition.**EdgeSetCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2025
