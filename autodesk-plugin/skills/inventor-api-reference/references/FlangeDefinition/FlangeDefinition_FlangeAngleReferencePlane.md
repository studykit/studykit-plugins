# FlangeDefinition.FlangeAngleReferencePlane Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Read-write property that gets and sets the reference plane(WorkPlane or Face) object that is used to define the flange angle. When set this property the WorkPlane or Face should be parallel to the flange edge, otherwise an error would occur.

## Syntax

FlangeDefinition.**FlangeAngleReferencePlane**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2021
