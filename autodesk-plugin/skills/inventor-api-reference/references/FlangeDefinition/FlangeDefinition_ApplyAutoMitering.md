# FlangeDefinition.ApplyAutoMitering Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Defines if auto mitering should be performed.

## Syntax

FlangeDefinition.**ApplyAutoMitering**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
