# CropViewAnnotationStyleSettings.Parent Property

Parent Object: [CropViewAnnotationStyleSettings](../CropViewAnnotationStyleSettings/CropViewAnnotationStyleSettings.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

CropViewAnnotationStyleSettings.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2025.1
