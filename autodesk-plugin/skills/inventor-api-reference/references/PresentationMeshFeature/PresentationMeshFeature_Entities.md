# PresentationMeshFeature.Entities Property

Parent Object: [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Description

Read-only property that returns the PresentationMeshFeatureEntitiesEnumerator collection object.

## Syntax

PresentationMeshFeature.**Entities**() As [PresentationMeshFeatureEntitiesEnumerator](../PresentationMeshFeatureEntitiesEnumerator/PresentationMeshFeatureEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationMeshFeatureEntitiesEnumerator](../PresentationMeshFeatureEntitiesEnumerator/PresentationMeshFeatureEntitiesEnumerator.md).

## Version

Introduced in version 2018
