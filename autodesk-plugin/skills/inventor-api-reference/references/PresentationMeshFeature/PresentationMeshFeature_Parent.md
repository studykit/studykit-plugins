# PresentationMeshFeature.Parent Property

Parent Object: [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PresentationMeshFeature.**Parent**() As [PresentationMeshFeatureSet](../PresentationMeshFeatureSet/PresentationMeshFeatureSet.md)

## Property Value

This is a read only property whose value is a [PresentationMeshFeatureSet](../PresentationMeshFeatureSet/PresentationMeshFeatureSet.md).

## Version

Introduced in version 2018
