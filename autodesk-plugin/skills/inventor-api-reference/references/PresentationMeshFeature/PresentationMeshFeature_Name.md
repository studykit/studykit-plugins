# PresentationMeshFeature.Name Property

Parent Object: [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Description

Read-only property that gets the name of the object.

## Syntax

PresentationMeshFeature.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
