# SweepProfileScalingEnum Enumerator

## Description

Enum indicating the type of profile scaling for the sweep feature.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kNoProfileScaling | 59395 | No scaling for profile. |
| kXProfileScaling | 59394 | Profile scaled along X axis only. |
| kXYProfileScaling | 59393 | Profile scaled along both X and Y axes. |

## Version

Introduced in version 11
