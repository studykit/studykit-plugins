# WeldSymbolStyle.ConvertToLocal Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

WeldSymbolStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2024
