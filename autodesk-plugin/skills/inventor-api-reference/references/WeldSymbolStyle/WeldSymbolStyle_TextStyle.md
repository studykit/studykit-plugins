# WeldSymbolStyle.TextStyle Property

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Gets and sets the text style used for the weld symbol style.

## Syntax

WeldSymbolStyle.**TextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2024
