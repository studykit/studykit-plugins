# WeldSymbolStyle.IdentificationLineType Property

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Gets and sets the default identification line type.

## Syntax

WeldSymbolStyle.**IdentificationLineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2024
