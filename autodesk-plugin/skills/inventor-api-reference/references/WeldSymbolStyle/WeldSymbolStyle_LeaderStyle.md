# WeldSymbolStyle.LeaderStyle Property

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Gets and sets the leader style used for the weld symbol style.

## Syntax

WeldSymbolStyle.**LeaderStyle**() As [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Property Value

This is a read/write property whose value is a [LeaderStyle](../LeaderStyle/LeaderStyle.md).

## Version

Introduced in version 2024
