# WeldSymbolStyle.GetBackingSymbolsFilter Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Method that gets the backing symbol filter.

## Syntax

WeldSymbolStyle.**GetBackingSymbolsFilter**( ***BackingSymbolFilter***() As [BackingSymbolTypeEnum](../BackingSymbolTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| BackingSymbolFilter | [BackingSymbolTypeEnum](../BackingSymbolTypeEnum.md) | Output an array of BackingSymbolsTypeEnum indicating available backing symbols. |

## Version

Introduced in version 2024
