# WeldSymbolStyle.IdentificationLinePlacement Property

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Gets and sets the default identification line placement.

## Syntax

WeldSymbolStyle.**IdentificationLinePlacement**() As [IdentificationLinePlacementEnum](../IdentificationLinePlacementEnum.md)

## Property Value

This is a read/write property whose value is an [IdentificationLinePlacementEnum](../IdentificationLinePlacementEnum.md).

## Version

Introduced in version 2024
