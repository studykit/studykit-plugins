# WeldSymbolStyle.SetContourSymbolFilterVariationType Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Sets the variation type of a contour symbol filter.

## Syntax

WeldSymbolStyle.**SetContourSymbolFilterVariationType**( ***ContourSymbolFilter*** As [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md), ***ContourSymbolVariation*** As [ContourSymbolVariationTypeEnum](../ContourSymbolVariationTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ContourSymbolFilter | [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md) | Input ContourSymbolTypeEnum to specify a contour symbol filter. |
| ContourSymbolVariation | [ContourSymbolVariationTypeEnum](../ContourSymbolVariationTypeEnum.md) | Intput ContourSymbolVariationTypeEnum value that indicates the included contour symbol filter variation type. |

## Version

Introduced in version 2024
