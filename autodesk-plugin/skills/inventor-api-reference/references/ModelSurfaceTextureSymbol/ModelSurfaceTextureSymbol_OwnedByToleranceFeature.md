# ModelSurfaceTextureSymbol.OwnedByToleranceFeature Property

Parent Object: [ModelSurfaceTextureSymbol](../ModelSurfaceTextureSymbol/ModelSurfaceTextureSymbol.md)

## Description

Returns the owning ModelToleranceFeature object.

## Syntax

ModelSurfaceTextureSymbol.**OwnedByToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2019
