# PrintManager.PaperWidth Property

Parent Object: [PrintManager](../PrintManager/PrintManager.md)

## Description

Gets/Sets the width of the paper to be printed (max value is 327.67cm). Setting this, sets PaperSize to kPaperSizeCustom.

## Syntax

PrintManager.**PaperWidth**() As Double

## Property Value

This is a read/write property whose value is a Double.
