# Light.Position Property

Parent Object: [Light](../Light/Light.md)

## Description

Gets and sets the position of the light in model space.

## Syntax

Light.**Position**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 10
