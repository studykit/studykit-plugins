# Light.LightDefinitionType Property

Parent Object: [Light](../Light/Light.md)

## Description

Returning the light definition type.

## Syntax

Light.**LightDefinitionType**() As [LightDefinitionTypeEnum](../LightDefinitionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LightDefinitionTypeEnum](../LightDefinitionTypeEnum.md).

## Version

Introduced in version 10
