# Light Object

## Description

The Light object provides access to all of the properties that define a specific light source.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Light/Light_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Color](../Light/Light_Color.md) | Gets and sets the color of the light source. |
| [ConstantAttenuation](../Light/Light_ConstantAttenuation.md) | Gets and sets the QuadraticAttenuation value. |
| [Direction](../Light/Light_Direction.md) | Gets and sets the direction of the light. |
| [Intensity](../Light/Light_Intensity.md) | Gets and sets the intensity value. |
| [LightDefinitionType](../Light/Light_LightDefinitionType.md) | Returning the light definition type. |
| [LightType](../Light/Light_LightType.md) | Returning kLightObject indicating the type of object. |
| [LinearAttenuation](../Light/Light_LinearAttenuation.md) | Gets and sets the LinearAttenuation value. |
| [On](../Light/Light_On.md) | Gets and sets whether this light is switched on. |
| [Parent](../Light/Light_Parent.md) | Property returning the parent of the object. |
| [Position](../Light/Light_Position.md) | Gets and sets the position of the light in model space. |
| [QuadraticAttenuation](../Light/Light_QuadraticAttenuation.md) | Gets and sets the QuadraticAttenuation value. |
| [SpotLightCutoffAngle](../Light/Light_SpotLightCutoffAngle.md) | Gets and sets the SpotLightCutoffAngle value. |
| [SpotLightFalloff](../Light/Light_SpotLightFalloff.md) | Gets and sets the SpotLightFalloff value. |
| [SpotLightInnerAngle](../Light/Light_SpotLightInnerAngle.md) | Gets and sets the SpotLightInnerAngle value. |
| [Type](../Light/Light_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Lights.Add](../Lights/Lights_Add.md), [Lights.Item](../Lights/Lights_Item.md)

## Version

Introduced in version 10
