# Light.LightType Property

Parent Object: [Light](../Light/Light.md)

## Description

Returning kLightObject indicating the type of object.

## Syntax

Light.**LightType**() As [LightTypeEnum](../LightTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LightTypeEnum](../LightTypeEnum.md).

## Version

Introduced in version 10
