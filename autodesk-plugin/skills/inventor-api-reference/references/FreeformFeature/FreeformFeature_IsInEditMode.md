# FreeformFeature.IsInEditMode Property

Parent Object: [FreeformFeature](../FreeformFeature/FreeformFeature.md)

## Description

Gets/Sets the edit mode of this freeform feature.

## Syntax

FreeformFeature.**IsInEditMode**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
