# FreeformFeature.Appearance Property

Parent Object: [FreeformFeature](../FreeformFeature/FreeformFeature.md)

## Description

Gets and sets the current appearance of the feature.

## Syntax

FreeformFeature.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2015
