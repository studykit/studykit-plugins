# RotateRotateConstraintProxy.ResultOfiMate Property

Parent Object: [RotateRotateConstraintProxy](../RotateRotateConstraintProxy/RotateRotateConstraintProxy.md)

## Description

Property that indicates if this iMateResult represents a composite iMate.

## Syntax

RotateRotateConstraintProxy.**ResultOfiMate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
