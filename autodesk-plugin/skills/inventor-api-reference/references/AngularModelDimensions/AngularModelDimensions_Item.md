# AngularModelDimensions.Item Property

Parent Object: [AngularModelDimensions](../AngularModelDimensions/AngularModelDimensions.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

AngularModelDimensions.**Item**( ***Index*** As Variant ) As [AngularModelDimension](../AngularModelDimension/AngularModelDimension.md)

## Property Value

This is a read only property whose value is an [AngularModelDimension](../AngularModelDimension/AngularModelDimension.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | The index or name of the item within the collection to return. |

## Version

Introduced in version 2018
