# SketchHolePlacementDefinition.Type Property

Parent Object: [SketchHolePlacementDefinition](../SketchHolePlacementDefinition/SketchHolePlacementDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchHolePlacementDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
