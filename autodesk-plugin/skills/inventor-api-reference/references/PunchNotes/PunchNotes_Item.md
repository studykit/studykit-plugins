# PunchNotes.Item Property

Parent Object: [PunchNotes](../PunchNotes/PunchNotes.md)

## Description

Returns the specified PunchNote object from the collection.

## Syntax

PunchNotes.**Item**( ***Index*** As Long ) As [PunchNote](../PunchNote/PunchNote.md)

## Property Value

This is a read only property whose value is a [PunchNote](../PunchNote/PunchNote.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
