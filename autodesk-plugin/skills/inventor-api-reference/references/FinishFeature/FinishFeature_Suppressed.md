# FinishFeature.Suppressed Property

Parent Object: [FinishFeature](../FinishFeature/FinishFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

FinishFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
