# DSDegreeOfFreedom.MaxBoundsValue Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets the maximum value of the joint. The value is in database units, which for linear bounds is centimeters and radial bounds is radians.

## Syntax

DSDegreeOfFreedom.**MaxBoundsValue**() As [DSValue](../DSValue/DSValue.md)

## Property Value

This is a read only property whose value is a [DSValue](../DSValue/DSValue.md).

## Version

Introduced in version 2013
