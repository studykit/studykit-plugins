# DSDegreeOfFreedom.DegreeOfFreedomType Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Read-only property that indicates if this degree of freedom is for a rotation or translation.

## Syntax

DSDegreeOfFreedom.**DegreeOfFreedomType**() As [DSDegreeOfFreedomTypeEnum](../DSDegreeOfFreedomTypeEnum.md)

## Property Value

This is a read only property whose value is a [DSDegreeOfFreedomTypeEnum](../DSDegreeOfFreedomTypeEnum.md).

## Version

Introduced in version 2013
