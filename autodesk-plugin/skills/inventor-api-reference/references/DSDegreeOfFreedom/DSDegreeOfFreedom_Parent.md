# DSDegreeOfFreedom.Parent Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Returns the DSJointDefinition object this degree of freedom is associated with.

## Syntax

DSDegreeOfFreedom.**Parent**() As [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md)

## Property Value

This is a read only property whose value is a [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md).

## Version

Introduced in version 2013
