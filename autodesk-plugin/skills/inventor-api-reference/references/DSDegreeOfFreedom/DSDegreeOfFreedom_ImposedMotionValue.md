# DSDegreeOfFreedom.ImposedMotionValue Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Read-only property returning a DSValue object that you can use to get and set the imposed motion. This property returns nothing in the case where the ImposedMotionType property is set to kNoImposedMotion.

## Syntax

DSDegreeOfFreedom.**ImposedMotionValue**() As [DSValue](../DSValue/DSValue.md)

## Property Value

This is a read only property whose value is a [DSValue](../DSValue/DSValue.md).

## Version

Introduced in version 2013
