# DSDegreeOfFreedom.IsMinBoundsEnabled Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets and sets if the minimum bounds of the joint are defined.

## Syntax

DSDegreeOfFreedom.**IsMinBoundsEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
