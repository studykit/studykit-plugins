# SketchEquationCurve.StartSketchPoint Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Gets the sketch point that defines the position of the start of the elliptical arc.

## Syntax

SketchEquationCurve.**StartSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 2014
