# SketchEquationCurve.CurvatureDisplay Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Specifies whether curvature information is displayed for the spline.

## Syntax

SketchEquationCurve.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
