# SketchEquationCurve.LineWeight Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Gets and sets the LineWeight applied to this sketch spline.

## Syntax

SketchEquationCurve.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
