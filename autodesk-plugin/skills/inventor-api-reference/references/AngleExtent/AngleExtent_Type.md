# AngleExtent.Type Property

Parent Object: [AngleExtent](../AngleExtent/AngleExtent.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

AngleExtent.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
