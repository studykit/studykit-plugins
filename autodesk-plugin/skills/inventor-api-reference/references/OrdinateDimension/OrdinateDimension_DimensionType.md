# OrdinateDimension.DimensionType Property

Parent Object: [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Description

Property that returns the dimension type.

## Syntax

OrdinateDimension.**DimensionType**() As [DimensionTypeEnum](../DimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [DimensionTypeEnum](../DimensionTypeEnum.md).

## Version

Introduced in version 2009
