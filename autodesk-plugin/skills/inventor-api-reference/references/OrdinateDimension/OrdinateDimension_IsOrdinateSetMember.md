# OrdinateDimension.IsOrdinateSetMember Property

Parent Object: [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Description

Property that returns whether this dimension is a member of an ordinate dimension set. If this property returns True, the OrdinateDimensionSet object is returned by the OrdinateDimensionSet property.

## Syntax

OrdinateDimension.**IsOrdinateSetMember**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
