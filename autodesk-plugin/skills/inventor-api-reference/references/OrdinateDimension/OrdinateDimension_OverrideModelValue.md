# OrdinateDimension.OverrideModelValue Property

Parent Object: [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Description

Gets and sets the NominalValue setting.

## Syntax

OrdinateDimension.**OverrideModelValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
