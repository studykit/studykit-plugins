# PointCloudCrop.KeepInside Property

Parent Object: [PointCloudCrop](../PointCloudCrop/PointCloudCrop.md)

## Description

Read-only property that returns whether the cloud points in the bounding box are kept or not in this crop.

## Syntax

PointCloudCrop.**KeepInside**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2016
