# DesignViewRepresentation.Delete Method

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Method that deletes the DesignViewRepresentation. The method returns an error if the DesignViewType is kMasterDesignViewType.

## Syntax

DesignViewRepresentation.**Delete**()

## Version

Introduced in version 11
