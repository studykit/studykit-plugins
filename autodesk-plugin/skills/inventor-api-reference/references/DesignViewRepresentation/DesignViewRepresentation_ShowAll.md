# DesignViewRepresentation.ShowAll Method

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Method that makes all components visible in this representation.

## Syntax

DesignViewRepresentation.**ShowAll**()

## Version

Introduced in version 11
