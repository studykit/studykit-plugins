# DesignViewRepresentation.AutoSaveCamera Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Gets and sets weather the design view representations camera is saved automatically.

## Syntax

DesignViewRepresentation.**AutoSaveCamera**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
