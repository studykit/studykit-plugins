# DesignViewRepresentation.DesignViewType Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Property that returns the design view representation type. This can return kPrimaryDesignViewType, kPrivateDesignViewType, kPublicDesignViewType and kTransientDesignViewType.

## Syntax

DesignViewRepresentation.**DesignViewType**() As [DesignViewTypeEnum](../DesignViewTypeEnum.md)

## Property Value

This is a read only property whose value is a [DesignViewTypeEnum](../DesignViewTypeEnum.md).

## Version

Introduced in version 11
