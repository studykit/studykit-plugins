# DesignViewRepresentation.Parent Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Property that returns the parent RepresentationsManager object.

## Syntax

DesignViewRepresentation.**Parent**() As [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md)

## Property Value

This is a read only property whose value is a [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md).

## Version

Introduced in version 11
