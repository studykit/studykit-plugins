# DesignViewRepresentation.Publish Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Gets and sets whether this design view representation is published.

## Syntax

DesignViewRepresentation.**Publish**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
