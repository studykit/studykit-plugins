# Parameters.FinishParameters Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Gets the collection of Finish Feature parameters.

## Syntax

Parameters.**FinishParameters**() As [FinishParameters](../FinishParameters/FinishParameters.md)

## Property Value

This is a read only property whose value is a [FinishParameters](../FinishParameters/FinishParameters.md).

## Version

Introduced in version 2024
