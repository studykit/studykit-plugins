# Parameters.ReferenceParameters Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Property that returns the ReferenceParameters collection object.

## Syntax

Parameters.**ReferenceParameters**() As [ReferenceParameters](../ReferenceParameters/ReferenceParameters.md)

## Property Value

This is a read only property whose value is a [ReferenceParameters](../ReferenceParameters/ReferenceParameters.md).

## Version

Introduced in version 4
