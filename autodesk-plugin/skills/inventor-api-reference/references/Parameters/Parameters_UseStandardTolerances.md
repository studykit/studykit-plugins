# Parameters.UseStandardTolerances Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Gets/Sets the boolean flag indicating whether to use standard tolerances for dimensions in this document.

## Syntax

Parameters.**UseStandardTolerances**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
