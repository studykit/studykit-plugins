# Parameters.CustomParameterGroups Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Property that returns the CustomParameterGroups collection object.

## Syntax

Parameters.**CustomParameterGroups**() As [CustomParameterGroups](../CustomParameterGroups/CustomParameterGroups.md)

## Property Value

This is a read only property whose value is a [CustomParameterGroups](../CustomParameterGroups/CustomParameterGroups.md).
