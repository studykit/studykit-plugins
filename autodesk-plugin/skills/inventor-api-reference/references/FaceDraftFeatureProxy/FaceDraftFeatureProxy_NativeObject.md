# FaceDraftFeatureProxy.NativeObject Property

Parent Object: [FaceDraftFeatureProxy](../FaceDraftFeatureProxy/FaceDraftFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FaceDraftFeatureProxy.**NativeObject**() As [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md)

## Property Value

This is a read only property whose value is a [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md).

## Version

Introduced in version 9
