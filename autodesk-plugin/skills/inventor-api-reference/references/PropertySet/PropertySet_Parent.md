# PropertySet.Parent Property

Parent Object: [PropertySet](../PropertySet/PropertySet.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

PropertySet.**Parent**() As [PropertySets](../PropertySets/PropertySets.md)

## Property Value

This is a read only property whose value is a [PropertySets](../PropertySets/PropertySets.md).

## Version

Introduced in version 4
