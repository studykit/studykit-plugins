# ContentFamiliesEnumerator Object

## Description

Object allowing enumeration of ContentFamily objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ContentFamiliesEnumerator/ContentFamiliesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ContentFamiliesEnumerator/ContentFamiliesEnumerator_Count.md) | Property that returns the number of ContentFamily objects in the collection. |
| [Item](../ContentFamiliesEnumerator/ContentFamiliesEnumerator_Item.md) | Returns the constant that indicates the type of this object. |
| [Type](../ContentFamiliesEnumerator/ContentFamiliesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ContentTreeViewNode.Families](../ContentTreeViewNode/ContentTreeViewNode_Families.md)

## Version

Introduced in version 2010
