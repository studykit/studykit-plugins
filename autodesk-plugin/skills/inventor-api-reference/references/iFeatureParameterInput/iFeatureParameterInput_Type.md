# iFeatureParameterInput.Type Property

Parent Object: [iFeatureParameterInput](../iFeatureParameterInput/iFeatureParameterInput.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iFeatureParameterInput.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
