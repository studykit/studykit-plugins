# iFeatureTableColumn.DisplayHeading Property

Parent Object: [iFeatureTableColumn](../iFeatureTableColumn/iFeatureTableColumn.md)

## Description

Property that returns the heading of the column as seen in the iFeature author command.

## Syntax

iFeatureTableColumn.**DisplayHeading**() As String

## Property Value

This is a read only property whose value is a String.
