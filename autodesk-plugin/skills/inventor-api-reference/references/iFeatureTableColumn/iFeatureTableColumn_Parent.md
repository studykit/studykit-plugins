# iFeatureTableColumn.Parent Property

Parent Object: [iFeatureTableColumn](../iFeatureTableColumn/iFeatureTableColumn.md)

## Description

Property that returns the parent iFeatureTable.

## Syntax

iFeatureTableColumn.**Parent**() As [iFeatureTable](../iFeatureTable/iFeatureTable.md)

## Property Value

This is a read only property whose value is an [iFeatureTable](../iFeatureTable/iFeatureTable.md).

## Version

Introduced in version 2009
