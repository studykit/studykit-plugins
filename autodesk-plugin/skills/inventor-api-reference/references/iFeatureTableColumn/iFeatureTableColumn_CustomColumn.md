# iFeatureTableColumn.CustomColumn Property

Parent Object: [iFeatureTableColumn](../iFeatureTableColumn/iFeatureTableColumn.md)

## Description

Gets and sets whether this is a custom parameter column.

## Syntax

iFeatureTableColumn.**CustomColumn**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
