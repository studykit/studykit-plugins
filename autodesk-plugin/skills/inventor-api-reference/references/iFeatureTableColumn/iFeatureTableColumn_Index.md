# iFeatureTableColumn.Index Property

Parent Object: [iFeatureTableColumn](../iFeatureTableColumn/iFeatureTableColumn.md)

## Description

Property that returns the index of this column within the iFeatureTableColumns collection where the first column has an index of 1.

## Syntax

iFeatureTableColumn.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
