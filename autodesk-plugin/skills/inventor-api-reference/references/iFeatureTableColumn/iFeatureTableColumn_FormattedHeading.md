# iFeatureTableColumn.FormattedHeading Property

Parent Object: [iFeatureTableColumn](../iFeatureTableColumn/iFeatureTableColumn.md)

## Description

Property that returns the heading of the column in XML format.

## Syntax

iFeatureTableColumn.**FormattedHeading**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2009
