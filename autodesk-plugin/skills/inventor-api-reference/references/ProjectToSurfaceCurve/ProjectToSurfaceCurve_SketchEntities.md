# ProjectToSurfaceCurve.SketchEntities Property

Parent Object: [ProjectToSurfaceCurve](../ProjectToSurfaceCurve/ProjectToSurfaceCurve.md)

## Description

Read-only property that returns a collection of sketch entities that belong to the project to surface curve. The sketch entities returned by this property cannot be edited or deleted because they are associated with the project to surface curve in the model. T.

## Syntax

ProjectToSurfaceCurve.**SketchEntities**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 2021
