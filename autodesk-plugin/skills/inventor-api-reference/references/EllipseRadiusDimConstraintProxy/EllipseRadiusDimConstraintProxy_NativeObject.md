# EllipseRadiusDimConstraintProxy.NativeObject Property

Parent Object: [EllipseRadiusDimConstraintProxy](../EllipseRadiusDimConstraintProxy/EllipseRadiusDimConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

EllipseRadiusDimConstraintProxy.**NativeObject**() As [EllipseRadiusDimConstraint](../EllipseRadiusDimConstraint/EllipseRadiusDimConstraint.md)

## Property Value

This is a read only property whose value is an [EllipseRadiusDimConstraint](../EllipseRadiusDimConstraint/EllipseRadiusDimConstraint.md).

## Version

Introduced in version 6
