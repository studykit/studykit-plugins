# DiameterModelDimension.TopToleranceFeature Property

Parent Object: [DiameterModelDimension](../DiameterModelDimension/DiameterModelDimension.md)

## Description

Returns the top ModelToleranceFeature object.

## Syntax

DiameterModelDimension.**TopToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2019
