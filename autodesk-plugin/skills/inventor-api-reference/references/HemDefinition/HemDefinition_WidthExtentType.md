# HemDefinition.WidthExtentType Property

Parent Object: [HemDefinition](../HemDefinition/HemDefinition.md)

## Description

Property that returns the method used to define the width extent.

## Syntax

HemDefinition.**WidthExtentType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).

## Version

Introduced in version 2009
