# HemDefinition.UnfoldMethod Property

Parent Object: [HemDefinition](../HemDefinition/HemDefinition.md)

## Description

Gets and sets the UnfoldMethod object that defines how the bends associated with this contour flange feature are unfolded.

## Syntax

HemDefinition.**UnfoldMethod**() As [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Property Value

This is a read/write property whose value is a [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md).

## Version

Introduced in version 2009
