# MidpointConstraint3D.Point Property

Parent Object: [MidpointConstraint3D](../MidpointConstraint3D/MidpointConstraint3D.md)

## Description

Property that returns the sketch point being constrained.

## Syntax

MidpointConstraint3D.**Point**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 2009
