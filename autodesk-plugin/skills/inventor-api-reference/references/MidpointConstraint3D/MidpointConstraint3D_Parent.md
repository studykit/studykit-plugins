# MidpointConstraint3D.Parent Property

Parent Object: [MidpointConstraint3D](../MidpointConstraint3D/MidpointConstraint3D.md)

## Description

Property that returns the parent sketch of the object.

## Syntax

MidpointConstraint3D.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2009
