# PointAndTangentWorkPlaneDef.Face Property

Parent Object: [PointAndTangentWorkPlaneDef](../PointAndTangentWorkPlaneDef/PointAndTangentWorkPlaneDef.md)

## Description

Property that returns the tangent face of the work plane. This face must either be a cylinder whose axis is parallel to the line, a cone that is positioned such that a valid tangent exists, a sphere or a bspline surface.

## Syntax

PointAndTangentWorkPlaneDef.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 10
