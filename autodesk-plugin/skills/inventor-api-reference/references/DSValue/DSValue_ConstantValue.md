# DSValue.ConstantValue Property

Parent Object: [DSValue](../DSValue/DSValue.md)

## Description

Get and set the value. This property is writable when the DSValue object is associated with a transient object.

## Syntax

DSValue.**ConstantValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2013
