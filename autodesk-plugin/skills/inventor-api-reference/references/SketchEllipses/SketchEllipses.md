# SketchEllipses Object

## Description

The SketchEllipses object provides access to all the objects in a sketch and provides methods to create additional sketch ellipses. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchEllipses/SketchEllipses_Add.md) | Method that creates a new sketch ellipse. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchEllipses/SketchEllipses_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchEllipses/SketchEllipses_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchEllipses/SketchEllipses_Item.md) | Returns the specified SketchEllipse object from the collection. |
| [Type](../SketchEllipses/SketchEllipses_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
