# SketchEllipses.Item Property

Parent Object: [SketchEllipses](../SketchEllipses/SketchEllipses.md)

## Description

Returns the specified SketchEllipse object from the collection.

## Syntax

SketchEllipses.**Item**( ***Index*** As Long ) As [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Property Value

This is a read only property whose value is a [SketchEllipse](../SketchEllipse/SketchEllipse.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 5
