# SketchEllipses.Type Property

Parent Object: [SketchEllipses](../SketchEllipses/SketchEllipses.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchEllipses.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
