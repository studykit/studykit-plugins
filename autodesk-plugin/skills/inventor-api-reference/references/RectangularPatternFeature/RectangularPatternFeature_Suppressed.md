# RectangularPatternFeature.Suppressed Property

Parent Object: [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

RectangularPatternFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
