# RectangularPatternFeature.HealthStatus Property

Parent Object: [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

RectangularPatternFeature.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 5
