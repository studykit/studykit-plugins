# RectangularPatternFeature.Shared Property

Parent Object: [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md)

## Description

Gets and sets whether the part feature is shared or not, applies only to surface features.

## Syntax

RectangularPatternFeature.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
