# HiddenLinesStatusEnum Enumerator

## Description

Constants identifying the visible status of hidden lines.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllHiddenLinesInvisible | 118275 | All hidden lines are invisible. |
| kAllHiddenLinesVisible | 118273 | All hidden lines are visible. |
| kSomeHiddenLinesVisible | 118274 | Some hidden lines are visible. |

## Version

Introduced in version 2022
