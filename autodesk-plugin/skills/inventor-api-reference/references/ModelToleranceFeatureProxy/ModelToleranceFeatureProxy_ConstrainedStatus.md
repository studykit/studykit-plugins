# ModelToleranceFeatureProxy.ConstrainedStatus Property

Parent Object: [ModelToleranceFeatureProxy](../ModelToleranceFeatureProxy/ModelToleranceFeatureProxy.md)

## Description

Gets or sets the constrained status of the tolerance feature.

## Syntax

ModelToleranceFeatureProxy.**ConstrainedStatus**() As [ModelToleranceFeatureConstrainedStatusEnum](../ModelToleranceFeatureConstrainedStatusEnum.md)

## Property Value

This is a read/write property whose value is a [ModelToleranceFeatureConstrainedStatusEnum](../ModelToleranceFeatureConstrainedStatusEnum.md).

## Version

Introduced in version 2019
