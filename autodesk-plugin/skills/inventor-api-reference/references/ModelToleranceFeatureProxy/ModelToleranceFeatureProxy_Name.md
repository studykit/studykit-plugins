# ModelToleranceFeatureProxy.Name Property

Parent Object: [ModelToleranceFeatureProxy](../ModelToleranceFeatureProxy/ModelToleranceFeatureProxy.md)

## Description

Gets or sets the name of the tolerance feature.

## Syntax

ModelToleranceFeatureProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
