# ModelToleranceFeatureProxy.Visible Property

Parent Object: [ModelToleranceFeatureProxy](../ModelToleranceFeatureProxy/ModelToleranceFeatureProxy.md)

## Description

Gets or sets the visibility of the tolerance feature.

## Syntax

ModelToleranceFeatureProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
