# MiniToolbarComboBox.OnItemHoverEnd Event

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Event that is fired when the user stops hovering over an item in the combobox in the expanded state.

## Syntax

MiniToolbarComboBox.**OnItemHoverEnd**( ***ListItem*** As [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ListItem | [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) | Input ListItem that indicates the item the user has stopped hovering over. |

## Version

Introduced in version 2012
