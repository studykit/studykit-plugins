# MiniToolbarComboBox.Count Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Read-only property that returns the number of items in the combobox, including separators.

## Syntax

MiniToolbarComboBox.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2012
