# MiniToolbarComboBox.Clear Method

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Method that removes all the items from the control, including any separators.

## Syntax

MiniToolbarComboBox.**Clear**()

## Version

Introduced in version 2012
