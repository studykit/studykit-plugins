# MiniToolbarComboBox.Enabled Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Read-write property that gets and sets whether the control is enabled.

## Syntax

MiniToolbarComboBox.**Enabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
