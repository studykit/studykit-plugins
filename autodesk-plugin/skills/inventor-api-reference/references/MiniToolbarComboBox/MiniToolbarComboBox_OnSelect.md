# MiniToolbarComboBox.OnSelect Event

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Event that is fired soon after the user changes the currently selected item using the drop down.

## Syntax

MiniToolbarComboBox.**OnSelect**( ***ListItem*** As [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ListItem | [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) | Input ListItem that indicates the item just selected. |
