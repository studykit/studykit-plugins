# MiniToolbarComboBox.Parent Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

MiniToolbarComboBox.**Parent**() As [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Property Value

This is a read only property whose value is a [MiniToolbar](../MiniToolbar/MiniToolbar.md).

## Version

Introduced in version 2012
