# MiniToolbarComboBox.ShowIcon Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Read-write property that gets and sets whether the icon associated with the currently selected item should be displayed on the face of the control.

## Syntax

MiniToolbarComboBox.**ShowIcon**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
