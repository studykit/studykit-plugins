# MiniToolbarComboBox.InternalName Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Read-only property that returns a string uniquely identifying this control within the toolbar.

## Syntax

MiniToolbarComboBox.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
