# MiniToolbar.Controls Property

Parent Object: [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Description

Read-only property that returns the MiniToolbarControls collection object.

## Syntax

MiniToolbar.**Controls**() As [MiniToolbarControls](../MiniToolbarControls/MiniToolbarControls.md)

## Property Value

This is a read only property whose value is a [MiniToolbarControls](../MiniToolbarControls/MiniToolbarControls.md).

## Version

Introduced in version 2012
