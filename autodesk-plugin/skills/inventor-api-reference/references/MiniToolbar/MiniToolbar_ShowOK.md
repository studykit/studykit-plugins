# MiniToolbar.ShowOK Property

Parent Object: [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Description

Read-write property that gets and sets whether the toolbar should display the OK button (represented by a green check mark).

## Syntax

MiniToolbar.**ShowOK**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
