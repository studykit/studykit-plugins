# MiniToolbar.ShowHandle Property

Parent Object: [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Description

Read-write property that gets and sets whether the toolbar should display the handle which is used to move the minitoolbar.

## Syntax

MiniToolbar.**ShowHandle**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
