# MiniToolbar.Width Property

Parent Object: [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Description

Read-only property that returns the width of the mini toolbar.

## Syntax

MiniToolbar.**Width**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2017
