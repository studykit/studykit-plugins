# MiniToolbar.EnableOK Property

Parent Object: [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Description

Read-write property that specifies whether the ‘OK’ button is enabled.

## Syntax

MiniToolbar.**EnableOK**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
