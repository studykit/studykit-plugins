# PositionalRepresentation Object

## Description

The PositionalRepresentation object represents a positional representation in the assembly.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Activate](../PositionalRepresentation/PositionalRepresentation_Activate.md) | Method that activates the positional representation. |
| [ClearRelationshipOverrides](../PositionalRepresentation/PositionalRepresentation_ClearRelationshipOverrides.md) | Method that clears all the overrides applied to the assembly constraint or joint object. |
| [Copy](../PositionalRepresentation/PositionalRepresentation_Copy.md) | Method that creates a copy of the PositionalRepresentation. The new created PositionalRepresentation is returned. The method returns an error if this is the 'master' positional representation. |
| [Delete](../PositionalRepresentation/PositionalRepresentation_Delete.md) | Method that deletes the PositionalRepresentation. The method returns an error if this is the 'master' positional representation. |

If not already overridden, the method has the effect of setting the 'Override' flag for the column offset. |

| [SetPositionalRepresentationOverride](../PositionalRepresentation/PositionalRepresentation_SetPositionalRepresentationOverride.md) | Method that sets an override positional representation on the input occurrence. If not already overridden, the method has the effect of setting the 'Override' flag for positional representation. |
| [SetRelationshipSuppressionOverride](../PositionalRepresentation/PositionalRepresentation_SetRelationshipSuppressionOverride.md) | Method that sets an overridden suppression state for the input assembly relationship. |
| [SetRelationshipValueOverride](../PositionalRepresentation/PositionalRepresentation_SetRelationshipValueOverride.md) | Method that sets an overridden value for the input assembly relationship. |
| [SetTransformOverride](../PositionalRepresentation/PositionalRepresentation_SetTransformOverride.md) | Method that sets an override transform on the input occurrence. If not already overridden, the method has the effect of setting the 'Override' flag for the transform (offset). |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PositionalRepresentation/PositionalRepresentation_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../PositionalRepresentation/PositionalRepresentation_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [Name](../PositionalRepresentation/PositionalRepresentation_Name.md) | Gets and sets the name of the PositionalRepresentation. The name must be unique with respect to all other PositionalRepresentation objects in the document. Cannot be set if this is the master positional representation. |
| [Parent](../PositionalRepresentation/PositionalRepresentation_Parent.md) | Property that returns the parent RepresentationsManager object. |
| [Primary](../PositionalRepresentation/PositionalRepresentation_Primary.md) | Gets whether this is the primary positional representation. |
| [Type](../PositionalRepresentation/PositionalRepresentation_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
