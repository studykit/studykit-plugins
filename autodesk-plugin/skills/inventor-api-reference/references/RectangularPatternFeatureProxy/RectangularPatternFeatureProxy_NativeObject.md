# RectangularPatternFeatureProxy.NativeObject Property

Parent Object: [RectangularPatternFeatureProxy](../RectangularPatternFeatureProxy/RectangularPatternFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RectangularPatternFeatureProxy.**NativeObject**() As [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md)

## Property Value

This is a read only property whose value is a [RectangularPatternFeature](../RectangularPatternFeature/RectangularPatternFeature.md).

## Version

Introduced in version 9
