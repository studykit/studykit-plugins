# RectangularPatternFeatureProxy.SurfacesOpaque Property

Parent Object: [RectangularPatternFeatureProxy](../RectangularPatternFeatureProxy/RectangularPatternFeatureProxy.md)

## Description

Gets and sets whether all the resulting surfaces are opaque or transparent.

## Syntax

RectangularPatternFeatureProxy.**SurfacesOpaque**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
