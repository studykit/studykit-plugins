# SketchDrivenPatternFeatures.Add Method

Parent Object: [SketchDrivenPatternFeatures](../SketchDrivenPatternFeatures/SketchDrivenPatternFeatures.md)

## Description

Method that creates a new sketch driven pattern feature. The new SketchDrivenPatternFeature object is returned.

## Syntax

SketchDrivenPatternFeatures.**Add**( ***Definition*** As [SketchDrivenPatternDefinition](../SketchDrivenPatternDefinition/SketchDrivenPatternDefinition.md) ) As [SketchDrivenPatternFeature](../SketchDrivenPatternFeature/SketchDrivenPatternFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [SketchDrivenPatternDefinition](../SketchDrivenPatternDefinition/SketchDrivenPatternDefinition.md) | Input SketchDrivenPatternDefinition objet that defines the SketchDrivenPatternFeature you want to create. |

## Version

Introduced in version 2017
