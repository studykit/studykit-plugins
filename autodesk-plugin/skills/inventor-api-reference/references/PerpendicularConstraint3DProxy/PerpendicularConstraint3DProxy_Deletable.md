# PerpendicularConstraint3DProxy.Deletable Property

Parent Object: [PerpendicularConstraint3DProxy](../PerpendicularConstraint3DProxy/PerpendicularConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

PerpendicularConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
