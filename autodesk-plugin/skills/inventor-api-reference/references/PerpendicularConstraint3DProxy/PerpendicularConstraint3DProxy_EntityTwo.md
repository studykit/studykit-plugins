# PerpendicularConstraint3DProxy.EntityTwo Property

Parent Object: [PerpendicularConstraint3DProxy](../PerpendicularConstraint3DProxy/PerpendicularConstraint3DProxy.md)

## Description

Property that returns the second sketch entity being constrained.

## Syntax

PerpendicularConstraint3DProxy.**EntityTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
