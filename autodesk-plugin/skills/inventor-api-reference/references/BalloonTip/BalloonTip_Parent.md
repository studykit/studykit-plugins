# BalloonTip.Parent Property

Parent Object: [BalloonTip](../BalloonTip/BalloonTip.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

BalloonTip.**Parent**() As [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Property Value

This is a read only property whose value is a [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md).

## Version

Introduced in version 2012
