# BalloonTip.Display Method

Parent Object: [BalloonTip](../BalloonTip/BalloonTip.md)

## Description

Method that displays the BalloonTip object.

## Syntax

BalloonTip.**Display**( [***Message***] As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Message | String | Input String to override the default message that was specified by the message property of BalloonTip. If an empty String is provided then the default message is used. |

## Version

Introduced in version 2012
