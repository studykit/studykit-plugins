# RotateRotateiMateDefinition.Suppressed Property

Parent Object: [RotateRotateiMateDefinition](../RotateRotateiMateDefinition/RotateRotateiMateDefinition.md)

## Description

Gets the boolean flag that indicates whether this iMateDefinition is suppressed or not.

## Syntax

RotateRotateiMateDefinition.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
