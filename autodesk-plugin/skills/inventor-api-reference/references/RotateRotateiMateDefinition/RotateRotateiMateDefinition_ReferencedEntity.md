# RotateRotateiMateDefinition.ReferencedEntity Property

Parent Object: [RotateRotateiMateDefinition](../RotateRotateiMateDefinition/RotateRotateiMateDefinition.md)

## Syntax

RotateRotateiMateDefinition.**ReferencedEntity**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value
