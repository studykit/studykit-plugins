# TangentConstraint3DProxy.Delete Method

Parent Object: [TangentConstraint3DProxy](../TangentConstraint3DProxy/TangentConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

TangentConstraint3DProxy.**Delete**()

## Version

Introduced in version 8
