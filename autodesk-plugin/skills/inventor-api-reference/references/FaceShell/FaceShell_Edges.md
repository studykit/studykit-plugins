# FaceShell.Edges Property

Parent Object: [FaceShell](../FaceShell/FaceShell.md)

## Description

Gets the referenced by this FaceShell.

## Syntax

FaceShell.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 4
