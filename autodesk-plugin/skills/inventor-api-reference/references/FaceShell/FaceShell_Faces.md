# FaceShell.Faces Property

Parent Object: [FaceShell](../FaceShell/FaceShell.md)

## Description

Property that returns a collection object generated as a result of the feature.

## Syntax

FaceShell.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 4
