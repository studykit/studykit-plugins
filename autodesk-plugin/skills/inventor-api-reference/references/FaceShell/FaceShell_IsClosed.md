# FaceShell.IsClosed Property

Parent Object: [FaceShell](../FaceShell/FaceShell.md)

## Description

Indicates whether the FaceShell is closed.

## Syntax

FaceShell.**IsClosed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
