# FaceShell.SurfaceBody Property

Parent Object: [FaceShell](../FaceShell/FaceShell.md)

## Description

Property that returns the surface body for this face shell.

## Syntax

FaceShell.**SurfaceBody**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 4
