# LightingStyles.Item Property

Parent Object: [LightingStyles](../LightingStyles/LightingStyles.md)

## Description

Returns the specified from the collection.

## Syntax

LightingStyles.**Item**( ***Index*** As Variant ) As [LightingStyle](../LightingStyle/LightingStyle.md)

## Property Value

This is a read only property whose value is a [LightingStyle](../LightingStyle/LightingStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Value that specifies the index number of the LightingStyle to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 10
