# LightingStyles Object

## Description

The LightingStyles collection object allows for the creation of new LightingStyle objects and provides access to existing LightingStyle objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../LightingStyles/LightingStyles_Add.md) | Method that creates a new LightingStyle object. The name of the new style is specified, while the rendering attributes are initialized with the same values as the Default style. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LightingStyles/LightingStyles_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LightingStyles/LightingStyles_Count.md) | Property that returns the number of objects in the collection. |
| [Item](../LightingStyles/LightingStyles_Item.md) | Returns the specified from the collection. |
| [Parent](../LightingStyles/LightingStyles_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../LightingStyles/LightingStyles_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[AssemblyDocument.LightingStyles](../AssemblyDocument/AssemblyDocument_LightingStyles.md), [PartDocument.LightingStyles](../PartDocument/PartDocument_LightingStyles.md), [PresentationDocument.LightingStyles](../PresentationDocument/PresentationDocument_LightingStyles.md)

## Version
