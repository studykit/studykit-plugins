# SketchEllipticalArcProxy.LineWeight Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets and sets the LineWeight applied to this elliptical arc.

## Syntax

SketchEllipticalArcProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
