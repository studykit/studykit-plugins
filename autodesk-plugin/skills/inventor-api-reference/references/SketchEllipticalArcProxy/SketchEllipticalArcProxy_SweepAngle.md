# SketchEllipticalArcProxy.SweepAngle Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Double property that returns the sweep angle of the elliptical arc. This is measured counterclockwise from the major axis in radians.

## Syntax

SketchEllipticalArcProxy.**SweepAngle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
