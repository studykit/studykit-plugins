# SketchEllipticalArcProxy.MajorAxisVector Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets and sets the major axis vector. This vector defines the direction of the major axis.

## Syntax

SketchEllipticalArcProxy.**MajorAxisVector**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read/write property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).

## Version

Introduced in version 6
