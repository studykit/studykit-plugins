# SketchEllipticalArcProxy.NativeObject Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchEllipticalArcProxy.**NativeObject**() As [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Property Value

This is a read only property whose value is a [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md).

## Version

Introduced in version 6
