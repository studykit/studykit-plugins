# SketchEllipticalArcProxy.Geometry3d Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Read-only property that returns elliptical arc geometry that represents this elliptical arc in model space.

## Syntax

SketchEllipticalArcProxy.**Geometry3d**() As [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Property Value

This is a read only property whose value is an [EllipticalArc](../EllipticalArc/EllipticalArc.md).

## Version

Introduced in version 2013
