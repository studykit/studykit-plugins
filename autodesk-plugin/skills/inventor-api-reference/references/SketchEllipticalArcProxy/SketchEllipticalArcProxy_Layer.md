# SketchEllipticalArcProxy.Layer Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets and sets the layer applied to this sketch entity.

## Syntax

SketchEllipticalArcProxy.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
