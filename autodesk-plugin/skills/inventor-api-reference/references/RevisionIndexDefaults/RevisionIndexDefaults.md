# RevisionIndexDefaults Object

## Description

RevisionIndexDefaults Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevisionIndexDefaults/RevisionIndexDefaults_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [AutoIndex](../RevisionIndexDefaults/RevisionIndexDefaults_AutoIndex.md) | Read-write property that gets and sets whether to automatically set the index value. |
| [IndexStartValue](../RevisionIndexDefaults/RevisionIndexDefaults_IndexStartValue.md) | Read-write property that gets and sets the index start value. |
| [Parent](../RevisionIndexDefaults/RevisionIndexDefaults_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Type](../RevisionIndexDefaults/RevisionIndexDefaults_Type.md) | Gets the constant that indicates the type of this object. |
| [UpdatePropertyOnRevisionNumberEdit](../RevisionIndexDefaults/RevisionIndexDefaults_UpdatePropertyOnRevisionNumberEdit.md) | Read-write property that gets and sets whether to update property on revision number edit or not. |
| [UseNumericIndex](../RevisionIndexDefaults/RevisionIndexDefaults_UseNumericIndex.md) | Read-write property that gets and sets whether the revision index is numeric or not. |

## Accessed From

[RevisionTableStyle.RevisionIndexDefaults](../RevisionTableStyle/RevisionTableStyle_RevisionIndexDefaults.md)
