# RevisionIndexDefaults.AutoIndex Property

Parent Object: [RevisionIndexDefaults](../RevisionIndexDefaults/RevisionIndexDefaults.md)

## Description

Read-write property that gets and sets whether to automatically set the index value.

## Syntax

RevisionIndexDefaults.**AutoIndex**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
