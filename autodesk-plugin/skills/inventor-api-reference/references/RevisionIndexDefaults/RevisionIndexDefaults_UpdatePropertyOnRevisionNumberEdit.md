# RevisionIndexDefaults.UpdatePropertyOnRevisionNumberEdit Property

Parent Object: [RevisionIndexDefaults](../RevisionIndexDefaults/RevisionIndexDefaults.md)

## Description

Read-write property that gets and sets whether to update property on revision number edit or not.

## Syntax

RevisionIndexDefaults.**UpdatePropertyOnRevisionNumberEdit**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
