# SurfaceTextureSymbolDefinition.SurfaceTextureType Property

Parent Object: [SurfaceTextureSymbolDefinition](../SurfaceTextureSymbolDefinition/SurfaceTextureSymbolDefinition.md)

## Description

Gets and sets the surface type for the symbol.

## Syntax

SurfaceTextureSymbolDefinition.**SurfaceTextureType**() As [SurfaceTextureTypeEnum](../SurfaceTextureTypeEnum.md)

## Property Value

This is a read/write property whose value is a [SurfaceTextureTypeEnum](../SurfaceTextureTypeEnum.md).

## Version

Introduced in version 2016
