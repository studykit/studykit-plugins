# HeightDatumTypeEnum Enumerator

## Description

Defines the constants used to specify how the height of a distance extent flange is calculated.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kHeightDatumInner | 75522 | Inner height datum type. |
| kHeightDatumInnerOrtho | 75525 | Inner-ortho height datum type. |
| kHeightDatumOuter | 75521 | Outer height datum type. |
| kHeightDatumOuterOrtho | 75524 | Outer-ortho height datum type. |
| kHeightDatumTangent | 75523 | Tangent height datum type. |

## Version

Introduced in version 2009
