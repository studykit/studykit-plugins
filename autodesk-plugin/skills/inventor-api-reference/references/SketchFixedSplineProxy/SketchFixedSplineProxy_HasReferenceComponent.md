# SketchFixedSplineProxy.HasReferenceComponent Property

Parent Object: [SketchFixedSplineProxy](../SketchFixedSplineProxy/SketchFixedSplineProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchFixedSplineProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
