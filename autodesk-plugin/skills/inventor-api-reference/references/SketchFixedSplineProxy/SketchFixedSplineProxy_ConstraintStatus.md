# SketchFixedSplineProxy.ConstraintStatus Property

Parent Object: [SketchFixedSplineProxy](../SketchFixedSplineProxy/SketchFixedSplineProxy.md)

## Description

Read-only property that returns the constraint status of the sketch. Possible return values are kFullyConstrainedConstraintStatus, kOverConstrainedConstraintStatus, kUnderConstrainedConstraintStatus, and kUnknownConstraintStatus.

## Syntax

SketchFixedSplineProxy.**ConstraintStatus**() As [ConstraintStatusEnum](../ConstraintStatusEnum.md)

## Property Value

This is a read only property whose value is a [ConstraintStatusEnum](../ConstraintStatusEnum.md).

## Version

Introduced in version 10
