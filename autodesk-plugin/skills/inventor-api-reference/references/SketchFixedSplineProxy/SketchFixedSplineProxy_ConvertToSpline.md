# SketchFixedSplineProxy.ConvertToSpline Method

Parent Object: [SketchFixedSplineProxy](../SketchFixedSplineProxy/SketchFixedSplineProxy.md)

## Description

Method that converts the fixed spline to a SketchSpline object.

## Syntax

SketchFixedSplineProxy.**ConvertToSpline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Version

Introduced in version 10
