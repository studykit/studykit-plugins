# TransactionManager.CurrentTransaction Property

Parent Object: [TransactionManager](../TransactionManager/TransactionManager.md)

## Description

Gets the current transaction in progress.

## Syntax

TransactionManager.**CurrentTransaction**() As [Transaction](../Transaction/Transaction.md)

## Property Value

This is a read only property whose value is a [Transaction](../Transaction/Transaction.md).

## Version

Introduced in version 5
