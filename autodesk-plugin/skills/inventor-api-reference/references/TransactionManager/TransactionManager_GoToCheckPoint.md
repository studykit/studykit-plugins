# TransactionManager.GoToCheckPoint Method

Parent Object: [TransactionManager](../TransactionManager/TransactionManager.md)

## Description

Aborts back up to the checkpoint specified.

## Syntax

TransactionManager.**GoToCheckPoint**( ***CheckPoint*** As [CheckPoint](../CheckPoint/CheckPoint.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CheckPoint | [CheckPoint](../CheckPoint/CheckPoint.md) | Input CheckPoint that specifies the checkpoint to abort back to. |

## Version

Introduced in version 5
