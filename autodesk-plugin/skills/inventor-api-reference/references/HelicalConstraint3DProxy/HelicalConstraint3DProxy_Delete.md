# HelicalConstraint3DProxy.Delete Method

Parent Object: [HelicalConstraint3DProxy](../HelicalConstraint3DProxy/HelicalConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

HelicalConstraint3DProxy.**Delete**()

## Version

Introduced in version 2010
