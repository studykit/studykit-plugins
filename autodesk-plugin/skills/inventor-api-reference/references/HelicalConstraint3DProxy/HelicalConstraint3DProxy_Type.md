# HelicalConstraint3DProxy.Type Property

Parent Object: [HelicalConstraint3DProxy](../HelicalConstraint3DProxy/HelicalConstraint3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

HelicalConstraint3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
