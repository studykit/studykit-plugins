# ClientResourceMap.Parent Property

Parent Object: [ClientResourceMap](../ClientResourceMap/ClientResourceMap.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ClientResourceMap.**Parent**() As [Application](../Application/Application.md)

## Property Value

This is a read only property whose value is an [Application](../Application/Application.md).

## Version

Introduced in version 2022
