# EmbossFeatures.Type Property

Parent Object: [EmbossFeatures](../EmbossFeatures/EmbossFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

EmbossFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
