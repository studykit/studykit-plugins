# ExtrudeFeatureProxy.Definition Property

Parent Object: [ExtrudeFeatureProxy](../ExtrudeFeatureProxy/ExtrudeFeatureProxy.md)

## Description

Read-write property that gets and sets the ExtrudeDefinition object associated with this feature.

## Syntax

ExtrudeFeatureProxy.**Definition**() As [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Property Value

This is a read/write property whose value is an [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md).

## Version

Introduced in version 2012
