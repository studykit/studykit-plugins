# ExtrudeFeatureProxy.EndFaces Property

Parent Object: [ExtrudeFeatureProxy](../ExtrudeFeatureProxy/ExtrudeFeatureProxy.md)

## Description

Property that returns the set of that cap one end of the extrusion.

## Syntax

ExtrudeFeatureProxy.**EndFaces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 9
