# DerivedAssemblyOccurrence.Name Property

Parent Object: [DerivedAssemblyOccurrence](../DerivedAssemblyOccurrence/DerivedAssemblyOccurrence.md)

## Description

Property that returns the name of the derived occurrence.

## Syntax

DerivedAssemblyOccurrence.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 5.3
