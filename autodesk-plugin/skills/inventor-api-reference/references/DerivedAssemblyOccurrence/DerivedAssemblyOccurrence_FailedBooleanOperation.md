# DerivedAssemblyOccurrence.FailedBooleanOperation Property

Parent Object: [DerivedAssemblyOccurrence](../DerivedAssemblyOccurrence/DerivedAssemblyOccurrence.md)

## Description

Read-only property that returns whether this occurrence failed the Boolean operation resulting in creation of separate, independent bodies. This occurs only if the IndependentSolidsOnFailedBoolean property on DerivedAssemblyDefinition is set to true.

## Syntax

DerivedAssemblyOccurrence.**FailedBooleanOperation**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
