# ComponentOccurrences.AllLeafOccurrences Property

Parent Object: [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md)

## Description

Return all unique leaf occurrences (optionally filtered to the specified definition) relative to this context.

## Syntax

ComponentOccurrences.**AllLeafOccurrences**( [***LeafDefinition***] As Variant ) As [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LeafDefinition | Variant | Occurence to be used as the basis for filtering the leaf space for a given leaf occurence. |

## Version

Introduced in version 8
