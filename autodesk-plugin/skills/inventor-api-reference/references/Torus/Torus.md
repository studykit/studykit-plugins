# Torus Object

## Description

The Torus object. The object created is a transient mathematical object and is not displayed graphically.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Torus/Torus_Copy.md) | Creates a copy of this Torus object. The result is entirely independent and can be edited without affecting the original Torus object. |
| [GetTorusData](../Torus/Torus_GetTorusData.md) | Get the data defining this torus. |
| [PutTorusData](../Torus/Torus_PutTorusData.md) | Method that sets the data defining this torus. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AxisVector](../Torus/Torus_AxisVector.md) | Specifies the direction of the axis of the torus. |
| [CenterPoint](../Torus/Torus_CenterPoint.md) | Specifies the center of the torus. |
| [Evaluator](../Torus/Torus_Evaluator.md) | Gets the surface evaluator for this torus. |
| [MajorRadius](../Torus/Torus_MajorRadius.md) | Specifies the major radius of the torus. |
| [MinorRadius](../Torus/Torus_MinorRadius.md) | Specifies the minor radius of the torus. |

## Accessed From

[Torus.Copy](../Torus/Torus_Copy.md), [TransientGeometry.CreateTorus](../TransientGeometry/TransientGeometry_CreateTorus.md)

## Version

Introduced in version 4
