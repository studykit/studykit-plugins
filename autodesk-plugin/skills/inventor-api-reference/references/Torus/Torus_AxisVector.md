# Torus.AxisVector Property

Parent Object: [Torus](../Torus/Torus.md)

## Description

Specifies the direction of the axis of the torus.

## Syntax

Torus.**AxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
