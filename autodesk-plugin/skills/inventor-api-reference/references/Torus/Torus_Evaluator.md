# Torus.Evaluator Property

Parent Object: [Torus](../Torus/Torus.md)

## Description

Gets the surface evaluator for this torus.

## Syntax

Torus.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
