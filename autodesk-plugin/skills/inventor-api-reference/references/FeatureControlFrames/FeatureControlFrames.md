# FeatureControlFrames Object

## Description

The FeatureControlFrames collection object provides access to all the feature control frame symbols on a sheet and provides methods to create additional ones.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../FeatureControlFrames/FeatureControlFrames_Add.md) | Method that creates a feature control frame symbol. |
| [CreateFeatureControlFrameRows](../FeatureControlFrames/FeatureControlFrames_CreateFeatureControlFrameRows.md) | Method that creates a new FeatureControlFrameRows object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FeatureControlFrames/FeatureControlFrames_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FeatureControlFrames/FeatureControlFrames_Count.md) | Property that returns the number of items in the collection. |
| [Item](../FeatureControlFrames/FeatureControlFrames_Item.md) | Method that returns the specified FeatureControlFrame object from the collection. |
| [Type](../FeatureControlFrames/FeatureControlFrames_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
