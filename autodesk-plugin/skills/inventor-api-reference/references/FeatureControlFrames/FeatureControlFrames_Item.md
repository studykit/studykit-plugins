# FeatureControlFrames.Item Property

Parent Object: [FeatureControlFrames](../FeatureControlFrames/FeatureControlFrames.md)

## Description

Method that returns the specified FeatureControlFrame object from the collection.

## Syntax

FeatureControlFrames.**Item**( ***Index*** As Long ) As [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Property Value

This is a read only property whose value is a [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2009
