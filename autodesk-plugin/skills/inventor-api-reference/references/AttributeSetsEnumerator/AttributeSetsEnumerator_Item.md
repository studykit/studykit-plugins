# AttributeSetsEnumerator.Item Property

Parent Object: [AttributeSetsEnumerator](../AttributeSetsEnumerator/AttributeSetsEnumerator.md)

## Description

Allows integer-indexed access to objects in the collection.

## Syntax

AttributeSetsEnumerator.**Item**( ***Index*** As Long ) As [AttributeSet](../AttributeSet/AttributeSet.md)

## Property Value

This is a read only property whose value is an [AttributeSet](../AttributeSet/AttributeSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 5
