# AttributeSetsEnumerator Object

## Description

The AttributeSetsEnumerator object provides access to a list of objects. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../AttributeSetsEnumerator/AttributeSetsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../AttributeSetsEnumerator/AttributeSetsEnumerator_Item.md) | Allows integer-indexed access to objects in the collection. |
| [Type](../AttributeSetsEnumerator/AttributeSetsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[AttributeManager.FindAttributeSets](../AttributeManager/AttributeManager_FindAttributeSets.md), [AttributeManager.OpenAttributeSets](../AttributeManager/AttributeManager_OpenAttributeSets.md)

## Version

Introduced in version 5
