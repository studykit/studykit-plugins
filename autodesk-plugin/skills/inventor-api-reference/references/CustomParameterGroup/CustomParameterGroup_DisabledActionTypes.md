# CustomParameterGroup.DisabledActionTypes Property

Parent Object: [CustomParameterGroup](../CustomParameterGroup/CustomParameterGroup.md)

## Description

Gets and sets the disabling of certain user actions on this object.

## Syntax

CustomParameterGroup.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 10
