# CustomParameterGroup.Add Method

Parent Object: [CustomParameterGroup](../CustomParameterGroup/CustomParameterGroup.md)

## Description

Method that adds a parameter to the group. Adding a parameter to a group does not remove it from group(s) that it currently resides in.

## Syntax

CustomParameterGroup.**Add**( ***Parameter*** As [Parameter](../Parameter/Parameter.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Parameter | [Parameter](../Parameter/Parameter.md) | Input Parameter object that represents the parameter to be added to this group. |
