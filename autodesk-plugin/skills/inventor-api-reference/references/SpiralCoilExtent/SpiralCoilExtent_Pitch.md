# SpiralCoilExtent.Pitch Property

Parent Object: [SpiralCoilExtent](../SpiralCoilExtent/SpiralCoilExtent.md)

## Description

Property that returns the parameter that controls the pitch of the coil feature.

## Syntax

SpiralCoilExtent.**Pitch**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 5.3
