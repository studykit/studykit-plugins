# Lights.Add Method

Parent Object: [Lights](../Lights/Lights.md)

## Description

Method that creates a new Light object.

## Syntax

Lights.**Add**( ***LightType*** As [LightTypeEnum](../LightTypeEnum.md) ) As [Light](../Light/Light.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LightType | [LightTypeEnum](../LightTypeEnum.md) | Specifies the light type - model space or view space. |

## Version

Introduced in version 10
