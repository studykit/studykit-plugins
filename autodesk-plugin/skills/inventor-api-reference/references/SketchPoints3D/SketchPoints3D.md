# SketchPoints3D Object

## Description

The SketchPoints3D object provides access to all the objects in a sketch. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchPoints3D/SketchPoints3D_Add.md) | Method that creates a new 3D sketch point at the specified coordinate. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchPoints3D/SketchPoints3D_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchPoints3D/SketchPoints3D_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchPoints3D/SketchPoints3D_Item.md) | Returns the specified SketchPoint3D object from the collection. |
| [Type](../SketchPoints3D/SketchPoints3D_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Sketch3D.SketchPoints3D](../Sketch3D/Sketch3D_SketchPoints3D.md), [Sketch3DProxy.SketchPoints3D](../Sketch3DProxy/Sketch3DProxy_SketchPoints3D.md)

## Version

Introduced in version 6
