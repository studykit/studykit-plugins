# FastenerFitType Enumerator

## Description

Constants info for hole clearance.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kCloseFitType | 84072097 | Hole is drilled (no countersink or counterboring). |
| kLooseFitType | 84072098 | Hole is counterbored. |
| kNormalFitType | 84072099 | Hole is countersunk. |

## Version

Introduced in version 2018
