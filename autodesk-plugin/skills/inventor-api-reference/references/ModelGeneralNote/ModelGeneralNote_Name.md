# ModelGeneralNote.Name Property

Parent Object: [ModelGeneralNote](../ModelGeneralNote/ModelGeneralNote.md)

## Description

Read-write property that gets and sets the name of the annotation.

## Syntax

ModelGeneralNote.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
