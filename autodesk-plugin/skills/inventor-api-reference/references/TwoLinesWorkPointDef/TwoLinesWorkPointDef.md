# TwoLinesWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point defined by the intersection of two lines.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../TwoLinesWorkPointDef/TwoLinesWorkPointDef_GetData.md) | Method that gets all of the data defining a work point at the intersection of two lines. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TwoLinesWorkPointDef/TwoLinesWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Line1](../TwoLinesWorkPointDef/TwoLinesWorkPointDef_Line1.md) | Property that returns the first line of a work point that is defined by the intersection of two lines. This object can be a linear Edge, WorkAxis, or SketchLine object. |
| [Line2](../TwoLinesWorkPointDef/TwoLinesWorkPointDef_Line2.md) | Property that returns the second line of a work point that is defined by the intersection of two lines. This object can be a linear Edge, WorkAxis, or SketchLine object.. |
| [Parent](../TwoLinesWorkPointDef/TwoLinesWorkPointDef_Parent.md) | Property returning the parent WorkPoint object. |
| [Type](../TwoLinesWorkPointDef/TwoLinesWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
