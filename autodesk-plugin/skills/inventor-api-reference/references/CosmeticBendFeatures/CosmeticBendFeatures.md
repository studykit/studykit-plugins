# CosmeticBendFeatures Object

## Description

The CosmeticBendFeatures collection object provides access to the CosmeticBendFeature objects that have been created within a flat pattern.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CosmeticBendFeatures/CosmeticBendFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CosmeticBendFeatures/CosmeticBendFeatures_Count.md) | Property that returns the number of LoftedFlangeFeature objects in the collection. |
| [Item](../CosmeticBendFeatures/CosmeticBendFeatures_Item.md) | Returns the specified CosmeticBendFeature object from the collection. |
| [Type](../CosmeticBendFeatures/CosmeticBendFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FlatPatternFeatures.CosmeticBendFeatures](../FlatPatternFeatures/FlatPatternFeatures_CosmeticBendFeatures.md)

## Version

Introduced in version 2010
