# OccurrencePatternElement.Suppressed Property

Parent Object: [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Description

Gets/Sets whether the element is suppressed or not.

## Syntax

OccurrencePatternElement.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
