# PointCloudPlane.Geometry Property

Parent Object: [PointCloudPlane](../PointCloudPlane/PointCloudPlane.md)

## Description

Read-only property that returns a Plane object which provides the location and orientation of the inferred plane.

## Syntax

PointCloudPlane.**Geometry**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).

## Version

Introduced in version 2015
