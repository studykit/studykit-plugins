# PointCloudPlane Object

## Description

The PointCloudPlane object represents a plane that’s inferred by points within a point cloud.

| --- | --- |

| [GetPlaneRectangle](../PointCloudPlane/PointCloudPlane_GetPlaneRectangle.md) | Method that gets the full definition of a bounded rectangular plane. |

| [Geometry](../PointCloudPlane/PointCloudPlane_Geometry.md) | Read-only property that returns a Plane object which provides the location and orientation of the inferred plane. |
