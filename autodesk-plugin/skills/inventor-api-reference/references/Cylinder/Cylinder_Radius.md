# Cylinder.Radius Property

Parent Object: [Cylinder](../Cylinder/Cylinder.md)

## Description

Specifies the radius of the cylinder.

## Syntax

Cylinder.**Radius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 4
