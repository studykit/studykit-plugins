# Cylinder Object

## Description

The Cylinder object. The object created is a transient mathematical object and is not displayed graphically

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Cylinder/Cylinder_Copy.md) | Creates a copy of this Cylinder object. The result is entirely independent and can be edited without affecting the original Cylinder object. |
| [GetCylinderData](../Cylinder/Cylinder_GetCylinderData.md) | Get the data defining this cylinder. |
| [PutCylinderData](../Cylinder/Cylinder_PutCylinderData.md) | Method that sets the data defining this cylinder. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AxisVector](../Cylinder/Cylinder_AxisVector.md) | Specifies the direction of the axis of the cylinder. |
| [BasePoint](../Cylinder/Cylinder_BasePoint.md) | Specifies the base of the cylinder. |
| [Evaluator](../Cylinder/Cylinder_Evaluator.md) | Gets the surface evaluator for this cylinder. |
| [Radius](../Cylinder/Cylinder_Radius.md) | Specifies the radius of the cylinder. |

## Accessed From

[Cylinder.Copy](../Cylinder/Cylinder_Copy.md), [TransientGeometry.CreateCylinder](../TransientGeometry/TransientGeometry_CreateCylinder.md)
