# Cylinder.AxisVector Property

Parent Object: [Cylinder](../Cylinder/Cylinder.md)

## Description

Specifies the direction of the axis of the cylinder.

## Syntax

Cylinder.**AxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).
