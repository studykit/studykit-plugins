# Cylinder.Evaluator Property

Parent Object: [Cylinder](../Cylinder/Cylinder.md)

## Description

Gets the surface evaluator for this cylinder.

## Syntax

Cylinder.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
