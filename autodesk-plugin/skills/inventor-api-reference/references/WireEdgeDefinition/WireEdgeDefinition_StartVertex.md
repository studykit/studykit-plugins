# WireEdgeDefinition.StartVertex Property

Parent Object: [WireEdgeDefinition](../WireEdgeDefinition/WireEdgeDefinition.md)

## Description

Gets and sets the starting vertex of the edge. This can be Nothing where no start vertex is specifically defined.

## Syntax

WireEdgeDefinition.**StartVertex**() As [VertexDefinition](../VertexDefinition/VertexDefinition.md)

## Property Value

This is a read/write property whose value is a [VertexDefinition](../VertexDefinition/VertexDefinition.md).

## Version

Introduced in version 2013
