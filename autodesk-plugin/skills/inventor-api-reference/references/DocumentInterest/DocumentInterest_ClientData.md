# DocumentInterest.ClientData Property

Parent Object: [DocumentInterest](../DocumentInterest/DocumentInterest.md)

## Description

Gets/Sets additional data that the client would like to store with the interest. This string is not interpreted by Inventor.

## Syntax

DocumentInterest.**ClientData**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
