# DocumentInterest.DataVersion Property

Parent Object: [DocumentInterest](../DocumentInterest/DocumentInterest.md)

## Description

Gets/Sets the current data version of the document. If the property returns -1 or is set to -1, this is assumed to be a 'non-migrating' type of interest.

## Syntax

DocumentInterest.**DataVersion**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2008
