# LinearGeneralDimension.GeneralDimensionType Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Returns an enum indicating the type of general dimension.

## Syntax

LinearGeneralDimension.**GeneralDimensionType**() As [GeneralDimensionTypeEnum](../GeneralDimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [GeneralDimensionTypeEnum](../GeneralDimensionTypeEnum.md).

## Version

Introduced in version 2008
