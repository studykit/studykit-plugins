# LinearGeneralDimension.VirtualArcPosition Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets the geometry that defines the position of the virtual arc.

## Syntax

LinearGeneralDimension.**VirtualArcPosition**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2015
