# LinearGeneralDimension.IsInspectionDimension Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets whether this is an inspection dimension.

## Syntax

LinearGeneralDimension.**IsInspectionDimension**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
