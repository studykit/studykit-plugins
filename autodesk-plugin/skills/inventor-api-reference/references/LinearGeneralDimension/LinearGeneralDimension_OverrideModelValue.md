# LinearGeneralDimension.OverrideModelValue Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets the NominalValue setting.

## Syntax

LinearGeneralDimension.**OverrideModelValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
