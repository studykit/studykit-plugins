# LinearGeneralDimension.ShowAllExtensionLines Method

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Method that un-hides all the extension lines associated with this dimension. If there are no hidden extension lines associated with this dimension, this method does nothing and returns a success.

## Syntax

LinearGeneralDimension.**ShowAllExtensionLines**()

## Version

Introduced in version 2011
