# LinearGeneralDimension.IsBaselineSetMember Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Property that returns whether this dimension is a member of a baseline dimension set. If this property returns True, the BaselineDimensionSet object is returned by the BaselineDimensionSet property.

## Syntax

LinearGeneralDimension.**IsBaselineSetMember**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
