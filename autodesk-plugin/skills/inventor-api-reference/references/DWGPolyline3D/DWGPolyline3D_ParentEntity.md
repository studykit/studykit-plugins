# DWGPolyline3D.ParentEntity Property

Parent Object: [DWGPolyline3D](../DWGPolyline3D/DWGPolyline3D.md)

## Description

Read-only property that returns the parent DWGEntity of the entity.

## Syntax

DWGPolyline3D.**ParentEntity**() As [DWGEntity](../DWGEntity/DWGEntity.md)

## Property Value

This is a read only property whose value is a [DWGEntity](../DWGEntity/DWGEntity.md).

## Version

Introduced in version 2017
