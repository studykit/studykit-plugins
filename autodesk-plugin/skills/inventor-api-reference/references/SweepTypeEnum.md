# SweepTypeEnum Enumerator

## Description

Enum describes the sweep type.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kPathAndGuidRailSweepType | 104450 | Sweep type that indicates the feature is defined with a path and guide rail. |
| kPathAndGuidSurfaceSweepType | 104451 | Sweep type that indicates the feature is defined with a path and guide surface. |
| kPathAndSectionTwistSweepType | 104452 | Sweep type that indicates the feature is defined with a path and section twist. |
| kPathSweepType | 104449 | Sweep type that indicates the feature is defined with a path. |

## Version

Introduced in version 2015
