# ApprenticeServerDocuments.Type Property

Parent Object: [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ApprenticeServerDocuments.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 4
