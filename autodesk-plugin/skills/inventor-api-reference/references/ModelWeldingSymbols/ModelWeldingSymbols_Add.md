# ModelWeldingSymbols.Add Method

Parent Object: [ModelWeldingSymbols](../ModelWeldingSymbols/ModelWeldingSymbols.md)

## Description

Method that creates a model welding symbol.

## Syntax

ModelWeldingSymbols.**Add**( ***Definitions*** As [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md) ) As [ModelWeldingSymbol](../ModelWeldingSymbol/ModelWeldingSymbol.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definitions | [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md) | Input ModelWeldingSymbolDefinitions object that defines the input for the ModelWeldingSymbol. A ModelWeldingSymbolDefinitions object can be created using the ModelWeldingSymbols.CreateDefinitions method. It can also be obtained from an existing ModelWeldingSymbol object. |

## Version

Introduced in version 2024
