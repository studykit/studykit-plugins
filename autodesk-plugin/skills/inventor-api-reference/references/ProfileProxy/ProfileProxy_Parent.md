# ProfileProxy.Parent Property

Parent Object: [ProfileProxy](../ProfileProxy/ProfileProxy.md)

## Description

Property that returns the sketch that the profile was derived from.

## Syntax

ProfileProxy.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 6
