# SurfaceTextureJISDefinition.SecondRequirement Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets the second requirement value for the surface.

## Syntax

SurfaceTextureJISDefinition.**SecondRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
