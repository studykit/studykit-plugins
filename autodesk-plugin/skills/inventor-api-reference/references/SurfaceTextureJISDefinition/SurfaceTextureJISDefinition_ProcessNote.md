# SurfaceTextureJISDefinition.ProcessNote Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets the process requirements for the surface.

## Syntax

SurfaceTextureJISDefinition.**ProcessNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
