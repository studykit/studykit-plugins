# SurfaceTextureJISDefinition.SurfaceTextureType Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets the surface type for the symbol.

## Syntax

SurfaceTextureJISDefinition.**SurfaceTextureType**() As [SurfaceTextureTypeEnum](../SurfaceTextureTypeEnum.md)

## Property Value

This is a read/write property whose value is a [SurfaceTextureTypeEnum](../SurfaceTextureTypeEnum.md).

## Version

Introduced in version 2016
