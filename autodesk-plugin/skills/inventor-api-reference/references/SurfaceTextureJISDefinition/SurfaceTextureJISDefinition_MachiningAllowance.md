# SurfaceTextureJISDefinition.MachiningAllowance Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets the machining allowance.

## Syntax

SurfaceTextureJISDefinition.**MachiningAllowance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
