# Box Object

## Description

The Box object is a mathematical utility object that represents a rectangular box whose faces are always parallel to the model XYZ planes. A common use of the Box object is as a means of passing the range box information of an entity and interacting with that range box.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Contains](../Box/Box_Contains.md) | Determines whether the specified point is contained within this Box. |
| [Copy](../Box/Box_Copy.md) | Creates a copy of this Box object. The result is entirely independent and can be edited without affecting the original Box object. |
| [Expand](../Box/Box_Expand.md) | Expands the Box on all sides by the specified distance. |
| [Extend](../Box/Box_Extend.md) | Extends the Box to include the specified point. |
| [GetBoxData](../Box/Box_GetBoxData.md) | Get the data defining this Box. |
| [IsDisjoint](../Box/Box_IsDisjoint.md) | Determines whether this Box intersects the specified Box. A return value of True indicates that the box do not intersect. |
| [PutBoxData](../Box/Box_PutBoxData.md) | Method that sets the data defining this Box. |

## Properties

| [MinPoint](../Box/Box_MinPoint.md) | Property that gets and sets the minimum corner of the box. |
