# Box.Extend Method

Parent Object: [Box](../Box/Box.md)

## Description

Extends the Box to include the specified point.

## Syntax

Box.**Extend**( ***Point*** As [Point](../Point/Point.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input Point object that specifies the coordinate to include in the box. |

## Version

Introduced in version 4
