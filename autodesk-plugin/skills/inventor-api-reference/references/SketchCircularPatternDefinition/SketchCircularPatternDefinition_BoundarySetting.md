# SketchCircularPatternDefinition.BoundarySetting Property

Parent Object: [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Description

Gets and set the PatternBoundarySetting object. When set this property a PatternBoundarySetting can be copied using PatternBoundarySetting.

## Syntax

SketchCircularPatternDefinition.**BoundarySetting**() As [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Property Value

This is a read/write property whose value is a [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md).

## Version

Introduced in version 2025.1
