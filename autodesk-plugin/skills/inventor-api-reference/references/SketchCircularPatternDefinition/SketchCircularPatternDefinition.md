# SketchCircularPatternDefinition Object

## Description

SketchCircularPatternDefinition Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Copy.md) | Creates a copy of this SketchCircularPatternDefinition object. The new SketchCircularPatternDefinition object is independent of any pattern. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Angle](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Angle.md) | Gets and sets the angle between instances. |
| [Application](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Associative](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Associative.md) | Gets and sets whether the pattern will update when changes are made to the part. If set this to False then the pattern constraints will be removed and the pattern will not update when change an element. |
| [AxisEntity](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_AxisEntity.md) | Gets and sets the axis entity. |
| [BoundarySetting](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_BoundarySetting.md) | Gets and set the PatternBoundarySetting object. When set this property a PatternBoundarySetting can be copied using PatternBoundarySetting. |
| [Count](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Count.md) | Gets and sets the number of instances. |
| [Fitted](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Fitted.md) | Gets and sets whether pattern elements are equally fitted within the specified angle. |
| [Geometries](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Geometries.md) | Gets and sets the ObjectCollection object that contains the sketch geometries to be patterned. |
| [HasBoundarySettingConfigured](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_HasBoundarySettingConfigured.md) | Gets and sets whether this pattern has boundary setting configured or not. |
| [NaturalAxisDirection](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_NaturalAxisDirection.md) | Gets and sets the flag that indicates if the direction of the pattern is in the natural direction of the AxisEntity or reversed. |
| [SuppressedElements](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_SuppressedElements.md) | Gets and sets the suppressed elements. |
| [Symmetric](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Symmetric.md) | Gets and sets if the occurrences are distributed on both sides of the original geometry. |
| [Type](../SketchCircularPatternDefinition/SketchCircularPatternDefinition_Type.md) | Gets the constant that indicates the type of this object. |
