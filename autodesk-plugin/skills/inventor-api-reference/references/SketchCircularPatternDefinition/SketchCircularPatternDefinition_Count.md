# SketchCircularPatternDefinition.Count Property

Parent Object: [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Description

Gets and sets the number of instances.

## Syntax

SketchCircularPatternDefinition.**Count**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2025.1
