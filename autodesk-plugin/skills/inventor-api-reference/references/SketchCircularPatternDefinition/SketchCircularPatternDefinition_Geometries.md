# SketchCircularPatternDefinition.Geometries Property

Parent Object: [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Description

Gets and sets the ObjectCollection object that contains the sketch geometries to be patterned.

## Syntax

SketchCircularPatternDefinition.**Geometries**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2025.1
