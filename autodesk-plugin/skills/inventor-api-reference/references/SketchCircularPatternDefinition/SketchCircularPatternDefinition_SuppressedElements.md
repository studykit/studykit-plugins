# SketchCircularPatternDefinition.SuppressedElements Property

Parent Object: [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Description

Gets and sets the suppressed elements.

## Syntax

SketchCircularPatternDefinition.**SuppressedElements**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2025.1
