# SketchLines.Item Property

Parent Object: [SketchLines](../SketchLines/SketchLines.md)

## Description

Returns the specified SketchLine object from the collection.

## Syntax

SketchLines.**Item**( ***Index*** As Long ) As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |
