# SimplifyDefinition.PreservedBodies Property

Parent Object: [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md)

## Description

Read-write property that gets and sets the bodies to preserve from removing.

## Syntax

SimplifyDefinition.**PreservedBodies**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2026
