# SimplifyDefinition.RemoveTunnelsStyle Property

Parent Object: [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md)

## Description

Read-write property that gets and sets the style to remove tunnels for the simplify feature. This is applicable when the EnvelopesReplaceStyle is kNoneReplaceStyle or kSelectedBodiesReplaceStyle.

## Syntax

SimplifyDefinition.**RemoveTunnelsStyle**() As [SimplificationRemoveStyleEnum](../SimplificationRemoveStyleEnum.md)

## Property Value

This is a read/write property whose value is a [SimplificationRemoveStyleEnum](../SimplificationRemoveStyleEnum.md).

## Version

Introduced in version 2026
