# SimplifyDefinition.Parent Property

Parent Object: [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SimplifyDefinition.**Parent**() As [SimplifyFeature](../SimplifyFeature/SimplifyFeature.md)

## Property Value

This is a read only property whose value is a [SimplifyFeature](../SimplifyFeature/SimplifyFeature.md).

## Version

Introduced in version 2026
