# NotebookOptions.ArrowColor Property

Parent Object: [NotebookOptions](../NotebookOptions/NotebookOptions.md)

## Description

Gets/Sets the color for arrows in design notes.

## Syntax

NotebookOptions.**ArrowColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
