# NotebookOptions Object

## Description

The NotebookOptions object provides access to properties that provide read and write access of the notebook related application options. This is somewhat equivalent to the Notebook tab of the Application Options dialog.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../NotebookOptions/NotebookOptions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [ArrowColor](../NotebookOptions/NotebookOptions_ArrowColor.md) | Gets/Sets the color for arrows in design notes. |
| [DisplayNoteIcons](../NotebookOptions/NotebookOptions_DisplayNoteIcons.md) | Gets/Sets whether enables or disables displaying icons for design notes in the graphics window. |
| [DisplayNoteText](../NotebookOptions/NotebookOptions_DisplayNoteText.md) | Gets/Sets whether enables or disables displaying text for design notes. |
| [KeepNotesOnDeletedObjects](../NotebookOptions/NotebookOptions_KeepNotesOnDeletedObjects.md) | Gets/Sets whether design notes associated with model geometry should be retained after the geometry has been deleted. |
| [NoteHighlightColor](../NotebookOptions/NotebookOptions_NoteHighlightColor.md) | Gets/Sets the color for the highlighted component in note views. |
| [TextBackgroundColor](../NotebookOptions/NotebookOptions_TextBackgroundColor.md) | Gets/Sets the background color for the comment boxes in design notes. |
| [Type](../NotebookOptions/NotebookOptions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Application.NotebookOptions](../Application/Application_NotebookOptions.md), [InventorServer.NotebookOptions](InventorServer_NotebookOptions.md), [InventorServerObject.NotebookOptions](InventorServerObject_NotebookOptions.md)
