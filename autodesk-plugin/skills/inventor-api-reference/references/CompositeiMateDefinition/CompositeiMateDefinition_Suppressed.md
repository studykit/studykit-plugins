# CompositeiMateDefinition.Suppressed Property

Parent Object: [CompositeiMateDefinition](../CompositeiMateDefinition/CompositeiMateDefinition.md)

## Description

Gets the boolean flag that indicates whether this iMateDefinition is suppressed or not.

## Syntax

CompositeiMateDefinition.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
