# CentermarkStyle.DefaultRadius Property

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Gets and sets the size of a sick center mark if it is left detached from geometry.

## Syntax

CentermarkStyle.**DefaultRadius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
