# SketchArc3DProxy.HasReferenceComponent Property

Parent Object: [SketchArc3DProxy](../SketchArc3DProxy/SketchArc3DProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchArc3DProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
