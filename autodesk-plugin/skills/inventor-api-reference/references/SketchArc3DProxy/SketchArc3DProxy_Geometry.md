# SketchArc3DProxy.Geometry Property

Parent Object: [SketchArc3DProxy](../SketchArc3DProxy/SketchArc3DProxy.md)

## Description

Gets and sets an Arc3d geometry object. The object returned represents a 'snapshot' view of the current state of the sketch arc.

## Syntax

SketchArc3DProxy.**Geometry**() As [Arc3d](../Arc3d/Arc3d.md)

## Property Value

This is a read/write property whose value is an [Arc3d](../Arc3d/Arc3d.md).

## Version

Introduced in version 6
