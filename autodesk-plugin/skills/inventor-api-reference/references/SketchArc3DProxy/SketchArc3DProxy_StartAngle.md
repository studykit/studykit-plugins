# SketchArc3DProxy.StartAngle Property

Parent Object: [SketchArc3DProxy](../SketchArc3DProxy/SketchArc3DProxy.md)

## Description

Gets the StartAngle of the arc.

## Syntax

SketchArc3DProxy.**StartAngle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
