# SketchArc3DProxy.CurvatureDisplay Property

Parent Object: [SketchArc3DProxy](../SketchArc3DProxy/SketchArc3DProxy.md)

## Description

Specifies whether curvature information is displayed for the arc.

## Syntax

SketchArc3DProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
