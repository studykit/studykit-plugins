# MarkStyle.MarkTextType Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that gets and sets the mark text type.

## Syntax

MarkStyle.**MarkTextType**() As [MarkTextTypeEnum](../MarkTextTypeEnum.md)

## Property Value

This is a read/write property whose value is a [MarkTextTypeEnum](../MarkTextTypeEnum.md).

## Version

Introduced in version 2023
