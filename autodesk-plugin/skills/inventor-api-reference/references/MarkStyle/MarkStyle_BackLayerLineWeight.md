# MarkStyle.BackLayerLineWeight Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that gets and sets the line weight for back Layer.

## Syntax

MarkStyle.**BackLayerLineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2023
