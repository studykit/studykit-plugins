# MarkStyle.StrokesFontName Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that gets and sets the mark text strokes font name.

## Syntax

MarkStyle.**StrokesFontName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2023
