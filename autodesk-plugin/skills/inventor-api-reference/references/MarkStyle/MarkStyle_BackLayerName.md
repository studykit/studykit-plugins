# MarkStyle.BackLayerName Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that gets and sets the layer name for back layer.

## Syntax

MarkStyle.**BackLayerName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2023
