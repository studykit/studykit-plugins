# MarkStyle.GetFrontCustomLineType Method

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Method that returns information regarding the custom line type in use.

## Syntax

MarkStyle.**GetFrontCustomLineType**( ***LineTypeName*** As String, ***LineTypeDescription*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LineTypeName | String | The name of the custom line type in use. |
| LineTypeDescription | String | The description of the line type in use. |

## Version

Introduced in version 2023
