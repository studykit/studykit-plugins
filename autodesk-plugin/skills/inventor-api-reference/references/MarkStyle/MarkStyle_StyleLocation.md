# MarkStyle.StyleLocation Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Gets the location status of the mark style.

## Syntax

MarkStyle.**StyleLocation**() As [StyleLocationEnum](../StyleLocationEnum.md)

## Property Value

This is a read only property whose value is a [StyleLocationEnum](../StyleLocationEnum.md).

## Version

Introduced in version 2023
