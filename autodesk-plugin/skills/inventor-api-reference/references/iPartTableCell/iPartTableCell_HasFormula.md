# iPartTableCell.HasFormula Property

Parent Object: [iPartTableCell](../iPartTableCell/iPartTableCell.md)

## Description

Property that returns whether a formula (equation) was input into this cell via Excel. Such cells show with a red background in the user interface and are not editable.

## Syntax

iPartTableCell.**HasFormula**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
