# FilletDefinition.FilletType Property

Parent Object: [FilletDefinition](../FilletDefinition/FilletDefinition.md)

## Description

Property that returns the type of the fillet. Possible return values are kEdgeFillet, kFaceFillet and kFullRoundFillet.

## Syntax

FilletDefinition.**FilletType**() As [FilletTypeEnum](../FilletTypeEnum.md)

## Property Value

This is a read only property whose value is a [FilletTypeEnum](../FilletTypeEnum.md).

## Version

Introduced in version 11
