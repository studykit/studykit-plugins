# FilletDefinition.FullRoundSet Property

Parent Object: [FilletDefinition](../FilletDefinition/FilletDefinition.md)

## Description

property that returns the FilletFullRoundSet object that defines the inputs used to create a full round fillet. This property returns Nothing if the FilletType is not kFullRoundFillet.

## Syntax

FilletDefinition.**FullRoundSet**() As [FilletFullRoundSet](../FilletFullRoundSet/FilletFullRoundSet.md)

## Property Value

This is a read only property whose value is a [FilletFullRoundSet](../FilletFullRoundSet/FilletFullRoundSet.md).

## Version

Introduced in version 11
