# FilletDefinition.FaceSet Property

Parent Object: [FilletDefinition](../FilletDefinition/FilletDefinition.md)

## Description

Property that returns the FilletConstantRadiusFaceSet object that defines the inputs used to create a face fillet. This property returns Nothing if the FilletType is not kFaceFillet.

## Syntax

FilletDefinition.**FaceSet**() As [FilletConstantRadiusFaceSet](../FilletConstantRadiusFaceSet/FilletConstantRadiusFaceSet.md)

## Property Value

This is a read only property whose value is a [FilletConstantRadiusFaceSet](../FilletConstantRadiusFaceSet/FilletConstantRadiusFaceSet.md).

## Version

Introduced in version 11
