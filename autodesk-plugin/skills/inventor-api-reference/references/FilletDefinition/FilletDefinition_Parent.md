# FilletDefinition.Parent Property

Parent Object: [FilletDefinition](../FilletDefinition/FilletDefinition.md)

## Description

Property that returns the parent FilletFeature of the definition.

## Syntax

FilletDefinition.**Parent**() As [FilletFeature](../FilletFeature/FilletFeature.md)

## Property Value

This is a read only property whose value is a [FilletFeature](../FilletFeature/FilletFeature.md).

## Version

Introduced in version 5.3
