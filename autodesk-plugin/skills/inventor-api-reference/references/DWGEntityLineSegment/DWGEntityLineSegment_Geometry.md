# DWGEntityLineSegment.Geometry Property

Parent Object: [DWGEntityLineSegment](../DWGEntityLineSegment/DWGEntityLineSegment.md)

## Description

Read-only property that returns the segment geometry.

## Syntax

DWGEntityLineSegment.**Geometry**() As [LineSegment](../LineSegment/LineSegment.md)

## Property Value

This is a read only property whose value is a [LineSegment](../LineSegment/LineSegment.md).

## Version

Introduced in version 2018
