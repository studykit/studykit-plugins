# LinearHolePlacementDefinition.ReferenceEntityOne Property

Parent Object: [LinearHolePlacementDefinition](../LinearHolePlacementDefinition/LinearHolePlacementDefinition.md)

## Description

Property that indicates the first entity referenced for dimensioning the placement of the hole. This can only be a linear Edge.

## Syntax

LinearHolePlacementDefinition.**ReferenceEntityOne**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
