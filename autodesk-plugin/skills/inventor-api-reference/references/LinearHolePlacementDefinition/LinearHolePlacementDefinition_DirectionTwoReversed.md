# LinearHolePlacementDefinition.DirectionTwoReversed Property

Parent Object: [LinearHolePlacementDefinition](../LinearHolePlacementDefinition/LinearHolePlacementDefinition.md)

## Description

Gets and sets whether the direction of the hole placement with respect to the second reference entity is reversed.

## Syntax

LinearHolePlacementDefinition.**DirectionTwoReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
