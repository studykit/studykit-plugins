# DefaultBorder.Parent Property

Parent Object: [DefaultBorder](../DefaultBorder/DefaultBorder.md)

## Description

Property returning the parent sheet object.

## Syntax

DefaultBorder.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 5.3
