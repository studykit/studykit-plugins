# DefaultBorder.TopMargin Property

Parent Object: [DefaultBorder](../DefaultBorder/DefaultBorder.md)

## Description

Property that returns the space between the top edge of the sheet and the border line. The distance is specified in centimeters.

## Syntax

DefaultBorder.**TopMargin**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 5.3
