# DefaultBorder.HorizontalZoneLabelMode Property

Parent Object: [DefaultBorder](../DefaultBorder/DefaultBorder.md)

## Description

Property that returns the label style for the horizontal labels. Valid values are kBorderLabelModeAlphabetical, kBorderLabelModeNumeric, and kBorderLabelModeNone.

## Syntax

DefaultBorder.**HorizontalZoneLabelMode**() As [BorderLabelModeEnum](../BorderLabelModeEnum.md)

## Property Value

This is a read only property whose value is a [BorderLabelModeEnum](../BorderLabelModeEnum.md).

## Version

Introduced in version 5.3
