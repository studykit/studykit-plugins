# DefaultBorder.BottomMargin Property

Parent Object: [DefaultBorder](../DefaultBorder/DefaultBorder.md)

## Description

Property that returns the space between the bottom edge of the sheet and the border line. The distance is specified in centimeters.

## Syntax

DefaultBorder.**BottomMargin**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 5.3
