# DefaultBorder.DelimitZonesByLines Property

Parent Object: [DefaultBorder](../DefaultBorder/DefaultBorder.md)

## Description

Property that returns if delimit zones are indicated by lines or arrowheads. A value of True indicates lines are used and False specifies arrowheads.

## Syntax

DefaultBorder.**DelimitZonesByLines**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5.3
