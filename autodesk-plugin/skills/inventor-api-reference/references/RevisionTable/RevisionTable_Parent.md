# RevisionTable.Parent Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Property that returns the parent object of this RevisionTable.

## Syntax

RevisionTable.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 10
