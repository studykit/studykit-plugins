# RevisionTable.WrapLeft Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets whether the sections of the revision table are moved to the left or right when the number of rows increase.

## Syntax

RevisionTable.**WrapLeft**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
