# RevisionTable.ShowTitle Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets whether to show the title of the revision table.

## Syntax

RevisionTable.**ShowTitle**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
