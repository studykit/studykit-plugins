# RevisionTable.Position Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets/Sets the position of the RevisionTable on the sheet.

## Syntax

RevisionTable.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
