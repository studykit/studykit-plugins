# RevisionTable.MaximumRows Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets the maximum number of rows per section.

## Syntax

RevisionTable.**MaximumRows**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2012
