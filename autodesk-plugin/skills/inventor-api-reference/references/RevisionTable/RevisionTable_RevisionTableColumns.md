# RevisionTable.RevisionTableColumns Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Property that returns the RevisionTableColumns collection object.

## Syntax

RevisionTable.**RevisionTableColumns**() As [RevisionTableColumns](../RevisionTableColumns/RevisionTableColumns.md)

## Property Value

This is a read only property whose value is a [RevisionTableColumns](../RevisionTableColumns/RevisionTableColumns.md).
