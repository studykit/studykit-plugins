# DoubleHemDefinition.Length Property

Parent Object: [DoubleHemDefinition](../DoubleHemDefinition/DoubleHemDefinition.md)

## Description

Property that returns the parameter controlling the length of a hem feature defined by a double bend style of bend.

## Syntax

DoubleHemDefinition.**Length**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
