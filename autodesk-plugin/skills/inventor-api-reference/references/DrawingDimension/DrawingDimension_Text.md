# DrawingDimension.Text Property

Parent Object: [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Description

Gets and sets the DimensionText setting.

## Syntax

DrawingDimension.**Text**() As [DimensionText](../DimensionText/DimensionText.md)

## Property Value

This is a read/write property whose value is a [DimensionText](../DimensionText/DimensionText.md).
