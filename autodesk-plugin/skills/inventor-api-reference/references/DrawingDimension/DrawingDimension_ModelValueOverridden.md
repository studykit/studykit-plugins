# DrawingDimension.ModelValueOverridden Property

Parent Object: [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Description

Read-write property that gets and sets whether the model value is overridden for the dimension.

## Syntax

DrawingDimension.**ModelValueOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
