# DrawingDimension.OverrideModelValue Property

Parent Object: [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Description

Gets and sets the NominalValue setting.

## Syntax

DrawingDimension.**OverrideModelValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
