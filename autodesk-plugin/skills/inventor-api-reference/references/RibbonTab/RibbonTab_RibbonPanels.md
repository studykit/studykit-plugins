# RibbonTab.RibbonPanels Property

Parent Object: [RibbonTab](../RibbonTab/RibbonTab.md)

## Description

Property that returns the RibbonPanels collection object.

## Syntax

RibbonTab.**RibbonPanels**() As [RibbonPanels](../RibbonPanels/RibbonPanels.md)

## Property Value

This is a read only property whose value is a [RibbonPanels](../RibbonPanels/RibbonPanels.md).
