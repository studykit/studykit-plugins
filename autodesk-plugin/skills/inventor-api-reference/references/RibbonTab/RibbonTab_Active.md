# RibbonTab.Active Property

Parent Object: [RibbonTab](../RibbonTab/RibbonTab.md)

## Description

Gets and sets whether this tab is currently active. Setting this property returns an error if the parent ribbon is not active.

## Syntax

RibbonTab.**Active**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
