# RibbonTab.KeyTip Property

Parent Object: [RibbonTab](../RibbonTab/RibbonTab.md)

## Description

Gets and sets the keyboard access key for the tab.

## Syntax

RibbonTab.**KeyTip**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
