# RibbonTab.Contextual Property

Parent Object: [RibbonTab](../RibbonTab/RibbonTab.md)

## Description

Gets and sets whether this tab is contextual.

## Syntax

RibbonTab.**Contextual**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
