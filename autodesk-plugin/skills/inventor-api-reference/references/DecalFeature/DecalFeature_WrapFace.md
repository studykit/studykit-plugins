# DecalFeature.WrapFace Property

Parent Object: [DecalFeature](../DecalFeature/DecalFeature.md)

## Description

Specifies if the decal is wrapped on the faces on which it is applied.

## Syntax

DecalFeature.**WrapFace**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
