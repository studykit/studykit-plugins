# TitleBlock.Parent Property

Parent Object: [TitleBlock](../TitleBlock/TitleBlock.md)

## Description

Property returning the parent sheet object.

## Syntax

TitleBlock.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 5.3
