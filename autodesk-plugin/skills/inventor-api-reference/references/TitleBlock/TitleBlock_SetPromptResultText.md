# TitleBlock.SetPromptResultText Method

Parent Object: [TitleBlock](../TitleBlock/TitleBlock.md)

## Description

Method that sets the text that was supplied for a specified title block that contains prompted text. The string displayed within this title block is changed.

## Syntax

TitleBlock.**SetPromptResultText**( ***DefinitionText*** As [TextBox](../TextBox/TextBox.md), ***NewValue*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DefinitionText | [TextBox](../TextBox/TextBox.md) | Input TextBox object from the referenced TitleBlockDefinition object. This text box is used to specify which prompted text box to set the text for. |
| NewValue | String | Input String that defines the new text to use for the specified prompted text. |

## Version

Introduced in version 5.3
