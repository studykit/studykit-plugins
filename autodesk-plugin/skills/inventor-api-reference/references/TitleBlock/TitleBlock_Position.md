# TitleBlock.Position Property

Parent Object: [TitleBlock](../TitleBlock/TitleBlock.md)

## Description

Property that returns the position on the sheet where the origin of the title block definition's sketch is positioned.

## Syntax

TitleBlock.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5.3
