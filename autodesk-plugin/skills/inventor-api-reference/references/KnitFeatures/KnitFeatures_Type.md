# KnitFeatures.Type Property

Parent Object: [KnitFeatures](../KnitFeatures/KnitFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

KnitFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
