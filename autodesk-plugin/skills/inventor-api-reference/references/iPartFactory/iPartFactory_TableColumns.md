# iPartFactory.TableColumns Property

Parent Object: [iPartFactory](../iPartFactory/iPartFactory.md)

## Description

Property that returns the iPartTableColumns collection object.

## Syntax

iPartFactory.**TableColumns**() As [iPartTableColumns](../iPartTableColumns/iPartTableColumns.md)

## Property Value

This is a read only property whose value is an [iPartTableColumns](../iPartTableColumns/iPartTableColumns.md).
