# iPartFactory.MemberEditScope Property

Parent Object: [iPartFactory](../iPartFactory/iPartFactory.md)

## Description

Gets and sets whether the edits to a member should affect just the active member or all members within the factory.

## Syntax

iPartFactory.**MemberEditScope**() As [MemberEditScopeEnum](../MemberEditScopeEnum.md)

## Property Value

This is a read/write property whose value is a [MemberEditScopeEnum](../MemberEditScopeEnum.md).

## Version

Introduced in version 11
