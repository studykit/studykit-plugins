# iPartFactory.DefaultRow Property

Parent Object: [iPartFactory](../iPartFactory/iPartFactory.md)

## Description

Gets and sets the default row for this iPart factory.

## Syntax

iPartFactory.**DefaultRow**() As [iPartTableRow](../iPartTableRow/iPartTableRow.md)

## Property Value

This is a read/write property whose value is an [iPartTableRow](../iPartTableRow/iPartTableRow.md).

## Version

Introduced in version 6
