# TextBoxes.Item Property

Parent Object: [TextBoxes](../TextBoxes/TextBoxes.md)

## Description

Returns a TextBox from the collection.

## Syntax

TextBoxes.**Item**( ***Index*** As Long ) As [TextBox](../TextBox/TextBox.md)

## Property Value

This is a read only property whose value is a [TextBox](../TextBox/TextBox.md).
