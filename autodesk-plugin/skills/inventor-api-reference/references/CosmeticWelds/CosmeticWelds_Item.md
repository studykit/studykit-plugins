# CosmeticWelds.Item Property

Parent Object: [CosmeticWelds](../CosmeticWelds/CosmeticWelds.md)

## Description

Returns the specified CosmeticWeld object from the collection.

## Syntax

CosmeticWelds.**Item**( ***Index*** As Variant ) As [CosmeticWeld](../CosmeticWeld/CosmeticWeld.md)

## Property Value

This is a read only property whose value is a [CosmeticWeld](../CosmeticWeld/CosmeticWeld.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the CosmeticWeld to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the CosmeticWeld name. If an out of range index or a name of a non-existent CosmeticWeld is provided, an error will occur. |

## Version

Introduced in version 8
