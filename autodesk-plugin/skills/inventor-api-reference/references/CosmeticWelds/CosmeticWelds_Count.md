# CosmeticWelds.Count Property

Parent Object: [CosmeticWelds](../CosmeticWelds/CosmeticWelds.md)

## Description

Property that returns the number of items in the collection

## Syntax

CosmeticWelds.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 8
