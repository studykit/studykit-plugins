# CosmeticWelds.Type Property

Parent Object: [CosmeticWelds](../CosmeticWelds/CosmeticWelds.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CosmeticWelds.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 8
