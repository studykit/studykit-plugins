# CosmeticWelds.Add Method

Parent Object: [CosmeticWelds](../CosmeticWelds/CosmeticWelds.md)

## Description

Method that creates a CosmeticWeld and returns the newly created CosmeticWeld object.

## Syntax

CosmeticWelds.**Add**( ***CosmeticWeldDefinition*** As [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md) ) As [CosmeticWeld](../CosmeticWeld/CosmeticWeld.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CosmeticWeldDefinition | [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md) | Input the CosmeticWeldDefinition object that defines the cosmetic weld you want to create. |

## Version

Introduced in version 2026
