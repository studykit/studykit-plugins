# CosmeticWelds Object

## Description

The CosmeticWelds object provides access to all the existing objects within a weldment.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../CosmeticWelds/CosmeticWelds_Add.md) | Method that creates a CosmeticWeld and returns the newly created CosmeticWeld object. |
| [CreateDefinition](../CosmeticWelds/CosmeticWelds_CreateDefinition.md) | Method that creates a CosmeticWeldDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CosmeticWelds/CosmeticWelds_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CosmeticWelds/CosmeticWelds_Count.md) | Property that returns the number of items in the collection |
| [Item](../CosmeticWelds/CosmeticWelds_Item.md) | Returns the specified CosmeticWeld object from the collection. |
| [Type](../CosmeticWelds/CosmeticWelds_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Welds.CosmeticWelds](../Welds/Welds_CosmeticWelds.md)

## Version

Introduced in version 8
