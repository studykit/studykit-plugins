# CoilFeatures.Item Property

Parent Object: [CoilFeatures](../CoilFeatures/CoilFeatures.md)

## Description

Returns the specified CoilFeature object from the collection.

## Syntax

CoilFeatures.**Item**( ***Index*** As Variant ) As [CoilFeature](../CoilFeature/CoilFeature.md)

## Property Value

This is a read only property whose value is a [CoilFeature](../CoilFeature/CoilFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the feature to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the feature's name. If an out of range index or a name of a non-existent feature is provided, an error occurs. |

## Version

Introduced in version 5
