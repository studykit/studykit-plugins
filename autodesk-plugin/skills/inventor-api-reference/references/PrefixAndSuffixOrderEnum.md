# PrefixAndSuffixOrderEnum Enumerator

## Description

Enum specifying the location of prefix and suffix.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAboveAndBelow | 42498 | The location of prefix and suffix to be above and below the text. |
| kBeforeAndAfter | 42497 | The location of prefix and suffix to be before and after the text. |

## Version

Introduced in version 9
