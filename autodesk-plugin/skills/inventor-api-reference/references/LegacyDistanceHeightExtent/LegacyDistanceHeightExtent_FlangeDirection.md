# LegacyDistanceHeightExtent.FlangeDirection Property

Parent Object: [LegacyDistanceHeightExtent](../LegacyDistanceHeightExtent/LegacyDistanceHeightExtent.md)

## Description

Gets and sets the direction of the flange.

## Syntax

LegacyDistanceHeightExtent.**FlangeDirection**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2009
