# LegacyDistanceHeightExtent.Distance Property

Parent Object: [LegacyDistanceHeightExtent](../LegacyDistanceHeightExtent/LegacyDistanceHeightExtent.md)

## Description

Property that returns the parameter controlling the extent distance of the flange.

## Syntax

LegacyDistanceHeightExtent.**Distance**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
