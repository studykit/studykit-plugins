# LegacyDistanceHeightExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The LegacyDistanceHeightExtent is used by the FlangeFeature object when the 'Old Method' is specified in the Flange feature dialog.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LegacyDistanceHeightExtent/LegacyDistanceHeightExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Distance](../LegacyDistanceHeightExtent/LegacyDistanceHeightExtent_Distance.md) | Property that returns the parameter controlling the extent distance of the flange. |
| [FlangeDirection](../LegacyDistanceHeightExtent/LegacyDistanceHeightExtent_FlangeDirection.md) | Gets and sets the direction of the flange. |
| [Parent](../LegacyDistanceHeightExtent/LegacyDistanceHeightExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Type](../LegacyDistanceHeightExtent/LegacyDistanceHeightExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2009
