# BossSet Object

## Description

Object containing boss definition of the rib.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../BossSet/BossSet_Delete.md) | Method that deletes the boss set from the collection. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BossSet/BossSet_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Centers](../BossSet/BossSet_Centers.md) | Read-write property that gets and sets the collection of SketchPoint objects which specify the boss centers. |
| [Diameter](../BossSet/BossSet_Diameter.md) | Read-write property that returns the boss diameter value. |
| [Offset](../BossSet/BossSet_Offset.md) | Read-write property that returns the value of boss offset from the sketch plane. |
| [Parent](../BossSet/BossSet_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [TaperAngle](../BossSet/BossSet_TaperAngle.md) | Read-write property that returns the boss taper angle value. |
| [Type](../BossSet/BossSet_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BossSets.Add](../BossSets/BossSets_Add.md), [BossSets.Item](../BossSets/BossSets_Item.md)

## Version

Introduced in version 2012
