# BossSet.Parent Property

Parent Object: [BossSet](../BossSet/BossSet.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

BossSet.**Parent**() As [RibDefinition](../RibDefinition/RibDefinition.md)

## Property Value

This is a read only property whose value is a [RibDefinition](../RibDefinition/RibDefinition.md).

## Version

Introduced in version 2012
