# SketchDrivenPatternDefinition.Operation Property

Parent Object: [SketchDrivenPatternDefinition](../SketchDrivenPatternDefinition/SketchDrivenPatternDefinition.md)

## Description

Gets and sets the operation type. Valid values are kNewBodyOperation and kJoinOperation.

## Syntax

SketchDrivenPatternDefinition.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2017
