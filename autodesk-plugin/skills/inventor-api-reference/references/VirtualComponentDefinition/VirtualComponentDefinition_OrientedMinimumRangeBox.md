# VirtualComponentDefinition.OrientedMinimumRangeBox Property

Parent Object: [VirtualComponentDefinition](../VirtualComponentDefinition/VirtualComponentDefinition.md)

## Description

Read-only property that returns the oriented minimum range box for this object.

## Syntax

VirtualComponentDefinition.**OrientedMinimumRangeBox**() As [OrientedBox](../OrientedBox/OrientedBox.md)

## Property Value

This is a read only property whose value is an [OrientedBox](../OrientedBox/OrientedBox.md).

## Version

Introduced in version 2024
