# VirtualComponentDefinition.ActiveMaterial Property

Parent Object: [VirtualComponentDefinition](../VirtualComponentDefinition/VirtualComponentDefinition.md)

## Description

Gets and sets the material for the VirtualComponentDefinition.

## Syntax

VirtualComponentDefinition.**ActiveMaterial**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2024
