# VirtualComponentDefinition.PropertySets Property

Parent Object: [VirtualComponentDefinition](../VirtualComponentDefinition/VirtualComponentDefinition.md)

## Description

Property that gets the PropertySets object associated with the virtual component.

## Syntax

VirtualComponentDefinition.**PropertySets**() As [PropertySets](../PropertySets/PropertySets.md)

## Property Value

This is a read only property whose value is a [PropertySets](../PropertySets/PropertySets.md).
