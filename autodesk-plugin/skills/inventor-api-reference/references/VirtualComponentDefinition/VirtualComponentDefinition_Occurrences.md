# VirtualComponentDefinition.Occurrences Property

Parent Object: [VirtualComponentDefinition](../VirtualComponentDefinition/VirtualComponentDefinition.md)

## Description

Property that returns the collection object.

## Syntax

VirtualComponentDefinition.**Occurrences**() As [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md).

## Version

Introduced in version 10
