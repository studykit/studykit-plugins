# TitleBlockAlignmentEnum Enumerator

## Description

Enum indicating the default title block location.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kLowerLeftTitleBlockAlignment | 66818 | Lower left title block. |
| kLowerRightTitleBlockAlignment | 66820 | Lower right title block. |
| kUpperLeftTitleBlockAlignment | 66817 | Upper left title block. |
| kUpperRightTitleBlockAlignment | 66819 | Upper right title block. |

## Version

Introduced in version 2008
