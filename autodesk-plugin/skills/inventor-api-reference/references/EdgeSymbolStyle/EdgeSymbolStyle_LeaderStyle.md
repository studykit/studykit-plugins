# EdgeSymbolStyle.LeaderStyle Property

Parent Object: [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md)

## Description

Read-write property that gets and sets the leader style used for the edge symbol style.

## Syntax

EdgeSymbolStyle.**LeaderStyle**() As [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Property Value

This is a read/write property whose value is a [LeaderStyle](../LeaderStyle/LeaderStyle.md).

## Version

Introduced in version 2024
