# EdgeSymbolStyle.ConvertToLocal Method

Parent Object: [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

EdgeSymbolStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2024
