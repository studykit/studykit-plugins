# MirrorFeatureDefinition.MirrorPlaneEntity Property

Parent Object: [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md)

## Description

Read-write property that gets and sets the mirror plane. This can be either a planar face or work plane.

## Syntax

MirrorFeatureDefinition.**MirrorPlaneEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2017
