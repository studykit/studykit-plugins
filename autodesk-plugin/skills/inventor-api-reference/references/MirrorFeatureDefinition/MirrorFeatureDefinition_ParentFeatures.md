# MirrorFeatureDefinition.ParentFeatures Property

Parent Object: [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md)

## Description

Read-write property that gets and sets the parent features of the pattern.

## Syntax

MirrorFeatureDefinition.**ParentFeatures**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2017
