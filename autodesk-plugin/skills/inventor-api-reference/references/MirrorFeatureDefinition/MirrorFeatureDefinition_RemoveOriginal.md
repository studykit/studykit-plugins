# MirrorFeatureDefinition.RemoveOriginal Property

Parent Object: [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md)

## Description

Read-write property that gets and sets whether to remove the original portion after the mirror operation. This property only applies if MirrorOfBody is True.

## Syntax

MirrorFeatureDefinition.**RemoveOriginal**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
