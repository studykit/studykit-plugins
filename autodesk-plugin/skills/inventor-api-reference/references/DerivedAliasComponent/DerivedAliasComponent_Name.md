# DerivedAliasComponent.Name Property

Parent Object: [DerivedAliasComponent](../DerivedAliasComponent/DerivedAliasComponent.md)

## Description

Property that returns the component's name.

## Syntax

DerivedAliasComponent.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2010
