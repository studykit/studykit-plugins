# iAssemblyMember.Row Property

Parent Object: [iAssemblyMember](../iAssemblyMember/iAssemblyMember.md)

## Description

Property that returns the row in the iAssembly table that this iAssemblyMember represents.

## Syntax

iAssemblyMember.**Row**() As [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md)

## Property Value

This is a read only property whose value is an [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md).

## Version

Introduced in version 11
