# ShrinkwrapDefinition.FullFileName Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets the full filename of the shrinkwrapped document.

## Syntax

ShrinkwrapDefinition.**FullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2019
