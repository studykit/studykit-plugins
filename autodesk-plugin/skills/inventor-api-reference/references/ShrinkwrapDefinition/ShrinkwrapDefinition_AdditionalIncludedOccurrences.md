# ShrinkwrapDefinition.AdditionalIncludedOccurrences Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the occurrences being included for the shrinkwrap.

## Syntax

ShrinkwrapDefinition.**AdditionalIncludedOccurrences**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2019
