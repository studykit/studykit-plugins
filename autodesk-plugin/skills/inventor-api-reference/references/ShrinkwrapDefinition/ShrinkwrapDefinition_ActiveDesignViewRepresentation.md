# ShrinkwrapDefinition.ActiveDesignViewRepresentation Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the name of the active Design View Representation for the shrinkwrap component.

## Syntax

ShrinkwrapDefinition.**ActiveDesignViewRepresentation**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
