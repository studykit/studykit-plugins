# ShrinkwrapDefinition.RemoveEmbossesStyle Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the style to remove emboss for the shrinkwrap.

## Syntax

ShrinkwrapDefinition.**RemoveEmbossesStyle**() As [ShrinkwrapRemoveStyleEnum](../ShrinkwrapRemoveStyleEnum.md)

## Property Value

This is a read/write property whose value is a [ShrinkwrapRemoveStyleEnum](../ShrinkwrapRemoveStyleEnum.md).

## Version

Introduced in version 2023
