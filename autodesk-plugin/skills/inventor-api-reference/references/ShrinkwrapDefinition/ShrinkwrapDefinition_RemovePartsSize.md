# ShrinkwrapDefinition.RemovePartsSize Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the bounding box diagonal length in centimeters of the parts being removed.

## Syntax

ShrinkwrapDefinition.**RemovePartsSize**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
