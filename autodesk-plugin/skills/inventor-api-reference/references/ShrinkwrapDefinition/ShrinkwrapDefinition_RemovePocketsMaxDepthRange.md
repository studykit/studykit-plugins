# ShrinkwrapDefinition.RemovePocketsMaxDepthRange Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the max depth of pockets.

## Syntax

ShrinkwrapDefinition.**RemovePocketsMaxDepthRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
