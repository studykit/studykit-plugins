# ShrinkwrapDefinition.RemoveChamfersDistanceRange Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the distance of chamfers in centimeters to remove for the shrinkwrap.

## Syntax

ShrinkwrapDefinition.**RemoveChamfersDistanceRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
