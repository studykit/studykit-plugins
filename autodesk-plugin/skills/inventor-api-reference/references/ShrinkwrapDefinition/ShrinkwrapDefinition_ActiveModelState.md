# ShrinkwrapDefinition.ActiveModelState Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Read-write property that gets or sets the name of active model state for the shrinkwrap component.

## Syntax

ShrinkwrapDefinition.**ActiveModelState**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
