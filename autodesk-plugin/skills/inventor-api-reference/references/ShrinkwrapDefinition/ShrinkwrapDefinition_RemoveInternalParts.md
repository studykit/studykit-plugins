# ShrinkwrapDefinition.RemoveInternalParts Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets whether remove internal parts or not.

## Syntax

ShrinkwrapDefinition.**RemoveInternalParts**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
