# ShrinkwrapDefinition.RenameComponent Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets whether rename the component or not.

## Syntax

ShrinkwrapDefinition.**RenameComponent**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
