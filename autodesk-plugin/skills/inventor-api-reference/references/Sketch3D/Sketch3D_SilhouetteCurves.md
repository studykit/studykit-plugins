# Sketch3D.SilhouetteCurves Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Gets the SilhouetteCurve collection Object.

## Syntax

Sketch3D.**SilhouetteCurves**() As [SilhouetteCurves](../SilhouetteCurves/SilhouetteCurves.md)

## Property Value

This is a read only property whose value is a [SilhouetteCurves](../SilhouetteCurves/SilhouetteCurves.md).

## Version

Introduced in version 2012
