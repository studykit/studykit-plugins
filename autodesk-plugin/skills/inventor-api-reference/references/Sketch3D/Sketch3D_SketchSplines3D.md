# Sketch3D.SketchSplines3D Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Property that returns the SketchSplines3D collection object. This collection provides access to the existing splines in the sketch.

## Syntax

Sketch3D.**SketchSplines3D**() As [SketchSplines3D](../SketchSplines3D/SketchSplines3D.md)

## Property Value

This is a read only property whose value is a [SketchSplines3D](../SketchSplines3D/SketchSplines3D.md).

## Version

Introduced in version 6
