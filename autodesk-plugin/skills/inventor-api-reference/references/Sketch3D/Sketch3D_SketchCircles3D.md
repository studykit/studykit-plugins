# Sketch3D.SketchCircles3D Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Property that returns the SketchCircles3D collection object. This collection provides access to the existing circles in the sketch.

## Syntax

Sketch3D.**SketchCircles3D**() As [SketchCircles3D](../SketchCircles3D/SketchCircles3D.md)

## Property Value

This is a read only property whose value is a [SketchCircles3D](../SketchCircles3D/SketchCircles3D.md).

## Version

Introduced in version 6
