# Sketch3D.OnFaceCurves Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Gets the 3D Sketch OnFaceCurves collection Object.

## Syntax

Sketch3D.**OnFaceCurves**() As [OnFaceCurves](../OnFaceCurves/OnFaceCurves.md)

## Property Value

This is a read only property whose value is an [OnFaceCurves](../OnFaceCurves/OnFaceCurves.md).

## Version

Introduced in version 2017
