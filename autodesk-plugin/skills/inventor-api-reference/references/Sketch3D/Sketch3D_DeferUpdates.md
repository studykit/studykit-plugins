# Sketch3D.DeferUpdates Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Gets and sets whether to defer the solving of the sketch.

## Syntax

Sketch3D.**DeferUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
