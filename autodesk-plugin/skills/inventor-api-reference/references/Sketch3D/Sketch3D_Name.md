# Sketch3D.Name Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Gets and sets name of the 3D sketch.

## Syntax

Sketch3D.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 6
