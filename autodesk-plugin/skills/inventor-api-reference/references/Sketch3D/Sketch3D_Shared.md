# Sketch3D.Shared Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Gets and sets whether the sketch is shared or not.

## Syntax

Sketch3D.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
