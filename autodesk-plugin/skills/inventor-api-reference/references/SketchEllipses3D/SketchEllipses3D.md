# SketchEllipses3D Object

## Description

The SketchEllipses3D object provides access to all the objects in a sketch. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchEllipses3D/SketchEllipses3D_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchEllipses3D/SketchEllipses3D_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchEllipses3D/SketchEllipses3D_Item.md) | Returns the specified SketchEllipse3D object from the collection. |
| [Type](../SketchEllipses3D/SketchEllipses3D_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Sketch3D.SketchEllipses3D](../Sketch3D/Sketch3D_SketchEllipses3D.md), [Sketch3DProxy.SketchEllipses3D](../Sketch3DProxy/Sketch3DProxy_SketchEllipses3D.md)

## Version

Introduced in version 6
