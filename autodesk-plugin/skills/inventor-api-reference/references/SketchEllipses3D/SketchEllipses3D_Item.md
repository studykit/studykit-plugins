# SketchEllipses3D.Item Property

Parent Object: [SketchEllipses3D](../SketchEllipses3D/SketchEllipses3D.md)

## Description

Returns the specified SketchEllipse3D object from the collection.

## Syntax

SketchEllipses3D.**Item**( ***Index*** As Long ) As [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Property Value

This is a read only property whose value is a [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
