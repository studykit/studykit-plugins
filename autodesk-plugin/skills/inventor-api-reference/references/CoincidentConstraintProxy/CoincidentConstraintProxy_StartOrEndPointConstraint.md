# CoincidentConstraintProxy.StartOrEndPointConstraint Property

Parent Object: [CoincidentConstraintProxy](../CoincidentConstraintProxy/CoincidentConstraintProxy.md)

## Description

Indicates whether the constraint is between the curve and its endpoint.

## Syntax

CoincidentConstraintProxy.**StartOrEndPointConstraint**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
