# CoincidentConstraintProxy.Deletable Property

Parent Object: [CoincidentConstraintProxy](../CoincidentConstraintProxy/CoincidentConstraintProxy.md)

## Description

Indicates whether this object is deletable.

## Syntax

CoincidentConstraintProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.
