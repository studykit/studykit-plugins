# Matrix2d.SetCoordinateSystem Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Sets this matrix2d to be the matrix2d that maps from the standard coordinate system to the specified coordinate system.

## Syntax

Matrix2d.**SetCoordinateSystem**( ***Origin*** As [Point2d](../Point2d/Point2d.md), ***XAxis*** As [Vector2d](../Vector2d/Vector2d.md), ***YAxis*** As [Vector2d](../Vector2d/Vector2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Origin | [Point2d](../Point2d/Point2d.md) | Origin point. |
| XAxis | [Vector2d](../Vector2d/Vector2d.md) | Vector for X axis of coordinate system. |
| YAxis | [Vector2d](../Vector2d/Vector2d.md) | Vector for Y axis of coordinate system. |

## Version

Introduced in version 11
