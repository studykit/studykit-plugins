# Matrix2d.SetTranslation Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Sets the translation portion of the matrix2d.

## Syntax

Matrix2d.**SetTranslation**( ***Translation*** As [Vector2d](../Vector2d/Vector2d.md), [***ResetRotation***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Translation | [Vector2d](../Vector2d/Vector2d.md) | Vector of translation |
| ResetRotation | Boolean | Optional. If True, the rotation portion of the matrix2d is reset to identity. |

## Version

Introduced in version 11
