# Matrix2d.PutMatrixData Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Method that sets the data defining this matrix.

## Syntax

Matrix2d.**PutMatrixData**( ***Cells***() As Double )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Cells | Double | Input/output Double that defines the matrix's cells. |

## Version

Introduced in version 4
