# Matrix2d.SetToRotateTo Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Sets to the matrix2d of rotation that would align the 'from' vector2d with the 'to' vector2d.

## Syntax

Matrix2d.**SetToRotateTo**( ***From*** As [Vector2d](../Vector2d/Vector2d.md), ***To*** As [Vector2d](../Vector2d/Vector2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| From | [Vector2d](../Vector2d/Vector2d.md) | Input Vector object to align from. |
| To | [Vector2d](../Vector2d/Vector2d.md) | Input Vector object to align to. |

## Version

Introduced in version 11
