# Matrix2d.SetToAlignCoordinateSystems Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Sets this matrix2d to be the matrix2d that maps from the 'from' coordinate system to the 'to' coordinate system.

## Syntax

Matrix2d.**SetToAlignCoordinateSystems**( ***FromOrigin*** As [Point2d](../Point2d/Point2d.md), ***FromXAxis*** As [Vector2d](../Vector2d/Vector2d.md), ***FromYAxis*** As [Vector2d](../Vector2d/Vector2d.md), ***ToOrigin*** As [Point2d](../Point2d/Point2d.md), ***ToXAxis*** As [Vector2d](../Vector2d/Vector2d.md), ***ToYAxis*** As [Vector2d](../Vector2d/Vector2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FromOrigin | [Point2d](../Point2d/Point2d.md) | Origin point of 'from' coordinate system. |
| FromXAxis | [Vector2d](../Vector2d/Vector2d.md) | Vector of X axis of 'from' coordinate system. |
| FromYAxis | [Vector2d](../Vector2d/Vector2d.md) | Vector of Y axis of 'from' coordinate system. |
| ToOrigin | [Point2d](../Point2d/Point2d.md) | Origin point of "to" coordinate system. |
| ToXAxis | [Vector2d](../Vector2d/Vector2d.md) | Vector of X axis of 'to' coordinate system. |
| ToYAxis | [Vector2d](../Vector2d/Vector2d.md) | Vector of Y axis of 'to' coordinate system. |

## Version

Introduced in version 11
