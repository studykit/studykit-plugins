# Matrix2d.Translation Property

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Get the translation portion of the matrix2d.

## Syntax

Matrix2d.**Translation**() As [Vector2d](../Vector2d/Vector2d.md)

## Property Value

This is a read only property whose value is a [Vector2d](../Vector2d/Vector2d.md).

## Version

Introduced in version 11
