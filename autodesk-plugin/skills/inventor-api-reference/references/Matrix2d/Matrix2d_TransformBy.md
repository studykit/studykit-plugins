# Matrix2d.TransformBy Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Set this matrix2d to the result of this transformation followed by the specified transformation (a pre multiplication of this matrix2d by the specified matrix2d).

## Syntax

Matrix2d.**TransformBy**( ***Matrix2d*** As [Matrix2d](../Matrix2d/Matrix2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix2d | [Matrix2d](../Matrix2d/Matrix2d.md) | Matrix to use for pre-multiplication. |

## Version

Introduced in version 11
