# Matrix2d.GetCoordinateSystem Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Get the coordinate system that this matrix2d maps to from the standard coordinate system.

## Syntax

Matrix2d.**GetCoordinateSystem**( ***Origin*** As [Point2d](../Point2d/Point2d.md), ***XAxis*** As [Vector2d](../Vector2d/Vector2d.md), ***YAxis*** As [Vector2d](../Vector2d/Vector2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Origin | [Point2d](../Point2d/Point2d.md) | Specifies the origin point. |
| XAxis | [Vector2d](../Vector2d/Vector2d.md) | Vextor for the X axis of the coordinate system. |
| YAxis | [Vector2d](../Vector2d/Vector2d.md) | Vextor for the Y axis of the coordinate system. |

## Version

Introduced in version 11
