# CornerFeatureProxy.Definition Property

Parent Object: [CornerFeatureProxy](../CornerFeatureProxy/CornerFeatureProxy.md)

## Description

Property that gets and sets the CornerDefinition object associated with this corner feature.

## Syntax

CornerFeatureProxy.**Definition**() As [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Property Value

This is a read only property whose value is a [CornerDefinition](../CornerDefinition/CornerDefinition.md).

## Version

Introduced in version 2009
