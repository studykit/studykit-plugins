# CornerFeatureProxy.Adaptive Property

Parent Object: [CornerFeatureProxy](../CornerFeatureProxy/CornerFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

CornerFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
