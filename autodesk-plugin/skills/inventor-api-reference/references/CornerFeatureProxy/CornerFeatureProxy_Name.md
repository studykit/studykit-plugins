# CornerFeatureProxy.Name Property

Parent Object: [CornerFeatureProxy](../CornerFeatureProxy/CornerFeatureProxy.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

CornerFeatureProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 9
