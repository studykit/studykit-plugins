# EdgeLoop.Face Property

Parent Object: [EdgeLoop](../EdgeLoop/EdgeLoop.md)

## Description

Gets the containing this EdgeLoop.

## Syntax

EdgeLoop.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 4
