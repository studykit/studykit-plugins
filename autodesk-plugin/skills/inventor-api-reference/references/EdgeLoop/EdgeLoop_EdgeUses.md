# EdgeLoop.EdgeUses Property

Parent Object: [EdgeLoop](../EdgeLoop/EdgeLoop.md)

## Description

Gets the referenced by this EdgeLoop.

## Syntax

EdgeLoop.**EdgeUses**() As [EdgeUses](../EdgeUses/EdgeUses.md)

## Property Value

This is a read only property whose value is an [EdgeUses](../EdgeUses/EdgeUses.md).
