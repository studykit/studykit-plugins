# GrillFeatureProxy.NativeObject Property

Parent Object: [GrillFeatureProxy](../GrillFeatureProxy/GrillFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

GrillFeatureProxy.**NativeObject**() As [GrillFeature](../GrillFeature/GrillFeature.md)

## Property Value

This is a read only property whose value is a [GrillFeature](../GrillFeature/GrillFeature.md).

## Version

Introduced in version 2010
