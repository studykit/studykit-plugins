# BaselineDimensionSets.Item Property

Parent Object: [BaselineDimensionSets](../BaselineDimensionSets/BaselineDimensionSets.md)

## Description

Method that returns the specified dimension set object from the collection.

## Syntax

BaselineDimensionSets.**Item**( ***Index*** As Long ) As [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md)

## Property Value

This is a read only property whose value is a [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
