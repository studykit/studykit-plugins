# AssemblyComponentDefinition.DefaultVirtualComponentMaterial Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Gets and sets the default material associated with the assembly component. This is the material that is used for newly created virtual components.

## Syntax

AssemblyComponentDefinition.**DefaultVirtualComponentMaterial**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
