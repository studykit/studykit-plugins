# AssemblyComponentDefinition.DataIO Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Gets the object that directly deals with I/O to and from a storage-medium, including Streams(IStream).

## Syntax

AssemblyComponentDefinition.**DataIO**() As [DataIO](../DataIO/DataIO.md)

## Property Value

This is a read only property whose value is a [DataIO](../DataIO/DataIO.md).

## Version

Introduced in version 4
