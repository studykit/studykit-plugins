# AssemblyComponentDefinition.Occurrences Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Property that returns the collection object.

## Syntax

AssemblyComponentDefinition.**Occurrences**() As [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md).

## Version

Introduced in version 4
