# AssemblyComponentDefinition.Sketches Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Gets the Sketches collection for this assembly component.

## Syntax

AssemblyComponentDefinition.**Sketches**() As [PlanarSketches](../PlanarSketches/PlanarSketches.md)

## Property Value

This is a read only property whose value is a [PlanarSketches](../PlanarSketches/PlanarSketches.md).

## Version

Introduced in version 11
