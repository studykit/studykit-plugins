# AssemblyComponentDefinition.BOM Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Property that returns the BOM object.

## Syntax

AssemblyComponentDefinition.**BOM**() As [BOM](../BOM/BOM.md)

## Property Value

This is a read only property whose value is a [BOM](../BOM/BOM.md).
