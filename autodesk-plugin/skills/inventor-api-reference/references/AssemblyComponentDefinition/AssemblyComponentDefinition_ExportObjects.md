# AssemblyComponentDefinition.ExportObjects Method

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Method that marks all the input objects as exported.

## Syntax

AssemblyComponentDefinition.**ExportObjects**( ***Objects*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Objects | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Collection of objects to mark as exported. |

## Version

Introduced in version 2011
