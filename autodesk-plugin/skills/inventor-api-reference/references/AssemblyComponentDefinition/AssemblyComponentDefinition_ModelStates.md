# AssemblyComponentDefinition.ModelStates Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Read-only property that returns the ModelStates object.

## Syntax

AssemblyComponentDefinition.**ModelStates**() As [ModelStates](../ModelStates/ModelStates.md)

## Property Value

This is a read only property whose value is a [ModelStates](../ModelStates/ModelStates.md).

## Version

Introduced in version 2022
