# TransactionPointEnum Enumerator

## Description

Symbolic indicators to specific points in the transaction sequence.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kCurrentTransaction | 3588 | Current transaction. |
| kNextTransaction | 3586 | Next transaction from the current one. |
| kPreviousTransaction | 3587 | Previous transaction from the current one. |
| kUnknownTransaction | 3585 | Unknown point in the transaction sequence. |
| kUptoSpecifiedTransaction | 3589 | Point in the transaction sequence specified by another transaction. |

## Version

Introduced in version 4
