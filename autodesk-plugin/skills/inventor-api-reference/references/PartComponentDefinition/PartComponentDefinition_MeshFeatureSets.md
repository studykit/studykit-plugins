# PartComponentDefinition.MeshFeatureSets Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Read-only property that returns the MeshFeatureSets collection object.

## Syntax

PartComponentDefinition.**MeshFeatureSets**() As [MeshFeatureSets](../MeshFeatureSets/MeshFeatureSets.md)

## Property Value

This is a read only property whose value is a [MeshFeatureSets](../MeshFeatureSets/MeshFeatureSets.md).

## Version

Introduced in version 2017
