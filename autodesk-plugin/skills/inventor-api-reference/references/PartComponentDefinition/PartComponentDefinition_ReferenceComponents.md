# PartComponentDefinition.ReferenceComponents Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the ReferenceComponents collection object.

## Syntax

PartComponentDefinition.**ReferenceComponents**() As [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Property Value

This is a read only property whose value is a [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md).
