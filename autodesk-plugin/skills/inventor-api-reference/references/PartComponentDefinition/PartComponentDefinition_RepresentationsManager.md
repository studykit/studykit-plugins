# PartComponentDefinition.RepresentationsManager Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Read-only property that returns the RepresentationsManager object.

## Syntax

PartComponentDefinition.**RepresentationsManager**() As [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md)

## Property Value

This is a read only property whose value is a [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md).

## Version

Introduced in version 2012
