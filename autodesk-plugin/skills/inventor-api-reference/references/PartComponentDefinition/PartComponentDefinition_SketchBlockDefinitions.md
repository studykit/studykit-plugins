# PartComponentDefinition.SketchBlockDefinitions Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the SketchBlockDefinitions collection object.

## Syntax

PartComponentDefinition.**SketchBlockDefinitions**() As [SketchBlockDefinitions](../SketchBlockDefinitions/SketchBlockDefinitions.md)

## Property Value

This is a read only property whose value is a [SketchBlockDefinitions](../SketchBlockDefinitions/SketchBlockDefinitions.md).
