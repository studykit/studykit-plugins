# PartComponentDefinition.PointClouds Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Gets the PointClouds collection object that encapsulates all of the point clouds defined in this Component Definition.

## Syntax

PartComponentDefinition.**PointClouds**() As [PointClouds](../PointClouds/PointClouds.md)

## Property Value

This is a read only property whose value is a [PointClouds](../PointClouds/PointClouds.md).

## Version

Introduced in version 2016
