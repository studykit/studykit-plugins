# PartComponentDefinition.iPartFactory Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the iPartFactory object. This property will fail in the case where the part is not an iPart Factory. You can determine this by using the IsiPartFactory property.

## Syntax

PartComponentDefinition.**iPartFactory**() As [iPartFactory](../iPartFactory/iPartFactory.md)

## Property Value

This is a read only property whose value is an [iPartFactory](../iPartFactory/iPartFactory.md).
