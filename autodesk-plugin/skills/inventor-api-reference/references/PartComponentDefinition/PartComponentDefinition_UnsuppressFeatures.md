# PartComponentDefinition.UnsuppressFeatures Method

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Method that unsuppresses the specified features.

## Syntax

PartComponentDefinition.**UnsuppressFeatures**( ***Features*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Features | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Collection of features to unsuppress. |

## Version

Introduced in version 2009
