# PartComponentDefinition.SurfaceBodies Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns all of the *result* SurfaceBody objects contained within this ComponentDefinition.

## Syntax

PartComponentDefinition.**SurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).

## Version

Introduced in version 4
