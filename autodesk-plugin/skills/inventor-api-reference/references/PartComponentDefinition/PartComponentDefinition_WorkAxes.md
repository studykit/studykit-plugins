# PartComponentDefinition.WorkAxes Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the WorkAxes collection object.

## Syntax

PartComponentDefinition.**WorkAxes**() As [WorkAxes](../WorkAxes/WorkAxes.md)

## Property Value

This is a read only property whose value is a [WorkAxes](../WorkAxes/WorkAxes.md).
