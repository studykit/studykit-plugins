# LipFeatureProxy.NativeObject Property

Parent Object: [LipFeatureProxy](../LipFeatureProxy/LipFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

LipFeatureProxy.**NativeObject**() As [LipFeature](../LipFeature/LipFeature.md)

## Property Value

This is a read only property whose value is a [LipFeature](../LipFeature/LipFeature.md).

## Version

Introduced in version 2010
