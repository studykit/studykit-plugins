# RibFeature.Adaptive Property

Parent Object: [RibFeature](../RibFeature/RibFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

RibFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
