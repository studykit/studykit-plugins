# RibFeature.Definition Property

Parent Object: [RibFeature](../RibFeature/RibFeature.md)

## Description

Read-write property that gets and sets the RibDefinition object associated with this feature. Modifying the RibDefinition object will edit the associated RibFeature. If you want to batch up the edits to the feature, you can set a new definition object to thi.

## Syntax

RibFeature.**Definition**() As [RibDefinition](../RibDefinition/RibDefinition.md)

## Property Value

This is a read/write property whose value is a [RibDefinition](../RibDefinition/RibDefinition.md).

## Version

Introduced in version 2012
