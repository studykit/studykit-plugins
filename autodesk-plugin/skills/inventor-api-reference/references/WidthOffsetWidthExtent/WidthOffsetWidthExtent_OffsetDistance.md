# WidthOffsetWidthExtent.OffsetDistance Property

Parent Object: [WidthOffsetWidthExtent](../WidthOffsetWidthExtent/WidthOffsetWidthExtent.md)

## Description

Property that returns the parameter controlling the distance of the flange from the offset entity.

## Syntax

WidthOffsetWidthExtent.**OffsetDistance**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
