# WidthOffsetWidthExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The WidthOffsetWidthExtent is used by sheet metal features when the position of a width defined feature is relative to another entity.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../WidthOffsetWidthExtent/WidthOffsetWidthExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [OffsetDistance](../WidthOffsetWidthExtent/WidthOffsetWidthExtent_OffsetDistance.md) | Property that returns the parameter controlling the distance of the flange from the offset entity. |
| [OffsetEntity](../WidthOffsetWidthExtent/WidthOffsetWidthExtent_OffsetEntity.md) | Property that gets and sets the entity the width is measured relative to. |
| [Parent](../WidthOffsetWidthExtent/WidthOffsetWidthExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [PositiveDirection](../WidthOffsetWidthExtent/WidthOffsetWidthExtent_PositiveDirection.md) | Property that specifies the side of the offset entity the flange is on. |
| [Type](../WidthOffsetWidthExtent/WidthOffsetWidthExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Width](../WidthOffsetWidthExtent/WidthOffsetWidthExtent_Width.md) | Property that returns the parameter controlling the width of the flange. |

## Version

Introduced in version 2009
