# WidthOffsetWidthExtent.Parent Property

Parent Object: [WidthOffsetWidthExtent](../WidthOffsetWidthExtent/WidthOffsetWidthExtent.md)

## Description

Property that returns the parent PartFeature of the definition.

## Syntax

WidthOffsetWidthExtent.**Parent**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 2009
