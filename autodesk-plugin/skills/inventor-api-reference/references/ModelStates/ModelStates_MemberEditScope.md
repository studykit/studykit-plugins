# ModelStates.MemberEditScope Property

Parent Object: [ModelStates](../ModelStates/ModelStates.md)

## Description

Read-write property that gets and sets whether the edits to a member should affect just the active member or all members.

## Syntax

ModelStates.**MemberEditScope**() As [MemberEditScopeEnum](../MemberEditScopeEnum.md)

## Property Value

This is a read/write property whose value is a [MemberEditScopeEnum](../MemberEditScopeEnum.md).
