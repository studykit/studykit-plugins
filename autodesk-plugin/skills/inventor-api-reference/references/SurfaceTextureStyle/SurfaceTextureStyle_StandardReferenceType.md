# SurfaceTextureStyle.StandardReferenceType Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets the drafting standard that controls surface texture symbol format.

## Syntax

SurfaceTextureStyle.**StandardReferenceType**() As [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md)

## Property Value

This is a read/write property whose value is a [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md).

## Version

Introduced in version 2025
