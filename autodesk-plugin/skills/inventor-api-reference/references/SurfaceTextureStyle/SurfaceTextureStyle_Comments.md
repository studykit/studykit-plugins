# SurfaceTextureStyle.Comments Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets comments associated with the style.

## Syntax

SurfaceTextureStyle.**Comments**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
