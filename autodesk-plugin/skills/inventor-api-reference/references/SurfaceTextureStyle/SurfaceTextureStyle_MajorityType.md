# SurfaceTextureStyle.MajorityType Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets the majority type of this style.

## Syntax

SurfaceTextureStyle.**MajorityType**() As [MajorityTypeEnum](../MajorityTypeEnum.md)

## Property Value

This is a read/write property whose value is a [MajorityTypeEnum](../MajorityTypeEnum.md).

## Version

Introduced in version 2023
