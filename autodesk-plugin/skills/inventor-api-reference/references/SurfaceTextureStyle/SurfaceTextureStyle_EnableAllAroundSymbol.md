# SurfaceTextureStyle.EnableAllAroundSymbol Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets whether to enable the all-around indicator to the symbol.

## Syntax

SurfaceTextureStyle.**EnableAllAroundSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
