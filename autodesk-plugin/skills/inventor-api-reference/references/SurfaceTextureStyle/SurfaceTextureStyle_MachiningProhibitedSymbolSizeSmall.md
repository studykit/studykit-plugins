# SurfaceTextureStyle.MachiningProhibitedSymbolSizeSmall Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets whether to use a large or small maching-prohibited symbol. If the property is set to True, the symbol with the smaller diameter is used.

## Syntax

SurfaceTextureStyle.**MachiningProhibitedSymbolSizeSmall**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
