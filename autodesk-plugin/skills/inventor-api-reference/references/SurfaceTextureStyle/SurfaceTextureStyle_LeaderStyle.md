# SurfaceTextureStyle.LeaderStyle Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets the leader style used for a surface texture symbol.

## Syntax

SurfaceTextureStyle.**LeaderStyle**() As [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Property Value

This is a read/write property whose value is a [LeaderStyle](../LeaderStyle/LeaderStyle.md).

## Version

Introduced in version 2009
