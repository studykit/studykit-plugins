# SurfaceTextureStyle.AvailableLaySymbols Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets a bit mask defining the machining lay symbols to be made available for surface texture symbol creation.

## Syntax

SurfaceTextureStyle.**AvailableLaySymbols**() As [LayDirectionTypeEnum](../LayDirectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LayDirectionTypeEnum](../LayDirectionTypeEnum.md).

## Version

Introduced in version 2009
