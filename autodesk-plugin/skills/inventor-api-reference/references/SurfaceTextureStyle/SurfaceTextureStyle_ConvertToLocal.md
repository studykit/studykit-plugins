# SurfaceTextureStyle.ConvertToLocal Method

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

SurfaceTextureStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2009
