# PatternConstraint Object

Derived from: [GeometricConstraint](../GeometricConstraint/GeometricConstraint.md) Object

## Description

The PatternConstraint object represents a constraint placed between elements of a pattern to make these elements associative.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../PatternConstraint/PatternConstraint_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../PatternConstraint/PatternConstraint_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

| [Deletable](../PatternConstraint/PatternConstraint_Deletable.md) | Indicates whether this object is deletable. |
