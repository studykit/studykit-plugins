# MeshFeatureEntityProxy.NativeObject Property

Parent Object: [MeshFeatureEntityProxy](../MeshFeatureEntityProxy/MeshFeatureEntityProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MeshFeatureEntityProxy.**NativeObject**() As [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Property Value

This is a read only property whose value is a [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md).

## Version

Introduced in version 2017
