# MeshFeatureEntityProxy.CoordinateIndexSet Property

Parent Object: [MeshFeatureEntityProxy](../MeshFeatureEntityProxy/MeshFeatureEntityProxy.md)

## Description

Read-only property that returns the coordinate index set of the mesh feature entity.

## Syntax

MeshFeatureEntityProxy.**CoordinateIndexSet**() As [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md)

## Property Value

This is a read only property whose value is a [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md).

## Version

Introduced in version 2017
