# LinkStatusEnum Enumerator

## Description

Constants signifying the state of a link.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAmbiguousStatus | 16 | Link is ambiguous. |
| kMissingStatus | 1 | No link exists. |
| kOutOfDateStatus | 4 | Link is out of date. |
| kSourceNewerStatus | 8 | Source is newer than target. |
| kUpToDateStatus | 2 | Link is up to date. |

## Version

Introduced in version 10
