# CircularPatternFeatureProxy.Suppressed Property

Parent Object: [CircularPatternFeatureProxy](../CircularPatternFeatureProxy/CircularPatternFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

CircularPatternFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
