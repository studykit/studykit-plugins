# CircularPatternFeatureProxy.ExtendedName Property

Parent Object: [CircularPatternFeatureProxy](../CircularPatternFeatureProxy/CircularPatternFeatureProxy.md)

## Description

Read-only property that returns the full feature name including any extended information.

## Syntax

CircularPatternFeatureProxy.**ExtendedName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
