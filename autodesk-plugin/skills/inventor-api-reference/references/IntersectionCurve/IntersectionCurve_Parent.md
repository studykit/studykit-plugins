# IntersectionCurve.Parent Property

Parent Object: [IntersectionCurve](../IntersectionCurve/IntersectionCurve.md)

## Description

Read-only property that returns the parent 3D sketch of the intersection curve.

## Syntax

IntersectionCurve.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2014
