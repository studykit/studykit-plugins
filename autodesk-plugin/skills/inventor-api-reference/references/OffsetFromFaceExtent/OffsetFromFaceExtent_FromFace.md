# OffsetFromFaceExtent.FromFace Property

Parent Object: [OffsetFromFaceExtent](../OffsetFromFaceExtent/OffsetFromFaceExtent.md)

## Description

Read-write property that gets and sets the offset from face, this can be a Face or WorkPlane object.

## Syntax

OffsetFromFaceExtent.**FromFace**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2018
