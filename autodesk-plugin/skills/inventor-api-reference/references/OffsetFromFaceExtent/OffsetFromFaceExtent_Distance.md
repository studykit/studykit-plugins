# OffsetFromFaceExtent.Distance Property

Parent Object: [OffsetFromFaceExtent](../OffsetFromFaceExtent/OffsetFromFaceExtent.md)

## Description

Read-only property that returns the parameter controlling the distance extent of the feature.

## Syntax

OffsetFromFaceExtent.**Distance**() As Variant

## Property Value

This is a read only property whose value is a Variant.

## Version

Introduced in version 2018
