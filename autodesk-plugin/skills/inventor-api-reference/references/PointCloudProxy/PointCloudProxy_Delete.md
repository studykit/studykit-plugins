# PointCloudProxy.Delete Method

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Method that deletes the point cloud.

## Syntax

PointCloudProxy.**Delete**( [***RetainDependents***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| RetainDependents | Boolean | Optional input Boolean that specifies whether to retain the dependents or not. |

## Version

Introduced in version 2016
