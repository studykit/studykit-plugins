# PointCloudProxy.Regions Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-only property that returns the point cloud regions collection object.

## Syntax

PointCloudProxy.**Regions**() As [PointCloudRegions](../PointCloudRegions/PointCloudRegions.md)

## Property Value

This is a read only property whose value is a [PointCloudRegions](../PointCloudRegions/PointCloudRegions.md).

## Version

Introduced in version 2016
