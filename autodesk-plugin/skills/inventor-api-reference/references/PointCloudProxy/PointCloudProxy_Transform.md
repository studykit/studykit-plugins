# PointCloudProxy.Transform Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-write property that gets and sets the matrix that defines the translation, rotation of the point cloud in model space.

## Syntax

PointCloudProxy.**Transform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2016
