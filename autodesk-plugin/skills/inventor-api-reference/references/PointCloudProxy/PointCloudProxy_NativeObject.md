# PointCloudProxy.NativeObject Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

PointCloudProxy.**NativeObject**() As [PointCloud](../PointCloud/PointCloud.md)

## Property Value

This is a read only property whose value is a [PointCloud](../PointCloud/PointCloud.md).

## Version

Introduced in version 2016
