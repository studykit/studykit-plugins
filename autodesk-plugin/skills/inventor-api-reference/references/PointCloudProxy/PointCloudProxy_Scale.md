# PointCloudProxy.Scale Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-write property that gets and sets the scale of the point cloud.

## Syntax

PointCloudProxy.**Scale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2016
