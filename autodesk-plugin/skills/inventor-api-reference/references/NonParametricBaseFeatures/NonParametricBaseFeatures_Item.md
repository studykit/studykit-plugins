# NonParametricBaseFeatures.Item Property

Parent Object: [NonParametricBaseFeatures](../NonParametricBaseFeatures/NonParametricBaseFeatures.md)

## Description

Returns the specified NonParametricBaseFeature object from the collection.

## Syntax

NonParametricBaseFeatures.**Item**( ***Index*** As Variant ) As [NonParametricBaseFeature](../NonParametricBaseFeature/NonParametricBaseFeature.md)

## Property Value

This is a read only property whose value is a [NonParametricBaseFeature](../NonParametricBaseFeature/NonParametricBaseFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the NonParametricBaseFeature to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the border definition's name. If an out-of-range index or a name of a non-existent border definition is specified, an error occurs. |

## Version

Introduced in version 6
