# PointInferenceEnumerator.Count Property

Parent Object: [PointInferenceEnumerator](../PointInferenceEnumerator/PointInferenceEnumerator.md)

## Description

Property that returns the number of PointInference objects in the collection.

## Syntax

PointInferenceEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5
