# SymmetryConstraint.EntityTwo Property

Parent Object: [SymmetryConstraint](../SymmetryConstraint/SymmetryConstraint.md)

## Description

Property that returns the sketch entity that is constrained to be symmetric to the entity returned by the EntityOne property.

## Syntax

SymmetryConstraint.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
