# CompositeiMateDefinitionProxy.SequenceIndex Property

Parent Object: [CompositeiMateDefinitionProxy](../CompositeiMateDefinitionProxy/CompositeiMateDefinitionProxy.md)

## Description

Gets the index of this iMateDefinition.

## Syntax

CompositeiMateDefinitionProxy.**SequenceIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.
