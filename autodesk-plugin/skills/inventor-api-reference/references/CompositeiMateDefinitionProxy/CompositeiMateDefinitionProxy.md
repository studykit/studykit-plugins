# CompositeiMateDefinitionProxy Object

Derived from: [CompositeiMateDefinition](../CompositeiMateDefinition/CompositeiMateDefinition.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../CompositeiMateDefinitionProxy/CompositeiMateDefinitionProxy_Delete.md) | Method that deletes the iMateDefinition. |
