# CompositeiMateDefinitionProxy.Suppressed Property

Parent Object: [CompositeiMateDefinitionProxy](../CompositeiMateDefinitionProxy/CompositeiMateDefinitionProxy.md)

## Description

Gets the boolean flag that indicates whether this iMateDefinition is suppressed or not.

## Syntax

CompositeiMateDefinitionProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
