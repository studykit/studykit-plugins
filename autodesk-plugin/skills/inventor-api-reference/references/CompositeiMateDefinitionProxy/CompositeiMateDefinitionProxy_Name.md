# CompositeiMateDefinitionProxy.Name Property

Parent Object: [CompositeiMateDefinitionProxy](../CompositeiMateDefinitionProxy/CompositeiMateDefinitionProxy.md)

## Description

Gets the name of this iMateDefinition.

## Syntax

CompositeiMateDefinitionProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.
