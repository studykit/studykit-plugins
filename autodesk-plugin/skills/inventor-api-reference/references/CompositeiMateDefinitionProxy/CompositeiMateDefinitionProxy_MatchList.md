# CompositeiMateDefinitionProxy.MatchList Property

Parent Object: [CompositeiMateDefinitionProxy](../CompositeiMateDefinitionProxy/CompositeiMateDefinitionProxy.md)

## Description

Gets and sets an array of strings that specifies the priority order for iMate definition matches.

## Syntax

CompositeiMateDefinitionProxy.**MatchList**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 11
