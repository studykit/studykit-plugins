# UnfoldFeature.FeatureDimensions Property

Parent Object: [UnfoldFeature](../UnfoldFeature/UnfoldFeature.md)

## Description

Property that returns the FeatureDimensions collection object.

## Syntax

UnfoldFeature.**FeatureDimensions**() As [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md)

## Property Value

This is a read only property whose value is a [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md).

## Version

Introduced in version 2010
