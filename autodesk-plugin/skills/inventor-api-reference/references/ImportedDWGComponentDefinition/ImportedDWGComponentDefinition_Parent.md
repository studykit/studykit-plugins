# ImportedDWGComponentDefinition.Parent Property

Parent Object: [ImportedDWGComponentDefinition](../ImportedDWGComponentDefinition/ImportedDWGComponentDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ImportedDWGComponentDefinition.**Parent**() As [ImportedComponent](../ImportedComponent/ImportedComponent.md)

## Property Value

This is a read only property whose value is an [ImportedComponent](../ImportedComponent/ImportedComponent.md).

## Version

Introduced in version 2016
