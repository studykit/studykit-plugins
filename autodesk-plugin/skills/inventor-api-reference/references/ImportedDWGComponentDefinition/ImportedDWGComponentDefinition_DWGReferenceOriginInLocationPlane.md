# ImportedDWGComponentDefinition.DWGReferenceOriginInLocationPlane Property

Parent Object: [ImportedDWGComponentDefinition](../ImportedDWGComponentDefinition/ImportedDWGComponentDefinition.md)

## Description

Read-write property that gets and sets the position of the imported DWG component origin.

## Syntax

ImportedDWGComponentDefinition.**DWGReferenceOriginInLocationPlane**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2017
