# PartFeaturesEnumerator.Type Property

Parent Object: [PartFeaturesEnumerator](../PartFeaturesEnumerator/PartFeaturesEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

PartFeaturesEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
