# PartFeaturesEnumerator Object

## Description

The PartFeaturesEnumerator object. Presents a snapshot of the set of objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../PartFeaturesEnumerator/PartFeaturesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../PartFeaturesEnumerator/PartFeaturesEnumerator_Item.md) | Returns the specified object in the collection. The index can be numeric or the object name. |
| [Type](../PartFeaturesEnumerator/PartFeaturesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SurfaceBody.AffectedByFeatures](../SurfaceBody/SurfaceBody_AffectedByFeatures.md), [SurfaceBodyProxy.AffectedByFeatures](../SurfaceBodyProxy/SurfaceBodyProxy_AffectedByFeatures.md)

## Version

Introduced in version 5
