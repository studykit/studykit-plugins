# PartFeaturesEnumerator.Count Property

Parent Object: [PartFeaturesEnumerator](../PartFeaturesEnumerator/PartFeaturesEnumerator.md)

## Description

Property that returns the number of items in this collection.

## Syntax

PartFeaturesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5
