# PartFeaturesEnumerator.Item Property

Parent Object: [PartFeaturesEnumerator](../PartFeaturesEnumerator/PartFeaturesEnumerator.md)

## Description

Returns the specified object in the collection. The index can be numeric or the object name.

## Syntax

PartFeaturesEnumerator.**Item**( ***Index*** As Variant ) As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the PartFeature to return. |

## Version

Introduced in version 5
