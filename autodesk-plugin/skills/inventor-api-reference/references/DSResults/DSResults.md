# DSResults Object

## Description

The DSResults collection object provides access to the results associated with a degree of freedom.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DSResults/DSResults_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DSResults/DSResults_Count.md) | Gets the number of items in this collection. |
| [Item](../DSResults/DSResults_Item.md) | Returns the specified DSResult object from the collection |
| [ItemByResultType](../DSResults/DSResults_ItemByResultType.md) | Gets the specified DSResult object from the collection. |
| [Type](../DSResults/DSResults_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DSDegreeOfFreedom.Results](../DSDegreeOfFreedom/DSDegreeOfFreedom_Results.md)

## Version

Introduced in version 2013
