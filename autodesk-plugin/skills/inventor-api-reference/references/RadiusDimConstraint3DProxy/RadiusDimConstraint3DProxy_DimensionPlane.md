# RadiusDimConstraint3DProxy.DimensionPlane Property

Parent Object: [RadiusDimConstraint3DProxy](../RadiusDimConstraint3DProxy/RadiusDimConstraint3DProxy.md)

## Description

Property that returns the transient dimension plane used to place and position the dimension text for this dimension constraint.

## Syntax

RadiusDimConstraint3DProxy.**DimensionPlane**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).

## Version

Introduced in version 2010
