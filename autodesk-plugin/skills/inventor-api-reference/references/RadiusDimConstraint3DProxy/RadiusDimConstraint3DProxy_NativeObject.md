# RadiusDimConstraint3DProxy.NativeObject Property

Parent Object: [RadiusDimConstraint3DProxy](../RadiusDimConstraint3DProxy/RadiusDimConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RadiusDimConstraint3DProxy.**NativeObject**() As [RadiusDimConstraint3D](../RadiusDimConstraint3D/RadiusDimConstraint3D.md)

## Property Value

This is a read only property whose value is a [RadiusDimConstraint3D](../RadiusDimConstraint3D/RadiusDimConstraint3D.md).

## Version

Introduced in version 2010
