# RadiusDimConstraint3DProxy.Entity Property

Parent Object: [RadiusDimConstraint3DProxy](../RadiusDimConstraint3DProxy/RadiusDimConstraint3DProxy.md)

## Description

Property that returns the circle or arc being constrained.

## Syntax

RadiusDimConstraint3DProxy.**Entity**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 2010
