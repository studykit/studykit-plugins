# FloatAssetValue.Type Property

Parent Object: [FloatAssetValue](../FloatAssetValue/FloatAssetValue.md)

## Description

Read-only property returning kAssetValueObject indicating this object’s type.

## Syntax

FloatAssetValue.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
