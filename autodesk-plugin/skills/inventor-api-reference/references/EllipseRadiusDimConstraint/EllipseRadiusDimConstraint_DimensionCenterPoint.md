# EllipseRadiusDimConstraint.DimensionCenterPoint Property

Parent Object: [EllipseRadiusDimConstraint](../EllipseRadiusDimConstraint/EllipseRadiusDimConstraint.md)

## Description

Gets the center of the dimension line.

## Syntax

EllipseRadiusDimConstraint.**DimensionCenterPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
