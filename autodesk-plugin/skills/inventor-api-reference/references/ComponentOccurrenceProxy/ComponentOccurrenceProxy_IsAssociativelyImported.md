# ComponentOccurrenceProxy.IsAssociativelyImported Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets whether this occurrence is an associatively imported occurrence.

## Syntax

ComponentOccurrenceProxy.**IsAssociativelyImported**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2016
