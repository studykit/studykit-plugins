# ComponentOccurrenceProxy.IsiPartMember Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that specifies if the ComponentOccurence is an iPartMember. The property is True if the ComponentOccurence is an iPartMember.

## Syntax

ComponentOccurrenceProxy.**IsiPartMember**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
