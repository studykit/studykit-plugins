# ComponentOccurrenceProxy.Adaptive Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

'Put' is Inventor Only: Gets/Sets the Boolean flag that specifies whether this Occurrence is adaptive or not.

## Syntax

ComponentOccurrenceProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 4
