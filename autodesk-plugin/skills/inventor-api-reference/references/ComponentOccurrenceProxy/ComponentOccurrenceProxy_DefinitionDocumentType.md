# ComponentOccurrenceProxy.DefinitionDocumentType Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets the type of the Document whose occurrence this is.

## Syntax

ComponentOccurrenceProxy.**DefinitionDocumentType**() As [DocumentTypeEnum](../DocumentTypeEnum.md)

## Property Value

This is a read only property whose value is a [DocumentTypeEnum](../DocumentTypeEnum.md).

## Version

Introduced in version 5.3
