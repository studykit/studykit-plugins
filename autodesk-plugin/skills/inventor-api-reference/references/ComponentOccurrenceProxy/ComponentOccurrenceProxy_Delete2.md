# ComponentOccurrenceProxy.Delete2 Method

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Deletes this component occurrence.

## Syntax

ComponentOccurrenceProxy.**Delete2**( [***SkipDocumentSave***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SkipDocumentSave | Boolean | Optional input Boolean value to specify if skip save the source document of the occurrence. This default to False if not specified. |

## Version

Introduced in version 2019
