# ComponentOccurrenceProxy.ActivePositionalRepresentation Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets and sets the active Positional Representation for this occurrence.

## Syntax

ComponentOccurrenceProxy.**ActivePositionalRepresentation**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
