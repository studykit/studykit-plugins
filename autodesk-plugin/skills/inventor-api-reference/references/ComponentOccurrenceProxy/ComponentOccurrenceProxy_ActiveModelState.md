# ComponentOccurrenceProxy.ActiveModelState Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Read-write property that gets and sets the name of the active model state for an assembly occurrence.

## Syntax

ComponentOccurrenceProxy.**ActiveModelState**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
