# ComponentOccurrenceProxy.ShowDegreesOfFreedom Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets and sets whether to display the degrees of freedom for the occurrence.

## Syntax

ComponentOccurrenceProxy.**ShowDegreesOfFreedom**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
