# ComponentOccurrenceProxy.HasBodyOverride Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that returns a Boolean specifying whether the occurrence path contains body-modifying assembly features.

## Syntax

ComponentOccurrenceProxy.**HasBodyOverride**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
