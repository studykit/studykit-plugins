# ComponentOccurrenceProxy.Suppressed Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that returns whether this occurrence is suppressed or not.

## Syntax

ComponentOccurrenceProxy.**Suppressed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
