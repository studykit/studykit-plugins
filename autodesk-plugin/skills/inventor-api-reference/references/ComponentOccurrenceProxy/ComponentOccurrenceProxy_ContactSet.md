# ComponentOccurrenceProxy.ContactSet Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets and sets whether the component belongs in the contact set.

## Syntax

ComponentOccurrenceProxy.**ContactSet**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
