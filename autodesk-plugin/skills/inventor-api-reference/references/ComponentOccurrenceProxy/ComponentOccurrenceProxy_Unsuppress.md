# ComponentOccurrenceProxy.Unsuppress Method

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Method that unsuppresses the occurrence.

## Syntax

ComponentOccurrenceProxy.**Unsuppress**()

## Version

Introduced in version 11
