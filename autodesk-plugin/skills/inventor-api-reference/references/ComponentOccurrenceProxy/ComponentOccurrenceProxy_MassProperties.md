# ComponentOccurrenceProxy.MassProperties Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that returns the MassProperties object. This supports performing mass properties calculations for this particular occurrence.

## Syntax

ComponentOccurrenceProxy.**MassProperties**() As [MassProperties](../MassProperties/MassProperties.md)

## Property Value

This is a read only property whose value is a [MassProperties](../MassProperties/MassProperties.md).

## Version

Introduced in version 5
