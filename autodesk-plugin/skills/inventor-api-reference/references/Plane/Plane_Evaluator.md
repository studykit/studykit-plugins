# Plane.Evaluator Property

Parent Object: [Plane](../Plane/Plane.md)

## Description

Gets the surface evaluator for this plane.

## Syntax

Plane.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
