# Plane.RootPoint Property

Parent Object: [Plane](../Plane/Plane.md)

## Description

Gets the root point for this plane.

## Syntax

Plane.**RootPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
