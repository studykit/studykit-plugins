# HoleTableColumn.Parent Property

Parent Object: [HoleTableColumn](../HoleTableColumn/HoleTableColumn.md)

## Description

Property that returns the parent HoleTable.

## Syntax

HoleTableColumn.**Parent**() As [HoleTable](../HoleTable/HoleTable.md)

## Property Value

This is a read only property whose value is a [HoleTable](../HoleTable/HoleTable.md).

## Version

Introduced in version 10
