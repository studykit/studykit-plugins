# HoleTableColumn Object

## Description

The HoleTableColumn object defines a column within the hole table, including the column width.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../HoleTableColumn/HoleTableColumn_Delete.md) | Method that deletes the column from the table. |
| [Reposition](../HoleTableColumn/HoleTableColumn_Reposition.md) | Method that repositions the column within the table. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HoleTableColumn/HoleTableColumn_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [CustomPropertyName](../HoleTableColumn/HoleTableColumn_CustomPropertyName.md) | Property that returns the name of the custom property associated with this column. |
| [Parent](../HoleTableColumn/HoleTableColumn_Parent.md) | Property that returns the parent HoleTable. |
| [PropertyType](../HoleTableColumn/HoleTableColumn_PropertyType.md) | Property that returns the property type associated with this column. |
| [Title](../HoleTableColumn/HoleTableColumn_Title.md) | Specifies the title of the HoleTable column. |
| [Type](../HoleTableColumn/HoleTableColumn_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [UnitsFormatting](../HoleTableColumn/HoleTableColumn_UnitsFormatting.md) | Gets the units formatting of the column. |
| [Width](../HoleTableColumn/HoleTableColumn_Width.md) | Gets/Sets the width of the HoleTable column. |

## Accessed From

[HoleTableColumns.Add](../HoleTableColumns/HoleTableColumns_Add.md), [HoleTableColumns.Item](../HoleTableColumns/HoleTableColumns_Item.md)

## Version

Introduced in version 10
