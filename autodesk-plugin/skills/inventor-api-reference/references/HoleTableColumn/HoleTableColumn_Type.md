# HoleTableColumn.Type Property

Parent Object: [HoleTableColumn](../HoleTableColumn/HoleTableColumn.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

HoleTableColumn.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
