# FeatureDimension Object

## Description

A FeatureDimension object represents a dimension associated with a feature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetReferenceKey](../FeatureDimension/FeatureDimension_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AnchorPoints](../FeatureDimension/FeatureDimension_AnchorPoints.md) | Gets the anchor points of dimension. |
| [Application](../FeatureDimension/FeatureDimension_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../FeatureDimension/FeatureDimension_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [DimensionType](../FeatureDimension/FeatureDimension_DimensionType.md) | Gets the type of dimension. |
| [Parameter](../FeatureDimension/FeatureDimension_Parameter.md) | Property that returns the parameter associated with the dimension. |
| [Parent](../FeatureDimension/FeatureDimension_Parent.md) | Property that returns the parent PartFeature object. |
| [TextPoint](../FeatureDimension/FeatureDimension_TextPoint.md) | Property that returns the position of the dimension text. |
| [Type](../FeatureDimension/FeatureDimension_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
