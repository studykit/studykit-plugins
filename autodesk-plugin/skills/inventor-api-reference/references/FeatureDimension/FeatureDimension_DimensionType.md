# FeatureDimension.DimensionType Property

Parent Object: [FeatureDimension](../FeatureDimension/FeatureDimension.md)

## Description

Gets the type of dimension.

## Syntax

FeatureDimension.**DimensionType**() As [FeatureDimensionTypeEnum](../FeatureDimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [FeatureDimensionTypeEnum](../FeatureDimensionTypeEnum.md).

## Version

Introduced in version 2012
