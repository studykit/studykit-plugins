# FeatureDimension.Parameter Property

Parent Object: [FeatureDimension](../FeatureDimension/FeatureDimension.md)

## Description

Property that returns the parameter associated with the dimension.

## Syntax

FeatureDimension.**Parameter**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).
