# MoveFaceDefinition.MoveFaceTypeDefinition Property

Parent Object: [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md)

## Description

Property that returns the definition object that defines the type of move.

## Syntax

MoveFaceDefinition.**MoveFaceTypeDefinition**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2011
