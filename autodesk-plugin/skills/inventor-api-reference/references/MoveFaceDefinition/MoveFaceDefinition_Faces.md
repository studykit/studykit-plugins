# MoveFaceDefinition.Faces Property

Parent Object: [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md)

## Description

Gets and sets the FaceCollection object that contains the faces that are used to define the Move Face feature.

## Syntax

MoveFaceDefinition.**Faces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2011
