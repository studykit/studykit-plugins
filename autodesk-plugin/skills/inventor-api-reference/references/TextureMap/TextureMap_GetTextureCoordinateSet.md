# TextureMap.GetTextureCoordinateSet Method

Parent Object: [TextureMap](../TextureMap/TextureMap.md)

## Description

Gets the TextureCoordinateSet for the input tolerance.

## Syntax

TextureMap.**GetTextureCoordinateSet**( ***Tolerance*** As Double ) As [TextureCoordinateSet](../TextureCoordinateSet/TextureCoordinateSet.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Tolerance | Double | Input tolerance value. |

## Version

Introduced in version 11
