# SketchSpline.Type Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchSpline.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
