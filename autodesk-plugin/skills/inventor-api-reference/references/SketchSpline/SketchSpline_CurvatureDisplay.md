# SketchSpline.CurvatureDisplay Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Specifies whether curvature information is displayed for the spline.

## Syntax

SketchSpline.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 8
