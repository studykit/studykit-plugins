# SketchSpline.EndSketchPoint Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Property that gets the that defines the position of the end of the spline.

## Syntax

SketchSpline.**EndSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 8
