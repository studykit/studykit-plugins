# SketchSpline.OverrideColor Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Gets and sets the color applied to this sketch spline.

## Syntax

SketchSpline.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
