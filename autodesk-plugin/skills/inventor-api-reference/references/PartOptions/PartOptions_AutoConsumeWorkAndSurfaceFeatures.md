# PartOptions.AutoConsumeWorkAndSurfaceFeatures Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Enables/disables automatically consuming work and surface features when they are used as inputs for creating other features.

## Syntax

PartOptions.**AutoConsumeWorkAndSurfaceFeatures**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
