# PartOptions.GeometricConstraintsBreakage Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Gets and Sets the preference for breaking geometric constraints during 3D grip edit.

## Syntax

PartOptions.**GeometricConstraintsBreakage**() As [GeometricConstraintsBreakageEnum](../GeometricConstraintsBreakageEnum.md)

## Property Value

This is a read/write property whose value is a [GeometricConstraintsBreakageEnum](../GeometricConstraintsBreakageEnum.md).

## Version

Introduced in version 11
