# PartOptions.DefaultDesignView Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Read-write property that gets and sets the default design view representation to use when opening part documents.

## Syntax

PartOptions.**DefaultDesignView**() As [DesignViewTypeEnum](../DesignViewTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DesignViewTypeEnum](../DesignViewTypeEnum.md).

## Version

Introduced in version 2012
