# PartOptions.DisplayGripsOnSelection Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Enables/disables display of glyphs for 3D grips when faces are selected.

## Syntax

PartOptions.**DisplayGripsOnSelection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
