# PartOptions.DimensionConstraintsRelaxation Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Gets and Sets the preference for relaxing dimension constraints during 3D grip edit.

## Syntax

PartOptions.**DimensionConstraintsRelaxation**() As [DimensionConstraintsRelaxationEnum](../DimensionConstraintsRelaxationEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionConstraintsRelaxationEnum](../DimensionConstraintsRelaxationEnum.md).

## Version

Introduced in version 11
