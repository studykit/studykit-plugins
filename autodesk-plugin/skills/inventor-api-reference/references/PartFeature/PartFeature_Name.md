# PartFeature.Name Property

Parent Object: [PartFeature](../PartFeature/PartFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

PartFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.
