# PartFeature.HealthStatus Property

Parent Object: [PartFeature](../PartFeature/PartFeature.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

PartFeature.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).
