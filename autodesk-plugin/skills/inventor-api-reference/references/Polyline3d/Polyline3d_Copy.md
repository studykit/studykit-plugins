# Polyline3d.Copy Method

Parent Object: [Polyline3d](../Polyline3d/Polyline3d.md)

## Description

Creates a copy of this Polyline3d object. The result is entirely independent and can be edited without affecting the original Polyline3d object.

## Syntax

Polyline3d.**Copy**() As [Polyline3d](../Polyline3d/Polyline3d.md)

## Version

Introduced in version 2013
