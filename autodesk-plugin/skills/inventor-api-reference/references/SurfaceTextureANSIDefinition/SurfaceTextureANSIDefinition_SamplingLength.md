# SurfaceTextureANSIDefinition.SamplingLength Property

Parent Object: [SurfaceTextureANSIDefinition](../SurfaceTextureANSIDefinition/SurfaceTextureANSIDefinition.md)

## Description

Gets and sets the sampling length for roughness value.

## Syntax

SurfaceTextureANSIDefinition.**SamplingLength**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
