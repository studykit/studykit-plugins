# SurfaceTextureANSIDefinition.ProcessNote Property

Parent Object: [SurfaceTextureANSIDefinition](../SurfaceTextureANSIDefinition/SurfaceTextureANSIDefinition.md)

## Description

Gets and sets the process requirements for the surface.

## Syntax

SurfaceTextureANSIDefinition.**ProcessNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
