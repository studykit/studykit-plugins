# SurfaceTextureANSIDefinition.AdditionalMinRoughness Property

Parent Object: [SurfaceTextureANSIDefinition](../SurfaceTextureANSIDefinition/SurfaceTextureANSIDefinition.md)

## Description

Gets and sets the other minimum roughness value.

## Syntax

SurfaceTextureANSIDefinition.**AdditionalMinRoughness**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
