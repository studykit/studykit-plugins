# CollinearConstraint.Type Property

Parent Object: [CollinearConstraint](../CollinearConstraint/CollinearConstraint.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CollinearConstraint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
