# iMateResultProxy.Delete Method

Parent Object: [iMateResultProxy](../iMateResultProxy/iMateResultProxy.md)

## Description

Method that deletes the iMateResult. This method will fail if the iMateResult is a part of a composite iMateResult.

## Syntax

iMateResultProxy.**Delete**()

## Version

Introduced in version 11
