# iMateResultProxy.Type Property

Parent Object: [iMateResultProxy](../iMateResultProxy/iMateResultProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iMateResultProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
