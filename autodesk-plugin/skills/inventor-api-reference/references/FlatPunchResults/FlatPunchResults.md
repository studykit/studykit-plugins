# FlatPunchResults Object

## Description

The FlatPunchResults collection object contains information about all the resulting punches in the flat pattern.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FlatPunchResults/FlatPunchResults_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FlatPunchResults/FlatPunchResults_Count.md) | Property that returns the number of items in the collection. |
| [Item](../FlatPunchResults/FlatPunchResults_Item.md) | Returns the specified FlatPunchResult object from the collection |
| [Type](../FlatPunchResults/FlatPunchResults_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FlatPattern.FlatPunchResults](../FlatPattern/FlatPattern_FlatPunchResults.md)

## Version

Introduced in version 2008
