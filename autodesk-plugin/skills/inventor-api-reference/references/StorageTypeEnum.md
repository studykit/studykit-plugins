# StorageTypeEnum Enumerator

## Description

Type to indicate the kind of storage medium being referred to.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kFileOrStreamStorage | 6148 | Storage medium could be either a file or a Structure Storage Stream. |
| kFileStorage | 6146 | Storage medium is a file on a persistent storage device. |
| kStorageStorage | 6149 | Storage medium is a Structured Storage Storage (accessed by IStorage\*). |
| kStreamStorage | 6147 | Storage medium is a Structured Storage Stream (accessed by IStream\*). |
| kUnknownStorage | 6145 | Storage medium is unknown. |

## Version

Introduced in version 4
