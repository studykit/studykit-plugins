# GeneralDimension.PromoteToSketch Method

Parent Object: [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Description

Method that copies the dimension to the underlying sketch. This method only works for dimensions associated with a draft view.

## Syntax

GeneralDimension.**PromoteToSketch**() As [DimensionConstraint](../DimensionConstraint/DimensionConstraint.md)

## Version

Introduced in version 9
