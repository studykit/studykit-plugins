# LeaderStylesEnumerator Object

## Description

The LeaderStylesEnumerator object provides access to the existing LeaderStyle objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LeaderStylesEnumerator/LeaderStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LeaderStylesEnumerator/LeaderStylesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../LeaderStylesEnumerator/LeaderStylesEnumerator_Item.md) | Returns the specified LeaderStyle object from the collection. |
| [Type](../LeaderStylesEnumerator/LeaderStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.LeaderStyles](../DrawingStylesManager/DrawingStylesManager_LeaderStyles.md)

## Version

Introduced in version 2009
