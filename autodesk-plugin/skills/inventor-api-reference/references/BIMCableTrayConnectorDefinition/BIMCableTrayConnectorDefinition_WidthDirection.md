# BIMCableTrayConnectorDefinition.WidthDirection Property

Parent Object: [BIMCableTrayConnectorDefinition](../BIMCableTrayConnectorDefinition/BIMCableTrayConnectorDefinition.md)

## Description

Read-only property that returns the direction of the connector width. This returns Nothing in the case where the width has not yet been defined.

## Syntax

BIMCableTrayConnectorDefinition.**WidthDirection**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2014
