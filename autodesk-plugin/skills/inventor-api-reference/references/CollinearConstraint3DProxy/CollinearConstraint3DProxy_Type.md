# CollinearConstraint3DProxy.Type Property

Parent Object: [CollinearConstraint3DProxy](../CollinearConstraint3DProxy/CollinearConstraint3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CollinearConstraint3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
