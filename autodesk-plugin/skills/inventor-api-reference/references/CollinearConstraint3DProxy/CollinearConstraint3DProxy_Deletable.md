# CollinearConstraint3DProxy.Deletable Property

Parent Object: [CollinearConstraint3DProxy](../CollinearConstraint3DProxy/CollinearConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

CollinearConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
