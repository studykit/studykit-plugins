# SketchCircularPattern.Parent Property

Parent Object: [SketchCircularPattern](../SketchCircularPattern/SketchCircularPattern.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SketchCircularPattern.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 2025.1
