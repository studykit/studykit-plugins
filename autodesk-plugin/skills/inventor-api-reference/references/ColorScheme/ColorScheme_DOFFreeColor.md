# ColorScheme.DOFFreeColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Property that returns the DOF free color.

## Syntax

ColorScheme.**DOFFreeColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
