# ColorScheme.GripHighlightingColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Gets the color of the active manipulator glyph.

## Syntax

ColorScheme.**GripHighlightingColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2021
