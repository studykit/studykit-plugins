# ColorScheme.Activate Method

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Method that causes this color scheme to become the active color scheme.

## Syntax

ColorScheme.**Activate**()

## Version

Introduced in version 9
