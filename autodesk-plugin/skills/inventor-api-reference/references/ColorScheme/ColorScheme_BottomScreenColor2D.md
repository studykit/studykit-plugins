# ColorScheme.BottomScreenColor2D Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Property that returns the 2D bottom screen color used when the BackgroundType is set to kGradientBackgroundType.

## Syntax

ColorScheme.**BottomScreenColor2D**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
