# ColorScheme.DOFFixedColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

property that returns the DOF fixed color.

## Syntax

ColorScheme.**DOFFixedColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
