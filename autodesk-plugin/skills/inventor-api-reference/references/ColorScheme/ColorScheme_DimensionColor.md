# ColorScheme.DimensionColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Gets the 2D&3D sketch dimension color.

## Syntax

ColorScheme.**DimensionColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2018
