# ColorScheme.Name Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

String property that returns the name of the color scheme.

## Syntax

ColorScheme.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 9
