# ColorScheme.ImageFullFileName Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Gets and sets the full file name of the image file to be used as the background.

## Syntax

ColorScheme.**ImageFullFileName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 10
