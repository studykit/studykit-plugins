# BOM.PartsOnlyViewNumberingScheme Property

Parent Object: [BOM](../BOM/BOM.md)

## Description

gets and sets the numbering scheme for a 'Parts Only' view.

## Syntax

BOM.**PartsOnlyViewNumberingScheme**() As [NumberingSchemeEnum](../NumberingSchemeEnum.md)

## Property Value

This is a read/write property whose value is a [NumberingSchemeEnum](../NumberingSchemeEnum.md).

## Version

Introduced in version 11
