# BOM.BOMViews Property

Parent Object: [BOM](../BOM/BOM.md)

## Description

Property that gets the BOMViews collection object.

## Syntax

BOM.**BOMViews**() As [BOMViews](../BOMViews/BOMViews.md)

## Property Value

This is a read only property whose value is a [BOMViews](../BOMViews/BOMViews.md).
