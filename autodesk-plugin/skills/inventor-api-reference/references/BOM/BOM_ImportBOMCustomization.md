# BOM.ImportBOMCustomization Method

Parent Object: [BOM](../BOM/BOM.md)

## Description

Import BOM column customization as XML file.

## Syntax

BOM.**ImportBOMCustomization**( ***FileName*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FileName | String | Input String that defines the name of an existing xml file that contains BOM customization information. |

## Version

Introduced in version 2015
