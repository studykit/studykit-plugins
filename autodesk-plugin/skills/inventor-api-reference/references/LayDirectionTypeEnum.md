# LayDirectionTypeEnum Enumerator

## Description

Bit field describing the various direction.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAngularInBothDirections | 4 | Angular in both directions. |
| kCircularRelativeToCenter | 16 | Circular relative to center. |
| kMultidirectional | 8 | Multiple directional. |
| kParallelToPlaneOfProjection | 1 | Parallel to plane projection. |
| kParticulateNondirectional | 64 | Particulate nondirectional. |
| kPerpendicularToPlaneOfProjection | 2 | Perpendicular to plane projection. |
| kRadialRelativeToCenter | 32 | Radial relative to center. |

## Version

Introduced in version 2009
