# RevolveFeatures.Type Property

Parent Object: [RevolveFeatures](../RevolveFeatures/RevolveFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RevolveFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
