# RevolveFeatures.Item Property

Parent Object: [RevolveFeatures](../RevolveFeatures/RevolveFeatures.md)

## Description

Returns the specified RevolveFeature object from the collection.

## Syntax

RevolveFeatures.**Item**( ***Index*** As Variant ) As [RevolveFeature](../RevolveFeature/RevolveFeature.md)

## Property Value

This is a read only property whose value is a [RevolveFeature](../RevolveFeature/RevolveFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the feature to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the feature name. If an out of range index or a name of a non-existent feature is provided, an error occurs. |

## Version

Introduced in version 5
