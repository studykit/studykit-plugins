# iFeatureInput.EntityType Property

Parent Object: [iFeatureInput](../iFeatureInput/iFeatureInput.md)

## Description

Property that returns the type of geometry that are valid. The value returned is the sum of values specifying the valid entity types.

## Syntax

iFeatureInput.**EntityType**() As [iFeatureEntityInputTypeEnum](../iFeatureEntityInputTypeEnum.md)

## Property Value

This is a read only property whose value is an [iFeatureEntityInputTypeEnum](../iFeatureEntityInputTypeEnum.md).

## Version

Introduced in version 6
