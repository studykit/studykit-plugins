# SplitFeatureProxy.SplitType Property

Parent Object: [SplitFeatureProxy](../SplitFeatureProxy/SplitFeatureProxy.md)

## Description

Property that returns the split feature type. Possible values are kSplitFaces and kSplitPart.

## Syntax

SplitFeatureProxy.**SplitType**() As [SplitTypeEnum](../SplitTypeEnum.md)

## Property Value

This is a read only property whose value is a [SplitTypeEnum](../SplitTypeEnum.md).

## Version

Introduced in version 9
