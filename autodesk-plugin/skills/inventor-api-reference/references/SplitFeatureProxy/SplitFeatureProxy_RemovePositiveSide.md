# SplitFeatureProxy.RemovePositiveSide Property

Parent Object: [SplitFeatureProxy](../SplitFeatureProxy/SplitFeatureProxy.md)

## Description

Gets and sets which side of the split body is removed for the trim body operation. This property fails in the case where the split feature defines a face split or a split body operation.

## Syntax

SplitFeatureProxy.**RemovePositiveSide**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
