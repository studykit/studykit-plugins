# SplitFeatureProxy.NativeObject Property

Parent Object: [SplitFeatureProxy](../SplitFeatureProxy/SplitFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SplitFeatureProxy.**NativeObject**() As [SplitFeature](../SplitFeature/SplitFeature.md)

## Property Value

This is a read only property whose value is a [SplitFeature](../SplitFeature/SplitFeature.md).

## Version

Introduced in version 9
