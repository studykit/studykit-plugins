# SplitFeatureProxy.AttributeSets Property

Parent Object: [SplitFeatureProxy](../SplitFeatureProxy/SplitFeatureProxy.md)

## Description

Property that returns the AttributeSets collection object associated with this object.

## Syntax

SplitFeatureProxy.**AttributeSets**() As [AttributeSets](../AttributeSets/AttributeSets.md)

## Property Value

This is a read only property whose value is an [AttributeSets](../AttributeSets/AttributeSets.md).

## Version

Introduced in version 9
