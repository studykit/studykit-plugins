# CommandCategory.Remove Method

Parent Object: [CommandCategory](../CommandCategory/CommandCategory.md)

## Description

Method that removes a control definition or a command bar from the command category.

## Syntax

CommandCategory.**Remove**( ***Index*** As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index within the collection of the item to return. |

## Version

Introduced in version 9
