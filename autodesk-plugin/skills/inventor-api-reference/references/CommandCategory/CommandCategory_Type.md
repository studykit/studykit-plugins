# CommandCategory.Type Property

Parent Object: [CommandCategory](../CommandCategory/CommandCategory.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CommandCategory.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
