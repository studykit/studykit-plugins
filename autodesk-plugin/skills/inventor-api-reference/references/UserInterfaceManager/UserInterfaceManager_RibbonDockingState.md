# UserInterfaceManager.RibbonDockingState Property

Parent Object: [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Description

Gets and sets the docking state of the ribbon.

## Syntax

UserInterfaceManager.**RibbonDockingState**() As [RibbonDockingStateEnum](../RibbonDockingStateEnum.md)

## Property Value

This is a read/write property whose value is a [RibbonDockingStateEnum](../RibbonDockingStateEnum.md).

## Version

Introduced in version 2010
