# UserInterfaceManager.BalloonTips Property

Parent Object: [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Description

Read-only property that returns the BalloonTips collection object.

## Syntax

UserInterfaceManager.**BalloonTips**() As [BalloonTips](../BalloonTips/BalloonTips.md)

## Property Value

This is a read only property whose value is a [BalloonTips](../BalloonTips/BalloonTips.md).

## Version

Introduced in version 2012
