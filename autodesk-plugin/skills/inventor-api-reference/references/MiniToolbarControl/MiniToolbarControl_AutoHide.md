# MiniToolbarControl.AutoHide Property

Parent Object: [MiniToolbarControl](../MiniToolbarControl/MiniToolbarControl.md)

## Description

Read-write property that gets and sets whether the control should be automatically hidden when the user moves the cursor away from the MiniToolbar.

## Syntax

MiniToolbarControl.**AutoHide**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
