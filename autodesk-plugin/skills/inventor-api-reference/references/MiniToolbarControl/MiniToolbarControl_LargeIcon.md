# MiniToolbarControl.LargeIcon Property

Parent Object: [MiniToolbarControl](../MiniToolbarControl/MiniToolbarControl.md)

## Description

Read-write property that gets and sets the large icon for the control.

## Syntax

MiniToolbarControl.**LargeIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2012
