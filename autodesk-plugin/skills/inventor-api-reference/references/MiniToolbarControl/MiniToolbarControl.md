# MiniToolbarControl Object

## Description

The MiniToolbarControl object is the base class for all controls within a MiniToolbar.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../MiniToolbarControl/MiniToolbarControl_Delete.md) | Method that deletes the control. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MiniToolbarControl/MiniToolbarControl_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AutoHide](../MiniToolbarControl/MiniToolbarControl_AutoHide.md) | Read-write property that gets and sets whether the control should be automatically hidden when the user moves the cursor away from the MiniToolbar. |
| [ControlType](../MiniToolbarControl/MiniToolbarControl_ControlType.md) | Read-only property that returns the control type. |
| [DisplayName](../MiniToolbarControl/MiniToolbarControl_DisplayName.md) | Read-write property that gets and sets the display name for the control. |
| [Enabled](../MiniToolbarControl/MiniToolbarControl_Enabled.md) | Read-write property that gets and sets whether the control is enabled. |
| [Index](../MiniToolbarControl/MiniToolbarControl_Index.md) | Read-only property that returns the index this control is currently positioned at within the mini toolbar. |
| [InternalName](../MiniToolbarControl/MiniToolbarControl_InternalName.md) | Read-only property that returns a string uniquely identifying this control within the toolbar. |
| [LargeIcon](../MiniToolbarControl/MiniToolbarControl_LargeIcon.md) | Read-write property that gets and sets the large icon for the control. |
| [Parent](../MiniToolbarControl/MiniToolbarControl_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [StandardIcon](../MiniToolbarControl/MiniToolbarControl_StandardIcon.md) | Read-write property that gets and sets the standard (small) icon for the control. |
| [ToolTipText](../MiniToolbarControl/MiniToolbarControl_ToolTipText.md) | Read-write property that gets and sets the tooltip text for the control. |
| [Type](../MiniToolbarControl/MiniToolbarControl_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Visible](../MiniToolbarControl/MiniToolbarControl_Visible.md) | Read-write property that gets and sets whether the control is visible. |
