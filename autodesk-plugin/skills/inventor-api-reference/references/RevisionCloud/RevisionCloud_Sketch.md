# RevisionCloud.Sketch Property

Parent Object: [RevisionCloud](../RevisionCloud/RevisionCloud.md)

## Description

Returns the associative DrawingSketch object.

## Syntax

RevisionCloud.**Sketch**() As [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Property Value

This is a read only property whose value is a [DrawingSketch](../DrawingSketch/DrawingSketch.md).

## Version

Introduced in version 2024
