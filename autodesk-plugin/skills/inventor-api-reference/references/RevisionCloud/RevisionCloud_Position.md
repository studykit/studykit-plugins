# RevisionCloud.Position Property

Parent Object: [RevisionCloud](../RevisionCloud/RevisionCloud.md)

## Description

Gets and sets the position of the revision cloud on the sheet.

## Syntax

RevisionCloud.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2024
