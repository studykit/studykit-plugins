# ComboBoxDefinition.OnSelect Event

Parent Object: [ComboBoxDefinition](../ComboBoxDefinition/ComboBoxDefinition.md)

## Description

Event that is fired when an end user changes the selection in the combo box or when the selection is changed using the ListIndex property. This event is not fired for built-in definitions.

## Syntax

ComboBoxDefinition.**OnSelect**( ***Context*** As [NameValueMap](../NameValueMap/NameValueMap.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Context | [NameValueMap](../NameValueMap/NameValueMap.md) | Input object that can be used to determine the context of why the event fired. |

## Version

Introduced in version 9
