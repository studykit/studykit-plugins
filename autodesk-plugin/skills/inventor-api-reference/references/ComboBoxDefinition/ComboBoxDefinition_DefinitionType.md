# ComboBoxDefinition.DefinitionType Property

Parent Object: [ComboBoxDefinition](../ComboBoxDefinition/ComboBoxDefinition.md)

## Description

Gets the ControlDefinitionType.

## Syntax

ComboBoxDefinition.**DefinitionType**() As [ControlDefinitionTypeEnum](../ControlDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [ControlDefinitionTypeEnum](../ControlDefinitionTypeEnum.md).

## Version

Introduced in version 9
