# Dictionary.Name Property

Parent Object: [Dictionary](../Dictionary/Dictionary.md)

## Description

Read-only property that returns the name of the Dictionary.

## Syntax

Dictionary.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2020
