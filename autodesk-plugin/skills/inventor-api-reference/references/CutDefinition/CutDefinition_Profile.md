# CutDefinition.Profile Property

Parent Object: [CutDefinition](../CutDefinition/CutDefinition.md)

## Description

Gets the Profile object that defines the shape of the cut.

## Syntax

CutDefinition.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 2009
