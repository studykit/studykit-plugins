# CutDefinition.Parent Property

Parent Object: [CutDefinition](../CutDefinition/CutDefinition.md)

## Description

Property that returns the parent CutFeature object of this CutDefinition object.

## Syntax

CutDefinition.**Parent**() As [CutFeature](../CutFeature/CutFeature.md)

## Property Value

This is a read only property whose value is a [CutFeature](../CutFeature/CutFeature.md).

## Version

Introduced in version 2009
