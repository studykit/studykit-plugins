# RuleFilletFeatureProxy.NativeObject Property

Parent Object: [RuleFilletFeatureProxy](../RuleFilletFeatureProxy/RuleFilletFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RuleFilletFeatureProxy.**NativeObject**() As [RuleFilletFeature](../RuleFilletFeature/RuleFilletFeature.md)

## Property Value

This is a read only property whose value is a [RuleFilletFeature](../RuleFilletFeature/RuleFilletFeature.md).

## Version

Introduced in version 2010
