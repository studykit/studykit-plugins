# RuleFilletFeatureProxy.Adaptive Property

Parent Object: [RuleFilletFeatureProxy](../RuleFilletFeatureProxy/RuleFilletFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

RuleFilletFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
