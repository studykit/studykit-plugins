# ImportedDWGComponent.Crop Property

Parent Object: [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Description

Read-write property that gets and sets the crop in LocationPlane space. Set this to Nothing will clear the crop.

## Syntax

ImportedDWGComponent.**Crop**() As [Box2d](../Box2d/Box2d.md)

## Property Value

This is a read/write property whose value is a [Box2d](../Box2d/Box2d.md).

## Version

Introduced in version 2017
