# FeatureDimensions Object

## Description

The FeatureDimensions collection object contains all the dimensions that are associated with a feature. These do not include the dimensions within sketches used by the feature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Show](../FeatureDimensions/FeatureDimensions_Show.md) | Method that displays all the dimensions for the feature. This is the equivalent of the 'Show Dimensions' command in the user interface. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FeatureDimensions/FeatureDimensions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FeatureDimensions/FeatureDimensions_Count.md) | Property that returns the number of items in the collection. |
| [Item](../FeatureDimensions/FeatureDimensions_Item.md) | Returns the specified FeatureDimension object from the collection. |
| [Type](../FeatureDimensions/FeatureDimensions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
