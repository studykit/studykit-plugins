# SketchArcProxy.CurvatureDisplay Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Specifies whether curvature information is displayed for the arc.

## Syntax

SketchArcProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
