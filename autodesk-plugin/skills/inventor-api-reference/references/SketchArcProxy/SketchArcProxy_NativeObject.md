# SketchArcProxy.NativeObject Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchArcProxy.**NativeObject**() As [SketchArc](../SketchArc/SketchArc.md)

## Property Value

This is a read only property whose value is a [SketchArc](../SketchArc/SketchArc.md).

## Version

Introduced in version 6
