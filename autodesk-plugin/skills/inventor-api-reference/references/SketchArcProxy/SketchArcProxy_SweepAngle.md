# SketchArcProxy.SweepAngle Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Double property that returns the sweep angle of the arc. This is measured counterclockwise from the start angle of the arc in radians.

## Syntax

SketchArcProxy.**SweepAngle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
