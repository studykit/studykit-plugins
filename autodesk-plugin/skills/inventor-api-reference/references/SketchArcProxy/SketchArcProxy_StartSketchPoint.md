# SketchArcProxy.StartSketchPoint Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Property that gets the that defines the position of the start of the arc.

## Syntax

SketchArcProxy.**StartSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 6
