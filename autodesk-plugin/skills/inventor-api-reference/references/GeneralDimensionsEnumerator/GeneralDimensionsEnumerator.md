# GeneralDimensionsEnumerator Object

## Description

The GeneralDimensionsEnumerator object provides access to the existing objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../GeneralDimensionsEnumerator/GeneralDimensionsEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../GeneralDimensionsEnumerator/GeneralDimensionsEnumerator_Item.md) | Returns the specified DrawingStandardStyle object from the collection. |
| [Type](../GeneralDimensionsEnumerator/GeneralDimensionsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BaselineDimensionSet.Members](../BaselineDimensionSet/BaselineDimensionSet_Members.md), [ChainDimensionSet.AddMembers](../ChainDimensionSet/ChainDimensionSet_AddMembers.md), [ChainDimensionSet.Members](../ChainDimensionSet/ChainDimensionSet_Members.md)

## Version

Introduced in version 9
