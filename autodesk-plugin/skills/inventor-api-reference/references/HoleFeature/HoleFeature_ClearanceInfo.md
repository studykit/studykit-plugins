# HoleFeature.ClearanceInfo Property

Parent Object: [HoleFeature](../HoleFeature/HoleFeature.md)

## Description

Gets and sets the HoleClearanceInfo object associated with the hole.

## Syntax

HoleFeature.**ClearanceInfo**() As [HoleClearanceInfo](../HoleClearanceInfo/HoleClearanceInfo.md)

## Property Value

This is a read/write property whose value is a [HoleClearanceInfo](../HoleClearanceInfo/HoleClearanceInfo.md).

## Version

Introduced in version 2018
