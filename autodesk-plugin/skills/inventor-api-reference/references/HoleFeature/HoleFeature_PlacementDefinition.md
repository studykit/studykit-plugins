# HoleFeature.PlacementDefinition Property

Parent Object: [HoleFeature](../HoleFeature/HoleFeature.md)

## Description

Gets the Placement Definition for the hole feature.

## Syntax

HoleFeature.**PlacementDefinition**() As [HolePlacementDefinition](../HolePlacementDefinition/HolePlacementDefinition.md)

## Property Value

This is a read/write property whose value is a [HolePlacementDefinition](../HolePlacementDefinition/HolePlacementDefinition.md).

## Version

Introduced in version 10
