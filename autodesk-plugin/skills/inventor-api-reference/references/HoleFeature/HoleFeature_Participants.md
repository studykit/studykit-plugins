# HoleFeature.Participants Property

Parent Object: [HoleFeature](../HoleFeature/HoleFeature.md)

## Description

Property that returns the list of participants for an assembly feature. This list is empty for features in a part.

## Syntax

HoleFeature.**Participants**() As [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md).

## Version

Introduced in version 10
