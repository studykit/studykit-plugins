# FlatBendResults Object

## Description

The FlatBendResults collection object contains information about all the resulting bends in the flat pattern.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FlatBendResults/FlatBendResults_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FlatBendResults/FlatBendResults_Count.md) | Property that returns the number of items in the collection. |
| [Item](../FlatBendResults/FlatBendResults_Item.md) | Returns the specified FlatBendResultobject from the collection. |
| [Type](../FlatBendResults/FlatBendResults_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FlatPattern.FlatBendResults](../FlatPattern/FlatPattern_FlatBendResults.md)

## Version

Introduced in version 2008
