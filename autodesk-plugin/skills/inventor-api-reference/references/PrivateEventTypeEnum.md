# PrivateEventTypeEnum Enumerator

## Description

Semantic description of data on Inventor's Private Event Queue.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBooleanEvent | 6662 | Boolean event corresponding to a Boolean. |
| kCommandNameEvent | 6661 | String event corresponding to a command name. |
| kDimensionEvent | 6659 | Float event corresponding to a dimensional value. |
| kFileNameEvent | 6657 | String event corresponding to a filename. |
| kNameEvent | 6658 | String event corresponding to a name. |
| kStringEvent | 6660 | String event corresponding to a generic string. |

## Version

Introduced in version 4
