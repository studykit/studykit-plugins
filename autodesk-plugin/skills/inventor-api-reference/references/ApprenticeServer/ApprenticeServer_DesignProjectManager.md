# ApprenticeServer.DesignProjectManager Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Property that returns the DesignProjectManager object.

## Syntax

ApprenticeServer.**DesignProjectManager**() As [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md)

## Property Value

This is a read only property whose value is a [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md).

## Version

Introduced in version 2011
