# ApprenticeServer.Close Method

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Closes the current file. After this call there is no top-level Document. An attempt is also made to close the children Documents.

## Syntax

ApprenticeServer.**Close**()

## Version

Introduced in version 4
