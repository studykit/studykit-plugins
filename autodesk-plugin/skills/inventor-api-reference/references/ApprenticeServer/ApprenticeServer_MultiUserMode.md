# ApprenticeServer.MultiUserMode Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Get or set the enum indicating the Multi-User mode.

## Syntax

ApprenticeServer.**MultiUserMode**() As [MultiUserModeEnum](../MultiUserModeEnum.md)

## Property Value

This is a read only property whose value is a [MultiUserModeEnum](../MultiUserModeEnum.md).

## Version

Introduced in version 6
