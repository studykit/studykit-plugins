# ApprenticeServer.DisplayOptions Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Get the DisplayOptions object.

## Syntax

ApprenticeServer.**DisplayOptions**() As [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Property Value

This is a read only property whose value is a [DisplayOptions](../DisplayOptions/DisplayOptions.md).

## Version

Introduced in version 11
