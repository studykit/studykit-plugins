# MidpointConstraint.Arc Property

Parent Object: [MidpointConstraint](../MidpointConstraint/MidpointConstraint.md)

## Description

Gets the sketch arc being constrained.

## Syntax

MidpointConstraint.**Arc**() As [SketchArc](../SketchArc/SketchArc.md)

## Property Value

This is a read only property whose value is a [SketchArc](../SketchArc/SketchArc.md).

## Version

Introduced in version 2015
