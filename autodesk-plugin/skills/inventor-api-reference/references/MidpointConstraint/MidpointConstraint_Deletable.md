# MidpointConstraint.Deletable Property

Parent Object: [MidpointConstraint](../MidpointConstraint/MidpointConstraint.md)

## Description

Indicates whether this object is deletable.

## Syntax

MidpointConstraint.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
