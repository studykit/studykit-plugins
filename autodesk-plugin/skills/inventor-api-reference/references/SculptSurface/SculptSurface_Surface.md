# SculptSurface.Surface Property

Parent Object: [SculptSurface](../SculptSurface/SculptSurface.md)

## Description

Gets and sets a boundary surface for the sculpt feature.

## Syntax

SculptSurface.**Surface**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 11
