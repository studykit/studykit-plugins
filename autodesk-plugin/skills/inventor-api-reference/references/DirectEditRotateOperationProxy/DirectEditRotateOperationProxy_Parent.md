# DirectEditRotateOperationProxy.Parent Property

Parent Object: [DirectEditRotateOperationProxy](../DirectEditRotateOperationProxy/DirectEditRotateOperationProxy.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DirectEditRotateOperationProxy.**Parent**() As [DirectEditFeature](../DirectEditFeature/DirectEditFeature.md)

## Property Value

This is a read only property whose value is a [DirectEditFeature](../DirectEditFeature/DirectEditFeature.md).

## Version

Introduced in version 2015
