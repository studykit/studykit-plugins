# ParallelToZAxisConstraint3DProxy.Line Property

Parent Object: [ParallelToZAxisConstraint3DProxy](../ParallelToZAxisConstraint3DProxy/ParallelToZAxisConstraint3DProxy.md)

## Description

Read-only property that returns the SketchLine3D being constrained.

## Syntax

ParallelToZAxisConstraint3DProxy.**Line**() As [SketchLine3D](../SketchLine3D/SketchLine3D.md)

## Property Value

This is a read only property whose value is a [SketchLine3D](../SketchLine3D/SketchLine3D.md).

## Version

Introduced in version 2017
