# HelicalConstraint3D.Type Property

Parent Object: [HelicalConstraint3D](../HelicalConstraint3D/HelicalConstraint3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

HelicalConstraint3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
