# TransitionSymbols.Item Property

Parent Object: [TransitionSymbols](../TransitionSymbols/TransitionSymbols.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

TransitionSymbols.**Item**( ***Index*** As Long ) As [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md)

## Property Value

This is a read only property whose value is a [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the TransitionSymbol to return. If an out of range index of a non-existent TransitionSymbol is provided, an error will occur . |

## Version

Introduced in version 2025.1
