# SplitFeature.SplitType Property

Parent Object: [SplitFeature](../SplitFeature/SplitFeature.md)

## Description

Property that returns the split feature type. Possible values are kSplitFaces and kSplitPart.

## Syntax

SplitFeature.**SplitType**() As [SplitTypeEnum](../SplitTypeEnum.md)

## Property Value

This is a read only property whose value is a [SplitTypeEnum](../SplitTypeEnum.md).

## Version

Introduced in version 6
