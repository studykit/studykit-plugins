# SplitFeature.SplitToolType Property

Parent Object: [SplitFeature](../SplitFeature/SplitFeature.md)

## Description

Property that returns the split tool type used to create the split feature.

## Syntax

SplitFeature.**SplitToolType**() As [SplitToolTypeEnum](../SplitToolTypeEnum.md)

## Property Value

This is a read only property whose value is a [SplitToolTypeEnum](../SplitToolTypeEnum.md).

## Version

Introduced in version 6
