# MacroControlDefinition.OverrideShortcutType Property

Parent Object: [MacroControlDefinition](../MacroControlDefinition/MacroControlDefinition.md)

## Description

Property that returns the type of override shortcut assigned to this command.

## Syntax

MacroControlDefinition.**OverrideShortcutType**() As [ShortcutTypeEnum](../ShortcutTypeEnum.md)

## Property Value

This is a read only property whose value is a [ShortcutTypeEnum](../ShortcutTypeEnum.md).

## Version

Introduced in version 2008
