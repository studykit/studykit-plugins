# MacroControlDefinition.Delete Method

Parent Object: [MacroControlDefinition](../MacroControlDefinition/MacroControlDefinition.md)

## Description

Method that deletes the control definition. This method fails for built-in definitions.

## Syntax

MacroControlDefinition.**Delete**()

## Version

Introduced in version 9
