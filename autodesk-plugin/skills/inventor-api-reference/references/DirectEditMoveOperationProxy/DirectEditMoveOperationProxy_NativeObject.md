# DirectEditMoveOperationProxy.NativeObject Property

Parent Object: [DirectEditMoveOperationProxy](../DirectEditMoveOperationProxy/DirectEditMoveOperationProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

DirectEditMoveOperationProxy.**NativeObject**() As [DirectEditMoveOperation](../DirectEditMoveOperation/DirectEditMoveOperation.md)

## Property Value

This is a read only property whose value is a [DirectEditMoveOperation](../DirectEditMoveOperation/DirectEditMoveOperation.md).

## Version

Introduced in version 2015
