# LeaderStyle.ArrowheadType Property

Parent Object: [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Description

Property that gets and sets the arrowhead style to use.

## Syntax

LeaderStyle.**ArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2009
