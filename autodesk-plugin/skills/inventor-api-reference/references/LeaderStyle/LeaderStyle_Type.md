# LeaderStyle.Type Property

Parent Object: [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

LeaderStyle.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
