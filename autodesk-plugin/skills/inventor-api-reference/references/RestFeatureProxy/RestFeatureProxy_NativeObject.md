# RestFeatureProxy.NativeObject Property

Parent Object: [RestFeatureProxy](../RestFeatureProxy/RestFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RestFeatureProxy.**NativeObject**() As [RestFeature](../RestFeature/RestFeature.md)

## Property Value

This is a read only property whose value is a [RestFeature](../RestFeature/RestFeature.md).

## Version

Introduced in version 2010
