# OnFaceConstraint3D.Type Property

Parent Object: [OnFaceConstraint3D](../OnFaceConstraint3D/OnFaceConstraint3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

OnFaceConstraint3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2017
