# ReferencedOpaqueFileDescriptor.Parent Property

Parent Object: [ReferencedOpaqueFileDescriptor](../ReferencedOpaqueFileDescriptor/ReferencedOpaqueFileDescriptor.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ReferencedOpaqueFileDescriptor.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2017
