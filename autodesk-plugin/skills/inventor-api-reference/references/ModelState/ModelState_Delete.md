# ModelState.Delete Method

Parent Object: [ModelState](../ModelState/ModelState.md)

## Description

Method that deletes the ModelState. The method returns an error if this is a built-in ModelState.

## Syntax

ModelState.**Delete**()

## Version

Introduced in version 2022
