# ModelState.FactoryDocument Property

Parent Object: [ModelState](../ModelState/ModelState.md)

## Description

Read-only property that returns the model state factory Document. If the model state factory document is not open, query this property will cause it open.

## Syntax

ModelState.**FactoryDocument**() As [Document](../Document/Document.md)

## Property Value

This is a read only property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 2022
