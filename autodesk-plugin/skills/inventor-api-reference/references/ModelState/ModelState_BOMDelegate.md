# ModelState.BOMDelegate Property

Parent Object: [ModelState](../ModelState/ModelState.md)

## Description

Read-only property that returns the ModelState which this ModelState's BOM is delegated to.

## Syntax

ModelState.**BOMDelegate**() As [ModelState](../ModelState/ModelState.md)

## Property Value

This is a read only property whose value is a [ModelState](../ModelState/ModelState.md).

## Version

Introduced in version 2022
