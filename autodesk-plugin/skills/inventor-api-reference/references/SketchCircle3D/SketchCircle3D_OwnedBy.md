# SketchCircle3D.OwnedBy Property

Parent Object: [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Description

Property indicating entity or entities that own this object.

## Syntax

SketchCircle3D.**OwnedBy**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 11
