# SketchCircle3D.CenterPoint Property

Parent Object: [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Description

Property that returns the point that defines the position of the center of the circle.

## Syntax

SketchCircle3D.**CenterPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
