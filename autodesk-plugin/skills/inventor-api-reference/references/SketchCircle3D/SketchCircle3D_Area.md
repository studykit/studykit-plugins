# SketchCircle3D.Area Property

Parent Object: [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Description

Double property that returns the area of the entity in square centimeters.

## Syntax

SketchCircle3D.**Area**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
