# EllipticalCylinder.BasePoint Property

Parent Object: [EllipticalCylinder](../EllipticalCylinder/EllipticalCylinder.md)

## Description

Specifies the base of the elliptical cylinder.

## Syntax

EllipticalCylinder.**BasePoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
