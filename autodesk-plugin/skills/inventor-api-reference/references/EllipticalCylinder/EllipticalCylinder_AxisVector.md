# EllipticalCylinder.AxisVector Property

Parent Object: [EllipticalCylinder](../EllipticalCylinder/EllipticalCylinder.md)

## Description

Specifies the direction of the axis of the elliptical cylinder.

## Syntax

EllipticalCylinder.**AxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
