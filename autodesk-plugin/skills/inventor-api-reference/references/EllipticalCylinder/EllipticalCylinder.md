# EllipticalCylinder Object

## Description

The EllipticalCylinder object. The object created is a transient mathematical object and is not displayed graphically.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../EllipticalCylinder/EllipticalCylinder_Copy.md) | Creates a copy of this EllipticalCylinder object. The result is entirely independent and can be edited without affecting the original EllipticalCylinder object. |
| [GetEllipticalCylinderData](../EllipticalCylinder/EllipticalCylinder_GetEllipticalCylinderData.md) | Get the data defining this elliptical cylinder. |
| [PutEllipticalCylinderData](../EllipticalCylinder/EllipticalCylinder_PutEllipticalCylinderData.md) | Method that sets the data defining this elliptical cylinder. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AxisVector](../EllipticalCylinder/EllipticalCylinder_AxisVector.md) | Specifies the direction of the axis of the elliptical cylinder. |
| [BasePoint](../EllipticalCylinder/EllipticalCylinder_BasePoint.md) | Specifies the base of the elliptical cylinder. |
| [Evaluator](../EllipticalCylinder/EllipticalCylinder_Evaluator.md) | Gets the surface evaluator for this elliptical cylinder. |
| [MajorAxisVector](../EllipticalCylinder/EllipticalCylinder_MajorAxisVector.md) | Specifies the direction of the major axis of the elliptical cylinder. |
| [MinorMajorRatio](../EllipticalCylinder/EllipticalCylinder_MinorMajorRatio.md) | Specifies the ratio of the minor and major axes of the elliptical cylinder. |

## Accessed From

[EllipticalCylinder.Copy](../EllipticalCylinder/EllipticalCylinder_Copy.md), [TransientGeometry.CreateEllipticalCylinder](../TransientGeometry/TransientGeometry_CreateEllipticalCylinder.md)
