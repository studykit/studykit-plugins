# EllipticalCylinder.Copy Method

Parent Object: [EllipticalCylinder](../EllipticalCylinder/EllipticalCylinder.md)

## Description

Creates a copy of this EllipticalCylinder object. The result is entirely independent and can be edited without affecting the original EllipticalCylinder object.

## Syntax

EllipticalCylinder.**Copy**() As [EllipticalCylinder](../EllipticalCylinder/EllipticalCylinder.md)

## Version

Introduced in version 2013
