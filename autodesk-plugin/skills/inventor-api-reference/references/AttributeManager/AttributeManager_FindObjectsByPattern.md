# AttributeManager.FindObjectsByPattern Method

Parent Object: [AttributeManager](../AttributeManager/AttributeManager.md)

## Description

Method that returns the objects whose AttributeSets, if converted to XML, satisfy the specified XSL pattern.

## Syntax

AttributeManager.**FindObjectsByPattern**( ***XPath*** As String ) As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

| --- | --- | --- |
| Name | Type | Description |
