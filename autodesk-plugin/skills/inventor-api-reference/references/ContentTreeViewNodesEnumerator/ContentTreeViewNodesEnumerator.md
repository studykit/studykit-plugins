# ContentTreeViewNodesEnumerator Object

## Description

The ContentFamiliesEnumerator is a collection object that provides access to a group of ContentFamily objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ContentTreeViewNodesEnumerator/ContentTreeViewNodesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ContentTreeViewNodesEnumerator/ContentTreeViewNodesEnumerator_Count.md) | Property that returns the number of ContentTreeViewNode objects in the collection. |
| [Item](../ContentTreeViewNodesEnumerator/ContentTreeViewNodesEnumerator_Item.md) | Returns the specified ContentTreeViewNode object from the collection. |
| [Type](../ContentTreeViewNodesEnumerator/ContentTreeViewNodesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
