# ComponentDefinitionsEnumerator.Type Property

Parent Object: [ComponentDefinitionsEnumerator](../ComponentDefinitionsEnumerator/ComponentDefinitionsEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ComponentDefinitionsEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
