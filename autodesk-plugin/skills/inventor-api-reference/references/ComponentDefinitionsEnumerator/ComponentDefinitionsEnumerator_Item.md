# ComponentDefinitionsEnumerator.Item Property

Parent Object: [ComponentDefinitionsEnumerator](../ComponentDefinitionsEnumerator/ComponentDefinitionsEnumerator.md)

## Description

Allows integer-indexed access to objects in the collection.

## Syntax

ComponentDefinitionsEnumerator.**Item**( ***Index*** As Long ) As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 10
