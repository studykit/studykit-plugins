# RevisionTableCell.StackedTextPosition Property

Parent Object: [RevisionTableCell](../RevisionTableCell/RevisionTableCell.md)

## Description

Gets and sets the position (alignment) of the stacked text with respect to regular text.

## Syntax

RevisionTableCell.**StackedTextPosition**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 2009
