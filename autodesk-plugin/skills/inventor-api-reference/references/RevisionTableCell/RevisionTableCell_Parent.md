# RevisionTableCell.Parent Property

Parent Object: [RevisionTableCell](../RevisionTableCell/RevisionTableCell.md)

## Description

Property that returns the parent RevisionTable object.

## Syntax

RevisionTableCell.**Parent**() As [RevisionTable](../RevisionTable/RevisionTable.md)

## Property Value

This is a read only property whose value is a [RevisionTable](../RevisionTable/RevisionTable.md).

## Version

Introduced in version 10
