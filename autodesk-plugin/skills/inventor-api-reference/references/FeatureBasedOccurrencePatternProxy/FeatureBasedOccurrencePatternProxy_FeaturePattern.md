# FeatureBasedOccurrencePatternProxy.FeaturePattern Property

Parent Object: [FeatureBasedOccurrencePatternProxy](../FeatureBasedOccurrencePatternProxy/FeatureBasedOccurrencePatternProxy.md)

## Description

Gets/Sets the feature pattern used as input for this occurrence pattern.

## Syntax

FeatureBasedOccurrencePatternProxy.**FeaturePattern**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2010
