# GraphicsImageSet.Type Property

Parent Object: [GraphicsImageSet](../GraphicsImageSet/GraphicsImageSet.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

GraphicsImageSet.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
