# UserCoordinateSystemProxy.Delete Method

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Method that deletes the UserCoordinateSystem.

## Syntax

UserCoordinateSystemProxy.**Delete**( [***RetainDependentFeaturesAndSketches***] As Boolean, [***RetainDependentWorkFeatures***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| RetainDependentFeaturesAndSketches | Boolean | Optional input Boolean that specifies whether to retain dependent features and sketches. Defaults to False. |
| RetainDependentWorkFeatures | Boolean | Optional input Boolean that specifies whether to retain dependent work features. Defaults to False.   This is an optional argument whose default value is False. |

## Version

Introduced in version 2010
