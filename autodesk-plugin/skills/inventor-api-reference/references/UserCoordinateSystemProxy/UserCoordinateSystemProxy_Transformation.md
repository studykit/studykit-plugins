# UserCoordinateSystemProxy.Transformation Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Gets and sets the transformation matrix for the coordinate system.

## Syntax

UserCoordinateSystemProxy.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2010
