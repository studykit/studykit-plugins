# UserCoordinateSystemProxy.OwnedBy Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Property that returns the PartFeature object that owns this object. This property returns Nothing if the IsOwnedByFeature property returns False.

## Syntax

UserCoordinateSystemProxy.**OwnedBy**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 2010
