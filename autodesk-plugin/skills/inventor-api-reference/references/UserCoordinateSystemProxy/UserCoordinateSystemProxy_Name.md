# UserCoordinateSystemProxy.Name Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Gets and sets the name of the UserCoordinateSystem.

## Syntax

UserCoordinateSystemProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
