# UserCoordinateSystemProxy.Parent Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Property that returns the parent ComponentDefinition object.

## Syntax

UserCoordinateSystemProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
