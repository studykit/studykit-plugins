# UserCoordinateSystemProxy.Origin Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Property that returns the work point that represents the origin of the coordinate system.

## Syntax

UserCoordinateSystemProxy.**Origin**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 2010
