# PitchAndHeightCoilExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The PitchAndHeightExtent object provides access to the information that defines the variables for a coil feature created using pitch and height values.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Height](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent_Height.md) | Property that gets the height of the coil feature. |
| [Parent](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Pitch](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent_Pitch.md) | Property that indicates the pitch of the coil feature. |
| [TaperAngle](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent_TaperAngle.md) | Property that indicates the taper angle of the coil feature. |
| [Type](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 5.3
