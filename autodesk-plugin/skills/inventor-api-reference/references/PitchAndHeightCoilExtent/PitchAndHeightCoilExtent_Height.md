# PitchAndHeightCoilExtent.Height Property

Parent Object: [PitchAndHeightCoilExtent](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent.md)

## Description

Property that gets the height of the coil feature.

## Syntax

PitchAndHeightCoilExtent.**Height**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
