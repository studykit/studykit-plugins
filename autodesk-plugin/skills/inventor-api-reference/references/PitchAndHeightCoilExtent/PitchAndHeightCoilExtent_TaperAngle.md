# PitchAndHeightCoilExtent.TaperAngle Property

Parent Object: [PitchAndHeightCoilExtent](../PitchAndHeightCoilExtent/PitchAndHeightCoilExtent.md)

## Description

Property that indicates the taper angle of the coil feature.

## Syntax

PitchAndHeightCoilExtent.**TaperAngle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 5.3
