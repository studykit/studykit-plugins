# PerpendicularConstraint3D.EntityOne Property

Parent Object: [PerpendicularConstraint3D](../PerpendicularConstraint3D/PerpendicularConstraint3D.md)

## Description

Property that returns the first sketch entity being constrained.

## Syntax

PerpendicularConstraint3D.**EntityOne**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
