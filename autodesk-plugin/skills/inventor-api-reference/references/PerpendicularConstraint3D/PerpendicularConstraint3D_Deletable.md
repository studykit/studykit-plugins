# PerpendicularConstraint3D.Deletable Property

Parent Object: [PerpendicularConstraint3D](../PerpendicularConstraint3D/PerpendicularConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

PerpendicularConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
