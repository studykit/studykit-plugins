# PerpendicularConstraint3D.Delete Method

Parent Object: [PerpendicularConstraint3D](../PerpendicularConstraint3D/PerpendicularConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

PerpendicularConstraint3D.**Delete**()

## Version

Introduced in version 10
