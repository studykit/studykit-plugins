# ModelDimensions.AngularModelDimensions Property

Parent Object: [ModelDimensions](../ModelDimensions/ModelDimensions.md)

## Description

Read-only property that returns a collection providing access to the existing angular dimensions and provides functionality to create new angular dimensions.

## Syntax

ModelDimensions.**AngularModelDimensions**() As [AngularModelDimensions](../AngularModelDimensions/AngularModelDimensions.md)

## Property Value

This is a read only property whose value is an [AngularModelDimensions](../AngularModelDimensions/AngularModelDimensions.md).

## Version

Introduced in version 2018
