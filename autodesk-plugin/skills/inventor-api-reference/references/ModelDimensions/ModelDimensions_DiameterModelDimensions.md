# ModelDimensions.DiameterModelDimensions Property

Parent Object: [ModelDimensions](../ModelDimensions/ModelDimensions.md)

## Description

Read-only property that returns a collection providing access to the existing diameter dimensions and provides functionality to create new diameter dimensions.

## Syntax

ModelDimensions.**DiameterModelDimensions**() As [DiameterModelDimensions](../DiameterModelDimensions/DiameterModelDimensions.md)

## Property Value

This is a read only property whose value is a [DiameterModelDimensions](../DiameterModelDimensions/DiameterModelDimensions.md).

## Version

Introduced in version 2018
