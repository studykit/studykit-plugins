# ModelDimensions.RadiusModelDimensions Property

Parent Object: [ModelDimensions](../ModelDimensions/ModelDimensions.md)

## Description

Read-only property that returns a collection providing access to the existing radial dimensions and provides functionality to create new radial dimensions.

## Syntax

ModelDimensions.**RadiusModelDimensions**() As [RadiusModelDimensions](../RadiusModelDimensions/RadiusModelDimensions.md)

## Property Value

This is a read only property whose value is a [RadiusModelDimensions](../RadiusModelDimensions/RadiusModelDimensions.md).

## Version

Introduced in version 2018
