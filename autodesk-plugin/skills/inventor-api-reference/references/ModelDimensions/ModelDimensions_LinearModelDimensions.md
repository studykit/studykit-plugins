# ModelDimensions.LinearModelDimensions Property

Parent Object: [ModelDimensions](../ModelDimensions/ModelDimensions.md)

## Description

Read-only property that returns a collection providing access to the existing linear dimensions and provides functionality to create new linear dimensions.

## Syntax

ModelDimensions.**LinearModelDimensions**() As [LinearModelDimensions](../LinearModelDimensions/LinearModelDimensions.md)

## Property Value

This is a read only property whose value is a [LinearModelDimensions](../LinearModelDimensions/LinearModelDimensions.md).

## Version

Introduced in version 2018
