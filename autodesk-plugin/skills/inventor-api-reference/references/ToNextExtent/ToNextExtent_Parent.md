# ToNextExtent.Parent Property

Parent Object: [ToNextExtent](../ToNextExtent/ToNextExtent.md)

## Description

Property that returns the parent PartFeature of the definition.

## Syntax

ToNextExtent.**Parent**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 5
