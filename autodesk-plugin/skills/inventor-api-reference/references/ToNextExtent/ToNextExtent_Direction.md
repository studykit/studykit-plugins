# ToNextExtent.Direction Property

Parent Object: [ToNextExtent](../ToNextExtent/ToNextExtent.md)

## Description

Property that gets and sets the direction of the feature. Valid \input is kPositiveExtentDirection or kNegativeExtentDirection. kPositiveExtentDirection defines the extrusion direction to be in the same direction as the normal of the sketch plane.

## Syntax

ToNextExtent.**Direction**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 5
