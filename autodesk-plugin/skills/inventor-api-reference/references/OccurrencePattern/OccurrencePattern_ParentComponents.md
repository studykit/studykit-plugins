# OccurrencePattern.ParentComponents Property

Parent Object: [OccurrencePattern](../OccurrencePattern/OccurrencePattern.md)

## Description

Property that gets and sets the components used as input for the pattern.

## Syntax

OccurrencePattern.**ParentComponents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2010
