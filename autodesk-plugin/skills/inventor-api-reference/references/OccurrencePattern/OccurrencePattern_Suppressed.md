# OccurrencePattern.Suppressed Property

Parent Object: [OccurrencePattern](../OccurrencePattern/OccurrencePattern.md)

## Description

Returns the suppress state of the occurrence pattern.

## Syntax

OccurrencePattern.**Suppressed**() As [OccurrencePatternSuppressionEnum](../OccurrencePatternSuppressionEnum.md)

## Property Value

This is a read only property whose value is an [OccurrencePatternSuppressionEnum](../OccurrencePatternSuppressionEnum.md).

## Version

Introduced in version 2022
