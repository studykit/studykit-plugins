# OccurrencePattern.Parent Property

Parent Object: [OccurrencePattern](../OccurrencePattern/OccurrencePattern.md)

## Description

Property that returns the parent of the object.

## Syntax

OccurrencePattern.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).

## Version

Introduced in version 6
