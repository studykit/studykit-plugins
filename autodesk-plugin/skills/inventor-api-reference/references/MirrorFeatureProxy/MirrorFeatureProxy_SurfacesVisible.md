# MirrorFeatureProxy.SurfacesVisible Property

Parent Object: [MirrorFeatureProxy](../MirrorFeatureProxy/MirrorFeatureProxy.md)

## Description

Gets and sets whether all the resulting surfaces are visible or not.

## Syntax

MirrorFeatureProxy.**SurfacesVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
