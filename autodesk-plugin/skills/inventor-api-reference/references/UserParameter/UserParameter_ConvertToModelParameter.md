# UserParameter.ConvertToModelParameter Method

Parent Object: [UserParameter](../UserParameter/UserParameter.md)

## Description

Method that causes the parameter to be converted to a model parameter. The new ModelParameter object is returned.

## Syntax

UserParameter.**ConvertToModelParameter**() As [ModelParameter](../ModelParameter/ModelParameter.md)

## Version

Introduced in version 10
