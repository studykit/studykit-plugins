# UserParameter.ConvertToReferenceParameter Method

Parent Object: [UserParameter](../UserParameter/UserParameter.md)

## Description

Method that causes the parameter to be converted to a reference parameter. The new ReferenceParameter object is returned.

## Syntax

UserParameter.**ConvertToReferenceParameter**() As [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Version

Introduced in version 10
