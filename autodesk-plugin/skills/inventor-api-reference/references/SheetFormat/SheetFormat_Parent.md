# SheetFormat.Parent Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that returns the parent DrawingDocument object.

## Syntax

SheetFormat.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 2009
