# SheetFormat.HasDrawingViews Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that returns whether this sheet format contains definitions for one or more drawing views.

## Syntax

SheetFormat.**HasDrawingViews**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
