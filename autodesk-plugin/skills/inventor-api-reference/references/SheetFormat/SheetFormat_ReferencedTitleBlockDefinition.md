# SheetFormat.ReferencedTitleBlockDefinition Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that returns the TitleBlockDefinition that this format references.

## Syntax

SheetFormat.**ReferencedTitleBlockDefinition**() As [TitleBlockDefinition](../TitleBlockDefinition/TitleBlockDefinition.md)

## Property Value

This is a read only property whose value is a [TitleBlockDefinition](../TitleBlockDefinition/TitleBlockDefinition.md).

## Version

Introduced in version 2009
