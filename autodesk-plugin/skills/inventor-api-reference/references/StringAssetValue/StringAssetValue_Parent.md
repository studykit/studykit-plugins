# StringAssetValue.Parent Property

Parent Object: [StringAssetValue](../StringAssetValue/StringAssetValue.md)

## Description

Read-only property that returns the parent Asset object.

## Syntax

StringAssetValue.**Parent**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read only property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
