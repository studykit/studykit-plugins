# MiniToolbarTextBox.Count Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Returns the number of items in the text box list.

## Syntax

MiniToolbarTextBox.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2018
