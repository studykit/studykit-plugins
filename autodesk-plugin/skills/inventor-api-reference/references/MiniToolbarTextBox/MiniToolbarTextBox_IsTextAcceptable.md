# MiniToolbarTextBox.IsTextAcceptable Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Gets and sets whether the text in the text box is acceptable.

## Syntax

MiniToolbarTextBox.**IsTextAcceptable**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
