# MiniToolbarTextBox.Clear Method

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Removes all the items from the control.

## Syntax

MiniToolbarTextBox.**Clear**()

## Version

Introduced in version 2018
