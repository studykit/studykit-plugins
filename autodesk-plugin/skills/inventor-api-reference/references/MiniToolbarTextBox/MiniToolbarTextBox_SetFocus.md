# MiniToolbarTextBox.SetFocus Method

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Sets the focus on this control and highlights the current content of the value editor.

## Syntax

MiniToolbarTextBox.**SetFocus**()

## Version

Introduced in version 2019
