# MiniToolbarTextBox.LargeIcon Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Read-write property that gets and sets the large icon for the control.

## Syntax

MiniToolbarTextBox.**LargeIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2018
