# DrawingPrintManager.PaperSize Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the size of the paper to be printed. Defaults to the printer's default.

## Syntax

DrawingPrintManager.**PaperSize**() As [PaperSizeEnum](../PaperSizeEnum.md)

## Property Value

This is a read/write property whose value is a [PaperSizeEnum](../PaperSizeEnum.md).
