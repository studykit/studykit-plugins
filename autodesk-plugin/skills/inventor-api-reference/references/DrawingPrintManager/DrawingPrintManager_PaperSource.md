# DrawingPrintManager.PaperSource Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the paper source (tray) on the printer. This is printer specific. Defaults to printer's default (denoted by -1).

## Syntax

DrawingPrintManager.**PaperSource**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 5
