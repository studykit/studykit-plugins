# DrawingPrintManager.Orientation Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the orientation of the printer. Defaults to the printer's default.

## Syntax

DrawingPrintManager.**Orientation**() As [PrintOrientationEnum](../PrintOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [PrintOrientationEnum](../PrintOrientationEnum.md).
