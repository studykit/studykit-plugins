# DrawingPrintManager.SetCurrentView Method

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Set the current view used for printing.

## Syntax

DrawingPrintManager.**SetCurrentView**( ***CurrentView*** As [ClientView](../ClientView/ClientView.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CurrentView | [ClientView](../ClientView/ClientView.md) | Input a ClientView for the print manager to print. |

## Version

Introduced in version 2019
