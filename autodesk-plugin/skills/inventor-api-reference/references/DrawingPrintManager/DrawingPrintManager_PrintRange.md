# DrawingPrintManager.PrintRange Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets information about which sheets are to be printed. Defaults to kPrintCurrentSheet.

## Syntax

DrawingPrintManager.**PrintRange**() As [PrintRangeEnum](../PrintRangeEnum.md)

## Property Value

This is a read/write property whose value is a [PrintRangeEnum](../PrintRangeEnum.md).
