# RevolveFeature.AxisEntity Property

Parent Object: [RevolveFeature](../RevolveFeature/RevolveFeature.md)

## Description

Gets and sets the entity used to define the axis of revolution.

## Syntax

RevolveFeature.**AxisEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 5
