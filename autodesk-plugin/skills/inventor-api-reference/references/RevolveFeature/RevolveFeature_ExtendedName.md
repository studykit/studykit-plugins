# RevolveFeature.ExtendedName Property

Parent Object: [RevolveFeature](../RevolveFeature/RevolveFeature.md)

## Description

Read-only property that returns the full feature name including any extended information.

## Syntax

RevolveFeature.**ExtendedName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
