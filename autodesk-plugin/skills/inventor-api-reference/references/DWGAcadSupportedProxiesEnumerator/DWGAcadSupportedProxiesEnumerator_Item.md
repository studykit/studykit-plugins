# DWGAcadSupportedProxiesEnumerator.Item Property

Parent Object: [DWGAcadSupportedProxiesEnumerator](../DWGAcadSupportedProxiesEnumerator/DWGAcadSupportedProxiesEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGAcadSupportedProxiesEnumerator.**Item**( ***Index*** As Variant ) As [DWGAcadSupportedProxy](../DWGAcadSupportedProxy/DWGAcadSupportedProxy.md)

## Property Value

This is a read only property whose value is a [DWGAcadSupportedProxy](../DWGAcadSupportedProxy/DWGAcadSupportedProxy.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2018
