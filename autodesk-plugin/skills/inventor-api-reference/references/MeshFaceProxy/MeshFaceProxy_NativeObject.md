# MeshFaceProxy.NativeObject Property

Parent Object: [MeshFaceProxy](../MeshFaceProxy/MeshFaceProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MeshFaceProxy.**NativeObject**() As [MeshFace](../MeshFace/MeshFace.md)

## Property Value

This is a read only property whose value is a [MeshFace](../MeshFace/MeshFace.md).

## Version

Introduced in version 2017
