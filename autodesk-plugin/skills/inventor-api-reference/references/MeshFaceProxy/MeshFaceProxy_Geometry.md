# MeshFaceProxy.Geometry Property

Parent Object: [MeshFaceProxy](../MeshFaceProxy/MeshFaceProxy.md)

## Description

Read-only property that returns the underlying geometry of the MeshFace.

## Syntax

MeshFaceProxy.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2017
