# ProjectedCuts.Item Property

Parent Object: [ProjectedCuts](../ProjectedCuts/ProjectedCuts.md)

## Description

Method that returns the specified ProjectedCut object from the collection.

## Syntax

ProjectedCuts.**Item**( ***Index*** As Variant ) As [ProjectedCut](../ProjectedCut/ProjectedCut.md)

## Property Value

This is a read only property whose value is a [ProjectedCut](../ProjectedCut/ProjectedCut.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the projected cut edges to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the name of the projected cut edges. If an out of range index or a name of a non\-existent projected cut edges is provided, an error occurs. |

## Version

Introduced in version 2010
