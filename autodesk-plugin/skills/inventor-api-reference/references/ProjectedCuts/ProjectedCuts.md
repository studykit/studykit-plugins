# ProjectedCuts Object

## Description

The ProjectedCuts object provides access to all of the projected cut edges on the sketch.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ProjectedCuts/ProjectedCuts_Add.md) | Method that adds projected cut edges by intersecting the model with the sketch plane and creating the resulting sketch entities. This method performs the same function as the Project Cut Edges command. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ProjectedCuts/ProjectedCuts_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ProjectedCuts/ProjectedCuts_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ProjectedCuts/ProjectedCuts_Item.md) | Method that returns the specified ProjectedCut object from the collection. |
| [Type](../ProjectedCuts/ProjectedCuts_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
