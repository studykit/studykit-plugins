# SurfaceTextureStylesEnumerator Object

## Description

The SurfaceTextureStylesEnumerator object provides access to the existing SurfaceTextureStyle objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SurfaceTextureStylesEnumerator/SurfaceTextureStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SurfaceTextureStylesEnumerator/SurfaceTextureStylesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SurfaceTextureStylesEnumerator/SurfaceTextureStylesEnumerator_Item.md) | Returns the specified SurfaceTextureStyle object from the collection. |
| [Type](../SurfaceTextureStylesEnumerator/SurfaceTextureStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.SurfaceTextureStyles](../DrawingStylesManager/DrawingStylesManager_SurfaceTextureStyles.md)

## Version

Introduced in version 2009
