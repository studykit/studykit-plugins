# BalloonStyle.ConvertToLocal Method

Parent Object: [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

BalloonStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2009
