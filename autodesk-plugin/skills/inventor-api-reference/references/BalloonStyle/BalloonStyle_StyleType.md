# BalloonStyle.StyleType Property

Parent Object: [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Description

Gets the type of the style.

## Syntax

BalloonStyle.**StyleType**() As [StyleTypeEnum](../StyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [StyleTypeEnum](../StyleTypeEnum.md).

## Version

Introduced in version 2009
