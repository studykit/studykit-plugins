# BalloonStyle.TextStyle Property

Parent Object: [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Description

Property that gets and sets the text style used to format the text in a balloon.

## Syntax

BalloonStyle.**TextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2009
