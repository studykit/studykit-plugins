# NativeBrowserNodeDefinition.OverrideStateIcon Property

Parent Object: [NativeBrowserNodeDefinition](../NativeBrowserNodeDefinition/NativeBrowserNodeDefinition.md)

## Description

Gets and sets state icon override on a existing definition object.

## Syntax

NativeBrowserNodeDefinition.**OverrideStateIcon**() As [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Property Value

This is a read/write property whose value is a [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md).

## Version

Introduced in version 2022
