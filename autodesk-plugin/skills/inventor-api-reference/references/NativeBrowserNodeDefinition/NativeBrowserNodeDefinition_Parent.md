# NativeBrowserNodeDefinition.Parent Property

Parent Object: [NativeBrowserNodeDefinition](../NativeBrowserNodeDefinition/NativeBrowserNodeDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

NativeBrowserNodeDefinition.**Parent**() As [Document](../Document/Document.md)

## Property Value

This is a read only property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 8
