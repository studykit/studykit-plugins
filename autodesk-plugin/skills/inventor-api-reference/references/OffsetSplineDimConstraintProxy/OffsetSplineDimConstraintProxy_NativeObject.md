# OffsetSplineDimConstraintProxy.NativeObject Property

Parent Object: [OffsetSplineDimConstraintProxy](../OffsetSplineDimConstraintProxy/OffsetSplineDimConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

OffsetSplineDimConstraintProxy.**NativeObject**() As [OffsetSplineDimConstraint](../OffsetSplineDimConstraint/OffsetSplineDimConstraint.md)

## Property Value

This is a read only property whose value is an [OffsetSplineDimConstraint](../OffsetSplineDimConstraint/OffsetSplineDimConstraint.md).

## Version

Introduced in version 11
