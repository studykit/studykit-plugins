# ChamferDefinition.AutomaticEdgeChain Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Gets whether or not to use all tangentially connected edges.

## Syntax

ChamferDefinition.**AutomaticEdgeChain**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
