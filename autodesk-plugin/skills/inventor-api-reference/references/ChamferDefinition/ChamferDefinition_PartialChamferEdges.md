# ChamferDefinition.PartialChamferEdges Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Gets and sets the PartialChamferEdges collection object which defines the partially chamfered edges info.

## Syntax

ChamferDefinition.**PartialChamferEdges**() As [PartialChamferEdges](../PartialChamferEdges/PartialChamferEdges.md)

## Property Value

This is a read/write property whose value is a [PartialChamferEdges](../PartialChamferEdges/PartialChamferEdges.md).

## Version

Introduced in version 2018
