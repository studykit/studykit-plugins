# DiameterDimConstraintProxy.NativeObject Property

Parent Object: [DiameterDimConstraintProxy](../DiameterDimConstraintProxy/DiameterDimConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DiameterDimConstraintProxy.**NativeObject**() As [DiameterDimConstraint](../DiameterDimConstraint/DiameterDimConstraint.md)

## Property Value

This is a read only property whose value is a [DiameterDimConstraint](../DiameterDimConstraint/DiameterDimConstraint.md).

## Version

Introduced in version 6
