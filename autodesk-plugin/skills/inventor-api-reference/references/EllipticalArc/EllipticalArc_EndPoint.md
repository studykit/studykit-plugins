# EllipticalArc.EndPoint Property

Parent Object: [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Description

Gets the end point for the arc.

## Syntax

EllipticalArc.**EndPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
