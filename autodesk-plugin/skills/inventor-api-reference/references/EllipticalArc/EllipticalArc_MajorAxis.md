# EllipticalArc.MajorAxis Property

Parent Object: [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Description

Specifies the major axis for the ellipse.

## Syntax

EllipticalArc.**MajorAxis**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 6
