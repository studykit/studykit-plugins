# EllipticalArc Object

## Description

The EllipticalArc object. The object created is a transient mathematical object and is not displayed graphically

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../EllipticalArc/EllipticalArc_Copy.md) | Creates a copy of this EllipticalArc object. The result is entirely independent and can be edited without affecting the original EllipticalArc object. |
| [GetEllipticalArcData](../EllipticalArc/EllipticalArc_GetEllipticalArcData.md) | Get the data defining this elliptical arc. |
| [PutEllipticalArcData](../EllipticalArc/EllipticalArc_PutEllipticalArcData.md) | Method that sets the data defining this elliptical arc. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Center](../EllipticalArc/EllipticalArc_Center.md) | Specifies the center point for the arc. |
| [EndPoint](../EllipticalArc/EllipticalArc_EndPoint.md) | Gets the end point for the arc. |
| [Evaluator](../EllipticalArc/EllipticalArc_Evaluator.md) | Gets the CurveEvaluator for the arc. |
| [MajorAxis](../EllipticalArc/EllipticalArc_MajorAxis.md) | Specifies the major axis for the ellipse. |
| [MajorRadius](../EllipticalArc/EllipticalArc_MajorRadius.md) | Specifies the major radius for the ellipse. |
| [MinorAxis](../EllipticalArc/EllipticalArc_MinorAxis.md) | Specifies the minor axis for the ellipse. |
| [MinorRadius](../EllipticalArc/EllipticalArc_MinorRadius.md) | Specifies the minor radius for the ellipse. |
| [StartAngle](../EllipticalArc/EllipticalArc_StartAngle.md) | Specifies the start angle for the arc. |
| [StartPoint](../EllipticalArc/EllipticalArc_StartPoint.md) | Gets the start point for the arc. |
| [SweepAngle](../EllipticalArc/EllipticalArc_SweepAngle.md) | Specifies the sweep angle from the start angle for the arc. |
