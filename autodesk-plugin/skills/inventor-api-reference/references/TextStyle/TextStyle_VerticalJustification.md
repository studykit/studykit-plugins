# TextStyle.VerticalJustification Property

Parent Object: [TextStyle](../TextStyle/TextStyle.md)

## Description

Gets and sets the vertical justification of the text style.

## Syntax

TextStyle.**VerticalJustification**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 5.3
