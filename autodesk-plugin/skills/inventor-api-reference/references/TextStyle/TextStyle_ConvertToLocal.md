# TextStyle.ConvertToLocal Method

Parent Object: [TextStyle](../TextStyle/TextStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

TextStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 9
