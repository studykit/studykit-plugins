# TextStyle.Underline Property

Parent Object: [TextStyle](../TextStyle/TextStyle.md)

## Description

Gets and sets whether the text style has underline formatting or not. True if the text style has underline formatting.

## Syntax

TextStyle.**Underline**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
