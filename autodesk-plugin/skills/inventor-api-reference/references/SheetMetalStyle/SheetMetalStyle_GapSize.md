# SheetMetalStyle.GapSize Property

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Gets and sets the expression that defines the value of the gap size for miters, seams, and rips.

## Syntax

SheetMetalStyle.**GapSize**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
