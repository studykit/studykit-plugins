# SheetMetalStyle.ConvertToLocal Method

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

SheetMetalStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2009
