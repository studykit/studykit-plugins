# SheetMetalStyle.UnfoldMethod Property

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Read-write property that gets and set the unfold method associated with the style.

## Syntax

SheetMetalStyle.**UnfoldMethod**() As [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Property Value

This is a read/write property whose value is a [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md).
