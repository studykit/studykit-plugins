# SheetMetalStyle.Name Property

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Gets/Sets the name of the Style.

## Syntax

SheetMetalStyle.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5.3
