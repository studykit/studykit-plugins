# SheetMetalStyle.PunchRepresentationType Property

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Gets and sets the default punch representation type.

## Syntax

SheetMetalStyle.**PunchRepresentationType**() As [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md).
