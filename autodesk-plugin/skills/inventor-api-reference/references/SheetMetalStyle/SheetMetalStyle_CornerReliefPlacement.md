# SheetMetalStyle.CornerReliefPlacement Property

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Gets and sets corner relief placement (valid only for square and round corners).

## Syntax

SheetMetalStyle.**CornerReliefPlacement**() As [CornerReliefPlacementEnum](../CornerReliefPlacementEnum.md)

## Property Value

This is a read/write property whose value is a [CornerReliefPlacementEnum](../CornerReliefPlacementEnum.md).

## Version

Introduced in version 2017
