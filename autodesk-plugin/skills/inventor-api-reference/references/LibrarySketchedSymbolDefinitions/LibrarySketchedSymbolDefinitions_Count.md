# LibrarySketchedSymbolDefinitions.Count Property

Parent Object: [LibrarySketchedSymbolDefinitions](../LibrarySketchedSymbolDefinitions/LibrarySketchedSymbolDefinitions.md)

## Description

Gets the number of items in this collection.

## Syntax

LibrarySketchedSymbolDefinitions.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
