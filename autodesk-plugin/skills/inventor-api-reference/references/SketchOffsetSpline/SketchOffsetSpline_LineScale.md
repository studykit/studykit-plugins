# SketchOffsetSpline.LineScale Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Gets and sets the LineScale applied to this sketch spline.

## Syntax

SketchOffsetSpline.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
