# SketchOffsetSpline.Geometry3d Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Read-only property that returns b-spline geometry that represents this spline in model space.

## Syntax

SketchOffsetSpline.**Geometry3d**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read only property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 2013
