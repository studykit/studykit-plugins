# SketchOffsetSpline.OverrideColor Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Gets and sets the color applied to this sketch spline.

## Syntax

SketchOffsetSpline.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
