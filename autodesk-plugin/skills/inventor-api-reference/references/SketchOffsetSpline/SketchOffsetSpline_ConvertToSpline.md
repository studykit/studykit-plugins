# SketchOffsetSpline.ConvertToSpline Method

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Method that converts the offset spline to a SketchSpline object and breaks the association to the spline from which it is offset.

## Syntax

SketchOffsetSpline.**ConvertToSpline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Version

Introduced in version 9
