# SketchOffsetSpline.CurvatureDisplay Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Specifies whether curvature information is displayed for the offset spline.

## Syntax

SketchOffsetSpline.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
