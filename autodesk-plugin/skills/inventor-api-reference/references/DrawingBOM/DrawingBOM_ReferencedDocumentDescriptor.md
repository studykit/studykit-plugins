# DrawingBOM.ReferencedDocumentDescriptor Property

Parent Object: [DrawingBOM](../DrawingBOM/DrawingBOM.md)

## Description

Property that gets the model document referenced by this DrawingBOM.

## Syntax

DrawingBOM.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 2009
