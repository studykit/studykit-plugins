# DrawingBOM.DrawingBOMColumns Property

Parent Object: [DrawingBOM](../DrawingBOM/DrawingBOM.md)

## Description

Property that gets the DrawingBOMColumns collection object.

## Syntax

DrawingBOM.**DrawingBOMColumns**() As [DrawingBOMColumns](../DrawingBOMColumns/DrawingBOMColumns.md)

## Property Value

This is a read only property whose value is a [DrawingBOMColumns](../DrawingBOMColumns/DrawingBOMColumns.md).

## Version

Introduced in version 2009
