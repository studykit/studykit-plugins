# DrawingBOM.DrawingBOMRows Property

Parent Object: [DrawingBOM](../DrawingBOM/DrawingBOM.md)

## Description

Property that gets the DrawingBOMRows collection object.

## Syntax

DrawingBOM.**DrawingBOMRows**() As [DrawingBOMRows](../DrawingBOMRows/DrawingBOMRows.md)

## Property Value

This is a read only property whose value is a [DrawingBOMRows](../DrawingBOMRows/DrawingBOMRows.md).

## Version

Introduced in version 2009
