# RestFeatures Object

## Description

The RestFeatures collection object provides access to the RestFeature objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RestFeatures/RestFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../RestFeatures/RestFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../RestFeatures/RestFeatures_Item.md) | Returns the specified RestFeature object from the collection. |
| [Type](../RestFeatures/RestFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.RestFeatures](../PartFeatures/PartFeatures_RestFeatures.md), [SheetMetalFeatures.RestFeatures](../SheetMetalFeatures/SheetMetalFeatures_RestFeatures.md)

## Version

Introduced in version 2010
