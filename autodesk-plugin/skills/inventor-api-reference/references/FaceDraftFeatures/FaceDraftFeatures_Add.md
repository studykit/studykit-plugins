# FaceDraftFeatures.Add Method

Parent Object: [FaceDraftFeatures](../FaceDraftFeatures/FaceDraftFeatures.md)

## Description

Method that creates a new FaceDraft feature.

## Syntax

FaceDraftFeatures.**Add**( ***FaceDraftDefinition*** As [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md) ) As [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FaceDraftDefinition | [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md) | Input FaceDraftDefinition object that defines the face draft feature you want to create. An FaceDraftDefinition object can be created using the FaceDraftFeatures.CreateFaceDraftDefinition method. It can also be obtained from an existing FaceDraftFeature object. |

## Version

Introduced in version 2016
