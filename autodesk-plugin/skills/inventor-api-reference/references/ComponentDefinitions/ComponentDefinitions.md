# ComponentDefinitions Object

## Description

Object that holds a collection of objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../ComponentDefinitions/ComponentDefinitions_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ComponentDefinitions/ComponentDefinitions_Item.md) | Allows integer-indexed access to objects in the collection. |
| [Type](../ComponentDefinitions/ComponentDefinitions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
