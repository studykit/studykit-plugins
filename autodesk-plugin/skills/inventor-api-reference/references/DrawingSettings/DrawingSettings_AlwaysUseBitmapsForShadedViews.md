# DrawingSettings.AlwaysUseBitmapsForShadedViews Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets whether bitmaps should always be used when displaying shaded drawing views.

## Syntax

DrawingSettings.**AlwaysUseBitmapsForShadedViews**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
