# DrawingSettings.ReflectionEnvironmentUseApplicationOptions Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets whether the reflection environment will use application options for shaded views.

## Syntax

DrawingSettings.**ReflectionEnvironmentUseApplicationOptions**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
