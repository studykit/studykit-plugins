# DrawingSettings.DeferUpdates Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets the state of the DeferUpdates setting.

## Syntax

DrawingSettings.**DeferUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 7
