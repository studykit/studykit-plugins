# DrawingSettings.AutomatedCenterlineSettings Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Property that returns an object that provides properties to set all of the various settings that are used as the default settings for this document when creating automatic centerlines.

## Syntax

DrawingSettings.**AutomatedCenterlineSettings**() As [AutomatedCenterlineSettings](../AutomatedCenterlineSettings/AutomatedCenterlineSettings.md)

## Property Value

This is a read only property whose value is an [AutomatedCenterlineSettings](../AutomatedCenterlineSettings/AutomatedCenterlineSettings.md).

## Version

Introduced in version 2010
