# DrawingSettings.OfflineImageFidelity Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets the printing quality for shaded drawing view when defer update is active.

## Syntax

DrawingSettings.**OfflineImageFidelity**() As [OfflineImageFidelityEnum](../OfflineImageFidelityEnum.md)

## Property Value

This is a read/write property whose value is an [OfflineImageFidelityEnum](../OfflineImageFidelityEnum.md).

## Version

Introduced in version 8
