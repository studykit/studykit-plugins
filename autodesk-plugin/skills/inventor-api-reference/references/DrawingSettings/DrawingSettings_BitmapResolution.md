# DrawingSettings.BitmapResolution Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets the bitmap resolution for shaded views.

## Syntax

DrawingSettings.**BitmapResolution**() As [BitmapResolutionTypeEnum](../BitmapResolutionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BitmapResolutionTypeEnum](../BitmapResolutionTypeEnum.md).

## Version

Introduced in version 2024
