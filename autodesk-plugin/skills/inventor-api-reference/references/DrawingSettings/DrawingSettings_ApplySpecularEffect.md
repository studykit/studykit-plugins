# DrawingSettings.ApplySpecularEffect Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets whether to apply specular effect or not.

## Syntax

DrawingSettings.**ApplySpecularEffect**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
