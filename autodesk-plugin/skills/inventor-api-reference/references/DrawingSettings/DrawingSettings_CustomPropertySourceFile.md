# DrawingSettings.CustomPropertySourceFile Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets the full file name of the model file that contains custom properties to copy to the active drawing or template.

## Syntax

DrawingSettings.**CustomPropertySourceFile**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
