# DrawingSettings Object

## Description

The DrawingSettings object provides access to properties that provide read and write access of the various drawing document specific settings. This is somewhat equivalent to the Drawing tab of the Document Settings dialog.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AlwaysUseBitmapsForShadedViews](../DrawingSettings/DrawingSettings_AlwaysUseBitmapsForShadedViews.md) | Gets and sets whether bitmaps should always be used when displaying shaded drawing views. |
| [Application](../DrawingSettings/DrawingSettings_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [ApplySpecularEffect](../DrawingSettings/DrawingSettings_ApplySpecularEffect.md) | Gets and sets whether to apply specular effect or not. |
| [AutomatedCenterlineSettings](../DrawingSettings/DrawingSettings_AutomatedCenterlineSettings.md) | Property that returns an object that provides properties to set all of the various settings that are used as the default settings for this document when creating automatic centerlines. |
| [BitmapResolution](../DrawingSettings/DrawingSettings_BitmapResolution.md) | Gets and sets the bitmap resolution for shaded views. |
| [CrossHatchClipping](../DrawingSettings/DrawingSettings_CrossHatchClipping.md) | Gets and sets whether cross hatches should be clipped about drawing annotations. |
| [CustomPropertySourceFile](../DrawingSettings/DrawingSettings_CustomPropertySourceFile.md) | Gets and sets the full file name of the model file that contains custom properties to copy to the active drawing or template. |
| [DeferUpdates](../DrawingSettings/DrawingSettings_DeferUpdates.md) | Gets and sets the state of the DeferUpdates setting. |
| [DimensionTextAlignment](../DrawingSettings/DrawingSettings_DimensionTextAlignment.md) | Gets and sets the state of dimension text update alignment. |
| [HighlightInvalidAnnotations](../DrawingSettings/DrawingSettings_HighlightInvalidAnnotations.md) | Gets and sets the state of the invalid annotation highlighting setting. |
| [InvalidFeatureBasedAnnotationsColor](../DrawingSettings/DrawingSettings_InvalidFeatureBasedAnnotationsColor.md) | Gets and sets the color for invalid feature-based annotations. |
| [OfflineImageFidelity](../DrawingSettings/DrawingSettings_OfflineImageFidelity.md) | Gets and sets the printing quality for shaded drawing view when defer update is active. |
| [Parent](../DrawingSettings/DrawingSettings_Parent.md) | Property that returns the parent DrawingDocument object. |
| [PreserveOrphanedAnnotations](../DrawingSettings/DrawingSettings_PreserveOrphanedAnnotations.md) | Gets and sets whether orphaned annotations should be preserved and continued to be displayed. |
| [ReflectionEnvironmentFilePath](../DrawingSettings/DrawingSettings_ReflectionEnvironmentFilePath.md) | Gets and sets the reflection environment file path. |
| [ReflectionEnvironmentUseApplicationOptions](../DrawingSettings/DrawingSettings_ReflectionEnvironmentUseApplicationOptions.md) | Gets and sets whether the reflection environment will use application options for shaded views. |
| [Type](../DrawingSettings/DrawingSettings_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingDocument.DrawingSettings](../DrawingDocument/DrawingDocument_DrawingSettings.md)
