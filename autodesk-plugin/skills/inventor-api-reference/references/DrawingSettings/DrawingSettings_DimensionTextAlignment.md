# DrawingSettings.DimensionTextAlignment Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets the state of dimension text update alignment.

## Syntax

DrawingSettings.**DimensionTextAlignment**() As [DimensionTextAlignmentEnum](../DimensionTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionTextAlignmentEnum](../DimensionTextAlignmentEnum.md).

## Version

Introduced in version 11
