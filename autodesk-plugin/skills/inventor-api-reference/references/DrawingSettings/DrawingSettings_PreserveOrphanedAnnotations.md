# DrawingSettings.PreserveOrphanedAnnotations Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets whether orphaned annotations should be preserved and continued to be displayed.

## Syntax

DrawingSettings.**PreserveOrphanedAnnotations**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
