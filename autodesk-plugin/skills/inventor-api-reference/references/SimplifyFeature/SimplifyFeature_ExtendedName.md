# SimplifyFeature.ExtendedName Property

Parent Object: [SimplifyFeature](../SimplifyFeature/SimplifyFeature.md)

## Description

Read-only property that returns the full feature name including any extended information.

## Syntax

SimplifyFeature.**ExtendedName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2026
