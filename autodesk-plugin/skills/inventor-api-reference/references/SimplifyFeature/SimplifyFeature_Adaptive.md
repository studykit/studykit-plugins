# SimplifyFeature.Adaptive Property

Parent Object: [SimplifyFeature](../SimplifyFeature/SimplifyFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

SimplifyFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2026
