# PresentationDocument.ActiveLightingStyle Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Gets and sets the lighting style to use for the presentation.

## Syntax

PresentationDocument.**ActiveLightingStyle**() As [LightingStyle](../LightingStyle/LightingStyle.md)

## Property Value

This is a read/write property whose value is a [LightingStyle](../LightingStyle/LightingStyle.md).

## Version

Introduced in version 2011
