# PresentationDocument.Update Method

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Performs compute operations on all of the entities within this Document's scope that may be out of date with respect to their driving entities.

## Syntax

PresentationDocument.**Update**()

## Version

Introduced in version 4
