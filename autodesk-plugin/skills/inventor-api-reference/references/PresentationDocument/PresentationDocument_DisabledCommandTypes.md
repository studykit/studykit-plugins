# PresentationDocument.DisabledCommandTypes Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Bit mask indicating the disabled command types.

## Syntax

PresentationDocument.**DisabledCommandTypes**() As [CommandTypesEnum](../CommandTypesEnum.md)

## Property Value

This is a read/write property whose value is a [CommandTypesEnum](../CommandTypesEnum.md).

## Version

Introduced in version 6
