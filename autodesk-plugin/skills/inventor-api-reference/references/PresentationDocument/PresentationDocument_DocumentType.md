# PresentationDocument.DocumentType Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Gets the constant that indicates the type of this document.

## Syntax

PresentationDocument.**DocumentType**() As [DocumentTypeEnum](../DocumentTypeEnum.md)

## Property Value

This is a read only property whose value is a [DocumentTypeEnum](../DocumentTypeEnum.md).

## Version

Introduced in version 4
