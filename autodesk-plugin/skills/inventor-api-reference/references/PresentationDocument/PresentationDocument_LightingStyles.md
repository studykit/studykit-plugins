# PresentationDocument.LightingStyles Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the LightingStyles collection object.

## Syntax

PresentationDocument.**LightingStyles**() As [LightingStyles](../LightingStyles/LightingStyles.md)

## Property Value

This is a read only property whose value is a [LightingStyles](../LightingStyles/LightingStyles.md).

## Version

Introduced in version 2010
