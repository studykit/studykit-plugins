# PresentationDocument.RevertReservedForWriteByMe Method

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Method that reverts the file checked out by the caller.

## Syntax

PresentationDocument.**RevertReservedForWriteByMe**()

## Version

Introduced in version 6
