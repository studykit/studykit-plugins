# PresentationDocument.ReferenceKeyManager Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Gets this document's ReferenceKeyManager.

## Syntax

PresentationDocument.**ReferenceKeyManager**() As [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md)

## Property Value

This is a read only property whose value is a [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md).

## Version

Introduced in version 5
