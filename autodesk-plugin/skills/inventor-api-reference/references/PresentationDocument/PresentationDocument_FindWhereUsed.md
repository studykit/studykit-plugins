# PresentationDocument.FindWhereUsed Method

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Method that obtains the set of documents that reference the given file within this document.

## Syntax

PresentationDocument.**FindWhereUsed**( ***FullFileName*** As String ) As [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input String that specifies the full file name of the file to search for references to. |

## Version

Introduced in version 5
