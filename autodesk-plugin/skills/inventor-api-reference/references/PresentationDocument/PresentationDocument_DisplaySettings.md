# PresentationDocument.DisplaySettings Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the DisplaySettings object. The DisplaySettings object provides access to various display appearance related document settings.

## Syntax

PresentationDocument.**DisplaySettings**() As [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Property Value

This is a read only property whose value is a [DisplaySettings](../DisplaySettings/DisplaySettings.md).

## Version

Introduced in version 2011
