# PresentationDocument.Open Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns whether this document is currently open. If False, the document has only been initialized and calling methods or properties on the document could cause it to open.

## Syntax

PresentationDocument.**Open**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
