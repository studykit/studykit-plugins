# PresentationDocument.UnitsOfMeasure Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the UnitsOfMeasure object.

## Syntax

PresentationDocument.**UnitsOfMeasure**() As [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md)

## Property Value

This is a read only property whose value is a [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md).

## Version

Introduced in version 4
