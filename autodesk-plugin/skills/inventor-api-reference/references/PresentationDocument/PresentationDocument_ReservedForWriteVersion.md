# PresentationDocument.ReservedForWriteVersion Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Gets the version within this file that has been reserved for write.

## Syntax

PresentationDocument.**ReservedForWriteVersion**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
