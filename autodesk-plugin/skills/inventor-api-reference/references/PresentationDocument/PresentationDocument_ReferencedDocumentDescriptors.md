# PresentationDocument.ReferencedDocumentDescriptors Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns an enumeration of descriptors that represent the native document references held by this document.

## Syntax

PresentationDocument.**ReferencedDocumentDescriptors**() As [DocumentDescriptorsEnumerator](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptorsEnumerator](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator.md).

## Version

Introduced in version 11
