# PresentationDocument.ReservedForWriteName Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Gets the name of the person who currently holds the reservation to write.

## Syntax

PresentationDocument.**ReservedForWriteName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
