# PresentationDocument.Scenes Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Read-only property that returns the PresentationScenes collection object.

## Syntax

PresentationDocument.**Scenes**() As [PresentationScenes](../PresentationScenes/PresentationScenes.md)

## Property Value

This is a read only property whose value is a [PresentationScenes](../PresentationScenes/PresentationScenes.md).

## Version

Introduced in version 2018
