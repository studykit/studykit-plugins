# PresentationDocument.DocumentEvents Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

This object supports a set of events that are specific to the document.

## Syntax

PresentationDocument.**DocumentEvents**() As [DocumentEvents](../DocumentEvents/DocumentEvents.md)

## Property Value

This is a read only property whose value is a [DocumentEvents](../DocumentEvents/DocumentEvents.md).

## Version

Introduced in version 4
