# PresentationDocument.SelectSet Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the SelectSet object.

## Syntax

PresentationDocument.**SelectSet**() As [SelectSet](../SelectSet/SelectSet.md)

## Property Value

This is a read only property whose value is a [SelectSet](../SelectSet/SelectSet.md).

## Version

Introduced in version 5
