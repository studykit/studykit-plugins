# PresentationDocument.AllReferencedDocuments Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns all the document references of this Document along with all of the recursively nested references.

## Syntax

PresentationDocument.**AllReferencedDocuments**() As [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md).

## Version

Introduced in version 11
