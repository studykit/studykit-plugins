# FeaturePatternElementProxy.NativeObject Property

Parent Object: [FeaturePatternElementProxy](../FeaturePatternElementProxy/FeaturePatternElementProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FeaturePatternElementProxy.**NativeObject**() As [FeaturePatternElement](../FeaturePatternElement/FeaturePatternElement.md)

## Property Value

This is a read only property whose value is a [FeaturePatternElement](../FeaturePatternElement/FeaturePatternElement.md).

## Version

Introduced in version 9
