# SketchSplineHandle3DProxy.StartSketchPoint Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Property that returns the that defines the position of the start of the line.

## Syntax

SketchSplineHandle3DProxy.**StartSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 2008
