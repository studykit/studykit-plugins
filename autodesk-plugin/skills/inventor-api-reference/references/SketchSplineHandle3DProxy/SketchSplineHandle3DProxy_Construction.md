# SketchSplineHandle3DProxy.Construction Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchSplineHandle3DProxy.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
