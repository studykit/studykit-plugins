# SketchSplineHandle3DProxy.Parent Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchSplineHandle3DProxy.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2008
