# SketchSplineHandle3DProxy.NativeObject Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchSplineHandle3DProxy.**NativeObject**() As [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md)

## Property Value

This is a read only property whose value is a [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md).

## Version

Introduced in version 2008
