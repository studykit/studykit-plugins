# SketchSplineHandle3DProxy.Tangent Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Gets and sets the tangent vector at the fit point associated with this handle.

## Syntax

SketchSplineHandle3DProxy.**Tangent**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2008
