# SketchSplineHandle3DProxy.Length Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchSplineHandle3DProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
