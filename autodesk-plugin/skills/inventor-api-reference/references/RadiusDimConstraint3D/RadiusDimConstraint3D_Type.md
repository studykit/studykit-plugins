# RadiusDimConstraint3D.Type Property

Parent Object: [RadiusDimConstraint3D](../RadiusDimConstraint3D/RadiusDimConstraint3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RadiusDimConstraint3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
