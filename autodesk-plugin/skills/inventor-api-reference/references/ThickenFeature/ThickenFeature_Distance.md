# ThickenFeature.Distance Property

Parent Object: [ThickenFeature](../ThickenFeature/ThickenFeature.md)

## Description

Property that returns the parameter that controls the thickness of the Thickness feature or the distance of the Offset feature.

## Syntax

ThickenFeature.**Distance**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 9
