# ThickenFeature.Operation Property

Parent Object: [ThickenFeature](../ThickenFeature/ThickenFeature.md)

## Description

Gets and sets the type of operation used to add the feature to the model.

## Syntax

ThickenFeature.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 9
