# PartComponentDefinitions.Item Property

Parent Object: [PartComponentDefinitions](../PartComponentDefinitions/PartComponentDefinitions.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PartComponentDefinitions.**Item**( ***Index*** As Long ) As [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Property Value

This is a read only property whose value is a [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the PartComponentDefinition to return. |

## Version

Introduced in version 4
