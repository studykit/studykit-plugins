# PartComponentDefinitions.Type Property

Parent Object: [PartComponentDefinitions](../PartComponentDefinitions/PartComponentDefinitions.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

PartComponentDefinitions.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 4
