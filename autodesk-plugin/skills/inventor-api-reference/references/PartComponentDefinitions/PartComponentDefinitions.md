# PartComponentDefinitions Object

## Description

The Part ComponentDefinitions object represents a collection of objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../PartComponentDefinitions/PartComponentDefinitions_Count.md) | Property that returns the number of items in this collection. |
| [Item](../PartComponentDefinitions/PartComponentDefinitions_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../PartComponentDefinitions/PartComponentDefinitions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
