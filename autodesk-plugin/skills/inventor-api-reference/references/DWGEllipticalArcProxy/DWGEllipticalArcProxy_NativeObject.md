# DWGEllipticalArcProxy.NativeObject Property

Parent Object: [DWGEllipticalArcProxy](../DWGEllipticalArcProxy/DWGEllipticalArcProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGEllipticalArcProxy.**NativeObject**() As [DWGEllipticalArc](../DWGEllipticalArc/DWGEllipticalArc.md)

## Property Value

This is a read only property whose value is a [DWGEllipticalArc](../DWGEllipticalArc/DWGEllipticalArc.md).

## Version

Introduced in version 2016
