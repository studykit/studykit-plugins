# DWGEllipticalArcProxy.Center Property

Parent Object: [DWGEllipticalArcProxy](../DWGEllipticalArcProxy/DWGEllipticalArcProxy.md)

## Description

Read-only property that returns the position of the center of the DWGEllipticalArc.

## Syntax

DWGEllipticalArcProxy.**Center**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2016
