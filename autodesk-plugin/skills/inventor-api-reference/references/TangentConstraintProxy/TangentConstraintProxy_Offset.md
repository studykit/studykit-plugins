# TangentConstraintProxy.Offset Property

Parent Object: [TangentConstraintProxy](../TangentConstraintProxy/TangentConstraintProxy.md)

## Description

Property that returns the Parameter object that controls the offset distance of the constraint.

## Syntax

TangentConstraintProxy.**Offset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 11
