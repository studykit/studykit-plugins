# TangentConstraintProxy.Type Property

Parent Object: [TangentConstraintProxy](../TangentConstraintProxy/TangentConstraintProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TangentConstraintProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
