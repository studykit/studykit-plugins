# ImportedModelEntity Object

## Description

ImportedModelEntity Object.

## Properties

|  |  |
| --- | --- |

| [EntityType](../ImportedModelEntity/ImportedModelEntity_EntityType.md) | Read-only property that returns the ImportedModelEntityTypeEnum indicating the type of the ImportedModelEntity. |

| [Name](../ImportedModelEntity/ImportedModelEntity_Name.md) | Read-only property that returns the name of this object. |
| [Parent](../ImportedModelEntity/ImportedModelEntity_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ParentEntity](../ImportedModelEntity/ImportedModelEntity_ParentEntity.md) | Read-only property that returns the parent ImportedModelEntity object. |
| [SubEntities](../ImportedModelEntity/ImportedModelEntity_SubEntities.md) | Read-only property that returns the sub ImportedModelEntities collection. |
| [Type](../ImportedModelEntity/ImportedModelEntity_Type.md) | Gets the constant that indicates the type of this object. |
