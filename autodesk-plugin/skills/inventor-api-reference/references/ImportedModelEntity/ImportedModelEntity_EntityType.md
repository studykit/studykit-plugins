# ImportedModelEntity.EntityType Property

Parent Object: [ImportedModelEntity](../ImportedModelEntity/ImportedModelEntity.md)

## Description

Read-only property that returns the ImportedModelEntityTypeEnum indicating the type of the ImportedModelEntity.

## Syntax

ImportedModelEntity.**EntityType**() As [ImportedModelEntityTypeEnum](../ImportedModelEntityTypeEnum.md)

## Property Value

This is a read only property whose value is an [ImportedModelEntityTypeEnum](../ImportedModelEntityTypeEnum.md).

## Version

Introduced in version 2016
