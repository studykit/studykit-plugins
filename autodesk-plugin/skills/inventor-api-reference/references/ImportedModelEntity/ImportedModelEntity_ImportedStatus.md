# ImportedModelEntity.ImportedStatus Property

Parent Object: [ImportedModelEntity](../ImportedModelEntity/ImportedModelEntity.md)

## Description

Read-write property that gets and set the imported status for the entity.

## Syntax

ImportedModelEntity.**ImportedStatus**() As [ImportedModelEntityStatusEnum](../ImportedModelEntityStatusEnum.md)

## Property Value

This is a read/write property whose value is an [ImportedModelEntityStatusEnum](../ImportedModelEntityStatusEnum.md).

## Version

Introduced in version 2016
