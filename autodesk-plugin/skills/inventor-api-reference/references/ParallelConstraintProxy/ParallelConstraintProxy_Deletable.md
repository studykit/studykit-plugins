# ParallelConstraintProxy.Deletable Property

Parent Object: [ParallelConstraintProxy](../ParallelConstraintProxy/ParallelConstraintProxy.md)

## Description

Indicates whether this object is deletable.

## Syntax

ParallelConstraintProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
