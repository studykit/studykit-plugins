# DrawingBOMCell Object

## Description

The DrawingBOMCell object represents a cell within the drawing BOM.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingBOMCell/DrawingBOMCell_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../DrawingBOMCell/DrawingBOMCell_Parent.md) | Property that returns the parent DrawingBOM object. |
| [Static](../DrawingBOMCell/DrawingBOMCell_Static.md) | Property that returns whether or not the contents of this cell are static. |
| [Type](../DrawingBOMCell/DrawingBOMCell_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Value](../DrawingBOMCell/DrawingBOMCell_Value.md) | Property that returns the value of the cell. |

## Accessed From

[DrawingBOMRow.Item](../DrawingBOMRow/DrawingBOMRow_Item.md)

## Version

Introduced in version 2009
