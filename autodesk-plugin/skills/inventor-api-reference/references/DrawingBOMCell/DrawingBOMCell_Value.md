# DrawingBOMCell.Value Property

Parent Object: [DrawingBOMCell](../DrawingBOMCell/DrawingBOMCell.md)

## Description

Property that returns the value of the cell.

## Syntax

DrawingBOMCell.**Value**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2009
