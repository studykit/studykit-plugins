# FeatureDimensionTypeEnum Enumerator

## Description

Enum indicating the type of dimensioning to use while placing linear dimensions.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAlongEdgeFeatureDimension | 93191 | AlongEdge feature dimension type. |
| kAngleFeatureDimension | 93185 | Angle feature dimension type. |
| kCircularCountFeatureDimension | 93186 | CircularCount feature dimension type. |
| kHoleFeatureDimension | 93187 | Hole feature dimension type. |
| kLinearFeatureDimension | 93188 | Linear feature dimension type. |
| kRadialFeatureDimension | 93189 | Radial feature dimension type. |
| kRectangularCountFeatureDimension | 93190 | RectangularCount feature dimension type. |

## Version

Introduced in version 2011
