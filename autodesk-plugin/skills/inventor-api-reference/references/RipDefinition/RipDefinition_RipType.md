# RipDefinition.RipType Property

Parent Object: [RipDefinition](../RipDefinition/RipDefinition.md)

## Description

Property that returns the method used to define the type of rip. The valid values for this property are kSinglePointRipType, kPointToPointRipType, and kFaceExtentsRipType.

## Syntax

RipDefinition.**RipType**() As [RipTypeEnum](../RipTypeEnum.md)

## Property Value

This is a read only property whose value is a [RipTypeEnum](../RipTypeEnum.md).

## Version

Introduced in version 2011
