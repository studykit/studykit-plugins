# RipDefinition.SetFaceExtentsRipType Method

Parent Object: [RipDefinition](../RipDefinition/RipDefinition.md)

## Description

Method that sets the RipDefinition so that it defines a rip that uses the entire rip face to determine the extent of the rip.

## Syntax

RipDefinition.**SetFaceExtentsRipType**( ***RipFace*** As [Face](../Face/Face.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| RipFace | [Face](../Face/Face.md) | The face that defines the extent of the rip. This must be a face this is valid for defining a rip extent. For example a face along the edge of the part where the thickness of the part is represented is not valid as input. |

## Version

Introduced in version 2011
