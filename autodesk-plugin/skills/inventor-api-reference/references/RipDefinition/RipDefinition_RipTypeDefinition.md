# RipDefinition.RipTypeDefinition Property

Parent Object: [RipDefinition](../RipDefinition/RipDefinition.md)

## Description

Property that returns the RipTypeDefinition object that defines the type of rip.

## Syntax

RipDefinition.**RipTypeDefinition**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2011
