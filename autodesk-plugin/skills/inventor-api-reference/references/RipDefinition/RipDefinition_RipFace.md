# RipDefinition.RipFace Property

Parent Object: [RipDefinition](../RipDefinition/RipDefinition.md)

## Description

Gets and sets the face that the rip is defined along.

## Syntax

RipDefinition.**RipFace**() As [Face](../Face/Face.md)

## Property Value

This is a read/write property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2011
