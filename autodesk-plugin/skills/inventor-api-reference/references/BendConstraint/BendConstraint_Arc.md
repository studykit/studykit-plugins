# BendConstraint.Arc Property

Parent Object: [BendConstraint](../BendConstraint/BendConstraint.md)

## Description

Property that returns the SketchArc3D object associated with the bend.

## Syntax

BendConstraint.**Arc**() As [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Property Value

This is a read only property whose value is a [SketchArc3D](../SketchArc3D/SketchArc3D.md).

## Version

Introduced in version 6
