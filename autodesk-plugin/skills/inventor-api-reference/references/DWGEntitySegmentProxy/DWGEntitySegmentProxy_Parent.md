# DWGEntitySegmentProxy.Parent Property

Parent Object: [DWGEntitySegmentProxy](../DWGEntitySegmentProxy/DWGEntitySegmentProxy.md)

## Description

Read-only property that returns the parent DWGEntity.

## Syntax

DWGEntitySegmentProxy.**Parent**() As [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Property Value

This is a read only property whose value is an [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md).

## Version

Introduced in version 2018
