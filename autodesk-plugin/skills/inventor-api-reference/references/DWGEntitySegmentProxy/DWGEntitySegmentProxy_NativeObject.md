# DWGEntitySegmentProxy.NativeObject Property

Parent Object: [DWGEntitySegmentProxy](../DWGEntitySegmentProxy/DWGEntitySegmentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGEntitySegmentProxy.**NativeObject**() As [DWGEntitySegment](../DWGEntitySegment/DWGEntitySegment.md)

## Property Value

This is a read only property whose value is a [DWGEntitySegment](../DWGEntitySegment/DWGEntitySegment.md).

## Version

Introduced in version 2018
