# DrawingHatchPatternsManager.Parent Property

Parent Object: [DrawingHatchPatternsManager](../DrawingHatchPatternsManager/DrawingHatchPatternsManager.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DrawingHatchPatternsManager.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 2021
