# BIMDuctConnectorDefinition.SystemType Property

Parent Object: [BIMDuctConnectorDefinition](../BIMDuctConnectorDefinition/BIMDuctConnectorDefinition.md)

## Description

Read-write property that specifies the system type for this connector.

## Syntax

BIMDuctConnectorDefinition.**SystemType**() As [BIMDuctSystemTypeEnum](../BIMDuctSystemTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BIMDuctSystemTypeEnum](../BIMDuctSystemTypeEnum.md).

## Version

Introduced in version 2011
