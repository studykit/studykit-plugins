# BIMDuctConnectorDefinition.Origin Property

Parent Object: [BIMDuctConnectorDefinition](../BIMDuctConnectorDefinition/BIMDuctConnectorDefinition.md)

## Description

Read-only property that returns the origin of the connector.

## Syntax

BIMDuctConnectorDefinition.**Origin**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2014
