# BIMDuctConnectorDefinition.ExposeFlowValueAsParameter Property

Parent Object: [BIMDuctConnectorDefinition](../BIMDuctConnectorDefinition/BIMDuctConnectorDefinition.md)

## Description

Read-write property that gets and sets if expose the flow value as parameter.

## Syntax

BIMDuctConnectorDefinition.**ExposeFlowValueAsParameter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
