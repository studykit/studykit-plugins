# BIMDuctConnectorDefinition.ConnectionType Property

Parent Object: [BIMDuctConnectorDefinition](../BIMDuctConnectorDefinition/BIMDuctConnectorDefinition.md)

## Description

Read-write property that gets and sets the connection type for this connector.

## Syntax

BIMDuctConnectorDefinition.**ConnectionType**() As [BIMDuctConnectionTypeEnum](../BIMDuctConnectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BIMDuctConnectionTypeEnum](../BIMDuctConnectionTypeEnum.md).

## Version

Introduced in version 2011
