# BIMDuctConnectorDefinition.ExposeLossValueAsParameter Property

Parent Object: [BIMDuctConnectorDefinition](../BIMDuctConnectorDefinition/BIMDuctConnectorDefinition.md)

## Description

Read-write property that gets and sets if expose the loss value as parameter.

## Syntax

BIMDuctConnectorDefinition.**ExposeLossValueAsParameter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
