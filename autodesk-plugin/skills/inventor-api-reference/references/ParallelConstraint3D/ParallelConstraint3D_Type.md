# ParallelConstraint3D.Type Property

Parent Object: [ParallelConstraint3D](../ParallelConstraint3D/ParallelConstraint3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ParallelConstraint3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
