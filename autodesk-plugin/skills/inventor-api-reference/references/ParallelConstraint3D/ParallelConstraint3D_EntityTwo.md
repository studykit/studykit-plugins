# ParallelConstraint3D.EntityTwo Property

Parent Object: [ParallelConstraint3D](../ParallelConstraint3D/ParallelConstraint3D.md)

## Description

Property that returns the second entity being constrained.

## Syntax

ParallelConstraint3D.**EntityTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
