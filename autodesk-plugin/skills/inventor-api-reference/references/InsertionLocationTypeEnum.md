# InsertionLocationTypeEnum Enumerator

## Description

Input the InsertionLocationTypeEnum indicating where to position the DragNodes.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kInsertAfter | 1 | The insertion location is after the target node. |
| kInsertBefore | 0 | The insertion location is before the target node. |
| kInsertUnder | 2 | The insertion location is under the target node. |

## Version

Introduced in version 2018
