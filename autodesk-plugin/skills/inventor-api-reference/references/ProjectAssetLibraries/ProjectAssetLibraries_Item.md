# ProjectAssetLibraries.Item Property

Parent Object: [ProjectAssetLibraries](../ProjectAssetLibraries/ProjectAssetLibraries.md)

## Description

Returns the specified ProjectAssetLibrary object from the collection.

## Syntax

ProjectAssetLibraries.**Item**( ***Index*** As Long ) As [ProjectAssetLibrary](../ProjectAssetLibrary/ProjectAssetLibrary.md)

## Property Value

This is a read only property whose value is a [ProjectAssetLibrary](../ProjectAssetLibrary/ProjectAssetLibrary.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the ProjectAssetLibrary to return. The first item in the collection has a index of 1. |

## Version

Introduced in version 2014
