# ModelSurfaceTextureSymbolProxy.IsMemberOfCompositeAnnotation Property

Parent Object: [ModelSurfaceTextureSymbolProxy](../ModelSurfaceTextureSymbolProxy/ModelSurfaceTextureSymbolProxy.md)

## Description

Returns whether this annotation is a member of a ModelCompositeAnnotation.

## Syntax

ModelSurfaceTextureSymbolProxy.**IsMemberOfCompositeAnnotation**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
