# SketchGeometricConstraintTypeEnum Enumerator

## Description

Constants indicating constraint types.

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kCoincident | 1 | Coincident constraint. |
| kColinear | 4 | Colinear constraint. |
| kConcentric | 2 | Concentric constraint. |
| kEqual | 8 | Equal constraint. |
| kFix | 16 | Fix constraint. |
| kHorizontal | 32 | Horizontal constraint. |
| kParallel | 64 | Parallel constraint. |
| kPerpendicular | 128 | Perpenticular constraint. |
| kSmooth | 256 | Smooth constraint. |

| kVertical | 2048 | Vertical constraint. |
