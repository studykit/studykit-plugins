# LoftRail.Condition Property

Parent Object: [LoftRail](../LoftRail/LoftRail.md)

## Description

Returns an enum indicating the condition of the loft at the rail if the rail has conditions associated with it.

## Syntax

LoftRail.**Condition**() As [LoftConditionEnum](../LoftConditionEnum.md)

## Property Value

This is a read/write property whose value is a [LoftConditionEnum](../LoftConditionEnum.md).

## Version

Introduced in version 11
