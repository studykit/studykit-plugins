# LoftRail.Impact Property

Parent Object: [LoftRail](../LoftRail/LoftRail.md)

## Description

Property that returns the Parameter object that controls the amount of influence the rail has on the shape of the entire loft.

## Syntax

LoftRail.**Impact**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 11
