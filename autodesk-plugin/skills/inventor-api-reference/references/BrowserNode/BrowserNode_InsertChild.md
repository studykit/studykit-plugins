# BrowserNode.InsertChild Method

Parent Object: [BrowserNode](../BrowserNode/BrowserNode.md)

## Description

Method that creates a new BrowserNode at the specified location. The new BrowserNode is returned.

## Syntax

BrowserNode.**InsertChild**( ***Definition*** As [BrowserNodeDefinition](../BrowserNodeDefinition/BrowserNodeDefinition.md), ***TargetNode*** As [BrowserNode](../BrowserNode/BrowserNode.md), ***InsertBefore*** As Boolean ) As [BrowserNode](../BrowserNode/BrowserNode.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [BrowserNodeDefinition](../BrowserNodeDefinition/BrowserNodeDefinition.md) | Input object that defines the input definition for the browser node. |
| TargetNode | [BrowserNode](../BrowserNode/BrowserNode.md) | Input BrowserNode object that specifies the node adjacent to which the new node should be inserted. |
| InsertBefore | Boolean | Input Boolean that specifies whether the new node should be inserted before or after the target node. |

## Version

Introduced in version 2011
