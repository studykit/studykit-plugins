# BrowserNode.AddChild Method

Parent Object: [BrowserNode](../BrowserNode/BrowserNode.md)

## Description

Method that creates a new as a nested child of this one. The new BrowserNode is returned.

## Remarks

It is not possible to add client nodes to built-in panes.

## Syntax

BrowserNode.**AddChild**( ***Definition*** As [BrowserNodeDefinition](../BrowserNodeDefinition/BrowserNodeDefinition.md) ) As [BrowserNode](../BrowserNode/BrowserNode.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [BrowserNodeDefinition](../BrowserNodeDefinition/BrowserNodeDefinition.md) | Input object that specifies the input definition for the browser node. |

## Version

Introduced in version 8
