# BrowserNode.BrowserNodes Property

Parent Object: [BrowserNode](../BrowserNode/BrowserNode.md)

## Description

Property that returns a collection of the top level BrowserNode objects contained under this node.

## Syntax

BrowserNode.**BrowserNodes**() As [BrowserNodesEnumerator](../BrowserNodesEnumerator/BrowserNodesEnumerator.md)

## Property Value

This is a read only property whose value is a [BrowserNodesEnumerator](../BrowserNodesEnumerator/BrowserNodesEnumerator.md).

## Version

Introduced in version 8
