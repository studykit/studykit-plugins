# Wire.EdgeUses Property

Parent Object: [Wire](../Wire/Wire.md)

## Description

Property that returns the EdgeUses collection object associated with this Wire.

## Syntax

Wire.**EdgeUses**() As [EdgeUses](../EdgeUses/EdgeUses.md)

## Property Value

This is a read only property whose value is an [EdgeUses](../EdgeUses/EdgeUses.md).

## Version

Introduced in version 2008
