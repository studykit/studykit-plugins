# ObjectsEnumerator Object

## Description

Generic enumerator for a group of objects. This enumerator will also support the IEnumxxx-style interface (IEnumUnknown, unless otherwise specified).

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../ObjectsEnumerator/ObjectsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ObjectsEnumerator/ObjectsEnumerator_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../ObjectsEnumerator/ObjectsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
