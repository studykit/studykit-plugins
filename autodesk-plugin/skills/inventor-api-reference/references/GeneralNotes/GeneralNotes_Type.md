# GeneralNotes.Type Property

Parent Object: [GeneralNotes](../GeneralNotes/GeneralNotes.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

GeneralNotes.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
