# GeneralNotes.Item Property

Parent Object: [GeneralNotes](../GeneralNotes/GeneralNotes.md)

## Description

Returns the specified GeneralNote object from the collection.

## Syntax

GeneralNotes.**Item**( ***Index*** As Long ) As [GeneralNote](../GeneralNote/GeneralNote.md)

## Property Value

This is a read only property whose value is a [GeneralNote](../GeneralNote/GeneralNote.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |
