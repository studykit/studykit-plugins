# ComponentDefinition.OrientedMinimumRangeBox Property

Parent Object: [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Description

Read-only property that returns the oriented minimum range box for this object.

## Syntax

ComponentDefinition.**OrientedMinimumRangeBox**() As [OrientedBox](../OrientedBox/OrientedBox.md)

## Property Value

This is a read only property whose value is an [OrientedBox](../OrientedBox/OrientedBox.md).

## Version

Introduced in version 2024
