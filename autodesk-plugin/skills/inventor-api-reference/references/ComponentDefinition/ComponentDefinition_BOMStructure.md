# ComponentDefinition.BOMStructure Property

Parent Object: [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Description

Gets and sets how the component is used/viewed in a BOM.

## Syntax

ComponentDefinition.**BOMStructure**() As [BOMStructureEnum](../BOMStructureEnum.md)

## Property Value

This is a read/write property whose value is a [BOMStructureEnum](../BOMStructureEnum.md).

## Version

Introduced in version 10
