# ComponentDefinition Object

## Description

The ComponentDefinition object is the base class for the PartComponentDefinition, AssemblyComponentDefinition and the FlatPattern objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [FindUsingPoint](../ComponentDefinition/ComponentDefinition_FindUsingPoint.md) | Method that finds all the entities of the specified type at the specified location. |
| [FindUsingVector](../ComponentDefinition/ComponentDefinition_FindUsingVector.md) | Method that finds all the entities of the specified type along the specified vector using either a cylinder or cone that to define the tolerance within the defined vector. |
| [GetUnusedGeometries](../ComponentDefinition/ComponentDefinition_GetUnusedGeometries.md) | Method that gets the unused sketches and work features. |
| [PurgeUnusedGeometries](../ComponentDefinition/ComponentDefinition_PurgeUnusedGeometries.md) | Method that purges unused sketches and work features. |
| [RepositionObject](../ComponentDefinition/ComponentDefinition_RepositionObject.md) | Method that repositions the specifies object(s) to the new position within the collection of the object in the document. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ComponentDefinition/ComponentDefinition_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../ComponentDefinition/ComponentDefinition_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [BOMQuantity](../ComponentDefinition/ComponentDefinition_BOMQuantity.md) | Property that returns the BOMQuantity object. |
| [BOMStructure](../ComponentDefinition/ComponentDefinition_BOMStructure.md) | Gets and sets how the component is used/viewed in a BOM. |
| [ClientGraphicsCollection](../ComponentDefinition/ComponentDefinition_ClientGraphicsCollection.md) | Property that returns the ClientGraphicsCollection object. |
| [DataIO](../ComponentDefinition/ComponentDefinition_DataIO.md) | Gets the object that directly deals with I/O to and from a storage-medium, including Streams(IStream). |
| [Document](../ComponentDefinition/ComponentDefinition_Document.md) | Property that returns the containing Document object. |
| [ModelGeometryVersion](../ComponentDefinition/ComponentDefinition_ModelGeometryVersion.md) | Property that returns a string that can be used to determine if the document has been modified. This version string is changed every time the assembly is modified. By saving a previous version string, you can compare the current version string to see if the assembly has been modified. |
| [Occurrences](../ComponentDefinition/ComponentDefinition_Occurrences.md) | Property that returns the collection object. |
| [OrientedMinimumRangeBox](../ComponentDefinition/ComponentDefinition_OrientedMinimumRangeBox.md) | Read-only property that returns the oriented minimum range box for this object. |
| [PreciseRangeBox](../ComponentDefinition/ComponentDefinition_PreciseRangeBox.md) | Gets a bounding box that tightly encloses all the solid and surface bodies under the ComponentDefinition. |
| [RangeBox](../ComponentDefinition/ComponentDefinition_RangeBox.md) | Property that returns a Box object which contains the opposing points of a rectangular box that is guaranteed to enclose this object. |
| [SurfaceBodies](../ComponentDefinition/ComponentDefinition_SurfaceBodies.md) | Property that returns all of the *result* SurfaceBody objects contained within this ComponentDefinition. |
| [Type](../ComponentDefinition/ComponentDefinition_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
