# SketchBlockDefinitionsEnumerator.Type Property

Parent Object: [SketchBlockDefinitionsEnumerator](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchBlockDefinitionsEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
