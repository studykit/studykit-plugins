# ImportedRVTComponentDefinition.Parent Property

Parent Object: [ImportedRVTComponentDefinition](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ImportedRVTComponentDefinition.**Parent**() As [ImportedComponent](../ImportedComponent/ImportedComponent.md)

## Property Value

This is a read only property whose value is an [ImportedComponent](../ImportedComponent/ImportedComponent.md).

## Version

Introduced in version 2021
