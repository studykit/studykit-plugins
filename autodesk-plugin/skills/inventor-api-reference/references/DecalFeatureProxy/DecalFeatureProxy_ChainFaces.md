# DecalFeatureProxy.ChainFaces Property

Parent Object: [DecalFeatureProxy](../DecalFeatureProxy/DecalFeatureProxy.md)

## Description

Specifies if the decal is also applied on any faces that are adjacent to the primary face.

## Syntax

DecalFeatureProxy.**ChainFaces**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
