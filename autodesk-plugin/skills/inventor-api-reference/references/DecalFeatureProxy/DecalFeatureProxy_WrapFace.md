# DecalFeatureProxy.WrapFace Property

Parent Object: [DecalFeatureProxy](../DecalFeatureProxy/DecalFeatureProxy.md)

## Description

Specifies if the decal is wrapped on the faces on which it is applied.

## Syntax

DecalFeatureProxy.**WrapFace**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
