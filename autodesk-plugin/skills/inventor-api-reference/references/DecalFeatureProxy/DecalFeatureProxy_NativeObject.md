# DecalFeatureProxy.NativeObject Property

Parent Object: [DecalFeatureProxy](../DecalFeatureProxy/DecalFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DecalFeatureProxy.**NativeObject**() As [DecalFeature](../DecalFeature/DecalFeature.md)

## Property Value

This is a read only property whose value is a [DecalFeature](../DecalFeature/DecalFeature.md).

## Version

Introduced in version 9
