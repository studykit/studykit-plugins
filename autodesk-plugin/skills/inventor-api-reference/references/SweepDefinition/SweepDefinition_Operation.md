# SweepDefinition.Operation Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the type of operation used to add the feature to the model.

## Syntax

SweepDefinition.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2015
