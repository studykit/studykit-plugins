# SweepDefinition.ProfileOrientation Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the orientation of sweep profile. This property is applicable when the SweepType is kPathSweepType and kPathAndSectionTwistSweepType.

## Syntax

SweepDefinition.**ProfileOrientation**() As [SweepProfileOrientationEnum](../SweepProfileOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [SweepProfileOrientationEnum](../SweepProfileOrientationEnum.md).

## Version

Introduced in version 2015
