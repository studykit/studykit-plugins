# SweepDefinition.Type Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SweepDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
