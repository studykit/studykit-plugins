# SweepDefinition.GuideRail Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the guide rail for the sweep which is the guide curve that controls the scaling and twist of the sweep profile. This property is applicable when the SweepType is kPathAndGuideRailSweepType.

## Syntax

SweepDefinition.**GuideRail**() As [Path](../Path/Path.md)

## Property Value

This is a read/write property whose value is a [Path](../Path/Path.md).

## Version

Introduced in version 2015
