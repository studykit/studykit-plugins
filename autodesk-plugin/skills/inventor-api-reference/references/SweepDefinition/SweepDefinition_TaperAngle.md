# SweepDefinition.TaperAngle Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the taper angle of sweep. This property is applicable when the SweepType is kPathSweepType and kPathAndSectionTwistSweepType and when the ProfileOrientation is kNormalToPath.

## Syntax

SweepDefinition.**TaperAngle**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2015
