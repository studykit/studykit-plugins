# RotateTranslateiMateDefinitionProxy.NativeObject Property

Parent Object: [RotateTranslateiMateDefinitionProxy](../RotateTranslateiMateDefinitionProxy/RotateTranslateiMateDefinitionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RotateTranslateiMateDefinitionProxy.**NativeObject**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version

Introduced in version 6
