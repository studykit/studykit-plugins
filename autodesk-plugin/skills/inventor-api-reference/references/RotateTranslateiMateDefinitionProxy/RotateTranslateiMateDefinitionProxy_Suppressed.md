# RotateTranslateiMateDefinitionProxy.Suppressed Property

Parent Object: [RotateTranslateiMateDefinitionProxy](../RotateTranslateiMateDefinitionProxy/RotateTranslateiMateDefinitionProxy.md)

## Description

Gets the boolean flag that indicates whether this iMateDefinition is suppressed or not.

## Syntax

RotateTranslateiMateDefinitionProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
