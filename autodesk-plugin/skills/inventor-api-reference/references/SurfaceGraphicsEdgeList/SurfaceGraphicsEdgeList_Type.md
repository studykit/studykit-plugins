# SurfaceGraphicsEdgeList.Type Property

## Syntax

SurfaceGraphicsEdgeList.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
