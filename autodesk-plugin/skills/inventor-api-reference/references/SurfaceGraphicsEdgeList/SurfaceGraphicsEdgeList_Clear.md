# SurfaceGraphicsEdgeList.Clear Method

Parent Object: [SurfaceGraphicsEdgeList](../SurfaceGraphicsEdgeList/SurfaceGraphicsEdgeList.md)

## Description

Method that removes all the edges from the list. No edges will be displayed after the method is called.

## Syntax

SurfaceGraphicsEdgeList.**Clear**()

## Version

Introduced in version 2009
