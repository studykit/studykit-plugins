# SurfaceGraphicsEdgeList.Count Property

Parent Object: [SurfaceGraphicsEdgeList](../SurfaceGraphicsEdgeList/SurfaceGraphicsEdgeList.md)

## Description

Property that returns the number of SurfaceGraphicsEdge objects in the list.

## Syntax

SurfaceGraphicsEdgeList.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2009
