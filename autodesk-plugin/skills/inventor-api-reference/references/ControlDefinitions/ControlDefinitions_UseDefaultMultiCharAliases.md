# ControlDefinitions.UseDefaultMultiCharAliases Property

Parent Object: [ControlDefinitions](../ControlDefinitions/ControlDefinitions.md)

## Description

Gets/Sets whether the multi-character Command Aliases delivered with Inventor should be used. The equivalent UI option is found under the 'Keyboard' tab of the Customize dialog.

## Syntax

ControlDefinitions.**UseDefaultMultiCharAliases**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
