# ControlDefinitions.Application Property

Parent Object: [ControlDefinitions](../ControlDefinitions/ControlDefinitions.md)

## Description

Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.

## Syntax

ControlDefinitions.**Application**() As [Application](../Application/Application.md)

## Property Value

This is a read only property whose value is an [Application](../Application/Application.md).

## Version

Introduced in version 7
