# BOMQuantity.Parent Property

Parent Object: [BOMQuantity](../BOMQuantity/BOMQuantity.md)

## Description

Property that returns the parent ComponentDefinition object.

## Syntax

BOMQuantity.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 10
