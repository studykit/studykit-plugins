# BOMQuantity.UnitQuantity Property

Parent Object: [BOMQuantity](../BOMQuantity/BOMQuantity.md)

## Description

Property that returns the unit quantity derived from two other properties, the BaseQuantity and the BaseUnits.

## Syntax

BOMQuantity.**UnitQuantity**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 10
