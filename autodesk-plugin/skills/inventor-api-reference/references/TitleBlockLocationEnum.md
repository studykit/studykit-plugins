# TitleBlockLocationEnum Enumerator

## Description

Constants identifying the position of the title block on the sheet.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBottomLeftPosition | 29443 | Constant identifying the bottom left corner of the sheet. |
| kBottomRightPosition | 29444 | Constant identifying the bottom right corner of the sheet. |
| kTopLeftPosition | 29441 | Constant identifying the top left corner of the sheet. |
| kTopRightPosition | 29442 | Constant identifying the top right corner of the sheet. |

## Version

Introduced in version 5.3
