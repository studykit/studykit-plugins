# CoilFeature.FlatStartType Property

Parent Object: [CoilFeature](../CoilFeature/CoilFeature.md)

## Description

Property that gets and sets the coil end type at the start of the coil feature.

## Syntax

CoilFeature.**FlatStartType**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
