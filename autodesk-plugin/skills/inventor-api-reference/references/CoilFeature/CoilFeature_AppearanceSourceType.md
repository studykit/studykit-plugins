# CoilFeature.AppearanceSourceType Property

Parent Object: [CoilFeature](../CoilFeature/CoilFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

CoilFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
