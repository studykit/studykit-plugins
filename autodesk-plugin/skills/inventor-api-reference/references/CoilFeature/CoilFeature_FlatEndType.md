# CoilFeature.FlatEndType Property

Parent Object: [CoilFeature](../CoilFeature/CoilFeature.md)

## Description

Property that gets and sets the coil end type at the end of the coil feature.

## Syntax

CoilFeature.**FlatEndType**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
