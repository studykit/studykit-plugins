# CoilFeature.Operation Property

Parent Object: [CoilFeature](../CoilFeature/CoilFeature.md)

## Description

Gets and sets the type of operation used to add the feature to the model.

## Syntax

CoilFeature.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 5.3
