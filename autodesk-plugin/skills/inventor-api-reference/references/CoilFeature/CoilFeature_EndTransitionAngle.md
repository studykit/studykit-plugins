# CoilFeature.EndTransitionAngle Property

Parent Object: [CoilFeature](../CoilFeature/CoilFeature.md)

## Description

Property that returns the parameter that controls the transition angle at the end of the coil feature. This property returns Nothing if ExtentType is kSpiral or FlatEndType is False.

## Syntax

CoilFeature.**EndTransitionAngle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 5.3
