# AutoCADBlockDefinition.GetPromptTags Method

Parent Object: [AutoCADBlockDefinition](../AutoCADBlockDefinition/AutoCADBlockDefinition.md)

## Description

Method that returns the tags and their corresponding prompts for field text (attributes) within the block definition.

## Syntax

AutoCADBlockDefinition.**GetPromptTags**( ***Tags*** As Variant, [***Prompts***] As Variant )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Tags | Variant | Output array of Strings that returns the unique tag strings indicating field texts in the block definition. |
| Prompts | Variant | Optional array of Strings that returns the corresponding prompt strings for the tags. |

## Version

Introduced in version 2011
