# WorkAxis.ReferencedEntity Property

Parent Object: [WorkAxis](../WorkAxis/WorkAxis.md)

## Description

Property indicating the object this entity is dependent on.

## Syntax

WorkAxis.**ReferencedEntity**() As [WorkAxis](../WorkAxis/WorkAxis.md)

## Property Value

This is a read only property whose value is a [WorkAxis](../WorkAxis/WorkAxis.md).

## Version

Introduced in version 5.3
