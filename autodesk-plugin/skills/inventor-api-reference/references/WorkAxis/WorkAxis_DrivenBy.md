# WorkAxis.DrivenBy Property

Parent Object: [WorkAxis](../WorkAxis/WorkAxis.md)

## Description

Property that returns the collection of objects that the work axis is dependent on.

## Syntax

WorkAxis.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
