# SketchesEnumerator.Item Property

Parent Object: [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md)

## Description

Returns the specified Sketch object from the collection.

## Syntax

SketchesEnumerator.**Item**( ***Index*** As Long ) As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the index of the Sketch to return. |

## Version

Introduced in version 5.3
