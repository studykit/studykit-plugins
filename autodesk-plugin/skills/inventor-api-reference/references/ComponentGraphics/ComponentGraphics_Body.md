# ComponentGraphics.Body Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Read-only property that returns the SurfaceBody object associated with the SurfaceGraphics.

## Syntax

ComponentGraphics.**Body**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2011
