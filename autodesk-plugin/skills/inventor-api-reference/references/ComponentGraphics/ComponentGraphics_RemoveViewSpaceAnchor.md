# ComponentGraphics.RemoveViewSpaceAnchor Method

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

ComponentGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 2011
