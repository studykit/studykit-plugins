# ComponentGraphics.LineType Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Read-write property that gets and sets the line type associated with the primitive.

## Syntax

ComponentGraphics.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2011
