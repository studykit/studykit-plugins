# UserInterfaceEvents.Parent Property

Parent Object: [UserInterfaceEvents](../UserInterfaceEvents/UserInterfaceEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

UserInterfaceEvents.**Parent**() As [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Property Value

This is a read only property whose value is a [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md).

## Version

Introduced in version 10
