# UserInterfaceEvents.OnResetRibbonInterface Event

Parent Object: [UserInterfaceEvents](../UserInterfaceEvents/UserInterfaceEvents.md)

## Description

Event that is fired when the ribbon user interface is reset.

## Syntax

UserInterfaceEvents.**OnResetRibbonInterface**( ***Context*** As [NameValueMap](../NameValueMap/NameValueMap.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Context | [NameValueMap](../NameValueMap/NameValueMap.md) | Input NameValueMap object that contains additional information about the event. Currently, this argument does not contain any information. |

## Version

Introduced in version 2010
