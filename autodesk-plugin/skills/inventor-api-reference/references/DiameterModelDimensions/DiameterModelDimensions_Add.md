# DiameterModelDimensions.Add Method

Parent Object: [DiameterModelDimensions](../DiameterModelDimensions/DiameterModelDimensions.md)

## Description

Method that creates a diameter dimension.

## Syntax

DiameterModelDimensions.**Add**( ***Definition*** As [DiameterModelDimensionDefinition](../DiameterModelDimensionDefinition/DiameterModelDimensionDefinition.md) ) As [DiameterModelDimension](../DiameterModelDimension/DiameterModelDimension.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [DiameterModelDimensionDefinition](../DiameterModelDimensionDefinition/DiameterModelDimensionDefinition.md) | Input DiameterModelDimensionDefinition object that defines the dimension to be created. |

## Version

Introduced in version 2018
