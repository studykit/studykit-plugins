# TransitionSymbol.Position Property

Parent Object: [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md)

## Description

Gets and sets the position of the transition symbol on the sheet.

## Syntax

TransitionSymbol.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2025.1
