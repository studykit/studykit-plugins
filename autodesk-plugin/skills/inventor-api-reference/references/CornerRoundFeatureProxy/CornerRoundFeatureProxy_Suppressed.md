# CornerRoundFeatureProxy.Suppressed Property

Parent Object: [CornerRoundFeatureProxy](../CornerRoundFeatureProxy/CornerRoundFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

CornerRoundFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
