# CornerRoundFeatureProxy.Adaptive Property

Parent Object: [CornerRoundFeatureProxy](../CornerRoundFeatureProxy/CornerRoundFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

CornerRoundFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
