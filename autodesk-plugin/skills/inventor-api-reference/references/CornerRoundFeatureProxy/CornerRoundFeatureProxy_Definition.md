# CornerRoundFeatureProxy.Definition Property

Parent Object: [CornerRoundFeatureProxy](../CornerRoundFeatureProxy/CornerRoundFeatureProxy.md)

## Description

Property that gets/sets the CornerRoundDefinition object associated with this fold feature.

## Syntax

CornerRoundFeatureProxy.**Definition**() As [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md)

## Property Value

This is a read only property whose value is a [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md).

## Version

Introduced in version 2009
