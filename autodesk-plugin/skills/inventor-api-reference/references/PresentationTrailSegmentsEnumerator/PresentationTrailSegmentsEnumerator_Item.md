# PresentationTrailSegmentsEnumerator.Item Property

Parent Object: [PresentationTrailSegmentsEnumerator](../PresentationTrailSegmentsEnumerator/PresentationTrailSegmentsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationTrailSegmentsEnumerator.**Item**( ***Index*** As Variant ) As [PresentationTrailSegment](../PresentationTrailSegment/PresentationTrailSegment.md)

## Property Value

This is a read only property whose value is a [PresentationTrailSegment](../PresentationTrailSegment/PresentationTrailSegment.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of an item to return. |

## Version

Introduced in version 2018
