# ModelLeaderNoteDefinition.AnnotationPlaneDefinition Property

Parent Object: [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md)

## Description

Read-write property that gets and sets the annotation plane definition for the dimension.

## Syntax

ModelLeaderNoteDefinition.**AnnotationPlaneDefinition**() As [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md).

## Version

Introduced in version 2018
