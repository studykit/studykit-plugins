# CosmeticBendFeatureProxy.ExtendedName Property

Parent Object: [CosmeticBendFeatureProxy](../CosmeticBendFeatureProxy/CosmeticBendFeatureProxy.md)

## Description

Read-only property that returns the full feature name including any extended information.

## Syntax

CosmeticBendFeatureProxy.**ExtendedName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2012
