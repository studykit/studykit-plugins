# CosmeticBendFeatureProxy.Suppressed Property

Parent Object: [CosmeticBendFeatureProxy](../CosmeticBendFeatureProxy/CosmeticBendFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

CosmeticBendFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
