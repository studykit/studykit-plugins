# ViewFramesEnumerator.Application Property

Parent Object: [ViewFramesEnumerator](../ViewFramesEnumerator/ViewFramesEnumerator.md)

## Description

Read-only property that returns the root Application object.

## Syntax

ViewFramesEnumerator.**Application**() As [Application](../Application/Application.md)

## Property Value

This is a read only property whose value is an [Application](../Application/Application.md).

## Version

Introduced in version 2021
