# PresentationBodiesEnumerator.Item Property

Parent Object: [PresentationBodiesEnumerator](../PresentationBodiesEnumerator/PresentationBodiesEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationBodiesEnumerator.**Item**( ***Index*** As Long ) As [PresentationBody](../PresentationBody/PresentationBody.md)

## Property Value

This is a read only property whose value is a [PresentationBody](../PresentationBody/PresentationBody.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long to specify the index of the object to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
