# ContourFlangeDefinition.IsDistanceWidthExtentAsymmetric Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Read-only property that returns whether the distance width extent is asymmetric or not.

## Syntax

ContourFlangeDefinition.**IsDistanceWidthExtentAsymmetric**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2023
