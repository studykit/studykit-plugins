# ContourFlangeDefinition.ApplyAutoMitering Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Defines if auto mitering should be performed.

## Syntax

ContourFlangeDefinition.**ApplyAutoMitering**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
