# ContourFlangeDefinition.Parent Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Property that returns the parent ContourFlangeFeature of this ContourFlangeDefinition object.

## Syntax

ContourFlangeDefinition.**Parent**() As [ContourFlangeFeature](../ContourFlangeFeature/ContourFlangeFeature.md)

## Property Value

This is a read only property whose value is a [ContourFlangeFeature](../ContourFlangeFeature/ContourFlangeFeature.md).

## Version

Introduced in version 2009
