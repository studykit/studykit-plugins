# ContourFlangeDefinition.ExtentDirection Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Gets and sets the extent direction which indicates which side of the sketch the projection of the flange is in.

## Syntax

ContourFlangeDefinition.**ExtentDirection**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2009
