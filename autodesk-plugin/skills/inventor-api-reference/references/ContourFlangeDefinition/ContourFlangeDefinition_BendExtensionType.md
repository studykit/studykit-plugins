# ContourFlangeDefinition.BendExtensionType Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Read-write property that gets and sets the bend extension type. This is applicable when the BendEdges is configured.

## Syntax

ContourFlangeDefinition.**BendExtensionType**() As [BendExtensionTypeEnum](../BendExtensionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BendExtensionTypeEnum](../BendExtensionTypeEnum.md).

## Version

Introduced in version 2026
