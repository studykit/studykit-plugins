# ContourFlangeDefinition.DistanceWidthExtentTwo Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Read-only property that returns the PartFeatureExtent object that defines how the second distance width extent of the contour flange feature is defined.

## Syntax

ContourFlangeDefinition.**DistanceWidthExtentTwo**() As [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md).

## Version

Introduced in version 2023
