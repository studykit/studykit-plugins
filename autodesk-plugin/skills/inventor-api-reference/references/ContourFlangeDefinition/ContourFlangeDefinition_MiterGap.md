# ContourFlangeDefinition.MiterGap Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Gets and sets the miter gap size of the Contour flange feature.

## Syntax

ContourFlangeDefinition.**MiterGap**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2009
