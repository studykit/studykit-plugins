# DrawingSketch.Profiles Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Gets the profiles collection associated with this drawing sketch.

## Syntax

DrawingSketch.**Profiles**() As [Profiles](../Profiles/Profiles.md)

## Property Value

This is a read only property whose value is a [Profiles](../Profiles/Profiles.md).
