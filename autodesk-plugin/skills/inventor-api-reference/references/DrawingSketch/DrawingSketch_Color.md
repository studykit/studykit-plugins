# DrawingSketch.Color Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Gets and Sets the color for the sketch.

## Syntax

DrawingSketch.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2010
