# DrawingSketch.TextBoxes Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Gets the TextBoxes collection associated with this Sketch.

## Syntax

DrawingSketch.**TextBoxes**() As [TextBoxes](../TextBoxes/TextBoxes.md)

## Property Value

This is a read only property whose value is a [TextBoxes](../TextBoxes/TextBoxes.md).

## Version

Introduced in version 5.3
