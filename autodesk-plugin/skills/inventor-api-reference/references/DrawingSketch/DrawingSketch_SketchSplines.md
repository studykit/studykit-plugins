# DrawingSketch.SketchSplines Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Property that returns the SketchSplines collection object.

## Syntax

DrawingSketch.**SketchSplines**() As [SketchSplines](../SketchSplines/SketchSplines.md)

## Property Value

This is a read only property whose value is a [SketchSplines](../SketchSplines/SketchSplines.md).

## Version

Introduced in version 5.3
