# DrawingSketch.SketchArcs Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

DrawingSketch.**SketchArcs**() As [SketchArcs](../SketchArcs/SketchArcs.md)

## Property Value

This is a read only property whose value is a [SketchArcs](../SketchArcs/SketchArcs.md).

## Version

Introduced in version 5.3
