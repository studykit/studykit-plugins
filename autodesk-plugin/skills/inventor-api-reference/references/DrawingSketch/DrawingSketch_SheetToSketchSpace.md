# DrawingSketch.SheetToSketchSpace Method

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Method that takes a coordinate in sheet space returns a Point2d object containing the resulting coordinate point in sketch space.

## Syntax

DrawingSketch.**SheetToSketchSpace**( ***SheetCoordinate*** As [Point2d](../Point2d/Point2d.md) ) As [Point2d](../Point2d/Point2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SheetCoordinate | [Point2d](../Point2d/Point2d.md) | Input object of a point within sheet space. |

## Version

Introduced in version 5.3
