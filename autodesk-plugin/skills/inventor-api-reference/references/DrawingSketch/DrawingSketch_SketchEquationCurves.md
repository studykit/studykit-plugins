# DrawingSketch.SketchEquationCurves Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Read-only property that returns the SketchEquationCurves collection object. This collection provides access to the existing equation curves in the sketch and provides functionality to create new equation curves.

## Syntax

DrawingSketch.**SketchEquationCurves**() As [SketchEquationCurves](../SketchEquationCurves/SketchEquationCurves.md)

## Property Value

This is a read only property whose value is a [SketchEquationCurves](../SketchEquationCurves/SketchEquationCurves.md).

## Version

Introduced in version 2014
