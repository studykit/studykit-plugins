# DrawingSketch.SketchFixedSplines Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Property that gets the collection object.

## Syntax

DrawingSketch.**SketchFixedSplines**() As [SketchFixedSplines](../SketchFixedSplines/SketchFixedSplines.md)

## Property Value

This is a read only property whose value is a [SketchFixedSplines](../SketchFixedSplines/SketchFixedSplines.md).

## Version

Introduced in version 9
