# DWGPointProxy.NativeObject Property

Parent Object: [DWGPointProxy](../DWGPointProxy/DWGPointProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGPointProxy.**NativeObject**() As [DWGPoint](../DWGPoint/DWGPoint.md)

## Property Value

This is a read only property whose value is a [DWGPoint](../DWGPoint/DWGPoint.md).

## Version

Introduced in version 2016
