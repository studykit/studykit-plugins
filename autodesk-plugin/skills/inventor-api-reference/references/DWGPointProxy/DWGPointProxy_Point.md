# DWGPointProxy.Point Property

Parent Object: [DWGPointProxy](../DWGPointProxy/DWGPointProxy.md)

## Description

Read-only property that returns a Point indicating the position of the DWGPoint in model space.

## Syntax

DWGPointProxy.**Point**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2016
