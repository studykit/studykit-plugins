# Centerline.Parent Property

Parent Object: [Centerline](../Centerline/Centerline.md)

## Description

Property that returns the parent Sheet object.

## Syntax

Centerline.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 2008
