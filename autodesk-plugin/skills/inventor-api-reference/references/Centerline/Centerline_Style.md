# Centerline.Style Property

Parent Object: [Centerline](../Centerline/Centerline.md)

## Description

Gets and sets the style associated with this centerline.

## Syntax

Centerline.**Style**() As [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Property Value

This is a read/write property whose value is a [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md).

## Version

Introduced in version 2010
