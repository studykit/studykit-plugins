# Centerline.StartPoint Property

Parent Object: [Centerline](../Centerline/Centerline.md)

## Description

Read-write property that gets and sets the start point of the centerline.

## Syntax

Centerline.**StartPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2009
