# SketchControlPointSplineProxy.Layer Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Gets and sets the layer applied to this sketch entity.

## Syntax

SketchControlPointSplineProxy.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2014
