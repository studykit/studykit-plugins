# SketchControlPointSplineProxy.CurvatureDisplay Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Specifies whether curvature information is displayed for the spline.

## Syntax

SketchControlPointSplineProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
