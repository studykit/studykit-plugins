# SketchControlPointSplineProxy.LineDefinitionSpace Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Gets and sets the LineDefinitionSpace applied to this sketch spline.

## Syntax

SketchControlPointSplineProxy.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2014
