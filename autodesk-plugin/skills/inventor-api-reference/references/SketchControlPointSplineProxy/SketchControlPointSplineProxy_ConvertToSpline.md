# SketchControlPointSplineProxy.ConvertToSpline Method

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Method that inserts a new control point into the existing control point spline.

## Syntax

SketchControlPointSplineProxy.**ConvertToSpline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Version

Introduced in version 2014
