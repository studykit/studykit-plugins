# SketchControlPointSplineProxy.LineWeight Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Gets and sets the LineWeight applied to this entity.

## Syntax

SketchControlPointSplineProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
