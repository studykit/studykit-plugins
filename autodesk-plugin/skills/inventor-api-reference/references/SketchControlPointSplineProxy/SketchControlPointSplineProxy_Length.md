# SketchControlPointSplineProxy.Length Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Returns the length of the entity in centimeters.

## Syntax

SketchControlPointSplineProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2014
