# SketchControlPointSplineProxy.ControlPointCount Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Read-only property that returns the number of control points defining the control point spline.

## Syntax

SketchControlPointSplineProxy.**ControlPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2014
