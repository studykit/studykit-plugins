# VertexProxy.Parent Property

Parent Object: [VertexProxy](../VertexProxy/VertexProxy.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

VertexProxy.**Parent**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 5
