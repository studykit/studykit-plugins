# DiameterGeneralDimension.Layer Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Gets and sets the layer applied to this dimension.

## Syntax

DiameterGeneralDimension.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 11
