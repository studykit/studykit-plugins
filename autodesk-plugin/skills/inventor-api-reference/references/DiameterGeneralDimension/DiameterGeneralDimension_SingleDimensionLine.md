# DiameterGeneralDimension.SingleDimensionLine Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Gets and sets whether to use a single dimension line.

## Syntax

DiameterGeneralDimension.**SingleDimensionLine**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
