# DiameterGeneralDimension.LeaderFromCenter Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Gets and sets whether the leader starts from the center of the arc or the circle.

## Syntax

DiameterGeneralDimension.**LeaderFromCenter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
