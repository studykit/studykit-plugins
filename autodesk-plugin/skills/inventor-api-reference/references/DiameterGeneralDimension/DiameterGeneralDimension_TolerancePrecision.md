# DiameterGeneralDimension.TolerancePrecision Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Gets and sets the precision of the tolerance text for the dimension.

## Syntax

DiameterGeneralDimension.**TolerancePrecision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2008
