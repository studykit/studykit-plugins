# DrawingStandardStyle.FrontViewPlane Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets the viewing plane to use as front view when creating a drawing view.

## Syntax

DrawingStandardStyle.**FrontViewPlane**() As [FrontViewPlaneEnum](../FrontViewPlaneEnum.md)

## Property Value

This is a read/write property whose value is a [FrontViewPlaneEnum](../FrontViewPlaneEnum.md).

## Version

Introduced in version 2010
