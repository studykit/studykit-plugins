# DrawingStandardStyle.GetViewAnnotationDefaults Method

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Method that gets the drawing view annotation defaults for the specified view type.

## Syntax

DrawingStandardStyle.**GetViewAnnotationDefaults**( ***ViewType*** As [DrawingViewTypeEnum](../DrawingViewTypeEnum.md), ***FormattedText*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ViewType | [DrawingViewTypeEnum](../DrawingViewTypeEnum.md) | Input DrawingViewTypeEnum value that specifies the view type for which to get the annotation defaults. Valid values are kAuxiliaryDrawingViewType, kDetailDrawingViewType and kSectionDrawingViewType, |
| FormattedText | String | Output String that gets the content of the view annotation defaults. |

## Version

Introduced in version 2026
