# DrawingStandardStyle.InternationalStandardReference Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Property that returns the International Standard on which this standard style is based.

## Syntax

DrawingStandardStyle.**InternationalStandardReference**() As [DraftingStandardEnum](../DraftingStandardEnum.md)

## Property Value

This is a read only property whose value is a [DraftingStandardEnum](../DraftingStandardEnum.md).

## Version

Introduced in version 2010
