# DrawingStandardStyle.ApplyExcludeCharactersToBendTables Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets whether to exclude characters specified in the ExcludeCharacters property from bend tables and its associated tags.

## Syntax

DrawingStandardStyle.**ApplyExcludeCharactersToBendTables**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
