# DrawingStandardStyle.StyleType Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets the type of the style.

## Syntax

DrawingStandardStyle.**StyleType**() As [StyleTypeEnum](../StyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [StyleTypeEnum](../StyleTypeEnum.md).

## Version

Introduced in version 2008
