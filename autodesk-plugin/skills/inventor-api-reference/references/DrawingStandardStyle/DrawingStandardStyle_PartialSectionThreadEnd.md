# DrawingStandardStyle.PartialSectionThreadEnd Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets thread edge display type for sectional views.

## Syntax

DrawingStandardStyle.**PartialSectionThreadEnd**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
