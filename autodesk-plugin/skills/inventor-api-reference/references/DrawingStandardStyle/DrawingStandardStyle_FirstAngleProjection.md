# DrawingStandardStyle.FirstAngleProjection Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets whether the projection type is first angle or third angle projection.

## Syntax

DrawingStandardStyle.**FirstAngleProjection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
