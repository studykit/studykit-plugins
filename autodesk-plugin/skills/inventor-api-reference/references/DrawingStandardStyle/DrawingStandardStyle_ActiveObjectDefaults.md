# DrawingStandardStyle.ActiveObjectDefaults Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Returns the active ObjectDefaultsStyle object.

## Syntax

DrawingStandardStyle.**ActiveObjectDefaults**() As [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Property Value

This is a read/write property whose value is an [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md).

## Version

Introduced in version 9
