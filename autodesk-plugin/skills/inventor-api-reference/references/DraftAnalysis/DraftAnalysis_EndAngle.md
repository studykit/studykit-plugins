# DraftAnalysis.EndAngle Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Gets/Sets the end angle for the draft analysis.

## Syntax

DraftAnalysis.**EndAngle**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
