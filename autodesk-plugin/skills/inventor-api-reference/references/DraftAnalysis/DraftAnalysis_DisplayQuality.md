# DraftAnalysis.DisplayQuality Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Gets/Sets the resolution or surface quality for the color gradient or color bands that represent the draft analysis results.

## Syntax

DraftAnalysis.**DisplayQuality**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2009
