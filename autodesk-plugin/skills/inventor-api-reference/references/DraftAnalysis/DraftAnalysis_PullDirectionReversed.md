# DraftAnalysis.PullDirectionReversed Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Gets/Sets whether the pull direction of the draft should be reversed.

## Syntax

DraftAnalysis.**PullDirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
