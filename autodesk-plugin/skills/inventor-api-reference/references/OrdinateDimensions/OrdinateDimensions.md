# OrdinateDimensions Object

## Description

The OrdinateDimensions collection object provides access to all of the ordinate dimensions on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../OrdinateDimensions/OrdinateDimensions_Add.md) | Method that creates an ordinate dimension. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../OrdinateDimensions/OrdinateDimensions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../OrdinateDimensions/OrdinateDimensions_Count.md) | Property that returns the number of items in the collection. |
| [Item](../OrdinateDimensions/OrdinateDimensions_Item.md) | Method that returns the specified dimension object from the collection. |
| [Type](../OrdinateDimensions/OrdinateDimensions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingDimensions.OrdinateDimensions](../DrawingDimensions/DrawingDimensions_OrdinateDimensions.md)

## Version

Introduced in version 2009
