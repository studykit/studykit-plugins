# OrdinateDimensions.Type Property

Parent Object: [OrdinateDimensions](../OrdinateDimensions/OrdinateDimensions.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

OrdinateDimensions.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2009
