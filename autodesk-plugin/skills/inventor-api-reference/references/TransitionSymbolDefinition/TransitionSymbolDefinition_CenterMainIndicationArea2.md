# TransitionSymbolDefinition.CenterMainIndicationArea2 Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets and sets the center main indication area value for combined maximum material and least material radius boundary specification.

## Syntax

TransitionSymbolDefinition.**CenterMainIndicationArea2**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025.1
