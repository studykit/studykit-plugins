# TransitionSymbolDefinition.UpperMainIndicationArea Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets and sets the upper main indication area value.

## Syntax

TransitionSymbolDefinition.**UpperMainIndicationArea**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025.1
