# TransitionSymbolDefinition.Style Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets and sets the transition symbol style associated with this object.

## Syntax

TransitionSymbolDefinition.**Style**() As [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Property Value

This is a read/write property whose value is a [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md).

## Version

Introduced in version 2025.1
