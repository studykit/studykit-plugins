# TransitionSymbolDefinition.CombinedMaximumAndLeastMaterial Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets and sets whether to combine maximum material and least material radius boundary specification for the transition symbol.

## Syntax

TransitionSymbolDefinition.**CombinedMaximumAndLeastMaterial**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
