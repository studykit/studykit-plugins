# TransitionSymbolDefinition.LowerMainIndicationArea Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets and sets the lower main indication area value.

## Syntax

TransitionSymbolDefinition.**LowerMainIndicationArea**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025.1
