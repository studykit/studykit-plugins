# MarkDefinition.Direction Property

Parent Object: [MarkDefinition](../MarkDefinition/MarkDefinition.md)

## Description

Read-write property that gets and sets the extent direction. Set this to kSymmetricExtentDirection is invalid when the method is set to kWrapToFaceMethodType.

## Syntax

MarkDefinition.**Direction**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2024
