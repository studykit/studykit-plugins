# MarkDefinition.MarkGeometrySetCount Property

Parent Object: [MarkDefinition](../MarkDefinition/MarkDefinition.md)

## Description

Property returns the number of mark geometry sets currently defined in the definition.

## Syntax

MarkDefinition.**MarkGeometrySetCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2023
