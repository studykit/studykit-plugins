# MarkDefinition.MarkGeometrySetItem Method

Parent Object: [MarkDefinition](../MarkDefinition/MarkDefinition.md)

## Description

Method that returns the specified MarkGeometrySet object from the collection.

## Syntax

MarkDefinition.**MarkGeometrySetItem**( ***Index*** As Long ) As [MarkGeometrySet](../MarkGeometrySet/MarkGeometrySet.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the MarkGeometrySet to return. |

## Version

Introduced in version 2023
