# RefoldFeatures Object

## Description

Object allowing iteration of the collection of RefoldFeature objects, and allows creation of RefoldFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../RefoldFeatures/RefoldFeatures_Add.md) | Method that creates a new refold feature. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RefoldFeatures/RefoldFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../RefoldFeatures/RefoldFeatures_Count.md) | Property that returns the number of RefoldFeature objects in the collection. |
| [Item](../RefoldFeatures/RefoldFeatures_Item.md) | Returns the specified RefoldFeature object from the collection. |
| [Type](../RefoldFeatures/RefoldFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SheetMetalFeatures.RefoldFeatures](../SheetMetalFeatures/SheetMetalFeatures_RefoldFeatures.md)

## Version

Introduced in version 2010
