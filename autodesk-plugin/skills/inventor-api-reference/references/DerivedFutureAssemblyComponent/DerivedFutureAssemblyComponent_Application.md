# DerivedFutureAssemblyComponent.Application Property

Parent Object: [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md)

## Description

Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.

## Syntax

DerivedFutureAssemblyComponent.**Application**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
