# DerivedFutureAssemblyComponent.SuppressAll Property

Parent Object: [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md)

## Description

Read-write property that gets and sets the suppress state for all of the Reference features created by this derived assembly component.

## Syntax

DerivedFutureAssemblyComponent.**SuppressAll**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
