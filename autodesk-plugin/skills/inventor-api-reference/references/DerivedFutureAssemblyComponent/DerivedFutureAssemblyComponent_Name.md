# DerivedFutureAssemblyComponent.Name Property

Parent Object: [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md)

## Description

Property that returns the component's name.

## Syntax

DerivedFutureAssemblyComponent.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
