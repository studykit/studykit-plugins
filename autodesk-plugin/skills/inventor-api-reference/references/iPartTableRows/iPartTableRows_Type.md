# iPartTableRows.Type Property

Parent Object: [iPartTableRows](../iPartTableRows/iPartTableRows.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iPartTableRows.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
