# TwoPointDistanceDimConstraint3D.AnchorPoints Property

Parent Object: [TwoPointDistanceDimConstraint3D](../TwoPointDistanceDimConstraint3D/TwoPointDistanceDimConstraint3D.md)

## Description

Gets the anchor points of dimension.

## Syntax

TwoPointDistanceDimConstraint3D.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
