# CustomParameterGroups.Type Property

Parent Object: [CustomParameterGroups](../CustomParameterGroups/CustomParameterGroups.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CustomParameterGroups.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
