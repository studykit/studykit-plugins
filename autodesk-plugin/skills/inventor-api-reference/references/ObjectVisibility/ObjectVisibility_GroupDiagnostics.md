# ObjectVisibility.GroupDiagnostics Property

Parent Object: [ObjectVisibility](../ObjectVisibility/ObjectVisibility.md)

## Description

Gets and sets whether group diagnostics should be visible in the construction environment of parts.

## Syntax

ObjectVisibility.**GroupDiagnostics**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
