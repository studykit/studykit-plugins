# NonParametricBaseFeature.IsComposite Property

Parent Object: [NonParametricBaseFeature](../NonParametricBaseFeature/NonParametricBaseFeature.md)

## Description

Property that indicates if the non-parametric base feature is a composite of multiple solids and/or surfaces or is a single solid or surface.

## Syntax

NonParametricBaseFeature.**IsComposite**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
