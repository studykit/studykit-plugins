# NonParametricBaseFeature.InputSurfaceBodies Property

Parent Object: [NonParametricBaseFeature](../NonParametricBaseFeature/NonParametricBaseFeature.md)

## Description

Property that returns the underlying SurfaceBody object associated with this feature.

## Syntax

NonParametricBaseFeature.**InputSurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).

## Version

Introduced in version 2009
