# ShrinkwrapComponents Object

## Description

The ShrinkwrapComponents collection object provides access to all of the existing ShrinkwrapComponent objects in a part and provides methods to create additional shrinkwrap components.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ShrinkwrapComponents/ShrinkwrapComponents_Add.md) | Creates a new ShrinkwrapComponent using the information supplied by the input ShrinkwrapDefinition object. |
| [CreateDefinition](../ShrinkwrapComponents/ShrinkwrapComponents_CreateDefinition.md) | Creates a new ShrinkwrapDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ShrinkwrapComponents/ShrinkwrapComponents_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ShrinkwrapComponents/ShrinkwrapComponents_Count.md) | Gets the number of items in this collection. |
| [Item](../ShrinkwrapComponents/ShrinkwrapComponents_Item.md) | Method that returns the specified ShrinkwrapComponent object from the collection. |
| [Type](../ShrinkwrapComponents/ShrinkwrapComponents_Type.md) | Read-only property returning kShrinkwrapComponentsObject indicating the type of object. |

## Accessed From

[ReferenceComponents.ShrinkwrapComponents](../ReferenceComponents/ReferenceComponents_ShrinkwrapComponents.md)

## Version

Introduced in version 2018
