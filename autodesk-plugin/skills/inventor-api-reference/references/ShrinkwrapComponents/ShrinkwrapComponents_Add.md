# ShrinkwrapComponents.Add Method

Parent Object: [ShrinkwrapComponents](../ShrinkwrapComponents/ShrinkwrapComponents.md)

## Description

Creates a new ShrinkwrapComponent using the information supplied by the input ShrinkwrapDefinition object.

## Syntax

ShrinkwrapComponents.**Add**( ***Definition*** As [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md) ) As [ShrinkwrapComponent](../ShrinkwrapComponent/ShrinkwrapComponent.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md) | Input ShrinkwrapDefinition object that defines the shrinkwrap component. |

## Version

Introduced in version 2019
