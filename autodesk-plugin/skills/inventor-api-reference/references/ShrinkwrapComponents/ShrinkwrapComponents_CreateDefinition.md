# ShrinkwrapComponents.CreateDefinition Method

Parent Object: [ShrinkwrapComponents](../ShrinkwrapComponents/ShrinkwrapComponents.md)

## Description

Creates a new ShrinkwrapDefinition object.

## Syntax

ShrinkwrapComponents.**CreateDefinition**( ***FullDocumentName*** As String ) As [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullDocumentName | String | Input string that specifies the name of the file to create the definition for. The file must be an existing assembly document. If an invalid file is specified an error will occur. |

## Version

Introduced in version 2019
