# ShrinkwrapComponents.Type Property

Parent Object: [ShrinkwrapComponents](../ShrinkwrapComponents/ShrinkwrapComponents.md)

## Description

Read-only property returning kShrinkwrapComponentsObject indicating the type of object.

## Syntax

ShrinkwrapComponents.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2018
