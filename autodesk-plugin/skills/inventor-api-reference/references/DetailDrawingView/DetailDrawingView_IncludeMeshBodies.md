# DetailDrawingView.IncludeMeshBodies Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that gets and sets whether to include the mesh bodies.

## Syntax

DetailDrawingView.**IncludeMeshBodies**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
