# DetailDrawingView.Scale Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets or sets the Model-to-PaperSpace scale of this particular Drawing View.

## Syntax

DetailDrawingView.**Scale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
