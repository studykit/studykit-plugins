# DetailDrawingView.InheritBreak Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets and sets whether the view should inherit the corresponding break from the parent view.

## Syntax

DetailDrawingView.**InheritBreak**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
