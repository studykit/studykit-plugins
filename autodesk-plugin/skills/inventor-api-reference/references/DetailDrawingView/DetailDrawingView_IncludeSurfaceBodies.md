# DetailDrawingView.IncludeSurfaceBodies Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that gets and sets whether to include the work surfaces.

## Syntax

DetailDrawingView.**IncludeSurfaceBodies**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
