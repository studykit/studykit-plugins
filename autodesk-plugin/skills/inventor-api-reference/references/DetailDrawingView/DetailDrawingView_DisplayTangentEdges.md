# DetailDrawingView.DisplayTangentEdges Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that gets and sets whether to display the tangent edges on the drawing view.

## Syntax

DetailDrawingView.**DisplayTangentEdges**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
