# DetailDrawingView.Width Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that specifies the width of the drawing view. The view width cannot be set but is defined by the contents of the view.

## Syntax

DetailDrawingView.**Width**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
