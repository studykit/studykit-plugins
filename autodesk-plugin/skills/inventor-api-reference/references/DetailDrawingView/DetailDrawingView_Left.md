# DetailDrawingView.Left Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that returns the position of the left edge of the drawing view.

## Syntax

DetailDrawingView.**Left**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 11
