# DetailDrawingView.IsUpdateComplete Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Indicates if the view has finished updating.

## Syntax

DetailDrawingView.**IsUpdateComplete**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
