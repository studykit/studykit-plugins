# DetailDrawingView.HiddenLineCalculationForAllBodies Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that gets and sets whether the hidden line are calculated for all bodies or reference data separately.

## Syntax

DetailDrawingView.**HiddenLineCalculationForAllBodies**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
