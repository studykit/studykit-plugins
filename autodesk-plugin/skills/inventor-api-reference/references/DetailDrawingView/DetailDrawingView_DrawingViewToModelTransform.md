# DetailDrawingView.DrawingViewToModelTransform Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that returns the transformation from drawing view coordinate space to the model space.

## Syntax

DetailDrawingView.**DrawingViewToModelTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 11
