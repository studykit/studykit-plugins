# DetailDrawingView.FenceRadius Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that gets and sets the radius of the circular fence. This property fails if the fence type is rectangular.

## Syntax

DetailDrawingView.**FenceRadius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
