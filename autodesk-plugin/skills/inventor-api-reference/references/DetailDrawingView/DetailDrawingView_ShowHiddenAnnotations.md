# DetailDrawingView.ShowHiddenAnnotations Method

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Method that displays all the annotations hidden by the user.

## Syntax

DetailDrawingView.**ShowHiddenAnnotations**()

## Version

Introduced in version 11
