# DetailDrawingView.IsTextPropertySource Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that gets and sets whether the drawing view is specified as the text property source.This is applicable only when the drawing view is a base view.

## Syntax

DetailDrawingView.**IsTextPropertySource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
