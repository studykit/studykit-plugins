# DetailDrawingView.GeneralDimensionType Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that indicates the type of dimension.

## Syntax

DetailDrawingView.**GeneralDimensionType**() As [GeneralDimensionTypeEnum](../GeneralDimensionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [GeneralDimensionTypeEnum](../GeneralDimensionTypeEnum.md).

## Version

Introduced in version 2008
