# DetailDrawingView.Label Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that returns the DrawingViewLabel object.

## Syntax

DetailDrawingView.**Label**() As [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Property Value

This is a read only property whose value is a [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md).

## Version

Introduced in version 2009
