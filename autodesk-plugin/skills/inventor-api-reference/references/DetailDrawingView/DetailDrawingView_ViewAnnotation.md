# DetailDrawingView.ViewAnnotation Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-only property that returns the drawing view annotation object.

## Syntax

DetailDrawingView.**ViewAnnotation**() As [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md)

## Property Value

This is a read only property whose value is a [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md).

## Version

Introduced in version 2026
