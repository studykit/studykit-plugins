# DetailDrawingView.InsertInModelSpace Method

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Inserts the view geometry into model space only if the drawing lives in an Inventor DWG file.

## Syntax

DetailDrawingView.**InsertInModelSpace**()

## Version

Introduced in version 2012
