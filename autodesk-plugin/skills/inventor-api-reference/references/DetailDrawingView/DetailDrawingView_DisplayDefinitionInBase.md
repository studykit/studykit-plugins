# DetailDrawingView.DisplayDefinitionInBase Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets and sets whether to display detail circles, section lines and the associated text in the base view.

## Syntax

DetailDrawingView.**DisplayDefinitionInBase**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
