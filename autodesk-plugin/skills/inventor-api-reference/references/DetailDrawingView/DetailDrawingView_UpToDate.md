# DetailDrawingView.UpToDate Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that returns whether the drawing view is up to date with respect to the model.

## Syntax

DetailDrawingView.**UpToDate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
