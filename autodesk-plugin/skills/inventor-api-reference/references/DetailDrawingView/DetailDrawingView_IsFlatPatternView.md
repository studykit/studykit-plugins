# DetailDrawingView.IsFlatPatternView Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets whether the drawing view is of a sheet metal flat pattern.

## Syntax

DetailDrawingView.**IsFlatPatternView**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
