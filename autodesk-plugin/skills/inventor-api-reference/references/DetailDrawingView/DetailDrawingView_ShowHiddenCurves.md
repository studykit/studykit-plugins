# DetailDrawingView.ShowHiddenCurves Method

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Method that displays all the curves explicitly hidden by the user.

## Syntax

DetailDrawingView.**ShowHiddenCurves**()

## Version

Introduced in version 11
