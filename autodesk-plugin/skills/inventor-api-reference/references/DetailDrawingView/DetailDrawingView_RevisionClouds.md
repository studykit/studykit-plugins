# DetailDrawingView.RevisionClouds Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Returns the collection of revision clouds on this drawing view.

## Syntax

DetailDrawingView.**RevisionClouds**() As [RevisionClouds](../RevisionClouds/RevisionClouds.md)

## Property Value

This is a read only property whose value is a [RevisionClouds](../RevisionClouds/RevisionClouds.md).

## Version

Introduced in version 2024
