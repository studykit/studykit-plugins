# ThreadFeature.DisplayInModel Property

Parent Object: [ThreadFeature](../ThreadFeature/ThreadFeature.md)

## Description

Property that gets and sets whether the threads are displayed on the part when viewed in any of the 3d environments, i.e. part or assembly.

## Syntax

ThreadFeature.**DisplayInModel**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
