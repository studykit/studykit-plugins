# ThreadFeature.StartEdge Property

Parent Object: [ThreadFeature](../ThreadFeature/ThreadFeature.md)

## Description

Property that returns the thread depth is measured from.

## Syntax

ThreadFeature.**StartEdge**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 5
