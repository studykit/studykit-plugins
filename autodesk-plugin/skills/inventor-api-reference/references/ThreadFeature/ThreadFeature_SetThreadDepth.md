# ThreadFeature.SetThreadDepth Method

Parent Object: [ThreadFeature](../ThreadFeature/ThreadFeature.md)

## Description

Method that sets the thread depth for the thread feature.

## Syntax

ThreadFeature.**SetThreadDepth**( ***ThreadDepth*** As Variant, ***StartEdge*** As [Edge](../Edge/Edge.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ThreadDepth | Variant | Input Variant that specifies the total depth of the thread, as measured from its starting edge. |
| StartEdge | [Edge](../Edge/Edge.md) | Input Edge that specifies the point at which to begin thread. |

## Version

Introduced in version 6
