# SketchLine3DProxy.Length Property

Parent Object: [SketchLine3DProxy](../SketchLine3DProxy/SketchLine3DProxy.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchLine3DProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
