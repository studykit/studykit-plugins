# SketchLine3DProxy.ReferenceComponent Property

Parent Object: [SketchLine3DProxy](../SketchLine3DProxy/SketchLine3DProxy.md)

## Description

Property that returns the ReferenceComponent that resulted in the creation of this feature.

## Syntax

SketchLine3DProxy.**ReferenceComponent**() As [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md)

## Property Value

This is a read only property whose value is a [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md).

## Version

Introduced in version 6
