# SketchLine3DProxy.StartSketchPoint Property

Parent Object: [SketchLine3DProxy](../SketchLine3DProxy/SketchLine3DProxy.md)

## Description

Property that returns the that defines the position of the start of the line.

## Syntax

SketchLine3DProxy.**StartSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 8
