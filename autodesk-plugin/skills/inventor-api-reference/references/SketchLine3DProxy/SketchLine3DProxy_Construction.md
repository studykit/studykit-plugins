# SketchLine3DProxy.Construction Property

Parent Object: [SketchLine3DProxy](../SketchLine3DProxy/SketchLine3DProxy.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchLine3DProxy.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
