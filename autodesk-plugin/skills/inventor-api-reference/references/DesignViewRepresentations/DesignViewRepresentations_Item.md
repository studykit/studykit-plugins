# DesignViewRepresentations.Item Property

Parent Object: [DesignViewRepresentations](../DesignViewRepresentations/DesignViewRepresentations.md)

## Description

Returns the specified DesignViewRepresentation object from the collection.

## Syntax

DesignViewRepresentations.**Item**( ***Index*** As Variant ) As [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Property Value

This is a read only property whose value is a [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the DesignViewRepresentation to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the DesignViewRepresentation name. If an out of range index or a name of a non-existent DesignViewRepresentation is provided, an error will occur . |

## Version

Introduced in version 11
