# CombineFeature.KeepToolBodies Property

Parent Object: [CombineFeature](../CombineFeature/CombineFeature.md)

## Description

Property that returns whether the tool bodies were retained during the creation of the combine feature.

## Syntax

CombineFeature.**KeepToolBodies**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
