# MeshFeaturesSuppressionEnum Enumerator

## Description

Defines the suppression types of the mesh features.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllMeshFeaturesSuppressed | 111617 | All mesh features are suppressed. |
| kNoMeshFeaturesSuppressed | 111618 | No mesh features are suppressed. |
| kSomeMeshFeaturesSuppressed | 111619 | Some mesh features are suppressed. |

## Version

Introduced in version 2017
