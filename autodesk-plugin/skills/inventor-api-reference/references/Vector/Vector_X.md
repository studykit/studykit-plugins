# Vector.X Property

Parent Object: [Vector](../Vector/Vector.md)

## Description

Specifies the X coordinate of the vector. If not supplied, the X value will default to 0.

## Syntax

Vector.**X**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 4
