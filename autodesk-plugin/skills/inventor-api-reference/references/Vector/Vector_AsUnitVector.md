# Vector.AsUnitVector Method

Parent Object: [Vector](../Vector/Vector.md)

## Description

Get the unit vector equivalent of this vector normalized.

## Syntax

Vector.**AsUnitVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Version

Introduced in version 4
