# Vector.CrossProduct Method

Parent Object: [Vector](../Vector/Vector.md)

## Description

Determine the cross product between this vector and the specified vector.

## Syntax

Vector.**CrossProduct**( ***Argument*** As [Vector](../Vector/Vector.md) ) As [Vector](../Vector/Vector.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [Vector](../Vector/Vector.md) | Input  used for comparison. |

## Version

Introduced in version 4
