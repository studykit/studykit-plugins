# Vector.IsParallelTo Method

Parent Object: [Vector](../Vector/Vector.md)

## Description

Determine if this vector is parallel to the specified vector.

## Syntax

Vector.**IsParallelTo**( ***Argument*** As [Vector](../Vector/Vector.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [Vector](../Vector/Vector.md) | Input  used for comparison. |
| Tolerance | Double | Input Double that specifies the tolerance to be used when determining whether the vectors are parallel. |

## Version

Introduced in version 4
