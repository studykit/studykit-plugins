# Vector.TransformBy Method

Parent Object: [Vector](../Vector/Vector.md)

## Description

Transform this vector by the specified matrix.

## Syntax

Vector.**TransformBy**( ***Matrix*** As [Matrix](../Matrix/Matrix.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix | [Matrix](../Matrix/Matrix.md) | Input Matrix object that specifies the position and orientation of the occurrence. |

## Version

Introduced in version 4
