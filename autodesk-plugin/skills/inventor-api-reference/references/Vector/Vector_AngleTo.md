# Vector.AngleTo Method

Parent Object: [Vector](../Vector/Vector.md)

## Description

Determines the angle between this vector and the specified vector.

## Remarks

When one of the input vectors is null (0,0,0) this method will return -NaN.

## Syntax

Vector.**AngleTo**( ***Argument*** As [Vector](../Vector/Vector.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [Vector](../Vector/Vector.md) | Input  used for comparison. |

## Version

Introduced in version 4
