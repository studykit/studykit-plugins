# DerivedPartComponent.SketchBlockDefinitions Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Property that returns the SketchBlockDefinition objects that were created as a result of this derived part.

## Syntax

DerivedPartComponent.**SketchBlockDefinitions**() As [SketchBlockDefinitionsEnumerator](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchBlockDefinitionsEnumerator](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator.md).

## Version

Introduced in version 2010
