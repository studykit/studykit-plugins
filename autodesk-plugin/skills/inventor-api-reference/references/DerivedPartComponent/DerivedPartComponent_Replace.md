# DerivedPartComponent.Replace Method

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Replaces current derived part component with another file.

## Syntax

DerivedPartComponent.**Replace**( ***FullFileName*** As String, ***Options*** As [NameValueMap](../NameValueMap/NameValueMap.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input String value that specifies the new file to be used to replace the existing component. |
| Options | [NameValueMap](../NameValueMap/NameValueMap.md) | Input NameValueMap value that specifies the options when replace the component. Valid options include: Name = ModelState. Value = String value that specifies the model state name. |

## Version

Introduced in version 2023
