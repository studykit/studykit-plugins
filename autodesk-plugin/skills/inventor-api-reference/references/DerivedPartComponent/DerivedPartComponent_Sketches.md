# DerivedPartComponent.Sketches Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Property that returns the PlanarSketch objects that were created as a result of this derived part.

## Syntax

DerivedPartComponent.**Sketches**() As [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md).

## Version

Introduced in version 5.3
