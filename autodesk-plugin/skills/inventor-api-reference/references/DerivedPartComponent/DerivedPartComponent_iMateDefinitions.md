# DerivedPartComponent.iMateDefinitions Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Property that returns the iMateDefinition objects that were created as a result of this derived part.

## Syntax

DerivedPartComponent.**iMateDefinitions**() As [iMateDefinitionsEnumerator](../iMateDefinitionsEnumerator/iMateDefinitionsEnumerator.md)

## Property Value

This is a read only property whose value is an [iMateDefinitionsEnumerator](../iMateDefinitionsEnumerator/iMateDefinitionsEnumerator.md).

## Version

Introduced in version 2010
