# DerivedPartComponent.Definition Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Gets the DerivedPartDefinition that defines the current state of the derived part.

## Syntax

DerivedPartComponent.**Definition**() As [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Property Value

This is a read/write property whose value is a [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md).

## Version

Introduced in version 5.3
