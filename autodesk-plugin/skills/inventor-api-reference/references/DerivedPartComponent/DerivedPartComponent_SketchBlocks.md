# DerivedPartComponent.SketchBlocks Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Property that returns the SketchBlock objects that were created as a result of this derived part.

## Syntax

DerivedPartComponent.**SketchBlocks**() As [SketchBlocksEnumerator](../SketchBlocksEnumerator/SketchBlocksEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchBlocksEnumerator](../SketchBlocksEnumerator/SketchBlocksEnumerator.md).

## Version

Introduced in version 2010
