# HelicalCurveConstantShapeDefinition.HelixStartEndType Property

Parent Object: [HelicalCurveConstantShapeDefinition](../HelicalCurveConstantShapeDefinition/HelicalCurveConstantShapeDefinition.md)

## Description

Gets and sets the helix start end type.

## Syntax

HelicalCurveConstantShapeDefinition.**HelixStartEndType**() As [HelixEndTypeEnum](../HelixEndTypeEnum.md)

## Property Value

This is a read/write property whose value is a [HelixEndTypeEnum](../HelixEndTypeEnum.md).

## Version

Introduced in version 2019
