# OnFaceCurveProxy.FitPointCount Property

Parent Object: [OnFaceCurveProxy](../OnFaceCurveProxy/OnFaceCurveProxy.md)

## Description

Read-only property that returns the number of fit points for the curve.

## Syntax

OnFaceCurveProxy.**FitPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2017
