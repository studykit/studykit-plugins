# OnFaceCurveProxy.Constraints3D Property

Parent Object: [OnFaceCurveProxy](../OnFaceCurveProxy/OnFaceCurveProxy.md)

## Description

Read-only property that returns a collection of sketch constraints that are tied directly to this entity. This collection consists of geometric and bend constraints.

## Syntax

OnFaceCurveProxy.**Constraints3D**() As [SketchConstraints3DEnumerator](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchConstraints3DEnumerator](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator.md).

## Version

Introduced in version 2017
