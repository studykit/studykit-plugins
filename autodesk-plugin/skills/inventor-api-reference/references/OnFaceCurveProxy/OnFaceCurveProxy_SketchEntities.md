# OnFaceCurveProxy.SketchEntities Property

Parent Object: [OnFaceCurveProxy](../OnFaceCurveProxy/OnFaceCurveProxy.md)

## Description

Read-only property that returns a collection of sketch entities that belong to the on face curve.

## Syntax

OnFaceCurveProxy.**SketchEntities**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 2017
