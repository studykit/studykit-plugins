# OnFaceCurveProxy.NativeObject Property

Parent Object: [OnFaceCurveProxy](../OnFaceCurveProxy/OnFaceCurveProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

OnFaceCurveProxy.**NativeObject**() As [OnFaceCurve](../OnFaceCurve/OnFaceCurve.md)

## Property Value

This is a read only property whose value is an [OnFaceCurve](../OnFaceCurve/OnFaceCurve.md).

## Version

Introduced in version 2017
