# ModelCompositeAnnotation.IsMemberOfCompositeAnnotation Property

Parent Object: [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md)

## Description

Returns whether this annotation is a member of a ModelCompositeAnnotation.

## Syntax

ModelCompositeAnnotation.**IsMemberOfCompositeAnnotation**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
