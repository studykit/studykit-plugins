# ModelCompositeAnnotation.Visible Property

Parent Object: [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelCompositeAnnotation.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
