# ModelCompositeAnnotation.Definition Property

Parent Object: [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md)

## Description

Gets and sets the ModelCompositeAnnotationDefinition associative with this object.

## Syntax

ModelCompositeAnnotation.**Definition**() As [ModelCompositeAnnotationDefinition](../ModelCompositeAnnotationDefinition/ModelCompositeAnnotationDefinition.md)

## Property Value

This is a read/write property whose value is a [ModelCompositeAnnotationDefinition](../ModelCompositeAnnotationDefinition/ModelCompositeAnnotationDefinition.md).

## Version

Introduced in version 2018
