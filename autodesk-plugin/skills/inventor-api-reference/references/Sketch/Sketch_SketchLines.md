# Sketch.SketchLines Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Property that returns the SketchLines collection object. This collection provides access to the existing lines in the sketch and provides functionality to create new lines.

## Syntax

Sketch.**SketchLines**() As [SketchLines](../SketchLines/SketchLines.md)

## Property Value

This is a read only property whose value is a [SketchLines](../SketchLines/SketchLines.md).
