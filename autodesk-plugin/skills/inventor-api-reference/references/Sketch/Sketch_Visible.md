# Sketch.Visible Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Gets and sets the visibility of the sketch.

## Syntax

Sketch.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
