# Sketch.GeometricConstraints Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Property that returns the collection of all geometric constraints on the sketch.

## Syntax

Sketch.**GeometricConstraints**() As [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md)

## Property Value

This is a read only property whose value is a [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md).
