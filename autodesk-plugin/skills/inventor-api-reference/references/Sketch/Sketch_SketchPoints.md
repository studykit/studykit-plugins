# Sketch.SketchPoints Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Property that returns the SketchPoints collection object.

## Syntax

Sketch.**SketchPoints**() As [SketchPoints](../SketchPoints/SketchPoints.md)

## Property Value

This is a read only property whose value is a [SketchPoints](../SketchPoints/SketchPoints.md).

## Version

Introduced in version 5.3
