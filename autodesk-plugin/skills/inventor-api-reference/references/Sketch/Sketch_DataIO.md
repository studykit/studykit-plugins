# Sketch.DataIO Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Gets the object through which this sketch's data content can be persisted.

## Syntax

Sketch.**DataIO**() As [DataIO](../DataIO/DataIO.md)

## Property Value

This is a read only property whose value is a [DataIO](../DataIO/DataIO.md).

## Version

Introduced in version 4
