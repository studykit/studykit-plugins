# BIMComponentPropertySet Object

## Description

BIMComponentPropertySet object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BIMComponentPropertySet/BIMComponentPropertySet_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BIMComponentPropertySet/BIMComponentPropertySet_Count.md) | Gets the number of items in this collection. |
| [InternalName](../BIMComponentPropertySet/BIMComponentPropertySet_InternalName.md) | Read-only property that gets the internal name of this property set. This name is consistent and can be used as a reliable index for this property set. |
| [Item](../BIMComponentPropertySet/BIMComponentPropertySet_Item.md) | Returns the specified BIMComponentProperty object from the collection. |
| [Name](../BIMComponentPropertySet/BIMComponentPropertySet_Name.md) | Read-only property that gets the visible name of this property set. This is the name shown to the end-user in the Component Properties list in the Export Building Components dialog. This name is localized and can change for different languages. |
| [Parent](../BIMComponentPropertySet/BIMComponentPropertySet_Parent.md) | Read-only property that returns the parent BIMComponent object of this object. |
| [Type](../BIMComponentPropertySet/BIMComponentPropertySet_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BIMComponentProperty.Parent](../BIMComponentProperty/BIMComponentProperty_Parent.md), [BIMComponentPropertySets.Item](../BIMComponentPropertySets/BIMComponentPropertySets_Item.md)

## Version

Introduced in version 2011
