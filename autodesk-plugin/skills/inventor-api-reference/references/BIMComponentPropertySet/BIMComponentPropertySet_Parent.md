# BIMComponentPropertySet.Parent Property

Parent Object: [BIMComponentPropertySet](../BIMComponentPropertySet/BIMComponentPropertySet.md)

## Description

Read-only property that returns the parent BIMComponent object of this object.

## Syntax

BIMComponentPropertySet.**Parent**() As [BIMComponent](../BIMComponent/BIMComponent.md)

## Property Value

This is a read only property whose value is a [BIMComponent](../BIMComponent/BIMComponent.md).

## Version

Introduced in version 2011
