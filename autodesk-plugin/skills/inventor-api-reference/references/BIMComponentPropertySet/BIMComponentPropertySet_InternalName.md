# BIMComponentPropertySet.InternalName Property

Parent Object: [BIMComponentPropertySet](../BIMComponentPropertySet/BIMComponentPropertySet.md)

## Description

Read-only property that gets the internal name of this property set. This name is consistent and can be used as a reliable index for this property set.

## Syntax

BIMComponentPropertySet.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2011
