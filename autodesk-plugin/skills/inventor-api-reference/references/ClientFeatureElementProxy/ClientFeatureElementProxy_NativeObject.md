# ClientFeatureElementProxy.NativeObject Property

Parent Object: [ClientFeatureElementProxy](../ClientFeatureElementProxy/ClientFeatureElementProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ClientFeatureElementProxy.**NativeObject**() As [ClientFeatureElement](../ClientFeatureElement/ClientFeatureElement.md)

## Property Value

This is a read only property whose value is a [ClientFeatureElement](../ClientFeatureElement/ClientFeatureElement.md).

## Version

Introduced in version 2008
