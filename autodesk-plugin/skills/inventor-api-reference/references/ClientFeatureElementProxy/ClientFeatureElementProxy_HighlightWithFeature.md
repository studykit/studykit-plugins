# ClientFeatureElementProxy.HighlightWithFeature Property

Parent Object: [ClientFeatureElementProxy](../ClientFeatureElementProxy/ClientFeatureElementProxy.md)

## Description

Read-write property that gets and sets whether this element should be pre-highlight, highlighted or selected with the client feature.

## Syntax

ClientFeatureElementProxy.**HighlightWithFeature**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
