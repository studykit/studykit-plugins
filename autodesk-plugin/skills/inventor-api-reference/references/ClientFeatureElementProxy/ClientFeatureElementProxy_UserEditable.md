# ClientFeatureElementProxy.UserEditable Property

Parent Object: [ClientFeatureElementProxy](../ClientFeatureElementProxy/ClientFeatureElementProxy.md)

## Description

Read-write property that gets and sets whether this element can be edited by the user.

## Syntax

ClientFeatureElementProxy.**UserEditable**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
