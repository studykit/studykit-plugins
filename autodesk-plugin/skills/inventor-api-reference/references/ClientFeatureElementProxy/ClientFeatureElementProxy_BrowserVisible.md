# ClientFeatureElementProxy.BrowserVisible Property

Parent Object: [ClientFeatureElementProxy](../ClientFeatureElementProxy/ClientFeatureElementProxy.md)

## Description

Read-write property that gets and sets whether this element is visible in the browser under the ClientFeature node.

## Syntax

ClientFeatureElementProxy.**BrowserVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
