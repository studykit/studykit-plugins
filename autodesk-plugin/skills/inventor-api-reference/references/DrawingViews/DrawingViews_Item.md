# DrawingViews.Item Property

Parent Object: [DrawingViews](../DrawingViews/DrawingViews.md)

## Description

Returns the specified object from the collection.

## Syntax

DrawingViews.**Item**( ***Index*** As Long ) As [DrawingView](../DrawingView/DrawingView.md)

## Property Value

This is a read only property whose value is a [DrawingView](../DrawingView/DrawingView.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 4
