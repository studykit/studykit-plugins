# LoftFeatureProxy.Definition Property

Parent Object: [LoftFeatureProxy](../LoftFeatureProxy/LoftFeatureProxy.md)

## Description

Gets and sets the definition for the loft feature.

## Syntax

LoftFeatureProxy.**Definition**() As [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Property Value

This is a read/write property whose value is a [LoftDefinition](../LoftDefinition/LoftDefinition.md).

## Version

Introduced in version 2008
