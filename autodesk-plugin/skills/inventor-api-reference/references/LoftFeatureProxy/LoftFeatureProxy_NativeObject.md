# LoftFeatureProxy.NativeObject Property

Parent Object: [LoftFeatureProxy](../LoftFeatureProxy/LoftFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

LoftFeatureProxy.**NativeObject**() As [LoftFeature](../LoftFeature/LoftFeature.md)

## Property Value

This is a read only property whose value is a [LoftFeature](../LoftFeature/LoftFeature.md).

## Version

Introduced in version 9
