# DSResult.ResultType Property

Parent Object: [DSResult](../DSResult/DSResult.md)

## Description

Gets the type of result this DSResult object represents.

## Syntax

DSResult.**ResultType**() As [DSResultTypeEnum](../DSResultTypeEnum.md)

## Property Value

This is a read only property whose value is a [DSResultTypeEnum](../DSResultTypeEnum.md).

## Version

Introduced in version 2013
