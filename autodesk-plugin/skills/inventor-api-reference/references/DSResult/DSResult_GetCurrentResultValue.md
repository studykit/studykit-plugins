# DSResult.GetCurrentResultValue Method

Parent Object: [DSResult](../DSResult/DSResult.md)

## Description

Returns the value of the current result.

## Syntax

DSResult.**GetCurrentResultValue**() As Double

## Version

Introduced in version 2013
