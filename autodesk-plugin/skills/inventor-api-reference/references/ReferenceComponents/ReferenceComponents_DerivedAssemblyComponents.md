# ReferenceComponents.DerivedAssemblyComponents Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Property that returns the DerivedAssemblyComponent collection object.

## Syntax

ReferenceComponents.**DerivedAssemblyComponents**() As [DerivedAssemblyComponents](../DerivedAssemblyComponents/DerivedAssemblyComponents.md)

## Property Value

This is a read only property whose value is a [DerivedAssemblyComponents](../DerivedAssemblyComponents/DerivedAssemblyComponents.md).
