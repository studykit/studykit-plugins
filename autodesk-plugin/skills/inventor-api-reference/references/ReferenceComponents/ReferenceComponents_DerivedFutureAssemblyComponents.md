# ReferenceComponents.DerivedFutureAssemblyComponents Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Gets the DerivedFutureAssemblyComponents collection object.

## Syntax

ReferenceComponents.**DerivedFutureAssemblyComponents**() As [DerivedFutureAssemblyComponents](../DerivedFutureAssemblyComponents/DerivedFutureAssemblyComponents.md)

## Property Value

This is a read only property whose value is a [DerivedFutureAssemblyComponents](../DerivedFutureAssemblyComponents/DerivedFutureAssemblyComponents.md).

## Version

Introduced in version 2018
