# ReferenceComponents.DerivedAliasComponents Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Property that returns the DerivedAliasComponents collection object.

## Syntax

ReferenceComponents.**DerivedAliasComponents**() As [DerivedAliasComponents](../DerivedAliasComponents/DerivedAliasComponents.md)

## Property Value

This is a read only property whose value is a [DerivedAliasComponents](../DerivedAliasComponents/DerivedAliasComponents.md).

## Version

Introduced in version 2010
