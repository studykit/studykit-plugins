# ReferenceComponents.ImportedComponents Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Read-only property that returns the ImportedComponents collection object. This collection provides access to existing ImportedComponent objects and provides functionality to create new ImportedComponent objects.

## Syntax

ReferenceComponents.**ImportedComponents**() As [ImportedComponents](../ImportedComponents/ImportedComponents.md)

## Property Value

This is a read only property whose value is an [ImportedComponents](../ImportedComponents/ImportedComponents.md).

## Version

Introduced in version 2016
