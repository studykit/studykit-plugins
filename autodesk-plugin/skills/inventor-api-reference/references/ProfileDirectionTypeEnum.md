# ProfileDirectionTypeEnum Enumerator

## Description

Bit field describing the various profile direction.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAtADefinedAngleToThePredominantDirection | 8 | At a defined angle to the predominant direction. |
| kCircularToTheCenterOfTheSurfaceToWhichTheSymbolApplies | 4 | Circular to the center of the surface to which the symbol applies. |
| kParallelToThePredominantDirection | 2 | Parallel to the predominant direction. |
| kPerpendicularToThePredominantDirection | 1 | Perpendicular to the predominant direction. |

## Version

Introduced in version 2025
