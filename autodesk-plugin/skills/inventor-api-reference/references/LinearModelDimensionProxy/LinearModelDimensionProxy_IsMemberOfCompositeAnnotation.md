# LinearModelDimensionProxy.IsMemberOfCompositeAnnotation Property

Parent Object: [LinearModelDimensionProxy](../LinearModelDimensionProxy/LinearModelDimensionProxy.md)

## Description

Returns whether this annotation is a member of a ModelCompositeAnnotation.

## Syntax

LinearModelDimensionProxy.**IsMemberOfCompositeAnnotation**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
