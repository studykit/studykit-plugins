# MirrorFeature.Definition Property

Parent Object: [MirrorFeature](../MirrorFeature/MirrorFeature.md)

## Description

Read-write property that gets and sets the MirrorFeatureDefinition object associated with this feature.

## Syntax

MirrorFeature.**Definition**() As [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md)

## Property Value

This is a read/write property whose value is a [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md).

## Version

Introduced in version 2017
