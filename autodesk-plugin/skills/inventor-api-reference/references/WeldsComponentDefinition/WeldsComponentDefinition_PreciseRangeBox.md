# WeldsComponentDefinition.PreciseRangeBox Property

Parent Object: [WeldsComponentDefinition](../WeldsComponentDefinition/WeldsComponentDefinition.md)

## Description

Gets a bounding box that tightly encloses all the solid and surface bodies under the ComponentDefinition.

## Syntax

WeldsComponentDefinition.**PreciseRangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2023
