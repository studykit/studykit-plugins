# ReferenceParameter.DisplayFormat Property

Parent Object: [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Description

Gets and sets the display format for the parameter.

## Syntax

ReferenceParameter.**DisplayFormat**() As [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md)

## Property Value

This is a read/write property whose value is a [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md).

## Version

Introduced in version 2011
