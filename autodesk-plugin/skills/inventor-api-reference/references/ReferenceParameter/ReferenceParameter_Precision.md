# ReferenceParameter.Precision Property

Parent Object: [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Description

Gets/(Sets) the the number of decimal places displayed for this parameter. Note that the precision is used when applying a standard tolerance to the parameter.

## Syntax

ReferenceParameter.**Precision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 6
