# ReferenceParameter.Renamed Property

Parent Object: [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Description

Gets whether the ModelParameter has been renamed by the user.

## Syntax

ReferenceParameter.**Renamed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
