# ReferenceParameter.ModelValueType Property

Parent Object: [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Description

Gets/(Sets) the setting which is used for determining the method used to compute the model value.

## Syntax

ReferenceParameter.**ModelValueType**() As [ModelValueTypeEnum](../ModelValueTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ModelValueTypeEnum](../ModelValueTypeEnum.md).

## Version

Introduced in version 6
