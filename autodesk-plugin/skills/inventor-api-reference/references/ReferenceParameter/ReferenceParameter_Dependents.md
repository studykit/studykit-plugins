# ReferenceParameter.Dependents Property

Parent Object: [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Description

Property that returns the collection of objects that have a direct dependency on the parameter.

## Syntax

ReferenceParameter.**Dependents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
