# CosmeticWeldDefinition Object

## Description

Cosmetic Weld Definition Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../CosmeticWeldDefinition/CosmeticWeldDefinition_Copy.md) | Creates a copy of this CosmeticWeldDefinition object. |
| [SetFromToExtent](../CosmeticWeldDefinition/CosmeticWeldDefinition_SetFromToExtent.md) | Method that set the extent type to from-to extent for the weld. Calling this method will set the ExtentType to kFromToExtent. |
| [SetThroughAllExtent](../CosmeticWeldDefinition/CosmeticWeldDefinition_SetThroughAllExtent.md) | Method that changes the extents to be “through all” extents. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CosmeticWeldDefinition/CosmeticWeldDefinition_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Area](../CosmeticWeldDefinition/CosmeticWeldDefinition_Area.md) | Read-write property that provides access to the area of the cosmetic weld. |
| [Edges](../CosmeticWeldDefinition/CosmeticWeldDefinition_Edges.md) | Read-write property that gets and sets the set of edges the cosmetic weld is on. |
| [ExtentType](../CosmeticWeldDefinition/CosmeticWeldDefinition_ExtentType.md) | Read only property that gets and set the extent type for the weld. |
| [FromFace](../CosmeticWeldDefinition/CosmeticWeldDefinition_FromFace.md) | Read only property that gets and sets a planar object as beginning plane to terminate the cosmetic weld. |
| [ToFace](../CosmeticWeldDefinition/CosmeticWeldDefinition_ToFace.md) | Read only property that gets and sets a planar object as ending plane to terminate the cosmetic weld. |
| [Type](../CosmeticWeldDefinition/CosmeticWeldDefinition_Type.md) | Gets the constant that indicates the type of this object. |
