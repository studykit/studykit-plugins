# CosmeticWeldDefinition.Edges Property

Parent Object: [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md)

## Description

Read-write property that gets and sets the set of edges the cosmetic weld is on.

## Syntax

CosmeticWeldDefinition.**Edges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2017
