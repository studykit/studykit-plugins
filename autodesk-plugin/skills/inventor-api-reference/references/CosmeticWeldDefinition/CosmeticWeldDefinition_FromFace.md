# CosmeticWeldDefinition.FromFace Property

Parent Object: [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md)

## Description

Read only property that gets and sets a planar object as beginning plane to terminate the cosmetic weld.

## Syntax

CosmeticWeldDefinition.**FromFace**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2017
