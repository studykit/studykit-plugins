# Vector2d.Y Property

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Gets the Y-component for this Vector2d.

## Syntax

Vector2d.**Y**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 4
