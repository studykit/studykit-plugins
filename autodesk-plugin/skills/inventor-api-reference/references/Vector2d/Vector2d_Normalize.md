# Vector2d.Normalize Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Normalize this vector2d to result in a length equal to 1.0.

## Syntax

Vector2d.**Normalize**()

## Version

Introduced in version 11
