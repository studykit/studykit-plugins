# Vector2d.IsEqualTo Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Compare this vector2d for equality to the specified vector2d.

## Syntax

Vector2d.**IsEqualTo**( ***Vector*** As [Vector2d](../Vector2d/Vector2d.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [Vector2d](../Vector2d/Vector2d.md) | Vector to compare to. |
| Tolerance | Double | Tolerance for comparison. |

## Version

Introduced in version 11
