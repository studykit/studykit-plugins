# Vector2d.SubtractVector Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Subtract the specified vector2d from this vector2d.

## Syntax

Vector2d.**SubtractVector**( ***Value*** As [Vector2d](../Vector2d/Vector2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | [Vector2d](../Vector2d/Vector2d.md) | Vector2d to subtract. |

## Version

Introduced in version 11
