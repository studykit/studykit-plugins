# Vector2d.DotProduct Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Determine the dot product of this vector2d to the specified vector2d.

## Syntax

Vector2d.**DotProduct**( ***Vector*** As [Vector2d](../Vector2d/Vector2d.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [Vector2d](../Vector2d/Vector2d.md) | Vector with which to calculate the dot product. |

## Version

Introduced in version 11
