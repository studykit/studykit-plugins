# Vector2d.TransformBy Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Transform this vector2d by the specified matrix.

## Syntax

Vector2d.**TransformBy**( ***Value*** As [Matrix2d](../Matrix2d/Matrix2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | [Matrix2d](../Matrix2d/Matrix2d.md) | Matrix2d for transform. |

## Version

Introduced in version 11
