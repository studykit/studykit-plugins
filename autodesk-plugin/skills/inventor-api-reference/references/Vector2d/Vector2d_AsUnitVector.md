# Vector2d.AsUnitVector Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Get the unit vector2d equivalent of this vector2d normalized.

## Syntax

Vector2d.**AsUnitVector**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Version

Introduced in version 11
