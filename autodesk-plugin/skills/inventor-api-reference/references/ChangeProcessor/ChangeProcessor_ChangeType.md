# ChangeProcessor.ChangeType Property

Parent Object: [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Description

Gets and sets the type of change (shape edit, query only, etc.) that this ChangeProcessor will make. Defaults to the type set on the ChangeDefinition, but can be overridden.

## Syntax

ChangeProcessor.**ChangeType**() As [CommandTypesEnum](../CommandTypesEnum.md)

## Property Value

This is a read/write property whose value is a [CommandTypesEnum](../CommandTypesEnum.md).

## Version

Introduced in version 2008
