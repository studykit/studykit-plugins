# ChangeProcessor.Execute Method

Parent Object: [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Description

Executes the ChangeProcessor on the specified document.

## Syntax

ChangeProcessor.**Execute**( ***Document*** As [Document](../Document/Document.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Document | [Document](../Document/Document.md) | Input Document object that specifies the document in which to make the changes. |

## Version

Introduced in version 9
