# ChangeProcessor.GlobalChangeProcessAborted Property

Parent Object: [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Description

Gets whether the global transaction for this change had to be aborted. This only applies if GlobalChangeProcess is True and an Execute is in progress.

## Syntax

ChangeProcessor.**GlobalChangeProcessAborted**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
