# ChangeProcessor.Parent Property

Parent Object: [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ChangeProcessor.**Parent**() As [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md)

## Property Value

This is a read only property whose value is a [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md).

## Version

Introduced in version 9
