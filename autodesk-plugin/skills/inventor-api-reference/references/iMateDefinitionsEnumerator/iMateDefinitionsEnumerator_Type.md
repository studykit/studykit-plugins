# iMateDefinitionsEnumerator.Type Property

Parent Object: [iMateDefinitionsEnumerator](../iMateDefinitionsEnumerator/iMateDefinitionsEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iMateDefinitionsEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
