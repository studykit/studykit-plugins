# PresentationTrails.Item Property

Parent Object: [PresentationTrails](../PresentationTrails/PresentationTrails.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationTrails.**Item**( ***Index*** As Long ) As [PresentationTrail](../PresentationTrail/PresentationTrail.md)

## Property Value

This is a read only property whose value is a [PresentationTrail](../PresentationTrail/PresentationTrail.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of an item to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
