# MateConstraint.OccurrenceOne Property

Parent Object: [MateConstraint](../MateConstraint/MateConstraint.md)

## Description

Property that returns the first of the two objects this constraint is between.

## Syntax

MateConstraint.**OccurrenceOne**() As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md).

## Version

Introduced in version 4
