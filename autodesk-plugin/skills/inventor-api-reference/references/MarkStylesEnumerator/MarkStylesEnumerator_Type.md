# MarkStylesEnumerator.Type Property

Parent Object: [MarkStylesEnumerator](../MarkStylesEnumerator/MarkStylesEnumerator.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

MarkStylesEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2023
