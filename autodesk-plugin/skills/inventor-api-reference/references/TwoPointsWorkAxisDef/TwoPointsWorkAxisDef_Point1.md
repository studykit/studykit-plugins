# TwoPointsWorkAxisDef.Point1 Property

Parent Object: [TwoPointsWorkAxisDef](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef.md)

## Description

Property that returns the first point of a two point defined work axis. This object can be a WorkPoint, Vertex, or SketchPoint object. The work axis runs between the two points.

## Syntax

TwoPointsWorkAxisDef.**Point1**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
