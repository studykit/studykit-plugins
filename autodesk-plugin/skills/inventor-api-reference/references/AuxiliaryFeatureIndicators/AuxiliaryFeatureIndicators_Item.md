# AuxiliaryFeatureIndicators.Item Property

Parent Object: [AuxiliaryFeatureIndicators](../AuxiliaryFeatureIndicators/AuxiliaryFeatureIndicators.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

AuxiliaryFeatureIndicators.**Item**( ***Index*** As Long ) As [AuxiliaryFeatureIndicator](../AuxiliaryFeatureIndicator/AuxiliaryFeatureIndicator.md)

## Property Value

This is a read only property whose value is an [AuxiliaryFeatureIndicator](../AuxiliaryFeatureIndicator/AuxiliaryFeatureIndicator.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the AuxiliaryFeatureIndicator to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2023
