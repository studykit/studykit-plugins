# LoftedFlangeDefinition.ExtentDirection Property

Parent Object: [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Description

Gets and sets the which side of the sketch entities is the direction for the material to be added.

## Syntax

LoftedFlangeDefinition.**ExtentDirection**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2011
