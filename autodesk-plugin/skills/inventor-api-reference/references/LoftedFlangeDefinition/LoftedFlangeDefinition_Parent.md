# LoftedFlangeDefinition.Parent Property

Parent Object: [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Description

Property that returns the parent LoftedFlangeFeature object.

## Syntax

LoftedFlangeDefinition.**Parent**() As [LoftedFlangeFeature](../LoftedFlangeFeature/LoftedFlangeFeature.md)

## Property Value

This is a read only property whose value is a [LoftedFlangeFeature](../LoftedFlangeFeature/LoftedFlangeFeature.md).

## Version

Introduced in version 2011
