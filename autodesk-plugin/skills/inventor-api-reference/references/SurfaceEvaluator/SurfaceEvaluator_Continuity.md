# SurfaceEvaluator.Continuity Property

Parent Object: [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Description

Gets the level of continuity of the curve. The continuity is the largest order of continuity maintained for the entire curve.

## Syntax

SurfaceEvaluator.**Continuity**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
