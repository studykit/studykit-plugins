# SurfaceEvaluator.IsRevolvedShape Property

Parent Object: [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Description

Read-only property that returns whether the surface resulted from a revolution.

## Syntax

SurfaceEvaluator.**IsRevolvedShape**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2017
