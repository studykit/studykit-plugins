# ImportedDWGComponentProxy.NativeObject Property

Parent Object: [ImportedDWGComponentProxy](../ImportedDWGComponentProxy/ImportedDWGComponentProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ImportedDWGComponentProxy.**NativeObject**() As [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Property Value

This is a read only property whose value is an [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md).

## Version

Introduced in version 2016
