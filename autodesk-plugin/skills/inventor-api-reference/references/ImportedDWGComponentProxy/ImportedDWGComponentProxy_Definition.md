# ImportedDWGComponentProxy.Definition Property

Parent Object: [ImportedDWGComponentProxy](../ImportedDWGComponentProxy/ImportedDWGComponentProxy.md)

## Description

Read-write property that gets and sets the ImportedComponentDefinition which defines the various inputs that were used to create the imported component.

## Syntax

ImportedDWGComponentProxy.**Definition**() As [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md)

## Property Value

This is a read/write property whose value is an [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md).

## Version

Introduced in version 2016
