# ImportedDWGComponentProxy.RangeBox Property

Parent Object: [ImportedDWGComponentProxy](../ImportedDWGComponentProxy/ImportedDWGComponentProxy.md)

## Description

Read-only property that returns the bounding box of the DWG graphics.

## Syntax

ImportedDWGComponentProxy.**RangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2018
