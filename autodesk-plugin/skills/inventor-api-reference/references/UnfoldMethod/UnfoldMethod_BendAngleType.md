# UnfoldMethod.BendAngleType Property

Parent Object: [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Description

Gets and sets whether the bend or open angle value is to be used for the angle in equations.

## Syntax

UnfoldMethod.**BendAngleType**() As [BendAngleTypeEnum](../BendAngleTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BendAngleTypeEnum](../BendAngleTypeEnum.md).

## Version

Introduced in version 2010
