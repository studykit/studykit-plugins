# LeaderNotes.Item Property

Parent Object: [LeaderNotes](../LeaderNotes/LeaderNotes.md)

## Description

Returns the specified LeaderNote object from the collection.

## Syntax

LeaderNotes.**Item**( ***Index*** As Long ) As [LeaderNote](../LeaderNote/LeaderNote.md)

## Property Value

This is a read only property whose value is a [LeaderNote](../LeaderNote/LeaderNote.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 10
