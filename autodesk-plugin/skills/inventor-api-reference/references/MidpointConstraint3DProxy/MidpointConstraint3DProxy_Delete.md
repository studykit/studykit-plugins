# MidpointConstraint3DProxy.Delete Method

Parent Object: [MidpointConstraint3DProxy](../MidpointConstraint3DProxy/MidpointConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

MidpointConstraint3DProxy.**Delete**()

## Version

Introduced in version 2009
