# MidpointConstraint3DProxy.NativeObject Property

Parent Object: [MidpointConstraint3DProxy](../MidpointConstraint3DProxy/MidpointConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MidpointConstraint3DProxy.**NativeObject**() As [MidpointConstraint3D](../MidpointConstraint3D/MidpointConstraint3D.md)

## Property Value

This is a read only property whose value is a [MidpointConstraint3D](../MidpointConstraint3D/MidpointConstraint3D.md).

## Version

Introduced in version 2009
