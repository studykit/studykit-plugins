# GraphicsNode.AddTriangleStripGraphics Method

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Method that creates a new TriangleStripGraphics graphic primitive.

## Syntax

GraphicsNode.**AddTriangleStripGraphics**() As [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Samples

|  |  |
| --- | --- |
| Name | Description |
| [Client Graphics - Triangle](../../sample-programs/GraphicsNode_AddTriangleFanGraphics_Sample.md) | This sample demonstrates the creation of client graphics triangles using triange fans and strips. It does this by drawing a cylinder. The end caps are triangle fans and the cylinder is made from a triangle strip. |

## Version

Introduced in version 5
