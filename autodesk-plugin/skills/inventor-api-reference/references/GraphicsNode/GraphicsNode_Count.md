# GraphicsNode.Count Property

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Property that returns the number of graphic primitive objects associated with this GraphicsNode object.

## Syntax

GraphicsNode.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5
