# GraphicsNode.Parent Property

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Property that returns the object this graphics node belongs to.

## Syntax

GraphicsNode.**Parent**() As [ClientGraphics](../ClientGraphics/ClientGraphics.md)

## Property Value

This is a read only property whose value is a [ClientGraphics](../ClientGraphics/ClientGraphics.md).

## Version

Introduced in version 5
