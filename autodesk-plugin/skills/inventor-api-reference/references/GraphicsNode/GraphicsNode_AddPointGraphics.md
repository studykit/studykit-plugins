# GraphicsNode.AddPointGraphics Method

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Method that creates a new PointGraphics graphic primitive.

## Syntax

GraphicsNode.**AddPointGraphics**() As [PointGraphics](../PointGraphics/PointGraphics.md)
