# CombineFeatureProxy.Operation Property

Parent Object: [CombineFeatureProxy](../CombineFeatureProxy/CombineFeatureProxy.md)

## Description

Gets and sets the operation type. Valid values are kJoinOperation, kCutOperation and kIntersectOperation.

## Syntax

CombineFeatureProxy.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2010
