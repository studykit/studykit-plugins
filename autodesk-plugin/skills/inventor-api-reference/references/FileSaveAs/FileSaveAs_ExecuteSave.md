# FileSaveAs.ExecuteSave Method

Parent Object: [FileSaveAs](../FileSaveAs/FileSaveAs.md)

## Description

Executes the Save operation on each of the documents specified via the AddFileToSave method.

## Syntax

FileSaveAs.**ExecuteSave**()

## Version

Introduced in version 4
