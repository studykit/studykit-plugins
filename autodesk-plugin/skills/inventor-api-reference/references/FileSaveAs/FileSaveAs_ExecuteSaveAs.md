# FileSaveAs.ExecuteSaveAs Method

Parent Object: [FileSaveAs](../FileSaveAs/FileSaveAs.md)

## Description

Executes the Save As operation on each of the documents specified via the AddFileToSave method.

## Syntax

FileSaveAs.**ExecuteSaveAs**()

## Version

Introduced in version 4
