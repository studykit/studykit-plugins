# AutoCADBlock.Delete Method

Parent Object: [AutoCADBlock](../AutoCADBlock/AutoCADBlock.md)

## Description

Method that deletes the AutoCAD block from the sheet.

## Syntax

AutoCADBlock.**Delete**()

## Version

Introduced in version 2011
