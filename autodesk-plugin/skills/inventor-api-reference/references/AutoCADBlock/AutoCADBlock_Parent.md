# AutoCADBlock.Parent Property

Parent Object: [AutoCADBlock](../AutoCADBlock/AutoCADBlock.md)

## Description

Property that returns the parent Sheet object.

## Syntax

AutoCADBlock.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 2011
