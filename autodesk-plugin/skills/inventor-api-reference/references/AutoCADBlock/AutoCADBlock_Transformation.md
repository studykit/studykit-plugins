# AutoCADBlock.Transformation Property

Parent Object: [AutoCADBlock](../AutoCADBlock/AutoCADBlock.md)

## Description

Property that provides the transform that is applied to display the associated AutoCAD block definition in the correct location on the sheet. The matrix defines the sheet to block transform.

## Syntax

AutoCADBlock.**Transformation**() As [Matrix2d](../Matrix2d/Matrix2d.md)

## Property Value

This is a read only property whose value is a [Matrix2d](../Matrix2d/Matrix2d.md).

## Version

Introduced in version 2011
