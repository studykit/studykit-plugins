# AutoCADBlock.Static Property

Parent Object: [AutoCADBlock](../AutoCADBlock/AutoCADBlock.md)

## Description

Read-write property that gets and sets whether to show the scale and rotation grip points on the block. If True, the grip points are disabled.

## Syntax

AutoCADBlock.**Static**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
