# SketchDrivenPatternFeatureProxy.Definition Property

Parent Object: [SketchDrivenPatternFeatureProxy](../SketchDrivenPatternFeatureProxy/SketchDrivenPatternFeatureProxy.md)

## Description

Read-write property that gets and sets the SketchDrivenPatternDefinition object associated with this feature.

## Syntax

SketchDrivenPatternFeatureProxy.**Definition**() As [SketchDrivenPatternDefinition](../SketchDrivenPatternDefinition/SketchDrivenPatternDefinition.md)

## Property Value

This is a read/write property whose value is a [SketchDrivenPatternDefinition](../SketchDrivenPatternDefinition/SketchDrivenPatternDefinition.md).

## Version

Introduced in version 2017
