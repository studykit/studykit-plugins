# SketchDrivenPatternFeatureProxy.Suppressed Property

Parent Object: [SketchDrivenPatternFeatureProxy](../SketchDrivenPatternFeatureProxy/SketchDrivenPatternFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

SketchDrivenPatternFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
