# FlatPatternFeatures.SketchDrivenPatternFeatures Property

Parent Object: [FlatPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures.md)

## Description

Gets the collection object that besides listing out the subset of part features that are SketchDrivenPatterns, allows the creations of new SketchDrivenPatterns.

## Syntax

FlatPatternFeatures.**SketchDrivenPatternFeatures**() As [SketchDrivenPatternFeatures](../SketchDrivenPatternFeatures/SketchDrivenPatternFeatures.md)

## Property Value

This is a read only property whose value is a [SketchDrivenPatternFeatures](../SketchDrivenPatternFeatures/SketchDrivenPatternFeatures.md).

## Version

Introduced in version 2017
