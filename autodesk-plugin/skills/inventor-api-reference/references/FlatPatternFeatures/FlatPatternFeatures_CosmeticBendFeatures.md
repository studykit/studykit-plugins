# FlatPatternFeatures.CosmeticBendFeatures Property

Parent Object: [FlatPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures.md)

## Description

Returns the CosmeticBendFeatures collection object.

## Syntax

FlatPatternFeatures.**CosmeticBendFeatures**() As [CosmeticBendFeatures](../CosmeticBendFeatures/CosmeticBendFeatures.md)

## Property Value

This is a read only property whose value is a [CosmeticBendFeatures](../CosmeticBendFeatures/CosmeticBendFeatures.md).

## Version

Introduced in version 2010
