# PresentationVerticesEnumerator.Item Property

Parent Object: [PresentationVerticesEnumerator](../PresentationVerticesEnumerator/PresentationVerticesEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationVerticesEnumerator.**Item**( ***Index*** As Long ) As [PresentationVertex](../PresentationVertex/PresentationVertex.md)

## Property Value

This is a read only property whose value is a [PresentationVertex](../PresentationVertex/PresentationVertex.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of an item to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
