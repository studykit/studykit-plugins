# TransactionEvents.Parent Property

Parent Object: [TransactionEvents](../TransactionEvents/TransactionEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

TransactionEvents.**Parent**() As [TransactionManager](../TransactionManager/TransactionManager.md)

## Property Value

This is a read only property whose value is a [TransactionManager](../TransactionManager/TransactionManager.md).

## Version

Introduced in version 5
