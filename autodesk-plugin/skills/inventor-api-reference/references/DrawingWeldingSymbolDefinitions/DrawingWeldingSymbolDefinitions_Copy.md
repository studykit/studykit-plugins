# DrawingWeldingSymbolDefinitions.Copy Method

Parent Object: [DrawingWeldingSymbolDefinitions](../DrawingWeldingSymbolDefinitions/DrawingWeldingSymbolDefinitions.md)

## Description

Method that creates a copy of this DrawingWeldingSymbolDefinitions object. The new DrawingWeldingSymbolDefinitions object is independent of any drawing welding symbol.

## Syntax

DrawingWeldingSymbolDefinitions.**Copy**() As [DrawingWeldingSymbolDefinitions](../DrawingWeldingSymbolDefinitions/DrawingWeldingSymbolDefinitions.md)

## Version

Introduced in version 2024
