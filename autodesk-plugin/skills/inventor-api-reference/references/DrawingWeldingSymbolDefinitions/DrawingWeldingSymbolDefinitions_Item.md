# DrawingWeldingSymbolDefinitions.Item Property

Parent Object: [DrawingWeldingSymbolDefinitions](../DrawingWeldingSymbolDefinitions/DrawingWeldingSymbolDefinitions.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

DrawingWeldingSymbolDefinitions.**Item**( ***Index*** As Long ) As [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md)

## Property Value

This is a read only property whose value is a [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of DrawingWeldingSymbolDefinition to return. |

## Version

Introduced in version 2024
