# PresentationEdge.Parent Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PresentationEdge.**Parent**() As [PresentationBody](../PresentationBody/PresentationBody.md)

## Property Value

This is a read only property whose value is a [PresentationBody](../PresentationBody/PresentationBody.md).

## Version

Introduced in version 2018
