# PresentationEdge.PointOnEdge Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Read-only property that returns a characteristic somewhere in the middle of the edge.

## Syntax

PresentationEdge.**PointOnEdge**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
