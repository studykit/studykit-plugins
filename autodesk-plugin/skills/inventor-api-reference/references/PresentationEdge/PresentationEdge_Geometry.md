# PresentationEdge.Geometry Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Read-only property that returns the underlying geometry of the edge (e.g. Arc2D, Circle, Cone etc.).

## Syntax

PresentationEdge.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
