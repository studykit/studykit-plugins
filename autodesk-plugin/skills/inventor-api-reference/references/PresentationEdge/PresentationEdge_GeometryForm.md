# PresentationEdge.GeometryForm Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Read-only property that returns the form of the underlying geometry as a combination of one or more CurveGeometryFormEnum values.

## Syntax

PresentationEdge.**GeometryForm**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2018
