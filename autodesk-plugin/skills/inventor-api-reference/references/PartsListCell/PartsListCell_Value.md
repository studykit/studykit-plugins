# PartsListCell.Value Property

Parent Object: [PartsListCell](../PartsListCell/PartsListCell.md)

## Description

Gets or sets the value of the cell in the parts list table.

## Syntax

PartsListCell.**Value**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5.3
