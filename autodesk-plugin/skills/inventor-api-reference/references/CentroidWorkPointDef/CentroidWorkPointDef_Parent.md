# CentroidWorkPointDef.Parent Property

Parent Object: [CentroidWorkPointDef](../CentroidWorkPointDef/CentroidWorkPointDef.md)

## Description

Property that returns the parent WorkPoint object.

## Syntax

CentroidWorkPointDef.**Parent**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 2008
