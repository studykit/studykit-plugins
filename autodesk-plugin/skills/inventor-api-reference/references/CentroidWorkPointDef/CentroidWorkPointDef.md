# CentroidWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point at the centroid of the input entities.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CentroidWorkPointDef/CentroidWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Entities](../CentroidWorkPointDef/CentroidWorkPointDef_Entities.md) | Property that returns either an Edge, an EdgeLoop or an EdgeCollection object. |
| [Parent](../CentroidWorkPointDef/CentroidWorkPointDef_Parent.md) | Property that returns the parent WorkPoint object. |
| [Type](../CentroidWorkPointDef/CentroidWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2008
