# BIMComponentProperty.Parent Property

Parent Object: [BIMComponentProperty](../BIMComponentProperty/BIMComponentProperty.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

BIMComponentProperty.**Parent**() As [BIMComponentPropertySet](../BIMComponentPropertySet/BIMComponentPropertySet.md)

## Property Value

This is a read only property whose value is a [BIMComponentPropertySet](../BIMComponentPropertySet/BIMComponentPropertySet.md).

## Version

Introduced in version 2011
