# PrintColorModeEnum Enumerator

## Description

Constants denoting the color mode being used.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kPrintColorPalette | 13313 | Print using color. |
| kPrintDefaultColorMode | 13315 | Default setting for color mode. |
| kPrintGrayScale | 13314 | Print using a gray scale. |

## Version

Introduced in version 2015
