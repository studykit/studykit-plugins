# OffsetConstraintProxy.Deletable Property

Parent Object: [OffsetConstraintProxy](../OffsetConstraintProxy/OffsetConstraintProxy.md)

## Description

Indicates whether this object is deletable.

## Syntax

OffsetConstraintProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
