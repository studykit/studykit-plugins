# RibbonPanel.Parent Property

Parent Object: [RibbonPanel](../RibbonPanel/RibbonPanel.md)

## Description

Property that returns the parent RibbonTab object.

## Syntax

RibbonPanel.**Parent**() As [RibbonTab](../RibbonTab/RibbonTab.md)

## Property Value

This is a read only property whose value is a [RibbonTab](../RibbonTab/RibbonTab.md).

## Version

Introduced in version 2010
