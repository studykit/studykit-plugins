# RibbonPanel.SlideoutControls Property

Parent Object: [RibbonPanel](../RibbonPanel/RibbonPanel.md)

## Description

Property that returns the CommandControls collection object that provides access to all controls in the slideout section of the panel.

![](../images/RibbonPanel_SlideOutControls.png)

## Syntax

RibbonPanel.**SlideoutControls**() As [CommandControls](../CommandControls/CommandControls.md)

## Property Value

This is a read only property whose value is a [CommandControls](../CommandControls/CommandControls.md).
