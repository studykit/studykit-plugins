# RibbonPanel.Docked Property

Parent Object: [RibbonPanel](../RibbonPanel/RibbonPanel.md)

## Description

Gets sets whether the ribbon panel docked within the ribbon.

## Syntax

RibbonPanel.**Docked**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
