# RibbonPanel.CommandControls Property

Parent Object: [RibbonPanel](../RibbonPanel/RibbonPanel.md)

## Description

Property that returns the CommandControls collection object that provides access to all the visible controls within the panel.

## Syntax

RibbonPanel.**CommandControls**() As [CommandControls](../CommandControls/CommandControls.md)

## Property Value

This is a read only property whose value is a [CommandControls](../CommandControls/CommandControls.md).
