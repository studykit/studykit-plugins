# SketchedSymbolDefinition.Name Property

Parent Object: [SketchedSymbolDefinition](../SketchedSymbolDefinition/SketchedSymbolDefinition.md)

## Description

Gets and sets the name of the sketched symbol definition.

## Syntax

SketchedSymbolDefinition.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5.3
