# iAssemblyTableRow.Type Property

Parent Object: [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iAssemblyTableRow.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
