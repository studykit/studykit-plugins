# iAssemblyTableRow.Copy Method

Parent Object: [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md)

## Description

Method that creates a new row with all values equal to the original row.

## Syntax

iAssemblyTableRow.**Copy**( [***NewMemberName***] As String ) As [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| NewMemberName | String | Name of new row member. |

## Version

Introduced in version 11
