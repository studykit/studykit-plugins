# DrawingNote.FormattedText Property

Parent Object: [DrawingNote](../DrawingNote/DrawingNote.md)

## Description

Gets/Sets formatted text of the DrawingNote.

## Syntax

DrawingNote.**FormattedText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 10
