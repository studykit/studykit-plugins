# DrawingNote.WidthScale Property

Parent Object: [DrawingNote](../DrawingNote/DrawingNote.md)

## Description

Gets/Sets the width scale factor of the DrawingNote.

## Syntax

DrawingNote.**WidthScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 10
