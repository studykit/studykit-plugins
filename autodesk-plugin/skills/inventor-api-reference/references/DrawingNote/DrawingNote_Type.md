# DrawingNote.Type Property

Parent Object: [DrawingNote](../DrawingNote/DrawingNote.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

DrawingNote.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
