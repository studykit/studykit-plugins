# VerticalAlignConstraintProxy.NativeObject Property

Parent Object: [VerticalAlignConstraintProxy](../VerticalAlignConstraintProxy/VerticalAlignConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

VerticalAlignConstraintProxy.**NativeObject**() As [VerticalAlignConstraint](../VerticalAlignConstraint/VerticalAlignConstraint.md)

## Property Value

This is a read only property whose value is a [VerticalAlignConstraint](../VerticalAlignConstraint/VerticalAlignConstraint.md).

## Version

Introduced in version 6
