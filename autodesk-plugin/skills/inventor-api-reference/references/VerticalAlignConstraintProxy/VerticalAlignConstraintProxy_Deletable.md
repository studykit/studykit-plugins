# VerticalAlignConstraintProxy.Deletable Property

Parent Object: [VerticalAlignConstraintProxy](../VerticalAlignConstraintProxy/VerticalAlignConstraintProxy.md)

## Description

Indicates whether this object is deletable.

## Syntax

VerticalAlignConstraintProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version
