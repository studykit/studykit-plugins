# CornerRoundEdgeSet Object

## Description

The CornerRoundEdgeSet object represents a set of edges and an associated fillet radius.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Edges](../CornerRoundEdgeSet/CornerRoundEdgeSet_Edges.md) | Gets and sets the edges in this edge set. |
| [Parent](../CornerRoundEdgeSet/CornerRoundEdgeSet_Parent.md) | Property that returns the parent CornerRoundDefinition of this CornerRoundEdgeSet object. |
| [Radius](../CornerRoundEdgeSet/CornerRoundEdgeSet_Radius.md) | Gets and sets the radius of a sheet metal corner round feature. |
| [Type](../CornerRoundEdgeSet/CornerRoundEdgeSet_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[CornerRoundDefinition.AddEdgeSet](../CornerRoundDefinition/CornerRoundDefinition_AddEdgeSet.md), [CornerRoundDefinition.EdgeSetItem](../CornerRoundDefinition/CornerRoundDefinition_EdgeSetItem.md)

## Version

Introduced in version 2009
