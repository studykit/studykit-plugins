# CornerRoundEdgeSet.Parent Property

Parent Object: [CornerRoundEdgeSet](../CornerRoundEdgeSet/CornerRoundEdgeSet.md)

## Description

Property that returns the parent CornerRoundDefinition of this CornerRoundEdgeSet object.

## Syntax

CornerRoundEdgeSet.**Parent**() As [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md)

## Property Value

This is a read only property whose value is a [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md).

## Version

Introduced in version 2009
