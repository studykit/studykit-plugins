# CornerRoundEdgeSet.Edges Property

Parent Object: [CornerRoundEdgeSet](../CornerRoundEdgeSet/CornerRoundEdgeSet.md)

## Description

Gets and sets the edges in this edge set.

## Syntax

CornerRoundEdgeSet.**Edges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2009
