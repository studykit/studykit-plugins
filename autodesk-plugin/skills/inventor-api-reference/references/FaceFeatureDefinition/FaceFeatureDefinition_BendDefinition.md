# FaceFeatureDefinition.BendDefinition Property

Parent Object: [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md)

## Description

Get and sets the BendDefinition object that defines the bend information for this face feature.

## Syntax

FaceFeatureDefinition.**BendDefinition**() As [BendDefinition](../BendDefinition/BendDefinition.md)

## Property Value

This is a read/write property whose value is a [BendDefinition](../BendDefinition/BendDefinition.md).

## Version

Introduced in version 2009
