# FaceFeatureDefinition.Parent Property

Parent Object: [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md)

## Description

Property that returns the parent FaceFeature object of this FaceFeatureDefinition object.

## Syntax

FaceFeatureDefinition.**Parent**() As [FaceFeature](../FaceFeature/FaceFeature.md)

## Property Value

This is a read only property whose value is a [FaceFeature](../FaceFeature/FaceFeature.md).

## Version

Introduced in version 2009
