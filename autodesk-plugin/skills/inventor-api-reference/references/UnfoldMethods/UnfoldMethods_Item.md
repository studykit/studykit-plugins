# UnfoldMethods.Item Property

Parent Object: [UnfoldMethods](../UnfoldMethods/UnfoldMethods.md)

## Description

Returns the specified UnfoldMethod object from the collection.

## Syntax

UnfoldMethods.**Item**( ***Index*** As Variant ) As [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Property Value

This is a read only property whose value is a [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the UnfoldMethod object to return. This can be either a numeric value indicating the index of the item in the collection or it can be a String indicating the UnfoldMethod name. If an out of range index or a name of a non-existent UnfoldMethod is provided, an error occurs. |

## Version

Introduced in version 5.3
