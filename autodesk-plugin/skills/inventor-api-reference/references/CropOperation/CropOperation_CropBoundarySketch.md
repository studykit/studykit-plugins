# CropOperation.CropBoundarySketch Property

Parent Object: [CropOperation](../CropOperation/CropOperation.md)

## Description

Read-only property that returns the drawing sketch used to specify the boundary for crop operation.

## Syntax

CropOperation.**CropBoundarySketch**() As [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Property Value

This is a read only property whose value is a [DrawingSketch](../DrawingSketch/DrawingSketch.md).

## Version

Introduced in version 2025.1
