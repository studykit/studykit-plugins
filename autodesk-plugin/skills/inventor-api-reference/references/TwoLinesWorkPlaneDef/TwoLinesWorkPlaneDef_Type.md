# TwoLinesWorkPlaneDef.Type Property

Parent Object: [TwoLinesWorkPlaneDef](../TwoLinesWorkPlaneDef/TwoLinesWorkPlaneDef.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TwoLinesWorkPlaneDef.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 4
