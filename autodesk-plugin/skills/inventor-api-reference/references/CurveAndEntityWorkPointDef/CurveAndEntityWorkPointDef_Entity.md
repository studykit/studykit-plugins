# CurveAndEntityWorkPointDef.Entity Property

Parent Object: [CurveAndEntityWorkPointDef](../CurveAndEntityWorkPointDef/CurveAndEntityWorkPointDef.md)

## Description

Property that returns the Face or WorkPlane object used in defining a work at the intersection of a curve and entity.

## Syntax

CurveAndEntityWorkPointDef.**Entity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
