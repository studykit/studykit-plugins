# DrawingWeldingSymbol.RetrievedFrom Property

Parent Object: [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Description

Read-only property that returns the object that this symbol was retrieved from.

## Syntax

DrawingWeldingSymbol.**RetrievedFrom**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2024
