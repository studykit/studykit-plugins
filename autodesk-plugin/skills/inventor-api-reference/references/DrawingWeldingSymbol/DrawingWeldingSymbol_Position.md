# DrawingWeldingSymbol.Position Property

Parent Object: [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Description

Gets and sets the position of the symbol on the sheet.

## Syntax

DrawingWeldingSymbol.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2024
