# DrawingWeldingSymbol.Style Property

Parent Object: [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Description

Gets and sets the associated WeldSymbolStyle object.

## Syntax

DrawingWeldingSymbol.**Style**() As [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Property Value

This is a read/write property whose value is a [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md).

## Version

Introduced in version 2024
