# DrawingWeldingSymbol.Retrieved Property

Parent Object: [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Description

Read-only property that indicates whether the symbol was created as a result of retrieving it from the model. Retrieved DrawingWeldingSymbol can not be edited.

## Syntax

DrawingWeldingSymbol.**Retrieved**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2024
