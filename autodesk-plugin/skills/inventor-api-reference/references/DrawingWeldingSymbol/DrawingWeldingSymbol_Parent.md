# DrawingWeldingSymbol.Parent Property

Parent Object: [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DrawingWeldingSymbol.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version
