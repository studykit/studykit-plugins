# DrawingViewHatchRegion.Visible Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that gets and sets whether the drawing view hatch region is visible or not.

## Syntax

DrawingViewHatchRegion.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
