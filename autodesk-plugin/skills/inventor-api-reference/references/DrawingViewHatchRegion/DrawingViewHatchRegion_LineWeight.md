# DrawingViewHatchRegion.LineWeight Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that specifies the line weight of the drawing view hatch region object in centimeters.

## Syntax

DrawingViewHatchRegion.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
