# DrawingViewHatchRegion.BoundaryGeometries Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-only property that returns the geometries that shape the profile of the drawing view hatch region.

## Syntax

DrawingViewHatchRegion.**BoundaryGeometries**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2022
