# DrawingViewHatchRegion.Scale Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that specifies the scale of the distance between lines.

## Syntax

DrawingViewHatchRegion.**Scale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
