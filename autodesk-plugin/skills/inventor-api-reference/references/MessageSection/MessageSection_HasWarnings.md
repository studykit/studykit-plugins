# MessageSection.HasWarnings Property

Parent Object: [MessageSection](../MessageSection/MessageSection.md)

## Description

Property that returns whether this message section has any warnings reported in it.

## Syntax

MessageSection.**HasWarnings**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.
