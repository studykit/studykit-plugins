# SoftwareVersion.DisplayVersion Property

Parent Object: [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Description

Returns the release version displayed to users. (e.g.: '2009' for Major version 13 or '2010' for Major version 14)

## Syntax

SoftwareVersion.**DisplayVersion**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2009
