# SoftwareVersion.BetaVersion Property

Parent Object: [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Description

Gets the Beta version. If this is not a Beta version software, this value will be zero.

## Syntax

SoftwareVersion.**BetaVersion**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
