# SoftwareVersion.BuildIdentifier Property

Parent Object: [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Description

Gets the number that completely identifies a particular build of Autodesk Inventor.

## Syntax

SoftwareVersion.**BuildIdentifier**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
