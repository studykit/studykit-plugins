# SoftwareVersion.NotProduction Property

Parent Object: [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Description

Gets a Boolean flag indicating whether this software is intended for purely internal purposes or not. Beta is considered external.

## Syntax

SoftwareVersion.**NotProduction**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
