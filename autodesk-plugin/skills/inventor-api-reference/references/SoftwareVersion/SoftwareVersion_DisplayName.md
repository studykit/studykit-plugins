# SoftwareVersion.DisplayName Property

Parent Object: [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Description

Gets the human-readable, displayable name designating this particular version (e.g.: 'Autodesk Inventor (tm) Release 2.5 Service Pack 3').

## Syntax

SoftwareVersion.**DisplayName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
