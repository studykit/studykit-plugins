# SoftwareVersion.ServicePack Property

Parent Object: [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Description

Gets the number of the Service Pack if this software has been updated with such. If none, a zero is returned. (e.g.: the '3' in Release 2.5 Service Pack 3).

## Syntax

SoftwareVersion.**ServicePack**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
