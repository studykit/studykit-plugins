# AssemblyDocument.ResolveFile Method

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Method that invoke the resolve file for the document if there are references missing.

## Syntax

AssemblyDocument.**ResolveFile**()

## Version

Introduced in version 2025
