# AssemblyDocument.PrintManager Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Property that returns the PrintManager, or when called from a DrawingDocument it returns a DrawingPrintManager object.

## Syntax

AssemblyDocument.**PrintManager**() As [PrintManager](../PrintManager/PrintManager.md)

## Property Value

This is a read only property whose value is a [PrintManager](../PrintManager/PrintManager.md).

## Version

Introduced in version 5
