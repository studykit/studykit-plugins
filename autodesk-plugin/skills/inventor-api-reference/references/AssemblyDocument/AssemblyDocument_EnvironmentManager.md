# AssemblyDocument.EnvironmentManager Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Property that returns the EnvironmentManager object.

## Syntax

AssemblyDocument.**EnvironmentManager**() As [EnvironmentManager](../EnvironmentManager/EnvironmentManager.md)

## Property Value

This is a read only property whose value is an [EnvironmentManager](../EnvironmentManager/EnvironmentManager.md).
