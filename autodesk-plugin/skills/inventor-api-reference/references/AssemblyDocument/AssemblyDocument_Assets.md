# AssemblyDocument.Assets Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Get Assets collection object that contains all of the assets, regardless of type, that are associated with the document.

## Syntax

AssemblyDocument.**Assets**() As [Assets](../Assets/Assets.md)

## Property Value

This is a read only property whose value is an [Assets](../Assets/Assets.md).
