# AssemblyDocument.DocumentType Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Gets the constant that indicates the type of this document.

## Syntax

AssemblyDocument.**DocumentType**() As [DocumentTypeEnum](../DocumentTypeEnum.md)

## Property Value

This is a read only property whose value is a [DocumentTypeEnum](../DocumentTypeEnum.md).

## Version

Introduced in version 4
