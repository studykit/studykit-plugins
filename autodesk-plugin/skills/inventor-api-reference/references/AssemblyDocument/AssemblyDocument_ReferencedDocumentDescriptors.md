# AssemblyDocument.ReferencedDocumentDescriptors Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Property that returns an enumeration of descriptors that represent the native document references held by this document.

## Syntax

AssemblyDocument.**ReferencedDocumentDescriptors**() As [DocumentDescriptorsEnumerator](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptorsEnumerator](../DocumentDescriptorsEnumerator/DocumentDescriptorsEnumerator.md).

## Version

Introduced in version 11
