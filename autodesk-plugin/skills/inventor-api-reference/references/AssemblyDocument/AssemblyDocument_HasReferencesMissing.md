# AssemblyDocument.HasReferencesMissing Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Read-only property that returns whether there are references missing for this document.

## Syntax

AssemblyDocument.**HasReferencesMissing**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025
