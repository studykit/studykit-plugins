# AssemblyDocument.PhysicalAssets Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Get AssetsEnumerator object that contains the physical properties associated with the document.

## Syntax

AssemblyDocument.**PhysicalAssets**() As [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md).
