# AssemblyDocument.LightingStyles Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Property that returns the LightingStyles collection object.

## Syntax

AssemblyDocument.**LightingStyles**() As [LightingStyles](../LightingStyles/LightingStyles.md)

## Property Value

This is a read only property whose value is a [LightingStyles](../LightingStyles/LightingStyles.md).

## Version

Introduced in version 10
