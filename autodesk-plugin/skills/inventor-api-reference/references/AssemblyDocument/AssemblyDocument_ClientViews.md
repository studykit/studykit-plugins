# AssemblyDocument.ClientViews Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Returns document's client views.

## Syntax

AssemblyDocument.**ClientViews**() As [ClientViews](../ClientViews/ClientViews.md)

## Property Value

This is a read only property whose value is a [ClientViews](../ClientViews/ClientViews.md).

## Version

Introduced in version 2017
