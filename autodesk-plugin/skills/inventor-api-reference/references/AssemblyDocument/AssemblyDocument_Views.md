# AssemblyDocument.Views Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Gets all the open views of this document in a collection.

## Syntax

AssemblyDocument.**Views**() As [Views](../Views/Views.md)

## Property Value

This is a read only property whose value is a [Views](../Views/Views.md).

## Version

Introduced in version 4
