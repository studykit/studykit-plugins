# DirectEditScaleOperation.DirectEditOperationType Property

Parent Object: [DirectEditScaleOperation](../DirectEditScaleOperation/DirectEditScaleOperation.md)

## Description

Gets the direct edit operation type.

## Syntax

DirectEditScaleOperation.**DirectEditOperationType**() As [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md)

## Property Value

This is a read only property whose value is a [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md).

## Version

Introduced in version 2016
