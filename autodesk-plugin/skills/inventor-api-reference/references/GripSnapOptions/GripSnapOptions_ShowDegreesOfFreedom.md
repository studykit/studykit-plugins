# GripSnapOptions.ShowDegreesOfFreedom Property

Parent Object: [GripSnapOptions](../GripSnapOptions/GripSnapOptions.md)

## Description

Gets and sets whether or not to add boxes to the end of the heads-up display (HUD) for translational and rotational degrees of freedom related to the active component or assembly selection.

## Syntax

GripSnapOptions.**ShowDegreesOfFreedom**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
