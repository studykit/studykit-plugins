# DerivedFuturePartDefinition.FullFileName Property

Parent Object: [DerivedFuturePartDefinition](../DerivedFuturePartDefinition/DerivedFuturePartDefinition.md)

## Description

Read-only property that gets the full filename of the derived document.

## Syntax

DerivedFuturePartDefinition.**FullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
