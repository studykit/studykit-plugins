# DerivedFuturePartDefinition.Surfaces Property

Parent Object: [DerivedFuturePartDefinition](../DerivedFuturePartDefinition/DerivedFuturePartDefinition.md)

## Description

Read-only property that gets the collection of surfaces available in the selected part along with the inclusion option for each surface.

## Syntax

DerivedFuturePartDefinition.**Surfaces**() As [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md)

## Property Value

This is a read only property whose value is a [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md).

## Version

Introduced in version 2018
