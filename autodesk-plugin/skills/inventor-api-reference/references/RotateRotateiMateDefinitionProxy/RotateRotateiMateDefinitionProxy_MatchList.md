# RotateRotateiMateDefinitionProxy.MatchList Property

Parent Object: [RotateRotateiMateDefinitionProxy](../RotateRotateiMateDefinitionProxy/RotateRotateiMateDefinitionProxy.md)

## Description

Gets and sets an array of strings that specifies the priority order for iMate definition matches.

## Syntax

RotateRotateiMateDefinitionProxy.**MatchList**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 11
