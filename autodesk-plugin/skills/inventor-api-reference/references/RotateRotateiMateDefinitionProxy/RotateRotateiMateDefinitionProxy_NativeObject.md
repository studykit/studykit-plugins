# RotateRotateiMateDefinitionProxy.NativeObject Property

Parent Object: [RotateRotateiMateDefinitionProxy](../RotateRotateiMateDefinitionProxy/RotateRotateiMateDefinitionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RotateRotateiMateDefinitionProxy.**NativeObject**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version

Introduced in version 6
