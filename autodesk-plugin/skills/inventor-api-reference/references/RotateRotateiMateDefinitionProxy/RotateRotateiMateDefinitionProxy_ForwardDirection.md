# RotateRotateiMateDefinitionProxy.ForwardDirection Property

Parent Object: [RotateRotateiMateDefinitionProxy](../RotateRotateiMateDefinitionProxy/RotateRotateiMateDefinitionProxy.md)

## Description

Boolean property that returns the direction of rotation with respect to the axis direction. If True, both objects will rotate in the same direction around their axes. If False, then they will rotate in opposite directions.

## Syntax

RotateRotateiMateDefinitionProxy.**ForwardDirection**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
