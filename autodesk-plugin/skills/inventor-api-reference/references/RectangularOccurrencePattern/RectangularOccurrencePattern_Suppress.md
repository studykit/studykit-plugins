# RectangularOccurrencePattern.Suppress Method

Parent Object: [RectangularOccurrencePattern](../RectangularOccurrencePattern/RectangularOccurrencePattern.md)

## Description

Suppress the occurrence pattern.

## Syntax

RectangularOccurrencePattern.**Suppress**( [***SkipDocumentSave***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SkipDocumentSave | Boolean | Optional input Boolean that specifies whether to skip document save or not. Default to False indicating the dirty documents will be saved. |

## Version

Introduced in version 2022
