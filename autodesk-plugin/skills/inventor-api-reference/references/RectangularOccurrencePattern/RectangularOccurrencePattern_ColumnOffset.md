# RectangularOccurrencePattern.ColumnOffset Property

Parent Object: [RectangularOccurrencePattern](../RectangularOccurrencePattern/RectangularOccurrencePattern.md)

## Description

Property returning the Parameter object that controls the distance between columns.

## Syntax

RectangularOccurrencePattern.**ColumnOffset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
