# RectangularOccurrencePattern.ColumnEntityNaturalDirection Property

Parent Object: [RectangularOccurrencePattern](../RectangularOccurrencePattern/RectangularOccurrencePattern.md)

## Description

Gets/Sets whether the column direction is in the natural direction of the column entity or not.

## Syntax

RectangularOccurrencePattern.**ColumnEntityNaturalDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
