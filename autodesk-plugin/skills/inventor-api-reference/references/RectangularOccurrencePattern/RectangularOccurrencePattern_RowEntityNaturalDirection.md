# RectangularOccurrencePattern.RowEntityNaturalDirection Property

Parent Object: [RectangularOccurrencePattern](../RectangularOccurrencePattern/RectangularOccurrencePattern.md)

## Description

Gets/Sets whether the row direction is in the natural direction of the row entity or not.

## Syntax

RectangularOccurrencePattern.**RowEntityNaturalDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
