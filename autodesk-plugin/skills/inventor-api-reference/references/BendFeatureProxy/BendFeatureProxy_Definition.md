# BendFeatureProxy.Definition Property

Parent Object: [BendFeatureProxy](../BendFeatureProxy/BendFeatureProxy.md)

## Description

Property that gets the BendDefinition object associated with this bend feature.

## Syntax

BendFeatureProxy.**Definition**() As [BendDefinition](../BendDefinition/BendDefinition.md)

## Property Value

This is a read only property whose value is a [BendDefinition](../BendDefinition/BendDefinition.md).

## Version

Introduced in version 2009
