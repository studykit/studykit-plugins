# PhysicalPropertyEnum Enumerator

## Description

Enum indicating physical properties associated with a model.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kPhysicalModelArea | 72450 | Area of the model. |
| kPhysicalModelDensity | 72452 | Density of the model. |
| kPhysicalModelMass | 72449 | Mass of the model. |
| kPhysicalModelVolume | 72451 | Volume of the model. |

## Version

Introduced in version 2008
