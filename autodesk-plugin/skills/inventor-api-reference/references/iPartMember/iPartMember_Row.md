# iPartMember.Row Property

Parent Object: [iPartMember](../iPartMember/iPartMember.md)

## Description

Property that returns the row in the iPart table that this iPartMember represents.

## Syntax

iPartMember.**Row**() As [iPartTableRow](../iPartTableRow/iPartTableRow.md)

## Property Value

This is a read only property whose value is an [iPartTableRow](../iPartTableRow/iPartTableRow.md).

## Version

Introduced in version 6
