# DrawingView.Label Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Property that returns the DrawingViewLabel object.

## Syntax

DrawingView.**Label**() As [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Property Value

This is a read only property whose value is a [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md).

## Version

Introduced in version 2009
