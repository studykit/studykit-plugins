# DrawingView.ViewStyle Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Gets and sets the drawing view style.

## Syntax

DrawingView.**ViewStyle**() As [DrawingViewStyleEnum](../DrawingViewStyleEnum.md)

## Property Value

This is a read/write property whose value is a [DrawingViewStyleEnum](../DrawingViewStyleEnum.md).

## Version

Introduced in version 11
