# DrawingView.ReferencedDocumentDescriptor Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Property that returns the model document referenced by this view.

## Syntax

DrawingView.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).
