# DrawingView.IsRasterView Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Read-write property that gets and sets whether the drawing view is raster view or not.

## Syntax

DrawingView.**IsRasterView**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
