# DrawingView.IsTextPropertySource Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Read-write property that gets and sets whether the drawing view is specified as the text property source.This is applicable only when the drawing view is a base view.

## Syntax

DrawingView.**IsTextPropertySource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
