# DrawingView.GraphicsDataSetsCollection Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Property that returns the object for the drawing view.

## Syntax

DrawingView.**GraphicsDataSetsCollection**() As [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md).

## Version

Introduced in version 9
