# DrawingView.Center Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Gets and sets the display center point of the drawing view on the owning sheet.

## Syntax

DrawingView.**Center**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).
