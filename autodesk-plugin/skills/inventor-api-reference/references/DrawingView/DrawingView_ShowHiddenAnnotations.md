# DrawingView.ShowHiddenAnnotations Method

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Method that displays all the annotations hidden by the user.

## Syntax

DrawingView.**ShowHiddenAnnotations**()

## Version

Introduced in version 11
