# DrawingView.InheritBreakOut Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Gets and sets whether the view should inherit the corresponding break out from the parent view.

## Syntax

DrawingView.**InheritBreakOut**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
