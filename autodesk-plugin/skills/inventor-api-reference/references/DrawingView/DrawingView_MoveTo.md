# DrawingView.MoveTo Method

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Method that moves this drawing view into the specified sheet. The moved object is returned.

## Syntax

DrawingView.**MoveTo**( ***TargetSheet*** As [Sheet](../Sheet/Sheet.md) ) As [DrawingView](../DrawingView/DrawingView.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TargetSheet | [Sheet](../Sheet/Sheet.md) | Input Sheet object that specifies the sheet that this drawing view is to be moved into. An error occurs if the input sheet is the parent sheet object. |

## Version

Introduced in version 9
