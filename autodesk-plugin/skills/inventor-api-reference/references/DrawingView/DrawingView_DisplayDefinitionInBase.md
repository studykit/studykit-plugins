# DrawingView.DisplayDefinitionInBase Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Gets and sets whether to display detail circles, section lines and the associated text in the base view.

## Syntax

DrawingView.**DisplayDefinitionInBase**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
