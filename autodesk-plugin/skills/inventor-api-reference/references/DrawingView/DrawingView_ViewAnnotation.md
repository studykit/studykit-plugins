# DrawingView.ViewAnnotation Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Read-only property that returns the drawing view annotation object.

## Syntax

DrawingView.**ViewAnnotation**() As [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md)

## Property Value

This is a read only property whose value is a [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md).
