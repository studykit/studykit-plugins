# DrawingView.DisplayTrails Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Gets and sets whether to display trails.

## Syntax

DrawingView.**DisplayTrails**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
