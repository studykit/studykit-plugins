# DrawingView.Include3DAnnotations Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Read-write property that gets and sets whether to include the 3D annotations.

## Syntax

DrawingView.**Include3DAnnotations**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
