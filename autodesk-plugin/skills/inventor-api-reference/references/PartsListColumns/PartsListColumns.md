# PartsListColumns Object

## Description

The PartsListColumns collection object provides access to all existing objects in the parts list.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../PartsListColumns/PartsListColumns_Add.md) | Method that creates a new PartsListColumn based on a property. The newly created PartsListColumn is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PartsListColumns/PartsListColumns_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../PartsListColumns/PartsListColumns_Count.md) | Property that returns the number of items in this collection. |
| [Item](../PartsListColumns/PartsListColumns_Item.md) | Returns the specified PartsListColumn object from the collection. This is the default property of the PartsListColumns collection object. |
| [Type](../PartsListColumns/PartsListColumns_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartsList.PartsListColumns](../PartsList/PartsList_PartsListColumns.md)

## Version

Introduced in version 5.3
