# RevisionTableRow.Delete Method

Parent Object: [RevisionTableRow](../RevisionTableRow/RevisionTableRow.md)

## Description

Method that deletes the RevisionTableRow and any associated revision tags.

## Syntax

RevisionTableRow.**Delete**()

## Version

Introduced in version 10
