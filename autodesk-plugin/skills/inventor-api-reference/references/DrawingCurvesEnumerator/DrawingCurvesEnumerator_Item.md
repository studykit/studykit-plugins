# DrawingCurvesEnumerator.Item Property

Parent Object: [DrawingCurvesEnumerator](../DrawingCurvesEnumerator/DrawingCurvesEnumerator.md)

## Description

Method that returns the specified DrawingCurve object from the collection. This is the default property of the DrawingCurves collection object.

## Syntax

DrawingCurvesEnumerator.**Item**( ***Index*** As Long ) As [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Property Value

This is a read only property whose value is a [DrawingCurve](../DrawingCurve/DrawingCurve.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the object to return. |
