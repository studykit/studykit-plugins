# DrawingCurvesEnumerator Object

## Description

The DrawingCurvesEnumerator object provides access to drawing view curves.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingCurvesEnumerator/DrawingCurvesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DrawingCurvesEnumerator/DrawingCurvesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DrawingCurvesEnumerator/DrawingCurvesEnumerator_Item.md) | Method that returns the specified DrawingCurve object from the collection. This is the default property of the DrawingCurves collection object. |
| [Type](../DrawingCurvesEnumerator/DrawingCurvesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DetailDrawingView.DrawingCurves](../DetailDrawingView/DetailDrawingView_DrawingCurves.md), [DrawingView.DrawingCurves](../DrawingView/DrawingView_DrawingCurves.md), [SectionDrawingView.DrawingCurves](../SectionDrawingView/SectionDrawingView_DrawingCurves.md)

## Version

Introduced in version 11
