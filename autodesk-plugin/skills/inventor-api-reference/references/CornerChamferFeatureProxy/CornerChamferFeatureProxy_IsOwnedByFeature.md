# CornerChamferFeatureProxy.IsOwnedByFeature Property

Parent Object: [CornerChamferFeatureProxy](../CornerChamferFeatureProxy/CornerChamferFeatureProxy.md)

## Description

Property that returns whether this object is owned by a feature. If True, the OwnedBy property returns the owning feature.

## Syntax

CornerChamferFeatureProxy.**IsOwnedByFeature**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
