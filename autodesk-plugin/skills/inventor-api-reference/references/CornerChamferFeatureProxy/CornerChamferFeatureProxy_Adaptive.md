# CornerChamferFeatureProxy.Adaptive Property

Parent Object: [CornerChamferFeatureProxy](../CornerChamferFeatureProxy/CornerChamferFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

CornerChamferFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
