# CornerChamferFeatureProxy.HealthStatus Property

Parent Object: [CornerChamferFeatureProxy](../CornerChamferFeatureProxy/CornerChamferFeatureProxy.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

CornerChamferFeatureProxy.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 9
