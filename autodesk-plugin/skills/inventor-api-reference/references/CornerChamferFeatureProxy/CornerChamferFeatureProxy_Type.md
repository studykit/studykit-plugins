# CornerChamferFeatureProxy.Type Property

Parent Object: [CornerChamferFeatureProxy](../CornerChamferFeatureProxy/CornerChamferFeatureProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CornerChamferFeatureProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
