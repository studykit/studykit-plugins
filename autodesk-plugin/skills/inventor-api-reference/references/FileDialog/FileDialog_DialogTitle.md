# FileDialog.DialogTitle Property

Parent Object: [FileDialog](../FileDialog/FileDialog.md)

## Description

The string displayed in the dialog's title bar.

## Syntax

FileDialog.**DialogTitle**() As String

## Property Value

This is a read/write property whose value is a String.
