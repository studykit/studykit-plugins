# FileDialog.FilterIndex Property

Parent Object: [FileDialog](../FileDialog/FileDialog.md)

## Description

The index of the default filter in the filter string (first index is 1).

## Syntax

FileDialog.**FilterIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.
