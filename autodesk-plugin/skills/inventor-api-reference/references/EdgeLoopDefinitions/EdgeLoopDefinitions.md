# EdgeLoopDefinitions Object

## Description

The EdgeLoopsDefinitions collection provides access to existing EdgeLoopDefinition objects and allows creation of new EdgeLoopDefinition objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../EdgeLoopDefinitions/EdgeLoopDefinitions_Add.md) | Method that creates a new EdgeLoopDefinition within the associated SurfaceBodyDefinition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../EdgeLoopDefinitions/EdgeLoopDefinitions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../EdgeLoopDefinitions/EdgeLoopDefinitions_Count.md) | Property that returns the number of items in this collection. |
| [Item](../EdgeLoopDefinitions/EdgeLoopDefinitions_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../EdgeLoopDefinitions/EdgeLoopDefinitions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FaceDefinition.EdgeLoopDefinitions](../FaceDefinition/FaceDefinition_EdgeLoopDefinitions.md)

## Version

Introduced in version 2011
