# ModelHoleThreadNoteProxy.CompositeAnnotation Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Read-only property that returns the ModelCompositeAnnotation object if this annotation is a member of it. This returns Nothing if the IsMemberOfCompositeAnnotation returns False.

## Syntax

ModelHoleThreadNoteProxy.**CompositeAnnotation**() As [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md)

## Property Value

This is a read only property whose value is a [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md).

## Version

Introduced in version 2018
