# ModelHoleThreadNoteProxy.NativeObject Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ModelHoleThreadNoteProxy.**NativeObject**() As [ModelHoleThreadNote](../ModelHoleThreadNote/ModelHoleThreadNote.md)

## Property Value

This is a read only property whose value is a [ModelHoleThreadNote](../ModelHoleThreadNote/ModelHoleThreadNote.md).

## Version

Introduced in version 2018
