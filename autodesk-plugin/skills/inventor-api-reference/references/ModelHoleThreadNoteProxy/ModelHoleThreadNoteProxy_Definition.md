# ModelHoleThreadNoteProxy.Definition Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Gets and sets the hole/thread defintion associated with the note.

## Syntax

ModelHoleThreadNoteProxy.**Definition**() As [ModelHoleThreadNoteDefinition](../ModelHoleThreadNoteDefinition/ModelHoleThreadNoteDefinition.md)

## Property Value

This is a read/write property whose value is a [ModelHoleThreadNoteDefinition](../ModelHoleThreadNoteDefinition/ModelHoleThreadNoteDefinition.md).

## Version

Introduced in version 2018
