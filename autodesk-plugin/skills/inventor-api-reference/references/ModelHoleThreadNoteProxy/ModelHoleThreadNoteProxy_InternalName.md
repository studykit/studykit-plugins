# ModelHoleThreadNoteProxy.InternalName Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelHoleThreadNoteProxy.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
