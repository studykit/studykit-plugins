# ModelHoleThreadNoteProxy.IsOwnedByToleranceFeature Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Returns whether this annotation is owned by a ModelToleranceFeature.

## Syntax

ModelHoleThreadNoteProxy.**IsOwnedByToleranceFeature**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2019
