# ModelHoleThreadNoteProxy.OwnedByToleranceFeature Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Returns the owning ModelToleranceFeature object.

## Syntax

ModelHoleThreadNoteProxy.**OwnedByToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2019
