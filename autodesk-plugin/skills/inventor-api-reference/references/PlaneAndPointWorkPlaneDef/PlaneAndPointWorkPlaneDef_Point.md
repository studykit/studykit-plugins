# PlaneAndPointWorkPlaneDef.Point Property

Parent Object: [PlaneAndPointWorkPlaneDef](../PlaneAndPointWorkPlaneDef/PlaneAndPointWorkPlaneDef.md)

## Description

Property that returns the point of a plane and point defined work plane. The point can be a WorkPoint, Vertex, or SketchPoint object. The work plane passes through the point.

## Syntax

PlaneAndPointWorkPlaneDef.**Point**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
