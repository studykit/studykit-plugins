# BendExtensionTypeEnum Enumerator

## Description

Defines the constants used to specify the bend extension type for contour flange feature.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBendExtensionMatchShape | 130049 | Defines the bend extension match shape type. |
| kBendExtensionUniform | 130050 | Defines the bend extension uniform type. |

## Version

Introduced in version 2026
