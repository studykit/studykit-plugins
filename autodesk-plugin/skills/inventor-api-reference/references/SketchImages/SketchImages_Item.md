# SketchImages.Item Property

Parent Object: [SketchImages](../SketchImages/SketchImages.md)

## Description

Method that returns the specified Image object from the collection.

## Syntax

SketchImages.**Item**( ***Index*** As Variant ) As [SketchImage](../SketchImage/SketchImage.md)

## Property Value

This is a read only property whose value is a [SketchImage](../SketchImage/SketchImage.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Value that specifies the image to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the image name. If an out of range index or a name of a non-existent image is provided, an error occurs. |

## Version

Introduced in version 11
