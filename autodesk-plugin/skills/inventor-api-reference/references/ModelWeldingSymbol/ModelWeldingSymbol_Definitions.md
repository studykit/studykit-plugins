# ModelWeldingSymbol.Definitions Property

Parent Object: [ModelWeldingSymbol](../ModelWeldingSymbol/ModelWeldingSymbol.md)

## Description

Gets and sets the ModelWeldingSymbolDefinitions object associated with this symbol.

## Syntax

ModelWeldingSymbol.**Definitions**() As [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Property Value

This is a read/write property whose value is a [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md).

## Version

Introduced in version 2024
