# ModelWeldingSymbol.Name Property

Parent Object: [ModelWeldingSymbol](../ModelWeldingSymbol/ModelWeldingSymbol.md)

## Description

Read-write property that gets and sets the name of the annotation.

## Syntax

ModelWeldingSymbol.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2024
