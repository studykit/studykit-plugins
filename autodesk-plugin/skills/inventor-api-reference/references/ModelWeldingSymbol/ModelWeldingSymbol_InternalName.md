# ModelWeldingSymbol.InternalName Property

Parent Object: [ModelWeldingSymbol](../ModelWeldingSymbol/ModelWeldingSymbol.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelWeldingSymbol.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2024
