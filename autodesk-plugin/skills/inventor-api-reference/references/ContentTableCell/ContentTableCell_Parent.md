# ContentTableCell.Parent Property

Parent Object: [ContentTableCell](../ContentTableCell/ContentTableCell.md)

## Description

Property that returns the parent ContentFamily object.

## Syntax

ContentTableCell.**Parent**() As [ContentFamily](../ContentFamily/ContentFamily.md)

## Property Value

This is a read only property whose value is a [ContentFamily](../ContentFamily/ContentFamily.md).

## Version

Introduced in version 2010
