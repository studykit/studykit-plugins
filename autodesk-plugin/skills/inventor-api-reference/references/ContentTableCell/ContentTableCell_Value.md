# ContentTableCell.Value Property

Parent Object: [ContentTableCell](../ContentTableCell/ContentTableCell.md)

## Description

Gets/Sets the value of the cell.

## Syntax

ContentTableCell.**Value**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
