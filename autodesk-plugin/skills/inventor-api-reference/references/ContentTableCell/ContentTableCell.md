# ContentTableCell Object

## Description

The ContentTableCell object represents a specific cell with a content center family table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ContentTableCell/ContentTableCell_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Column](../ContentTableCell/ContentTableCell_Column.md) | Returns the column index of this cell within the parent ContentFamily. |
| [IsOverridden](../ContentTableCell/ContentTableCell_IsOverridden.md) | Property that indicates if the value of this sell has been overwritten from its original value. |
| [Parent](../ContentTableCell/ContentTableCell_Parent.md) | Property that returns the parent ContentFamily object. |
| [Row](../ContentTableCell/ContentTableCell_Row.md) | Returns the row index of this cell within the parent ContentFamily. |
| [Type](../ContentTableCell/ContentTableCell_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Value](../ContentTableCell/ContentTableCell_Value.md) | Gets/Sets the value of the cell. |

## Accessed From

[ContentFamily.ActiveCell](../ContentFamily/ContentFamily_ActiveCell.md), [ContentTableRow.Item](../ContentTableRow/ContentTableRow_Item.md)

## Version

Introduced in version 2010
