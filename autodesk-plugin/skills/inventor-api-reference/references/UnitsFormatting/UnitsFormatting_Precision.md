# UnitsFormatting.Precision Property

Parent Object: [UnitsFormatting](../UnitsFormatting/UnitsFormatting.md)

## Description

Gets and sets the precision.

## Syntax

UnitsFormatting.**Precision**() As [CustomPropertyPrecisionEnum](../CustomPropertyPrecisionEnum.md)

## Property Value

This is a read/write property whose value is a [CustomPropertyPrecisionEnum](../CustomPropertyPrecisionEnum.md).

## Version

Introduced in version 2021
