# WorkPointProxy.Point Property

Parent Object: [WorkPointProxy](../WorkPointProxy/WorkPointProxy.md)

## Description

Property that returns a Point geometry object. The Point object returned provides information about the position of the work point.

## Syntax

WorkPointProxy.**Point**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 5
