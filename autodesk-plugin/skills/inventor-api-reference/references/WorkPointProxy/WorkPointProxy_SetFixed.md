# WorkPointProxy.SetFixed Method

Parent Object: [WorkPointProxy](../WorkPointProxy/WorkPointProxy.md)

## Description

Method that redefines the work point to be at the position specified by the input point.

## Syntax

WorkPointProxy.**SetFixed**( ***Point*** As [Point](../Point/Point.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input Point object. |

## Version

Introduced in version 2008
