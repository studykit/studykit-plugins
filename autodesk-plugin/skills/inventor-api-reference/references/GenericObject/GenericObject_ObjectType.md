# GenericObject.ObjectType Property

Parent Object: [GenericObject](../GenericObject/GenericObject.md)

## Description

Gets the constant that indicates the real type of this object.

## Syntax

GenericObject.**ObjectType**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
