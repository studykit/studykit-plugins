# RotateTranslateConstraintProxy.Visible Property

Parent Object: [RotateTranslateConstraintProxy](../RotateTranslateConstraintProxy/RotateTranslateConstraintProxy.md)

## Description

Gets/Sets the Visible indicating whether this constraint is visible.

## Syntax

RotateTranslateConstraintProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
