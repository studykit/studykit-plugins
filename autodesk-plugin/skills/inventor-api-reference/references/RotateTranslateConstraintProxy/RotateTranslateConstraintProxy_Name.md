# RotateTranslateConstraintProxy.Name Property

Parent Object: [RotateTranslateConstraintProxy](../RotateTranslateConstraintProxy/RotateTranslateConstraintProxy.md)

## Description

Gets/Sets the displayable name of this constraint.

## Syntax

RotateTranslateConstraintProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
