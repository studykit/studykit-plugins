# PluginLicenseIdleModeEnum Enumerator

## Description

Constants that define the plugin license idle mode.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDeferToIdleMode | 121090 | Defer to enter idle mode. |
| kEnterIdleMode | 121089 | Enter idle mode. |
| kExitIdleMode | 121091 | Exit idle mode. |

## Version

Introduced in version 2023
