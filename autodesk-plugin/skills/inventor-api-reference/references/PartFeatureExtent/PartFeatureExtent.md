# PartFeatureExtent Object

## Description

The PartFeatureExtent object is the base class that defines the extent for certain part features--those that are profile-based.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PartFeatureExtent/PartFeatureExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../PartFeatureExtent/PartFeatureExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Type](../PartFeatureExtent/PartFeatureExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
