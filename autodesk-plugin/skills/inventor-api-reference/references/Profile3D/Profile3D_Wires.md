# Profile3D.Wires Property

Parent Object: [Profile3D](../Profile3D/Profile3D.md)

## Description

Property returning the Wires collection object associated with this Profile3D.

## Syntax

Profile3D.**Wires**() As [Wires](../Wires/Wires.md)

## Property Value

This is a read only property whose value is a [Wires](../Wires/Wires.md).

## Version

Introduced in version 2009
