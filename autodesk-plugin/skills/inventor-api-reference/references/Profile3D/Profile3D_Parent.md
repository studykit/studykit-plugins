# Profile3D.Parent Property

Parent Object: [Profile3D](../Profile3D/Profile3D.md)

## Description

Property that returns the 3D sketch that the profile was derived from.

## Syntax

Profile3D.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 6
