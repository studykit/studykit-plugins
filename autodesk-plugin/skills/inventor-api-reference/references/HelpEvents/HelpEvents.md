# HelpEvents Object

## Description

Object that provides access to help related events.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HelpEvents/HelpEvents_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Type](../HelpEvents/HelpEvents_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Events

|  |  |
| --- | --- |
| Name | Description |
| [OnApplicationHelp](../HelpEvents/HelpEvents_OnApplicationHelp.md) | Event that fires before Application Help is displayed. |

## Accessed From

[HelpManager.HelpEvents](../HelpManager/HelpManager_HelpEvents.md)

## Version

Introduced in version 2012
