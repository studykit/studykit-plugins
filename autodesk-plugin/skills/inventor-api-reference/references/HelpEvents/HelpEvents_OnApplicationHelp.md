# HelpEvents.OnApplicationHelp Event

Parent Object: [HelpEvents](../HelpEvents/HelpEvents.md)

## Description

Event that fires before Application Help is displayed.

## Syntax

HelpEvents.**OnApplicationHelp**( ***Context*** As [NameValueMap](../NameValueMap/NameValueMap.md), ***HandlingCode*** As [HandlingCodeEnum](../HandlingCodeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Context | [NameValueMap](../NameValueMap/NameValueMap.md) | Input NameValueMap object that can be used to determine the context of why the event fired. |
| HandlingCode | [HandlingCodeEnum](../HandlingCodeEnum.md) | Output HandlingCodeEnum that indicates how you are handling the event. |

## Version

Introduced in version 2012
