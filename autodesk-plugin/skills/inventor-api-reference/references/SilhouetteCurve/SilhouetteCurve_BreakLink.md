# SilhouetteCurve.BreakLink Method

Parent Object: [SilhouetteCurve](../SilhouetteCurve/SilhouetteCurve.md)

## Description

Method that breaks the link between the silhouette curve and the model.

## Syntax

SilhouetteCurve.**BreakLink**()

## Version

Introduced in version 2012
