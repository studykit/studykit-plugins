# SilhouetteCurve.Parent Property

Parent Object: [SilhouetteCurve](../SilhouetteCurve/SilhouetteCurve.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SilhouetteCurve.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2012
