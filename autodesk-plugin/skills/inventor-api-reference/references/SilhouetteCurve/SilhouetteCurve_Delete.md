# SilhouetteCurve.Delete Method

Parent Object: [SilhouetteCurve](../SilhouetteCurve/SilhouetteCurve.md)

## Description

Method that deletes the silhouette curve. This will delete all of the related sketch entities.

## Syntax

SilhouetteCurve.**Delete**()

## Version

Introduced in version 2012
