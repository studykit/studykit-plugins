# DerivedFuturePartComponent.SuppressLinkToFile Property

Parent Object: [DerivedFuturePartComponent](../DerivedFuturePartComponent/DerivedFuturePartComponent.md)

## Description

Read-write property that suppresses or unsuppresses the link to the base part.

## Syntax

DerivedFuturePartComponent.**SuppressLinkToFile**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
