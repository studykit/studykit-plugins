# ControlDefinition.StandardIcon Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Gets/Sets StandardIcon. For MacroControlDefinition this property is read-only.

## Syntax

ControlDefinition.**StandardIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2011
