# ControlDefinition.ProgressiveToolTip Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Property that returns a ProgressiveToolTip object providing access to enhanced tooltip display for controls in the ribbon interface.

## Syntax

ControlDefinition.**ProgressiveToolTip**() As [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md)

## Property Value

This is a read only property whose value is a [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md).

## Version

Introduced in version 2011
