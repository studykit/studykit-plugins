# ControlDefinition.OverrideShortcutType Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Property that returns the type of override shortcut assigned to this command.

## Syntax

ControlDefinition.**OverrideShortcutType**() As [ShortcutTypeEnum](../ShortcutTypeEnum.md)

## Property Value

This is a read only property whose value is a [ShortcutTypeEnum](../ShortcutTypeEnum.md).

## Version

Introduced in version 2008
