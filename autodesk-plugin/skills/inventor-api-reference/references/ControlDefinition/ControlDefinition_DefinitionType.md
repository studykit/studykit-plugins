# ControlDefinition.DefinitionType Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Property that returns the control definition type. The possible return values are kButtonDefinition, kComboBoxDefinition, and kMacroControlDefinition.

## Syntax

ControlDefinition.**DefinitionType**() As [ControlDefinitionTypeEnum](../ControlDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [ControlDefinitionTypeEnum](../ControlDefinitionTypeEnum.md).

## Version

Introduced in version 9
