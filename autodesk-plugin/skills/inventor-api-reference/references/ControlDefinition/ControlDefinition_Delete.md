# ControlDefinition.Delete Method

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Method that deletes the control definition. This method fails for built-in definitions.

## Syntax

ControlDefinition.**Delete**()

## Version

Introduced in version 9
