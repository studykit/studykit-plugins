# ControlDefinition.IsShortcutOverridden Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Property that returns whether the default (internal) shortcut has been overridden by the user or through the API.

## Syntax

ControlDefinition.**IsShortcutOverridden**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
