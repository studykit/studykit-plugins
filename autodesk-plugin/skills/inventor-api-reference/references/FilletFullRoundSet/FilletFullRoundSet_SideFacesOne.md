# FilletFullRoundSet.SideFacesOne Property

Parent Object: [FilletFullRoundSet](../FilletFullRoundSet/FilletFullRoundSet.md)

## Description

Property that returns the first set of side faces for the full round fillet feature.

## Syntax

FilletFullRoundSet.**SideFacesOne**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read only property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 11
