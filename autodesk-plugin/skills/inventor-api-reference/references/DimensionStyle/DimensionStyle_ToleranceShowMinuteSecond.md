# DimensionStyle.ToleranceShowMinuteSecond Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to show angles using fractional degrees or minutes and seconds.

## Syntax

DimensionStyle.**ToleranceShowMinuteSecond**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
