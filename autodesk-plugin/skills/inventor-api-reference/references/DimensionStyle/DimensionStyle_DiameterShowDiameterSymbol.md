# DimensionStyle.DiameterShowDiameterSymbol Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the diameter symbol should be shown in a diameter dimension.

## Syntax

DimensionStyle.**DiameterShowDiameterSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
