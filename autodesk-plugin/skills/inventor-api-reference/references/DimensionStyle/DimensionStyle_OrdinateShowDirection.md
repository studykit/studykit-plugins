# DimensionStyle.OrdinateShowDirection Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the direction arrow should be shown or not for ordinate dimensions.

## Syntax

DimensionStyle.**OrdinateShowDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
