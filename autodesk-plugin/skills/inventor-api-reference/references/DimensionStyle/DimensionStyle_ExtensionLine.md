# DimensionStyle.ExtensionLine Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the length of the extension line in centimeters.

## Syntax

DimensionStyle.**ExtensionLine**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2026
