# DimensionStyle.DisplayFormat Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the display format for all non-angular dimensions.

## Syntax

DimensionStyle.**DisplayFormat**() As [DisplayFormatEnum](../DisplayFormatEnum.md)

## Property Value

This is a read/write property whose value is a [DisplayFormatEnum](../DisplayFormatEnum.md).

## Version

Introduced in version 2009
