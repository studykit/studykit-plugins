# DimensionStyle.LinearMultiLineTextOrientation Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Read-write property that gets and sets the position of multiple lines of text relative to the landing line in a linear dimension. Valid values are kAllAboveLandingLine and kFirstLineAboveLandingLine.

## Syntax

DimensionStyle.**LinearMultiLineTextOrientation**() As [MultiLineDimensionTextEnum](../MultiLineDimensionTextEnum.md)

## Property Value

This is a read/write property whose value is a [MultiLineDimensionTextEnum](../MultiLineDimensionTextEnum.md).

## Version

Introduced in version 2014
