# DimensionStyle.AngularUseQuadrant Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to use a quadrant or ignore the quadrant angular dimension text.

## Syntax

DimensionStyle.**AngularUseQuadrant**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
