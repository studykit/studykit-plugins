# DimensionStyle.LinearHideExtensionLineTwo Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the second extension line of a linear dimension is hidden.

## Syntax

DimensionStyle.**LinearHideExtensionLineTwo**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
