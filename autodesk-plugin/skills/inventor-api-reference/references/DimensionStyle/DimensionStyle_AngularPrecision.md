# DimensionStyle.AngularPrecision Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the precision for angular dimensions in the drawing.

## Syntax

DimensionStyle.**AngularPrecision**() As [AngularPrecisionEnum](../AngularPrecisionEnum.md)

## Property Value

This is a read/write property whose value is an [AngularPrecisionEnum](../AngularPrecisionEnum.md).
