# DimensionStyle.RadialJoggedLeader Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the leader line of a radial dimension is jogged or not.

## Syntax

DimensionStyle.**RadialJoggedLeader**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
