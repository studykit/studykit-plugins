# DimensionStyle.AlternateLinearPrecision Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the precision for linear alternate dimensions in the drawing.

## Syntax

DimensionStyle.**AlternateLinearPrecision**() As [LinearPrecisionEnum](../LinearPrecisionEnum.md)

## Property Value

This is a read/write property whose value is a [LinearPrecisionEnum](../LinearPrecisionEnum.md).

## Version

Introduced in version 9
