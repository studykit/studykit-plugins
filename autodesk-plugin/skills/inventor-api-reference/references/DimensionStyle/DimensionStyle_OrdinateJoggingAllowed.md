# DimensionStyle.OrdinateJoggingAllowed Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if jogged leaders for ordinate dimensions is allowed or not.

## Syntax

DimensionStyle.**OrdinateJoggingAllowed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
