# DimensionStyle.ShowBreakSymbol Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display break symbols for dimensions on a broken view.

## Syntax

DimensionStyle.**ShowBreakSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
