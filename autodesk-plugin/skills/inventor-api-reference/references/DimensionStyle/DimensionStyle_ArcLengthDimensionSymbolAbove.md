# DimensionStyle.ArcLengthDimensionSymbolAbove Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether the arc length dimension symbol is above the arc length dimension value or before it.

## Syntax

DimensionStyle.**ArcLengthDimensionSymbolAbove**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
