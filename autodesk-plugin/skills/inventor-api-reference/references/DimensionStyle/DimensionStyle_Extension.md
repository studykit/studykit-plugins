# DimensionStyle.Extension Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the distance the extension line is drawn past the dimension line.

## Syntax

DimensionStyle.**Extension**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
