# DimensionStyle.HoleNoteDualFormat Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the arrangement of the alternate dimension display for hole notes.

## Syntax

DimensionStyle.**HoleNoteDualFormat**() As [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md).

## Version

Introduced in version 2014
