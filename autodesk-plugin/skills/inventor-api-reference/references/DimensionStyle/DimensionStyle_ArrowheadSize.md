# DimensionStyle.ArrowheadSize Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the size of the arrowhead.

## Syntax

DimensionStyle.**ArrowheadSize**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
