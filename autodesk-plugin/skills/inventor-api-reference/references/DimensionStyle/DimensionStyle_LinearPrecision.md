# DimensionStyle.LinearPrecision Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the precision value for linear dimensions in the drawing.

## Syntax

DimensionStyle.**LinearPrecision**() As [LinearPrecisionEnum](../LinearPrecisionEnum.md)

## Property Value

This is a read/write property whose value is a [LinearPrecisionEnum](../LinearPrecisionEnum.md).
