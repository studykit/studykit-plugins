# DimensionStyle.OrdinatePositiveBothDirections Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if positive values should be shown in both directions or not for an ordinate dimension.

## Syntax

DimensionStyle.**OrdinatePositiveBothDirections**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
