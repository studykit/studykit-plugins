# DimensionStyle.IncludedZeroDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the included zeroes for dimensions in the drawing.

## Syntax

DimensionStyle.**IncludedZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
