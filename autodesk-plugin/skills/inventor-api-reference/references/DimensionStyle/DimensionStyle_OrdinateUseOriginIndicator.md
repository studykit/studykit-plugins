# DimensionStyle.OrdinateUseOriginIndicator Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the origin indicator is to be used or not.

## Syntax

DimensionStyle.**OrdinateUseOriginIndicator**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
