# DimensionStyle.AlternateFractionalTextScale Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the size of the stacked text relative to the default text size when using alternate units. The valid range of values is 0.25 to 1.25.

## Syntax

DimensionStyle.**AlternateFractionalTextScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
