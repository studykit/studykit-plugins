# DimensionStyle.ToleranceLeadingZeroDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the leading zeroes for tolerances.

## Syntax

DimensionStyle.**ToleranceLeadingZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
