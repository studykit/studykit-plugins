# DimensionStyle.ToleranceAlternateIncludedZeroDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the included zeroes for tolerances in alternate units.

## Syntax

DimensionStyle.**ToleranceAlternateIncludedZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
