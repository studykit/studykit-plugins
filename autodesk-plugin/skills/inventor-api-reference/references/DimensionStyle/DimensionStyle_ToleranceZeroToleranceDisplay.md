# DimensionStyle.ToleranceZeroToleranceDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets how to display zero tolerances.

## Syntax

DimensionStyle.**ToleranceZeroToleranceDisplay**() As [DimensionZeroToleranceDisplayEnum](../DimensionZeroToleranceDisplayEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionZeroToleranceDisplayEnum](../DimensionZeroToleranceDisplayEnum.md).

## Version

Introduced in version 2014
