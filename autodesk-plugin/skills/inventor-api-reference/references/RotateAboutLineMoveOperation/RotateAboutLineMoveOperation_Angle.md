# RotateAboutLineMoveOperation.Angle Property

Parent Object: [RotateAboutLineMoveOperation](../RotateAboutLineMoveOperation/RotateAboutLineMoveOperation.md)

## Description

Read-write property that provides access to the angle of a rotate about line operation on a move body feature.

## Syntax

RotateAboutLineMoveOperation.**Angle**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2013
