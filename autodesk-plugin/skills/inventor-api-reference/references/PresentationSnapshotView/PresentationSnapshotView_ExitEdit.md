# PresentationSnapshotView.ExitEdit Method

Parent Object: [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

Method that exits the presentation snapshot view edit mode.

## Syntax

PresentationSnapshotView.**ExitEdit**()
