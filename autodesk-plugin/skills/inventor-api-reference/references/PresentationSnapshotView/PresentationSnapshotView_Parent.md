# PresentationSnapshotView.Parent Property

Parent Object: [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PresentationSnapshotView.**Parent**() As [PresentationScene](../PresentationScene/PresentationScene.md)

## Property Value

This is a read only property whose value is a [PresentationScene](../PresentationScene/PresentationScene.md).

## Version

Introduced in version 2018
