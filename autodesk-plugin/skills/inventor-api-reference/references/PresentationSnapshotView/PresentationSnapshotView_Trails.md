# PresentationSnapshotView.Trails Property

Parent Object: [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Description

Read-only property that returns the PresentationTrails collection object.

## Syntax

PresentationSnapshotView.**Trails**() As [PresentationTrails](../PresentationTrails/PresentationTrails.md)

## Property Value

This is a read only property whose value is a [PresentationTrails](../PresentationTrails/PresentationTrails.md).

## Version

Introduced in version 2018
