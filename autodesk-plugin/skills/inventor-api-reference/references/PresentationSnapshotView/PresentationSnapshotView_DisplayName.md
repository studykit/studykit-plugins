# PresentationSnapshotView.DisplayName Property

Parent Object: [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Description

Read-write property that gets and sets the presentation snapshot view name.

## Syntax

PresentationSnapshotView.**DisplayName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
