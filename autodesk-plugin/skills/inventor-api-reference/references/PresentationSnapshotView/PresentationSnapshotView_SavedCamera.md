# PresentationSnapshotView.SavedCamera Property

Parent Object: [PresentationSnapshotView](../PresentationSnapshotView/PresentationSnapshotView.md)

## Description

Read-only property that returns the saved camera for current presentation snapshot view.

## Syntax

PresentationSnapshotView.**SavedCamera**() As [Camera](../Camera/Camera.md)

## Property Value

This is a read only property whose value is a [Camera](../Camera/Camera.md).

## Version

Introduced in version 2018
