# SketchCircleProxy.Length Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Property that returns the length of the entity in centimeters.

## Syntax

SketchCircleProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
