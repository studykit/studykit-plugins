# CoincidentConstraint3D.Deletable Property

Parent Object: [CoincidentConstraint3D](../CoincidentConstraint3D/CoincidentConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

CoincidentConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
