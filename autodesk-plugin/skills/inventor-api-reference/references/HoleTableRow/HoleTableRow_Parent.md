# HoleTableRow.Parent Property

Parent Object: [HoleTableRow](../HoleTableRow/HoleTableRow.md)

## Description

Property that returns the parent HoleTable.

## Syntax

HoleTableRow.**Parent**() As [HoleTable](../HoleTable/HoleTable.md)

## Property Value

This is a read only property whose value is a [HoleTable](../HoleTable/HoleTable.md).

## Version

Introduced in version 10
