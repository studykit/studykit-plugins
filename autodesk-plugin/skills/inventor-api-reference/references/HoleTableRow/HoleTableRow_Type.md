# HoleTableRow.Type Property

Parent Object: [HoleTableRow](../HoleTableRow/HoleTableRow.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

HoleTableRow.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
