# HoleTableRow Object

## Description

The HoleTableRow object defines a row within the hole table, including the row height.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../HoleTableRow/HoleTableRow_Delete.md) | Method that deletes the HoleTableRow and the associated hole tag. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HoleTableRow/HoleTableRow_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../HoleTableRow/HoleTableRow_Count.md) | Property that returns the number of items in the collection. |
| [Height](../HoleTableRow/HoleTableRow_Height.md) | Gets/Sets the height of the HoleTable row. |
| [HoleTag](../HoleTableRow/HoleTableRow_HoleTag.md) | Property that returns the hole tag associated with this hole in the table. |
| [Item](../HoleTableRow/HoleTableRow_Item.md) | Returns the specified Cell object from the collection. |
| [Parent](../HoleTableRow/HoleTableRow_Parent.md) | Property that returns the parent HoleTable. |
| [ReferencedHole](../HoleTableRow/HoleTableRow_ReferencedHole.md) | Property that returns the hole associated with this row. |
| [Type](../HoleTableRow/HoleTableRow_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[HoleTableRows.Add](../HoleTableRows/HoleTableRows_Add.md), [HoleTableRows.Item](../HoleTableRows/HoleTableRows_Item.md), [HoleTag.Parent](../HoleTag/HoleTag_Parent.md)
