# SplineFitPointConstraintProxy.NativeObject Property

Parent Object: [SplineFitPointConstraintProxy](../SplineFitPointConstraintProxy/SplineFitPointConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SplineFitPointConstraintProxy.**NativeObject**() As [SplineFitPointConstraint](../SplineFitPointConstraint/SplineFitPointConstraint.md)

## Property Value

This is a read only property whose value is a [SplineFitPointConstraint](../SplineFitPointConstraint/SplineFitPointConstraint.md).

## Version

Introduced in version 6
