# SplineFitPointConstraintProxy.Point Property

Parent Object: [SplineFitPointConstraintProxy](../SplineFitPointConstraintProxy/SplineFitPointConstraintProxy.md)

## Description

Property that returns the sketch point the spline is fit through.

## Syntax

SplineFitPointConstraintProxy.**Point**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 6
