# SplineFitPointConstraintProxy.Spline Property

Parent Object: [SplineFitPointConstraintProxy](../SplineFitPointConstraintProxy/SplineFitPointConstraintProxy.md)

## Description

Property that returns the spline entity being constrained.

## Syntax

SplineFitPointConstraintProxy.**Spline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Property Value

This is a read only property whose value is a [SketchSpline](../SketchSpline/SketchSpline.md).

## Version

Introduced in version 6
