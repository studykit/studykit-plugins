# SplineFitPointConstraintProxy.Deletable Property

Parent Object: [SplineFitPointConstraintProxy](../SplineFitPointConstraintProxy/SplineFitPointConstraintProxy.md)

## Description

Indicates whether this object is deletable.

## Syntax

SplineFitPointConstraintProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
