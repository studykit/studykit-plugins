# PerpendicularConstraintProxy.Deletable Property

Parent Object: [PerpendicularConstraintProxy](../PerpendicularConstraintProxy/PerpendicularConstraintProxy.md)

## Description

Indicates whether this object is deletable.

## Syntax

PerpendicularConstraintProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
