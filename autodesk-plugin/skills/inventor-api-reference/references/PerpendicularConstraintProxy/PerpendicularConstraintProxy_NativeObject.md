# PerpendicularConstraintProxy.NativeObject Property

Parent Object: [PerpendicularConstraintProxy](../PerpendicularConstraintProxy/PerpendicularConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

PerpendicularConstraintProxy.**NativeObject**() As [PerpendicularConstraint](../PerpendicularConstraint/PerpendicularConstraint.md)

## Property Value

This is a read only property whose value is a [PerpendicularConstraint](../PerpendicularConstraint/PerpendicularConstraint.md).

## Version

Introduced in version 6
