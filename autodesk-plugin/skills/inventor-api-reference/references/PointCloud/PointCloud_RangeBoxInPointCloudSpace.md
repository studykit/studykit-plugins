# PointCloud.RangeBoxInPointCloudSpace Property

Parent Object: [PointCloud](../PointCloud/PointCloud.md)

## Description

Read-only property that returns a Box object which contains the opposing points of a rectangular box that is guaranteed to enclose this object in point cloud space.

## Syntax

PointCloud.**RangeBoxInPointCloudSpace**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2016
