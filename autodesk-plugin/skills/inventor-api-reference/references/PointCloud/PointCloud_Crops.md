# PointCloud.Crops Property

Parent Object: [PointCloud](../PointCloud/PointCloud.md)

## Description

Read-only property that returns the PointCloudCrops collection object.

## Syntax

PointCloud.**Crops**() As [PointCloudCrops](../PointCloudCrops/PointCloudCrops.md)

## Property Value

This is a read only property whose value is a [PointCloudCrops](../PointCloudCrops/PointCloudCrops.md).

## Version

Introduced in version 2016
