# PointCloud.SourceFullFileName Property

Parent Object: [PointCloud](../PointCloud/PointCloud.md)

## Description

Read-only property that returns the point cloud source full file name.

## Syntax

PointCloud.**SourceFullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
