# PointCloud.ModelToPointCloudSpace Method

Parent Object: [PointCloud](../PointCloud/PointCloud.md)

## Description

Method that takes a 3d coordinate in model space and returns a Point object containing the coordinate point in point cloud space.

## Syntax

PointCloud.**ModelToPointCloudSpace**( ***ModelCoordinate*** As [Point](../Point/Point.md) ) As [Point](../Point/Point.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ModelCoordinate | [Point](../Point/Point.md) | Input Point object of a point in the model space. |

## Version

Introduced in version 2016
