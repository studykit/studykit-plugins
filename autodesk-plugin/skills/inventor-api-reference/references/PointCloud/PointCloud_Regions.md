# PointCloud.Regions Property

Parent Object: [PointCloud](../PointCloud/PointCloud.md)

## Description

Read-only property that returns the point cloud regions collection object.

## Syntax

PointCloud.**Regions**() As [PointCloudRegions](../PointCloudRegions/PointCloudRegions.md)

## Property Value

This is a read only property whose value is a [PointCloudRegions](../PointCloudRegions/PointCloudRegions.md).

## Version

Introduced in version 2016
