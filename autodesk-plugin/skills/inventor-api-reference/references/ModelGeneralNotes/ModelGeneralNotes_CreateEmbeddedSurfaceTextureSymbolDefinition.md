# ModelGeneralNotes.CreateEmbeddedSurfaceTextureSymbolDefinition Method

Parent Object: [ModelGeneralNotes](../ModelGeneralNotes/ModelGeneralNotes.md)

## Description

Method that creates a ModelSurfaceTextureSymbolDefinition. This ModelSurfaceTextureSymbolDefinition is to define the model surface texture symbol which is embedded in the model general note.

## Syntax

ModelGeneralNotes.**CreateEmbeddedSurfaceTextureSymbolDefinition**() As [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Version

Introduced in version 2018
