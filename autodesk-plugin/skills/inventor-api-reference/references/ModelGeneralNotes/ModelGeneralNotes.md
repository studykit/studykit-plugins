# ModelGeneralNotes Object

## Description

The ModelGeneralNotes collection object provides access to all of the general notes in a part or assembly and provides functionality to create new general notes.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ModelGeneralNotes/ModelGeneralNotes_Add.md) | Method that creates a general note. |
| [CreateDefinition](../ModelGeneralNotes/ModelGeneralNotes_CreateDefinition.md) | Method that creates a general note definition. This is a not a general note but an object that encapsulates all of the information that defines a general note. |
| [CreateEmbeddedFeatureControlFrameDefinition](../ModelGeneralNotes/ModelGeneralNotes_CreateEmbeddedFeatureControlFrameDefinition.md) | Method that creates a ModelFeatureControlFrameDefinition. This ModelFeatureControlFrameDefinition is to define the model feature control frame symbol which is embedded in the model general note. |
| [CreateEmbeddedSurfaceTextureSymbolDefinition](../ModelGeneralNotes/ModelGeneralNotes_CreateEmbeddedSurfaceTextureSymbolDefinition.md) | Method that creates a ModelSurfaceTextureSymbolDefinition. This ModelSurfaceTextureSymbolDefinition is to define the model surface texture symbol which is embedded in the model general note. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ModelGeneralNotes/ModelGeneralNotes_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../ModelGeneralNotes/ModelGeneralNotes_Count.md) | Gets the number of items in this collection. |
| [Item](../ModelGeneralNotes/ModelGeneralNotes_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../ModelGeneralNotes/ModelGeneralNotes_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From
