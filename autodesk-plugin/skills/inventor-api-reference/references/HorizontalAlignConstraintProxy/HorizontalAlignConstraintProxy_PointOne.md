# HorizontalAlignConstraintProxy.PointOne Property

Parent Object: [HorizontalAlignConstraintProxy](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy.md)

## Description

Property that returns the first sketch point being constrained.

## Syntax

HorizontalAlignConstraintProxy.**PointOne**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 6
