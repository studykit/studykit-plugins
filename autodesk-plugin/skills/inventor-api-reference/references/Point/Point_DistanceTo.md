# Point.DistanceTo Method

Parent Object: [Point](../Point/Point.md)

## Description

Determine the distance between this point and the specified point.

## Syntax

Point.**DistanceTo**( ***Point*** As [Point](../Point/Point.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input object that specifies the point. |

## Version

Introduced in version 4
