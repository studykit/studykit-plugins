# Point Object

## Description

The Point object. The object is a transient mathematical object and is not displayed graphically. For more information see the Transient Geometry article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Point/Point_Copy.md) | Creates a copy of this Point object. The result is entirely independent and can be edited without affecting the original Point object. |
| [DistanceTo](../Point/Point_DistanceTo.md) | Determine the distance between this point and the specified point. |
| [GetPointData](../Point/Point_GetPointData.md) | Get the data defining this point. |
| [IsEqualTo](../Point/Point_IsEqualTo.md) | Compares this point for equality with the specified point. |
| [PutPointData](../Point/Point_PutPointData.md) | Method that sets the data defining this point. |
| [TransformBy](../Point/Point_TransformBy.md) | Transform this point by the specified matrix. |
| [TranslateBy](../Point/Point_TranslateBy.md) | Translate this point by the specified vector. |
| [VectorTo](../Point/Point_VectorTo.md) | Gets the vector of translation between this point and the specified point. |
