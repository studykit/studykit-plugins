# Point.Z Property

Parent Object: [Point](../Point/Point.md)

## Description

Read-write proerty that gets and sets the Z coordinate value.

## Syntax

Point.**Z**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 4
