# Point.TranslateBy Method

Parent Object: [Point](../Point/Point.md)

## Description

Translate this point by the specified vector.

## Syntax

Point.**TranslateBy**( ***Vector*** As [Vector](../Vector/Vector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [Vector](../Vector/Vector.md) | Input Vector object that specifies the vector to translate by. |
