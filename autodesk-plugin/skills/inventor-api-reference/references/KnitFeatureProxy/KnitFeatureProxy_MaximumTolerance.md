# KnitFeatureProxy.MaximumTolerance Property

Parent Object: [KnitFeatureProxy](../KnitFeatureProxy/KnitFeatureProxy.md)

## Description

Gets and sets the maximum distance between the edges of two faces before being knit together.

## Syntax

KnitFeatureProxy.**MaximumTolerance**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
