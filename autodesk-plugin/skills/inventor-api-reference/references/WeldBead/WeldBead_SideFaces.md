# WeldBead.SideFaces Property

Parent Object: [WeldBead](../WeldBead/WeldBead.md)

## Description

Gets the set of faces that define the sides of the weld bead.

## Syntax

WeldBead.**SideFaces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 11
