# WeldBead.BeadFaces Property

Parent Object: [WeldBead](../WeldBead/WeldBead.md)

## Description

Property that returns the set of  that define the geometric result of the weld bead.

## Syntax

WeldBead.**BeadFaces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 8
