# WeldBead.EndFaces Property

Parent Object: [WeldBead](../WeldBead/WeldBead.md)

## Description

Gets the set of faces that define the ends of the weld bead.

## Syntax

WeldBead.**EndFaces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 11
