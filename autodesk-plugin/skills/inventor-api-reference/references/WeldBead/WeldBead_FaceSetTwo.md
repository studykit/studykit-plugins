# WeldBead.FaceSetTwo Property

Parent Object: [WeldBead](../WeldBead/WeldBead.md)

## Description

Returns the second set of faces the weld bead is on.

## Syntax

WeldBead.**FaceSetTwo**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2011
