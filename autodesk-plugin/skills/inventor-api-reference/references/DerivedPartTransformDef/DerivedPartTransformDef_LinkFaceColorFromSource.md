# DerivedPartTransformDef.LinkFaceColorFromSource Property

Parent Object: [DerivedPartTransformDef](../DerivedPartTransformDef/DerivedPartTransformDef.md)

## Description

Read-write Boolean property that specifies whether to link face color from source component or not.

## Syntax

DerivedPartTransformDef.**LinkFaceColorFromSource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
