# CentermarkStylesEnumerator.Item Property

Parent Object: [CentermarkStylesEnumerator](../CentermarkStylesEnumerator/CentermarkStylesEnumerator.md)

## Description

Returns the specified CentermarkStyle object from the collection.

## Syntax

CentermarkStylesEnumerator.**Item**( ***Index*** As Variant ) As [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Property Value

This is a read only property whose value is a [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the CentermarkStyle to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the CentermarkStyle name. If an out of range index or a name of a non-existent CentermarkStyle is provided, an error will occur. |

## Version

Introduced in version 2010
