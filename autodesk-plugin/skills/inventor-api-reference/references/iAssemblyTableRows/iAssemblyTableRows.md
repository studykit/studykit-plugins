# iAssemblyTableRows Object

## Description

The iAssemblyTableRows collection object provides access to all existing iAssemblyTableRow objects in the iAssembly factory table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../iAssemblyTableRows/iAssemblyTableRows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../iAssemblyTableRows/iAssemblyTableRows_Count.md) | Property that specifies the number of items in the collection. |
| [Item](../iAssemblyTableRows/iAssemblyTableRows_Item.md) | Returns an item from the collection. |
| [Type](../iAssemblyTableRows/iAssemblyTableRows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[iAssemblyFactory.TableRows](../iAssemblyFactory/iAssemblyFactory_TableRows.md)

## Version

Introduced in version 11
