# SingleHemDefinition.Gap Property

Parent Object: [SingleHemDefinition](../SingleHemDefinition/SingleHemDefinition.md)

## Description

Property that returns the parameter controlling the gap of a hem feature defined by a single bend.

## Syntax

SingleHemDefinition.**Gap**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
