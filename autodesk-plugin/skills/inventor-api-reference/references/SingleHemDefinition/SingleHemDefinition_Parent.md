# SingleHemDefinition.Parent Property

Parent Object: [SingleHemDefinition](../SingleHemDefinition/SingleHemDefinition.md)

## Description

Property that returns the parent hem definition feature.

## Syntax

SingleHemDefinition.**Parent**() As [HemDefinition](../HemDefinition/HemDefinition.md)

## Property Value

This is a read only property whose value is a [HemDefinition](../HemDefinition/HemDefinition.md).

## Version

Introduced in version 2009
