# BrowserPane.Activate Method

Parent Object: [BrowserPane](../BrowserPane/BrowserPane.md)

## Description

Method that causes the pane to become the active browser pane.

## Syntax

BrowserPane.**Activate**()

## Version

Introduced in version 5
