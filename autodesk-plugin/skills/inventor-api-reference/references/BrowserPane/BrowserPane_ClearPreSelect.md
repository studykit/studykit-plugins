# BrowserPane.ClearPreSelect Method

Parent Object: [BrowserPane](../BrowserPane/BrowserPane.md)

## Description

Clears any pre-selection(s) for this pane.

## Syntax

BrowserPane.**ClearPreSelect**()

## Version

Introduced in version 2018
