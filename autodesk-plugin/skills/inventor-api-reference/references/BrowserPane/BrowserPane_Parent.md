# BrowserPane.Parent Property

Parent Object: [BrowserPane](../BrowserPane/BrowserPane.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

BrowserPane.**Parent**() As [Document](../Document/Document.md)

## Property Value

This is a read only property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 5
