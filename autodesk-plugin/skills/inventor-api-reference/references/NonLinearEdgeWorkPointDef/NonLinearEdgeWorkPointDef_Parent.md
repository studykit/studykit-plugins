# NonLinearEdgeWorkPointDef.Parent Property

Parent Object: [NonLinearEdgeWorkPointDef](../NonLinearEdgeWorkPointDef/NonLinearEdgeWorkPointDef.md)

## Description

Property that returns the parent WorkPoint object.

## Syntax

NonLinearEdgeWorkPointDef.**Parent**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 11
