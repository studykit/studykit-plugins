# TriangleFanGraphics.RemoveViewSpaceAnchor Method

Parent Object: [TriangleFanGraphics](../TriangleFanGraphics/TriangleFanGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

TriangleFanGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
