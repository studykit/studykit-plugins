# TriangleFanGraphics.NormalBinding Property

Parent Object: [TriangleFanGraphics](../TriangleFanGraphics/TriangleFanGraphics.md)

## Description

Gets and sets how the normals are defined for the triangles.

## Syntax

TriangleFanGraphics.**NormalBinding**() As [NormalBindingEnum](../NormalBindingEnum.md)

## Property Value

This is a read/write property whose value is a [NormalBindingEnum](../NormalBindingEnum.md).

## Version

Introduced in version 5
