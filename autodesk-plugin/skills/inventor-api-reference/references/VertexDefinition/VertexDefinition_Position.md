# VertexDefinition.Position Property

Parent Object: [VertexDefinition](../VertexDefinition/VertexDefinition.md)

## Description

Gets and sets the defined location of the vertex.

## Syntax

VertexDefinition.**Position**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2011
