# SketchSplineHandleProxy.Reference Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Gets and sets whether this entity is a reference entity or not. It is only valid to set this property to False.

## Syntax

SketchSplineHandleProxy.**Reference**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
