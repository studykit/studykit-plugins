# SketchSplineHandleProxy.Weight Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Gets and sets the tangent weight at the fit point associated with this handle.

## Syntax

SketchSplineHandleProxy.**Weight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
