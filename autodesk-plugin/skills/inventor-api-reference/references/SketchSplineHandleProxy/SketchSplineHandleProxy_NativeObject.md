# SketchSplineHandleProxy.NativeObject Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchSplineHandleProxy.**NativeObject**() As [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md)

## Property Value

This is a read only property whose value is a [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md).

## Version

Introduced in version 2008
