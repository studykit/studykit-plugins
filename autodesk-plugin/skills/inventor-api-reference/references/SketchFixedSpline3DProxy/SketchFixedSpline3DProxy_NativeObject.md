# SketchFixedSpline3DProxy.NativeObject Property

Parent Object: [SketchFixedSpline3DProxy](../SketchFixedSpline3DProxy/SketchFixedSpline3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchFixedSpline3DProxy.**NativeObject**() As [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md).

## Version

Introduced in version 9
