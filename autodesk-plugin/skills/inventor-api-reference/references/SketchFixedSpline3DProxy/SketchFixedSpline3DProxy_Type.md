# SketchFixedSpline3DProxy.Type Property

Parent Object: [SketchFixedSpline3DProxy](../SketchFixedSpline3DProxy/SketchFixedSpline3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchFixedSpline3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
