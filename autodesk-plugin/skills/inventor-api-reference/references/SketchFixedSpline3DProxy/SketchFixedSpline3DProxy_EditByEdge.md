# SketchFixedSpline3DProxy.EditByEdge Method

Parent Object: [SketchFixedSpline3DProxy](../SketchFixedSpline3DProxy/SketchFixedSpline3DProxy.md)

## Description

Method that edits the geometry of a fixed sketch spline based on an input transient Edge.

## Syntax

SketchFixedSpline3DProxy.**EditByEdge**( ***TransientEdge*** As [Edge](../Edge/Edge.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TransientEdge | [Edge](../Edge/Edge.md) | Input transient Edge object. |

## Version

Introduced in version 2010
