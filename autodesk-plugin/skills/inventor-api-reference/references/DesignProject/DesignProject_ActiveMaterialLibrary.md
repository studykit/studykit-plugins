# DesignProject.ActiveMaterialLibrary Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Read-only property that returns the currently active material library.

## Syntax

DesignProject.**ActiveMaterialLibrary**() As [ProjectAssetLibrary](../ProjectAssetLibrary/ProjectAssetLibrary.md)

## Property Value

This is a read only property whose value is a [ProjectAssetLibrary](../ProjectAssetLibrary/ProjectAssetLibrary.md).

## Version

Introduced in version 2014
