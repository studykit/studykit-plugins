# DesignProject.FullFileName Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Property that returns the fully qualified file name for this design project.

## Syntax

DesignProject.**FullFileName**() As String

## Property Value

This is a read only property whose value is a String.
