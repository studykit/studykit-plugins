# DesignProject.Shortcuts Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets an array of strings specifying the shortcuts for the project.

## Syntax

DesignProject.**Shortcuts**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2011
