# DesignProject.ContentCenterPathOverridden Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets whether the content center path is overridden by this project. Setting the property to False clears the override and path in the application option is used.

## Syntax

DesignProject.**ContentCenterPathOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
