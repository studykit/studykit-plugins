# DesignProject.MaterialLibraries Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Read-only property that returns the collection of material libraries associated with this project.

## Syntax

DesignProject.**MaterialLibraries**() As [ProjectAssetLibraries](../ProjectAssetLibraries/ProjectAssetLibraries.md)

## Property Value

This is a read only property whose value is a [ProjectAssetLibraries](../ProjectAssetLibraries/ProjectAssetLibraries.md).

## Version

Introduced in version 2014
