# DesignProject.OldVersionsToKeep Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets the number of versions to store in the OldVersions folder for each file saved. The first time a file is saved.

## Syntax

DesignProject.**OldVersionsToKeep**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2011
