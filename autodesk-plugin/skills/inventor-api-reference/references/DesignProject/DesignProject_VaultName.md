# DesignProject.VaultName Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets the vault name. Applicable only if the ProjectType is set to kVaultMode.

## Syntax

DesignProject.**VaultName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
