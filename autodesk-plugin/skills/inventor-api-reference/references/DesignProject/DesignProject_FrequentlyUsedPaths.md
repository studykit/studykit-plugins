# DesignProject.FrequentlyUsedPaths Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Property that returns a ProjectPaths object that contains the list of frequently used paths and provides ability to add/remove paths.

## Syntax

DesignProject.**FrequentlyUsedPaths**() As [ProjectPaths](../ProjectPaths/ProjectPaths.md)

## Property Value

This is a read only property whose value is a [ProjectPaths](../ProjectPaths/ProjectPaths.md).

## Version

Introduced in version 2011
