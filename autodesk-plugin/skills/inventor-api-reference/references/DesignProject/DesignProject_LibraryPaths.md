# DesignProject.LibraryPaths Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Property that returns a ProjectPaths object that contains the list of library paths and provides ability to add/remove paths.

## Syntax

DesignProject.**LibraryPaths**() As [ProjectPaths](../ProjectPaths/ProjectPaths.md)

## Property Value

This is a read only property whose value is a [ProjectPaths](../ProjectPaths/ProjectPaths.md).
