# DesignProject.ReleaseId Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets version of the project.

## Syntax

DesignProject.**ReleaseId**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
