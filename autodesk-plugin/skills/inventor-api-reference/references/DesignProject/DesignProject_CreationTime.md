# DesignProject.CreationTime Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets the creation time of the project file.

## Syntax

DesignProject.**CreationTime**() As Date

## Property Value

This is a read/write property whose value is a Date.

## Version

Introduced in version 2011
