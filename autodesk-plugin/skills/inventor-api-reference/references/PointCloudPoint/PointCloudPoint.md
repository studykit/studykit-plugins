# PointCloudPoint Object

## Description

The PointCloudPoint object represents a point within a point cloud.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointCloudPoint/PointCloudPoint_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Geometry](../PointCloudPoint/PointCloudPoint_Geometry.md) | Read-only property that returns a Point object which provides the coordinates of this point. |
| [Parent](../PointCloudPoint/PointCloudPoint_Parent.md) | Read-only property that returns the parent point cloud of this point. |
| [Type](../PointCloudPoint/PointCloudPoint_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
