# SketchLineProxy.Construction Property

Parent Object: [SketchLineProxy](../SketchLineProxy/SketchLineProxy.md)

## Description

Gets and sets whether the entity behaves as a construction entity or not.

## Syntax

SketchLineProxy.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
