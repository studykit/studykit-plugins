# SketchLineProxy.Centerline Property

Parent Object: [SketchLineProxy](../SketchLineProxy/SketchLineProxy.md)

## Description

Gets and sets whether the line behaves as a centerline or not.

## Syntax

SketchLineProxy.**Centerline**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
