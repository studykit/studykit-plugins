# SketchLineProxy.Geometry3d Property

Parent Object: [SketchLineProxy](../SketchLineProxy/SketchLineProxy.md)

## Description

Read-only property that returns line segment geometry that represents this line in model space.

## Syntax

SketchLineProxy.**Geometry3d**() As [LineSegment](../LineSegment/LineSegment.md)

## Property Value

This is a read only property whose value is a [LineSegment](../LineSegment/LineSegment.md).

## Version

Introduced in version 2013
