# SketchLineProxy.LineType Property

Parent Object: [SketchLineProxy](../SketchLineProxy/SketchLineProxy.md)

## Description

Gets and sets the LineType applied to this sketch line.

## Syntax

SketchLineProxy.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 9
