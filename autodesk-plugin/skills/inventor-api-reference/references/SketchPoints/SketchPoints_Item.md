# SketchPoints.Item Property

Parent Object: [SketchPoints](../SketchPoints/SketchPoints.md)

## Description

Returns the specified SketchPoint object from the collection.

## Syntax

SketchPoints.**Item**( ***Index*** As Long ) As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 5
