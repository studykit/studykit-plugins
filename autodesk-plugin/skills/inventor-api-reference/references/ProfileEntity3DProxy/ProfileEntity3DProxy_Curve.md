# ProfileEntity3DProxy.Curve Property

Parent Object: [ProfileEntity3DProxy](../ProfileEntity3DProxy/ProfileEntity3DProxy.md)

## Description

Property that returns the geometry of the entity. The geometry is returned such that the entities are connected in a head-to-tail fashion.

## Syntax

ProfileEntity3DProxy.**Curve**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
