# ProfileEntity3DProxy.StartSketchPoint Property

Parent Object: [ProfileEntity3DProxy](../ProfileEntity3DProxy/ProfileEntity3DProxy.md)

## Description

Property that gets the that defines the start of the profile entity. This property is not valid in the case where the curve is a periodic curve, i.e. circle, ellipse, etc.

## Syntax

ProfileEntity3DProxy.**StartSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 6
