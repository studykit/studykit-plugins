# PartsListStyle.SaveToGlobal Method

Parent Object: [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

PartsListStyle.**SaveToGlobal**()

## Version

Introduced in version 2011
