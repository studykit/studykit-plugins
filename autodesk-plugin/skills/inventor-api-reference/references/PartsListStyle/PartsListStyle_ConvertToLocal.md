# PartsListStyle.ConvertToLocal Method

Parent Object: [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

PartsListStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2011
