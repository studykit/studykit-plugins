# PartsListStyle.StyleType Property

Parent Object: [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Description

Gets the type of the style.

## Syntax

PartsListStyle.**StyleType**() As [StyleTypeEnum](../StyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [StyleTypeEnum](../StyleTypeEnum.md).

## Version

Introduced in version 2011
