# PartsListStyle.HeadingGap Property

Parent Object: [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Description

Gets and sets the heading gap in centimeters.

## Syntax

PartsListStyle.**HeadingGap**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
