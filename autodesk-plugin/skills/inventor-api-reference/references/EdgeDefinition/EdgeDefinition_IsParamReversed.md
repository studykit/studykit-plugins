# EdgeDefinition.IsParamReversed Property

Parent Object: [EdgeDefinition](../EdgeDefinition/EdgeDefinition.md)

## Description

Gets and sets if the orientation of this EdgeUse is in the same direction or not relative to the associated EdgeDefinition object.

## Syntax

EdgeDefinition.**IsParamReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
