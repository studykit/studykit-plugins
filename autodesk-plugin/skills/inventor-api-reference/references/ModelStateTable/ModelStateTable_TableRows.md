# ModelStateTable.TableRows Property

Parent Object: [ModelStateTable](../ModelStateTable/ModelStateTable.md)

## Description

Read-only property that returns the ModelStateTableRows collection object.

## Syntax

ModelStateTable.**TableRows**() As [ModelStateTableRows](../ModelStateTableRows/ModelStateTableRows.md)

## Property Value

This is a read only property whose value is a [ModelStateTableRows](../ModelStateTableRows/ModelStateTableRows.md).

## Version

Introduced in version 2022
