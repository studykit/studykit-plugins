# ModelStateTable.Parent Property

Parent Object: [ModelStateTable](../ModelStateTable/ModelStateTable.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelStateTable.**Parent**() As [ModelStates](../ModelStates/ModelStates.md)

## Property Value

This is a read only property whose value is a [ModelStates](../ModelStates/ModelStates.md).

## Version

Introduced in version 2022
