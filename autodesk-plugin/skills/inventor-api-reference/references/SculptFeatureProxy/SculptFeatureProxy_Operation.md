# SculptFeatureProxy.Operation Property

Parent Object: [SculptFeatureProxy](../SculptFeatureProxy/SculptFeatureProxy.md)

## Description

Gets and sets the type of operation used to add the feature to the model.

## Syntax

SculptFeatureProxy.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2011
