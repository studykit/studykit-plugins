# SculptFeatureProxy.NativeObject Property

Parent Object: [SculptFeatureProxy](../SculptFeatureProxy/SculptFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SculptFeatureProxy.**NativeObject**() As [SculptFeature](../SculptFeature/SculptFeature.md)

## Property Value

This is a read only property whose value is a [SculptFeature](../SculptFeature/SculptFeature.md).

## Version

Introduced in version 11
