# SculptFeatureProxy.Surfaces Property

Parent Object: [SculptFeatureProxy](../SculptFeatureProxy/SculptFeatureProxy.md)

## Description

Gets and sets the set of surfaces and their respective directions associated with the sculpt feature.

## Syntax

SculptFeatureProxy.**Surfaces**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 11
