# LicenseManager Object

## Description

LicenseManager Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LicenseManager/LicenseManager_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [LicenseModel](../LicenseManager/LicenseManager_LicenseModel.md) | Read-only property that returns the license model. |
| [LicenseUsage](../LicenseManager/LicenseManager_LicenseUsage.md) | Read-only property that returns the license usage. |
| [Parent](../LicenseManager/LicenseManager_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Type](../LicenseManager/LicenseManager_Type.md) | Gets the constant that indicates the type of this object. |

## Version

Introduced in version 2022
