# DataMedium.IDataObject Property

Parent Object: [DataMedium](../DataMedium/DataMedium.md)

## Description

Specifies a IDataObject containing the data.

## Syntax

DataMedium.**IDataObject**() As Unknown

## Property Value

This is a read/write property whose value is a Unknown.

## Version

Introduced in version 5.3
