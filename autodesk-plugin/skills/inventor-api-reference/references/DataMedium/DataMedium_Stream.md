# DataMedium.Stream Property

Parent Object: [DataMedium](../DataMedium/DataMedium.md)

## Description

Specifies an IStream containing the data.

## Syntax

DataMedium.**Stream**() As Unknown

## Property Value

This is a read/write property whose value is a Unknown.

## Version

Introduced in version 4
