# CheckPoint.Id Property

Parent Object: [CheckPoint](../CheckPoint/CheckPoint.md)

## Description

Gets the numeric identifier of this check point.

## Syntax

CheckPoint.**Id**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5
