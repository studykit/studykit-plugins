# LibraryFolders.Item Property

Parent Object: [LibraryFolders](../LibraryFolders/LibraryFolders.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

LibraryFolders.**Item**( ***Index*** As Variant ) As [LibraryFolder](../LibraryFolder/LibraryFolder.md)

## Property Value

This is a read only property whose value is a [LibraryFolder](../LibraryFolder/LibraryFolder.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of library folder in the library. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the name of the library folder. If an out of range index is specified, an error occurs. |

## Version

Introduced in version 2016
