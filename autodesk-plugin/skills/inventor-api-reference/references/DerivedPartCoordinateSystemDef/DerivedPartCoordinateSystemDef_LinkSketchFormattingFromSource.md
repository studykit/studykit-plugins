# DerivedPartCoordinateSystemDef.LinkSketchFormattingFromSource Property

Parent Object: [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Description

Read-write Boolean property that specifies whether to link sketch formatting from source component or not.

## Syntax

DerivedPartCoordinateSystemDef.**LinkSketchFormattingFromSource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
