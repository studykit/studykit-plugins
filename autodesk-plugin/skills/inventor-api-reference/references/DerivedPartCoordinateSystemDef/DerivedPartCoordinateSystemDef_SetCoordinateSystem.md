# DerivedPartCoordinateSystemDef.SetCoordinateSystem Method

Parent Object: [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Description

Method that sets the coordinate system.

## Syntax

DerivedPartCoordinateSystemDef.**SetCoordinateSystem**( ***Origin*** As [Point](../Point/Point.md), ***XAxis*** As [Vector](../Vector/Vector.md), ***YAxis*** As [Vector](../Vector/Vector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Origin | [Point](../Point/Point.md) | Input that specifies the origin of the coordinate system. |
| XAxis | [Vector](../Vector/Vector.md) | Input that specifies the x-axis of the coordinate system. |
| YAxis | [Vector](../Vector/Vector.md) | Input that specifies the y-axis of the coordinate system. |

## Version

Introduced in version 6
