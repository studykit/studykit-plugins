# DerivedPartCoordinateSystemDef.DeriveStyle Property

Parent Object: [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Description

Gets and sets how bodies are derived. Valid options are kDeriveAsSingleBodyWithSeams, kDeriveAsSingleBodyNoSeams, kDeriveAsMultipleBodies and kDeriveAsWorkSurface.

## Syntax

DerivedPartCoordinateSystemDef.**DeriveStyle**() As [DerivedComponentStyleEnum](../DerivedComponentStyleEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentStyleEnum](../DerivedComponentStyleEnum.md).

## Version

Introduced in version 2010
