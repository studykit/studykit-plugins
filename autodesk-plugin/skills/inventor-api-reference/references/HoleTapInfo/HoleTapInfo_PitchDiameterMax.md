# HoleTapInfo.PitchDiameterMax Property

Parent Object: [HoleTapInfo](../HoleTapInfo/HoleTapInfo.md)

## Description

Property that returns the maximum pitch diameter. This is in millimeters if the Metric argument is True and inches if False. This property contains a Double value or can be Nothing if the value is not set.

## Syntax

HoleTapInfo.**PitchDiameterMax**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 5
