# HoleTapInfo.ThreadDepth Property

Parent Object: [HoleTapInfo](../HoleTapInfo/HoleTapInfo.md)

## Description

Property that returns the parameter that controls the depth of the thread. Even though the parameter for the thread depth is always created and accessible through this property, it is only used in the case where the FullDepth property is False.

## Syntax

HoleTapInfo.**ThreadDepth**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 5
