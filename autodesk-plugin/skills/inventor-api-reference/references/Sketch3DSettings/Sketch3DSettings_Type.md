# Sketch3DSettings.Type Property

Parent Object: [Sketch3DSettings](../Sketch3DSettings/Sketch3DSettings.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

Sketch3DSettings.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 8
