# Sketch3DSettings.Parent Property

Parent Object: [Sketch3DSettings](../Sketch3DSettings/Sketch3DSettings.md)

## Description

Property that returns the parent Document object.

## Syntax

Sketch3DSettings.**Parent**() As [Document](../Document/Document.md)

## Property Value

This is a read only property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 8
