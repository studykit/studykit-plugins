# CornerFeatures Object

## Description

The CornerFeatures collection object provides access to all of the objects in a sheet metal component definition.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../CornerFeatures/CornerFeatures_Add.md) | Method that creates a new sheet metal corner feature. |
| [CreateCornerDefinition](../CornerFeatures/CornerFeatures_CreateCornerDefinition.md) | Method that creates a new sheet metal corner feature. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CornerFeatures/CornerFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CornerFeatures/CornerFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../CornerFeatures/CornerFeatures_Item.md) | Returns the specified CornerFeature object from the collection. This is the default property of the CornerFeatures collection object. |
| [Type](../CornerFeatures/CornerFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SheetMetalFeatures.CornerFeatures](../SheetMetalFeatures/SheetMetalFeatures_CornerFeatures.md)

## Version

Introduced in version 5.3
