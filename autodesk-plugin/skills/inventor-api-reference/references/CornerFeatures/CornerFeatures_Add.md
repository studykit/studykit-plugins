# CornerFeatures.Add Method

Parent Object: [CornerFeatures](../CornerFeatures/CornerFeatures.md)

## Description

Method that creates a new sheet metal corner feature.

## Syntax

CornerFeatures.**Add**( ***CornerDefinition*** As [CornerDefinition](../CornerDefinition/CornerDefinition.md) ) As [CornerFeature](../CornerFeature/CornerFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CornerDefinition | [CornerDefinition](../CornerDefinition/CornerDefinition.md) | Input CornerDefinition object that defines the corner feature you want to create. A CornerDefinition object can be created using the CornerFeature.CreateCornerDefinition method. It can also be obtained from an existing CornerFeature object. |

## Version

Introduced in version 2010
