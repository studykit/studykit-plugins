# EdgeProxy.EdgeUses Property

Parent Object: [EdgeProxy](../EdgeProxy/EdgeProxy.md)

## Description

Gets the that reference this Edge.

## Syntax

EdgeProxy.**EdgeUses**() As [EdgeUses](../EdgeUses/EdgeUses.md)

## Property Value

This is a read only property whose value is an [EdgeUses](../EdgeUses/EdgeUses.md).

## Version

Introduced in version 4
