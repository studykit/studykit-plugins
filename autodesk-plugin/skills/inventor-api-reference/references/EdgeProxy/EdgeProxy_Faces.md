# EdgeProxy.Faces Property

Parent Object: [EdgeProxy](../EdgeProxy/EdgeProxy.md)

## Description

Property that returns the that this Edge is referenced from.

## Syntax

EdgeProxy.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 4
