# UserCoordinateSystem.XZPlane Property

Parent Object: [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Description

Property that returns the work plane that represents the X-Z plane of the coordinate system.

## Syntax

UserCoordinateSystem.**XZPlane**() As [WorkPlane](../WorkPlane/WorkPlane.md)

## Property Value

This is a read only property whose value is a [WorkPlane](../WorkPlane/WorkPlane.md).

## Version

Introduced in version 2010
