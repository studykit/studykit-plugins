# UserCoordinateSystem.YAxis Property

Parent Object: [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Description

Property that returns the work axis that represents the y-axis of the coordinate system.

## Syntax

UserCoordinateSystem.**YAxis**() As [WorkAxis](../WorkAxis/WorkAxis.md)

## Property Value

This is a read only property whose value is a [WorkAxis](../WorkAxis/WorkAxis.md).

## Version

Introduced in version 2010
