# UserCoordinateSystem.XOffset Property

Parent Object: [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Description

Property that returns the parameter associated with the X offset value.

## Syntax

UserCoordinateSystem.**XOffset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
