# AssetTexture.Type Property

Parent Object: [AssetTexture](../AssetTexture/AssetTexture.md)

## Description

Read-only property returning kAssetTextureObject indicating this object’s type.

## Syntax

AssetTexture.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
