# ModelDatumTarget.AreaDiameter Property

Parent Object: [ModelDatumTarget](../ModelDatumTarget/ModelDatumTarget.md)

## Description

Returns the parameter that controls the datum target area diameter.

## Syntax

ModelDatumTarget.**AreaDiameter**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2023
