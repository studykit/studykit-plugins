# ModelDatumTarget.TopToleranceFeature Property

Parent Object: [ModelDatumTarget](../ModelDatumTarget/ModelDatumTarget.md)

## Description

Returns the top ModelToleranceFeature object.

## Syntax

ModelDatumTarget.**TopToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2023
