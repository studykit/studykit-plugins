# ModelDatumTarget.Name Property

Parent Object: [ModelDatumTarget](../ModelDatumTarget/ModelDatumTarget.md)

## Description

Read-write property that gets and sets the name of the annotation.

## Syntax

ModelDatumTarget.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2023
