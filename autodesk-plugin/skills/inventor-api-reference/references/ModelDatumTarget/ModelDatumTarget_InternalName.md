# ModelDatumTarget.InternalName Property

Parent Object: [ModelDatumTarget](../ModelDatumTarget/ModelDatumTarget.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelDatumTarget.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2023
