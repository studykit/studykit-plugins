# SinglePointRipTypeDef.Point Property

Parent Object: [SinglePointRipTypeDef](../SinglePointRipTypeDef/SinglePointRipTypeDef.md)

## Description

Gets and sets the point that defines the location of the rip.

## Syntax

SinglePointRipTypeDef.**Point**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2011
