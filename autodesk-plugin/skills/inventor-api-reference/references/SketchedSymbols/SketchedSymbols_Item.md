# SketchedSymbols.Item Property

Parent Object: [SketchedSymbols](../SketchedSymbols/SketchedSymbols.md)

## Description

Method that returns the specified SketchedSymbol object from the collection.

## Syntax

SketchedSymbols.**Item**( ***Index*** As Long ) As [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Property Value

This is a read only property whose value is a [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the SketchedSymbol to return. |

## Version

Introduced in version 5.3
