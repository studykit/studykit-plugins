# PathEntity.OpposedToSketchEntity Property

Parent Object: [PathEntity](../PathEntity/PathEntity.md)

## Description

Property that returns a Boolean indicating if the parametric flow of the path entity is in the same direction as the sketch entity it was derived from.

## Syntax

PathEntity.**OpposedToSketchEntity**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
