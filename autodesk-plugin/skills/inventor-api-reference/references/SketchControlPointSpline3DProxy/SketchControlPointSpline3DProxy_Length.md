# SketchControlPointSpline3DProxy.Length Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Gets the length of the entity in centimeters.

## Syntax

SketchControlPointSpline3DProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2014
