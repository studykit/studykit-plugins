# SketchControlPointSpline3DProxy.NativeObject Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchControlPointSpline3DProxy.**NativeObject**() As [SketchControlPointSpline3D](../SketchControlPointSpline3D/SketchControlPointSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchControlPointSpline3D](../SketchControlPointSpline3D/SketchControlPointSpline3D.md).

## Version

Introduced in version 2014
