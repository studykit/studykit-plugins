# SketchControlPointSpline3DProxy.CurvatureDisplay Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Specifies whether curvature information is displayed for the spline.

## Syntax

SketchControlPointSpline3DProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
