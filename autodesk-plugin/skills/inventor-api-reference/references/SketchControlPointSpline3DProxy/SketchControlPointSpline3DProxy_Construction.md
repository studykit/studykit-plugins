# SketchControlPointSpline3DProxy.Construction Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchControlPointSpline3DProxy.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
