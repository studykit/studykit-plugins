# SketchControlPointSpline3DProxy.ConvertToSpline Method

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Method that inserts a new control point into the existing control point spline.

## Syntax

SketchControlPointSpline3DProxy.**ConvertToSpline**() As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Version

Introduced in version 2014
