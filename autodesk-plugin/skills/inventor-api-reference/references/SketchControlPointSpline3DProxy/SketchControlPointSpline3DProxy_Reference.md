# SketchControlPointSpline3DProxy.Reference Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Gets and sets whether this entity is a reference entity or not.

## Syntax

SketchControlPointSpline3DProxy.**Reference**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version
