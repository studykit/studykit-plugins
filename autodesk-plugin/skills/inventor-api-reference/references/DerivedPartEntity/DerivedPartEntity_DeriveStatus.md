# DerivedPartEntity.DeriveStatus Property

Parent Object: [DerivedPartEntity](../DerivedPartEntity/DerivedPartEntity.md)

## Description

Read-write property that gets and sets the derive status for this entity.

## Syntax

DerivedPartEntity.**DeriveStatus**() As [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md).

## Version

Introduced in version 2024
