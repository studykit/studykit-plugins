# EnvironmentList.Type Property

Parent Object: [EnvironmentList](../EnvironmentList/EnvironmentList.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

EnvironmentList.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
