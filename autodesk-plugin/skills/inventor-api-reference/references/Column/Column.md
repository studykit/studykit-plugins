# Column Object

## Description

The Column object represents a single column in this custom table.See the Custom Tables overview for more information.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../Column/Column_Delete.md) | Method that deletes this Column object. |
| [Reposition](../Column/Column_Reposition.md) | Method that repositions the column within the table. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Column/Column_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [InternalTitle](../Column/Column_InternalTitle.md) | Property that returns the internal name of the column title. |
| [Parent](../Column/Column_Parent.md) | Property that returns the parent Table. |
| [Title](../Column/Column_Title.md) | Specifies the title of the column. |
| [TitleHorizontalJustification](../Column/Column_TitleHorizontalJustification.md) | Specifies the horizontal justification of the column title. |
| [Type](../Column/Column_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [ValueHorizontalJustification](../Column/Column_ValueHorizontalJustification.md) | Specifies the horizontal justification of the column values. |
| [Width](../Column/Column_Width.md) | Specifies the width of the column. |
