# Column.TitleHorizontalJustification Property

Parent Object: [Column](../Column/Column.md)

## Description

Specifies the horizontal justification of the column title.

## Syntax

Column.**TitleHorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 9
