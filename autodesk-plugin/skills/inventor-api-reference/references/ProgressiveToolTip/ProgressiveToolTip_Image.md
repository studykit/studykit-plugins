# ProgressiveToolTip.Image Property

Parent Object: [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md)

## Description

Gets and sets the image to use for the progressive tooltip. The standard recommended size is 304 x 188 pixels. This property can either be set to an IPictureDisp object or a string indicating a URI (uniform resource identifier).

## Syntax

ProgressiveToolTip.**Image**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2011
