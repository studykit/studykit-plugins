# ProgressiveToolTip.Title Property

Parent Object: [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md)

## Description

Gets and sets the title of the progressive tooltip.

## Syntax

ProgressiveToolTip.**Title**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
