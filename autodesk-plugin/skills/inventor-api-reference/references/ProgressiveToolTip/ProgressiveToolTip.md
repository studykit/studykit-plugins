# ProgressiveToolTip Object

## Description

A ProgressiveToolTip object provides access to enhanced tooltip display for controls in the ribbon interface.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ProgressiveToolTip/ProgressiveToolTip_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Description](../ProgressiveToolTip/ProgressiveToolTip_Description.md) | Gets and sets the short description of the progressive tooltip. |
| [ExpandedDescription](../ProgressiveToolTip/ProgressiveToolTip_ExpandedDescription.md) | Gets and sets the long description of the progressive tooltip. |
| [Image](../ProgressiveToolTip/ProgressiveToolTip_Image.md) | Gets and sets the image to use for the progressive tooltip. The standard recommended size is 304 x 188 pixels. This property can either be set to an IPictureDisp object or a string indicating a URI (uniform resource identifier). |
| [IsProgressive](../ProgressiveToolTip/ProgressiveToolTip_IsProgressive.md) | Gets and sets whether the second level content (ExpandedDescription and Image) is displayed after a delay. |
| [Parent](../ProgressiveToolTip/ProgressiveToolTip_Parent.md) | Property that returns the parent of the ControlDefinition object. |
| [Title](../ProgressiveToolTip/ProgressiveToolTip_Title.md) | Gets and sets the title of the progressive tooltip. |
| [Type](../ProgressiveToolTip/ProgressiveToolTip_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Video](../ProgressiveToolTip/ProgressiveToolTip_Video.md) | Gets and sets the the video clip to use for the progressive tooltip. The standard recommended size is 304 x 188 pixels. This property can either be set to the full file name indicating the video clip or a string indicating a URI (uniform resource identifier). |
