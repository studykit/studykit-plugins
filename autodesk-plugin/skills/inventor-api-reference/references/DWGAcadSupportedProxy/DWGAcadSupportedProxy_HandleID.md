# DWGAcadSupportedProxy.HandleID Property

Parent Object: [DWGAcadSupportedProxy](../DWGAcadSupportedProxy/DWGAcadSupportedProxy.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGAcadSupportedProxy.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
