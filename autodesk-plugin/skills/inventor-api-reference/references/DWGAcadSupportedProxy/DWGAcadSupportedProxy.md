# DWGAcadSupportedProxy Object

Derived from: [DWGEntity](../DWGEntity/DWGEntity.md) Object

## Description

The DWGAcadSupportedProxy object represents a DWG Acad support proxy within a DWGModelSpaceDefinition.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DWGAcadSupportedProxy/DWGAcadSupportedProxy_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [HandleID](../DWGAcadSupportedProxy/DWGAcadSupportedProxy_HandleID.md) | Read-only property that returns the HandleID for the DWGEntity. |
| [Parent](../DWGAcadSupportedProxy/DWGAcadSupportedProxy_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ParentEntity](../DWGAcadSupportedProxy/DWGAcadSupportedProxy_ParentEntity.md) | Read-only property that returns the parent DWGEntity of the entity. |
| [Segments](../DWGAcadSupportedProxy/DWGAcadSupportedProxy_Segments.md) | Read-only property that returns the DWGEntitySegmentsEnumerator object. |
| [Type](../DWGAcadSupportedProxy/DWGAcadSupportedProxy_Type.md) | Gets the constant that indicates the type of this object. |
