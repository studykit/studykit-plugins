# BOMView.ViewType Property

Parent Object: [BOMView](../BOMView/BOMView.md)

## Description

Property that returns the BOM View type. Possible return values are kModelDataBOMViewType (for the 'raw' view), kStructuredBOMViewType (for the structured view) and kPartsOnlyBOMViewType (for the parts-only view).

## Syntax

BOMView.**ViewType**() As [BOMViewTypeEnum](../BOMViewTypeEnum.md)

## Property Value

This is a read only property whose value is a [BOMViewTypeEnum](../BOMViewTypeEnum.md).

## Version

Introduced in version 11
