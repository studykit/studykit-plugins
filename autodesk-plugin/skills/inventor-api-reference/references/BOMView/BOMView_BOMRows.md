# BOMView.BOMRows Property

Parent Object: [BOMView](../BOMView/BOMView.md)

## Description

Property that gets the BOMRows enumerator object containing the top level BOM rows.

## Syntax

BOMView.**BOMRows**() As [BOMRowsEnumerator](../BOMRowsEnumerator/BOMRowsEnumerator.md)

## Property Value

This is a read only property whose value is a [BOMRowsEnumerator](../BOMRowsEnumerator/BOMRowsEnumerator.md).

## Version

Introduced in version 10
