# BOMView.Parent Property

Parent Object: [BOMView](../BOMView/BOMView.md)

## Description

Property that returns the parent BOM object.

## Syntax

BOMView.**Parent**() As [BOM](../BOM/BOM.md)

## Property Value

This is a read only property whose value is a [BOM](../BOM/BOM.md).

## Version

Introduced in version 10
