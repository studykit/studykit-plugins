# TextBoxProxy.Style Property

Parent Object: [TextBoxProxy](../TextBoxProxy/TextBoxProxy.md)

## Description

Gets/Sets the text style associated with the text box.

## Syntax

TextBoxProxy.**Style**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 10
