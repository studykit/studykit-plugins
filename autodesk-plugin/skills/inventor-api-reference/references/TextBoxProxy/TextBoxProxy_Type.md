# TextBoxProxy.Type Property

Parent Object: [TextBoxProxy](../TextBoxProxy/TextBoxProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TextBoxProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
