# TextBoxProxy.Origin Property

Parent Object: [TextBoxProxy](../TextBoxProxy/TextBoxProxy.md)

## Description

Gets and sets the position of upper-left corner of the text box.

## Syntax

TextBoxProxy.**Origin**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
