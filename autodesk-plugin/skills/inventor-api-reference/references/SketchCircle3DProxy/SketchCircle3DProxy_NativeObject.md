# SketchCircle3DProxy.NativeObject Property

Parent Object: [SketchCircle3DProxy](../SketchCircle3DProxy/SketchCircle3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchCircle3DProxy.**NativeObject**() As [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Property Value

This is a read only property whose value is a [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md).

## Version

Introduced in version 6
