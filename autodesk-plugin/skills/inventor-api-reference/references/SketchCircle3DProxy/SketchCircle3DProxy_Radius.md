# SketchCircle3DProxy.Radius Property

Parent Object: [SketchCircle3DProxy](../SketchCircle3DProxy/SketchCircle3DProxy.md)

## Description

Gets the Radius of the circle.

## Syntax

SketchCircle3DProxy.**Radius**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
