# SpellCheckOptions Object

## Description

SpellCheckOptions object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [ActiveCustomDictionary](../SpellCheckOptions/SpellCheckOptions_ActiveCustomDictionary.md) | Read-write property that gets and sets the active custom dictionary. |
| [Application](../SpellCheckOptions/SpellCheckOptions_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [CustomDictionaries](../SpellCheckOptions/SpellCheckOptions_CustomDictionaries.md) | Read-only property that returns the CustomDictionaries object containing all the custom dictionaries. |
| [DictionaryLanguage](../SpellCheckOptions/SpellCheckOptions_DictionaryLanguage.md) | Read-write property that gets and sets dictionary language. |
| [IgnoreCapitalizedWords](../SpellCheckOptions/SpellCheckOptions_IgnoreCapitalizedWords.md) | Read-write property that gets and sets whether or not to ignore the capitalized words (e.g. Autodesk). |
| [IgnoreUppercaseWords](../SpellCheckOptions/SpellCheckOptions_IgnoreUppercaseWords.md) | Read-write property that gets and sets whether or not to ignore the uppercase words (e.g. DWG). |
| [IgnoreWordsContainingPunctuation](../SpellCheckOptions/SpellCheckOptions_IgnoreWordsContainingPunctuation.md) | Read-write property that gets and sets whether or not to ignore the words with punctuation (e.g. A/3). |
| [IgnoreWordsWithNumbers](../SpellCheckOptions/SpellCheckOptions_IgnoreWordsWithNumbers.md) | Read-write property that gets and sets whether or not to ignore the words with numbers (e.g. ISO9000). |
| [IgnoreWordWithMixedCase](../SpellCheckOptions/SpellCheckOptions_IgnoreWordWithMixedCase.md) | Read-write property that gets and sets whether or not to ignore the word with mixed case(e.g. TrueType). |
| [Type](../SpellCheckOptions/SpellCheckOptions_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[GeneralOptions.SpellCheckOptions](../GeneralOptions/GeneralOptions_SpellCheckOptions.md)

## Version

Introduced in version 2020
