# SpellCheckOptions.DictionaryLanguage Property

Parent Object: [SpellCheckOptions](../SpellCheckOptions/SpellCheckOptions.md)

## Description

Read-write property that gets and sets dictionary language.

## Syntax

SpellCheckOptions.**DictionaryLanguage**() As [DictionaryLanguageTypeEnum](../DictionaryLanguageTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DictionaryLanguageTypeEnum](../DictionaryLanguageTypeEnum.md).

## Version

Introduced in version 2020
