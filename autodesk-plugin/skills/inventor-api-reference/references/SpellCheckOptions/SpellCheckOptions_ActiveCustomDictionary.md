# SpellCheckOptions.ActiveCustomDictionary Property

Parent Object: [SpellCheckOptions](../SpellCheckOptions/SpellCheckOptions.md)

## Description

Read-write property that gets and sets the active custom dictionary.

## Syntax

SpellCheckOptions.**ActiveCustomDictionary**() As [CustomDictionary](../CustomDictionary/CustomDictionary.md)

## Property Value

This is a read/write property whose value is a [CustomDictionary](../CustomDictionary/CustomDictionary.md).

## Version

Introduced in version 2020
