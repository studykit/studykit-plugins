# SketchRectangularPattern.Constraints Property

Parent Object: [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Description

Returns a collection of sketch constraints that are tied directly to this entity. This collection consists of both geometric and dimension constraints.

## Syntax

SketchRectangularPattern.**Constraints**() As [SketchConstraintsEnumerator](../SketchConstraintsEnumerator/SketchConstraintsEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchConstraintsEnumerator](../SketchConstraintsEnumerator/SketchConstraintsEnumerator.md).

## Version

Introduced in version 2025.1
