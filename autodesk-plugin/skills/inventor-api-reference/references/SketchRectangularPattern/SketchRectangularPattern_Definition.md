# SketchRectangularPattern.Definition Property

Parent Object: [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Description

Gets and sets the associative definition of the sketch pattern.

## Syntax

SketchRectangularPattern.**Definition**() As [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Property Value

This is a read/write property whose value is a [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md).

## Version

Introduced in version 2025.1
