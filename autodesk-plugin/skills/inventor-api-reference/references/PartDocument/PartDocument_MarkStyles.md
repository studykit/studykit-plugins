# PartDocument.MarkStyles Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Get the document's mark styles.

## Syntax

PartDocument.**MarkStyles**() As [MarkStylesEnumerator](../MarkStylesEnumerator/MarkStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [MarkStylesEnumerator](../MarkStylesEnumerator/MarkStylesEnumerator.md).

## Version

Introduced in version 2023
