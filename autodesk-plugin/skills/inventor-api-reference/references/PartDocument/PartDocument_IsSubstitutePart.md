# PartDocument.IsSubstitutePart Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Read-write property that gets and sets whether this part is intended to be used as a substitute part.

## Syntax

PartDocument.**IsSubstitutePart**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
