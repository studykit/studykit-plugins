# PartDocument.DocumentEvents Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

This object supports a set of events that are specific to the document.

## Syntax

PartDocument.**DocumentEvents**() As [DocumentEvents](../DocumentEvents/DocumentEvents.md)

## Property Value

This is a read only property whose value is a [DocumentEvents](../DocumentEvents/DocumentEvents.md).

## Version

Introduced in version 4
