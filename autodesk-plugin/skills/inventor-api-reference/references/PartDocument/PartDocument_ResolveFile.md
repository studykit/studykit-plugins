# PartDocument.ResolveFile Method

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Method that invoke the resolve file for the document if there are references missing.

## Syntax

PartDocument.**ResolveFile**()

## Version

Introduced in version 2025
