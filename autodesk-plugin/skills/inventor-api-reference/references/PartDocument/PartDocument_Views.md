# PartDocument.Views Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets all the open views of this document in a collection.

## Syntax

PartDocument.**Views**() As [Views](../Views/Views.md)

## Property Value

This is a read only property whose value is a [Views](../Views/Views.md).

## Version

Introduced in version 4
