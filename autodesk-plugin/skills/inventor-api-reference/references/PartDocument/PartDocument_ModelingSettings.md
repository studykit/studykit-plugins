# PartDocument.ModelingSettings Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Property that returns the ModelingSettings object. The ModelingSettings object provides access to various modeling related document options. This is somewhat equivalent to the Modeling tab of the Document Settings dialog.

## Syntax

PartDocument.**ModelingSettings**() As [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Property Value

This is a read only property whose value is a [ModelingSettings](../ModelingSettings/ModelingSettings.md).

## Version

Introduced in version 8
