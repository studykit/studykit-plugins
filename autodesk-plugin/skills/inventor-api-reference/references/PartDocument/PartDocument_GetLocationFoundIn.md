# PartDocument.GetLocationFoundIn Method

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Obtains the name of the location this file was found in.

## Syntax

PartDocument.**GetLocationFoundIn**( ***LocationName*** As String, ***LocationType*** As [LocationTypeEnum](../LocationTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LocationName | String | Output String that specifies the name of the location. |
| LocationType | [LocationTypeEnum](../LocationTypeEnum.md) | Output String that specifies the type of the location. |

## Version

Introduced in version 4
