# PartDocument.ReservedForWriteLogin Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the login of the person who currently holds the reservation to write.

## Syntax

PartDocument.**ReservedForWriteLogin**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
