# PartDocument.IsEmbeddedDocument Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Read-only property that returns whether this document is embedded into another document.

## Syntax

PartDocument.**IsEmbeddedDocument**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2016
