# PartDocument.InternalName Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the Internal Name (a GUID) for this document.

## Syntax

PartDocument.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 4
