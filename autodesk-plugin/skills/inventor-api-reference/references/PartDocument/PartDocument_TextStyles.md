# PartDocument.TextStyles Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the TextStylesEnumerator object.

## Syntax

PartDocument.**TextStyles**() As [TextStylesEnumerator](../TextStylesEnumerator/TextStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [TextStylesEnumerator](../TextStylesEnumerator/TextStylesEnumerator.md).

## Version

Introduced in version 2017
