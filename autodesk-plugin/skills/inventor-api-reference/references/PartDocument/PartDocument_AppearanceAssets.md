# PartDocument.AppearanceAssets Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Returns Assets collection object that contains the appearances associated with the document.

## Syntax

PartDocument.**AppearanceAssets**() As [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md).
