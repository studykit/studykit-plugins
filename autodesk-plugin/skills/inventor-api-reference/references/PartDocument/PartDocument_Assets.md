# PartDocument.Assets Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Read-only property that returns an Assets collection object that contains all of the assets, regardless of their type, that are associated with the document.

## Syntax

PartDocument.**Assets**() As [Assets](../Assets/Assets.md)

## Property Value

This is a read only property whose value is an [Assets](../Assets/Assets.md).
