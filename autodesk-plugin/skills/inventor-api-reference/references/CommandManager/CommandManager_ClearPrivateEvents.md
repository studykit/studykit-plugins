# CommandManager.ClearPrivateEvents Method

Parent Object: [CommandManager](../CommandManager/CommandManager.md)

## Description

Clears the contents of Autodesk Inventor's internal clipboard.

## Syntax

CommandManager.**ClearPrivateEvents**()

## Version

Introduced in version 10
