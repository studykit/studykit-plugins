# CommandManager.StopActiveCommand Method

Parent Object: [CommandManager](../CommandManager/CommandManager.md)

## Description

Method that causes the currently running command to be terminated.

## Syntax

CommandManager.**StopActiveCommand**()

## Version

Introduced in version 4
