# CommandManager.CommandCategories Property

Parent Object: [CommandManager](../CommandManager/CommandManager.md)

## Description

Property that returns the collection.

## Syntax

CommandManager.**CommandCategories**() As [CommandCategories](../CommandCategories/CommandCategories.md)

## Property Value

This is a read only property whose value is a [CommandCategories](../CommandCategories/CommandCategories.md).

## Version

Introduced in version 9
