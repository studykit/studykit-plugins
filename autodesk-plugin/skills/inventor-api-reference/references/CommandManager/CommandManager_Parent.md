# CommandManager.Parent Property

Parent Object: [CommandManager](../CommandManager/CommandManager.md)

## Description

Property that returns the parent document of the object.

## Syntax

CommandManager.**Parent**() As [Application](../Application/Application.md)

## Property Value

This is a read only property whose value is an [Application](../Application/Application.md).

## Version

Introduced in version 4
