# PointCloudPlaneProxy.NativeObject Property

Parent Object: [PointCloudPlaneProxy](../PointCloudPlaneProxy/PointCloudPlaneProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

PointCloudPlaneProxy.**NativeObject**() As [PointCloudPlane](../PointCloudPlane/PointCloudPlane.md)

## Property Value

This is a read only property whose value is a [PointCloudPlane](../PointCloudPlane/PointCloudPlane.md).

## Version

Introduced in version 2016
