# PointCloudPlaneProxy.Geometry Property

Parent Object: [PointCloudPlaneProxy](../PointCloudPlaneProxy/PointCloudPlaneProxy.md)

## Description

Read-only property that returns a Plane object which provides the location and orientation of the inferred plane.

## Syntax

PointCloudPlaneProxy.**Geometry**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).

## Version

Introduced in version 2016
