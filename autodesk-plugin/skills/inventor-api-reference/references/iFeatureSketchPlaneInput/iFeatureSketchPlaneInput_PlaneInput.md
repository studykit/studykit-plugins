# iFeatureSketchPlaneInput.PlaneInput Property

Parent Object: [iFeatureSketchPlaneInput](../iFeatureSketchPlaneInput/iFeatureSketchPlaneInput.md)

## Description

Gets and sets the geometry matching the sketch plane input.

## Syntax

iFeatureSketchPlaneInput.**PlaneInput**() As Object

## Property Value

This is a read/write property whose value is an Object.
