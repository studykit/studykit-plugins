# SketchedSymbolDefinitionLibrary.IsReadOnly Property

Parent Object: [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md)

## Description

Read-only property that returns whether this sketched symbol definition library is read only.

## Syntax

SketchedSymbolDefinitionLibrary.**IsReadOnly**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2016
