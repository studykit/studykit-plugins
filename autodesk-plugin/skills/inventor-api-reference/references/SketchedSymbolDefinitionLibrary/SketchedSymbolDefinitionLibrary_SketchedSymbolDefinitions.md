# SketchedSymbolDefinitionLibrary.SketchedSymbolDefinitions Property

Parent Object: [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md)

## Description

Read-only property that returns the LibrarySketchedSymbolDefinitions collection object.

## Syntax

SketchedSymbolDefinitionLibrary.**SketchedSymbolDefinitions**() As [LibrarySketchedSymbolDefinitions](../LibrarySketchedSymbolDefinitions/LibrarySketchedSymbolDefinitions.md)

## Property Value

This is a read only property whose value is a [LibrarySketchedSymbolDefinitions](../LibrarySketchedSymbolDefinitions/LibrarySketchedSymbolDefinitions.md).

## Version

Introduced in version 2016
