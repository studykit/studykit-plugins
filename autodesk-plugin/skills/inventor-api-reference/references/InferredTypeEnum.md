# InferredTypeEnum Enumerator

## Description

Constants that define the types of geometry that can be inferred from different faces.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kInferredLine | 24834 | Inferenced line inference type. |
| kInferredPoint | 24835 | Inferenced point inference type. |
| kNoInference | 24833 | No inference inference type. |

## Version

Introduced in version 5
