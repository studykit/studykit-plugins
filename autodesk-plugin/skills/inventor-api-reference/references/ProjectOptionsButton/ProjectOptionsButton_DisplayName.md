# ProjectOptionsButton.DisplayName Property

Parent Object: [ProjectOptionsButton](../ProjectOptionsButton/ProjectOptionsButton.md)

## Description

Read-write property that gets and sets the display name of the button.

## Syntax

ProjectOptionsButton.**DisplayName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2012
