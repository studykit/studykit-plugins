# ProjectOptionsButton Object

## Description

Project Options Buttons events object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../ProjectOptionsButton/ProjectOptionsButton_Delete.md) | Method that deletes the button from the projects dialog. |
| [FireOnClick](../ProjectOptionsButton/ProjectOptionsButton_FireOnClick.md) | Method that fires the OnClick event. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ProjectOptionsButton/ProjectOptionsButton_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [DisplayName](../ProjectOptionsButton/ProjectOptionsButton_DisplayName.md) | Read-write property that gets and sets the display name of the button. |
| [Type](../ProjectOptionsButton/ProjectOptionsButton_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Events

|  |  |
| --- | --- |
| Name | Description |
| [OnClick](../ProjectOptionsButton/ProjectOptionsButton_OnClick.md) | Event that is fired whenever click the button. |

## Accessed From

[DesignProjectManager.AddOptionsButton](../DesignProjectManager/DesignProjectManager_AddOptionsButton.md)

## Version

Introduced in version 2012
