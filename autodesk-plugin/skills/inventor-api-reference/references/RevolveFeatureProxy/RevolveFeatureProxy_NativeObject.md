# RevolveFeatureProxy.NativeObject Property

Parent Object: [RevolveFeatureProxy](../RevolveFeatureProxy/RevolveFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RevolveFeatureProxy.**NativeObject**() As [RevolveFeature](../RevolveFeature/RevolveFeature.md)

## Property Value

This is a read only property whose value is a [RevolveFeature](../RevolveFeature/RevolveFeature.md).

## Version

Introduced in version 9
