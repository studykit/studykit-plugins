# RevolveFeatureProxy.Adaptive Property

Parent Object: [RevolveFeatureProxy](../RevolveFeatureProxy/RevolveFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

RevolveFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
