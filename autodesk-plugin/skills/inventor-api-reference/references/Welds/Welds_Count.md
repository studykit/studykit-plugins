# Welds.Count Property

Parent Object: [Welds](../Welds/Welds.md)

## Description

Property that returns the number of items in this collection.

## Syntax

Welds.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 8
