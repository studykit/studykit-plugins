# Welds.Item Property

Parent Object: [Welds](../Welds/Welds.md)

## Description

Returns the specified Weld object from the collection.

## Syntax

Welds.**Item**( ***Index*** As Variant ) As [Weld](../Weld/Weld.md)

## Property Value

This is a read only property whose value is a [Weld](../Weld/Weld.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the Weld to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the Weld name. If an out of range index or a name of a non-existent Weld is provided, an error will occur. |

## Version

Introduced in version 8
