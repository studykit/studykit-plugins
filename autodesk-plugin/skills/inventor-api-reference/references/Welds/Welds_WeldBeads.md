# Welds.WeldBeads Property

Parent Object: [Welds](../Welds/Welds.md)

## Description

Property that returns the WeldBeads object.

## Syntax

Welds.**WeldBeads**() As [WeldBeads](../WeldBeads/WeldBeads.md)

## Property Value

This is a read only property whose value is a [WeldBeads](../WeldBeads/WeldBeads.md).

## Version

Introduced in version 8
