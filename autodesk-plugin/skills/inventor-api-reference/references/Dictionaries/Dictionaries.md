# Dictionaries Object

## Description

Dictionaries object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Dictionaries/Dictionaries_Add.md) | Method that creates a new Dictionary. The new created Dictionary is returned. |
| [AddExisting](../Dictionaries/Dictionaries_AddExisting.md) | Method that adds an existing Dictionary to the list of available dictionaries. The newly added Dictionary is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Dictionaries/Dictionaries_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../Dictionaries/Dictionaries_Count.md) | Gets the number of items in this collection. |
| [Item](../Dictionaries/Dictionaries_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../Dictionaries/Dictionaries_Type.md) | Gets the constant that indicates the type of this object. |

## Version

Introduced in version 2020
