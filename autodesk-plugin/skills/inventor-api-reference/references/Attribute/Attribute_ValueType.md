# Attribute.ValueType Property

Parent Object: [Attribute](../Attribute/Attribute.md)

## Description

Property that returns the type of the value stored within this attribute.

## Syntax

Attribute.**ValueType**() As [ValueTypeEnum](../ValueTypeEnum.md)

## Property Value

This is a read only property whose value is a [ValueTypeEnum](../ValueTypeEnum.md).

## Version

Introduced in version 5
