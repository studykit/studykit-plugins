# Attribute Object

## Description

The Attribute object stores a name and value and belongs to a specific attribute set.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../Attribute/Attribute_Delete.md) | Method that deletes the attribute from the attribute set. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Name](../Attribute/Attribute_Name.md) | Gets/Sets the name of the Attribute. |
| [Parent](../Attribute/Attribute_Parent.md) | Property that returns the parent AttributeSet. |
| [Type](../Attribute/Attribute_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Value](../Attribute/Attribute_Value.md) | Gets/Sets the value of the Attribute. |
| [ValueType](../Attribute/Attribute_ValueType.md) | Property that returns the type of the value stored within this attribute. |

## Accessed From

[AttributesEnumerator.Item](../AttributesEnumerator/AttributesEnumerator_Item.md), [AttributeSet.Add](../AttributeSet/AttributeSet_Add.md), [AttributeSet.Item](../AttributeSet/AttributeSet_Item.md)

## Version

Introduced in version 5
