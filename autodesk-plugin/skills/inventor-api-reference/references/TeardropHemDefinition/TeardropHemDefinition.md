# TeardropHemDefinition Object

## Description

The TeardropHemDefinition object defines all of the inputs that are unique to a tear drop style of hem feature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Angle](../TeardropHemDefinition/TeardropHemDefinition_Angle.md) | Property that returns the parameter controlling the bend angle of a hem feature defined by a tear drop style of bend. |
| [Parent](../TeardropHemDefinition/TeardropHemDefinition_Parent.md) | Property that returns the parent hem definition feature. |
| [Radius](../TeardropHemDefinition/TeardropHemDefinition_Radius.md) | Property that returns the parameter controlling the radius of a hem feature defined by a tear drop style of bend. |

## Version

Introduced in version 2009
