# ModelWeldingSymbolDefinition.IdentificationLinePlacement Property

Parent Object: [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Description

Gets and sets the placement of the identification line against the reference line. Available for ISO and DIN only.

## Syntax

ModelWeldingSymbolDefinition.**IdentificationLinePlacement**() As [IdentificationLinePlacementEnum](../IdentificationLinePlacementEnum.md)

## Property Value

This is a read/write property whose value is an [IdentificationLinePlacementEnum](../IdentificationLinePlacementEnum.md).

## Version

Introduced in version 2024
