# ModelWeldingSymbolDefinition.AllAroundSymbol Property

Parent Object: [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Description

Gets and sets whether to add the all-around indicator to the symbol.

## Syntax

ModelWeldingSymbolDefinition.**AllAroundSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
