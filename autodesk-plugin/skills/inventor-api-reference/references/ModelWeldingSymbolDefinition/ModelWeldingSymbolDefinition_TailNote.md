# ModelWeldingSymbolDefinition.TailNote Property

Parent Object: [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Description

Gets and sets the description to the reference line.

## Syntax

ModelWeldingSymbolDefinition.**TailNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version
