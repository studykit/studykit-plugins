# RadialMarkingMenu.Name Property

Parent Object: [RadialMarkingMenu](../RadialMarkingMenu/RadialMarkingMenu.md)

## Description

Read-write property that gets and sets the name of the radial marking menu.

## Syntax

RadialMarkingMenu.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2013
