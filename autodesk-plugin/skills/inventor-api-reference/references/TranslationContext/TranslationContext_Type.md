# TranslationContext.Type Property

Parent Object: [TranslationContext](../TranslationContext/TranslationContext.md)

## Description

Specifies the IO operation type of the translation.

## Syntax

TranslationContext.**Type**() As [IOMechanismEnum](../IOMechanismEnum.md)

## Property Value

This is a read/write property whose value is an [IOMechanismEnum](../IOMechanismEnum.md).
