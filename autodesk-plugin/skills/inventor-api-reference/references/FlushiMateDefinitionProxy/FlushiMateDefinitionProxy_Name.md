# FlushiMateDefinitionProxy.Name Property

Parent Object: [FlushiMateDefinitionProxy](../FlushiMateDefinitionProxy/FlushiMateDefinitionProxy.md)

## Description

Gets the name of this iMateDefinition.

## Syntax

FlushiMateDefinitionProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 6
