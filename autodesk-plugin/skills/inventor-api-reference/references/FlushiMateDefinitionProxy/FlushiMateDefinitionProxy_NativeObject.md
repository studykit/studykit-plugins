# FlushiMateDefinitionProxy.NativeObject Property

Parent Object: [FlushiMateDefinitionProxy](../FlushiMateDefinitionProxy/FlushiMateDefinitionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FlushiMateDefinitionProxy.**NativeObject**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version

Introduced in version 6
