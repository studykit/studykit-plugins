# FlushiMateDefinitionProxy.Entity Property

Parent Object: [FlushiMateDefinitionProxy](../FlushiMateDefinitionProxy/FlushiMateDefinitionProxy.md)

## Description

Property that returns the entity used for the iMate definition.

## Syntax

FlushiMateDefinitionProxy.**Entity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
