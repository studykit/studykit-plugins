# EqualLengthConstraintProxy.NativeObject Property

Parent Object: [EqualLengthConstraintProxy](../EqualLengthConstraintProxy/EqualLengthConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

EqualLengthConstraintProxy.**NativeObject**() As [EqualLengthConstraint](../EqualLengthConstraint/EqualLengthConstraint.md)

## Property Value

This is a read only property whose value is an [EqualLengthConstraint](../EqualLengthConstraint/EqualLengthConstraint.md).

## Version

Introduced in version 6
