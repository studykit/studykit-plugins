# TwoPlanesWorkAxisDef Object

## Description

Object that allows you to get and set the information that specifies a work axis along a line.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../TwoPlanesWorkAxisDef/TwoPlanesWorkAxisDef_GetData.md) | Method that gets all of the data defining a work axis at the intersection of two planes. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TwoPlanesWorkAxisDef/TwoPlanesWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../TwoPlanesWorkAxisDef/TwoPlanesWorkAxisDef_Parent.md) | Property returning the parent WorkAxis object. |
| [Plane1](../TwoPlanesWorkAxisDef/TwoPlanesWorkAxisDef_Plane1.md) | Property that returns the plane of a two plane defined work axis. This object can be a planar Face, WorkPlane, or Sketch object. The work axis is at the intersection of the two planes. |
| [Plane2](../TwoPlanesWorkAxisDef/TwoPlanesWorkAxisDef_Plane2.md) | Property that returns the plane of a two plane defined work axis. This object can be a planar Face, WorkPlane, or Sketch object. The work axis is at the intersection of the two planes. |
| [Type](../TwoPlanesWorkAxisDef/TwoPlanesWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
