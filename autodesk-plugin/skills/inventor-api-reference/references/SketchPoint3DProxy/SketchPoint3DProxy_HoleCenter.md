# SketchPoint3DProxy.HoleCenter Property

Parent Object: [SketchPoint3DProxy](../SketchPoint3DProxy/SketchPoint3DProxy.md)

## Description

Defines if the sketch point is being displayed as a hole center.

## Syntax

SketchPoint3DProxy.**HoleCenter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
