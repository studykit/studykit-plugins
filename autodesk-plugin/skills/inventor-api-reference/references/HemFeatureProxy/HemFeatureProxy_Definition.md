# HemFeatureProxy.Definition Property

Parent Object: [HemFeatureProxy](../HemFeatureProxy/HemFeatureProxy.md)

## Description

Property that gets the HemDefinition object associated with this fold feature.

## Syntax

HemFeatureProxy.**Definition**() As [HemDefinition](../HemDefinition/HemDefinition.md)

## Property Value

This is a read only property whose value is a [HemDefinition](../HemDefinition/HemDefinition.md).

## Version

Introduced in version 2009
