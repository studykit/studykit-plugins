# HemFeatureProxy.SetAffectedBodies Method

Parent Object: [HemFeatureProxy](../HemFeatureProxy/HemFeatureProxy.md)

## Description

Method that sets a collection of SurfaceBody objects affected by this feature.

## Syntax

HemFeatureProxy.**SetAffectedBodies**( ***Bodies*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Bodies | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | ObjectCollection of SurfaceBody objects. |

## Version

Introduced in version 2010
