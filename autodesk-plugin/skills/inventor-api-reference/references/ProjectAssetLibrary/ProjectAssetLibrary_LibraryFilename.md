# ProjectAssetLibrary.LibraryFilename Property

Parent Object: [ProjectAssetLibrary](../ProjectAssetLibrary/ProjectAssetLibrary.md)

## Description

Read-only property that returns the full filename of the library file.

## Syntax

ProjectAssetLibrary.**LibraryFilename**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2014
