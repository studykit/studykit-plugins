# FileVersionEnum Enumerator

## Description

Constants indicating the type of FileVersion.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kOpenCurrentVersion | 57858 | Open a current version file. |
| kOpenOldVersion | 57857 | Open an old version file. |
| kRestoreOldVersionToCurrent | 57859 | Restore the old version file to current version. |

## Version

Introduced in version 11
