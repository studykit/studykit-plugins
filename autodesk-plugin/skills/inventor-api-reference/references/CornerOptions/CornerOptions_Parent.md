# CornerOptions.Parent Property

Parent Object: [CornerOptions](../CornerOptions/CornerOptions.md)

## Description

Property that returns the parent object of this CornerOptions object.

## Syntax

CornerOptions.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2009
