# CornerOptions.CornerReliefPlacement Property

Parent Object: [CornerOptions](../CornerOptions/CornerOptions.md)

## Description

Defines the two bend relief placement to use for the associated sheet metal feature (valid only for round and square corners).

## Syntax

CornerOptions.**CornerReliefPlacement**() As [CornerReliefPlacementEnum](../CornerReliefPlacementEnum.md)

## Property Value

This is a read/write property whose value is a [CornerReliefPlacementEnum](../CornerReliefPlacementEnum.md).

## Version

Introduced in version 2017
