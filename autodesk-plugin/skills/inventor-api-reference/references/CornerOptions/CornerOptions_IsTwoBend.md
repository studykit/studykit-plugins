# CornerOptions.IsTwoBend Property

Parent Object: [CornerOptions](../CornerOptions/CornerOptions.md)

## Description

Property that indicates if the corner this CornerOptions object is associated with is a two or three bend corner.

## Syntax

CornerOptions.**IsTwoBend**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
