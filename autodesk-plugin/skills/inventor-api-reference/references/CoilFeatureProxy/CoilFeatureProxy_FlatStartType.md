# CoilFeatureProxy.FlatStartType Property

Parent Object: [CoilFeatureProxy](../CoilFeatureProxy/CoilFeatureProxy.md)

## Description

Property that gets and sets the coil end type at the start of the coil feature.

## Syntax

CoilFeatureProxy.**FlatStartType**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
