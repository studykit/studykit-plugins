# CoilFeatureProxy.ClockwiseRotation Property

Parent Object: [CoilFeatureProxy](../CoilFeatureProxy/CoilFeatureProxy.md)

## Description

Property that gets and sets whether the rotation is clockwise or counter-clockwise. A value of True indicates clockwise rotation.

## Syntax

CoilFeatureProxy.**ClockwiseRotation**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
