# Balloon.Position Property

Parent Object: [Balloon](../Balloon/Balloon.md)

## Description

Gets and sets the position of the balloon on the Sheet.

## Syntax

Balloon.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 9
