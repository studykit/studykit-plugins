# Balloon.Parent Property

Parent Object: [Balloon](../Balloon/Balloon.md)

## Description

Property returning the parent sheet (the sheet object this balloon is associated with).

## Syntax

Balloon.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 9
