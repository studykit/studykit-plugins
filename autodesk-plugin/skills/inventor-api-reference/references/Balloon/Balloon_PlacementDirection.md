# Balloon.PlacementDirection Property

Parent Object: [Balloon](../Balloon/Balloon.md)

## Description

Gets and sets the direction in which to place attached balloons.

## Syntax

Balloon.**PlacementDirection**() As [BalloonDirectionEnum](../BalloonDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [BalloonDirectionEnum](../BalloonDirectionEnum.md).
