# Balloon.Leader Property

Parent Object: [Balloon](../Balloon/Balloon.md)

## Description

Property that returns the leader associated with the balloon.

## Syntax

Balloon.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 2009
