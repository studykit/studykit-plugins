# TangentConstraint.iMateResult Property

Parent Object: [TangentConstraint](../TangentConstraint/TangentConstraint.md)

## Description

Property that returns the iMateResult object that resulted in the creation of this constraint.

## Syntax

TangentConstraint.**iMateResult**() As [iMateResult](../iMateResult/iMateResult.md)

## Property Value

This is a read only property whose value is an [iMateResult](../iMateResult/iMateResult.md).

## Version

Introduced in version 6
