# TangentConstraint.InsideTangency Property

Parent Object: [TangentConstraint](../TangentConstraint/TangentConstraint.md)

## Description

Property that returns whether the curvatures of the surfaces at the point of tangency are on the same side. True if they are on the same side.

## Syntax

TangentConstraint.**InsideTangency**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
