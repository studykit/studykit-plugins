# TangentConstraint.Type Property

Parent Object: [TangentConstraint](../TangentConstraint/TangentConstraint.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TangentConstraint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 4
