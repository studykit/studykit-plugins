# DSLoads Object

## Description

DSLoads Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DSLoads/DSLoads_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DSLoads/DSLoads_Count.md) | Property that returns the number of items in this collection. |
| [Item](../DSLoads/DSLoads_Item.md) | Input Variant value that specifies the DSLoad to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the DSLoad name. If an out of range index or a name of a non-existent DSLoad is provided, an error will occur. |
| [Type](../DSLoads/DSLoads_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DynamicSimulation.DSLoads](../DynamicSimulation/DynamicSimulation_DSLoads.md)

## Version

Introduced in version 2013
