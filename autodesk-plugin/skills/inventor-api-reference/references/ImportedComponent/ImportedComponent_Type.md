# ImportedComponent.Type Property

Parent Object: [ImportedComponent](../ImportedComponent/ImportedComponent.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ImportedComponent.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2016
