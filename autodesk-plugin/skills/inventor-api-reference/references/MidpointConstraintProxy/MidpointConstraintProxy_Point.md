# MidpointConstraintProxy.Point Property

Parent Object: [MidpointConstraintProxy](../MidpointConstraintProxy/MidpointConstraintProxy.md)

## Description

Property that returns the sketch point being constrained.

## Syntax

MidpointConstraintProxy.**Point**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 6
