# MidpointConstraintProxy Object

Derived from: [MidpointConstraint](../MidpointConstraint/MidpointConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../MidpointConstraintProxy/MidpointConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../MidpointConstraintProxy/MidpointConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MidpointConstraintProxy/MidpointConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Arc](../MidpointConstraintProxy/MidpointConstraintProxy_Arc.md) | Gets the sketch arc being constrained. |
| [AttributeSets](../MidpointConstraintProxy/MidpointConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../MidpointConstraintProxy/MidpointConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../MidpointConstraintProxy/MidpointConstraintProxy_Deletable.md) | Indicates whether this object is deletable. |
| [Line](../MidpointConstraintProxy/MidpointConstraintProxy_Line.md) | Property that returns the sketch line being constrained. |
| [NativeObject](../MidpointConstraintProxy/MidpointConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../MidpointConstraintProxy/MidpointConstraintProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Point](../MidpointConstraintProxy/MidpointConstraintProxy_Point.md) | Property that returns the sketch point being constrained. |
| [Type](../MidpointConstraintProxy/MidpointConstraintProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
