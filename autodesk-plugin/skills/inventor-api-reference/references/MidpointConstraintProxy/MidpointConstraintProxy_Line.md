# MidpointConstraintProxy.Line Property

Parent Object: [MidpointConstraintProxy](../MidpointConstraintProxy/MidpointConstraintProxy.md)

## Description

Property that returns the sketch line being constrained.

## Syntax

MidpointConstraintProxy.**Line**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 6
