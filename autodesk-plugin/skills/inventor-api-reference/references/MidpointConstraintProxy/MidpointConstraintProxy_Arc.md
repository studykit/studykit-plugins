# MidpointConstraintProxy.Arc Property

Parent Object: [MidpointConstraintProxy](../MidpointConstraintProxy/MidpointConstraintProxy.md)

## Description

Gets the sketch arc being constrained.

## Syntax

MidpointConstraintProxy.**Arc**() As [SketchArc](../SketchArc/SketchArc.md)

## Property Value

This is a read only property whose value is a [SketchArc](../SketchArc/SketchArc.md).

## Version

Introduced in version 2015
