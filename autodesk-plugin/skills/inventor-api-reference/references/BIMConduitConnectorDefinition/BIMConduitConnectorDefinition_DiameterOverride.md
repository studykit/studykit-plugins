# BIMConduitConnectorDefinition.DiameterOverride Property

Parent Object: [BIMConduitConnectorDefinition](../BIMConduitConnectorDefinition/BIMConduitConnectorDefinition.md)

## Description

Read-write property that gets and sets the diameter override for this connector.

## Syntax

BIMConduitConnectorDefinition.**DiameterOverride**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2011
