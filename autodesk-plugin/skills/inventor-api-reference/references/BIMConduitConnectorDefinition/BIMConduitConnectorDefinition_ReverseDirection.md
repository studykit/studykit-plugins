# BIMConduitConnectorDefinition.ReverseDirection Method

Parent Object: [BIMConduitConnectorDefinition](../BIMConduitConnectorDefinition/BIMConduitConnectorDefinition.md)

## Description

Method that will reverse the direction of the connection.

## Syntax

BIMConduitConnectorDefinition.**ReverseDirection**()

## Version

Introduced in version 2011
