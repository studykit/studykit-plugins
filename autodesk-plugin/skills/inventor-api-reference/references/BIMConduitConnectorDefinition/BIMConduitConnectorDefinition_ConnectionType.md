# BIMConduitConnectorDefinition.ConnectionType Property

Parent Object: [BIMConduitConnectorDefinition](../BIMConduitConnectorDefinition/BIMConduitConnectorDefinition.md)

## Description

Read-write property that gets and sets the connection type for this connector.

## Syntax

BIMConduitConnectorDefinition.**ConnectionType**() As [BIMConduitConnectionTypeEnum](../BIMConduitConnectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BIMConduitConnectionTypeEnum](../BIMConduitConnectionTypeEnum.md).

## Version

Introduced in version 2011
