# ModelStateTableColumn.Index Property

Parent Object: [ModelStateTableColumn](../ModelStateTableColumn/ModelStateTableColumn.md)

## Description

Gets the the index of this column within the ModelStateTableColumns collection where the first column has an index of 1.

## Syntax

ModelStateTableColumn.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2022
