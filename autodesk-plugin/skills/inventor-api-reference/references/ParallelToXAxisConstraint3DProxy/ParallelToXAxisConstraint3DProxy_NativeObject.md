# ParallelToXAxisConstraint3DProxy.NativeObject Property

Parent Object: [ParallelToXAxisConstraint3DProxy](../ParallelToXAxisConstraint3DProxy/ParallelToXAxisConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ParallelToXAxisConstraint3DProxy.**NativeObject**() As [ParallelToXAxisConstraint3D](../ParallelToXAxisConstraint3D/ParallelToXAxisConstraint3D.md)

## Property Value

This is a read only property whose value is a [ParallelToXAxisConstraint3D](../ParallelToXAxisConstraint3D/ParallelToXAxisConstraint3D.md).

## Version

Introduced in version 2017
