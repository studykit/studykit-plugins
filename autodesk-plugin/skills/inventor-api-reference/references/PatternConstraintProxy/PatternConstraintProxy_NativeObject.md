# PatternConstraintProxy.NativeObject Property

Parent Object: [PatternConstraintProxy](../PatternConstraintProxy/PatternConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

PatternConstraintProxy.**NativeObject**() As [PatternConstraint](../PatternConstraint/PatternConstraint.md)

## Property Value

This is a read only property whose value is a [PatternConstraint](../PatternConstraint/PatternConstraint.md).

## Version

Introduced in version 6
