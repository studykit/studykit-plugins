# ChamferFeature.Type Property

Parent Object: [ChamferFeature](../ChamferFeature/ChamferFeature.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ChamferFeature.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
