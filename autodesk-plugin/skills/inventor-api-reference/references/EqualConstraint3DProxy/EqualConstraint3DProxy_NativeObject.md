# EqualConstraint3DProxy.NativeObject Property

Parent Object: [EqualConstraint3DProxy](../EqualConstraint3DProxy/EqualConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

EqualConstraint3DProxy.**NativeObject**() As [EqualConstraint3D](../EqualConstraint3D/EqualConstraint3D.md)

## Property Value

This is a read only property whose value is an [EqualConstraint3D](../EqualConstraint3D/EqualConstraint3D.md).

## Version

Introduced in version 2017
