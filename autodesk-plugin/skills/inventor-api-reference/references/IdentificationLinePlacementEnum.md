# IdentificationLinePlacementEnum Enumerator

## Description

Constants that define the identification line placement types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kIdentificationLineAbove | 122882 | Identification line above the reference line. |
| kIdentificationLineBelow | 122883 | Identification line below the reference line. |
| kNoIdentificationLine | 122881 | No identification line. |
