# Sketches3D Object

## Description

Provides access to all of the objects owned by a particular PartComponentDefinition. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Sketches3D/Sketches3D_Add.md) | Method that creates a new 3D sketch. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Sketches3D/Sketches3D_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Sketches3D/Sketches3D_Count.md) | Property that returns the number of items in the collection. |
| [Item](../Sketches3D/Sketches3D_Item.md) | Returns the specified Sketch3D object from the collection. |
| [Type](../Sketches3D/Sketches3D_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FlatPattern.Sketches3D](../FlatPattern/FlatPattern_Sketches3D.md), [PartComponentDefinition.Sketches3D](../PartComponentDefinition/PartComponentDefinition_Sketches3D.md), [SheetMetalComponentDefinition.Sketches3D](../SheetMetalComponentDefinition/SheetMetalComponentDefinition_Sketches3D.md)

## Version
