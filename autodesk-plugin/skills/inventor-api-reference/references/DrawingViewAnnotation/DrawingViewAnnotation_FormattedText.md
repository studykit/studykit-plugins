# DrawingViewAnnotation.FormattedText Property

Parent Object: [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md)

## Description

Gets and sets the formatted string value of the view annotation text.

## Syntax

DrawingViewAnnotation.**FormattedText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2026
