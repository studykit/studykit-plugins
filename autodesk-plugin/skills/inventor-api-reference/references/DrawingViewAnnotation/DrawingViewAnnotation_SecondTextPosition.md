# DrawingViewAnnotation.SecondTextPosition Property

Parent Object: [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md)

## Description

Gets and sets the position of the second view annotation text.

## Syntax

DrawingViewAnnotation.**SecondTextPosition**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2026
