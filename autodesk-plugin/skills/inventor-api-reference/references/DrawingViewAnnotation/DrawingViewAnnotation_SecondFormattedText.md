# DrawingViewAnnotation.SecondFormattedText Property

Parent Object: [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md)

## Description

Gets and sets the formatted string value of the second view annotation text.

## Syntax

DrawingViewAnnotation.**SecondFormattedText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2026
