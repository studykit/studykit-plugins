# DrawingViewAnnotation.SecondText Property

Parent Object: [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md)

## Description

Returns the second string value of the view annotation text.

## Syntax

DrawingViewAnnotation.**SecondText**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2026
