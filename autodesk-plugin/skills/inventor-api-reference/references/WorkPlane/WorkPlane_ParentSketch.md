# WorkPlane.ParentSketch Property

Parent Object: [WorkPlane](../WorkPlane/WorkPlane.md)

## Description

Gets the parent 3d sketch if this work plane belongs to a 3d sketch, Gets Nothing otherwise.

## Syntax

WorkPlane.**ParentSketch**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2008
