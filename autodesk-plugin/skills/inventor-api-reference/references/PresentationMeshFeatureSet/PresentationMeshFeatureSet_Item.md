# PresentationMeshFeatureSet.Item Property

Parent Object: [PresentationMeshFeatureSet](../PresentationMeshFeatureSet/PresentationMeshFeatureSet.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationMeshFeatureSet.**Item**( ***Index*** As Long ) As [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Property Value

This is a read only property whose value is a [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the PresentationMeshFeature to return. |

## Version

Introduced in version 2018
