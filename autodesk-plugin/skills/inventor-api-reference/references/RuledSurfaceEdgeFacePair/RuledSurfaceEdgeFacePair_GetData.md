# RuledSurfaceEdgeFacePair.GetData Method

Parent Object: [RuledSurfaceEdgeFacePair](../RuledSurfaceEdgeFacePair/RuledSurfaceEdgeFacePair.md)

## Description

Method that gets data from the edge face pair.

## Syntax

RuledSurfaceEdgeFacePair.**GetData**( ***Edge*** As [Edge](../Edge/Edge.md), ***Face*** As [Face](../Face/Face.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Edge | [Edge](../Edge/Edge.md) | Output Edge object. |
| Face | [Face](../Face/Face.md) | Output Face object. |

## Version

Introduced in version 2017
