# SketchEllipticalArc.Length Property

Parent Object: [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchEllipticalArc.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.
