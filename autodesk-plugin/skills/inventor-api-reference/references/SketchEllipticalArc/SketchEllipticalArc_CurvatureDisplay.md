# SketchEllipticalArc.CurvatureDisplay Property

Parent Object: [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Description

Specifies whether curvature information is displayed for the elliptical arc.

## Syntax

SketchEllipticalArc.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
