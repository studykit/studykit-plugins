# SketchEllipticalArc.Geometry Property

Parent Object: [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Description

Property that returns an EllipticalArc2d geometry object. The object returned represents a snapshot view of the current state of the sketch elliptical arc.

## Syntax

SketchEllipticalArc.**Geometry**() As [EllipticalArc2d](../EllipticalArc2d/EllipticalArc2d.md)

## Property Value

This is a read only property whose value is an [EllipticalArc2d](../EllipticalArc2d/EllipticalArc2d.md).

## Version

Introduced in version 5
