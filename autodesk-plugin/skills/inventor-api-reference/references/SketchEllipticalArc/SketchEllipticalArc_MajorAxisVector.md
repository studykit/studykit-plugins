# SketchEllipticalArc.MajorAxisVector Property

Parent Object: [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Description

Gets and sets the major axis vector. This vector defines the direction of the major axis.

## Syntax

SketchEllipticalArc.**MajorAxisVector**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read/write property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).
