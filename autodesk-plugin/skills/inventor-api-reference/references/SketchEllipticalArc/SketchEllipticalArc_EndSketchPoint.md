# SketchEllipticalArc.EndSketchPoint Property

Parent Object: [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Description

Property that returns the end of the elliptical arc.

## Syntax

SketchEllipticalArc.**EndSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 5
