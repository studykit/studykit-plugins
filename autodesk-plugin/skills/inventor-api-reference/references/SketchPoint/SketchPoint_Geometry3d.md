# SketchPoint.Geometry3d Property

Parent Object: [SketchPoint](../SketchPoint/SketchPoint.md)

## Description

Read-only property that returns point geometry that represents this point in model space.

## Syntax

SketchPoint.**Geometry3d**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2013
