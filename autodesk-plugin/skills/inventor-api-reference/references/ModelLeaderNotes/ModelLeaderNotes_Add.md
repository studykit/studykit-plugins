# ModelLeaderNotes.Add Method

Parent Object: [ModelLeaderNotes](../ModelLeaderNotes/ModelLeaderNotes.md)

## Description

Method that creates a leader note.

## Syntax

ModelLeaderNotes.**Add**( ***Definition*** As [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md) ) As [ModelLeaderNote](../ModelLeaderNote/ModelLeaderNote.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md) | Input ModelLeaderNoteDefinition object that defines the leader note to be created |

## Version

Introduced in version 2015
