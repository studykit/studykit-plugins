# DSSettings Object

## Description

The DSDegreeOfFreedom object represents a degree of freedom associated with a joint.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DSSettings/DSSettings_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Type](../DSSettings/DSSettings_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DynamicSimulation.DSSettings](../DynamicSimulation/DynamicSimulation_DSSettings.md)

## Version

Introduced in version 2013
