# SurfaceTextureGOSTDefinition.TextureDirection Property

Parent Object: [SurfaceTextureGOSTDefinition](../SurfaceTextureGOSTDefinition/SurfaceTextureGOSTDefinition.md)

## Description

Gets and sets the direction of texture for the surface.

## Syntax

SurfaceTextureGOSTDefinition.**TextureDirection**() As [LayDirectionTypeEnum](../LayDirectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LayDirectionTypeEnum](../LayDirectionTypeEnum.md).

## Version

Introduced in version 2016
