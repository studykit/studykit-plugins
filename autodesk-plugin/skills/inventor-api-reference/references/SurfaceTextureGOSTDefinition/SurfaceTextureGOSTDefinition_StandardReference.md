# SurfaceTextureGOSTDefinition.StandardReference Property

Parent Object: [SurfaceTextureGOSTDefinition](../SurfaceTextureGOSTDefinition/SurfaceTextureGOSTDefinition.md)

## Description

Returns the standard reference type.

## Syntax

SurfaceTextureGOSTDefinition.**StandardReference**() As [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md)

## Property Value

This is a read only property whose value is a [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md).

## Version

Introduced in version 2025
