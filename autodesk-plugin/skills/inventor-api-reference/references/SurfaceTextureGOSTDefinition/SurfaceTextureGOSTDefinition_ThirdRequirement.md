# SurfaceTextureGOSTDefinition.ThirdRequirement Property

Parent Object: [SurfaceTextureGOSTDefinition](../SurfaceTextureGOSTDefinition/SurfaceTextureGOSTDefinition.md)

## Description

Gets and sets the third requirement value for the surface.

## Syntax

SurfaceTextureGOSTDefinition.**ThirdRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
