# SketchSpline3D.CurvatureDisplay Property

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Specifies whether curvature information is displayed for the spline.

## Syntax

SketchSpline3D.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 8
