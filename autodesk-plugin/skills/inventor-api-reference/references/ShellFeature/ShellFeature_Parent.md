# ShellFeature.Parent Property

Parent Object: [ShellFeature](../ShellFeature/ShellFeature.md)

## Description

Property that returns the parent of the feature.

## Syntax

ShellFeature.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 5
