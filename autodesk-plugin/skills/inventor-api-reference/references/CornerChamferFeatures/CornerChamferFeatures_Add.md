# CornerChamferFeatures.Add Method

Parent Object: [CornerChamferFeatures](../CornerChamferFeatures/CornerChamferFeatures.md)

## Description

Method that creates a new corner chamfer feature. The newly created CornerChamferFeature object is returned.

## Syntax

CornerChamferFeatures.**Add**( ***CornerChamferDefinition*** As [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md) ) As [CornerChamferFeature](../CornerChamferFeature/CornerChamferFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CornerChamferDefinition | [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md) | Input CornerChamferFeature object that defines the corner chamfer feature you want to create. A CornerChamferDefinition object can be created using the CornerChamferFeatures.CreateCornerChamferDefinition method. It can also be obtained from an existing CornerChamferFeature object. |

## Version

Introduced in version 2010
