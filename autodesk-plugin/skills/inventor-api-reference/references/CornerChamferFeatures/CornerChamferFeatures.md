# CornerChamferFeatures Object

## Description

The CornerChamferFeatures collection object provides access to all of the objects in a sheet metal component definition.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../CornerChamferFeatures/CornerChamferFeatures_Add.md) | Method that creates a new corner chamfer feature. The newly created CornerChamferFeature object is returned. |
| [CreateCornerChamferDefinition](../CornerChamferFeatures/CornerChamferFeatures_CreateCornerChamferDefinition.md) | Method that creates a new CornerChamferDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CornerChamferFeatures/CornerChamferFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CornerChamferFeatures/CornerChamferFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../CornerChamferFeatures/CornerChamferFeatures_Item.md) | Returns the specified CornerChamferFeature object from the collection. This is the default property of the CornerChamferFeatures collection object. |
| [Type](../CornerChamferFeatures/CornerChamferFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FlatPatternFeatures.CornerChamferFeatures](../FlatPatternFeatures/FlatPatternFeatures_CornerChamferFeatures.md), [SheetMetalFeatures.CornerChamferFeatures](../SheetMetalFeatures/SheetMetalFeatures_CornerChamferFeatures.md)

## Version

Introduced in version 5.3
