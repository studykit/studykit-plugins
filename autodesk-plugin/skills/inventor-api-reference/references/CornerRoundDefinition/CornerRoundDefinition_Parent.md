# CornerRoundDefinition.Parent Property

Parent Object: [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md)

## Description

Property that returns the parent CornerRoundFeature of this CornerRoundDefinition object.

## Syntax

CornerRoundDefinition.**Parent**() As [CornerRoundFeature](../CornerRoundFeature/CornerRoundFeature.md)

## Property Value

This is a read only property whose value is a [CornerRoundFeature](../CornerRoundFeature/CornerRoundFeature.md).

## Version

Introduced in version 2009
