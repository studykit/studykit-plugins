# CornerRoundDefinition.EdgeSetItem Property

Parent Object: [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md)

## Description

Method that returns the specified CornerRoundEdgeSet object from this CornerRoundDefinition object.

## Syntax

CornerRoundDefinition.**EdgeSetItem**( ***Index*** As Long ) As [CornerRoundEdgeSet](../CornerRoundEdgeSet/CornerRoundEdgeSet.md)

## Property Value

This is a read only property whose value is a [CornerRoundEdgeSet](../CornerRoundEdgeSet/CornerRoundEdgeSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the CornerRoundEdgeSet to return. If an out of range index is provided, an error occurs. |

## Version

Introduced in version 2009
