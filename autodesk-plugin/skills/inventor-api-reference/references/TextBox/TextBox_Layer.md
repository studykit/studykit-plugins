# TextBox.Layer Property

Parent Object: [TextBox](../TextBox/TextBox.md)

## Description

Gets and sets the layer applied to this text box.

## Syntax

TextBox.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
