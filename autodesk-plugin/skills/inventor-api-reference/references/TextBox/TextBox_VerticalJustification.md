# TextBox.VerticalJustification Property

Parent Object: [TextBox](../TextBox/TextBox.md)

## Description

Gets and sets the vertical justification override of the text box.

## Syntax

TextBox.**VerticalJustification**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).
