# FlangeReferenceTypeEnum Enumerator

## Description

Defines the constants used to specify the reference type for flange feature.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kReferencePlaneAtSideFace | 126467 | Defines the Reference Plane as the Side Face next to the selected edge. |
| kReferencePlaneThroughPartnerEdge | 126465 | Defines the Reference Plane through the partner Edge of the selected edge. |
| kReferencePlaneThroughSelectedEdge | 126466 | Defines the Reference Plane through the selected edge. |
| kReferenceSelectedFaceOrWorkPlane | 126468 | Defines the Reference Plane as a user selected Work Plane or Planar Face. |

## Version

Introduced in version 2025
