# DerivedAssemblyComponentProxy.SuppressAll Property

Parent Object: [DerivedAssemblyComponentProxy](../DerivedAssemblyComponentProxy/DerivedAssemblyComponentProxy.md)

## Description

Property that gets and sets the suppress state for all of the Reference features created by this derived assembly component.

## Syntax

DerivedAssemblyComponentProxy.**SuppressAll**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
