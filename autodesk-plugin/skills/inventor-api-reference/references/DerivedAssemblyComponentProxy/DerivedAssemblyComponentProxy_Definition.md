# DerivedAssemblyComponentProxy.Definition Property

Parent Object: [DerivedAssemblyComponentProxy](../DerivedAssemblyComponentProxy/DerivedAssemblyComponentProxy.md)

## Description

Gets or sets DerivedAssemblyDefinition that defines the current state of this derived assembly.

## Syntax

DerivedAssemblyComponentProxy.**Definition**() As [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Property Value

This is a read/write property whose value is a [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md).

## Version

Introduced in version 2008
