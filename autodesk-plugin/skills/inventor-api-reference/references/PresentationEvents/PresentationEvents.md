# PresentationEvents Object

## Description

The Presentation Events Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PresentationEvents/PresentationEvents_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Parent](../PresentationEvents/PresentationEvents_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Type](../PresentationEvents/PresentationEvents_Type.md) | Gets the constant that indicates the type of this object. |

## Events

|  |  |
| --- | --- |
| Name | Description |
| [OnDelete](../PresentationEvents/PresentationEvents_OnDelete.md) | Event that fires just before and after an entity is deleted. |
| [OnNewPublication](../PresentationEvents/PresentationEvents_OnNewPublication.md) | Event that fires just before and after a publication is created. |
| [OnNewPublicationMarkedView](../PresentationEvents/PresentationEvents_OnNewPublicationMarkedView.md) | Event that fires just before and after a publication marked view is added. |
| [OnPublicationComponentChange](../PresentationEvents/PresentationEvents_OnPublicationComponentChange.md) | Event that fires just before and after a publication component is changed. |
| [OnPublicationDesignViewChange](../PresentationEvents/PresentationEvents_OnPublicationDesignViewChange.md) | Event that fires just before and after a publication design view is changed. |
| [OnPublicationModelingDataUpdate](../PresentationEvents/PresentationEvents_OnPublicationModelingDataUpdate.md) | Event that fires just before and after a publication component is changed. |
| [OnPublicationNameChange](../PresentationEvents/PresentationEvents_OnPublicationNameChange.md) | Event that fires just before and after a publication component’s name is changed. |
| [OnStoryboardChange](../PresentationEvents/PresentationEvents_OnStoryboardChange.md) | Event that fires just before and after a publication is updated because of modeling data changes. |

## Version

Introduced in version 2017
