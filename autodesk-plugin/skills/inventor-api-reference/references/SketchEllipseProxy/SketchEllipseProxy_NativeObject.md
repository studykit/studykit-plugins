# SketchEllipseProxy.NativeObject Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchEllipseProxy.**NativeObject**() As [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Property Value

This is a read only property whose value is a [SketchEllipse](../SketchEllipse/SketchEllipse.md).

## Version

Introduced in version 6
