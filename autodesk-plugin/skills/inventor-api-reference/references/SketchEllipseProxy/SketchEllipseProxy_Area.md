# SketchEllipseProxy.Area Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Double property that returns the area of the entity in square centimeters.

## Syntax

SketchEllipseProxy.**Area**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
