# SketchEllipseProxy.Construction Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Gets and sets whether the entity behaves as a construction entity or not.

## Syntax

SketchEllipseProxy.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
