# CosmeticWeld.Name Property

Parent Object: [CosmeticWeld](../CosmeticWeld/CosmeticWeld.md)

## Description

Gets and sets the name of the Weld.

## Syntax

CosmeticWeld.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 8
