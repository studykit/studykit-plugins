# CosmeticWeld.Type Property

Parent Object: [CosmeticWeld](../CosmeticWeld/CosmeticWeld.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CosmeticWeld.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 8
