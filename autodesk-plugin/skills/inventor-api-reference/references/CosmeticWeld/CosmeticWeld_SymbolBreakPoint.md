# CosmeticWeld.SymbolBreakPoint Property

Parent Object: [CosmeticWeld](../CosmeticWeld/CosmeticWeld.md)

## Description

Property that returns the coordinate where the weld symbol leader line break point is.

## Syntax

CosmeticWeld.**SymbolBreakPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 8
