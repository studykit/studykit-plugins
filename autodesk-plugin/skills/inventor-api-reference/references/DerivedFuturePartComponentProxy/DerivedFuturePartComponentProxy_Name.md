# DerivedFuturePartComponentProxy.Name Property

Parent Object: [DerivedFuturePartComponentProxy](../DerivedFuturePartComponentProxy/DerivedFuturePartComponentProxy.md)

## Description

Property that returns the component's name.

## Syntax

DerivedFuturePartComponentProxy.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
