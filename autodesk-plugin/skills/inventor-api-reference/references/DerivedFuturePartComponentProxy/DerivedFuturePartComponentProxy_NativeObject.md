# DerivedFuturePartComponentProxy.NativeObject Property

Parent Object: [DerivedFuturePartComponentProxy](../DerivedFuturePartComponentProxy/DerivedFuturePartComponentProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DerivedFuturePartComponentProxy.**NativeObject**() As [DerivedFuturePartComponent](../DerivedFuturePartComponent/DerivedFuturePartComponent.md)

## Property Value

This is a read only property whose value is a [DerivedFuturePartComponent](../DerivedFuturePartComponent/DerivedFuturePartComponent.md).

## Version

Introduced in version 2018
