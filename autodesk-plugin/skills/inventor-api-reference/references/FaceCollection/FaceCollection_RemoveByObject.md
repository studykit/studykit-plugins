# FaceCollection.RemoveByObject Method

Parent Object: [FaceCollection](../FaceCollection/FaceCollection.md)

## Description

Method that removes the specified object from the generic object collection.

## Syntax

FaceCollection.**RemoveByObject**( ***Object*** As Object )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Object | Object | Input Object to remove. |

## Version

Introduced in version 9
