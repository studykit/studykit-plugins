# FaceCollection.Remove Method

Parent Object: [FaceCollection](../FaceCollection/FaceCollection.md)

## Description

Method that removes the specified object from the generic collection.

## Syntax

FaceCollection.**Remove**( ***Index*** As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to remove. |

## Version

Introduced in version 9
