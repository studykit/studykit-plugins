# ClientView.Document Property

Parent Object: [ClientView](../ClientView/ClientView.md)

## Description

Property that returns the document object this is a view of.

## Syntax

ClientView.**Document**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
