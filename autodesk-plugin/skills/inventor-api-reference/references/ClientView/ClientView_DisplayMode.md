# ClientView.DisplayMode Property

Parent Object: [ClientView](../ClientView/ClientView.md)

## Description

Rendering mode of view (shaded, hidden, hidden line).

## Syntax

ClientView.**DisplayMode**() As [DisplayModeEnum](../DisplayModeEnum.md)

## Property Value

This is a read/write property whose value is a [DisplayModeEnum](../DisplayModeEnum.md).

## Version

Introduced in version 6
