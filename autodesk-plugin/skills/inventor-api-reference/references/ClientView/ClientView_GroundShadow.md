# ClientView.GroundShadow Property

Parent Object: [ClientView](../ClientView/ClientView.md)

## Description

Gets and sets the ground shadow setting on the view!/s window.

## Syntax

ClientView.**GroundShadow**() As [GroundShadowEnum](../GroundShadowEnum.md)

## Property Value

This is a read/write property whose value is a [GroundShadowEnum](../GroundShadowEnum.md).

## Version

Introduced in version 2008
