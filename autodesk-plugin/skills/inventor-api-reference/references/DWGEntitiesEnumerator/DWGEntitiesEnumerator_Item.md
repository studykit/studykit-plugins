# DWGEntitiesEnumerator.Item Property

Parent Object: [DWGEntitiesEnumerator](../DWGEntitiesEnumerator/DWGEntitiesEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGEntitiesEnumerator.**Item**( ***Index*** As Variant ) As [DWGEntity](../DWGEntity/DWGEntity.md)

## Property Value

This is a read only property whose value is a [DWGEntity](../DWGEntity/DWGEntity.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
