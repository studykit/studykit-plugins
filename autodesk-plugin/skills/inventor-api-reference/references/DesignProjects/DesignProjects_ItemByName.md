# DesignProjects.ItemByName Property

Parent Object: [DesignProjects](../DesignProjects/DesignProjects.md)

## Description

Returns the specified DesignProject object from the collection.

## Syntax

DesignProjects.**ItemByName**( ***NameOrFileName*** As String ) As [DesignProject](../DesignProject/DesignProject.md)

## Property Value

This is a read only property whose value is a [DesignProject](../DesignProject/DesignProject.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| NameOrFileName | String | Input String that specifies either the FullFileName or the Name of the design project to return. |
