# TextureMaps.Item Property

Parent Object: [TextureMaps](../TextureMaps/TextureMaps.md)

## Description

Returns the specified object from the collection.

## Syntax

TextureMaps.**Item**( ***Index*** As Long ) As [TextureMap](../TextureMap/TextureMap.md)

## Property Value

This is a read only property whose value is a [TextureMap](../TextureMap/TextureMap.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the object to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the name of the item. |

## Version

Introduced in version 10
