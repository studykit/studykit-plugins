# ImportedDataExchangeComponent.BreakLinkToFile Method

Parent Object: [ImportedDataExchangeComponent](../ImportedDataExchangeComponent/ImportedDataExchangeComponent.md)

## Description

Method that breaks the connection of the derived component with the part or assembly from which it was created. When the link is broken the ReferencedFile property will return Nothing.

## Syntax

ImportedDataExchangeComponent.**BreakLinkToFile**()

## Version

Introduced in version 2023
