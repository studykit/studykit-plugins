# ModelGeneralNoteDefinition.SymbolDefinitions Property

Parent Object: [ModelGeneralNoteDefinition](../ModelGeneralNoteDefinition/ModelGeneralNoteDefinition.md)

## Description

Returns the GeneralNoteSymbolDefinitions object, which allows you add model annotation definitions that can be used to display their symbols in the general note.

## Syntax

ModelGeneralNoteDefinition.**SymbolDefinitions**() As [GeneralNoteSymbolDefinitions](../GeneralNoteSymbolDefinitions/GeneralNoteSymbolDefinitions.md)

## Property Value

This is a read only property whose value is a [GeneralNoteSymbolDefinitions](../GeneralNoteSymbolDefinitions/GeneralNoteSymbolDefinitions.md).

## Version

Introduced in version 2018
