# EmbossFeature.TopFaceAppearanceSourceType Property

Parent Object: [EmbossFeature](../EmbossFeature/EmbossFeature.md)

## Description

Gets and sets the source of the appearance for the top face of the emboss feature.

## Syntax

EmbossFeature.**TopFaceAppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
