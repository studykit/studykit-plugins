# EmbossFeature.TopFaceAppearance Property

Parent Object: [EmbossFeature](../EmbossFeature/EmbossFeature.md)

## Description

Gets and sets the top face appearance of the emboss feature.

## Syntax

EmbossFeature.**TopFaceAppearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
