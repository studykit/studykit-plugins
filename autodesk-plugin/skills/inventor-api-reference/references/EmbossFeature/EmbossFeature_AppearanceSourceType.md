# EmbossFeature.AppearanceSourceType Property

Parent Object: [EmbossFeature](../EmbossFeature/EmbossFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

EmbossFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
