# EmbossFeature.Taper Property

Parent Object: [EmbossFeature](../EmbossFeature/EmbossFeature.md)

## Description

Property that returns the parameter controlling the taper angle of the emboss feature.

## Syntax

EmbossFeature.**Taper**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2008
