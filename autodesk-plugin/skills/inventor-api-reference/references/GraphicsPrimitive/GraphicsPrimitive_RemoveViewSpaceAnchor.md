# GraphicsPrimitive.RemoveViewSpaceAnchor Method

Parent Object: [GraphicsPrimitive](../GraphicsPrimitive/GraphicsPrimitive.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

GraphicsPrimitive.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
