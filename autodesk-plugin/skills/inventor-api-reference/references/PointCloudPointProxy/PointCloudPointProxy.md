# PointCloudPointProxy Object

Derived from: [PointCloudPoint](../PointCloudPoint/PointCloudPoint.md) Object

## Description

PointCloudPointProxy Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointCloudPointProxy/PointCloudPointProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [ContainingOccurrence](../PointCloudPointProxy/PointCloudPointProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Geometry](../PointCloudPointProxy/PointCloudPointProxy_Geometry.md) | Read-only property that returns a Point object which provides the coordinates of this point. |
| [NativeObject](../PointCloudPointProxy/PointCloudPointProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../PointCloudPointProxy/PointCloudPointProxy_Parent.md) | Read-only property that returns the parent point cloud of this point. |
| [Type](../PointCloudPointProxy/PointCloudPointProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2016
