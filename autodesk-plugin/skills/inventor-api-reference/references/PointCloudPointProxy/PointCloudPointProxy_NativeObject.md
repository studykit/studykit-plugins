# PointCloudPointProxy.NativeObject Property

Parent Object: [PointCloudPointProxy](../PointCloudPointProxy/PointCloudPointProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

PointCloudPointProxy.**NativeObject**() As [PointCloudPoint](../PointCloudPoint/PointCloudPoint.md)

## Property Value

This is a read only property whose value is a [PointCloudPoint](../PointCloudPoint/PointCloudPoint.md).

## Version

Introduced in version 2016
