# Cone.AxisVector Property

Parent Object: [Cone](../Cone/Cone.md)

## Description

Specifies the direction of the axis of the cone.

## Syntax

Cone.**AxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
