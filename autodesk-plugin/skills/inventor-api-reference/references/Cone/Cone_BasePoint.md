# Cone.BasePoint Property

Parent Object: [Cone](../Cone/Cone.md)

## Description

Specifies the base of the cone.

## Syntax

Cone.**BasePoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
