# Cone Object

## Description

The Cone object. The object is a transient mathematical object and is not displayed graphically.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Cone/Cone_Copy.md) | Creates a copy of this Cone object. The result is entirely independent and can be edited without affecting the original Cone object. |
| [GetConeData](../Cone/Cone_GetConeData.md) | Get the data defining this cone. |
| [PutConeData](../Cone/Cone_PutConeData.md) | Method that sets the data defining this cone. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AxisVector](../Cone/Cone_AxisVector.md) | Specifies the direction of the axis of the cone. |
| [BasePoint](../Cone/Cone_BasePoint.md) | Specifies the base of the cone. |
| [Evaluator](../Cone/Cone_Evaluator.md) | Gets the surface evaluator for this cone. |
| [HalfAngle](../Cone/Cone_HalfAngle.md) | Specifies the taper angle of the cone. This is the angle measured from the axis to the side of the cone. |
| [IsExpanding](../Cone/Cone_IsExpanding.md) | Indicates whether the taper of the cone is expanding outwards or inwards in the direction of the axis vector. If True and the HalfAngle is positive, then the cone is expanding outward along the axis vector. |
| [Radius](../Cone/Cone_Radius.md) | Specifies the radius of the cone at the base point. |

## Accessed From

[Cone.Copy](../Cone/Cone_Copy.md), [TransientGeometry.CreateCone](../TransientGeometry/TransientGeometry_CreateCone.md)

## Version

Introduced in version 4
