# FaceShellProxy.NativeObject Property

Parent Object: [FaceShellProxy](../FaceShellProxy/FaceShellProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FaceShellProxy.**NativeObject**() As [FaceShell](../FaceShell/FaceShell.md)

## Property Value

This is a read only property whose value is a [FaceShell](../FaceShell/FaceShell.md).

## Version

Introduced in version 4
