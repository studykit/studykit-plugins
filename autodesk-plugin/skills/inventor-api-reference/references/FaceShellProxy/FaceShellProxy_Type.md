# FaceShellProxy.Type Property

Parent Object: [FaceShellProxy](../FaceShellProxy/FaceShellProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

FaceShellProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
