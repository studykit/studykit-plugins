# DWGBlockReference.HandleID Property

Parent Object: [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGBlockReference.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
