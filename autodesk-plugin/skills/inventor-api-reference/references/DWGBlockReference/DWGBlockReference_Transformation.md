# DWGBlockReference.Transformation Property

Parent Object: [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md)

## Description

Read-only property that returns the transformation matrix for the DWGBlockReference instance.

## Syntax

DWGBlockReference.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2016
