# TransientGeometry.CreateCircle2dByThreePoints Method

Parent Object: [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Description

Method that creates a new Circle2D object by three points. The object created is a transient mathematical object and is not displayed graphically.

## Syntax

TransientGeometry.**CreateCircle2dByThreePoints**( ***PointOne*** As [Point2d](../Point2d/Point2d.md), ***PointTwo*** As [Point2d](../Point2d/Point2d.md), ***PointThree*** As [Point2d](../Point2d/Point2d.md) ) As [Circle2d](../Circle2d/Circle2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PointOne | [Point2d](../Point2d/Point2d.md) | First of three points. |
| PointTwo | [Point2d](../Point2d/Point2d.md) | Second of three points. |
| PointThree | [Point2d](../Point2d/Point2d.md) | Third of three points. |

## Version

Introduced in version 2008
