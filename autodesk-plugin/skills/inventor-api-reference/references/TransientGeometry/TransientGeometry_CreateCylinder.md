# TransientGeometry.CreateCylinder Method

Parent Object: [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Description

Method that creates a new Cylinder object. The object created is a transient mathematical object and is not displayed graphically.

## Syntax

TransientGeometry.**CreateCylinder**( ***RootPoint*** As [Point](../Point/Point.md), ***Axis*** As [UnitVector](../UnitVector/UnitVector.md), ***Radius*** As Double ) As [Cylinder](../Cylinder/Cylinder.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| RootPoint | [Point](../Point/Point.md) | Input Point object that specifies the base of the cylinder. |
| Axis | [UnitVector](../UnitVector/UnitVector.md) | Input UnitVector object that specifies the direction of the axis of the cylinder. |
| Radius | Double | Input Double that specifies the radius of the cylinder. |

## Version

Introduced in version 4
