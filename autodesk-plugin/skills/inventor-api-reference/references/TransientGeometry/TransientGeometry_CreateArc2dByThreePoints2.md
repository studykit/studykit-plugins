# TransientGeometry.CreateArc2dByThreePoints2 Method

Parent Object: [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Description

Method that creates a Arc2d object.

## Syntax

TransientGeometry.**CreateArc2dByThreePoints2**( ***PointOne*** As [Point2d](../Point2d/Point2d.md), ***PointTwo*** As [Point2d](../Point2d/Point2d.md), ***PointThree*** As [Point2d](../Point2d/Point2d.md) ) As [Arc2d](../Arc2d/Arc2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PointOne | [Point2d](../Point2d/Point2d.md) | Input Point2d that defines one of ending points of the arc. |
| PointTwo | [Point2d](../Point2d/Point2d.md) | Input Point2d that defines a point on the arc. |
| PointThree | [Point2d](../Point2d/Point2d.md) | Input Point2d that defines the other ending point of the arc. |

## Version

Introduced in version 2025.1
