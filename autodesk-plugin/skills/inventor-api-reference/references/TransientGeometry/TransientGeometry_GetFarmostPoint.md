# TransientGeometry.GetFarmostPoint Method

Parent Object: [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Description

Gets the farmost point along the direction.

## Syntax

TransientGeometry.**GetFarmostPoint**( ***Entity*** As Object, ***Direction*** As [UnitVector](../UnitVector/UnitVector.md) ) As [Point](../Point/Point.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Entity | Object | Input the entity object from which to get the farmost point along specified direction. Valid entity can be Edge, Face and SurfaceBody. |
| Direction | [UnitVector](../UnitVector/UnitVector.md) | Input the UnitVector object to specify the direction. |

## Version

Introduced in version 2015
