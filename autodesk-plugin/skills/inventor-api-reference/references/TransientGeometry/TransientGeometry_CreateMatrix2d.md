# TransientGeometry.CreateMatrix2d Method

Parent Object: [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Description

Method that creates a new 3x3 Matrix object. The matrix is initialized with an identity matrix.

## Syntax

TransientGeometry.**CreateMatrix2d**() As [Matrix2d](../Matrix2d/Matrix2d.md)

## Version

Introduced in version 4
