# FaceDefinition.IsParamReversed Property

Parent Object: [FaceDefinition](../FaceDefinition/FaceDefinition.md)

## Description

Read write property that indicates if the normal of this face is reversed with respect to the geometry associated with this face definition.

## Syntax

FaceDefinition.**IsParamReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
