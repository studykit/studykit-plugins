# IntersectionCurveProxy.NativeObject Property

Parent Object: [IntersectionCurveProxy](../IntersectionCurveProxy/IntersectionCurveProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

IntersectionCurveProxy.**NativeObject**() As [IntersectionCurve](../IntersectionCurve/IntersectionCurve.md)

## Property Value

This is a read only property whose value is an [IntersectionCurve](../IntersectionCurve/IntersectionCurve.md).

## Version

Introduced in version 2014
