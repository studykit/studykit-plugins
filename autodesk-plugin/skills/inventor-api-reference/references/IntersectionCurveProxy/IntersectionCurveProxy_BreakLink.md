# IntersectionCurveProxy.BreakLink Method

Parent Object: [IntersectionCurveProxy](../IntersectionCurveProxy/IntersectionCurveProxy.md)

## Description

Method that breaks the link between the intersection curve and the model. This breaks the associativity to the model, allowing you to edit the individual sketch entities.

## Syntax

IntersectionCurveProxy.**BreakLink**()

## Version

Introduced in version 2014
