# ModelFeatureControlFrameRow.ToleranceModifierPrefix Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the symbol to use as a prefix for the tolerance.

## Syntax

ModelFeatureControlFrameRow.**ToleranceModifierPrefix**() As [GDTSymbolEnum](../GDTSymbolEnum.md)

## Property Value

This is a read/write property whose value is a [GDTSymbolEnum](../GDTSymbolEnum.md).

## Version

Introduced in version 2018
