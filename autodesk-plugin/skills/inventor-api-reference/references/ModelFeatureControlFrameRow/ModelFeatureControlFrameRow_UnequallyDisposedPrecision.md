# ModelFeatureControlFrameRow.UnequallyDisposedPrecision Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the Unequally Disposed precision associated with this geometric characteristic. The value is a short integer.

## Syntax

ModelFeatureControlFrameRow.**UnequallyDisposedPrecision**() As Integer

## Property Value

This is a read/write property whose value is an Integer.

## Version

Introduced in version 2019
