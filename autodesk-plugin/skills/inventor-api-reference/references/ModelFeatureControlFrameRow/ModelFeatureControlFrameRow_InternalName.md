# ModelFeatureControlFrameRow.InternalName Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets the internal name (GUID) of the model feature control frame row.

## Syntax

ModelFeatureControlFrameRow.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
