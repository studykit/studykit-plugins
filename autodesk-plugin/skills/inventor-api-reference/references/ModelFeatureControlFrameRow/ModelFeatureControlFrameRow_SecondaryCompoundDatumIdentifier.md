# ModelFeatureControlFrameRow.SecondaryCompoundDatumIdentifier Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the secondary compound datum.

## Syntax

ModelFeatureControlFrameRow.**SecondaryCompoundDatumIdentifier**() As [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md)

## Property Value

This is a read/write property whose value is a [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md).

## Version

Introduced in version 2018
