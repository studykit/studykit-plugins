# ModelFeatureControlFrameRow.DisplayCommonZoneModifier Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets whether to display the common zone modifier or not.

## Syntax

ModelFeatureControlFrameRow.**DisplayCommonZoneModifier**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
