# ModelFeatureControlFrameRow.SecondaryCompoundDatumModifier Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the secondary compound datum modifier.

## Syntax

ModelFeatureControlFrameRow.**SecondaryCompoundDatumModifier**() As [GDTSymbolEnum](../GDTSymbolEnum.md)

## Property Value

This is a read/write property whose value is a [GDTSymbolEnum](../GDTSymbolEnum.md).

## Version

Introduced in version 2018
