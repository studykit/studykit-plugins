# ModelFeatureControlFrameRow.DisplayProjectedToleranceZoneModifier Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets whether to display the projected tolerance zone modifier or not.

## Syntax

ModelFeatureControlFrameRow.**DisplayProjectedToleranceZoneModifier**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
