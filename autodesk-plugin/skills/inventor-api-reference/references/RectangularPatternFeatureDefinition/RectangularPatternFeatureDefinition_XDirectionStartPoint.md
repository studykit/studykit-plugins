# RectangularPatternFeatureDefinition.XDirectionStartPoint Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the X direction start point. This can be a vertex, planar sketch point or a 3D sketch point.

## Syntax

RectangularPatternFeatureDefinition.**XDirectionStartPoint**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2017
