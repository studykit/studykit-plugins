# RectangularPatternFeatureDefinition.OrientationMethod Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the type of operation used to add the feature to the model. Valid values are kNewBodyOperation and kJoinOperation.

## Syntax

RectangularPatternFeatureDefinition.**OrientationMethod**() As [PatternOrientationEnum](../PatternOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [PatternOrientationEnum](../PatternOrientationEnum.md).

## Version

Introduced in version 2017
