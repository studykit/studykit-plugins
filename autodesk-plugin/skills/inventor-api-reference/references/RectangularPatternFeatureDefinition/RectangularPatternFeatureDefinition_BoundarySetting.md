# RectangularPatternFeatureDefinition.BoundarySetting Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the PatternBoundarySetting object.

## Syntax

RectangularPatternFeatureDefinition.**BoundarySetting**() As [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Property Value

This is a read/write property whose value is a [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md).
