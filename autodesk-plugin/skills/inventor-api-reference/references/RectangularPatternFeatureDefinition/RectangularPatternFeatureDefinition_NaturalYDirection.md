# RectangularPatternFeatureDefinition.NaturalYDirection Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the flag that indicates if the direction of the pattern is in the natural direction of the YDirectionEntity or reversed.

## Syntax

RectangularPatternFeatureDefinition.**NaturalYDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
