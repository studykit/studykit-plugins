# RectangularPatternFeatureDefinition.YDirectionMidPlanePattern Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets whether the patterned occurrences should fall on either side of the original in the Y direction.

## Syntax

RectangularPatternFeatureDefinition.**YDirectionMidPlanePattern**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
