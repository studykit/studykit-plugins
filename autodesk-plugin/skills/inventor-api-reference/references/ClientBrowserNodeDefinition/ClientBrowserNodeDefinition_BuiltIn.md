# ClientBrowserNodeDefinition.BuiltIn Property

Parent Object: [ClientBrowserNodeDefinition](../ClientBrowserNodeDefinition/ClientBrowserNodeDefinition.md)

## Description

Specifies if the node is a standard Autodesk Inventor node or not.

## Syntax

ClientBrowserNodeDefinition.**BuiltIn**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 8
