# ClientBrowserNodeDefinition.Id Property

Parent Object: [ClientBrowserNodeDefinition](../ClientBrowserNodeDefinition/ClientBrowserNodeDefinition.md)

## Description

Gets the client-supplied Id of the client browser node.

## Syntax

ClientBrowserNodeDefinition.**Id**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 8
