# ClientBrowserNodeDefinition.DisplayState Property

Parent Object: [ClientBrowserNodeDefinition](../ClientBrowserNodeDefinition/ClientBrowserNodeDefinition.md)

## Description

Gets and sets the display state of browsernodes that use this definition.

## Syntax

ClientBrowserNodeDefinition.**DisplayState**() As [BrowserNodeDisplayStateEnum](../BrowserNodeDisplayStateEnum.md)

## Property Value

This is a read/write property whose value is a [BrowserNodeDisplayStateEnum](../BrowserNodeDisplayStateEnum.md).

## Version

Introduced in version 9
