# AssetCategories.Type Property

Parent Object: [AssetCategories](../AssetCategories/AssetCategories.md)

## Description

Read-only property returning kAssetCategoriesObject indicating this object’s type.

## Syntax

AssetCategories.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
