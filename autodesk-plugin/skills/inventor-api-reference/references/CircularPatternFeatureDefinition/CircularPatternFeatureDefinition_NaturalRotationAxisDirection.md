# CircularPatternFeatureDefinition.NaturalRotationAxisDirection Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the flag that indicates if the direction of the pattern is in the natural direction of the RotationAxis or reversed.

## Syntax

CircularPatternFeatureDefinition.**NaturalRotationAxisDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
