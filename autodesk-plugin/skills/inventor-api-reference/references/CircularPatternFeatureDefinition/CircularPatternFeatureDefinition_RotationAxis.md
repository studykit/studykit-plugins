# CircularPatternFeatureDefinition.RotationAxis Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the entity as rotation axis. This can be a linear Edge, a WorkAxis, a cylindrical or conical Face (axis is used).

## Syntax

CircularPatternFeatureDefinition.**RotationAxis**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2017
