# CircularPatternFeatureDefinition.LockRotation Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets whether the patterned occurrences keep the same rotation as their parent features.

## Syntax

CircularPatternFeatureDefinition.**LockRotation**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
