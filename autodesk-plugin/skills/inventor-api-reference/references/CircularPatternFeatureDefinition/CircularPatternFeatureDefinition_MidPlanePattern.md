# CircularPatternFeatureDefinition.MidPlanePattern Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets whether the patterned occurrences should fall on either side of the original.

## Syntax

CircularPatternFeatureDefinition.**MidPlanePattern**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
