# CircularPatternFeatureDefinition.PatternRadiusPoint Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the point used to define the pattern radius against the rotation axis.

## Syntax

CircularPatternFeatureDefinition.**PatternRadiusPoint**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2017
