# CircularPatternFeatureDefinition.Operation Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the type of operation used to add the feature to the model. Valid values are kNewBodyOperation and kJoinOperation.

## Syntax

CircularPatternFeatureDefinition.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2017
