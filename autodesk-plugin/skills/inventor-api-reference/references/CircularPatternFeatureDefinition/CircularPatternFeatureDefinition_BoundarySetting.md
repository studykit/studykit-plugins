# CircularPatternFeatureDefinition.BoundarySetting Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the PatternBoundarySetting object.

## Syntax

CircularPatternFeatureDefinition.**BoundarySetting**() As [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Property Value

This is a read/write property whose value is a [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md).

## Version

Introduced in version 2025
