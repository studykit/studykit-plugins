# CircularPatternFeatureDefinition.Count Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the number of instances.

## Syntax

CircularPatternFeatureDefinition.**Count**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2017
