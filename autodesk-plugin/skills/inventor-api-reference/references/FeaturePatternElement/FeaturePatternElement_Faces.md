# FeaturePatternElement.Faces Property

Parent Object: [FeaturePatternElement](../FeaturePatternElement/FeaturePatternElement.md)

## Description

Property that returns a collection object generated as a result of the feature.

## Syntax

FeaturePatternElement.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 5
