# FeaturePatternElement.Parent Property

Parent Object: [FeaturePatternElement](../FeaturePatternElement/FeaturePatternElement.md)

## Description

Property that returns the feature pattern this element is a member of.

## Syntax

FeaturePatternElement.**Parent**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 5
