# TwoLineAngleDimConstraint3D.AnchorPoints Property

Parent Object: [TwoLineAngleDimConstraint3D](../TwoLineAngleDimConstraint3D/TwoLineAngleDimConstraint3D.md)

## Description

Gets the anchor points of dimension.

## Syntax

TwoLineAngleDimConstraint3D.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
