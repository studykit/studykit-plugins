# TwoLineAngleDimConstraint3D.Parameter Property

Parent Object: [TwoLineAngleDimConstraint3D](../TwoLineAngleDimConstraint3D/TwoLineAngleDimConstraint3D.md)

## Description

Property that returns the parameter associated with this dimension constraint. If the Driven property of the constraint is True, this will return a ReferenceParameter object. Otherwise it will return a ModelParameter object.

## Syntax

TwoLineAngleDimConstraint3D.**Parameter**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 11
