# ThemesEnumerator.Item Property

Parent Object: [ThemesEnumerator](../ThemesEnumerator/ThemesEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

ThemesEnumerator.**Item**( ***Index*** As Long ) As [Theme](../Theme/Theme.md)

## Property Value

This is a read only property whose value is a [Theme](../Theme/Theme.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the item to return. |

## Version

Introduced in version 2021
