# PresentationTrailSegment.Geometry Property

Parent Object: [PresentationTrailSegment](../PresentationTrailSegment/PresentationTrailSegment.md)

## Description

Read-only property that returns the tweak trail representative geometry.

## Syntax

PresentationTrailSegment.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
