# PresentationTrailSegment.Visible Property

Parent Object: [PresentationTrailSegment](../PresentationTrailSegment/PresentationTrailSegment.md)

## Description

Read-write property that gets and sets whether the presentation trail segment is visible or not.

## Syntax

PresentationTrailSegment.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
