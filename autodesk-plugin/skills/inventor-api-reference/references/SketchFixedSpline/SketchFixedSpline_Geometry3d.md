# SketchFixedSpline.Geometry3d Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Read-only property that returns B-Spline geometry that represents this spline in model space.

## Syntax

SketchFixedSpline.**Geometry3d**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read only property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 2013
