# SketchFixedSpline.OverrideColor Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Gets and sets the color applied to this sketch fixed spline.

## Syntax

SketchFixedSpline.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
