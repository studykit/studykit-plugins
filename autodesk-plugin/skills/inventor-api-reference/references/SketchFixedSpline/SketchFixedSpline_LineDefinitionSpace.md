# SketchFixedSpline.LineDefinitionSpace Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Gets and sets the LineDefinitionSpace applied to this sketch fixed spline.

## Syntax

SketchFixedSpline.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 9
