# SketchFixedSpline.Construction Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Gets and sets whether the entity behaves as a construction entity or not.

## Syntax

SketchFixedSpline.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
