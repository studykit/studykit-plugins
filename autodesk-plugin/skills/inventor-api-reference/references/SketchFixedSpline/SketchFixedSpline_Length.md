# SketchFixedSpline.Length Property

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchFixedSpline.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 9
