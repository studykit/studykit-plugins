# SketchFixedSpline3D.ConvertToSpline Method

Parent Object: [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Description

Method that converts the fixed spline to a SketchSpline3D object. This method is currently only valid for SketchFixedSpline3D objects that were created using a BSplineCurve as input.

## Syntax

SketchFixedSpline3D.**ConvertToSpline**() As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Version

Introduced in version 10
