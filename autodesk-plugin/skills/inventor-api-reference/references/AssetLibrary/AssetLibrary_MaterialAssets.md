# AssetLibrary.MaterialAssets Property

Parent Object: [AssetLibrary](../AssetLibrary/AssetLibrary.md)

## Description

Gets the Assets collection which provides access to all of the materials in this library.

## Syntax

AssetLibrary.**MaterialAssets**() As [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md).
