# PresentationComponent.SurfaceBodies Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that returns all the PresentationBody objects contained within the presentation component.

## Syntax

PresentationComponent.**SurfaceBodies**() As [PresentationBodiesEnumerator](../PresentationBodiesEnumerator/PresentationBodiesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationBodiesEnumerator](../PresentationBodiesEnumerator/PresentationBodiesEnumerator.md).

## Version

Introduced in version 2018
