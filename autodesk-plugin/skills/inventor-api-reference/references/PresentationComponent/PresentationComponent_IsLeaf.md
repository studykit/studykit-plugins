# PresentationComponent.IsLeaf Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that returns whether this component is leaf component or not.

## Syntax

PresentationComponent.**IsLeaf**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
