# PresentationComponent.MeshFeatureSets Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that returns all the PresentationMeshFeatureSet objects contained within the presentation component.

## Syntax

PresentationComponent.**MeshFeatureSets**() As [PresentationMeshFeatureSetsEnumerator](../PresentationMeshFeatureSetsEnumerator/PresentationMeshFeatureSetsEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationMeshFeatureSetsEnumerator](../PresentationMeshFeatureSetsEnumerator/PresentationMeshFeatureSetsEnumerator.md).

## Version

Introduced in version 2018
