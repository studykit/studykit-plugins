# DirectEditSizeOperation.Name Property

Parent Object: [DirectEditSizeOperation](../DirectEditSizeOperation/DirectEditSizeOperation.md)

## Description

Gets the name of this direct edit operation within the scope of this Document.

## Syntax

DirectEditSizeOperation.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2015
