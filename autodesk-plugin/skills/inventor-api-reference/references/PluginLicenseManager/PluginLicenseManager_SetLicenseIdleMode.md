# PluginLicenseManager.SetLicenseIdleMode Method

Parent Object: [PluginLicenseManager](../PluginLicenseManager/PluginLicenseManager.md)

## Description

Method that sets plugin license mode.

## Syntax

PluginLicenseManager.**SetLicenseIdleMode**( ***Value*** As [PluginLicenseIdleModeEnum](../PluginLicenseIdleModeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | [PluginLicenseIdleModeEnum](../PluginLicenseIdleModeEnum.md) | Input PluginLicenseIdleModeEnum value that specifies the idle mode to set. |

## Version

Introduced in version 2023
