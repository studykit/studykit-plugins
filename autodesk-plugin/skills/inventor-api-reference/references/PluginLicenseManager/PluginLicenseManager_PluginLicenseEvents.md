# PluginLicenseManager.PluginLicenseEvents Property

Parent Object: [PluginLicenseManager](../PluginLicenseManager/PluginLicenseManager.md)

## Description

Gets the plugin license events.

## Syntax

PluginLicenseManager.**PluginLicenseEvents**() As [PluginLicenseManagerEvents](../PluginLicenseManagerEvents/PluginLicenseManagerEvents.md)

## Property Value

This is a read only property whose value is a [PluginLicenseManagerEvents](../PluginLicenseManagerEvents/PluginLicenseManagerEvents.md).

## Version

Introduced in version 2020
