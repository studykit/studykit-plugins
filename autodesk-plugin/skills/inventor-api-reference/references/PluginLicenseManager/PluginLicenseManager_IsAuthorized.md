# PluginLicenseManager.IsAuthorized Property

Parent Object: [PluginLicenseManager](../PluginLicenseManager/PluginLicenseManager.md)

## Description

Gets whether the license is authorized.

## Syntax

PluginLicenseManager.**IsAuthorized**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2020
