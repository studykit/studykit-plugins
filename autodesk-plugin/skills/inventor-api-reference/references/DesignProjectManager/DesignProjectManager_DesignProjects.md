# DesignProjectManager.DesignProjects Property

Parent Object: [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md)

## Description

Property that returns the DesignProjects collection object containing all the projects.

## Syntax

DesignProjectManager.**DesignProjects**() As [DesignProjects](../DesignProjects/DesignProjects.md)

## Property Value

This is a read only property whose value is a [DesignProjects](../DesignProjects/DesignProjects.md).
