# DWGBlockDefinition.Entities Property

Parent Object: [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Description

Read-only property that returns the DWGEntitiesEnumerator collection object.

## Syntax

DWGBlockDefinition.**Entities**() As [DWGEntitiesEnumerator](../DWGEntitiesEnumerator/DWGEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEntitiesEnumerator](../DWGEntitiesEnumerator/DWGEntitiesEnumerator.md).

## Version

Introduced in version 2016
