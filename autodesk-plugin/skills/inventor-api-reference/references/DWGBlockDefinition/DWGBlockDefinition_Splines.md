# DWGBlockDefinition.Splines Property

Parent Object: [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Description

Read-only property that returns the DWGSplinesEnumerator collection object.

## Syntax

DWGBlockDefinition.**Splines**() As [DWGSplinesEnumerator](../DWGSplinesEnumerator/DWGSplinesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGSplinesEnumerator](../DWGSplinesEnumerator/DWGSplinesEnumerator.md).

## Version

Introduced in version 2016
