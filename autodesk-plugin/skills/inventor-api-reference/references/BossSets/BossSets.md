# BossSets Object

## Description

Collection Object containing boss definitions of the rib.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../BossSets/BossSets_Add.md) | Method that creates a new Boss set. The newly created BossSet object is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BossSets/BossSets_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BossSets/BossSets_Count.md) | Gets the number of items in this collection. |
| [Item](../BossSets/BossSets_Item.md) | Property that returns an item from the collection. You can provide either the index of the item in the collection, where the first item is index 1, or you can provide the name of the object. |
| [Type](../BossSets/BossSets_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[RibDefinition.BossSets](../RibDefinition/RibDefinition_BossSets.md)

## Version

Introduced in version 2012
