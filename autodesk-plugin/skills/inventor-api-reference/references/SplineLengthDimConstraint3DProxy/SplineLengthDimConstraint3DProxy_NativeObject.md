# SplineLengthDimConstraint3DProxy.NativeObject Property

Parent Object: [SplineLengthDimConstraint3DProxy](../SplineLengthDimConstraint3DProxy/SplineLengthDimConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SplineLengthDimConstraint3DProxy.**NativeObject**() As [SplineLengthDimConstraint3D](../SplineLengthDimConstraint3D/SplineLengthDimConstraint3D.md)

## Property Value

This is a read only property whose value is a [SplineLengthDimConstraint3D](../SplineLengthDimConstraint3D/SplineLengthDimConstraint3D.md).

## Version

Introduced in version 2019
