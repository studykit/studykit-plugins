# SplineLengthDimConstraint3DProxy.AnchorPoints Property

Parent Object: [SplineLengthDimConstraint3DProxy](../SplineLengthDimConstraint3DProxy/SplineLengthDimConstraint3DProxy.md)

## Syntax

SplineLengthDimConstraint3DProxy.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).
