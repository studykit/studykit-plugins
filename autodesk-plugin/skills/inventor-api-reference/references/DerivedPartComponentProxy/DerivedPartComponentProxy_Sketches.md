# DerivedPartComponentProxy.Sketches Property

Parent Object: [DerivedPartComponentProxy](../DerivedPartComponentProxy/DerivedPartComponentProxy.md)

## Description

Property that returns the PlanarSketch objects that were created as a result of this derived part.

## Syntax

DerivedPartComponentProxy.**Sketches**() As [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md).

## Version

Introduced in version 2008
