# DerivedPartComponentProxy.SketchBlocks Property

Parent Object: [DerivedPartComponentProxy](../DerivedPartComponentProxy/DerivedPartComponentProxy.md)

## Description

Property that returns the SketchBlock objects that were created as a result of this derived part.

## Syntax

DerivedPartComponentProxy.**SketchBlocks**() As [SketchBlocksEnumerator](../SketchBlocksEnumerator/SketchBlocksEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchBlocksEnumerator](../SketchBlocksEnumerator/SketchBlocksEnumerator.md).

## Version

Introduced in version 2010
