# DerivedPartComponentProxy.Definition Property

Parent Object: [DerivedPartComponentProxy](../DerivedPartComponentProxy/DerivedPartComponentProxy.md)

## Description

Gets the DerivedPartDefinition that defines the current state of the derived part.

## Syntax

DerivedPartComponentProxy.**Definition**() As [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Property Value

This is a read/write property whose value is a [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md).

## Version

Introduced in version 2008
