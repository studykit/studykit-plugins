# DerivedPartComponentProxy.NativeObject Property

Parent Object: [DerivedPartComponentProxy](../DerivedPartComponentProxy/DerivedPartComponentProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DerivedPartComponentProxy.**NativeObject**() As [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Property Value

This is a read only property whose value is a [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md).

## Version

Introduced in version 2008
