# DWGEllipticalArc.Center Property

Parent Object: [DWGEllipticalArc](../DWGEllipticalArc/DWGEllipticalArc.md)

## Description

Read-only property that returns the position of the center of the DWGEllipticalArc.

## Syntax

DWGEllipticalArc.**Center**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2016
