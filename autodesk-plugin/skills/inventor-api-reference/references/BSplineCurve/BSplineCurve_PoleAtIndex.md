# BSplineCurve.PoleAtIndex Property

Parent Object: [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Description

Property that returns the x, y, z position of a specified pole.

## Syntax

BSplineCurve.**PoleAtIndex**( ***Index*** As Long ) As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).
