# CircularPatternFeature.Name Property

Parent Object: [CircularPatternFeature](../CircularPatternFeature/CircularPatternFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

CircularPatternFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5
