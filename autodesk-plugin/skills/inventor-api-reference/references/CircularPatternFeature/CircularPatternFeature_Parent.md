# CircularPatternFeature.Parent Property

Parent Object: [CircularPatternFeature](../CircularPatternFeature/CircularPatternFeature.md)

## Description

Property that returns the parent of the feature.

## Syntax

CircularPatternFeature.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 5
