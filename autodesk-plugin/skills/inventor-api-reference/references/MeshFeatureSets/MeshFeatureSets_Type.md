# MeshFeatureSets.Type Property

Parent Object: [MeshFeatureSets](../MeshFeatureSets/MeshFeatureSets.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

MeshFeatureSets.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2017
