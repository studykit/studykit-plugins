# TableStylesEnumerator.Item Property

Parent Object: [TableStylesEnumerator](../TableStylesEnumerator/TableStylesEnumerator.md)

## Description

Returns the specified TableStyle object from the collection.

## Syntax

TableStylesEnumerator.**Item**( ***Index*** As Variant ) As [TableStyle](../TableStyle/TableStyle.md)

## Property Value

This is a read only property whose value is a [TableStyle](../TableStyle/TableStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the TableStyle to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the TableStyle name. If an out of range index or a name of a non\-existent TableStyle is provided, an error will occur. |

## Version

Introduced in version 2011
