# TableStylesEnumerator Object

## Description

The TableStylesEnumerator object provides access to the existing TableStyle objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TableStylesEnumerator/TableStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../TableStylesEnumerator/TableStylesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../TableStylesEnumerator/TableStylesEnumerator_Item.md) | Returns the specified TableStyle object from the collection. |
| [Type](../TableStylesEnumerator/TableStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.TableStyles](../DrawingStylesManager/DrawingStylesManager_TableStyles.md)

## Version

Introduced in version 2011
