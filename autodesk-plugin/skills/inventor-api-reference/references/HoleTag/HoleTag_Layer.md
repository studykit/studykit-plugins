# HoleTag.Layer Property

Parent Object: [HoleTag](../HoleTag/HoleTag.md)

## Description

Property that gets and sets the layer associated with this object.

## Syntax

HoleTag.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 11
