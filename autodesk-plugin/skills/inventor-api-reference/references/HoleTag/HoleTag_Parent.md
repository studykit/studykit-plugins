# HoleTag.Parent Property

Parent Object: [HoleTag](../HoleTag/HoleTag.md)

## Description

Property that returns the parent HoleTableRow object.

## Syntax

HoleTag.**Parent**() As [HoleTableRow](../HoleTableRow/HoleTableRow.md)

## Property Value

This is a read only property whose value is a [HoleTableRow](../HoleTableRow/HoleTableRow.md).

## Version

Introduced in version 10
