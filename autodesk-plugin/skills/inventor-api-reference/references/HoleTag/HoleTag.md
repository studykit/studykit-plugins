# HoleTag Object

## Description

The HoleTag object represents a hole tag associated with a hole table row.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HoleTag/HoleTag_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [DimensionStyle](../HoleTag/HoleTag_DimensionStyle.md) | Property that gets and sets the associated DimensionStyle object. |
| [FormattedText](../HoleTag/HoleTag_FormattedText.md) | Gets/Sets formatted text of the HoleTag. |
| [Layer](../HoleTag/HoleTag_Layer.md) | Property that gets and sets the layer associated with this object. |
| [Leader](../HoleTag/HoleTag_Leader.md) | Property that returns the Leader object associated with the hole tag. |
| [Parent](../HoleTag/HoleTag_Parent.md) | Property that returns the parent HoleTableRow object. |
| [Position](../HoleTag/HoleTag_Position.md) | Gets/Sets the position of the HoleTag on the sheet. |
| [RangeBox](../HoleTag/HoleTag_RangeBox.md) | Property that returns a Box2D object which contains the lower-left and upper-right corners of a rectangle that is guaranteed to enclose this object. |
| [ShowLeader](../HoleTag/HoleTag_ShowLeader.md) | Property that gets and sets whether to display the leader line. |
| [StackedTextPosition](../HoleTag/HoleTag_StackedTextPosition.md) | Gets and sets the position (alignment) of the stacked text with respect to regular text. |
| [Text](../HoleTag/HoleTag_Text.md) | Gets/Sets the text of the HoleTag. |
| [Type](../HoleTag/HoleTag_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Visible](../HoleTag/HoleTag_Visible.md) | Property that gets and sets whether the tag is visible. |

## Accessed From

[HoleTableRow.HoleTag](../HoleTableRow/HoleTableRow_HoleTag.md)

## Version

Introduced in version 10
