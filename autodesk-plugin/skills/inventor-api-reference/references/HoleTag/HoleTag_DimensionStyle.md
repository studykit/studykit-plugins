# HoleTag.DimensionStyle Property

Parent Object: [HoleTag](../HoleTag/HoleTag.md)

## Description

Property that gets and sets the associated DimensionStyle object.

## Syntax

HoleTag.**DimensionStyle**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 10
