# HoleClearanceInfo.FastenerFitType Property

Parent Object: [HoleClearanceInfo](../HoleClearanceInfo/HoleClearanceInfo.md)

## Description

Read-write property that gets and sets the fastener fit type.

## Syntax

HoleClearanceInfo.**FastenerFitType**() As [FastenerFitType](../FastenerFitType/FastenerFitType.md)

## Property Value

This is a read/write property whose value is a [FastenerFitType](../FastenerFitType/FastenerFitType.md).

## Version

Introduced in version 2018
