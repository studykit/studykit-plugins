# FaceFeatureProxy.BendFeature Property

Parent Object: [FaceFeatureProxy](../FaceFeatureProxy/FaceFeatureProxy.md)

## Description

Property that returns the associated bend feature, if one exists. If there isn't an associated bend feature this property returns Nothing.

## Syntax

FaceFeatureProxy.**BendFeature**() As [BendFeature](../BendFeature/BendFeature.md)

## Property Value

This is a read only property whose value is a [BendFeature](../BendFeature/BendFeature.md).

## Version

Introduced in version 2009
