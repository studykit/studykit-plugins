# FaceFeatureProxy.AppearanceSourceType Property

Parent Object: [FaceFeatureProxy](../FaceFeatureProxy/FaceFeatureProxy.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

FaceFeatureProxy.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
