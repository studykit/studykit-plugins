# TriadSegmentEnum Enumerator

## Description

Triad segment type enum.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllSegments | 0 | Toggle all segments. |
| kOriginSegment | 1 | Toggle the origin segment. |
| kXAxisRotationSegment | 16 | Toggle X axis Rotation segment. |
| kXAxisTranslationSegment | 2 | Toggle X axis translation segment. |
| kXYPlaneTranslationSegment | 128 | Toggle XY plane translation segment. |
| kXZPlaneTranslationSegment | 256 | Toggle XZ plane translation segment. |
| kYAxisRotationSegment | 32 | Toggle Y axis Rotation segment. |
| kYAxisTranslationSegment | 4 | Toggle Y axis translation segment. |
| kYZPlaneTranslationSegment | 512 | Toggle YZ plane translation segment. |
| kZAxisRotationSegment | 64 | Toggle Z axis Rotation segment. |
| kZAxisTranslationSegment | 8 | Toggle Z axis translation segment. |

## Version

Introduced in version 9
