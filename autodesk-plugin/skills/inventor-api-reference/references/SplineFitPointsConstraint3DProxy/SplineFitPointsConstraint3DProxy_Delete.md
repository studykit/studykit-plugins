# SplineFitPointsConstraint3DProxy.Delete Method

Parent Object: [SplineFitPointsConstraint3DProxy](../SplineFitPointsConstraint3DProxy/SplineFitPointsConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

SplineFitPointsConstraint3DProxy.**Delete**()

## Version

Introduced in version 8
