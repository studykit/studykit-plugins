# SplineFitPointsConstraint3DProxy.Points Property

Parent Object: [SplineFitPointsConstraint3DProxy](../SplineFitPointsConstraint3DProxy/SplineFitPointsConstraint3DProxy.md)

## Description

Property that returns the sketch points the spline is fit through.

## Syntax

SplineFitPointsConstraint3DProxy.**Points**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 8
