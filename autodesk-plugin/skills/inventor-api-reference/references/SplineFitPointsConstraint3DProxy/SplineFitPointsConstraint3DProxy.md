# SplineFitPointsConstraint3DProxy Object

Derived from: [SplineFitPointsConstraint3D](../SplineFitPointsConstraint3D/SplineFitPointsConstraint3D.md) Object

This is an assembly-context proxy object derived from its native definition-context object.

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../SplineFitPointsConstraint3DProxy/SplineFitPointsConstraint3DProxy_Delete.md) | Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted. |
