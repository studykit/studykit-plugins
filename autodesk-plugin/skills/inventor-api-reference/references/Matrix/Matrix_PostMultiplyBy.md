# Matrix.PostMultiplyBy Method

Parent Object: [Matrix](../Matrix/Matrix.md)

## Description

Method that post-multiplies this matrix by the specified matrix, setting this matrix to the result.

## Syntax

Matrix.**PostMultiplyBy**( ***Matrix*** As [Matrix](../Matrix/Matrix.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix | [Matrix](../Matrix/Matrix.md) | Input Matrix object that specifies the matrix to post-multiply by. |

## Version

Introduced in version 6
