# Matrix.SetCoordinateSystem Method

Parent Object: [Matrix](../Matrix/Matrix.md)

## Description

Sets this matrix to be the matrix that maps from the standard coordinate system to the specified coordinate system.

## Syntax

Matrix.**SetCoordinateSystem**( ***Origin*** As [Point](../Point/Point.md), ***XAxis*** As [Vector](../Vector/Vector.md), ***YAxis*** As [Vector](../Vector/Vector.md), ***ZAxis*** As [Vector](../Vector/Vector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Origin | [Point](../Point/Point.md) | Input Point object that defines the origin of the coordinate system. |
| XAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the X-axis vector of the coordinate system. |
| YAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the Y-axis vector of the coordinate system. |
| ZAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the Z-axis vector of the coordinate system. |

## Version

Introduced in version 4
