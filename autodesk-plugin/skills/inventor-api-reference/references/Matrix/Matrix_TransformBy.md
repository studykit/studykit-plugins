# Matrix.TransformBy Method

Parent Object: [Matrix](../Matrix/Matrix.md)

## Description

Method that sets this matrix to the result of this transformation followed by the specified transformation (a pre-multiplication of this matrix by the specified matrix).

## Syntax

Matrix.**TransformBy**( ***Matrix*** As [Matrix](../Matrix/Matrix.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix | [Matrix](../Matrix/Matrix.md) | Input Matrix object that specifies the matrix to transform by. |
