# InsertConstraintProxy.iMateResult Property

Parent Object: [InsertConstraintProxy](../InsertConstraintProxy/InsertConstraintProxy.md)

## Description

Property that returns the iMateResult object that resulted in the creation of this constraint.

## Syntax

InsertConstraintProxy.**iMateResult**() As [iMateResult](../iMateResult/iMateResult.md)

## Property Value

This is a read only property whose value is an [iMateResult](../iMateResult/iMateResult.md).

## Version

Introduced in version 11
