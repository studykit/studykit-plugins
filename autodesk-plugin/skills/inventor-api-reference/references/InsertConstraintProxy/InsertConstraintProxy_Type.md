# InsertConstraintProxy.Type Property

Parent Object: [InsertConstraintProxy](../InsertConstraintProxy/InsertConstraintProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

InsertConstraintProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
