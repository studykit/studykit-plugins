# ModelToleranceFeaturesEnumerator Object

## Description

The ModelToleranceFeaturesEnumerator collection object provides access to a set of tolerance features.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ModelToleranceFeaturesEnumerator/ModelToleranceFeaturesEnumerator_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../ModelToleranceFeaturesEnumerator/ModelToleranceFeaturesEnumerator_Count.md) | Gets the number of items in this collection. |
| [Item](../ModelToleranceFeaturesEnumerator/ModelToleranceFeaturesEnumerator_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../ModelToleranceFeaturesEnumerator/ModelToleranceFeaturesEnumerator_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[ModelToleranceFeatureDefinition.SubFeatures](ModelToleranceFeatureDefinition_SubFeatures.md)

## Version

Introduced in version 2018
