# SketchArc3D.SweepAngle Property

Parent Object: [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Description

Gets the SweepAngle of the arc.

## Syntax

SketchArc3D.**SweepAngle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
