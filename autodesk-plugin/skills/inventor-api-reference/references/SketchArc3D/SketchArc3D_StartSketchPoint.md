# SketchArc3D.StartSketchPoint Property

Parent Object: [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Description

Property that gets the that defines the position of the start of the arc.

## Syntax

SketchArc3D.**StartSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 8
