# SketchArc3D.Construction Property

Parent Object: [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchArc3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
