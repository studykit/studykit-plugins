# SketchArc3D.StartAngle Property

Parent Object: [SketchArc3D](../SketchArc3D/SketchArc3D.md)

## Description

Gets the StartAngle of the arc.

## Syntax

SketchArc3D.**StartAngle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
