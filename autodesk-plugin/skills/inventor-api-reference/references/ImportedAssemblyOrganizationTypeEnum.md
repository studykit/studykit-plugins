# ImportedAssemblyOrganizationTypeEnum Enumerator

## Description

Specifies the assembly organization type.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAImportedAsMultibodyPart | 108546 | Indicates assembly structure is converted to multibody parts. |
| kImportedAsAssembly | 108545 | Indicates the assembly structure is retained. |
| kImportedAsCompositeFeaturePart | 108547 | Indicates assembly structure is converted to composite features. |
| kImportedAsMultibodyPart | 108546 | Indicates assembly structure is converted to multibody parts. |

## Version

Introduced in version 2016
