# TwoLineAngleDimConstraintProxy.LineOne Property

Parent Object: [TwoLineAngleDimConstraintProxy](../TwoLineAngleDimConstraintProxy/TwoLineAngleDimConstraintProxy.md)

## Description

Property that returns the first sketch line being constrained.

## Syntax

TwoLineAngleDimConstraintProxy.**LineOne**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 6
