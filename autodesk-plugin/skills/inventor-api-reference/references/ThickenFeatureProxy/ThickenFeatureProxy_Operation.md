# ThickenFeatureProxy.Operation Property

Parent Object: [ThickenFeatureProxy](../ThickenFeatureProxy/ThickenFeatureProxy.md)

## Description

Gets and sets the type of operation used to add the feature to the model.

## Syntax

ThickenFeatureProxy.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 9
