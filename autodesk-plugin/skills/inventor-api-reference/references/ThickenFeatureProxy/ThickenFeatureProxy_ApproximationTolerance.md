# ThickenFeatureProxy.ApproximationTolerance Property

Parent Object: [ThickenFeatureProxy](../ThickenFeatureProxy/ThickenFeatureProxy.md)

## Description

Gets and set the approximation tolerance.

## Syntax

ThickenFeatureProxy.**ApproximationTolerance**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 11
