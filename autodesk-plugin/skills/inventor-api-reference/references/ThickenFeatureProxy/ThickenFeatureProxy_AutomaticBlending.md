# ThickenFeatureProxy.AutomaticBlending Property

Parent Object: [ThickenFeatureProxy](../ThickenFeatureProxy/ThickenFeatureProxy.md)

## Description

Property that returns whether the offset operation should propagate to adjacent tangent faces.

## Syntax

ThickenFeatureProxy.**AutomaticBlending**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2011
