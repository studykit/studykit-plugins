# ThickenFeatureProxy.ApproximationOptimized Property

Parent Object: [ThickenFeatureProxy](../ThickenFeatureProxy/ThickenFeatureProxy.md)

## Description

Gets and sets whether the approximation is optimized.

## Syntax

ThickenFeatureProxy.**ApproximationOptimized**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
