# ThickenFeatureProxy.Appearance Property

Parent Object: [ThickenFeatureProxy](../ThickenFeatureProxy/ThickenFeatureProxy.md)

## Description

Gets and sets the current appearance of the feature.

## Syntax

ThickenFeatureProxy.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
