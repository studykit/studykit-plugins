# MaterialAsset.IsUsed Property

Parent Object: [MaterialAsset](../MaterialAsset/MaterialAsset.md)

## Description

Gets the boolean flag that indicates if this asset is being used in the document or in a material definition.

## Syntax

MaterialAsset.**IsUsed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
