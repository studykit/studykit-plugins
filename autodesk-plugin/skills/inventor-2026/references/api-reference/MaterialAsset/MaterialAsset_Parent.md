# MaterialAsset.Parent Property

Parent Object: [MaterialAsset](../MaterialAsset/MaterialAsset.md)

## Description

Read-only property that returns the parent object of this asset. This can return an AssetLibrary, or Document object.

## Syntax

MaterialAsset.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2014
