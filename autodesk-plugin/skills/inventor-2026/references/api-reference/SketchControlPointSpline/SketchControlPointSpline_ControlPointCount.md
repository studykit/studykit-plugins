# SketchControlPointSpline.ControlPointCount Property

Parent Object: [SketchControlPointSpline](../SketchControlPointSpline/SketchControlPointSpline.md)

## Description

Read-only property that returns the number of control points defining the control point spline.

## Syntax

SketchControlPointSpline.**ControlPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2014
