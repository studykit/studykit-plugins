# GroundConstraint.Type Property

Parent Object: [GroundConstraint](../GroundConstraint/GroundConstraint.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

GroundConstraint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
