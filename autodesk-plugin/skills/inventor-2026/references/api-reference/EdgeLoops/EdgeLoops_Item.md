# EdgeLoops.Item Property

Parent Object: [EdgeLoops](../EdgeLoops/EdgeLoops.md)

## Description

Allows integer-indexed access to objects in the collection.

## Syntax

EdgeLoops.**Item**( ***Index*** As Long ) As [EdgeLoop](../EdgeLoop/EdgeLoop.md)

## Property Value

This is a read only property whose value is an [EdgeLoop](../EdgeLoop/EdgeLoop.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 4
