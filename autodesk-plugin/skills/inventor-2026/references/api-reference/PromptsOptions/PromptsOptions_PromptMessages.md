# PromptsOptions.PromptMessages Property

Parent Object: [PromptsOptions](../PromptsOptions/PromptsOptions.md)

## Description

Read-only property that returns the collection of prompt messages.

## Syntax

PromptsOptions.**PromptMessages**() As [PromptMessages](../PromptMessages/PromptMessages.md)

## Property Value

This is a read only property whose value is a [PromptMessages](../PromptMessages/PromptMessages.md).
