# PromptsOptions.DoNotShowMessagesForPreexistingProblems Property

Parent Object: [PromptsOptions](../PromptsOptions/PromptsOptions.md)

## Description

Read-write Boolean property that gets and sets whether or not to show messages for pre-existing problems.

## Syntax

PromptsOptions.**DoNotShowMessagesForPreexistingProblems**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
