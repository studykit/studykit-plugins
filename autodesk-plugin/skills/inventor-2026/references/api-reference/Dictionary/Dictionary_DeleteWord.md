# Dictionary.DeleteWord Method

Parent Object: [Dictionary](../Dictionary/Dictionary.md)

## Description

Method that deletes a word from this dictionary.

## Syntax

Dictionary.**DeleteWord**( ***Word*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Word | String | Input String indicating a word. |

## Version

Introduced in version 2020
