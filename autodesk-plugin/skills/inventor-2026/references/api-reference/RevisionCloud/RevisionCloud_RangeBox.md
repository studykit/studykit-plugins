# RevisionCloud.RangeBox Property

Parent Object: [RevisionCloud](../RevisionCloud/RevisionCloud.md)

## Description

Returns a Box2d object which contains the lower-left and upper-right corners of a rectangle that is guaranteed to enclose this object.

## Syntax

RevisionCloud.**RangeBox**() As [Box2d](../Box2d/Box2d.md)

## Property Value

This is a read only property whose value is a [Box2d](../Box2d/Box2d.md).

## Version

Introduced in version 2024
