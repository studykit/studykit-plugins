# SweepDefinition.Path Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the entity that defines the path of the sweep.

## Syntax

SweepDefinition.**Path**() As [Path](../Path/Path.md)

## Property Value

This is a read/write property whose value is a [Path](../Path/Path.md).

## Version

Introduced in version 2015
