# DimensionText.BackgroundColor Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Gets and sets the BackgroundColor associated with the DimensionText.

## Syntax

DimensionText.**BackgroundColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2025
