# DimensionText.Color Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Gets and sets the Color Object of the text.

## Syntax

DimensionText.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
