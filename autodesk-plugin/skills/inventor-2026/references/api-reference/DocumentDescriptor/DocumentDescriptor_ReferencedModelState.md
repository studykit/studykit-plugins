# DocumentDescriptor.ReferencedModelState Property

Parent Object: [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Description

Read-only property that returns the model state type of the referenced document.

## Syntax

DocumentDescriptor.**ReferencedModelState**() As [ModelStateTypeEnum](../ModelStateTypeEnum.md)

## Property Value

This is a read only property whose value is a [ModelStateTypeEnum](../ModelStateTypeEnum.md).

## Version

Introduced in version 2022
