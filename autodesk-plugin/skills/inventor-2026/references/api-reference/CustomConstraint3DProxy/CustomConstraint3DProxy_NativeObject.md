# CustomConstraint3DProxy.NativeObject Property

Parent Object: [CustomConstraint3DProxy](../CustomConstraint3DProxy/CustomConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

CustomConstraint3DProxy.**NativeObject**() As [CustomConstraint3D](../CustomConstraint3D/CustomConstraint3D.md)

## Property Value

This is a read only property whose value is a [CustomConstraint3D](../CustomConstraint3D/CustomConstraint3D.md).

## Version

Introduced in version 11
