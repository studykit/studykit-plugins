# SketchSplines.Item Property

Parent Object: [SketchSplines](../SketchSplines/SketchSplines.md)

## Description

Returns the specified SketchSpline object from the collection.

## Syntax

SketchSplines.**Item**( ***Index*** As Long ) As [SketchSpline](../SketchSpline/SketchSpline.md)

## Property Value

This is a read only property whose value is a [SketchSpline](../SketchSpline/SketchSpline.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 5
