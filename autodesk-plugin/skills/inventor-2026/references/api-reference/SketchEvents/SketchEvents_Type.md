# SketchEvents.Type Property

Parent Object: [SketchEvents](../SketchEvents/SketchEvents.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchEvents.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
