# SketchEntities3DEnumerator Object

## Description

The SketchEntities3DEnumerator object provides access to a set of 3D sketch entities. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../SketchEntities3DEnumerator/SketchEntities3DEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchEntities3DEnumerator/SketchEntities3DEnumerator_Item.md) | Returns the specified SketchEntity3D object from the collection. |
| [Type](../SketchEntities3DEnumerator/SketchEntities3DEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
