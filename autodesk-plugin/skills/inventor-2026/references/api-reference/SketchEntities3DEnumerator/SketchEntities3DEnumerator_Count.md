# SketchEntities3DEnumerator.Count Property

Parent Object: [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Description

Property that returns the number of items in the collection.

## Syntax

SketchEntities3DEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 6
