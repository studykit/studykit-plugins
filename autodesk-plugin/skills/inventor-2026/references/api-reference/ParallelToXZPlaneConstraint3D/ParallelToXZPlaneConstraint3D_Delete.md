# ParallelToXZPlaneConstraint3D.Delete Method

Parent Object: [ParallelToXZPlaneConstraint3D](../ParallelToXZPlaneConstraint3D/ParallelToXZPlaneConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ParallelToXZPlaneConstraint3D.**Delete**()

## Version

Introduced in version 2017
