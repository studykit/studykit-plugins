# ParallelToXZPlaneConstraint3D.Entity Property

Parent Object: [ParallelToXZPlaneConstraint3D](../ParallelToXZPlaneConstraint3D/ParallelToXZPlaneConstraint3D.md)

## Description

Read-only property that returns the 3D sketch entity being constrained.

## Syntax

ParallelToXZPlaneConstraint3D.**Entity**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 2017
