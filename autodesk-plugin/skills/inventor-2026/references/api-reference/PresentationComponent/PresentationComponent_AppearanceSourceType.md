# PresentationComponent.AppearanceSourceType Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that gets and sets the source of the appearance for this presentation component. This can be set to kMaterialAppearance to clear the override.

## Syntax

PresentationComponent.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read only property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2018
