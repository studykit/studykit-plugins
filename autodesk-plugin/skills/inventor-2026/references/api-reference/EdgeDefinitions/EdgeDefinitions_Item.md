# EdgeDefinitions.Item Property

Parent Object: [EdgeDefinitions](../EdgeDefinitions/EdgeDefinitions.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

EdgeDefinitions.**Item**( ***Index*** As Long ) As [EdgeDefinition](../EdgeDefinition/EdgeDefinition.md)

## Property Value

This is a read only property whose value is an [EdgeDefinition](../EdgeDefinition/EdgeDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the EdgeDefinition object within the collection to return. |

## Version

Introduced in version 2011
