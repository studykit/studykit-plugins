# PlanarSketch.Edit Method

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Method that causes the Sketch environment to be invoked with this sketch available for interactive edit.

## Syntax

PlanarSketch.**Edit**()
