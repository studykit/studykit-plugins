# PlanarSketch.Profiles Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the Profiles collection object.

## Syntax

PlanarSketch.**Profiles**() As [Profiles](../Profiles/Profiles.md)

## Property Value

This is a read only property whose value is a [Profiles](../Profiles/Profiles.md).
