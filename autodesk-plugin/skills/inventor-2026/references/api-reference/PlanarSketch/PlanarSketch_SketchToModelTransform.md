# PlanarSketch.SketchToModelTransform Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the transformation from the 2D sketch coordinate space to model space.

## Syntax

PlanarSketch.**SketchToModelTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 6
