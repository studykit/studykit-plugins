# GraphicsNodeProxy.AddScalableTextGraphics Method

Parent Object: [GraphicsNodeProxy](../GraphicsNodeProxy/GraphicsNodeProxy.md)

## Description

Method that creates a new (scalable) TextGraphics graphic primitive.

## Syntax

GraphicsNodeProxy.**AddScalableTextGraphics**() As [TextGraphics](../TextGraphics/TextGraphics.md)

## Version

Introduced in version 2008
