# GraphicsNodeProxy.Appearance Property

Parent Object: [GraphicsNodeProxy](../GraphicsNodeProxy/GraphicsNodeProxy.md)

## Description

Gets and sets the appearance asset associated with the graphics node.

## Syntax

GraphicsNodeProxy.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
