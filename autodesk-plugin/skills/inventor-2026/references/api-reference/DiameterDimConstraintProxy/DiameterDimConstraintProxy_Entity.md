# DiameterDimConstraintProxy.Entity Property

Parent Object: [DiameterDimConstraintProxy](../DiameterDimConstraintProxy/DiameterDimConstraintProxy.md)

## Description

Property that returns the circle or arc being constrained.

## Syntax

DiameterDimConstraintProxy.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
