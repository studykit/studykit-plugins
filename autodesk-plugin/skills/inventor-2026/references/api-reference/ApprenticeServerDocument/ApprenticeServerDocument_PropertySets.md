# ApprenticeServerDocument.PropertySets Property

Parent Object: [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md)

## Description

Gets the object that manages the File Properties.

## Syntax

ApprenticeServerDocument.**PropertySets**() As [PropertySets](../PropertySets/PropertySets.md)

## Property Value

This is a read only property whose value is a [PropertySets](../PropertySets/PropertySets.md).

## Version

Introduced in version 4
