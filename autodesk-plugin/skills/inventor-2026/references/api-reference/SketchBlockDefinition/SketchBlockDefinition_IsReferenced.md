# SketchBlockDefinition.IsReferenced Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns whether this sketch block definition is currently referenced within the document.

## Syntax

SketchBlockDefinition.**IsReferenced**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
