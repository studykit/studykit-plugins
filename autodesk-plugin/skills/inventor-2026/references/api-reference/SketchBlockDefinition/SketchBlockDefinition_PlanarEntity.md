# SketchBlockDefinition.PlanarEntity Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and sets the planar object that defines the planar object the sketch is to be built on.

## Syntax

SketchBlockDefinition.**PlanarEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2010
