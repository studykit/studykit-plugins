# SketchBlockDefinition.DimensionConstraints Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets the collection of all dimension constraints on the sketch.

## Syntax

SketchBlockDefinition.**DimensionConstraints**() As [DimensionConstraints](../DimensionConstraints/DimensionConstraints.md)

## Property Value

This is a read only property whose value is a [DimensionConstraints](../DimensionConstraints/DimensionConstraints.md).

## Version

Introduced in version 2010
