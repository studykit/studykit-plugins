# SketchedSymbolDefinitionLibraries.Add Method

Parent Object: [SketchedSymbolDefinitionLibraries](../SketchedSymbolDefinitionLibraries/SketchedSymbolDefinitionLibraries.md)

## Description

Creates a new sketch symbol library.

## Syntax

SketchedSymbolDefinitionLibraries.**Add**( ***Name*** As String ) As [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Name | String | Input String that specifies the sketched symbol library name. The name should be unique in the libraries collection |
