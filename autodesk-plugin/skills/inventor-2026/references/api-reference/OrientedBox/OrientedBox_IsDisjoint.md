# OrientedBox.IsDisjoint Method

Parent Object: [OrientedBox](../OrientedBox/OrientedBox.md)

## Description

Determines whether this box does not intersect with the specified oriented box.

## Syntax

OrientedBox.**IsDisjoint**( ***OrientedBox*** As [OrientedBox](../OrientedBox/OrientedBox.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| OrientedBox | [OrientedBox](../OrientedBox/OrientedBox.md) | Input OrientedBox object to compare. |

## Version

Introduced in version 2016
