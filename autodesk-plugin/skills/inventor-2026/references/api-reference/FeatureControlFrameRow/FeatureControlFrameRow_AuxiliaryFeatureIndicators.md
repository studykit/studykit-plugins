# FeatureControlFrameRow.AuxiliaryFeatureIndicators Property

Parent Object: [FeatureControlFrameRow](../FeatureControlFrameRow/FeatureControlFrameRow.md)

## Description

Gets the AuxiliaryFeatureIndicators collection object.

## Syntax

FeatureControlFrameRow.**AuxiliaryFeatureIndicators**() As [AuxiliaryFeatureIndicators](../AuxiliaryFeatureIndicators/AuxiliaryFeatureIndicators.md)

## Property Value

This is a read only property whose value is an [AuxiliaryFeatureIndicators](../AuxiliaryFeatureIndicators/AuxiliaryFeatureIndicators.md).

## Version

Introduced in version 2023
