# Curve2dEvaluator.Continuity Property

Parent Object: [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md)

## Description

Gets the level of continuity of the curve. The continuity is the largest order of continuity maintained for the entire curve.

## Syntax

Curve2dEvaluator.**Continuity**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
