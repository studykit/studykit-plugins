# PartsListColumns.Item Property

Parent Object: [PartsListColumns](../PartsListColumns/PartsListColumns.md)

## Description

Returns the specified PartsListColumn object from the collection. This is the default property of the PartsListColumns collection object.

## Syntax

PartsListColumns.**Item**( ***Index*** As Variant ) As [PartsListColumn](../PartsListColumn/PartsListColumn.md)

## Property Value

This is a read only property whose value is a [PartsListColumn](../PartsListColumn/PartsListColumn.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the PartsListColumn to return. |

## Version

Introduced in version 5.3
