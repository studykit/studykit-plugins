# UserCoordinateSystemProxy.NativeObject Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

UserCoordinateSystemProxy.**NativeObject**() As [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Property Value

This is a read only property whose value is a [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md).

## Version

Introduced in version 2010
