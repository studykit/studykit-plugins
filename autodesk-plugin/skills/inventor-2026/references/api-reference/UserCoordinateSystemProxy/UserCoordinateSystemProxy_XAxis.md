# UserCoordinateSystemProxy.XAxis Property

Parent Object: [UserCoordinateSystemProxy](../UserCoordinateSystemProxy/UserCoordinateSystemProxy.md)

## Description

Property that returns the work axis that represents the x-axis of the coordinate system.

## Syntax

UserCoordinateSystemProxy.**XAxis**() As [WorkAxis](../WorkAxis/WorkAxis.md)

## Property Value

This is a read only property whose value is a [WorkAxis](../WorkAxis/WorkAxis.md).

## Version

Introduced in version 2010
