# ReferenceFeaturesEnumerator.Count Property

Parent Object: [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md)

## Description

Property that returns the number of items in this collection.

## Syntax

ReferenceFeaturesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5.3
