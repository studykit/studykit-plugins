# SketchOffsetSplineProxy.ConvertToSpline Method

Parent Object: [SketchOffsetSplineProxy](../SketchOffsetSplineProxy/SketchOffsetSplineProxy.md)

## Description

Method that converts the offset spline to a SketchSpline object and breaks the association to the spline from which it is offset.

## Syntax

SketchOffsetSplineProxy.**ConvertToSpline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Version

Introduced in version 9
