# SketchOffsetSplineProxy.LineScale Property

Parent Object: [SketchOffsetSplineProxy](../SketchOffsetSplineProxy/SketchOffsetSplineProxy.md)

## Description

Gets and sets the LineScale applied to this sketch spline.

## Syntax

SketchOffsetSplineProxy.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
