# FaceShellDefinitions.Item Property

Parent Object: [FaceShellDefinitions](../FaceShellDefinitions/FaceShellDefinitions.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

FaceShellDefinitions.**Item**( ***Index*** As Long ) As [FaceShellDefinition](../FaceShellDefinition/FaceShellDefinition.md)

## Property Value

This is a read only property whose value is a [FaceShellDefinition](../FaceShellDefinition/FaceShellDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the FaceShellDefinition object within the collection to return. |

## Version

Introduced in version 2011
