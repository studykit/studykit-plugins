# NativeBrowserNodeDefinition.OverrideExpandedIcon Property

Parent Object: [NativeBrowserNodeDefinition](../NativeBrowserNodeDefinition/NativeBrowserNodeDefinition.md)

## Description

Gets and sets expanded icon override on a existing definition object.

## Syntax

NativeBrowserNodeDefinition.**OverrideExpandedIcon**() As [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Property Value

This is a read/write property whose value is a [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md).

## Version

Introduced in version 9
