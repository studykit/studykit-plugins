# RadialMarkingMenu.NortheastControl Property

Parent Object: [RadialMarkingMenu](../RadialMarkingMenu/RadialMarkingMenu.md)

## Description

Gets and sets the command to display in the north-east control.

## Syntax

RadialMarkingMenu.**NortheastControl**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2012
