# Profiles3D.Item Property

Parent Object: [Profiles3D](../Profiles3D/Profiles3D.md)

## Description

Method that returns the specified Profile3D object from the collection.

## Syntax

Profiles3D.**Item**( ***Index*** As Long ) As [Profile3D](../Profile3D/Profile3D.md)

## Property Value

This is a read only property whose value is a [Profile3D](../Profile3D/Profile3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index of the Profile3D to return. |

## Version

Introduced in version 6
