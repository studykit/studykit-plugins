# SurfaceTextureStyle.TextStyle Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets the text style used to format the text in a surface texture symbol.

## Syntax

SurfaceTextureStyle.**TextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2009
