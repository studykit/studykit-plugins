# SurfaceTextureStyle.AvailableProfileDirectionSymbols Property

Parent Object: [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Description

Gets and sets a bit mask defining the profile direction symbols to be made available for surface texture symbol creation.

## Syntax

SurfaceTextureStyle.**AvailableProfileDirectionSymbols**() As [ProfileDirectionTypeEnum](../ProfileDirectionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ProfileDirectionTypeEnum](../ProfileDirectionTypeEnum.md).

## Version

Introduced in version 2025
