# ApprenticeServerComponent.HardwareOptions Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the Hardware Options.

## Syntax

ApprenticeServerComponent.**HardwareOptions**() As [HardwareOptions](../HardwareOptions/HardwareOptions.md)

## Property Value

This is a read only property whose value is a [HardwareOptions](../HardwareOptions/HardwareOptions.md).

## Version

Introduced in version 11
