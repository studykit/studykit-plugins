# PartsListFilterItem.Parent Property

Parent Object: [PartsListFilterItem](../PartsListFilterItem/PartsListFilterItem.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PartsListFilterItem.**Parent**() As [PartsListFilterSettings](../PartsListFilterSettings/PartsListFilterSettings.md)

## Property Value

This is a read only property whose value is a [PartsListFilterSettings](../PartsListFilterSettings/PartsListFilterSettings.md).

## Version

Introduced in version 2024.1
