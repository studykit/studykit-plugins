# DisplaySettings.NewWindowShowAmbientShadows Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets whether to display ambient shadows (occlusions) for new windows.

## Syntax

DisplaySettings.**NewWindowShowAmbientShadows**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
