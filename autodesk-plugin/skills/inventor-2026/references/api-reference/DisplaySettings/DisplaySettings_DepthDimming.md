# DisplaySettings.DepthDimming Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Enables or disables the depth dimming effect that displays entities that are nearer with a brighter light to better convey the depth of the model.

## Syntax

DisplaySettings.**DepthDimming**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
