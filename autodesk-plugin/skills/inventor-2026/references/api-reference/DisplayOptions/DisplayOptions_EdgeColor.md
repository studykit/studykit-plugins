# DisplayOptions.EdgeColor Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that gets and sets the color for the edge display (including silhouettes) for active components.

## Syntax

DisplayOptions.**EdgeColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2011
