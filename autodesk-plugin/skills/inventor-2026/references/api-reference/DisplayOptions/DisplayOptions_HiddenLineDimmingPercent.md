# DisplayOptions.HiddenLineDimmingPercent Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets the dimming factor (expressed as a percentage) applied to the display of hidden lines in the model.

## Syntax

DisplayOptions.**HiddenLineDimmingPercent**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 11
