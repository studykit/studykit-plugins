# DisplayOptions.NewWindowShowGroundReflections Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that gets and sets whether to display ground plane reflections for new windows.

## Syntax

DisplayOptions.**NewWindowShowGroundReflections**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
