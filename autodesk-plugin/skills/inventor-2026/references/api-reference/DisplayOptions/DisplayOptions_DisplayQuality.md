# DisplayOptions.DisplayQuality Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Specifies the resolution quality to apply to the display of the model.

## Syntax

DisplayOptions.**DisplayQuality**() As [DisplayQualityEnum](../DisplayQualityEnum.md)

## Property Value

This is a read/write property whose value is a [DisplayQualityEnum](../DisplayQualityEnum.md).

## Version

Introduced in version 11
