# DisplayOptions.NewWindowDisplayMode Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets the default display mode for new windows.

## Syntax

DisplayOptions.**NewWindowDisplayMode**() As [DisplayModeEnum](../DisplayModeEnum.md)

## Property Value

This is a read/write property whose value is a [DisplayModeEnum](../DisplayModeEnum.md).

## Version

Introduced in version 2010
