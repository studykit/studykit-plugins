# HoleThreadNotes.Item Property

Parent Object: [HoleThreadNotes](../HoleThreadNotes/HoleThreadNotes.md)

## Description

Returns the specified HoleThreadNote object from the collection.

## Syntax

HoleThreadNotes.**Item**( ***Index*** As Long ) As [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Property Value

This is a read only property whose value is a [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
