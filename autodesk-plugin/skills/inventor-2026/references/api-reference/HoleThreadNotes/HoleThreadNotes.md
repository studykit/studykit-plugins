# HoleThreadNotes Object

## Description

The HoleThreadNotes collection object provides access to all of the hole and thread notes on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../HoleThreadNotes/HoleThreadNotes_Add.md) | Method that creates a hole or thread note on the sheet. Different results are possible depending on the input provided, as discussed below. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HoleThreadNotes/HoleThreadNotes_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../HoleThreadNotes/HoleThreadNotes_Count.md) | Property that returns the number of items in the collection. |
| [Item](../HoleThreadNotes/HoleThreadNotes_Item.md) | Returns the specified HoleThreadNote object from the collection. |
| [Type](../HoleThreadNotes/HoleThreadNotes_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
