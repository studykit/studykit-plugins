# AssemblySymmetryConstraint.EntityOneInferredType Property

Parent Object: [AssemblySymmetryConstraint](../AssemblySymmetryConstraint/AssemblySymmetryConstraint.md)

## Description

Read-only property that returns an enum indicating how the geometry of entity one is interpreted.

## Syntax

AssemblySymmetryConstraint.**EntityOneInferredType**() As [InferredTypeEnum](../InferredTypeEnum.md)

## Property Value

This is a read only property whose value is an [InferredTypeEnum](../InferredTypeEnum.md).

## Version

Introduced in version 2014
