# FixedWorkAxisDef Object

## Description

Object that allows you to get and set the information that specifies a fixed work axis.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../FixedWorkAxisDef/FixedWorkAxisDef_GetData.md) | Method that gets all of the data defining a fixed work axis. |
| [PutData](../FixedWorkAxisDef/FixedWorkAxisDef_PutData.md) | Method that sets the data defining a fixed work axis. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FixedWorkAxisDef/FixedWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Axis](../FixedWorkAxisDef/FixedWorkAxisDef_Axis.md) | Property that returns the X-axis vector of the work plane. |
| [OriginPoint](../FixedWorkAxisDef/FixedWorkAxisDef_OriginPoint.md) | Property that indicates the origin point of the work axis. |
| [Parent](../FixedWorkAxisDef/FixedWorkAxisDef_Parent.md) | Property returning the parent WorkAxis object. |
| [Type](../FixedWorkAxisDef/FixedWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
