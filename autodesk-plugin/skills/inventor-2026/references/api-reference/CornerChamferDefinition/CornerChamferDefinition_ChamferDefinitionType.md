# CornerChamferDefinition.ChamferDefinitionType Property

Parent Object: [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md)

## Description

Property that returns an enum indicating how the chamfer is defined.

## Syntax

CornerChamferDefinition.**ChamferDefinitionType**() As [ChamferDefinitionTypeEnum](../ChamferDefinitionTypeEnum.md)

## Property Value

This is a read only property whose value is a [ChamferDefinitionTypeEnum](../ChamferDefinitionTypeEnum.md).

## Version

Introduced in version 2009
