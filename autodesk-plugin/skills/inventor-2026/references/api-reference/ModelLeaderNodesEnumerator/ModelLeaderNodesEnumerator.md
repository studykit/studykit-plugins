# ModelLeaderNodesEnumerator Object

## Description

The ModelLeaderNodesEnumerator object provides an enumeration of ModelLeaderNode objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ModelLeaderNodesEnumerator/ModelLeaderNodesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ModelLeaderNodesEnumerator/ModelLeaderNodesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ModelLeaderNodesEnumerator/ModelLeaderNodesEnumerator_Item.md) | Property that returns an item from the collection. You can provide either the index of the item in the collection, where the first item is index 1, or you can provide the name of the object. |
| [Type](../ModelLeaderNodesEnumerator/ModelLeaderNodesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
