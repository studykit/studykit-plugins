# DrawingDimension.Tolerance Property

Parent Object: [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Description

Property that returns the Tolerance object associated with this dimension.

## Syntax

DrawingDimension.**Tolerance**() As [Tolerance](../Tolerance/Tolerance.md)

## Property Value

This is a read only property whose value is a [Tolerance](../Tolerance/Tolerance.md).

## Version

Introduced in version 9
