# DWGLine.EndPoint Property

Parent Object: [DWGLine](../DWGLine/DWGLine.md)

## Description

Read-only property that returns the point that defines the position of the end of the line.

## Syntax

DWGLine.**EndPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2016
