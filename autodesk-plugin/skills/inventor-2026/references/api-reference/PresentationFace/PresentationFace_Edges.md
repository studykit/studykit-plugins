# PresentationFace.Edges Property

Parent Object: [PresentationFace](../PresentationFace/PresentationFace.md)

## Description

Read-only property that returns all the PresentationEdge objects contained within the presentation face.

## Syntax

PresentationFace.**Edges**() As [PresentationEdgesEnumerator](../PresentationEdgesEnumerator/PresentationEdgesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationEdgesEnumerator](../PresentationEdgesEnumerator/PresentationEdgesEnumerator.md).

## Version

Introduced in version 2018
