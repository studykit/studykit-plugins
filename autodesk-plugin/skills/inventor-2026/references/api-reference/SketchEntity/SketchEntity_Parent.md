# SketchEntity.Parent Property

Parent Object: [SketchEntity](../SketchEntity/SketchEntity.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchEntity.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 5
