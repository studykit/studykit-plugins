# UnitsFormatting.DisplayFormat Property

Parent Object: [UnitsFormatting](../UnitsFormatting/UnitsFormatting.md)

## Description

Gets and sets the display format.

## Syntax

UnitsFormatting.**DisplayFormat**() As [DisplayFormatEnum](../DisplayFormatEnum.md)

## Property Value

This is a read/write property whose value is a [DisplayFormatEnum](../DisplayFormatEnum.md).

## Version

Introduced in version 2021
