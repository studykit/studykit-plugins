# UnitsFormatting.DecimalMarkerType Property

Parent Object: [UnitsFormatting](../UnitsFormatting/UnitsFormatting.md)

## Description

Gets and sets the decimal marker type.

## Syntax

UnitsFormatting.**DecimalMarkerType**() As [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md).

## Version

Introduced in version 2021
