# ComboBoxDefinition.Classification Property

Parent Object: [ComboBoxDefinition](../ComboBoxDefinition/ComboBoxDefinition.md)

## Description

Gets the classification.

## Syntax

ComboBoxDefinition.**Classification**() As [CommandTypesEnum](../CommandTypesEnum.md)

## Property Value

This is a read only property whose value is a [CommandTypesEnum](../CommandTypesEnum.md).

## Version

Introduced in version 9
