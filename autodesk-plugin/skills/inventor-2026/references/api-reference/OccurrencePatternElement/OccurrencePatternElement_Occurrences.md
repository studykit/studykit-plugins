# OccurrencePatternElement.Occurrences Property

Parent Object: [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Description

Property that returns the set of occurrences that were created for this particular instance.

## Syntax

OccurrencePatternElement.**Occurrences**() As [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md).

## Version

Introduced in version 6
