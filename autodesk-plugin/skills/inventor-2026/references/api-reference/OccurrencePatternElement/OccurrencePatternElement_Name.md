# OccurrencePatternElement.Name Property

Parent Object: [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Description

Property that returns the name of the element.

## Syntax

OccurrencePatternElement.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 6
