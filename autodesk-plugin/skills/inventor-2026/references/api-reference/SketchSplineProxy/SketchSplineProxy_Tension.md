# SketchSplineProxy.Tension Property

Parent Object: [SketchSplineProxy](../SketchSplineProxy/SketchSplineProxy.md)

## Description

Gets and sets the tension for the spline.

## Syntax

SketchSplineProxy.**Tension**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 9
