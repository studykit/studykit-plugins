# DrawingView.BreakOutOperations Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Property that returns the BreakOutOperations object containing information about all the break out operations applied to this view as well as methods to add break outs. Only local break out operations are returned.

## Syntax

DrawingView.**BreakOutOperations**() As [BreakOutOperations](../BreakOutOperations/BreakOutOperations.md)

## Property Value

This is a read only property whose value is a [BreakOutOperations](../BreakOutOperations/BreakOutOperations.md).

## Version

Introduced in version 2010
