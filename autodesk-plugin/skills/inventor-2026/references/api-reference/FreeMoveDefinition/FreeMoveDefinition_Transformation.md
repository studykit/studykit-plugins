# FreeMoveDefinition.Transformation Property

Parent Object: [FreeMoveDefinition](../FreeMoveDefinition/FreeMoveDefinition.md)

## Description

Gets and sets the Matrix defining the free move transform of the faces.

## Syntax

FreeMoveDefinition.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2011
