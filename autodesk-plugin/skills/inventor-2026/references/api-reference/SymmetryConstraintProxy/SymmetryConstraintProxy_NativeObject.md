# SymmetryConstraintProxy.NativeObject Property

Parent Object: [SymmetryConstraintProxy](../SymmetryConstraintProxy/SymmetryConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SymmetryConstraintProxy.**NativeObject**() As [SymmetryConstraint](../SymmetryConstraint/SymmetryConstraint.md)

## Property Value

This is a read only property whose value is a [SymmetryConstraint](../SymmetryConstraint/SymmetryConstraint.md).

## Version

Introduced in version 6
