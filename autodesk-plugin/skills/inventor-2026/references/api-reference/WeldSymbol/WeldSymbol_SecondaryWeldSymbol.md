# WeldSymbol.SecondaryWeldSymbol Property

Parent Object: [WeldSymbol](../WeldSymbol/WeldSymbol.md)

## Description

Returns the WeldSymbol object indicating the secondary fillet weld.

## Syntax

WeldSymbol.**SecondaryWeldSymbol**() As [WeldSymbol](../WeldSymbol/WeldSymbol.md)

## Property Value

This is a read only property whose value is a [WeldSymbol](../WeldSymbol/WeldSymbol.md).

## Version

Introduced in version 2024
