# UnitVector.DotProduct Method

Parent Object: [UnitVector](../UnitVector/UnitVector.md)

## Description

Determine the dot product of this vector to the specified vector.

## Syntax

UnitVector.**DotProduct**( ***Argument*** As [UnitVector](../UnitVector/UnitVector.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [UnitVector](../UnitVector/UnitVector.md) | Input  used for comparison. |

## Version

Introduced in version 4
