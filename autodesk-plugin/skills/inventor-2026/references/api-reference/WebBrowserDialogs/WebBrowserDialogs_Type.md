# WebBrowserDialogs.Type Property

Parent Object: [WebBrowserDialogs](../WebBrowserDialogs/WebBrowserDialogs.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

WebBrowserDialogs.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2017
