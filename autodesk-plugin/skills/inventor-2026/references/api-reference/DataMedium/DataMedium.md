# DataMedium Object

## Description

The DataMedium object is used when exporting/importing to various formats using the TranslatorAddIn APIs. See the 'export' samples under TranslatorAddin for examples of its use.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [FileName](../DataMedium/DataMedium_FileName.md) | Specifies the name of a data file. |
| [IDataObject](../DataMedium/DataMedium_IDataObject.md) | Specifies a IDataObject containing the data. |
| [MediumType](../DataMedium/DataMedium_MediumType.md) | Specifies the type of the data. |
| [Stream](../DataMedium/DataMedium_Stream.md) | Specifies an IStream containing the data. |
| [String](../DataMedium/DataMedium_String.md) | Specifies a String containing the data. |

## Accessed From

[TransientObjects.CreateDataMedium](../TransientObjects/TransientObjects_CreateDataMedium.md)
