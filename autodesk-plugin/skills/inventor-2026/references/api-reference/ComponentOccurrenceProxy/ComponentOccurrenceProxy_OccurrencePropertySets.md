# ComponentOccurrenceProxy.OccurrencePropertySets Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Read-only property that returns a PropertySets object associated with the occurence.

## Syntax

ComponentOccurrenceProxy.**OccurrencePropertySets**() As [PropertySets](../PropertySets/PropertySets.md)

## Property Value

This is a read only property whose value is a [PropertySets](../PropertySets/PropertySets.md).

## Version

Introduced in version 2022
