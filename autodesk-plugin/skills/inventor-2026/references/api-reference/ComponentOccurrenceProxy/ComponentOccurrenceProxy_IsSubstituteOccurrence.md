# ComponentOccurrenceProxy.IsSubstituteOccurrence Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that returns whether this occurrence references a substitute part in the context of a substitute model state.

## Syntax

ComponentOccurrenceProxy.**IsSubstituteOccurrence**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
