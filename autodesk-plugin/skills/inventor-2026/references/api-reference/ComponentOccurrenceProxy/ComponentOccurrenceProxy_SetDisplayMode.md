# ComponentOccurrenceProxy.SetDisplayMode Method

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Sets the display mode and type - default or override.

## Syntax

ComponentOccurrenceProxy.**SetDisplayMode**( ***DisplayModeSourceType*** As [DisplayModeSourceTypeEnum](../DisplayModeSourceTypeEnum.md), [***DisplayMode***] As Variant )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DisplayModeSourceType | [DisplayModeSourceTypeEnum](../DisplayModeSourceTypeEnum.md) | Enum indicating the display mode type. |
| DisplayMode | Variant | The display mode. |

## Version

Introduced in version 10
