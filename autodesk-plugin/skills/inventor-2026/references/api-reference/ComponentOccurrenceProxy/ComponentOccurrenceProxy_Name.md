# ComponentOccurrenceProxy.Name Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets/Sets the human-readable name of this Component Occurrence.

## Syntax

ComponentOccurrenceProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 4
