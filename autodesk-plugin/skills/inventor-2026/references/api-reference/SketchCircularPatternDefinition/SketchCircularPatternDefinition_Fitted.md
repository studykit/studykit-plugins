# SketchCircularPatternDefinition.Fitted Property

Parent Object: [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Description

Gets and sets whether pattern elements are equally fitted within the specified angle.

## Syntax

SketchCircularPatternDefinition.**Fitted**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
