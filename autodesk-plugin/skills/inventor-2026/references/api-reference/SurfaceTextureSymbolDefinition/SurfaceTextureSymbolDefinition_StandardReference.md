# SurfaceTextureSymbolDefinition.StandardReference Property

Parent Object: [SurfaceTextureSymbolDefinition](../SurfaceTextureSymbolDefinition/SurfaceTextureSymbolDefinition.md)

## Description

Returns the standard reference type.

## Syntax

SurfaceTextureSymbolDefinition.**StandardReference**() As [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md)

## Property Value

This is a read only property whose value is a [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md).

## Version

Introduced in version 2025
