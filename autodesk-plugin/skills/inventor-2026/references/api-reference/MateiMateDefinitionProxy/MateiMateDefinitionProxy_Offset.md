# MateiMateDefinitionProxy.Offset Property

Parent Object: [MateiMateDefinitionProxy](../MateiMateDefinitionProxy/MateiMateDefinitionProxy.md)

## Description

Property that returns the Parameter object that controls the offset of the mate iMate definition.

## Syntax

MateiMateDefinitionProxy.**Offset**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
