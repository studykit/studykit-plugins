# MateiMateDefinitionProxy.Name Property

Parent Object: [MateiMateDefinitionProxy](../MateiMateDefinitionProxy/MateiMateDefinitionProxy.md)

## Description

Gets the name of this iMateDefinition.

## Syntax

MateiMateDefinitionProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 6
