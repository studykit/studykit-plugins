# Columns Object

## Description

The Columns collection object represents the collection of objects for this table.See the Custom Tables overview for more information.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Columns/Columns_Add.md) | Method that creates a new Column. The new created Column is returned. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Columns/Columns_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Columns/Columns_Count.md) | Property that returns the number of items in the collection. |
| [Item](../Columns/Columns_Item.md) | Returns the specified Column object from the collection. |
| [Type](../Columns/Columns_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
