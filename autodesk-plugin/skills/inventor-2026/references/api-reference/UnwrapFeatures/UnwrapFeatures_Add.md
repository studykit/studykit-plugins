# UnwrapFeatures.Add Method

Parent Object: [UnwrapFeatures](../UnwrapFeatures/UnwrapFeatures.md)

## Description

Method that creates a new Unwrap feature. The newly created UnwrapFeature object is returned.

## Syntax

UnwrapFeatures.**Add**( ***Definition*** As [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md) ) As [UnwrapFeature](../UnwrapFeature/UnwrapFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md) | Input UnwrapDefinition object that defins the unwrap feature you want to create.An UnwrapDefinition object can be created using the UnwrapFeatures.CreateDefinition method. It can also be obtained from an existing UnwrapFeature object. |

## Version

Introduced in version 2020
