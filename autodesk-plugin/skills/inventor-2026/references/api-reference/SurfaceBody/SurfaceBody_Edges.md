# SurfaceBody.Edges Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Gets the referenced by this SurfaceBody.

## Syntax

SurfaceBody.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 4
