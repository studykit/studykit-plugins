# ToExtent.Direction Property

Parent Object: [ToExtent](../ToExtent/ToExtent.md)

## Description

Gets and sets the direction of the feature.

## Syntax

ToExtent.**Direction**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2008
