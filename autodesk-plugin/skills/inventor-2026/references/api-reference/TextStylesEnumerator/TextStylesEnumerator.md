# TextStylesEnumerator Object

## Description

The TextStylesEnumerator Collection Object allows iteration of the available text styles.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TextStylesEnumerator/TextStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../TextStylesEnumerator/TextStylesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../TextStylesEnumerator/TextStylesEnumerator_Item.md) | Returns the specified object in the collection. |
| [Type](../TextStylesEnumerator/TextStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[AssemblyDocument.TextStyles](../AssemblyDocument/AssemblyDocument_TextStyles.md), [DrawingStylesManager.TextStyles](../DrawingStylesManager/DrawingStylesManager_TextStyles.md), [PartDocument.TextStyles](../PartDocument/PartDocument_TextStyles.md)

## Version

Introduced in version 9
