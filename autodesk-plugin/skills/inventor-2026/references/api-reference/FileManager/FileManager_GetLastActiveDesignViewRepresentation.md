# FileManager.GetLastActiveDesignViewRepresentation Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Method that returns the name of the design view representation that was active when the file was last saved.

## Syntax

FileManager.**GetLastActiveDesignViewRepresentation**( ***FullFileName*** As String ) As String

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input String that specifies the full file name of a part or assembly document. |

## Version

Introduced in version 2010
