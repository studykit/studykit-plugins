# FileManager.Files Property

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Property that returns the FilesEnumerator object

## Syntax

FileManager.**Files**() As [FilesEnumerator](../FilesEnumerator/FilesEnumerator.md)

## Property Value

This is a read only property whose value is a [FilesEnumerator](../FilesEnumerator/FilesEnumerator.md).

## Version

Introduced in version 11
