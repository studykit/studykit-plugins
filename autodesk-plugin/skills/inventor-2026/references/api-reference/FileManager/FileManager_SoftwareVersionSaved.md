# FileManager.SoftwareVersionSaved Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Returns the object that encapsulates the version of the software with which this file was last saved.

## Syntax

FileManager.**SoftwareVersionSaved**( ***FullFileName*** As String ) As [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input string value that indicates the full filename of an Inventor file. |

## Version

Introduced in version 2011
