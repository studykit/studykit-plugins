# FileManager.GetIdentifierFromFileName Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Creates and returns a unique, persistent identifier for the specified file.

## Syntax

FileManager.**GetIdentifierFromFileName**( ***FullFileName*** As String, ***Identifier***() As Byte, [***AbsolutePath***] As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input String that specifies the full path and filename of the file. |
| Identifier | Byte | The Identifier for the file. |
| AbsolutePath | String | String specifying the absolute path to the file. |

## Version

Introduced in version 9
