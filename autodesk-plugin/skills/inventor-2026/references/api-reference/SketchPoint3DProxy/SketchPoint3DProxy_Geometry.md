# SketchPoint3DProxy.Geometry Property

Parent Object: [SketchPoint3DProxy](../SketchPoint3DProxy/SketchPoint3DProxy.md)

## Description

Gets and sets a Point geometry object. The object returned represents a 'snapshot' view of the current state of the sketch point.

## Syntax

SketchPoint3DProxy.**Geometry**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
