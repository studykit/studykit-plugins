# CommandManager.PeekPrivateEvent Method

Parent Object: [CommandManager](../CommandManager/CommandManager.md)

## Description

Method that queries data contained by Autodesk Inventor's internal clipboard.

## Syntax

CommandManager.**PeekPrivateEvent**( ***DataType*** As [PrivateEventTypeEnum](../PrivateEventTypeEnum.md), ***Data*** As Variant )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DataType | [PrivateEventTypeEnum](../PrivateEventTypeEnum.md) | Indicates the type of value contained by the clipboard. |
| Data | Variant | The actual data contained by the clipboard. |

## Version

Introduced in version 10
