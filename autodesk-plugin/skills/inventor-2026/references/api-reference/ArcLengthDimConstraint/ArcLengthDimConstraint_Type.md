# ArcLengthDimConstraint.Type Property

Parent Object: [ArcLengthDimConstraint](../ArcLengthDimConstraint/ArcLengthDimConstraint.md)

## Description

## Syntax

ArcLengthDimConstraint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
