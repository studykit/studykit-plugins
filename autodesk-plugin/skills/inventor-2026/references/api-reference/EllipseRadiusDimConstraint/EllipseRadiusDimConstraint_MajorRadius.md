# EllipseRadiusDimConstraint.MajorRadius Property

Parent Object: [EllipseRadiusDimConstraint](../EllipseRadiusDimConstraint/EllipseRadiusDimConstraint.md)

## Description

Property that returns whether the proxy constraint is controlling the major or minor radius of the ellipse. Returns true if it is the major radius.

## Syntax

EllipseRadiusDimConstraint.**MajorRadius**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5
