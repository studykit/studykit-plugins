# AssemblyJointProxy.Definition Property

Parent Object: [AssemblyJointProxy](../AssemblyJointProxy/AssemblyJointProxy.md)

## Description

Read-write property that gets and sets the AssemblyJointDefinition object associated with this assembly joint object.

## Syntax

AssemblyJointProxy.**Definition**() As [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Property Value

This is a read/write property whose value is an [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md).

## Version

Introduced in version 2014
