# SnapFitFeatureProxy.Shared Property

Parent Object: [SnapFitFeatureProxy](../SnapFitFeatureProxy/SnapFitFeatureProxy.md)

## Description

Gets and sets whether the part feature is shared or not, applies only to surface features.

## Syntax

SnapFitFeatureProxy.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
