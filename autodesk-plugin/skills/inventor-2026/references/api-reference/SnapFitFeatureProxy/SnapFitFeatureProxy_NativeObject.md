# SnapFitFeatureProxy.NativeObject Property

Parent Object: [SnapFitFeatureProxy](../SnapFitFeatureProxy/SnapFitFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SnapFitFeatureProxy.**NativeObject**() As [SnapFitFeature](../SnapFitFeature/SnapFitFeature.md)

## Property Value

This is a read only property whose value is a [SnapFitFeature](../SnapFitFeature/SnapFitFeature.md).

## Version

Introduced in version 2010
