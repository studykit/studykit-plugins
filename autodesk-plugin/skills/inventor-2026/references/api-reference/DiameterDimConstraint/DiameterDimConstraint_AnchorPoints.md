# DiameterDimConstraint.AnchorPoints Property

Parent Object: [DiameterDimConstraint](../DiameterDimConstraint/DiameterDimConstraint.md)

## Description

Gets the anchor points of dimension.

## Syntax

DiameterDimConstraint.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
