# HoleTableColumn.PropertyType Property

Parent Object: [HoleTableColumn](../HoleTableColumn/HoleTableColumn.md)

## Description

Property that returns the property type associated with this column.

## Syntax

HoleTableColumn.**PropertyType**() As [HolePropertyEnum](../HolePropertyEnum.md)

## Property Value

This is a read only property whose value is a [HolePropertyEnum](../HolePropertyEnum.md).

## Version

Introduced in version 2009
