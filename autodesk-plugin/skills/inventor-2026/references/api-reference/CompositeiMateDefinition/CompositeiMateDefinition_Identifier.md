# CompositeiMateDefinition.Identifier Property

Parent Object: [CompositeiMateDefinition](../CompositeiMateDefinition/CompositeiMateDefinition.md)

## Description

Property that returns the internal unique identifier for this iMateDefinition.

## Syntax

CompositeiMateDefinition.**Identifier**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 6
