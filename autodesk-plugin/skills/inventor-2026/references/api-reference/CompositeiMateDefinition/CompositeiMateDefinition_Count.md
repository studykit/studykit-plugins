# CompositeiMateDefinition.Count Property

Parent Object: [CompositeiMateDefinition](../CompositeiMateDefinition/CompositeiMateDefinition.md)

## Description

Property that returns the number of iMateDefinitions in this composite iMate.

## Syntax

CompositeiMateDefinition.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 6
