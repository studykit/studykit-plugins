# HoleFeatureProxy.ClearanceInfo Property

Parent Object: [HoleFeatureProxy](../HoleFeatureProxy/HoleFeatureProxy.md)

## Description

Gets and sets the HoleClearanceInfo object associated with the hole.

## Syntax

HoleFeatureProxy.**ClearanceInfo**() As [HoleClearanceInfo](../HoleClearanceInfo/HoleClearanceInfo.md)

## Property Value

This is a read/write property whose value is a [HoleClearanceInfo](../HoleClearanceInfo/HoleClearanceInfo.md).

## Version

Introduced in version 2018
