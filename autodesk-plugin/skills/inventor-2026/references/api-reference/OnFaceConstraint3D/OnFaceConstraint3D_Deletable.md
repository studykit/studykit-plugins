# OnFaceConstraint3D.Deletable Property

Parent Object: [OnFaceConstraint3D](../OnFaceConstraint3D/OnFaceConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

OnFaceConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2017
