# DriveConstraintSettings.PauseDelay Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Gets and sets the delay between steps in seconds.

## Syntax

DriveConstraintSettings.**PauseDelay**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2012
