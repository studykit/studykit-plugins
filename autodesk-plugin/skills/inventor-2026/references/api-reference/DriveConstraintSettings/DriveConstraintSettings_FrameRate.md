# DriveConstraintSettings.FrameRate Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Gets and sets the increments at which a "snapshot" is taken for inclusion as a frame in a recorded animation.

## Syntax

DriveConstraintSettings.**FrameRate**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2012
