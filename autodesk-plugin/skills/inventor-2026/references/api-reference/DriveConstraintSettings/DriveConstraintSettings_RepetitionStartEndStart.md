# DriveConstraintSettings.RepetitionStartEndStart Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Gets and sets whether to drive the constraint from the start value to the end value and then in reverse to the start value.

## Syntax

DriveConstraintSettings.**RepetitionStartEndStart**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
