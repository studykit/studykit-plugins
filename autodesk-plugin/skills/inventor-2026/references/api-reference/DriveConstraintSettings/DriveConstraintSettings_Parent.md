# DriveConstraintSettings.Parent Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DriveConstraintSettings.**Parent**() As [AssemblyConstraint](../AssemblyConstraint/AssemblyConstraint.md)

## Property Value

This is a read only property whose value is an [AssemblyConstraint](../AssemblyConstraint/AssemblyConstraint.md).

## Version

Introduced in version 2012
