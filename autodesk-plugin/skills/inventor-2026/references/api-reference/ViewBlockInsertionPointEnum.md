# ViewBlockInsertionPointEnum Enumerator

## Description

Type to specify default insertion point for a view block.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kModelOriginInsertion | 86786 | Use model origin as insertion point. |
| kViewCenterInsertion | 86785 | Use view center as insertion point. |

## Version

Introduced in version 2010
