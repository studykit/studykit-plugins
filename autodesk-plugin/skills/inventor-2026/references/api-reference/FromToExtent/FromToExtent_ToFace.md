# FromToExtent.ToFace Property

Parent Object: [FromToExtent](../FromToExtent/FromToExtent.md)

## Description

Property that gets and sets the face that defines the 'to' extents of the feature.

## Syntax

FromToExtent.**ToFace**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 5
