# DerivedFutureAssemblyComponentProxy.NativeObject Property

Parent Object: [DerivedFutureAssemblyComponentProxy](../DerivedFutureAssemblyComponentProxy/DerivedFutureAssemblyComponentProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

DerivedFutureAssemblyComponentProxy.**NativeObject**() As [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md)

## Property Value

This is a read only property whose value is a [DerivedFutureAssemblyComponent](../DerivedFutureAssemblyComponent/DerivedFutureAssemblyComponent.md).

## Version

Introduced in version 2018
