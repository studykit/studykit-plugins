# MidpointConstraint3DProxy.Point Property

Parent Object: [MidpointConstraint3DProxy](../MidpointConstraint3DProxy/MidpointConstraint3DProxy.md)

## Description

Property that returns the sketch point being constrained.

## Syntax

MidpointConstraint3DProxy.**Point**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 2009
