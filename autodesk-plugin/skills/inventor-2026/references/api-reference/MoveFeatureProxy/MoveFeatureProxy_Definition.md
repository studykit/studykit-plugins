# MoveFeatureProxy.Definition Property

Parent Object: [MoveFeatureProxy](../MoveFeatureProxy/MoveFeatureProxy.md)

## Description

Gets and sets the definition for the move feature.

## Syntax

MoveFeatureProxy.**Definition**() As [MoveDefinition](../MoveDefinition/MoveDefinition.md)

## Property Value

This is a read/write property whose value is a [MoveDefinition](../MoveDefinition/MoveDefinition.md).

## Version

Introduced in version 2013
