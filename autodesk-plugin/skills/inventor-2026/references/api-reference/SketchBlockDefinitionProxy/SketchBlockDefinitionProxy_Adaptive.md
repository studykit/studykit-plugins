# SketchBlockDefinitionProxy.Adaptive Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and sets whether the sketch is adaptive or not.

## Syntax

SketchBlockDefinitionProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
