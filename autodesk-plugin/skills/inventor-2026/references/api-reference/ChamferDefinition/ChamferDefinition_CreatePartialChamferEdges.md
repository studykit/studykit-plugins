# ChamferDefinition.CreatePartialChamferEdges Method

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Create a new transient PartialChamferEdges collection object which can be used to define the partially chamfered edges info.

## Syntax

ChamferDefinition.**CreatePartialChamferEdges**() As [PartialChamferEdges](../PartialChamferEdges/PartialChamferEdges.md)

## Version

Introduced in version 2018
