# PartialChamferEdge.EndDistance Property

Parent Object: [PartialChamferEdge](../PartialChamferEdge/PartialChamferEdge.md)

## Description

Gets and sets the end distance of the chamfer on the Edge or Edge chain.

## Syntax

PartialChamferEdge.**EndDistance**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2018
