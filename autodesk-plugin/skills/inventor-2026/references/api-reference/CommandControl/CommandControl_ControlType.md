# CommandControl.ControlType Property

Parent Object: [CommandControl](../CommandControl/CommandControl.md)

## Description

Property that returns the control type.

## Syntax

CommandControl.**ControlType**() As [ControlTypeEnum](ControlTypeEnum.md)

## Property Value

This is a read only property whose value is a [ControlTypeEnum](ControlTypeEnum.md).
