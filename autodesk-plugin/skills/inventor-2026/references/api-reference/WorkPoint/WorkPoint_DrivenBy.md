# WorkPoint.DrivenBy Property

Parent Object: [WorkPoint](../WorkPoint/WorkPoint.md)

## Description

Property that returns the collection of objects that the work point is dependent on.

## Syntax

WorkPoint.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
