# WorkPoint.Parent Property

Parent Object: [WorkPoint](../WorkPoint/WorkPoint.md)

## Description

Property returning the parent  object.

## Syntax

WorkPoint.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 4
