# MirrorFeature.SurfacesVisible Property

Parent Object: [MirrorFeature](../MirrorFeature/MirrorFeature.md)

## Description

Gets and sets whether all the resulting surfaces are visible or not.

## Syntax

MirrorFeature.**SurfacesVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
