# DrawingViewHatchRegion.Shift Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that specifies the hatch pattern offset from original hatch pattern position.

## Syntax

DrawingViewHatchRegion.**Shift**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
