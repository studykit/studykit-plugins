# SplineFitPointConstraint.Parent Property

Parent Object: [SplineFitPointConstraint](../SplineFitPointConstraint/SplineFitPointConstraint.md)

## Description

Property that returns the parent sketch of the object.

## Syntax

SplineFitPointConstraint.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 5
