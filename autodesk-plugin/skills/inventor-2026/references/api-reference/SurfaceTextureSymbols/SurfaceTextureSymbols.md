# SurfaceTextureSymbols Object

## Description

The SurfaceTextureSymbols collection object provides access to all the surface texture symbols on a sheet and provides methods to create additional ones.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddByDefinition](../SurfaceTextureSymbols/SurfaceTextureSymbols_AddByDefinition.md) | Method that creates a surface texture symbol. |
| [CreateDefinition](../SurfaceTextureSymbols/SurfaceTextureSymbols_CreateDefinition.md) | Method that creates a surface texture definition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SurfaceTextureSymbols/SurfaceTextureSymbols_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SurfaceTextureSymbols/SurfaceTextureSymbols_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SurfaceTextureSymbols/SurfaceTextureSymbols_Item.md) | Method that returns the specified SurfaceTextureSymbol object from the collection. |
| [Type](../SurfaceTextureSymbols/SurfaceTextureSymbols_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
