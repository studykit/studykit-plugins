# AnnotationPlane.Type Property

Parent Object: [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Description

Read-only property returning kAnnotationPlaneObject indicating this objects type.

## Syntax

AnnotationPlane.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2018
