# SketchRectangularPatternDefinition.NaturalYDirection Property

Parent Object: [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Description

Gets and sets the flag that indicates if the direction of the pattern is in the natural direction of the YDirectionEntity or reversed.

## Syntax

SketchRectangularPatternDefinition.**NaturalYDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
