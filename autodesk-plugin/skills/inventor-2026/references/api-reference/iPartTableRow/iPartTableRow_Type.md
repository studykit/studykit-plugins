# iPartTableRow.Type Property

Parent Object: [iPartTableRow](../iPartTableRow/iPartTableRow.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

iPartTableRow.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
