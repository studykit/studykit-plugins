# PointAndPlaneDistanceDimConstraint3D.Plane Property

Parent Object: [PointAndPlaneDistanceDimConstraint3D](../PointAndPlaneDistanceDimConstraint3D/PointAndPlaneDistanceDimConstraint3D.md)

## Description

Property that returns the planar Face or WorkPlane object that defines the distance to the 3D sketch point being constrained.

## Syntax

PointAndPlaneDistanceDimConstraint3D.**Plane**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
