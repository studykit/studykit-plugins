# MeshFeatureProxy.AppearanceSourceType Property

Parent Object: [MeshFeatureProxy](../MeshFeatureProxy/MeshFeatureProxy.md)

## Description

Read-write property that gets and sets the source of the appearance for the object.

## Syntax

MeshFeatureProxy.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2017
