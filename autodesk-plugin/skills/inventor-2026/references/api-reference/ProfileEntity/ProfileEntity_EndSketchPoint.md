# ProfileEntity.EndSketchPoint Property

Parent Object: [ProfileEntity](../ProfileEntity/ProfileEntity.md)

## Description

Property that returns the that defines the end of the profile entity. This property is not valid in the case where the curve is a periodic curve, i.e. circle, ellipse, etc.

## Syntax

ProfileEntity.**EndSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 5
