# ProfileEntity.CurveType Property

Parent Object: [ProfileEntity](../ProfileEntity/ProfileEntity.md)

## Description

Property that returns the type of the curve referenced by the profile entity. This property allows you to determine what type of object will be returned by the Curve property.

## Syntax

ProfileEntity.**CurveType**() As [Curve2dTypeEnum](../Curve2dTypeEnum.md)

## Property Value

This is a read only property whose value is a [Curve2dTypeEnum](../Curve2dTypeEnum.md).

## Version

Introduced in version 5
