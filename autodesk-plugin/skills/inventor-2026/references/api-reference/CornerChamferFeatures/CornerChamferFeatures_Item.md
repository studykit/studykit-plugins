# CornerChamferFeatures.Item Property

Parent Object: [CornerChamferFeatures](../CornerChamferFeatures/CornerChamferFeatures.md)

## Description

Returns the specified CornerChamferFeature object from the collection. This is the default property of the CornerChamferFeatures collection object.

## Syntax

CornerChamferFeatures.**Item**( ***Index*** As Variant ) As [CornerChamferFeature](../CornerChamferFeature/CornerChamferFeature.md)

## Property Value

This is a read only property whose value is a [CornerChamferFeature](../CornerChamferFeature/CornerChamferFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the derived occurrence's name. If an out of range index or a name of a non-existent occurrence is provided, an error occurs. |

## Version

Introduced in version 5.3
