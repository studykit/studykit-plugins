# DecalFeature.Image Property

Parent Object: [DecalFeature](../DecalFeature/DecalFeature.md)

## Description

Gets and sets the sketch image used to apply the decal.

## Syntax

DecalFeature.**Image**() As [SketchImage](../SketchImage/SketchImage.md)

## Property Value

This is a read/write property whose value is a [SketchImage](../SketchImage/SketchImage.md).

## Version

Introduced in version 2008
