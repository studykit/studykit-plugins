# AssemblyOptions.PlaceAndGroundFirstComponentAtOrigin Property

Parent Object: [AssemblyOptions](../AssemblyOptions/AssemblyOptions.md)

## Description

Place and ground first component at origin.

## Syntax

AssemblyOptions.**PlaceAndGroundFirstComponentAtOrigin**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2020
