# EdgeProxy.GeometryType Property

Parent Object: [EdgeProxy](../EdgeProxy/EdgeProxy.md)

## Description

Get the curve type of the curve that will be returned from the Geometry property.

## Syntax

EdgeProxy.**GeometryType**() As [CurveTypeEnum](../CurveTypeEnum.md)

## Property Value

This is a read only property whose value is a [CurveTypeEnum](../CurveTypeEnum.md).

## Version

Introduced in version 2008
