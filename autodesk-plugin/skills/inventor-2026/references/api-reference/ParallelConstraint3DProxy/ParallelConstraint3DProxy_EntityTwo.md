# ParallelConstraint3DProxy.EntityTwo Property

Parent Object: [ParallelConstraint3DProxy](../ParallelConstraint3DProxy/ParallelConstraint3DProxy.md)

## Description

Property that returns the second entity being constrained.

## Syntax

ParallelConstraint3DProxy.**EntityTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
