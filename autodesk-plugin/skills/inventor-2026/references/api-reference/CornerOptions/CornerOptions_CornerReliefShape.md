# CornerOptions.CornerReliefShape Property

Parent Object: [CornerOptions](../CornerOptions/CornerOptions.md)

## Description

Defines the type of two bend relief shape to use for the associated sheet metal feature.

## Syntax

CornerOptions.**CornerReliefShape**() As [CornerReliefShapeEnum](../CornerReliefShapeEnum.md)

## Property Value

This is a read/write property whose value is a [CornerReliefShapeEnum](../CornerReliefShapeEnum.md).

## Version

Introduced in version 2009
