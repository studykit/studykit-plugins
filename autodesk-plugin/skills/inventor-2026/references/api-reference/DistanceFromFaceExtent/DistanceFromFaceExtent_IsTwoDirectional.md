# DistanceFromFaceExtent.IsTwoDirectional Property

Parent Object: [DistanceFromFaceExtent](../DistanceFromFaceExtent/DistanceFromFaceExtent.md)

## Description

Read-write property that gets and sets the distance from face, this can be a Face or WorkPlane object. This is applicable when the Direction is not set to kSymmetricExtentDirection.

## Syntax

DistanceFromFaceExtent.**IsTwoDirectional**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
