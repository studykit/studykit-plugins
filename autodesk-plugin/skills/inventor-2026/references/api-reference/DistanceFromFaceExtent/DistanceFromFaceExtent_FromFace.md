# DistanceFromFaceExtent.FromFace Property

Parent Object: [DistanceFromFaceExtent](../DistanceFromFaceExtent/DistanceFromFaceExtent.md)

## Description

Read-write property that gets and sets the distance from face, this can be a Face or WorkPlane object.

## Syntax

DistanceFromFaceExtent.**FromFace**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2018
