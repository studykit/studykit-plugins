# ClientView.Type Property

Parent Object: [ClientView](../ClientView/ClientView.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ClientView.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
