# DrawingStylesManager.TextStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that gets the TextStyles collection object.

## Syntax

DrawingStylesManager.**TextStyles**() As [TextStylesEnumerator](../TextStylesEnumerator/TextStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [TextStylesEnumerator](../TextStylesEnumerator/TextStylesEnumerator.md).

## Version

Introduced in version 9
