# DrawingStylesManager.ReplaceStyles Method

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Method that replaces styles.

## Syntax

DrawingStylesManager.**ReplaceStyles**( ***ReplacedStyles*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md), ***ReplaceStyle*** As [Style](../Style/Style.md), [***PurgeReplacedStyles***] As Variant )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ReplacedStyles | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection object that specifies the styles to be replaced. |
| ReplaceStyle | [Style](../Style/Style.md) | Input Style object that specifies the replace style. |
| PurgeReplacedStyles | Variant | Optional input Boolean that specifies whether to purge the replaced styles or not. This default to False if not specified. |

## Version

Introduced in version 2021
