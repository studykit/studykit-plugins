# DrawingStylesManager.EdgeSymbolStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Read-only property that returns the EdgeSymbolStylesEnumerator object. This object provides access to all the edge symbol styles.

## Syntax

DrawingStylesManager.**EdgeSymbolStyles**() As [EdgeSymbolStylesEnumerator](../EdgeSymbolStylesEnumerator/EdgeSymbolStylesEnumerator.md)

## Property Value

This is a read only property whose value is an [EdgeSymbolStylesEnumerator](../EdgeSymbolStylesEnumerator/EdgeSymbolStylesEnumerator.md).

## Version

Introduced in version 2024
