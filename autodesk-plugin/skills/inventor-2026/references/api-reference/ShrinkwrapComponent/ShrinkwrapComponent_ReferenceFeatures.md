# ShrinkwrapComponent.ReferenceFeatures Property

Parent Object: [ShrinkwrapComponent](../ShrinkwrapComponent/ShrinkwrapComponent.md)

## Description

Gets the ReferenceFeature objects that were created as a result of this shrinkwrap.

## Syntax

ShrinkwrapComponent.**ReferenceFeatures**() As [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md)

## Property Value

This is a read only property whose value is a [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md).

## Version

Introduced in version 2019
