# ContourRollFeature.Name Property

Parent Object: [ContourRollFeature](../ContourRollFeature/ContourRollFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

ContourRollFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
