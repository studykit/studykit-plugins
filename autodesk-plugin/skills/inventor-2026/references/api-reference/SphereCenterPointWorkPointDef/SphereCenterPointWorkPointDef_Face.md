# SphereCenterPointWorkPointDef.Face Property

Parent Object: [SphereCenterPointWorkPointDef](../SphereCenterPointWorkPointDef/SphereCenterPointWorkPointDef.md)

## Description

Gets the face object whose geometry is a Sphere surface, the center point of which defines the position of the work point.

## Syntax

SphereCenterPointWorkPointDef.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2014
