# HelpManager.HelpEvents Property

Parent Object: [HelpManager](../HelpManager/HelpManager.md)

## Description

Gets the object that fires Help related events.

## Syntax

HelpManager.**HelpEvents**() As [HelpEvents](../HelpEvents/HelpEvents.md)

## Property Value

This is a read only property whose value is a [HelpEvents](../HelpEvents/HelpEvents.md).

## Version

Introduced in version 2012
