# DrawingBOMRow.Ballooned Property

Parent Object: [DrawingBOMRow](../DrawingBOMRow/DrawingBOMRow.md)

## Description

Property that returns whether the item represented by this row in the drawing BOM has been ballooned in the drawing.

## Syntax

DrawingBOMRow.**Ballooned**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
