# LoftDefinition.FirstSectionTangentPlane Property

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Property that specifies the tangent plane for the starting section in the case that the section is a point.

## Syntax

LoftDefinition.**FirstSectionTangentPlane**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2008
