# iPartTableColumns Object

## Description

The iPartTableColumns collection object provides access to all existing objects in the iPart factory table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../iPartTableColumns/iPartTableColumns_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../iPartTableColumns/iPartTableColumns_Count.md) | Property that returns the number of items in the collection. |
| [Item](../iPartTableColumns/iPartTableColumns_Item.md) | Returns the specified iPartTableColumn object from the collection. This is the default property of the iPartTableColumns collection object. |
| [Type](../iPartTableColumns/iPartTableColumns_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[iPartFactory.TableColumns](../iPartFactory/iPartFactory_TableColumns.md)

## Version

Introduced in version 6
