# HemFeatureProxy.Parameters Property

Parent Object: [HemFeatureProxy](../HemFeatureProxy/HemFeatureProxy.md)

## Description

Property that returns all the parameters associated with the feature.

## Syntax

HemFeatureProxy.**Parameters**() As [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md)

## Property Value

This is a read only property whose value is a [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md).

## Version

Introduced in version 11
