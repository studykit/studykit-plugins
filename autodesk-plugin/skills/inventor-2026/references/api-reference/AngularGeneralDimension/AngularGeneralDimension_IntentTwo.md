# AngularGeneralDimension.IntentTwo Property

Parent Object: [AngularGeneralDimension](../AngularGeneralDimension/AngularGeneralDimension.md)

## Description

Gets and sets the second geometry associated with the dimension.

## Syntax

AngularGeneralDimension.**IntentTwo**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 11
