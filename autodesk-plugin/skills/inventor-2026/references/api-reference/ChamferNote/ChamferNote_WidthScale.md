# ChamferNote.WidthScale Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Gets/Sets the width scale factor of the DrawingNote.

## Syntax

ChamferNote.**WidthScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
