# ChamferNote.UseBackgroundColor Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Gets/Sets the UseBackgroundColor associated with the LeaderNote.

## Syntax

ChamferNote.**UseBackgroundColor**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
