# DockableWindow.Parent Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Property that returns the parent UserInterfaceManager object.

## Syntax

DockableWindow.**Parent**() As [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Property Value

This is a read only property whose value is a [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md).

## Version

Introduced in version 2011
