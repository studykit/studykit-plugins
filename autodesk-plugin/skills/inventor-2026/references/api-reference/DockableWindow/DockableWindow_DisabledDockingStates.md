# DockableWindow.DisabledDockingStates Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Read-write property that gets and sets the bit-mask that specifies the docking positions to disallow.

## Syntax

DockableWindow.**DisabledDockingStates**() As [DockingStateEnum](../DockingStateEnum.md)

## Property Value

This is a read/write property whose value is a [DockingStateEnum](../DockingStateEnum.md).
