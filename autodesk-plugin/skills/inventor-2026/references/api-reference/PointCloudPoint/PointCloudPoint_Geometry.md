# PointCloudPoint.Geometry Property

Parent Object: [PointCloudPoint](../PointCloudPoint/PointCloudPoint.md)

## Description

Read-only property that returns a Point object which provides the coordinates of this point.

## Syntax

PointCloudPoint.**Geometry**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2013
