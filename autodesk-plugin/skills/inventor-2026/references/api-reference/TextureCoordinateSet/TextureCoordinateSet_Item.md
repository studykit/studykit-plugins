# TextureCoordinateSet.Item Property

Parent Object: [TextureCoordinateSet](../TextureCoordinateSet/TextureCoordinateSet.md)

## Description

Returns the specified coordinates from the set.

## Syntax

TextureCoordinateSet.**Item**( ***Index*** As Long ) As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Specifies the object to return. |

## Version

Introduced in version 10
