# PromptMessage.Restrictions Property

Parent Object: [PromptMessage](../PromptMessage/PromptMessage.md)

## Description

Read-write property that gets and sets the restrictions for the prompt.

## Syntax

PromptMessage.**Restrictions**() As [PromptMessageRestrictionsEnum](../PromptMessageRestrictionsEnum.md)

## Property Value

This is a read/write property whose value is a [PromptMessageRestrictionsEnum](../PromptMessageRestrictionsEnum.md).

## Version

Introduced in version 2025.1
