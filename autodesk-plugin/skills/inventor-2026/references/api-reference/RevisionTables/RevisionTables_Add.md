# RevisionTables.Add Method

Parent Object: [RevisionTables](../RevisionTables/RevisionTables.md)

## Description

Method that creates a new RevisionTable. The new created RevisionTable is returned.

## Syntax

RevisionTables.**Add**( ***PlacementPoint*** As [Point2d](../Point2d/Point2d.md) ) As [RevisionTable](../RevisionTable/RevisionTable.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PlacementPoint | [Point2d](../Point2d/Point2d.md) | Input Point2d object that specifies the placement point of the revision table on the sheet. |

## Version

Introduced in version 10
