# InstallationStatusEnum Enumerator

## Description

Describe the installation status.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kInstallationFailedStatus | 127746 | Installation failed. |
| kInstallationInProcessStatus | 127748 | Installation is in process. |
| kInstalledStatus | 127747 | The specified version has been installed. |
| kUnsupportedVersionStatus | 127745 | The specified version is unsupported to install. |

## Version

Introduced in version 2025
