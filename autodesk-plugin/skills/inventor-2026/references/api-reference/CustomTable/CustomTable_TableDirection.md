# CustomTable.TableDirection Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the vertical direction of the table.

## Syntax

CustomTable.**TableDirection**() As [TableDirectionEnum](../TableDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [TableDirectionEnum](../TableDirectionEnum.md).

## Version

Introduced in version 2009
