# BIMConnectorLink.ConnectorTwo Property

Parent Object: [BIMConnectorLink](../BIMConnectorLink/BIMConnectorLink.md)

## Description

Read-only property that gets the first of two connectors that this link is between.

## Syntax

BIMConnectorLink.**ConnectorTwo**() As [BIMConnector](../BIMConnector/BIMConnector.md)

## Property Value

This is a read only property whose value is a [BIMConnector](../BIMConnector/BIMConnector.md).

## Version

Introduced in version 2011
