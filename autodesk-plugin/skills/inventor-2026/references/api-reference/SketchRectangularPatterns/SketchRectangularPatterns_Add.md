# SketchRectangularPatterns.Add Method

Parent Object: [SketchRectangularPatterns](../SketchRectangularPatterns/SketchRectangularPatterns.md)

## Description

Creates a new sketch rectangular pattern. The new SketchRectangularPattern object is returned.

## Syntax

SketchRectangularPatterns.**Add**( ***Definition*** As [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md) ) As [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md) | Input SketchRectangularPatternDefinition objet that defines the SketchRectangularPattern to create. |

## Version

Introduced in version 2025.1
