# DerivedParameter.DisplayFormat Property

Parent Object: [DerivedParameter](../DerivedParameter/DerivedParameter.md)

## Description

Gets and sets the display format for the parameter.

## Syntax

DerivedParameter.**DisplayFormat**() As [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md)

## Property Value

This is a read/write property whose value is a [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md).

## Version

Introduced in version 2011
