# WeldmentComponentDefinition.DataIO Property

Parent Object: [WeldmentComponentDefinition](../WeldmentComponentDefinition/WeldmentComponentDefinition.md)

## Description

Gets the object that directly deals with I/O to and from a storage-medium, including Streams(IStream).

## Syntax

WeldmentComponentDefinition.**DataIO**() As [DataIO](../DataIO/DataIO.md)

## Property Value

This is a read only property whose value is a [DataIO](../DataIO/DataIO.md).

## Version

Introduced in version 8
