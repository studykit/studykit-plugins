# WeldmentComponentDefinition.iMateResults Property

Parent Object: [WeldmentComponentDefinition](../WeldmentComponentDefinition/WeldmentComponentDefinition.md)

## Description

Property that returns the iMateResults collection object associated with this assembly.

## Syntax

WeldmentComponentDefinition.**iMateResults**() As [iMateResults](../iMateResults/iMateResults.md)

## Property Value

This is a read only property whose value is an [iMateResults](../iMateResults/iMateResults.md).

## Version

Introduced in version 8
