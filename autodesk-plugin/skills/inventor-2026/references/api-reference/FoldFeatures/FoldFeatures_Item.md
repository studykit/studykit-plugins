# FoldFeatures.Item Property

Parent Object: [FoldFeatures](../FoldFeatures/FoldFeatures.md)

## Description

Returns the specified FoldFeature object from the collection. This is the default property of the FoldFeatures collection object.

## Syntax

FoldFeatures.**Item**( ***Index*** As Variant ) As [FoldFeature](../FoldFeature/FoldFeature.md)

## Property Value

This is a read only property whose value is a [FoldFeature](../FoldFeature/FoldFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the to return. |

## Version

Introduced in version 5.3
