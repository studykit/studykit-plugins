# PointCloud.Scans Property

Parent Object: [PointCloud](../PointCloud/PointCloud.md)

## Description

Read-only property that returns the point cloud scans collection object.

## Syntax

PointCloud.**Scans**() As [PointCloudScans](../PointCloudScans/PointCloudScans.md)

## Property Value

This is a read only property whose value is a [PointCloudScans](../PointCloudScans/PointCloudScans.md).

## Version

Introduced in version 2016
