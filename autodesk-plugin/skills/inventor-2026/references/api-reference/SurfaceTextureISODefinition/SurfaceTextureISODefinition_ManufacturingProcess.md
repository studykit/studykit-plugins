# SurfaceTextureISODefinition.ManufacturingProcess Property

Parent Object: [SurfaceTextureISODefinition](../SurfaceTextureISODefinition/SurfaceTextureISODefinition.md)

## Description

Gets and sets the manufacturing process.

## Syntax

SurfaceTextureISODefinition.**ManufacturingProcess**() As String

## Property Value

This is a read/write property whose value is a String.
