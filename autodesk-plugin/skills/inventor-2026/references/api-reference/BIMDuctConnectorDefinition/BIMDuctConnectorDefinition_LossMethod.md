# BIMDuctConnectorDefinition.LossMethod Property

Parent Object: [BIMDuctConnectorDefinition](../BIMDuctConnectorDefinition/BIMDuctConnectorDefinition.md)

## Description

Read-write property that gets and sets the loss method used for this connector.

## Syntax

BIMDuctConnectorDefinition.**LossMethod**() As [BIMDuctLossMethodEnum](../BIMDuctLossMethodEnum.md)

## Property Value

This is a read/write property whose value is a [BIMDuctLossMethodEnum](../BIMDuctLossMethodEnum.md).

## Version

Introduced in version 2011
