# UserCoordinateSystemSettings Object

## Description

The UserCoordinateSystemSettings object provides access to various UCS related document level settings. This object is obtained from the ModelingSettings.UserCoordinateSystemSettings property.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [DefaultPlane](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_DefaultPlane.md) | Gets and sets the default plane to use for user coordinate systems. |
| [NamingPrefix](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_NamingPrefix.md) | Gets and sets the prefix to use for the name of new user coordinate systems. |
| [ShowOrigin](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowOrigin.md) | Gets and sets whether whether or not to make the origin visible. |
| [ShowTriad](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowTriad.md) | Gets and sets whether whether or not to display the UCS triad. |
| [ShowXAxis](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowXAxis.md) | Gets and sets whether whether or not to make the X axis visible. |
| [ShowXYPlane](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowXYPlane.md) | Gets and sets whether whether or not to make the XY plane visible. |
| [ShowXZPlane](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowXZPlane.md) | Gets and sets whether whether or not to make the XZ plane visible. |
| [ShowYAxis](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowYAxis.md) | Gets and sets whether whether or not to make the Y axis visible. |
| [ShowYZPlane](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowYZPlane.md) | Gets and sets whether whether or not to make the YZ plane visible. |
| [ShowZAxis](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_ShowZAxis.md) | Gets and sets whether whether or not to make the Z axis visible. |
| [Type](../UserCoordinateSystemSettings/UserCoordinateSystemSettings_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ModelingSettings.UserCoordinateSystemSettings](../ModelingSettings/ModelingSettings_UserCoordinateSystemSettings.md)

## Version

Introduced in version 2010
