# UserCoordinateSystemSettings.DefaultPlane Property

Parent Object: [UserCoordinateSystemSettings](../UserCoordinateSystemSettings/UserCoordinateSystemSettings.md)

## Description

Gets and sets the default plane to use for user coordinate systems.

## Syntax

UserCoordinateSystemSettings.**DefaultPlane**() As [CoordinateSystemPlaneEnum](../CoordinateSystemPlaneEnum.md)

## Property Value

This is a read/write property whose value is a [CoordinateSystemPlaneEnum](../CoordinateSystemPlaneEnum.md).

## Version

Introduced in version 2010
