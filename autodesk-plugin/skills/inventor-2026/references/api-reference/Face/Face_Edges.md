# Face.Edges Property

Parent Object: [Face](../Face/Face.md)

## Description

Gets the Edges referenced by this Face.

## Syntax

Face.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).
