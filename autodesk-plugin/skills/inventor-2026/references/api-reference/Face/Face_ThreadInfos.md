# Face.ThreadInfos Property

Parent Object: [Face](../Face/Face.md)

## Description

Property that returns a ThreadInfo object for each thread affecting this face.

## Syntax

Face.**ThreadInfos**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2010
