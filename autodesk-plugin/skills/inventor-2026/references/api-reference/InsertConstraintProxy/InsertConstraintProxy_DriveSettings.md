# InsertConstraintProxy.DriveSettings Property

Parent Object: [InsertConstraintProxy](../InsertConstraintProxy/InsertConstraintProxy.md)

## Description

Returns the DriveSettings object for the constraint.

## Syntax

InsertConstraintProxy.**DriveSettings**() As [DriveSettings](../DriveSettings/DriveSettings.md)

## Property Value

This is a read only property whose value is a [DriveSettings](../DriveSettings/DriveSettings.md).

## Version

Introduced in version 2014
