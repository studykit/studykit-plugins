# OffsetDimConstraint.Line Property

Parent Object: [OffsetDimConstraint](../OffsetDimConstraint/OffsetDimConstraint.md)

## Description

Property that returns the proxy SketchLine being constrained.

## Syntax

OffsetDimConstraint.**Line**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 5
