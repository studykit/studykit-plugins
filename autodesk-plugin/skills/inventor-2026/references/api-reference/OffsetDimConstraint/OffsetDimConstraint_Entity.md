# OffsetDimConstraint.Entity Property

Parent Object: [OffsetDimConstraint](../OffsetDimConstraint/OffsetDimConstraint.md)

## Description

Property that returns the circle or arc being constrained.

## Syntax

OffsetDimConstraint.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
