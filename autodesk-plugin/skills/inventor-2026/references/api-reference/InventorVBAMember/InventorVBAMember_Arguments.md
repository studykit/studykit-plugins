# InventorVBAMember.Arguments Property

Parent Object: [InventorVBAMember](../InventorVBAMember/InventorVBAMember.md)

## Description

Property that returns the collection containing the member's arguments.

## Syntax

InventorVBAMember.**Arguments**() As [InventorVBAArguments](../InventorVBAArguments/InventorVBAArguments.md)

## Property Value

This is a read only property whose value is an [InventorVBAArguments](../InventorVBAArguments/InventorVBAArguments.md).

## Version

Introduced in version 6
