# OffsetConstraint.EntityTwo Property

Parent Object: [OffsetConstraint](../OffsetConstraint/OffsetConstraint.md)

## Description

Property that returns one of the entities.

## Syntax

OffsetConstraint.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
