# SurfaceTextureGBDefinition.AllAroundSymbol Property

Parent Object: [SurfaceTextureGBDefinition](../SurfaceTextureGBDefinition/SurfaceTextureGBDefinition.md)

## Description

Gets and sets whether to add the all-around indicator to the symbol.

## Syntax

SurfaceTextureGBDefinition.**AllAroundSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
