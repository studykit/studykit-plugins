# SurfaceTextureGBDefinition.IsForceTailShown Property

Parent Object: [SurfaceTextureGBDefinition](../SurfaceTextureGBDefinition/SurfaceTextureGBDefinition.md)

## Description

Gets and sets whether to add a tail to the symbol.

## Syntax

SurfaceTextureGBDefinition.**IsForceTailShown**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
