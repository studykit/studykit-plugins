# CosmeticWeldDefinition.ToFace Property

Parent Object: [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md)

## Description

Read only property that gets and sets a planar object as ending plane to terminate the cosmetic weld.

## Syntax

CosmeticWeldDefinition.**ToFace**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2017
