# SketchOffsetSpline.Length Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchOffsetSpline.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 9
