# HelicalCurveVariableShapeDefinition.Parent Property

Parent Object: [HelicalCurveVariableShapeDefinition](../HelicalCurveVariableShapeDefinition/HelicalCurveVariableShapeDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

HelicalCurveVariableShapeDefinition.**Parent**() As [HelicalCurve](../HelicalCurve/HelicalCurve.md)

## Property Value

This is a read only property whose value is a [HelicalCurve](../HelicalCurve/HelicalCurve.md).

## Version

Introduced in version 2019
