# ComponentGraphics.DisplayedEdges Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Read-only property that returns a SurfaceGraphicsEdgeList object. This list provides access to all edges that are currently displayed.

## Syntax

ComponentGraphics.**DisplayedEdges**() As [SurfaceGraphicsEdgeList](../SurfaceGraphicsEdgeList/SurfaceGraphicsEdgeList.md)

## Property Value

This is a read only property whose value is a [SurfaceGraphicsEdgeList](../SurfaceGraphicsEdgeList/SurfaceGraphicsEdgeList.md).

## Version

Introduced in version 2011
