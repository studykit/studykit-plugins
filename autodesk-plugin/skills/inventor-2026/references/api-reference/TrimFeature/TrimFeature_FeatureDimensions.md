# TrimFeature.FeatureDimensions Property

Parent Object: [TrimFeature](../TrimFeature/TrimFeature.md)

## Description

Property that returns the FeatureDimensions collection object.

## Syntax

TrimFeature.**FeatureDimensions**() As [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md)

## Property Value

This is a read only property whose value is a [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md).

## Version

Introduced in version 2008
