# FloatAssetValue.Values Property

Parent Object: [FloatAssetValue](../FloatAssetValue/FloatAssetValue.md)

## Description

Gets and sets the values associated with this asset value. HasMultipleValues property indicates if this property will be returning more than one value.

## Syntax

FloatAssetValue.**Values**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
