# Features.SweepFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the SweepFeatures collection object. This collection provides access to existing SweepFeature objects.

## Syntax

Features.**SweepFeatures**() As [SweepFeatures](../SweepFeatures/SweepFeatures.md)

## Property Value

This is a read only property whose value is a [SweepFeatures](../SweepFeatures/SweepFeatures.md).
