# Features.RevolveFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the RevolveFeatures collection object. This collection provides access to existing RevolveFeature objects.

## Syntax

Features.**RevolveFeatures**() As [RevolveFeatures](../RevolveFeatures/RevolveFeatures.md)

## Property Value

This is a read only property whose value is a [RevolveFeatures](../RevolveFeatures/RevolveFeatures.md).

## Version

Introduced in version 10
