# Features.ClientFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Gets the Client Features collection object.

## Syntax

Features.**ClientFeatures**() As [ClientFeatures](../ClientFeatures/ClientFeatures.md)

## Property Value

This is a read only property whose value is a [ClientFeatures](../ClientFeatures/ClientFeatures.md).

## Version

Introduced in version 2009
