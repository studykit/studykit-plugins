# Features.ChamferFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the ChamferFeatures collection object. This collection provides access to existing ChamferFeature objects.

## Syntax

Features.**ChamferFeatures**() As [ChamferFeatures](../ChamferFeatures/ChamferFeatures.md)

## Property Value

This is a read only property whose value is a [ChamferFeatures](../ChamferFeatures/ChamferFeatures.md).

## Version

Introduced in version 10
