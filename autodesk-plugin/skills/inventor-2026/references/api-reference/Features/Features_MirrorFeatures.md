# Features.MirrorFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the MirrorFeatures collection object. This collection provides access to existing MirrorFeature objects.

## Syntax

Features.**MirrorFeatures**() As [MirrorFeatures](../MirrorFeatures/MirrorFeatures.md)

## Property Value

This is a read only property whose value is a [MirrorFeatures](../MirrorFeatures/MirrorFeatures.md).

## Version

Introduced in version 10
