# Features.CircularPatternFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the CircularPatternFeatures collection object. This collection provides access to existing CircularPatternFeature objects.

## Syntax

Features.**CircularPatternFeatures**() As [CircularPatternFeatures](../CircularPatternFeatures/CircularPatternFeatures.md)

## Property Value

This is a read only property whose value is a [CircularPatternFeatures](../CircularPatternFeatures/CircularPatternFeatures.md).

## Version

Introduced in version 10
