# Features.ThreadFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Gets the collection object that besides listing out the subset of features that are Threads, allows the creations of new Threads.

## Syntax

Features.**ThreadFeatures**() As [ThreadFeatures](../ThreadFeatures/ThreadFeatures.md)

## Property Value

This is a read only property whose value is a [ThreadFeatures](../ThreadFeatures/ThreadFeatures.md).

## Version

Introduced in version 2020
