# ChainDimensionSet.Members Property

Parent Object: [ChainDimensionSet](../ChainDimensionSet/ChainDimensionSet.md)

## Description

Property that returns all the member LinearGeneralDimension objects.

## Syntax

ChainDimensionSet.**Members**() As [GeneralDimensionsEnumerator](../GeneralDimensionsEnumerator/GeneralDimensionsEnumerator.md)

## Property Value

This is a read only property whose value is a [GeneralDimensionsEnumerator](../GeneralDimensionsEnumerator/GeneralDimensionsEnumerator.md).

## Version

Introduced in version 2011
