# OnFaceCurve.FitPointCount Property

Parent Object: [OnFaceCurve](../OnFaceCurve/OnFaceCurve.md)

## Description

Read-only property that returns the number of fit points for the curve.

## Syntax

OnFaceCurve.**FitPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2017
