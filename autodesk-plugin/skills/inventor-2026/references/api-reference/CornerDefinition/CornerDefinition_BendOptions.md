# CornerDefinition.BendOptions Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Gets and sets the BendOptions object that defines the bend options for this bend feature.

## Syntax

CornerDefinition.**BendOptions**() As [BendOptions](../BendOptions/BendOptions.md)

## Property Value

This is a read/write property whose value is a [BendOptions](../BendOptions/BendOptions.md).

## Version

Introduced in version 2009
