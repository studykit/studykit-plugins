# CornerDefinition.Edges Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Gets the EdgeCollection object that specifies either the two edges that represent the two faces to create the seam between or the set of the create a ripped edge seam on.

## Syntax

CornerDefinition.**Edges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2009
