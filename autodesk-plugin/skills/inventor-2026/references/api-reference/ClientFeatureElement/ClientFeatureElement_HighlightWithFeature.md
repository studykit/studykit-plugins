# ClientFeatureElement.HighlightWithFeature Property

Parent Object: [ClientFeatureElement](../ClientFeatureElement/ClientFeatureElement.md)

## Description

Read-write property that gets and sets whether this element should be pre-highlight, highlighted or selected with the client feature.

## Syntax

ClientFeatureElement.**HighlightWithFeature**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
