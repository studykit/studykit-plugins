# MeshFeatureEntityProxy.ColorSet Property

Parent Object: [MeshFeatureEntityProxy](../MeshFeatureEntityProxy/MeshFeatureEntityProxy.md)

## Description

Read-only property that returns the color set of the mesh feature entity.

## Syntax

MeshFeatureEntityProxy.**ColorSet**() As [GraphicsColorSet](../GraphicsColorSet/GraphicsColorSet.md)

## Property Value

This is a read only property whose value is a [GraphicsColorSet](../GraphicsColorSet/GraphicsColorSet.md).

## Version

Introduced in version 2017
