# MeshFeatureEntityProxy.ColorBinding Property

Parent Object: [MeshFeatureEntityProxy](../MeshFeatureEntityProxy/MeshFeatureEntityProxy.md)

## Description

Read-only property that returns the color binding of the mesh feature entity.

## Syntax

MeshFeatureEntityProxy.**ColorBinding**() As [ColorBindingEnum](../ColorBindingEnum.md)

## Property Value

This is a read only property whose value is a [ColorBindingEnum](../ColorBindingEnum.md).

## Version

Introduced in version 2017
