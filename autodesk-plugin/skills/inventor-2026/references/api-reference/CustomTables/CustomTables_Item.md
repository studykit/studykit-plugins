# CustomTables.Item Property

Parent Object: [CustomTables](../CustomTables/CustomTables.md)

## Description

Returns the specified object in the collection.

## Syntax

CustomTables.**Item**( ***Index*** As Long ) As [CustomTable](../CustomTable/CustomTable.md)

## Property Value

This is a read only property whose value is a [CustomTable](../CustomTable/CustomTable.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index within the collection of the item to return. |

## Version

Introduced in version 9
