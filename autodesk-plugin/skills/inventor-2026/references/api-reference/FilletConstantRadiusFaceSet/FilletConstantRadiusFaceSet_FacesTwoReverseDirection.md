# FilletConstantRadiusFaceSet.FacesTwoReverseDirection Property

Parent Object: [FilletConstantRadiusFaceSet](../FilletConstantRadiusFaceSet/FilletConstantRadiusFaceSet.md)

## Description

Boolean that indicates whether the side of the surface on which the fillet was created is flipped. This only applies to faces of a surface and is ignored for faces of a solid.

## Syntax

FilletConstantRadiusFaceSet.**FacesTwoReverseDirection**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
