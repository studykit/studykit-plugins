# PartsListRow.Expanded Property

Parent Object: [PartsListRow](../PartsListRow/PartsListRow.md)

## Description

Gets or sets whether this row is expanded if this row is a nested one i.e. it references an assembly.

## Syntax

PartsListRow.**Expanded**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
