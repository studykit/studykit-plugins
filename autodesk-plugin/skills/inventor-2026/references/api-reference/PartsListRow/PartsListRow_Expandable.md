# PartsListRow.Expandable Property

Parent Object: [PartsListRow](../PartsListRow/PartsListRow.md)

## Description

Property that returns whether this row is a nested one. (i.e. it references an assembly.)

## Syntax

PartsListRow.**Expandable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
