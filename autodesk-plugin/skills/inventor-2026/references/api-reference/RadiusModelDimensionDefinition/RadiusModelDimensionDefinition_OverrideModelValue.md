# RadiusModelDimensionDefinition.OverrideModelValue Property

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Read-write property that gets and sets the display value for the dimension.

## Syntax

RadiusModelDimensionDefinition.**OverrideModelValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
