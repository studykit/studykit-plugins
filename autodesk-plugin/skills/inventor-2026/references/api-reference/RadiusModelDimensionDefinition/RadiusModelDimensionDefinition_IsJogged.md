# RadiusModelDimensionDefinition.IsJogged Property

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Read-write property that gets and sets whether the dimension is jogged.

## Syntax

RadiusModelDimensionDefinition.**IsJogged**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
