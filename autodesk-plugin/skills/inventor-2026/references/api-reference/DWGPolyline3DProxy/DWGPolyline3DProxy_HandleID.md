# DWGPolyline3DProxy.HandleID Property

Parent Object: [DWGPolyline3DProxy](../DWGPolyline3DProxy/DWGPolyline3DProxy.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGPolyline3DProxy.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
