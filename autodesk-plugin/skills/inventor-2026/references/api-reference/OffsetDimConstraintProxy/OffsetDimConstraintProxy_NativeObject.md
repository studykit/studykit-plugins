# OffsetDimConstraintProxy.NativeObject Property

Parent Object: [OffsetDimConstraintProxy](../OffsetDimConstraintProxy/OffsetDimConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

OffsetDimConstraintProxy.**NativeObject**() As [OffsetDimConstraint](../OffsetDimConstraint/OffsetDimConstraint.md)

## Property Value

This is a read only property whose value is an [OffsetDimConstraint](../OffsetDimConstraint/OffsetDimConstraint.md).

## Version

Introduced in version 6
