# iFeatureOptions.SheetMetalPunchesRootPath Property

Parent Object: [iFeatureOptions](../iFeatureOptions/iFeatureOptions.md)

## Description

Gets/Sets the location of iFeature files used by the sheet metal Punch Tool dialog box.

## Syntax

iFeatureOptions.**SheetMetalPunchesRootPath**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
