# DrawingSketch.GeometricConstraints Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Property that returns the collection of all geometric constraints on the sketch.

## Syntax

DrawingSketch.**GeometricConstraints**() As [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md)

## Property Value

This is a read only property whose value is a [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md).

## Version

Introduced in version 5.3
