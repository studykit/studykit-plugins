# DerivedPartCoordinateSystemDef.IncludeAll Method

Parent Object: [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Description

Method that causes all entities within the derived part component to be set to kDerivedIncludeAll.

## Syntax

DerivedPartCoordinateSystemDef.**IncludeAll**()

## Version

Introduced in version 6
