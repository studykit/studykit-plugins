# MeshFeatureSetProxy.Name Property

Parent Object: [MeshFeatureSetProxy](../MeshFeatureSetProxy/MeshFeatureSetProxy.md)

## Description

Read-write property that gets and sets the name of the object.

## Syntax

MeshFeatureSetProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2017
