# MeshFeatureSetProxy.HealthStatus Property

Parent Object: [MeshFeatureSetProxy](../MeshFeatureSetProxy/MeshFeatureSetProxy.md)

## Description

Read-only property that returns an enum indicating the current state of the object.

## Syntax

MeshFeatureSetProxy.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2017
