# NonParametricBaseFeatures.Type Property

Parent Object: [NonParametricBaseFeatures](../NonParametricBaseFeatures/NonParametricBaseFeatures.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

NonParametricBaseFeatures.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
