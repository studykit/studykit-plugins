# DetailDrawingView.Margin Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-write property that gets and sets the amount of area outside the normal view boundary that the view boundary extends.

## Syntax

DetailDrawingView.**Margin**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2024
