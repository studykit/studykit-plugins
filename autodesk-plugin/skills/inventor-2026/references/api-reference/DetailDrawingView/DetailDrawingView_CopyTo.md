# DetailDrawingView.CopyTo Method

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Method that copies this drawing view into the specified sheet. The new object is returned.

## Syntax

DetailDrawingView.**CopyTo**( ***TargetSheet*** As [Sheet](../Sheet/Sheet.md) ) As [DrawingView](../DrawingView/DrawingView.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TargetSheet | [Sheet](../Sheet/Sheet.md) | Input Sheet object that specifies the sheet that this drawing view is to be copied into. An error occurs if the input sheet is the parent sheet object. |

## Version

Introduced in version 11
