# DetailDrawingView.Camera Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that returns a Camera object that defines the orientation of the model within the view.

## Syntax

DetailDrawingView.**Camera**() As [Camera](../Camera/Camera.md)

## Property Value

This is a read only property whose value is a [Camera](../Camera/Camera.md).

## Version

Introduced in version 11
