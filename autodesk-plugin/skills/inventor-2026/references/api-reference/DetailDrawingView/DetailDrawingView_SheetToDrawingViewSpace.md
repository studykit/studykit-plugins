# DetailDrawingView.SheetToDrawingViewSpace Method

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Method that takes a 2d coordinate in sheet space, and returns a Point2d object containing the coordinate point in drawing view space.

## Syntax

DetailDrawingView.**SheetToDrawingViewSpace**( ***SheetCoordinate*** As [Point2d](../Point2d/Point2d.md) ) As [Point2d](../Point2d/Point2d.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SheetCoordinate | [Point2d](../Point2d/Point2d.md) | Input Point2d object of a point within sheet space. |

## Version

Introduced in version 11
