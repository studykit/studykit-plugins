# DetailDrawingView.ScaleString Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets or sets the Model-to-PaperSpace scale in string format of this particular Drawing View.

## Syntax

DetailDrawingView.**ScaleString**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
