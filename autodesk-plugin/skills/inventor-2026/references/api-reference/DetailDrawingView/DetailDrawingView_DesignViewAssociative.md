# DetailDrawingView.DesignViewAssociative Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-only property that gets whether the DesignViewRepresentation for a drawing view of a model is associative or not.

## Syntax

DetailDrawingView.**DesignViewAssociative**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2021
