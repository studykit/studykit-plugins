# DetailDrawingView.SheetToModelTransform Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that returns the transformation from sheet coordinate space to the model space.

## Syntax

DetailDrawingView.**SheetToModelTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 11
