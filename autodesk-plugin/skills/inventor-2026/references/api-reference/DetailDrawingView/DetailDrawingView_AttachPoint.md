# DetailDrawingView.AttachPoint Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that gets and sets geometry to attach the view to. Use Sheet.CreateGeometryIntent method to create a new GeometryIntent object. This property can be set to Nothing to un-attach the view from a geometry.

## Syntax

DetailDrawingView.**AttachPoint**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 11
