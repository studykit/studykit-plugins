# WeldSymbolStyle.SetContourSymbolsFilter Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Method that sets the contour symbol filter.

## Syntax

WeldSymbolStyle.**SetContourSymbolsFilter**( ***ContourSymbolFilter***() As [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ContourSymbolFilter | [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md) | Input an array of ContourSymbolTypeEnum values to set contour symbol filter. |

## Version

Introduced in version 2024
