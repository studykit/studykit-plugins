# WeldSymbolStyle.GetWeldingSymbolsFilter Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Method that gets the welding symbol filter.

## Syntax

WeldSymbolStyle.**GetWeldingSymbolsFilter**( ***WeldingSymbolFilter***() As [WeldSymbolTypeEnum](../WeldSymbolTypeEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| WeldingSymbolFilter | [WeldSymbolTypeEnum](../WeldSymbolTypeEnum.md) | Output an array of WeldSymbolTypeEnum indicating available welding symbols. |

## Version

Introduced in version 2024
