# OrdinateDimensionSet.OriginArrowheadType Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Read-write property that gets and sets the arrowhead type for the origin member of the ordinate dimension set.

## Syntax

OrdinateDimensionSet.**OriginArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2012
