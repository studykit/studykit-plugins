# CentermarkStyle.OvershootDistance Property

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Gets and sets the distance that center mark extension lines extend beyond the edges of the features that they define.

## Syntax

CentermarkStyle.**OvershootDistance**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
