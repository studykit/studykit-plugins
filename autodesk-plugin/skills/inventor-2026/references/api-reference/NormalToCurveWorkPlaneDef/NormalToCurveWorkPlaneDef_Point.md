# NormalToCurveWorkPlaneDef.Point Property

Parent Object: [NormalToCurveWorkPlaneDef](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef.md)

## Description

Property that returns the point of a normal to curve defined work plane. The point can be a WorkPoint, Vertex, SketchPoint3D or SketchPoint object. The work plane passes through the point.

## Syntax

NormalToCurveWorkPlaneDef.**Point**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
