# HelicalCurveDefinition.CurveStart Property

Parent Object: [HelicalCurveDefinition](../HelicalCurveDefinition/HelicalCurveDefinition.md)

## Description

Gets and sets the start point of the helical curve.

## Syntax

HelicalCurveDefinition.**CurveStart**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2019
