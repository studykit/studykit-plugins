# ModelLeaderNoteProxy.NativeObject Property

Parent Object: [ModelLeaderNoteProxy](../ModelLeaderNoteProxy/ModelLeaderNoteProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ModelLeaderNoteProxy.**NativeObject**() As [ModelLeaderNote](../ModelLeaderNote/ModelLeaderNote.md)

## Property Value

This is a read only property whose value is a [ModelLeaderNote](../ModelLeaderNote/ModelLeaderNote.md).

## Version

Introduced in version 2018
