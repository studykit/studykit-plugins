# UnfoldMethod.StyleType Property

Parent Object: [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Description

Gets the type of the style.

## Syntax

UnfoldMethod.**StyleType**() As [StyleTypeEnum](../StyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [StyleTypeEnum](../StyleTypeEnum.md).

## Version

Introduced in version 2009
