# UnfoldMethod.EquationType Property

Parent Object: [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Description

Property that specifies the type of equation defined for the unfold method. This property is only valid in the case where the UnfoldMethodType property returns kCustomEquationUnfoldMethod. In all other cases the value of this property should be ignored.

## Syntax

UnfoldMethod.**EquationType**() As [UnfoldMethodEquationTypeEnum](../UnfoldMethodEquationTypeEnum.md)

## Property Value

This is a read only property whose value is a [UnfoldMethodEquationTypeEnum](../UnfoldMethodEquationTypeEnum.md).

## Version

Introduced in version 2010
