# VirtualComponentDefinition.RangeBox Property

Parent Object: [VirtualComponentDefinition](../VirtualComponentDefinition/VirtualComponentDefinition.md)

## Description

Property that returns a Box object which contains the opposing points of a rectangular box that is guaranteed to enclose this object.

## Syntax

VirtualComponentDefinition.**RangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 11
