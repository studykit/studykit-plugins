# TableStyle.InsideLineColor Property

Parent Object: [TableStyle](../TableStyle/TableStyle.md)

## Description

Gets and sets the color of the inner lines of the table.

## Syntax

TableStyle.**InsideLineColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2011
