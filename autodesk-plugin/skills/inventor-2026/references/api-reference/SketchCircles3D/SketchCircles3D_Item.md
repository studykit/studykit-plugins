# SketchCircles3D.Item Property

Parent Object: [SketchCircles3D](../SketchCircles3D/SketchCircles3D.md)

## Description

Returns the specified SketchCircle3D object from the collection.

## Syntax

SketchCircles3D.**Item**( ***Index*** As Long ) As [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Property Value

This is a read only property whose value is a [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
