# BSplineSurface.Copy Method

Parent Object: [BSplineSurface](../BSplineSurface/BSplineSurface.md)

## Description

Creates a copy of this BSplineSurface object. The result is entirely independent and can be edited without affecting the original BSplineSurface object.

## Syntax

BSplineSurface.**Copy**() As [BSplineSurface](../BSplineSurface/BSplineSurface.md)

## Version

Introduced in version 2013
