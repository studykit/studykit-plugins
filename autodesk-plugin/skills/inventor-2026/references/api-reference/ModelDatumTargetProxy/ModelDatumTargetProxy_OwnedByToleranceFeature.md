# ModelDatumTargetProxy.OwnedByToleranceFeature Property

Parent Object: [ModelDatumTargetProxy](../ModelDatumTargetProxy/ModelDatumTargetProxy.md)

## Description

Returns the owning ModelToleranceFeature object.

## Syntax

ModelDatumTargetProxy.**OwnedByToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2023
