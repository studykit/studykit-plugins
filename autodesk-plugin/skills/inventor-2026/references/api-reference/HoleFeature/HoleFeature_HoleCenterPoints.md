# HoleFeature.HoleCenterPoints Property

Parent Object: [HoleFeature](../HoleFeature/HoleFeature.md)

## Description

Property that returns the set of points that define the centers of the holes of this feature.

## Syntax

HoleFeature.**HoleCenterPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 5
