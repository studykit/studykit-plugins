# MiniToolbarButton.Visible Property

Parent Object: [MiniToolbarButton](../MiniToolbarButton/MiniToolbarButton.md)

## Description

Read-write property that gets and sets whether the control is visible.

## Syntax

MiniToolbarButton.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
