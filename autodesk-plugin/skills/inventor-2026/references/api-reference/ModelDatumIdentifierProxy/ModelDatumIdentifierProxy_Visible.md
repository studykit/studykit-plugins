# ModelDatumIdentifierProxy.Visible Property

Parent Object: [ModelDatumIdentifierProxy](../ModelDatumIdentifierProxy/ModelDatumIdentifierProxy.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelDatumIdentifierProxy.**Visible**() As Boolean

This is a read/write property whose value is a Boolean.
