# GeneralDimension.SecondArrowheadType Property

Parent Object: [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Description

Read-write property that gets and sets the type of the second arrowhead.

## Syntax

GeneralDimension.**SecondArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2011
