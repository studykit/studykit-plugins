# GeneralOptions.UpdatePropertiesOnSaveForFileType Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Gets and sets whether properties should be updated when part or assembly files are saved.

## Syntax

GeneralOptions.**UpdatePropertiesOnSaveForFileType**() As [UpdatePropertiesOnSaveForFileTypeEnum](../UpdatePropertiesOnSaveForFileTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UpdatePropertiesOnSaveForFileTypeEnum](../UpdatePropertiesOnSaveForFileTypeEnum.md).

## Version

Introduced in version 2008
