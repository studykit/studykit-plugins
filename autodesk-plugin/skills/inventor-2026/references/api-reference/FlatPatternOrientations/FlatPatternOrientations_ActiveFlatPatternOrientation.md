# FlatPatternOrientations.ActiveFlatPatternOrientation Property

Parent Object: [FlatPatternOrientations](../FlatPatternOrientations/FlatPatternOrientations.md)

## Description

Read-only property that returns the active FlatPatternOrientation object.

## Syntax

FlatPatternOrientations.**ActiveFlatPatternOrientation**() As [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Property Value

This is a read only property whose value is a [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md).

## Version

Introduced in version 2015
