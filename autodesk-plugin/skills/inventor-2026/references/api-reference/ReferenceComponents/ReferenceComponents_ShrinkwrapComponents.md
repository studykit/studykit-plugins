# ReferenceComponents.ShrinkwrapComponents Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Gets the ShrinkwrapComponents collection object.

## Syntax

ReferenceComponents.**ShrinkwrapComponents**() As [ShrinkwrapComponents](../ShrinkwrapComponents/ShrinkwrapComponents.md)

## Property Value

This is a read only property whose value is a [ShrinkwrapComponents](../ShrinkwrapComponents/ShrinkwrapComponents.md).

## Version

Introduced in version 2018
