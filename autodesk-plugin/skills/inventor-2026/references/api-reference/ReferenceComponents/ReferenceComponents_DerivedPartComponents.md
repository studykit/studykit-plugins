# ReferenceComponents.DerivedPartComponents Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Property that returns the DerivedPartComponents collection object.

## Syntax

ReferenceComponents.**DerivedPartComponents**() As [DerivedPartComponents](../DerivedPartComponents/DerivedPartComponents.md)

## Property Value

This is a read only property whose value is a [DerivedPartComponents](../DerivedPartComponents/DerivedPartComponents.md).
