# ReplaceFaceDefinition.Parent Property

Parent Object: [ReplaceFaceDefinition](../ReplaceFaceDefinition/ReplaceFaceDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ReplaceFaceDefinition.**Parent**() As [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md)

## Property Value

This is a read only property whose value is a [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md).

## Version

Introduced in version 2024.1
