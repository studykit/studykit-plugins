# SketchEquationCurveProxy.LineWeight Property

Parent Object: [SketchEquationCurveProxy](../SketchEquationCurveProxy/SketchEquationCurveProxy.md)

## Description

Gets and sets the LineWeight applied to this sketch spline.

## Syntax

SketchEquationCurveProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
