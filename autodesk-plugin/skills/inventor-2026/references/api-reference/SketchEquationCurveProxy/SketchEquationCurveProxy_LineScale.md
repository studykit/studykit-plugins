# SketchEquationCurveProxy.LineScale Property

Parent Object: [SketchEquationCurveProxy](../SketchEquationCurveProxy/SketchEquationCurveProxy.md)

## Description

Gets and sets the LineScale applied to this sketch spline.

## Syntax

SketchEquationCurveProxy.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
