# ViewTabGroupsEnumerator.Item Property

Parent Object: [ViewTabGroupsEnumerator](../ViewTabGroupsEnumerator/ViewTabGroupsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

ViewTabGroupsEnumerator.**Item**( ***Index*** As Long ) As [ViewTabGroup](../ViewTabGroup/ViewTabGroup.md)

## Property Value

This is a read only property whose value is a [ViewTabGroup](../ViewTabGroup/ViewTabGroup.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the ViewTabGroup to return. |

## Version

Introduced in version 2022
