# MouseButtonEnum Enumerator

## Description

Enumeration of the buttons on a mouse.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kLeftMouseButton | 1 | Left mouse button. |
| kMiddleMouseButton | 4 | Middle mouse button. |
| kNoneMouseButton | 0 | No mouse button. |
| kRightMouseButton | 2 | Right mouse button. |

## Version

Introduced in version 5
