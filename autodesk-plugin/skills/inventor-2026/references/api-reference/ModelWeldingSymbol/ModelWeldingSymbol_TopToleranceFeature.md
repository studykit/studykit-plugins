# ModelWeldingSymbol.TopToleranceFeature Property

Parent Object: [ModelWeldingSymbol](../ModelWeldingSymbol/ModelWeldingSymbol.md)

## Description

Returns the top ModelToleranceFeature object.

## Syntax

ModelWeldingSymbol.**TopToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2024
