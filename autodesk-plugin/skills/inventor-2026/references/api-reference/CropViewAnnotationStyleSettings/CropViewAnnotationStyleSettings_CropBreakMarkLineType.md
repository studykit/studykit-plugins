# CropViewAnnotationStyleSettings.CropBreakMarkLineType Property

Parent Object: [CropViewAnnotationStyleSettings](../CropViewAnnotationStyleSettings/CropViewAnnotationStyleSettings.md)

## Description

Read-write property that gets and sets break mark line type for crop view.

## Syntax

CropViewAnnotationStyleSettings.**CropBreakMarkLineType**() As [CropViewBreakMarkLineTypeEnum](../CropViewBreakMarkLineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [CropViewBreakMarkLineTypeEnum](../CropViewBreakMarkLineTypeEnum.md).

## Version

Introduced in version 2025.1
