# CropViewAnnotationStyleSettings.DistanceBetweenSymbols Property

Parent Object: [CropViewAnnotationStyleSettings](../CropViewAnnotationStyleSettings/CropViewAnnotationStyleSettings.md)

## Description

Read-write property that gets and sets the distance between symbols.

## Syntax

CropViewAnnotationStyleSettings.**DistanceBetweenSymbols**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2025.1
