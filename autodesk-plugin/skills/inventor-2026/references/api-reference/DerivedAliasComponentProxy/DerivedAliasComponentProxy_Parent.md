# DerivedAliasComponentProxy.Parent Property

Parent Object: [DerivedAliasComponentProxy](../DerivedAliasComponentProxy/DerivedAliasComponentProxy.md)

## Description

Property that returns the parent object.

## Syntax

DerivedAliasComponentProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
