# DerivedAliasComponentProxy.Name Property

Parent Object: [DerivedAliasComponentProxy](../DerivedAliasComponentProxy/DerivedAliasComponentProxy.md)

## Description

Property that returns the component's name.

## Syntax

DerivedAliasComponentProxy.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2010
