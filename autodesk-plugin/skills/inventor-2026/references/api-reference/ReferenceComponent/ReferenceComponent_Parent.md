# ReferenceComponent.Parent Property

Parent Object: [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md)

## Description

Property that returns the parent object.

## Syntax

ReferenceComponent.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 5.3
