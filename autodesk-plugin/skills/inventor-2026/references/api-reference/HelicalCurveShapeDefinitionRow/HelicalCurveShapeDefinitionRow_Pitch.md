# HelicalCurveShapeDefinitionRow.Pitch Property

Parent Object: [HelicalCurveShapeDefinitionRow](../HelicalCurveShapeDefinitionRow/HelicalCurveShapeDefinitionRow.md)

## Description

Gets and sets the pitch of the helical curve.

## Syntax

HelicalCurveShapeDefinitionRow.**Pitch**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2019
