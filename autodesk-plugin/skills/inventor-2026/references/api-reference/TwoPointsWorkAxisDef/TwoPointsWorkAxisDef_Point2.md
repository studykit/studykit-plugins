# TwoPointsWorkAxisDef.Point2 Property

Parent Object: [TwoPointsWorkAxisDef](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef.md)

## Description

Property that returns the second point of a two point defined work axis. This object can be a WorkPoint, Vertex, or SketchPoint object. The work axis runs between the two points.

## Syntax

TwoPointsWorkAxisDef.**Point2**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
