# TwoPointsWorkAxisDef Object

## Description

Object that allows you to get and set the information that specifies a work axis along a line.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef_GetData.md) | Method that gets all of the data defining a work axis between two points. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef_Parent.md) | Property returning the parent WorkAxis object. |
| [Point1](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef_Point1.md) | Property that returns the first point of a two point defined work axis. This object can be a WorkPoint, Vertex, or SketchPoint object. The work axis runs between the two points. |
| [Point2](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef_Point2.md) | Property that returns the second point of a two point defined work axis. This object can be a WorkPoint, Vertex, or SketchPoint object. The work axis runs between the two points. |
| [Type](../TwoPointsWorkAxisDef/TwoPointsWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
