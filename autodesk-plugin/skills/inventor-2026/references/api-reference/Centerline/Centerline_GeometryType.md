# Centerline.GeometryType Property

Parent Object: [Centerline](../Centerline/Centerline.md)

## Description

Property that returns the type of the curve used to define the centerline. This property allows you to determine what type of object will be returned by the Geometry.

## Syntax

Centerline.**GeometryType**() As [CurveTypeEnum](../CurveTypeEnum.md)

## Property Value

This is a read only property whose value is a [CurveTypeEnum](../CurveTypeEnum.md).

## Version

Introduced in version 2009
