# RotateRotateConstraintProxy.Name Property

Parent Object: [RotateRotateConstraintProxy](../RotateRotateConstraintProxy/RotateRotateConstraintProxy.md)

## Description

Gets/Sets the displayable name of this constraint.

## Syntax

RotateRotateConstraintProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
