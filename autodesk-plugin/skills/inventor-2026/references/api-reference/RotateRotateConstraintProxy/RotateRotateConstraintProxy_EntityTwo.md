# RotateRotateConstraintProxy.EntityTwo Property

Parent Object: [RotateRotateConstraintProxy](../RotateRotateConstraintProxy/RotateRotateConstraintProxy.md)

## Description

Property that indicates the second of the geometric entities (Face, Axis, Edge, etc.) being constrained.

## Syntax

RotateRotateConstraintProxy.**EntityTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
