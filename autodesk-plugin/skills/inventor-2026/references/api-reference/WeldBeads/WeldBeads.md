# WeldBeads Object

## Description

The WeldBeads object provides access to all the existing  objects within a weldment.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../WeldBeads/WeldBeads_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../WeldBeads/WeldBeads_Count.md) | Property that returns the number of items in the collection. |
| [Item](../WeldBeads/WeldBeads_Item.md) | Returns the specified WeldBead object from the collection |
| [Type](../WeldBeads/WeldBeads_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Welds.WeldBeads](../Welds/Welds_WeldBeads.md)

## Version

Introduced in version 8
