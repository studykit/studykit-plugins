# MarkDefinition.AddGeometrySet Method

Parent Object: [MarkDefinition](../MarkDefinition/MarkDefinition.md)

## Description

Method that creates a new mark geometry set.

## Syntax

MarkDefinition.**AddGeometrySet**( ***Geometries*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md), ***pStyle*** As [MarkStyle](../MarkStyle/MarkStyle.md) ) As [MarkGeometrySet](../MarkGeometrySet/MarkGeometrySet.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Geometries | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection object that specifies the sketch geometries to create mark feature. |
| pStyle | [MarkStyle](../MarkStyle/MarkStyle.md) | Input MarkStyle object that specifies mark style. |

## Version

Introduced in version 2023
