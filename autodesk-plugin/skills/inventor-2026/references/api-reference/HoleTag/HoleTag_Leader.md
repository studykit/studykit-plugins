# HoleTag.Leader Property

Parent Object: [HoleTag](../HoleTag/HoleTag.md)

## Description

Property that returns the Leader object associated with the hole tag.

## Syntax

HoleTag.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 2009
