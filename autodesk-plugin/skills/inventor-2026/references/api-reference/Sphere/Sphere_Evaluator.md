# Sphere.Evaluator Property

Parent Object: [Sphere](../Sphere/Sphere.md)

## Description

Gets the surface evaluator for this sphere.

## Syntax

Sphere.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
