# Sphere.CenterPoint Property

Parent Object: [Sphere](../Sphere/Sphere.md)

## Description

Specifies the center of the sphere.

## Syntax

Sphere.**CenterPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
