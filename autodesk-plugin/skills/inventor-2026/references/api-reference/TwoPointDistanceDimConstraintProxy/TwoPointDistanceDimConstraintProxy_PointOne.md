# TwoPointDistanceDimConstraintProxy.PointOne Property

Parent Object: [TwoPointDistanceDimConstraintProxy](../TwoPointDistanceDimConstraintProxy/TwoPointDistanceDimConstraintProxy.md)

## Description

Property that returns the first sketch point being constrained.

## Syntax

TwoPointDistanceDimConstraintProxy.**PointOne**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 6
