# ContentFamily.FamilyType Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that returns the constant that indicates what type of family this family is; kContentPartFamily or kContentFeatureFamily.

## Syntax

ContentFamily.**FamilyType**() As [ContentFamilyTypeEnum](../ContentFamilyTypeEnum.md)

## Property Value

This is a read only property whose value is a [ContentFamilyTypeEnum](../ContentFamilyTypeEnum.md).

## Version

Introduced in version 2011
