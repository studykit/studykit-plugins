# ContentFamily.ContentIdentifier Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that returns the identifier for this object. This can be used with the GetContentObject method of the ContentCenter object to obtain this object at a later time.

## Syntax

ContentFamily.**ContentIdentifier**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2011
