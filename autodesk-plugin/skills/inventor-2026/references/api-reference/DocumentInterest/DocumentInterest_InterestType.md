# DocumentInterest.InterestType Property

Parent Object: [DocumentInterest](../DocumentInterest/DocumentInterest.md)

## Description

Gets/Sets the type of interest that the client has in this document.

## Syntax

DocumentInterest.**InterestType**() As [DocumentInterestTypeEnum](../DocumentInterestTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DocumentInterestTypeEnum](../DocumentInterestTypeEnum.md).

## Version

Introduced in version 2008
