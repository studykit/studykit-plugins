# iFeatureProxy.iFeatureDefinition Property

Parent Object: [iFeatureProxy](../iFeatureProxy/iFeatureProxy.md)

## Description

Property that returns the associated iFeatureDefinition.

## Syntax

iFeatureProxy.**iFeatureDefinition**() As [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md)

## Property Value

This is a read only property whose value is an [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md).

## Version

Introduced in version 2009
