# iFeatureProxy.Sketches Property

Parent Object: [iFeatureProxy](../iFeatureProxy/iFeatureProxy.md)

## Description

Property that returns the sketches that were created when the iFeature was placed. In most case there will be a single sketch.

## Syntax

iFeatureProxy.**Sketches**() As [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md).

## Version

Introduced in version 2009
