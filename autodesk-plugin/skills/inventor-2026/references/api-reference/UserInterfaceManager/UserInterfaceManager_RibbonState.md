# UserInterfaceManager.RibbonState Property

Parent Object: [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Description

Gets and sets the display state of the ribbon. Valid values are kFullRibbon, kMinimizeToTabs, kMinimizeToPanelTitles and kMinimizeToPanelButtons.

## Syntax

UserInterfaceManager.**RibbonState**() As [RibbonStateEnum](../RibbonStateEnum.md)

## Property Value

This is a read/write property whose value is a [RibbonStateEnum](../RibbonStateEnum.md).

## Version

Introduced in version 2010
