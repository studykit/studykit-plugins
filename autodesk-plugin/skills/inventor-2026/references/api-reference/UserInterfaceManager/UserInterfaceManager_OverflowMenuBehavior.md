# UserInterfaceManager.OverflowMenuBehavior Property

Parent Object: [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Description

Read-write property that gets and sets the behavior of the overflow menu.

## Syntax

UserInterfaceManager.**OverflowMenuBehavior**() As [OverflowMenuBehaviorEnum](../OverflowMenuBehaviorEnum.md)

## Property Value

This is a read/write property whose value is an [OverflowMenuBehaviorEnum](../OverflowMenuBehaviorEnum.md).

## Version

Introduced in version 2013
