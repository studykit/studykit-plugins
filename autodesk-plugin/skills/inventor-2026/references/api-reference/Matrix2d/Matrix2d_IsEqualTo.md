# Matrix2d.IsEqualTo Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Compares this matrix2d for equality with the specified matrix2d

## Syntax

Matrix2d.**IsEqualTo**( ***Matrix2d*** As [Matrix2d](../Matrix2d/Matrix2d.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix2d | [Matrix2d](../Matrix2d/Matrix2d.md) | Matrix2s to compare to. |
| Tolerance | Double | Tolerance value for comparison. |

## Version

Introduced in version 11
