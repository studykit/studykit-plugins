# ProfileEntityProxy.SketchEntity Property

Parent Object: [ProfileEntityProxy](../ProfileEntityProxy/ProfileEntityProxy.md)

## Description

Property that returns the sketch entity this profile entity was derived from.

## Syntax

ProfileEntityProxy.**SketchEntity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
