# ImportedGenericComponentProxy.Type Property

Parent Object: [ImportedGenericComponentProxy](../ImportedGenericComponentProxy/ImportedGenericComponentProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ImportedGenericComponentProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2016
