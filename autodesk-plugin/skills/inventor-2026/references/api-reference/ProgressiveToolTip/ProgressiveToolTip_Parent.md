# ProgressiveToolTip.Parent Property

Parent Object: [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md)

## Description

Property that returns the parent of the ControlDefinition object.

## Syntax

ProgressiveToolTip.**Parent**() As [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Property Value

This is a read only property whose value is a [ControlDefinition](../ControlDefinition/ControlDefinition.md).

## Version

Introduced in version 2011
