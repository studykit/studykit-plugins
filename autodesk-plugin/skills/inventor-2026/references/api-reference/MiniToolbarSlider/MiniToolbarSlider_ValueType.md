# MiniToolbarSlider.ValueType Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets if the slider will handle integer or decimal numbers. Valid values are kDoubleType or kIntegerType.

## Syntax

MiniToolbarSlider.**ValueType**() As [ValueTypeEnum](../ValueTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ValueTypeEnum](../ValueTypeEnum.md).

## Version

Introduced in version 2013
