# MiniToolbarSlider.LargeIcon Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets the large icon for the control.

## Syntax

MiniToolbarSlider.**LargeIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2013
