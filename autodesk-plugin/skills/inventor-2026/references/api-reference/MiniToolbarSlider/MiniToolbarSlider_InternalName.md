# MiniToolbarSlider.InternalName Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-only property that returns a string uniquely identifying this control within the toolbar.

## Syntax

MiniToolbarSlider.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2013
