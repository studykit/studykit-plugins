# DSJoint.JointType Property

Parent Object: [DSJoint](../DSJoint/DSJoint.md)

## Description

Returns the type of simulation joint this object represents

## Syntax

DSJoint.**JointType**() As [DSJointTypeEnum](../DSJointTypeEnum.md)

## Property Value

This is a read only property whose value is a [DSJointTypeEnum](../DSJointTypeEnum.md).

## Version

Introduced in version 2013
