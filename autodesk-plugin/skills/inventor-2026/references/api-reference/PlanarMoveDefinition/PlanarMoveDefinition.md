# PlanarMoveDefinition Object

## Description

The PlanarMoveDefinition object defines all of the inputs that are unique to a move face feature defined using two planar points.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PlanarMoveDefinition/PlanarMoveDefinition_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../PlanarMoveDefinition/PlanarMoveDefinition_Parent.md) | Property that returns the parent MoveFaceDefinition object. |
| [Plane](../PlanarMoveDefinition/PlanarMoveDefinition_Plane.md) | Gets and sets the plane for the planar move. |
| [PointOne](../PlanarMoveDefinition/PlanarMoveDefinition_PointOne.md) | Gets and sets the first point for the planar move. |
| [PointTwo](../PlanarMoveDefinition/PlanarMoveDefinition_PointTwo.md) | Gets and sets the second point for the planar move. |
| [Type](../PlanarMoveDefinition/PlanarMoveDefinition_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2011
