# EdgeSymbolDefinition.HorizontalValueLower Property

Parent Object: [EdgeSymbolDefinition](../EdgeSymbolDefinition/EdgeSymbolDefinition.md)

## Description

Read-write property that gets and sets the horizontal value lower of the EdgeSymbol.

## Syntax

EdgeSymbolDefinition.**HorizontalValueLower**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2024
