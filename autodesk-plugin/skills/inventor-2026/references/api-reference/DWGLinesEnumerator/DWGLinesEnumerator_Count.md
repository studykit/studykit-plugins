# DWGLinesEnumerator.Count Property

Parent Object: [DWGLinesEnumerator](../DWGLinesEnumerator/DWGLinesEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGLinesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
