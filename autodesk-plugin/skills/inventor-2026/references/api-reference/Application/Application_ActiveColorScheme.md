# Application.ActiveColorScheme Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the ColorScheme that is currently active.

## Syntax

Application.**ActiveColorScheme**() As [ColorScheme](../ColorScheme/ColorScheme.md)

## Property Value

This is a read only property whose value is a [ColorScheme](../ColorScheme/ColorScheme.md).

## Version

Introduced in version 9
