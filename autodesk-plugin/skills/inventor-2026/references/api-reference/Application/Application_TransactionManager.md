# Application.TransactionManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the TransactionManager object that controls all the database transactions taking place in the system.

## Syntax

Application.**TransactionManager**() As [TransactionManager](../TransactionManager/TransactionManager.md)

## Property Value

This is a read only property whose value is a [TransactionManager](../TransactionManager/TransactionManager.md).
