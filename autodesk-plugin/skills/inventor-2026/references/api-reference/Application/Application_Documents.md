# Application.Documents Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets all the in-memory documents in a collection.

## Syntax

Application.**Documents**() As [Documents](../Documents/Documents.md)

## Property Value

This is a read only property whose value is a [Documents](../Documents/Documents.md).
