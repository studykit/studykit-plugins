# Application.iFeatureOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the iFeatureOptions object. The iFeatureOptions object provides access to various iFeature related application level options. This is somewhat equivalent to the iFeature tab of the Application Options dialog.

## Syntax

Application.**iFeatureOptions**() As [iFeatureOptions](../iFeatureOptions/iFeatureOptions.md)

## Property Value

This is a read only property whose value is an [iFeatureOptions](../iFeatureOptions/iFeatureOptions.md).

## Version

Introduced in version 11
