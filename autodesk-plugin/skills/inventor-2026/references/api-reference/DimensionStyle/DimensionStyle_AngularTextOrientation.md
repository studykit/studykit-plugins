# DimensionStyle.AngularTextOrientation Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the orientation of angular dimension text.

## Syntax

DimensionStyle.**AngularTextOrientation**() As [DimensionTextOrientationEnum](../DimensionTextOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionTextOrientationEnum](../DimensionTextOrientationEnum.md).

## Version

Introduced in version 2014
