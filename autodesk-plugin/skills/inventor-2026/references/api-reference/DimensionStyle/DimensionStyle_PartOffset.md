# DimensionStyle.PartOffset Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the distance from the referenced part edge to a parallel dimension line.

## Syntax

DimensionStyle.**PartOffset**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
