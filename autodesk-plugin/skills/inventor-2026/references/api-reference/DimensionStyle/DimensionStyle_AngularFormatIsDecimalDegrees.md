# DimensionStyle.AngularFormatIsDecimalDegrees Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display angular dimension values as decimal degrees or degrees-minutes-seconds.

## Syntax

DimensionStyle.**AngularFormatIsDecimalDegrees**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
