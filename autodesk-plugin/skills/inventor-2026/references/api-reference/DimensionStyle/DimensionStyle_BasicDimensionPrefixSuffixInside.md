# DimensionStyle.BasicDimensionPrefixSuffixInside Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether the prefix and suffix are inside or outside the basic dimension box.

## Syntax

DimensionStyle.**BasicDimensionPrefixSuffixInside**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
