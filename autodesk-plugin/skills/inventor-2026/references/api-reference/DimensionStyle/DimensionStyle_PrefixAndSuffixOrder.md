# DimensionStyle.PrefixAndSuffixOrder Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the location of prefix and suffix.

## Syntax

DimensionStyle.**PrefixAndSuffixOrder**() As [PrefixAndSuffixOrderEnum](../PrefixAndSuffixOrderEnum.md)

## Property Value

This is a read/write property whose value is a [PrefixAndSuffixOrderEnum](../PrefixAndSuffixOrderEnum.md).

## Version

Introduced in version 9
