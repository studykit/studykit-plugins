# DimensionStyle.PunchNoteDualFormat Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the arrangement of the alternate dimension display for punch notes.

## Syntax

DimensionStyle.**PunchNoteDualFormat**() As [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md).

## Version

Introduced in version 2014
