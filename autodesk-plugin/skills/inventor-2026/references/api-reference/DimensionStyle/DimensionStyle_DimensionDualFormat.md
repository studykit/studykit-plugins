# DimensionStyle.DimensionDualFormat Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Read-write property that gets and sets the arrangement of the alternate dimension display for dimensions.

## Syntax

DimensionStyle.**DimensionDualFormat**() As [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionStyleFormatEnum](../DimensionStyleFormatEnum.md).

## Version

Introduced in version 2014
