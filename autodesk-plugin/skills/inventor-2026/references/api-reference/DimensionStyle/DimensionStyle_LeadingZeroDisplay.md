# DimensionStyle.LeadingZeroDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the leading zeroes for dimensions in the drawing.

## Syntax

DimensionStyle.**LeadingZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
