# ImportedModelEntity.Parent Property

Parent Object: [ImportedModelEntity](../ImportedModelEntity/ImportedModelEntity.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ImportedModelEntity.**Parent**() As [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md)

## Property Value

This is a read only property whose value is an [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md).

## Version

Introduced in version 2016
