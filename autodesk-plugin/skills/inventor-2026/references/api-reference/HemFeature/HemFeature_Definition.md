# HemFeature.Definition Property

Parent Object: [HemFeature](../HemFeature/HemFeature.md)

## Description

Property that gets the HemDefinition object associated with this fold feature.

## Syntax

HemFeature.**Definition**() As [HemDefinition](../HemDefinition/HemDefinition.md)

## Property Value

This is a read only property whose value is a [HemDefinition](../HemDefinition/HemDefinition.md).

## Version

Introduced in version 2009
