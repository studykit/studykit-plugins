# MeshFace.Face Property

Parent Object: [MeshFace](../MeshFace/MeshFace.md)

## Description

Read-only property that returns the transient Brep Face for the MeshFace.

## Syntax

MeshFace.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2017
