# PresentationScene.ActiveModelState Property

Parent Object: [PresentationScene](../PresentationScene/PresentationScene.md)

## Description

Read-write property that gets and sets the name of active model state used for this presentation scene.

## Syntax

PresentationScene.**ActiveModelState**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025.2
