# PresentationScene.TopSceneComponent Property

Parent Object: [PresentationScene](../PresentationScene/PresentationScene.md)

## Description

Read-only property that returns the top PresentationComponent object in the presentation scene.

## Syntax

PresentationScene.**TopSceneComponent**() As [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Property Value

This is a read only property whose value is a [PresentationComponent](../PresentationComponent/PresentationComponent.md).

## Version

Introduced in version 2018
