# ModelParameter.DrivenBy Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Method that returns the collection objects that the parameter is dependent on.

## Syntax

ModelParameter.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
