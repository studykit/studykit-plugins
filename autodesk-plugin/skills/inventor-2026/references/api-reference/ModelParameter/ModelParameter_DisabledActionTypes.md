# ModelParameter.DisabledActionTypes Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Gets and sets the action types valid for this parameter.

## Syntax

ModelParameter.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 9
