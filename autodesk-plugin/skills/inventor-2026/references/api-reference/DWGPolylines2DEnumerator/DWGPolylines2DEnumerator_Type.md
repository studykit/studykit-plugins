# DWGPolylines2DEnumerator.Type Property

Parent Object: [DWGPolylines2DEnumerator](../DWGPolylines2DEnumerator/DWGPolylines2DEnumerator.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

DWGPolylines2DEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2016
