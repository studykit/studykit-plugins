# AnnotationPlaneDefinition.Type Property

Parent Object: [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Description

Read-only property returning kAnnotationPlaneDefinitionObject indicating this object’s type.

## Syntax

AnnotationPlaneDefinition.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2018
