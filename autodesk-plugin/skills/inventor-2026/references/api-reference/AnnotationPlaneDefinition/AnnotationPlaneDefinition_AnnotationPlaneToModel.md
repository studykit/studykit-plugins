# AnnotationPlaneDefinition.AnnotationPlaneToModel Method

Parent Object: [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Description

Method that takes a 2d coordinate in annotation plane space, and returns a Point3d containing the coordinates of the point in model space.

## Syntax

AnnotationPlaneDefinition.**AnnotationPlaneToModel**( ***AnnotationPlaneCoordinate*** As [Point2d](../Point2d/Point2d.md) ) As [Point](../Point/Point.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| AnnotationPlaneCoordinate | [Point2d](../Point2d/Point2d.md) | Input Point2d object defining the 2d coordinate in annotation plane space. |

## Version

Introduced in version 2018
