# SketchCircleProxy.CenterSketchPoint Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Property that gets the sketch point that defines the position of the center of the circle.

## Syntax

SketchCircleProxy.**CenterSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 6
