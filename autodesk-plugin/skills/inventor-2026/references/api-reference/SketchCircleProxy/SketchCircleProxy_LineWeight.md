# SketchCircleProxy.LineWeight Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Gets and sets the LineWeight applied to this sketch circle.

## Syntax

SketchCircleProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
