# SketchCircleProxy.NativeObject Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchCircleProxy.**NativeObject**() As [SketchCircle](../SketchCircle/SketchCircle.md)

## Property Value

This is a read only property whose value is a [SketchCircle](../SketchCircle/SketchCircle.md).

## Version

Introduced in version 6
