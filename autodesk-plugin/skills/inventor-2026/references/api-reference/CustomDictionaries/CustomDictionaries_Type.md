# CustomDictionaries.Type Property

Parent Object: [CustomDictionaries](../CustomDictionaries/CustomDictionaries.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

CustomDictionaries.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2020
