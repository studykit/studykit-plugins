# FixedWorkPlaneDef.YAxis Property

Parent Object: [FixedWorkPlaneDef](../FixedWorkPlaneDef/FixedWorkPlaneDef.md)

## Description

Property that returns the Y-axis vector of the work plane.

## Syntax

FixedWorkPlaneDef.**YAxis**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
