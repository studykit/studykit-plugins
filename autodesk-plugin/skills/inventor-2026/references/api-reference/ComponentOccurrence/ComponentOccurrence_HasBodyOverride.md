# ComponentOccurrence.HasBodyOverride Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Property that returns a Boolean specifying whether the occurrence path contains body-modifying assembly features.

## Syntax

ComponentOccurrence.**HasBodyOverride**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
