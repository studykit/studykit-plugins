# ComponentOccurrence.IsAssociativeToDesignViewRepresentation Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Gets and sets whether this occurrence is associative to a design view representation.

## Syntax

ComponentOccurrence.**IsAssociativeToDesignViewRepresentation**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
