# View.DisplayMode Property

Parent Object: [View](../View/View.md)

## Description

Gets/Sets the Display Mode on this View's window.

## Syntax

View.**DisplayMode**() As [DisplayModeEnum](../DisplayModeEnum.md)

## Property Value

This is a read/write property whose value is a [DisplayModeEnum](../DisplayModeEnum.md).
