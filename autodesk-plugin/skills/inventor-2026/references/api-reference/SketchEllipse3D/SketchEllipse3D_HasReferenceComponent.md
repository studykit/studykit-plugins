# SketchEllipse3D.HasReferenceComponent Property

Parent Object: [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchEllipse3D.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
