# SimplifyDefinition.RemoveEmbossMaxHeightRange Property

Parent Object: [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md)

## Description

Read-write property that gets and sets the max height of embosses in centimeters to remove for the simplify feature. The embosses which height is less than or equal to this value will be removed. This is only applicable when the RemoveEmbossesStyle is set to k.

## Syntax

SimplifyDefinition.**RemoveEmbossMaxHeightRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2026
