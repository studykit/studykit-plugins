# iFeatureWorkPlaneInput.PlaneInput Property

Parent Object: [iFeatureWorkPlaneInput](../iFeatureWorkPlaneInput/iFeatureWorkPlaneInput.md)

## Description

Gets and sets the geometry matching the plane input.

## Syntax

iFeatureWorkPlaneInput.**PlaneInput**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 6
