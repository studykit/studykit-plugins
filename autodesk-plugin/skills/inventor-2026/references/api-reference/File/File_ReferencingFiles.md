# File.ReferencingFiles Property

Parent Object: [File](../File/File.md)

## Description

Property that returns all the files in memory that reference this file.

## Syntax

File.**ReferencingFiles**() As [FilesEnumerator](../FilesEnumerator/FilesEnumerator.md)

## Property Value

This is a read only property whose value is a [FilesEnumerator](../FilesEnumerator/FilesEnumerator.md).

## Version

Introduced in version 11
