# DistanceHeightExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The DistanceHeightExtent is used by the FlangeFeature object when the height extent type is defined as a distance extent.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DistanceHeightExtent/DistanceHeightExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Distance](../DistanceHeightExtent/DistanceHeightExtent_Distance.md) | Property that returns the parameter controlling the extent distance of the flange. |
| [FlangeDirection](../DistanceHeightExtent/DistanceHeightExtent_FlangeDirection.md) | Gets and sets the direction of the flange. |
| [HeightDatumType](../DistanceHeightExtent/DistanceHeightExtent_HeightDatumType.md) | Gets and set the how the height of the flange is defined when the height extent is a distance extent. |
| [Parent](../DistanceHeightExtent/DistanceHeightExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Type](../DistanceHeightExtent/DistanceHeightExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2009
