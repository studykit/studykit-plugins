# OrdinateDimension.JogPointTwo Property

Parent Object: [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Description

Read-write property that gets and sets the second jog point for the ordinate dimension.

## Syntax

OrdinateDimension.**JogPointTwo**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2009
