# TransitionSymbolStyle.ConvertToLocal Method

Parent Object: [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

TransitionSymbolStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2025.1
