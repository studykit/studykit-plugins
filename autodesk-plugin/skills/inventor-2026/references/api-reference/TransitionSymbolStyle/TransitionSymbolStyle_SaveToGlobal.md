# TransitionSymbolStyle.SaveToGlobal Method

Parent Object: [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

TransitionSymbolStyle.**SaveToGlobal**()

## Version

Introduced in version 2025.1
