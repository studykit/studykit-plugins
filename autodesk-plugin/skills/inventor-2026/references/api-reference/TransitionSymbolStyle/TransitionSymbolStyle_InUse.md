# TransitionSymbolStyle.InUse Property

Parent Object: [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Description

Property that indicates if this style is in use.

## Syntax

TransitionSymbolStyle.**InUse**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025.1
