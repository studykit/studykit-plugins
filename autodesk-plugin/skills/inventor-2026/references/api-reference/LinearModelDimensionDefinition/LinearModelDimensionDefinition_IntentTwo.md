# LinearModelDimensionDefinition.IntentTwo Property

Parent Object: [LinearModelDimensionDefinition](../LinearModelDimensionDefinition/LinearModelDimensionDefinition.md)

## Description

Read-write property that gets and sets the second geometry associated with the dimension.

## Syntax

LinearModelDimensionDefinition.**IntentTwo**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2018
