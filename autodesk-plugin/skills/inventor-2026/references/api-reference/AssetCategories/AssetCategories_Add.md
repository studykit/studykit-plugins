# AssetCategories.Add Method

Parent Object: [AssetCategories](../AssetCategories/AssetCategories.md)

## Description

Method that creates a new asset category in the library.

## Syntax

AssetCategories.**Add**( ***DisplayName*** As String, ***Asset*** As [Asset](../Asset/Asset.md) ) As [AssetCategory](../AssetCategory/AssetCategory.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DisplayName | String | Displayed name of the category as shown to the user in the browser. |
| Asset | [Asset](../Asset/Asset.md) | An Asset that will be used as the first item in the category. The asset must exist in the same library or document as the category being created. |

## Version

Introduced in version 2014
