# AssetCategories.Item Property

Parent Object: [AssetCategories](../AssetCategories/AssetCategories.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

AssetCategories.**Item**( ***Index*** As Variant ) As [AssetCategory](../AssetCategory/AssetCategory.md)

## Property Value

This is a read only property whose value is an [AssetCategory](../AssetCategory/AssetCategory.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the category to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the category display name. |

## Version

Introduced in version 2014
