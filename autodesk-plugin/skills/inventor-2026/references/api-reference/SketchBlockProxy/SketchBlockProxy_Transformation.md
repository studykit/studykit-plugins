# SketchBlockProxy.Transformation Property

Parent Object: [SketchBlockProxy](../SketchBlockProxy/SketchBlockProxy.md)

## Description

Gets and sets the transformation matrix for the sketch block.

## Syntax

SketchBlockProxy.**Transformation**() As [Matrix2d](../Matrix2d/Matrix2d.md)

## Property Value

This is a read/write property whose value is a [Matrix2d](../Matrix2d/Matrix2d.md).

## Version

Introduced in version 2010
