# ProfileEntity3DProxy.NativeObject Property

Parent Object: [ProfileEntity3DProxy](../ProfileEntity3DProxy/ProfileEntity3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ProfileEntity3DProxy.**NativeObject**() As [ProfileEntity3D](../ProfileEntity3D/ProfileEntity3D.md)

## Property Value

This is a read only property whose value is a [ProfileEntity3D](../ProfileEntity3D/ProfileEntity3D.md).

## Version

Introduced in version 6
