# TextBoxConstraintProxy.TextBox Property

Parent Object: [TextBoxConstraintProxy](../TextBoxConstraintProxy/TextBoxConstraintProxy.md)

## Description

The text box that is constrained.

## Syntax

TextBoxConstraintProxy.**TextBox**() As [TextBox](../TextBox/TextBox.md)

## Property Value

This is a read only property whose value is a [TextBox](../TextBox/TextBox.md).

## Version

Introduced in version 11
