# SketchArcProxy.LineScale Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Gets and sets the lineScale applied to this sketch line.

## Syntax

SketchArcProxy.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
