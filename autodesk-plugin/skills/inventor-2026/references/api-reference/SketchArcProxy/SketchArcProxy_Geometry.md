# SketchArcProxy.Geometry Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Property that returns an geometry object. The object returned represents a 'snapshot' view of the current state of the sketch arc.

## Syntax

SketchArcProxy.**Geometry**() As [Arc2d](../Arc2d/Arc2d.md)

## Property Value

This is a read only property whose value is an [Arc2d](../Arc2d/Arc2d.md).

## Version

Introduced in version 6
