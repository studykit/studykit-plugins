# PluginLicenseManager.LicenseExpirationDate Property

Parent Object: [PluginLicenseManager](../PluginLicenseManager/PluginLicenseManager.md)

## Description

Gets the license expiration date.

## Syntax

PluginLicenseManager.**LicenseExpirationDate**() As Date

## Property Value

This is a read only property whose value is a Date.

## Version

Introduced in version 2020
