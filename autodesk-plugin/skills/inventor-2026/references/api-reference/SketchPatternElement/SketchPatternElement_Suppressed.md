# SketchPatternElement.Suppressed Property

Parent Object: [SketchPatternElement](../SketchPatternElement/SketchPatternElement.md)

## Description

Gets and sets whether this pattern element is suppressed or not.

## Syntax

SketchPatternElement.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
