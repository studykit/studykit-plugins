# PartFeature.SurfaceBodies Property

Parent Object: [PartFeature](../PartFeature/PartFeature.md)

## Description

Property that returns the bodies that this feature has created or modified.

## Syntax

PartFeature.**SurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).
