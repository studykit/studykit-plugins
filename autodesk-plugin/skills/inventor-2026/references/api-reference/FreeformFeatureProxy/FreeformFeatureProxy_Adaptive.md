# FreeformFeatureProxy.Adaptive Property

Parent Object: [FreeformFeatureProxy](../FreeformFeatureProxy/FreeformFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

FreeformFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
