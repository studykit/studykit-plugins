# DWGSplinesEnumerator.Count Property

Parent Object: [DWGSplinesEnumerator](../DWGSplinesEnumerator/DWGSplinesEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGSplinesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
