# RevisionCloudControlPoint.Index Property

Parent Object: [RevisionCloudControlPoint](../RevisionCloudControlPoint/RevisionCloudControlPoint.md)

## Description

Returns the index of the control point in the RevisionCloudControlPoint collection.

## Syntax

RevisionCloudControlPoint.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2024
