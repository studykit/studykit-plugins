# DSJointDefinition.IsSuppressed Property

Parent Object: [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md)

## Description

Gets and sets whether the joint is suppressed.

## Syntax

DSJointDefinition.**IsSuppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
