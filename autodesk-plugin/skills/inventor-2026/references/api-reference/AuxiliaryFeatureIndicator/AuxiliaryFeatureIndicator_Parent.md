# AuxiliaryFeatureIndicator.Parent Property

Parent Object: [AuxiliaryFeatureIndicator](../AuxiliaryFeatureIndicator/AuxiliaryFeatureIndicator.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

AuxiliaryFeatureIndicator.**Parent**() As [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Property Value

This is a read only property whose value is a [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md).

## Version

Introduced in version 2023
