# MiniToolbarDropdown.AutoHide Property

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Read-write property that gets and sets whether the control should be automatically hidden when the user moves the cursor away from the MiniToolbar.

## Syntax

MiniToolbarDropdown.**AutoHide**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
