# DynamicSimulation.LastComputedTimeStep Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Gets the last computed time step of the simulation.

## Syntax

DynamicSimulation.**LastComputedTimeStep**() As Long

## Property Value

This is a read only property whose value is a Long.
