# DynamicSimulation.SimulationLength Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Gets and sets the simulation length of the simulation in seconds.

## Syntax

DynamicSimulation.**SimulationLength**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2013
