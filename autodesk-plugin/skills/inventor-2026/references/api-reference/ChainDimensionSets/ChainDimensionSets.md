# ChainDimensionSets Object

## Description

The ChainDimensionSets collection object provides access to all of the chain dimension sets on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ChainDimensionSets/ChainDimensionSets_Add.md) | Method that creates a chain dimension set using the specified geometries. Returns the newly created ChainDimensionSets object. |
| [AddUsingBaseDimension](../ChainDimensionSets/ChainDimensionSets_AddUsingBaseDimension.md) | Method that creates a chain dimension set based on the specified base dimension and using specified additional geometry. Returns the newly created ChainDimensionSet object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ChainDimensionSets/ChainDimensionSets_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ChainDimensionSets/ChainDimensionSets_Count.md) | Property that returns the number of items in the collection. |
| [Item](../ChainDimensionSets/ChainDimensionSets_Item.md) | Method that returns the specified dimension set object from the collection. |
| [Type](../ChainDimensionSets/ChainDimensionSets_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
