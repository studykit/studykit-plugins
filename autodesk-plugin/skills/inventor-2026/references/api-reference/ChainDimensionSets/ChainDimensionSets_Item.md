# ChainDimensionSets.Item Property

Parent Object: [ChainDimensionSets](../ChainDimensionSets/ChainDimensionSets.md)

## Description

Method that returns the specified dimension set object from the collection.

## Syntax

ChainDimensionSets.**Item**( ***Index*** As Long ) As [ChainDimensionSet](../ChainDimensionSet/ChainDimensionSet.md)

## Property Value

This is a read only property whose value is a [ChainDimensionSet](../ChainDimensionSet/ChainDimensionSet.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2011
