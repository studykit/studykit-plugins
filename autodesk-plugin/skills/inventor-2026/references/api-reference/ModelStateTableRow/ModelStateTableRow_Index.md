# ModelStateTableRow.Index Property

Parent Object: [ModelStateTableRow](../ModelStateTableRow/ModelStateTableRow.md)

## Description

Gets the index of this row within the ModelStateTableRows collection where the first row has an index of 1.

## Syntax

ModelStateTableRow.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2022
