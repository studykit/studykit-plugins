# ModelStateTableRow.Parent Property

Parent Object: [ModelStateTableRow](../ModelStateTableRow/ModelStateTableRow.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelStateTableRow.**Parent**() As [ModelStateTable](../ModelStateTable/ModelStateTable.md)

## Property Value

This is a read only property whose value is a [ModelStateTable](../ModelStateTable/ModelStateTable.md).

## Version

Introduced in version 2022
