# InterferenceResult.InterferenceBody Property

Parent Object: [InterferenceResult](../InterferenceResult/InterferenceResult.md)

## Description

Gets the transient body representing the interference volume.

## Syntax

InterferenceResult.**InterferenceBody**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2012
