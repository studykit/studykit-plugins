# SketchBlock.ContainingSketchBlock Property

Parent Object: [SketchBlock](../SketchBlock/SketchBlock.md)

## Description

Property that returns the sketch block that in turn contains this sketch block. This is the same SketchBlock returned as the last item in the SketchBlockPath property. This property \returns Nothing if this sketch block does not belong to another sketch block and lives directly under a sketch.

## Syntax

SketchBlock.**ContainingSketchBlock**() As [SketchBlock](../SketchBlock/SketchBlock.md)

## Property Value

This is a read only property whose value is a [SketchBlock](../SketchBlock/SketchBlock.md).

## Version

Introduced in version 2010
