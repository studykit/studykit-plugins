# PresentationDocument.ReservedForWriteTime Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Gets the time at which the reservation was made.

## Syntax

PresentationDocument.**ReservedForWriteTime**() As Date

## Property Value

This is a read only property whose value is a Date.

## Version

Introduced in version 4
