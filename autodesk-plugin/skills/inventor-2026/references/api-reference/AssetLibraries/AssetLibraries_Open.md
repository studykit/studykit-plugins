# AssetLibraries.Open Method

Parent Object: [AssetLibraries](../AssetLibraries/AssetLibraries.md)

## Description

Method that open an existing asset library. The opened AssetLibrary object is returned.

## Syntax

AssetLibraries.**Open**( ***FullFileName*** As String ) As [AssetLibrary](../AssetLibrary/AssetLibrary.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | The full filename of the library to open. |

## Version

Introduced in version 2014
