# HoleTable.RowMergeType Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets the row merge option for the hole table.

## Syntax

HoleTable.**RowMergeType**() As [RowMergeTypeEnum](../RowMergeTypeEnum.md)

## Property Value

This is a read/write property whose value is a [RowMergeTypeEnum](../RowMergeTypeEnum.md).

## Version

Introduced in version 2009
