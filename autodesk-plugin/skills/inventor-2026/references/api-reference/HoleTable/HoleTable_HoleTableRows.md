# HoleTable.HoleTableRows Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Property that returns the HoleTableRows collection object.

## Syntax

HoleTable.**HoleTableRows**() As [HoleTableRows](../HoleTableRows/HoleTableRows.md)

## Property Value

This is a read only property whose value is a [HoleTableRows](../HoleTableRows/HoleTableRows.md).

## Version

Introduced in version 10
