# HoleTable.HoleTableType Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Property that returns the type of the hole table.

## Syntax

HoleTable.**HoleTableType**() As [HoleTableTypeEnum](../HoleTableTypeEnum.md)

## Property Value

This is a read only property whose value is a [HoleTableTypeEnum](../HoleTableTypeEnum.md).

## Version

Introduced in version 2009
