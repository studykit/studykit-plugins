# HoleTable.Layer Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets and sets the layer used by the HoleTable.

## Syntax

HoleTable.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
