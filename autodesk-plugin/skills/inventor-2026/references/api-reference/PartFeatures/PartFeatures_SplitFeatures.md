# PartFeatures.SplitFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the SplitFeatures collection object. This collection provides access to existing SplitFeature objects.

## Syntax

PartFeatures.**SplitFeatures**() As [SplitFeatures](../SplitFeatures/SplitFeatures.md)

## Property Value

This is a read only property whose value is a [SplitFeatures](../SplitFeatures/SplitFeatures.md).

## Version

Introduced in version 5
