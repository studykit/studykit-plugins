# PartFeatures.RuleFilletFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are RuleFilletFeatures.

## Syntax

PartFeatures.**RuleFilletFeatures**() As [RuleFilletFeatures](../RuleFilletFeatures/RuleFilletFeatures.md)

## Property Value

This is a read only property whose value is a [RuleFilletFeatures](../RuleFilletFeatures/RuleFilletFeatures.md).

## Version

Introduced in version 2010
