# PartFeatures.MoveFaceFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the MoveFaceFeatures collection object.

## Syntax

PartFeatures.**MoveFaceFeatures**() As [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md)

## Property Value

This is a read only property whose value is a [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md).

## Version

Introduced in version 10
