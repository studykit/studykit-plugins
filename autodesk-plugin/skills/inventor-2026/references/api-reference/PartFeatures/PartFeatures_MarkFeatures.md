# PartFeatures.MarkFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are MarkFeatures.

## Syntax

PartFeatures.**MarkFeatures**() As [MarkFeatures](../MarkFeatures/MarkFeatures.md)

## Property Value

This is a read only property whose value is a [MarkFeatures](../MarkFeatures/MarkFeatures.md).

## Version

Introduced in version 2023
