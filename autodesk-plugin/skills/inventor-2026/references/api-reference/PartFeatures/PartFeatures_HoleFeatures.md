# PartFeatures.HoleFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the HoleFeatures collection object. This collection provides access to existing HoleFeature objects and provides functionality to create new HoleFeature objects.

## Syntax

PartFeatures.**HoleFeatures**() As [HoleFeatures](../HoleFeatures/HoleFeatures.md)

## Property Value

This is a read only property whose value is a [HoleFeatures](../HoleFeatures/HoleFeatures.md).
