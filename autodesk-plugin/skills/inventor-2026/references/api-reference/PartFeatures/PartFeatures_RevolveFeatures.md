# PartFeatures.RevolveFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the RevolveFeatures collection object. This collection provides access to existing RevolveFeature objects and provides functionality to create new RevolveFeature objects.

## Syntax

PartFeatures.**RevolveFeatures**() As [RevolveFeatures](../RevolveFeatures/RevolveFeatures.md)

## Property Value

This is a read only property whose value is a [RevolveFeatures](../RevolveFeatures/RevolveFeatures.md).

## Version

Introduced in version 5
