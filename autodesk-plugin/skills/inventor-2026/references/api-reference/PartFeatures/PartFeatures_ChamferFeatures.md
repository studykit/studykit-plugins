# PartFeatures.ChamferFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the ChamferFeatures collection object. This collection provides access to existing ChamferFeature objects and provides functionality to create new ChamferFeature objects.

## Syntax

PartFeatures.**ChamferFeatures**() As [ChamferFeatures](../ChamferFeatures/ChamferFeatures.md)

## Property Value

This is a read only property whose value is a [ChamferFeatures](../ChamferFeatures/ChamferFeatures.md).

## Version

Introduced in version 5
