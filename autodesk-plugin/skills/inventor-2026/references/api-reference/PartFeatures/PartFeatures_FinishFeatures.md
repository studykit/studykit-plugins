# PartFeatures.FinishFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are FinishFeatures.

## Syntax

PartFeatures.**FinishFeatures**() As [FinishFeatures](../FinishFeatures/FinishFeatures.md)

## Property Value

This is a read only property whose value is a [FinishFeatures](../FinishFeatures/FinishFeatures.md).

## Version

Introduced in version 2024
