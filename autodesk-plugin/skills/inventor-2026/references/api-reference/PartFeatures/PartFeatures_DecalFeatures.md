# PartFeatures.DecalFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the DecalFeatures collection object. This collection provides access to existing DecalFeature objects.

## Syntax

PartFeatures.**DecalFeatures**() As [DecalFeatures](../DecalFeatures/DecalFeatures.md)

## Property Value

This is a read only property whose value is a [DecalFeatures](../DecalFeatures/DecalFeatures.md).
