# TangentDistanceDimConstraintProxy.EntityOne Property

Parent Object: [TangentDistanceDimConstraintProxy](../TangentDistanceDimConstraintProxy/TangentDistanceDimConstraintProxy.md)

## Description

Property that returns the first circle or line being constrained.

## Syntax

TangentDistanceDimConstraintProxy.**EntityOne**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
