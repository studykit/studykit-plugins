# BIMPipeConnectorDefinition.SystemType Property

Parent Object: [BIMPipeConnectorDefinition](../BIMPipeConnectorDefinition/BIMPipeConnectorDefinition.md)

## Description

Read-write property that specifies the system type for this connector.

## Syntax

BIMPipeConnectorDefinition.**SystemType**() As [BIMPipeSystemTypeEnum](../BIMPipeSystemTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BIMPipeSystemTypeEnum](../BIMPipeSystemTypeEnum.md).

## Version

Introduced in version 2011
