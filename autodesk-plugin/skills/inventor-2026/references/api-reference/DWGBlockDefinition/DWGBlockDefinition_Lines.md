# DWGBlockDefinition.Lines Property

Parent Object: [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Description

Read-only property that returns the DWGLinesEnumerator collection object.

## Syntax

DWGBlockDefinition.**Lines**() As [DWGLinesEnumerator](../DWGLinesEnumerator/DWGLinesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGLinesEnumerator](../DWGLinesEnumerator/DWGLinesEnumerator.md).

## Version

Introduced in version 2016
