# DWGBlockDefinition.AcadSupportedProxies Property

Parent Object: [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Description

Read-only property that returns the DWGAcadSupportedProxiesEnumerator collection object.

## Syntax

DWGBlockDefinition.**AcadSupportedProxies**() As [DWGAcadSupportedProxiesEnumerator](../DWGAcadSupportedProxiesEnumerator/DWGAcadSupportedProxiesEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGAcadSupportedProxiesEnumerator](../DWGAcadSupportedProxiesEnumerator/DWGAcadSupportedProxiesEnumerator.md).

## Version

Introduced in version 2018
