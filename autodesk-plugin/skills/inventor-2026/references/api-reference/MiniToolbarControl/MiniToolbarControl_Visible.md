# MiniToolbarControl.Visible Property

Parent Object: [MiniToolbarControl](../MiniToolbarControl/MiniToolbarControl.md)

## Description

Read-write property that gets and sets whether the control is visible.

## Syntax

MiniToolbarControl.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
