# MiniToolbarControl.Enabled Property

Parent Object: [MiniToolbarControl](../MiniToolbarControl/MiniToolbarControl.md)

## Description

Read-write property that gets and sets whether the control is enabled.

## Syntax

MiniToolbarControl.**Enabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
