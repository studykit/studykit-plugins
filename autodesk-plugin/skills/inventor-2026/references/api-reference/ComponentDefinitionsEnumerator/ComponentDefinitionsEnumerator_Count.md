# ComponentDefinitionsEnumerator.Count Property

Parent Object: [ComponentDefinitionsEnumerator](../ComponentDefinitionsEnumerator/ComponentDefinitionsEnumerator.md)

## Description

Property that returns the number of items in this collection.

## Syntax

ComponentDefinitionsEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 10
