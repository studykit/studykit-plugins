# GroundConstraint3DProxy.NativeObject Property

Parent Object: [GroundConstraint3DProxy](../GroundConstraint3DProxy/GroundConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

GroundConstraint3DProxy.**NativeObject**() As [GroundConstraint3D](../GroundConstraint3D/GroundConstraint3D.md)

## Property Value

This is a read only property whose value is a [GroundConstraint3D](../GroundConstraint3D/GroundConstraint3D.md).

## Version

Introduced in version 10
