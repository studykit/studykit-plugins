# PointCloudCrop.BoundingBox Property

Parent Object: [PointCloudCrop](../PointCloudCrop/PointCloudCrop.md)

## Description

Read-only property that returns the oriented box presents the bounding volume of the point cloud crop.

## Syntax

PointCloudCrop.**BoundingBox**() As [OrientedBox](../OrientedBox/OrientedBox.md)

## Property Value

This is a read only property whose value is an [OrientedBox](../OrientedBox/OrientedBox.md).

## Version

Introduced in version 2016
