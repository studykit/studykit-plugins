# PointCloudCrop.Parent Property

Parent Object: [PointCloudCrop](../PointCloudCrop/PointCloudCrop.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PointCloudCrop.**Parent**() As [PointCloud](../PointCloud/PointCloud.md)

## Property Value

This is a read only property whose value is a [PointCloud](../PointCloud/PointCloud.md).

## Version

Introduced in version 2016
