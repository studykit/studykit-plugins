# ModelFeatureControlFrameRows Object

## Description

Object provides access to all rows of a feature control frame in a part or assembly.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ModelFeatureControlFrameRows/ModelFeatureControlFrameRows_Add.md) | Method that adds a row to the feature control frame. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ModelFeatureControlFrameRows/ModelFeatureControlFrameRows_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../ModelFeatureControlFrameRows/ModelFeatureControlFrameRows_Count.md) | Gets the number of items in this collection. |
| [Item](../ModelFeatureControlFrameRows/ModelFeatureControlFrameRows_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../ModelFeatureControlFrameRows/ModelFeatureControlFrameRows_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[ModelFeatureControlFrameDefinition.FeatureControlFrameRows](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition_FeatureControlFrameRows.md)

## Version

Introduced in version 2015
