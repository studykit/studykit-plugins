# Plane.IsCoplanarTo Property

Parent Object: [Plane](../Plane/Plane.md)

## Description

Property that gets whether this Plane is coplanar with the specified Plane.

## Syntax

Plane.**IsCoplanarTo**( ***Plane*** As [Plane](../Plane/Plane.md), [***Tolerance***] As Double ) As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Plane | [Plane](../Plane/Plane.md) | Plane to perform coplanar comparison against. |
| Tolerance | Double | Input Double that specifies the tolerance. |

## Version

Introduced in version 9
