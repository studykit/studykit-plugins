# SculptSurface.Direction Property

Parent Object: [SculptSurface](../SculptSurface/SculptSurface.md)

## Description

Gets and sets the direction associated with the surface.

## Syntax

SculptSurface.**Direction**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 11
