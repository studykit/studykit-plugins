# BIMConnector.Definition Property

Parent Object: [BIMConnector](../BIMConnector/BIMConnector.md)

## Description

Read-only property that gets the definition object associated with this connector.

## Syntax

BIMConnector.**Definition**() As [BIMConnectorDefinition](../BIMConnectorDefinition/BIMConnectorDefinition.md)

## Property Value

This is a read only property whose value is a [BIMConnectorDefinition](../BIMConnectorDefinition/BIMConnectorDefinition.md).

## Version

Introduced in version 2011
