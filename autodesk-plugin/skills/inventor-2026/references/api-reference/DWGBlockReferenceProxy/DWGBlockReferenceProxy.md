# DWGBlockReferenceProxy Object

Derived from: [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md) Object

## Description

DWGBlockReferenceProxy Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetObject](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_GetObject.md) | Method that returns the corresponding object in the block reference for the given object from its definition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [ContainingOccurrence](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_ContainingOccurrence.md) | Get the component occurrence context through which the native object is being seen through. |
| [Definition](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_Definition.md) | Read-only property that returns the definition object that this DWGBlockReference is an instance of. |
| [HandleID](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_HandleID.md) | Read-only property that returns the HandleID for the DWGEntity. |
| [NativeObject](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_NativeObject.md) | Get the object in the context of the definition instead of the containing assembly. |
| [Parent](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ParentEntity](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_ParentEntity.md) | Read-only property that returns the parent DWGEntity of the entity. |
| [Transformation](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_Transformation.md) | Read-only property that returns the transformation matrix for the DWGBlockReference instance. |
| [Type](../DWGBlockReferenceProxy/DWGBlockReferenceProxy_Type.md) | Gets the constant that indicates the type of this object. |
