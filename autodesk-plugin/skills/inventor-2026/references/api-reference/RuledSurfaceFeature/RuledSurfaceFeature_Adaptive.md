# RuledSurfaceFeature.Adaptive Property

Parent Object: [RuledSurfaceFeature](../RuledSurfaceFeature/RuledSurfaceFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

RuledSurfaceFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
