# BendsEnumerator Object

## Description

The BendsEnumerator object represents all of the bends in a sheet metal part.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BendsEnumerator/BendsEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BendsEnumerator/BendsEnumerator_Count.md) | Property that returns the number of Bend objects in the collection. |
| [Item](../BendsEnumerator/BendsEnumerator_Item.md) | Returns the specified Bend object from the collection. |
| [Type](../BendsEnumerator/BendsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
