# PresentationEdge.Evaluator Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Read-only property that returns the CurveEvaluator for this presentation edge.

## Syntax

PresentationEdge.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 2018
