# RotateTranslateiMateDefinition Object

Derived from: [iMateDefinition](../iMateDefinition/iMateDefinition.md) Object

## Description

The RotateTranslateiMateDefinition object represents a rotate translate iMate definition. It is derived from the iMateDefinition object.
