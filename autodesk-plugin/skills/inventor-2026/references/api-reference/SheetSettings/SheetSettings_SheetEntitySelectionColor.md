# SheetSettings.SheetEntitySelectionColor Property

Parent Object: [SheetSettings](../SheetSettings/SheetSettings.md)

## Description

Gets/Sets the color used to illuminate elements on the drawing sheet when they are selected.

## Syntax

SheetSettings.**SheetEntitySelectionColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
