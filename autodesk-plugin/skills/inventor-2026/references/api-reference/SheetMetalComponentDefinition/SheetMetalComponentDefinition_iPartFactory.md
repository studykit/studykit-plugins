# SheetMetalComponentDefinition.iPartFactory Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the iPartFactory object. This property will fail in the case where the part is not an iPart Factory. You can determine this by using the IsiPartFactory property.

## Syntax

SheetMetalComponentDefinition.**iPartFactory**() As [iPartFactory](../iPartFactory/iPartFactory.md)

## Property Value

This is a read only property whose value is an [iPartFactory](../iPartFactory/iPartFactory.md).

## Version

Introduced in version 6
