# SheetMetalComponentDefinition.RolledBackForEdit Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that gets whether the model is currently rolled back to a previous point in the feature history.

## Syntax

SheetMetalComponentDefinition.**RolledBackForEdit**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 10
