# SheetMetalComponentDefinition.AnnotationPlanes Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-only property that returns the AnnotationPlanes collection object. This object provides access to all of the annotation planes in the part.

## Syntax

SheetMetalComponentDefinition.**AnnotationPlanes**() As [AnnotationPlanes](../AnnotationPlanes/AnnotationPlanes.md)

## Property Value

This is a read only property whose value is an [AnnotationPlanes](../AnnotationPlanes/AnnotationPlanes.md).

## Version

Introduced in version 2018
