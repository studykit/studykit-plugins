# SheetMetalComponentDefinition.RepresentationsManager Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-only property that returns the RepresentationsManager object.

## Syntax

SheetMetalComponentDefinition.**RepresentationsManager**() As [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md)

## Property Value

This is a read only property whose value is a [RepresentationsManager](../RepresentationsManager/RepresentationsManager.md).

## Version

Introduced in version 2012
