# SheetMetalComponentDefinition.UseSheetMetalStyleUnfoldMethod Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-write property that specifies whether to use the unfold method specified by the sheet metal style or not.

## Syntax

SheetMetalComponentDefinition.**UseSheetMetalStyleUnfoldMethod**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
