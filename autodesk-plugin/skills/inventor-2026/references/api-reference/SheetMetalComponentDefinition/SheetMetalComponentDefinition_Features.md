# SheetMetalComponentDefinition.Features Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the PartFeatures collection object.

## Syntax

SheetMetalComponentDefinition.**Features**() As [PartFeatures](../PartFeatures/PartFeatures.md)

## Property Value

This is a read only property whose value is a [PartFeatures](../PartFeatures/PartFeatures.md).
