# SheetMetalComponentDefinition.WorkAxes Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the WorkAxes collection object.

## Syntax

SheetMetalComponentDefinition.**WorkAxes**() As [WorkAxes](../WorkAxes/WorkAxes.md)

## Property Value

This is a read only property whose value is a [WorkAxes](../WorkAxes/WorkAxes.md).

## Version

Introduced in version 5.3
