# SheetMetalComponentDefinition.BOMQuantity Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the BOMQuantity object.

## Syntax

SheetMetalComponentDefinition.**BOMQuantity**() As [BOMQuantity](../BOMQuantity/BOMQuantity.md)

## Property Value

This is a read only property whose value is a [BOMQuantity](../BOMQuantity/BOMQuantity.md).

## Version

Introduced in version 10
