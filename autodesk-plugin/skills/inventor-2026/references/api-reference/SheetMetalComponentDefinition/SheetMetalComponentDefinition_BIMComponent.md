# SheetMetalComponentDefinition.BIMComponent Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-only property that returns the BIMComponent object associated with this component definition.

## Syntax

SheetMetalComponentDefinition.**BIMComponent**() As [BIMComponent](../BIMComponent/BIMComponent.md)

## Property Value

This is a read only property whose value is a [BIMComponent](../BIMComponent/BIMComponent.md).

## Version

Introduced in version 2011
