# DerivedAssemblyComponents.Item Property

Parent Object: [DerivedAssemblyComponents](../DerivedAssemblyComponents/DerivedAssemblyComponents.md)

## Description

Returns the specified object from the collection.

## Syntax

DerivedAssemblyComponents.**Item**( ***Index*** As Long ) As [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Property Value

This is a read only property whose value is a [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the to return. |

## Version

Introduced in version 5.3
