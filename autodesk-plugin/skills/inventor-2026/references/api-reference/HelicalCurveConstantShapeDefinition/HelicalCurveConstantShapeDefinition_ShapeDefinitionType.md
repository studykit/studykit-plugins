# HelicalCurveConstantShapeDefinition.ShapeDefinitionType Property

Parent Object: [HelicalCurveConstantShapeDefinition](../HelicalCurveConstantShapeDefinition/HelicalCurveConstantShapeDefinition.md)

## Description

Gets and sets the shape definition type of the helical curve.

## Syntax

HelicalCurveConstantShapeDefinition.**ShapeDefinitionType**() As [HelicalShapeDefinitionTypeEnum](../HelicalShapeDefinitionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [HelicalShapeDefinitionTypeEnum](../HelicalShapeDefinitionTypeEnum.md).

## Version

Introduced in version 2019
