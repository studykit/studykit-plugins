# InventorVBAProjects.Item Property

Parent Object: [InventorVBAProjects](../InventorVBAProjects/InventorVBAProjects.md)

## Description

Returns the specified InventorVBAProject object from the collection.

## Syntax

InventorVBAProjects.**Item**( ***Index*** As Long ) As [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Property Value

This is a read only property whose value is an [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
