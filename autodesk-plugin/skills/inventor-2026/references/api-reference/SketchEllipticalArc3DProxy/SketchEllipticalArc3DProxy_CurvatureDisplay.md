# SketchEllipticalArc3DProxy.CurvatureDisplay Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Specifies whether curvature information is displayed for the elliptical arc.

## Syntax

SketchEllipticalArc3DProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
