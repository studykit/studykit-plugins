# SketchCircle.OverrideColor Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Gets and sets the color applied to this sketch circl.

## Syntax

SketchCircle.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
