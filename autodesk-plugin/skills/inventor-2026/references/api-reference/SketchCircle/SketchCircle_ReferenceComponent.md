# SketchCircle.ReferenceComponent Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Property that returns the ReferenceComponent that resulted in the creation of this feature.

## Syntax

SketchCircle.**ReferenceComponent**() As [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md)

## Property Value

This is a read only property whose value is a [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md).

## Version

Introduced in version 5.3
