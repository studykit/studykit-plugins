# DirectEditMoveOperationProxy.ContainingOccurrence Property

Parent Object: [DirectEditMoveOperationProxy](../DirectEditMoveOperationProxy/DirectEditMoveOperationProxy.md)

## Description

Get the component occurrence context through which the native object is being seen through.

## Syntax

DirectEditMoveOperationProxy.**ContainingOccurrence**() As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md).

## Version

Introduced in version 2015
