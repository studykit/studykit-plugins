# SketchedSymbolDefinitionLibrary.Name Property

Parent Object: [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md)

## Description

Read-only property that returns the name of the sketched symbol definition library.

## Syntax

SketchedSymbolDefinitionLibrary.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
