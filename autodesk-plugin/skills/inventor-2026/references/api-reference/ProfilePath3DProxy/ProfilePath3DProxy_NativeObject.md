# ProfilePath3DProxy.NativeObject Property

Parent Object: [ProfilePath3DProxy](../ProfilePath3DProxy/ProfilePath3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ProfilePath3DProxy.**NativeObject**() As [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Property Value

This is a read only property whose value is a [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md).

## Version

Introduced in version 6
