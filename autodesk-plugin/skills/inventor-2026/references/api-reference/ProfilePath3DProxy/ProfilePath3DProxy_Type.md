# ProfilePath3DProxy.Type Property

Parent Object: [ProfilePath3DProxy](../ProfilePath3DProxy/ProfilePath3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ProfilePath3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
