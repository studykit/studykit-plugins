# ProfilePath3DProxy.Parent Property

Parent Object: [ProfilePath3DProxy](../ProfilePath3DProxy/ProfilePath3DProxy.md)

## Description

Property that returns the parent Profile3D.

## Syntax

ProfilePath3DProxy.**Parent**() As [Profile3D](../Profile3D/Profile3D.md)

## Property Value

This is a read only property whose value is a [Profile3D](../Profile3D/Profile3D.md).

## Version

Introduced in version 6
