# SketchEquationCurve.Construction Property

Parent Object: [SketchEquationCurve](../SketchEquationCurve/SketchEquationCurve.md)

## Description

Gets and sets whether the entity behaves as a construction entity or not.

## Syntax

SketchEquationCurve.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
