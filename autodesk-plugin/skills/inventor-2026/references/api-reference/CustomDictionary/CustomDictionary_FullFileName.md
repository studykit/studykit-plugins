# CustomDictionary.FullFileName Property

Parent Object: [CustomDictionary](../CustomDictionary/CustomDictionary.md)

## Description

Read-only property that returns the fully qualified file name of the CustomDictionary.

## Syntax

CustomDictionary.**FullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2020
