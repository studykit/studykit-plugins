# CustomDictionary.DeleteWord Method

Parent Object: [CustomDictionary](../CustomDictionary/CustomDictionary.md)

## Description

Method that deletes a word from this custom dictionary.

## Syntax

CustomDictionary.**DeleteWord**( ***Word*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Word | String | Input String indicating a word. |

## Version

Introduced in version 2020
