# AssemblyComponentDefinition.SuppressFeatures Method

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Property that returns the SurfaceBodies collection. Surface bodies within the assembly is not currently supported by Inventor. The returned SurfaceBodies collection will be a collection of zero.

## Syntax

AssemblyComponentDefinition.**SuppressFeatures**( ***Features*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Features | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection that contains the list of features to suppress. |

## Version

Introduced in version 2009
