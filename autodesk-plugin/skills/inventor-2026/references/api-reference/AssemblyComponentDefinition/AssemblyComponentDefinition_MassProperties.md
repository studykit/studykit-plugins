# AssemblyComponentDefinition.MassProperties Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Property that returns the MassProperties object. This supports performing mass properties calculations for the entire assembly.

## Syntax

AssemblyComponentDefinition.**MassProperties**() As [MassProperties](../MassProperties/MassProperties.md)

## Property Value

This is a read only property whose value is a [MassProperties](../MassProperties/MassProperties.md).

## Version

Introduced in version 5
