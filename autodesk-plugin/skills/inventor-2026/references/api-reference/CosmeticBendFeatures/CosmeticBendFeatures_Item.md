# CosmeticBendFeatures.Item Property

Parent Object: [CosmeticBendFeatures](../CosmeticBendFeatures/CosmeticBendFeatures.md)

## Description

Returns the specified CosmeticBendFeature object from the collection.

## Syntax

CosmeticBendFeatures.**Item**( ***Index*** As Variant ) As [CosmeticBendFeature](../CosmeticBendFeature/CosmeticBendFeature.md)

## Property Value

This is a read only property whose value is a [CosmeticBendFeature](../CosmeticBendFeature/CosmeticBendFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long or String that specifies the CosmeticBendFeature object within the collection to return. Inputting a Long indicates the index of the CosmeticBendFeature within the collection to return. A String can also be used to specify the name of the feature. The property will fail is the index is out of range or the name does not exist. |

## Version

Introduced in version 2010
