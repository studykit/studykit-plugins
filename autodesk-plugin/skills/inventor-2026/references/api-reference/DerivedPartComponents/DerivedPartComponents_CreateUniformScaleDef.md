# DerivedPartComponents.CreateUniformScaleDef Method

Parent Object: [DerivedPartComponents](../DerivedPartComponents/DerivedPartComponents.md)

## Description

Method that creates a new DerivedPartUniformScaleDef object.

## Syntax

DerivedPartComponents.**CreateUniformScaleDef**( ***FullFileName*** As String ) As [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input string that specifies the name of the file to create the definition for. |
