# CollinearConstraint3D.EntityOne Property

Parent Object: [CollinearConstraint3D](../CollinearConstraint3D/CollinearConstraint3D.md)

## Description

Property that returns the first entity being constrained. This can either be a SketchLine3D or a WorkAxis.

## Syntax

CollinearConstraint3D.**EntityOne**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
