# EdgeUse.Edge Property

Parent Object: [EdgeUse](../EdgeUse/EdgeUse.md)

## Description

Gets the that this EdgeUse references.

## Syntax

EdgeUse.**Edge**() As [Edge](../Edge/Edge.md)

## Property Value

This is a read only property whose value is an [Edge](../Edge/Edge.md).
