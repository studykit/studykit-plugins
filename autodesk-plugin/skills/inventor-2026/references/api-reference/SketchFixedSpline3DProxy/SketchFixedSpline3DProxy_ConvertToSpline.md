# SketchFixedSpline3DProxy.ConvertToSpline Method

Parent Object: [SketchFixedSpline3DProxy](../SketchFixedSpline3DProxy/SketchFixedSpline3DProxy.md)

## Description

Method that converts the fixed spline to a SketchSpline3D object. This method is currently only valid for SketchFixedSpline3D objects that were created using a BSplineCurve as input.

## Syntax

SketchFixedSpline3DProxy.**ConvertToSpline**() As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Version

Introduced in version 10
