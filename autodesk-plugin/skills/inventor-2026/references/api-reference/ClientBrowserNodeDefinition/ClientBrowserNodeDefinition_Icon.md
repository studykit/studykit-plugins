# ClientBrowserNodeDefinition.Icon Property

Parent Object: [ClientBrowserNodeDefinition](../ClientBrowserNodeDefinition/ClientBrowserNodeDefinition.md)

## Description

Gets and sets the icon on a client definition object.

## Syntax

ClientBrowserNodeDefinition.**Icon**() As [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Property Value

This is a read/write property whose value is a [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md).

## Version

Introduced in version 8
