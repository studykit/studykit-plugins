# ModelHoleThreadNoteProxy.AttributeSets Property

Parent Object: [ModelHoleThreadNoteProxy](../ModelHoleThreadNoteProxy/ModelHoleThreadNoteProxy.md)

## Description

Property that returns the AttributeSets collection object associated with this object.

## Syntax

ModelHoleThreadNoteProxy.**AttributeSets**() As [AttributeSets](../AttributeSets/AttributeSets.md)

## Property Value

This is a read only property whose value is an [AttributeSets](../AttributeSets/AttributeSets.md).

## Version

Introduced in version 2018
