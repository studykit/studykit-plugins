# LineSegment.StartPoint Property

Parent Object: [LineSegment](../LineSegment/LineSegment.md)

## Description

Specifies the start point of the line segment.

## Syntax

LineSegment.**StartPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
