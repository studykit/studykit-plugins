# MoveDefinition.Parent Property

Parent Object: [MoveDefinition](../MoveDefinition/MoveDefinition.md)

## Description

Read-only property that returns the parent MoveFeature of the definition. This can return Nothing in the case where this definition is transient and is not associated with any feature.

## Syntax

MoveDefinition.**Parent**() As [MoveFeature](../MoveFeature/MoveFeature.md)

## Property Value

This is a read only property whose value is a [MoveFeature](../MoveFeature/MoveFeature.md).

## Version

Introduced in version 2013
