# SketchDrivenPatternFeature.ExtendedName Property

Parent Object: [SketchDrivenPatternFeature](../SketchDrivenPatternFeature/SketchDrivenPatternFeature.md)

## Description

Read-only property that returns the full feature name including any extended information.

## Syntax

SketchDrivenPatternFeature.**ExtendedName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2017
