# FlangeDefinition.UnfoldMethod Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Gets and sets the UnfoldMethod object that defines how the bends associated with this flange feature are unfolded.

## Syntax

FlangeDefinition.**UnfoldMethod**() As [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Property Value

This is a read/write property whose value is a [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md).

## Version

Introduced in version 2009
