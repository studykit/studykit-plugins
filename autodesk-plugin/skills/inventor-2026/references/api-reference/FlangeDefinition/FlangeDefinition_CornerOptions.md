# FlangeDefinition.CornerOptions Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Gets and sets the CornerOptions object that defines how the corners of the bend are modeled.

## Syntax

FlangeDefinition.**CornerOptions**() As [CornerOptions](../CornerOptions/CornerOptions.md)

## Property Value

This is a read/write property whose value is a [CornerOptions](../CornerOptions/CornerOptions.md).

## Version

Introduced in version 2009
