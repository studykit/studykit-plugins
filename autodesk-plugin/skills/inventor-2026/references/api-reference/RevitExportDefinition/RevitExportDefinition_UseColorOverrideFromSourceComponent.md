# RevitExportDefinition.UseColorOverrideFromSourceComponent Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets whether use the color override from source component.

## Syntax

RevitExportDefinition.**UseColorOverrideFromSourceComponent**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
