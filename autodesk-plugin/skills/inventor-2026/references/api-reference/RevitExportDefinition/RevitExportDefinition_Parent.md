# RevitExportDefinition.Parent Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

RevitExportDefinition.**Parent**() As [RevitExport](../RevitExport/RevitExport.md)

## Property Value

This is a read only property whose value is a [RevitExport](../RevitExport/RevitExport.md).

## Version

Introduced in version 2022
