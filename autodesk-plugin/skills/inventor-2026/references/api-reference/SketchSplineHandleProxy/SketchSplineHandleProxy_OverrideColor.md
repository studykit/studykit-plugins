# SketchSplineHandleProxy.OverrideColor Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Gets and sets the color applied to this sketch line.

## Syntax

SketchSplineHandleProxy.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2008
