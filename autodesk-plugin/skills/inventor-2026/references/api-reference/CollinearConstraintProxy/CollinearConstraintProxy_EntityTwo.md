# CollinearConstraintProxy.EntityTwo Property

Parent Object: [CollinearConstraintProxy](../CollinearConstraintProxy/CollinearConstraintProxy.md)

## Description

Property that returns the second sketch entity being constrained.

## Syntax

CollinearConstraintProxy.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
