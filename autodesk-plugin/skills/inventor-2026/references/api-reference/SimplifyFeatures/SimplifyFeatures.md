# SimplifyFeatures Object

## Description

Part Simplify Features Collection Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SimplifyFeatures/SimplifyFeatures_Add.md) | Method that creates a new SimplifyFeature. The new created SimplifyFeature is returned. |
| [CreateDefinition](../SimplifyFeatures/SimplifyFeatures_CreateDefinition.md) | Method that creates a SimplifyDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SimplifyFeatures/SimplifyFeatures_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../SimplifyFeatures/SimplifyFeatures_Count.md) | Gets the number of items in this collection. |
| [Item](../SimplifyFeatures/SimplifyFeatures_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../SimplifyFeatures/SimplifyFeatures_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PartFeatures.SimplifyFeatures](../PartFeatures/PartFeatures_SimplifyFeatures.md), [SheetMetalFeatures.SimplifyFeatures](../SheetMetalFeatures/SheetMetalFeatures_SimplifyFeatures.md)

## Version

Introduced in version 2026
