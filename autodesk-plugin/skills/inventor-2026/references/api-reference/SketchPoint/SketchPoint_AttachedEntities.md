# SketchPoint.AttachedEntities Property

Parent Object: [SketchPoint](../SketchPoint/SketchPoint.md)

## Description

Property that returns the collection of objects that are connected to this point.

## Syntax

SketchPoint.**AttachedEntities**() As [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md).

## Version

Introduced in version 5
