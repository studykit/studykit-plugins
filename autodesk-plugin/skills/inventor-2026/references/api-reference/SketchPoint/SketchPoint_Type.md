# SketchPoint.Type Property

Parent Object: [SketchPoint](../SketchPoint/SketchPoint.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchPoint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
