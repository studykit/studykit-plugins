# EdgeSymbolStyle.TextStyle Property

Parent Object: [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md)

## Description

Read-write property that gets and sets the text style used for the edge symbol style.

## Syntax

EdgeSymbolStyle.**TextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2024
