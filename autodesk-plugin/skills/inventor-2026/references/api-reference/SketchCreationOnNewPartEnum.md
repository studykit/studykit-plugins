# SketchCreationOnNewPartEnum Enumerator

## Description

Constants enumerating the options for sketch creation.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kNoNewSketchCreation | 55041 | No New Sketch will be created. |
| kXYPlaneSketchCreation | 55042 | Create a new Sketch on XY plane. |
| kXZPlaneSketchCreation | 55044 | Create a new Sketch on XZ plane. |
| kYZPlaneSketchCreation | 55043 | Create a new Sketch on YZ plane. |

## Version

Introduced in version 11
