# TriangleFanGraphics.BackFaceCulling Property

Parent Object: [TriangleFanGraphics](../TriangleFanGraphics/TriangleFanGraphics.md)

## Description

Gets and sets how the back face culling are defined for the triangles.

## Syntax

TriangleFanGraphics.**BackFaceCulling**() As [BackFaceCullingEnum](../BackFaceCullingEnum.md)

## Property Value

This is a read/write property whose value is a [BackFaceCullingEnum](../BackFaceCullingEnum.md).

## Version

Introduced in version 2013
