# ImportedDWGComponentProxy.Name Property

Parent Object: [ImportedDWGComponentProxy](../ImportedDWGComponentProxy/ImportedDWGComponentProxy.md)

## Description

Property that returns the component's name.

## Syntax

ImportedDWGComponentProxy.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
