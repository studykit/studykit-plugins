# OnFaceConstraint3DProxy.Face Property

Parent Object: [OnFaceConstraint3DProxy](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy.md)

## Description

Read-only property that returns the Face which the 3D sketch entity is constrained to.

## Syntax

OnFaceConstraint3DProxy.**Face**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2017
