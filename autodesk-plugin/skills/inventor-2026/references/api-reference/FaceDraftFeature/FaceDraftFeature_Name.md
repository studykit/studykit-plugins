# FaceDraftFeature.Name Property

Parent Object: [FaceDraftFeature](../FaceDraftFeature/FaceDraftFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

FaceDraftFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5
