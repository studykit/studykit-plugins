# SurfaceTextureDINDefinition.Parent Property

Parent Object: [SurfaceTextureDINDefinition](../SurfaceTextureDINDefinition/SurfaceTextureDINDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SurfaceTextureDINDefinition.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 2016
