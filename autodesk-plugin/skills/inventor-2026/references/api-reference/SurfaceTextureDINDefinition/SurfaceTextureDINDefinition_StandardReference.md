# SurfaceTextureDINDefinition.StandardReference Property

Parent Object: [SurfaceTextureDINDefinition](../SurfaceTextureDINDefinition/SurfaceTextureDINDefinition.md)

## Description

Returns the standard reference type.

## Syntax

SurfaceTextureDINDefinition.**StandardReference**() As [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md)

## Property Value

This is a read only property whose value is a [SurfaceTextureStandardReferenceTypeEnum](../SurfaceTextureStandardReferenceTypeEnum.md).

## Version

Introduced in version 2025
