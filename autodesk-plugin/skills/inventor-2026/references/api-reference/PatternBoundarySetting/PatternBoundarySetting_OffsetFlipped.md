# PatternBoundarySetting.OffsetFlipped Property

Parent Object: [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Description

Read-write property that gets and sets whether the offset is flipped or not.

## Syntax

PatternBoundarySetting.**OffsetFlipped**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
