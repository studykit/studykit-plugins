# AliasFreeformFeatureProxy.SurfaceBodies Property

Parent Object: [AliasFreeformFeatureProxy](../AliasFreeformFeatureProxy/AliasFreeformFeatureProxy.md)

## Description

Property that returns the bodies that this feature has created or modified.

## Syntax

AliasFreeformFeatureProxy.**SurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).

## Version

Introduced in version 2011
