# Style.StyleLocation Property

Parent Object: [Style](../Style/Style.md)

## Description

Property that returns the location of this style, i.e. local to the document, cached locally in the document, exists in the library. Styles that exist in the library cannot be edited.

## Syntax

Style.**StyleLocation**() As [StyleLocationEnum](../StyleLocationEnum.md)

## Property Value

This is a read only property whose value is a [StyleLocationEnum](../StyleLocationEnum.md).
