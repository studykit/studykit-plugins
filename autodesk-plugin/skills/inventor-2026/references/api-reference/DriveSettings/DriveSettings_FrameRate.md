# DriveSettings.FrameRate Property

Parent Object: [DriveSettings](../DriveSettings/DriveSettings.md)

## Description

Read-write property that gets and sets the increments at which a snapshot is taken for inclusion as a frame in a recorded animation.

## Syntax

DriveSettings.**FrameRate**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2014
