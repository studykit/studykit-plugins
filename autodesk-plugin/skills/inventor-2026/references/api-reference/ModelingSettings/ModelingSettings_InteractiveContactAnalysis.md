# ModelingSettings.InteractiveContactAnalysis Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets the type of components to use for interactive contact analysis.

## Syntax

ModelingSettings.**InteractiveContactAnalysis**() As [InteractiveContactAnalysisEnum](../InteractiveContactAnalysisEnum.md)

## Property Value

This is a read/write property whose value is an [InteractiveContactAnalysisEnum](../InteractiveContactAnalysisEnum.md).

## Version

Introduced in version 2008
