# ModelingSettings.AdaptivelyUsedInAssembly Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets the indicator that part is user adaptive in an assembly.

## Syntax

ModelingSettings.**AdaptivelyUsedInAssembly**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 8
