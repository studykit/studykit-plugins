# SmoothConstraintProxy.NativeObject Property

Parent Object: [SmoothConstraintProxy](../SmoothConstraintProxy/SmoothConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SmoothConstraintProxy.**NativeObject**() As [SmoothConstraint](../SmoothConstraint/SmoothConstraint.md)

## Property Value

This is a read only property whose value is a [SmoothConstraint](../SmoothConstraint/SmoothConstraint.md).

## Version

Introduced in version 11
