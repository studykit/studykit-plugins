# PointAndTangentWorkPlaneDef.Parent Property

Parent Object: [PointAndTangentWorkPlaneDef](../PointAndTangentWorkPlaneDef/PointAndTangentWorkPlaneDef.md)

## Description

Property that returns the parent WorkPlane object.

## Syntax

PointAndTangentWorkPlaneDef.**Parent**() As [WorkPlane](../WorkPlane/WorkPlane.md)

## Property Value

This is a read only property whose value is a [WorkPlane](../WorkPlane/WorkPlane.md).

## Version

Introduced in version 10
