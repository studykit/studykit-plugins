# FeatureControlFrameStyle.ConvertToLocal Method

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

FeatureControlFrameStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2009
