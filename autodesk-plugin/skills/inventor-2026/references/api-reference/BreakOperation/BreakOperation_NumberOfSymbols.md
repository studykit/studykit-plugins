# BreakOperation.NumberOfSymbols Property

Parent Object: [BreakOperation](../BreakOperation/BreakOperation.md)

## Description

Read-write property that gets and sets the number of break symbols to use for a structural style break.

## Syntax

BreakOperation.**NumberOfSymbols**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
