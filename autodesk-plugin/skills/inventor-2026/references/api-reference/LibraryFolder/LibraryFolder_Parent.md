# LibraryFolder.Parent Property

Parent Object: [LibraryFolder](../LibraryFolder/LibraryFolder.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

LibraryFolder.**Parent**() As [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md)

## Property Value

This is a read only property whose value is a [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md).

## Version

Introduced in version 2016
