# FaceShellProxy.IsVoid Property

Parent Object: [FaceShellProxy](../FaceShellProxy/FaceShellProxy.md)

## Description

Indicates whether the FaceShell is a void.

## Syntax

FaceShellProxy.**IsVoid**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
