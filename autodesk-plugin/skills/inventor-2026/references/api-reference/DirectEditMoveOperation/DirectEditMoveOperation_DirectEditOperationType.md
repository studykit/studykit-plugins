# DirectEditMoveOperation.DirectEditOperationType Property

Parent Object: [DirectEditMoveOperation](../DirectEditMoveOperation/DirectEditMoveOperation.md)

## Description

Gets the direct edit operation type.

## Syntax

DirectEditMoveOperation.**DirectEditOperationType**() As [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md)

## Property Value

This is a read only property whose value is a [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md).

## Version

Introduced in version 2015
