## Description

SketchRectangularPattern Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../SketchRectangularPattern/SketchRectangularPattern_Delete.md) | Deletes the sketch pattern. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchRectangularPattern/SketchRectangularPattern_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [AttributeSets](../SketchRectangularPattern/SketchRectangularPattern_AttributeSets.md) | Returns the AttributeSets collection object associated with this object. |
| [Constraints](../SketchRectangularPattern/SketchRectangularPattern_Constraints.md) | Returns a collection of sketch constraints that are tied directly to this entity. This collection consists of both geometric and dimension constraints. |
| [ConstraintStatus](../SketchRectangularPattern/SketchRectangularPattern_ConstraintStatus.md) | Returns the constraint status of the sketch entity. Possible return values are kFullyConstrainedConstraintStatus, kOverConstrainedConstraintStatus, kUnderConstrainedConstraintStatus and kUnknownConstraintStatus. |
| [Definition](../SketchRectangularPattern/SketchRectangularPattern_Definition.md) | Gets and sets the associative definition of the sketch pattern. |
| [Elements](../SketchRectangularPattern/SketchRectangularPattern_Elements.md) | Returns the SketchPatternElements collection object. |
| [Parent](../SketchRectangularPattern/SketchRectangularPattern_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [RangeBox](../SketchRectangularPattern/SketchRectangularPattern_RangeBox.md) | Returns a Box2d object which contains the diagonal points of a rectangle that tightly encloses the entity. |
| [Type](../SketchRectangularPattern/SketchRectangularPattern_Type.md) | Gets the constant that indicates the type of this object. |
