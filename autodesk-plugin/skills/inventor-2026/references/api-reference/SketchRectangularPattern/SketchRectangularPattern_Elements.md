# SketchRectangularPattern.Elements Property

Parent Object: [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Description

Returns the SketchPatternElements collection object.

## Syntax

SketchRectangularPattern.**Elements**() As [SketchPatternElements](../SketchPatternElements/SketchPatternElements.md)

## Property Value

This is a read only property whose value is a [SketchPatternElements](../SketchPatternElements/SketchPatternElements.md).

## Version

Introduced in version 2025.1
