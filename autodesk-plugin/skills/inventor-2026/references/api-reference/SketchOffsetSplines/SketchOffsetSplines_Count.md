# SketchOffsetSplines.Count Property

Parent Object: [SketchOffsetSplines](../SketchOffsetSplines/SketchOffsetSplines.md)

## Description

Property that specifies the number of items in the collection.

## Syntax

SketchOffsetSplines.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 9
