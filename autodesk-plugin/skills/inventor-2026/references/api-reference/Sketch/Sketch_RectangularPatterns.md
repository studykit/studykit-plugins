# Sketch.RectangularPatterns Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Gets the SketchRectangularPatterns collection object.

## Syntax

Sketch.**RectangularPatterns**() As [SketchRectangularPatterns](../SketchRectangularPatterns/SketchRectangularPatterns.md)

## Property Value

This is a read only property whose value is a [SketchRectangularPatterns](../SketchRectangularPatterns/SketchRectangularPatterns.md).

## Version

Introduced in version 2025.1
