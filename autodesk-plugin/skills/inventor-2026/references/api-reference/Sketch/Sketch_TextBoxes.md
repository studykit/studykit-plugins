# Sketch.TextBoxes Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Gets the TextBoxes collection associated with this Sketch.

## Syntax

Sketch.**TextBoxes**() As [TextBoxes](../TextBoxes/TextBoxes.md)

## Property Value

This is a read only property whose value is a [TextBoxes](../TextBoxes/TextBoxes.md).
