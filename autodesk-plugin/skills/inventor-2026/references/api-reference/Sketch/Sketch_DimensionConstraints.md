# Sketch.DimensionConstraints Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Gets the collection of all dimension constraints on the sketch.

## Syntax

Sketch.**DimensionConstraints**() As [DimensionConstraints](../DimensionConstraints/DimensionConstraints.md)

## Property Value

This is a read only property whose value is a [DimensionConstraints](../DimensionConstraints/DimensionConstraints.md).

## Version

Introduced in version 5.3
