# Sketch.SketchSplines Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Property that returns the SketchSplines collection object.

## Syntax

Sketch.**SketchSplines**() As [SketchSplines](../SketchSplines/SketchSplines.md)

## Property Value

This is a read only property whose value is a [SketchSplines](../SketchSplines/SketchSplines.md).
