# Sketch.DisabledActionTypes Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Gets and sets the action types valid for this sketch.

## Syntax

Sketch.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 9
