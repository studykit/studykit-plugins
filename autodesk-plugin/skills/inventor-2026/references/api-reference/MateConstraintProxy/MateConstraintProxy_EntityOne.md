# MateConstraintProxy.EntityOne Property

Parent Object: [MateConstraintProxy](../MateConstraintProxy/MateConstraintProxy.md)

## Description

Property that indicates the first of the geometric entities (Face, Axis, Edge, etc.) being constrained.

## Syntax

MateConstraintProxy.**EntityOne**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
