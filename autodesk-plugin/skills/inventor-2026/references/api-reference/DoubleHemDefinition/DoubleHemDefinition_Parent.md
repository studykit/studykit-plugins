# DoubleHemDefinition.Parent Property

Parent Object: [DoubleHemDefinition](../DoubleHemDefinition/DoubleHemDefinition.md)

## Description

Property that returns the parent hem definition feature.

## Syntax

DoubleHemDefinition.**Parent**() As [HemDefinition](../HemDefinition/HemDefinition.md)

## Property Value

This is a read only property whose value is a [HemDefinition](../HemDefinition/HemDefinition.md).

## Version

Introduced in version 2009
