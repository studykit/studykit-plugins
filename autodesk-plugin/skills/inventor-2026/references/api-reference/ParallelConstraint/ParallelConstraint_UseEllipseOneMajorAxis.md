# ParallelConstraint.UseEllipseOneMajorAxis Property

Parent Object: [ParallelConstraint](../ParallelConstraint/ParallelConstraint.md)

## Description

Property used in the case where the entity returned by the EntityOne property is an ellipse. This property specifies if the major or minor axis of the ellipse is parallel to EntityTwo. True if it is the major axis.

## Syntax

ParallelConstraint.**UseEllipseOneMajorAxis**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5
