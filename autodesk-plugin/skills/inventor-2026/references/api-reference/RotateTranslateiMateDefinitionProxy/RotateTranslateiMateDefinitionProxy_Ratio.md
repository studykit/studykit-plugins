# RotateTranslateiMateDefinitionProxy.Ratio Property

Parent Object: [RotateTranslateiMateDefinitionProxy](../RotateTranslateiMateDefinitionProxy/RotateTranslateiMateDefinitionProxy.md)

## Description

Property that returns the Parameter object that controls the ratio of the iMate definition.

## Syntax

RotateTranslateiMateDefinitionProxy.**Ratio**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
