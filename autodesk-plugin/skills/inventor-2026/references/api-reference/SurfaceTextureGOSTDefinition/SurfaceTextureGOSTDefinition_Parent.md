# SurfaceTextureGOSTDefinition.Parent Property

## Syntax

SurfaceTextureGOSTDefinition.**Parent**() As [Sheet](../Sheet/Sheet.md)

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).
