# ModelCompositeAnnotationDefinition.Parent Property

Parent Object: [ModelCompositeAnnotationDefinition](../ModelCompositeAnnotationDefinition/ModelCompositeAnnotationDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelCompositeAnnotationDefinition.**Parent**() As [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md)

## Property Value

This is a read only property whose value is a [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md).

## Version

Introduced in version 2018
