# SketchLine3DProxy.OwnedBy Property

Parent Object: [SketchLine3DProxy](../SketchLine3DProxy/SketchLine3DProxy.md)

## Description

Property indicating entity or entities that own this object.

## Syntax

SketchLine3DProxy.**OwnedBy**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 11
