# SketchSplineHandle.LineScale Property

Parent Object: [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md)

## Description

Gets and sets the LineScale applied to this sketch line.

## Syntax

SketchSplineHandle.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
