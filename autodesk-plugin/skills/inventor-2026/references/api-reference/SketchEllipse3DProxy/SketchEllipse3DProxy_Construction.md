# SketchEllipse3DProxy.Construction Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchEllipse3DProxy.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
