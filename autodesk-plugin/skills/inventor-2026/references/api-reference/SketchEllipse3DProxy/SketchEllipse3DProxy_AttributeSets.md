# SketchEllipse3DProxy.AttributeSets Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Property that returns the AttributeSets collection object associated with this object.

## Syntax

SketchEllipse3DProxy.**AttributeSets**() As [AttributeSets](../AttributeSets/AttributeSets.md)

## Property Value

This is a read only property whose value is an [AttributeSets](../AttributeSets/AttributeSets.md).

## Version

Introduced in version 6
