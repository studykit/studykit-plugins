# iMateResultsEnumerator.Item Property

Parent Object: [iMateResultsEnumerator](../iMateResultsEnumerator/iMateResultsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

iMateResultsEnumerator.**Item**( ***Index*** As Long ) As [iMateResult](../iMateResult/iMateResult.md)

## Property Value

This is a read only property whose value is an [iMateResult](../iMateResult/iMateResult.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 6
