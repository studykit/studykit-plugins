# DerivedAliasComponents.Count Property

Parent Object: [DerivedAliasComponents](../DerivedAliasComponents/DerivedAliasComponents.md)

## Description

Property that returns the number of items in this collection.

## Syntax

DerivedAliasComponents.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
