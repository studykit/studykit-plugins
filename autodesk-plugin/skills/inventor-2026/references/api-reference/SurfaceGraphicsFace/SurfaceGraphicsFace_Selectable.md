# SurfaceGraphicsFace.Selectable Property

Parent Object: [SurfaceGraphicsFace](../SurfaceGraphicsFace/SurfaceGraphicsFace.md)

## Description

Read-write property that specifies whether the surface graphics face can be selected or not.

## Syntax

SurfaceGraphicsFace.**Selectable**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
