# PositionalRepresentation.IsTransformOverridden Method

Parent Object: [PositionalRepresentation](../PositionalRepresentation/PositionalRepresentation.md)

## Description

Method that returns whether the transform of the input occurrence is overridden for the positional representation and returns the transform.

## Syntax

PositionalRepresentation.**IsTransformOverridden**( ***Occurrence*** As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md), ***Transform*** As [Matrix](../Matrix/Matrix.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Occurrence | [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md) | ComponentOccurrence object that specifies the occurrence to be queried for the override. |
| Transform | [Matrix](../Matrix/Matrix.md) | Matrix object that returns the transform (whether it is overridden or not). |

## Version

Introduced in version 11
