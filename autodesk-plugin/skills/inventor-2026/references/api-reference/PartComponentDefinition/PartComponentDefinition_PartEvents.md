# PartComponentDefinition.PartEvents Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the PartEvents source object.

## Syntax

PartComponentDefinition.**PartEvents**() As [PartEvents](../PartEvents/PartEvents.md)

## Property Value

This is a read only property whose value is a [PartEvents](../PartEvents/PartEvents.md).

## Version

Introduced in version 4
