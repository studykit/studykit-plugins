# PartComponentDefinition.Features Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the PartFeatures collection object.

## Syntax

PartComponentDefinition.**Features**() As [PartFeatures](../PartFeatures/PartFeatures.md)

## Property Value

This is a read only property whose value is a [PartFeatures](../PartFeatures/PartFeatures.md).
