# PartComponentDefinition.RolledBackForEdit Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that gets whether the model is currently rolled back to a previous point in the feature history.

## Syntax

PartComponentDefinition.**RolledBackForEdit**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 10
