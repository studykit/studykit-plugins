# ModelStateTableCell.Column Property

Parent Object: [ModelStateTableCell](../ModelStateTableCell/ModelStateTableCell.md)

## Description

Read-only property that returns the index of this cell within the parent ModelStatesTableColumn.

## Syntax

ModelStateTableCell.**Column**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2022
