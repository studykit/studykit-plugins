# ModelStateTableCell.HasFormula Property

Parent Object: [ModelStateTableCell](../ModelStateTableCell/ModelStateTableCell.md)

## Description

Read-only property that returns whether a formula(equation) was input into this cell via Excel.

## Syntax

ModelStateTableCell.**HasFormula**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2022
