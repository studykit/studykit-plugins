# MoveAlongRayMoveOperation.Offset Property

Parent Object: [MoveAlongRayMoveOperation](../MoveAlongRayMoveOperation/MoveAlongRayMoveOperation.md)

## Description

Read-write property that provides access to the offset of a move along ray operation on a move body feature.

## Syntax

MoveAlongRayMoveOperation.**Offset**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2013
