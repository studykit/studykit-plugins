# ViewTabGroup.ViewFrame Property

Parent Object: [ViewTabGroup](../ViewTabGroup/ViewTabGroup.md)

## Description

Read-only property that returns the ViewFrame this view tab Group is located in.

## Syntax

ViewTabGroup.**ViewFrame**() As [ViewFrame](../ViewFrame/ViewFrame.md)

## Property Value

This is a read only property whose value is a [ViewFrame](../ViewFrame/ViewFrame.md).

## Version

Introduced in version 2022
