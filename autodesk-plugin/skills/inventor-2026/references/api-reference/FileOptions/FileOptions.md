# FileOptions Object

## Description

The FileOptions object provides access to properties that provide read and write access of the file related application options.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FileOptions/FileOptions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [ContentCenterPath](../FileOptions/FileOptions_ContentCenterPath.md) | Gets and sets the location of content center components. |
| [DefaultTemplateDrawingStandard](../FileOptions/FileOptions_DefaultTemplateDrawingStandard.md) | Gets and sets the standard for the default template. |
| [DefaultTemplateUnitsAreInches](../FileOptions/FileOptions_DefaultTemplateUnitsAreInches.md) | Gets and sets the units for the default template, the options are either inches or millimeters. |
| [DefaultVBAProjectFileFullFilename](../FileOptions/FileOptions_DefaultVBAProjectFileFullFilename.md) | Gets and sets the full filename of the VBA project file. |
| [DesignDataPath](../FileOptions/FileOptions_DesignDataPath.md) | Gets and sets the location of design data files such as Style files. |
| [EnvironmentPath](../FileOptions/FileOptions_EnvironmentPath.md) | Gets and sets the location of environment files. |
| [FileOpenOptions](../FileOptions/FileOptions_FileOpenOptions.md) | Property that gets the FileOpenOptions object. |
| [PresetsPath](../FileOptions/FileOptions_PresetsPath.md) | Gets and sets the location of presets files. |
| [ProjectsPath](../FileOptions/FileOptions_ProjectsPath.md) | Gets and sets the location of a new project file. |
| [SymbolLibraryPath](../FileOptions/FileOptions_SymbolLibraryPath.md) | Gets and sets the location of sketch symbol library. |
| [TeamWebFullFilename](../FileOptions/FileOptions_TeamWebFullFilename.md) | Gets and sets the full filename of the html file. |
| [TemplatesPath](../FileOptions/FileOptions_TemplatesPath.md) | Gets and sets the location of Inventor template files. |
| [TexturePath](../FileOptions/FileOptions_TexturePath.md) | Gets and sets the location of texture files. |
| [Type](../FileOptions/FileOptions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [UndoPath](../FileOptions/FileOptions_UndoPath.md) | Gets and sets the location of the temporary file used for transactions. |
