# SpellCheckOptions.IgnoreWordsContainingPunctuation Property

Parent Object: [SpellCheckOptions](../SpellCheckOptions/SpellCheckOptions.md)

## Description

Read-write property that gets and sets whether or not to ignore the words with punctuation (e.g. A/3).

## Syntax

SpellCheckOptions.**IgnoreWordsContainingPunctuation**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2020
