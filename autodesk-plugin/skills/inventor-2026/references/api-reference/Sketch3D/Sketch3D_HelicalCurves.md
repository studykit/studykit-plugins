# Sketch3D.HelicalCurves Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Gets the HelicalCurves collection object.

## Syntax

Sketch3D.**HelicalCurves**() As [HelicalCurves](../HelicalCurves/HelicalCurves.md)

## Property Value

This is a read only property whose value is a [HelicalCurves](../HelicalCurves/HelicalCurves.md).

## Version

Introduced in version 2019
