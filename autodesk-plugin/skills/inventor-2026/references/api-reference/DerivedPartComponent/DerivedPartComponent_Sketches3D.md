# DerivedPartComponent.Sketches3D Property

Parent Object: [DerivedPartComponent](../DerivedPartComponent/DerivedPartComponent.md)

## Description

Property that returns the Sketch3D objects that were created as a result of this derived part.

## Syntax

DerivedPartComponent.**Sketches3D**() As [Sketches3DEnumerator](../Sketches3DEnumerator/Sketches3DEnumerator.md)

## Property Value

This is a read only property whose value is a [Sketches3DEnumerator](../Sketches3DEnumerator/Sketches3DEnumerator.md).

## Version

Introduced in version 2009
