# FilenameAssetValue.DisplayName Property

Parent Object: [FilenameAssetValue](../FilenameAssetValue/FilenameAssetValue.md)

## Description

Gets the name of this value as seen in the Material or Appearance Browser. This can change based on the current Inventor language.

## Syntax

FilenameAssetValue.**DisplayName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2014
