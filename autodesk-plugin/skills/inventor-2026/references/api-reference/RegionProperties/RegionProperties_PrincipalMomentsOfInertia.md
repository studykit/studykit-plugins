# RegionProperties.PrincipalMomentsOfInertia Method

Parent Object: [RegionProperties](../RegionProperties/RegionProperties.md)

## Description

Method that returns the moments of inertia about the principal axes.

## Syntax

RegionProperties.**PrincipalMomentsOfInertia**( ***Ix*** As Double, ***Iy*** As Double, ***Iz*** As Double )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Ix | Double | Output Double that specifies the first moment of inertia. |
| Iy | Double | Output Double that specifies the second moment of inertia. |
| Iz | Double | Output Double that specifies the third moment of inertia. |
