# RegionProperties.GetTorsionProperties Method

Parent Object: [RegionProperties](../RegionProperties/RegionProperties.md)

## Description

Method that returns torsion coefficients of sketch profile.

## Syntax

RegionProperties.**GetTorsionProperties**( ***Torsion*** As Double, ***WorstStressValue*** As Double, ***WorstStressPosition*** As [Point](../Point/Point.md), [***FiniteElementNodeCount***] As Long )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Torsion | Double | The torsion value. |
| WorstStressValue | Double | WorstStressValue |
| WorstStressPosition | [Point](../Point/Point.md) | WorstStressPosition |
| FiniteElementNodeCount | Long | FiniteElementNodeCount |

## Version

Introduced in version 2009
