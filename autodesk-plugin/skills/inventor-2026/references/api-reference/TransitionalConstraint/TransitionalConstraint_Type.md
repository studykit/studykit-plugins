# TransitionalConstraint.Type Property

Parent Object: [TransitionalConstraint](../TransitionalConstraint/TransitionalConstraint.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

TransitionalConstraint.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
