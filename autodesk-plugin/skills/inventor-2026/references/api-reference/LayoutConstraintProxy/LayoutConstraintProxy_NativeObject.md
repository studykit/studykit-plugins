# LayoutConstraintProxy.NativeObject Property

Parent Object: [LayoutConstraintProxy](../LayoutConstraintProxy/LayoutConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

LayoutConstraintProxy.**NativeObject**() As [LayoutConstraint](../LayoutConstraint/LayoutConstraint.md)

## Property Value

This is a read only property whose value is a [LayoutConstraint](../LayoutConstraint/LayoutConstraint.md).

## Version

Introduced in version 2010
