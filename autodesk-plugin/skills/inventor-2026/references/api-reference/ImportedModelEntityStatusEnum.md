# ImportedModelEntityStatusEnum Enumerator

## Description

Specifies the include state for a model entity.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAExcludedStatus | 109570 | Indicates a model entity is excluded from the result. |
| kExcludedStatus | 109570 | Indicates a model entity is excluded from the result. |
| kImportedAsIndividualSurfacesStatus | 109571 | Indicates that the surfaces within an entity will be created as individual surfaces. |
| kImportedIndividualStatus | 109572 | Indicates a model entity's individual status. |
| kIncludedStatus | 109569 | Indicates a model entity is included in the result. |

## Version

Introduced in version 2016
