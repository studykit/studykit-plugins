# MergedCell.Parent Property

Parent Object: [MergedCell](../MergedCell/MergedCell.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

MergedCell.**Parent**() As [CustomTable](../CustomTable/CustomTable.md)

## Property Value

This is a read only property whose value is a [CustomTable](../CustomTable/CustomTable.md).

## Version

Introduced in version 2025.1
