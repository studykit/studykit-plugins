# MassProperties.CacheResultsOnCompute Property

Parent Object: [MassProperties](../MassProperties/MassProperties.md)

## Description

Property that deterimines whether mass property calculations are cached with the component.

## Syntax

MassProperties.**CacheResultsOnCompute**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
