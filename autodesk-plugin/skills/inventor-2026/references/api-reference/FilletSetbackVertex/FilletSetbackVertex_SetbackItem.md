# FilletSetbackVertex.SetbackItem Property

Parent Object: [FilletSetbackVertex](../FilletSetbackVertex/FilletSetbackVertex.md)

## Description

Method that returns the specified FilletSetback object from this vertex.

## Syntax

FilletSetbackVertex.**SetbackItem**( ***Index*** As Long ) As [FilletSetback](../FilletSetback/FilletSetback.md)

## Property Value

This is a read only property whose value is a [FilletSetback](../FilletSetback/FilletSetback.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies index of the object to return. |

## Version

Introduced in version 5.3
