# LightingStyle.Lights Property

Parent Object: [LightingStyle](../LightingStyle/LightingStyle.md)

## Description

Property that returns the Lights collection object.

## Syntax

LightingStyle.**Lights**() As [Lights](../Lights/Lights.md)

## Property Value

This is a read only property whose value is a [Lights](../Lights/Lights.md).

## Version

Introduced in version 10
