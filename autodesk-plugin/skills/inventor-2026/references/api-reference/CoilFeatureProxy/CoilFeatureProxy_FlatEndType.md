# CoilFeatureProxy.FlatEndType Property

Parent Object: [CoilFeatureProxy](../CoilFeatureProxy/CoilFeatureProxy.md)

## Description

Property that gets and sets the coil end type at the end of the coil feature.

## Syntax

CoilFeatureProxy.**FlatEndType**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
