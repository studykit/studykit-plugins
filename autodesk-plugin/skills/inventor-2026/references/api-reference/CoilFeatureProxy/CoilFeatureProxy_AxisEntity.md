# CoilFeatureProxy.AxisEntity Property

Parent Object: [CoilFeatureProxy](../CoilFeatureProxy/CoilFeatureProxy.md)

## Description

Property that gets and sets the entity used to define the axis of the coil. Valid input is either a sketch line or work axis. The axis entity must not intersect the profile.

## Syntax

CoilFeatureProxy.**AxisEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 9
