# DirectEditDeleteOperation.DirectEditOperationType Property

Parent Object: [DirectEditDeleteOperation](../DirectEditDeleteOperation/DirectEditDeleteOperation.md)

## Description

Gets the direct edit operation type.

## Syntax

DirectEditDeleteOperation.**DirectEditOperationType**() As [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md)

## Property Value

This is a read only property whose value is a [DirectEditOperationTypeEnum](../DirectEditOperationTypeEnum.md).

## Version

Introduced in version 2015
