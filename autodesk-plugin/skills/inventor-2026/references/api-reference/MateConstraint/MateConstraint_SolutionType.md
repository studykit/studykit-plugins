# MateConstraint.SolutionType Property

Parent Object: [MateConstraint](../MateConstraint/MateConstraint.md)

## Description

Read-only property that returns the solution type for the mate constraint.

## Syntax

MateConstraint.**SolutionType**() As [MateConstraintSolutionTypeEnum](../MateConstraintSolutionTypeEnum.md)

## Property Value

This is a read only property whose value is a [MateConstraintSolutionTypeEnum](../MateConstraintSolutionTypeEnum.md).

## Version

Introduced in version 2019
