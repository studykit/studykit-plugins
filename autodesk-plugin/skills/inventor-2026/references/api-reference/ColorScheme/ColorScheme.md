# ColorScheme Object

## Description

The ColorScheme object represents a single color scheme. It provides access to the various components that make up a color scheme.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Activate](../ColorScheme/ColorScheme_Activate.md) | Method that causes this color scheme to become the active color scheme. |
