# ColorScheme.DisplaySeparateColors Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Read-write property that gets and sets the colors that are used for display separate colors setting. Currently 32 colors are supported for display separate colors.

## Syntax

ColorScheme.**DisplaySeparateColors**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2025
