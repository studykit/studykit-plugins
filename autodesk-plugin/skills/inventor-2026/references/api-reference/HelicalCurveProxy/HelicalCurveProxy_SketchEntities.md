# HelicalCurveProxy.SketchEntities Property

Parent Object: [HelicalCurveProxy](../HelicalCurveProxy/HelicalCurveProxy.md)

## Description

Returns a collection of sketch entities that belong to the helical curve.

## Syntax

HelicalCurveProxy.**SketchEntities**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 2019
