# HelicalCurveProxy.OwnedBy Property

Parent Object: [HelicalCurveProxy](../HelicalCurveProxy/HelicalCurveProxy.md)

## Description

Property indicating entity or entities that own this object.

## Syntax

HelicalCurveProxy.**OwnedBy**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 2019
