# FeatureBasedOccurrencePatternProxy.ParentComponents Property

Parent Object: [FeatureBasedOccurrencePatternProxy](../FeatureBasedOccurrencePatternProxy/FeatureBasedOccurrencePatternProxy.md)

## Description

Property that gets and sets the components used as input for the pattern.

## Syntax

FeatureBasedOccurrencePatternProxy.**ParentComponents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2010
