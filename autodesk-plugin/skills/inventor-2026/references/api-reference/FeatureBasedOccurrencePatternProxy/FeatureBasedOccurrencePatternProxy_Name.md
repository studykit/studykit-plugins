# FeatureBasedOccurrencePatternProxy.Name Property

Parent Object: [FeatureBasedOccurrencePatternProxy](../FeatureBasedOccurrencePatternProxy/FeatureBasedOccurrencePatternProxy.md)

## Description

Gets/Sets the name for the pattern.

## Syntax

FeatureBasedOccurrencePatternProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
