# ObjectDefaultsStyle.BendTableStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for bend tables.

## Syntax

ObjectDefaultsStyle.**BendTableStyle**() As [TableStyle](../TableStyle/TableStyle.md)

## Property Value

This is a read/write property whose value is a [TableStyle](../TableStyle/TableStyle.md).

## Version

Introduced in version 2011
