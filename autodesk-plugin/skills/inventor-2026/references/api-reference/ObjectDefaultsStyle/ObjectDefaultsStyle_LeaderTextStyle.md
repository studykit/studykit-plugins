# ObjectDefaultsStyle.LeaderTextStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default text style for leader text.

## Syntax

ObjectDefaultsStyle.**LeaderTextStyle**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 9
