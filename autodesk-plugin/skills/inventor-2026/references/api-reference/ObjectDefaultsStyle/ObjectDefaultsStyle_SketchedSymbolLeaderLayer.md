# ObjectDefaultsStyle.SketchedSymbolLeaderLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default layer for user defined symbols.

## Syntax

ObjectDefaultsStyle.**SketchedSymbolLeaderLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2010
