# InventorVBAComponents Object

## Description

The InventorVBAComponents collection object represents a collection of objects in an InventorVBAProject, and provides methods to create new InventorVBAComponent objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../InventorVBAComponents/InventorVBAComponents_Add.md) | Method that adds a component to the collection. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../InventorVBAComponents/InventorVBAComponents_Count.md) | Property that returns the number of items in this collection. |
| [Item](../InventorVBAComponents/InventorVBAComponents_Item.md) | Returns the specified InventorVBAComponent object from the collection. This is the default property of the InventorVBAComponents collection object. |
| [Parent](../InventorVBAComponents/InventorVBAComponents_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../InventorVBAComponents/InventorVBAComponents_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[InventorVBAProject.InventorVBAComponents](../InventorVBAProject/InventorVBAProject_InventorVBAComponents.md)

## Version

Introduced in version 6
