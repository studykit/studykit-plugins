# InventorVBAComponents.Add Method

Parent Object: [InventorVBAComponents](../InventorVBAComponents/InventorVBAComponents.md)

## Description

Method that adds a component to the collection.

## Syntax

InventorVBAComponents.**Add**( ***Name*** As String ) As [InventorVBAComponent](../InventorVBAComponent/InventorVBAComponent.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Name | String | Input String that specifies the name of the new component to add to the collection. |

## Version

Introduced in version 6
