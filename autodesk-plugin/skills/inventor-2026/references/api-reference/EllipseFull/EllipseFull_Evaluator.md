# EllipseFull.Evaluator Property

Parent Object: [EllipseFull](../EllipseFull/EllipseFull.md)

## Description

Gets the CurveEvaluator for this ellipse.

## Syntax

EllipseFull.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 4
