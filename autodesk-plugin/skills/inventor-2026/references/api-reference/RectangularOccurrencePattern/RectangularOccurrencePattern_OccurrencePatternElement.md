# RectangularOccurrencePattern.OccurrencePatternElement Property

Parent Object: [RectangularOccurrencePattern](../RectangularOccurrencePattern/RectangularOccurrencePattern.md)

## Description

Property returning a specific element within a pattern. The pattern element returned for row 1, column 1 will contain the components that were provided as input for the pattern.

## Syntax

RectangularOccurrencePattern.**OccurrencePatternElement**( ***Row*** As Long, ***Column*** As Long ) As [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Property Value

This is a read only property whose value is an [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Row | Long | Input Long that specifies the row index of the element to return. |
| Column | Long | Input Long that specifies the column index of the element to return. |

## Version

Introduced in version 6
