# FlangeFeatureProxy.Adaptive Property

Parent Object: [FlangeFeatureProxy](../FlangeFeatureProxy/FlangeFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

FlangeFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
