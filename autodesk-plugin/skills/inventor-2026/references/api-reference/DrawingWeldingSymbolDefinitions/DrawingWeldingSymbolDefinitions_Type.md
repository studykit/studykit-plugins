# DrawingWeldingSymbolDefinitions.Type Property

## Syntax

DrawingWeldingSymbolDefinitions.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
