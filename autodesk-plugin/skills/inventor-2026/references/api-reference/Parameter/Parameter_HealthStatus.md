# Parameter.HealthStatus Property

Parent Object: [Parameter](../Parameter/Parameter.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

Parameter.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).
