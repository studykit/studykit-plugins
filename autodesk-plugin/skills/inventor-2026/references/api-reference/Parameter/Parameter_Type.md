# Parameter.Type Property

Parent Object: [Parameter](../Parameter/Parameter.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

Parameter.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
