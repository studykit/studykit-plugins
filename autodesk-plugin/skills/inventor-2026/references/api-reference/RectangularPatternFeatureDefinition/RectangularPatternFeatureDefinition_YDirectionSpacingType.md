# RectangularPatternFeatureDefinition.YDirectionSpacingType Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the enum that indicates if the occurrences in the y-direction have been fitted within a specified distance. Valid values are kDefault, kFitted and kFitToPathLength.

## Syntax

RectangularPatternFeatureDefinition.**YDirectionSpacingType**() As [PatternSpacingTypeEnum](../PatternSpacingTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PatternSpacingTypeEnum](../PatternSpacingTypeEnum.md).

## Version

Introduced in version 2017
