# RectangularPatternFeatureDefinition.YCount Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the number of instances in the Y direction.

## Syntax

RectangularPatternFeatureDefinition.**YCount**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2017
