# RectangularPatternFeatureDefinition.XDirectionEntity Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the X direction entity. This can be a linear Edge, a WorkAxis, a WorkPlane (normal is used), a planar Face (normal is used), or a GeometryIntent.

## Syntax

RectangularPatternFeatureDefinition.**XDirectionEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2017
