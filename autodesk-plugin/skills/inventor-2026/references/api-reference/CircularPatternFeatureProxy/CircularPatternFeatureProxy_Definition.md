# CircularPatternFeatureProxy.Definition Property

Parent Object: [CircularPatternFeatureProxy](../CircularPatternFeatureProxy/CircularPatternFeatureProxy.md)

## Description

Read-write property that gets and sets the CircularPatternFeatureDefinition object associated with this feature.

## Syntax

CircularPatternFeatureProxy.**Definition**() As [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Property Value

This is a read/write property whose value is a [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md).

## Version

Introduced in version 2017
