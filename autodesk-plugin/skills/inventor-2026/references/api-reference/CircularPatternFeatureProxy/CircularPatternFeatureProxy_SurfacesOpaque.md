# CircularPatternFeatureProxy.SurfacesOpaque Property

Parent Object: [CircularPatternFeatureProxy](../CircularPatternFeatureProxy/CircularPatternFeatureProxy.md)

## Description

Gets and sets whether all the resulting surfaces are opaque or transparent.

## Syntax

CircularPatternFeatureProxy.**SurfacesOpaque**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
