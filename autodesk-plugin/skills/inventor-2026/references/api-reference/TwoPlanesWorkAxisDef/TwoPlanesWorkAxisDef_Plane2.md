# TwoPlanesWorkAxisDef.Plane2 Property

Parent Object: [TwoPlanesWorkAxisDef](../TwoPlanesWorkAxisDef/TwoPlanesWorkAxisDef.md)

## Description

Property that returns the plane of a two plane defined work axis. This object can be a planar Face, WorkPlane, or Sketch object. The work axis is at the intersection of the two planes.

## Syntax

TwoPlanesWorkAxisDef.**Plane2**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
