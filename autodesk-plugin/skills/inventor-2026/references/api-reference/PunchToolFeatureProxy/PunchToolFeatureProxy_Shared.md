# PunchToolFeatureProxy.Shared Property

Parent Object: [PunchToolFeatureProxy](../PunchToolFeatureProxy/PunchToolFeatureProxy.md)

## Description

Gets and sets whether the part feature is shared or not, applies only to surface features.

## Syntax

PunchToolFeatureProxy.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
