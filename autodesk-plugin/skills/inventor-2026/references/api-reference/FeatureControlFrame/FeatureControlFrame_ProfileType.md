# FeatureControlFrame.ProfileType Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Gets and sets the profile type for the feature control frame.

## Syntax

FeatureControlFrame.**ProfileType**() As [FeatureControlFrameProfileTypeEnum](../FeatureControlFrameProfileTypeEnum.md)

## Property Value

This is a read/write property whose value is a [FeatureControlFrameProfileTypeEnum](../FeatureControlFrameProfileTypeEnum.md).

## Version

Introduced in version 2023
