# BendConstraintProxy.Radius Property

Parent Object: [BendConstraintProxy](../BendConstraintProxy/BendConstraintProxy.md)

## Description

Property that returns the parameter controlling the radius of the bend.

## Syntax

BendConstraintProxy.**Radius**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
