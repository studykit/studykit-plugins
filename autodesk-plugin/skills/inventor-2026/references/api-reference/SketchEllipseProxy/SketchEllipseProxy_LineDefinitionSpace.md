# SketchEllipseProxy.LineDefinitionSpace Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Gets and sets the LineDefinitionSpace applied to this sketch line.

## Syntax

SketchEllipseProxy.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 9
