# SketchEllipseProxy.MajorRadius Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Gets and sets the major radius.

## Syntax

SketchEllipseProxy.**MajorRadius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 6
