# SketchEllipseProxy.CurvatureDisplay Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Specifies whether curvature information is displayed for the ellipse.

## Syntax

SketchEllipseProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
