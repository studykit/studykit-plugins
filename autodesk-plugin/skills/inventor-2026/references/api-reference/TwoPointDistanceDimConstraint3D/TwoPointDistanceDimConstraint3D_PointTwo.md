# TwoPointDistanceDimConstraint3D.PointTwo Property

Parent Object: [TwoPointDistanceDimConstraint3D](../TwoPointDistanceDimConstraint3D/TwoPointDistanceDimConstraint3D.md)

## Description

Property that returns the SketchPoint3D object that defines the second point being constrained.

## Syntax

TwoPointDistanceDimConstraint3D.**PointTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
