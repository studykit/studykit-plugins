# SketchCircle3DProxy.CurvatureDisplay Property

Parent Object: [SketchCircle3DProxy](../SketchCircle3DProxy/SketchCircle3DProxy.md)

## Description

Specifies whether curvature information is displayed for the circle.

## Syntax

SketchCircle3DProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
