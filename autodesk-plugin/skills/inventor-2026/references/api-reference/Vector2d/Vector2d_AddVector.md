# Vector2d.AddVector Method

Parent Object: [Vector2d](../Vector2d/Vector2d.md)

## Description

Add the specified vector2d to this vector2d.

## Syntax

Vector2d.**AddVector**( ***Value*** As [Vector2d](../Vector2d/Vector2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | [Vector2d](../Vector2d/Vector2d.md) | Vector2d to add. |

## Version

Introduced in version 11
