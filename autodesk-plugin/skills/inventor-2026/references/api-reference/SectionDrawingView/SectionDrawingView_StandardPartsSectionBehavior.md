# SectionDrawingView.StandardPartsSectionBehavior Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets the sectioning behavior for standard parts in drawing views of assemblies.

## Syntax

SectionDrawingView.**StandardPartsSectionBehavior**() As [StandardPartsSectionBehaviorEnum](../StandardPartsSectionBehaviorEnum.md)

## Property Value

This is a read/write property whose value is a [StandardPartsSectionBehaviorEnum](../StandardPartsSectionBehaviorEnum.md).

## Version

Introduced in version 2010
