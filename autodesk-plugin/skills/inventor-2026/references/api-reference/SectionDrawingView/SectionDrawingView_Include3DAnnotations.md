# SectionDrawingView.Include3DAnnotations Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Read-write property that gets and sets whether to include the 3D annotations.

## Syntax

SectionDrawingView.**Include3DAnnotations**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
