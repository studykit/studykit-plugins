# SectionDrawingView.GraphicsDataSetsCollection Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that returns the object for the drawing view.

## Syntax

SectionDrawingView.**GraphicsDataSetsCollection**() As [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md).

## Version

Introduced in version 9
