# Light.Parent Property

Parent Object: [Light](../Light/Light.md)

## Description

Property returning the parent of the object.

## Syntax

Light.**Parent**() As [LightingStyle](../LightingStyle/LightingStyle.md)

## Property Value

This is a read only property whose value is a [LightingStyle](../LightingStyle/LightingStyle.md).

## Version

Introduced in version 10
