# Environment.RadialMarkingMenus Property

Parent Object: [Environment](../Environment/Environment.md)

## Description

Returns the RadialMarkingMenus object that can be used to query and set the selection based radial menu.

## Syntax

Environment.**RadialMarkingMenus**() As [RadialMarkingMenus](../RadialMarkingMenus/RadialMarkingMenus.md)

## Property Value

This is a read only property whose value is a [RadialMarkingMenus](../RadialMarkingMenus/RadialMarkingMenus.md).

## Version

Introduced in version 2013
