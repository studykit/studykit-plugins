# Environment.DisabledCommandTypes Property

Parent Object: [Environment](../Environment/Environment.md)

## Description

Gets list of disabled command types for the Environment.

## Syntax

Environment.**DisabledCommandTypes**() As [CommandTypesEnum](../CommandTypesEnum.md)

## Property Value

This is a read/write property whose value is a [CommandTypesEnum](../CommandTypesEnum.md).

## Version

Introduced in version 10
