# CoilFeature.ExtentType Property

Parent Object: [CoilFeature](../CoilFeature/CoilFeature.md)

## Description

Property that returns an enum indicating how the extent of the feature is defined. This indicates the type of object returned by the Extent property.

## Syntax

CoilFeature.**ExtentType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).

## Version

Introduced in version 5.3
