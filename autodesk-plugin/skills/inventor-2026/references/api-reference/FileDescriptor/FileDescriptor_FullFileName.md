# FileDescriptor.FullFileName Property

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

Property that returns the full path name of the file as currently found (or the last known full file name, if the reference is not found).

## Syntax

FileDescriptor.**FullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 11
