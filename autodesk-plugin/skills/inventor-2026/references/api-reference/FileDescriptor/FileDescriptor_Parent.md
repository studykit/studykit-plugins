# FileDescriptor.Parent Property

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

Property that returns the parent File object of this descriptor.

## Syntax

FileDescriptor.**Parent**() As [File](../File/File.md)

## Property Value

This is a read only property whose value is a [File](../File/File.md).

## Version

Introduced in version 11
