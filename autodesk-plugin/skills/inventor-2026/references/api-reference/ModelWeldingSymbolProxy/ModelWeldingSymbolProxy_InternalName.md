# ModelWeldingSymbolProxy.InternalName Property

Parent Object: [ModelWeldingSymbolProxy](../ModelWeldingSymbolProxy/ModelWeldingSymbolProxy.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelWeldingSymbolProxy.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2024
