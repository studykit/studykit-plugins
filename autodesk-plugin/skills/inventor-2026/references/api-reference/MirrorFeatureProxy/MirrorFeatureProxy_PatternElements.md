# MirrorFeatureProxy.PatternElements Property

Parent Object: [MirrorFeatureProxy](../MirrorFeatureProxy/MirrorFeatureProxy.md)

## Description

Gets the FeaturePatternElements collection that contains the elements created by this pattern.

## Syntax

MirrorFeatureProxy.**PatternElements**() As [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md)

## Property Value

This is a read only property whose value is a [FeaturePatternElements](../FeaturePatternElements/FeaturePatternElements.md).

## Version

Introduced in version 2010
