# UserParameter.DisplayFormat Property

Parent Object: [UserParameter](../UserParameter/UserParameter.md)

## Description

Gets and sets the display format for the parameter.

## Syntax

UserParameter.**DisplayFormat**() As [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md)

## Property Value

This is a read/write property whose value is a [ParameterDisplayFormatEnum](../ParameterDisplayFormatEnum.md).

## Version

Introduced in version 2011
