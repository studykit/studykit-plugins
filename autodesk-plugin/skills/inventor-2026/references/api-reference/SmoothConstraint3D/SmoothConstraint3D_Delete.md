# SmoothConstraint3D.Delete Method

Parent Object: [SmoothConstraint3D](../SmoothConstraint3D/SmoothConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

SmoothConstraint3D.**Delete**()

## Version

Introduced in version 11
