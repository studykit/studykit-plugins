# Weld.SymbolBreakPoint Property

Parent Object: [Weld](../Weld/Weld.md)

## Description

Property that returns the coordinate where the weld symbol leader line break point is.

## Syntax

Weld.**SymbolBreakPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 8
