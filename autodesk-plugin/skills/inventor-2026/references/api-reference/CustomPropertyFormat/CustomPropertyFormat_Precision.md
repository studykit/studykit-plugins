# CustomPropertyFormat.Precision Property

Parent Object: [CustomPropertyFormat](../CustomPropertyFormat/CustomPropertyFormat.md)

## Description

Gets/Sets the precision to use when formatting the parameter value to be displayed as an iProperty.

## Syntax

CustomPropertyFormat.**Precision**() As [CustomPropertyPrecisionEnum](../CustomPropertyPrecisionEnum.md)

## Property Value

This is a read/write property whose value is a [CustomPropertyPrecisionEnum](../CustomPropertyPrecisionEnum.md).

## Version

Introduced in version 2010
