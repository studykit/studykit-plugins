# ModelDatumTarget.TextPosition Property

Parent Object: [ModelDatumTarget](../ModelDatumTarget/ModelDatumTarget.md)

## Description

Gets and sets the position of the datum target text.

## Syntax

ModelDatumTarget.**TextPosition**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2023
