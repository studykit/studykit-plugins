# SketchSplineHandle3DProxy.Type Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchSplineHandle3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2008
