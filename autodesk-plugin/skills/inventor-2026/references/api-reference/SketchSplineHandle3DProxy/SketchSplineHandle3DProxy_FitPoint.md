# SketchSplineHandle3DProxy.FitPoint Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Property that returns the fit point associated with the handle.

## Syntax

SketchSplineHandle3DProxy.**FitPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 2008
