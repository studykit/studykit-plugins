# ShrinkwrapDefinition.EnvelopesReplaceStyle Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Read-write property that gets and sets envelopes replace style. When a new definition is created, this defaults to kNoneReplaceStyle.

## Syntax

ShrinkwrapDefinition.**EnvelopesReplaceStyle**() As [EnvelopesReplaceStyleEnum](../EnvelopesReplaceStyleEnum.md)

## Property Value

This is a read/write property whose value is an [EnvelopesReplaceStyleEnum](../EnvelopesReplaceStyleEnum.md).

## Version

Introduced in version 2022
