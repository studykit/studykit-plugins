# BaselineDimensionSets Object

## Description

The BaselineDimensionSets collection object provides access to all of the baseline dimension sets on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../BaselineDimensionSets/BaselineDimensionSets_Add.md) | Method that creates a baseline dimension set and returns the newly created BaselineDimensionSet object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BaselineDimensionSets/BaselineDimensionSets_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BaselineDimensionSets/BaselineDimensionSets_Count.md) | Property that returns the number of items in the collection. |
| [Item](../BaselineDimensionSets/BaselineDimensionSets_Item.md) | Method that returns the specified dimension set object from the collection. |
| [Type](../BaselineDimensionSets/BaselineDimensionSets_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
