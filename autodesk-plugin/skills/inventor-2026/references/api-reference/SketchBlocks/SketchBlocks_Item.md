# SketchBlocks.Item Property

Parent Object: [SketchBlocks](../SketchBlocks/SketchBlocks.md)

## Description

Returns the specified SketchBlock object from the collection.

## Syntax

SketchBlocks.**Item**( ***Index*** As Variant ) As [SketchBlock](../SketchBlock/SketchBlock.md)

## Property Value

This is a read only property whose value is a [SketchBlock](../SketchBlock/SketchBlock.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the SketchBlock to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the SketchBlock name. If an out of range index or a name of a non\-existent SketchBlock is provided, an error will occur. |

## Version

Introduced in version 2010
