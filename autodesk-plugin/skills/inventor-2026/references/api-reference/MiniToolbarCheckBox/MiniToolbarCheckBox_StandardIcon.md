# MiniToolbarCheckBox.StandardIcon Property

Parent Object: [MiniToolbarCheckBox](../MiniToolbarCheckBox/MiniToolbarCheckBox.md)

## Description

Read-write property that gets and sets the standard (small) icon for the control.

## Syntax

MiniToolbarCheckBox.**StandardIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2012
