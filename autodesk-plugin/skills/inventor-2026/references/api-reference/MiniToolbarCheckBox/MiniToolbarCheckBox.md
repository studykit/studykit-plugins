# MiniToolbarCheckBox Object

## Description

The MiniToolbarCheckBox object represents a check box control within a MiniToolbar.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../MiniToolbarCheckBox/MiniToolbarCheckBox_Delete.md) | Method that deletes the control. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MiniToolbarCheckBox/MiniToolbarCheckBox_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AutoHide](../MiniToolbarCheckBox/MiniToolbarCheckBox_AutoHide.md) | Read-write property that gets and sets whether the control should be automatically hidden when the user moves the cursor away from the MiniToolbar. |
| [Checked](../MiniToolbarCheckBox/MiniToolbarCheckBox_Checked.md) | Read-write property that gets and sets whether the check box is currently checked. |
| [ControlType](../MiniToolbarCheckBox/MiniToolbarCheckBox_ControlType.md) | Read-only property that returns the control type. |
| [DisplayName](../MiniToolbarCheckBox/MiniToolbarCheckBox_DisplayName.md) | Read-write property that gets and sets the display name for the control. |
| [Enabled](../MiniToolbarCheckBox/MiniToolbarCheckBox_Enabled.md) | Read-write property that gets and sets whether the control is enabled. |
| [Index](../MiniToolbarCheckBox/MiniToolbarCheckBox_Index.md) | Read-only property that returns the index this control is currently positioned at within the mini toolbar. |
| [InternalName](../MiniToolbarCheckBox/MiniToolbarCheckBox_InternalName.md) | Read-only property that returns a string uniquely identifying this control within the toolbar. |
| [LargeIcon](../MiniToolbarCheckBox/MiniToolbarCheckBox_LargeIcon.md) | Read-write property that gets and sets the large icon for the control. |
| [Parent](../MiniToolbarCheckBox/MiniToolbarCheckBox_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [StandardIcon](../MiniToolbarCheckBox/MiniToolbarCheckBox_StandardIcon.md) | Read-write property that gets and sets the standard (small) icon for the control. |
| [ToolTipText](../MiniToolbarCheckBox/MiniToolbarCheckBox_ToolTipText.md) | Read-write property that gets and sets the tooltip text for the control. |
| [Type](../MiniToolbarCheckBox/MiniToolbarCheckBox_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [Visible](../MiniToolbarCheckBox/MiniToolbarCheckBox_Visible.md) | Read-write property that gets and sets whether the control is visible. |
