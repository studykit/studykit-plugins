# EllipticalArc.Copy Method

Parent Object: [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Description

Creates a copy of this EllipticalArc object. The result is entirely independent and can be edited without affecting the original EllipticalArc object.

## Syntax

EllipticalArc.**Copy**() As [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Version

Introduced in version 2013
