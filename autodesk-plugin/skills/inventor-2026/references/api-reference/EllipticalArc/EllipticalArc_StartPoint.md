# EllipticalArc.StartPoint Property

Parent Object: [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Description

Gets the start point for the arc.

## Syntax

EllipticalArc.**StartPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
