# EllipticalArc.Center Property

Parent Object: [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Description

Specifies the center point for the arc.

## Syntax

EllipticalArc.**Center**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
