# DSDegreeOfFreedom.Results Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets the DSResults object that provides access to the simulation results associated with this degree of freedom.

## Syntax

DSDegreeOfFreedom.**Results**() As [DSResults](../DSResults/DSResults.md)

## Property Value

This is a read only property whose value is a [DSResults](../DSResults/DSResults.md).

## Version

Introduced in version 2013
