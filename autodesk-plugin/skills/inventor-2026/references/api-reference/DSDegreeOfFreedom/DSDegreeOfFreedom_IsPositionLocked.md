# DSDegreeOfFreedom.IsPositionLocked Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets and sets if the position of this joint is locked.

## Syntax

DSDegreeOfFreedom.**IsPositionLocked**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
