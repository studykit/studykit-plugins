# DSDegreeOfFreedom.LoadDamping Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets a DSValue object that you can use to get and set the load damping. This property returns Nothing in the case where IsLoadEnabled is False.

## Syntax

DSDegreeOfFreedom.**LoadDamping**() As [DSValue](../DSValue/DSValue.md)

## Property Value

This is a read only property whose value is a [DSValue](../DSValue/DSValue.md).

## Version

Introduced in version 2013
