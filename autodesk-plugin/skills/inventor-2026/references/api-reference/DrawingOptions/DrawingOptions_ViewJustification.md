# DrawingOptions.ViewJustification Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the default drawing view justification.

## Syntax

DrawingOptions.**ViewJustification**() As [ViewJustificationEnum](../ViewJustificationEnum.md)

## Property Value

This is a read/write property whose value is a [ViewJustificationEnum](../ViewJustificationEnum.md).

## Version

Introduced in version 2008
