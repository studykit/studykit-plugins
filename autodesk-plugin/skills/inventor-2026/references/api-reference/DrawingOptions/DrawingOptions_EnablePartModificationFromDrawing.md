# DrawingOptions.EnablePartModificationFromDrawing Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Enables/disables part modification from within a drawings.

## Syntax

DrawingOptions.**EnablePartModificationFromDrawing**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
