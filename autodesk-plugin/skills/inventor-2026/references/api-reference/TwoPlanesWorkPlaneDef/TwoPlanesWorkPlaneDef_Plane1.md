# TwoPlanesWorkPlaneDef.Plane1 Property

Parent Object: [TwoPlanesWorkPlaneDef](../TwoPlanesWorkPlaneDef/TwoPlanesWorkPlaneDef.md)

## Description

Property that returns the first plane of the two-plane-defined work plane This object can be a planar Face or a WorkPlane object.

## Syntax

TwoPlanesWorkPlaneDef.**Plane1**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
