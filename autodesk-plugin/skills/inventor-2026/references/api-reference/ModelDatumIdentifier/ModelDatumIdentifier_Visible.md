# ModelDatumIdentifier.Visible Property

Parent Object: [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md)

## Description

Gets and sets the visibility of the annotation.

## Syntax

ModelDatumIdentifier.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
