# DrawingCurve.EndPoint Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the end point of the curve in sheet space. This property returns Nothing for circular curves.

## Syntax

DrawingCurve.**EndPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 11
