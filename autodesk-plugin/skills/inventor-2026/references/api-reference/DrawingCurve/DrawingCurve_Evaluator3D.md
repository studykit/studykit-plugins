# DrawingCurve.Evaluator3D Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the CurveEvaluator object for this curve in 3d model view space.

## Syntax

DrawingCurve.**Evaluator3D**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 11
