# DrawingCurve.LineType Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Gets and sets the line type override for the curve.

## Syntax

DrawingCurve.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2009
