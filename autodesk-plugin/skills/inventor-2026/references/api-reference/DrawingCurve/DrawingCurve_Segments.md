# DrawingCurve.Segments Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the collection of the selectable segments representing this curve.

## Syntax

DrawingCurve.**Segments**() As [DrawingCurveSegments](../DrawingCurveSegments/DrawingCurveSegments.md)

## Property Value

This is a read only property whose value is a [DrawingCurveSegments](../DrawingCurveSegments/DrawingCurveSegments.md).

## Version

Introduced in version 11
