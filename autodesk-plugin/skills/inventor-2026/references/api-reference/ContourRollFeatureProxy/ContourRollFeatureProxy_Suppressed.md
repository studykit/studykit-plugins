# ContourRollFeatureProxy.Suppressed Property

Parent Object: [ContourRollFeatureProxy](../ContourRollFeatureProxy/ContourRollFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

ContourRollFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
