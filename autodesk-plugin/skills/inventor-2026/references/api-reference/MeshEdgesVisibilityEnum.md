# MeshEdgesVisibilityEnum Enumerator

## Description

Defines the visibility types of the mesh edges.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllMeshEdgesVisible | 111361 | All mesh edges are visiable. |
| kNoMeshEdgesVisible | 111362 | No mesh edges are visiable. |
| kSomeMeshEdgesVisible | 111363 | Some mesh edges are visiable. |

## Version

Introduced in version 2017
