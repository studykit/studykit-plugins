# ModelStates.ModelStateTable Property

Parent Object: [ModelStates](../ModelStates/ModelStates.md)

## Description

Read-only property that gets the ModelStateTable object that represents the model states table. If the ModelStates is from a model state member document then this property will return Nothing.

## Syntax

ModelStates.**ModelStateTable**() As [ModelStateTable](../ModelStateTable/ModelStateTable.md)

## Property Value

This is a read only property whose value is a [ModelStateTable](../ModelStateTable/ModelStateTable.md).

## Version

Introduced in version 2022
