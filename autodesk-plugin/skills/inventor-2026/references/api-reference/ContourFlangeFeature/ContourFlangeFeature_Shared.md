# ContourFlangeFeature.Shared Property

Parent Object: [ContourFlangeFeature](../ContourFlangeFeature/ContourFlangeFeature.md)

## Description

Gets and sets whether the part feature is shared or not, applies only to surface features.

## Syntax

ContourFlangeFeature.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
