# SketchSpline3D.StartTangentScale Property

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Gets and sets the tangent scale parameter at the start point of the spline.

## Syntax

SketchSpline3D.**StartTangentScale**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 8
