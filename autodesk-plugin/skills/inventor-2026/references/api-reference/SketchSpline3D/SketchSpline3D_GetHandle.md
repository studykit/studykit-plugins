# SketchSpline3D.GetHandle Method

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Method that returns the fit point handle at the given fit point. The method returns Nothing if the handle is not enabled at the input fit point.

## Syntax

SketchSpline3D.**GetHandle**( ***FitPoint*** As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md) ) As [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FitPoint | [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md) | Input SketchPoint3D object that specifies the fit point at which to obtain the handle. |

## Version

Introduced in version 2008
