# SketchSpline3D.FitPointCount Property

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Property that returns the number of fit points for the spline.

## Syntax

SketchSpline3D.**FitPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 8
