# SketchSpline3D.Closed Property

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Specifies whether the curve is closed. Setting a curve to be closed will cause it to close and be periodic.

## Syntax

SketchSpline3D.**Closed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
