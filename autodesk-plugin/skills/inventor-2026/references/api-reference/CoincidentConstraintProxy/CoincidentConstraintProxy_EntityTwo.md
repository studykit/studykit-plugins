# CoincidentConstraintProxy.EntityTwo Property

Parent Object: [CoincidentConstraintProxy](../CoincidentConstraintProxy/CoincidentConstraintProxy.md)

## Description

Property that returns the sketch entity being constrained.

## Syntax

CoincidentConstraintProxy.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
