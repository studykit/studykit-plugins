# CoincidentConstraintProxy.NativeObject Property

Parent Object: [CoincidentConstraintProxy](../CoincidentConstraintProxy/CoincidentConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

CoincidentConstraintProxy.**NativeObject**() As [CoincidentConstraint](../CoincidentConstraint/CoincidentConstraint.md)

## Property Value

This is a read only property whose value is a [CoincidentConstraint](../CoincidentConstraint/CoincidentConstraint.md).

## Version

Introduced in version 6
