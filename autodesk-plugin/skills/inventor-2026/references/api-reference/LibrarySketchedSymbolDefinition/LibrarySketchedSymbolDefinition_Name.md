# LibrarySketchedSymbolDefinition.Name Property

Parent Object: [LibrarySketchedSymbolDefinition](../LibrarySketchedSymbolDefinition/LibrarySketchedSymbolDefinition.md)

## Description

Read-only property that returns the name of the library sketched symbol definition.

## Syntax

LibrarySketchedSymbolDefinition.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
