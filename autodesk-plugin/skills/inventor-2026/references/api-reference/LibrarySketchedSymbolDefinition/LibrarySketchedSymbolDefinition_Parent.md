# LibrarySketchedSymbolDefinition.Parent Property

Parent Object: [LibrarySketchedSymbolDefinition](../LibrarySketchedSymbolDefinition/LibrarySketchedSymbolDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

LibrarySketchedSymbolDefinition.**Parent**() As [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md)

## Property Value

This is a read only property whose value is a [SketchedSymbolDefinitionLibrary](../SketchedSymbolDefinitionLibrary/SketchedSymbolDefinitionLibrary.md).

## Version

Introduced in version 2016
