# SketchArc.DisabledActionTypes Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Gets and sets whether specific user actions are blocked on this sketch geometry.

## Syntax

SketchArc.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 9
