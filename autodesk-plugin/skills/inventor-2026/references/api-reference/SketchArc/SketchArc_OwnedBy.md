# SketchArc.OwnedBy Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Indicates entity or entities that own this object.

## Syntax

SketchArc.**OwnedBy**() As [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md).

## Version

Introduced in version 11
