# SketchArc.CenterSketchPoint Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Property that gets the sketch point that defines the position of the center of the arc.

## Syntax

SketchArc.**CenterSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).
