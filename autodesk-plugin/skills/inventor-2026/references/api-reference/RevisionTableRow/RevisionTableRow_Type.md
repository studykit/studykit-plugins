# RevisionTableRow.Type Property

Parent Object: [RevisionTableRow](../RevisionTableRow/RevisionTableRow.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RevisionTableRow.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 10
