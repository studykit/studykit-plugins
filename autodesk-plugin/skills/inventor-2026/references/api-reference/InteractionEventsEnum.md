# InteractionEventsEnum Enumerator

## Description

Constants identifying which events are active during Interaction.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kKeyboardInteraction | 4 | Only the Keyboard Events is active. |
| kMouseAndKeyboardInteraction | 6 | Both, the Mouse and Keyboard Events are active. |
| kMouseInteraction | 2 | Only the MouseEvents is active. |
| kNoInteraction | 0 | None of the Events are active. |
| kSelectAndKeyboardInteraction | 5 | Both, the Select and Keyboard Events are active. |
| kSelectInteraction | 1 | Only the Select Events is active. |

## Version

Introduced in version 5
