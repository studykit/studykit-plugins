# EdgeUseProxy.CurveType Property

Parent Object: [EdgeUseProxy](../EdgeUseProxy/EdgeUseProxy.md)

## Description

Gets the type of the underlying curve geometry that this EdgeUse is defined by.

## Syntax

EdgeUseProxy.**CurveType**() As [Curve2dTypeEnum](../Curve2dTypeEnum.md)

## Property Value

This is a read only property whose value is a [Curve2dTypeEnum](../Curve2dTypeEnum.md).

## Version

Introduced in version 4
