# FinishParameter.DrivenBy Property

Parent Object: [FinishParameter](../FinishParameter/FinishParameter.md)

## Description

Method that returns the collection objects that the parameter is dependent on.

## Syntax

FinishParameter.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2024
