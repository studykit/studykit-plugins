# DrawingViewLabel.HorizontalJustification Property

Parent Object: [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Description

Read-write property that gets and sets the horizontal justification override of the drawing view label.

## Syntax

DrawingViewLabel.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 2009
