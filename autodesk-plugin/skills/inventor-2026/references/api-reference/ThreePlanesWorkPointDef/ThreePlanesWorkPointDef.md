# ThreePlanesWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point defined by the intersection of three planes.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef_GetData.md) | Method that gets all of the data defining a work point at the intersection of three planes. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef_Parent.md) | Property returning the parent WorkPoint object. |
| [Plane1](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef_Plane1.md) | Property that returns the first plane of a work point that is defined by the intersection of three planes. This object can be a planar Face, WorkPlane, or Sketch object. |
| [Plane2](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef_Plane2.md) | Property that returns the second plane of a work point that is defined by the intersection of three planes. This object can be a planar Face, WorkPlane, or Sketch object. |
| [Plane3](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef_Plane3.md) | Property that returns the third plane of a work point that is defined by the intersection of three planes. This object can be a planar Face, WorkPlane, or Sketch object. |
| [Type](../ThreePlanesWorkPointDef/ThreePlanesWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
