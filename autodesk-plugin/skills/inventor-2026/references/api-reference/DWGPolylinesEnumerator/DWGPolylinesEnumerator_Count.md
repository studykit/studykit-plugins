# DWGPolylinesEnumerator.Count Property

Parent Object: [DWGPolylinesEnumerator](../DWGPolylinesEnumerator/DWGPolylinesEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGPolylinesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
