# FeaturePatternElementProxy.Index Property

Parent Object: [FeaturePatternElementProxy](../FeaturePatternElementProxy/FeaturePatternElementProxy.md)

## Description

Property that returns the index of this element within the pattern.

## Syntax

FeaturePatternElementProxy.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
