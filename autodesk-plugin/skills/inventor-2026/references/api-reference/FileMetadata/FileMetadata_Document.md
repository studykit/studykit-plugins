# FileMetadata.Document Property

Parent Object: [FileMetadata](../FileMetadata/FileMetadata.md)

## Description

Gets and sets the document object for which the file name and properties are being requested. This can be set to Nothing if the document hasn’t yet been created.

## Syntax

FileMetadata.**Document**() As [Document](../Document/Document.md)

## Property Value

This is a read/write property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 2009
