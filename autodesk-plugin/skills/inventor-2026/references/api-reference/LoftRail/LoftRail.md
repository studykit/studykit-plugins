# LoftRail Object

## Description

The LoftRail object is used to define a rail in a loft feature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../LoftRail/LoftRail_Delete.md) | Method that deletes this loft rail. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LoftRail/LoftRail_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Condition](../LoftRail/LoftRail_Condition.md) | Returns an enum indicating the condition of the loft at the rail if the rail has conditions associated with it. |
| [HasConditions](../LoftRail/LoftRail_HasConditions.md) | Property that specifies whether the rail has conditions associated with it. |
| [Impact](../LoftRail/LoftRail_Impact.md) | Property that returns the Parameter object that controls the amount of influence the rail has on the shape of the entire loft. |
| [Rail](../LoftRail/LoftRail_Rail.md) | Property that gets the rail for the loft feature. |
| [Type](../LoftRail/LoftRail_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[LoftRails.Add](../LoftRails/LoftRails_Add.md), [LoftRails.Item](../LoftRails/LoftRails_Item.md)

## Version

Introduced in version 11
