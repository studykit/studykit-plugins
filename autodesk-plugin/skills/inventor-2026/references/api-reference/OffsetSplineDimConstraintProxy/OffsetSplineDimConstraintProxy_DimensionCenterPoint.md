# OffsetSplineDimConstraintProxy.DimensionCenterPoint Property

Parent Object: [OffsetSplineDimConstraintProxy](../OffsetSplineDimConstraintProxy/OffsetSplineDimConstraintProxy.md)

## Description

Gets the center of the dimension line.

## Syntax

OffsetSplineDimConstraintProxy.**DimensionCenterPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 11
