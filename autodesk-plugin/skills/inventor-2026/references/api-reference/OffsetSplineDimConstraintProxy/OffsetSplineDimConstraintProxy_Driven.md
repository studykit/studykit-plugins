# OffsetSplineDimConstraintProxy.Driven Property

Parent Object: [OffsetSplineDimConstraintProxy](../OffsetSplineDimConstraintProxy/OffsetSplineDimConstraintProxy.md)

## Description

Gets and sets whether this dimension constraint is driving the geometry or the geometry is defining the value associated with the constraint.

## Syntax

OffsetSplineDimConstraintProxy.**Driven**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
