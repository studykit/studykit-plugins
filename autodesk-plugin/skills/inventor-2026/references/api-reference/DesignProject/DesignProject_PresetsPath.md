# DesignProject.PresetsPath Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets the directory in which to look for the Presets path. If set, this value overrides the corresponding directory.

## Syntax

DesignProject.**PresetsPath**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2020
