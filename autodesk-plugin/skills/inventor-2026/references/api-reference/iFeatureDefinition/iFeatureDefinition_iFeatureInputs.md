# iFeatureDefinition.iFeatureInputs Property

Parent Object: [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md)

## Description

Property that indicates the input information of the iFeature.

## Syntax

iFeatureDefinition.**iFeatureInputs**() As [iFeatureInputs](../iFeatureInputs/iFeatureInputs.md)

## Property Value

This is a read only property whose value is an [iFeatureInputs](../iFeatureInputs/iFeatureInputs.md).
