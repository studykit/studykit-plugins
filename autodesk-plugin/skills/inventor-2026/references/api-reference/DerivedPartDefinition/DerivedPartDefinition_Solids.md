# DerivedPartDefinition.Solids Property

Parent Object: [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Description

Property that gets the collection of solids available in the selected part along with the inclusion option for each solid.

## Syntax

DerivedPartDefinition.**Solids**() As [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md)

## Property Value

This is a read only property whose value is a [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md).

## Version

Introduced in version 2010
