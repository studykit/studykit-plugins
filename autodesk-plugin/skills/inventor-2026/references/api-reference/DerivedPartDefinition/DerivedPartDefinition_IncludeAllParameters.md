# DerivedPartDefinition.IncludeAllParameters Property

Parent Object: [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Description

Property that defines whether exported parameters are included in the derived part. This property is initialized to True when the DerivedPartDefinition object is created.

## Syntax

DerivedPartDefinition.**IncludeAllParameters**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
