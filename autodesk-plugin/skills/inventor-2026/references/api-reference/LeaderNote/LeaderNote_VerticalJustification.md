# LeaderNote.VerticalJustification Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Gets/Sets vertical justification of the DrawingNote.

## Syntax

LeaderNote.**VerticalJustification**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 10
