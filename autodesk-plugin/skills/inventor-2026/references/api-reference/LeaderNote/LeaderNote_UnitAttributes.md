# LeaderNote.UnitAttributes Property

Parent Object: [LeaderNote](../LeaderNote/LeaderNote.md)

## Description

Get the unit attributes associated with the LeaderNote.

## Syntax

LeaderNote.**UnitAttributes**() As [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Property Value

This is a read only property whose value is a [UnitAttributes](../UnitAttributes/UnitAttributes.md).

## Version

Introduced in version 2016
