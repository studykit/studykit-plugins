# DrawingStandardStyle.PresetLineWeights Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets an array of Doubles that specify the set of default values for line weights.

## Syntax

DrawingStandardStyle.**PresetLineWeights**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2010
