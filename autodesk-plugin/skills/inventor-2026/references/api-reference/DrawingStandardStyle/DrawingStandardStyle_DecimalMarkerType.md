# DrawingStandardStyle.DecimalMarkerType Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets the character to use as a decimal marker.

## Syntax

DrawingStandardStyle.**DecimalMarkerType**() As [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md).

## Version

Introduced in version 2010
