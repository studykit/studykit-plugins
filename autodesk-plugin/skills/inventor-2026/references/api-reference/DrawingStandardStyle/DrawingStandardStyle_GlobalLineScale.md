# DrawingStandardStyle.GlobalLineScale Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets the line scale for all line styles in the drawing.

## Syntax

DrawingStandardStyle.**GlobalLineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
