# DrawingStandardStyle.SaveToGlobal Method

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

DrawingStandardStyle.**SaveToGlobal**()

## Version

Introduced in version 9
