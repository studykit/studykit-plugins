# DrawingStandardStyle.PresetScales Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets an array of Strings that specify the set of default values for drawing view scales.

## Syntax

DrawingStandardStyle.**PresetScales**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2010
