# ReferenceFeature.Suppressed Property

Parent Object: [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

ReferenceFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
