# ReferenceFeature.Name Property

Parent Object: [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

ReferenceFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5.3
