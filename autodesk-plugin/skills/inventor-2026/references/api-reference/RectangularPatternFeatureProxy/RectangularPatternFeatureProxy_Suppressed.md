# RectangularPatternFeatureProxy.Suppressed Property

Parent Object: [RectangularPatternFeatureProxy](../RectangularPatternFeatureProxy/RectangularPatternFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

RectangularPatternFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
