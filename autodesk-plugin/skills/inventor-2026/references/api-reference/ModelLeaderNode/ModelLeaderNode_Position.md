# ModelLeaderNode.Position Property

Parent Object: [ModelLeaderNode](../ModelLeaderNode/ModelLeaderNode.md)

## Description

Gets and sets the leader node location on the sheet.

## Syntax

ModelLeaderNode.**Position**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
