# SketchHolePlacementDefinition.HoleCenterPoints Property

Parent Object: [SketchHolePlacementDefinition](../SketchHolePlacementDefinition/SketchHolePlacementDefinition.md)

## Description

Gets and sets the set of sketch points that define the centers of the holes of this feature.

## Syntax

SketchHolePlacementDefinition.**HoleCenterPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 10
