# FileOpenOptions.DefaultDesignViewInPart Property

Parent Object: [FileOpenOptions](../FileOpenOptions/FileOpenOptions.md)

## Description

Gets and sets the default design view representation to use when opening part documents.

## Syntax

FileOpenOptions.**DefaultDesignViewInPart**() As [DesignViewTypeEnum](../DesignViewTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DesignViewTypeEnum](../DesignViewTypeEnum.md).

## Version

Introduced in version 2022
