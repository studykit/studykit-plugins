# BoundaryPatchFeatureProxy.Definition Property

Parent Object: [BoundaryPatchFeatureProxy](../BoundaryPatchFeatureProxy/BoundaryPatchFeatureProxy.md)

## Description

Property that returns the BoundaryPatchDefinition object which defines the various input used to create the boundary patch feature.

## Syntax

BoundaryPatchFeatureProxy.**Definition**() As [BoundaryPatchDefinition](../BoundaryPatchDefinition/BoundaryPatchDefinition.md)

## Property Value

This is a read only property whose value is a [BoundaryPatchDefinition](../BoundaryPatchDefinition/BoundaryPatchDefinition.md).

## Version

Introduced in version 2008
