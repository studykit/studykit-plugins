# Matrix.GetCoordinateSystem Method

Parent Object: [Matrix](../Matrix/Matrix.md)

## Description

Get the coordinate system that this matrix maps to from the standard coordinate system.

## Syntax

Matrix.**GetCoordinateSystem**( ***Origin*** As [Point](../Point/Point.md), ***XAxis*** As [Vector](../Vector/Vector.md), ***YAxis*** As [Vector](../Vector/Vector.md), ***ZAxis*** As [Vector](../Vector/Vector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Origin | [Point](../Point/Point.md) | Output Point object that defines the origin of the coordinate system. |
| XAxis | [Vector](../Vector/Vector.md) | Output Vector object that defines the X-axis vector of the coordinate system. |
| YAxis | [Vector](../Vector/Vector.md) | Output Vector object that defines the Y-axis vector of the coordinate system. |
| ZAxis | [Vector](../Vector/Vector.md) | Output Vector object that defines the Z-axis vector of the coordinate system. |

## Version

Introduced in version 4
