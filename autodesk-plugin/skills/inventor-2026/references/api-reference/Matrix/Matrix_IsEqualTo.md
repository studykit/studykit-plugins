# Matrix.IsEqualTo Method

Parent Object: [Matrix](../Matrix/Matrix.md)

## Description

Compares this matrix for equality with the specified matrix.

## Syntax

Matrix.**IsEqualTo**( ***Matrix*** As [Matrix](../Matrix/Matrix.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix | [Matrix](../Matrix/Matrix.md) | Input Matrix object that specifies matrix to compare. |
| Tolerance | Double | Input Double that specifies the tolerance to be used when determining whether the matrices are equal. |

## Version

Introduced in version 4
