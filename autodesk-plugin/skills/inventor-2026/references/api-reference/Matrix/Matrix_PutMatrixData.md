# Matrix.PutMatrixData Method

Parent Object: [Matrix](../Matrix/Matrix.md)

## Description

Method that sets the data defining this matrix.

## Syntax

Matrix.**PutMatrixData**( ***Cells***() As Double )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Cells | Double | Input/output Double that defines the matrix's cells. |

## Version

Introduced in version 4
