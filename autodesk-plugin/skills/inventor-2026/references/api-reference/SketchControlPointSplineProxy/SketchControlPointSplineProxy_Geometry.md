# SketchControlPointSplineProxy.Geometry Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Read-only property that returns a BSplineCurve2d geometry object. The object returned represents a 'snapshot' view of the current state of the spline.

## Syntax

SketchControlPointSplineProxy.**Geometry**() As [BSplineCurve2d](../BSplineCurve2d/BSplineCurve2d.md)

## Property Value

This is a read only property whose value is a [BSplineCurve2d](../BSplineCurve2d/BSplineCurve2d.md).

## Version

Introduced in version 2014
