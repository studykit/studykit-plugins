# SketchControlPointSplineProxy.IsClosed Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Read-only property that returns whether the curve is closed.

## Syntax

SketchControlPointSplineProxy.**IsClosed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
