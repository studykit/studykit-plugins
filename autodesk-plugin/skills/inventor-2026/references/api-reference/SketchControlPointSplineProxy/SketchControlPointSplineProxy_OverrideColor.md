# SketchControlPointSplineProxy.OverrideColor Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Gets and sets the color applied to this sketch spline.

## Syntax

SketchControlPointSplineProxy.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2014
