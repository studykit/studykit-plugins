# SketchControlPointSplineProxy.LineScale Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Gets and sets the LineScale applied to this sketch spline.

## Syntax

SketchControlPointSplineProxy.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2014
