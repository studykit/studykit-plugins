# FlatBendResult.Bend Property

Parent Object: [FlatBendResult](../FlatBendResult/FlatBendResult.md)

## Description

Property that returns the Bend object associated with this bend result.

## Syntax

FlatBendResult.**Bend**() As [Bend](../Bend/Bend.md)

## Property Value

This is a read only property whose value is a [Bend](../Bend/Bend.md).

## Version

Introduced in version 2010
