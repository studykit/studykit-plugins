# ButtonDefinition.Parent Property

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ButtonDefinition.**Parent**() As [CommandManager](../CommandManager/CommandManager.md)

## Property Value

This is a read only property whose value is a [CommandManager](../CommandManager/CommandManager.md).

## Version

Introduced in version 9
