# ModelCompositeAnnotationProxy.InternalName Property

Parent Object: [ModelCompositeAnnotationProxy](../ModelCompositeAnnotationProxy/ModelCompositeAnnotationProxy.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelCompositeAnnotationProxy.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
