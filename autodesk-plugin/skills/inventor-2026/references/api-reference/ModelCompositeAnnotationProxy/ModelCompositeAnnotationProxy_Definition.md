# ModelCompositeAnnotationProxy.Definition Property

Parent Object: [ModelCompositeAnnotationProxy](../ModelCompositeAnnotationProxy/ModelCompositeAnnotationProxy.md)

## Description

Gets and sets the ModelCompositeAnnotationDefinition associative with this object.

## Syntax

ModelCompositeAnnotationProxy.**Definition**() As [ModelCompositeAnnotationDefinition](../ModelCompositeAnnotationDefinition/ModelCompositeAnnotationDefinition.md)

## Property Value

This is a read/write property whose value is a [ModelCompositeAnnotationDefinition](../ModelCompositeAnnotationDefinition/ModelCompositeAnnotationDefinition.md).

## Version

Introduced in version 2018
