# RadiusModelDimensionProxy.InternalName Property

Parent Object: [RadiusModelDimensionProxy](../RadiusModelDimensionProxy/RadiusModelDimensionProxy.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

RadiusModelDimensionProxy.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
