# TranslateTranslateConstraintProxy.LayoutConstraint Property

Parent Object: [TranslateTranslateConstraintProxy](../TranslateTranslateConstraintProxy/TranslateTranslateConstraintProxy.md)

## Description

Property that returns the layout constraint this constraint is a member of. Returns Nothing in the case where this constraint is not a member of a layout constraint. You can also determine if a constraint is a member of a layout constraint by using the ResultOfLayoutConstraint property.

## Syntax

TranslateTranslateConstraintProxy.**LayoutConstraint**() As [LayoutConstraint](../LayoutConstraint/LayoutConstraint.md)

## Property Value

This is a read only property whose value is a [LayoutConstraint](../LayoutConstraint/LayoutConstraint.md).

## Version

Introduced in version 2010
