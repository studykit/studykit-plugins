# CombineFeatureProxy.KeepToolBodies Property

Parent Object: [CombineFeatureProxy](../CombineFeatureProxy/CombineFeatureProxy.md)

## Description

Property that returns whether the tool bodies were retained during the creation of the combine feature.

## Syntax

CombineFeatureProxy.**KeepToolBodies**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
