# MeshFeatureEntity.NormalSet Property

Parent Object: [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Description

Read-only property that returns the normal set of the mesh feature entity.

## Syntax

MeshFeatureEntity.**NormalSet**() As [GraphicsNormalSet](../GraphicsNormalSet/GraphicsNormalSet.md)

## Property Value

This is a read only property whose value is a [GraphicsNormalSet](../GraphicsNormalSet/GraphicsNormalSet.md).

## Version

Introduced in version 2017
