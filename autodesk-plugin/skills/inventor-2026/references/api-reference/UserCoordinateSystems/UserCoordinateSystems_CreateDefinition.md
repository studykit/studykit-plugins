# UserCoordinateSystems.CreateDefinition Method

Parent Object: [UserCoordinateSystems](../UserCoordinateSystems/UserCoordinateSystems.md)

## Description

Method that creates a new UserCoordinateSystemDefinition object.

## Syntax

UserCoordinateSystems.**CreateDefinition**() As [UserCoordinateSystemDefinition](../UserCoordinateSystemDefinition/UserCoordinateSystemDefinition.md)
