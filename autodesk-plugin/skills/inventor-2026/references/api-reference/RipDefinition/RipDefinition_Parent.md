# RipDefinition.Parent Property

Parent Object: [RipDefinition](../RipDefinition/RipDefinition.md)

## Description

Property that returns the parent rip definition feature.

## Syntax

RipDefinition.**Parent**() As [RipFeature](../RipFeature/RipFeature.md)

## Property Value

This is a read only property whose value is a [RipFeature](../RipFeature/RipFeature.md).

## Version

Introduced in version 2011
