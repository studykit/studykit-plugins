# TriangleStripGraphics.BackFaceCulling Property

Parent Object: [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Description

Gets and sets how the back face culling are defined for the triangles.

## Syntax

TriangleStripGraphics.**BackFaceCulling**() As [BackFaceCullingEnum](../BackFaceCullingEnum.md)

## Property Value

This is a read/write property whose value is a [BackFaceCullingEnum](../BackFaceCullingEnum.md).

## Version

Introduced in version 2013
