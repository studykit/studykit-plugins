# SketchFixedSplines3D.Item Property

Parent Object: [SketchFixedSplines3D](../SketchFixedSplines3D/SketchFixedSplines3D.md)

## Description

Method that returns the specified SketchFixedSpline3D object from the collection.

## Syntax

SketchFixedSplines3D.**Item**( ***Index*** As Long ) As [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 9
