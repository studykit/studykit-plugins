# HelicalCurve.ReferenceComponent Property

Parent Object: [HelicalCurve](../HelicalCurve/HelicalCurve.md)

## Description

Property that returns the ReferenceComponent that resulted in the creation of this feature.

## Syntax

HelicalCurve.**ReferenceComponent**() As [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md)

## Property Value

This is a read only property whose value is a [ReferenceComponent](../ReferenceComponent/ReferenceComponent.md).

## Version

Introduced in version 2019
