# ShellDefinition.InputFaces Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Gets and sets the faces that were specified to create or edit the Shell feature.

## Syntax

ShellDefinition.**InputFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 9
