# DirectEditFeature.Participants Property

Parent Object: [DirectEditFeature](../DirectEditFeature/DirectEditFeature.md)

## Description

Property that returns the list of participants for an assembly feature. This list is empty for features in a part.

## Syntax

DirectEditFeature.**Participants**() As [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md).

## Version

Introduced in version 2015
