# RevisionCloudDefinition.Inverted Property

Parent Object: [RevisionCloudDefinition](../RevisionCloudDefinition/RevisionCloudDefinition.md)

## Description

Gets and sets whether the revision cloud is inverted or not.

## Syntax

RevisionCloudDefinition.**Inverted**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
