# UnitVector2d.TransformBy Method

Parent Object: [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Description

Transform this unit vector2d by the specified matrix.

## Syntax

UnitVector2d.**TransformBy**( ***Value*** As [Matrix2d](../Matrix2d/Matrix2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | [Matrix2d](../Matrix2d/Matrix2d.md) | Matrix2d for transform. |

## Version

Introduced in version 11
