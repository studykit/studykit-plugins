# UnitVector2d.Copy Method

Parent Object: [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Description

Creates a copy of this UnitVector2d object. The result is entirely independent and can be edited without affecting the original UnitVector2d object.

## Syntax

UnitVector2d.**Copy**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Version

Introduced in version 2013
