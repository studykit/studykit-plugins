# EllipticalCone.Copy Method

Parent Object: [EllipticalCone](../EllipticalCone/EllipticalCone.md)

## Description

Creates a copy of this EllipticalCone object. The result is entirely independent and can be edited without affecting the original EllipticalCone object.

## Syntax

EllipticalCone.**Copy**() As [EllipticalCone](../EllipticalCone/EllipticalCone.md)

## Version

Introduced in version 2013
