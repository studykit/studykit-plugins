# EllipticalCone.Evaluator Property

Parent Object: [EllipticalCone](../EllipticalCone/EllipticalCone.md)

## Description

Gets the surface evaluator for this elliptical cone.

## Syntax

EllipticalCone.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
