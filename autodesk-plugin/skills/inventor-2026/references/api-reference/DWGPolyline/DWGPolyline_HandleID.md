# DWGPolyline.HandleID Property

Parent Object: [DWGPolyline](../DWGPolyline/DWGPolyline.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGPolyline.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
