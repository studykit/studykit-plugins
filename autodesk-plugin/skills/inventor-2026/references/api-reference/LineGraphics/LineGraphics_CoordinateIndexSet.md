# LineGraphics.CoordinateIndexSet Property

Parent Object: [LineGraphics](../LineGraphics/LineGraphics.md)

## Description

Gets and sets the GraphicsCoordinateIndexSet that defines the indices within the coordinate set to use for the lines of the set.

## Syntax

LineGraphics.**CoordinateIndexSet**() As [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md).

## Version

Introduced in version 5
