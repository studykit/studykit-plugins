# FlangeEdgeSet.Edges Property

Parent Object: [FlangeEdgeSet](../FlangeEdgeSet/FlangeEdgeSet.md)

## Description

Read-write property that gets and sets the collection of edges in the edge set.

## Syntax

FlangeEdgeSet.**Edges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2025
