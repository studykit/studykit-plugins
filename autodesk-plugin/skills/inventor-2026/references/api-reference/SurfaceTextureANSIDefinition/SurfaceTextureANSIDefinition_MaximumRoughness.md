# SurfaceTextureANSIDefinition.MaximumRoughness Property

Parent Object: [SurfaceTextureANSIDefinition](../SurfaceTextureANSIDefinition/SurfaceTextureANSIDefinition.md)

## Description

Gets and sets the maximum roughness average value.

## Syntax

SurfaceTextureANSIDefinition.**MaximumRoughness**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
