# SurfaceTextureANSIDefinition.AdditionalMaxRoughness Property

Parent Object: [SurfaceTextureANSIDefinition](../SurfaceTextureANSIDefinition/SurfaceTextureANSIDefinition.md)

## Description

Gets and sets the other maximum roughness value for the surface.

## Syntax

SurfaceTextureANSIDefinition.**AdditionalMaxRoughness**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
