# CutFeatureProxy.Suppressed Property

Parent Object: [CutFeatureProxy](../CutFeatureProxy/CutFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

CutFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
