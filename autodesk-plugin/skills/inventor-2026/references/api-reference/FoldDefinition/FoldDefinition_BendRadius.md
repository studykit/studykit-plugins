# FoldDefinition.BendRadius Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets the bend radius of a sheet metal fold feature.

## Syntax

FoldDefinition.**BendRadius**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2009
