# RotateAboutLineMoveOperation.Parent Property

Parent Object: [RotateAboutLineMoveOperation](../RotateAboutLineMoveOperation/RotateAboutLineMoveOperation.md)

## Description

Read-only property returning the associated MoveDefinition object.

## Syntax

RotateAboutLineMoveOperation.**Parent**() As [MoveDefinition](../MoveDefinition/MoveDefinition.md)

## Property Value

This is a read only property whose value is a [MoveDefinition](../MoveDefinition/MoveDefinition.md).

## Version

Introduced in version 2013
