# RectangularOccurrencePatternProxy.NativeObject Property

Parent Object: [RectangularOccurrencePatternProxy](../RectangularOccurrencePatternProxy/RectangularOccurrencePatternProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RectangularOccurrencePatternProxy.**NativeObject**() As [RectangularOccurrencePattern](../RectangularOccurrencePattern/RectangularOccurrencePattern.md)

## Property Value

This is a read only property whose value is a [RectangularOccurrencePattern](../RectangularOccurrencePattern/RectangularOccurrencePattern.md).

## Version

Introduced in version 2010
