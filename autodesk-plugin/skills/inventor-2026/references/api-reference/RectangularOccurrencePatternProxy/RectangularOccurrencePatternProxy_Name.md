# RectangularOccurrencePatternProxy.Name Property

Parent Object: [RectangularOccurrencePatternProxy](../RectangularOccurrencePatternProxy/RectangularOccurrencePatternProxy.md)

## Description

Gets/Sets the name for the pattern.

## Syntax

RectangularOccurrencePatternProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
