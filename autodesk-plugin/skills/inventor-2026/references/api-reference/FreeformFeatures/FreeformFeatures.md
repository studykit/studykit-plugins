# FreeformFeatures Object

## Description

Part Freeform Features Collection Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FreeformFeatures/FreeformFeatures_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../FreeformFeatures/FreeformFeatures_Count.md) | Gets the number of items in this collection. |
| [Item](../FreeformFeatures/FreeformFeatures_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../FreeformFeatures/FreeformFeatures_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PartFeatures.FreeformFeatures](../PartFeatures/PartFeatures_FreeformFeatures.md), [SheetMetalFeatures.FreeformFeatures](../SheetMetalFeatures/SheetMetalFeatures_FreeformFeatures.md)

## Version

Introduced in version 2015
