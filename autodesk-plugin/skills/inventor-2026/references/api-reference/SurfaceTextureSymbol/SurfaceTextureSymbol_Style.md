# SurfaceTextureSymbol.Style Property

Parent Object: [SurfaceTextureSymbol](../SurfaceTextureSymbol/SurfaceTextureSymbol.md)

## Description

Gets and sets the associated SurfaceTextureStyle object.

## Syntax

SurfaceTextureSymbol.**Style**() As [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md)

## Property Value

This is a read/write property whose value is a [SurfaceTextureStyle](../SurfaceTextureStyle/SurfaceTextureStyle.md).

## Version

Introduced in version 2009
