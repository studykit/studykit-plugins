# SurfaceGraphicsVertexList.Count Property

Parent Object: [SurfaceGraphicsVertexList](../SurfaceGraphicsVertexList/SurfaceGraphicsVertexList.md)

## Description

Read-only property that returns the number of SurfaceGraphicsVertex objects in the list.

## Syntax

SurfaceGraphicsVertexList.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2013
