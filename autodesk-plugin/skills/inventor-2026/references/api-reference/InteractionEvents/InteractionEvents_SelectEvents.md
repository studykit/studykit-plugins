# InteractionEvents.SelectEvents Property

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Gets the Selection Events object.

## Syntax

InteractionEvents.**SelectEvents**() As [SelectEvents](../SelectEvents/SelectEvents.md)

## Property Value

This is a read only property whose value is a [SelectEvents](../SelectEvents/SelectEvents.md).
