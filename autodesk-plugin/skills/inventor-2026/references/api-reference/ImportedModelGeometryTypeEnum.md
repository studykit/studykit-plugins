# ImportedModelGeometryTypeEnum Enumerator

## Description

Specifies the model geometry type.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kMeshesModelGeometryType | 4 | Indicates meshes are included. |
| kSolidsModelGeometryType | 1 | Indicates solids are included. |
| kSurfacesModelGeometryType | 2 | Indicates surfaces are included. |
| kWiresModelGeometryType | 8 | Indicates wires are included. |

## Version

Introduced in version 2016
