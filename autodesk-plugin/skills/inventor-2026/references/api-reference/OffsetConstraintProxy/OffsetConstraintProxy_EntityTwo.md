# OffsetConstraintProxy.EntityTwo Property

Parent Object: [OffsetConstraintProxy](../OffsetConstraintProxy/OffsetConstraintProxy.md)

## Description

Property that returns one of the entities.

## Syntax

OffsetConstraintProxy.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
