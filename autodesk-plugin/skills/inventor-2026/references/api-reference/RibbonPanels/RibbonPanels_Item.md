# RibbonPanels.Item Property

Parent Object: [RibbonPanels](../RibbonPanels/RibbonPanels.md)

## Description

Returns the specified RibbonPanel object from the collection.

## Syntax

RibbonPanels.**Item**( ***Index*** As Variant ) As [RibbonPanel](../RibbonPanel/RibbonPanel.md)

## Property Value

This is a read only property whose value is a [RibbonPanel](../RibbonPanel/RibbonPanel.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the RibbonPanel to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the RibbonPanel name. If an out of range index or a name of a non\-existent RibbonPanel is provided, an error will occur. |

## Version

Introduced in version 2010
