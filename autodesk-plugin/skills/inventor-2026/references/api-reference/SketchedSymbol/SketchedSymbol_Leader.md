# SketchedSymbol.Leader Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Property that returns the Leader object.

## Syntax

SketchedSymbol.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 11
