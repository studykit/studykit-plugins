# SketchedSymbol.Callout Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Property that gets whether this sketched symbol is a callout symbol.

## Syntax

SketchedSymbol.**Callout**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 10
