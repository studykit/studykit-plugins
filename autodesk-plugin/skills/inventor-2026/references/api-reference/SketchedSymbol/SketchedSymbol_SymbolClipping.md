# SketchedSymbol.SymbolClipping Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets whether to trim annotations applied to the SketchedSymbol.

## Syntax

SketchedSymbol.**SymbolClipping**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
