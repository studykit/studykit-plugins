# SketchedSymbol.LineWeight Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets the line weight override for the symbol.

## Syntax

SketchedSymbol.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
