# SketchedSymbol.LeaderVisible Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets whether the leader of the SketchedSymbol is visible.

## Syntax

SketchedSymbol.**LeaderVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
