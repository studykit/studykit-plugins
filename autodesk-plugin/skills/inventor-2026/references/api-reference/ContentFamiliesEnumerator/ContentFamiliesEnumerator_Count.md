# ContentFamiliesEnumerator.Count Property

Parent Object: [ContentFamiliesEnumerator](../ContentFamiliesEnumerator/ContentFamiliesEnumerator.md)

## Description

Property that returns the number of ContentFamily objects in the collection.

## Syntax

ContentFamiliesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
