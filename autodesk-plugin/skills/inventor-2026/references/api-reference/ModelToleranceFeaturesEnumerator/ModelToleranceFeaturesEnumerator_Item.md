# ModelToleranceFeaturesEnumerator.Item Property

Parent Object: [ModelToleranceFeaturesEnumerator](../ModelToleranceFeaturesEnumerator/ModelToleranceFeaturesEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

ModelToleranceFeaturesEnumerator.**Item**( ***Index*** As Variant ) As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant |  |
