# ModelDatumProxy.HealthStatus Property

Parent Object: [ModelDatumProxy](../ModelDatumProxy/ModelDatumProxy.md)

## Description

Returns the current health status of the model datum.

## Syntax

ModelDatumProxy.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2023
