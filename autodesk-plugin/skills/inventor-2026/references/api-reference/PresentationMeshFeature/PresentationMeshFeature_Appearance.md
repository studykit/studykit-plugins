# PresentationMeshFeature.Appearance Property

Parent Object: [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Description

Read-only property that gets the appearance of this object.

## Syntax

PresentationMeshFeature.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read only property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2018
