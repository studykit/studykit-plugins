# PresentationMeshFeature.Suppressed Property

Parent Object: [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Description

Read-only property that gets whether this object is suppressed or not.

## Syntax

PresentationMeshFeature.**Suppressed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
