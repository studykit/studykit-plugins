# OccurrencePatternElementProxy.Name Property

Parent Object: [OccurrencePatternElementProxy](../OccurrencePatternElementProxy/OccurrencePatternElementProxy.md)

## Description

Property that returns the name of the element.

## Syntax

OccurrencePatternElementProxy.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2010
