# HelpEvents.Type Property

Parent Object: [HelpEvents](../HelpEvents/HelpEvents.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

HelpEvents.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2012
