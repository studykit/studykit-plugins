# InsertConstraint.HealthStatus Property

Parent Object: [InsertConstraint](../InsertConstraint/InsertConstraint.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

InsertConstraint.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 5
