# FreeDragMoveOperation.XOffset Property

Parent Object: [FreeDragMoveOperation](../FreeDragMoveOperation/FreeDragMoveOperation.md)

## Description

Read-write property that provides access to the x offset of a free drag operation on a move body feature.

## Syntax

FreeDragMoveOperation.**XOffset**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2013
