# SketchDrivenPatternFeatureProxy.SurfacesOpaque Property

Parent Object: [SketchDrivenPatternFeatureProxy](../SketchDrivenPatternFeatureProxy/SketchDrivenPatternFeatureProxy.md)

## Description

Gets and sets whether all the resulting surfaces are opaque or transparent.

## Syntax

SketchDrivenPatternFeatureProxy.**SurfacesOpaque**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
