# TwoPointDistanceDimConstraint3DProxy.TextPoint Property

Parent Object: [TwoPointDistanceDimConstraint3DProxy](../TwoPointDistanceDimConstraint3DProxy/TwoPointDistanceDimConstraint3DProxy.md)

## Description

Gets and sets the position of the dimension text.

## Syntax

TwoPointDistanceDimConstraint3DProxy.**TextPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 11
