# DrawingDimensions.OrdinateDimensionSets Property

Parent Object: [DrawingDimensions](../DrawingDimensions/DrawingDimensions.md)

## Description

Property that returns the OrdinateDimensionSets collection object.

## Syntax

DrawingDimensions.**OrdinateDimensionSets**() As [OrdinateDimensionSets](../OrdinateDimensionSets/OrdinateDimensionSets.md)

## Property Value

This is a read only property whose value is an [OrdinateDimensionSets](../OrdinateDimensionSets/OrdinateDimensionSets.md).

## Version

Introduced in version 2010
