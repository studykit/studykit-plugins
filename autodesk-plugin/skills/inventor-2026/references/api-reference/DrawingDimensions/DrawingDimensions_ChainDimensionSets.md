# DrawingDimensions.ChainDimensionSets Property

Parent Object: [DrawingDimensions](../DrawingDimensions/DrawingDimensions.md)

## Description

Property that returns the ChainDimensionSets collection object. This object provides access to all of the chain dimension sets on the sheet and provides functionality to create new chain dimension sets.

## Syntax

DrawingDimensions.**ChainDimensionSets**() As [ChainDimensionSets](../ChainDimensionSets/ChainDimensionSets.md)

## Property Value

This is a read only property whose value is a [ChainDimensionSets](../ChainDimensionSets/ChainDimensionSets.md).

## Version

Introduced in version 2011
