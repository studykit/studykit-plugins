# PartOptions.AutoHideInLineWorkFeatures Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Enables/disables the behavior of automatically hiding an in-line work feature.

## Syntax

PartOptions.**AutoHideInLineWorkFeatures**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
