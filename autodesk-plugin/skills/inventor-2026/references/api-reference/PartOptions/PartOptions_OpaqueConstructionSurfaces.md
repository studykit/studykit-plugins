# PartOptions.OpaqueConstructionSurfaces Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Enables/disables whether construction surfaces are displayed opaque or translucent.

## Syntax

PartOptions.**OpaqueConstructionSurfaces**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
