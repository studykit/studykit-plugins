# PartOptions.DefaultDesignViewIsAssociative Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Read-write Boolean property that gets and sets whether the default design view representation is associative.

## Syntax

PartOptions.**DefaultDesignViewIsAssociative**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
