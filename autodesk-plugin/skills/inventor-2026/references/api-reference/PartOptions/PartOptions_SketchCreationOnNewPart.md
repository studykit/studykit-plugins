# PartOptions.SketchCreationOnNewPart Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Gets and sets the preference for creating a sketch when a new part file is created.

## Syntax

PartOptions.**SketchCreationOnNewPart**() As [SketchCreationOnNewPartEnum](../SketchCreationOnNewPartEnum.md)

## Property Value

This is a read/write property whose value is a [SketchCreationOnNewPartEnum](../SketchCreationOnNewPartEnum.md).

## Version

Introduced in version 11
