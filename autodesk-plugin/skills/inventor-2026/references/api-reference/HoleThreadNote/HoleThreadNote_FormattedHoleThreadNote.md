# HoleThreadNote.FormattedHoleThreadNote Property

Parent Object: [HoleThreadNote](../HoleThreadNote/HoleThreadNote.md)

## Description

Gets and sets the fully formatted string that defines the contents of the hole/thread note.

## Syntax

HoleThreadNote.**FormattedHoleThreadNote**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
