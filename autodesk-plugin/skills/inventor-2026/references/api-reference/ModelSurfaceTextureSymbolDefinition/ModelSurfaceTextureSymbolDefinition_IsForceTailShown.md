# ModelSurfaceTextureSymbolDefinition.IsForceTailShown Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets whether a tail is added to the symbol.

## Syntax

ModelSurfaceTextureSymbolDefinition.**IsForceTailShown**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
