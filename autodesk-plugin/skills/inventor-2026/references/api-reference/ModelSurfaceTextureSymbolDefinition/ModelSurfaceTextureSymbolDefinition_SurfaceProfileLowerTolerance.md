# ModelSurfaceTextureSymbolDefinition.SurfaceProfileLowerTolerance Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Gets and sets the surface profile lower tolerance.

## Syntax

ModelSurfaceTextureSymbolDefinition.**SurfaceProfileLowerTolerance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025
