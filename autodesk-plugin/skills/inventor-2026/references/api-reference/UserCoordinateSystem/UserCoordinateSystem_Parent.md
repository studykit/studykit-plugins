# UserCoordinateSystem.Parent Property

Parent Object: [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Description

Property that returns the parent ComponentDefinition object.

## Syntax

UserCoordinateSystem.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
