# SaveOptions.TranslatorReportLocation Property

Parent Object: [SaveOptions](../SaveOptions/SaveOptions.md)

## Description

Gets/Sets whether to create a tranlation report and where to save it.

## Syntax

SaveOptions.**TranslatorReportLocation**() As [ReportLocationEnum](../ReportLocationEnum.md)

## Property Value

This is a read/write property whose value is a [ReportLocationEnum](../ReportLocationEnum.md).

## Version

Introduced in version 2016
