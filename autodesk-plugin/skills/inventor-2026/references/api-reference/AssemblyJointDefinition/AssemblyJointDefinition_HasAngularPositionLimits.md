# AssemblyJointDefinition.HasAngularPositionLimits Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-write property that indicates if the assembly joint has angular position start and end limits that the angular position value should not exceed to be valid.

## Syntax

AssemblyJointDefinition.**HasAngularPositionLimits**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
