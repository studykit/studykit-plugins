# BrowserNode.BrowserFolders Property

Parent Object: [BrowserNode](../BrowserNode/BrowserNode.md)

## Description

Property that returns a collection of all browser folders contained directly under this node in the browser.

## Syntax

BrowserNode.**BrowserFolders**() As [BrowserFoldersEnumerator](../BrowserFoldersEnumerator/BrowserFoldersEnumerator.md)

## Property Value

This is a read only property whose value is a [BrowserFoldersEnumerator](../BrowserFoldersEnumerator/BrowserFoldersEnumerator.md).

## Version

Introduced in version 2010
