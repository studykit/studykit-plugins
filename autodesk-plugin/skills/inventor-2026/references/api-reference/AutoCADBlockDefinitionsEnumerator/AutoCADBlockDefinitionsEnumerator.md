# AutoCADBlockDefinitionsEnumerator Object

## Description

The AutoCADBlockDefinitionsEnumerator object enumerates AutoCADBlockDefinition objects

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../AutoCADBlockDefinitionsEnumerator/AutoCADBlockDefinitionsEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../AutoCADBlockDefinitionsEnumerator/AutoCADBlockDefinitionsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../AutoCADBlockDefinitionsEnumerator/AutoCADBlockDefinitionsEnumerator_Item.md) | Returns the specified AutoCADBlockDefinition object from the collection. |
| [Type](../AutoCADBlockDefinitionsEnumerator/AutoCADBlockDefinitionsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
