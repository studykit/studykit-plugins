# SurfaceTextureJISDefinition.IsForceTailShown Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets whether to add a tail to the symbol.

## Syntax

SurfaceTextureJISDefinition.**IsForceTailShown**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
