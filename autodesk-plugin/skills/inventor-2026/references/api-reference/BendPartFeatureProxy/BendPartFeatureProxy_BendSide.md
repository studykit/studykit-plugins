# BendPartFeatureProxy.BendSide Property

Parent Object: [BendPartFeatureProxy](../BendPartFeatureProxy/BendPartFeatureProxy.md)

## Description

Property that gets and sets the bend side.

## Syntax

BendPartFeatureProxy.**BendSide**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2010
