# SketchConstraintSettings.EnableRelaxMode Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets whether to enable relax mode.

## Syntax

SketchConstraintSettings.**EnableRelaxMode**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
