# SketchConstraintSettings.EnableInferConstraints Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets whether to enable inferred constraints.

## Syntax

SketchConstraintSettings.**EnableInferConstraints**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
