# ChamferFeatureProxy.NativeObject Property

Parent Object: [ChamferFeatureProxy](../ChamferFeatureProxy/ChamferFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ChamferFeatureProxy.**NativeObject**() As [ChamferFeature](../ChamferFeature/ChamferFeature.md)

## Property Value

This is a read only property whose value is a [ChamferFeature](../ChamferFeature/ChamferFeature.md).

## Version

Introduced in version 9
