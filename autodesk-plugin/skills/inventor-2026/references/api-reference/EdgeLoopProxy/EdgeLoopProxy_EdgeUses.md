# EdgeLoopProxy.EdgeUses Property

Parent Object: [EdgeLoopProxy](../EdgeLoopProxy/EdgeLoopProxy.md)

## Description

Gets the referenced by this EdgeLoop.

## Syntax

EdgeLoopProxy.**EdgeUses**() As [EdgeUses](../EdgeUses/EdgeUses.md)

## Property Value

This is a read only property whose value is an [EdgeUses](../EdgeUses/EdgeUses.md).

## Version

Introduced in version 4
