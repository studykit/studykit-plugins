# SketchEllipticalArc.OverrideColor Property

Parent Object: [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Description

Gets and sets the color applied to this elliptical arc.

## Syntax

SketchEllipticalArc.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
