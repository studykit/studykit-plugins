# SketchEllipticalArc.LineScale Property

Parent Object: [SketchEllipticalArc](../SketchEllipticalArc/SketchEllipticalArc.md)

## Description

Gets and sets the LineScale applied to this elliptical arc.

## Syntax

SketchEllipticalArc.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
