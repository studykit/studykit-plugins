# MiniToolbarComboBox.SelectedIndex Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Read-write property that gets and set the index of the item currently selected.

## Syntax

MiniToolbarComboBox.**SelectedIndex**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2012
