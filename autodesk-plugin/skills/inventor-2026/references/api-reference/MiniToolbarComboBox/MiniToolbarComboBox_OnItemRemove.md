# MiniToolbarComboBox.OnItemRemove Event

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Event that is fired when the user removes an item from the combobox.

## Syntax

MiniToolbarComboBox.**OnItemRemove**( ***ListItem*** As [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ListItem | [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) | Input ListItem that indicates the item being deleted. |

## Version

Introduced in version 2012
