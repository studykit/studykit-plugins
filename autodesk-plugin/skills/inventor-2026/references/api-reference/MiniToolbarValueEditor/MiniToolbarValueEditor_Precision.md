# MiniToolbarValueEditor.Precision Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets number of decimal places to be used when displaying the parameter associated with this control.

## Syntax

MiniToolbarValueEditor.**Precision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2012
