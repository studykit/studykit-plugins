# MiniToolbarValueEditor.MaximumValue Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the maximum valid value for this control in database units.

## Syntax

MiniToolbarValueEditor.**MaximumValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2012
