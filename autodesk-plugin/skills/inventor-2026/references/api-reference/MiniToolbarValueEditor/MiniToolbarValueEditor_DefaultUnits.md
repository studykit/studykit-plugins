# MiniToolbarValueEditor.DefaultUnits Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the default units for the value editor. If the Expression includes string that has no units specified, the units of this property will be used to calculate the value.

## Syntax

MiniToolbarValueEditor.**DefaultUnits**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).

## Version

Introduced in version 2018
