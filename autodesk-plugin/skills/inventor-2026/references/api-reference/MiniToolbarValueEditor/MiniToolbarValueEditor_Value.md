# MiniToolbarValueEditor.Value Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the evaluated value of the current expression in database units.

## Syntax

MiniToolbarValueEditor.**Value**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2012
