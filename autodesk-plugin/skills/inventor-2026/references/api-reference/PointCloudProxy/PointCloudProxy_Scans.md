# PointCloudProxy.Scans Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-only property that returns the point cloud scans collection object.

## Syntax

PointCloudProxy.**Scans**() As [PointCloudScans](../PointCloudScans/PointCloudScans.md)

## Property Value

This is a read only property whose value is a [PointCloudScans](../PointCloudScans/PointCloudScans.md).

## Version

Introduced in version 2016
