# SketchSettings.LineWeightType Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Gets and sets the type of line weight to apply when displaying lines.

## Syntax

SketchSettings.**LineWeightType**() As [LineWeightTypeEnum](../LineWeightTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineWeightTypeEnum](../LineWeightTypeEnum.md).

## Version

Introduced in version 2008
