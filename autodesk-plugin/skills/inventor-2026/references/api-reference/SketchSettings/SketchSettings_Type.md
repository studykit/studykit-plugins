# SketchSettings.Type Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchSettings.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 8
