# RevisionTable.Layer Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets the layer used by the revision table.

## Syntax

RevisionTable.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
