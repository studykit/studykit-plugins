# EqualConstraint3DProxy Object

Derived from: [EqualConstraint3D](../EqualConstraint3D/EqualConstraint3D.md) Object

## Description

EqualConstraint3DProxy Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../EqualConstraint3DProxy/EqualConstraint3DProxy_Delete.md) | Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted. |
| [GetReferenceKey](../EqualConstraint3DProxy/EqualConstraint3DProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../EqualConstraint3DProxy/EqualConstraint3DProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../EqualConstraint3DProxy/EqualConstraint3DProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../EqualConstraint3DProxy/EqualConstraint3DProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../EqualConstraint3DProxy/EqualConstraint3DProxy_Deletable.md) | Indicates whether this object can be deleted. |
| [EntityOne](../EqualConstraint3DProxy/EqualConstraint3DProxy_EntityOne.md) | Read-only property that returns the first 3D sketch entity being constrained. |
| [EntityTwo](../EqualConstraint3DProxy/EqualConstraint3DProxy_EntityTwo.md) | Read-only property that returns the second 3D sketch entity being constrained. |
| [NativeObject](../EqualConstraint3DProxy/EqualConstraint3DProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../EqualConstraint3DProxy/EqualConstraint3DProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Type](../EqualConstraint3DProxy/EqualConstraint3DProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2017
