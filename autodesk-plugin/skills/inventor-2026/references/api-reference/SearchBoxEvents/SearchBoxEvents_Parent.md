# SearchBoxEvents.Parent Property

Parent Object: [SearchBoxEvents](../SearchBoxEvents/SearchBoxEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SearchBoxEvents.**Parent**() As [SearchBox](../SearchBox/SearchBox.md)

## Property Value

This is a read only property whose value is a [SearchBox](../SearchBox/SearchBox.md).

## Version

Introduced in version 2018
