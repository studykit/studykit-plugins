# PartDocument.AttributeManager Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Returns the AttributeManager object.

## Syntax

PartDocument.**AttributeManager**() As [AttributeManager](../AttributeManager/AttributeManager.md)

## Property Value

This is a read only property whose value is an [AttributeManager](../AttributeManager/AttributeManager.md).

## Version

Introduced in version 5
