# PartDocument.ActiveMaterial Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets and sets the default material for the part.

## Syntax

PartDocument.**ActiveMaterial**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
