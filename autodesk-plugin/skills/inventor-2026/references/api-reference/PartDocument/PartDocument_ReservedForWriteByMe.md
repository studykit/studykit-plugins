# PartDocument.ReservedForWriteByMe Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets/Sets the Boolean flag indicating whether this file has been reserved for write by the caller.

## Syntax

PartDocument.**ReservedForWriteByMe**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 4
