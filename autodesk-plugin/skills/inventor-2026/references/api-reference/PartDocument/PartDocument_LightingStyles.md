# PartDocument.LightingStyles Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Property that returns the LightingStyles collection object.

## Syntax

PartDocument.**LightingStyles**() As [LightingStyles](../LightingStyles/LightingStyles.md)

## Property Value

This is a read only property whose value is a [LightingStyles](../LightingStyles/LightingStyles.md).

## Version

Introduced in version 10
