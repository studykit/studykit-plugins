# PartDocument.HasReferencesMissing Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Read-only property that returns whether there are references missing for this document.

## Syntax

PartDocument.**HasReferencesMissing**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025
