# DerivedAssemblyDefinition.ActiveModelState Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Read-write property that gets or sets the name of active model state for the derived assembly.

## Syntax

DerivedAssemblyDefinition.**ActiveModelState**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2022
