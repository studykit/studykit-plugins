# RadiusGeneralDimension.TolerancePrecision Property

Parent Object: [RadiusGeneralDimension](../RadiusGeneralDimension/RadiusGeneralDimension.md)

## Description

Gets and sets the precision of the tolerance text for the dimension.

## Syntax

RadiusGeneralDimension.**TolerancePrecision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2008
