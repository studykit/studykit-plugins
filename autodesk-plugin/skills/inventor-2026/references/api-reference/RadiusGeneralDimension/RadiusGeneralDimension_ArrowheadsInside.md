# RadiusGeneralDimension.ArrowheadsInside Property

Parent Object: [RadiusGeneralDimension](../RadiusGeneralDimension/RadiusGeneralDimension.md)

## Description

Gets and sets the ArrowheadsInside setting.

## Syntax

RadiusGeneralDimension.**ArrowheadsInside**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
