# EllipticalArc2d Object

## Description

The EllipticalArc2d object. The object created is a transient mathematical object and is not displayed graphically.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../EllipticalArc2d/EllipticalArc2d_Copy.md) | Creates a copy of this EllipticalArc2d object. The result is entirely independent and can be edited without affecting the original EllipticalArc2d object. |
| [GetEllipticalArcData](../EllipticalArc2d/EllipticalArc2d_GetEllipticalArcData.md) | Get the data defining this elliptical arc. |
| [PutEllipticalArcData](../EllipticalArc2d/EllipticalArc2d_PutEllipticalArcData.md) | Method that sets the data defining this elliptical arc. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Center](../EllipticalArc2d/EllipticalArc2d_Center.md) | Specifies the center point for the arc. |
| [EndPoint](../EllipticalArc2d/EllipticalArc2d_EndPoint.md) | Gets the end point for the arc. |
| [Evaluator](../EllipticalArc2d/EllipticalArc2d_Evaluator.md) | Gets the Curve2DEvaluator for the arc. |
| [MajorAxis](../EllipticalArc2d/EllipticalArc2d_MajorAxis.md) | Specifies the major axis for the ellipse. |
| [MajorRadius](../EllipticalArc2d/EllipticalArc2d_MajorRadius.md) | Specifies the major radius for the ellipse. |
| [MinorRadius](../EllipticalArc2d/EllipticalArc2d_MinorRadius.md) | Specifies the minor radius for the ellipse. |
| [StartAngle](../EllipticalArc2d/EllipticalArc2d_StartAngle.md) | Specifies the start angle for the arc. |
| [StartPoint](../EllipticalArc2d/EllipticalArc2d_StartPoint.md) | Gets the start point for the arc. |
| [SweepAngle](../EllipticalArc2d/EllipticalArc2d_SweepAngle.md) | Specifies the sweep angle from the start angle for the arc. |
