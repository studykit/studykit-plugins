# LeaderNode.ChildNodes Property

Parent Object: [LeaderNode](../LeaderNode/LeaderNode.md)

## Description

Property that returns the children nodes under this node. The count will return 0 if the node is a leaf node.

## Syntax

LeaderNode.**ChildNodes**() As [LeaderNodesEnumerator](../LeaderNodesEnumerator/LeaderNodesEnumerator.md)

## Property Value

This is a read only property whose value is a [LeaderNodesEnumerator](../LeaderNodesEnumerator/LeaderNodesEnumerator.md).
