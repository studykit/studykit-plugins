# DerivedPartTransformDef.IsAssociativeDesignView Property

Parent Object: [DerivedPartTransformDef](../DerivedPartTransformDef/DerivedPartTransformDef.md)

## Description

Read-write property that gets and sets if there is an associative link to the specified design view.

## Syntax

DerivedPartTransformDef.**IsAssociativeDesignView**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
