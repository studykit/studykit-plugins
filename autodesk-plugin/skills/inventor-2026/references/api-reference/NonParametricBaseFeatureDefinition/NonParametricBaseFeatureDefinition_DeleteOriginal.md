# NonParametricBaseFeatureDefinition.DeleteOriginal Property

Parent Object: [NonParametricBaseFeatureDefinition](../NonParametricBaseFeatureDefinition/NonParametricBaseFeatureDefinition.md)

## Description

Read-write property that defines if the original geometry in the construction environment is deleted.

## Syntax

NonParametricBaseFeatureDefinition.**DeleteOriginal**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
