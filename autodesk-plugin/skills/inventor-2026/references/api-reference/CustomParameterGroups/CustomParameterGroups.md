# CustomParameterGroups Object

## Description

The CustomParameterGroups collection object provides access to all existing custom parameter groups and provides methods to create additional groups

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../CustomParameterGroups/CustomParameterGroups_Add.md) | Method that adds a custom parameter group. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CustomParameterGroups/CustomParameterGroups_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CustomParameterGroups/CustomParameterGroups_Count.md) | Property that returns the number of items in the collection. |
| [Item](../CustomParameterGroups/CustomParameterGroups_Item.md) | Returns the specified CustomParameterGroup object from the collection. |
| [Type](../CustomParameterGroups/CustomParameterGroups_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
