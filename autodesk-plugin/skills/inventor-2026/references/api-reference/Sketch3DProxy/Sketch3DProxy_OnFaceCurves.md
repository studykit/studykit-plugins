# Sketch3DProxy.OnFaceCurves Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Gets the 3D Sketch OnFaceCurves collection Object.

## Syntax

Sketch3DProxy.**OnFaceCurves**() As [OnFaceCurves](../OnFaceCurves/OnFaceCurves.md)

## Property Value

This is a read only property whose value is an [OnFaceCurves](../OnFaceCurves/OnFaceCurves.md).

## Version

Introduced in version 2017
