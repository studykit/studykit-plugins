# LeaderStyle.Color Property

Parent Object: [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Description

Property that gets and sets the color for the style. Setting the property to Nothing restores the style to the color defined by the layer.

## Syntax

LeaderStyle.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2010
