# ProfileEntity3D.Parent Property

Parent Object: [ProfileEntity3D](../ProfileEntity3D/ProfileEntity3D.md)

## Description

Property that returns the parent ProfilePath3D of the entity.

## Syntax

ProfileEntity3D.**Parent**() As [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Property Value

This is a read only property whose value is a [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md).

## Version

Introduced in version 6
