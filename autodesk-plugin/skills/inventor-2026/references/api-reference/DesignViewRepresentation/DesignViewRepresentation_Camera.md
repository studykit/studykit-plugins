# DesignViewRepresentation.Camera Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Gets and sets the camera associated with this design view.

## Syntax

DesignViewRepresentation.**Camera**() As [Camera](../Camera/Camera.md)

## Property Value

This is a read/write property whose value is a [Camera](../Camera/Camera.md).

## Version

Introduced in version 2014
