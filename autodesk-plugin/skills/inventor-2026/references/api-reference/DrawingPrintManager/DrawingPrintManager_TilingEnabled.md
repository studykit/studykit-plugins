# DrawingPrintManager.TilingEnabled Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the Boolean flag indicating whether Tiling is enabled during print.

## Syntax

DrawingPrintManager.**TilingEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
