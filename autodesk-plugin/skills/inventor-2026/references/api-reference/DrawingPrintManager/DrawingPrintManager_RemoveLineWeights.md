# DrawingPrintManager.RemoveLineWeights Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the Boolean flag indicating whether the Drawing line weights should be removed in the print.

## Syntax

DrawingPrintManager.**RemoveLineWeights**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
