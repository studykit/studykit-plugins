# DrawingPrintManager.ColorMode Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the color mode to be used for printing. Defaults to using kPrintGrayScale.

## Syntax

DrawingPrintManager.**ColorMode**() As [PrintColorModeEnum](../PrintColorModeEnum.md)

## Property Value

This is a read/write property whose value is a [PrintColorModeEnum](../PrintColorModeEnum.md).
