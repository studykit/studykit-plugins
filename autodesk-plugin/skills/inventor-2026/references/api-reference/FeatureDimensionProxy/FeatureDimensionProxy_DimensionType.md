# FeatureDimensionProxy.DimensionType Property

Parent Object: [FeatureDimensionProxy](../FeatureDimensionProxy/FeatureDimensionProxy.md)

## Description

Gets the type of dimension.

## Syntax

FeatureDimensionProxy.**DimensionType**() As [FeatureDimensionTypeEnum](../FeatureDimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [FeatureDimensionTypeEnum](../FeatureDimensionTypeEnum.md).

## Version

Introduced in version 2012
