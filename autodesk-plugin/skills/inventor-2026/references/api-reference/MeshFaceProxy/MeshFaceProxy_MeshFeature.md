# MeshFaceProxy.MeshFeature Property

Parent Object: [MeshFaceProxy](../MeshFaceProxy/MeshFaceProxy.md)

## Description

Read-only property that returns the MeshFeature this MeshFace is located on.

## Syntax

MeshFaceProxy.**MeshFeature**() As [MeshFeature](../MeshFeature/MeshFeature.md)

## Property Value

This is a read only property whose value is a [MeshFeature](../MeshFeature/MeshFeature.md).

## Version

Introduced in version 2017
