# MeshFaceProxy.Evaluator Property

Parent Object: [MeshFaceProxy](../MeshFaceProxy/MeshFaceProxy.md)

## Description

Read-only property that returns the surface evaluator for this MeshFace.This evaluator differs from the one that could be obtained from the surface geometry definition in that this evaluator accounts for the topological orientation.

## Syntax

MeshFaceProxy.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 2017
