# SketchFixedSpline3D.Type Property

Parent Object: [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchFixedSpline3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
