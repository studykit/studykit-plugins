# SketchFixedSpline3D.HasReferenceComponent Property

Parent Object: [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchFixedSpline3D.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
