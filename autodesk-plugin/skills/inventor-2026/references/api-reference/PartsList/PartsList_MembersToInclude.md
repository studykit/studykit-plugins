# PartsList.MembersToInclude Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the names of the iPart/iAssembly members to include as columns in the parts list.

## Syntax

PartsList.**MembersToInclude**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
