# RadiusDimConstraint.AnchorPoints Property

Parent Object: [RadiusDimConstraint](../RadiusDimConstraint/RadiusDimConstraint.md)

## Description

Gets the anchor points of dimension.

## Syntax

RadiusDimConstraint.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
