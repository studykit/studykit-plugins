# ManipulatorEvents.SetAppearance Method

Parent Object: [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md)

## Description

Sets the visible and enabled manipulator elements.

## Syntax

ManipulatorEvents.**SetAppearance**( ***VisibleElements*** As [ManipulatorElementEnum](../ManipulatorElementEnum.md), ***EnabledElements*** As [ManipulatorElementEnum](../ManipulatorElementEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| VisibleElements | [ManipulatorElementEnum](../ManipulatorElementEnum.md) | Input ManipulatorElementEnum that specifies which manipulator elements would be enabled. This could be a bit mask defining various of manipulator elements. |
| EnabledElements | [ManipulatorElementEnum](../ManipulatorElementEnum.md) | Input ManipulatorElementEnum that specifies which manipulator elements would be enabled. This could be a bit mask defining various of manipulator elements. |

## Version

Introduced in version 2017
