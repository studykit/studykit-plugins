# ApprenticePrintManager.ColorMode Property

Parent Object: [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Description

Gets/Sets the color mode to be used for printing. Defaults to using kPrintGrayScale.

## Syntax

ApprenticePrintManager.**ColorMode**() As [PrintColorModeEnum](../PrintColorModeEnum.md)

## Property Value

This is a read/write property whose value is a [PrintColorModeEnum](../PrintColorModeEnum.md).

## Version

Introduced in version 9
