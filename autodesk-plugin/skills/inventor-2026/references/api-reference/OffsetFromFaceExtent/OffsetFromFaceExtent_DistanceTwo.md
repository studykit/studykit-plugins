# OffsetFromFaceExtent.DistanceTwo Property

Parent Object: [OffsetFromFaceExtent](../OffsetFromFaceExtent/OffsetFromFaceExtent.md)

## Description

Read-only property that returns the parameter controlling the distance extent of the feature in the other direction. This returns Nothing if the IsTwoDirectional returns False.

## Syntax

OffsetFromFaceExtent.**DistanceTwo**() As Variant

## Property Value

This is a read only property whose value is a Variant.

## Version

Introduced in version 2018
