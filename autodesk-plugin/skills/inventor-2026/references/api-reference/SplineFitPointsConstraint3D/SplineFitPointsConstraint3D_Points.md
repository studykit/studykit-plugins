# SplineFitPointsConstraint3D.Points Property

Parent Object: [SplineFitPointsConstraint3D](../SplineFitPointsConstraint3D/SplineFitPointsConstraint3D.md)

## Description

Property that returns the sketch points the spline is fit through.

## Syntax

SplineFitPointsConstraint3D.**Points**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 8
