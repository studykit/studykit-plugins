# ApprenticeServerDrawingDocument.Sheets Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Property that returns the collection of all the sheets making up this document.

## Syntax

ApprenticeServerDrawingDocument.**Sheets**() As [Sheets](../Sheets/Sheets.md)

## Property Value

This is a read only property whose value is a [Sheets](../Sheets/Sheets.md).

## Version

Introduced in version 4
