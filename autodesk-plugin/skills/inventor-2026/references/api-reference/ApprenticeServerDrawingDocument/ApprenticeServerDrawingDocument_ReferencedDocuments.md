# ApprenticeServerDrawingDocument.ReferencedDocuments Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Property that returns all the documents referenced by this document.

## Syntax

ApprenticeServerDrawingDocument.**ReferencedDocuments**() As [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md)

## Property Value

This is a read only property whose value is an [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md).

## Version

Introduced in version 11
