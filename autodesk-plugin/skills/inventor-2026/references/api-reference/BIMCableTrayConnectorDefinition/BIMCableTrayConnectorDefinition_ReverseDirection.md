# BIMCableTrayConnectorDefinition.ReverseDirection Method

Parent Object: [BIMCableTrayConnectorDefinition](../BIMCableTrayConnectorDefinition/BIMCableTrayConnectorDefinition.md)

## Description

Method that will reverse the direction of the connection.

## Syntax

BIMCableTrayConnectorDefinition.**ReverseDirection**()

## Version

Introduced in version 2011
