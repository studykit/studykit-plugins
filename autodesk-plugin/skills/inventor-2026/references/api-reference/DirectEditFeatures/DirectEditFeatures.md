# DirectEditFeatures Object

## Description

Part DirectEdit Features Collection Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DirectEditFeatures/DirectEditFeatures_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../DirectEditFeatures/DirectEditFeatures_Count.md) | Gets the number of items in this collection. |
| [Item](../DirectEditFeatures/DirectEditFeatures_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../DirectEditFeatures/DirectEditFeatures_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PartFeatures.DirectEditFeatures](../PartFeatures/PartFeatures_DirectEditFeatures.md), [SheetMetalFeatures.DirectEditFeatures](../SheetMetalFeatures/SheetMetalFeatures_DirectEditFeatures.md)

## Version

Introduced in version 2015
