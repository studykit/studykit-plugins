# SheetMetalStyle.SaveToGlobal Method

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

SheetMetalStyle.**SaveToGlobal**()

## Version

Introduced in version 2009
