# ContentTreeViewNodesEnumerator.Count Property

Parent Object: [ContentTreeViewNodesEnumerator](../ContentTreeViewNodesEnumerator/ContentTreeViewNodesEnumerator.md)

## Description

Property that returns the number of ContentTreeViewNode objects in the collection.

## Syntax

ContentTreeViewNodesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
