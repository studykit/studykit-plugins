# ControlDefinition.OverrideShortcut Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Gets/Sets the override shortcut assigned to the command. Setting this property to a null string clears the override.

## Syntax

ControlDefinition.**OverrideShortcut**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
