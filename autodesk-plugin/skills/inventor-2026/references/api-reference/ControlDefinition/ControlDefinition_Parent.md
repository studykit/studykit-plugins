# ControlDefinition.Parent Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

ControlDefinition.**Parent**() As [CommandManager](../CommandManager/CommandManager.md)

## Property Value

This is a read only property whose value is a [CommandManager](../CommandManager/CommandManager.md).

## Version

Introduced in version 9
