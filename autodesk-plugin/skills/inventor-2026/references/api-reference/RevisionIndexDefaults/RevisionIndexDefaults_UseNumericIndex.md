# RevisionIndexDefaults.UseNumericIndex Property

Parent Object: [RevisionIndexDefaults](../RevisionIndexDefaults/RevisionIndexDefaults.md)

## Description

Read-write property that gets and sets whether the revision index is numeric or not.

## Syntax

RevisionIndexDefaults.**UseNumericIndex**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
