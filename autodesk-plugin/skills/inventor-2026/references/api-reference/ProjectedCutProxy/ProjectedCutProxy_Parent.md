# ProjectedCutProxy.Parent Property

Parent Object: [ProjectedCutProxy](../ProjectedCutProxy/ProjectedCutProxy.md)

## Description

Property that returns the parent sketch of the projected cut edges.

## Syntax

ProjectedCutProxy.**Parent**() As [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Property Value

This is a read only property whose value is a [PlanarSketch](../PlanarSketch/PlanarSketch.md).

## Version

Introduced in version 2010
