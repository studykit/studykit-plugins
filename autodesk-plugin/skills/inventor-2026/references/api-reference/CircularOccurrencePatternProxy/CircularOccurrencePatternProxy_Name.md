# CircularOccurrencePatternProxy.Name Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Gets/Sets the name for the pattern.

## Syntax

CircularOccurrencePatternProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
