# CircularOccurrencePatternProxy.LockRotation Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Read-write property that gets and sets whether the patterned elements keep the same rotation as their parent occurrences. This defaults to False when the CircularOccurrencePattern is just created indicates that the patterned elements will be rotated.

## Syntax

CircularOccurrencePatternProxy.**LockRotation**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
