# CircularOccurrencePatternProxy.AxisEntity Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Gets/Sets the entity used to define the axis of the pattern.

## Syntax

CircularOccurrencePatternProxy.**AxisEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2010
