# BreakOutOperation.Type Property

Parent Object: [BreakOutOperation](../BreakOutOperation/BreakOutOperation.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

BreakOutOperation.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
