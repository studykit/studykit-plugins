# CurveGraphics.CurveType Property

Parent Object: [CurveGraphics](../CurveGraphics/CurveGraphics.md)

## Description

Property that returns the type of the underlying curve geometry that defines this curve.

## Syntax

CurveGraphics.**CurveType**() As [CurveTypeEnum](../CurveTypeEnum.md)

## Property Value

This is a read only property whose value is a [CurveTypeEnum](../CurveTypeEnum.md).

## Version

Introduced in version 2008
