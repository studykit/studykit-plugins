# Leader.ArrowheadType Property

Parent Object: [Leader](../Leader/Leader.md)

## Description

Gets and sets the arrowhead type of the leader.

## Syntax

Leader.**ArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2008
