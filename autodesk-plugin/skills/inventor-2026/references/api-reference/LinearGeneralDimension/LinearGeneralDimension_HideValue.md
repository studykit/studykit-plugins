# LinearGeneralDimension.HideValue Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets the HideValue setting.

## Syntax

LinearGeneralDimension.**HideValue**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
