# SurfaceGraphicsVertex.Vertex Property

Parent Object: [SurfaceGraphicsVertex](../SurfaceGraphicsVertex/SurfaceGraphicsVertex.md)

## Description

Read-only property that returns the Vertex object associated with the SurfaceGraphicsVertex.

## Syntax

SurfaceGraphicsVertex.**Vertex**() As [Vertex](../Vertex/Vertex.md)

## Property Value

This is a read only property whose value is a [Vertex](../Vertex/Vertex.md).

## Version

Introduced in version 2013
