# LoftSectionDimension.Position Property

Parent Object: [LoftSectionDimension](../LoftSectionDimension/LoftSectionDimension.md)

## Description

Read-write property that gets and sets the positional distance (from the starting section) of the placed section for an area-graph type loft.

## Syntax

LoftSectionDimension.**Position**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
