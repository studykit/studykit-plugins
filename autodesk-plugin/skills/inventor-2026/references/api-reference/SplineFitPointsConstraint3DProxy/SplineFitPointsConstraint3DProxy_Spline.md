# SplineFitPointsConstraint3DProxy.Spline Property

Parent Object: [SplineFitPointsConstraint3DProxy](../SplineFitPointsConstraint3DProxy/SplineFitPointsConstraint3DProxy.md)

## Description

Property that returns the spline entity being constrained.

## Syntax

SplineFitPointsConstraint3DProxy.**Spline**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 8
