# Sheet.RevisionTables Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the RevisionTables collection object.

## Syntax

Sheet.**RevisionTables**() As [RevisionTables](../RevisionTables/RevisionTables.md)

## Property Value

This is a read only property whose value is a [RevisionTables](../RevisionTables/RevisionTables.md).
