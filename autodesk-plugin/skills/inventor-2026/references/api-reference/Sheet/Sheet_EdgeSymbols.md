# Sheet.EdgeSymbols Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Read-only property that returns the collection of edge symbols on this sheet.

## Syntax

Sheet.**EdgeSymbols**() As [EdgeSymbols](../EdgeSymbols/EdgeSymbols.md)

## Property Value

This is a read only property whose value is an [EdgeSymbols](../EdgeSymbols/EdgeSymbols.md).

## Version

Introduced in version 2024
