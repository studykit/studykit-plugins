# KnitFeature.Surfaces Property

Parent Object: [KnitFeature](../KnitFeature/KnitFeature.md)

## Description

Gets and sets the set of surfaces associated with the Knit Feature.

## Syntax

KnitFeature.**Surfaces**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2008
