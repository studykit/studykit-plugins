# MeshEdge.MeshFeature Property

Parent Object: [MeshEdge](../MeshEdge/MeshEdge.md)

## Description

Read-only property that returns the MeshFeature this MeshEdge is located on.

## Syntax

MeshEdge.**MeshFeature**() As [MeshFeature](../MeshFeature/MeshFeature.md)

## Property Value

This is a read only property whose value is a [MeshFeature](../MeshFeature/MeshFeature.md).

## Version

Introduced in version 2017
