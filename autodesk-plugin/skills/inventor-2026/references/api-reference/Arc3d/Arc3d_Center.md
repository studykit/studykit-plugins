# Arc3d.Center Property

Parent Object: [Arc3d](../Arc3d/Arc3d.md)

## Description

Specifies the center point for this arc.

## Syntax

Arc3d.**Center**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
