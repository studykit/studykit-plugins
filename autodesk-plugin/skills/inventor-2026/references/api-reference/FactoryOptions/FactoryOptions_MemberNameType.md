# FactoryOptions.MemberNameType Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the type of value to use for the member name.

## Syntax

FactoryOptions.**MemberNameType**() As [FactoryOptionsMemberNameTypeEnum](../FactoryOptionsMemberNameTypeEnum.md)

## Property Value

This is a read/write property whose value is a [FactoryOptionsMemberNameTypeEnum](../FactoryOptionsMemberNameTypeEnum.md).

## Version

Introduced in version 2010
