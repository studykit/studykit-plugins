# ParallelToYAxisConstraint3DProxy.NativeObject Property

Parent Object: [ParallelToYAxisConstraint3DProxy](../ParallelToYAxisConstraint3DProxy/ParallelToYAxisConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ParallelToYAxisConstraint3DProxy.**NativeObject**() As [ParallelToYAxisConstraint3D](../ParallelToYAxisConstraint3D/ParallelToYAxisConstraint3D.md)

## Property Value

This is a read only property whose value is a [ParallelToYAxisConstraint3D](../ParallelToYAxisConstraint3D/ParallelToYAxisConstraint3D.md).

## Version

Introduced in version 2017
