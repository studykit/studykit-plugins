# SplineLengthDimConstraint3DProxy.TextPoint Property

Parent Object: [SplineLengthDimConstraint3DProxy](../SplineLengthDimConstraint3DProxy/SplineLengthDimConstraint3DProxy.md)

## Description

Gets and sets the position of the dimension text.

## Syntax

SplineLengthDimConstraint3DProxy.**TextPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2019
