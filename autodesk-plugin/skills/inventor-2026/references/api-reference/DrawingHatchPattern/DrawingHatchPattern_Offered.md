# DrawingHatchPattern.Offered Property

Parent Object: [DrawingHatchPattern](../DrawingHatchPattern/DrawingHatchPattern.md)

## Description

Read-write property that gets and sets the offered status of the hatch pattern.

## Syntax

DrawingHatchPattern.**Offered**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2021
