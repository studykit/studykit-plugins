# ImportedComponentProxy.Definition Property

Parent Object: [ImportedComponentProxy](../ImportedComponentProxy/ImportedComponentProxy.md)

## Description

Read-write property that gets and sets the ImportedComponentDefinition which defines the various inputs that were used to create the imported component.

## Syntax

ImportedComponentProxy.**Definition**() As [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md)

## Property Value

This is a read only property whose value is an [ImportedComponentDefinition](../ImportedComponentDefinition/ImportedComponentDefinition.md).

## Version

Introduced in version 2016
