# SketchSplineHandle3D.Tangent Property

Parent Object: [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md)

## Description

Gets and sets the tangent vector at the fit point associated with this handle.

## Syntax

SketchSplineHandle3D.**Tangent**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2008
