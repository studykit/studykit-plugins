# SketchSplineHandle3D.Weight Property

Parent Object: [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md)

## Description

Gets and sets the tangent weight at the fit point associated with this handle.

## Syntax

SketchSplineHandle3D.**Weight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
