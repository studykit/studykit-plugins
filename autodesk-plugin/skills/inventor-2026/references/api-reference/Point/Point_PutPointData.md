# Point.PutPointData Method

Parent Object: [Point](../Point/Point.md)

## Description

Method that sets the data defining this point.

## Syntax

Point.**PutPointData**( ***Coords***() As Double )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Coords | Double | Input/output Double that specifies the entity's coordinates. |

## Version

Introduced in version 4
