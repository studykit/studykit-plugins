# TransitionSymbol.AttachmentType Property

Parent Object: [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md)

## Description

Returns the attachment type for the transition symbol when it is created.

## Syntax

TransitionSymbol.**AttachmentType**() As [TransitionSymbolAttachmentTypeEnum](../TransitionSymbolAttachmentTypeEnum.md)

## Property Value

This is a read only property whose value is a [TransitionSymbolAttachmentTypeEnum](../TransitionSymbolAttachmentTypeEnum.md).

## Version

Introduced in version 2025.1
