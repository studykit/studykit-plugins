# TransitionSymbol.Definition Property

Parent Object: [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md)

## Description

Gets and sets the TransitionSymbolDefinition object associated with this transition symbol.

## Syntax

TransitionSymbol.**Definition**() As [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Property Value

This is a read/write property whose value is a [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md).

## Version

Introduced in version 2025.1
