# FlatPattern.BOMQuantity Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the BOMQuantity object.

## Syntax

FlatPattern.**BOMQuantity**() As [BOMQuantity](../BOMQuantity/BOMQuantity.md)

## Property Value

This is a read only property whose value is a [BOMQuantity](../BOMQuantity/BOMQuantity.md).

## Version

Introduced in version 2010
