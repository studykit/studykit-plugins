# FlatPattern.Parent Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the parent SheetMetalComponentDefinition object.

## Syntax

FlatPattern.**Parent**() As [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Property Value

This is a read only property whose value is a [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md).

## Version

Introduced in version 2008
