# FlatPattern.Occurrences Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the collection object.

## Syntax

FlatPattern.**Occurrences**() As [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md).

## Version

Introduced in version 2010
