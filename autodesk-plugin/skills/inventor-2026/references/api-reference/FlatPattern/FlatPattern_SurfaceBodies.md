# FlatPattern.SurfaceBodies Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns all of the *result* SurfaceBody objects contained within this ComponentDefinition.

## Syntax

FlatPattern.**SurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).

## Version

Introduced in version 2010
