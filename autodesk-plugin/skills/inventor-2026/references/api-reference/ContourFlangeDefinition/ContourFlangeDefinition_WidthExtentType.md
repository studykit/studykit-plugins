# ContourFlangeDefinition.WidthExtentType Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Read-only property that returns the width extent type. The valid values for this property are kDistanceExten, kFromToExtent, and kToExtent. This is applicable when the WidthExtentsFromSketchPlane property is True.

## Syntax

ContourFlangeDefinition.**WidthExtentType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).

## Version

Introduced in version 2026
