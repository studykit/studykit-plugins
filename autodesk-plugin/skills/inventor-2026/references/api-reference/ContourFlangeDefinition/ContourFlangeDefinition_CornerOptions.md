# ContourFlangeDefinition.CornerOptions Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Gets and sets the CornerOptions object that defines how the corners of the contour flange are modeled.

## Syntax

ContourFlangeDefinition.**CornerOptions**() As [CornerOptions](../CornerOptions/CornerOptions.md)

## Property Value

This is a read/write property whose value is a [CornerOptions](../CornerOptions/CornerOptions.md).

## Version

Introduced in version 2009
