# ContourFlangeDefinition.WidthExtent Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Read-only property that returns the PartFeatureExtent object that defines how the width extent of the contour flange feature is defined. This is applicable when the WidthExtentsFromSketchPlane is True.

## Syntax

ContourFlangeDefinition.**WidthExtent**() As [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md).

## Version

Introduced in version 2026
