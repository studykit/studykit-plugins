# RevolveFeatureProxy.AxisEntity Property

Parent Object: [RevolveFeatureProxy](../RevolveFeatureProxy/RevolveFeatureProxy.md)

## Description

Gets and sets the entity used to define the axis of revolution.

## Syntax

RevolveFeatureProxy.**AxisEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 9
