# Circle.Evaluator Property

Parent Object: [Circle](../Circle/Circle.md)

## Description

Gets the CurveEvaluator for this circle.

## Syntax

Circle.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 4
