# DrawingDocument.SelectionPreferences Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Gets the SelectionPreferences.

## Syntax

DrawingDocument.**SelectionPreferences**() As [SelectionPreferences](../SelectionPreferences/SelectionPreferences.md)

## Property Value

This is a read only property whose value is a [SelectionPreferences](../SelectionPreferences/SelectionPreferences.md).

## Version

Introduced in version 2016
