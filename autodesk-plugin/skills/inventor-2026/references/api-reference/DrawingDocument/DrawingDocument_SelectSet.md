# DrawingDocument.SelectSet Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the SelectSet object.

## Syntax

DrawingDocument.**SelectSet**() As [SelectSet](../SelectSet/SelectSet.md)

## Property Value

This is a read only property whose value is a [SelectSet](../SelectSet/SelectSet.md).
