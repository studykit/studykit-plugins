# DrawingDocument.PrintManager Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the PrintManager, or when called from a DrawingDocument it returns a DrawingPrintManager object.

## Syntax

DrawingDocument.**PrintManager**() As [PrintManager](../PrintManager/PrintManager.md)

## Property Value

This is a read only property whose value is a [PrintManager](../PrintManager/PrintManager.md).
