# DrawingDocument.ClientViews Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Returns document's client views.

## Syntax

DrawingDocument.**ClientViews**() As [ClientViews](../ClientViews/ClientViews.md)

## Property Value

This is a read only property whose value is a [ClientViews](../ClientViews/ClientViews.md).

## Version

Introduced in version 2017
