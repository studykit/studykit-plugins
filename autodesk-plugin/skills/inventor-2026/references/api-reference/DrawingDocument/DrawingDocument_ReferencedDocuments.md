# DrawingDocument.ReferencedDocuments Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns all the documents directly referenced by this document.

## Syntax

DrawingDocument.**ReferencedDocuments**() As [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md).

## Version

Introduced in version 11
