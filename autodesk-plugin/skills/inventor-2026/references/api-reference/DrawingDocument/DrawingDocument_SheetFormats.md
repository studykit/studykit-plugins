# DrawingDocument.SheetFormats Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the SheetFormats collection object.

## Syntax

DrawingDocument.**SheetFormats**() As [SheetFormats](../SheetFormats/SheetFormats.md)

## Property Value

This is a read only property whose value is a [SheetFormats](../SheetFormats/SheetFormats.md).
