# AngleConstraintProxy.Parent Property

Parent Object: [AngleConstraintProxy](../AngleConstraintProxy/AngleConstraintProxy.md)

## Description

Property that returns the parent of the object.

## Syntax

AngleConstraintProxy.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).
