## Description

SketchCircularPattern Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../SketchCircularPattern/SketchCircularPattern_Delete.md) | Deletes the sketch pattern. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchCircularPattern/SketchCircularPattern_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [AttributeSets](../SketchCircularPattern/SketchCircularPattern_AttributeSets.md) | Returns the AttributeSets collection object associated with this object. |
| [Constraints](../SketchCircularPattern/SketchCircularPattern_Constraints.md) | Returns a collection of sketch constraints that are tied directly to this entity. This collection consists of both geometric and dimension constraints. |
| [ConstraintStatus](../SketchCircularPattern/SketchCircularPattern_ConstraintStatus.md) | Returns the constraint status of the sketch entity. Possible return values are kFullyConstrainedConstraintStatus, kOverConstrainedConstraintStatus, kUnderConstrainedConstraintStatus and kUnknownConstraintStatus. |
| [Definition](../SketchCircularPattern/SketchCircularPattern_Definition.md) | Gets and sets the associative definition of the sketch patter. |
| [Elements](../SketchCircularPattern/SketchCircularPattern_Elements.md) | Returns the SketchPatternElements collection objec. |
| [Parent](../SketchCircularPattern/SketchCircularPattern_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [RangeBox](../SketchCircularPattern/SketchCircularPattern_RangeBox.md) | Returns a Box2d object which contains the diagonal points of a rectangle that tightly encloses the entity. |
| [Type](../SketchCircularPattern/SketchCircularPattern_Type.md) | Gets the constant that indicates the type of this object. |
