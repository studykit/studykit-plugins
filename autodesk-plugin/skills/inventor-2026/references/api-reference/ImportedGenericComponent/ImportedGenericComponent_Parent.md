# ImportedGenericComponent.Parent Property

Parent Object: [ImportedGenericComponent](../ImportedGenericComponent/ImportedGenericComponent.md)

## Description

Property that returns the parent object.

## Syntax

ImportedGenericComponent.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2016
