# RevisionTableStyle.RevisionTagLeaderStyle Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Gets and sets the leader style used for revision tags.

## Syntax

RevisionTableStyle.**RevisionTagLeaderStyle**() As [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Property Value

This is a read/write property whose value is a [LeaderStyle](../LeaderStyle/LeaderStyle.md).

## Version

Introduced in version 2011
