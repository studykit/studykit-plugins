# RevisionTableStyle.Type Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RevisionTableStyle.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
