# RevisionTableStyle.InsideLineColor Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Gets and sets the color of the inner lines of the table.

## Syntax

RevisionTableStyle.**InsideLineColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2011
