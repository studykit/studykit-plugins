# PromptMessages.Item Property

Parent Object: [PromptMessages](../PromptMessages/PromptMessages.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

PromptMessages.**Item**( ***Index*** As Variant ) As [PromptMessage](../PromptMessage/PromptMessage.md)

## Property Value

This is a read only property whose value is a [PromptMessage](../PromptMessage/PromptMessage.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value to specify a PromptMessage to return. This can be a Long value to indicate the index of the prompt message, or a String value to indicate the Id of the prompt message. |

## Version

Introduced in version 2025.1
