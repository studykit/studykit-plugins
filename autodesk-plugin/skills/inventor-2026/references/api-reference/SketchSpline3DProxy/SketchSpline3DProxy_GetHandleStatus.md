# SketchSpline3DProxy.GetHandleStatus Method

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Method that returns whether the handle at a given fit point is enabled or disabled.

## Syntax

SketchSpline3DProxy.**GetHandleStatus**( ***FitPoint*** As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FitPoint | [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md) | Input SketchPoint3D object that specifies the fit point at which to get the handle status. |

## Version

Introduced in version 2008
