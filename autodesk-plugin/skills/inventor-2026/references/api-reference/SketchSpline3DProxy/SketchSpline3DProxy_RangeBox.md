# SketchSpline3DProxy.RangeBox Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Property that returns a Box object which contains the opposing points of a rectangular box that is guaranteed to enclose this object.

## Syntax

SketchSpline3DProxy.**RangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 6
