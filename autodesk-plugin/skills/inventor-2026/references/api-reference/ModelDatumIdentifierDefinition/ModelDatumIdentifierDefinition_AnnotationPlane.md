# ModelDatumIdentifierDefinition.AnnotationPlane Property

Parent Object: [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md)

## Description

Read-write property that gets and sets the annotation plane for the datum identifier symbol.

## Syntax

ModelDatumIdentifierDefinition.**AnnotationPlane**() As [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md).

## Version

Introduced in version 2018
