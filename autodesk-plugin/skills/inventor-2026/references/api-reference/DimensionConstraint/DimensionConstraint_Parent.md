# DimensionConstraint.Parent Property

Parent Object: [DimensionConstraint](../DimensionConstraint/DimensionConstraint.md)

## Description

Property that returns the parent sketch of the object.

## Syntax

DimensionConstraint.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 5
