# SketchEquationCurve3D.Parent Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchEquationCurve3D.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 2014
