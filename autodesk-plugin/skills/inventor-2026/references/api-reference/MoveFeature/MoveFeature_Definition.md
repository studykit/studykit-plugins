# MoveFeature.Definition Property

Parent Object: [MoveFeature](../MoveFeature/MoveFeature.md)

## Description

Gets and sets the definition for the move feature.

## Syntax

MoveFeature.**Definition**() As [MoveDefinition](../MoveDefinition/MoveDefinition.md)

## Property Value

This is a read/write property whose value is a [MoveDefinition](../MoveDefinition/MoveDefinition.md).

## Version

Introduced in version 2013
