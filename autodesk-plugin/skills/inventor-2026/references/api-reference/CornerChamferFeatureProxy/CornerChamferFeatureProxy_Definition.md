# CornerChamferFeatureProxy.Definition Property

Parent Object: [CornerChamferFeatureProxy](../CornerChamferFeatureProxy/CornerChamferFeatureProxy.md)

## Description

Property that gets the CornerChamferDefinition object associated with this fold feature.

## Syntax

CornerChamferFeatureProxy.**Definition**() As [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md)

## Property Value

This is a read only property whose value is a [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md).

## Version

Introduced in version 2009
