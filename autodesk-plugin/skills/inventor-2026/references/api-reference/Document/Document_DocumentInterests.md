# Document.DocumentInterests Property

Parent Object: [Document](../Document/Document.md)

## Description

Property that returns the DocumentInterests collection object.

## Syntax

Document.**DocumentInterests**() As [DocumentInterests](../DocumentInterests/DocumentInterests.md)

## Property Value

This is a read only property whose value is a [DocumentInterests](../DocumentInterests/DocumentInterests.md).

## Version

Introduced in version 2008
