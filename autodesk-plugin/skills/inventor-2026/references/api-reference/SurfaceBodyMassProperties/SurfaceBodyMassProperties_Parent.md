# SurfaceBodyMassProperties.Parent Property

Parent Object: [SurfaceBodyMassProperties](../SurfaceBodyMassProperties/SurfaceBodyMassProperties.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

SurfaceBodyMassProperties.**Parent**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2023
