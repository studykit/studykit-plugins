# BendDefinition.UnfoldMethod Property

Parent Object: [BendDefinition](../BendDefinition/BendDefinition.md)

## Description

Gets and sets the UnfoldMethod object that defines how this bend is unfolded.

## Syntax

BendDefinition.**UnfoldMethod**() As [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Property Value

This is a read/write property whose value is a [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md).

## Version

Introduced in version 2009
