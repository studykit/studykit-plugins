# DiameterGeneralDimension.HideValue Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Gets and sets the HideValue setting.

## Syntax

DiameterGeneralDimension.**HideValue**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
