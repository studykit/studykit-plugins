# DiameterGeneralDimension.ExtensionLineTwo Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Property that returns the second extension line of the dimension.

## Syntax

DiameterGeneralDimension.**ExtensionLineTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2008
