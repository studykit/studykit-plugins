# DiameterGeneralDimension.PromoteToSketch Method

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Method that copies the dimension to the underlying sketch. This method only works for dimensions associated with a draft view.

## Syntax

DiameterGeneralDimension.**PromoteToSketch**() As [DimensionConstraint](../DimensionConstraint/DimensionConstraint.md)

## Version

Introduced in version 11
