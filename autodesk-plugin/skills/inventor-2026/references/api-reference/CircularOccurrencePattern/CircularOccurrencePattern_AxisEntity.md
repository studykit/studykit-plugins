# CircularOccurrencePattern.AxisEntity Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Gets/Sets the entity used to define the axis of the pattern.

## Syntax

CircularOccurrencePattern.**AxisEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 6
