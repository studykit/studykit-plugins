# CircularOccurrencePattern.PositioningMethod Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Read-write property that gets and sets the enum indicating the positioning method used for the component pattern.

## Syntax

CircularOccurrencePattern.**PositioningMethod**() As [PatternPositioningMethodEnum](../PatternPositioningMethodEnum.md)

## Property Value

This is a read/write property whose value is a [PatternPositioningMethodEnum](../PatternPositioningMethodEnum.md).

## Version

Introduced in version 2024
