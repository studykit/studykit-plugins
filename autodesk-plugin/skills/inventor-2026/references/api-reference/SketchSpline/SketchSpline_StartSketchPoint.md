# SketchSpline.StartSketchPoint Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Property that returns the that defines the position of the start of the line.

## Syntax

SketchSpline.**StartSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 8
