# SketchSpline.FitPointCount Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Property that returns the number of fit points for the spline.

## Syntax

SketchSpline.**FitPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.
