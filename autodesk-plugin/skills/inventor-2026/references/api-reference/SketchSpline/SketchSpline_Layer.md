# SketchSpline.Layer Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Gets and sets the layer applied to this sketch entity.

## Syntax

SketchSpline.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
