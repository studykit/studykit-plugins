# MiniToolbarTextBox.Text Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Gets and sets the text displayed in the text box.

## Syntax

MiniToolbarTextBox.**Text**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
