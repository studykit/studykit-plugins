# MiniToolbarTextBox.AutoHide Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Read-write property that gets and sets whether the control should be automatically hidden when the user moves the cursor away from the MiniToolbar.

## Syntax

MiniToolbarTextBox.**AutoHide**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
