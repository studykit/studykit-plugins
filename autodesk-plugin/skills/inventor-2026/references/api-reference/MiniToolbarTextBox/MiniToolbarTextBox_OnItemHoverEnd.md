# MiniToolbarTextBox.OnItemHoverEnd Event

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Event that is fired when the user stops hovering over an item in the drop down in the expanded state.

## Syntax

MiniToolbarTextBox.**OnItemHoverEnd**()

## Version

Introduced in version 2018
