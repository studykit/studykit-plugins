# MiniToolbarTextBox.SelectedItem Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Returns the item currently selected.

## Syntax

MiniToolbarTextBox.**SelectedItem**() As [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Property Value

This is a read only property whose value is a [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md).

## Version

Introduced in version 2018
