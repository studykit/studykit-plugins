# MiniToolbarTextBox.DisplayName Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Read-write property that gets and sets the display name for the control.

## Syntax

MiniToolbarTextBox.**DisplayName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
