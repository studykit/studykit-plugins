# SketchBlocksEnumerator Object

## Description

The SketchBlocksEnumerator allows iteration of available SketchBlock objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../SketchBlocksEnumerator/SketchBlocksEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchBlocksEnumerator/SketchBlocksEnumerator_Item.md) | Property that returns an item from the collection. You can provide either the index of the item in the collection, where the first item is index 1, or you can provide the name of the object. |
| [Type](../SketchBlocksEnumerator/SketchBlocksEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
