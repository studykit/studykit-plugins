# ClientFeatureProxy.AppearanceSourceType Property

Parent Object: [ClientFeatureProxy](../ClientFeatureProxy/ClientFeatureProxy.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

ClientFeatureProxy.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
