# Profile3DProxy.Parent Property

Parent Object: [Profile3DProxy](../Profile3DProxy/Profile3DProxy.md)

## Description

Property that returns the 3D sketch that the profile was derived from.

## Syntax

Profile3DProxy.**Parent**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 6
