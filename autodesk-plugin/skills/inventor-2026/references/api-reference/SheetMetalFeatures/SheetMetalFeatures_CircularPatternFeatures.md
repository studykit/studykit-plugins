# SheetMetalFeatures.CircularPatternFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the CircularPatternFeatures collection object. This collection provides access to existing CircularPatternFeature objects and provides functionality to create new CircularPatternFeature objects.

## Syntax

SheetMetalFeatures.**CircularPatternFeatures**() As [CircularPatternFeatures](../CircularPatternFeatures/CircularPatternFeatures.md)

## Property Value

This is a read only property whose value is a [CircularPatternFeatures](../CircularPatternFeatures/CircularPatternFeatures.md).

## Version

Introduced in version 5.3
