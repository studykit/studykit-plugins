# SheetMetalFeatures.MoveFaceFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the MoveFaceFeatures collection object.

## Syntax

SheetMetalFeatures.**MoveFaceFeatures**() As [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md)

## Property Value

This is a read only property whose value is a [MoveFaceFeatures](../MoveFaceFeatures/MoveFaceFeatures.md).

## Version

Introduced in version 10
