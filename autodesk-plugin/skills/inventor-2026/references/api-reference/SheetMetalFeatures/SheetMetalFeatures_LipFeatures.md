# SheetMetalFeatures.LipFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are LipFeatures.

## Syntax

SheetMetalFeatures.**LipFeatures**() As [LipFeatures](../LipFeatures/LipFeatures.md)

## Property Value

This is a read only property whose value is a [LipFeatures](../LipFeatures/LipFeatures.md).

## Version

Introduced in version 2010
