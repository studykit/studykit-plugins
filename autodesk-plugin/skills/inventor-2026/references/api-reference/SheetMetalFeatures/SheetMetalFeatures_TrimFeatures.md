# SheetMetalFeatures.TrimFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the TrimFeatures collection object. This collection provides access to existing TrimFeature objects and provides functionality to create new TrimFeature objects.

## Syntax

SheetMetalFeatures.**TrimFeatures**() As [TrimFeatures](../TrimFeatures/TrimFeatures.md)

## Property Value

This is a read only property whose value is a [TrimFeatures](../TrimFeatures/TrimFeatures.md).

## Version

Introduced in version 11
