# SheetMetalFeatures.CornerChamferFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the CornerChamferFeatures collection object.

## Syntax

SheetMetalFeatures.**CornerChamferFeatures**() As [CornerChamferFeatures](../CornerChamferFeatures/CornerChamferFeatures.md)

## Property Value

This is a read only property whose value is a [CornerChamferFeatures](../CornerChamferFeatures/CornerChamferFeatures.md).

## Version

Introduced in version 5.3
