# SheetMetalFeatures.SimplifyFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Read-only property that returns the SimplifyFeatures collection object.

## Syntax

SheetMetalFeatures.**SimplifyFeatures**() As [SimplifyFeatures](../SimplifyFeatures/SimplifyFeatures.md)

## Property Value

This is a read only property whose value is a [SimplifyFeatures](../SimplifyFeatures/SimplifyFeatures.md).

## Version

Introduced in version 2026
