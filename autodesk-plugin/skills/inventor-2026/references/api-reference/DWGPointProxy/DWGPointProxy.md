# DWGPointProxy Object

Derived from: [DWGPoint](../DWGPoint/DWGPoint.md) Object

## Description

DWGPointProxy Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DWGPointProxy/DWGPointProxy_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [ContainingOccurrence](../DWGPointProxy/DWGPointProxy_ContainingOccurrence.md) | Get the component occurrence context through which the native object is being seen through. |
| [HandleID](../DWGPointProxy/DWGPointProxy_HandleID.md) | Read-only property that returns the HandleID for the DWGEntity. |
| [NativeObject](../DWGPointProxy/DWGPointProxy_NativeObject.md) | Get the object in the context of the definition instead of the containing assembly. |
| [Parent](../DWGPointProxy/DWGPointProxy_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ParentEntity](../DWGPointProxy/DWGPointProxy_ParentEntity.md) | Read-only property that returns the parent DWGEntity of the entity. |
| [Point](../DWGPointProxy/DWGPointProxy_Point.md) | Read-only property that returns a Point indicating the position of the DWGPoint in model space. |
| [Type](../DWGPointProxy/DWGPointProxy_Type.md) | Gets the constant that indicates the type of this object. |
