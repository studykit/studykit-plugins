# RadialMarkingMenus.Type Property

Parent Object: [RadialMarkingMenus](../RadialMarkingMenus/RadialMarkingMenus.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

RadialMarkingMenus.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2013
