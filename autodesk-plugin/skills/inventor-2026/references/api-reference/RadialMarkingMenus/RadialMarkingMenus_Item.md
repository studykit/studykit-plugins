# RadialMarkingMenus.Item Property

Parent Object: [RadialMarkingMenus](../RadialMarkingMenus/RadialMarkingMenus.md)

## Description

Returns the specified RadialMarkingMenu from the collection.

## Syntax

RadialMarkingMenus.**Item**( ***Index*** As Variant ) As [RadialMarkingMenu](../RadialMarkingMenu/RadialMarkingMenu.md)

## Property Value

This is a read only property whose value is a [RadialMarkingMenu](../RadialMarkingMenu/RadialMarkingMenu.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long that specifies the object within the list to return. |

## Version

Introduced in version 2013
