# FlatPatternOrientation.FlipAlignmentAxis Property

Parent Object: [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Description

Read-write property that gets and sets whether alignment direction of flat pattern orientation is the opposite of the natural direction of the alignment axis.

## Syntax

FlatPatternOrientation.**FlipAlignmentAxis**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
