# ArcLengthDimConstraintProxy.Entity Property

Parent Object: [ArcLengthDimConstraintProxy](../ArcLengthDimConstraintProxy/ArcLengthDimConstraintProxy.md)

## Description

Gets the SketchArc2d object being constrained.

## Syntax

ArcLengthDimConstraintProxy.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 2013
