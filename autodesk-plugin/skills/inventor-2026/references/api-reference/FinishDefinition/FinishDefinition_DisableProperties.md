# FinishDefinition.DisableProperties Method

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Method that disables or enables the properties in this finish definition.

## Syntax

FinishDefinition.**DisableProperties**( ***PropertyNames***() As String, ***Disabled*** As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| PropertyNames | String | Input String array that specifies the properties to disable or enable for this finish definition. The properties can be:  Appearance  Comment  Description  ShortDescription  Thickness  Hardness  Depth |
| Disabled | Boolean | Input Boolean value that specifies whether the properties should be disabled or enabled for this finish definition. |

## Version

Introduced in version 2025
