# FinishDefinition.Hardness Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that provides access to the hardness of the feature.

## Syntax

FinishDefinition.**Hardness**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2024
