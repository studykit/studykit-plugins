# ApprenticeServer.HardwareOptions Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Get the HardwareOptions object.

## Syntax

ApprenticeServer.**HardwareOptions**() As [HardwareOptions](../HardwareOptions/HardwareOptions.md)

## Property Value

This is a read only property whose value is a [HardwareOptions](../HardwareOptions/HardwareOptions.md).

## Version

Introduced in version 11
