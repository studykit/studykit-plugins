# iFeatureTemplateDescriptors Object

## Description

The iFeatureTemplateDescriptors collection object provides access to all of the objects in a document.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../iFeatureTemplateDescriptors/iFeatureTemplateDescriptors_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../iFeatureTemplateDescriptors/iFeatureTemplateDescriptors_Count.md) | Property that returns the number of items in this collection. |
| [Item](../iFeatureTemplateDescriptors/iFeatureTemplateDescriptors_Item.md) | Returns the specified iFeatureTemplateDescriptor object from the collection. |
| [Type](../iFeatureTemplateDescriptors/iFeatureTemplateDescriptors_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ReferenceComponents.iFeatureTemplateDescriptors](../ReferenceComponents/ReferenceComponents_iFeatureTemplateDescriptors.md)

## Version

Introduced in version 5.3
