# ModelLeaderNoteDefinition.Leader Property

Parent Object: [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md)

## Description

Read-only property that returns the leader associated with the leader note. This property will return Nothing in the cases where this object is not associated with a leader.

## Syntax

ModelLeaderNoteDefinition.**Leader**() As [ModelLeader](../ModelLeader/ModelLeader.md)

## Property Value

This is a read only property whose value is a [ModelLeader](../ModelLeader/ModelLeader.md).

## Version

Introduced in version 2018
