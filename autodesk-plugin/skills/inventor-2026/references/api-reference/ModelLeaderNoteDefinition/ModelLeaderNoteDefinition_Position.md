# ModelLeaderNoteDefinition.Position Property

Parent Object: [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md)

## Description

Read-write property that gets and sets the position of the note in the part or assembly. The point is projected onto the orientation plane.

## Syntax

ModelLeaderNoteDefinition.**Position**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
