# CustomGraphicsText.parent Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Graphics/CustomGraphicsText.h>

## Description

Returns the parent Component for a top-level group or the CustomGraphicsGroup object for graphics entities and child groups.
