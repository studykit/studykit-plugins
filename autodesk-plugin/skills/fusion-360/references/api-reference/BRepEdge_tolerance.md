# BRepEdge.tolerance Property

Parent Object: [BRepEdge](BRepEdge.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/BRep/BRepEdge.h>

## Description

Returns the tolerance used by a tolerant edge. This value is only useful when isTolerant is true.

## Syntax

* [Python](#Python)
* [C++](#C++)

"bRepEdge\_var" is a variable referencing a BRepEdge object. |

"bRepEdge\_var" is a variable referencing a BRepEdge object. ```` ``` #include <Fusion/BRep/BRepEdge.h>  // Get the value of the property. double propertyValue = bRepEdge_var->tolerance(); ``` ```` |

## Property Value

This is a read only property whose value is a double.
