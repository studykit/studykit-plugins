# Preferences.generalPreferences Property

Parent Object: [Preferences](Preferences.htm)
Defined in namespace "adsk::core" and the header file is <Core/Application/Preferences.h>

## Description

Gets the GeneralPreferences object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"preferences\_var" is a variable referencing a Preferences object. |

"preferences\_var" is a variable referencing a Preferences object. ```` ``` #include <Core/Application/Preferences.h>  // Get the value of the property. Ptr<GeneralPreferences> propertyValue = preferences_var->generalPreferences(); ``` ```` |

## Property Value

This is a read only property whose value is a [GeneralPreferences](GeneralPreferences.htm).
