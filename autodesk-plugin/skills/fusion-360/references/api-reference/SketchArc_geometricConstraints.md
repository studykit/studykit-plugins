# SketchArc.geometricConstraints Property

Parent Object: [SketchArc](SketchArc.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Sketch/SketchArc.h>

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

* [Python](#Python)
* [C++](#C++)

"sketchArc\_var" is a variable referencing a SketchArc object. |

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.htm).
