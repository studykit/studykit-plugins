# Arc3D.copy Method

Parent Object: [Arc3D](Arc3D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Arc3D.h>

## Description

Creates and returns an independent copy of this Arc3D object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"arc3D\_var" is a variable referencing an [Arc3D](Arc3D.htm) object.```` ``` returnValue = arc3D_var.copy() ``` ```` |
