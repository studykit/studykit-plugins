# StitchFeature.objectType Property

Parent Object: [StitchFeature](StitchFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/StitchFeature.h>

## Description

This property is supported by all objects in the API and returns a string that contains the full name (namespace::objecttype) describing the type of the object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"stitchFeature\_var" is a variable referencing a StitchFeature object.  ```` ``` # Get the value of the property. propertyValue = stitchFeature_var.objectType ``` ```` |

"stitchFeature\_var" is a variable referencing a StitchFeature object. ```` ``` #include <Fusion/Features/StitchFeature.h>  // Get the value of the property. string propertyValue = stitchFeature_var->objectType(); ``` ```` |

## Property Value

This is a read only property whose value is a string.
