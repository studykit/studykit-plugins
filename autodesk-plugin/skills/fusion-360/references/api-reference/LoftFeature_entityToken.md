# LoftFeature.entityToken Property

Parent Object: [LoftFeature](LoftFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/LoftFeature.h>

## Description

Returns a token for the Feature object. This can be saved and used at a later time with the Design.findEntityByToken method to get back the same feature.

## Syntax

* [Python](#Python)
* [C++](#C++)

"loftFeature\_var" is a variable referencing a LoftFeature object.  ```` ``` # Get the value of the property. propertyValue = loftFeature_var.entityToken ``` ```` |

"loftFeature\_var" is a variable referencing a LoftFeature object. ```` ``` #include <Fusion/Features/LoftFeature.h>  // Get the value of the property. string propertyValue = loftFeature_var->entityToken(); ``` ```` |

## Property Value

This is a read only property whose value is a string.
