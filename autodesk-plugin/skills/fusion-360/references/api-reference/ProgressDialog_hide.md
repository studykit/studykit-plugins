# ProgressDialog.hide Method

Parent Object: [ProgressDialog](ProgressDialog.htm)
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/ProgressDialog.h>

## Description

Hides the progress dialog. This should be used when the process has completed.

* [Python](#Python)
* [C++](#C++)

"progressDialog\_var" is a variable referencing a [ProgressDialog](ProgressDialog.htm) object.```` ``` returnValue = progressDialog_var.hide() ``` ```` |
