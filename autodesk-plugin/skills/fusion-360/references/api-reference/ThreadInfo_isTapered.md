# ThreadInfo.isTapered Property

Parent Object: [ThreadInfo](ThreadInfo.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/ThreadInfo.h>

## Description

Indicates if this ThreadInfo object defines a standard or tapered thread.

## Syntax

* [Python](#Python)
* [C++](#C++)

"threadInfo\_var" is a variable referencing a ThreadInfo object. |

"threadInfo\_var" is a variable referencing a ThreadInfo object. ```` ``` #include <Fusion/Features/ThreadInfo.h>  // Get the value of the property. boolean propertyValue = threadInfo_var->isTapered(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
