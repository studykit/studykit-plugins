# SceneSettings.restoreDefaults Method

Parent Object: [SceneSettings](SceneSettings.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Render/SceneSettings.h>

## Description

Changes all of the scene settings to the default values.

## Syntax

* [Python](#Python)
* [C++](#C++)

"sceneSettings\_var" is a variable referencing a [SceneSettings](SceneSettings.htm) object.```` ``` returnValue = sceneSettings_var.restoreDefaults() ``` ```` |

"sceneSettings\_var" is a variable referencing a [SceneSettings](SceneSettings.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if setting to the default settings was successful. |
