# CAMTemplate.save Method

Parent Object: [CAMTemplate](CAMTemplate.htm)
Defined in namespace "adsk::cam" and the header file is <Cam/CAMTemplate/CAMTemplate.h>

## Description

Save the CAMTemplate to a file

## Syntax

* [Python](#Python)
* [C++](#C++)

"cAMTemplate\_var" is a variable referencing a [CAMTemplate](CAMTemplate.htm) object.```` ``` returnValue = cAMTemplate_var.save(filePath) ``` ```` |

"cAMTemplate\_var" is a variable referencing a [CAMTemplate](CAMTemplate.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if the template was saved successfully, false otherwise. |

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| filePath | string | The path to the file you wish to save |
