# Material.deleteMe Method

Parent Object: [Material](Material.htm)
Defined in namespace "adsk::core" and the header file is <Core/Materials/Material.h>

## Description

Deletes the material from the Design. This method only applies to materials in a Design that are unused

## Syntax

* [Python](#Python)
* [C++](#C++)

"material\_var" is a variable referencing a [Material](Material.htm) object.```` ``` returnValue = material_var.deleteMe() ``` ```` |

"material\_var" is a variable referencing a [Material](Material.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if the delete was successful. |
