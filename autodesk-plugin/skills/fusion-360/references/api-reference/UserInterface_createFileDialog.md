# UserInterface.createFileDialog Method

Parent Object: [UserInterface](UserInterface.htm)
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/UserInterface.h>

## Description

Creates a new FileDialog object which provides the ability to show a standard file selection dialog to the user.

## Syntax

* [Python](#Python)
* [C++](#C++)

"userInterface\_var" is a variable referencing a [UserInterface](UserInterface.htm) object.```` ``` returnValue = userInterface_var.createFileDialog() ``` ```` |

"userInterface\_var" is a variable referencing a [UserInterface](UserInterface.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| [FileDialog](FileDialog.htm) | Returns the created FileDialog object that you can use to define the contents of and display a standard file dialog. |
