# CanvasInput.flipHorizontal Method

Parent Object: [CanvasInput](CanvasInput.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Image/CanvasInput.h>

## Description

Flips the image along the horizontal axis. This is a convenience method that flips the direction of the X axis of the transform.

## Syntax

* [Python](#Python)
* [C++](#C++)

"canvasInput\_var" is a variable referencing a [CanvasInput](CanvasInput.htm) object.```` ``` returnValue = canvasInput_var.flipHorizontal() ``` ```` |

"canvasInput\_var" is a variable referencing a [CanvasInput](CanvasInput.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if the flip was successful. |
