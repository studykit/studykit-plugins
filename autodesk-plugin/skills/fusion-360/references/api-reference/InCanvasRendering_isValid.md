# InCanvasRendering.isValid Property

Parent Object: [InCanvasRendering](InCanvasRendering.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Render/InCanvasRendering.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"inCanvasRendering\_var" is a variable referencing an InCanvasRendering object. |

"inCanvasRendering\_var" is a variable referencing an InCanvasRendering object. ```` ``` #include <Fusion/Render/InCanvasRendering.h>  // Get the value of the property. boolean propertyValue = inCanvasRendering_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
