# SketchEllipse.geometricConstraints Property

Parent Object: [SketchEllipse](SketchEllipse.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Sketch/SketchEllipse.h>

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

* [C++](#C++)

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.htm).
