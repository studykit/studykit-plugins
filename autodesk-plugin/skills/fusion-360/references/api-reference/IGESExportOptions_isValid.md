# IGESExportOptions.isValid Property

Parent Object: [IGESExportOptions](IGESExportOptions.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Fusion/IGESExportOptions.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"iGESExportOptions\_var" is a variable referencing an IGESExportOptions object. |

"iGESExportOptions\_var" is a variable referencing an IGESExportOptions object. ```` ``` #include <Fusion/Fusion/IGESExportOptions.h>  // Get the value of the property. boolean propertyValue = iGESExportOptions_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
