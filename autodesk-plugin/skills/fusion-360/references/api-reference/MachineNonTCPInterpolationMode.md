# MachineNonTCPInterpolationMode Enumerator

## Description

Interpolation modes available for non-TCP motions.
Defined in namespace "adsk::cam" and the header file is <Cam\CamTypeDefs.h>

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| MachineNonTCPInterpolationMode\_IndependentAxes | 1 | Moves the axes together, completing the motion simultaneously, although the tool's tip may deviate from the direct line between the start and finish points. |
| MachineNonTCPInterpolationMode\_SynchronizedAxes | 0 | Moves the axes independently at maximum speed, potentially resulting in different completion times for each axis |

## Version

Introduced in version September 2025
