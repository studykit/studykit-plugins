# Matrix3D.asArray Method

Parent Object: [Matrix3D](Matrix3D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Matrix3D.h>

## Description

Returns the contents of the matrix as a 16 element array.
