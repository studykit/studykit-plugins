# SketchLine.geometricConstraints Property

Parent Object: [SketchLine](SketchLine.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Sketch/SketchLine.h>

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

* [Python](#Python)
* [C++](#C++)

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.htm).
