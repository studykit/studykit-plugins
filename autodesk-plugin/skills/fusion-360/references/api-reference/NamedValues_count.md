# NamedValues.count Property

Parent Object: [NamedValues](NamedValues.htm)
Defined in namespace "adsk::core" and the header file is <Core/Application/NamedValues.h>

## Description

Returns the number of name value pairs in this object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"namedValues\_var" is a variable referencing a NamedValues object. |

"namedValues\_var" is a variable referencing a NamedValues object. ```` ``` #include <Core/Application/NamedValues.h>  // Get the value of the property. uinteger propertyValue = namedValues_var->count(); ``` ```` |

## Property Value

This is a read only property whose value is a uinteger.
