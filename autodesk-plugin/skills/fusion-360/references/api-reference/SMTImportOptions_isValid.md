# SMTImportOptions.isValid Property

Parent Object: [SMTImportOptions](SMTImportOptions.htm)
Defined in namespace "adsk::core" and the header file is <Core/Application/SMTImportOptions.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"sMTImportOptions\_var" is a variable referencing a SMTImportOptions object. |

"sMTImportOptions\_var" is a variable referencing a SMTImportOptions object. ```` ``` #include <Core/Application/SMTImportOptions.h>  // Get the value of the property. boolean propertyValue = sMTImportOptions_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
