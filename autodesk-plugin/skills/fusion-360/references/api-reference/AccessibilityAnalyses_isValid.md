# AccessibilityAnalyses.isValid Property

Parent Object: [AccessibilityAnalyses](AccessibilityAnalyses.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Fusion/AccessibilityAnalyses.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"accessibilityAnalyses\_var" is a variable referencing an AccessibilityAnalyses object. |

"accessibilityAnalyses\_var" is a variable referencing an AccessibilityAnalyses object. ```` ``` #include <Fusion/Fusion/AccessibilityAnalyses.h>  // Get the value of the property. boolean propertyValue = accessibilityAnalyses_var->isValid(); ``` ```` |

## Property Value

This is a read only property whose value is a boolean.
