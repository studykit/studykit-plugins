# SceneSettings.saveAsDefaults Method

Parent Object: [SceneSettings](SceneSettings.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Render/SceneSettings.h>

## Description

Saves all of the scene settings as the default settings for this Design.

## Syntax

* [Python](#Python)
* [C++](#C++)

"sceneSettings\_var" is a variable referencing a [SceneSettings](SceneSettings.htm) object.```` ``` returnValue = sceneSettings_var.saveAsDefaults() ``` ```` |

"sceneSettings\_var" is a variable referencing a [SceneSettings](SceneSettings.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if saving the defaults was successful. |
