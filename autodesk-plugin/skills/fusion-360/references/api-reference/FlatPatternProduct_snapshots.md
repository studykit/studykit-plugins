# FlatPatternProduct.snapshots Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/SheetMetal/FlatPatternProduct.h>

## Description

Returns the Snapshots object associated with this design which provides access to the existing snapshots and the creation of new snapshots.

## Syntax

* [Python](#Python)
* [C++](#C++)

"flatPatternProduct\_var" is a variable referencing a FlatPatternProduct object. |

"flatPatternProduct\_var" is a variable referencing a FlatPatternProduct object. ```` ``` #include <Fusion/SheetMetal/FlatPatternProduct.h>  // Get the value of the property. Ptr<Snapshots> propertyValue = flatPatternProduct_var->snapshots(); ``` ```` |

## Property Value

This is a read only property whose value is a [Snapshots](Snapshots.htm).

Introduced in version October 2022
