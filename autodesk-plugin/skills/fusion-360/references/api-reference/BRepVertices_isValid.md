# BRepVertices.isValid Property

Parent Object: [BRepVertices](BRepVertices.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/BRep/BRepVertices.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"bRepVertices\_var" is a variable referencing a BRepVertices object. |

## Property Value

This is a read only property whose value is a boolean.
