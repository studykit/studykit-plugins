# Features.unstitchFeatures Property

Parent Object: [Features](Features.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/Features.h>

## Description

Returns the collection that provides access to the Unstitch features within the component and supports the creation of new Unstitch features.

## Syntax

* [Python](#Python)
* [C++](#C++)

"features\_var" is a variable referencing a Features object. |

"features\_var" is a variable referencing a Features object. ```` ``` #include <Fusion/Features/Features.h>  // Get the value of the property. Ptr<UnstitchFeatures> propertyValue = features_var->unstitchFeatures(); ``` ```` |

## Property Value

This is a read only property whose value is a [UnstitchFeatures](UnstitchFeatures.htm).

## Version

Introduced in version July 2015
