# ThickenFeature.objectType Property

Parent Object: [ThickenFeature](ThickenFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/ThickenFeature.h>

## Description

This property is supported by all objects in the API and returns a string that contains the full name (namespace::objecttype) describing the type of the object.

## Syntax

* [Python](#Python)
* [C++](#C++)

"thickenFeature\_var" is a variable referencing a ThickenFeature object.  ```` ``` # Get the value of the property. propertyValue = thickenFeature_var.objectType ``` ```` |

"thickenFeature\_var" is a variable referencing a ThickenFeature object. ```` ``` #include <Fusion/Features/ThickenFeature.h>  // Get the value of the property. string propertyValue = thickenFeature_var->objectType(); ``` ```` |

## Property Value

This is a read only property whose value is a string.
