# BaseFeature.finishEdit Method

Parent Object: [BaseFeature](BaseFeature.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/BaseFeature.h>

## Description

Exits from edit mode in the user-interface. If this base feature in not in edit mode, then nothing happens.
