# CopyPasteBodies.isValid Property

Parent Object: [CopyPasteBodies](CopyPasteBodies.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Features/CopyPasteBodies.h>

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

* [Python](#Python)
* [C++](#C++)

"copyPasteBodies\_var" is a variable referencing a CopyPasteBodies object. |

## Property Value

This is a read only property whose value is a boolean.
