# CurvatureMapAnalyses Object

Derived from: [Base](Base.htm) Object

Defined in namespace "adsk::fusion" and the header file is <Fusion/Fusion/CurvatureMapAnalyses.h>

## Description

Provides access to any curvature map analyses results in the design.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [classType](CurvatureMapAnalyses_classType.htm) | Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
| [item](CurvatureMapAnalyses_item.htm) | A method that returns the specified CurvatureMapAnalysis object using an index into the collection. |
| [itemByName](CurvatureMapAnalyses_itemByName.htm) | A method that returns the specified CurvatureMapAnalysis object using the name of the analysis as displayed in the browser. |

## Properties

|  |  |
| --- | --- |
| Name | Description |

| [isValid](CurvatureMapAnalyses_isValid.htm) | Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference. |
| [objectType](CurvatureMapAnalyses_objectType.htm) | This property is supported by all objects in the API and returns a string that contains the full name (namespace::objecttype) describing the type of the object. |
