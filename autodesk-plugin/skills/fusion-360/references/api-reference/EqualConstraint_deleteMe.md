# EqualConstraint.deleteMe Method

Parent Object: [EqualConstraint](EqualConstraint.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Sketch/EqualConstraint.h>

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.
