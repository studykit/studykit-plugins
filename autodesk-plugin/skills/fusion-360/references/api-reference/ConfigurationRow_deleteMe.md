# ConfigurationRow.deleteMe Method

Parent Object: [ConfigurationRow](ConfigurationRow.htm)
Defined in namespace "adsk::fusion" and the header file is <Fusion/Configurations/ConfigurationRow.h>

## Description

Deletes this row from the table. The first row of the top table cannot be deleted, and this method will fail.

* [C++](#C++)
