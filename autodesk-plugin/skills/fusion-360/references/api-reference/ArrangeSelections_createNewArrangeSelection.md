# ArrangeSelections.createNewArrangeSelection Method

Parent Object: [ArrangeSelections](ArrangeSelections.htm)
Defined in namespace "adsk::cam" and the header file is <Cam/GeometrySelections/ArrangeSelections.h>

## Description

Creates a new occurrence selection meant for arrange operations and adds it to the end of the collection.
