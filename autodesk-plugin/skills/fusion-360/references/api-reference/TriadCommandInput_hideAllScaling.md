# TriadCommandInput.hideAllScaling Method

Parent Object: [TriadCommandInput](TriadCommandInput.htm)
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/TriadCommandInput.h>

## Description

Sets all scaling related controls to be invisible. This is useful if you are only using translations or rotations.
