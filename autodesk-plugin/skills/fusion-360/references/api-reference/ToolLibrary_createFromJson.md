# ToolLibrary.createFromJson Method

Parent Object: [ToolLibrary](ToolLibrary.htm)
Defined in namespace "adsk::cam" and the header file is <Cam/Tools/ToolLibrary.h>

## Description

Creates a ToolLibrary by given JSON-string. Raises an error if the given JSON is invalid.
