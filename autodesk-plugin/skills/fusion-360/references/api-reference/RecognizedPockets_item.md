# RecognizedPockets.item Method

Parent Object: [RecognizedPockets](RecognizedPockets.htm)
Defined in namespace "adsk::cam" and the header file is <Cam/PocketRecognition/RecognizedPockets.h>

## Description

Returns the pocket at the specified index from this collection of pockets.

## Syntax

* [Python](#Python)
* [C++](#C++)

"recognizedPockets\_var" is a variable referencing a [RecognizedPockets](RecognizedPockets.htm) object.```` ``` returnValue = recognizedPockets_var.item(index) ``` ```` |

"recognizedPockets\_var" is a variable referencing a [RecognizedPockets](RecognizedPockets.htm) object. |

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| index | uinteger | The index of the pocket within this pocket collection to return. The first pocket in this collection has an index of 0. |
