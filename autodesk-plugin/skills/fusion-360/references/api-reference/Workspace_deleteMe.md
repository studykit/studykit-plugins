# Workspace.deleteMe Method

Parent Object: [Workspace](Workspace.htm)
Defined in namespace "adsk::core" and the header file is <Core/UserInterface/Workspace.h>

## Description

Deletes this workspace. Only a workspace added by the API can be deleted, (IsNative is false).

## Syntax

* [Python](#Python)
* [C++](#C++)

"workspace\_var" is a variable referencing a [Workspace](Workspace.htm) object.```` ``` returnValue = workspace_var.deleteMe() ``` ```` |

"workspace\_var" is a variable referencing a [Workspace](Workspace.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Boolean return that indicates if the deletion was successful or not. |
