# Document.activate Method

Parent Object: [Document](Document.htm)
Defined in namespace "adsk::core" and the header file is <Core/Application/Document.h>

## Description

Causes this document to become the active document in the user interface.

## Syntax

* [Python](#Python)
* [C++](#C++)

"document\_var" is a variable referencing a [Document](Document.htm) object.```` ``` returnValue = document_var.activate() ``` ```` |

"document\_var" is a variable referencing a [Document](Document.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| boolean | Returns true if the activation was successful. |
