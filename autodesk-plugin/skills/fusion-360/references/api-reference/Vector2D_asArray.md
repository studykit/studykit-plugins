# Vector2D.asArray Method

Parent Object: [Vector2D](Vector2D.htm)
Defined in namespace "adsk::core" and the header file is <Core/Geometry/Vector2D.h>

## Description

Returns the vector values as an array [x, y].

## Syntax

* [Python](#Python)
* [C++](#C++)

"vector2D\_var" is a variable referencing a [Vector2D](Vector2D.htm) object.```` ``` returnValue = vector2D_var.asArray() ``` ```` |

"vector2D\_var" is a variable referencing a [Vector2D](Vector2D.htm) object. |

## Return Value

|  |  |
| --- | --- |
| Type | Description |
| double[] | Returns an array of the vector's values [x, y]. |
