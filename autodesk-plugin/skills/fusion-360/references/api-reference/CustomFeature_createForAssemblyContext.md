## Description

Creates or returns a proxy for the native object - i.e. a new object that represents this object but adds the assembly context defined by the input occurrence.

## Syntax

* [Python](#Python)
* [C++](#C++)

"customFeature\_var" is a variable referencing a [CustomFeature](CustomFeature.htm) object.```` ``` returnValue = customFeature_var.createForAssemblyContext(occurrence) ``` ```` |
