# ComponentOccurrence.ImportedComponent Property

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Read-only property that returns the associative ImportedComponent object if the HasAssociativeImportedComponent returns True.

## Syntax

ComponentOccurrence.**ImportedComponent**() As [ImportedComponent](../ImportedComponent/ImportedComponent.md)

## Property Value

This is a read only property whose value is an [ImportedComponent](../ImportedComponent/ImportedComponent.md).

## Version

Introduced in version 2016
