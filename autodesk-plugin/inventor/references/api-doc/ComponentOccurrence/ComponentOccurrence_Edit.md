# ComponentOccurrence.Edit Method

Parent Object: [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Description

Activates this component occurrence for editing by the end user via the user interface.

## Syntax

ComponentOccurrence.**Edit**()

## Version

Introduced in version 6
