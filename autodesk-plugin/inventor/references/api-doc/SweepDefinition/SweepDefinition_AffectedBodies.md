# SweepDefinition.AffectedBodies Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the collection of bodies affected by this feature. If this property is not set for multi-body parts, the default solid body is used as the affected body. This property applies only to features in a part.

## Syntax

SweepDefinition.**AffectedBodies**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2015
