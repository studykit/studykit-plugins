# SweepDefinition.Profile Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the profile object used to define the shape of the sweep.

## Syntax

SweepDefinition.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 2015
