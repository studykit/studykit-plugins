# SweepDefinition.ProfileScaling Property

Parent Object: [SweepDefinition](../SweepDefinition/SweepDefinition.md)

## Description

Read-write property that gets and sets the scaling direction for sweep profile. This property is applicable when the SweepType is kPathAndGuideRailSweepType.

## Syntax

SweepDefinition.**ProfileScaling**() As [SweepProfileScalingEnum](../SweepProfileScalingEnum.md)

## Property Value

This is a read/write property whose value is a [SweepProfileScalingEnum](../SweepProfileScalingEnum.md).

## Version

Introduced in version 2015
