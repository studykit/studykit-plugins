# TwoLineAngleDimConstraint3D.LineOne Property

Parent Object: [TwoLineAngleDimConstraint3D](../TwoLineAngleDimConstraint3D/TwoLineAngleDimConstraint3D.md)

## Description

Property that returns the first 3D sketch line being constrained.

## Syntax

TwoLineAngleDimConstraint3D.**LineOne**() As [SketchLine3D](../SketchLine3D/SketchLine3D.md)

## Property Value

This is a read only property whose value is a [SketchLine3D](../SketchLine3D/SketchLine3D.md).

## Version

Introduced in version 11
