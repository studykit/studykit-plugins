# SketchPoint.Merge Method

Parent Object: [SketchPoint](../SketchPoint/SketchPoint.md)

## Description

Method that merges this sketch point with the input sketch point. Any objects dependent on this sketch point will change their dependency to the new sketch point.

## Syntax

SketchPoint.**Merge**( ***SketchPoint*** As [SketchPoint](../SketchPoint/SketchPoint.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SketchPoint | [SketchPoint](../SketchPoint/SketchPoint.md) | Input SketchPoint object. |

## Version

Introduced in version 5
