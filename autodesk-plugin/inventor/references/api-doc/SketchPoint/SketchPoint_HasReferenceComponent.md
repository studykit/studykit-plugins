# SketchPoint.HasReferenceComponent Property

Parent Object: [SketchPoint](../SketchPoint/SketchPoint.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchPoint.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5.3
