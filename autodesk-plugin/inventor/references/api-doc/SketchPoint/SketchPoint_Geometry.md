# SketchPoint.Geometry Property

Parent Object: [SketchPoint](../SketchPoint/SketchPoint.md)

## Description

Property that returns a Point2d geometry object. The object returned represents a snapshot view of the current state of the sketch point.

## Syntax

SketchPoint.**Geometry**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
