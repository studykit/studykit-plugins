# BOMViews Object

## Description

The BOMViews collection object provides access to the existing BOMView objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BOMViews/BOMViews_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BOMViews/BOMViews_Count.md) | Property that returns the number of items in the collection. |
| [Item](../BOMViews/BOMViews_Item.md) | Returns the specified BOMView object from the collection. |
| [Type](../BOMViews/BOMViews_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BOM.BOMViews](../BOM/BOM_BOMViews.md)

## Version

Introduced in version 10
