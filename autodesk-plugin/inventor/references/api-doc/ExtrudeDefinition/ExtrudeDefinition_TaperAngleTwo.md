# ExtrudeDefinition.TaperAngleTwo Property

Parent Object: [ExtrudeDefinition](../ExtrudeDefinition/ExtrudeDefinition.md)

## Description

Read-write property that provides access to the second direction taper angle of an asymmetric extrude feature. This property is not applicable and ret.

## Syntax

ExtrudeDefinition.**TaperAngleTwo**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2012
