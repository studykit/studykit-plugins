# BSplineCurveDefinition Object

## Description

The BSplineCurveDefinition object is a transient object. It is a convenient way to define a spline, perhaps to then create a BSplineCurve object. For more information, see the Transient Geometry overview.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddPoint](../BSplineCurveDefinition/BSplineCurveDefinition_AddPoint.md) | Method that adds point and its corresponding weight and tangent information to the definition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [FitMethod](../BSplineCurveDefinition/BSplineCurveDefinition_FitMethod.md) | Gets and sets the FitMethod setting. |
| [FixedLength](../BSplineCurveDefinition/BSplineCurveDefinition_FixedLength.md) | Gets and sets whether the curve should have a fixed length. |
| [Length](../BSplineCurveDefinition/BSplineCurveDefinition_Length.md) | Gets and sets the length for the curve. |
| [Type](../BSplineCurveDefinition/BSplineCurveDefinition_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[TransientGeometry.CreateBSplineCurveDefinition](../TransientGeometry/TransientGeometry_CreateBSplineCurveDefinition.md)
