# DrawingDocument.IsInventorDWG Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns whether this is an Inventor DWG document.

## Syntax

DrawingDocument.**IsInventorDWG**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
