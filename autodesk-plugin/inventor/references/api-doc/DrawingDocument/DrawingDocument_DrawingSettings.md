# DrawingDocument.DrawingSettings Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

The DrawingSettings object provides access to various drawing related document settings. This is somewhat equivalent to the Drawing tab of the Document Settings dialog.

## Syntax

DrawingDocument.**DrawingSettings**() As [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Property Value

This is a read only property whose value is a [DrawingSettings](../DrawingSettings/DrawingSettings.md).

## Version

Introduced in version 7
