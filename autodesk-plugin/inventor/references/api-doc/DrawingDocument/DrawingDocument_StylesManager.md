# DrawingDocument.StylesManager Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the DrawingStylesManager object.

## Syntax

DrawingDocument.**StylesManager**() As [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Property Value

This is a read only property whose value is a [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md).
