# DrawingDocument.Dirty Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Gets/Sets a Boolean flag indicating if the Document has been written into, since opened.

## Syntax

DrawingDocument.**Dirty**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 4
