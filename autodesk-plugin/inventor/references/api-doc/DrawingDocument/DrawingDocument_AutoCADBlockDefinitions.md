# DrawingDocument.AutoCADBlockDefinitions Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the AutoCADBlockDefinitions collection object.

## Syntax

DrawingDocument.**AutoCADBlockDefinitions**() As [AutoCADBlockDefinitions](../AutoCADBlockDefinitions/AutoCADBlockDefinitions.md)

## Property Value

This is a read only property whose value is an [AutoCADBlockDefinitions](../AutoCADBlockDefinitions/AutoCADBlockDefinitions.md).
