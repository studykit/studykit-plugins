# DrawingDocument.ReferencingDocuments Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns all the documents in memory that reference this document. A referencing document may or may not be fully open (i.e. may just be initialized).

## Syntax

DrawingDocument.**ReferencingDocuments**() As [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DocumentsEnumerator](../DocumentsEnumerator/DocumentsEnumerator.md).

## Version

Introduced in version 11
