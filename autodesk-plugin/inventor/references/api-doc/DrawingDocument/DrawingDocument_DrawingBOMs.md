# DrawingDocument.DrawingBOMs Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the collection of locally stored BOMs in this drawing document.

## Syntax

DrawingDocument.**DrawingBOMs**() As [DrawingBOMs](../DrawingBOMs/DrawingBOMs.md)

## Property Value

This is a read only property whose value is a [DrawingBOMs](../DrawingBOMs/DrawingBOMs.md).

## Version

Introduced in version 2009
