# DrawingDocument.Rebuild Method

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Performs compute operations on all of the entities within this document's scope as if all of the driving entities had been 'dirtied.'

## Syntax

DrawingDocument.**Rebuild**()

## Version

Introduced in version 4
