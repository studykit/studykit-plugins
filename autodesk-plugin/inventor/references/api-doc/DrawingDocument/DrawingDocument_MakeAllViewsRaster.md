# DrawingDocument.MakeAllViewsRaster Method

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Set all drawing views in the document to be raster views.

## Syntax

DrawingDocument.**MakeAllViewsRaster**()

## Version

Introduced in version 2014
