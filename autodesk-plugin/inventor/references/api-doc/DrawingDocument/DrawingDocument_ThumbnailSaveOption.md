# DrawingDocument.ThumbnailSaveOption Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Property that returns the current thumbnail (preview picture) save option.

## Syntax

DrawingDocument.**ThumbnailSaveOption**() As [ThumbnailSaveOptionEnum](../ThumbnailSaveOptionEnum.md)

## Property Value

This is a read only property whose value is a [ThumbnailSaveOptionEnum](../ThumbnailSaveOptionEnum.md).

## Version

Introduced in version 2010
