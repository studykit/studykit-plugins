# DrawingDocument.Activate Method

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Makes this document the active one (receives user-focus).

## Syntax

DrawingDocument.**Activate**()

## Version

Introduced in version 4
