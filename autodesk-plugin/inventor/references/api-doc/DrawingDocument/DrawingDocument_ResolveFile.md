# DrawingDocument.ResolveFile Method

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Method that invoke the resolve file for the document if there are references missing.

## Syntax

DrawingDocument.**ResolveFile**()

## Version

Introduced in version 2025
