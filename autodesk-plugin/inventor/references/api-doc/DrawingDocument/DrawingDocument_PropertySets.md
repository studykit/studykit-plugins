# DrawingDocument.PropertySets Property

Parent Object: [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Description

Gets the Property Sets object that controls the file's published-format properties.

## Syntax

DrawingDocument.**PropertySets**() As [PropertySets](../PropertySets/PropertySets.md)

## Property Value

This is a read only property whose value is a [PropertySets](../PropertySets/PropertySets.md).
