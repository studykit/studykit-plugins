# iFeatureTableCell.Column Property

Parent Object: [iFeatureTableCell](../iFeatureTableCell/iFeatureTableCell.md)

## Description

Property that returns the column this cell is within.

## Syntax

iFeatureTableCell.**Column**() As [iFeatureTableColumn](../iFeatureTableColumn/iFeatureTableColumn.md)

## Property Value

This is a read only property whose value is an [iFeatureTableColumn](../iFeatureTableColumn/iFeatureTableColumn.md).

## Version

Introduced in version 2009
