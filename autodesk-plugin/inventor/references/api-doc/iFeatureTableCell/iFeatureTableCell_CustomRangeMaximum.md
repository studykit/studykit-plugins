# iFeatureTableCell.CustomRangeMaximum Property

Parent Object: [iFeatureTableCell](../iFeatureTableCell/iFeatureTableCell.md)

## Description

Gets and sets the maximum value for a custom parameter cell.

## Syntax

iFeatureTableCell.**CustomRangeMaximum**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
