# TextGraphics.HorizontalAlignment Property

Parent Object: [TextGraphics](../TextGraphics/TextGraphics.md)

## Description

Read-write property that allows you to specify the horizontal justification of the text with respect to the coordinate point for the text.

## Syntax

TextGraphics.**HorizontalAlignment**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).
