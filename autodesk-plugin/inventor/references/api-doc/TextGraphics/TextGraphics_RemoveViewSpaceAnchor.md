# TextGraphics.RemoveViewSpaceAnchor Method

Parent Object: [TextGraphics](../TextGraphics/TextGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

TextGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
