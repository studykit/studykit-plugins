# FlangeDefinition.BendOptions Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Gets and sets the bend options associated with this bend feature.

## Syntax

FlangeDefinition.**BendOptions**() As [BendOptions](../BendOptions/BendOptions.md)

## Property Value

This is a read/write property whose value is a [BendOptions](../BendOptions/BendOptions.md).

## Version

Introduced in version 2009
