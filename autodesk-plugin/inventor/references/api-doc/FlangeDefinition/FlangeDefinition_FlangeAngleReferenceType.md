# FlangeDefinition.FlangeAngleReferenceType Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Read-write property that gets and sets the flange angle reference type. When change this from flange angle value type to reference plane type the SetFlangeAngleReferenceType should be used.

## Syntax

FlangeDefinition.**FlangeAngleReferenceType**() As [FlangeReferenceTypeEnum](../FlangeReferenceTypeEnum.md)

## Property Value

This is a read/write property whose value is a [FlangeReferenceTypeEnum](../FlangeReferenceTypeEnum.md).

## Version

Introduced in version 2025
