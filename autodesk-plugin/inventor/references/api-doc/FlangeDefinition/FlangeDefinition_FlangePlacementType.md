# FlangeDefinition.FlangePlacementType Property

Parent Object: [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Description

Read-write property that gets and sets the flange placement type.

## Syntax

FlangeDefinition.**FlangePlacementType**() As [DirectionEnum](../DirectionEnum.md)

## Property Value

This is a read/write property whose value is a [DirectionEnum](../DirectionEnum.md).

## Version

Introduced in version 2025
