# iAssemblyTableCell.IsValid Property

Parent Object: [iAssemblyTableCell](../iAssemblyTableCell/iAssemblyTableCell.md)

## Description

Property that returns whether the contents of this cell are valid. Invalid cells show with a yellow background in the user interface.

## Syntax

iAssemblyTableCell.**IsValid**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
