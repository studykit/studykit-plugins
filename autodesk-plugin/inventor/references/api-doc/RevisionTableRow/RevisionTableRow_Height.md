# RevisionTableRow.Height Property

Parent Object: [RevisionTableRow](../RevisionTableRow/RevisionTableRow.md)

## Description

Gets/Sets the height of the RevisionTable row.

## Syntax

RevisionTableRow.**Height**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
