# GeometricConstraints.AddMidPointToArc Method

Parent Object: [GeometricConstraints](../GeometricConstraints/GeometricConstraints.md)

## Description

Creates a new midpoint constraint between the point and arc.

## Syntax

GeometricConstraints.**AddMidPointToArc**( ***Point*** As [SketchPoint](../SketchPoint/SketchPoint.md), ***Arc*** As [SketchArc](../SketchArc/SketchArc.md) ) As [MidpointConstraint](../MidpointConstraint/MidpointConstraint.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [SketchPoint](../SketchPoint/SketchPoint.md) | Input SketchPoint entity. |
| Arc | [SketchArc](../SketchArc/SketchArc.md) | Input SketchLine entity. |

## Version

Introduced in version 2015
