# BendNote.BendEdge Property

Parent Object: [BendNote](../BendNote/BendNote.md)

## Description

Property that returns the bend edge associated with the note.

## Syntax

BendNote.**BendEdge**() As [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Property Value

This is a read only property whose value is a [DrawingCurve](../DrawingCurve/DrawingCurve.md).

## Version

Introduced in version 2010
