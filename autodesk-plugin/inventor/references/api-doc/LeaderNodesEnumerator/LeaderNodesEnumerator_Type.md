# LeaderNodesEnumerator.Type Property

Parent Object: [LeaderNodesEnumerator](../LeaderNodesEnumerator/LeaderNodesEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

LeaderNodesEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 11
