# Assets.Type Property

Parent Object: [Assets](../Assets/Assets.md)

## Description

Read-only property returning kAssetsObject indicating this object’s type.

## Syntax

Assets.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
