# ThreadFeature.ThreadInfoType Property

Parent Object: [ThreadFeature](../ThreadFeature/ThreadFeature.md)

## Description

Property that returns a indicating the thread type.

## Syntax

ThreadFeature.**ThreadInfoType**() As [ThreadTypeEnum](../ThreadTypeEnum.md)

## Property Value

This is a read only property whose value is a [ThreadTypeEnum](../ThreadTypeEnum.md).

## Version

Introduced in version 5.3
