# ThreadFeature.Type Property

Parent Object: [ThreadFeature](../ThreadFeature/ThreadFeature.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ThreadFeature.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 5
