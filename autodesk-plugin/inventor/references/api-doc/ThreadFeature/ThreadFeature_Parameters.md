# ThreadFeature.Parameters Property

Parent Object: [ThreadFeature](../ThreadFeature/ThreadFeature.md)

## Description

Property that returns all the parameters associated with the feature.

## Syntax

ThreadFeature.**Parameters**() As [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md)

## Property Value

This is a read only property whose value is a [ParametersEnumerator](../ParametersEnumerator/ParametersEnumerator.md).

## Version

Introduced in version 11
