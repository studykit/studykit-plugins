# DrawingBOMRows Object

## Description

The DrawingBOMRows collection object represents the collection of all rows of a drawing BOM.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingBOMRows/DrawingBOMRows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DrawingBOMRows/DrawingBOMRows_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DrawingBOMRows/DrawingBOMRows_Item.md) | Returns the specified DrawingBOMRow object from the collection. |
| [Type](../DrawingBOMRows/DrawingBOMRows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingBOM.DrawingBOMRows](../DrawingBOM/DrawingBOM_DrawingBOMRows.md)

## Version

Introduced in version 2009
