# LineSegment2d.StartPoint Property

Parent Object: [LineSegment2d](../LineSegment2d/LineSegment2d.md)

## Description

Specifies the start point of the line segment.

## Syntax

LineSegment2d.**StartPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
