# LineSegment2d.MidPoint Property

Parent Object: [LineSegment2d](../LineSegment2d/LineSegment2d.md)

## Description

Gets the mid point of the line segment.

## Syntax

LineSegment2d.**MidPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
