# iFeature.iFeatureDefinition Property

Parent Object: [iFeature](../iFeature/iFeature.md)

## Description

Property that returns the associated iFeatureDefinition.

## Syntax

iFeature.**iFeatureDefinition**() As [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md)

## Property Value

This is a read only property whose value is an [iFeatureDefinition](../iFeatureDefinition/iFeatureDefinition.md).
