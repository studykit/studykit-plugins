# RuledSurfaceFeatureProxy.Suppressed Property

Parent Object: [RuledSurfaceFeatureProxy](../RuledSurfaceFeatureProxy/RuledSurfaceFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

RuledSurfaceFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
