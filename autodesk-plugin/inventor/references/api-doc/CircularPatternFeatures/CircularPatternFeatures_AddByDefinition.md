# CircularPatternFeatures.AddByDefinition Method

Parent Object: [CircularPatternFeatures](../CircularPatternFeatures/CircularPatternFeatures.md)

## Description

Method that creates a new CircularPattern feature.

## Syntax

CircularPatternFeatures.**AddByDefinition**( ***Definition*** As [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md) ) As [CircularPatternFeature](../CircularPatternFeature/CircularPatternFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md) | Input CircularPatternFeatureDefinition objet that defines the CircularPatternFeature you want to create. |

## Version

Introduced in version 2017
