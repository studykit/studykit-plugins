# FeatureControlFrame.MergeSymbolOverridden Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Gets whether the OverrideMergeSymbol is set to override the MergeSymbol setting in FeatureControlFrameStyle. Set this to False will clear the override setting.

## Syntax

FeatureControlFrame.**MergeSymbolOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
