# FeatureControlFrame.Rotation Property

Parent Object: [FeatureControlFrame](../FeatureControlFrame/FeatureControlFrame.md)

## Description

Gets and sets the absolute rotation angle of the symbol in radians. The value can either be positive (counter-clockwise rotation) or negative (clockwise rotation).

## Syntax

FeatureControlFrame.**Rotation**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
