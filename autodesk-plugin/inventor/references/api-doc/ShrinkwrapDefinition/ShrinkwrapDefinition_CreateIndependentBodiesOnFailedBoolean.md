# ShrinkwrapDefinition.CreateIndependentBodiesOnFailedBoolean Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets whether multi-body part is created when a Boolean operation fails on a single solid body style option.

## Syntax

ShrinkwrapDefinition.**CreateIndependentBodiesOnFailedBoolean**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
