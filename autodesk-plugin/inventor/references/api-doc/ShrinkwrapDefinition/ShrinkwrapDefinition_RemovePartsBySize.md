# ShrinkwrapDefinition.RemovePartsBySize Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets whether remove parts by size or not.

## Syntax

ShrinkwrapDefinition.**RemovePartsBySize**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
