# ShrinkwrapDefinition.DeriveStyle Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets how bodies are derived.

## Syntax

ShrinkwrapDefinition.**DeriveStyle**() As [DerivedComponentStyleEnum](../DerivedComponentStyleEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentStyleEnum](../DerivedComponentStyleEnum.md).

## Version

Introduced in version 2018
