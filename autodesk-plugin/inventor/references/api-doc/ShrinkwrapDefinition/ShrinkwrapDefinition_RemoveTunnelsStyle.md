# ShrinkwrapDefinition.RemoveTunnelsStyle Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets the style to remove voids for the shrinkwrap.

## Syntax

ShrinkwrapDefinition.**RemoveTunnelsStyle**() As [ShrinkwrapRemoveStyleEnum](../ShrinkwrapRemoveStyleEnum.md)

## Property Value

This is a read/write property whose value is a [ShrinkwrapRemoveStyleEnum](../ShrinkwrapRemoveStyleEnum.md).

## Version

Introduced in version 2022
