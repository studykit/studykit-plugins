# ShrinkwrapDefinition.LinkFaceColorFromSource Property

Parent Object: [ShrinkwrapDefinition](../ShrinkwrapDefinition/ShrinkwrapDefinition.md)

## Description

Gets or Sets whether use the color override from source component.

## Syntax

ShrinkwrapDefinition.**LinkFaceColorFromSource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
