# MateConstraint.Parent Property

Parent Object: [MateConstraint](../MateConstraint/MateConstraint.md)

## Description

Property that returns the parent of the object.

## Syntax

MateConstraint.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).

## Version

Introduced in version 4
