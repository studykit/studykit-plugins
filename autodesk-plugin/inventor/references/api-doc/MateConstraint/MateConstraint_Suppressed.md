# MateConstraint.Suppressed Property

Parent Object: [MateConstraint](../MateConstraint/MateConstraint.md)

## Description

Gets/Sets the Boolean flag indicating whether this constraint is suppressed.

## Syntax

MateConstraint.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 4
