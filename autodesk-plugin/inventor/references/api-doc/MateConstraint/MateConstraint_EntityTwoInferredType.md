# MateConstraint.EntityTwoInferredType Property

Parent Object: [MateConstraint](../MateConstraint/MateConstraint.md)

## Description

Property that returns an enum indicating how the geometry of entity two is interpreted.

## Syntax

MateConstraint.**EntityTwoInferredType**() As [InferredTypeEnum](../InferredTypeEnum.md)

## Property Value

This is a read only property whose value is an [InferredTypeEnum](../InferredTypeEnum.md).

## Version

Introduced in version 6
