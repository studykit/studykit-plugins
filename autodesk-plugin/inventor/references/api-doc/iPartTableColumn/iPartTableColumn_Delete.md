# iPartTableColumn.Delete Method

Parent Object: [iPartTableColumn](../iPartTableColumn/iPartTableColumn.md)

## Description

Method that deletes this column in the factory. The "Member" and "Part Number" columns cannot be deleted.

## Syntax

iPartTableColumn.**Delete**()

## Version

Introduced in version 2011
