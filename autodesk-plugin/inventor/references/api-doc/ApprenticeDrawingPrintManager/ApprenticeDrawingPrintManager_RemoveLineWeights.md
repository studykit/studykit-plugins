# ApprenticeDrawingPrintManager.RemoveLineWeights Property

Parent Object: [ApprenticeDrawingPrintManager](../ApprenticeDrawingPrintManager/ApprenticeDrawingPrintManager.md)

## Description

Gets/Sets the Boolean flag indicating whether the Drawing line weights should be removed in the print.

## Syntax

ApprenticeDrawingPrintManager.**RemoveLineWeights**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
