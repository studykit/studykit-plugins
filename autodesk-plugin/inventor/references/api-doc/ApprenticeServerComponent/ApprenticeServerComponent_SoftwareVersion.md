# ApprenticeServerComponent.SoftwareVersion Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the object that encapsulates the version of the current software.

## Syntax

ApprenticeServerComponent.**SoftwareVersion**() As [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md)

## Property Value

This is a read only property whose value is a [SoftwareVersion](../SoftwareVersion/SoftwareVersion.md).

## Version

Introduced in version 4
