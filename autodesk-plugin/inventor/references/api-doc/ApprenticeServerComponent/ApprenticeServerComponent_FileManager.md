# ApprenticeServerComponent.FileManager Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets the file manager object.

## Syntax

ApprenticeServerComponent.**FileManager**() As [FileManager](../FileManager/FileManager.md)

## Property Value

This is a read only property whose value is a [FileManager](../FileManager/FileManager.md).

## Version

Introduced in version 6
