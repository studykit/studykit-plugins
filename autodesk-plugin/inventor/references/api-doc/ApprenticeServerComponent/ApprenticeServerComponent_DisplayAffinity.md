# ApprenticeServerComponent.DisplayAffinity Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets/Sets the Boolean flag that fine tunes the performance of Apprentice when it is being used to display Inventor files. Defaults to false.

## Syntax

ApprenticeServerComponent.**DisplayAffinity**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
