# ApprenticeServerComponent.MultiUserMode Property

Parent Object: [ApprenticeServerComponent](../ApprenticeServerComponent/ApprenticeServerComponent.md)

## Description

Gets/Sets the enum indicating the Multi-User mode.

## Syntax

ApprenticeServerComponent.**MultiUserMode**() As [MultiUserModeEnum](../MultiUserModeEnum.md)

## Property Value

This is a read/write property whose value is a [MultiUserModeEnum](../MultiUserModeEnum.md).

## Version

Introduced in version 6
