# SurfaceBody.Appearance Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Gets and sets the current appearance of the body.

## Syntax

SurfaceBody.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
