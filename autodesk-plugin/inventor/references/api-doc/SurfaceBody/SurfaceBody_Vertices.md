# SurfaceBody.Vertices Property

Parent Object: [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Description

Property that retrieves all vertices on the body.

## Syntax

SurfaceBody.**Vertices**() As [Vertices](../Vertices/Vertices.md)

## Property Value

This is a read only property whose value is a [Vertices](../Vertices/Vertices.md).

## Version

Introduced in version 6
