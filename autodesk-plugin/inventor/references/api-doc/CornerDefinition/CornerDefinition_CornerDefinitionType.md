# CornerDefinition.CornerDefinitionType Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Property that specifies the geometry that is used to measure the corner seam or miter gap.

## Syntax

CornerDefinition.**CornerDefinitionType**() As [CornerDefinitionTypeEnum](../CornerDefinitionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [CornerDefinitionTypeEnum](../CornerDefinitionTypeEnum.md).

## Version

Introduced in version 2010
