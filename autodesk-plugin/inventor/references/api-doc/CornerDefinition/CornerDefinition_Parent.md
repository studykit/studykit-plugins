# CornerDefinition.Parent Property

Parent Object: [CornerDefinition](../CornerDefinition/CornerDefinition.md)

## Description

Property that returns the parent CornerFeature object of this CornerDefinition object.

## Syntax

CornerDefinition.**Parent**() As [CornerFeature](../CornerFeature/CornerFeature.md)

## Property Value

This is a read only property whose value is a [CornerFeature](../CornerFeature/CornerFeature.md).

## Version

Introduced in version 2009
