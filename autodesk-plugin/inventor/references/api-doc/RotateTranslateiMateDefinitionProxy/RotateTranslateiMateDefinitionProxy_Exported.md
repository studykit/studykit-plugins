# RotateTranslateiMateDefinitionProxy.Exported Property

Parent Object: [RotateTranslateiMateDefinitionProxy](../RotateTranslateiMateDefinitionProxy/RotateTranslateiMateDefinitionProxy.md)

## Description

Read-write property that gets and sets whether the object is exported. Objects must be marked for export in order for them to be derived.

## Syntax

RotateTranslateiMateDefinitionProxy.**Exported**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
