# LoftRails Object

## Description

The LoftRails object is used to define the rails and their conditions in a loft feature.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../LoftRails/LoftRails_Add.md) | Method that creates a new LoftRail that represents a rail for a loft feature. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LoftRails/LoftRails_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LoftRails/LoftRails_Count.md) | Property that specifies the number of LoftRail objects in the collection. |
| [Item](../LoftRails/LoftRails_Item.md) | Method that returns the specified LoftRail object from the collection. |
| [Type](../LoftRails/LoftRails_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[LoftDefinition.LoftRails](../LoftDefinition/LoftDefinition_LoftRails.md)

## Version

Introduced in version 11
