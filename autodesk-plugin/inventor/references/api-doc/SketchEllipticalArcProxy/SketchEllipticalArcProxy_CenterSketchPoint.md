# SketchEllipticalArcProxy.CenterSketchPoint Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Property that gets the sketch point that defines the position of the center of the elliptical arc.

## Syntax

SketchEllipticalArcProxy.**CenterSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 6
