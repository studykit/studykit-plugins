# SketchEllipticalArcProxy.MinorRadius Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets and sets the minor radius.

## Syntax

SketchEllipticalArcProxy.**MinorRadius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 6
