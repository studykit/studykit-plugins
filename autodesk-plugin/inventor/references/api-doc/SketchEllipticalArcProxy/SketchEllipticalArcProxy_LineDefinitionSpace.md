# SketchEllipticalArcProxy.LineDefinitionSpace Property

Parent Object: [SketchEllipticalArcProxy](../SketchEllipticalArcProxy/SketchEllipticalArcProxy.md)

## Description

Gets and sets the LineDefinitionSpace applied to this elliptical arc.

## Syntax

SketchEllipticalArcProxy.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 9
