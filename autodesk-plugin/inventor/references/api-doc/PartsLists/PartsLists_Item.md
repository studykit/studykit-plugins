# PartsLists.Item Property

Parent Object: [PartsLists](../PartsLists/PartsLists.md)

## Description

Returns the specified PartsList object from the collection. This is the default property of the PartsLists collection object.

## Syntax

PartsLists.**Item**( ***Index*** As Long ) As [PartsList](../PartsList/PartsList.md)

## Property Value

This is a read only property whose value is a [PartsList](../PartsList/PartsList.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the index of the PartsList to return. |
