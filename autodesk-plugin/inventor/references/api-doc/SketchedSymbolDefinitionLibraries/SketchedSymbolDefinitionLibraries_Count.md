# SketchedSymbolDefinitionLibraries.Count Property

Parent Object: [SketchedSymbolDefinitionLibraries](../SketchedSymbolDefinitionLibraries/SketchedSymbolDefinitionLibraries.md)

## Description

Gets the number of items in this collection.

## Syntax

SketchedSymbolDefinitionLibraries.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
