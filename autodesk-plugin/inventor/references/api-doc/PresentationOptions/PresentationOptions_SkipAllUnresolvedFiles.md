# PresentationOptions.SkipAllUnresolvedFiles Property

Parent Object: [PresentationOptions](../PresentationOptions/PresentationOptions.md)

## Description

Enables/disables skip all unresolved files during openning.

## Syntax

PresentationOptions.**SkipAllUnresolvedFiles**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
