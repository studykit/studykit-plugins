# ChamferNote.ChamferEdgeTwo Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Property that returns that second chamfer edge. This can either be a linear DrawingCurve object or a SketchLine object from a sheet sketch.

## Syntax

ChamferNote.**ChamferEdgeTwo**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2010
