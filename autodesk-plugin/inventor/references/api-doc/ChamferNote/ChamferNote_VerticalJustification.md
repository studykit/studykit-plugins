# ChamferNote.VerticalJustification Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Gets/Sets vertical justification of the DrawingNote.

## Syntax

ChamferNote.**VerticalJustification**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 2010
