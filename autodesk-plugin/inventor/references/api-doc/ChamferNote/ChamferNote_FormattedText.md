# ChamferNote.FormattedText Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Gets/Sets formatted text of the DrawingNote.

## Syntax

ChamferNote.**FormattedText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
