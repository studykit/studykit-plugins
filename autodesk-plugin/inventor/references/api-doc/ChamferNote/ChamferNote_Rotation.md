# ChamferNote.Rotation Property

Parent Object: [ChamferNote](../ChamferNote/ChamferNote.md)

## Description

Gets/Sets the rotation of the LeaderNote.

## Syntax

ChamferNote.**Rotation**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2016
