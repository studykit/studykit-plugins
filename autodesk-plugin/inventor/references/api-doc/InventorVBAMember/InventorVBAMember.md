# InventorVBAMember Object

## Description

Represents a member within a InventorVBAComponent.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Execute](../InventorVBAMember/InventorVBAMember_Execute.md) | Method that is used to execute this member. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Arguments](../InventorVBAMember/InventorVBAMember_Arguments.md) | Property that returns the collection containing the member's arguments. |
| [MemberType](../InventorVBAMember/InventorVBAMember_MemberType.md) | Property that returns a constant that specifies the member type. |
| [Name](../InventorVBAMember/InventorVBAMember_Name.md) | Property that indicates the member's display name. |
| [Parent](../InventorVBAMember/InventorVBAMember_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../InventorVBAMember/InventorVBAMember_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [VBMember](../InventorVBAMember/InventorVBAMember_VBMember.md) | Property that returns the member object that corresponds to this InventorVBAMember object. |

## Accessed From

[InventorVBAArgument.Parent](../InventorVBAArgument/InventorVBAArgument_Parent.md), [InventorVBAArguments.Parent](../InventorVBAArguments/InventorVBAArguments_Parent.md), [InventorVBAMembers.Item](../InventorVBAMembers/InventorVBAMembers_Item.md)
