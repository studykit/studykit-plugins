# InventorVBAMember.MemberType Property

Parent Object: [InventorVBAMember](../InventorVBAMember/InventorVBAMember.md)

## Description

Property that returns a constant that specifies the member type.

## Syntax

InventorVBAMember.**MemberType**() As [MemberTypeEnum](../MemberTypeEnum.md)

## Property Value

This is a read only property whose value is a [MemberTypeEnum](../MemberTypeEnum.md).

## Version

Introduced in version 6
