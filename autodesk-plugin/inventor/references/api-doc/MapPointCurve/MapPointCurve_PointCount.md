# MapPointCurve.PointCount Property

Parent Object: [MapPointCurve](../MapPointCurve/MapPointCurve.md)

## Description

Property that specifies the number of MapPoints in the set.

## Syntax

MapPointCurve.**PointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 6
