# ShrinkwrapRemoveStyleEnum Enumerator

## Description

Constants identifying how holes/chamfers/fillets/pockets are removed.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kShrinkwrapRemoveAll | 114690 | Remove all. |
| kShrinkwrapRemoveByRange | 114691 | Remove in range. |
| kShrinkwrapRemoveNone | 114689 | Do not remove any. |

## Version

Introduced in version 2018
