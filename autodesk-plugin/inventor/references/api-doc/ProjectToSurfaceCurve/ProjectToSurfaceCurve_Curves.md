# ProjectToSurfaceCurve.Curves Property

Parent Object: [ProjectToSurfaceCurve](../ProjectToSurfaceCurve/ProjectToSurfaceCurve.md)

## Description

Read-write property that gets and sets the sketch curves to project. Valid objects are 2D&3D sketch entities.

## Syntax

ProjectToSurfaceCurve.**Curves**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2021
