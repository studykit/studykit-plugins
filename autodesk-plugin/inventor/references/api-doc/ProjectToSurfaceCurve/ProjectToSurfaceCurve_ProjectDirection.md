# ProjectToSurfaceCurve.ProjectDirection Property

Parent Object: [ProjectToSurfaceCurve](../ProjectToSurfaceCurve/ProjectToSurfaceCurve.md)

## Description

Read-write property that defines the project direction. The direction is bidirectional. If the ProjectionType is not kProjectAlongVectorType setting this property will fail. If the ProjectionType is kProjectAlongVectorType, set this to Nothing to use the norma.

## Syntax

ProjectToSurfaceCurve.**ProjectDirection**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2021
