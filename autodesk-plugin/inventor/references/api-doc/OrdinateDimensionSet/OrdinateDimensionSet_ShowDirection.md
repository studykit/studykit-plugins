# OrdinateDimensionSet.ShowDirection Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets whether to place an arrow on the origin dimension pointing in the positive direction.

## Syntax

OrdinateDimensionSet.**ShowDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
