# OrdinateDimensionSet.DimensionType Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Property that returns the dimension type of the ordinate set. Possible values are kAlignedDimensionType, kHorizontalDimensionType and kVerticalDimensionType.

## Syntax

OrdinateDimensionSet.**DimensionType**() As [DimensionTypeEnum](../DimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [DimensionTypeEnum](../DimensionTypeEnum.md).

## Version

Introduced in version 2010
