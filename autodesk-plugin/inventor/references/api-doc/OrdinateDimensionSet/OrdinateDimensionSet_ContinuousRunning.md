# OrdinateDimensionSet.ContinuousRunning Property

Parent Object: [OrdinateDimensionSet](../OrdinateDimensionSet/OrdinateDimensionSet.md)

## Description

Gets and sets whether to display continuity line between the set members.

## Syntax

OrdinateDimensionSet.**ContinuousRunning**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
