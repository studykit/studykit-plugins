# PitchAndRevolutionCoilExtent.Parent Property

Parent Object: [PitchAndRevolutionCoilExtent](../PitchAndRevolutionCoilExtent/PitchAndRevolutionCoilExtent.md)

## Description

Property that returns the parent PartFeature of the definition.

## Syntax

PitchAndRevolutionCoilExtent.**Parent**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 5.3
