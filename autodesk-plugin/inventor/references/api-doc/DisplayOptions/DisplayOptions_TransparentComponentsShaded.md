# DisplayOptions.TransparentComponentsShaded Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that enables or disables contrasting shading for the display of transparent components.

## Syntax

DisplayOptions.**TransparentComponentsShaded**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.2
