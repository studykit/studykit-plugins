# DisplayOptions.UseRayTracingForRealisticDisplay Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets and sets whether to enable ray tracing in realistic visual styles for new windows.

## Syntax

DisplayOptions.**UseRayTracingForRealisticDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
