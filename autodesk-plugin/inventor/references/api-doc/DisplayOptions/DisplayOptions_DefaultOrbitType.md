# DisplayOptions.DefaultOrbitType Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Gets/sets the default behavior for the orbit command.

## Syntax

DisplayOptions.**DefaultOrbitType**() As [OrbitTypeEnum](../OrbitTypeEnum.md)

## Property Value

This is a read/write property whose value is an [OrbitTypeEnum](../OrbitTypeEnum.md).

## Version

Introduced in version 2010
