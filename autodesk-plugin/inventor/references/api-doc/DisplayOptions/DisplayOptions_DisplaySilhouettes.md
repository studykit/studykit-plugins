# DisplayOptions.DisplaySilhouettes Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that specifies whether to display silhouette edges for active components.

## Syntax

DisplayOptions.**DisplaySilhouettes**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
