# DisplayOptions.InactiveComponentsEdgeDisplayFollowVisibility Property

Parent Object: [DisplayOptions](../DisplayOptions/DisplayOptions.md)

## Description

Read-write property that specifies whether inactive component edges display should follow the visibility setting of visual styles.

## Syntax

DisplayOptions.**InactiveComponentsEdgeDisplayFollowVisibility**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.2
