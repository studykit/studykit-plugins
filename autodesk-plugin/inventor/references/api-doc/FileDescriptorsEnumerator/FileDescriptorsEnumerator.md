# FileDescriptorsEnumerator Object

## Description

The FileDescriptorsEnumerator object provides information about file references held, both native and foreign.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FileDescriptorsEnumerator/FileDescriptorsEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FileDescriptorsEnumerator/FileDescriptorsEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../FileDescriptorsEnumerator/FileDescriptorsEnumerator_Item.md) | Returns the specified FileDescriptor object from the collection. |
| [Type](../FileDescriptorsEnumerator/FileDescriptorsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[File.ReferencedFileDescriptors](../File/File_ReferencedFileDescriptors.md)

## Version

Introduced in version 11
