# DerivedPartTransformDef.IncludeAll Method

Parent Object: [DerivedPartTransformDef](../DerivedPartTransformDef/DerivedPartTransformDef.md)

## Description

Method that causes all entities within the derived part component to be set to kDerivedIncludeAll.

## Syntax

DerivedPartTransformDef.**IncludeAll**()

## Version

Introduced in version 6
