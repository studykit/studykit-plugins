# ModelLeaderNode.Parent Property

Parent Object: [ModelLeaderNode](../ModelLeaderNode/ModelLeaderNode.md)

## Description

Read-only property that returns the parent of the leader.

## Syntax

ModelLeaderNode.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
