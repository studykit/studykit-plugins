# ModelLeaderNode.Delete Method

Parent Object: [ModelLeaderNode](../ModelLeaderNode/ModelLeaderNode.md)

## Description

Method that deletes this leader node, optionally retaining any dependent nodes.

## Syntax

ModelLeaderNode.**Delete**( [***RetainDependentNodes***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| RetainDependentNodes | Boolean | Input optional value that specifies if the any dependent nodes should also be deleted. Defaults to deleting all dependent nodes. |

## Version

Introduced in version 2018
