# Bend.BackFaces Property

Parent Object: [Bend](../Bend/Bend.md)

## Description

Property that returns the collection of front faces in the model that represent this bend.

## Syntax

Bend.**BackFaces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2010
