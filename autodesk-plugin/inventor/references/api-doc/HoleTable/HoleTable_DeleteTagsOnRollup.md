# HoleTable.DeleteTagsOnRollup Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets whether to delete tags except for the first in a series of hole tags when the row merge type is kRollupRowMerge.

## Syntax

HoleTable.**DeleteTagsOnRollup**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
