# HoleTable.TitleTextStyle Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Gets/Sets the text style used for the title of the table.

## Syntax

HoleTable.**TitleTextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).

## Version

Introduced in version 2009
