# HoleTable.HoleTableColumns Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Property that returns the HoleTableColumns collection object.

## Syntax

HoleTable.**HoleTableColumns**() As [HoleTableColumns](../HoleTableColumns/HoleTableColumns.md)

## Property Value

This is a read only property whose value is a [HoleTableColumns](../HoleTableColumns/HoleTableColumns.md).

## Version

Introduced in version 10
