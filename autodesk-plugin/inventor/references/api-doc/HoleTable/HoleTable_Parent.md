# HoleTable.Parent Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Property that returns the parent object of this HoleTable.

## Syntax

HoleTable.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 10
