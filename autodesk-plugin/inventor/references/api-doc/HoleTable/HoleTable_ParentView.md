# HoleTable.ParentView Property

Parent Object: [HoleTable](../HoleTable/HoleTable.md)

## Description

Property that returns the drawing view that this table is associated with.

## Syntax

HoleTable.**ParentView**() As [DrawingView](../DrawingView/DrawingView.md)

## Property Value

This is a read only property whose value is a [DrawingView](../DrawingView/DrawingView.md).

## Version

Introduced in version 2009
