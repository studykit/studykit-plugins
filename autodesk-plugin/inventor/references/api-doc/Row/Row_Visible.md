# Row.Visible Property

Parent Object: [Row](../Row/Row.md)

## Description

Gets and sets whether the row is visible.

## Syntax

Row.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
