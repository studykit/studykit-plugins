# FileDescriptor.ReferencedFile Property

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

Property that returns the File object being referenced. The property returns Nothing for foreign file references.

## Syntax

FileDescriptor.**ReferencedFile**() As [File](../File/File.md)

## Property Value

This is a read only property whose value is a [File](../File/File.md).

## Version

Introduced in version 11
