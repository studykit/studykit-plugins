# FileDescriptor.RelativeFileName Property

Parent Object: [FileDescriptor](../FileDescriptor/FileDescriptor.md)

## Description

Gets the relative file name from the resolved location of this reference. This property is only valid when ReferenceMissing is false.

## Syntax

FileDescriptor.**RelativeFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2008
