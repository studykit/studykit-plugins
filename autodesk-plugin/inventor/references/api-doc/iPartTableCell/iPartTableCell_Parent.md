# iPartTableCell.Parent Property

Parent Object: [iPartTableCell](../iPartTableCell/iPartTableCell.md)

## Description

Property that returns the parent iPartTableRow object.

## Syntax

iPartTableCell.**Parent**() As [iPartFactory](../iPartFactory/iPartFactory.md)

## Property Value

This is a read only property whose value is an [iPartFactory](../iPartFactory/iPartFactory.md).

## Version

Introduced in version 6
