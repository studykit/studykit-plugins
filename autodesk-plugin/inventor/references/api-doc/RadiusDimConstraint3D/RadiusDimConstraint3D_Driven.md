# RadiusDimConstraint3D.Driven Property

Parent Object: [RadiusDimConstraint3D](../RadiusDimConstraint3D/RadiusDimConstraint3D.md)

## Description

Gets and sets whether this dimension constraint is driving the geometry or the geometry is defining the value associated with the constraint.

## Syntax

RadiusDimConstraint3D.**Driven**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
