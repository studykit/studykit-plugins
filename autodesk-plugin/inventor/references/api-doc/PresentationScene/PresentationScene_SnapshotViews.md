# PresentationScene.SnapshotViews Property

Parent Object: [PresentationScene](../PresentationScene/PresentationScene.md)

## Description

Read-only property that returns PresentationSnapshotViews collection object.

## Syntax

PresentationScene.**SnapshotViews**() As [PresentationSnapshotViews](../PresentationSnapshotViews/PresentationSnapshotViews.md)

## Property Value

This is a read only property whose value is a [PresentationSnapshotViews](../PresentationSnapshotViews/PresentationSnapshotViews.md).

## Version

Introduced in version 2018
