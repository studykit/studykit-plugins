# PresentationScene.Parent Property

Parent Object: [PresentationScene](../PresentationScene/PresentationScene.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PresentationScene.**Parent**() As [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Property Value

This is a read only property whose value is a [PresentationDocument](../PresentationDocument/PresentationDocument.md).

## Version

Introduced in version 2018
