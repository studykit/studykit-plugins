# Edge.Wire Property

Parent Object: [Edge](../Edge/Edge.md)

## Description

Get the Wire containing this Edge. Returns Nothing if this Edge does not belong to a Wire.

## Syntax

Edge.**Wire**() As [Wire](../Wire/Wire.md)

## Property Value

This is a read only property whose value is a [Wire](../Wire/Wire.md).

## Version

Introduced in version 2008
