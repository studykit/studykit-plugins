# BalloonTip Object

## Description

BalloonTip Object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../BalloonTip/BalloonTip_Delete.md) | Method that deletes the BalloonTip object. |
| [Display](../BalloonTip/BalloonTip_Display.md) | Method that displays the BalloonTip object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BalloonTip/BalloonTip_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [ClientData](../BalloonTip/BalloonTip_ClientData.md) | Gets/sets property that contains some additional data that the client would like to store with the BalloonTip. |
| [Enabled](../BalloonTip/BalloonTip_Enabled.md) | Read-write property that gets and sets if the BalloonTip object is enabled or not. |
| [InternalName](../BalloonTip/BalloonTip_InternalName.md) | Read-only property that returns the internal name of the BalloonTip object. |
| [Interval](../BalloonTip/BalloonTip_Interval.md) | Read-write property that gets and sets the time interval for displaying the BalloonTip object. |
| [Message](../BalloonTip/BalloonTip_Message.md) | Gets/sets the message of the BalloonTip object. This message supports the : <Hyperlink></Hyperlink> tag to create hyperlink string. |
| [Parent](../BalloonTip/BalloonTip_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Type](../BalloonTip/BalloonTip_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
