# EdgeUse.Next Property

Parent Object: [EdgeUse](../EdgeUse/EdgeUse.md)

## Description

Gets the next in the connected list of EdgeUses in the EdgeLoop.

## Syntax

EdgeUse.**Next**() As [EdgeUse](../EdgeUse/EdgeUse.md)

## Property Value

This is a read only property whose value is an [EdgeUse](../EdgeUse/EdgeUse.md).

## Version

Introduced in version 4
