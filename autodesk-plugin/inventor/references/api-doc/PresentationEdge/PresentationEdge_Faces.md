# PresentationEdge.Faces Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Read-only property that returns the PresentationFace objects that are adjacent to this edge.

## Syntax

PresentationEdge.**Faces**() As [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md)

## Property Value

This is a read only property whose value is a [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md).

## Version

Introduced in version 2018
