# PresentationEdge.GeometryType Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Read-only property that returns the curve type of the curve that will be returned from the Geometry property.

## Syntax

PresentationEdge.**GeometryType**() As [CurveTypeEnum](../CurveTypeEnum.md)

## Property Value

This is a read only property whose value is a [CurveTypeEnum](../CurveTypeEnum.md).

## Version

Introduced in version 2018
