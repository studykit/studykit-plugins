# PresentationEdge.StopVertex Property

Parent Object: [PresentationEdge](../PresentationEdge/PresentationEdge.md)

## Description

Read-only property that returns the PresentationVertex object referenced at the end of this presentation edge.

## Syntax

PresentationEdge.**StopVertex**() As [PresentationVertex](../PresentationVertex/PresentationVertex.md)

## Property Value

This is a read only property whose value is a [PresentationVertex](../PresentationVertex/PresentationVertex.md).

## Version

Introduced in version 2018
