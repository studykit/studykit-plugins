# ProfilePath3DProxy Object

Derived from: [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetReferenceKey](../ProfilePath3DProxy/ProfilePath3DProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ProfilePath3DProxy/ProfilePath3DProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../ProfilePath3DProxy/ProfilePath3DProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [Closed](../ProfilePath3DProxy/ProfilePath3DProxy_Closed.md) | Property that returns a Boolean indicating if the path is closed or not. Returns True in the case of a closed path. |
| [ContainingOccurrence](../ProfilePath3DProxy/ProfilePath3DProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Count](../ProfilePath3DProxy/ProfilePath3DProxy_Count.md) | Property that returns the number of items in the collection. |
| [Item](../ProfilePath3DProxy/ProfilePath3DProxy_Item.md) | Method that returns the specified ProfileEntity3D object from the collection. |
| [NativeObject](../ProfilePath3DProxy/ProfilePath3DProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../ProfilePath3DProxy/ProfilePath3DProxy_Parent.md) | Property that returns the parent Profile3D. |
| [Type](../ProfilePath3DProxy/ProfilePath3DProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version
