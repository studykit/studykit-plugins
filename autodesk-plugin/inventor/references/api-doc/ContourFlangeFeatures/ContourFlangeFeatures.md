# ContourFlangeFeatures Object

## Description

The ContourFlangeFeatures collection object provides access to the ContourFlangeFeature objects. It"s also through the ContourFlangeFeatures object that you create new ContourFlangeFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ContourFlangeFeatures/ContourFlangeFeatures_Add.md) | Method that creates a new contour flange feature. The newly created ContourFlangeFeature object is returned. |
| [CreateDefinition](../ContourFlangeFeatures/ContourFlangeFeatures_CreateDefinition.md) | Creates a new ContourFlangeDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ContourFlangeFeatures/ContourFlangeFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ContourFlangeFeatures/ContourFlangeFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ContourFlangeFeatures/ContourFlangeFeatures_Item.md) | Returns the specified ContourFlangeFeature object from the collection. This is the default property of the ContourFlangeFeatures collection object. |
| [Type](../ContourFlangeFeatures/ContourFlangeFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SheetMetalFeatures.ContourFlangeFeatures](../SheetMetalFeatures/SheetMetalFeatures_ContourFlangeFeatures.md)

## Version

Introduced in version 5.3
