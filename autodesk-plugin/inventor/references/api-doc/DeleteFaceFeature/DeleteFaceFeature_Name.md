# DeleteFaceFeature.Name Property

Parent Object: [DeleteFaceFeature](../DeleteFaceFeature/DeleteFaceFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

DeleteFaceFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 6
