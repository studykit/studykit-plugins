# MiniToolbarDropdown.OnSelect Event

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Event that is fired soon after the user changes the currently selected item using the drop down.

## Syntax

MiniToolbarDropdown.**OnSelect**( ***ListItem*** As [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ListItem | [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) | Input MiniToolbarListItem that is the item just selected. |
