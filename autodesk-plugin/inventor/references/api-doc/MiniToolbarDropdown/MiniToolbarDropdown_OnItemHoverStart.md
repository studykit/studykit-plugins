# MiniToolbarDropdown.OnItemHoverStart Event

Parent Object: [MiniToolbarDropdown](../MiniToolbarDropdown/MiniToolbarDropdown.md)

## Description

Event that is fired when the user starts to hover over an item in the drop down in the expanded state.

## Syntax

MiniToolbarDropdown.**OnItemHoverStart**( ***ListItem*** As [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ListItem | [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md) | Input Long that is the index of the item the user is starting to hover over. |

## Version

Introduced in version 2012
