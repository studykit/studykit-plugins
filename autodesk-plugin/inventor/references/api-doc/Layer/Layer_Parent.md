# Layer.Parent Property

Parent Object: [Layer](../Layer/Layer.md)

## Description

Property returning the parent of this object.

## Syntax

Layer.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
