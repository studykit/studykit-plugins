# SweepDefinitionTypeEnum Enumerator

## Description

Enum indicating the type of the sweep definition.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kPathAndGuideRailSweepDef | 59138 | Sweep defined by a path and a guide rail. |
| kPathAndGuideSurfaceSweepDef | 59139 | Sweep defined by a path and a guide surface. |
| kPathAndSectionTwistsSweepDef | 59140 | Sweep defined by a path and section twists. |
| kPathSweepDef | 59137 | Sweep defined by path and profile orientation. |

## Version

Introduced in version 11
