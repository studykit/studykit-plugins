# DWGPolyline3D.HandleID Property

Parent Object: [DWGPolyline3D](../DWGPolyline3D/DWGPolyline3D.md)

## Description

Read-only property that returns the HandleID for the DWGEntity.

## Syntax

DWGPolyline3D.**HandleID**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
