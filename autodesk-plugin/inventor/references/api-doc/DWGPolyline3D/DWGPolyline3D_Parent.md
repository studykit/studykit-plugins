# DWGPolyline3D.Parent Property

Parent Object: [DWGPolyline3D](../DWGPolyline3D/DWGPolyline3D.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

DWGPolyline3D.**Parent**() As [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Property Value

This is a read only property whose value is an [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md).

## Version

Introduced in version 2016
