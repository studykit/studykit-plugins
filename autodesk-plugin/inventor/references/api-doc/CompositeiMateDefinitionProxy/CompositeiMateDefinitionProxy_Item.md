# CompositeiMateDefinitionProxy.Item Property

Parent Object: [CompositeiMateDefinitionProxy](../CompositeiMateDefinitionProxy/CompositeiMateDefinitionProxy.md)

## Description

Returns the specified iMateDefinition object.

## Syntax

CompositeiMateDefinitionProxy.**Item**( ***Index*** As Long ) As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 6
