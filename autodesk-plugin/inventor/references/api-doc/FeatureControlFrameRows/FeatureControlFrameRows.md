# FeatureControlFrameRows Object

## Description

The FeatureControlFrameRows object provides access to all rows of a feature control frame.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../FeatureControlFrameRows/FeatureControlFrameRows_Add.md) | Method that adds a row to the feature control frame. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FeatureControlFrameRows/FeatureControlFrameRows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FeatureControlFrameRows/FeatureControlFrameRows_Count.md) | Property that returns the number of items in the collection. |
| [Item](../FeatureControlFrameRows/FeatureControlFrameRows_Item.md) | Method that returns the specified FeatureControlFrameRow object from the collection. |
| [Type](../FeatureControlFrameRows/FeatureControlFrameRows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
