# CosmeticWelds.CreateDefinition Method

Parent Object: [CosmeticWelds](../CosmeticWelds/CosmeticWelds.md)

## Description

Method that creates a CosmeticWeldDefinition object.

## Syntax

CosmeticWelds.**CreateDefinition**( ***Edges*** As [EdgeCollection](../EdgeCollection/EdgeCollection.md) ) As [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Edges | [EdgeCollection](../EdgeCollection/EdgeCollection.md) | Input the EdgeCollection object that specifies the edges on which to create the cosmetic weld. |

## Version

Introduced in version 2026
