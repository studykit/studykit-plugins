# CoincidentConstraint3D.EntityOne Property

Parent Object: [CoincidentConstraint3D](../CoincidentConstraint3D/CoincidentConstraint3D.md)

## Description

Property that returns the first entity being constrained. This can be a SketchPoint3D, SketchLine3D, WorkPlane or a Face.

## Syntax

CoincidentConstraint3D.**EntityOne**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
