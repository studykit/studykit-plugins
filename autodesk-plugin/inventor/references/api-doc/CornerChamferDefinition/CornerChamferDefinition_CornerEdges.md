# CornerChamferDefinition.CornerEdges Property

Parent Object: [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md)

## Description

Gets the edges used for this corner chamfer.

## Syntax

CornerChamferDefinition.**CornerEdges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2009
