# DrawingHatchPattern.Description Property

Parent Object: [DrawingHatchPattern](../DrawingHatchPattern/DrawingHatchPattern.md)

## Description

Read-only property that returns the description of the hatch pattern.

## Syntax

DrawingHatchPattern.**Description**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2021
