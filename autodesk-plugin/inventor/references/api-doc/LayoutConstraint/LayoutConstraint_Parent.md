# LayoutConstraint.Parent Property

Parent Object: [LayoutConstraint](../LayoutConstraint/LayoutConstraint.md)

## Description

Property that returns the parent of the object.

## Syntax

LayoutConstraint.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).

## Version

Introduced in version 2010
