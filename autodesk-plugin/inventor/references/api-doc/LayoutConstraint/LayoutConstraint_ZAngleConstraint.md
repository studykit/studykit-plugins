# LayoutConstraint.ZAngleConstraint Property

Parent Object: [LayoutConstraint](../LayoutConstraint/LayoutConstraint.md)

## Description

Property that returns the angle constraint that defines the rotation about the Z axis of the layout constraint.

## Syntax

LayoutConstraint.**ZAngleConstraint**() As [AngleConstraint](../AngleConstraint/AngleConstraint.md)

## Property Value

This is a read only property whose value is an [AngleConstraint](../AngleConstraint/AngleConstraint.md).

## Version

Introduced in version 2010
