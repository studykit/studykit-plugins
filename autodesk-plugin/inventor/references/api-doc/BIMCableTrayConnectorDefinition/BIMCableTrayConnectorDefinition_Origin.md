# BIMCableTrayConnectorDefinition.Origin Property

Parent Object: [BIMCableTrayConnectorDefinition](../BIMCableTrayConnectorDefinition/BIMCableTrayConnectorDefinition.md)

## Description

Read-only property that returns the origin of the connector.

## Syntax

BIMCableTrayConnectorDefinition.**Origin**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2014
