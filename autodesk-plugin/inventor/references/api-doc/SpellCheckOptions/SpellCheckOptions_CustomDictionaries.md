# SpellCheckOptions.CustomDictionaries Property

Parent Object: [SpellCheckOptions](../SpellCheckOptions/SpellCheckOptions.md)

## Description

Read-only property that returns the CustomDictionaries object containing all the custom dictionaries.

## Syntax

SpellCheckOptions.**CustomDictionaries**() As [CustomDictionaries](../CustomDictionaries/CustomDictionaries.md)

## Property Value

This is a read only property whose value is a [CustomDictionaries](../CustomDictionaries/CustomDictionaries.md).

## Version

Introduced in version 2020
