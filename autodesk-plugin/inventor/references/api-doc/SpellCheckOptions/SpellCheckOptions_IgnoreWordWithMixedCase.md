# SpellCheckOptions.IgnoreWordWithMixedCase Property

Parent Object: [SpellCheckOptions](../SpellCheckOptions/SpellCheckOptions.md)

## Description

Read-write property that gets and sets whether or not to ignore the word with mixed case(e.g. TrueType).

## Syntax

SpellCheckOptions.**IgnoreWordWithMixedCase**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2020
