# BalloonStylesEnumerator.Item Property

Parent Object: [BalloonStylesEnumerator](../BalloonStylesEnumerator/BalloonStylesEnumerator.md)

## Description

Returns the specified BalloonStyle object from the collection.

## Syntax

BalloonStylesEnumerator.**Item**( ***Index*** As Variant ) As [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Property Value

This is a read only property whose value is a [BalloonStyle](../BalloonStyle/BalloonStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the BalloonStyle to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the BalloonStyle name. If an out of range index or a name of a non-existent BalloonStyle is provided, an error will occur. |

## Version

Introduced in version 2009
