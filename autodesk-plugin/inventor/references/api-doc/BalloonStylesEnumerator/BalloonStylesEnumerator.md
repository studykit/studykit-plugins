# BalloonStylesEnumerator Object

## Description

The BalloonStylesEnumerator object provides access to the existing BalloonStyle objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BalloonStylesEnumerator/BalloonStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../BalloonStylesEnumerator/BalloonStylesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../BalloonStylesEnumerator/BalloonStylesEnumerator_Item.md) | Returns the specified BalloonStyle object from the collection. |
| [Type](../BalloonStylesEnumerator/BalloonStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.BalloonStyles](../DrawingStylesManager/DrawingStylesManager_BalloonStyles.md)

## Version

Introduced in version 2009
