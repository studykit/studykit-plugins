# EllipticalArc.Evaluator Property

Parent Object: [EllipticalArc](../EllipticalArc/EllipticalArc.md)

## Description

Gets the CurveEvaluator for the arc.

## Syntax

EllipticalArc.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 6
