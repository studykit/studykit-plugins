# TriangleGraphics.TextureCoordinateSet Property

Parent Object: [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Description

Read-write property that gets and sets the GraphicsCoordinateSet associated with the set.

## Syntax

TriangleGraphics.**TextureCoordinateSet**() As [GraphicsTextureCoordinateSet](../GraphicsTextureCoordinateSet/GraphicsTextureCoordinateSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsTextureCoordinateSet](../GraphicsTextureCoordinateSet/GraphicsTextureCoordinateSet.md).
