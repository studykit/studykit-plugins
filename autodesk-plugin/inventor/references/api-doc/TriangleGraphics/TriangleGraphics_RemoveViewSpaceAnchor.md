# TriangleGraphics.RemoveViewSpaceAnchor Method

Parent Object: [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

TriangleGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
