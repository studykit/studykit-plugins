# Circle.Radius Property

Parent Object: [Circle](../Circle/Circle.md)

## Description

The radius of the circle.

## Syntax

Circle.**Radius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 4
