# DWGEntityLineSegment.Parent Property

Parent Object: [DWGEntityLineSegment](../DWGEntityLineSegment/DWGEntityLineSegment.md)

## Description

Read-only property that returns the parent DWGEntity.

## Syntax

DWGEntityLineSegment.**Parent**() As [DWGEntity](../DWGEntity/DWGEntity.md)

## Property Value

This is a read only property whose value is a [DWGEntity](../DWGEntity/DWGEntity.md).

## Version

Introduced in version 2018
