# ReferenceKeyEvents Object

## Description

Object that provides access to events associated with custom reference keys.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ReferenceKeyEvents/ReferenceKeyEvents_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Type](../ReferenceKeyEvents/ReferenceKeyEvents_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Events

|  |  |
| --- | --- |
| Name | Description |
| [OnBindKeyToObject](../ReferenceKeyEvents/ReferenceKeyEvents_OnBindKeyToObject.md) | Event that fires when either the BindKeyToObject or CanBindKeyToObject method of the ReferenceKeyManager object is called with a custom reference key. |

## Version

Introduced in version 2012
