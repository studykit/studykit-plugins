# SplitTypeEnum Enumerator

## Description

Constants identifying the kind of split.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kSplitBody | 32771 | Split feature caused body to split. |
| kSplitFaces | 32770 | Split feature caused faces to split. |
| kSplitPart | 32769 | Split feature caused part to split. |
| kTrimSolid | 32769 | Split feature caused solid to be trimmed. |
