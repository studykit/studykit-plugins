# AssetCategories Object

## Description

The AssetCategories collection object provides access to the categories associated with a library.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../AssetCategories/AssetCategories_Add.md) | Method that creates a new asset category in the library. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../AssetCategories/AssetCategories_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../AssetCategories/AssetCategories_Count.md) | Gets the number of items in this collection. |
| [Item](../AssetCategories/AssetCategories_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../AssetCategories/AssetCategories_Type.md) | Read-only property returning kAssetCategoriesObject indicating this object’s type. |

## Accessed From

[AssetLibrary.AppearanceAssetCategories](../AssetLibrary/AssetLibrary_AppearanceAssetCategories.md), [AssetLibrary.MaterialAssetCategories](../AssetLibrary/AssetLibrary_MaterialAssetCategories.md)

## Version

Introduced in version 2014
