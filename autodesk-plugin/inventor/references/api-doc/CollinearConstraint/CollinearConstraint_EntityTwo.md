# CollinearConstraint.EntityTwo Property

Parent Object: [CollinearConstraint](../CollinearConstraint/CollinearConstraint.md)

## Description

Property that returns the second sketch entity being constrained.

## Syntax

CollinearConstraint.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
