# CustomConstraint3DProxy.Deletable Property

Parent Object: [CustomConstraint3DProxy](../CustomConstraint3DProxy/CustomConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

CustomConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
