# MacroControlDefinition.AutoAddToGUI Method

Parent Object: [MacroControlDefinition](../MacroControlDefinition/MacroControlDefinition.md)

## Description

Method that automatically adds a control based on this definition to the General panel of the Add-Ins tab in the ribbon interface.

## Syntax

MacroControlDefinition.**AutoAddToGUI**()

## Version

Introduced in version 9
