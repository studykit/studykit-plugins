# OffsetFromFaceExtent.IsTwoDirectional Property

Parent Object: [OffsetFromFaceExtent](../OffsetFromFaceExtent/OffsetFromFaceExtent.md)

## Description

Read-write property that gets and sets the offset from face, this can be a Face or WorkPlane object. This is applicable when the Direction is not set to kSymmetricExtentDirection.

## Syntax

OffsetFromFaceExtent.**IsTwoDirectional**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
