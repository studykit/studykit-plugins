# OffsetFromFaceExtent.Parent Property

Parent Object: [OffsetFromFaceExtent](../OffsetFromFaceExtent/OffsetFromFaceExtent.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

OffsetFromFaceExtent.**Parent**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 2018
