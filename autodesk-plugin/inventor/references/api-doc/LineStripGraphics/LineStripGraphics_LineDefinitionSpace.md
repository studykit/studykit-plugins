# LineStripGraphics.LineDefinitionSpace Property

Parent Object: [LineStripGraphics](../LineStripGraphics/LineStripGraphics.md)

## Description

Gets and sets the LineDefinitionSpace applied to this line strip graphics.

## Syntax

LineStripGraphics.**LineDefinitionSpace**() As [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md)

## Property Value

This is a read/write property whose value is a [LineDefinitionSpaceEnum](../LineDefinitionSpaceEnum.md).

## Version

Introduced in version 2008
