# iAssemblyTableRow Object

## Description

The iAssemblyTableRow object represents a row in the iAssembly factory table.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../iAssemblyTableRow/iAssemblyTableRow_Copy.md) | Method that creates a new row with all values equal to the original row. |
| [Delete](../iAssemblyTableRow/iAssemblyTableRow_Delete.md) | Method that deletes this row in the factory. |
| [GetReferenceKey](../iAssemblyTableRow/iAssemblyTableRow_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |
| [IsExcluded](../iAssemblyTableRow/iAssemblyTableRow_IsExcluded.md) | Method that returns whether the input object is excluded in this member. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../iAssemblyTableRow/iAssemblyTableRow_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../iAssemblyTableRow/iAssemblyTableRow_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [Count](../iAssemblyTableRow/iAssemblyTableRow_Count.md) | Property that specifies the number of items in the collection. |
| [DocumentName](../iAssemblyTableRow/iAssemblyTableRow_DocumentName.md) | Gets and sets the document name corresponding to this row in the iAssembly table. |
| [DocumentNameModifiable](../iAssemblyTableRow/iAssemblyTableRow_DocumentNameModifiable.md) | Property that returns whether the name of the document corresponding to this member can be modified. |
| [Index](../iAssemblyTableRow/iAssemblyTableRow_Index.md) | Property that returns the index of this row within the iAssembly factory table. |
| [IsDocumentNameBasedOnMemberName](../iAssemblyTableRow/iAssemblyTableRow_IsDocumentNameBasedOnMemberName.md) | Gets whether the member's filename is based on the member name or an explicitly set filename. |
| [Item](../iAssemblyTableRow/iAssemblyTableRow_Item.md) | Returns the specified iAssemblyTableCell object from the collection. |
| [MemberName](../iAssemblyTableRow/iAssemblyTableRow_MemberName.md) | Property that returns the member name corresponding to this row in the iAssembly table. |
| [Parent](../iAssemblyTableRow/iAssemblyTableRow_Parent.md) | Property that returns the parent iAssemblyFactory object. |
| [Type](../iAssemblyTableRow/iAssemblyTableRow_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
