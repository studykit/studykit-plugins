# IntegerAssetValue.HasMultipleValues Property

Parent Object: [IntegerAssetValue](../IntegerAssetValue/IntegerAssetValue.md)

## Description

Gets the boolean flag that indicates if this value has multiple values or not.

## Syntax

IntegerAssetValue.**HasMultipleValues**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2014
