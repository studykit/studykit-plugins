# ComponentDefinition.Occurrences Property

Parent Object: [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Description

Property that returns the collection object.

## Syntax

ComponentDefinition.**Occurrences**() As [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrences](../ComponentOccurrences/ComponentOccurrences.md).
