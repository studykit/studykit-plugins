# ComponentDefinition.SurfaceBodies Property

Parent Object: [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Description

Property that returns all of the *result* SurfaceBody objects contained within this ComponentDefinition.

## Syntax

ComponentDefinition.**SurfaceBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).
