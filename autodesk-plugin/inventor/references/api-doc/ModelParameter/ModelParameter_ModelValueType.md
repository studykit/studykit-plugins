# ModelParameter.ModelValueType Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Gets/(Sets) the setting which is used for determining the method used to compute the model value.

## Syntax

ModelParameter.**ModelValueType**() As [ModelValueTypeEnum](../ModelValueTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ModelValueTypeEnum](../ModelValueTypeEnum.md).

## Version

Introduced in version 6
