# ModelParameter.ExposedAsProperty Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Gets/(Sets) the Boolean indicating if this parameter is exposed as a property of this Document.

## Syntax

ModelParameter.**ExposedAsProperty**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
