# ModelParameter.Renamed Property

Parent Object: [ModelParameter](../ModelParameter/ModelParameter.md)

## Description

Gets whether the ModelParameter has been renamed by the user.

## Syntax

ModelParameter.**Renamed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
