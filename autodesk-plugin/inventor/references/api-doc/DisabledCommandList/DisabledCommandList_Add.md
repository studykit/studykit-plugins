# DisabledCommandList.Add Method

Parent Object: [DisabledCommandList](../DisabledCommandList/DisabledCommandList.md)

## Description

Method that adds the specified command to the list.

## Syntax

DisabledCommandList.**Add**( ***Command*** As [ControlDefinition](../ControlDefinition/ControlDefinition.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Command | [ControlDefinition](../ControlDefinition/ControlDefinition.md) | Input ControlDefinition object that is to be added to the disabled command list. |
