# DisabledCommandList.Count Property

Parent Object: [DisabledCommandList](../DisabledCommandList/DisabledCommandList.md)

## Description

Read only property that returns the number of commands in the list.

## Syntax

DisabledCommandList.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 10
