# BoundaryPatchFeature.SetAffectedBodies Method

Parent Object: [BoundaryPatchFeature](../BoundaryPatchFeature/BoundaryPatchFeature.md)

## Description

Method that sets a collection of SurfaceBody objects affected by this feature.

## Syntax

BoundaryPatchFeature.**SetAffectedBodies**( ***Bodies*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Bodies | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | ObjectCollection of SurfaceBody objects. |

## Version

Introduced in version 2010
