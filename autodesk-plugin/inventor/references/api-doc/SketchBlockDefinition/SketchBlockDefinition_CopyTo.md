# SketchBlockDefinition.CopyTo Method

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Method that copies the definition to the target document.

## Syntax

SketchBlockDefinition.**CopyTo**( ***TargetDocument*** As [Document](../Document/Document.md), [***Reserved***] As Boolean ) As [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TargetDocument | [Document](../Document/Document.md) | Input Document object that specifies the document to copy the definition into. This can be the source document object if a copy of the definition is to be created in the same document. |
| Reserved | Boolean | Optional input Boolean reserved for future use. This value is currently ignored. |

## Version

Introduced in version 2010
