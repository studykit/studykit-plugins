# SketchBlockDefinition.ReferencedEntity Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the referenced sketch in the cases where this sketch was created as a result of a "derive" operation or copied over to the sheet metal flat pattern from the folded model.

## Syntax

SketchBlockDefinition.**ReferencedEntity**() As [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Property Value

This is a read only property whose value is a [PlanarSketch](../PlanarSketch/PlanarSketch.md).

## Version

Introduced in version 2010
