# SketchBlockDefinition.SketchFixedSplines Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that gets the collection object.

## Syntax

SketchBlockDefinition.**SketchFixedSplines**() As [SketchFixedSplines](../SketchFixedSplines/SketchFixedSplines.md)

## Property Value

This is a read only property whose value is a [SketchFixedSplines](../SketchFixedSplines/SketchFixedSplines.md).

## Version

Introduced in version 2010
