# SketchBlockDefinition.UpdateProfiles Method

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Method that updates all the profiles within the sketch.

## Syntax

SketchBlockDefinition.**UpdateProfiles**()

## Version

Introduced in version 2010
