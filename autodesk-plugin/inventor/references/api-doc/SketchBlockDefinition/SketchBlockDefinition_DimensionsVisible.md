# SketchBlockDefinition.DimensionsVisible Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Gets and sets whether the dimensions on the sketch are visible.

## Syntax

SketchBlockDefinition.**DimensionsVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
