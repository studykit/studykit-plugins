# SketchBlockDefinition.SketchToModelTransform Property

Parent Object: [SketchBlockDefinition](../SketchBlockDefinition/SketchBlockDefinition.md)

## Description

Property that returns the transformation from the 2D sketch coordinate space to model space.

## Syntax

SketchBlockDefinition.**SketchToModelTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2010
