# DrawingDimensions.OrdinateDimensions Property

Parent Object: [DrawingDimensions](../DrawingDimensions/DrawingDimensions.md)

## Description

Property that returns the OrdinateDimensions collection object.

## Syntax

DrawingDimensions.**OrdinateDimensions**() As [OrdinateDimensions](../OrdinateDimensions/OrdinateDimensions.md)

## Property Value

This is a read only property whose value is an [OrdinateDimensions](../OrdinateDimensions/OrdinateDimensions.md).
