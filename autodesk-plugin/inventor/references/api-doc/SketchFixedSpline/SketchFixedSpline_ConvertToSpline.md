# SketchFixedSpline.ConvertToSpline Method

Parent Object: [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Description

Method that converts the fixed spline to a SketchSpline object.

## Syntax

SketchFixedSpline.**ConvertToSpline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Version

Introduced in version 10
