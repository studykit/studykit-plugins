# OccurrencePatternElementProxy.Occurrences Property

Parent Object: [OccurrencePatternElementProxy](../OccurrencePatternElementProxy/OccurrencePatternElementProxy.md)

## Description

Property that returns the set of occurrences that were created for this particular instance.

## Syntax

OccurrencePatternElementProxy.**Occurrences**() As [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrencesEnumerator](../ComponentOccurrencesEnumerator/ComponentOccurrencesEnumerator.md).

## Version

Introduced in version 2010
