# SketchBlock.ChildBlocks Property

Parent Object: [SketchBlock](../SketchBlock/SketchBlock.md)

## Description

Property that returns an enumerator of SketchBlocks objects found directly under this sketch block.

## Syntax

SketchBlock.**ChildBlocks**() As [SketchBlocksEnumerator](../SketchBlocksEnumerator/SketchBlocksEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchBlocksEnumerator](../SketchBlocksEnumerator/SketchBlocksEnumerator.md).

## Version

Introduced in version 2010
