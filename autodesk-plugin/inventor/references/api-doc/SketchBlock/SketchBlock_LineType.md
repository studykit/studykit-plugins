# SketchBlock.LineType Property

Parent Object: [SketchBlock](../SketchBlock/SketchBlock.md)

## Description

Gets and sets the line type override for the sketch block.

## Syntax

SketchBlock.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 2010
