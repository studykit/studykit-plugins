# SketchBlock.Flexible Property

Parent Object: [SketchBlock](../SketchBlock/SketchBlock.md)

## Description

Gets and sets whether this sketch block is flexible.

## Syntax

SketchBlock.**Flexible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
