# FaceShellDefinitions.Count Property

Parent Object: [FaceShellDefinitions](../FaceShellDefinitions/FaceShellDefinitions.md)

## Description

Property that returns the number of items in this collection.

## Syntax

FaceShellDefinitions.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2011
