# HeadsUpDisplayOptions Object

## Description

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [DimensionInputEnabled](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_DimensionInputEnabled.md) | Gets and sets whether to enable dimension input where possible. |
| [DimensionInputInCartesianCoordinates](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_DimensionInputInCartesianCoordinates.md) | Gets and sets whether dimension input is in Cartesian or Polar coordinates. If set to False, input is in Polar coordinates. |
| [Enabled](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_Enabled.md) | Gets and sets whether heads-up display is enabled for sketches. |
| [Parent](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_Parent.md) | Property that returns the parent SketchOptions object. |
| [PointerInputEnabled](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_PointerInputEnabled.md) | Gets and sets whether pointer input is enabled for sketches. |
| [PointerInputInCartesianCoordinates](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_PointerInputInCartesianCoordinates.md) | Gets and sets whether pointer input is in Cartesian or Polar coordinates. If set to False, input is in Polar coordinates. |
| [Type](../HeadsUpDisplayOptions/HeadsUpDisplayOptions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
