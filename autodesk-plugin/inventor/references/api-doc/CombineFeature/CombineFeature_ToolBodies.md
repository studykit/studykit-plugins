# CombineFeature.ToolBodies Property

Parent Object: [CombineFeature](../CombineFeature/CombineFeature.md)

## Description

Property that returns the input tool bodies used to create the feature. If the KeepToolBodies property returns False, the end of part marker needs to be moved above this feature in order to query the bodies.

## Syntax

CombineFeature.**ToolBodies**() As [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md)

## Property Value

This is a read only property whose value is a [SurfaceBodies](../SurfaceBodies/SurfaceBodies.md).

## Version

Introduced in version 2010
