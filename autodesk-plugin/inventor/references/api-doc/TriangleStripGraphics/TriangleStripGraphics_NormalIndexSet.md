# TriangleStripGraphics.NormalIndexSet Property

Parent Object: [TriangleStripGraphics](../TriangleStripGraphics/TriangleStripGraphics.md)

## Description

Gets and sets the GraphicsIndexSet that defines the indices within the normal set to use for the triangles of the set.

## Syntax

TriangleStripGraphics.**NormalIndexSet**() As [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md)

## Property Value

This is a read/write property whose value is a [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md).

## Version

Introduced in version 5
