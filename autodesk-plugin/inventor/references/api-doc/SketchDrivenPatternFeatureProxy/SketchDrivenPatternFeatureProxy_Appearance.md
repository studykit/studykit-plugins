# SketchDrivenPatternFeatureProxy.Appearance Property

Parent Object: [SketchDrivenPatternFeatureProxy](../SketchDrivenPatternFeatureProxy/SketchDrivenPatternFeatureProxy.md)

## Description

Gets and sets the current appearance of the feature.

## Syntax

SketchDrivenPatternFeatureProxy.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2017
