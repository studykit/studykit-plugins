# DrawingCurveSegment.Visible Property

Parent Object: [DrawingCurveSegment](../DrawingCurveSegment/DrawingCurveSegment.md)

## Description

Gets and sets whether this object is graphically visible.

## Syntax

DrawingCurveSegment.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
