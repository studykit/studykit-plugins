# VerticalConstraintProxy.NativeObject Property

Parent Object: [VerticalConstraintProxy](../VerticalConstraintProxy/VerticalConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

VerticalConstraintProxy.**NativeObject**() As [VerticalConstraint](../VerticalConstraint/VerticalConstraint.md)

## Property Value

This is a read only property whose value is a [VerticalConstraint](../VerticalConstraint/VerticalConstraint.md).

## Version

Introduced in version 6
