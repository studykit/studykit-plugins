# WorkPointProxy.NativeObject Property

Parent Object: [WorkPointProxy](../WorkPointProxy/WorkPointProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

WorkPointProxy.**NativeObject**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 5
