# PlanarSketchProxy.Delete Method

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Method that deletes the sketch. This method is only valid for sketches that are not used by a feature.

## Syntax

PlanarSketchProxy.**Delete**()

## Version

Introduced in version 6
