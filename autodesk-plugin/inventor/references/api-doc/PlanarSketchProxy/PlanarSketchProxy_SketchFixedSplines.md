# PlanarSketchProxy.SketchFixedSplines Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that gets the collection object.

## Syntax

PlanarSketchProxy.**SketchFixedSplines**() As [SketchFixedSplines](../SketchFixedSplines/SketchFixedSplines.md)

## Property Value

This is a read only property whose value is a [SketchFixedSplines](../SketchFixedSplines/SketchFixedSplines.md).

## Version
