# PlanarSketchProxy.DeferUpdates Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Gets and Sets whether to defer the solving of the sketch or not.

## Syntax

PlanarSketchProxy.**DeferUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
