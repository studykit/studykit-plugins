# PlanarSketchProxy.Edit Method

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Method that causes the Sketch environment to be invoked with this sketch available for interactive edit.

## Syntax

PlanarSketchProxy.**Edit**()

## Version

Introduced in version 6
