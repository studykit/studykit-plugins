# PlanarSketchProxy.SketchPoints Property

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Property that returns the SketchPoints collection object.

## Syntax

PlanarSketchProxy.**SketchPoints**() As [SketchPoints](../SketchPoints/SketchPoints.md)

## Property Value

This is a read only property whose value is a [SketchPoints](../SketchPoints/SketchPoints.md).

## Version

Introduced in version 6
