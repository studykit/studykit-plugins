# PlanarSketchProxy.BreakLink Method

Parent Object: [PlanarSketchProxy](../PlanarSketchProxy/PlanarSketchProxy.md)

## Description

Method that breaks the link to the source sketch.

## Syntax

PlanarSketchProxy.**BreakLink**()

## Version
