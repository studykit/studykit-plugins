# ContourFlangeFeatureProxy.Definition Property

Parent Object: [ContourFlangeFeatureProxy](../ContourFlangeFeatureProxy/ContourFlangeFeatureProxy.md)

## Description

Read-write property that gets and sets the ContourFlangeDefinition object associated with this contour flange feature.

## Syntax

ContourFlangeFeatureProxy.**Definition**() As [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Property Value

This is a read/write property whose value is a [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md).

## Version

Introduced in version 2009
