# ContourFlangeFeatureProxy.HealthStatus Property

Parent Object: [ContourFlangeFeatureProxy](../ContourFlangeFeatureProxy/ContourFlangeFeatureProxy.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

ContourFlangeFeatureProxy.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 9
