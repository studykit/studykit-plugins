# DSLoad.Definition Property

Parent Object: [DSLoad](../DSLoad/DSLoad.md)

## Description

Gets and sets the load definition object associated with this load.

## Syntax

DSLoad.**Definition**() As [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Property Value

This is a read/write property whose value is a [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md).

## Version

Introduced in version 2013
