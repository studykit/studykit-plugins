# SketchEllipse3DProxy.MinorRadius Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Gets the MinorRadius of the ellipse.

## Syntax

SketchEllipse3DProxy.**MinorRadius**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
