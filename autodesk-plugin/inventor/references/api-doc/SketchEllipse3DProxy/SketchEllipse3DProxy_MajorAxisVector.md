# SketchEllipse3DProxy.MajorAxisVector Property

Parent Object: [SketchEllipse3DProxy](../SketchEllipse3DProxy/SketchEllipse3DProxy.md)

## Description

Gets the MajorAxisVector of the ellipse.

## Syntax

SketchEllipse3DProxy.**MajorAxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2010
