# BIMExchangeServer.GetBIMComponent Method

Parent Object: [BIMExchangeServer](../BIMExchangeServer/BIMExchangeServer.md)

## Description

Method that returns the BIMComponent associated with the provided Inventor part or assembly component definition.

## Syntax

BIMExchangeServer.**GetBIMComponent**( ***ComponentDefinition*** As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md) ) As [BIMComponent](../BIMComponent/BIMComponent.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ComponentDefinition | [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md) | Input PartComponentDefinition or AssemblyComponentDefinition object that specifies which Inventor part or assembly you want to get the associated BIMComponent for. |

## Version

Introduced in version 2011
