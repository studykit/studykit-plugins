# HoleTableCell.Parent Property

Parent Object: [HoleTableCell](../HoleTableCell/HoleTableCell.md)

## Description

Property that returns the parent HoleTable object.

## Syntax

HoleTableCell.**Parent**() As [HoleTable](../HoleTable/HoleTable.md)

## Property Value

This is a read only property whose value is a [HoleTable](../HoleTable/HoleTable.md).

## Version

Introduced in version 10
