# LoftedFlangeDefinition.Copy Method

Parent Object: [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Description

Method that creates a copy of this LoftedFlangeDefinition object. The new LoftedFlangeDefinition object is independent of any feature. It can edited and used as input to edit an existing feature or to create a new lofted flange feature.

## Syntax

LoftedFlangeDefinition.**Copy**() As [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Version

Introduced in version 2011
