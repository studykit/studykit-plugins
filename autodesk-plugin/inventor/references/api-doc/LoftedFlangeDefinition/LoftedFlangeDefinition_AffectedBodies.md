# LoftedFlangeDefinition.AffectedBodies Property

Parent Object: [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Description

Gets and sets the collection of bodies affected by this feature. If this property is not set for multi-body parts, the default solid body is used as the affected body.

## Syntax

LoftedFlangeDefinition.**AffectedBodies**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2016
