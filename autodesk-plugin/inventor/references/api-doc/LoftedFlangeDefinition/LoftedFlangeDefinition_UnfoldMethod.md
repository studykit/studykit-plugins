# LoftedFlangeDefinition.UnfoldMethod Property

Parent Object: [LoftedFlangeDefinition](../LoftedFlangeDefinition/LoftedFlangeDefinition.md)

## Description

Gets and set the UnfoldMethod object that defines how the bends associated with this lofted flange feature are unfolded.

## Syntax

LoftedFlangeDefinition.**UnfoldMethod**() As [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Property Value

This is a read/write property whose value is a [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md).

## Version

Introduced in version 2011
