# DerivedParameterTable.ReferencedDocumentDescriptor Property

Parent Object: [DerivedParameterTable](../DerivedParameterTable/DerivedParameterTable.md)

## Description

Property that returns the referenced Inventor document.

## Syntax

DerivedParameterTable.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 11
