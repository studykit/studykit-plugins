# ImportedRVTComponentProxy.NativeObject Property

Parent Object: [ImportedRVTComponentProxy](../ImportedRVTComponentProxy/ImportedRVTComponentProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

ImportedRVTComponentProxy.**NativeObject**() As [ImportedRVTComponent](../ImportedRVTComponent/ImportedRVTComponent.md)

## Property Value

This is a read only property whose value is an [ImportedRVTComponent](../ImportedRVTComponent/ImportedRVTComponent.md).

## Version

Introduced in version 2021
