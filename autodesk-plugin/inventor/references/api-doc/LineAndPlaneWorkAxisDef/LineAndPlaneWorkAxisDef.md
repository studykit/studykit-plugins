# LineAndPlaneWorkAxisDef Object

## Description

Object that allows you to get and set the information that specifies a work axis that is along a line defined by projecting a line onto a plane along the plane normal.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../LineAndPlaneWorkAxisDef/LineAndPlaneWorkAxisDef_GetData.md) | Method that gets all of the data defining a work axis that is along a line defined by projecting a line onto a plane along the plane normal. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LineAndPlaneWorkAxisDef/LineAndPlaneWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Line](../LineAndPlaneWorkAxisDef/LineAndPlaneWorkAxisDef_Line.md) | Property that returns the point of a work axis that is along a line defined by projecting a line onto a plane along the plane normal. This object can be a linear Edge, WorkAxis, or SketchLine object. |
| [Parent](../LineAndPlaneWorkAxisDef/LineAndPlaneWorkAxisDef_Parent.md) | Property returning the parent WorkAxis object. |
| [Plane](../LineAndPlaneWorkAxisDef/LineAndPlaneWorkAxisDef_Plane.md) | Property that indicates the plane of a work axis that is along a line defined by projecting a line onto a plane along the plane normal. This object can be a planar Face, WorkPlane, or Sketch object. |
| [Type](../LineAndPlaneWorkAxisDef/LineAndPlaneWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 4
