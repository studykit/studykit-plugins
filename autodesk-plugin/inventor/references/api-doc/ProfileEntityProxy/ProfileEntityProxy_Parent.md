# ProfileEntityProxy.Parent Property

Parent Object: [ProfileEntityProxy](../ProfileEntityProxy/ProfileEntityProxy.md)

## Description

Property that returns the parent ProfilePath of the entity.

## Syntax

ProfileEntityProxy.**Parent**() As [ProfilePath](../ProfilePath/ProfilePath.md)

## Property Value

This is a read only property whose value is a [ProfilePath](../ProfilePath/ProfilePath.md).

## Version

Introduced in version 6
