# EqualRadiusConstraintProxy.EntityOne Property

Parent Object: [EqualRadiusConstraintProxy](../EqualRadiusConstraintProxy/EqualRadiusConstraintProxy.md)

## Description

Property that returns the sketch circle or arc being constrained.

## Syntax

EqualRadiusConstraintProxy.**EntityOne**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
