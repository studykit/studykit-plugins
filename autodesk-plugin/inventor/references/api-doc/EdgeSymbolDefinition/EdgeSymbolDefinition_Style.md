# EdgeSymbolDefinition.Style Property

Parent Object: [EdgeSymbolDefinition](../EdgeSymbolDefinition/EdgeSymbolDefinition.md)

## Description

Read-write property that gets and sets the edge symbol style associated with this object.

## Syntax

EdgeSymbolDefinition.**Style**() As [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md)

## Property Value

This is a read/write property whose value is an [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md).

## Version

Introduced in version 2024
