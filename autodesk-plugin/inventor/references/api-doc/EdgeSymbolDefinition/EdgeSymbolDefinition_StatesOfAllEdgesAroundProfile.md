# EdgeSymbolDefinition.StatesOfAllEdgesAroundProfile Property

Parent Object: [EdgeSymbolDefinition](../EdgeSymbolDefinition/EdgeSymbolDefinition.md)

## Description

Read-write property that gets and sets whether to enable the states of all edges around the profile of a part.

## Syntax

EdgeSymbolDefinition.**StatesOfAllEdgesAroundProfile**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
