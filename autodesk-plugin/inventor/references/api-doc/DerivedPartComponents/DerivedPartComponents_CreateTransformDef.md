# DerivedPartComponents.CreateTransformDef Method

Parent Object: [DerivedPartComponents](../DerivedPartComponents/DerivedPartComponents.md)

## Description

Method that creates a new DerivedPartTransformDef object.

## Syntax

DerivedPartComponents.**CreateTransformDef**( ***FullFileName*** As String ) As [DerivedPartTransformDef](../DerivedPartTransformDef/DerivedPartTransformDef.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FullFileName | String | Input string that specifies the name of the file to create the definition for. |

## Version

Introduced in version 6
