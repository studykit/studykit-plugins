# ModelLeaderNoteProxy.TopToleranceFeature Property

Parent Object: [ModelLeaderNoteProxy](../ModelLeaderNoteProxy/ModelLeaderNoteProxy.md)

## Description

Returns the top ModelToleranceFeature object.

## Syntax

ModelLeaderNoteProxy.**TopToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2019
