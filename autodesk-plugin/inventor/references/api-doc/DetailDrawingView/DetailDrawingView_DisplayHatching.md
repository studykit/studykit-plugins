# DetailDrawingView.DisplayHatching Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets and sets whether to display hatches for section and slice views.

## Syntax

DetailDrawingView.**DisplayHatching**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
