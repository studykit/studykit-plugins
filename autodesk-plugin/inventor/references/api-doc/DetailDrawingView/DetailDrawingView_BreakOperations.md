# DetailDrawingView.BreakOperations Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that returns the BreakOperations object containing information about all the break operations applied to this view as well as methods to add breaks. Both local as well as inherited break operations are returned in the collection.

## Syntax

DetailDrawingView.**BreakOperations**() As [BreakOperations](../BreakOperations/BreakOperations.md)

## Property Value

This is a read only property whose value is a [BreakOperations](../BreakOperations/BreakOperations.md).

## Version

Introduced in version 2010
