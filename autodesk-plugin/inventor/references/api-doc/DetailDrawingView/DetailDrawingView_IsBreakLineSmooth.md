# DetailDrawingView.IsBreakLineSmooth Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that gets and sets whether the cut shape of the fence is smooth or jogged.

## Syntax

DetailDrawingView.**IsBreakLineSmooth**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
