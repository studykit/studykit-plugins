# DetailDrawingView.BoundaryLayer Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that gets and sets the layer associated with the boundary geometry.

## Syntax

DetailDrawingView.**BoundaryLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2008
