# DetailDrawingView.ShowTrails Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Gets and set whether to show the trails or not for a drawing view of a presentation.

## Syntax

DetailDrawingView.**ShowTrails**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2026
