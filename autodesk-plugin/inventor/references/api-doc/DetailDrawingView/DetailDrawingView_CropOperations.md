# DetailDrawingView.CropOperations Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Read-only property that returns the CropOperations object containing information about all the crop operations applied to this view as well as methods to add crops.

## Syntax

DetailDrawingView.**CropOperations**() As [CropOperations](../CropOperations/CropOperations.md)

## Property Value

This is a read only property whose value is a [CropOperations](../CropOperations/CropOperations.md).

## Version

Introduced in version 2025.1
