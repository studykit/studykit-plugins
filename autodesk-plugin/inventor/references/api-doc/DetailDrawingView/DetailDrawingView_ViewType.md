# DetailDrawingView.ViewType Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property returning an DrawingViewTypeEnum constant indicating the type of view.

## Syntax

DetailDrawingView.**ViewType**() As [DrawingViewTypeEnum](../DrawingViewTypeEnum.md)

## Property Value

This is a read only property whose value is a [DrawingViewTypeEnum](../DrawingViewTypeEnum.md).

## Version

Introduced in version 11
