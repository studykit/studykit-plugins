# DetailDrawingView.HasOriginIndicator Property

Parent Object: [DetailDrawingView](../DetailDrawingView/DetailDrawingView.md)

## Description

Property that specifies whether the origin indicator for ordinate dimensions and hole tables has been created.

## Syntax

DetailDrawingView.**HasOriginIndicator**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
