# iAssemblyMember.ReferencedDocumentDescriptor Property

Parent Object: [iAssemblyMember](../iAssemblyMember/iAssemblyMember.md)

## Description

Property that returns the descriptor of the iAssembly factory document.

## Syntax

iAssemblyMember.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 11
