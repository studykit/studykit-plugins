# TransitionSymbol.Leader Property

Parent Object: [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md)

## Description

Returns the Leader object.

## Syntax

TransitionSymbol.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 2025.1
