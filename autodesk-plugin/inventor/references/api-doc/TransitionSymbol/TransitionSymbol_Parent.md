# TransitionSymbol.Parent Property

Parent Object: [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

TransitionSymbol.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 2025.1
