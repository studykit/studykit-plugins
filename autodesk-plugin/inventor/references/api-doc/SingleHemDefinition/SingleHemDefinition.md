# SingleHemDefinition Object

## Description

The SingleHemDefinition object defines all of the inputs that are unique to a single style of hem feature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Gap](../SingleHemDefinition/SingleHemDefinition_Gap.md) | Property that returns the parameter controlling the gap of a hem feature defined by a single bend. |
| [Length](../SingleHemDefinition/SingleHemDefinition_Length.md) | Property that returns the parameter controlling the length of a hem feature defined by a single bend. |
| [Parent](../SingleHemDefinition/SingleHemDefinition_Parent.md) | Property that returns the parent hem definition feature. |

## Version

Introduced in version 2009
