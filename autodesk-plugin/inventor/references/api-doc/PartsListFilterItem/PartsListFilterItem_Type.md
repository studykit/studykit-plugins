# PartsListFilterItem.Type Property

Parent Object: [PartsListFilterItem](../PartsListFilterItem/PartsListFilterItem.md)

## Description

Gets the constant that indicates the type of this object.

## Syntax

PartsListFilterItem.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2024.1
