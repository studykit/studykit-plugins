# WeldSymbol.WeldSymbolType Property

Parent Object: [WeldSymbol](../WeldSymbol/WeldSymbol.md)

## Description

Gets and sets the weld symbol type. This can be a value of WeldSymbolTypeEnum or BackingSymbolTypeEnum.

## Syntax

WeldSymbol.**WeldSymbolType**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2024
