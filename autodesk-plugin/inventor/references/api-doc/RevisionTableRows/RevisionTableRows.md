# RevisionTableRows Object

## Description

This collection object represents row objects in the revision table.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../RevisionTableRows/RevisionTableRows_Add.md) | Method that creates a new RevisionTableRow. The new created RevisionTableRow is returned. |
| [AddCustom](../RevisionTableRows/RevisionTableRows_AddCustom.md) | Creates a new custom RevisionTableRow. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevisionTableRows/RevisionTableRows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../RevisionTableRows/RevisionTableRows_Count.md) | Property that returns the number of items in the collection. |
| [Item](../RevisionTableRows/RevisionTableRows_Item.md) | Returns the specified row object from the collection. |
| [Type](../RevisionTableRows/RevisionTableRows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[RevisionTable.RevisionTableRows](../RevisionTable/RevisionTable_RevisionTableRows.md)

## Version

Introduced in version 10
