# RevisionTableRows.Add Method

Parent Object: [RevisionTableRows](../RevisionTableRows/RevisionTableRows.md)

## Description

Method that creates a new RevisionTableRow. The new created RevisionTableRow is returned.

## Syntax

RevisionTableRows.**Add**() As [RevisionTableRow](../RevisionTableRow/RevisionTableRow.md)

## Version

Introduced in version 10
