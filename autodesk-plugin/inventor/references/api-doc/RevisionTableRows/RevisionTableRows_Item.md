# RevisionTableRows.Item Property

Parent Object: [RevisionTableRows](../RevisionTableRows/RevisionTableRows.md)

## Description

Returns the specified row object from the collection.

## Syntax

RevisionTableRows.**Item**( ***Index*** As Long ) As [RevisionTableRow](../RevisionTableRow/RevisionTableRow.md)

## Property Value

This is a read only property whose value is a [RevisionTableRow](../RevisionTableRow/RevisionTableRow.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the RevisionTableRow to return. This is a numeric value indicating the index of the item in the collection. If an out of range index is input, an error occurs. |
