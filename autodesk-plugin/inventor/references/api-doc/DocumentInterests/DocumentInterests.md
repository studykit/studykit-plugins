# DocumentInterests Object

## Description

Description of the collection object. Below are some common methods and properties.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../DocumentInterests/DocumentInterests_Add.md) | Method that creates a new DocumentInterest. |
| [HasInterest](../DocumentInterests/DocumentInterests_HasInterest.md) | Method that returns whether an AddIn with the input ClientId/Name has expressed interest in this document. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DocumentInterests/DocumentInterests_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DocumentInterests/DocumentInterests_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DocumentInterests/DocumentInterests_Item.md) | Returns the specified DocumentInterest object from the collection |
| [Type](../DocumentInterests/DocumentInterests_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
