# ModelGeneralNoteDefinition.ScreenQuadrant Property

Parent Object: [ModelGeneralNoteDefinition](../ModelGeneralNoteDefinition/ModelGeneralNoteDefinition.md)

## Description

Gets or sets the location where the general note is displayed on screen. Setting this property is only applicable when the OnScreen returns True.

## Syntax

ModelGeneralNoteDefinition.**ScreenQuadrant**() As [ScreenQuadrantEnum](../ScreenQuadrantEnum.md)

## Property Value

This is a read/write property whose value is a [ScreenQuadrantEnum](../ScreenQuadrantEnum.md).

## Version

Introduced in version 2018
