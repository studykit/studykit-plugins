# TableParameter.DrivenBy Property

Parent Object: [TableParameter](../TableParameter/TableParameter.md)

## Description

Method that returns the collection objects that the parameter is dependent on.

## Syntax

TableParameter.**DrivenBy**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
