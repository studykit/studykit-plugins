# ModelingSettings.TappedHoleDiameter Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets the tapped hole diameter.

## Syntax

ModelingSettings.**TappedHoleDiameter**() As [ModelDiameterFromThreadEnum](../ModelDiameterFromThreadEnum.md)

## Property Value

This is a read/write property whose value is a [ModelDiameterFromThreadEnum](../ModelDiameterFromThreadEnum.md).

## Version

Introduced in version 9
