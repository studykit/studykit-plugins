# ModelingSettings.AdvancedFeatureValidation Property

Parent Object: [ModelingSettings](../ModelingSettings/ModelingSettings.md)

## Description

Gets and sets the advanced feature validation setting.

## Syntax

ModelingSettings.**AdvancedFeatureValidation**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
