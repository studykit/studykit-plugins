# CoincidentConstraint Object

Derived from: [GeometricConstraint](../GeometricConstraint/GeometricConstraint.md) Object

## Description

The CoincidentConstraint object represents a coincident constraint within a sketch.

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../CoincidentConstraint/CoincidentConstraint_Delete.md) | Method that deletes the constraint. |
