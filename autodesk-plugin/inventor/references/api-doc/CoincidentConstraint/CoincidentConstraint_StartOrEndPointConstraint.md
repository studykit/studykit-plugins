# CoincidentConstraint.StartOrEndPointConstraint Property

Parent Object: [CoincidentConstraint](../CoincidentConstraint/CoincidentConstraint.md)

## Description

Indicates whether the constraint is between the curve and its endpoint.

## Syntax

CoincidentConstraint.**StartOrEndPointConstraint**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
