# DerivedPartDefinition.LinkFaceColorFromSource Property

Parent Object: [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Description

Read-write Boolean property that specifies whether to link face color from source component or not.

## Syntax

DerivedPartDefinition.**LinkFaceColorFromSource**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
