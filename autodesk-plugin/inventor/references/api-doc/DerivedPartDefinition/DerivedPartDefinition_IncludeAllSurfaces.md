# DerivedPartDefinition.IncludeAllSurfaces Property

Parent Object: [DerivedPartDefinition](../DerivedPartDefinition/DerivedPartDefinition.md)

## Description

Property that defines whether all surfaces are included in the derived part. Valid input for this property is kDerivedIncludeAll, kDerivedExcludeAll, and kDerivedIndividualDefined.

## Syntax

DerivedPartDefinition.**IncludeAllSurfaces**() As [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md).

## Version

Introduced in version 5.3
