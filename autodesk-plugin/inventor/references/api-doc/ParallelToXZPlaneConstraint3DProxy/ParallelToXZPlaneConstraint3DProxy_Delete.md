# ParallelToXZPlaneConstraint3DProxy.Delete Method

Parent Object: [ParallelToXZPlaneConstraint3DProxy](../ParallelToXZPlaneConstraint3DProxy/ParallelToXZPlaneConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ParallelToXZPlaneConstraint3DProxy.**Delete**()

## Version

Introduced in version 2017
