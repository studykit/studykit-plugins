# ImportedDWGComponent.ModelSpaceDefinition Property

Parent Object: [ImportedDWGComponent](../ImportedDWGComponent/ImportedDWGComponent.md)

## Description

Read-only property that returns the DWGModelSpaceDefinition object.

## Syntax

ImportedDWGComponent.**ModelSpaceDefinition**() As [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md)

## Property Value

This is a read only property whose value is a [DWGBlockDefinition](../DWGBlockDefinition/DWGBlockDefinition.md).
