# ModelFeatureControlFrameRow.GeometricCharacteristic Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the geometric characteristic symbol for the row.

## Syntax

ModelFeatureControlFrameRow.**GeometricCharacteristic**() As [GeometricCharacteristicEnum](../GeometricCharacteristicEnum.md)

## Property Value

This is a read/write property whose value is a [GeometricCharacteristicEnum](../GeometricCharacteristicEnum.md).

## Version

Introduced in version 2018
