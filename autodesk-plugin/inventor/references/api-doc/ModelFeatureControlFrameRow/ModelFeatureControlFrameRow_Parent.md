# ModelFeatureControlFrameRow.Parent Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelFeatureControlFrameRow.**Parent**() As [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Property Value

This is a read only property whose value is a [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md).

## Version

Introduced in version 2018
