# ModelFeatureControlFrameRow.TolerancePrecision Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the tolerance precision associated with this geometric characteristic. The value is a short integer.

## Syntax

ModelFeatureControlFrameRow.**TolerancePrecision**() As Integer

## Property Value

This is a read/write property whose value is an Integer.

## Version

Introduced in version 2018
