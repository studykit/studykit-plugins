# ModelFeatureControlFrameRow.PrimaryDatumModifier Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the the material condition symbol to append to the primary datum.

## Syntax

ModelFeatureControlFrameRow.**PrimaryDatumModifier**() As [GDTSymbolEnum](../GDTSymbolEnum.md)

## Property Value

This is a read/write property whose value is a [GDTSymbolEnum](../GDTSymbolEnum.md).

## Version

Introduced in version 2018
