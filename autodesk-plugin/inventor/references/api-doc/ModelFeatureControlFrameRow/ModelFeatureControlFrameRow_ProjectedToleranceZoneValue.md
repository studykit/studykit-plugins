# ModelFeatureControlFrameRow.ProjectedToleranceZoneValue Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the value for the projected tolerance zone modifier.

## Syntax

ModelFeatureControlFrameRow.**ProjectedToleranceZoneValue**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
