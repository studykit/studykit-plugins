# ModelFeatureControlFrameRow.GeneralProfileOfSurfaceModifier Property

Parent Object: [ModelFeatureControlFrameRow](../ModelFeatureControlFrameRow/ModelFeatureControlFrameRow.md)

## Description

Gets and sets the flag whether this FCF represents general surface profile tolerance.

## Syntax

ModelFeatureControlFrameRow.**GeneralProfileOfSurfaceModifier**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
