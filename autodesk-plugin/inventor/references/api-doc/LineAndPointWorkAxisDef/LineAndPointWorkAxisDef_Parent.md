# LineAndPointWorkAxisDef.Parent Property

Parent Object: [LineAndPointWorkAxisDef](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef.md)

## Description

Property that returns the parent WorkAxis object.

## Syntax

LineAndPointWorkAxisDef.**Parent**() As [WorkAxis](../WorkAxis/WorkAxis.md)

## Property Value

This is a read only property whose value is a [WorkAxis](../WorkAxis/WorkAxis.md).

## Version

Introduced in version 11
