# LineAndPointWorkAxisDef Object

## Description

Object that allows you to get and set the information that specifies a work axis that is along a line and passes through a point.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef_GetData.md) | Method that gets all of the data defining a work axis that is along a line and passes through a point. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Line](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef_Line.md) | Property that returns the line used to define a work axis that is along a line and passes through a point. This object can be a linear Edge, WorkAxis, SketchLine, or SketchLine3D object. |
| [Parent](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef_Parent.md) | Property that returns the parent WorkAxis object. |
| [Point](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef_Point.md) | Property that returns the point used to define a work axis that is along a line and passes through a point.. This object can be a vertex, WorkPoint, SketchPoint, or SketchPoint3D object. |
| [Type](../LineAndPointWorkAxisDef/LineAndPointWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 11
