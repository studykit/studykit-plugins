# Ribbon.QuickAccessControls Property

Parent Object: [Ribbon](../Ribbon/Ribbon.md)

## Description

Property that returns a CommandControls collection containing the controls in the Quick Access Toolbar (QAT).

## Syntax

Ribbon.**QuickAccessControls**() As [CommandControls](../CommandControls/CommandControls.md)

## Property Value

This is a read only property whose value is a [CommandControls](../CommandControls/CommandControls.md).
