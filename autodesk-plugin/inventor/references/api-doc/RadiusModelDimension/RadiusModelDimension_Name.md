# RadiusModelDimension.Name Property

Parent Object: [RadiusModelDimension](../RadiusModelDimension/RadiusModelDimension.md)

## Description

Read-write property that gets and sets the name of the annotation.

## Syntax

RadiusModelDimension.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
