# ErrorManager.IsMessageSectionActive Property

Parent Object: [ErrorManager](../ErrorManager/ErrorManager.md)

## Description

Property that returns whether there is any message section active.

## Syntax

ErrorManager.**IsMessageSectionActive**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
