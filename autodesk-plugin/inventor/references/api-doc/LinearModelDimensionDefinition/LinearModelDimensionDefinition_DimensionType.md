# LinearModelDimensionDefinition.DimensionType Property

Parent Object: [LinearModelDimensionDefinition](../LinearModelDimensionDefinition/LinearModelDimensionDefinition.md)

## Description

Read-only property that returns the dimension type. Possible values are kAlignedDimensionType, kHorizontalDimensionType, kVerticalDimensionType, kArcLengthDimensionType, kSymmetricDimensionType and kDiametricDimensionType.

## Syntax

LinearModelDimensionDefinition.**DimensionType**() As [DimensionTypeEnum](../DimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [DimensionTypeEnum](../DimensionTypeEnum.md).

## Version

Introduced in version 2018
