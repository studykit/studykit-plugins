# LinearModelDimensionDefinition.IsExtensionLineOneVisible Property

Parent Object: [LinearModelDimensionDefinition](../LinearModelDimensionDefinition/LinearModelDimensionDefinition.md)

## Description

Read-write property that gets and sets whether the first extension line is visible.

## Syntax

LinearModelDimensionDefinition.**IsExtensionLineOneVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
