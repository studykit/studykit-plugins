# ChamferDefinition.ChamferedEdges Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Gets the collection of edges that are chamfered by this feature.

## Syntax

ChamferDefinition.**ChamferedEdges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2018
