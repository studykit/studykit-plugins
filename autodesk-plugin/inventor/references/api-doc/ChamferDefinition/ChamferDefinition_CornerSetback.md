# ChamferDefinition.CornerSetback Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Gets and sets whether a setback is to be applied to the chamfer feature.

## Syntax

ChamferDefinition.**CornerSetback**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
