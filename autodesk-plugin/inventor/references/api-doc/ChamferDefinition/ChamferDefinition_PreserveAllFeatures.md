# ChamferDefinition.PreserveAllFeatures Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Gets whether or not to preserve all features.

## Syntax

ChamferDefinition.**PreserveAllFeatures**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
