# ChamferDefinition.Distance Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Returns the parameter that controls the distance used for the chamfer feature. This is applicable when the DefinitionType is kDistance.

## Syntax

ChamferDefinition.**Distance**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2018
