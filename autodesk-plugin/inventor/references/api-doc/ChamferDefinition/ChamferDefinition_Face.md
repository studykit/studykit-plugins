# ChamferDefinition.Face Property

Parent Object: [ChamferDefinition](../ChamferDefinition/ChamferDefinition.md)

## Description

Returns the face used to define the chamfer feature created using a distance and an angle or two distances.

## Syntax

ChamferDefinition.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2018
