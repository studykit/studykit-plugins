# DirectEditFeatureProxy.Suppressed Property

Parent Object: [DirectEditFeatureProxy](../DirectEditFeatureProxy/DirectEditFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

DirectEditFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
