# SolidSweepDefinition.Orientation Property

Parent Object: [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md)

## Description

Read-write property that gets and sets the orientation of sweep.

## Syntax

SolidSweepDefinition.**Orientation**() As [SweepProfileOrientationEnum](../SweepProfileOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [SweepProfileOrientationEnum](../SweepProfileOrientationEnum.md).

## Version

Introduced in version 2020
