# SolidSweepDefinition.ToolBody Property

Parent Object: [SolidSweepDefinition](../SolidSweepDefinition/SolidSweepDefinition.md)

## Description

Read-write property that gets and sets the SurfaceBody object used to define the toolbody of the sweep.

## Syntax

SolidSweepDefinition.**ToolBody**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read/write property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2019
