# RibDefinition.DraftProfileEnds Property

Parent Object: [RibDefinition](../RibDefinition/RibDefinition.md)

## Description

Read-write property that gets and sets the draft profile ends value.

## Syntax

RibDefinition.**DraftProfileEnds**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
