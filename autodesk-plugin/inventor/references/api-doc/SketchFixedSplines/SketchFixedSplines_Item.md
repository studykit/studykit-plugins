# SketchFixedSplines.Item Property

Parent Object: [SketchFixedSplines](../SketchFixedSplines/SketchFixedSplines.md)

## Description

Method that returns the specified SketchFixedSpline object from the collection.

## Syntax

SketchFixedSplines.**Item**( ***Index*** As Long ) As [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Property Value

This is a read only property whose value is a [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 9
