# SketchBlockDefinitionsEnumerator Object

## Description

The SketchBlockDefinitionsEnumerator allows iteration of available sketchBlockDefinition objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DerivedPartComponent.SketchBlockDefinitions](../DerivedPartComponent/DerivedPartComponent_SketchBlockDefinitions.md), [DerivedPartComponentProxy.SketchBlockDefinitions](../DerivedPartComponentProxy/DerivedPartComponentProxy_SketchBlockDefinitions.md)

## Version
