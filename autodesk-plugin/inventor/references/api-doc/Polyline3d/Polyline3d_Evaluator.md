# Polyline3d.Evaluator Property

Parent Object: [Polyline3d](../Polyline3d/Polyline3d.md)

## Description

Property that returns an evaluator object for the curve. The evaluator supports general curve evaluation functions.

## Syntax

Polyline3d.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 2009
