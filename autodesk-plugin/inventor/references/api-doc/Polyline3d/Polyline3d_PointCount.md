# Polyline3d.PointCount Property

Parent Object: [Polyline3d](../Polyline3d/Polyline3d.md)

## Description

Property that returns the number of points defining the polyline.

## Syntax

Polyline3d.**PointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2009
