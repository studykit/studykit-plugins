# Polyline3d.PointAtIndex Property

Parent Object: [Polyline3d](../Polyline3d/Polyline3d.md)

## Description

Gets or sets the x, y, z position of a specified point.

## Syntax

Polyline3d.**PointAtIndex**( ***Index*** As Long ) As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).
