# AddInLicenseStatusEnum Enumerator

## Description

License Status of AddIn.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAddInLicenseStatusAvailable | 76290 | The AddIn has attempted to license itself and a valid license is available for the AddIn. |
| kAddInLicenseStatusUnavailable | 76291 | The AddIn has attempted to license itself however a valid license is not available for the AddIn. |
| kAddInLicenseStatusUnlicensed | 76289 | The AddIn has not attempted to license itself therefor it is not possible to tell whether it is Licensed or Unlicensed. |

## Version

Introduced in version 2009
