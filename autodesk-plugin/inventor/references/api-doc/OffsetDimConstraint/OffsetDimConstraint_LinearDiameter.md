# OffsetDimConstraint.LinearDiameter Property

Parent Object: [OffsetDimConstraint](../OffsetDimConstraint/OffsetDimConstraint.md)

## Description

Property that returns whether the dimension is a linear diameter style of dimension.

## Syntax

OffsetDimConstraint.**LinearDiameter**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5
