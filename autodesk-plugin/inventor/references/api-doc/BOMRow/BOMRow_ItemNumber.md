# BOMRow.ItemNumber Property

Parent Object: [BOMRow](../BOMRow/BOMRow.md)

## Description

Gets and sets the item number of the BOM item.

## Syntax

BOMRow.**ItemNumber**() As String

## Property Value

This is a read/write property whose value is a String.
