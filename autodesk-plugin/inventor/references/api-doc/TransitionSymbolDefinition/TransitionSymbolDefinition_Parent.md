# TransitionSymbolDefinition.Parent Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

TransitionSymbolDefinition.**Parent**() As [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md)

## Property Value

This is a read only property whose value is a [TransitionSymbol](../TransitionSymbol/TransitionSymbol.md).

## Version

Introduced in version 2025.1
