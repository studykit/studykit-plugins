# TransitionSymbolDefinition.SeveralTolerancedFeaturesIndication Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets and sets the several tolerance features indication value.

## Syntax

TransitionSymbolDefinition.**SeveralTolerancedFeaturesIndication**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2025.1
