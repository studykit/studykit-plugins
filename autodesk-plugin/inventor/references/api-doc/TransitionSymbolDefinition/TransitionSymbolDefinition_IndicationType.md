# TransitionSymbolDefinition.IndicationType Property

Parent Object: [TransitionSymbolDefinition](../TransitionSymbolDefinition/TransitionSymbolDefinition.md)

## Description

Gets and sets the indication type of the transition symbol.

## Syntax

TransitionSymbolDefinition.**IndicationType**() As [TransitionSymbolIndicationTypeEnum](../TransitionSymbolIndicationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [TransitionSymbolIndicationTypeEnum](../TransitionSymbolIndicationTypeEnum.md).

## Version

Introduced in version 2025.1
