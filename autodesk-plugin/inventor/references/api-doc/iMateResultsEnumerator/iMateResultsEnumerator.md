# iMateResultsEnumerator Object

## Description

Object that represents an iMateResultsEnumerator. Presents a snapshot of the set of objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../iMateResultsEnumerator/iMateResultsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../iMateResultsEnumerator/iMateResultsEnumerator_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../iMateResultsEnumerator/iMateResultsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From
