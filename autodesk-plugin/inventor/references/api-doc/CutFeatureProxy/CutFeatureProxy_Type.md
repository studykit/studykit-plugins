# CutFeatureProxy.Type Property

Parent Object: [CutFeatureProxy](../CutFeatureProxy/CutFeatureProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CutFeatureProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
