# ColorScheme.JoinPreviewColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Property that returns the preview color used when using the 'Join' option during feature creation.

## Syntax

ColorScheme.**JoinPreviewColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
