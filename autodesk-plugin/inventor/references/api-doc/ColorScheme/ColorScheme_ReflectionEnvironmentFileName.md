# ColorScheme.ReflectionEnvironmentFileName Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Read-write property that gets and sets the image full filename for the reflection map.

## Syntax

ColorScheme.**ReflectionEnvironmentFileName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2021
