# ColorScheme.TertiarySelectColor Property

Parent Object: [ColorScheme](../ColorScheme/ColorScheme.md)

## Description

Gets the TertiarySelectColor Color setting.

## Syntax

ColorScheme.**TertiarySelectColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2009
