# CoincidentConstraintProxy Object

Derived from: [CoincidentConstraint](../CoincidentConstraint/CoincidentConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../CoincidentConstraintProxy/CoincidentConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../CoincidentConstraintProxy/CoincidentConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CoincidentConstraintProxy/CoincidentConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../CoincidentConstraintProxy/CoincidentConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../CoincidentConstraintProxy/CoincidentConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../CoincidentConstraintProxy/CoincidentConstraintProxy_Deletable.md) | Indicates whether this object is deletable. |
| [EntityOne](../CoincidentConstraintProxy/CoincidentConstraintProxy_EntityOne.md) | Property that returns the sketch entity being constrained. |
| [EntityTwo](../CoincidentConstraintProxy/CoincidentConstraintProxy_EntityTwo.md) | Property that returns the sketch entity being constrained. |
| [NativeObject](../CoincidentConstraintProxy/CoincidentConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../CoincidentConstraintProxy/CoincidentConstraintProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [StartOrEndPointConstraint](../CoincidentConstraintProxy/CoincidentConstraintProxy_StartOrEndPointConstraint.md) | Indicates whether the constraint is between the curve and its endpoint. |
| [Type](../CoincidentConstraintProxy/CoincidentConstraintProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
