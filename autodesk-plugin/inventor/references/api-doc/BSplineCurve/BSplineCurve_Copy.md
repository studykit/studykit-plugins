# BSplineCurve.Copy Method

Parent Object: [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Description

Creates a copy of this BSplineCurve object. The result is entirely independent and can be edited without affecting the original BSplineCurve object.

## Syntax

BSplineCurve.**Copy**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Version

Introduced in version 2013
