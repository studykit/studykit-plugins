# FaceFeatureDefinition.Operation Property

Parent Object: [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md)

## Description

Gets and sets the type of operation used to add the feature to the model. Valid inputs are kNewBodyOperation, kJoinOperation.

## Syntax

FaceFeatureDefinition.**Operation**() As [PartFeatureOperationEnum](../PartFeatureOperationEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureOperationEnum](../PartFeatureOperationEnum.md).

## Version

Introduced in version 2016
