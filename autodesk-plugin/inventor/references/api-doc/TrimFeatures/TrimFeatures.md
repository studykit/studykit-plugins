# TrimFeatures Object

## Description

The TrimFeatures collection provides access to all of the TrimFeature objects in a component definition.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TrimFeatures/TrimFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../TrimFeatures/TrimFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../TrimFeatures/TrimFeatures_Item.md) | Property that returns an item from the collection. You can provide either the index of the item in the collection, where the first item is index 1, or you can provide the name of the object. |
| [Type](../TrimFeatures/TrimFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.TrimFeatures](../PartFeatures/PartFeatures_TrimFeatures.md), [SheetMetalFeatures.TrimFeatures](../SheetMetalFeatures/SheetMetalFeatures_TrimFeatures.md)

## Version

Introduced in version 11
