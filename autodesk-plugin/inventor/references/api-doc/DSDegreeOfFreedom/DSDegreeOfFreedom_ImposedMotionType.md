# DSDegreeOfFreedom.ImposedMotionType Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets and sets the type of imposed motion defined on a degree of freedom.

## Syntax

DSDegreeOfFreedom.**ImposedMotionType**() As [DSDOFImposedMotionTypeEnum](../DSDOFImposedMotionTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DSDOFImposedMotionTypeEnum](../DSDOFImposedMotionTypeEnum.md).

## Version

Introduced in version 2013
