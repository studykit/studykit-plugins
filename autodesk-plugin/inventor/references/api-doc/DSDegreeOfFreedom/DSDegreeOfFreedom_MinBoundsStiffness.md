# DSDegreeOfFreedom.MinBoundsStiffness Property

Parent Object: [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Description

Gets the minimum stiffness value.

## Syntax

DSDegreeOfFreedom.**MinBoundsStiffness**() As [DSValue](../DSValue/DSValue.md)

## Property Value

This is a read only property whose value is a [DSValue](../DSValue/DSValue.md).

## Version

Introduced in version 2013
