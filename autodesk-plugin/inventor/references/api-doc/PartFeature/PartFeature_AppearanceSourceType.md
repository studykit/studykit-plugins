# PartFeature.AppearanceSourceType Property

Parent Object: [PartFeature](../PartFeature/PartFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

PartFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).
