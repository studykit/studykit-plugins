# PartFeature.FeatureDimensions Property

Parent Object: [PartFeature](../PartFeature/PartFeature.md)

## Description

Property that returns the FeatureDimensions collection object.

## Syntax

PartFeature.**FeatureDimensions**() As [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md)

## Property Value

This is a read only property whose value is a [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md).
