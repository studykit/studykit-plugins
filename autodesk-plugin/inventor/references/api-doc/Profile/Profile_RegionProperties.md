# Profile.RegionProperties Property

Parent Object: [Profile](../Profile/Profile.md)

## Description

Returns the RegionProperties object.

## Syntax

Profile.**RegionProperties**() As [RegionProperties](../RegionProperties/RegionProperties.md)

## Property Value

This is a read only property whose value is a [RegionProperties](../RegionProperties/RegionProperties.md).
