# ConcentricConstraintProxy.EntityTwo Property

Parent Object: [ConcentricConstraintProxy](../ConcentricConstraintProxy/ConcentricConstraintProxy.md)

## Description

Gets the sketch entity being constrained.

## Syntax

ConcentricConstraintProxy.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
