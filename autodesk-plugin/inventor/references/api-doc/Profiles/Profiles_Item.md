# Profiles.Item Property

Parent Object: [Profiles](../Profiles/Profiles.md)

## Description

Returns the specified Profile object from the collection.

## Syntax

Profiles.**Item**( ***Index*** As Long ) As [Profile](../Profile/Profile.md)

## Property Value

This is a read only property whose value is a [Profile](../Profile/Profile.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the index of the Profile to return. |

## Version

Introduced in version 5
