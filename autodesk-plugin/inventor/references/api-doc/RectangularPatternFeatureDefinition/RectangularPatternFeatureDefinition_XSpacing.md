# RectangularPatternFeatureDefinition.XSpacing Property

Parent Object: [RectangularPatternFeatureDefinition](../RectangularPatternFeatureDefinition/RectangularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the spacing between instances in the X direction.

## Syntax

RectangularPatternFeatureDefinition.**XSpacing**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2017
