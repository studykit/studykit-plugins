# Centerline.PatternCenter Property

Parent Object: [Centerline](../Centerline/Centerline.md)

## Description

Property that returns the geometry the center that defines the center of the centerline pattern. This property is only valid for kCenteredPatternCenterline type centerlines and will return Nothing in all other cases.

## Syntax

Centerline.**PatternCenter**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read only property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2010
