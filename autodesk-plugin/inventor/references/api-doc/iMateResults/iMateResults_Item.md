# iMateResults.Item Property

Parent Object: [iMateResults](../iMateResults/iMateResults.md)

## Description

Returns the specified iMateResult object from the collection.

## Syntax

iMateResults.**Item**( ***Index*** As Variant ) As [iMateResult](../iMateResult/iMateResult.md)

## Property Value

This is a read only property whose value is an [iMateResult](../iMateResult/iMateResult.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the name of the iMateResult object. If an out-of-range index or a name of a non-existent iMateResult is provided, an error will occur. |

## Version

Introduced in version 6
