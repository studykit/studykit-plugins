# UnfoldFeature.Bends Property

Parent Object: [UnfoldFeature](../UnfoldFeature/UnfoldFeature.md)

## Description

Property that returns the collection of Bend objects unfolded by this feature.

## Syntax

UnfoldFeature.**Bends**() As [BendsEnumerator](../BendsEnumerator/BendsEnumerator.md)

## Property Value

This is a read only property whose value is a [BendsEnumerator](../BendsEnumerator/BendsEnumerator.md).

## Version

Introduced in version 2010
