# CustomConstraint3D.Entity Property

Parent Object: [CustomConstraint3D](../CustomConstraint3D/CustomConstraint3D.md)

## Description

Property that returns the entity to which this custom constraint has been applied.

## Syntax

CustomConstraint3D.**Entity**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 11
