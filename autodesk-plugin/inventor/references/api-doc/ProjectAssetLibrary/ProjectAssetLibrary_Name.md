# ProjectAssetLibrary.Name Property

Parent Object: [ProjectAssetLibrary](../ProjectAssetLibrary/ProjectAssetLibrary.md)

## Description

Read-only property that returns the name of the library. This is the name displayed in the Projects dialog.

## Syntax

ProjectAssetLibrary.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2014
