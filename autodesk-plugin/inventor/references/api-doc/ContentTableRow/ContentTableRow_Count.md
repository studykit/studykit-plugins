# ContentTableRow.Count Property

Parent Object: [ContentTableRow](../ContentTableRow/ContentTableRow.md)

## Description

Property that returns the number of ContentTableCell objects in the collection.

## Syntax

ContentTableRow.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
