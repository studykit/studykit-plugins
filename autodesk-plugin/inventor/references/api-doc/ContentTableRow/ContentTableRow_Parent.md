# ContentTableRow.Parent Property

Parent Object: [ContentTableRow](../ContentTableRow/ContentTableRow.md)

## Description

Property that returns the parent ContentFamily object.

## Syntax

ContentTableRow.**Parent**() As [ContentFamily](../ContentFamily/ContentFamily.md)

## Property Value

This is a read only property whose value is a [ContentFamily](../ContentFamily/ContentFamily.md).

## Version

Introduced in version 2010
