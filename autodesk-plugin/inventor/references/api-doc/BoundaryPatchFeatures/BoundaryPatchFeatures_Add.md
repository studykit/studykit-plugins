# BoundaryPatchFeatures.Add Method

Parent Object: [BoundaryPatchFeatures](../BoundaryPatchFeatures/BoundaryPatchFeatures.md)

## Description

Method that creates a new boundary patch feature.

## Syntax

BoundaryPatchFeatures.**Add**( ***Definition*** As [BoundaryPatchDefinition](../BoundaryPatchDefinition/BoundaryPatchDefinition.md) ) As [BoundaryPatchFeature](../BoundaryPatchFeature/BoundaryPatchFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [BoundaryPatchDefinition](../BoundaryPatchDefinition/BoundaryPatchDefinition.md) | Input BoundaryPatchDefinition object that defines the input definition for the boundary patch feature. The BoundaryPatchDefinition object can be created using the CreateBoundaryPatchDefinition method of the BoundaryPatchFeatures object. The BoundaryPatchDefinition object defines the boundary loops for creating the boundary patch feature. |

## Version

Introduced in version 2008
