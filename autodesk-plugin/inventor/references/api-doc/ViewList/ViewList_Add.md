# ViewList.Add Method

Parent Object: [ViewList](../ViewList/ViewList.md)

## Description

Method that adds a View object to the collection.

## Syntax

ViewList.**Add**( ***View*** As [View](../View/View.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| View | [View](../View/View.md) | View object to add. |

## Version

Introduced in version 9
