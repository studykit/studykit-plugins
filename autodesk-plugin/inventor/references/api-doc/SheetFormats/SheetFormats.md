# SheetFormats Object

## Description

The SheetFormats collection object provides access to all the existing SheetFormat objects in a drawing document and provides methods to create additional sheet formats.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SheetFormats/SheetFormats_Add.md) | Method that creates a new SheetFormat. The newly created SheetFormat is returned. |
| [AddWithOptions](../SheetFormats/SheetFormats_AddWithOptions.md) | Creates a new SheetFormat object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SheetFormats/SheetFormats_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SheetFormats/SheetFormats_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SheetFormats/SheetFormats_Item.md) | Returns the specified SheetFormat object from the collection. |
| [Type](../SheetFormats/SheetFormats_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingDocument.SheetFormats](../DrawingDocument/DrawingDocument_SheetFormats.md)

## Version

Introduced in version 2009
