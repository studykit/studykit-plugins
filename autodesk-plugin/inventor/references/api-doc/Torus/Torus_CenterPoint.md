# Torus.CenterPoint Property

Parent Object: [Torus](../Torus/Torus.md)

## Description

Specifies the center of the torus.

## Syntax

Torus.**CenterPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
