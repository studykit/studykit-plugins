# FieldFlagDirectionTypeEnum Enumerator

## Description

Drawing weld symbol field flag direction type enum.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAutomaticDirection | 123137 | The Field flag changes direction based on the leader direction. |
| kFixedLeftDirection | 123138 | The Field flag always points to the left ignoring the direction of the leader. |
| kFixedRightDirection | 123139 | The Field flag always points to the right ignoring the direction of the leader. |

## Version

Introduced in version 2024
