# ReferenceKeyManager.KeyToString Method

Parent Object: [ReferenceKeyManager](../ReferenceKeyManager/ReferenceKeyManager.md)

## Description

Converts a ReferenceKey byte array to a Base64 encoded string.

## Syntax

ReferenceKeyManager.**KeyToString**( ***ReferenceKey***() As Byte ) As String

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ReferenceKey | Byte | The reference key to convert. |

## Version

Introduced in version 10
