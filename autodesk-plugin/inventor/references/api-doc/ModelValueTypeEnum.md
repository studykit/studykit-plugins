# ModelValueTypeEnum Enumerator

## Description

Settings for computing model value of a parameter.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kLowerValue | 31490 | Use Lower Tolerance value. |
| kMedianValue | 31492 | Use Median Tolerance value. |
| kNominalValue | 31489 | Use Nominal value. |
| kUpperValue | 31491 | Use Upper Tolerance value. |

## Version

Introduced in version 6
