# BossSet.Centers Property

Parent Object: [BossSet](../BossSet/BossSet.md)

## Description

Read-write property that gets and sets the collection of SketchPoint objects which specify the boss centers.

## Syntax

BossSet.**Centers**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2012
