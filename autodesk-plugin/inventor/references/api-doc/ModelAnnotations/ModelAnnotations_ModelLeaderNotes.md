# ModelAnnotations.ModelLeaderNotes Property

Parent Object: [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md)

## Description

Read-only property that returns the ModelLeaderNotes collection object. This object provides access to all of the model dimensions in the part or assembly.

## Syntax

ModelAnnotations.**ModelLeaderNotes**() As [ModelLeaderNotes](../ModelLeaderNotes/ModelLeaderNotes.md)

## Property Value

This is a read only property whose value is a [ModelLeaderNotes](../ModelLeaderNotes/ModelLeaderNotes.md).

## Version

Introduced in version 2018
