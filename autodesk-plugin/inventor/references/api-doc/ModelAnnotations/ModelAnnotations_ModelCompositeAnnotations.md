# ModelAnnotations.ModelCompositeAnnotations Property

Parent Object: [ModelAnnotations](../ModelAnnotations/ModelAnnotations.md)

## Description

Read-only property that returns ModelCompositeAnnotations collection object.

## Syntax

ModelAnnotations.**ModelCompositeAnnotations**() As [ModelCompositeAnnotations](../ModelCompositeAnnotations/ModelCompositeAnnotations.md)

## Property Value

This is a read only property whose value is a [ModelCompositeAnnotations](../ModelCompositeAnnotations/ModelCompositeAnnotations.md).

## Version

Introduced in version 2018
