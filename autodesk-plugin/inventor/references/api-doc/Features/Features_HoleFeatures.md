# Features.HoleFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Property that returns the HoleFeatures collection object. This collection provides access to existing HoleFeature objects.

## Syntax

Features.**HoleFeatures**() As [HoleFeatures](../HoleFeatures/HoleFeatures.md)

## Property Value

This is a read only property whose value is a [HoleFeatures](../HoleFeatures/HoleFeatures.md).

## Version

Introduced in version 10
