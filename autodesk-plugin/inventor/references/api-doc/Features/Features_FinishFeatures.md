# Features.FinishFeatures Property

Parent Object: [Features](../Features/Features.md)

## Description

Gets the collection object that besides listing out the subset of features that are FinishFeatures.

## Syntax

Features.**FinishFeatures**() As [FinishFeatures](../FinishFeatures/FinishFeatures.md)

## Property Value

This is a read only property whose value is a [FinishFeatures](../FinishFeatures/FinishFeatures.md).

## Version

Introduced in version 2024
