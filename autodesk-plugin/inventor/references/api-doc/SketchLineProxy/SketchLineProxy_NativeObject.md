# SketchLineProxy.NativeObject Property

Parent Object: [SketchLineProxy](../SketchLineProxy/SketchLineProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchLineProxy.**NativeObject**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 6
