# PointCloudProxy.MaximumPointCount Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-write property that gets and sets the maximum point count that can be displayed in current point cloud.

## Syntax

PointCloudProxy.**MaximumPointCount**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2016
