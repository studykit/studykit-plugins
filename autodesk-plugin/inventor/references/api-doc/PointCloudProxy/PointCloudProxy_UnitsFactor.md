# PointCloudProxy.UnitsFactor Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-only property that returns the units ratio between the point cloud source units and the Inventor database length unit(centimeters).

## Syntax

PointCloudProxy.**UnitsFactor**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2016
