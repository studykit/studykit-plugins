# PointCloudProxy.Name Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-only property that returns the name of the point cloud. This is the name that’s visible in the browser.

## Syntax

PointCloudProxy.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2016
