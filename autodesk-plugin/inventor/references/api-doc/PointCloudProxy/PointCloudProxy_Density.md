# PointCloudProxy.Density Property

Parent Object: [PointCloudProxy](../PointCloudProxy/PointCloudProxy.md)

## Description

Read-write property that gets and sets the density of the points displayed. The range of this property is from 1 to 10.

## Syntax

PointCloudProxy.**Density**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2016
