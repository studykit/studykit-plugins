# MeshFeatureSetProxy.NativeObject Property

Parent Object: [MeshFeatureSetProxy](../MeshFeatureSetProxy/MeshFeatureSetProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MeshFeatureSetProxy.**NativeObject**() As [MeshFeatureSet](../MeshFeatureSet/MeshFeatureSet.md)

## Property Value

This is a read only property whose value is a [MeshFeatureSet](../MeshFeatureSet/MeshFeatureSet.md).

## Version

Introduced in version 2017
