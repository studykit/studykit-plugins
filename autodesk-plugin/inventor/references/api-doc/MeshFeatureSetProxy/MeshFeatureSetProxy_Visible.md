# MeshFeatureSetProxy.Visible Property

Parent Object: [MeshFeatureSetProxy](../MeshFeatureSetProxy/MeshFeatureSetProxy.md)

## Description

Read-write property that gets and sets whether all of the mesh features within this set are visible.

## Syntax

MeshFeatureSetProxy.**Visible**() As [MeshFeaturesVisibilityEnum](../MeshFeaturesVisibilityEnum.md)

## Property Value

This is a read/write property whose value is a [MeshFeaturesVisibilityEnum](../MeshFeaturesVisibilityEnum.md).

## Version

Introduced in version 2017
