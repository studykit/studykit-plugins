# TransitionSymbolStyle.Name Property

Parent Object: [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Description

Gets/Sets the name of the Style.

## Syntax

TransitionSymbolStyle.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025.1
