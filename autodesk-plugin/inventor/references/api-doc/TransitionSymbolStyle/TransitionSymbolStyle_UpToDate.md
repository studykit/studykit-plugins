# TransitionSymbolStyle.UpToDate Property

Parent Object: [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Description

Property that gets the up-to-date status of the style against the global repository.

## Syntax

TransitionSymbolStyle.**UpToDate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025.1
