# ModelState.Parent Property

Parent Object: [ModelState](../ModelState/ModelState.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelState.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2022
