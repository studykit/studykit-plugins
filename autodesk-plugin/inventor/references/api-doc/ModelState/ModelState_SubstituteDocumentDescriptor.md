# ModelState.SubstituteDocumentDescriptor Property

Parent Object: [ModelState](../ModelState/ModelState.md)

## Description

Read-only property that returns the DocumentDescriptor of the document associated with the substitute occurrence. This property returns Nothing if the ModelState property returns anything but kSubstituteModelState.

## Syntax

ModelState.**SubstituteDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 2022
