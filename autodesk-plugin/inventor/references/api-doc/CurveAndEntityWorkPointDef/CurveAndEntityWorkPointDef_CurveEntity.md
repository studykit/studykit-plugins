# CurveAndEntityWorkPointDef.CurveEntity Property

Parent Object: [CurveAndEntityWorkPointDef](../CurveAndEntityWorkPointDef/CurveAndEntityWorkPointDef.md)

## Description

Property that returns the curve of a curve and entity defined work point. . This object can be an Edge or any 2d or 3d sketch object.

## Syntax

CurveAndEntityWorkPointDef.**CurveEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
