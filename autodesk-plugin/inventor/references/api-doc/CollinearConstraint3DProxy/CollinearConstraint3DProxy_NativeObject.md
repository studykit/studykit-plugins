# CollinearConstraint3DProxy.NativeObject Property

Parent Object: [CollinearConstraint3DProxy](../CollinearConstraint3DProxy/CollinearConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

CollinearConstraint3DProxy.**NativeObject**() As [CollinearConstraint3D](../CollinearConstraint3D/CollinearConstraint3D.md)

## Property Value

This is a read only property whose value is a [CollinearConstraint3D](../CollinearConstraint3D/CollinearConstraint3D.md).

## Version

Introduced in version 10
