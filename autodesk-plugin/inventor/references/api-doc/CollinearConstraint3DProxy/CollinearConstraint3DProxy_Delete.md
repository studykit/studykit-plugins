# CollinearConstraint3DProxy.Delete Method

Parent Object: [CollinearConstraint3DProxy](../CollinearConstraint3DProxy/CollinearConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

CollinearConstraint3DProxy.**Delete**()

## Version

Introduced in version 10
