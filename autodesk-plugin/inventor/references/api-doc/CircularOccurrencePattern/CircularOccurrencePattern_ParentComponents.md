# CircularOccurrencePattern.ParentComponents Property

Parent Object: [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Description

Property that gets and sets the components used as input for the pattern.

## Syntax

CircularOccurrencePattern.**ParentComponents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2010
