# MeshFace.MeshFeatureEntity Property

Parent Object: [MeshFace](../MeshFace/MeshFace.md)

## Description

Read-only property that returns the MeshFeatureEntity this MeshFace is located on.

## Syntax

MeshFace.**MeshFeatureEntity**() As [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Property Value

This is a read only property whose value is a [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md).

## Version

Introduced in version 2017
