# LinearDimensionTypeEnum Enumerator

## Description

Enum indicating the type of dimensioning to use while placing linear dimensions.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDiametricLinearDimension | 66306 | Linear diametric dimension type. |
| kRegularLinearDimension | 66305 | Linear regular dimension type. |
| kSymmetricLinearDimension | 66307 | Linear symmetric dimension type. |

## Version

Introduced in version 2008
