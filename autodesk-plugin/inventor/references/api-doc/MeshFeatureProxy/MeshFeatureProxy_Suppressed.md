# MeshFeatureProxy.Suppressed Property

Parent Object: [MeshFeatureProxy](../MeshFeatureProxy/MeshFeatureProxy.md)

## Description

Read-write property that gets and sets the whether this object is suppressed or not.

## Syntax

MeshFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
