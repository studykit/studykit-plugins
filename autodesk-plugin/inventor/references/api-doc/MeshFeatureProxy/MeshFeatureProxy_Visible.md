# MeshFeatureProxy.Visible Property

Parent Object: [MeshFeatureProxy](../MeshFeatureProxy/MeshFeatureProxy.md)

## Description

Read-write property that gets and sets the visiblity of this object.

## Syntax

MeshFeatureProxy.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
