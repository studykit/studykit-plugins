# SketchSplineHandle3D.Type Property

Parent Object: [SketchSplineHandle3D](../SketchSplineHandle3D/SketchSplineHandle3D.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchSplineHandle3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2008
