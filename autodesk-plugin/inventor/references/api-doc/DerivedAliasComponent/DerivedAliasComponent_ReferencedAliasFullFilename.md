# DerivedAliasComponent.ReferencedAliasFullFilename Property

Parent Object: [DerivedAliasComponent](../DerivedAliasComponent/DerivedAliasComponent.md)

## Description

Read-only property that returns the full filename of the derived Alias file.

## Syntax

DerivedAliasComponent.**ReferencedAliasFullFilename**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2021
