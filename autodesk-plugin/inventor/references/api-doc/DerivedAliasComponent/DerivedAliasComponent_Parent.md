# DerivedAliasComponent.Parent Property

Parent Object: [DerivedAliasComponent](../DerivedAliasComponent/DerivedAliasComponent.md)

## Description

Property that returns the parent object.

## Syntax

DerivedAliasComponent.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
