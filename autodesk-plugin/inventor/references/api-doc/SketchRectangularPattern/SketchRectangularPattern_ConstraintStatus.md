# SketchRectangularPattern.ConstraintStatus Property

Parent Object: [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Description

Returns the constraint status of the sketch entity. Possible return values are kFullyConstrainedConstraintStatus, kOverConstrainedConstraintStatus, kUnderConstrainedConstraintStatus and kUnknownConstraintStatus.

## Syntax

SketchRectangularPattern.**ConstraintStatus**() As [ConstraintStatusEnum](../ConstraintStatusEnum.md)

## Property Value

This is a read only property whose value is a [ConstraintStatusEnum](../ConstraintStatusEnum.md).

## Version

Introduced in version 2025.1
