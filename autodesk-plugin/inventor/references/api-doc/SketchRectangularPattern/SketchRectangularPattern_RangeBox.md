# SketchRectangularPattern.RangeBox Property

Parent Object: [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Description

Returns a Box2d object which contains the diagonal points of a rectangle that tightly encloses the entity.

## Syntax

SketchRectangularPattern.**RangeBox**() As [Box2d](../Box2d/Box2d.md)

## Property Value

This is a read only property whose value is a [Box2d](../Box2d/Box2d.md).

## Version

Introduced in version 2025.1
