# Sketches3DEnumerator.Count Property

Parent Object: [Sketches3DEnumerator](../Sketches3DEnumerator/Sketches3DEnumerator.md)

## Description

Property that returns the number of items in this collection.

## Syntax

Sketches3DEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2009
