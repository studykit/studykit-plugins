# SketchFillRegions.Item Property

Parent Object: [SketchFillRegions](../SketchFillRegions/SketchFillRegions.md)

## Description

Method that returns the specified object from the collection.

## Syntax

SketchFillRegions.**Item**( ***Index*** As Long ) As [SketchFillRegion](../SketchFillRegion/SketchFillRegion.md)

## Property Value

This is a read only property whose value is a [SketchFillRegion](../SketchFillRegion/SketchFillRegion.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 10
