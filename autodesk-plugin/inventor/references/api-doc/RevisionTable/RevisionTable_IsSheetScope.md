# RevisionTable.IsSheetScope Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets whether the scope of the revision table is the entire drawing or the parent sheet.

## Syntax

RevisionTable.**IsSheetScope**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
