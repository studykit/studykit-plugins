# RevisionTable.RevisionTableRows Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Property that returns the RevisionTableRows collection object.

## Syntax

RevisionTable.**RevisionTableRows**() As [RevisionTableRows](../RevisionTableRows/RevisionTableRows.md)

## Property Value

This is a read only property whose value is a [RevisionTableRows](../RevisionTableRows/RevisionTableRows.md).
