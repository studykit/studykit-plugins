# RevisionTable.HeadingPlacement Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets the location of the heading.

## Syntax

RevisionTable.**HeadingPlacement**() As [HeadingPlacementEnum](../HeadingPlacementEnum.md)

## Property Value

This is a read/write property whose value is a [HeadingPlacementEnum](../HeadingPlacementEnum.md).

## Version

Introduced in version 2012
