# RevisionTable.Style Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets the style associated with this object.

## Syntax

RevisionTable.**Style**() As [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Property Value

This is a read/write property whose value is a [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md).

## Version

Introduced in version 2011
