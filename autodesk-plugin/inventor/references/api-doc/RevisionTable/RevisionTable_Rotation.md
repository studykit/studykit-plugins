# RevisionTable.Rotation Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets and sets the absolute rotation angle of the revision table in radians.

## Syntax

RevisionTable.**Rotation**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2012
