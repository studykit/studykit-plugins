# RevisionTable.RowGap Property

Parent Object: [RevisionTable](../RevisionTable/RevisionTable.md)

## Description

Gets the row gap.

## Syntax

RevisionTable.**RowGap**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2012
