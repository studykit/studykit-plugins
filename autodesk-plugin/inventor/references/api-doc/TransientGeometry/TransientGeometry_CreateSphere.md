# TransientGeometry.CreateSphere Method

Parent Object: [TransientGeometry](../TransientGeometry/TransientGeometry.md)

## Description

Method that creates a new Sphere object. The object created is a transient mathematical object and is not displayed graphically

## Syntax

TransientGeometry.**CreateSphere**( ***CenterPoint*** As [Point](../Point/Point.md), ***Radius*** As Double ) As [Sphere](../Sphere/Sphere.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CenterPoint | [Point](../Point/Point.md) | Input Point object that specifies the center of the sphere. |
| Radius | Double | Input Double value that specifies the radius of the sphere. |

## Version

Introduced in version 4
