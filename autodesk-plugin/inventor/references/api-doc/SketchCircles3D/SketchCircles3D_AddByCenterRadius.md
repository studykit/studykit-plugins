# SketchCircles3D.AddByCenterRadius Method

Parent Object: [SketchCircles3D](../SketchCircles3D/SketchCircles3D.md)

## Description

Method that creates a new sketch circle at a specified location and radius.

## Syntax

SketchCircles3D.**AddByCenterRadius**( ***CenterPoint*** As Object, ***Normal*** As [UnitVector](../UnitVector/UnitVector.md), ***Radius*** As Double ) As [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CenterPoint | Object | Input object that defines the center point. This can currently only be a Point object. |
| Normal | [UnitVector](../UnitVector/UnitVector.md) | Input UnitVector that defines the normal of the circle. |
| Radius | Double | Input Double that defines the radius of the circle in centimeters |

## Version

Introduced in version 2008
