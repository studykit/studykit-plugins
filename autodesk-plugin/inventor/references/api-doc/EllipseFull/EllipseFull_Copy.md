# EllipseFull.Copy Method

Parent Object: [EllipseFull](../EllipseFull/EllipseFull.md)

## Description

Creates a copy of this EllipseFull object. The result is entirely independent and can be edited without affecting the original EllipseFull object.

## Syntax

EllipseFull.**Copy**() As [EllipseFull](../EllipseFull/EllipseFull.md)

## Version

Introduced in version 2013
