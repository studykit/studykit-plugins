# EllipseFull.Center Property

Parent Object: [EllipseFull](../EllipseFull/EllipseFull.md)

## Description

Specifies the center of the ellipse.

## Syntax

EllipseFull.**Center**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
