# ModelStateTableRow.MemberName Property

Parent Object: [ModelStateTableRow](../ModelStateTableRow/ModelStateTableRow.md)

## Description

Read-only property that returns the member name of the ModelStateTableRow.

## Syntax

ModelStateTableRow.**MemberName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
