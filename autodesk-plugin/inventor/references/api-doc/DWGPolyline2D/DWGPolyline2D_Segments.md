# DWGPolyline2D.Segments Property

Parent Object: [DWGPolyline2D](../DWGPolyline2D/DWGPolyline2D.md)

## Description

Read-only property that returns the DWGEntitySegmentsEnumerator object.

## Syntax

DWGPolyline2D.**Segments**() As [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md).

## Version

Introduced in version 2018
