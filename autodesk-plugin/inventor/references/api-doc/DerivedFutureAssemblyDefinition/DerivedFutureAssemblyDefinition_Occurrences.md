# DerivedFutureAssemblyDefinition.Occurrences Property

Parent Object: [DerivedFutureAssemblyDefinition](../DerivedFutureAssemblyDefinition/DerivedFutureAssemblyDefinition.md)

## Description

Read-only property that returns the derived assemblies.

## Syntax

DerivedFutureAssemblyDefinition.**Occurrences**() As [DerivedFutureAssemblyOccurrences](../DerivedFutureAssemblyOccurrences/DerivedFutureAssemblyOccurrences.md)

## Property Value

This is a read only property whose value is a [DerivedFutureAssemblyOccurrences](../DerivedFutureAssemblyOccurrences/DerivedFutureAssemblyOccurrences.md).

## Version

Introduced in version 2018
