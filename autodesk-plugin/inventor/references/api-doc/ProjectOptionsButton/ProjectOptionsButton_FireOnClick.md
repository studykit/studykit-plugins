# ProjectOptionsButton.FireOnClick Method

Parent Object: [ProjectOptionsButton](../ProjectOptionsButton/ProjectOptionsButton.md)

## Description

Method that fires the OnClick event.

## Syntax

ProjectOptionsButton.**FireOnClick**( ***Context*** As [NameValueMap](../NameValueMap/NameValueMap.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Context | [NameValueMap](../NameValueMap/NameValueMap.md) | Input NameValueMap object that can be used to determine the context of why the event fired. |

## Version

Introduced in version 2012
