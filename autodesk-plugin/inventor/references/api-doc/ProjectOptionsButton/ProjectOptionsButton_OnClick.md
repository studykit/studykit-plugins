# ProjectOptionsButton.OnClick Event

Parent Object: [ProjectOptionsButton](../ProjectOptionsButton/ProjectOptionsButton.md)

## Description

Event that is fired whenever click the button.

## Syntax

ProjectOptionsButton.**OnClick**( ***Context*** As [NameValueMap](../NameValueMap/NameValueMap.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Context | [NameValueMap](../NameValueMap/NameValueMap.md) | Input object that can be used to determine the context of why the event fired. |

## Version

Introduced in version 2012
