# RibbonTab.Parent Property

Parent Object: [RibbonTab](../RibbonTab/RibbonTab.md)

## Description

Property that returns the parent Ribbon object.

## Syntax

RibbonTab.**Parent**() As [Ribbon](../Ribbon/Ribbon.md)

## Property Value

This is a read only property whose value is a [Ribbon](../Ribbon/Ribbon.md).

## Version

Introduced in version 2010
