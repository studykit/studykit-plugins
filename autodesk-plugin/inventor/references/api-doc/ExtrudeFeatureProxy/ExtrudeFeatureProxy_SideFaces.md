# ExtrudeFeatureProxy.SideFaces Property

Parent Object: [ExtrudeFeatureProxy](../ExtrudeFeatureProxy/ExtrudeFeatureProxy.md)

## Description

Property that returns a object that provides access to all of the faces created around the perimeter of the feature.

## Syntax

ExtrudeFeatureProxy.**SideFaces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 9
