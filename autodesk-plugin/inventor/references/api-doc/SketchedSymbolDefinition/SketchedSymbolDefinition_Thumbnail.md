# SketchedSymbolDefinition.Thumbnail Property

Parent Object: [SketchedSymbolDefinition](../SketchedSymbolDefinition/SketchedSymbolDefinition.md)

## Description

Returns the thumbnail of this objec.

## Syntax

SketchedSymbolDefinition.**Thumbnail**() As IPictureDisp

## Property Value

This is a read only property whose value is an IPictureDisp.

## Version

Introduced in version 2025.1
