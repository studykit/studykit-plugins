# SketchedSymbolDefinition.Parent Property

Parent Object: [SketchedSymbolDefinition](../SketchedSymbolDefinition/SketchedSymbolDefinition.md)

## Description

Property returning the parent .

## Syntax

SketchedSymbolDefinition.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 5.3
