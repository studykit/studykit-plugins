# BIMComponentDescription.ViewCubeOrientationOrigin Property

Parent Object: [BIMComponentDescription](../BIMComponentDescription/BIMComponentDescription.md)

## Description

Read-write property that gets and set the origin of the coordinate system when the orientation is defined using the ViewCube.

## Syntax

BIMComponentDescription.**ViewCubeOrientationOrigin**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2014
