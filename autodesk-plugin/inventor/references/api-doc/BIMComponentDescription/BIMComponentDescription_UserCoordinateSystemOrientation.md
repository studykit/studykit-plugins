# BIMComponentDescription.UserCoordinateSystemOrientation Property

Parent Object: [BIMComponentDescription](../BIMComponentDescription/BIMComponentDescription.md)

## Description

Read-write property that defines which user coordinate system will be used when exporting the component. If the OrientationType is not kUserCoordinateSystemOrientationType this property returns Nothing.

## Syntax

BIMComponentDescription.**UserCoordinateSystemOrientation**() As [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md)

## Property Value

This is a read/write property whose value is a [UserCoordinateSystem](../UserCoordinateSystem/UserCoordinateSystem.md).

## Version

Introduced in version 2014
