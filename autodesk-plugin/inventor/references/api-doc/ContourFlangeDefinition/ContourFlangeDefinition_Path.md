# ContourFlangeDefinition.Path Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Gets and sets the Path object that defines the shape of the contour flange.

## Syntax

ContourFlangeDefinition.**Path**() As [Path](../Path/Path.md)

## Property Value

This is a read/write property whose value is a [Path](../Path/Path.md).

## Version

Introduced in version 2009
