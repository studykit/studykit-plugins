# ContourFlangeDefinition.WidthExtentsFromSketchPlane Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Read-only property that get and sets the width extents type is from sketch plane or along selected edge. This is applicable when the Operation is kJoinOperation.

## Syntax

ContourFlangeDefinition.**WidthExtentsFromSketchPlane**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2026
