# ContourFlangeDefinition.AddContourFlangeEdgeSet Method

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Method that adds a new contour flange edge set. The new FlangeEdgeSet is returned. This is applicable when the WidthExtentsFromSketchPlane is set to False.

## Syntax

ContourFlangeDefinition.**AddContourFlangeEdgeSet**( ***Edges*** As [EdgeCollection](../EdgeCollection/EdgeCollection.md) ) As [FlangeEdgeSet](../FlangeEdgeSet/FlangeEdgeSet.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Edges | [EdgeCollection](../EdgeCollection/EdgeCollection.md) | Input EdgeCollection containing Edge objects used to create the contour flange edge set. |

## Version

Introduced in version 2026
