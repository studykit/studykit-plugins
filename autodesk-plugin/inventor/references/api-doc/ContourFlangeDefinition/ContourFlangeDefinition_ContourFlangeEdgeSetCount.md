# ContourFlangeDefinition.ContourFlangeEdgeSetCount Property

Parent Object: [ContourFlangeDefinition](../ContourFlangeDefinition/ContourFlangeDefinition.md)

## Description

Read-only property that returns the contour flange edge set count currently defined in this contour flange definition.

## Syntax

ContourFlangeDefinition.**ContourFlangeEdgeSetCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2026
