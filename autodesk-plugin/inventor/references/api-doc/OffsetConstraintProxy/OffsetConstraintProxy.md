# OffsetConstraintProxy Object

Derived from: [OffsetConstraint](../OffsetConstraint/OffsetConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../OffsetConstraintProxy/OffsetConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../OffsetConstraintProxy/OffsetConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../OffsetConstraintProxy/OffsetConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../OffsetConstraintProxy/OffsetConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../OffsetConstraintProxy/OffsetConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../OffsetConstraintProxy/OffsetConstraintProxy_Deletable.md) | Indicates whether this object is deletable. |
| [EntityFour](../OffsetConstraintProxy/OffsetConstraintProxy_EntityFour.md) | Property that returns one of the entities. |
| [EntityOne](../OffsetConstraintProxy/OffsetConstraintProxy_EntityOne.md) | Property that returns one of the entities. |
| [EntityThree](../OffsetConstraintProxy/OffsetConstraintProxy_EntityThree.md) | Property that returns one of the entities. |
| [EntityTwo](../OffsetConstraintProxy/OffsetConstraintProxy_EntityTwo.md) | Property that returns one of the entities. |
| [NativeObject](../OffsetConstraintProxy/OffsetConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../OffsetConstraintProxy/OffsetConstraintProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Type](../OffsetConstraintProxy/OffsetConstraintProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
