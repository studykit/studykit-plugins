# AuxiliaryFeatureIndicator.SymbolType Property

Parent Object: [AuxiliaryFeatureIndicator](../AuxiliaryFeatureIndicator/AuxiliaryFeatureIndicator.md)

## Description

Read-write property that gets and sets the symbol for the auxiliary feature indicator.

## Syntax

AuxiliaryFeatureIndicator.**SymbolType**() As [AuxiliaryFeatureIndicatorSymbolTypeEnum](../AuxiliaryFeatureIndicatorSymbolTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AuxiliaryFeatureIndicatorSymbolTypeEnum](../AuxiliaryFeatureIndicatorSymbolTypeEnum.md).

## Version

Introduced in version 2023
