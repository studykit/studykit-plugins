# NormalToCurveWorkPlaneDef.CurveEntity Property

Parent Object: [NormalToCurveWorkPlaneDef](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef.md)

## Description

Property that returns the curve entity of a normal to curve defined work plane. This object can be an Edge, 3D Sketch entity, or 2D Sketch entity.

## Syntax

NormalToCurveWorkPlaneDef.**CurveEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
