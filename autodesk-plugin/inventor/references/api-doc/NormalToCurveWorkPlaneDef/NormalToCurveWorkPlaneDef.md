# NormalToCurveWorkPlaneDef Object

## Description

The NormalToCurveWorkPlaneDef object that allows you to get and set the information that specifies a work plane defined as normal to a curve and through a point.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef_GetData.md) | Method that returns all of the data defining a work plane defined by a curve and a point. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [CurveEntity](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef_CurveEntity.md) | Property that returns the curve entity of a normal to curve defined work plane. This object can be an Edge, 3D Sketch entity, or 2D Sketch entity. |
| [Parent](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Point](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef_Point.md) | Property that returns the point of a normal to curve defined work plane. The point can be a WorkPoint, Vertex, SketchPoint3D or SketchPoint object. The work plane passes through the point. |
| [Type](../NormalToCurveWorkPlaneDef/NormalToCurveWorkPlaneDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 6
