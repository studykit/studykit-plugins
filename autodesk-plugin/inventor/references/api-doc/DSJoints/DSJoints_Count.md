# DSJoints.Count Property

Parent Object: [DSJoints](../DSJoints/DSJoints.md)

## Description

Gets the number of items in this collection.

## Syntax

DSJoints.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2013
