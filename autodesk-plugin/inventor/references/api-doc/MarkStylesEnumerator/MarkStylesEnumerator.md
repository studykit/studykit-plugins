# MarkStylesEnumerator Object

## Description

The MarkStylesEnumerator collection object provides access to mark styles.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MarkStylesEnumerator/MarkStylesEnumerator_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../MarkStylesEnumerator/MarkStylesEnumerator_Count.md) | Gets the number of items in this collection. |
| [Item](../MarkStylesEnumerator/MarkStylesEnumerator_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../MarkStylesEnumerator/MarkStylesEnumerator_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PartDocument.MarkStyles](../PartDocument/PartDocument_MarkStyles.md)

## Version

Introduced in version 2023
