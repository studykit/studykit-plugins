# RevisionTableColumn.TitleHorizontalJustification Property

Parent Object: [RevisionTableColumn](../RevisionTableColumn/RevisionTableColumn.md)

## Description

Specifies the horizontal justification of the column title.

## Syntax

RevisionTableColumn.**TitleHorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 2008
