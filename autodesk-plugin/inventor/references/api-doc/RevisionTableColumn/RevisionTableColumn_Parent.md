# RevisionTableColumn.Parent Property

Parent Object: [RevisionTableColumn](../RevisionTableColumn/RevisionTableColumn.md)

## Description

Property that returns the parent RevisionTable.

## Syntax

RevisionTableColumn.**Parent**() As [RevisionTable](../RevisionTable/RevisionTable.md)

## Property Value

This is a read only property whose value is a [RevisionTable](../RevisionTable/RevisionTable.md).

## Version

Introduced in version 10
