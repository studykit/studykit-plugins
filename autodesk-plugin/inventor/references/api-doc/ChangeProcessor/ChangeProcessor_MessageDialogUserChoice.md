# ChangeProcessor.MessageDialogUserChoice Property

Parent Object: [ChangeProcessor](../ChangeProcessor/ChangeProcessor.md)

## Description

Read-only property that returns a ButtonTypeEnum value indicating the user’s choice in the message dialog.

## Syntax

ChangeProcessor.**MessageDialogUserChoice**() As [ButtonTypeEnum](../ButtonTypeEnum.md)

## Property Value

This is a read only property whose value is a [ButtonTypeEnum](../ButtonTypeEnum.md).

## Version

Introduced in version 2010
