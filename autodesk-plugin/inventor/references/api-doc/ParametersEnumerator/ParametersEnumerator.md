# ParametersEnumerator Object

## Description

The ParametersEnumerator object allows enumeration through available parameters.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../ParametersEnumerator/ParametersEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ParametersEnumerator/ParametersEnumerator_Item.md) | Returns the specified Parameter object from the collection. This is the default property of the ParametersEnumerator collection object. |
| [Type](../ParametersEnumerator/ParametersEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
