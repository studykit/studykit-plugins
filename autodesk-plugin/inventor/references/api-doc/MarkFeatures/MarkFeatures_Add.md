# MarkFeatures.Add Method

Parent Object: [MarkFeatures](../MarkFeatures/MarkFeatures.md)

## Description

Method that creates a new mark feature.

## Syntax

MarkFeatures.**Add**( ***pDefinition*** As [MarkDefinition](../MarkDefinition/MarkDefinition.md) ) As [MarkFeature](../MarkFeature/MarkFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| pDefinition | [MarkDefinition](../MarkDefinition/MarkDefinition.md) | Input MarkDefinition object that defines the mark feature you want to create. An MarkDefinition object can be created using the MarkFeatures.CreateMarkDefinition method. It can also be obtained from an existing MarkFeature object. |
