# MarkFeatures.CreateMarkDefinition Method

Parent Object: [MarkFeatures](../MarkFeatures/MarkFeatures.md)

## Description

Method that creates a new MarkDefinition objec.

## Syntax

MarkFeatures.**CreateMarkDefinition**( ***pGeoms*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md), ***pStyle*** As [MarkStyle](../MarkStyle/MarkStyle.md) ) As [MarkDefinition](../MarkDefinition/MarkDefinition.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| pGeoms | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Input ObjectCollection object that specifies the sketch geometries to create mark feature. |
| pStyle | [MarkStyle](../MarkStyle/MarkStyle.md) | Input MarkStyle object that specifies mark style. |
