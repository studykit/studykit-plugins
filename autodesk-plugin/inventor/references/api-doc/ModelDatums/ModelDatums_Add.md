# ModelDatums.Add Method

Parent Object: [ModelDatums](../ModelDatums/ModelDatums.md)

## Description

Method that creates a model datum.

## Syntax

ModelDatums.**Add**( ***Definition*** As [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md) ) As [ModelDatum](../ModelDatum/ModelDatum.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md) | Input ModelDatumDefinition object that defines the model datum to be created. |

## Version

Introduced in version 2023
