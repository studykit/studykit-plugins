# ModelDatums.NextAvailableDatumText Property

Parent Object: [ModelDatums](../ModelDatums/ModelDatums.md)

## Description

Returns a string value that indicates an unused datum text.

## Syntax

ModelDatums.**NextAvailableDatumText**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2023
