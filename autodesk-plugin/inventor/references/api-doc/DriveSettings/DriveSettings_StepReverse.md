# DriveSettings.StepReverse Method

Parent Object: [DriveSettings](../DriveSettings/DriveSettings.md)

## Description

Method that reverses the animation by one step.

## Syntax

DriveSettings.**StepReverse**()

## Version

Introduced in version 2014
