# Centerlines Object

## Description

The Centerlines collection object provides access to all the centerlines on a sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../Centerlines/Centerlines_Add.md) | Method that creates a new centerline. The centerline created with this method is kRegularCenterline type. |
| [AddBisector](../Centerlines/Centerlines_AddBisector.md) | Method that creates a new bisector centerline. The centerline created with this method is kBisectorCenterline type. |
| [AddByWorkFeature](../Centerlines/Centerlines_AddByWorkFeature.md) | Method that creates a center line that represents the work feature within the drawing view. |
| [AddCenteredPattern](../Centerlines/Centerlines_AddCenteredPattern.md) | Method that creates a new centerline pattern. The centerline created with this method is kCenteredPatternCenterline type. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Centerlines/Centerlines_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../Centerlines/Centerlines_Count.md) | Property that returns the number of items in the collection. |
| [Item](../Centerlines/Centerlines_Item.md) | Method that returns the specified Centerline object from the collection. |
| [Type](../Centerlines/Centerlines_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Sheet.Centerlines](../Sheet/Sheet_Centerlines.md)

## Version

Introduced in version 2008
