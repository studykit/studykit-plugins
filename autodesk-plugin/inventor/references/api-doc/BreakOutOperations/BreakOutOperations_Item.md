# BreakOutOperations.Item Property

Parent Object: [BreakOutOperations](../BreakOutOperations/BreakOutOperations.md)

## Description

Method that returns the specified break out operation object from the collection.

## Syntax

BreakOutOperations.**Item**( ***Index*** As Long ) As [BreakOutOperation](../BreakOutOperation/BreakOutOperation.md)

## Property Value

This is a read only property whose value is a [BreakOutOperation](../BreakOutOperation/BreakOutOperation.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2010
