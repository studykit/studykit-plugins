# FlatPatternFeatures.MarkFeatures Property

Parent Object: [FlatPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures.md)

## Description

Property that returns the MarkFeature collection object. This collection provides access to existing MarkFeature objects and provides functionality to create new MarkFeature objects.

## Syntax

FlatPatternFeatures.**MarkFeatures**() As [MarkFeatures](../MarkFeatures/MarkFeatures.md)

## Property Value

This is a read only property whose value is a [MarkFeatures](../MarkFeatures/MarkFeatures.md).

## Version

Introduced in version 2024
