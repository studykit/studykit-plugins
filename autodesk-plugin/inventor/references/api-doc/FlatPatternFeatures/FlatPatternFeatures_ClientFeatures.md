# FlatPatternFeatures.ClientFeatures Property

Parent Object: [FlatPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures.md)

## Description

Gets the ClientFeatures object associated with the flat pattern.

## Syntax

FlatPatternFeatures.**ClientFeatures**() As [ClientFeatures](../ClientFeatures/ClientFeatures.md)

## Property Value

This is a read only property whose value is a [ClientFeatures](../ClientFeatures/ClientFeatures.md).

## Version

Introduced in version 2013
