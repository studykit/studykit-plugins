# MirrorFeatures.AddByDefinition Method

Parent Object: [MirrorFeatures](../MirrorFeatures/MirrorFeatures.md)

## Description

Method that creates a new mirror feature.

## Syntax

MirrorFeatures.**AddByDefinition**( ***Definition*** As [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md) ) As [MirrorFeature](../MirrorFeature/MirrorFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md) | Input MirrorFeatureDefinition objet that defines the MirrorFeature you want to create. |

## Version

Introduced in version 2017
