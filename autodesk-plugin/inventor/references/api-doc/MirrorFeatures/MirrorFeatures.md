# MirrorFeatures Object

## Description

The MirrorFeatures object provides access to all of the objects in a component definition and provides methods to create additional MirrorFeature objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddByDefinition](../MirrorFeatures/MirrorFeatures_AddByDefinition.md) | Method that creates a new mirror feature. |
| [CreateDefinition](../MirrorFeatures/MirrorFeatures_CreateDefinition.md) | Method that creates a new MirrorFeatureDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../MirrorFeatures/MirrorFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../MirrorFeatures/MirrorFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../MirrorFeatures/MirrorFeatures_Item.md) | Returns the specified MirrorFeature objects from the collection. This is the default property of the MirrorFeatures collection object. |
| [Type](../MirrorFeatures/MirrorFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Features.MirrorFeatures](../Features/Features_MirrorFeatures.md), [FlatPatternFeatures.MirrorFeatures](../FlatPatternFeatures/FlatPatternFeatures_MirrorFeatures.md), [PartFeatures.MirrorFeatures](../PartFeatures/PartFeatures_MirrorFeatures.md), [SheetMetalFeatures.MirrorFeatures](../SheetMetalFeatures/SheetMetalFeatures_MirrorFeatures.md)

## Version

Introduced in version 5
