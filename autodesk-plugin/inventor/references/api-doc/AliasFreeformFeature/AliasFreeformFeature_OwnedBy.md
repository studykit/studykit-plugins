# AliasFreeformFeature.OwnedBy Property

Parent Object: [AliasFreeformFeature](../AliasFreeformFeature/AliasFreeformFeature.md)

## Description

Property that returns the owning PartFeature object. This property returns Nothing if the IsOwnedByFeature property returns False.

## Syntax

AliasFreeformFeature.**OwnedBy**() As [PartFeature](../PartFeature/PartFeature.md)

## Property Value

This is a read only property whose value is a [PartFeature](../PartFeature/PartFeature.md).

## Version

Introduced in version 2011
