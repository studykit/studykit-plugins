# Leader.RootNode Property

Parent Object: [Leader](../Leader/Leader.md)

## Description

Property that returns the root node of this leader. This property will return Nothing if the HasRootNode property returns False.

## Syntax

Leader.**RootNode**() As [LeaderNode](../LeaderNode/LeaderNode.md)

## Property Value

This is a read only property whose value is a [LeaderNode](../LeaderNode/LeaderNode.md).
