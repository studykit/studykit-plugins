# IntersectionCurves.Type Property

Parent Object: [IntersectionCurves](../IntersectionCurves/IntersectionCurves.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

IntersectionCurves.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
