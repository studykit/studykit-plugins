# Transaction.ParentTransaction Property

Parent Object: [Transaction](../Transaction/Transaction.md)

## Description

Property that returns the parent transaction, if one exists; else returns a NULL pointer.

## Syntax

Transaction.**ParentTransaction**() As [Transaction](../Transaction/Transaction.md)

## Property Value

This is a read only property whose value is a [Transaction](../Transaction/Transaction.md).

## Version

Introduced in version 5
