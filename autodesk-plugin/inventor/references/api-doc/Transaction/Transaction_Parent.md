# Transaction.Parent Property

Parent Object: [Transaction](../Transaction/Transaction.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

Transaction.**Parent**() As [TransactionManager](../TransactionManager/TransactionManager.md)

## Property Value

This is a read only property whose value is a [TransactionManager](../TransactionManager/TransactionManager.md).

## Version

Introduced in version 5
