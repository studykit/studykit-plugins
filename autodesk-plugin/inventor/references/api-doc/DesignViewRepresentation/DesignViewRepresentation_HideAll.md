# DesignViewRepresentation.HideAll Method

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Method that makes all components invisible in this representation.

## Syntax

DesignViewRepresentation.**HideAll**()

## Version

Introduced in version 11
