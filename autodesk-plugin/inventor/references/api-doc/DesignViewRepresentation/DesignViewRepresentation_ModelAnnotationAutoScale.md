# DesignViewRepresentation.ModelAnnotationAutoScale Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Gets and sets the design view representations model annotation scale mode.

## Syntax

DesignViewRepresentation.**ModelAnnotationAutoScale**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
