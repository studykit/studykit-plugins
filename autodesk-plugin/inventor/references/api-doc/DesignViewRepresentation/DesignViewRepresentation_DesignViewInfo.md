# DesignViewRepresentation.DesignViewInfo Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Property that returns the design view representation information as a String containing XML formatted data. This is applicable for assembly documents only. For more information on the XML format see [More XML Design View Info...](MoreXMLDesignViewInfo_Overview.md)

## Syntax

DesignViewRepresentation.**DesignViewInfo**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 11
