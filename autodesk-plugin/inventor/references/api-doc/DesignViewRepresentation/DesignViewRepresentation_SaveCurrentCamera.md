# DesignViewRepresentation.SaveCurrentCamera Method

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Method that saves the camera of the active view to the design view.

## Syntax

DesignViewRepresentation.**SaveCurrentCamera**()

## Version

Introduced in version 2014
