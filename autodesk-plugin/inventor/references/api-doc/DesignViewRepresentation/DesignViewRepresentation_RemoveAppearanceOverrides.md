# DesignViewRepresentation.RemoveAppearanceOverrides Method

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Resets all components to their default appearance.

## Syntax

DesignViewRepresentation.**RemoveAppearanceOverrides**()

## Version

Introduced in version 2014
