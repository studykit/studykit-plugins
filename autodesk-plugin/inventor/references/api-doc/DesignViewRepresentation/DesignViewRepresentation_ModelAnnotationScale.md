# DesignViewRepresentation.ModelAnnotationScale Property

Parent Object: [DesignViewRepresentation](../DesignViewRepresentation/DesignViewRepresentation.md)

## Description

Gets and sets the design view representations model annotation scale.

## Syntax

DesignViewRepresentation.**ModelAnnotationScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2018
