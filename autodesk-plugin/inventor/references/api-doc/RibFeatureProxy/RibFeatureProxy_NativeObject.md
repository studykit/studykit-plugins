# RibFeatureProxy.NativeObject Property

Parent Object: [RibFeatureProxy](../RibFeatureProxy/RibFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RibFeatureProxy.**NativeObject**() As [RibFeature](../RibFeature/RibFeature.md)

## Property Value

This is a read only property whose value is a [RibFeature](../RibFeature/RibFeature.md).

## Version

Introduced in version 9
