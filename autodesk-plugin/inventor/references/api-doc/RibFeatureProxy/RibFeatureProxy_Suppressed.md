# RibFeatureProxy.Suppressed Property

Parent Object: [RibFeatureProxy](../RibFeatureProxy/RibFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

RibFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
