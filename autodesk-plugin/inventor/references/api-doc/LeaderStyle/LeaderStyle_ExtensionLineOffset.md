# LeaderStyle.ExtensionLineOffset Property

Parent Object: [LeaderStyle](../LeaderStyle/LeaderStyle.md)

## Description

Property that gets and sets the space from the end of the edge to which the leader attaches to the beginning of the leader extension line.

## Syntax

LeaderStyle.**ExtensionLineOffset**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
