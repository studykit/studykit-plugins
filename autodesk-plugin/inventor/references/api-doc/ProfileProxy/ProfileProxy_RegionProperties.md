# ProfileProxy.RegionProperties Property

Parent Object: [ProfileProxy](../ProfileProxy/ProfileProxy.md)

## Description

Returns the RegionProperties object.

## Syntax

ProfileProxy.**RegionProperties**() As [RegionProperties](../RegionProperties/RegionProperties.md)

## Property Value

This is a read only property whose value is a [RegionProperties](../RegionProperties/RegionProperties.md).

## Version

Introduced in version 2008
