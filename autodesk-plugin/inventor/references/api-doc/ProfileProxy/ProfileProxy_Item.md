# ProfileProxy.Item Property

Parent Object: [ProfileProxy](../ProfileProxy/ProfileProxy.md)

## Description

Returns the specified ProfilePath object from the collection.

## Syntax

ProfileProxy.**Item**( ***Index*** As Long ) As [ProfilePath](../ProfilePath/ProfilePath.md)

## Property Value

This is a read only property whose value is a [ProfilePath](../ProfilePath/ProfilePath.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Specifies the index of the object to return. |

## Version

Introduced in version 6
