# ProfileProxy.Wires Property

Parent Object: [ProfileProxy](../ProfileProxy/ProfileProxy.md)

## Description

Property returning the Wires collection object associated with this Profile.

## Syntax

ProfileProxy.**Wires**() As [Wires](../Wires/Wires.md)

## Property Value

This is a read only property whose value is a [Wires](../Wires/Wires.md).

## Version

Introduced in version 2009
