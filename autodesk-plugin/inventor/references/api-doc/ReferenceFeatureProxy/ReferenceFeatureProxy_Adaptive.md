# ReferenceFeatureProxy.Adaptive Property

Parent Object: [ReferenceFeatureProxy](../ReferenceFeatureProxy/ReferenceFeatureProxy.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

ReferenceFeatureProxy.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
