# ReferenceFeatureProxy.NativeObject Property

Parent Object: [ReferenceFeatureProxy](../ReferenceFeatureProxy/ReferenceFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ReferenceFeatureProxy.**NativeObject**() As [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md)

## Property Value

This is a read only property whose value is a [ReferenceFeature](../ReferenceFeature/ReferenceFeature.md).

## Version

Introduced in version 9
