# UnwrapDefinition.ResultAlignment Property

Parent Object: [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md)

## Description

Read-write property that gets and sets alignment of the deformed result.

## Syntax

UnwrapDefinition.**ResultAlignment**() As [UnwrapResultAlignmentEnum](../UnwrapResultAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [UnwrapResultAlignmentEnum](../UnwrapResultAlignmentEnum.md).

## Version

Introduced in version 2020
