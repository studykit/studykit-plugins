# UnwrapDefinition.Origin Property

Parent Object: [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md)

## Description

Read-write property that gets and sets the vertex as origin.

## Syntax

UnwrapDefinition.**Origin**() As [Vertex](../Vertex/Vertex.md)

## Property Value

This is a read/write property whose value is a [Vertex](../Vertex/Vertex.md).

## Version

Introduced in version 2020
