# DecalFeature.Adaptive Property

Parent Object: [DecalFeature](../DecalFeature/DecalFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

DecalFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
