# Cone.Evaluator Property

Parent Object: [Cone](../Cone/Cone.md)

## Description

Gets the surface evaluator for this cone.

## Syntax

Cone.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
