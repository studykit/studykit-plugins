# FeatureDimension.TextPoint Property

Parent Object: [FeatureDimension](../FeatureDimension/FeatureDimension.md)

## Description

Property that returns the position of the dimension text.

## Syntax

FeatureDimension.**TextPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).
