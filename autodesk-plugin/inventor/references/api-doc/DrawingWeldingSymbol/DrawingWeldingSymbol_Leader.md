# DrawingWeldingSymbol.Leader Property

Parent Object: [DrawingWeldingSymbol](../DrawingWeldingSymbol/DrawingWeldingSymbol.md)

## Description

Read-only property that returns the leader associated with the drawing welding symbol.

## Syntax

DrawingWeldingSymbol.**Leader**() As [Leader](../Leader/Leader.md)

## Property Value

This is a read only property whose value is a [Leader](../Leader/Leader.md).

## Version

Introduced in version 2024
