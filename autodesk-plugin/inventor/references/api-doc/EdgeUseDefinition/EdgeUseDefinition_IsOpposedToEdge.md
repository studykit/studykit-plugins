# EdgeUseDefinition.IsOpposedToEdge Property

Parent Object: [EdgeUseDefinition](../EdgeUseDefinition/EdgeUseDefinition.md)

## Description

Gets and sets if the orientation of this EdgeUse is in the same direction or not relative to the associated EdgeDefinition object.

## Syntax

EdgeUseDefinition.**IsOpposedToEdge**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
