# Arc3d.StartPoint Property

Parent Object: [Arc3d](../Arc3d/Arc3d.md)

## Description

Property that gets the start point of the arc.

## Syntax

Arc3d.**StartPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).
