# DrawingPrintManager.AllColorsAsBlack Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the Boolean flag indicating whether all the Drawing colors should be black in the print.

## Syntax

DrawingPrintManager.**AllColorsAsBlack**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
