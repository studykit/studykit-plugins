# DrawingPrintManager.Rotate90Degrees Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the Boolean flag indicating whether the Drawing should be rotated 90-deg in the print.

## Syntax

DrawingPrintManager.**Rotate90Degrees**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
