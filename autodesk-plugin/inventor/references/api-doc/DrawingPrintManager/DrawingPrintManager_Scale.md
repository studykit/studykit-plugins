# DrawingPrintManager.Scale Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the scale of the print. Meaningless when the ScaleMode is not kPrintCustomScale.

## Syntax

DrawingPrintManager.**Scale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 5
