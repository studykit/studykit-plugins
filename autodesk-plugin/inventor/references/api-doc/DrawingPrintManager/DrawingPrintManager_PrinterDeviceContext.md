# DrawingPrintManager.PrinterDeviceContext Property

Parent Object: [DrawingPrintManager](../DrawingPrintManager/DrawingPrintManager.md)

## Description

Gets/Sets the printer's device context.

## Syntax

DrawingPrintManager.**PrinterDeviceContext**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 9
