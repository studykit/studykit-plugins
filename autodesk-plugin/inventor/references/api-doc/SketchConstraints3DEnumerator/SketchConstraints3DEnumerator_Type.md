# SketchConstraints3DEnumerator.Type Property

Parent Object: [SketchConstraints3DEnumerator](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchConstraints3DEnumerator.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
