# PresentationDocument.AttributeManager Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Returns the AttributeManager object.

## Syntax

PresentationDocument.**AttributeManager**() As [AttributeManager](../AttributeManager/AttributeManager.md)

## Property Value

This is a read only property whose value is an [AttributeManager](../AttributeManager/AttributeManager.md).

## Version

Introduced in version 5
