# PresentationDocument.ActiveScene Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Read-only property that returns the active PresentationScene object.

## Syntax

PresentationDocument.**ActiveScene**() As [PresentationScene](../PresentationScene/PresentationScene.md)

## Property Value

This is a read only property whose value is a [PresentationScene](../PresentationScene/PresentationScene.md).

## Version

Introduced in version 2018
