# PresentationDocument.ResolveFile Method

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Method that invoke the resolve file for the document if there are references missing.

## Syntax

PresentationDocument.**ResolveFile**()

## Version

Introduced in version 2025
