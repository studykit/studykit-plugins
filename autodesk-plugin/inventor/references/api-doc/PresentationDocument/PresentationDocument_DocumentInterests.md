# PresentationDocument.DocumentInterests Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the DocumentInterests collection object.

## Syntax

PresentationDocument.**DocumentInterests**() As [DocumentInterests](../DocumentInterests/DocumentInterests.md)

## Property Value

This is a read only property whose value is a [DocumentInterests](../DocumentInterests/DocumentInterests.md).

## Version

Introduced in version 2008
