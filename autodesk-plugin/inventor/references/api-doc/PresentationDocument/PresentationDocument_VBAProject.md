# PresentationDocument.VBAProject Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the Autodesk Inventor VBA project for this document.

## Syntax

PresentationDocument.**VBAProject**() As [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Property Value

This is a read only property whose value is an [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md).

## Version

Introduced in version 6
