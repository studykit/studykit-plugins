# PresentationDocument.ThumbnailSaveOption Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the current thumbnail (preview picture) save option.

## Syntax

PresentationDocument.**ThumbnailSaveOption**() As [ThumbnailSaveOptionEnum](../ThumbnailSaveOptionEnum.md)

## Property Value

This is a read only property whose value is a [ThumbnailSaveOptionEnum](../ThumbnailSaveOptionEnum.md).

## Version

Introduced in version 2010
