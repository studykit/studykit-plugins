# PresentationDocument.GraphicsDataSetsCollection Property

Parent Object: [PresentationDocument](../PresentationDocument/PresentationDocument.md)

## Description

Property that returns the object for the document.

## Syntax

PresentationDocument.**GraphicsDataSetsCollection**() As [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md)

## Property Value

This is a read only property whose value is a [GraphicsDataSetsCollection](../GraphicsDataSetsCollection/GraphicsDataSetsCollection.md).

## Version

Introduced in version 5
