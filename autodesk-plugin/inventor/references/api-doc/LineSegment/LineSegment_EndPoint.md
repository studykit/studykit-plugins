# LineSegment.EndPoint Property

Parent Object: [LineSegment](../LineSegment/LineSegment.md)

## Description

Specifies the end point of the line segment.

## Syntax

LineSegment.**EndPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).
