# FixedWorkAxisDef.Axis Property

Parent Object: [FixedWorkAxisDef](../FixedWorkAxisDef/FixedWorkAxisDef.md)

## Description

Property that returns the X-axis vector of the work plane.

## Syntax

FixedWorkAxisDef.**Axis**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 4
