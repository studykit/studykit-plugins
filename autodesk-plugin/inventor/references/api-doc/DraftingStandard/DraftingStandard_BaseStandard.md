# DraftingStandard.BaseStandard Property

Parent Object: [DraftingStandard](../DraftingStandard/DraftingStandard.md)

## Description

Gets the base drafting standard associated with this particular object.

## Syntax

DraftingStandard.**BaseStandard**() As [DraftingStandardEnum](../DraftingStandardEnum.md)

## Property Value

This is a read only property whose value is a [DraftingStandardEnum](../DraftingStandardEnum.md).

## Version

Introduced in version 4
