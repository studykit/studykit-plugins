# Parameters.DimensionDisplayType Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Gets/Sets the dimension display type for dimensions in this document.

## Syntax

Parameters.**DimensionDisplayType**() As [DimensionDisplayTypeEnum](../DimensionDisplayTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionDisplayTypeEnum](../DimensionDisplayTypeEnum.md).

## Version

Introduced in version 6
