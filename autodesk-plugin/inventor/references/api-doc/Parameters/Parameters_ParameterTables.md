# Parameters.ParameterTables Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Property that returns the ParameterTables collection object.

## Syntax

Parameters.**ParameterTables**() As [ParameterTables](../ParameterTables/ParameterTables.md)

## Property Value

This is a read only property whose value is a [ParameterTables](../ParameterTables/ParameterTables.md).

|  |  |
| --- | --- |
| Name | Description |
| [Table Parameters](../../sample-programs/ParameterTables_Sample.md) | This sample demonstrates how to access the Parameters object, and from it in turn the TableParameters collection that represents the collection of parameters that have been linked/embedded from an external spreadsheet. |
