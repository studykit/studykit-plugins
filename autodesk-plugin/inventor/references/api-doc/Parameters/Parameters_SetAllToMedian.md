# Parameters.SetAllToMedian Method

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Sets the values of all parameters to be the median value as defined by the parameter's tolerance.

## Syntax

Parameters.**SetAllToMedian**()

## Version

Introduced in version 10
