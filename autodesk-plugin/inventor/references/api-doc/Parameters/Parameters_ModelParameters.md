# Parameters.ModelParameters Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Property that returns the ModelParameters collection object.

## Syntax

Parameters.**ModelParameters**() As [ModelParameters](../ModelParameters/ModelParameters.md)

## Property Value

This is a read only property whose value is a [ModelParameters](../ModelParameters/ModelParameters.md).
