# Parameters.DerivedParameterTables Property

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Property that returns the DerivedParameterTables collection object.

## Syntax

Parameters.**DerivedParameterTables**() As [DerivedParameterTables](../DerivedParameterTables/DerivedParameterTables.md)

## Property Value

This is a read only property whose value is a [DerivedParameterTables](../DerivedParameterTables/DerivedParameterTables.md).

## Version

Introduced in version 11
