# Parameters.SetAllToNominal Method

Parent Object: [Parameters](../Parameters/Parameters.md)

## Description

Method that sets the values of all parameters to be the nominal value as defined by the parameter's tolerance.

## Syntax

Parameters.**SetAllToNominal**()

## Version

Introduced in version 6
