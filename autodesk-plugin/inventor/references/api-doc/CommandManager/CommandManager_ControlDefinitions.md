# CommandManager.ControlDefinitions Property

Parent Object: [CommandManager](../CommandManager/CommandManager.md)

## Description

Property that returns the ControlDefinitions collection.

## Syntax

CommandManager.**ControlDefinitions**() As [ControlDefinitions](../ControlDefinitions/ControlDefinitions.md)

## Property Value

This is a read only property whose value is a [ControlDefinitions](../ControlDefinitions/ControlDefinitions.md).
