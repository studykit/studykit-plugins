# DrawingViewHatchRegion.Pattern Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that gets and sets the DrawingHatchPattern of the drawing view hatch region.

## Syntax

DrawingViewHatchRegion.**Pattern**() As [DrawingHatchPattern](../DrawingHatchPattern/DrawingHatchPattern.md)

## Property Value

This is a read/write property whose value is a [DrawingHatchPattern](../DrawingHatchPattern/DrawingHatchPattern.md).

## Version

Introduced in version 2022
