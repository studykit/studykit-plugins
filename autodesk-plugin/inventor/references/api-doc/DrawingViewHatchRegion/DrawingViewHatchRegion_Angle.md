# DrawingViewHatchRegion.Angle Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-write property that gets and sets the rotation angle of the pattern in radians.

## Syntax

DrawingViewHatchRegion.**Angle**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
