# DrawingViewHatchRegion.HatchAreas Property

Parent Object: [DrawingViewHatchRegion](../DrawingViewHatchRegion/DrawingViewHatchRegion.md)

## Description

Read-only property that returns the DrawingViewHatchAreas collection object.

## Syntax

DrawingViewHatchRegion.**HatchAreas**() As [DrawingViewHatchAreas](../DrawingViewHatchAreas/DrawingViewHatchAreas.md)

## Property Value

This is a read only property whose value is a [DrawingViewHatchAreas](../DrawingViewHatchAreas/DrawingViewHatchAreas.md).

## Version

Introduced in version 2022
