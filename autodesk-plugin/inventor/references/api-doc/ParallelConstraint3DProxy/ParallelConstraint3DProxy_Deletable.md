# ParallelConstraint3DProxy.Deletable Property

Parent Object: [ParallelConstraint3DProxy](../ParallelConstraint3DProxy/ParallelConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

ParallelConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
