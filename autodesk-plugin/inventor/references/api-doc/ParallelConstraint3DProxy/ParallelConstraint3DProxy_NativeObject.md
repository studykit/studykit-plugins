# ParallelConstraint3DProxy.NativeObject Property

Parent Object: [ParallelConstraint3DProxy](../ParallelConstraint3DProxy/ParallelConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ParallelConstraint3DProxy.**NativeObject**() As [ParallelConstraint3D](../ParallelConstraint3D/ParallelConstraint3D.md)

## Property Value

This is a read only property whose value is a [ParallelConstraint3D](../ParallelConstraint3D/ParallelConstraint3D.md).

## Version

Introduced in version 10
