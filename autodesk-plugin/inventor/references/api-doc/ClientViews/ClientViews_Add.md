# ClientViews.Add Method

Parent Object: [ClientViews](../ClientViews/ClientViews.md)

## Description

Method that adds a new to the collection.

## Syntax

ClientViews.**Add**( ***HWND*** As Long ) As [ClientView](../ClientView/ClientView.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| HWND | Long | Input Long that defines the hWnd for the window to add. |

## Version

Introduced in version 6
