# SurfaceTextureGOSTDefinition.ProductionMethod Property

Parent Object: [SurfaceTextureGOSTDefinition](../SurfaceTextureGOSTDefinition/SurfaceTextureGOSTDefinition.md)

## Description

Gets and sets the productuction method for the surface.

## Syntax

SurfaceTextureGOSTDefinition.**ProductionMethod**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
