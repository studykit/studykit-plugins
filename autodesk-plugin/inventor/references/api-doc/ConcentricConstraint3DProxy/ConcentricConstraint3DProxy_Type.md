# ConcentricConstraint3DProxy.Type Property

Parent Object: [ConcentricConstraint3DProxy](../ConcentricConstraint3DProxy/ConcentricConstraint3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

ConcentricConstraint3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2010
