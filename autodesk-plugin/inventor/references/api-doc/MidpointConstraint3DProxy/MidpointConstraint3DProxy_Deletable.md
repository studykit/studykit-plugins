# MidpointConstraint3DProxy.Deletable Property

Parent Object: [MidpointConstraint3DProxy](../MidpointConstraint3DProxy/MidpointConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

MidpointConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
