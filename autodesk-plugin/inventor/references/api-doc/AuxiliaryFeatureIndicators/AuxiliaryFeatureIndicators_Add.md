# AuxiliaryFeatureIndicators.Add Method

Parent Object: [AuxiliaryFeatureIndicators](../AuxiliaryFeatureIndicators/AuxiliaryFeatureIndicators.md)

## Description

Method that creates a new AuxiliaryFeatureIndicator. The newly created AuxiliaryFeatureIndicator object is returned.

## Syntax

AuxiliaryFeatureIndicators.**Add**( ***SymbolType*** As [AuxiliaryFeatureIndicatorSymbolTypeEnum](../AuxiliaryFeatureIndicatorSymbolTypeEnum.md), [***FieldOneText***] As String, [***FieldTwoText***] As String ) As [AuxiliaryFeatureIndicator](../AuxiliaryFeatureIndicator/AuxiliaryFeatureIndicator.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| SymbolType | [AuxiliaryFeatureIndicatorSymbolTypeEnum](../AuxiliaryFeatureIndicatorSymbolTypeEnum.md) | Input AuxiliaryFeatureIndicatorSymbolTypeEnum value that specifies the symbol type of the auxiliary feature indicator. |
| FieldOneText | String | Input String value that specifies the formatted text for the first text field of auxiliary feature indicator. |
| FieldTwoText | String | Input String value that specifies the formatted text for the second text field of auxiliary feature indicator.   This is an optional argument whose default value is "". |

## Version

Introduced in version 2023
