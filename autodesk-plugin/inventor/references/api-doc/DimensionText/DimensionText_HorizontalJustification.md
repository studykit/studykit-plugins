# DimensionText.HorizontalJustification Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Gets and sets the horizontal justification of the text.

## Syntax

DimensionText.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 9
