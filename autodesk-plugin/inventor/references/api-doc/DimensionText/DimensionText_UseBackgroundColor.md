# DimensionText.UseBackgroundColor Property

Parent Object: [DimensionText](../DimensionText/DimensionText.md)

## Description

Gets and sets the UseBackgroundColor associated with the DimensionText.

## Syntax

DimensionText.**UseBackgroundColor**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
