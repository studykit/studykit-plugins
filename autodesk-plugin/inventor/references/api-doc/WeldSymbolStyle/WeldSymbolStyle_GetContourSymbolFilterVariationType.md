# WeldSymbolStyle.GetContourSymbolFilterVariationType Method

Parent Object: [WeldSymbolStyle](../WeldSymbolStyle/WeldSymbolStyle.md)

## Description

Gets the variation type of a contour symbol filter.

## Syntax

WeldSymbolStyle.**GetContourSymbolFilterVariationType**( ***ContourSymbolFilter*** As [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md) ) As [ContourSymbolVariationTypeEnum](../ContourSymbolVariationTypeEnum.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ContourSymbolFilter | [ContourSymbolTypeEnum](../ContourSymbolTypeEnum.md) | Input ContourSymbolTypeEnum to specify a contour symbol filter. |

## Version

Introduced in version 2024
