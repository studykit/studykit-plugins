# MiniToolbarValueEditor.OnEnter Event

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Event that is fired just before the control gets focus.

## Syntax

MiniToolbarValueEditor.**OnEnter**()

## Version

Introduced in version 2012
