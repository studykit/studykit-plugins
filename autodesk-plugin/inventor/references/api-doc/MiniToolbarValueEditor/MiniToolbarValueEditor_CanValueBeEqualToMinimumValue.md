# MiniToolbarValueEditor.CanValueBeEqualToMinimumValue Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets whether the minimum value is valid value or not.

## Syntax

MiniToolbarValueEditor.**CanValueBeEqualToMinimumValue**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
