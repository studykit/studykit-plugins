# MiniToolbarValueEditor.SetFocus Method

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Method that sets the focus on this control and highlights the current content of the value editor.

## Syntax

MiniToolbarValueEditor.**SetFocus**()

## Version

Introduced in version 2012
