# MiniToolbarValueEditor.UnitsType Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-only property that returns the units type assigned to the control during creation.

## Syntax

MiniToolbarValueEditor.**UnitsType**() As [ValueUnitsTypeEnum](../ValueUnitsTypeEnum.md)

## Property Value

This is a read only property whose value is a [ValueUnitsTypeEnum](../ValueUnitsTypeEnum.md).

## Version

Introduced in version 2012
