# MiniToolbarValueEditor.OnExit Event

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Event that is fired just before the control loses focus.

## Syntax

MiniToolbarValueEditor.**OnExit**()

## Version

Introduced in version 2012
