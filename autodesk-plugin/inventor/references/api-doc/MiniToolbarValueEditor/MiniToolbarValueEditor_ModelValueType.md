# MiniToolbarValueEditor.ModelValueType Property

Parent Object: [MiniToolbarValueEditor](../MiniToolbarValueEditor/MiniToolbarValueEditor.md)

## Description

Read-write property that gets and sets the model value type.

## Syntax

MiniToolbarValueEditor.**ModelValueType**() As [ModelValueTypeEnum](../ModelValueTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ModelValueTypeEnum](../ModelValueTypeEnum.md).

## Version

Introduced in version 2012
