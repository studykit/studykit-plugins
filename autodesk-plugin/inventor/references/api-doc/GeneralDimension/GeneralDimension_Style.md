# GeneralDimension.Style Property

Parent Object: [GeneralDimension](../GeneralDimension/GeneralDimension.md)

## Description

Gets and sets the DimensionStyle associated with the dimension.

## Syntax

GeneralDimension.**Style**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).
