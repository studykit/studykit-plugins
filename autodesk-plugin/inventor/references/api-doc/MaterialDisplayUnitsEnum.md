# MaterialDisplayUnitsEnum Enumerator

## Description

units used when material information is displayed in the user interface.

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kMaterialDisplayUnitsEnglishFoot | 98824 | Values are displayed using ft, lbmass, psf. |

| kMaterialDisplayUnitsEnglishStandard | 98822 | Values are displayed using ksi, psi, BTU. |
| kMaterialDisplayUnitsMetricCGS | 98820 | Values are displayed using cm, g, dyne. |
| kMaterialDisplayUnitsMetricMKS | 98818 | Values are displayed using m, kg, Pa. |
| kMaterialDisplayUnitsMetricMMNS | 98819 | Values are displayed using mm, N, MPa. |
| kMaterialDisplayUnitsMetricStandard | 98817 | Values are displayed using MPa, kg, J. |
| kMaterialDisplayUnitsMetricUMNS | 98821 | Values are displayed using µm, N, N/µm^2. |
