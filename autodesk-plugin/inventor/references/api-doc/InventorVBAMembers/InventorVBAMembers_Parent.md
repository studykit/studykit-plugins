# InventorVBAMembers.Parent Property

Parent Object: [InventorVBAMembers](../InventorVBAMembers/InventorVBAMembers.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

InventorVBAMembers.**Parent**() As [InventorVBAComponent](../InventorVBAComponent/InventorVBAComponent.md)

## Property Value

This is a read only property whose value is an [InventorVBAComponent](../InventorVBAComponent/InventorVBAComponent.md).

## Version

Introduced in version 6
