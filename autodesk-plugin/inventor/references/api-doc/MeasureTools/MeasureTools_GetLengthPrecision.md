# MeasureTools.GetLengthPrecision Method

Parent Object: [MeasureTools](../MeasureTools/MeasureTools.md)

## Description

Get length precision.

## Syntax

MeasureTools.**GetLengthPrecision**( ***Document*** As [Document](../Document/Document.md) ) As Long

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Document | [Document](../Document/Document.md) | Input Document object that gets the measure tools length precision from. Valid input includes PartDocument and AssemblyDocument. |

## Version

Introduced in version 2021
