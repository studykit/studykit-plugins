# MeasureTools.GetAnglePrecision Method

Parent Object: [MeasureTools](../MeasureTools/MeasureTools.md)

## Description

Get angle precision.

## Syntax

MeasureTools.**GetAnglePrecision**( ***Document*** As [Document](../Document/Document.md) ) As Long

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Document | [Document](../Document/Document.md) | Input Document object that gets the measure tools angle precision from. Valid input includes PartDocument and AssemblyDocument. |

## Version

Introduced in version 2021
