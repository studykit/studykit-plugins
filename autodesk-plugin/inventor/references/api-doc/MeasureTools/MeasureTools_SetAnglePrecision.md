# MeasureTools.SetAnglePrecision Method

Parent Object: [MeasureTools](../MeasureTools/MeasureTools.md)

## Description

Set angle precision.

## Syntax

MeasureTools.**SetAnglePrecision**( ***Document*** As [Document](../Document/Document.md), ***Precision*** As Long ) As Long

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Document | [Document](../Document/Document.md) | Input Document object that sets the measure tools angle precision to. Valid input includes PartDocument and AssemblyDocument. |
| Precision | Long | Input Long value that to set the measure tools angle precision. Valid value is from -1 to 8. When specify this to -1 the All Decimals is set. |

## Version

Introduced in version 2021
