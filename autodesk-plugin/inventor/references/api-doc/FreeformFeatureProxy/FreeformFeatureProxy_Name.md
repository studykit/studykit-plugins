# FreeformFeatureProxy.Name Property

Parent Object: [FreeformFeatureProxy](../FreeformFeatureProxy/FreeformFeatureProxy.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

FreeformFeatureProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2015
