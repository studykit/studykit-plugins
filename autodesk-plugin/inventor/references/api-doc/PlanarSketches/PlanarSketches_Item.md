# PlanarSketches.Item Property

Parent Object: [PlanarSketches](../PlanarSketches/PlanarSketches.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PlanarSketches.**Item**( ***Index*** As Variant ) As [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Property Value

This is a read only property whose value is a [PlanarSketch](../PlanarSketch/PlanarSketch.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Value that specifies the index of the object to return. |
