# TangentSketchConstraint.EntityOne Property

Parent Object: [TangentSketchConstraint](../TangentSketchConstraint/TangentSketchConstraint.md)

## Description

Property that returns the sketch entity that is constrained to be tangent to entity two.

## Syntax

TangentSketchConstraint.**EntityOne**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
