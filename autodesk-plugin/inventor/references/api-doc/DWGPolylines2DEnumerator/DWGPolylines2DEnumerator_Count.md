# DWGPolylines2DEnumerator.Count Property

Parent Object: [DWGPolylines2DEnumerator](../DWGPolylines2DEnumerator/DWGPolylines2DEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGPolylines2DEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
