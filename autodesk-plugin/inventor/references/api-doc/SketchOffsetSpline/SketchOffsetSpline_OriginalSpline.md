# SketchOffsetSpline.OriginalSpline Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Returns the original spline that was used as the source to create the offset spline.

## Syntax

SketchOffsetSpline.**OriginalSpline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Property Value

This is a read only property whose value is a [SketchSpline](../SketchSpline/SketchSpline.md).

## Version

Introduced in version 11
