# SketchOffsetSpline.EndSketchPoint Property

Parent Object: [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Description

Property that returns the that defines the position of the end of the spline.

## Syntax

SketchOffsetSpline.**EndSketchPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 9
