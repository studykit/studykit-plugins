# BendOptions.BendTransition Property

Parent Object: [BendOptions](../BendOptions/BendOptions.md)

## Description

Defines the type of bend transition to use for the associated sheet metal feature.

## Syntax

BendOptions.**BendTransition**() As [BendTransitionEnum](../BendTransitionEnum.md)

## Property Value

This is a read/write property whose value is a [BendTransitionEnum](../BendTransitionEnum.md).

## Version

Introduced in version 2009
