# DrawingViewHatchArea.RangeBox Property

Parent Object: [DrawingViewHatchArea](../DrawingViewHatchArea/DrawingViewHatchArea.md)

## Description

Read-only property that returns the range box of the DrawingViewHatchArea.

## Syntax

DrawingViewHatchArea.**RangeBox**() As [Box2d](../Box2d/Box2d.md)

## Property Value

This is a read only property whose value is a [Box2d](../Box2d/Box2d.md).

## Version

Introduced in version 2022
