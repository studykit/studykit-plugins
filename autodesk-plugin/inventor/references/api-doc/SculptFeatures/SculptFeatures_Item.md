# SculptFeatures.Item Property

Parent Object: [SculptFeatures](../SculptFeatures/SculptFeatures.md)

## Description

Method that returns the specified SculptFeature object from the collection.

## Syntax

SculptFeatures.**Item**( ***Index*** As Variant ) As [SculptFeature](../SculptFeature/SculptFeature.md)

## Property Value

This is a read only property whose value is a [SculptFeature](../SculptFeature/SculptFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Value that specifies the feature to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the feature name. If an out of range index or a name of a non-existent feature is provided, an error occurs. |

## Version

Introduced in version 11
