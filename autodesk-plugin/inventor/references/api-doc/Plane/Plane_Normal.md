# Plane.Normal Property

Parent Object: [Plane](../Plane/Plane.md)

## Description

Property that indicates the vector for a specific normal in the set.

## Syntax

Plane.**Normal**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).
