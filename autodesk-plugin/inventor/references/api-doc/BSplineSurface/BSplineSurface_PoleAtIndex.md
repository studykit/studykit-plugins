# BSplineSurface.PoleAtIndex Property

Parent Object: [BSplineSurface](../BSplineSurface/BSplineSurface.md)

## Description

Indicates the pole coordinate point at the specified UV indices into the spline's pole array.

## Syntax

BSplineSurface.**PoleAtIndex**( ***IndexU*** As Long, ***IndexV*** As Long ) As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).
