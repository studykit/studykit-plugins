# FilletFullRoundSet Object

## Description

The FilletFullRoundSet object provides access to the inputs used to create a full round fillet feature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FilletFullRoundSet/FilletFullRoundSet_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [CenterFaces](../FilletFullRoundSet/FilletFullRoundSet_CenterFaces.md) | Property that returns the center faces for the full round fillet feature. |
| [IncludeTangentFaces](../FilletFullRoundSet/FilletFullRoundSet_IncludeTangentFaces.md) | Property that returns whether to include faces that are tangential to the faces specified in the SideFacesOne, CenterFaces and SideFacesTwo properties for the fillet creation. |
| [SideFacesOne](../FilletFullRoundSet/FilletFullRoundSet_SideFacesOne.md) | Property that returns the first set of side faces for the full round fillet feature. |
| [SideFacesTwo](../FilletFullRoundSet/FilletFullRoundSet_SideFacesTwo.md) | Property that returns the second set of side faces for the full round fillet feature. |
| [Type](../FilletFullRoundSet/FilletFullRoundSet_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FilletDefinition.AddFullRoundSet](../FilletDefinition/FilletDefinition_AddFullRoundSet.md), [FilletDefinition.FullRoundSet](../FilletDefinition/FilletDefinition_FullRoundSet.md)

## Version

Introduced in version 11
