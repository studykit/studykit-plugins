# FilletFullRoundSet.IncludeTangentFaces Property

Parent Object: [FilletFullRoundSet](../FilletFullRoundSet/FilletFullRoundSet.md)

## Description

Property that returns whether to include faces that are tangential to the faces specified in the SideFacesOne, CenterFaces and SideFacesTwo properties for the fillet creation.

## Syntax

FilletFullRoundSet.**IncludeTangentFaces**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
