# ThreadFeatureProxy.ThreadInfoType Property

Parent Object: [ThreadFeatureProxy](../ThreadFeatureProxy/ThreadFeatureProxy.md)

## Description

Property that returns a indicating the thread type.

## Syntax

ThreadFeatureProxy.**ThreadInfoType**() As [ThreadTypeEnum](../ThreadTypeEnum.md)

## Property Value

This is a read only property whose value is a [ThreadTypeEnum](../ThreadTypeEnum.md).

## Version

Introduced in version 9
