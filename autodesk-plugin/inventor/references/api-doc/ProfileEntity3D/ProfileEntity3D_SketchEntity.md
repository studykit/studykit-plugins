# ProfileEntity3D.SketchEntity Property

Parent Object: [ProfileEntity3D](../ProfileEntity3D/ProfileEntity3D.md)

## Description

Property that gets the sketch entity this profile entity was derived from.

## Syntax

ProfileEntity3D.**SketchEntity**() As [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md)

## Property Value

This is a read only property whose value is a [SketchEntity3D](../SketchEntity3D/SketchEntity3D.md).

## Version

Introduced in version 6
