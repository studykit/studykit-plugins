# OffsetSplineDimConstraint.OffsetSpline Property

Parent Object: [OffsetSplineDimConstraint](../OffsetSplineDimConstraint/OffsetSplineDimConstraint.md)

## Description

Gets the SketchOffsetSpline being constrained to.

## Syntax

OffsetSplineDimConstraint.**OffsetSpline**() As [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Property Value

This is a read only property whose value is a [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md).

## Version

Introduced in version 11
