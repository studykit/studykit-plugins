# iAssemblyFactory.TableColumns Property

Parent Object: [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Description

Property that returns the iAssemblyTableColumns collection object.

## Syntax

iAssemblyFactory.**TableColumns**() As [iAssemblyTableColumns](../iAssemblyTableColumns/iAssemblyTableColumns.md)

## Property Value

This is a read only property whose value is an [iAssemblyTableColumns](../iAssemblyTableColumns/iAssemblyTableColumns.md).

## Version

Introduced in version 11
