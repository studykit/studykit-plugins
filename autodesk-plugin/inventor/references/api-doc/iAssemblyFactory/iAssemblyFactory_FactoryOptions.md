# iAssemblyFactory.FactoryOptions Property

Parent Object: [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Description

Property that gets the options for the factory.

## Syntax

iAssemblyFactory.**FactoryOptions**() As [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Property Value

This is a read only property whose value is a [FactoryOptions](../FactoryOptions/FactoryOptions.md).

## Version

Introduced in version 2010
