# iAssemblyFactory.DefaultRow Property

Parent Object: [iAssemblyFactory](../iAssemblyFactory/iAssemblyFactory.md)

## Description

Read-write property that gets and sets the default row for this iAssemblyFactory.

## Syntax

iAssemblyFactory.**DefaultRow**() As [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md)

## Property Value

This is a read/write property whose value is an [iAssemblyTableRow](../iAssemblyTableRow/iAssemblyTableRow.md).

## Version

Introduced in version 11
