# ContentTreeViewNode.Icon Property

Parent Object: [ContentTreeViewNode](../ContentTreeViewNode/ContentTreeViewNode.md)

## Description

Property that returns the icon used for this node.

## Syntax

ContentTreeViewNode.**Icon**() As IPictureDisp

## Property Value

This is a read only property whose value is an IPictureDisp.

## Version

Introduced in version 2010
