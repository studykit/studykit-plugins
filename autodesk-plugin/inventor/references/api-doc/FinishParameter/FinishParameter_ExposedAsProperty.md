# FinishParameter.ExposedAsProperty Property

Parent Object: [FinishParameter](../FinishParameter/FinishParameter.md)

## Description

Gets/(Sets) the Boolean indicating if this parameter is exposed as a property of this Document.

## Syntax

FinishParameter.**ExposedAsProperty**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
