# FinishParameter.Parent Property

Parent Object: [FinishParameter](../FinishParameter/FinishParameter.md)

## Description

Property that returns the parent Document of this parameter.

## Syntax

FinishParameter.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2024
