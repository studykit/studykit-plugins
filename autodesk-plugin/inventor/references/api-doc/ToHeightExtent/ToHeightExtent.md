# ToHeightExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The ToHeightExtent is used by the FlangeFeature object when the height extent type is defined as a 'To' extent.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ToHeightExtent/ToHeightExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Offset](../ToHeightExtent/ToHeightExtent_Offset.md) | Property that returns the parameter controlling the offset distance of the flange relative to the 'To' point. |
| [Parent](../ToHeightExtent/ToHeightExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [ToEntity](../ToHeightExtent/ToHeightExtent_ToEntity.md) | Property that gets and sets the entity defining the 'To' extent. |
| [Type](../ToHeightExtent/ToHeightExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2009
