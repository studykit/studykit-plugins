# FaceShellDefinition.FaceDefinitions Property

Parent Object: [FaceShellDefinition](../FaceShellDefinition/FaceShellDefinition.md)

## Description

Property that returns the collection of FaceDefinition objects associated with this FaceShellDefinition object.

## Syntax

FaceShellDefinition.**FaceDefinitions**() As [FaceDefinitions](../FaceDefinitions/FaceDefinitions.md)

## Property Value

This is a read only property whose value is a [FaceDefinitions](../FaceDefinitions/FaceDefinitions.md).
