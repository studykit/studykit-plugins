# Faces.Item Property

Parent Object: [Faces](../Faces/Faces.md)

## Description

Allows integer-indexed access to objects in the collection.

## Syntax

Faces.**Item**( ***Index*** As Long ) As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |
