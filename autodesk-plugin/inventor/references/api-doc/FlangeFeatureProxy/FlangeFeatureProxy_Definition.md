# FlangeFeatureProxy.Definition Property

Parent Object: [FlangeFeatureProxy](../FlangeFeatureProxy/FlangeFeatureProxy.md)

## Description

Read-only property that returns the FlangeDefinition object that provides access to all of the various settings that define this flange feature.

## Syntax

FlangeFeatureProxy.**Definition**() As [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md)

## Property Value

This is a read/write property whose value is a [FlangeDefinition](../FlangeDefinition/FlangeDefinition.md).

## Version

Introduced in version 2009
