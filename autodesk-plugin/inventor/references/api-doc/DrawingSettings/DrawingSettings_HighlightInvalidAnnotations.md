# DrawingSettings.HighlightInvalidAnnotations Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets the state of the invalid annotation highlighting setting.

## Syntax

DrawingSettings.**HighlightInvalidAnnotations**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 8
