# DrawingSettings.Parent Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Property that returns the parent DrawingDocument object.

## Syntax

DrawingSettings.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 7
