# DrawingSettings.CrossHatchClipping Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets whether cross hatches should be clipped about drawing annotations.

## Syntax

DrawingSettings.**CrossHatchClipping**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
