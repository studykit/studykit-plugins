# DrawingSettings.InvalidFeatureBasedAnnotationsColor Property

Parent Object: [DrawingSettings](../DrawingSettings/DrawingSettings.md)

## Description

Gets and sets the color for invalid feature-based annotations.

## Syntax

DrawingSettings.**InvalidFeatureBasedAnnotationsColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2008
