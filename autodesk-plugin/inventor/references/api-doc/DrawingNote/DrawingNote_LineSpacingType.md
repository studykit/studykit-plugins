# DrawingNote.LineSpacingType Property

Parent Object: [DrawingNote](../DrawingNote/DrawingNote.md)

## Description

Gets/Sets the LineSpacingType used by the DrawingNote.

## Syntax

DrawingNote.**LineSpacingType**() As [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md)

## Property Value

This is a read/write property whose value is a [TextLineSpacingTypeEnum](../TextLineSpacingTypeEnum.md).

## Version

Introduced in version 10
