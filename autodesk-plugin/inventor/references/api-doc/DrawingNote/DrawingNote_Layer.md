# DrawingNote.Layer Property

Parent Object: [DrawingNote](../DrawingNote/DrawingNote.md)

## Description

Gets/Sets the layer used by the DrawingNote.

## Syntax

DrawingNote.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
