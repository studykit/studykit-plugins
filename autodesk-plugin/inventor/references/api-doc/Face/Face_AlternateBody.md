# Face.AlternateBody Property

Parent Object: [Face](../Face/Face.md)

## Description

Property that returns the alternate SurfaceBody.

## Syntax

Face.**AlternateBody**( ***AlternateForm*** As Long ) As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| AlternateForm | Long | Input Long that specifies the requested alternate form as a combination of values. |

## Version

Introduced in version 4
