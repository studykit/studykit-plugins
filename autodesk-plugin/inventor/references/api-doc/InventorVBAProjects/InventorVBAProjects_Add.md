# InventorVBAProjects.Add Method

Parent Object: [InventorVBAProjects](../InventorVBAProjects/InventorVBAProjects.md)

## Description

Method that creates a new Autodesk Inventor VBA project. Please note that this method works only for user-defined VBA projects.

## Syntax

InventorVBAProjects.**Add**() As [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Version

Introduced in version 6
