# InventorVBAProjects Object

## Description

A collection of Autodesk Inventor's VBA projects ( objects).

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../InventorVBAProjects/InventorVBAProjects_Add.md) | Method that creates a new Autodesk Inventor VBA project. Please note that this method works only for user-defined VBA projects. |
| [Open](../InventorVBAProjects/InventorVBAProjects_Open.md) | Method that opens the specified VBA project. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../InventorVBAProjects/InventorVBAProjects_Count.md) | Property that returns the number of items in the collection. |
| [Item](../InventorVBAProjects/InventorVBAProjects_Item.md) | Returns the specified InventorVBAProject object from the collection. |
| [Parent](../InventorVBAProjects/InventorVBAProjects_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../InventorVBAProjects/InventorVBAProjects_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Application.VBAProjects](../Application/Application_VBAProjects.md), [InventorServer.VBAProjects](InventorServer_VBAProjects.md), [InventorServerObject.VBAProjects](InventorServerObject_VBAProjects.md)

## Version

Introduced in version 6
