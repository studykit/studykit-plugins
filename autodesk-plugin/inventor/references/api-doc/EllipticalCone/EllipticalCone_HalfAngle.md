# EllipticalCone.HalfAngle Property

Parent Object: [EllipticalCone](../EllipticalCone/EllipticalCone.md)

## Description

Specifies the taper angle of the elliptical cone. This is the angle measured from the axis to the side of the elliptical cone.

## Syntax

EllipticalCone.**HalfAngle**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 4
