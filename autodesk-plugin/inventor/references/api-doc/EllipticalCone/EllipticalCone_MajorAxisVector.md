# EllipticalCone.MajorAxisVector Property

Parent Object: [EllipticalCone](../EllipticalCone/EllipticalCone.md)

## Description

Specifies the direction of the major axis of the elliptical cone.

## Syntax

EllipticalCone.**MajorAxisVector**() As [Vector](../Vector/Vector.md)

## Property Value

This is a read/write property whose value is a [Vector](../Vector/Vector.md).

## Version

Introduced in version 4
