# EllipticalCone.BasePoint Property

Parent Object: [EllipticalCone](../EllipticalCone/EllipticalCone.md)

## Description

Specifies the base of the elliptical cone.

## Syntax

EllipticalCone.**BasePoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
