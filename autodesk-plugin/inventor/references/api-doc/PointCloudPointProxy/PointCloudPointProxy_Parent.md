# PointCloudPointProxy.Parent Property

Parent Object: [PointCloudPointProxy](../PointCloudPointProxy/PointCloudPointProxy.md)

## Description

Read-only property that returns the parent point cloud of this point.

## Syntax

PointCloudPointProxy.**Parent**() As [PointCloud](../PointCloud/PointCloud.md)

## Property Value

This is a read only property whose value is a [PointCloud](../PointCloud/PointCloud.md).

## Version

Introduced in version 2016
