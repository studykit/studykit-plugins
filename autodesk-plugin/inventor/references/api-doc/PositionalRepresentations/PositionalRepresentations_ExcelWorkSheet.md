# PositionalRepresentations.ExcelWorkSheet Property

Parent Object: [PositionalRepresentations](../PositionalRepresentations/PositionalRepresentations.md)

## Description

Property that returns the Excel spreadsheet that contains the positional representations data.

## Syntax

PositionalRepresentations.**ExcelWorkSheet**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
