# AutoCADBlock.GetCustomLineType Method

Parent Object: [AutoCADBlock](../AutoCADBlock/AutoCADBlock.md)

## Description

Method that returns information regarding the custom line type in use. The method returns a failure if the return value of the LineType property is not kCustomLineType.

## Syntax

AutoCADBlock.**GetCustomLineType**( ***LineTypeName*** As String, ***LineTypeDescription*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| LineTypeName | String | String that returns the name of the line type. |
| LineTypeDescription | String | String that returns the description of the line type. |

## Version

Introduced in version 2011
