# ApplicationAddIn.LicenseStatus Property

Parent Object: [ApplicationAddIn](../ApplicationAddIn/ApplicationAddIn.md)

## Description

Gets the license status of the AddIn.

## Syntax

ApplicationAddIn.**LicenseStatus**() As [AddInLicenseStatusEnum](../AddInLicenseStatusEnum.md)

## Property Value

This is a read only property whose value is an [AddInLicenseStatusEnum](../AddInLicenseStatusEnum.md).

## Version

Introduced in version 2009
