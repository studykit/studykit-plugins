# GeneralOptions.UseNegativeIntegralForInertialProperties Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Gets/Sets whether the inertial properties should be calculated using the negative integral.

## Syntax

GeneralOptions.**UseNegativeIntegralForInertialProperties**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
