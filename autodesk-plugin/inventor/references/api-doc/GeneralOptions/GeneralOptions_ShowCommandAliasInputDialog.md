# GeneralOptions.ShowCommandAliasInputDialog Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Gets and sets whether the input dialog should be displayed for command aliases.

## Syntax

GeneralOptions.**ShowCommandAliasInputDialog**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
