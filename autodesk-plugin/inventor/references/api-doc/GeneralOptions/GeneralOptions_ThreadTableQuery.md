# GeneralOptions.ThreadTableQuery Property

Parent Object: [GeneralOptions](../GeneralOptions/GeneralOptions.md)

## Description

Property that returns the ThreadTableQuery object. This object has methods to query the thread table data contained in the Thread.xls spreadsheet.

## Syntax

GeneralOptions.**ThreadTableQuery**() As [ThreadTableQuery](../ThreadTableQuery/ThreadTableQuery.md)

## Property Value

This is a read only property whose value is a [ThreadTableQuery](../ThreadTableQuery/ThreadTableQuery.md).
