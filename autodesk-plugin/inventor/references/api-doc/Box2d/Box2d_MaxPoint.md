# Box2d.MaxPoint Property

Parent Object: [Box2d](../Box2d/Box2d.md)

## Description

Property that gets and sets the maximum corner of the box.

## Syntax

Box2d.**MaxPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).
