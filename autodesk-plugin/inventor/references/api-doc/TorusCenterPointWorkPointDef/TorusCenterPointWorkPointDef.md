# TorusCenterPointWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point defined to be at the center of a torus.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../TorusCenterPointWorkPointDef/TorusCenterPointWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Face](../TorusCenterPointWorkPointDef/TorusCenterPointWorkPointDef_Face.md) | Property that indicates the Face object whose geometry is a torus surface, the center point of which defines the position of the work point. |
| [Parent](../TorusCenterPointWorkPointDef/TorusCenterPointWorkPointDef_Parent.md) | Property that returns the parent WorkPoint object. |
| [Type](../TorusCenterPointWorkPointDef/TorusCenterPointWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2008
