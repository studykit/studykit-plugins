# DisplaySettings.AreTexturesOn Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets whether to turn textures on.

## Syntax

DisplaySettings.**AreTexturesOn**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
