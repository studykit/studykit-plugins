# DisplaySettings.HiddenLineDimmingPercent Property

Parent Object: [DisplaySettings](../DisplaySettings/DisplaySettings.md)

## Description

Gets and sets the dimming factor (expressed as a percentage) applied to the display of hidden lines in the model.

## Syntax

DisplaySettings.**HiddenLineDimmingPercent**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2011
