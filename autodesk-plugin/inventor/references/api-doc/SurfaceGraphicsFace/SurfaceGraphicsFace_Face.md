# SurfaceGraphicsFace.Face Property

Parent Object: [SurfaceGraphicsFace](../SurfaceGraphicsFace/SurfaceGraphicsFace.md)

## Description

Property that returns the Face object associated with the SurfaceGraphicsFace.

## Syntax

SurfaceGraphicsFace.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2009
