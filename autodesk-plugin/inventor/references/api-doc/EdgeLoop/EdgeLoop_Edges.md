# EdgeLoop.Edges Property

Parent Object: [EdgeLoop](../EdgeLoop/EdgeLoop.md)

## Description

Gets the referenced by this EdgeLoop.

## Syntax

EdgeLoop.**Edges**() As [Edges](../Edges/Edges.md)

## Property Value

This is a read only property whose value is an [Edges](../Edges/Edges.md).

## Version

Introduced in version 4
