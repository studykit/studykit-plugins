# GraphicsNodeProxy.AddTriangleGraphics Method

Parent Object: [GraphicsNodeProxy](../GraphicsNodeProxy/GraphicsNodeProxy.md)

## Description

Method that creates a new TriangleGraphics graphic primitive.

## Syntax

GraphicsNodeProxy.**AddTriangleGraphics**() As [TriangleGraphics](../TriangleGraphics/TriangleGraphics.md)

## Version

Introduced in version 2008
