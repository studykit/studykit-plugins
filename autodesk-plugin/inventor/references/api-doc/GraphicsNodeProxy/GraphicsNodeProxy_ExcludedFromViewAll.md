# GraphicsNodeProxy.ExcludedFromViewAll Property

Parent Object: [GraphicsNodeProxy](../GraphicsNodeProxy/GraphicsNodeProxy.md)

## Description

Specifies that the node is not considered when doing a view all.

## Syntax

GraphicsNodeProxy.**ExcludedFromViewAll**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
