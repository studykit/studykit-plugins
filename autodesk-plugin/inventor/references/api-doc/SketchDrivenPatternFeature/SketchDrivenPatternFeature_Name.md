# SketchDrivenPatternFeature.Name Property

Parent Object: [SketchDrivenPatternFeature](../SketchDrivenPatternFeature/SketchDrivenPatternFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

SketchDrivenPatternFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2017
