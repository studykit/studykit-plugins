# ComponentGraphics.LineScale Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Read-write property that gets and sets the line scale applied to this primitive.

## Syntax

ComponentGraphics.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
