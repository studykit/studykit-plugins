# ComponentGraphics.DisplaySilhouettes Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Read-write property that specifies whether or not to display the silhouette edges of bodies. The property defaults to True when the ComponentGraphics object is created.

## Syntax

ComponentGraphics.**DisplaySilhouettes**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
