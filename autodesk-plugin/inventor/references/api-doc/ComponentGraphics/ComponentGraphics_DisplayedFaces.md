# ComponentGraphics.DisplayedFaces Property

Parent Object: [ComponentGraphics](../ComponentGraphics/ComponentGraphics.md)

## Description

Read-only property that returns a SurfaceGraphicsFaceList object. This list provides access to all faces that are currently displayed.

## Syntax

ComponentGraphics.**DisplayedFaces**() As [SurfaceGraphicsFaceList](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList.md)

## Property Value

This is a read only property whose value is a [SurfaceGraphicsFaceList](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList.md).

## Version

Introduced in version 2011
