# WireDefinitions.Item Property

Parent Object: [WireDefinitions](../WireDefinitions/WireDefinitions.md)

## Description

Returns the specified WireDefinition object from the collection

## Syntax

WireDefinitions.**Item**( ***Index*** As Long ) As [WireDefinition](../WireDefinition/WireDefinition.md)

## Property Value

This is a read only property whose value is a [WireDefinition](../WireDefinition/WireDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the WireDefinition object within the collection to return. |

## Version

Introduced in version 2013
