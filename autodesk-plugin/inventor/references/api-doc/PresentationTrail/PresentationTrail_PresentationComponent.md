# PresentationTrail.PresentationComponent Property

Parent Object: [PresentationTrail](../PresentationTrail/PresentationTrail.md)

## Description

Read-only property that returns the PresentationComponent object this trail is associative with.

## Syntax

PresentationTrail.**PresentationComponent**() As [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Property Value

This is a read only property whose value is a [PresentationComponent](../PresentationComponent/PresentationComponent.md).

## Version

Introduced in version 2018
