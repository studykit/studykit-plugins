# PresentationTrail.Segments Property

Parent Object: [PresentationTrail](../PresentationTrail/PresentationTrail.md)

## Description

Read-only property that returns the PresentationTrailSegments collection object.

## Syntax

PresentationTrail.**Segments**() As [PresentationTrailSegments](../PresentationTrailSegments/PresentationTrailSegments.md)

## Property Value

This is a read only property whose value is a [PresentationTrailSegments](../PresentationTrailSegments/PresentationTrailSegments.md).

## Version

Introduced in version 2018
