# FromToExtent.Direction Property

Parent Object: [FromToExtent](../FromToExtent/FromToExtent.md)

## Description

Gets and sets the direction of the feature.

## Syntax

FromToExtent.**Direction**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2008
