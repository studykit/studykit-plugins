# EqualLengthConstraintProxy Object

Derived from: [EqualLengthConstraint](../EqualLengthConstraint/EqualLengthConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_Deletable.md) | Indicates whether this object is deletable. |
| [LineOne](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_LineOne.md) | Property that returns the sketch line being constrained. |
| [LineTwo](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_LineTwo.md) | Property that returns the sketch line being constrained. |
| [NativeObject](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Type](../EqualLengthConstraintProxy/EqualLengthConstraintProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
