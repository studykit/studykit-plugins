# EqualLengthConstraintProxy.LineOne Property

Parent Object: [EqualLengthConstraintProxy](../EqualLengthConstraintProxy/EqualLengthConstraintProxy.md)

## Description

Property that returns the sketch line being constrained.

## Syntax

EqualLengthConstraintProxy.**LineOne**() As [SketchLine](../SketchLine/SketchLine.md)

## Property Value

This is a read only property whose value is a [SketchLine](../SketchLine/SketchLine.md).

## Version

Introduced in version 6
