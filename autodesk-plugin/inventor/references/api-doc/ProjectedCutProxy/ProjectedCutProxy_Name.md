# ProjectedCutProxy.Name Property

Parent Object: [ProjectedCutProxy](../ProjectedCutProxy/ProjectedCutProxy.md)

## Description

Gets and sets name of the projected cut edges.

## Syntax

ProjectedCutProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
