# ProjectedCutProxy.SketchEntities Property

Parent Object: [ProjectedCutProxy](../ProjectedCutProxy/ProjectedCutProxy.md)

## Description

Property that returns a collection of sketch entities that belong to the projected cut edges. The sketch entities returned by this property cannot be edited or deleted because they are associated with the projected edges in the model.

## Syntax

ProjectedCutProxy.**SketchEntities**() As [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntitiesEnumerator](../SketchEntitiesEnumerator/SketchEntitiesEnumerator.md).

## Version

Introduced in version 2010
