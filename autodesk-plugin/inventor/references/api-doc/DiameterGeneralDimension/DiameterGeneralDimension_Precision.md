# DiameterGeneralDimension.Precision Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Gets and sets the Precision setting.

## Syntax

DiameterGeneralDimension.**Precision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 11
