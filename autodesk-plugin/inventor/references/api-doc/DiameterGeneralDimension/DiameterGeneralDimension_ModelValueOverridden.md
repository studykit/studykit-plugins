# DiameterGeneralDimension.ModelValueOverridden Property

Parent Object: [DiameterGeneralDimension](../DiameterGeneralDimension/DiameterGeneralDimension.md)

## Description

Read-write property that gets and sets whether the model value is overridden for the dimension.

## Syntax

DiameterGeneralDimension.**ModelValueOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
