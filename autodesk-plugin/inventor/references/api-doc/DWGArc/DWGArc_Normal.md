# DWGArc.Normal Property

Parent Object: [DWGArc](../DWGArc/DWGArc.md)

## Description

Read-only property that returns the UnitVector indicating the normal of the arc.

## Syntax

DWGArc.**Normal**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2016
