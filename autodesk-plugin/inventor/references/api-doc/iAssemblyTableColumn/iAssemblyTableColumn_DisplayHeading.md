# iAssemblyTableColumn.DisplayHeading Property

Parent Object: [iAssemblyTableColumn](../iAssemblyTableColumn/iAssemblyTableColumn.md)

## Description

Property that returns the localized heading of the column.

## Syntax

iAssemblyTableColumn.**DisplayHeading**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 11
