# MarkFeatureProxy.Suppressed Property

Parent Object: [MarkFeatureProxy](../MarkFeatureProxy/MarkFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

MarkFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
