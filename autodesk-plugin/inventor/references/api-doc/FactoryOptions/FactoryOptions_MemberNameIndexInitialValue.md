# FactoryOptions.MemberNameIndexInitialValue Property

Parent Object: [FactoryOptions](../FactoryOptions/FactoryOptions.md)

## Description

Gets and sets the starting value of the member name index.

## Syntax

FactoryOptions.**MemberNameIndexInitialValue**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
