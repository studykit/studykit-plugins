# ExtendFeatureProxy.NativeObject Property

Parent Object: [ExtendFeatureProxy](../ExtendFeatureProxy/ExtendFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ExtendFeatureProxy.**NativeObject**() As [ExtendFeature](../ExtendFeature/ExtendFeature.md)

## Property Value

This is a read only property whose value is an [ExtendFeature](../ExtendFeature/ExtendFeature.md).

## Version

Introduced in version 11
