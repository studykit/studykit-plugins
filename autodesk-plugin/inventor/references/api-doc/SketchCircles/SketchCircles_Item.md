# SketchCircles.Item Property

Parent Object: [SketchCircles](../SketchCircles/SketchCircles.md)

## Description

Returns the specified SketchCircle object from the collection.

## Syntax

SketchCircles.**Item**( ***Index*** As Long ) As [SketchCircle](../SketchCircle/SketchCircle.md)

## Property Value

This is a read only property whose value is a [SketchCircle](../SketchCircle/SketchCircle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 5
