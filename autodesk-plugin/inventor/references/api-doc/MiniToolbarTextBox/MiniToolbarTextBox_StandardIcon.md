# MiniToolbarTextBox.StandardIcon Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Read-write property that gets and sets the standard (small) icon for the control.

## Syntax

MiniToolbarTextBox.**StandardIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2018
