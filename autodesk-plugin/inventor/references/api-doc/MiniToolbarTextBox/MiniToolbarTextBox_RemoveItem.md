# MiniToolbarTextBox.RemoveItem Method

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Removes the specified item from the control.

## Syntax

MiniToolbarTextBox.**RemoveItem**( ***Index*** As Variant )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the item or the separator to be removed. This is a numeric value indicating the (1-based) index of the item/separator in the drop down. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
