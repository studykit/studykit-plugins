# MiniToolbarTextBox.OnItemRemove Event

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Event that is fired when the user removes an item from the drop down.

## Syntax

MiniToolbarTextBox.**OnItemRemove**()

## Version

Introduced in version 2018
