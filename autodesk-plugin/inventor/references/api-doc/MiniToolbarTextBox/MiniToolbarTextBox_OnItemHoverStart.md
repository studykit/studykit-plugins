# MiniToolbarTextBox.OnItemHoverStart Event

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Event that is fired when the user starts to hover over an item in the drop down in the expanded state.

## Syntax

MiniToolbarTextBox.**OnItemHoverStart**()

## Version

Introduced in version 2018
