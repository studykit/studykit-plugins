# MiniToolbarTextBox.OnEnter Event

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Event that is fired just before the control gets focus.

## Syntax

MiniToolbarTextBox.**OnEnter**()

## Version

Introduced in version 2018
