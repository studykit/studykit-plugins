# MiniToolbarTextBox.OnExit Event

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Event that is fired just before the control loses focus.

## Syntax

MiniToolbarTextBox.**OnExit**()

## Version

Introduced in version 2018
