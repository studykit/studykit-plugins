# MiniToolbarTextBox.SelectionStartPosition Property

Parent Object: [MiniToolbarTextBox](../MiniToolbarTextBox/MiniToolbarTextBox.md)

## Description

Returns the position of the selected characters start position in the text box.

## Syntax

MiniToolbarTextBox.**SelectionStartPosition**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2018
