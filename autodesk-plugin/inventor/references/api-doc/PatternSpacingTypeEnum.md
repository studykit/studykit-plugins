# PatternSpacingTypeEnum Enumerator

## Description

Constants identifying the pattern spacing type.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDefault | 33537 | Occurrences separated by specified distance. |
| kFitted | 33538 | All occurrences fitted within specified distance. |
| kFitToPathLength | 33539 | All occurrences fitted within path length. |
