# PointCloudScan.Origin Property

Parent Object: [PointCloudScan](../PointCloudScan/PointCloudScan.md)

## Description

Read-only property that returns a point indicating the origin of the PointCloudScan.

## Syntax

PointCloudScan.**Origin**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2016
