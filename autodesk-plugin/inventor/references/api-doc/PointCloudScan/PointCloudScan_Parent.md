# PointCloudScan.Parent Property

Parent Object: [PointCloudScan](../PointCloudScan/PointCloudScan.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

PointCloudScan.**Parent**() As [PointCloud](../PointCloud/PointCloud.md)

## Property Value

This is a read only property whose value is a [PointCloud](../PointCloud/PointCloud.md).

## Version

Introduced in version 2016
