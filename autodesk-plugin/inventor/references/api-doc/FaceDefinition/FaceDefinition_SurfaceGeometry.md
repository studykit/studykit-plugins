# FaceDefinition.SurfaceGeometry Property

Parent Object: [FaceDefinition](../FaceDefinition/FaceDefinition.md)

## Description

Gets the collection of EdgeLoopDefinition objects associated with this FaceDefinition object.

## Syntax

FaceDefinition.**SurfaceGeometry**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2011
