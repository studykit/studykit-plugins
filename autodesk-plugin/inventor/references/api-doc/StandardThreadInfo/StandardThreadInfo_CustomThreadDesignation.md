# StandardThreadInfo.CustomThreadDesignation Property

Parent Object: [StandardThreadInfo](../StandardThreadInfo/StandardThreadInfo.md)

## Description

Indicates the custom thread designation from the Custom Thread Designation column of the thread data spreadsheet.

## Syntax

StandardThreadInfo.**CustomThreadDesignation**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 9
