# StandardThreadInfo.Internal Property

Parent Object: [StandardThreadInfo](../StandardThreadInfo/StandardThreadInfo.md)

## Description

Gets and sets whether this is an internal thread or external (False).

## Syntax

StandardThreadInfo.**Internal**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
