# SketchLine3DProxy.Type Property

Parent Object: [SketchLine3DProxy](../SketchLine3DProxy/SketchLine3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchLine3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
