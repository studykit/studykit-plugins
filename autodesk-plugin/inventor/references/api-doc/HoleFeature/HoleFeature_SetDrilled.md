# HoleFeature.SetDrilled Method

Parent Object: [HoleFeature](../HoleFeature/HoleFeature.md)

## Description

Method that changes the hole to be a drilled hole type.

## Syntax

HoleFeature.**SetDrilled**()

## Version

Introduced in version 5
