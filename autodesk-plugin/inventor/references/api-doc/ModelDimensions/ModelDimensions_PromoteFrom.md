# ModelDimensions.PromoteFrom Method

Parent Object: [ModelDimensions](../ModelDimensions/ModelDimensions.md)

## Description

Method that promotes a dimension constraint to ModelDimension.

## Syntax

ModelDimensions.**PromoteFrom**( ***pDimConstr*** As Object ) As [ModelDimension](../ModelDimension/ModelDimension.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| pDimConstr | Object | Input dimension constraint object to be promoted as model dimension. Valid object is DimensionConstraint. |

## Version

Introduced in version 2023
