# RadiusModelDimensionDefinition.IsArrowheadInside Property

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Read-write property that gets and sets whether the arrowhead is inside or not.

## Syntax

RadiusModelDimensionDefinition.**IsArrowheadInside**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
