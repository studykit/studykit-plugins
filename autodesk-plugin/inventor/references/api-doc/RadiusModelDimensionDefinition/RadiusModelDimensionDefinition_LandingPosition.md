# RadiusModelDimensionDefinition.LandingPosition Property

Parent Object: [RadiusModelDimensionDefinition](../RadiusModelDimensionDefinition/RadiusModelDimensionDefinition.md)

## Description

Read-write property that gets and sets the landing position of the dimension. The point is projected onto the orientation plane.

## Syntax

RadiusModelDimensionDefinition.**LandingPosition**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
