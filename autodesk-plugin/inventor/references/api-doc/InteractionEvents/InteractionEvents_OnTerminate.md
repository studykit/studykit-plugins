# InteractionEvents.OnTerminate Event

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Event that notifies the command has been terminated. This can happen when the user selects another command or presses escape. It can be forced by calling the Stop method on the InteractionEvents object as well.

## Syntax

InteractionEvents.**OnTerminate**()
