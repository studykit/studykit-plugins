# InteractionEvents.OnSuspend Event

Parent Object: [InteractionEvents](../InteractionEvents/InteractionEvents.md)

## Description

Event that notifies the InteractionEvents object to temporarily suspend itself. This happens when the user selects a command that stacks on the current command. Invocation of any of the view commands will cause the stacking behavior.

## Syntax

InteractionEvents.**OnSuspend**()

## Version

Introduced in version 5
