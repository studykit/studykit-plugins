# DrawingStandardStyle.GetAvailableStyles Method

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Method that returns the ObjectCollection containing the styles which are available for the standard style.

## Syntax

DrawingStandardStyle.**GetAvailableStyles**( ***StyleType*** As [StyleTypeEnum](../StyleTypeEnum.md) ) As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| StyleType | [StyleTypeEnum](../StyleTypeEnum.md) | Input StyleTypeEnum value that specifies style type to get the available styles for the standard style. |

## Version

Introduced in version 2026
