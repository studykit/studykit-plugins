# DrawingStandardStyle.ExcludeCharacters Property

Parent Object: [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Description

Gets and sets comma separated list of characters to be excluded from automatic alphabetical indexing.

## Syntax

DrawingStandardStyle.**ExcludeCharacters**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
