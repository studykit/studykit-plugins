# DSLoadDefinition.DirectionEntity Property

Parent Object: [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Description

Gets and sets Entity that defines the direction of the load. Valid geometry includes planar and cylindrical faces, and linear edges.

## Syntax

DSLoadDefinition.**DirectionEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2013
