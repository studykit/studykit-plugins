# DSLoadDefinition.IsDisplayed Property

Parent Object: [DSLoadDefinition](../DSLoadDefinition/DSLoadDefinition.md)

## Description

Gets and sets if the arrow for the force or torque is displayed during the playback of the simulation.

## Syntax

DSLoadDefinition.**IsDisplayed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
