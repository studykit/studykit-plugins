# iMateResultProxy.NativeObject Property

Parent Object: [iMateResultProxy](../iMateResultProxy/iMateResultProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

iMateResultProxy.**NativeObject**() As [iMateResult](../iMateResult/iMateResult.md)

## Property Value

This is a read only property whose value is an [iMateResult](../iMateResult/iMateResult.md).

## Version

Introduced in version 11
