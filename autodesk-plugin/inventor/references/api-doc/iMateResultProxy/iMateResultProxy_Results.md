# iMateResultProxy.Results Property

Parent Object: [iMateResultProxy](../iMateResultProxy/iMateResultProxy.md)

## Description

Property that returns an iMateResultsEnumerator object that provides access to the iMate results object that are part of a composite iMate. This property is only valid in the case when the iMateResult object it is being called from is a composite. This can be checked for by using the IsComposite property of the iMateResult object.

## Syntax

iMateResultProxy.**Results**() As [iMateResultsEnumerator](../iMateResultsEnumerator/iMateResultsEnumerator.md)

## Property Value

This is a read only property whose value is an [iMateResultsEnumerator](../iMateResultsEnumerator/iMateResultsEnumerator.md).

## Version

Introduced in version 11
