# MiniToolbarComboBox.StandardIcon Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Read-write property that gets and sets the standard (small) icon for the control.

## Syntax

MiniToolbarComboBox.**StandardIcon**() As IPictureDisp

## Property Value

This is a read/write property whose value is an IPictureDisp.

## Version

Introduced in version 2012
