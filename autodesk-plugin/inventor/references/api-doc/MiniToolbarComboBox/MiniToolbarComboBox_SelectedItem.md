# MiniToolbarComboBox.SelectedItem Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Returns the item currently selected.

## Syntax

MiniToolbarComboBox.**SelectedItem**() As [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Property Value

This is a read only property whose value is a [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md).

## Version

Introduced in version 2012
