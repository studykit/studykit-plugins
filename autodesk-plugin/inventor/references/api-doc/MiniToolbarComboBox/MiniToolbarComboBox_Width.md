# MiniToolbarComboBox.Width Property

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Read-write property that gets and sets the width of the combobox in pixels.

## Syntax

MiniToolbarComboBox.**Width**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2012
