# MiniToolbarComboBox.OnPreMenuDisplay Event

Parent Object: [MiniToolbarComboBox](../MiniToolbarComboBox/MiniToolbarComboBox.md)

## Description

Event that is fired when the user clicks the combobox.

## Syntax

MiniToolbarComboBox.**OnPreMenuDisplay**()

## Version

Introduced in version 2012
