# FlangeEdgeSet.WidthExtentType Property

Parent Object: [FlangeEdgeSet](../FlangeEdgeSet/FlangeEdgeSet.md)

## Description

Read-only property that returns width extent type of this FlangeEdgeSet.

## Syntax

FlangeEdgeSet.**WidthExtentType**() As [PartFeatureExtentEnum](../PartFeatureExtentEnum.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtentEnum](../PartFeatureExtentEnum.md).

## Version

Introduced in version 2025
