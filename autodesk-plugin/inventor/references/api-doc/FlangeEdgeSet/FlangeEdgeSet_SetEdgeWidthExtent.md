# FlangeEdgeSet.SetEdgeWidthExtent Method

Parent Object: [FlangeEdgeSet](../FlangeEdgeSet/FlangeEdgeSet.md)

## Description

Method that changes the flange edge set object to define a flange whose width is defined by the length of the input edge(s).

## Syntax

FlangeEdgeSet.**SetEdgeWidthExtent**()

## Version

Introduced in version 2025
