# FlatBendResult.InternalName Property

Parent Object: [FlatBendResult](../FlatBendResult/FlatBendResult.md)

## Description

Property that returns a string that uniquely identifies a bend instance within the flat pattern.

## Syntax

FlatBendResult.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2008
