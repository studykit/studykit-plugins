# FlatBendResult.Angle Property

Parent Object: [FlatBendResult](../FlatBendResult/FlatBendResult.md)

## Description

Property that returns the angle of the bend.

## Syntax

FlatBendResult.**Angle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
