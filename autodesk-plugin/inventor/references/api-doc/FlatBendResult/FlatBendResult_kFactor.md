# FlatBendResult.kFactor Property

Parent Object: [FlatBendResult](../FlatBendResult/FlatBendResult.md)

## Description

Read-only property that returns the K-Factor value associated with this bend.

## Syntax

FlatBendResult.**kFactor**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2009
