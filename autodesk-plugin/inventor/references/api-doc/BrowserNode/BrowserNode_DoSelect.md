# BrowserNode.DoSelect Method

Parent Object: [BrowserNode](../BrowserNode/BrowserNode.md)

## Description

This method will simulate a mouse click on the browser node.

## Syntax

BrowserNode.**DoSelect**()

## Version

Introduced in version 8
