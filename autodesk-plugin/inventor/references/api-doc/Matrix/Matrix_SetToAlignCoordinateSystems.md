# Matrix.SetToAlignCoordinateSystems Method

Parent Object: [Matrix](../Matrix/Matrix.md)

## Description

Sets this matrix to be the matrix that maps from the 'from' coordinate system to the 'to' coordinate system.

## Syntax

Matrix.**SetToAlignCoordinateSystems**( ***FromOrigin*** As [Point](../Point/Point.md), ***FromXAxis*** As [Vector](../Vector/Vector.md), ***FromYAxis*** As [Vector](../Vector/Vector.md), ***FromZAxis*** As [Vector](../Vector/Vector.md), ***ToOrigin*** As [Point](../Point/Point.md), ***ToXAxis*** As [Vector](../Vector/Vector.md), ***ToYAxis*** As [Vector](../Vector/Vector.md), ***ToZAxis*** As [Vector](../Vector/Vector.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FromOrigin | [Point](../Point/Point.md) | Input Point object that defines the origin to map the coordinate system from. |
| FromXAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the X-axis to map the coordinate system from. |
| FromYAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the Y-axis to map the coordinate system from. |
| FromZAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the Z-axis to map the coordinate system from. |
| ToOrigin | [Point](../Point/Point.md) | Input Point object that defines the origin to map the coordinate system to. |
| ToXAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the X-axis to map the coordinate system to. |
| ToYAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the Y-axis to map the coordinate system to. |
| ToZAxis | [Vector](../Vector/Vector.md) | Input Vector object that defines the Z-axis to map the coordinate system to. |

## Version

Introduced in version 4
