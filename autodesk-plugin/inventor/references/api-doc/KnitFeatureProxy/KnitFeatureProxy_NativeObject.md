# KnitFeatureProxy.NativeObject Property

Parent Object: [KnitFeatureProxy](../KnitFeatureProxy/KnitFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

KnitFeatureProxy.**NativeObject**() As [KnitFeature](../KnitFeature/KnitFeature.md)

## Property Value

This is a read only property whose value is a [KnitFeature](../KnitFeature/KnitFeature.md).

## Version

Introduced in version 9
