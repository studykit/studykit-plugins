# CustomPropertyFormat.PropertyType Property

Parent Object: [CustomPropertyFormat](../CustomPropertyFormat/CustomPropertyFormat.md)

## Description

Gets/Sets whether the custom iProperty created for this parameter is a text (String) or number (Double) parameter.

## Syntax

CustomPropertyFormat.**PropertyType**() As [CustomPropertyTypeEnum](../CustomPropertyTypeEnum.md)

## Property Value

This is a read/write property whose value is a [CustomPropertyTypeEnum](../CustomPropertyTypeEnum.md).

## Version

Introduced in version 2010
