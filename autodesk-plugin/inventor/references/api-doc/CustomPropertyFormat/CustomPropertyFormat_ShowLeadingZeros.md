# CustomPropertyFormat.ShowLeadingZeros Property

Parent Object: [CustomPropertyFormat](../CustomPropertyFormat/CustomPropertyFormat.md)

## Description

Gets/Sets if leading zeros should be shown in the formatted iProperty value.

## Syntax

CustomPropertyFormat.**ShowLeadingZeros**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
