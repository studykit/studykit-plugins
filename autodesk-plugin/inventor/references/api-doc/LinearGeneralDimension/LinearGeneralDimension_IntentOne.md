# LinearGeneralDimension.IntentOne Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets the sets first geometry associated with the dimension.

## Syntax

LinearGeneralDimension.**IntentOne**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 11
