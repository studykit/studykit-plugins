# LinearGeneralDimension.ExtensionLineOneVisible Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets whether the first extension line is visible.

## Syntax

LinearGeneralDimension.**ExtensionLineOneVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
