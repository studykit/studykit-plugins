# LinearGeneralDimension.SecondArrowheadPosition Property

Parent Object: [LinearGeneralDimension](../LinearGeneralDimension/LinearGeneralDimension.md)

## Description

Gets and sets the position of the second arrowhead.

## Syntax

LinearGeneralDimension.**SecondArrowheadPosition**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2015
