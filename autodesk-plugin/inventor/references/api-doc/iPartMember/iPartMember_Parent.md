# iPartMember.Parent Property

Parent Object: [iPartMember](../iPartMember/iPartMember.md)

## Description

Property that returns the parent object.

## Syntax

iPartMember.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 6
