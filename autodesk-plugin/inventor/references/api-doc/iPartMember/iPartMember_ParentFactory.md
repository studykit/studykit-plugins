# iPartMember.ParentFactory Property

Parent Object: [iPartMember](../iPartMember/iPartMember.md)

## Description

Property that returns the iPart factory that created this iPartMember.

## Syntax

iPartMember.**ParentFactory**() As [iPartFactory](../iPartFactory/iPartFactory.md)

## Property Value

This is a read only property whose value is an [iPartFactory](../iPartFactory/iPartFactory.md).

## Version

Introduced in version 6
