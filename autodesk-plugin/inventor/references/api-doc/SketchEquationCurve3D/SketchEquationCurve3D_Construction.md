# SketchEquationCurve3D.Construction Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchEquationCurve3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
