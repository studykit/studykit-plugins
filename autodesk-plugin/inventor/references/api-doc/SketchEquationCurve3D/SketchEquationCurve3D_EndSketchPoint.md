# SketchEquationCurve3D.EndSketchPoint Property

Parent Object: [SketchEquationCurve3D](../SketchEquationCurve3D/SketchEquationCurve3D.md)

## Description

Read-only property that returns the sketch point that defines the position of the end of the curve.

## Syntax

SketchEquationCurve3D.**EndSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 2014
