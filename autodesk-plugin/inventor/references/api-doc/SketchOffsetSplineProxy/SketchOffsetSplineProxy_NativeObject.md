# SketchOffsetSplineProxy.NativeObject Property

Parent Object: [SketchOffsetSplineProxy](../SketchOffsetSplineProxy/SketchOffsetSplineProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchOffsetSplineProxy.**NativeObject**() As [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Property Value

This is a read only property whose value is a [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md).

## Version

Introduced in version 9
