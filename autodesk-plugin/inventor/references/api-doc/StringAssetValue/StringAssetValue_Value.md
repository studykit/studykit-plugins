# StringAssetValue.Value Property

Parent Object: [StringAssetValue](../StringAssetValue/StringAssetValue.md)

## Description

Gets and sets this asset value.

## Syntax

StringAssetValue.**Value**() As String

## Property Value

This is a read/write property whose value is a String.
