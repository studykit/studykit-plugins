# ModelToleranceFeatureProxy.ParentToleranceFeature Property

Parent Object: [ModelToleranceFeatureProxy](../ModelToleranceFeatureProxy/ModelToleranceFeatureProxy.md)

## Description

Returns the parent ModelToleranceFeature object if this ModelToleranceFeature is a sub feature of it.

## Syntax

ModelToleranceFeatureProxy.**ParentToleranceFeature**() As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Version

Introduced in version 2018
