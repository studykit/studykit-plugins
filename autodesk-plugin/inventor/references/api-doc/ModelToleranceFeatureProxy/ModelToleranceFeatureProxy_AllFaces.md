# ModelToleranceFeatureProxy.AllFaces Property

Parent Object: [ModelToleranceFeatureProxy](../ModelToleranceFeatureProxy/ModelToleranceFeatureProxy.md)

## Description

Gets all faces associated with the tolerance feature and its subfeatures.

## Syntax

ModelToleranceFeatureProxy.**AllFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read only property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2018
