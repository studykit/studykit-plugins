# ModelToleranceFeatureProxy.ClientId Property

Parent Object: [ModelToleranceFeatureProxy](../ModelToleranceFeatureProxy/ModelToleranceFeatureProxy.md)

## Description

Returns string that uniquely identifies the client.

## Syntax

ModelToleranceFeatureProxy.**ClientId**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
