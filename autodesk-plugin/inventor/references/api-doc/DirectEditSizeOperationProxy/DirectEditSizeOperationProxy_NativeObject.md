# DirectEditSizeOperationProxy.NativeObject Property

Parent Object: [DirectEditSizeOperationProxy](../DirectEditSizeOperationProxy/DirectEditSizeOperationProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

DirectEditSizeOperationProxy.**NativeObject**() As [DirectEditSizeOperation](../DirectEditSizeOperation/DirectEditSizeOperation.md)

## Property Value

This is a read only property whose value is a [DirectEditSizeOperation](../DirectEditSizeOperation/DirectEditSizeOperation.md).

## Version

Introduced in version 2015
