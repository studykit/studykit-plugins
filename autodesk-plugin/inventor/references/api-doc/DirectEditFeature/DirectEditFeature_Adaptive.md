# DirectEditFeature.Adaptive Property

Parent Object: [DirectEditFeature](../DirectEditFeature/DirectEditFeature.md)

## Description

Gets and sets whether this feature is adaptive or not.

## Syntax

DirectEditFeature.**Adaptive**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
