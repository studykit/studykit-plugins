# ReferenceComponents.DerivedFuturePartComponents Property

Parent Object: [ReferenceComponents](../ReferenceComponents/ReferenceComponents.md)

## Description

Gets the DerivedFuturePartComponents collection object.

## Syntax

ReferenceComponents.**DerivedFuturePartComponents**() As [DerivedFuturePartComponents](../DerivedFuturePartComponents/DerivedFuturePartComponents.md)

## Property Value

This is a read only property whose value is a [DerivedFuturePartComponents](../DerivedFuturePartComponents/DerivedFuturePartComponents.md).

## Version

Introduced in version 2018
