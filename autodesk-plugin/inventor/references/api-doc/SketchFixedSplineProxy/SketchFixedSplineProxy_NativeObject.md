# SketchFixedSplineProxy.NativeObject Property

Parent Object: [SketchFixedSplineProxy](../SketchFixedSplineProxy/SketchFixedSplineProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchFixedSplineProxy.**NativeObject**() As [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md)

## Property Value

This is a read only property whose value is a [SketchFixedSpline](../SketchFixedSpline/SketchFixedSpline.md).

## Version

Introduced in version 9
