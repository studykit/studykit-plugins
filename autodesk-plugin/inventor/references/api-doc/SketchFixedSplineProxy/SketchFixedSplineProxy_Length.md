# SketchFixedSplineProxy.Length Property

Parent Object: [SketchFixedSplineProxy](../SketchFixedSplineProxy/SketchFixedSplineProxy.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchFixedSplineProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 9
