# Point.TransformBy Method

Parent Object: [Point](../Point/Point.md)

## Description

Transform this point by the specified matrix.

## Syntax

Point.**TransformBy**( ***Matrix*** As [Matrix](../Matrix/Matrix.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix | [Matrix](../Matrix/Matrix.md) | Input Matrix object that specifies the matrix to transform by. |

## Version

Introduced in version 4
