# Point.VectorTo Method

Parent Object: [Point](../Point/Point.md)

## Description

Gets the vector of translation between this point and the specified point.

## Syntax

Point.**VectorTo**( ***Point*** As [Point](../Point/Point.md) ) As [Vector](../Vector/Vector.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input object that specifies the point. |
