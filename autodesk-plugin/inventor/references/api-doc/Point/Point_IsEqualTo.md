# Point.IsEqualTo Method

Parent Object: [Point](../Point/Point.md)

## Description

Compares this point for equality with the specified point.

## Syntax

Point.**IsEqualTo**( ***Point*** As [Point](../Point/Point.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input Point object that specifies the coordinate to compare. |
| Tolerance | Double | Input Double that specifies the tolerance to be used when determining whether the points are equal. |
