# ButtonDefinition.ProgressiveToolTip Property

Parent Object: [ButtonDefinition](../ButtonDefinition/ButtonDefinition.md)

## Description

Returns a ProgressiveToolTip object providing access to enhanced tooltip display for controls in the ribbon interface.

## Syntax

ButtonDefinition.**ProgressiveToolTip**() As [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md)

## Property Value

This is a read only property whose value is a [ProgressiveToolTip](../ProgressiveToolTip/ProgressiveToolTip.md).

## Version

Introduced in version 2011
