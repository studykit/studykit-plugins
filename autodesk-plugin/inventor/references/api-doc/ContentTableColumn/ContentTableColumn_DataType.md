# ContentTableColumn.DataType Property

Parent Object: [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md)

## Description

Property that indicates the type of value defined for this column. Values this property can return are kIntegerType, kDoubleType, kStringType, and kBooleanType.

## Syntax

ContentTableColumn.**DataType**() As [ValueTypeEnum](../ValueTypeEnum.md)

## Property Value

This is a read only property whose value is a [ValueTypeEnum](../ValueTypeEnum.md).

## Version

Introduced in version 2010
