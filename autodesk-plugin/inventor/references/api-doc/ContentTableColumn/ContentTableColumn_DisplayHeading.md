# ContentTableColumn.DisplayHeading Property

Parent Object: [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md)

## Description

Gets/Sets the display name of this column.

## Syntax

ContentTableColumn.**DisplayHeading**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
