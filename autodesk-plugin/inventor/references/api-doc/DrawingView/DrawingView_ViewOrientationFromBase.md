# DrawingView.ViewOrientationFromBase Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Gets and sets whether the view should inherit the orientation from the base view.

## Syntax

DrawingView.**ViewOrientationFromBase**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
