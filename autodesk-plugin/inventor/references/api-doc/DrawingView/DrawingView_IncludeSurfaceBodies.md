# DrawingView.IncludeSurfaceBodies Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Read-write property that gets and sets whether to include the work surfaces.

## Syntax

DrawingView.**IncludeSurfaceBodies**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
