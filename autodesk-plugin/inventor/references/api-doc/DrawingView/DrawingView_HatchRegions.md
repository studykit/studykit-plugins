# DrawingView.HatchRegions Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Read-only property that returns the DrawingViewHatchRegions collection object.

## Syntax

DrawingView.**HatchRegions**() As [DrawingViewHatchRegions](../DrawingViewHatchRegions/DrawingViewHatchRegions.md)

## Property Value

This is a read only property whose value is a [DrawingViewHatchRegions](../DrawingViewHatchRegions/DrawingViewHatchRegions.md).

## Version

Introduced in version 2022
