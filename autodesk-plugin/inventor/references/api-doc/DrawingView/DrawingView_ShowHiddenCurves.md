# DrawingView.ShowHiddenCurves Method

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Method that displays all the curves explicitly hidden by the user.

## Syntax

DrawingView.**ShowHiddenCurves**()

## Version

Introduced in version 11
