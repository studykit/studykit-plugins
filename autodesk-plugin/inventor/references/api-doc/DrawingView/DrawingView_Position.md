# DrawingView.Position Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Gets and sets the point used to position the drawing view on the sheet.

## Syntax

DrawingView.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 2008
