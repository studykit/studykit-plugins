# DrawingView.GetAutomatedCenterlineSettings Method

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Method that returns the settings that define how automatic center lines and center marks are to be calculated for this view.

## Syntax

DrawingView.**GetAutomatedCenterlineSettings**( ***AutomatedCenterlineSettings*** As [AutomatedCenterlineSettings](../AutomatedCenterlineSettings/AutomatedCenterlineSettings.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| AutomatedCenterlineSettings | [AutomatedCenterlineSettings](../AutomatedCenterlineSettings/AutomatedCenterlineSettings.md) | Output AutomatedCenterlineSettings object that defines the automatic center line and center mark creation settings for this drawing view. |

## Version

Introduced in version 2010
