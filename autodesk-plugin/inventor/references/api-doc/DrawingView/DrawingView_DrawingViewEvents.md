# DrawingView.DrawingViewEvents Property

Parent Object: [DrawingView](../DrawingView/DrawingView.md)

## Description

Property that returns the object, which provides event notifcation. For example, view updates.

## Syntax

DrawingView.**DrawingViewEvents**() As [DrawingViewEvents](../DrawingViewEvents/DrawingViewEvents.md)

## Property Value

This is a read only property whose value is a [DrawingViewEvents](../DrawingViewEvents/DrawingViewEvents.md).

## Version

Introduced in version 9
