# ApplicationAddIns.Item Property

Parent Object: [ApplicationAddIns](../ApplicationAddIns/ApplicationAddIns.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

ApplicationAddIns.**Item**( ***Index*** As Long ) As [ApplicationAddIn](../ApplicationAddIn/ApplicationAddIn.md)

## Property Value

This is a read only property whose value is an [ApplicationAddIn](../ApplicationAddIn/ApplicationAddIn.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 4
