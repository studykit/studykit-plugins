# SketchEquationCurve3DProxy.CurvatureDisplay Property

Parent Object: [SketchEquationCurve3DProxy](../SketchEquationCurve3DProxy/SketchEquationCurve3DProxy.md)

## Description

Specifies whether curvature information is displayed for the curve.

## Syntax

SketchEquationCurve3DProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
