# SketchEquationCurve3DProxy.Length Property

Parent Object: [SketchEquationCurve3DProxy](../SketchEquationCurve3DProxy/SketchEquationCurve3DProxy.md)

## Description

Gets the length of the entity in centimeters.

## Syntax

SketchEquationCurve3DProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2014
