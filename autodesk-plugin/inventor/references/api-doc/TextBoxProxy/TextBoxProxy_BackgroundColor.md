# TextBoxProxy.BackgroundColor Property

Parent Object: [TextBoxProxy](../TextBoxProxy/TextBoxProxy.md)

## Description

Gets/Sets the BackgroundColor associated with the TextBox.

## Syntax

TextBoxProxy.**BackgroundColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2017
