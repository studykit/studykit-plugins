# TextBoxProxy.Color Property

Parent Object: [TextBoxProxy](../TextBoxProxy/TextBoxProxy.md)

## Description

Gets/Sets the color of the text box.

## Syntax

TextBoxProxy.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
