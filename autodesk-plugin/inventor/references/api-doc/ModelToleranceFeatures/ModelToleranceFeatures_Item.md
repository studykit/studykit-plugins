# ModelToleranceFeatures.Item Property

Parent Object: [ModelToleranceFeatures](../ModelToleranceFeatures/ModelToleranceFeatures.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

ModelToleranceFeatures.**Item**( ***Index*** As Variant ) As [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md)

## Property Value

This is a read only property whose value is a [ModelToleranceFeature](../ModelToleranceFeature/ModelToleranceFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant |  |

## Version

Introduced in version 2018
