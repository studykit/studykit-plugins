# ModelToleranceFeatures.UpdateStatusVisualization Method

Parent Object: [ModelToleranceFeatures](../ModelToleranceFeatures/ModelToleranceFeatures.md)

## Description

Updates the status face coloring if it is switched on.

## Syntax

ModelToleranceFeatures.**UpdateStatusVisualization**()

## Version

Introduced in version 2019
