# LumpDefinitions.Item Property

Parent Object: [LumpDefinitions](../LumpDefinitions/LumpDefinitions.md)

## Description

Returns the specified LumpDefinition object from the collection.

## Syntax

LumpDefinitions.**Item**( ***Index*** As Long ) As [LumpDefinition](../LumpDefinition/LumpDefinition.md)

## Property Value

This is a read only property whose value is a [LumpDefinition](../LumpDefinition/LumpDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the LumpDefinition object within the collection to return. |

## Version

Introduced in version 2011
