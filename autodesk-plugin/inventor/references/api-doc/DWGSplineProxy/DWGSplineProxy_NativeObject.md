# DWGSplineProxy.NativeObject Property

Parent Object: [DWGSplineProxy](../DWGSplineProxy/DWGSplineProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGSplineProxy.**NativeObject**() As [DWGSpline](../DWGSpline/DWGSpline.md)

## Property Value

This is a read only property whose value is a [DWGSpline](../DWGSpline/DWGSpline.md).

## Version

Introduced in version 2016
