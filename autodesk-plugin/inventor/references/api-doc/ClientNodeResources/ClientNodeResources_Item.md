# ClientNodeResources.Item Property

Parent Object: [ClientNodeResources](../ClientNodeResources/ClientNodeResources.md)

## Description

Returns the specified NodeResource object from the collection.

## Syntax

ClientNodeResources.**Item**( ***Index*** As Long ) As [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Property Value

This is a read only property whose value is a [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the NodeResource to return. This is the numeric value indicating the index of the item in the collection. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 8
