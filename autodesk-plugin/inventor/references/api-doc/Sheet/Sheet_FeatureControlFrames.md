# Sheet.FeatureControlFrames Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the FeatureControlFrames collection object.

## Syntax

Sheet.**FeatureControlFrames**() As [FeatureControlFrames](../FeatureControlFrames/FeatureControlFrames.md)

## Property Value

This is a read only property whose value is a [FeatureControlFrames](../FeatureControlFrames/FeatureControlFrames.md).
