# Sheet.ClientViews Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Property that returns the collection for the drawing sheet.

## Syntax

Sheet.**ClientViews**() As [ClientViews](../ClientViews/ClientViews.md)

## Property Value

This is a read only property whose value is a [ClientViews](../ClientViews/ClientViews.md).

## Version

Introduced in version 6
