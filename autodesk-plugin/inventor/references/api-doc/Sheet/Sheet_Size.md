# Sheet.Size Property

Parent Object: [Sheet](../Sheet/Sheet.md)

## Description

Gets or sets the size of the Sheet as value in an enumeration of standard sheet sizes.

## Syntax

Sheet.**Size**() As [DrawingSheetSizeEnum](../DrawingSheetSizeEnum.md)

## Property Value

This is a read/write property whose value is a [DrawingSheetSizeEnum](../DrawingSheetSizeEnum.md).

## Version

Introduced in version 4
