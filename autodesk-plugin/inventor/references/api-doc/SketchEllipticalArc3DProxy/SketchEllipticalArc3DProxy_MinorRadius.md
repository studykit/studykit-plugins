# SketchEllipticalArc3DProxy.MinorRadius Property

Parent Object: [SketchEllipticalArc3DProxy](../SketchEllipticalArc3DProxy/SketchEllipticalArc3DProxy.md)

## Description

Gets the MinorRadius of the elliptical arc.

## Syntax

SketchEllipticalArc3DProxy.**MinorRadius**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
