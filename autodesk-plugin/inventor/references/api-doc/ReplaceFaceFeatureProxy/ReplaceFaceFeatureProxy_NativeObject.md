# ReplaceFaceFeatureProxy.NativeObject Property

Parent Object: [ReplaceFaceFeatureProxy](../ReplaceFaceFeatureProxy/ReplaceFaceFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ReplaceFaceFeatureProxy.**NativeObject**() As [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md)

## Property Value

This is a read only property whose value is a [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md).

## Version

Introduced in version 9
