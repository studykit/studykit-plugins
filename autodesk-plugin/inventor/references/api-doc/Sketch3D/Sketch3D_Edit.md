# Sketch3D.Edit Method

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Method that causes the Sketch environment to be invoked with this sketch available for interactive edit.

## Syntax

Sketch3D.**Edit**()

## Version

Introduced in version 6
