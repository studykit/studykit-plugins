# Sketch3D.CheckSumValue Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Gets sketch checksum value.

## Syntax

Sketch3D.**CheckSumValue**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2020
