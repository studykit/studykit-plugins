# Sketch3D.HasReferenceComponent Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

Sketch3D.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
