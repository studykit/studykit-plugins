# Sketch3D.OverrideColor Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Specifies the override color, if any, for this sketch.

## Syntax

Sketch3D.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 8
