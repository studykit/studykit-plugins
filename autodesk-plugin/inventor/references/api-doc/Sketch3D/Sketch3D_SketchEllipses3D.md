# Sketch3D.SketchEllipses3D Property

Parent Object: [Sketch3D](../Sketch3D/Sketch3D.md)

## Description

Property that returns the SketchEllipses3D collection object. This collection provides access to the existing ellipses in the sketch.

## Syntax

Sketch3D.**SketchEllipses3D**() As [SketchEllipses3D](../SketchEllipses3D/SketchEllipses3D.md)

## Property Value

This is a read only property whose value is a [SketchEllipses3D](../SketchEllipses3D/SketchEllipses3D.md).

## Version

Introduced in version 6
