# SketchPatternElement.Parent Property

Parent Object: [SketchPatternElement](../SketchPatternElement/SketchPatternElement.md)

## Description

Returns the parent pattern this element is a member of.

## Syntax

SketchPatternElement.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2025.1
