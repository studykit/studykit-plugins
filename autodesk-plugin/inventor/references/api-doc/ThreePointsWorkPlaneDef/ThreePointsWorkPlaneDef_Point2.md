# ThreePointsWorkPlaneDef.Point2 Property

Parent Object: [ThreePointsWorkPlaneDef](../ThreePointsWorkPlaneDef/ThreePointsWorkPlaneDef.md)

## Description

Property that returns the second point of a three point defined work plane. The second point defines the X-axis in relation to the first point. This object can be a WorkPoint, Vertex, or SketchPoint object.

## Syntax

ThreePointsWorkPlaneDef.**Point2**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
