# EdgeProxy.StopVertex Property

Parent Object: [EdgeProxy](../EdgeProxy/EdgeProxy.md)

## Description

Gets the Vertex referenced at the end of this Edge.

## Syntax

EdgeProxy.**StopVertex**() As [Vertex](../Vertex/Vertex.md)

## Property Value

This is a read only property whose value is a [Vertex](../Vertex/Vertex.md).

## Version

Introduced in version 4
