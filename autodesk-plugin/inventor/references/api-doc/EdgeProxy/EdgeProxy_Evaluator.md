# EdgeProxy.Evaluator Property

Parent Object: [EdgeProxy](../EdgeProxy/EdgeProxy.md)

## Description

Gets the CurveEvaluator for this Edge.

## Syntax

EdgeProxy.**Evaluator**() As [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md)

## Property Value

This is a read only property whose value is a [CurveEvaluator](../CurveEvaluator/CurveEvaluator.md).

## Version

Introduced in version 4
