# StyleLocationEnum Enumerator

## Description

Enum for the different style location types.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBothStyleLocation | 51201 | style is in both local and global locations. |
| kLibraryStyleLocation | 51203 | style is only in the library. |
| kLocalStyleLocation | 51202 | style is only local. |

## Version

Introduced in version 9
