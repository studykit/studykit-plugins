# SketchSettings.DisplayLineWeights Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Gets and sets the display of unique line weights.

## Syntax

SketchSettings.**DisplayLineWeights**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
