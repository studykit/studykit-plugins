# SketchSettings.Parent Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Property that returns the parent Document object.

## Syntax

SketchSettings.**Parent**() As [Document](../Document/Document.md)

## Property Value

This is a read only property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 8
