# SketchSettings.MinorLinesPerMajorGridLine Property

Parent Object: [SketchSettings](../SketchSettings/SketchSettings.md)

## Description

Gets and sets the number of minor lines to appear between major lines.

## Syntax

SketchSettings.**MinorLinesPerMajorGridLine**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 8
