# RevisionTableCell Object

## Description

The RevisionTableCell object represents an individual cell in the revision table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevisionTableCell/RevisionTableCell_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [FormattedText](../RevisionTableCell/RevisionTableCell_FormattedText.md) | Gets/Sets formatted text of the RevisionTable cell. |
| [Parent](../RevisionTableCell/RevisionTableCell_Parent.md) | Property that returns the parent RevisionTable object. |
| [StackedTextPosition](../RevisionTableCell/RevisionTableCell_StackedTextPosition.md) | Gets and sets the position (alignment) of the stacked text with respect to regular text. |
| [Text](../RevisionTableCell/RevisionTableCell_Text.md) | Gets/Sets the text of the RevisionTable cell. |
| [Type](../RevisionTableCell/RevisionTableCell_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
