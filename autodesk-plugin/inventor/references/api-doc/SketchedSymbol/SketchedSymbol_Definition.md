# SketchedSymbol.Definition Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Property returning the sketched symbol definition object referenced by the sketched symbol.

## Syntax

SketchedSymbol.**Definition**() As [SketchedSymbolDefinition](../SketchedSymbolDefinition/SketchedSymbolDefinition.md)

## Property Value

This is a read only property whose value is a [SketchedSymbolDefinition](../SketchedSymbolDefinition/SketchedSymbolDefinition.md).

## Version

Introduced in version 5.3
