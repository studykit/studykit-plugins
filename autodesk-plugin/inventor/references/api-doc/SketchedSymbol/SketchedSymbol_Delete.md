# SketchedSymbol.Delete Method

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Method that deletes the sketched symbol (e.g. the title block) from the sheet.

## Syntax

SketchedSymbol.**Delete**()

## Version

Introduced in version 5.3
