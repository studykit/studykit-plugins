# SketchedSymbol.Position Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets the origin position of the symbol on the sheet.

## Syntax

SketchedSymbol.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5.3
