# SketchedSymbol.Parent Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Property that gets the parent object from whom this object can logically be reached.

## Syntax

SketchedSymbol.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 5.3
