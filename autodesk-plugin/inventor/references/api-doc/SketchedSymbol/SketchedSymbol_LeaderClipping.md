# SketchedSymbol.LeaderClipping Property

Parent Object: [SketchedSymbol](../SketchedSymbol/SketchedSymbol.md)

## Description

Gets and sets whether the symbol's leader should be clipped where it overlaps the symbol.

## Syntax

SketchedSymbol.**LeaderClipping**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
