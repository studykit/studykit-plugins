# RotateRotateConstraintProxy.NativeObject Property

Parent Object: [RotateRotateConstraintProxy](../RotateRotateConstraintProxy/RotateRotateConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

RotateRotateConstraintProxy.**NativeObject**() As [RotateRotateConstraint](../RotateRotateConstraint/RotateRotateConstraint.md)

## Property Value

This is a read only property whose value is a [RotateRotateConstraint](../RotateRotateConstraint/RotateRotateConstraint.md).

## Version

Introduced in version 11
