# MeshVertex.MeshFeature Property

Parent Object: [MeshVertex](../MeshVertex/MeshVertex.md)

## Description

Read-only property that returns the MeshFeature this MeshVertex is located on.

## Syntax

MeshVertex.**MeshFeature**() As [MeshFeature](../MeshFeature/MeshFeature.md)

## Property Value

This is a read only property whose value is a [MeshFeature](../MeshFeature/MeshFeature.md).

## Version

Introduced in version 2017
