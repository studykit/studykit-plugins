# MeshVertex.MeshFeatureEntity Property

Parent Object: [MeshVertex](../MeshVertex/MeshVertex.md)

## Description

Read-only property that returns the MeshFeatureEntity this MeshVertex is located on.

## Syntax

MeshVertex.**MeshFeatureEntity**() As [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Property Value

This is a read only property whose value is a [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md).

## Version

Introduced in version 2017
