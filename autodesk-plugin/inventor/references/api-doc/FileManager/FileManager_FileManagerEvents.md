# FileManager.FileManagerEvents Property

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Property that gets the object that fires the File Manager related events.

## Syntax

FileManager.**FileManagerEvents**() As [FileManagerEvents](../FileManagerEvents/FileManagerEvents.md)

## Property Value

This is a read only property whose value is a [FileManagerEvents](../FileManagerEvents/FileManagerEvents.md).

## Version

Introduced in version 9
