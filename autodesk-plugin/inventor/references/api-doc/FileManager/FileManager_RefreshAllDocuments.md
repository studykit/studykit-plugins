# FileManager.RefreshAllDocuments Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Refreshes all documents in memory to the latest version on disk.

## Syntax

FileManager.**RefreshAllDocuments**()

## Version

Introduced in version 2010
