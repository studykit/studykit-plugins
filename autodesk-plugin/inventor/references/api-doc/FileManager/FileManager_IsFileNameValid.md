# FileManager.IsFileNameValid Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Validate the input filename and replace illegal filename characters with something more appropriate.

## Syntax

FileManager.**IsFileNameValid**( ***FileName*** As String, ***ValidFileName*** As String ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FileName | String | Filename to validate. |
| ValidFileName | String | Validated filename. |

## Version

Introduced in version 11
