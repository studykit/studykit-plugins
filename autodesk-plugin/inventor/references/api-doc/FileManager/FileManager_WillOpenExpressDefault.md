# FileManager.WillOpenExpressDefault Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Returns whether the input assembly will open in Express mode by default.

## Syntax

FileManager.**WillOpenExpressDefault**( ***FullFileName*** As String ) As Boolean
