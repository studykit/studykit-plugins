# FileManager.GetRevitEngineInstallationStatus Method

Parent Object: [FileManager](../FileManager/FileManager.md)

## Description

Method that returns the Revit engine installation status for available versions.

## Syntax

FileManager.**GetRevitEngineInstallationStatus**() As [NameValueMap](../NameValueMap/NameValueMap.md)

## Version

Introduced in version 2025
