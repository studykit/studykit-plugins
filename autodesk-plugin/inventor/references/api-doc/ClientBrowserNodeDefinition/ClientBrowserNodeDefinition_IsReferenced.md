# ClientBrowserNodeDefinition.IsReferenced Property

Parent Object: [ClientBrowserNodeDefinition](../ClientBrowserNodeDefinition/ClientBrowserNodeDefinition.md)

## Description

Gets whether this ClientBrowserNodeDefinition object is used by any browser nodes.

## Syntax

ClientBrowserNodeDefinition.**IsReferenced**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 10
