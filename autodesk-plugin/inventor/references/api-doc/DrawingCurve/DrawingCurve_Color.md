# DrawingCurve.Color Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Gets and sets the color for the curve.

## Syntax

DrawingCurve.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2010
