# DrawingCurve.ModelGeometry Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the corresponding geometry from the model. This property returns Nothing in the case the model is not available.

## Syntax

DrawingCurve.**ModelGeometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
