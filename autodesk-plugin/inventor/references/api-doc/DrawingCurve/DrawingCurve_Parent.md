# DrawingCurve.Parent Property

Parent Object: [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Description

Property that returns the parent drawing view of the curve.

## Syntax

DrawingCurve.**Parent**() As [DrawingView](../DrawingView/DrawingView.md)

## Property Value

This is a read only property whose value is a [DrawingView](../DrawingView/DrawingView.md).
