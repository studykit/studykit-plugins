# FreeformFeature.Suppressed Property

Parent Object: [FreeformFeature](../FreeformFeature/FreeformFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

FreeformFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
