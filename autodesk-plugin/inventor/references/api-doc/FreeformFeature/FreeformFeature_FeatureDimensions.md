# FreeformFeature.FeatureDimensions Property

Parent Object: [FreeformFeature](../FreeformFeature/FreeformFeature.md)

## Description

Property that returns the FeatureDimensions collection object.

## Syntax

FreeformFeature.**FeatureDimensions**() As [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md)

## Property Value

This is a read only property whose value is a [FeatureDimensions](../FeatureDimensions/FeatureDimensions.md).

## Version

Introduced in version 2015
