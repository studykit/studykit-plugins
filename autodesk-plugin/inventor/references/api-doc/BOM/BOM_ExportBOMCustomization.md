# BOM.ExportBOMCustomization Method

Parent Object: [BOM](../BOM/BOM.md)

## Description

Export BOM column customization as XML file.

## Syntax

BOM.**ExportBOMCustomization**( ***FileName*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FileName | String | Input String that defines the name of the xml file to write the BOM customization information to. |

## Version

Introduced in version 2015
