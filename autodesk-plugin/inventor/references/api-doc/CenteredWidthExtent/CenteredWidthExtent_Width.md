# CenteredWidthExtent.Width Property

Parent Object: [CenteredWidthExtent](../CenteredWidthExtent/CenteredWidthExtent.md)

## Description

Property that returns the parameter controlling the width of the flange.

## Syntax

CenteredWidthExtent.**Width**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2010
