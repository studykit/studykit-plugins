# SketchPoints3D.Add Method

Parent Object: [SketchPoints3D](../SketchPoints3D/SketchPoints3D.md)

## Description

Method that creates a new 3D sketch point at the specified coordinate.

## Syntax

SketchPoints3D.**Add**( ***Point*** As [Point](../Point/Point.md), [***HoleCenter***] As Boolean ) As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Point | [Point](../Point/Point.md) | Input object that defines the position of the 3D sketch point. |
| HoleCenter | Boolean | Optional input Boolean that specifies if the point is a hole center or not. |

## Version

Introduced in version 11
