# FlatPatternOrientation.AlignmentRotation Property

Parent Object: [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Description

Read-only property that returns the parameter object controlling the alignment rotation angle of flat pattern.

## Syntax

FlatPatternOrientation.**AlignmentRotation**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2015
