## Properties

|  |  |
| --- | --- |
| Name | Description |
| [AlignmentAxis](../FlatPatternOrientation/FlatPatternOrientation_AlignmentAxis.md) | Read-write property that gets and sets the alignment axis of flat pattern orientation. |
| [AlignmentRotation](../FlatPatternOrientation/FlatPatternOrientation_AlignmentRotation.md) | Read-only property that returns the parameter object controlling the alignment rotation angle of flat pattern. |
| [AlignmentType](../FlatPatternOrientation/FlatPatternOrientation_AlignmentType.md) | Read-write property that gets and sets the alignment type of flat pattern orientation. |
| [Application](../FlatPatternOrientation/FlatPatternOrientation_Application.md) | Gets the root Application object. | [FlipAlignmentAxis](../FlatPatternOrientation/FlatPatternOrientation_FlipAlignmentAxis.md) | Read-write property that gets and sets whether alignment direction of flat pattern orientation is the opposite of the natural direction of the alignment axis. |
| [FlipBaseFace](../FlatPatternOrientation/FlatPatternOrientation_FlipBaseFace.md) | Read-write property that gets and sets whether to flip the flat pattern by swapping the top and bottom faces or not. |
| [Name](../FlatPatternOrientation/FlatPatternOrientation_Name.md) | Read-write property that gets and sets the name for the FlatPatternOrientation object. |
| [Parent](../FlatPatternOrientation/FlatPatternOrientation_Parent.md) | Gets the parent object from whom this object can logically be reached. |
