# FlatPatternOrientation.Name Property

Parent Object: [FlatPatternOrientation](../FlatPatternOrientation/FlatPatternOrientation.md)

## Description

Read-write property that gets and sets the name for the FlatPatternOrientation object.

## Syntax

FlatPatternOrientation.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2015
