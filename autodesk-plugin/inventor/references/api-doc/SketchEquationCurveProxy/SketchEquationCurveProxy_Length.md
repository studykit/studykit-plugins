# SketchEquationCurveProxy.Length Property

Parent Object: [SketchEquationCurveProxy](../SketchEquationCurveProxy/SketchEquationCurveProxy.md)

## Description

Gets the length of the entity in centimeters.

## Syntax

SketchEquationCurveProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2014
