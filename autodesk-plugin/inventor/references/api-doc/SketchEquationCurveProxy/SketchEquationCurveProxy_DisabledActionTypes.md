# SketchEquationCurveProxy.DisabledActionTypes Property

Parent Object: [SketchEquationCurveProxy](../SketchEquationCurveProxy/SketchEquationCurveProxy.md)

## Description

Gets and sets whether specific user actions are blocked on this sketch geometry.

## Syntax

SketchEquationCurveProxy.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 2014
