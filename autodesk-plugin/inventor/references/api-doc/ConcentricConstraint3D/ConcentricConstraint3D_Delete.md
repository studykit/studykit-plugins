# ConcentricConstraint3D.Delete Method

Parent Object: [ConcentricConstraint3D](../ConcentricConstraint3D/ConcentricConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ConcentricConstraint3D.**Delete**()

## Version

Introduced in version 2010
