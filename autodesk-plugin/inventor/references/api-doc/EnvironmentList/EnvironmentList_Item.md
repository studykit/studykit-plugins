# EnvironmentList.Item Property

Parent Object: [EnvironmentList](../EnvironmentList/EnvironmentList.md)

## Description

Returns the specified Environment object from the list.

## Syntax

EnvironmentList.**Item**( ***Index*** As Long ) As [Environment](../Environment/Environment.md)

## Property Value

This is a read only property whose value is an [Environment](../Environment/Environment.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long that specifies the object within the list to return. |

## Version

Introduced in version 10
