# MirrorFeature.AddParticipant Method

Parent Object: [MirrorFeature](../MirrorFeature/MirrorFeature.md)

## Description

Add the specified occurrence from the list of participants for this feature. This method only applies to assembly features.

## Syntax

MirrorFeature.**AddParticipant**( ***Occurrenc*** As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Occurrenc | [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md) |  |

## Version

Introduced in version 2010
