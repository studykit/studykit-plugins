# MirrorFeature.SurfacesOpaque Property

Parent Object: [MirrorFeature](../MirrorFeature/MirrorFeature.md)

## Description

Gets and sets whether all the resulting surfaces are opaque or transparent.

## Syntax

MirrorFeature.**SurfacesOpaque**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
