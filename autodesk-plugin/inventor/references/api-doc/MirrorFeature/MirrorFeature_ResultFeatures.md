# MirrorFeature.ResultFeatures Property

Parent Object: [MirrorFeature](../MirrorFeature/MirrorFeature.md)

## Description

Property that returns the features that were created for this mirror. These can be work planes, work axes, work points and work surfaces.

## Syntax

MirrorFeature.**ResultFeatures**() As [ObjectsEnumerator](../ObjectsEnumerator/ObjectsEnumerator.md)

## Property Value

This is a read only property whose value is an [ObjectsEnumerator](../ObjectsEnumerator/ObjectsEnumerator.md).

## Version

Introduced in version 11
