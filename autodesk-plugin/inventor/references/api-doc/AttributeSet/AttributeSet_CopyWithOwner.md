# AttributeSet.CopyWithOwner Property

Parent Object: [AttributeSet](../AttributeSet/AttributeSet.md)

## Description

Gets the Boolean flag that controls the behavior of whether this set will copied onto this Inventor Object's copy.

## Syntax

AttributeSet.**CopyWithOwner**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
