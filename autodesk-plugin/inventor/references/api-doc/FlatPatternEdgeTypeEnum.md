## Description

Edge types for flat pattern edges.

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |

| kBendUpFlatPatternEdge | 64004 | Bend up edges. |
| kTangentFlatPatternEdge | 64001 | Tangent edges. |
