# InteractionGraphics.UpdateOverlayGraphics Method

Parent Object: [InteractionGraphics](../InteractionGraphics/InteractionGraphics.md)

## Description

Redraw just the overlay graphics only in a specified view.

## Syntax

InteractionGraphics.**UpdateOverlayGraphics**( ***TheView*** As [View](../View/View.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TheView | [View](../View/View.md) | View object representing the view to redraw. |
