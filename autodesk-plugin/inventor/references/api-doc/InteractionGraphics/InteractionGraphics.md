# InteractionGraphics Object

## Description

Inventor supports the creation of preview and overlay graphics.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [UpdateOverlayGraphics](../InteractionGraphics/InteractionGraphics_UpdateOverlayGraphics.md) | Redraw just the overlay graphics only in a specified view. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [GraphicsDataSets](../InteractionGraphics/InteractionGraphics_GraphicsDataSets.md) | The object that can be used with the preview and overlay graphics properties. |
| [OverlayClientGraphics](../InteractionGraphics/InteractionGraphics_OverlayClientGraphics.md) | The top level object for graphics that are rendered independently from the main scene. |
| [PreviewClientGraphics](../InteractionGraphics/InteractionGraphics_PreviewClientGraphics.md) | The top level object for graphics that are rendered with the main scene. |
| [Type](../InteractionGraphics/InteractionGraphics_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[InteractionEvents.InteractionGraphics](../InteractionEvents/InteractionEvents_InteractionGraphics.md)
