# ASideDefinition.Faces Property

Parent Object: [ASideDefinition](../ASideDefinition/ASideDefinition.md)

## Description

Read-only property that returns all the faces that are tangentially connected with each other to define the A-side face definition, includes the ASideFace.

## Syntax

ASideDefinition.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2016
