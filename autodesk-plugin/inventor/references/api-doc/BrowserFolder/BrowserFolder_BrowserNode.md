# BrowserFolder.BrowserNode Property

Parent Object: [BrowserFolder](../BrowserFolder/BrowserFolder.md)

## Description

Property that returns the corresponding browser node for the folder. The contents of the folder (including sub-folders) can be accessed via the browser node.

## Syntax

BrowserFolder.**BrowserNode**() As [BrowserNode](../BrowserNode/BrowserNode.md)

## Property Value

This is a read only property whose value is a [BrowserNode](../BrowserNode/BrowserNode.md).

## Version

Introduced in version 2010
