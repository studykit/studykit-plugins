# SurfaceGraphicsFaceList.Clear Method

Parent Object: [SurfaceGraphicsFaceList](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList.md)

## Description

Method that removes all the faces from the list. No faces will be displayed after the method is called.

## Syntax

SurfaceGraphicsFaceList.**Clear**()

## Version

Introduced in version 2009
