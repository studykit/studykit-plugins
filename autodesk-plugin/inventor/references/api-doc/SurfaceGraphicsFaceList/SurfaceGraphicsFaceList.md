# SurfaceGraphicsFaceList Object

## Description

The SurfaceGraphicsFaceList object contains a list of faces currently displayed by a SurfaceGraphics primitive and allows you to add faces to the list.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Add.md) | Method that specifies additional faces to be displayed. |
| [Clear](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Clear.md) | Method that removes all the faces from the list. No faces will be displayed after the method is called. |
| [Remove](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Remove.md) | Method that removes a face from the list. The face will no longer be displayed. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Count.md) | Property that returns the number of SurfaceGraphicsFace objects in the list. |
| [Item](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Item.md) | Returns a SurfaeGraphicsFace object. |
| [Type](../SurfaceGraphicsFaceList/SurfaceGraphicsFaceList_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ComponentGraphics.DisplayedFaces](../ComponentGraphics/ComponentGraphics_DisplayedFaces.md), [SurfaceGraphics.DisplayedFaces](../SurfaceGraphics/SurfaceGraphics_DisplayedFaces.md)

## Version

Introduced in version 2009
