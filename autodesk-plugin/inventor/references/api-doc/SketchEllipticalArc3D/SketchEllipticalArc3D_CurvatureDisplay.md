# SketchEllipticalArc3D.CurvatureDisplay Property

Parent Object: [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md)

## Description

Specifies whether curvature information is displayed for the elliptical arc.

## Syntax

SketchEllipticalArc3D.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
