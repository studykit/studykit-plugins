# FaceShellProxy.IsClosed Property

Parent Object: [FaceShellProxy](../FaceShellProxy/FaceShellProxy.md)

## Description

Indicates whether the FaceShell is closed.

## Syntax

FaceShellProxy.**IsClosed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
