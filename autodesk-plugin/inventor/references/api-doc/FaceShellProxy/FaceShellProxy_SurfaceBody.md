# FaceShellProxy.SurfaceBody Property

Parent Object: [FaceShellProxy](../FaceShellProxy/FaceShellProxy.md)

## Description

Property that returns the surface body for this face shell.

## Syntax

FaceShellProxy.**SurfaceBody**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 4
