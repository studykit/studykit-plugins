# UnitsOfMeasure.LengthUnits Property

Parent Object: [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md)

## Description

Gets/Sets the default unit of length for this Document.

## Syntax

UnitsOfMeasure.**LengthUnits**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).
