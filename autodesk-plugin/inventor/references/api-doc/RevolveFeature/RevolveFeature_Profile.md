# RevolveFeature.Profile Property

Parent Object: [RevolveFeature](../RevolveFeature/RevolveFeature.md)

## Description

Gets and sets the Profile that defines the shape of the feature.

## Syntax

RevolveFeature.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 5
