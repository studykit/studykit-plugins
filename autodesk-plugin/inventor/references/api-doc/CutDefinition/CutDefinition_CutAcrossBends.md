# CutDefinition.CutAcrossBends Property

Parent Object: [CutDefinition](../CutDefinition/CutDefinition.md)

## Description

Property that returns whether this cut feature goes across bends or not.

## Syntax

CutDefinition.**CutAcrossBends**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
