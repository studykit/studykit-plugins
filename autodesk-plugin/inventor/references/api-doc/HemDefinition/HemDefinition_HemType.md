# HemDefinition.HemType Property

Parent Object: [HemDefinition](../HemDefinition/HemDefinition.md)

## Description

Property that returns the method used to define the type of hem.

## Syntax

HemDefinition.**HemType**() As [HemTypeEnum](../HemTypeEnum.md)

## Property Value

This is a read only property whose value is a [HemTypeEnum](../HemTypeEnum.md).

## Version

Introduced in version 2009
