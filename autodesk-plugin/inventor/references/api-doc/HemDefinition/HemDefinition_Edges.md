# HemDefinition.Edges Property

Parent Object: [HemDefinition](../HemDefinition/HemDefinition.md)

## Description

Gets and sets the EdgeCollection object that contains the edges that are used to define the hem feature.

## Syntax

HemDefinition.**Edges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2009
