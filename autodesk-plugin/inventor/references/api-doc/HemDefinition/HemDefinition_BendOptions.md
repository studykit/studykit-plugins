# HemDefinition.BendOptions Property

Parent Object: [HemDefinition](../HemDefinition/HemDefinition.md)

## Description

Gets and sets the bend options associated with this contour flange feature.

## Syntax

HemDefinition.**BendOptions**() As [BendOptions](../BendOptions/BendOptions.md)

## Property Value

This is a read/write property whose value is a [BendOptions](../BendOptions/BendOptions.md).

## Version

Introduced in version 2009
