# DrawingViewAnnotation.TextPosition Property

Parent Object: [DrawingViewAnnotation](../DrawingViewAnnotation/DrawingViewAnnotation.md)

## Description

Gets and sets the position of the view annotation text.

## Syntax

DrawingViewAnnotation.**TextPosition**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).
