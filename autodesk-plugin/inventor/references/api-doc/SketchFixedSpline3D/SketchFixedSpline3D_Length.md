# SketchFixedSpline3D.Length Property

Parent Object: [SketchFixedSpline3D](../SketchFixedSpline3D/SketchFixedSpline3D.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchFixedSpline3D.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 9
