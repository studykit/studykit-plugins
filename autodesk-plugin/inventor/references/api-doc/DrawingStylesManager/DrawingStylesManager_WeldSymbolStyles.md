# DrawingStylesManager.WeldSymbolStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Read-only property that returns the WeldSymbolStylesEnumerator object.

## Syntax

DrawingStylesManager.**WeldSymbolStyles**() As [WeldSymbolStylesEnumerator](../WeldSymbolStylesEnumerator/WeldSymbolStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [WeldSymbolStylesEnumerator](../WeldSymbolStylesEnumerator/WeldSymbolStylesEnumerator.md).

## Version

Introduced in version 2024
