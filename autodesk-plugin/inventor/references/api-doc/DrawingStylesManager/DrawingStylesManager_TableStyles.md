# DrawingStylesManager.TableStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the TableStylesEnumerator object. This object provides access to all the general table styles.

## Syntax

DrawingStylesManager.**TableStyles**() As [TableStylesEnumerator](../TableStylesEnumerator/TableStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [TableStylesEnumerator](../TableStylesEnumerator/TableStylesEnumerator.md).

## Version

Introduced in version 2011
