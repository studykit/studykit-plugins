# DrawingStylesManager.BalloonStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the BalloonStylesEnumerator object.

## Syntax

DrawingStylesManager.**BalloonStyles**() As [BalloonStylesEnumerator](../BalloonStylesEnumerator/BalloonStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [BalloonStylesEnumerator](../BalloonStylesEnumerator/BalloonStylesEnumerator.md).

## Version

Introduced in version 2009
