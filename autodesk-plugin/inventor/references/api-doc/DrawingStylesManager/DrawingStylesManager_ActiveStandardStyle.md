# DrawingStylesManager.ActiveStandardStyle Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Gets and sets the active standard style.

## Syntax

DrawingStylesManager.**ActiveStandardStyle**() As [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md)

## Property Value

This is a read/write property whose value is a [DrawingStandardStyle](../DrawingStandardStyle/DrawingStandardStyle.md).

## Version

Introduced in version 9
