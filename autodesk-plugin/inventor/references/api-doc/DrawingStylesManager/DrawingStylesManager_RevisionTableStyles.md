# DrawingStylesManager.RevisionTableStyles Property

Parent Object: [DrawingStylesManager](../DrawingStylesManager/DrawingStylesManager.md)

## Description

Property that returns the RevisionTableStylesEnumerator object. This object provides access to all the revision table styles.

## Syntax

DrawingStylesManager.**RevisionTableStyles**() As [RevisionTableStylesEnumerator](../RevisionTableStylesEnumerator/RevisionTableStylesEnumerator.md)

## Property Value

This is a read only property whose value is a [RevisionTableStylesEnumerator](../RevisionTableStylesEnumerator/RevisionTableStylesEnumerator.md).

## Version

Introduced in version 2011
