# BendPartFeature.BendSide Property

Parent Object: [BendPartFeature](../BendPartFeature/BendPartFeature.md)

## Description

Property that gets and sets the bend side.

## Syntax

BendPartFeature.**BendSide**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2010
