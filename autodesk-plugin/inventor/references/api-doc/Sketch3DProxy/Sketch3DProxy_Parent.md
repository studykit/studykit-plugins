# Sketch3DProxy.Parent Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that returns the parent of the object. This will return the parent PartComponentDefinition.

## Syntax

Sketch3DProxy.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 6
