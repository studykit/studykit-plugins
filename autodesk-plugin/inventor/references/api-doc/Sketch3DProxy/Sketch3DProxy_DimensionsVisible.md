# Sketch3DProxy.DimensionsVisible Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Gets and sets whether the dimensions on the sketch are visible.

## Syntax

Sketch3DProxy.**DimensionsVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
