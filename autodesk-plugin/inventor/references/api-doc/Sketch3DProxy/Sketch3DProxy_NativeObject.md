# Sketch3DProxy.NativeObject Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

Sketch3DProxy.**NativeObject**() As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Version

Introduced in version 6
