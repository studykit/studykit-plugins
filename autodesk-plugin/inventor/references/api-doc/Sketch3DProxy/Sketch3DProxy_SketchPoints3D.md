# Sketch3DProxy.SketchPoints3D Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Property that returns the SketchPoints3D collection object. This collection provides access to the existing points in the sketch.

## Syntax

Sketch3DProxy.**SketchPoints3D**() As [SketchPoints3D](../SketchPoints3D/SketchPoints3D.md)

## Property Value

This is a read only property whose value is a [SketchPoints3D](../SketchPoints3D/SketchPoints3D.md).

## Version

Introduced in version 6
