# Sketch3DProxy.DeferUpdates Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Gets and sets whether to defer the solving of the sketch.

## Syntax

Sketch3DProxy.**DeferUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
