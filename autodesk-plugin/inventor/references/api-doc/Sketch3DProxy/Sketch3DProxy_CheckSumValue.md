# Sketch3DProxy.CheckSumValue Property

Parent Object: [Sketch3DProxy](../Sketch3DProxy/Sketch3DProxy.md)

## Description

Gets sketch checksum value.

## Syntax

Sketch3DProxy.**CheckSumValue**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2020
