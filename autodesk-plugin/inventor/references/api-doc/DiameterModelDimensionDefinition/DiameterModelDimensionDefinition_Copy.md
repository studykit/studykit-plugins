# DiameterModelDimensionDefinition.Copy Method

Parent Object: [DiameterModelDimensionDefinition](../DiameterModelDimensionDefinition/DiameterModelDimensionDefinition.md)

## Description

Method that creates a copy of this DiameterModelDimensionDefinition object. The new DiameterModelDimensionDefinition object is independent of any dimension. It can edited and used as input to edit an existing dimension or to create a new dimension.

## Syntax

DiameterModelDimensionDefinition.**Copy**() As [DiameterModelDimensionDefinition](../DiameterModelDimensionDefinition/DiameterModelDimensionDefinition.md)

## Version

Introduced in version 2018
