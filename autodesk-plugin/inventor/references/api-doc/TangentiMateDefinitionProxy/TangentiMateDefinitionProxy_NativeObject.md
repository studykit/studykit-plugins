# TangentiMateDefinitionProxy.NativeObject Property

Parent Object: [TangentiMateDefinitionProxy](../TangentiMateDefinitionProxy/TangentiMateDefinitionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

TangentiMateDefinitionProxy.**NativeObject**() As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Version

Introduced in version 6
