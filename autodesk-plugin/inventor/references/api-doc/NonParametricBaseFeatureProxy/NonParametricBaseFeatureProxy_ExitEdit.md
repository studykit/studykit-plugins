# NonParametricBaseFeatureProxy.ExitEdit Method

Parent Object: [NonParametricBaseFeatureProxy](../NonParametricBaseFeatureProxy/NonParametricBaseFeatureProxy.md)

## Description

Exits the edit context of the base feature.

## Syntax

NonParametricBaseFeatureProxy.**ExitEdit**()

## Version

Introduced in version 11
