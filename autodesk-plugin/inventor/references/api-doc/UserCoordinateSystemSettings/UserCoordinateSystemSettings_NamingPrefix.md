# UserCoordinateSystemSettings.NamingPrefix Property

Parent Object: [UserCoordinateSystemSettings](../UserCoordinateSystemSettings/UserCoordinateSystemSettings.md)

## Description

Gets and sets the prefix to use for the name of new user coordinate systems.

## Syntax

UserCoordinateSystemSettings.**NamingPrefix**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
