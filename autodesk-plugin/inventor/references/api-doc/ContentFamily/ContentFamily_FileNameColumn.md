# ContentFamily.FileNameColumn Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that returns the column representing family file name values.

## Syntax

ContentFamily.**FileNameColumn**() As [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md)

## Property Value

This is a read only property whose value is a [ContentTableColumn](../ContentTableColumn/ContentTableColumn.md).

## Version

Introduced in version 2011
