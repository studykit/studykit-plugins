# ContentFamily.StandardRevision Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Gets/Sets the standard revision this family is based on.

## Syntax

ContentFamily.**StandardRevision**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
