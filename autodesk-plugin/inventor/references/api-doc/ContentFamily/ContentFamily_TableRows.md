# ContentFamily.TableRows Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that returns the collection of rows for the table associated with this family.

## Syntax

ContentFamily.**TableRows**() As [ContentTableRows](../ContentTableRows/ContentTableRows.md)

## Property Value

This is a read only property whose value is a [ContentTableRows](../ContentTableRows/ContentTableRows.md).
