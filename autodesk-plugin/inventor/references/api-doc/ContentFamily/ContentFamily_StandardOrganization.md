# ContentFamily.StandardOrganization Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Gets/Sets the standard organization this family is part of.

## Syntax

ContentFamily.**StandardOrganization**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
