# ContentFamily.IsModifiable Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that indicates if this library is writable or not. Returns True in the case where the library is modifiable.

## Syntax

ContentFamily.**IsModifiable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2010
