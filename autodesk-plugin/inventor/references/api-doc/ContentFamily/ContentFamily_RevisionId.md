# ContentFamily.RevisionId Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Property that returns the family revision Id, which is a GUID in string format.

## Syntax

ContentFamily.**RevisionId**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2011
