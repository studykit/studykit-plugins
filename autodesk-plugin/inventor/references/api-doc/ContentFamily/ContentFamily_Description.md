# ContentFamily.Description Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Gets/Sets the description of this family.

## Syntax

ContentFamily.**Description**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
