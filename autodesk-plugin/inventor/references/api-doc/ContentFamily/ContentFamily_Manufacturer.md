# ContentFamily.Manufacturer Property

Parent Object: [ContentFamily](../ContentFamily/ContentFamily.md)

## Description

Gets/Sets the manufacturer of the parts within this family.

## Syntax

ContentFamily.**Manufacturer**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
