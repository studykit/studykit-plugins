# AssetTexture.TextureType Property

Parent Object: [AssetTexture](../AssetTexture/AssetTexture.md)

## Description

Gets the current texture type.

## Syntax

AssetTexture.**TextureType**() As [AssetTextureTypeEnum](../AssetTextureTypeEnum.md)

## Property Value

This is a read only property whose value is an [AssetTextureTypeEnum](../AssetTextureTypeEnum.md).
