# FaceDraftFeatures Object

## Description

The FaceDraftFeatures object represents a collection of face draft features ( objects) on a part.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../FaceDraftFeatures/FaceDraftFeatures_Add.md) | Method that creates a new FaceDraft feature. |
| [CreateFaceDraftDefinition](../FaceDraftFeatures/FaceDraftFeatures_CreateFaceDraftDefinition.md) | Method that creates a new FaceDraftDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FaceDraftFeatures/FaceDraftFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FaceDraftFeatures/FaceDraftFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../FaceDraftFeatures/FaceDraftFeatures_Item.md) | Returns the specified object in the collection. The index can be numeric or the object name. |
| [Type](../FaceDraftFeatures/FaceDraftFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.FaceDraftFeatures](../PartFeatures/PartFeatures_FaceDraftFeatures.md), [SheetMetalFeatures.FaceDraftFeatures](../SheetMetalFeatures/SheetMetalFeatures_FaceDraftFeatures.md)

## Version

Introduced in version 5
