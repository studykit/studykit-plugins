# UnitsFormatting.Units Property

Parent Object: [UnitsFormatting](../UnitsFormatting/UnitsFormatting.md)

## Description

Gets and sets the units.

## Syntax

UnitsFormatting.**Units**() As [UnitsTypeEnum](../UnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [UnitsTypeEnum](../UnitsTypeEnum.md).

## Version

Introduced in version 2021
