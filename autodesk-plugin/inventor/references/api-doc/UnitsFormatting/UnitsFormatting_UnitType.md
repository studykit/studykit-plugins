# UnitsFormatting.UnitType Property

Parent Object: [UnitsFormatting](../UnitsFormatting/UnitsFormatting.md)

## Description

Gets and sets the value units type.

## Syntax

UnitsFormatting.**UnitType**() As [ValueUnitsTypeEnum](../ValueUnitsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [ValueUnitsTypeEnum](../ValueUnitsTypeEnum.md).

## Version

Introduced in version 2021
