# CosmeticWeld.Definition Property

Parent Object: [CosmeticWeld](../CosmeticWeld/CosmeticWeld.md)

## Description

Read-write property that gets and sets the definition object which defines the various inputs that were used to create the cosmetic weld.

## Syntax

CosmeticWeld.**Definition**() As [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md)

## Property Value

This is a read/write property whose value is a [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md).

## Version

Introduced in version 2017
