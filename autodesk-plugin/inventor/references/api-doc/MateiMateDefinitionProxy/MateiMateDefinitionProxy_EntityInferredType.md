# MateiMateDefinitionProxy.EntityInferredType Property

Parent Object: [MateiMateDefinitionProxy](../MateiMateDefinitionProxy/MateiMateDefinitionProxy.md)

## Description

Property that returns an enum indicating how the geometry of the entity is interpreted.

## Syntax

MateiMateDefinitionProxy.**EntityInferredType**() As [InferredTypeEnum](../InferredTypeEnum.md)

## Property Value

This is a read only property whose value is an [InferredTypeEnum](../InferredTypeEnum.md).

## Version

Introduced in version 6
