# MateiMateDefinitionProxy.Suppressed Property

Parent Object: [MateiMateDefinitionProxy](../MateiMateDefinitionProxy/MateiMateDefinitionProxy.md)

## Description

Gets the boolean flag that indicates whether this iMateDefinition is suppressed or not.

## Syntax

MateiMateDefinitionProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
