# PrintOrientationEnum Enumerator

## Description

Constants denoting the orientation of the print.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kDefaultOrientation | 13571 | Default setting for print orientation. |
| kLandscapeOrientation | 13570 | Print with Landscape orientation. |
| kPortraitOrientation | 13569 | Print with Portrait orientation. |

## Version

Introduced in version 5
