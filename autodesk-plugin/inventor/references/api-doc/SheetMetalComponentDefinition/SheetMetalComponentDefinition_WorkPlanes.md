# SheetMetalComponentDefinition.WorkPlanes Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the WorkPlanes collection object.

## Syntax

SheetMetalComponentDefinition.**WorkPlanes**() As [WorkPlanes](../WorkPlanes/WorkPlanes.md)

## Property Value

This is a read only property whose value is a [WorkPlanes](../WorkPlanes/WorkPlanes.md).

## Version

Introduced in version 5.3
