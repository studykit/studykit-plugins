# SheetMetalComponentDefinition.WorkPoints Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the WorkPoints collection object.

## Syntax

SheetMetalComponentDefinition.**WorkPoints**() As [WorkPoints](../WorkPoints/WorkPoints.md)

## Property Value

This is a read only property whose value is a [WorkPoints](../WorkPoints/WorkPoints.md).

## Version

Introduced in version 5.3
