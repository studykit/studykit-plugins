# SheetMetalComponentDefinition.DataIO Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Gets the object that directly deals with I/O to and from a storage-medium, including Streams(IStream).

## Syntax

SheetMetalComponentDefinition.**DataIO**() As [DataIO](../DataIO/DataIO.md)

## Property Value

This is a read only property whose value is a [DataIO](../DataIO/DataIO.md).

## Version

Introduced in version 5.3
