# SheetMetalComponentDefinition.UserCoordinateSystems Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the UserCoordinateSystems collection object.

## Syntax

SheetMetalComponentDefinition.**UserCoordinateSystems**() As [UserCoordinateSystems](../UserCoordinateSystems/UserCoordinateSystems.md)

## Property Value

This is a read only property whose value is a [UserCoordinateSystems](../UserCoordinateSystems/UserCoordinateSystems.md).

## Version

Introduced in version 2010
