# SheetMetalComponentDefinition.SketchBlockDefinitions Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the SketchBlockDefinitions collection object.

## Syntax

SheetMetalComponentDefinition.**SketchBlockDefinitions**() As [SketchBlockDefinitions](../SketchBlockDefinitions/SketchBlockDefinitions.md)

## Property Value

This is a read only property whose value is a [SketchBlockDefinitions](../SketchBlockDefinitions/SketchBlockDefinitions.md).

## Version

Introduced in version 2010
