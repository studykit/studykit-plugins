# SheetMetalComponentDefinition.Document Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the containing Document object.

## Syntax

SheetMetalComponentDefinition.**Document**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 5.3
