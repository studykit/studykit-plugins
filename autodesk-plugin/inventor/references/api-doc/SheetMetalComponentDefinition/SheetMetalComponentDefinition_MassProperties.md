# SheetMetalComponentDefinition.MassProperties Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Mass properties for this ComponentDefinition.

## Syntax

SheetMetalComponentDefinition.**MassProperties**() As [MassProperties](../MassProperties/MassProperties.md)

## Property Value

This is a read only property whose value is a [MassProperties](../MassProperties/MassProperties.md).

## Version

Introduced in version 5.3
