# SheetMetalComponentDefinition.ActiveSheetMetalStyle Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the active SheetMetalStyle object.

## Syntax

SheetMetalComponentDefinition.**ActiveSheetMetalStyle**() As [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Property Value

This is a read only property whose value is a [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md).
