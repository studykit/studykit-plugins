# SheetMetalComponentDefinition.ExportObjects Method

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Marks all the input objects as exported.

## Syntax

SheetMetalComponentDefinition.**ExportObjects**( ***Objects*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Objects | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Object collection to mark as exported. |

## Version

Introduced in version 2011
