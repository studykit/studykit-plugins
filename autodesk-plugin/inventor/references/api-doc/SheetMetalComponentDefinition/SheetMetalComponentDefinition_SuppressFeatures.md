# SheetMetalComponentDefinition.SuppressFeatures Method

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Method that suppresses the specified features.

## Syntax

SheetMetalComponentDefinition.**SuppressFeatures**( ***Features*** As [ObjectCollection](../ObjectCollection/ObjectCollection.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Features | [ObjectCollection](../ObjectCollection/ObjectCollection.md) | Collection of features to suppress. |

## Version

Introduced in version 2009
