# SheetMetalComponentDefinition.AnalysisManager Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Property that returns the AnalysisManager object.

## Syntax

SheetMetalComponentDefinition.**AnalysisManager**() As [AnalysisManager](../AnalysisManager/AnalysisManager.md)

## Property Value

This is a read only property whose value is an [AnalysisManager](../AnalysisManager/AnalysisManager.md).

## Version

Introduced in version 2009
