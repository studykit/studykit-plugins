# SheetMetalComponentDefinition.ModelStates Property

Parent Object: [SheetMetalComponentDefinition](../SheetMetalComponentDefinition/SheetMetalComponentDefinition.md)

## Description

Read-only property that returns the ModelStates object.

## Syntax

SheetMetalComponentDefinition.**ModelStates**() As [ModelStates](../ModelStates/ModelStates.md)

## Property Value

This is a read only property whose value is a [ModelStates](../ModelStates/ModelStates.md).

## Version

Introduced in version 2022
