# ShellFeatureProxy.NativeObject Property

Parent Object: [ShellFeatureProxy](../ShellFeatureProxy/ShellFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ShellFeatureProxy.**NativeObject**() As [ShellFeature](../ShellFeature/ShellFeature.md)

## Property Value

This is a read only property whose value is a [ShellFeature](../ShellFeature/ShellFeature.md).

## Version

Introduced in version 9
