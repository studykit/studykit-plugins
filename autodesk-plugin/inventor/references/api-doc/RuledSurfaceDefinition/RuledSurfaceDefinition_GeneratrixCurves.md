# RuledSurfaceDefinition.GeneratrixCurves Property

Parent Object: [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md)

## Description

Read-write property that gets and sets the generatrix curves base on which to create the ruled surface.

## Syntax

RuledSurfaceDefinition.**GeneratrixCurves**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2017
