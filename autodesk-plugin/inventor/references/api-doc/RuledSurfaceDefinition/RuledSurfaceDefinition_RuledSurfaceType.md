# RuledSurfaceDefinition.RuledSurfaceType Property

Parent Object: [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md)

## Description

Read-only property that returns the ruled surface type.

## Syntax

RuledSurfaceDefinition.**RuledSurfaceType**() As [RuledSurfaceTypeEnum](../RuledSurfaceTypeEnum.md)

## Property Value

This is a read only property whose value is a [RuledSurfaceTypeEnum](../RuledSurfaceTypeEnum.md).

## Version

Introduced in version 2016
