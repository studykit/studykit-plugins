# RuledSurfaceDefinition.FlipExtendDirection Property

Parent Object: [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md)

## Description

Read-write property that gets and sets whether to flip the extend direction of the ruled surface.

## Syntax

RuledSurfaceDefinition.**FlipExtendDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
