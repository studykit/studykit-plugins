# ModelDatumTargetProxy.TargetPoint Property

Parent Object: [ModelDatumTargetProxy](../ModelDatumTargetProxy/ModelDatumTargetProxy.md)

## Description

Gets and sets the placement point of this model datum target object.

## Syntax

ModelDatumTargetProxy.**TargetPoint**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2023
