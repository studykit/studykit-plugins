# DrawingViewLabel.ConstrainToBorder Property

Parent Object: [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Description

Gets and sets whether to constrain the view label to the view.

## Syntax

DrawingViewLabel.**ConstrainToBorder**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
