# DrawingViewLabel.RestoreViewLabel Method

Parent Object: [DrawingViewLabel](../DrawingViewLabel/DrawingViewLabel.md)

## Description

Method that restores the drawing view label to default position.

## Syntax

DrawingViewLabel.**RestoreViewLabel**()

## Version

Introduced in version 2025.2
