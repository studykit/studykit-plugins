# OfflineImageFidelityEnum Enumerator

## Description

Enum indicating the printing quality for a shaded drawing view when defer updates is active.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kBestImageFidelity | 37380 | Printing quality of 200 dpi. |
| kBetterImageFidelity | 37379 | Printing quality of 150 dpi. |
| kLowerImageFidelity | 37377 | Printing quality of 50 dpi. |
| kStandardImageFidelity | 37378 | Printing quality of 100 dpi. |

## Version

Introduced in version 8
