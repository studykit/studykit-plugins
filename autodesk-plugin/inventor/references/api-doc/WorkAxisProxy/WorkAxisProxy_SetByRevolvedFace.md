# WorkAxisProxy.SetByRevolvedFace Method

Parent Object: [WorkAxisProxy](../WorkAxisProxy/WorkAxisProxy.md)

## Description

Method that redefines the work axis to be at the axis of the input revolved face.

## Syntax

WorkAxisProxy.**SetByRevolvedFace**( ***Face*** As [Face](../Face/Face.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Face | [Face](../Face/Face.md) | Input Face whose geometry is a revolved surface. Valid surface types include cylinders, cones, and toroids. |

## Version

Introduced in version 2008
