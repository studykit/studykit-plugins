# WorkAxisProxy.SetByAnalyticEdge Method

Parent Object: [WorkAxisProxy](../WorkAxisProxy/WorkAxisProxy.md)

## Description

Method that redefines the work axis to be based on the input edge.

## Syntax

WorkAxisProxy.**SetByAnalyticEdge**( ***Edge*** As [Edge](../Edge/Edge.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Edge | [Edge](../Edge/Edge.md) | Input Edge object that can be a circle, arc, or an ellipse. |

## Version

Introduced in version 2008
