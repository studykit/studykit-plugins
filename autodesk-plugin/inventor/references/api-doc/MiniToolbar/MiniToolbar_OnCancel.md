# MiniToolbar.OnCancel Event

Parent Object: [MiniToolbar](../MiniToolbar/MiniToolbar.md)

## Description

Event that fires when the ‘Cancel’ button is clicked.

## Syntax

MiniToolbar.**OnCancel**()
