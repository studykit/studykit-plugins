# ModelCompositeAnnotationProxy.NativeObject Property

Parent Object: [ModelCompositeAnnotationProxy](../ModelCompositeAnnotationProxy/ModelCompositeAnnotationProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ModelCompositeAnnotationProxy.**NativeObject**() As [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md)

## Property Value

This is a read only property whose value is a [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md).

## Version

Introduced in version 2018
