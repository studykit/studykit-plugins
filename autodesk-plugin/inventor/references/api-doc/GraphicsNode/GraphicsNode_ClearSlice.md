# GraphicsNode.ClearSlice Method

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Method that clears all the slicing applied to the graphics node.

## Syntax

GraphicsNode.**ClearSlice**()

## Version

Introduced in version 2010
