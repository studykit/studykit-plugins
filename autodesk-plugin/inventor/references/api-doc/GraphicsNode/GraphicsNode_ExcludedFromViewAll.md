# GraphicsNode.ExcludedFromViewAll Property

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Specifies that the node is not considered when doing a view all.

## Syntax

GraphicsNode.**ExcludedFromViewAll**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
