# GraphicsNode.AddScalableTextGraphics Method

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Method that creates a new (scalable) TextGraphics graphic primitive.

## Syntax

GraphicsNode.**AddScalableTextGraphics**() As [TextGraphics](../TextGraphics/TextGraphics.md)

## Version

Introduced in version 2008
