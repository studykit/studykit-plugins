# GraphicsNode.ItemById Property

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Returns the specified GraphicsPrimitive from the collection using its Id as index.

## Syntax

GraphicsNode.**ItemById**( ***DataSetId*** As Long ) As [GraphicsPrimitive](../GraphicsPrimitive/GraphicsPrimitive.md)

## Property Value

This is a read only property whose value is a [GraphicsPrimitive](../GraphicsPrimitive/GraphicsPrimitive.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DataSetId | Long | Input Long value that indicates the Id of a GraphicsPrimitive object to return. |

## Version

Introduced in version 2017
