# GraphicsNode.Item Property

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Returns the specified graphic entity from the collection.

## Syntax

GraphicsNode.**Item**( ***Index*** As Long ) As [GraphicsPrimitive](../GraphicsPrimitive/GraphicsPrimitive.md)

## Property Value

This is a read only property whose value is a [GraphicsPrimitive](../GraphicsPrimitive/GraphicsPrimitive.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Specifies which object to return. This is an index into the collection. |

## Version

Introduced in version 5
