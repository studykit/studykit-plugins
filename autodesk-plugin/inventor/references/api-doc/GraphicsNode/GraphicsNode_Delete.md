# GraphicsNode.Delete Method

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Method that deletes the GraphicsNode. This also deletes all associated graphic primitives.

## Syntax

GraphicsNode.**Delete**()

## Version

Introduced in version 5
