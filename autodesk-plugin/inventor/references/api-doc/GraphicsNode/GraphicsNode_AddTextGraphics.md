# GraphicsNode.AddTextGraphics Method

Parent Object: [GraphicsNode](../GraphicsNode/GraphicsNode.md)

## Description

Method that creates a new TextGraphics graphic primitive.

## Syntax

GraphicsNode.**AddTextGraphics**() As [TextGraphics](../TextGraphics/TextGraphics.md)
