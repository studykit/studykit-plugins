# CornerRoundFeatures.Add Method

Parent Object: [CornerRoundFeatures](../CornerRoundFeatures/CornerRoundFeatures.md)

## Description

Method that creates a new sheet metal corner round feature.

## Syntax

CornerRoundFeatures.**Add**( ***CornerRoundDefinition*** As [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md) ) As [CornerRoundFeature](../CornerRoundFeature/CornerRoundFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| CornerRoundDefinition | [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md) | Input CornerRoundDefinition object that defines the various edges and radii. |

## Version

Introduced in version 2010
