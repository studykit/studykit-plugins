# CornerRoundFeatures Object

## Description

The CornerRoundFeatures collection object provides access to all of the objects in a sheet metal component definition.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../CornerRoundFeatures/CornerRoundFeatures_Add.md) | Method that creates a new sheet metal corner round feature. |
| [CreateCornerRoundDefinition](../CornerRoundFeatures/CornerRoundFeatures_CreateCornerRoundDefinition.md) | Method that creates a new CornerRoundDefinition object. |
| [GetEdgesFromFeature](../CornerRoundFeatures/CornerRoundFeatures_GetEdgesFromFeature.md) | Method that gets the edges from a part feature that can be used to create corner round feature. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CornerRoundFeatures/CornerRoundFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CornerRoundFeatures/CornerRoundFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../CornerRoundFeatures/CornerRoundFeatures_Item.md) | Returns the specified CornerRoundFeature object from the collection. This is the default property of the CornerRoundFeatures collection object. |
| [Type](../CornerRoundFeatures/CornerRoundFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[FlatPatternFeatures.CornerRoundFeatures](../FlatPatternFeatures/FlatPatternFeatures_CornerRoundFeatures.md), [SheetMetalFeatures.CornerRoundFeatures](../SheetMetalFeatures/SheetMetalFeatures_CornerRoundFeatures.md)

## Version

Introduced in version 5.3
