# DiameterDimConstraintProxy.DimensionCenterPoint Property

Parent Object: [DiameterDimConstraintProxy](../DiameterDimConstraintProxy/DiameterDimConstraintProxy.md)

## Description

Gets the center of the dimension line.

## Syntax

DiameterDimConstraintProxy.**DimensionCenterPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
