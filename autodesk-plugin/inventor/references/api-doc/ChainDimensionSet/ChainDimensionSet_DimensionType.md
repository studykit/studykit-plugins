# ChainDimensionSet.DimensionType Property

Parent Object: [ChainDimensionSet](../ChainDimensionSet/ChainDimensionSet.md)

## Description

Property that returns the dimension type of the chain set. Possible values are kAlignedDimensionType, kHorizontalDimensionType and kVerticalDimensionType.

## Syntax

ChainDimensionSet.**DimensionType**() As [DimensionTypeEnum](../DimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [DimensionTypeEnum](../DimensionTypeEnum.md).

## Version

Introduced in version 2011
