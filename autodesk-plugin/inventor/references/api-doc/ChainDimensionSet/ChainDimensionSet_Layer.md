# ChainDimensionSet.Layer Property

Parent Object: [ChainDimensionSet](../ChainDimensionSet/ChainDimensionSet.md)

## Description

Read-write property that gets and sets the layer for the dimension set.

## Syntax

ChainDimensionSet.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2011
