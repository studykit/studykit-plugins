# DerivedAssemblyOccurrence.InclusionOption Property

Parent Object: [DerivedAssemblyOccurrence](../DerivedAssemblyOccurrence/DerivedAssemblyOccurrence.md)

## Description

Property that defines if this occurrence is included and if it is how it is used in the derived assembly. Valid values for this property are kDerivedIncludeAll, kDerivedSubtractAll, kDerivedExcludeAll, kDerivedBoundingBox, kDerivedIntersect and kDerivedIndividualDefined.

## Syntax

DerivedAssemblyOccurrence.**InclusionOption**() As [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md).

## Version

Introduced in version 5.3
