# PunchNote.DimensionStyle Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets/Sets the DimensionStyle associated with the LeaderNote.

## Syntax

PunchNote.**DimensionStyle**() As [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Property Value

This is a read/write property whose value is a [DimensionStyle](../DimensionStyle/DimensionStyle.md).

## Version

Introduced in version 2010
