# PunchNote.UseBackgroundColor Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets/Sets the UseBackgroundColor associated with the LeaderNote.

## Syntax

PunchNote.**UseBackgroundColor**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
