# PunchNote.QuantityDefinition Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets and sets how the quantity value is set for the note.

## Syntax

PunchNote.**QuantityDefinition**() As [PunchNoteQuantityEnum](../PunchNoteQuantityEnum.md)

## Property Value

This is a read/write property whose value is a [PunchNoteQuantityEnum](../PunchNoteQuantityEnum.md).

## Version

Introduced in version 2010
