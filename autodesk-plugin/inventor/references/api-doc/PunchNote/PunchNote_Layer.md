# PunchNote.Layer Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets/Sets the layer used by the DrawingNote.

## Syntax

PunchNote.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2010
