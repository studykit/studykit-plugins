# PunchNote.WidthScale Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets/Sets the width scale factor of the DrawingNote.

## Syntax

PunchNote.**WidthScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
