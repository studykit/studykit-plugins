# PunchNote.StackedTextPosition Property

Parent Object: [PunchNote](../PunchNote/PunchNote.md)

## Description

Gets and sets the position (alignment) of the stacked text with respect to regular text.

## Syntax

PunchNote.**StackedTextPosition**() As [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [VerticalTextAlignmentEnum](../VerticalTextAlignmentEnum.md).

## Version

Introduced in version 2010
