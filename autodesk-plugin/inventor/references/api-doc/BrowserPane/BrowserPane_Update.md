# BrowserPane.Update Method

Parent Object: [BrowserPane](../BrowserPane/BrowserPane.md)

## Description

Refreshes the browser view. Should be used after changing browser node status (e.g. DisplayState).

## Syntax

BrowserPane.**Update**()

## Version

Introduced in version 11
