# BrowserPane.TopNode Property

Parent Object: [BrowserPane](../BrowserPane/BrowserPane.md)

## Description

Gets the top browser node.

## Syntax

BrowserPane.**TopNode**() As [BrowserNode](../BrowserNode/BrowserNode.md)

## Property Value

This is a read only property whose value is a [BrowserNode](../BrowserNode/BrowserNode.md).
