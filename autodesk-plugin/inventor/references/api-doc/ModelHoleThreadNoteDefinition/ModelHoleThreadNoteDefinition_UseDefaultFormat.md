# ModelHoleThreadNoteDefinition.UseDefaultFormat Property

Parent Object: [ModelHoleThreadNoteDefinition](../ModelHoleThreadNoteDefinition/ModelHoleThreadNoteDefinition.md)

## Description

Gets and sets whether to use the default format for hole notes.

## Syntax

ModelHoleThreadNoteDefinition.**UseDefaultFormat**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
