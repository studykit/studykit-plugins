# SketchArcProxy.LineWeight Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Gets and sets the lineWeight applied to this sketch line.

## Syntax

SketchArcProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
