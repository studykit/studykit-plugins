# SketchArcProxy.Radius Property

Parent Object: [SketchArcProxy](../SketchArcProxy/SketchArcProxy.md)

## Description

Property that returns the radius of the arc.

## Syntax

SketchArcProxy.**Radius**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 6
