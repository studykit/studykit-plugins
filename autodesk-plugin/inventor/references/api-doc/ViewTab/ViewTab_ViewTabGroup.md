# ViewTab.ViewTabGroup Property

Parent Object: [ViewTab](../ViewTab/ViewTab.md)

## Description

Read-only property that returns the ViewTabGroup this view tab is located in. This returns Nothing if the ViewTab is not grouped.

## Syntax

ViewTab.**ViewTabGroup**() As [ViewTabGroup](../ViewTabGroup/ViewTabGroup.md)

## Property Value

This is a read only property whose value is a [ViewTabGroup](../ViewTabGroup/ViewTabGroup.md).

## Version

Introduced in version 2022
