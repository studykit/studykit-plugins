# ModelLeader.RootNode Property

Parent Object: [ModelLeader](../ModelLeader/ModelLeader.md)

## Description

Read-only property that returns the root node of this leader. This property will return Nothing if the HasRootNode property returns False.

## Syntax

ModelLeader.**RootNode**() As [ModelLeaderNode](../ModelLeaderNode/ModelLeaderNode.md)

## Property Value

This is a read only property whose value is a [ModelLeaderNode](../ModelLeaderNode/ModelLeaderNode.md).

## Version

Introduced in version 2018
