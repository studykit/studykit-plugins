# MoveFaceFeature.Suppressed Property

Parent Object: [MoveFaceFeature](../MoveFaceFeature/MoveFaceFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

MoveFaceFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 10
