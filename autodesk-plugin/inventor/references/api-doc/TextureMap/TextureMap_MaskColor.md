# TextureMap.MaskColor Property

Parent Object: [TextureMap](../TextureMap/TextureMap.md)

## Description

Gets the mask color for the texture map image.

## Syntax

TextureMap.**MaskColor**() As [Color](../Color/Color.md)

## Property Value

This is a read only property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
