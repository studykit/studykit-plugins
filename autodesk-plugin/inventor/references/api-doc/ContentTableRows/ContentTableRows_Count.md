# ContentTableRows.Count Property

Parent Object: [ContentTableRows](../ContentTableRows/ContentTableRows.md)

## Description

Property that returns the number of ContentTableRow objects in the collection.

## Syntax

ContentTableRows.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.
