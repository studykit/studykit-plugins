# SketchFixedSpline3DProxy.Geometry Property

Parent Object: [SketchFixedSpline3DProxy](../SketchFixedSpline3DProxy/SketchFixedSpline3DProxy.md)

## Description

Gets and sets a BSplineCurve3d geometry object. The object returned represents a 'snapshot' view of the current state of the sketch fixed spline.

## Syntax

SketchFixedSpline3DProxy.**Geometry**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read/write property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 9
