# Column.Parent Property

Parent Object: [Column](../Column/Column.md)

## Description

Property that returns the parent Table.

## Syntax

Column.**Parent**() As [CustomTable](../CustomTable/CustomTable.md)

## Property Value

This is a read only property whose value is a [CustomTable](../CustomTable/CustomTable.md).

## Version

Introduced in version 9
