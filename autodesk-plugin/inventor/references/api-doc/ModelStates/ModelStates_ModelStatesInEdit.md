# ModelStates.ModelStatesInEdit Property

Parent Object: [ModelStates](../ModelStates/ModelStates.md)

## Description

Read-write property that gets and sets the model states in edit.

## Syntax

ModelStates.**ModelStatesInEdit**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).
