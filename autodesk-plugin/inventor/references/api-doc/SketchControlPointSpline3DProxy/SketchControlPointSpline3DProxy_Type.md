# SketchControlPointSpline3DProxy.Type Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchControlPointSpline3DProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2014
