# SketchControlPointSpline3DProxy.ControlPointCount Property

Parent Object: [SketchControlPointSpline3DProxy](../SketchControlPointSpline3DProxy/SketchControlPointSpline3DProxy.md)

## Description

Read-only property that returns the number of control points defining the control point spline.

## Syntax

SketchControlPointSpline3DProxy.**ControlPointCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2014
