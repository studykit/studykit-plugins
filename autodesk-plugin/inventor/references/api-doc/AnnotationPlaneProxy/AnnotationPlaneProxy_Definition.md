# AnnotationPlaneProxy.Definition Property

Parent Object: [AnnotationPlaneProxy](../AnnotationPlaneProxy/AnnotationPlaneProxy.md)

## Description

Read-only property that returns the annotation plane definition object associated with this annotation plane.

## Syntax

AnnotationPlaneProxy.**Definition**() As [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Property Value

This is a read only property whose value is an [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md).

## Version

Introduced in version 2018
