AnnotationPlaneProxy Object.

## Methods

|  |  |
| --- | --- |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Annotations](../AnnotationPlaneProxy/AnnotationPlaneProxy_Annotations.md) | Read-only property that returns the annotations associated with this annotation plane. |
| [Application](../AnnotationPlaneProxy/AnnotationPlaneProxy_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [AttributeSets](../AnnotationPlaneProxy/AnnotationPlaneProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../AnnotationPlaneProxy/AnnotationPlaneProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Definition](../AnnotationPlaneProxy/AnnotationPlaneProxy_Definition.md) | Read-only property that returns the annotation plane definition object associated with this annotation plane. |
| [NativeObject](../AnnotationPlaneProxy/AnnotationPlaneProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../AnnotationPlaneProxy/AnnotationPlaneProxy_Parent.md) | Read-only property that returns the parent component definition of the object. |
| [Type](../AnnotationPlaneProxy/AnnotationPlaneProxy_Type.md) | Read-only property returning kAnnotationPlaneObject indicating this objects type. |
