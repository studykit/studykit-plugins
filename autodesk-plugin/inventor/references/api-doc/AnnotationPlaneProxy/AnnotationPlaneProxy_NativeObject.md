# AnnotationPlaneProxy.NativeObject Property

Parent Object: [AnnotationPlaneProxy](../AnnotationPlaneProxy/AnnotationPlaneProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

AnnotationPlaneProxy.**NativeObject**() As [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md)

## Property Value

This is a read only property whose value is an [AnnotationPlane](../AnnotationPlane/AnnotationPlane.md).

## Version

Introduced in version 2018
