# ModelWeldingSymbolDefinitions.AssociatedGeometries Property

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Gets and sets the associated geometries. Valid geometries include faces, edges and vertices.

## Syntax

ModelWeldingSymbolDefinitions.**AssociatedGeometries**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2024
