# ModelWeldingSymbolDefinitions.Copy Method

Parent Object: [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Description

Method that creates a copy of this ModelWeldingSymbolDefinitions object. The new ModelWeldingSymbolDefinitions object is independent of any model welding symbol. It can be edited and used as input to edit an existing model welding symbol or to create a new m.

## Syntax

ModelWeldingSymbolDefinitions.**Copy**() As [ModelWeldingSymbolDefinitions](../ModelWeldingSymbolDefinitions/ModelWeldingSymbolDefinitions.md)

## Version

Introduced in version 2024
