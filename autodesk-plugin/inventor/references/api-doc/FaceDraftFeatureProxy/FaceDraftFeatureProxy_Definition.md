# FaceDraftFeatureProxy.Definition Property

Parent Object: [FaceDraftFeatureProxy](../FaceDraftFeatureProxy/FaceDraftFeatureProxy.md)

## Description

Read-write property that gets and sets the FaceDraftDefinition object associated with this feature.

## Syntax

FaceDraftFeatureProxy.**Definition**() As [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Property Value

This is a read/write property whose value is a [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md).

## Version

Introduced in version 2016
