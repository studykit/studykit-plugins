# TitleBlock.Location Property

Parent Object: [TitleBlock](../TitleBlock/TitleBlock.md)

## Description

Property that returns the position of the title block on the sheet.

## Syntax

TitleBlock.**Location**() As [TitleBlockLocationEnum](../TitleBlockLocationEnum.md)

## Property Value

This is a read only property whose value is a [TitleBlockLocationEnum](../TitleBlockLocationEnum.md).

## Version

Introduced in version 5.3
