# MarkStyle.MarkDepthType Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that gets and sets the mark depth type.

## Syntax

MarkStyle.**MarkDepthType**() As [MarkTypeEnum](../MarkTypeEnum.md)

## Property Value

This is a read/write property whose value is a [MarkTypeEnum](../MarkTypeEnum.md).

## Version

Introduced in version 2023
