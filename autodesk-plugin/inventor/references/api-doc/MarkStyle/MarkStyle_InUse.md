# MarkStyle.InUse Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that returns whether this style is in use.

## Syntax

MarkStyle.**InUse**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2023
