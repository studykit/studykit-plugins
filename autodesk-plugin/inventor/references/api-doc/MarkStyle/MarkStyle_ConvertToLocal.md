# MarkStyle.ConvertToLocal Method

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

MarkStyle.**ConvertToLocal**() As [MarkStyle](../MarkStyle/MarkStyle.md)

## Version

Introduced in version 2023
