# MarkStyle.InternalName Property

Parent Object: [MarkStyle](../MarkStyle/MarkStyle.md)

## Description

Property that returns the unique name of the style.

## Syntax

MarkStyle.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2023
