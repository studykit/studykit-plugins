# DimensionStyle.UseCapitalXForEquallySpacedFeatures Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to use capital X for equally spaced features or not.

## Syntax

DimensionStyle.**UseCapitalXForEquallySpacedFeatures**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2026
