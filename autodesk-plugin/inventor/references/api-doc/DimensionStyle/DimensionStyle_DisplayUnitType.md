# DimensionStyle.DisplayUnitType Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display units for dimensions in the drawing.

## Syntax

DimensionStyle.**DisplayUnitType**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
