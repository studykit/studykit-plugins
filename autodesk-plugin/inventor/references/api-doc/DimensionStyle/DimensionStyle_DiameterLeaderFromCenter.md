# DimensionStyle.DiameterLeaderFromCenter Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the leader line of a diameter dimension extends to the center or not.

## Syntax

DimensionStyle.**DiameterLeaderFromCenter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
