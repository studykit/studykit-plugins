# DimensionStyle.ShowDimensionLine Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the dimension line when arrowheads are outside the dimension extension lines.

## Syntax

DimensionStyle.**ShowDimensionLine**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
