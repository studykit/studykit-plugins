# DimensionStyle.OrdinateDimensionOriginArrowheadType Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the arrowhead type for the origin member of ordinate dimension sets.

## Syntax

DimensionStyle.**OrdinateDimensionOriginArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2012
