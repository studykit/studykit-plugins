# DimensionStyle.ToleranceLinearPrecision Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the number of decimal places visible to the right of the decimal marker in tolerance text for linear dimensions.

## Syntax

DimensionStyle.**ToleranceLinearPrecision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 9
