# DimensionStyle.UpToDate Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Property that gets the up-to-date status of the style against the global repository.

## Syntax

DimensionStyle.**UpToDate**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 9
