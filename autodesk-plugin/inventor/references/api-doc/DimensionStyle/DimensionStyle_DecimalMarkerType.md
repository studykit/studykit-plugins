# DimensionStyle.DecimalMarkerType Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the character to use as a decimal marker for dimensions.

## Syntax

DimensionStyle.**DecimalMarkerType**() As [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DecimalMarkerTypeEnum](../DecimalMarkerTypeEnum.md).

## Version

Introduced in version 9
