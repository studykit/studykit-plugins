# DimensionStyle.OriginOffset Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the distance from an edge or point to the endpoint of the extension line.

## Syntax

DimensionStyle.**OriginOffset**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
