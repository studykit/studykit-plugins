# DimensionStyle.ToleranceAlternateTrailingZeroDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the trailing zeroes for tolerances in alternate units.

## Syntax

DimensionStyle.**ToleranceAlternateTrailingZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
