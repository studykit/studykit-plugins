# DimensionStyle.ArrowheadHeight Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the height of the arrowhead relative to the associated line.

## Syntax

DimensionStyle.**ArrowheadHeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2011
