# DimensionStyle.RadialMultiLineTextOrientation Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the position of multiple lines of text relative to the landing line in a radial dimension.

## Syntax

DimensionStyle.**RadialMultiLineTextOrientation**() As [MultiLineDimensionTextEnum](../MultiLineDimensionTextEnum.md)

## Property Value

This is a read/write property whose value is a [MultiLineDimensionTextEnum](../MultiLineDimensionTextEnum.md).

## Version

Introduced in version 2014
