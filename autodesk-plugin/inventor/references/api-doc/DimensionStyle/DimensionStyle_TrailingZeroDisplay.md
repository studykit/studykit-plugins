# DimensionStyle.TrailingZeroDisplay Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets whether to display the trailing zeroes for dimensions in the drawing.

## Syntax

DimensionStyle.**TrailingZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
