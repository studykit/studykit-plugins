# DimensionStyle.RadialTextOrientation Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the orientation of text in a radial dimension.

## Syntax

DimensionStyle.**RadialTextOrientation**() As [DimensionTextOrientationEnum](../DimensionTextOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [DimensionTextOrientationEnum](../DimensionTextOrientationEnum.md).

## Version

Introduced in version 2014
