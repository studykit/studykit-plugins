# DimensionStyle.ArrowheadTypeForChainDimensions Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the internal arrowhead styles to use for chain dimensions.

## Syntax

DimensionStyle.**ArrowheadTypeForChainDimensions**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2011
