# DimensionStyle.DiameterArrowsInside Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets inside or outside placement of arrows for diameter dimensions.

## Syntax

DimensionStyle.**DiameterArrowsInside**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
