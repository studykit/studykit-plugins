# DimensionStyle.OrdinateHideOriginIndicator Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if the origin indicator is to be hidden or not.

## Syntax

DimensionStyle.**OrdinateHideOriginIndicator**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
