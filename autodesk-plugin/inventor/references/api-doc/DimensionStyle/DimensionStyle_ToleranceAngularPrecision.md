# DimensionStyle.ToleranceAngularPrecision Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the precision of the tolerance text for angular dimensions.

## Syntax

DimensionStyle.**ToleranceAngularPrecision**() As [AngularPrecisionEnum](../AngularPrecisionEnum.md)

## Property Value

This is a read/write property whose value is an [AngularPrecisionEnum](../AngularPrecisionEnum.md).

## Version

Introduced in version 9
