# DimensionStyle.DiameterDualDimensionLine Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets if single or dual diameter dimension lines should be displayed.

## Syntax

DimensionStyle.**DiameterDualDimensionLine**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
