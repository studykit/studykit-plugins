# DimensionStyle.ToleranceFontSize Property

Parent Object: [DimensionStyle](../DimensionStyle/DimensionStyle.md)

## Description

Gets and sets the size of the font used for tolerances.

## Syntax

DimensionStyle.**ToleranceFontSize**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
