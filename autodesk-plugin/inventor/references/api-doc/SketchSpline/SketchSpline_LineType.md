# SketchSpline.LineType Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Gets and sets the LineType applied to this sketch spline.

## Syntax

SketchSpline.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 9
