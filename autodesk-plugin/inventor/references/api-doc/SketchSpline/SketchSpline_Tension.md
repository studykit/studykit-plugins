# SketchSpline.Tension Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Gets and sets the tension for the spline.

## Syntax

SketchSpline.**Tension**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 9
