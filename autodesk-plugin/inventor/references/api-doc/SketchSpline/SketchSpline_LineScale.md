# SketchSpline.LineScale Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Gets and sets the LineScale applied to this sketch spline.

## Syntax

SketchSpline.**LineScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
