# SketchSpline.Geometry3d Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Read-only property that returns the BSplineCurve geometry that represents this sketch spline in model space.

## Syntax

SketchSpline.**Geometry3d**() As [BSplineCurve](../BSplineCurve/BSplineCurve.md)

## Property Value

This is a read only property whose value is a [BSplineCurve](../BSplineCurve/BSplineCurve.md).

## Version

Introduced in version 2013
