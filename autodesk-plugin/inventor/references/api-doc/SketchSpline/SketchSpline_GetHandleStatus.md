# SketchSpline.GetHandleStatus Method

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Method that returns whether the handle at a given fit point is enabled or disabled.

## Syntax

SketchSpline.**GetHandleStatus**( ***FitPoint*** As [SketchPoint](../SketchPoint/SketchPoint.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FitPoint | [SketchPoint](../SketchPoint/SketchPoint.md) | Input SketchPoint object that specifies the fit point at which to get the handle status. |

## Version

Introduced in version 2008
