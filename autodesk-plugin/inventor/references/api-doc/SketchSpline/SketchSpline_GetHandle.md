# SketchSpline.GetHandle Method

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Method that returns the fit point handle at the given fit point. The method returns Nothing if the handle is not enabled at the input fit point.

## Syntax

SketchSpline.**GetHandle**( ***FitPoint*** As [SketchPoint](../SketchPoint/SketchPoint.md) ) As [SketchSplineHandle](../SketchSplineHandle/SketchSplineHandle.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FitPoint | [SketchPoint](../SketchPoint/SketchPoint.md) | Input SketchPoint object that specifies the fit point at which to obtain the handle. |

## Version

Introduced in version 2008
