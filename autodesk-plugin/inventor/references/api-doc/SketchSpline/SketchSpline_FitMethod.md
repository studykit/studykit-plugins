# SketchSpline.FitMethod Property

Parent Object: [SketchSpline](../SketchSpline/SketchSpline.md)

## Description

Specifies the fit method for the spline.

## Syntax

SketchSpline.**FitMethod**() As [SplineFitMethodEnum](../SplineFitMethodEnum.md)

## Property Value

This is a read/write property whose value is a [SplineFitMethodEnum](../SplineFitMethodEnum.md).

## Version

Introduced in version 5
