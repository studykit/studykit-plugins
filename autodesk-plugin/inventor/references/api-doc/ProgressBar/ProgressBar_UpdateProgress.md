# ProgressBar.UpdateProgress Method

Parent Object: [ProgressBar](../ProgressBar/ProgressBar.md)

## Description

Updates the progress bar by incrementing the progress step by one. This method should be called as many times as the number of steps provided in the CreateProgressBar method.

## Syntax

ProgressBar.**UpdateProgress**()
