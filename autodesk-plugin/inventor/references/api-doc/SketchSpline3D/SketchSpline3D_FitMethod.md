# SketchSpline3D.FitMethod Property

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Specifies the fit method for the spline.

## Syntax

SketchSpline3D.**FitMethod**() As [SplineFitMethodEnum](../SplineFitMethodEnum.md)

## Property Value

This is a read/write property whose value is a [SplineFitMethodEnum](../SplineFitMethodEnum.md).

## Version

Introduced in version 8
