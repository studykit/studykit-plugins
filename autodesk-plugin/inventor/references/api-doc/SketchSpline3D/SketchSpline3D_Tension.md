# SketchSpline3D.Tension Property

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Gets and sets the tension for the spline.

## Syntax

SketchSpline3D.**Tension**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 8
