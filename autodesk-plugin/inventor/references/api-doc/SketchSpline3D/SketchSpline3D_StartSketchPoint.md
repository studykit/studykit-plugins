# SketchSpline3D.StartSketchPoint Property

Parent Object: [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Description

Property that returns the that defines the position of the start of the spline.

## Syntax

SketchSpline3D.**StartSketchPoint**() As [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Property Value

This is a read only property whose value is a [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md).

## Version

Introduced in version 8
