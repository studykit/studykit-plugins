# DWGArcProxy.Center Property

Parent Object: [DWGArcProxy](../DWGArcProxy/DWGArcProxy.md)

## Description

Read-only property that returns center of the arc in model space.

## Syntax

DWGArcProxy.**Center**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2016
