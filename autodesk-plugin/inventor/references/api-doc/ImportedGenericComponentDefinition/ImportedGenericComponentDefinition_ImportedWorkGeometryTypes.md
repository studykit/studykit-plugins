# ImportedGenericComponentDefinition.ImportedWorkGeometryTypes Property

Parent Object: [ImportedGenericComponentDefinition](../ImportedGenericComponentDefinition/ImportedGenericComponentDefinition.md)

## Description

Read-write property that gets and sets a bit mask defining the imported work geometry types. A combination of enum values can be provided.

## Syntax

ImportedGenericComponentDefinition.**ImportedWorkGeometryTypes**() As [ImportedWorkGeometryTypeEnum](../ImportedWorkGeometryTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ImportedWorkGeometryTypeEnum](../ImportedWorkGeometryTypeEnum.md).

## Version

Introduced in version 2016
