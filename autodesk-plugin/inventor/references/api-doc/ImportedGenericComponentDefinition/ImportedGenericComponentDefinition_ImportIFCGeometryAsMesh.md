# ImportedGenericComponentDefinition.ImportIFCGeometryAsMesh Property

Parent Object: [ImportedGenericComponentDefinition](../ImportedGenericComponentDefinition/ImportedGenericComponentDefinition.md)

## Description

Read-write property that gets and sets whether to import IFC geometry as mesh. This is only applicable when import IFC file.

## Syntax

ImportedGenericComponentDefinition.**ImportIFCGeometryAsMesh**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
