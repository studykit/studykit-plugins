# ImportedRVTComponentDefinition.ImportedAssemblyOrganizationType Property

Parent Object: [ImportedRVTComponentDefinition](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition.md)

## Description

Read-write property that gets and sets the imported assembly organization type.

## Syntax

ImportedRVTComponentDefinition.**ImportedAssemblyOrganizationType**() As [ImportedAssemblyOrganizationTypeEnum](../ImportedAssemblyOrganizationTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ImportedAssemblyOrganizationTypeEnum](../ImportedAssemblyOrganizationTypeEnum.md).

## Version

Introduced in version 2021
