# ImportedRVTComponentDefinition.ImportedLanguage Property

Parent Object: [ImportedRVTComponentDefinition](../ImportedRVTComponentDefinition/ImportedRVTComponentDefinition.md)

## Description

Read-write property that gets and sets the imported Revit language type.

## Syntax

ImportedRVTComponentDefinition.**ImportedLanguage**() As [ImportedRevitLanguageTypeEnum](../ImportedRevitLanguageTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ImportedRevitLanguageTypeEnum](../ImportedRevitLanguageTypeEnum.md).

## Version

Introduced in version 2021
