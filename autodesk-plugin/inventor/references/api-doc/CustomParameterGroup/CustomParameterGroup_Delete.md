# CustomParameterGroup.Delete Method

Parent Object: [CustomParameterGroup](../CustomParameterGroup/CustomParameterGroup.md)

## Description

Method that deletes the group and optionally deletes the parameters that it contains.

## Syntax

CustomParameterGroup.**Delete**( [***DeleteParameters***] As Boolean )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| DeleteParameters | Boolean | Optional input Boolean value that indicates whether or not to additionally delete the parameters contained in the group. A value of True specifies that the parameters should be deleted. If not specified, a default of False is used indicating that the parameters in the group will not be deleted. |

## Version

Introduced in version 10
