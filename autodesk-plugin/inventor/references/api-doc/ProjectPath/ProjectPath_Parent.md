# ProjectPath.Parent Property

Parent Object: [ProjectPath](../ProjectPath/ProjectPath.md)

## Description

Property that returns the parent DesignProject object.

## Syntax

ProjectPath.**Parent**() As [DesignProject](../DesignProject/DesignProject.md)

## Property Value

This is a read only property whose value is a [DesignProject](../DesignProject/DesignProject.md).

## Version

Introduced in version 2011
