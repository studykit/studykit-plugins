# EdgeSymbol.Definition Property

Parent Object: [EdgeSymbol](../EdgeSymbol/EdgeSymbol.md)

## Description

Read-write property that gets and sets the EdgeSymbolDefinition object associated with this edge symbol.

## Syntax

EdgeSymbol.**Definition**() As [EdgeSymbolDefinition](../EdgeSymbolDefinition/EdgeSymbolDefinition.md)

## Property Value

This is a read/write property whose value is an [EdgeSymbolDefinition](../EdgeSymbolDefinition/EdgeSymbolDefinition.md).

## Version

Introduced in version 2024
