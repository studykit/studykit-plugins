# AssemblySymmetryConstraint.OccurrenceOne Property

Parent Object: [AssemblySymmetryConstraint](../AssemblySymmetryConstraint/AssemblySymmetryConstraint.md)

## Description

Property that returns the first of the two objects this constraint is between.

## Syntax

AssemblySymmetryConstraint.**OccurrenceOne**() As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md).

## Version

Introduced in version 2014
