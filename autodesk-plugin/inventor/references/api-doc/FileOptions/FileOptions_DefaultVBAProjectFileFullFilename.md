# FileOptions.DefaultVBAProjectFileFullFilename Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Gets and sets the full filename of the VBA project file.

## Syntax

FileOptions.**DefaultVBAProjectFileFullFilename**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 9
