# FileOptions.FileOpenOptions Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Property that gets the FileOpenOptions object.

## Syntax

FileOptions.**FileOpenOptions**() As [FileOpenOptions](../FileOpenOptions/FileOpenOptions.md)

## Property Value

This is a read only property whose value is a [FileOpenOptions](../FileOpenOptions/FileOpenOptions.md).

## Version

Introduced in version 2011
