# FileOptions.TexturePath Property

Parent Object: [FileOptions](../FileOptions/FileOptions.md)

## Description

Gets and sets the location of texture files.

## Syntax

FileOptions.**TexturePath**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2017
