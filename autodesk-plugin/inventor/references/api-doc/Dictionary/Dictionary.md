# Dictionary Object

## Description

Dictionary object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddWord](../Dictionary/Dictionary_AddWord.md) | Method that adds a word to this dictionary. |
| [DeleteWord](../Dictionary/Dictionary_DeleteWord.md) | Method that deletes a word from this dictionary. |
| [GetWords](../Dictionary/Dictionary_GetWords.md) | Method that gets all words in this dictionary. |
| [ImportWordsFromExistingDictionary](../Dictionary/Dictionary_ImportWordsFromExistingDictionary.md) | Method that imports words to this dictionary from an existing dictionary. |
| [Remove](../Dictionary/Dictionary_Remove.md) | Method that removes the Dictionary from the dictionaries. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Dictionary/Dictionary_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [FullFileName](../Dictionary/Dictionary_FullFileName.md) | Read-only property that returns the fully qualified file name of the Dictionary. |
| [Name](../Dictionary/Dictionary_Name.md) | Read-only property that returns the name of the Dictionary. |
| [Type](../Dictionary/Dictionary_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[Dictionaries.Add](../Dictionaries/Dictionaries_Add.md), [Dictionaries.AddExisting](../Dictionaries/Dictionaries_AddExisting.md), [Dictionaries.Item](../Dictionaries/Dictionaries_Item.md)

## Version

Introduced in version 2020
