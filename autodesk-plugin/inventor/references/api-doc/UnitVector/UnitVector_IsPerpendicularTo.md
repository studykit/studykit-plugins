# UnitVector.IsPerpendicularTo Method

Parent Object: [UnitVector](../UnitVector/UnitVector.md)

## Description

Determine if this vector is perpendicular to the specified vector.

## Syntax

UnitVector.**IsPerpendicularTo**( ***Argument*** As [UnitVector](../UnitVector/UnitVector.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [UnitVector](../UnitVector/UnitVector.md) | Input  used for comparison. |
| Tolerance | Double | Input Double that specifies the tolerance to be used when determining whether the vectors are perpendicular. |

## Version

Introduced in version 4
