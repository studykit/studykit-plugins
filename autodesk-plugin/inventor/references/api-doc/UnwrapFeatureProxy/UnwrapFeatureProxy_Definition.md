# UnwrapFeatureProxy.Definition Property

Parent Object: [UnwrapFeatureProxy](../UnwrapFeatureProxy/UnwrapFeatureProxy.md)

## Description

Property that gets and sets the UnwrapDefinition object associated with this unwrap feature.

## Syntax

UnwrapFeatureProxy.**Definition**() As [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md)

## Property Value

This is a read/write property whose value is a [UnwrapDefinition](../UnwrapDefinition/UnwrapDefinition.md).

## Version

Introduced in version 2020
