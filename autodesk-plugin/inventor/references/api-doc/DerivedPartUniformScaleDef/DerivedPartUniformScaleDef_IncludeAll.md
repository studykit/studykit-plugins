# DerivedPartUniformScaleDef.IncludeAll Method

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Method that causes all entities within the derived part component to be set to kDerivedIncludeAll.

## Syntax

DerivedPartUniformScaleDef.**IncludeAll**()

## Version

Introduced in version 6
