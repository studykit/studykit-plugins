# DerivedPartUniformScaleDef.Sketches3D Property

Parent Object: [DerivedPartUniformScaleDef](../DerivedPartUniformScaleDef/DerivedPartUniformScaleDef.md)

## Description

Property that gets the collection of 3D sketches available in the selected part along with the inclusion option for each sketch.

## Syntax

DerivedPartUniformScaleDef.**Sketches3D**() As [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md)

## Property Value

This is a read only property whose value is a [DerivedPartEntities](../DerivedPartEntities/DerivedPartEntities.md).

## Version

Introduced in version 11
