# DWGBlockDefinitionProxy.IsModelSpaceDefinition Property

Parent Object: [DWGBlockDefinitionProxy](../DWGBlockDefinitionProxy/DWGBlockDefinitionProxy.md)

## Description

Read-only property that returns whether this DWGBlockDefinition is indicating the model space definition.

## Syntax

DWGBlockDefinitionProxy.**IsModelSpaceDefinition**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2016
