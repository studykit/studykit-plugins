# DerivedAssemblyComponentProxy.ReferenceFeatures Property

Parent Object: [DerivedAssemblyComponentProxy](../DerivedAssemblyComponentProxy/DerivedAssemblyComponentProxy.md)

## Description

Property that returns the collection containing the ReferenceFeatures that was created as a result of the derived component.

## Syntax

DerivedAssemblyComponentProxy.**ReferenceFeatures**() As [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md)

## Property Value

This is a read only property whose value is a [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md).

## Version

Introduced in version 2008
