# DerivedAssemblyComponentProxy.SuppressLinkToFile Property

Parent Object: [DerivedAssemblyComponentProxy](../DerivedAssemblyComponentProxy/DerivedAssemblyComponentProxy.md)

## Description

Gets and Sets a Boolean flag indicating whether the link to the assembly file is suppressed.

## Syntax

DerivedAssemblyComponentProxy.**SuppressLinkToFile**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
