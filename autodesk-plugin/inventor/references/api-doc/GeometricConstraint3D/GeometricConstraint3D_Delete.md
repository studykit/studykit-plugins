# GeometricConstraint3D.Delete Method

Parent Object: [GeometricConstraint3D](../GeometricConstraint3D/GeometricConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

GeometricConstraint3D.**Delete**()

## Version

Introduced in version 6
