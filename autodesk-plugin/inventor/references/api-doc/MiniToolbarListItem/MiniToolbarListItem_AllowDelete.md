# MiniToolbarListItem.AllowDelete Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-write property that specifies if this item can be deleted. This will always be false for a separator.

## Syntax

MiniToolbarListItem.**AllowDelete**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
