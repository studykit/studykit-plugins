# MiniToolbarListItem.IsChecked Property

Parent Object: [MiniToolbarListItem](../MiniToolbarListItem/MiniToolbarListItem.md)

## Description

Read-write property that specifies if the check box beside this item is checked or not.

## Syntax

MiniToolbarListItem.**IsChecked**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
