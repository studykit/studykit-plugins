# HorizontalAlignConstraintProxy Object

Derived from: [HorizontalAlignConstraint](../HorizontalAlignConstraint/HorizontalAlignConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_Deletable.md) | Indicates whether this object is deletable. |
| [NativeObject](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [PointOne](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_PointOne.md) | Property that returns the first sketch point being constrained. |
| [PointTwo](../HorizontalAlignConstraintProxy/HorizontalAlignConstraintProxy_PointTwo.md) | Property that returns the second sketch point being constrained. |
