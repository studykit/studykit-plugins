# LineWorkAxisDef.Line Property

Parent Object: [LineWorkAxisDef](../LineWorkAxisDef/LineWorkAxisDef.md)

## Description

Property that returns the line of a line defined work axis. This object can be a linear Edge, WorkAxis, or SketchLine object.

## Syntax

LineWorkAxisDef.**Line**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 4
