# DWGPolylineProxy.Segments Property

Parent Object: [DWGPolylineProxy](../DWGPolylineProxy/DWGPolylineProxy.md)

## Description

Read-only property that returns the DWGEntitySegmentsEnumerator object.

## Syntax

DWGPolylineProxy.**Segments**() As [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md)

## Property Value

This is a read only property whose value is a [DWGEntitySegmentsEnumerator](../DWGEntitySegmentsEnumerator/DWGEntitySegmentsEnumerator.md).

## Version

Introduced in version 2018
