# AnalysisManager.DraftAnalyses Property

Parent Object: [AnalysisManager](../AnalysisManager/AnalysisManager.md)

## Description

Property that returns the DraftAnalyses collection object.

## Syntax

AnalysisManager.**DraftAnalyses**() As [DraftAnalyses](../DraftAnalyses/DraftAnalyses.md)

## Property Value

This is a read only property whose value is a [DraftAnalyses](../DraftAnalyses/DraftAnalyses.md).
