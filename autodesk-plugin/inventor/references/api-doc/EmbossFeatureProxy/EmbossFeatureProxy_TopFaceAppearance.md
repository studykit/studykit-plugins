# EmbossFeatureProxy.TopFaceAppearance Property

Parent Object: [EmbossFeatureProxy](../EmbossFeatureProxy/EmbossFeatureProxy.md)

## Description

Gets and sets the top face appearance of the emboss feature.

## Syntax

EmbossFeatureProxy.**TopFaceAppearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
