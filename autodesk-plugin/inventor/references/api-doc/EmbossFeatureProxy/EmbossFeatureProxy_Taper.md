# EmbossFeatureProxy.Taper Property

Parent Object: [EmbossFeatureProxy](../EmbossFeatureProxy/EmbossFeatureProxy.md)

## Description

Property that returns the parameter controlling the taper angle of the emboss feature.

## Syntax

EmbossFeatureProxy.**Taper**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2008
