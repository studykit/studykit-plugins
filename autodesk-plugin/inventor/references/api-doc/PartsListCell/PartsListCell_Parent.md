# PartsListCell.Parent Property

Parent Object: [PartsListCell](../PartsListCell/PartsListCell.md)

## Description

Property that gets the parent object from whom this object can logically be reached.

## Syntax

PartsListCell.**Parent**() As [PartsList](../PartsList/PartsList.md)

## Property Value

This is a read only property whose value is a [PartsList](../PartsList/PartsList.md).

## Version

Introduced in version 5.3
