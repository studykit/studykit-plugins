# ProjectedCut.BreakLink Method

Parent Object: [ProjectedCut](../ProjectedCut/ProjectedCut.md)

## Description

Method that breaks the link between the projected cut edges and the model.

## Syntax

ProjectedCut.**BreakLink**()

## Version

Introduced in version 2010
