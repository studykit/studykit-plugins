# ModelDatumTarget.IsMemberOfCompositeAnnotation Property

Parent Object: [ModelDatumTarget](../ModelDatumTarget/ModelDatumTarget.md)

## Description

Returns whether this annotation is a member of a ModelCompositeAnnotation.

## Syntax

ModelDatumTarget.**IsMemberOfCompositeAnnotation**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2023
