# iMateDefinitions.Item Property

Parent Object: [iMateDefinitions](../iMateDefinitions/iMateDefinitions.md)

## Description

Returns the specified iMateDefinition object from the collection.

## Syntax

iMateDefinitions.**Item**( ***Index*** As Long ) As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Specifies the index of the to return. |
