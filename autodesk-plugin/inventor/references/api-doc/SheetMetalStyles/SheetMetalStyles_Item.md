# SheetMetalStyles.Item Property

Parent Object: [SheetMetalStyles](../SheetMetalStyles/SheetMetalStyles.md)

## Description

Method that returns the specified SheetMetalStyle object from the collection.

## Syntax

SheetMetalStyles.**Item**( ***Index*** As Variant ) As [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Property Value

This is a read only property whose value is a [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the sheet metal style to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the style name. If an out-of-range index or a name of a non-existent style is provided, an error occurs. |

## Version

Introduced in version 5.3
