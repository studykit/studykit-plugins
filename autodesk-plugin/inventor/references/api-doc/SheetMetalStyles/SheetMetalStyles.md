# SheetMetalStyles Object

## Description

The SheetMetalStyles collection object provides access to all of the objects in a part component definition and provides methods to create additional sheet metal styles.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SheetMetalStyles/SheetMetalStyles_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SheetMetalStyles/SheetMetalStyles_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SheetMetalStyles/SheetMetalStyles_Item.md) | Method that returns the specified SheetMetalStyle object from the collection. |
| [Type](../SheetMetalStyles/SheetMetalStyles_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SheetMetalComponentDefinition.SheetMetalStyles](../SheetMetalComponentDefinition/SheetMetalComponentDefinition_SheetMetalStyles.md)

## Version

Introduced in version 5.3
