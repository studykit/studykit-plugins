# iFeatureSketchPlaneInput.FlipOption Property

Parent Object: [iFeatureSketchPlaneInput](../iFeatureSketchPlaneInput/iFeatureSketchPlaneInput.md)

## Description

Gets and sets whether the geometry selection needs to be flipped.

## Syntax

iFeatureSketchPlaneInput.**FlipOption**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
