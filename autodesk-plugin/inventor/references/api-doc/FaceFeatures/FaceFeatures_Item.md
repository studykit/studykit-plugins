# FaceFeatures.Item Property

Parent Object: [FaceFeatures](../FaceFeatures/FaceFeatures.md)

## Description

Returns the specified FaceFeature object from the collection. This is the default property of the FaceFeatures collection object.

## Syntax

FaceFeatures.**Item**( ***Index*** As Variant ) As [FaceFeature](../FaceFeature/FaceFeature.md)

## Property Value

This is a read only property whose value is a [FaceFeature](../FaceFeature/FaceFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the to return. |

## Version

Introduced in version 5.3
