# DockableWindow.Left Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Property that returns the distance between the left edge of the screen and left edge of the window.

## Syntax

DockableWindow.**Left**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2011
