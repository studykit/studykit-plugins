# DockableWindow.InternalName Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Property that returns the unique internal name of the window.

## Syntax

DockableWindow.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2011
