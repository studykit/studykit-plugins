# DockableWindow.Top Property

Parent Object: [DockableWindow](../DockableWindow/DockableWindow.md)

## Description

Property that returns the distance between the top of the screen and top of the window.

## Syntax

DockableWindow.**Top**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2011
