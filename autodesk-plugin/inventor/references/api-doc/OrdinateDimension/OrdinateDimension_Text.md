# OrdinateDimension.Text Property

Parent Object: [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Description

Gets and sets the DimensionText setting.

## Syntax

OrdinateDimension.**Text**() As [DimensionText](../DimensionText/DimensionText.md)

## Property Value

This is a read/write property whose value is a [DimensionText](../DimensionText/DimensionText.md).

## Version

Introduced in version 2009
