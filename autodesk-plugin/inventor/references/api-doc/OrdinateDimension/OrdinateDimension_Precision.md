# OrdinateDimension.Precision Property

Parent Object: [OrdinateDimension](../OrdinateDimension/OrdinateDimension.md)

## Description

Gets and sets the Precision setting.

## Syntax

OrdinateDimension.**Precision**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2009
