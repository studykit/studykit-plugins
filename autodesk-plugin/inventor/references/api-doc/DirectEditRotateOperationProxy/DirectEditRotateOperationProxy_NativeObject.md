# DirectEditRotateOperationProxy.NativeObject Property

Parent Object: [DirectEditRotateOperationProxy](../DirectEditRotateOperationProxy/DirectEditRotateOperationProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

DirectEditRotateOperationProxy.**NativeObject**() As [DirectEditRotateOperation](../DirectEditRotateOperation/DirectEditRotateOperation.md)

## Property Value

This is a read only property whose value is a [DirectEditRotateOperation](../DirectEditRotateOperation/DirectEditRotateOperation.md).

## Version

Introduced in version 2015
