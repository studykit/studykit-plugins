# CosmeticWeldDefinition.SetThroughAllExtent Method

Parent Object: [CosmeticWeldDefinition](../CosmeticWeldDefinition/CosmeticWeldDefinition.md)

## Description

Method that changes the extents to be “through all” extents.

## Syntax

CosmeticWeldDefinition.**SetThroughAllExtent**()

## Version

Introduced in version 2017
