# SketchConstraintsEnumerator Object

## Description

The SketchConstraintEnumerator object provides access to a list of SketchConstraint objects. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../SketchConstraintsEnumerator/SketchConstraintsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchConstraintsEnumerator/SketchConstraintsEnumerator_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../SketchConstraintsEnumerator/SketchConstraintsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
