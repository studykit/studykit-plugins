# PartsList.RowLineSpacing Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the line spacing of the rows.

## Syntax

PartsList.**RowLineSpacing**() As [LineSpacingEnum](../LineSpacingEnum.md)

## Property Value

This is a read/write property whose value is a [LineSpacingEnum](../LineSpacingEnum.md).

## Version

Introduced in version 9
