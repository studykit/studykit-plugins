# PartsList.ReferencedDocumentDescriptor Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Property that gets the document referenced by this parts list.

## Syntax

PartsList.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 11
