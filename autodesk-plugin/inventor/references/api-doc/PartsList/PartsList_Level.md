# PartsList.Level Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Property that returns the type of numbering for the parts list. Possible values are kFirstLevelComponents and kPartsOnly.

## Syntax

PartsList.**Level**() As [PartsListLevelEnum](../PartsListLevelEnum.md)

## Property Value

This is a read only property whose value is a [PartsListLevelEnum](../PartsListLevelEnum.md).

## Version

Introduced in version 10
