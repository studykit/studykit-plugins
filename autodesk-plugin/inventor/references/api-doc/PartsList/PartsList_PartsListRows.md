# PartsList.PartsListRows Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets the PartsListRows collection object for this parts list table.

## Syntax

PartsList.**PartsListRows**() As [PartsListRows](../PartsListRows/PartsListRows.md)

## Property Value

This is a read only property whose value is a [PartsListRows](../PartsListRows/PartsListRows.md).
