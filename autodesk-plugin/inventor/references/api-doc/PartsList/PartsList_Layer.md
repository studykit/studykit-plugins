# PartsList.Layer Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the layer used by the parts list.

## Syntax

PartsList.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
