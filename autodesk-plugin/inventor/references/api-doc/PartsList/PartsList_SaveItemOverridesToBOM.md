# PartsList.SaveItemOverridesToBOM Method

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Saves any overrides to the item values in the balloon to the model BOM.

## Syntax

PartsList.**SaveItemOverridesToBOM**()

## Version

Introduced in version 11
