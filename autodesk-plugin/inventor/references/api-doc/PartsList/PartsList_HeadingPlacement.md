# PartsList.HeadingPlacement Property

Parent Object: [PartsList](../PartsList/PartsList.md)

## Description

Gets and sets the placement position of the table heading.

## Syntax

PartsList.**HeadingPlacement**() As [HeadingPlacementEnum](../HeadingPlacementEnum.md)

## Property Value

This is a read/write property whose value is a [HeadingPlacementEnum](../HeadingPlacementEnum.md).

## Version

Introduced in version 9
