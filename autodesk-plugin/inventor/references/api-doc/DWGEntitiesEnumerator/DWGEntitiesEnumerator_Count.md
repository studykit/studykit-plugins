# DWGEntitiesEnumerator.Count Property

Parent Object: [DWGEntitiesEnumerator](../DWGEntitiesEnumerator/DWGEntitiesEnumerator.md)

## Description

Gets the number of items in this collection.

## Syntax

DWGEntitiesEnumerator.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2016
