# PresentationFacesEnumerator.Item Property

Parent Object: [PresentationFacesEnumerator](../PresentationFacesEnumerator/PresentationFacesEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

PresentationFacesEnumerator.**Item**( ***Index*** As Long ) As [PresentationFace](../PresentationFace/PresentationFace.md)

## Property Value

This is a read only property whose value is a [PresentationFace](../PresentationFace/PresentationFace.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long to specify the index of the object to return. If an out of range index is provided, an error will occur. |

## Version

Introduced in version 2018
