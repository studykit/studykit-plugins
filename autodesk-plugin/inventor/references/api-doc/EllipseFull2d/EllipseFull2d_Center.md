# EllipseFull2d.Center Property

Parent Object: [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md)

## Description

Specifies the center of the ellipse.

## Syntax

EllipseFull2d.**Center**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 4
