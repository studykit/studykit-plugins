# EllipseFull2d.Copy Method

Parent Object: [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md)

## Description

Creates a copy of this EllipseFull2d object. The result is entirely independent and can be edited without affecting the original EllipseFull2d object.

## Syntax

EllipseFull2d.**Copy**() As [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md)

## Version

Introduced in version 2013
