# DWGACMStandardPartsEnumerator.Item Property

Parent Object: [DWGACMStandardPartsEnumerator](../DWGACMStandardPartsEnumerator/DWGACMStandardPartsEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGACMStandardPartsEnumerator.**Item**( ***Index*** As Variant ) As [DWGACMStandardPart](../DWGACMStandardPart/DWGACMStandardPart.md)

## Property Value

This is a read only property whose value is a [DWGACMStandardPart](../DWGACMStandardPart/DWGACMStandardPart.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant |  |

Introduced in version 2017
