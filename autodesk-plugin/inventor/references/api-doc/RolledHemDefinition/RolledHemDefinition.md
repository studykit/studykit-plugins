# RolledHemDefinition Object

## Description

The RolledHemDefinition object defines all of the inputs that are unique to a rolled style of hem feature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Angle](../RolledHemDefinition/RolledHemDefinition_Angle.md) | Property that returns the parameter controlling the bend angle of a hem feature defined by a rolled style of bend. |
| [Parent](../RolledHemDefinition/RolledHemDefinition_Parent.md) | Property that returns the parent hem definition feature. |
| [Radius](../RolledHemDefinition/RolledHemDefinition_Radius.md) | Property that returns the parameter controlling the radius of a hem feature defined by a rolled style of bend. |

## Version

Introduced in version 2009
