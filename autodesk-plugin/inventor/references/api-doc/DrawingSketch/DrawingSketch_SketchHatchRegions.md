# DrawingSketch.SketchHatchRegions Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Read-only property that returns the SketchHatchRegions collection object.

## Syntax

DrawingSketch.**SketchHatchRegions**() As [SketchHatchRegions](../SketchHatchRegions/SketchHatchRegions.md)

## Property Value

This is a read only property whose value is a [SketchHatchRegions](../SketchHatchRegions/SketchHatchRegions.md).

## Version

Introduced in version 2021
