# DrawingSketch.SketchCircles Property

Parent Object: [DrawingSketch](../DrawingSketch/DrawingSketch.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

DrawingSketch.**SketchCircles**() As [SketchCircles](../SketchCircles/SketchCircles.md)

## Property Value

This is a read only property whose value is a [SketchCircles](../SketchCircles/SketchCircles.md).
