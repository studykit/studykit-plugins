# OccurrencePatternElement.Independent Property

Parent Object: [OccurrencePatternElement](../OccurrencePatternElement/OccurrencePatternElement.md)

## Description

Gets/Sets whether the element is independent of the pattern or not.

## Syntax

OccurrencePatternElement.**Independent**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
