# SketchCircle3D.Construction Property

Parent Object: [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchCircle3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
