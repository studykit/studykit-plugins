# SketchCircle3D.CurvatureDisplay Property

Parent Object: [SketchCircle3D](../SketchCircle3D/SketchCircle3D.md)

## Description

Specifies whether curvature information is displayed for the circle.

## Syntax

SketchCircle3D.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
