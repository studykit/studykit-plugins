# ObjectVisibility.ComponentAnnotations3D Property

Parent Object: [ObjectVisibility](../ObjectVisibility/ObjectVisibility.md)

## Description

Gets and sets whether component 3d annotations should be visible.

## Syntax

ObjectVisibility.**ComponentAnnotations3D**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
