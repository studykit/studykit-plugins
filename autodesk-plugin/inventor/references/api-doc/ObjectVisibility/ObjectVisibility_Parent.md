# ObjectVisibility.Parent Property

Parent Object: [ObjectVisibility](../ObjectVisibility/ObjectVisibility.md)

## Description

Property that returns the parent Document object.

## Syntax

ObjectVisibility.**Parent**() As [Document](../Document/Document.md)

## Property Value

This is a read only property whose value is a [Document](../Document/Document.md).

## Version

Introduced in version 2011
