# LineGraphics.RemoveViewSpaceAnchor Method

Parent Object: [LineGraphics](../LineGraphics/LineGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

LineGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
