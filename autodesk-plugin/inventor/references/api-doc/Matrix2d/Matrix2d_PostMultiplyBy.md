# Matrix2d.PostMultiplyBy Method

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Post multiply this matrix2d by the specified matrix2d, setting this matrix2d to the result.

## Syntax

Matrix2d.**PostMultiplyBy**( ***Matrix2d*** As [Matrix2d](../Matrix2d/Matrix2d.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Matrix2d | [Matrix2d](../Matrix2d/Matrix2d.md) | Matrix2d to multiply by. |

## Version

Introduced in version 11
