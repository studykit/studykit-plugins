# Matrix2d.Determinant Property

Parent Object: [Matrix2d](../Matrix2d/Matrix2d.md)

## Description

Get the determinant of this matrix2d.

## Syntax

Matrix2d.**Determinant**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 11
