# PartsListColumn.TitleHorizontalJustification Property

Parent Object: [PartsListColumn](../PartsListColumn/PartsListColumn.md)

## Description

Gets and sets the justification of the column title.

## Syntax

PartsListColumn.**TitleHorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 10
