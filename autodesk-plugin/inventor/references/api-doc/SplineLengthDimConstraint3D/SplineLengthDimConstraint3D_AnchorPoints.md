# SplineLengthDimConstraint3D.AnchorPoints Property

Parent Object: [SplineLengthDimConstraint3D](../SplineLengthDimConstraint3D/SplineLengthDimConstraint3D.md)

## Description

Gets the anchor points of dimension.

## Syntax

SplineLengthDimConstraint3D.**AnchorPoints**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2019
