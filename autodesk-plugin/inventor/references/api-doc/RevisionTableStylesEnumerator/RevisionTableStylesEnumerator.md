# RevisionTableStylesEnumerator Object

## Description

The RevisionTableStylesEnumerator object provides access to the existing RevisionTableStyle objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevisionTableStylesEnumerator/RevisionTableStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../RevisionTableStylesEnumerator/RevisionTableStylesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../RevisionTableStylesEnumerator/RevisionTableStylesEnumerator_Item.md) | Returns the specified RevisionTableStyle object from the collection. |
| [Type](../RevisionTableStylesEnumerator/RevisionTableStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.RevisionTableStyles](../DrawingStylesManager/DrawingStylesManager_RevisionTableStyles.md)

## Version

Introduced in version 2011
