# RevitExportDefinition.RemoveChamfersDistanceRange Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the distance of chamfers in centimeters to remove for the RevitExport. This is only applicable when the RemoveChamfersStyle is set to kRevitExportRemoveByRange.

## Syntax

RevitExportDefinition.**RemoveChamfersDistanceRange**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2022
