# RevitExportDefinition.AdditionalExcludedOccurrences Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the occurrences being excluded for the RevitExport.

## Syntax

RevitExportDefinition.**AdditionalExcludedOccurrences**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2022
