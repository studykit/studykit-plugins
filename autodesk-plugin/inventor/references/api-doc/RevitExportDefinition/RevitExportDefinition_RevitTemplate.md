# RevitExportDefinition.RevitTemplate Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets the Revit template file used to create the RevitExport.

## Syntax

RevitExportDefinition.**RevitTemplate**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2026
