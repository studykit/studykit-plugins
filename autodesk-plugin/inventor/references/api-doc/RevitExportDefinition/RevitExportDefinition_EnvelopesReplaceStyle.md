# RevitExportDefinition.EnvelopesReplaceStyle Property

Parent Object: [RevitExportDefinition](../RevitExportDefinition/RevitExportDefinition.md)

## Description

Read-write property that gets and sets envelopes replace style. When a new definition is created, this defaults to kNoneReplaceStyle.

## Syntax

RevitExportDefinition.**EnvelopesReplaceStyle**() As [EnvelopesReplaceStyleEnum](../EnvelopesReplaceStyleEnum.md)

## Property Value

This is a read/write property whose value is an [EnvelopesReplaceStyleEnum](../EnvelopesReplaceStyleEnum.md).

## Version

Introduced in version 2022
