# CornerChamferFeatureProxy.Suppressed Property

Parent Object: [CornerChamferFeatureProxy](../CornerChamferFeatureProxy/CornerChamferFeatureProxy.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

CornerChamferFeatureProxy.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
