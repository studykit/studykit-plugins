# BIMDuctConnectorDefinition.HeightDirection Property

Parent Object: [BIMDuctConnectorDefinition](../BIMDuctConnectorDefinition/BIMDuctConnectorDefinition.md)

## Description

Read-only property that returns the direction of the connector height. This returns Nothing in the case where the height has not yet been defined.

## Syntax

BIMDuctConnectorDefinition.**HeightDirection**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2014
