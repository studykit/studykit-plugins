# GraphicsTextureCoordinateSet.Id Property

Parent Object: [GraphicsTextureCoordinateSet](../GraphicsTextureCoordinateSet/GraphicsTextureCoordinateSet.md)

## Description

Property returning the unique id of this GraphicsDataSet.

## Syntax

GraphicsTextureCoordinateSet.**Id**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
