# DataMedium.String Property

Parent Object: [DataMedium](../DataMedium/DataMedium.md)

## Description

Specifies a String containing the data.

## Syntax

DataMedium.**String**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
