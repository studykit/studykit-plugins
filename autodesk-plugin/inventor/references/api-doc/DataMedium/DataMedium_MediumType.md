# DataMedium.MediumType Property

Parent Object: [DataMedium](../DataMedium/DataMedium.md)

## Description

Specifies the type of the data.

## Syntax

DataMedium.**MediumType**() As [MediumTypeEnum](../MediumTypeEnum.md)

## Property Value

This is a read/write property whose value is a [MediumTypeEnum](../MediumTypeEnum.md).

## Version

Introduced in version 11
