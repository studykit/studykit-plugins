# DerivedParameter.DisabledActionTypes Property

Parent Object: [DerivedParameter](../DerivedParameter/DerivedParameter.md)

## Description

Gets and sets the action types valid for this parameter.

## Syntax

DerivedParameter.**DisabledActionTypes**() As [ActionTypeEnum](../ActionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ActionTypeEnum](../ActionTypeEnum.md).

## Version

Introduced in version 11
