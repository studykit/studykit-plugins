# iFeatureVectorInput Object

Derived from: [iFeatureInput](../iFeatureInput/iFeatureInput.md) Object

## Description

The iFeatureVectorInput object represents the vector input information of an iFeature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../iFeatureVectorInput/iFeatureVectorInput_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [EntityType](../iFeatureVectorInput/iFeatureVectorInput_EntityType.md) | Property that returns the type of geometry that are valid. The value returned is the sum of values specifying the valid entity types. |
| [FlipOption](../iFeatureVectorInput/iFeatureVectorInput_FlipOption.md) | Gets and sets whether the geometry selection needs to be flipped. |
| [LineInput](../iFeatureVectorInput/iFeatureVectorInput_LineInput.md) | Gets and sets the geometry matching the vector input. |
| [Name](../iFeatureVectorInput/iFeatureVectorInput_Name.md) | Property that gets the name associated with this input. When placing an iFeature using the API you can use the name to identify each input when assigning the desired values and entities. |
| [Prompt](../iFeatureVectorInput/iFeatureVectorInput_Prompt.md) | Property that gets the prompt that is used for this input during the placement of the iFeature. |
| [Type](../iFeatureVectorInput/iFeatureVectorInput_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 6
