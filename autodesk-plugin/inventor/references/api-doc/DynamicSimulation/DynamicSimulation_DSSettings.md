# DynamicSimulation.DSSettings Property

Parent Object: [DynamicSimulation](../DynamicSimulation/DynamicSimulation.md)

## Description

Gets the DSSettings object which provides access to settings associated with this simulation.

## Syntax

DynamicSimulation.**DSSettings**() As [DSSettings](../DSSettings/DSSettings.md)

## Property Value

This is a read only property whose value is a [DSSettings](../DSSettings/DSSettings.md).

## Version

Introduced in version 2013
