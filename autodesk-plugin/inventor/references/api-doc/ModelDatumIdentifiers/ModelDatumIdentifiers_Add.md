# ModelDatumIdentifiers.Add Method

Parent Object: [ModelDatumIdentifiers](../ModelDatumIdentifiers/ModelDatumIdentifiers.md)

## Description

Method that creates a datum identifier symbol.

## Syntax

ModelDatumIdentifiers.**Add**( ***Definition*** As [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md) ) As [ModelDatumIdentifier](../ModelDatumIdentifier/ModelDatumIdentifier.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [ModelDatumIdentifierDefinition](../ModelDatumIdentifierDefinition/ModelDatumIdentifierDefinition.md) | Input ModelDatumIdentifierDefinition object that defines the datum identifer to be created. |

## Version

Introduced in version 2018
