# PartOptions.DisplayExtendedName Property

Parent Object: [PartOptions](../PartOptions/PartOptions.md)

## Description

Read-write property that gets and sets whether to display extended information after the feature name in the browser.

## Syntax

PartOptions.**DisplayExtendedName**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
