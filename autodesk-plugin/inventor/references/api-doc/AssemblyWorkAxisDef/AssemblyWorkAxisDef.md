# AssemblyWorkAxisDef Object

## Description

The AssemblyWorkAxisDef object allows you to get and set the information that specifies a fixed work axis.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetData](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_GetData.md) | Method that gets all of the data defining an assembly work axis. |
| [PutData](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_PutData.md) | Method that sets all of the data defining an assembly work axis. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Axis](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_Axis.md) | Property that returns the X-axis vector of the work axis. |
| [Constraints](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_Constraints.md) | Property that returns the collection of constraints that are acting on the work axis. This returns all of the constraints, including those returned by the DefinitionConstraints property. |
| [OriginPoint](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_OriginPoint.md) | Property that returns the origin point of the work axis. |
| [Parent](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_Parent.md) | Property that returns the parent object from whom this object can logically be reached. |
| [Type](../AssemblyWorkAxisDef/AssemblyWorkAxisDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 6
