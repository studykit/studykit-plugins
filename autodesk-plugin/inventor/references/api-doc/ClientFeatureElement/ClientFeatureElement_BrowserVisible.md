# ClientFeatureElement.BrowserVisible Property

Parent Object: [ClientFeatureElement](../ClientFeatureElement/ClientFeatureElement.md)

## Description

Read-write property that gets and sets whether this element is visible in the browser under the ClientFeature node.

## Syntax

ClientFeatureElement.**BrowserVisible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
