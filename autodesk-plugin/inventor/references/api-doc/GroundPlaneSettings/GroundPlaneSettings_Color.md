# GroundPlaneSettings.Color Property

Parent Object: [GroundPlaneSettings](../GroundPlaneSettings/GroundPlaneSettings.md)

## Description

Read-write property that gets and sets the color for the ground plane for the area inside the margin.

## Syntax

GroundPlaneSettings.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 2011
