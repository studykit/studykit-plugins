# DerivedAssemblyComponent.ReferencedDocumentDescriptor Property

Parent Object: [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Description

Property that returns the DocumentDescriptor object.

## Syntax

DerivedAssemblyComponent.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 11
