# DerivedAssemblyComponent.Name Property

Parent Object: [DerivedAssemblyComponent](../DerivedAssemblyComponent/DerivedAssemblyComponent.md)

## Description

Property that returns the component's name.

## Syntax

DerivedAssemblyComponent.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 5.3
