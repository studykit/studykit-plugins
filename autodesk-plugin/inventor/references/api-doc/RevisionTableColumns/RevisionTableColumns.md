# RevisionTableColumns Object

## Description

This collection object represents the column objects in a revision table.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../RevisionTableColumns/RevisionTableColumns_Add.md) | Method that creates a new revision table column based on a property. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RevisionTableColumns/RevisionTableColumns_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../RevisionTableColumns/RevisionTableColumns_Count.md) | Property that returns the number of items in the collection. |
| [Item](../RevisionTableColumns/RevisionTableColumns_Item.md) | Returns the specified Column object from the collection. |
| [Type](../RevisionTableColumns/RevisionTableColumns_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
