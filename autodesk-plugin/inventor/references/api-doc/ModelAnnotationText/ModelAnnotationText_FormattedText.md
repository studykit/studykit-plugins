# ModelAnnotationText.FormattedText Property

Parent Object: [ModelAnnotationText](../ModelAnnotationText/ModelAnnotationText.md)

## Description

Property that gets and sets the fully formatted string that defines the contents of the dimension text.

## Syntax

ModelAnnotationText.**FormattedText**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
