# ModelAnnotationText Object

## Description

The ModelAnnotationText object represents text in a model annotation or dimension.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ModelAnnotationText/ModelAnnotationText_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Color](../ModelAnnotationText/ModelAnnotationText_Color.md) | Property that gets and sets the color of the annotation text. |
| [Font](../ModelAnnotationText/ModelAnnotationText_Font.md) | Property that gets and sets the default font. |
| [FormattedText](../ModelAnnotationText/ModelAnnotationText_FormattedText.md) | Property that gets and sets the fully formatted string that defines the contents of the dimension text. |
| [HorizontalJustification](../ModelAnnotationText/ModelAnnotationText_HorizontalJustification.md) | Property that gets and sets the horizontal justification override of the annotation text. |
| [LineSpacing](../ModelAnnotationText/ModelAnnotationText_LineSpacing.md) | Property that gets and sets the line spacing of the dimension text. |
| [LineSpacingType](../ModelAnnotationText/ModelAnnotationText_LineSpacingType.md) | Property that gets and sets the method used to define the line spacing. |
| [Parent](../ModelAnnotationText/ModelAnnotationText_Parent.md) | Read-only property that returns the parent of the object. |
| [Position](../ModelAnnotationText/ModelAnnotationText_Position.md) | Property that gets and sets the position of the origin of the dimension text. |
| [Size](../ModelAnnotationText/ModelAnnotationText_Size.md) | Property that gets and sets the default size of the text in centimeters. |
| [StackedTextPosition](../ModelAnnotationText/ModelAnnotationText_StackedTextPosition.md) | Property that gets and sets the position (alignment) of the stacked text with respect to regular text. |
| [Text](../ModelAnnotationText/ModelAnnotationText_Text.md) | Read-only property that gets the string that defines the dimension text. When getting this property, the returned string has all formatting removed and contains the actual text characters displayed in the text. |
| [Type](../ModelAnnotationText/ModelAnnotationText_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
| [VerticalJustification](../ModelAnnotationText/ModelAnnotationText_VerticalJustification.md) | Property that gets and sets the vertical justification of the ModelAnnotationText. |
| [WidthScale](../ModelAnnotationText/ModelAnnotationText_WidthScale.md) | Property that gets and sets the width scale factor for the dimension text. |
