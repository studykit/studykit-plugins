# FeaturePatternElement.Index Property

Parent Object: [FeaturePatternElement](../FeaturePatternElement/FeaturePatternElement.md)

## Description

Property that returns the index of this element within the pattern.

## Syntax

FeaturePatternElement.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
