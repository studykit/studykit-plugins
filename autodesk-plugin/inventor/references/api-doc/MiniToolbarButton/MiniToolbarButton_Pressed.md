# MiniToolbarButton.Pressed Property

Parent Object: [MiniToolbarButton](../MiniToolbarButton/MiniToolbarButton.md)

## Description

Read-write property that gets and sets whether the button is currently pressed (active).

## Syntax

MiniToolbarButton.**Pressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
