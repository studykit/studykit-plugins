# ShrinkwrapComponents.Count Property

Parent Object: [ShrinkwrapComponents](../ShrinkwrapComponents/ShrinkwrapComponents.md)

## Description

Gets the number of items in this collection.

## Syntax

ShrinkwrapComponents.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2018
