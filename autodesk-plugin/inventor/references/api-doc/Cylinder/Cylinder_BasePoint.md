# Cylinder.BasePoint Property

Parent Object: [Cylinder](../Cylinder/Cylinder.md)

## Description

Specifies the base of the cylinder.

## Syntax

Cylinder.**BasePoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).
