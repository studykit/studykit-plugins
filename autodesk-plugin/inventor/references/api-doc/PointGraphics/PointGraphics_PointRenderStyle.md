# PointGraphics.PointRenderStyle Property

Parent Object: [PointGraphics](../PointGraphics/PointGraphics.md)

## Description

Gets and sets the point style associated with the PointGraphics object.

## Syntax

PointGraphics.**PointRenderStyle**() As [PointRenderStyleEnum](../PointRenderStyleEnum.md)

## Property Value

This is a read/write property whose value is a [PointRenderStyleEnum](../PointRenderStyleEnum.md).
