# PointGraphics.RemoveViewSpaceAnchor Method

Parent Object: [PointGraphics](../PointGraphics/PointGraphics.md)

## Description

The RemoveViewSpaceAnchor method removes the view space anchor from the object, and sets the Anchored property to false.

## Syntax

PointGraphics.**RemoveViewSpaceAnchor**()

## Version

Introduced in version 9
