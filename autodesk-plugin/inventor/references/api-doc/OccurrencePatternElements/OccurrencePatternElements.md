# OccurrencePatternElements Object

## Description

The OccurrencePatternElements collection object provides access to the instances ( objects) that result from the creation of a pattern.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../OccurrencePatternElements/OccurrencePatternElements_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../OccurrencePatternElements/OccurrencePatternElements_Count.md) | Property that returns the number of items in the collection. |
| [Item](../OccurrencePatternElements/OccurrencePatternElements_Item.md) | Method that returns the specified OccurrencePatternElement object from the collection. This is the default method of the OccurrencePatternElements collection object. |
| [Parent](../OccurrencePatternElements/OccurrencePatternElements_Parent.md) | Property that returns the parent OccurrencePattern object. |
| [Type](../OccurrencePatternElements/OccurrencePatternElements_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
