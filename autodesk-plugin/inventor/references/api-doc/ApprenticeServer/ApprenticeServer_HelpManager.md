# ApprenticeServer.HelpManager Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Gets the Help Manager object that provides access to the help-related activity taking place in the system.

## Syntax

ApprenticeServer.**HelpManager**() As [HelpManager](../HelpManager/HelpManager.md)

## Property Value

This is a read only property whose value is a [HelpManager](../HelpManager/HelpManager.md).

## Version

Introduced in version 2013
