# ApprenticeServer.FileManager Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Gets the file manager object.

## Syntax

ApprenticeServer.**FileManager**() As [FileManager](../FileManager/FileManager.md)

## Property Value

This is a read only property whose value is a [FileManager](../FileManager/FileManager.md).

## Version

Introduced in version 6
