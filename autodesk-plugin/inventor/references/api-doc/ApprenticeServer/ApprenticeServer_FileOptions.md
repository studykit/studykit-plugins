# ApprenticeServer.FileOptions Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Gets the File Options object.

## Syntax

ApprenticeServer.**FileOptions**() As [FileOptions](../FileOptions/FileOptions.md)

## Property Value

This is a read only property whose value is a [FileOptions](../FileOptions/FileOptions.md).

## Version

Introduced in version 9
