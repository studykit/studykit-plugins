# ApprenticeServer.Document Property

Parent Object: [ApprenticeServer](../ApprenticeServer/ApprenticeServer.md)

## Description

Gets the top-level Apprentice Server Document.

## Syntax

ApprenticeServer.**Document**() As [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md)

## Property Value

This is a read only property whose value is an [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md).

## Version

Introduced in version 4
