# ReferencedOpaqueFileDescriptor.Delete Method

Parent Object: [ReferencedOpaqueFileDescriptor](../ReferencedOpaqueFileDescriptor/ReferencedOpaqueFileDescriptor.md)

## Description

Deletes the opaque reference (does not delete the referenced file).

## Syntax

ReferencedOpaqueFileDescriptor.**Delete**()

## Version

Introduced in version 2017
