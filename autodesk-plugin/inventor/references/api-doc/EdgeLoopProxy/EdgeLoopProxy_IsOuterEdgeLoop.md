# EdgeLoopProxy.IsOuterEdgeLoop Property

Parent Object: [EdgeLoopProxy](../EdgeLoopProxy/EdgeLoopProxy.md)

## Description

Gets whether this EdgeLoop is an external loop, or a loop that encloses material as opposed to a void.

## Syntax

EdgeLoopProxy.**IsOuterEdgeLoop**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
