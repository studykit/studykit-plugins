# SymmetryConstraintProxy.EntityTwo Property

Parent Object: [SymmetryConstraintProxy](../SymmetryConstraintProxy/SymmetryConstraintProxy.md)

## Description

Property that returns the sketch entity that is constrained to be symmetric to the entity returned by the EntityOne property.

## Syntax

SymmetryConstraintProxy.**EntityTwo**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
