# SmoothConstraint3D.Deletable Property

Parent Object: [SmoothConstraint3D](../SmoothConstraint3D/SmoothConstraint3D.md)

## Description

Indicates whether this object can be deleted.

## Syntax

SmoothConstraint3D.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
