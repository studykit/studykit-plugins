# DriveConstraintSettings.OnCollision Event

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Event that is fired whenever a collision occurs during the animation/play of the constraint.

## Syntax

DriveConstraintSettings.**OnCollision**()

## Version

Introduced in version 2012
