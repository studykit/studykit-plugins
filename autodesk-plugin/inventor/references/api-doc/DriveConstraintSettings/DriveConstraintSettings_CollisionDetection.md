# DriveConstraintSettings.CollisionDetection Property

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Gets and sets whether to drive the constrained assembly until a collision is detected.

## Syntax

DriveConstraintSettings.**CollisionDetection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
