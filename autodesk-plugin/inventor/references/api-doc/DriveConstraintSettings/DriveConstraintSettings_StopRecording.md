# DriveConstraintSettings.StopRecording Method

Parent Object: [DriveConstraintSettings](../DriveConstraintSettings/DriveConstraintSettings.md)

## Description

Stops recording the drive constraint animation and generates the animation file.

## Syntax

DriveConstraintSettings.**StopRecording**()

## Version

Introduced in version 2012
