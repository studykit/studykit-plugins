# HelicalCurve.SketchEntities Property

Parent Object: [HelicalCurve](../HelicalCurve/HelicalCurve.md)

## Description

Returns a collection of sketch entities that belong to the helical curve.

## Syntax

HelicalCurve.**SketchEntities**() As [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchEntities3DEnumerator](../SketchEntities3DEnumerator/SketchEntities3DEnumerator.md).

## Version

Introduced in version 2019
