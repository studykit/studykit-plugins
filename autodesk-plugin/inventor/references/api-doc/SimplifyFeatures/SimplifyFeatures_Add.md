# SimplifyFeatures.Add Method

Parent Object: [SimplifyFeatures](../SimplifyFeatures/SimplifyFeatures.md)

## Description

Method that creates a new SimplifyFeature. The new created SimplifyFeature is returned.

## Syntax

SimplifyFeatures.**Add**( ***pDefinition*** As [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md) ) As [SimplifyFeature](../SimplifyFeature/SimplifyFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| pDefinition | [SimplifyDefinition](../SimplifyDefinition/SimplifyDefinition.md) | Input SimplifyDefinition object created using CreateDefinition method. |
