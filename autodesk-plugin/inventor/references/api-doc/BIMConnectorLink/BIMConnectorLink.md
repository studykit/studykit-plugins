# BIMConnectorLink Object

## Description

BIMConnectorLink object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../BIMConnectorLink/BIMConnectorLink_Delete.md) | Method that deletes the connectorlink. |
| [GetReferenceKey](../BIMConnectorLink/BIMConnectorLink_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../BIMConnectorLink/BIMConnectorLink_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [ConnectorOne](../BIMConnectorLink/BIMConnectorLink_ConnectorOne.md) | Read-only property that gets the first of two connectors that this link is between. |
| [ConnectorTwo](../BIMConnectorLink/BIMConnectorLink_ConnectorTwo.md) | Read-only property that gets the first of two connectors that this link is between. |
| [Name](../BIMConnectorLink/BIMConnectorLink_Name.md) | Read-write property that gets and sets the displayed name of the connectorlink. This is the name that is visible in the browser and is editable by the end-user. |
| [Type](../BIMConnectorLink/BIMConnectorLink_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[BIMConnectorLinks.Add](../BIMConnectorLinks/BIMConnectorLinks_Add.md), [BIMConnectorLinks.Item](../BIMConnectorLinks/BIMConnectorLinks_Item.md)

## Version

Introduced in version 2011
