# FilesEnumerator Object

## Description

The FilesEnumerator object provides access to all Inventor files.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FilesEnumerator/FilesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../FilesEnumerator/FilesEnumerator_Count.md) | Property that returns the number of items in the collection. |
| [Item](../FilesEnumerator/FilesEnumerator_Item.md) | Returns the specified File object from the collection. |
| [Type](../FilesEnumerator/FilesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[File.AllReferencedFiles](../File/File_AllReferencedFiles.md), [File.ReferencedFiles](../File/File_ReferencedFiles.md), [File.ReferencingFiles](../File/File_ReferencingFiles.md), [FileManager.Files](../FileManager/FileManager_Files.md)

## Version

Introduced in version 11
