# ToNextExtent.Terminator Property

Parent Object: [ToNextExtent](../ToNextExtent/ToNextExtent.md)

## Description

Gets and sets the SurfaceBody that specifies the solid or the surface on which to terminate the revolution.

## Syntax

ToNextExtent.**Terminator**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read/write property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2008
