# Preparations Object

## Description

Represents the Preparations environment in a weldment assembly.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Edit](../Preparations/Preparations_Edit.md) | Method that activates this object for editing by the user. |
| [ExitEdit](../Preparations/Preparations_ExitEdit.md) | Method that causes the current editing environment to be closed and the user interface to return to weldment editing environment. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../Preparations/Preparations_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Type](../Preparations/Preparations_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[WeldmentComponentDefinition.Preparations](../WeldmentComponentDefinition/WeldmentComponentDefinition_Preparations.md)

## Version

Introduced in version 2010
