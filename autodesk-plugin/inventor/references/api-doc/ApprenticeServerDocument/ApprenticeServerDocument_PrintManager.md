# ApprenticeServerDocument.PrintManager Property

Parent Object: [ApprenticeServerDocument](../ApprenticeServerDocument/ApprenticeServerDocument.md)

## Description

Property that returns the object for this document.

## Syntax

ApprenticeServerDocument.**PrintManager**() As [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md)

## Property Value

This is a read only property whose value is an [ApprenticePrintManager](../ApprenticePrintManager/ApprenticePrintManager.md).

## Version

Introduced in version 9
