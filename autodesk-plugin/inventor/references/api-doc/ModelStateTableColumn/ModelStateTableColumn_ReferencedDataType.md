# ModelStateTableColumn.ReferencedDataType Property

Parent Object: [ModelStateTableColumn](../ModelStateTableColumn/ModelStateTableColumn.md)

## Description

Read - only property that returns the data type contained in the column.

## Syntax

ModelStateTableColumn.**ReferencedDataType**() As [iComponentColumnTypeEnum](../iComponentColumnTypeEnum.md)

## Property Value

This is a read only property whose value is an [iComponentColumnTypeEnum](../iComponentColumnTypeEnum.md).

## Version

Introduced in version 2022
