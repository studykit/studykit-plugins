# DerivedPartComponentProxy.SurfaceBodies Property

Parent Object: [DerivedPartComponentProxy](../DerivedPartComponentProxy/DerivedPartComponentProxy.md)

## Description

Property that returns the SurfaceBody objects that represent work surfaces that were created as a result of this derived part.

## Syntax

DerivedPartComponentProxy.**SurfaceBodies**() As [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md)

## Property Value

This is a read only property whose value is a [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md).

## Version

Introduced in version 2008
