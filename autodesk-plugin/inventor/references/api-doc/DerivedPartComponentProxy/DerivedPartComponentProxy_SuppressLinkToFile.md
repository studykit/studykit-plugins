# DerivedPartComponentProxy.SuppressLinkToFile Property

Parent Object: [DerivedPartComponentProxy](../DerivedPartComponentProxy/DerivedPartComponentProxy.md)

## Description

Gets and Sets a Boolean flag indicating whether the link to the part file is suppressed.

## Syntax

DerivedPartComponentProxy.**SuppressLinkToFile**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
