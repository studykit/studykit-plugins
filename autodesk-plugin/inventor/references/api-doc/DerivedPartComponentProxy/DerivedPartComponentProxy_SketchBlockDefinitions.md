# DerivedPartComponentProxy.SketchBlockDefinitions Property

Parent Object: [DerivedPartComponentProxy](../DerivedPartComponentProxy/DerivedPartComponentProxy.md)

## Description

Property that returns the SketchBlockDefinition objects that were created as a result of this derived part.

## Syntax

DerivedPartComponentProxy.**SketchBlockDefinitions**() As [SketchBlockDefinitionsEnumerator](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchBlockDefinitionsEnumerator](../SketchBlockDefinitionsEnumerator/SketchBlockDefinitionsEnumerator.md).

## Version

Introduced in version 2010
