# SketchRectangularPatterns.Item Property

Parent Object: [SketchRectangularPatterns](../SketchRectangularPatterns/SketchRectangularPatterns.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

SketchRectangularPatterns.**Item**( ***Index*** As Long ) As [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md)

## Property Value

This is a read only property whose value is a [SketchRectangularPattern](../SketchRectangularPattern/SketchRectangularPattern.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the pattern to return. If an out of range index is provided, an error occurs |

## Version

Introduced in version 2025.1
