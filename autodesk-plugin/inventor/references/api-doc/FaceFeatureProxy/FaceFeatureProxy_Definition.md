# FaceFeatureProxy.Definition Property

Parent Object: [FaceFeatureProxy](../FaceFeatureProxy/FaceFeatureProxy.md)

## Description

Property that gets the FaceFeatureDefinition object associated with this face feature.

## Syntax

FaceFeatureProxy.**Definition**() As [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md)

## Property Value

This is a read only property whose value is a [FaceFeatureDefinition](../FaceFeatureDefinition/FaceFeatureDefinition.md).

## Version

Introduced in version 2009
