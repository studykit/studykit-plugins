# HoleTableRow.Delete Method

Parent Object: [HoleTableRow](../HoleTableRow/HoleTableRow.md)

## Description

Method that deletes the HoleTableRow and the associated hole tag.

## Syntax

HoleTableRow.**Delete**()

## Version

Introduced in version 10
