# SculptSurface Object

## Description

The SculptSurface object provides access to a sculpt surface and its associated direction. The surface defines a boundary of the sculpt feature and the associated direction defines the side of the surface used to create the sculpt feature.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Direction](../SculptSurface/SculptSurface_Direction.md) | Gets and sets the direction associated with the surface. |
| [Surface](../SculptSurface/SculptSurface_Surface.md) | Gets and sets a boundary surface for the sculpt feature. |
| [Type](../SculptSurface/SculptSurface_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[SculptFeatures.CreateSculptSurface](../SculptFeatures/SculptFeatures_CreateSculptSurface.md)

## Version

Introduced in version 11
