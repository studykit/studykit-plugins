# RevolveFeatureProxy.Extent Property

Parent Object: [RevolveFeatureProxy](../RevolveFeatureProxy/RevolveFeatureProxy.md)

## Description

Property that returns the object that defines how the extent of the feature is defined. The type of extent object that this property will return can be determined by using the ExtentType property.

## Syntax

RevolveFeatureProxy.**Extent**() As [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md)

## Property Value

This is a read only property whose value is a [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md).

## Version

Introduced in version 9
