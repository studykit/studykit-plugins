# RevolveFeatureProxy.Profile Property

Parent Object: [RevolveFeatureProxy](../RevolveFeatureProxy/RevolveFeatureProxy.md)

## Description

Gets and sets the Profile that defines the shape of the feature.

## Syntax

RevolveFeatureProxy.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 9
