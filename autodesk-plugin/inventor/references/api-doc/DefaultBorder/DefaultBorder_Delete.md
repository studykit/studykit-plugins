# DefaultBorder.Delete Method

Parent Object: [DefaultBorder](../DefaultBorder/DefaultBorder.md)

## Description

Method that deletes the title block from the sheet.

## Syntax

DefaultBorder.**Delete**()

## Version

Introduced in version 5.3
