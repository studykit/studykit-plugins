# DefaultBorder.HorizontalZoneCount Property

Parent Object: [DefaultBorder](../DefaultBorder/DefaultBorder.md)

## Description

Property that returns the number of horizontal zones to be created for the border.

## Syntax

DefaultBorder.**HorizontalZoneCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 5.3
