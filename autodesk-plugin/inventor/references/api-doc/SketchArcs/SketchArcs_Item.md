# SketchArcs.Item Property

Parent Object: [SketchArcs](../SketchArcs/SketchArcs.md)

## Description

Returns the specified SketchArcobject from the collection.

## Syntax

SketchArcs.**Item**( ***Index*** As Long ) As [SketchArc](../SketchArc/SketchArc.md)

## Property Value

This is a read only property whose value is a [SketchArc](../SketchArc/SketchArc.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 5
