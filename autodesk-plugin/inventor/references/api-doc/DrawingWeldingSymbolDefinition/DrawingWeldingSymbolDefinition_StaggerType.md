# DrawingWeldingSymbolDefinition.StaggerType Property

Parent Object: [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md)

## Description

Gets and sets the stagger type of the welding symbols for fillets. This is applicable only if fillet welding symbols are set on both sides of the reference line.

## Syntax

DrawingWeldingSymbolDefinition.**StaggerType**() As [StaggerTypeEnum](../StaggerTypeEnum.md)

## Property Value

This is a read/write property whose value is a [StaggerTypeEnum](../StaggerTypeEnum.md).

## Version

Introduced in version 2024
