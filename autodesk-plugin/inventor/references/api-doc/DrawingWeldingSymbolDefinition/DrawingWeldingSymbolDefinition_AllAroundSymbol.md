# DrawingWeldingSymbolDefinition.AllAroundSymbol Property

Parent Object: [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md)

## Description

Gets and sets whether to add the all-around indicator to the symbol.

## Syntax

DrawingWeldingSymbolDefinition.**AllAroundSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
