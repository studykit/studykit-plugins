# DrawingWeldingSymbolDefinition.UseSpacer Property

Parent Object: [DrawingWeldingSymbolDefinition](../DrawingWeldingSymbolDefinition/DrawingWeldingSymbolDefinition.md)

## Description

Gets and sets whether to use spacer or not.

## Syntax

DrawingWeldingSymbolDefinition.**UseSpacer**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
