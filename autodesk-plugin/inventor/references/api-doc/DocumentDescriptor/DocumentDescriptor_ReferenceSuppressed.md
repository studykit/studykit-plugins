# DocumentDescriptor.ReferenceSuppressed Property

Parent Object: [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Description

Property that returns whether the reference is currently suppressed.

## Syntax

DocumentDescriptor.**ReferenceSuppressed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
