# DocumentDescriptor Object

## Description

The DocumentDescriptor object represents a native document reference.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DocumentDescriptor/DocumentDescriptor_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [DisplayName](../DocumentDescriptor/DocumentDescriptor_DisplayName.md) | Property that returns the display name of the referenced document as currently found (or the last known display name, if the reference is not found). |
| [FullDocumentName](../DocumentDescriptor/DocumentDescriptor_FullDocumentName.md) | Property that returns the full path name of the referenced document as currently found (or the last known full document name, if the reference is not found). |
| [Parent](../DocumentDescriptor/DocumentDescriptor_Parent.md) | Property that returns the parent of this object. |
| [ReferencedDocument](../DocumentDescriptor/DocumentDescriptor_ReferencedDocument.md) | Property that returns the Document object being referenced. |
| [ReferencedDocumentType](../DocumentDescriptor/DocumentDescriptor_ReferencedDocumentType.md) | Property that returns type of the referenced document. |
| [ReferencedFileDescriptor](../DocumentDescriptor/DocumentDescriptor_ReferencedFileDescriptor.md) | Property that returns the descriptor object for the file object that contains the referenced document. |
| [ReferenceDisabled](../DocumentDescriptor/DocumentDescriptor_ReferenceDisabled.md) | Gets whether the link to this file is disabled. |
| [ReferencedModelState](../DocumentDescriptor/DocumentDescriptor_ReferencedModelState.md) | Read-only property that returns the model state type of the referenced document. |
| [ReferenceInternalNameDifferent](../DocumentDescriptor/DocumentDescriptor_ReferenceInternalNameDifferent.md) | Gets whether the internal name of the referenced file is different from the last time the parent (referencing) file was saved. A value of True indicates that the currently referenced file has a different internal name than the originally referenced file. |
| [ReferenceLocationDifferent](../DocumentDescriptor/DocumentDescriptor_ReferenceLocationDifferent.md) | Gets whether the file path or the file name of the referenced document is different from the last time the parent (referencing) document was saved. A value of True indicates that the currently referenced document has a different file path or file name. |
| [ReferenceMissing](../DocumentDescriptor/DocumentDescriptor_ReferenceMissing.md) | Property that returns whether the referenced document is missing (unresolved or suppressed). |
| [ReferenceSuppressed](../DocumentDescriptor/DocumentDescriptor_ReferenceSuppressed.md) | Property that returns whether the reference is currently suppressed. |
| [Type](../DocumentDescriptor/DocumentDescriptor_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
