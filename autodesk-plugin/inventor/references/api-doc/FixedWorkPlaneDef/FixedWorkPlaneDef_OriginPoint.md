# FixedWorkPlaneDef.OriginPoint Property

Parent Object: [FixedWorkPlaneDef](../FixedWorkPlaneDef/FixedWorkPlaneDef.md)

## Description

Property that returns the origin point of the work plane.

## Syntax

FixedWorkPlaneDef.**OriginPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
