# FoldFeature.AppearanceSourceType Property

Parent Object: [FoldFeature](../FoldFeature/FoldFeature.md)

## Description

Gets and sets the source of the appearance for the feature.

## Syntax

FoldFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
