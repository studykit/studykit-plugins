# PresentationMeshFeature.AppearanceSourceType Property

Parent Object: [PresentationMeshFeature](../PresentationMeshFeature/PresentationMeshFeature.md)

## Description

Read-only property that gets the source of the appearance for the object.

## Syntax

PresentationMeshFeature.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read only property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2018
