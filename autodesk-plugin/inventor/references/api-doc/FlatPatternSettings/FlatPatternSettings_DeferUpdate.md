# FlatPatternSettings.DeferUpdate Property

Parent Object: [FlatPatternSettings](../FlatPatternSettings/FlatPatternSettings.md)

## Description

Read-write property that gets and sets Flat Pattern defer update status.

## Syntax

FlatPatternSettings.**DeferUpdate**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2017
