# GraphicsVisibilityEnum Enumerator

## Description

Indicates whether some, all, or none of the client graphics are visible.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllGraphicsVisible | 25091 | All client graphics are selectable. |
| kNoGraphicsVisible | 25089 | No client graphics are selectable. |
| kSomeGraphicsVisible | 25090 | Some, but not all, client graphics are selectable. |

## Version

Introduced in version 5
