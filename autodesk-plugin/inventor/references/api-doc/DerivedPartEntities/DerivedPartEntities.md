# DerivedPartEntities Object

## Description

The DerivedPartEntities collection object provides access to all of the objects associated with a specific DerivedPartDefinition object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DerivedPartEntities/DerivedPartEntities_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DerivedPartEntities/DerivedPartEntities_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DerivedPartEntities/DerivedPartEntities_Item.md) | Returns the specified DerivedPartEntity object from the collection. |
| [Type](../DerivedPartEntities/DerivedPartEntities_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
