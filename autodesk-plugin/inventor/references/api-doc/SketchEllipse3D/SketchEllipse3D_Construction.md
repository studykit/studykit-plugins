# SketchEllipse3D.Construction Property

Parent Object: [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchEllipse3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
