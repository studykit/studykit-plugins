# SketchEllipse3D.MinorRadius Property

Parent Object: [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Description

Gets the MinorRadius of the ellipse.

## Syntax

SketchEllipse3D.**MinorRadius**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
