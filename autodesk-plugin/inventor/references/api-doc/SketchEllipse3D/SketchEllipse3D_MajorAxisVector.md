# SketchEllipse3D.MajorAxisVector Property

Parent Object: [SketchEllipse3D](../SketchEllipse3D/SketchEllipse3D.md)

## Description

Gets the MajorAxisVector of the ellipse.

## Syntax

SketchEllipse3D.**MajorAxisVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read only property whose value is a [UnitVector](../UnitVector/UnitVector.md).

## Version

Introduced in version 2010
