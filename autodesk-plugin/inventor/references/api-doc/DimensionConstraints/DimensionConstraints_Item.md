# DimensionConstraints.Item Property

Parent Object: [DimensionConstraints](../DimensionConstraints/DimensionConstraints.md)

## Description

Returns the specified sketch dimension constraint object from the collection.

## Syntax

DimensionConstraints.**Item**( ***Index*** As Long ) As [DimensionConstraint](../DimensionConstraint/DimensionConstraint.md)

## Property Value

This is a read only property whose value is a [DimensionConstraint](../DimensionConstraint/DimensionConstraint.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 5
