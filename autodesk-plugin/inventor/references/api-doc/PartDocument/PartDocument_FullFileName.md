# PartDocument.FullFileName Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets/Sets the fully qualified file-name of this Document.

## Syntax

PartDocument.**FullFileName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 4
