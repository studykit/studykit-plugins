# PartDocument.ModelStateName Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Read-only property that returns the model state that this document represents.

## Syntax

PartDocument.**ModelStateName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
