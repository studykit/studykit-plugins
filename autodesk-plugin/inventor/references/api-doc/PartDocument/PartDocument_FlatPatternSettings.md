# PartDocument.FlatPatternSettings Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the FlatPatternSettings object.

## Syntax

PartDocument.**FlatPatternSettings**() As [FlatPatternSettings](../FlatPatternSettings/FlatPatternSettings.md)

## Property Value

This is a read only property whose value is a [FlatPatternSettings](../FlatPatternSettings/FlatPatternSettings.md).

## Version

Introduced in version 2017
