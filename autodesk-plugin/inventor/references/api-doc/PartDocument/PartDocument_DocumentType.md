# PartDocument.DocumentType Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Gets the constant that indicates the type of this document.

## Syntax

PartDocument.**DocumentType**() As [DocumentTypeEnum](../DocumentTypeEnum.md)

## Property Value

This is a read only property whose value is a [DocumentTypeEnum](../DocumentTypeEnum.md).

## Version

Introduced in version 4
