# PartDocument.ActiveMarkStyle Property

Parent Object: [PartDocument](../PartDocument/PartDocument.md)

## Description

Property to get or set the active mark style object.

## Syntax

PartDocument.**ActiveMarkStyle**() As [MarkStyle](../MarkStyle/MarkStyle.md)

## Property Value

This is a read/write property whose value is a [MarkStyle](../MarkStyle/MarkStyle.md).

## Version

Introduced in version 2023
