# InterferenceResult.Centroid Property

Parent Object: [InterferenceResult](../InterferenceResult/InterferenceResult.md)

## Description

Property that returns the centroid of the volume of interference.

## Syntax

InterferenceResult.**Centroid**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 6
