# PartComponentDefinition.IsModelStateMember Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Read-only property that returns whether the document is a model state member document or not.

## Syntax

PartComponentDefinition.**IsModelStateMember**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2022
