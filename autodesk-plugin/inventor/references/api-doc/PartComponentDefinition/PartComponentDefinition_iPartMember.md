# PartComponentDefinition.iPartMember Property

Parent Object: [PartComponentDefinition](../PartComponentDefinition/PartComponentDefinition.md)

## Description

Property that returns the iPartMember object.

## Syntax

PartComponentDefinition.**iPartMember**() As [iPartMember](../iPartMember/iPartMember.md)

## Property Value

This is a read only property whose value is an [iPartMember](../iPartMember/iPartMember.md).

## Version

Introduced in version 6
