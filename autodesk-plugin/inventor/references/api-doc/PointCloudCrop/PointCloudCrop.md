# PointCloudCrop Object

## Description

The PointCloudCrop object represents a crop within a point cloud.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PointCloudCrop/PointCloudCrop_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [BoundingBox](../PointCloudCrop/PointCloudCrop_BoundingBox.md) | Read-only property that returns the oriented box presents the bounding volume of the point cloud crop. |
| [KeepInside](../PointCloudCrop/PointCloudCrop_KeepInside.md) | Read-only property that returns whether the cloud points in the bounding box are kept or not in this crop. |
| [Parent](../PointCloudCrop/PointCloudCrop_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Type](../PointCloudCrop/PointCloudCrop_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PointCloudCrops.Add](../PointCloudCrops/PointCloudCrops_Add.md), [PointCloudCrops.Item](../PointCloudCrops/PointCloudCrops_Item.md)

## Version

Introduced in version 2016
