# WidthOffsetWidthExtent.OffsetEntity Property

Parent Object: [WidthOffsetWidthExtent](../WidthOffsetWidthExtent/WidthOffsetWidthExtent.md)

## Description

Property that gets and sets the entity the width is measured relative to.

## Syntax

WidthOffsetWidthExtent.**OffsetEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2009
