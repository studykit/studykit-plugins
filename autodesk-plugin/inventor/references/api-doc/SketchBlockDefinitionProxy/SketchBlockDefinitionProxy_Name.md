# SketchBlockDefinitionProxy.Name Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and sets name of the sketch.

## Syntax

SketchBlockDefinitionProxy.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
