# SketchBlockDefinitionProxy.InsertionPoint Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Property that returns the insertion point of the sketch block definition.

## Syntax

SketchBlockDefinitionProxy.**InsertionPoint**() As [SketchPoint](../SketchPoint/SketchPoint.md)

## Property Value

This is a read only property whose value is a [SketchPoint](../SketchPoint/SketchPoint.md).

## Version

Introduced in version 2010
