# SketchBlockDefinitionProxy.LineWeight Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and Sets the line weight override for the sketch.

## Syntax

SketchBlockDefinitionProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
