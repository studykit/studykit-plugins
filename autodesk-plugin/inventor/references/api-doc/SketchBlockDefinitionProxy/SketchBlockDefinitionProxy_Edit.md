# SketchBlockDefinitionProxy.Edit Method

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Method that causes the Sketch environment to be invoked with this sketch available for interactive edit.

## Syntax

SketchBlockDefinitionProxy.**Edit**()

## Version

Introduced in version 2010
