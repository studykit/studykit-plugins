# SketchBlockDefinitionProxy.Delete Method

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Method that deletes the sketch. This method is only valid for sketches that are not used by a feature.

## Syntax

SketchBlockDefinitionProxy.**Delete**()

## Version

Introduced in version 2010
