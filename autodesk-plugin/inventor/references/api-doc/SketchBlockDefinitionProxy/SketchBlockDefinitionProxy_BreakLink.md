# SketchBlockDefinitionProxy.BreakLink Method

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

Method that breaks the link to the source sketch.

## Syntax

SketchBlockDefinitionProxy.**BreakLink**()
