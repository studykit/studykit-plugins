# SketchBlockDefinitionProxy.Shared Property

Parent Object: [SketchBlockDefinitionProxy](../SketchBlockDefinitionProxy/SketchBlockDefinitionProxy.md)

## Description

Gets and sets whether the profile is shared or not.

## Syntax

SketchBlockDefinitionProxy.**Shared**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
