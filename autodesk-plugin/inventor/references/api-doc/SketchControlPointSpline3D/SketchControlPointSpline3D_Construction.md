# SketchControlPointSpline3D.Construction Property

Parent Object: [SketchControlPointSpline3D](../SketchControlPointSpline3D/SketchControlPointSpline3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchControlPointSpline3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
