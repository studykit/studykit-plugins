# HelicalConstraint3DProxy.NativeObject Property

Parent Object: [HelicalConstraint3DProxy](../HelicalConstraint3DProxy/HelicalConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

HelicalConstraint3DProxy.**NativeObject**() As [HelicalConstraint3D](../HelicalConstraint3D/HelicalConstraint3D.md)

## Property Value

This is a read only property whose value is a [HelicalConstraint3D](../HelicalConstraint3D/HelicalConstraint3D.md).

## Version

Introduced in version 2010
