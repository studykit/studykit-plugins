# DrawingDimension.Parent Property

Parent Object: [DrawingDimension](../DrawingDimension/DrawingDimension.md)

## Description

Property that returns the parent sheet of the object.

## Syntax

DrawingDimension.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 9
