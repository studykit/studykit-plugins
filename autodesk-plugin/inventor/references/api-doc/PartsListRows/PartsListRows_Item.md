# PartsListRows.Item Property

Parent Object: [PartsListRows](../PartsListRows/PartsListRows.md)

## Description

Returns the specified PartsListRows object from the collection. This is the default property of the PartsListRows collection object.

## Syntax

PartsListRows.**Item**( ***Index*** As Long ) As [PartsListRow](../PartsListRow/PartsListRow.md)

## Property Value

This is a read only property whose value is a [PartsListRow](../PartsListRow/PartsListRow.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Value that specifies the index of the PartsListRow to return. |
