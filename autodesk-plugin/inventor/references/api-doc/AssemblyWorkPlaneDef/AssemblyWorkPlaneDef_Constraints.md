# AssemblyWorkPlaneDef.Constraints Property

Parent Object: [AssemblyWorkPlaneDef](../AssemblyWorkPlaneDef/AssemblyWorkPlaneDef.md)

## Description

Property that returns the collection of constraints that are acting on the work plane. This returns all of the constraints, including those returned by the DefinitionConstraints property

## Syntax

AssemblyWorkPlaneDef.**Constraints**() As [AssemblyConstraintsEnumerator](../AssemblyConstraintsEnumerator/AssemblyConstraintsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssemblyConstraintsEnumerator](../AssemblyConstraintsEnumerator/AssemblyConstraintsEnumerator.md).

## Version

Introduced in version 6
