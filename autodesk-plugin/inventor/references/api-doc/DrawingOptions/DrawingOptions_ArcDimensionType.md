# DrawingOptions.ArcDimensionType Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the type of dimensioning to use while placing arc dimensions on drawing sheets.

## Syntax

DrawingOptions.**ArcDimensionType**() As [ArcDimensionTypeEnum](../ArcDimensionTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArcDimensionTypeEnum](../ArcDimensionTypeEnum.md).

## Version

Introduced in version 2008
