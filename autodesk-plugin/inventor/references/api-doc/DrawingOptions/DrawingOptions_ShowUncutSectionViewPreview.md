# DrawingOptions.ShowUncutSectionViewPreview Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the section view previews should be displayed in uncut mode.

## Syntax

DrawingOptions.**ShowUncutSectionViewPreview**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
