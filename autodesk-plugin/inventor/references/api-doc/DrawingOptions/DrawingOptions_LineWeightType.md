# DrawingOptions.LineWeightType Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the type of line weight to apply when displaying lines in the drawing.

## Syntax

DrawingOptions.**LineWeightType**() As [LineWeightTypeEnum](../LineWeightTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineWeightTypeEnum](../LineWeightTypeEnum.md).

## Version

Introduced in version 2008
