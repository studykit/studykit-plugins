# DrawingOptions.TitleBlockAlignment Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets the default title block location.

## Syntax

DrawingOptions.**TitleBlockAlignment**() As [TitleBlockAlignmentEnum](../TitleBlockAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [TitleBlockAlignmentEnum](../TitleBlockAlignmentEnum.md).

## Version

Introduced in version 2008
