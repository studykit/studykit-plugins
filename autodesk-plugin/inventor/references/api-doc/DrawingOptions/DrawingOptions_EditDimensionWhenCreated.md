# DrawingOptions.EditDimensionWhenCreated Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets whether the edit dimension dialog should be shown when a dimension is created using the 'General Dimension' command.

## Syntax

DrawingOptions.**EditDimensionWhenCreated**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
