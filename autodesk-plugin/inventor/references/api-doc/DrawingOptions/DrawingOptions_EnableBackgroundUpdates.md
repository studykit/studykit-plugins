# DrawingOptions.EnableBackgroundUpdates Property

Parent Object: [DrawingOptions](../DrawingOptions/DrawingOptions.md)

## Description

Gets/Sets whether drawing views are computed as a background process or not.

## Syntax

DrawingOptions.**EnableBackgroundUpdates**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
