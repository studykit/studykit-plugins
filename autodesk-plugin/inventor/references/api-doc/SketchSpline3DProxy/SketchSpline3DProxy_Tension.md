# SketchSpline3DProxy.Tension Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Gets and sets the tension for the spline.

## Syntax

SketchSpline3DProxy.**Tension**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 8
