# SketchSpline3DProxy.NativeObject Property

Parent Object: [SketchSpline3DProxy](../SketchSpline3DProxy/SketchSpline3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SketchSpline3DProxy.**NativeObject**() As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md).

## Version

Introduced in version 6
