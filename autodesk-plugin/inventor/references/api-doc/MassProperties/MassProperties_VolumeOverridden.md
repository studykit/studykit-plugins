# MassProperties.VolumeOverridden Property

Parent Object: [MassProperties](../MassProperties/MassProperties.md)

## Description

Gets and sets whether the volume value is overridden.

## Syntax

MassProperties.**VolumeOverridden**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
