# ClientFeatureElements.Count Property

Parent Object: [ClientFeatureElements](../ClientFeatureElements/ClientFeatureElements.md)

## Description

Property that specifies the number of items in the collection.

## Syntax

ClientFeatureElements.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2008
