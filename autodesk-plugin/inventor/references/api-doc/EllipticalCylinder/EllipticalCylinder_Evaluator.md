# EllipticalCylinder.Evaluator Property

Parent Object: [EllipticalCylinder](../EllipticalCylinder/EllipticalCylinder.md)

## Description

Gets the surface evaluator for this elliptical cylinder.

## Syntax

EllipticalCylinder.**Evaluator**() As [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md)

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](../SurfaceEvaluator/SurfaceEvaluator.md).

## Version

Introduced in version 4
