# NotebookOptions.KeepNotesOnDeletedObjects Property

Parent Object: [NotebookOptions](../NotebookOptions/NotebookOptions.md)

## Description

Gets/Sets whether design notes associated with model geometry should be retained after the geometry has been deleted.

## Syntax

NotebookOptions.**KeepNotesOnDeletedObjects**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
