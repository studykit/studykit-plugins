# NotebookOptions.TextBackgroundColor Property

Parent Object: [NotebookOptions](../NotebookOptions/NotebookOptions.md)

## Description

Gets/Sets the background color for the comment boxes in design notes.

## Syntax

NotebookOptions.**TextBackgroundColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
