# NotebookOptions.NoteHighlightColor Property

Parent Object: [NotebookOptions](../NotebookOptions/NotebookOptions.md)

## Description

Gets/Sets the color for the highlighted component in note views.

## Syntax

NotebookOptions.**NoteHighlightColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
