# FinishDefinition.Depth Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that provides access to the depth of the feature.

## Syntax

FinishDefinition.**Depth**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2024
