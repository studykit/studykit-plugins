# FinishDefinition.ProcessName Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that gets and sets the process name for the finish feature.

## Syntax

FinishDefinition.**ProcessName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2024
