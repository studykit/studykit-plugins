# FinishDefinition.IncludedEntities Property

Parent Object: [FinishDefinition](../FinishDefinition/FinishDefinition.md)

## Description

Read-write property that gets and sets the Face and SurfaceBody objects and their proxies on which the finish feature is created. Valid objects in the ObjectCollection can be Face, SurfaceBody and the proxies, or FaceCollection that contains Face or FaceProxy.

## Syntax

FinishDefinition.**IncludedEntities**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2025.1
