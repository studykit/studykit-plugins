# iPartFactory.TableRows Property

Parent Object: [iPartFactory](../iPartFactory/iPartFactory.md)

## Description

Property that returns the iPartTableRows collection object.

## Syntax

iPartFactory.**TableRows**() As [iPartTableRows](../iPartTableRows/iPartTableRows.md)

## Property Value

This is a read only property whose value is an [iPartTableRows](../iPartTableRows/iPartTableRows.md).
