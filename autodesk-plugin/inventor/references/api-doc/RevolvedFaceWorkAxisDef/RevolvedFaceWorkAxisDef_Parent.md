# RevolvedFaceWorkAxisDef.Parent Property

Parent Object: [RevolvedFaceWorkAxisDef](../RevolvedFaceWorkAxisDef/RevolvedFaceWorkAxisDef.md)

## Description

Property returning the parent WorkAxis object.

## Syntax

RevolvedFaceWorkAxisDef.**Parent**() As [WorkAxis](../WorkAxis/WorkAxis.md)

## Property Value

This is a read only property whose value is a [WorkAxis](../WorkAxis/WorkAxis.md).

## Version

Introduced in version 4
