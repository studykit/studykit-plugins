# RevolvedFaceWorkAxisDef.Face Property

Parent Object: [RevolvedFaceWorkAxisDef](../RevolvedFaceWorkAxisDef/RevolvedFaceWorkAxisDef.md)

## Description

Property that returns the used to define the revolved face work axis. This object must be a Face object whose geometry is a revolved surface. Valid surface types include cylinders, cones, and toroids.

## Syntax

RevolvedFaceWorkAxisDef.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read only property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 4
