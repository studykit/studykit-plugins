# OnFaceCurveProxy.BreakLink Method

Parent Object: [OnFaceCurveProxy](../OnFaceCurveProxy/OnFaceCurveProxy.md)

## Description

Method that breaks the link between the intersection curve and the model.

## Syntax

OnFaceCurveProxy.**BreakLink**()

## Version

Introduced in version 2017
