# StylesManager Object

## Description

The StylesManager object provides utilities to manage the various styles in a document.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [GetStyleNameFromKey](../StylesManager/StylesManager_GetStyleNameFromKey.md) | Method that extracts and returns the internal and display name of a style from the input reference key of the style. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../StylesManager/StylesManager_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../StylesManager/StylesManager_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [Type](../StylesManager/StylesManager_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Application.StylesManager](../Application/Application_StylesManager.md), [InventorServer.StylesManager](InventorServer_StylesManager.md), [InventorServerObject.StylesManager](InventorServerObject_StylesManager.md)
