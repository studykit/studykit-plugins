# CommandControlsEnumerator Object

## Description

The CommandControlsEnumerator object provides access to a list of CommandControl objects.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../CommandControlsEnumerator/CommandControlsEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../CommandControlsEnumerator/CommandControlsEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../CommandControlsEnumerator/CommandControlsEnumerator_Item.md) | Property that returns an item from the collection. The index of the first item in the collection is 1. |
| [Type](../CommandControlsEnumerator/CommandControlsEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[UserInterfaceManager.AllReferencedControls](../UserInterfaceManager/UserInterfaceManager_AllReferencedControls.md)

## Version

Introduced in version 2010
