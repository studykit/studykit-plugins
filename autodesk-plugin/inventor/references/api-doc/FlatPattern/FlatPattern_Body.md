# FlatPattern.Body Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the corresponding SurfaceBody.

## Syntax

FlatPattern.**Body**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read only property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 6
