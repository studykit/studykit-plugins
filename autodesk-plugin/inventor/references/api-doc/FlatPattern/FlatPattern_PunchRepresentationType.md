# FlatPattern.PunchRepresentationType Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Gets and sets the punch representation to use for the flat pattern.

## Syntax

FlatPattern.**PunchRepresentationType**() As [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PunchRepresentationTypeEnum](../PunchRepresentationTypeEnum.md).

## Version

Introduced in version 2008
