# FlatPattern.Features Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the collection object that provides access to the features associated with the flat pattern.

## Syntax

FlatPattern.**Features**() As [FlatPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures.md)

## Property Value

This is a read only property whose value is a [FlatPatternFeatures](../FlatPatternFeatures/FlatPatternFeatures.md).
