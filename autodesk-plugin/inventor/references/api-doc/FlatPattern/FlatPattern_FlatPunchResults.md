# FlatPattern.FlatPunchResults Property

Parent Object: [FlatPattern](../FlatPattern/FlatPattern.md)

## Description

Property that returns the FlatPunchResults collection object that contains information about all the punches in the flat pattern.

## Syntax

FlatPattern.**FlatPunchResults**() As [FlatPunchResults](../FlatPunchResults/FlatPunchResults.md)

## Property Value

This is a read only property whose value is a [FlatPunchResults](../FlatPunchResults/FlatPunchResults.md).

## Version

Introduced in version 2008
