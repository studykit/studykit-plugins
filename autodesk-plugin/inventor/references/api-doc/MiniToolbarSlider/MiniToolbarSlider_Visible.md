# MiniToolbarSlider.Visible Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets whether the control is visible.

## Syntax

MiniToolbarSlider.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2013
