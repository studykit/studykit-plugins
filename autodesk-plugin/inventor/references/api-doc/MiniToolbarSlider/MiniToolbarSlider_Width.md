# MiniToolbarSlider.Width Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets the width of the slider in pixels.

## Syntax

MiniToolbarSlider.**Width**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2013
