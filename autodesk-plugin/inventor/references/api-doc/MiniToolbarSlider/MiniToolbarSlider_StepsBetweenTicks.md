# MiniToolbarSlider.StepsBetweenTicks Property

Parent Object: [MiniToolbarSlider](../MiniToolbarSlider/MiniToolbarSlider.md)

## Description

Read-write property that gets and sets the number of steps the slider will move to go from one tick to the next. This controls the resolution and the corresponding values that can be returned by the slider.

## Syntax

MiniToolbarSlider.**StepsBetweenTicks**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2013
