# SheetMetalFeatures.ShellFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the ShellFeatures collection object. This collection provides access to existing ShellFeature objects.

## Syntax

SheetMetalFeatures.**ShellFeatures**() As [ShellFeatures](../ShellFeatures/ShellFeatures.md)

## Property Value

This is a read only property whose value is a [ShellFeatures](../ShellFeatures/ShellFeatures.md).

## Version

Introduced in version 5.3
