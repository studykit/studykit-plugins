# SheetMetalFeatures.ClientFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property returning the client features collection object.

## Syntax

SheetMetalFeatures.**ClientFeatures**() As [ClientFeatures](../ClientFeatures/ClientFeatures.md)

## Property Value

This is a read only property whose value is a [ClientFeatures](../ClientFeatures/ClientFeatures.md).

## Version

Introduced in version 2008
