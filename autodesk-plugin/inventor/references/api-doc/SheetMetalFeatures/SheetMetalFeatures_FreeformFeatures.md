# SheetMetalFeatures.FreeformFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that besides listing out the subset of part features that are Freeform features, allows the creations of new Freeform features.

## Syntax

SheetMetalFeatures.**FreeformFeatures**() As [FreeformFeatures](../FreeformFeatures/FreeformFeatures.md)

## Property Value

This is a read only property whose value is a [FreeformFeatures](../FreeformFeatures/FreeformFeatures.md).

## Version

Introduced in version 2015
