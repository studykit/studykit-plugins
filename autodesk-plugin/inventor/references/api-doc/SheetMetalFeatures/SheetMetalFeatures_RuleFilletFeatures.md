# SheetMetalFeatures.RuleFilletFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are RuleFilletFeatures.

## Syntax

SheetMetalFeatures.**RuleFilletFeatures**() As [RuleFilletFeatures](../RuleFilletFeatures/RuleFilletFeatures.md)

## Property Value

This is a read only property whose value is a [RuleFilletFeatures](../RuleFilletFeatures/RuleFilletFeatures.md).

## Version

Introduced in version 2010
