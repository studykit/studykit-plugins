# SheetMetalFeatures.ContourFlangeFeatures Property

Parent Object: [SheetMetalFeatures](../SheetMetalFeatures/SheetMetalFeatures.md)

## Description

Property that returns the CounterFlangeFeatures collection object.

## Syntax

SheetMetalFeatures.**ContourFlangeFeatures**() As [ContourFlangeFeatures](../ContourFlangeFeatures/ContourFlangeFeatures.md)

## Property Value

This is a read only property whose value is a [ContourFlangeFeatures](../ContourFlangeFeatures/ContourFlangeFeatures.md).

## Version

Introduced in version 5.3
