# PathEntity.Parent Property

Parent Object: [PathEntity](../PathEntity/PathEntity.md)

## Description

Property that returns the parent Path of the entity.

## Syntax

PathEntity.**Parent**() As [Path](../Path/Path.md)

## Property Value

This is a read only property whose value is a [Path](../Path/Path.md).

## Version

Introduced in version 6
