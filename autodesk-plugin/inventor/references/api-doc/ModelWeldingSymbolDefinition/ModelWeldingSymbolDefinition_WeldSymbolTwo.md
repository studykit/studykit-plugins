# ModelWeldingSymbolDefinition.WeldSymbolTwo Property

Parent Object: [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Description

Returns the WeldSymbol object that is below the reference line.

## Syntax

ModelWeldingSymbolDefinition.**WeldSymbolTwo**() As [WeldSymbol](../WeldSymbol/WeldSymbol.md)

## Property Value

This is a read only property whose value is a [WeldSymbol](../WeldSymbol/WeldSymbol.md).

## Version

Introduced in version 2024
