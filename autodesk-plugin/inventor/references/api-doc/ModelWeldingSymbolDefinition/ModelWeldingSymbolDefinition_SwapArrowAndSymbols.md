# ModelWeldingSymbolDefinition.SwapArrowAndSymbols Property

Parent Object: [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Description

Gets and sets whether to switch the arrow and symbols above or below the reference line.

## Syntax

ModelWeldingSymbolDefinition.**SwapArrowAndSymbols**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
