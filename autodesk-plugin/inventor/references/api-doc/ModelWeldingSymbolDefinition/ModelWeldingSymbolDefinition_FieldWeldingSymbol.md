# ModelWeldingSymbolDefinition.FieldWeldingSymbol Property

Parent Object: [ModelWeldingSymbolDefinition](../ModelWeldingSymbolDefinition/ModelWeldingSymbolDefinition.md)

## Description

Gets and sets whether to add a flag indicating a field or site weld to the selected reference line or not.

## Syntax

ModelWeldingSymbolDefinition.**FieldWeldingSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2024
