# TextStyle.Color Property

Parent Object: [TextStyle](../TextStyle/TextStyle.md)

## Description

Gets and sets the current color of the text.

## Syntax

TextStyle.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 11
