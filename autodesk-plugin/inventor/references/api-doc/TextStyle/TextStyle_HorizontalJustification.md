# TextStyle.HorizontalJustification Property

Parent Object: [TextStyle](../TextStyle/TextStyle.md)

## Description

Gets and sets the horizontal justification of the text style.

## Syntax

TextStyle.**HorizontalJustification**() As [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md)

## Property Value

This is a read/write property whose value is a [HorizontalTextAlignmentEnum](../HorizontalTextAlignmentEnum.md).

## Version

Introduced in version 5.3
