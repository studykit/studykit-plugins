# RevisionTableStyle.InUse Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Property that indicates if this style is in use.

## Syntax

RevisionTableStyle.**InUse**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2011
