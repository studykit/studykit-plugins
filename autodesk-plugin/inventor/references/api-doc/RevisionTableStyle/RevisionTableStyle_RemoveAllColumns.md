# RevisionTableStyle.RemoveAllColumns Method

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Method that removes all columns from the style.

## Syntax

RevisionTableStyle.**RemoveAllColumns**()

## Version

Introduced in version 2011
