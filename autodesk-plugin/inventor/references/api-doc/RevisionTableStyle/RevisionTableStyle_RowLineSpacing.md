# RevisionTableStyle.RowLineSpacing Property

Parent Object: [RevisionTableStyle](../RevisionTableStyle/RevisionTableStyle.md)

## Description

Gets and sets the spacing between the rows.

## Syntax

RevisionTableStyle.**RowLineSpacing**() As [LineSpacingEnum](../LineSpacingEnum.md)

## Property Value

This is a read/write property whose value is a [LineSpacingEnum](../LineSpacingEnum.md).

## Version

Introduced in version 2011
