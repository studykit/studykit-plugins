# EdgeSymbolStyle.SaveToGlobal Method

Parent Object: [EdgeSymbolStyle](../EdgeSymbolStyle/EdgeSymbolStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

EdgeSymbolStyle.**SaveToGlobal**()

## Version

Introduced in version 2024
