# RibFeatures.Item Property

Parent Object: [RibFeatures](../RibFeatures/RibFeatures.md)

## Description

Returns the specified object in the collection. The index can be numeric or the object name.

## Syntax

RibFeatures.**Item**( ***Index*** As Variant ) As [RibFeature](../RibFeature/RibFeature.md)

## Property Value

This is a read only property whose value is a [RibFeature](../RibFeature/RibFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the RibFeature to return. |

## Version

Introduced in version 5
