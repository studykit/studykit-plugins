# Camera.Target Property

Parent Object: [Camera](../Camera/Camera.md)

## Description

Specifies the position of the Target point the observer's viewing in the scene (View's Z-axis).

## Syntax

Camera.**Target**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).
