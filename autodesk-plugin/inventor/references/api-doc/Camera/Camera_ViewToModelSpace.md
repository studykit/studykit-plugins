# Camera.ViewToModelSpace Method

Parent Object: [Camera](../Camera/Camera.md)

## Description

Method that converts a point in view space to the equivalent point in the model.

## Syntax

Camera.**ViewToModelSpace**( ***ViewCoordinate*** As [Point2d](../Point2d/Point2d.md) ) As [Point](../Point/Point.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ViewCoordinate | [Point2d](../Point2d/Point2d.md) | Input view point to convert to model space. |

## Version

Introduced in version 6
