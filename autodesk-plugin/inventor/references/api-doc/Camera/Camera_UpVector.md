# Camera.UpVector Property

Parent Object: [Camera](../Camera/Camera.md)

## Description

Specifies the vector defining what is 'up' for the observer.

## Syntax

Camera.**UpVector**() As [UnitVector](../UnitVector/UnitVector.md)

## Property Value

This is a read/write property whose value is a [UnitVector](../UnitVector/UnitVector.md).
