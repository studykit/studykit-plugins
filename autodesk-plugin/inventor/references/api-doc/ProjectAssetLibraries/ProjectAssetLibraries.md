# ProjectAssetLibraries Object

## Description

The ProjectAssetLibraries collection object provides access to the list of material and appearance libraries associated with a project. Through the collection object you can add additional materials or appearances and access the specific library references.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ProjectAssetLibraries/ProjectAssetLibraries_Add.md) | Method that adds a reference to a material library to the project. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ProjectAssetLibraries/ProjectAssetLibraries_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ProjectAssetLibraries/ProjectAssetLibraries_Count.md) | Gets the number of items in this collection. |
| [Item](../ProjectAssetLibraries/ProjectAssetLibraries_Item.md) | Returns the specified ProjectAssetLibrary object from the collection. |
| [Type](../ProjectAssetLibraries/ProjectAssetLibraries_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DesignProject.AppearanceLibraries](../DesignProject/DesignProject_AppearanceLibraries.md), [DesignProject.MaterialLibraries](../DesignProject/DesignProject_MaterialLibraries.md)

## Version

Introduced in version 2014
