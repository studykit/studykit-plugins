# DiameterDimConstraint.Entity Property

Parent Object: [DiameterDimConstraint](../DiameterDimConstraint/DiameterDimConstraint.md)

## Description

Property that returns the circle or arc being constrained.

## Syntax

DiameterDimConstraint.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 5
