# DerivedPartCoordinateSystemDef.MirrorPlane Property

Parent Object: [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Description

Sets the plane about which the base part will be mirrored to create the derived part.

## Syntax

DerivedPartCoordinateSystemDef.**MirrorPlane**() As [DerivedPartMirrorPlaneEnum](../DerivedPartMirrorPlaneEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedPartMirrorPlaneEnum](../DerivedPartMirrorPlaneEnum.md).

## Version

Introduced in version 6
