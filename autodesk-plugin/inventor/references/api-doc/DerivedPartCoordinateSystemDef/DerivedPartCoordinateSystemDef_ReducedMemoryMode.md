# DerivedPartCoordinateSystemDef.ReducedMemoryMode Property

Parent Object: [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Description

Read-write property that gets and sets the Reduced Memory Mode.

## Syntax

DerivedPartCoordinateSystemDef.**ReducedMemoryMode**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2012
