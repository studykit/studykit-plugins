# DerivedPartCoordinateSystemDef.LinkSheetMetalStyles Property

Parent Object: [DerivedPartCoordinateSystemDef](../DerivedPartCoordinateSystemDef/DerivedPartCoordinateSystemDef.md)

## Description

Read-write Boolean property that specifies whether to link sheet metal styles or not. This is applicable only when derive another sheet metal document to current sheet metal document.

## Syntax

DerivedPartCoordinateSystemDef.**LinkSheetMetalStyles**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025.1
