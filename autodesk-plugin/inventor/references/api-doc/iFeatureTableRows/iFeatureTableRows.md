# iFeatureTableRows Object

## Description

The iFeatureTableRows collection object provides access to the rows of an iFeature table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../iFeatureTableRows/iFeatureTableRows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../iFeatureTableRows/iFeatureTableRows_Count.md) | Property that specifies the number of items in the collection. |
| [Item](../iFeatureTableRows/iFeatureTableRows_Item.md) | Method that returns the specified iFeatureTableRow object from the collection. |
| [Type](../iFeatureTableRows/iFeatureTableRows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[iFeatureTable.iFeatureTableRows](../iFeatureTable/iFeatureTable_iFeatureTableRows.md)

## Version

Introduced in version 2009
