# CoincidentConstraint3DProxy.ConnectedEntity Property

Parent Object: [CoincidentConstraint3DProxy](../CoincidentConstraint3DProxy/CoincidentConstraint3DProxy.md)

## Description

Property that returns the entity connected to this constraint. This can be a SketchPoint, Vertex or a WorkPoint. This property returns Nothing in the case where there isn't a connected entity.

## Syntax

CoincidentConstraint3DProxy.**ConnectedEntity**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
