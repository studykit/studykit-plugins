# EmbossFeature.Profile Property

Parent Object: [EmbossFeature](../EmbossFeature/EmbossFeature.md)

## Description

Gets and sets the Profile object that defines the shape of the emboss.

## Syntax

EmbossFeature.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read/write property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 2008
