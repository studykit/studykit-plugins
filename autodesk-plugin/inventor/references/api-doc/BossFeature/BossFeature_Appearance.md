# BossFeature.Appearance Property

Parent Object: [BossFeature](../BossFeature/BossFeature.md)

## Description

Gets and sets the current appearance of the feature.

## Syntax

BossFeature.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read/write property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2014
