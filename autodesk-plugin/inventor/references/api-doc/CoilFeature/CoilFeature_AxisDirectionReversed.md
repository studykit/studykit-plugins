# CoilFeature.AxisDirectionReversed Property

Parent Object: [CoilFeature](../CoilFeature/CoilFeature.md)

## Description

Property that gets and sets whether the axis direction of the coil is reversed.

## Syntax

CoilFeature.**AxisDirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5.3
