# MirrorFeatureDefinition.ComputeType Property

Parent Object: [MirrorFeatureDefinition](../MirrorFeatureDefinition/MirrorFeatureDefinition.md)

## Description

Read-write property that that indicates the method of solution for the pattern.

## Syntax

MirrorFeatureDefinition.**ComputeType**() As [PatternComputeTypeEnum](../PatternComputeTypeEnum.md)

## Property Value

This is a read/write property whose value is a [PatternComputeTypeEnum](../PatternComputeTypeEnum.md).

## Version

Introduced in version 2017
