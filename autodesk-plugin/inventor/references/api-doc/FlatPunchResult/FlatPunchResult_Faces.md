# FlatPunchResult.Faces Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns the faces in the flat pattern that belong to the punch result. This property returns Nothing if punch representation is not formed feature representation.

## Syntax

FlatPunchResult.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2008
