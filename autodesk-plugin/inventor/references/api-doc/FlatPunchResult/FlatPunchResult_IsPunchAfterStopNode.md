# FlatPunchResult.IsPunchAfterStopNode Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns whether the punch feature that resulted in this punch result is currently after the stop node.

## Syntax

FlatPunchResult.**IsPunchAfterStopNode**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
