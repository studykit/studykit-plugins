# FlatPunchResult.Angle Property

Parent Object: [FlatPunchResult](../FlatPunchResult/FlatPunchResult.md)

## Description

Property that returns an angle value indicating the punch orientation. This property may return an error if this information is not available.

## Syntax

FlatPunchResult.**Angle**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
