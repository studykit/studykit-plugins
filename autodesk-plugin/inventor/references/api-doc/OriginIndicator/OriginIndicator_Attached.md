# OriginIndicator.Attached Property

Parent Object: [OriginIndicator](../OriginIndicator/OriginIndicator.md)

## Description

Property that specifies whether the origin indicator is attached to an origin point.

## Syntax

OriginIndicator.**Attached**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2009
