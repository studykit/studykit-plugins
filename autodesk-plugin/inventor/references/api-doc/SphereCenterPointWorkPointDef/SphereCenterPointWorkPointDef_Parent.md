# SphereCenterPointWorkPointDef.Parent Property

Parent Object: [SphereCenterPointWorkPointDef](../SphereCenterPointWorkPointDef/SphereCenterPointWorkPointDef.md)

## Description

Read-only property that returns the parent WorkPoint object.

## Syntax

SphereCenterPointWorkPointDef.**Parent**() As [WorkPoint](../WorkPoint/WorkPoint.md)

## Property Value

This is a read only property whose value is a [WorkPoint](../WorkPoint/WorkPoint.md).

## Version

Introduced in version 2014
