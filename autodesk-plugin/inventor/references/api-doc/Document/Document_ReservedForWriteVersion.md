# Document.ReservedForWriteVersion Property

Parent Object: [Document](../Document/Document.md)

## Description

Gets the version within this file that has been reserved for write.

## Syntax

Document.**ReservedForWriteVersion**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 4
