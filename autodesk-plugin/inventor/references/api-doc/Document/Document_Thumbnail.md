# Document.Thumbnail Property

Parent Object: [Document](../Document/Document.md)

## Description

Property that returns a bitmap picture of the document.

## Syntax

Document.**Thumbnail**() As IPictureDisp

## Property Value

This is a read only property whose value is an IPictureDisp.

## Version

Introduced in version 6
