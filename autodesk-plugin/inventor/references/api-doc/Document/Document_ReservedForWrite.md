# Document.ReservedForWrite Property

Parent Object: [Document](../Document/Document.md)

## Description

Gets the Boolean flag indicating whether this file has been reserved for write by someone.

## Syntax

Document.**ReservedForWrite**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 4
