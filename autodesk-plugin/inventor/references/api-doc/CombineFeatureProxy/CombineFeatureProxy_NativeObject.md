# CombineFeatureProxy.NativeObject Property

Parent Object: [CombineFeatureProxy](../CombineFeatureProxy/CombineFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

CombineFeatureProxy.**NativeObject**() As [CombineFeature](../CombineFeature/CombineFeature.md)

## Property Value

This is a read only property whose value is a [CombineFeature](../CombineFeature/CombineFeature.md).

## Version

Introduced in version 2010
