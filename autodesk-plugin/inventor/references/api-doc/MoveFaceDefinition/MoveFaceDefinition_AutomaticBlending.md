# MoveFaceDefinition.AutomaticBlending Property

Parent Object: [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md)

## Description

Gets and sets whether the move operation should propagate to adjacent tangent faces.

## Syntax

MoveFaceDefinition.**AutomaticBlending**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
