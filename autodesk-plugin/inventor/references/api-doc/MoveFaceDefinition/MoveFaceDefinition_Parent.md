# MoveFaceDefinition.Parent Property

Parent Object: [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md)

## Description

Property that returns the parent MoveFaceFeature of this MoveFaceDefinition object.

## Syntax

MoveFaceDefinition.**Parent**() As [MoveFaceFeature](../MoveFaceFeature/MoveFaceFeature.md)

## Property Value

This is a read only property whose value is a [MoveFaceFeature](../MoveFaceFeature/MoveFaceFeature.md).

## Version

Introduced in version 2011
