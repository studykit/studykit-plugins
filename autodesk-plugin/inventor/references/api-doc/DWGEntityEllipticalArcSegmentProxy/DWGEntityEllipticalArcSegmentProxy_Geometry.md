# DWGEntityEllipticalArcSegmentProxy.Geometry Property

Parent Object: [DWGEntityEllipticalArcSegmentProxy](../DWGEntityEllipticalArcSegmentProxy/DWGEntityEllipticalArcSegmentProxy.md)

## Description

Read-only property that returns the segment geometry.

## Syntax

DWGEntityEllipticalArcSegmentProxy.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2018
