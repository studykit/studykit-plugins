# FaceProxy.SurfaceType Property

Parent Object: [FaceProxy](../FaceProxy/FaceProxy.md)

## Description

Property that returns a SurfaceTypeEnum that specifies the surface type for this Face.

## Syntax

FaceProxy.**SurfaceType**() As [SurfaceTypeEnum](../SurfaceTypeEnum.md)

## Property Value

This is a read only property whose value is a [SurfaceTypeEnum](../SurfaceTypeEnum.md).

## Version

Introduced in version 4
