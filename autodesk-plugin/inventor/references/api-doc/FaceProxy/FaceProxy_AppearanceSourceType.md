# FaceProxy.AppearanceSourceType Property

Parent Object: [FaceProxy](../FaceProxy/FaceProxy.md)

## Description

Gets and sets the source of the appearance for the face.

## Syntax

FaceProxy.**AppearanceSourceType**() As [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AppearanceSourceTypeEnum](../AppearanceSourceTypeEnum.md).

## Version

Introduced in version 2014
