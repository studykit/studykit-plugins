# FeatureBasedOccurrencePattern.Visible Property

Parent Object: [FeatureBasedOccurrencePattern](../FeatureBasedOccurrencePattern/FeatureBasedOccurrencePattern.md)

## Description

Gets/Sets the Visible property for the pattern.

## Syntax

FeatureBasedOccurrencePattern.**Visible**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
