# SketchCircularPatternDefinition.Copy Method

Parent Object: [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Description

Creates a copy of this SketchCircularPatternDefinition object. The new SketchCircularPatternDefinition object is independent of any pattern.

## Syntax

SketchCircularPatternDefinition.**Copy**() As [SketchCircularPatternDefinition](../SketchCircularPatternDefinition/SketchCircularPatternDefinition.md)

## Version

Introduced in version 2025.1
