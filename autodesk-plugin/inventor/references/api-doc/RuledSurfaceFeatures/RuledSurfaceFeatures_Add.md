# RuledSurfaceFeatures.Add Method

Parent Object: [RuledSurfaceFeatures](../RuledSurfaceFeatures/RuledSurfaceFeatures.md)

## Description

Method that creates a new RuledSurfaceFeature Object.

## Syntax

RuledSurfaceFeatures.**Add**( ***Definition*** As [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md) ) As [RuledSurfaceFeature](../RuledSurfaceFeature/RuledSurfaceFeature.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Definition | [RuledSurfaceDefinition](../RuledSurfaceDefinition/RuledSurfaceDefinition.md) | Input RuledSurfaceDefinition object that defines the ruled surface feature you want to create. An RuledSurfaceDefinition object can be created using the RuledSurfaceFeatures.CreateDefinition method. It can also be obtained from an existing RuledSurfaceFeature object. |

## Version

Introduced in version 2016
