# NonLinearEdgeWorkPointDef Object

## Description

Object that allows you to get and set the information that specifies a work point using a non-linear edge.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../NonLinearEdgeWorkPointDef/NonLinearEdgeWorkPointDef_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Edge](../NonLinearEdgeWorkPointDef/NonLinearEdgeWorkPointDef_Edge.md) | Property that returns a non-linear Edge object. |
| [Parent](../NonLinearEdgeWorkPointDef/NonLinearEdgeWorkPointDef_Parent.md) | Property that returns the parent WorkPoint object. |
| [Type](../NonLinearEdgeWorkPointDef/NonLinearEdgeWorkPointDef_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 11
