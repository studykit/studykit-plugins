# NonLinearEdgeWorkPointDef.Edge Property

Parent Object: [NonLinearEdgeWorkPointDef](../NonLinearEdgeWorkPointDef/NonLinearEdgeWorkPointDef.md)

## Description

Property that returns a non-linear Edge object.

## Syntax

NonLinearEdgeWorkPointDef.**Edge**() As [Edge](../Edge/Edge.md)

## Property Value

This is a read only property whose value is an [Edge](../Edge/Edge.md).

## Version

Introduced in version 11
