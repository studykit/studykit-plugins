# TangentConstraint3DProxy.NativeObject Property

Parent Object: [TangentConstraint3DProxy](../TangentConstraint3DProxy/TangentConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

TangentConstraint3DProxy.**NativeObject**() As [TangentConstraint3D](../TangentConstraint3D/TangentConstraint3D.md)

## Property Value

This is a read only property whose value is a [TangentConstraint3D](../TangentConstraint3D/TangentConstraint3D.md).

## Version

Introduced in version 8
