# DerivedParameterTables.Item Property

Parent Object: [DerivedParameterTables](../DerivedParameterTables/DerivedParameterTables.md)

## Description

Method that returns the specified DerivedParameterTable object from the collection. This is the default method of the DerivedParameterTables collection object.

## Syntax

DerivedParameterTables.**Item**( ***Value*** As Long ) As [DerivedParameterTable](../DerivedParameterTable/DerivedParameterTable.md)

## Property Value

This is a read only property whose value is a [DerivedParameterTable](../DerivedParameterTable/DerivedParameterTable.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Value | Long | Value that specifies the object to return. |

## Version

Introduced in version 11
