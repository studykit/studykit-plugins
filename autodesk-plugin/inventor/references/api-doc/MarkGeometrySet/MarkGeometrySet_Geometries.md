# MarkGeometrySet.Geometries Property

Parent Object: [MarkGeometrySet](../MarkGeometrySet/MarkGeometrySet.md)

## Description

Property that gets and sets the geometries for the mark geometry set.

## Syntax

MarkGeometrySet.**Geometries**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2023
