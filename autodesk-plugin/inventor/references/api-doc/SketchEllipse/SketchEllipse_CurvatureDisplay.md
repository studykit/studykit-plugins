# SketchEllipse.CurvatureDisplay Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Specifies whether curvature information is displayed for the ellipse.

## Syntax

SketchEllipse.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
