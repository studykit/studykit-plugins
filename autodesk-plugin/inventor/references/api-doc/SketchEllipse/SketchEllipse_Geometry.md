# SketchEllipse.Geometry Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Property that returns an EllipseFull2d geometry object. The object returned represents a snapshot view of the current state of the sketch ellipse.

## Syntax

SketchEllipse.**Geometry**() As [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md)

## Property Value

This is a read only property whose value is an [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md).

## Version

Introduced in version 5
