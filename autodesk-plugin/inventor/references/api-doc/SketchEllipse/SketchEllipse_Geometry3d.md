# SketchEllipse.Geometry3d Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Read-only property that returns ellipse geometry that represents this ellipse in model space.

## Syntax

SketchEllipse.**Geometry3d**() As [EllipseFull](../EllipseFull/EllipseFull.md)

## Property Value

This is a read only property whose value is an [EllipseFull](../EllipseFull/EllipseFull.md).

## Version

Introduced in version 2013
