# SketchEllipse.MajorRadius Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Gets and sets the major radius.

## Syntax

SketchEllipse.**MajorRadius**() As Double

## Property Value

This is a read/write property whose value is a Double.
