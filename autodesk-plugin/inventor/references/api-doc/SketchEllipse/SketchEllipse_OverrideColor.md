# SketchEllipse.OverrideColor Property

Parent Object: [SketchEllipse](../SketchEllipse/SketchEllipse.md)

## Description

Gets and sets the color applied to this sketch line.

## Syntax

SketchEllipse.**OverrideColor**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 9
