# HoleTag.Position Property

Parent Object: [HoleTag](../HoleTag/HoleTag.md)

## Description

Gets/Sets the position of the HoleTag on the sheet.

## Syntax

HoleTag.**Position**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 10
