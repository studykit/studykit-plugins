# HoleTag.Text Property

Parent Object: [HoleTag](../HoleTag/HoleTag.md)

## Description

Gets/Sets the text of the HoleTag.

## Syntax

HoleTag.**Text**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 10
