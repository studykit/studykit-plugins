# Box.IsDisjoint Method

Parent Object: [Box](../Box/Box.md)

## Description

Determines whether this Box intersects the specified Box. A return value of True indicates that the box do not intersect.

## Syntax

Box.**IsDisjoint**( ***Box*** As [Box](../Box/Box.md) ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Box | [Box](../Box/Box.md) | Input Box object to compare. |

## Version

Introduced in version 4
