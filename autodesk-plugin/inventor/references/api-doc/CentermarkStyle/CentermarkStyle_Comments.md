# CentermarkStyle.Comments Property

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Gets and sets comments associated with the style.

## Syntax

CentermarkStyle.**Comments**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
