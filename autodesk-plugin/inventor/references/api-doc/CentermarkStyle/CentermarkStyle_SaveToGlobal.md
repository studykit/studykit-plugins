# CentermarkStyle.SaveToGlobal Method

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

CentermarkStyle.**SaveToGlobal**()

## Version

Introduced in version 2010
