# CentermarkStyle.Parent Property

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Property returning the parent of this object.

## Syntax

CentermarkStyle.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2010
