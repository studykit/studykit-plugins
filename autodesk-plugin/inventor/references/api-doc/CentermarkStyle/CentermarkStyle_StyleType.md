# CentermarkStyle.StyleType Property

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Gets the type of the style.

## Syntax

CentermarkStyle.**StyleType**() As [StyleTypeEnum](../StyleTypeEnum.md)

## Property Value

This is a read only property whose value is a [StyleTypeEnum](../StyleTypeEnum.md).

## Version

Introduced in version 2010
