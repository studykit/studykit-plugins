# CentermarkStyle.ExtensionLength Property

Parent Object: [CentermarkStyle](../CentermarkStyle/CentermarkStyle.md)

## Description

Gets and sets the minimum length of center mark extension lines.

## Syntax

CentermarkStyle.**ExtensionLength**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
