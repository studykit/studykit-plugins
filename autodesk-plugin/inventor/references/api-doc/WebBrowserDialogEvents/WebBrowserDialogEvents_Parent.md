# WebBrowserDialogEvents.Parent Property

Parent Object: [WebBrowserDialogEvents](../WebBrowserDialogEvents/WebBrowserDialogEvents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

WebBrowserDialogEvents.**Parent**() As [WebBrowserDialog](../WebBrowserDialog/WebBrowserDialog.md)

## Property Value

This is a read only property whose value is a [WebBrowserDialog](../WebBrowserDialog/WebBrowserDialog.md).

## Version

Introduced in version 2017
