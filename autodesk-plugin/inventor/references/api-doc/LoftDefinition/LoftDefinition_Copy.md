# LoftDefinition.Copy Method

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Method that creates and returns a copy of this loft definition.

## Syntax

LoftDefinition.**Copy**() As [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Version

Introduced in version 2008
