# LoftDefinition.FirstSectionImpact Property

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Property that specifies the impact the starting section's condition has on the shape of the entire loft.

## Syntax

LoftDefinition.**FirstSectionImpact**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2008
