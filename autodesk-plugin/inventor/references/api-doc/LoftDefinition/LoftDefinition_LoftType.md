# LoftDefinition.LoftType Property

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Property that gets the type of loft feature.

## Syntax

LoftDefinition.**LoftType**() As [LoftTypeEnum](../LoftTypeEnum.md)

## Property Value

This is a read only property whose value is a [LoftTypeEnum](../LoftTypeEnum.md).

## Version

Introduced in version 2008
