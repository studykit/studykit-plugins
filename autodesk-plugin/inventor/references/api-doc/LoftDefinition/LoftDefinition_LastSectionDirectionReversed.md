# LoftDefinition.LastSectionDirectionReversed Property

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Property that specifies whether the takeoff direction for the ending section should be reversed from its default direction.

## Syntax

LoftDefinition.**LastSectionDirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
