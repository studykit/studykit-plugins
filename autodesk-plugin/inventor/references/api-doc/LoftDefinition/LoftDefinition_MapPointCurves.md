# LoftDefinition.MapPointCurves Property

Parent Object: [LoftDefinition](../LoftDefinition/LoftDefinition.md)

## Description

Property that specifies the mapping to use between sections.

## Syntax

LoftDefinition.**MapPointCurves**() As [MapPointCurves](../MapPointCurves/MapPointCurves.md)

## Property Value

This is a read/write property whose value is a [MapPointCurves](../MapPointCurves/MapPointCurves.md).

## Version

Introduced in version 2008
