# MeshEdgeProxy.Geometry Property

Parent Object: [MeshEdgeProxy](../MeshEdgeProxy/MeshEdgeProxy.md)

## Description

Read-only property that returns the underlying geometry of the MeshEdge.

## Syntax

MeshEdgeProxy.**Geometry**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2017
