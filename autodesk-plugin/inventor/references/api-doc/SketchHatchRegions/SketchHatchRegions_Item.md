# SketchHatchRegions.Item Property

Parent Object: [SketchHatchRegions](../SketchHatchRegions/SketchHatchRegions.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

SketchHatchRegions.**Item**( ***Index*** As Long ) As [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md)

## Property Value

This is a read only property whose value is a [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the item to return. |

## Version

Introduced in version 2021
