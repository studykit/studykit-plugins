# ReplaceFaceFeature.Suppressed Property

Parent Object: [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md)

## Description

Gets and sets whether this feature is suppressed or not.

## Syntax

ReplaceFaceFeature.**Suppressed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
