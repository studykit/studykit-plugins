# ReplaceFaceFeature.Name Property

Parent Object: [ReplaceFaceFeature](../ReplaceFaceFeature/ReplaceFaceFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

ReplaceFaceFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 6
