# MassPropertiesAccuracyEnum Enumerator

## Description

Constants specifying the different computational accuracies of the mass property calculations.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| k\_High | 24579 | High Accuracy. |
| k\_Low | 24577 | Low Accuracy. |
| k\_Medium | 24578 | Medium Accuracy. |
| k\_None | 24581 | No Mass Properties Available. |
| k\_VeryHigh | 24580 | VeryHigh Accuracy. |

## Version

Introduced in version 5
