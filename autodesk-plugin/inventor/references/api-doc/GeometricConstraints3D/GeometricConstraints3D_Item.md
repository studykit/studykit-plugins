# GeometricConstraints3D.Item Property

Parent Object: [GeometricConstraints3D](../GeometricConstraints3D/GeometricConstraints3D.md)

## Description

Returns the specified geometric sketch constraint object from the collection.

## Syntax

GeometricConstraints3D.**Item**( ***Index*** As Long ) As [GeometricConstraint3D](../GeometricConstraint3D/GeometricConstraint3D.md)

## Property Value

This is a read only property whose value is a [GeometricConstraint3D](../GeometricConstraint3D/GeometricConstraint3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
