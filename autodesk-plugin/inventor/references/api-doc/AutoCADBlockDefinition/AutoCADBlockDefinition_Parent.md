# AutoCADBlockDefinition.Parent Property

Parent Object: [AutoCADBlockDefinition](../AutoCADBlockDefinition/AutoCADBlockDefinition.md)

## Description

Property returning the parent DrawingDocument object.

## Syntax

AutoCADBlockDefinition.**Parent**() As [DrawingDocument](../DrawingDocument/DrawingDocument.md)

## Property Value

This is a read only property whose value is a [DrawingDocument](../DrawingDocument/DrawingDocument.md).

## Version

Introduced in version 2011
