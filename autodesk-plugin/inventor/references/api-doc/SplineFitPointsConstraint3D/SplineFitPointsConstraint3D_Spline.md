# SplineFitPointsConstraint3D.Spline Property

Parent Object: [SplineFitPointsConstraint3D](../SplineFitPointsConstraint3D/SplineFitPointsConstraint3D.md)

## Description

Property that returns the spline entity being constrained.

## Syntax

SplineFitPointsConstraint3D.**Spline**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 8
