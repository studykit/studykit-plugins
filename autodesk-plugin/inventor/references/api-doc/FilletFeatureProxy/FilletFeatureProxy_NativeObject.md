# FilletFeatureProxy.NativeObject Property

Parent Object: [FilletFeatureProxy](../FilletFeatureProxy/FilletFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FilletFeatureProxy.**NativeObject**() As [FilletFeature](../FilletFeature/FilletFeature.md)

## Property Value

This is a read only property whose value is a [FilletFeature](../FilletFeature/FilletFeature.md).

## Version

Introduced in version 9
