# BreakOperation.Parent Property

Parent Object: [BreakOperation](../BreakOperation/BreakOperation.md)

## Description

Property that returns the parent drawing view from which this BreakOperation was retrieved.

## Syntax

BreakOperation.**Parent**() As [DrawingView](../DrawingView/DrawingView.md)

## Property Value

This is a read only property whose value is a [DrawingView](../DrawingView/DrawingView.md).

## Version

Introduced in version 2010
