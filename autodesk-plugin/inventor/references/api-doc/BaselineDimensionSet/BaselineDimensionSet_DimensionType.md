# BaselineDimensionSet.DimensionType Property

Parent Object: [BaselineDimensionSet](../BaselineDimensionSet/BaselineDimensionSet.md)

## Description

Property that returns the dimension type of the baseline set. Possible values are kAlignedDimensionType, kHorizontalDimensionType and kVerticalDimensionType.

## Syntax

BaselineDimensionSet.**DimensionType**() As [DimensionTypeEnum](../DimensionTypeEnum.md)

## Property Value

This is a read only property whose value is a [DimensionTypeEnum](../DimensionTypeEnum.md).

## Version

Introduced in version 2010
