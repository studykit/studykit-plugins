# GeneralNote.Color Property

Parent Object: [GeneralNote](../GeneralNote/GeneralNote.md)

## Description

Gets/Sets color of the DrawingNote.

## Syntax

GeneralNote.**Color**() As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

## Version

Introduced in version 10
