# GeneralNote.TextStyle Property

Parent Object: [GeneralNote](../GeneralNote/GeneralNote.md)

## Description

Gets/Sets the TextStyle of the GeneralNote.

## Syntax

GeneralNote.**TextStyle**() As [TextStyle](../TextStyle/TextStyle.md)

## Property Value

This is a read/write property whose value is a [TextStyle](../TextStyle/TextStyle.md).
