# SinglePointRipTypeDef.GapSide Property

Parent Object: [SinglePointRipTypeDef](../SinglePointRipTypeDef/SinglePointRipTypeDef.md)

## Description

Gets and sets the value that indicates the position of the gap.

## Syntax

SinglePointRipTypeDef.**GapSide**() As [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md)

## Property Value

This is a read/write property whose value is a [PartFeatureExtentDirectionEnum](../PartFeatureExtentDirectionEnum.md).

## Version

Introduced in version 2011
