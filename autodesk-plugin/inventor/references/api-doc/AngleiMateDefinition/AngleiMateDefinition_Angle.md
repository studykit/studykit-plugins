# AngleiMateDefinition.Angle Property

Parent Object: [AngleiMateDefinition](../AngleiMateDefinition/AngleiMateDefinition.md)

## Description

Property that returns the Parameter object that controls the angle of the angle iMate definition.

## Syntax

AngleiMateDefinition.**Angle**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 6
