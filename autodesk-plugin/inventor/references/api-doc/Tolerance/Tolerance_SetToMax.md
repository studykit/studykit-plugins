# Tolerance.SetToMax Method

Parent Object: [Tolerance](../Tolerance/Tolerance.md)

## Description

Sets the type of this tolerance to be a MAX type of tolerance.

## Syntax

Tolerance.**SetToMax**()

## Version

Introduced in version 10
