# Tolerance.SetToFits Method

Parent Object: [Tolerance](../Tolerance/Tolerance.md)

## Description

Method that sets the type of this tolerance to be a fits tolerance.

## Syntax

Tolerance.**SetToFits**( ***FitsToleranceType*** As [ToleranceTypeEnum](../ToleranceTypeEnum.md), ***HoleTolerance*** As String, ***ShaftTolerance*** As String )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| FitsToleranceType | [ToleranceTypeEnum](../ToleranceTypeEnum.md) | Input constant that specifies the type of tolerance. |
| HoleTolerance | String | Input String that specifies the tolerance for the hole feature. |
| ShaftTolerance | String | Input String that specifies the tolerance for the shaft. |

## Version

Introduced in version 6
