# SketchControlPointSplineProxy.RangeBox Property

Parent Object: [SketchControlPointSplineProxy](../SketchControlPointSplineProxy/SketchControlPointSplineProxy.md)

## Description

Property that returns a Box2D object which contains the lower-left and upper-right corners of a rectangle that is guaranteed to enclose this object.

## Syntax

SketchControlPointSplineProxy.**RangeBox**() As [Box2d](../Box2d/Box2d.md)

## Property Value

This is a read only property whose value is a [Box2d](../Box2d/Box2d.md).

## Version

Introduced in version 2014
