# MeshFeatureEntitiesEnumerator.Item Property

Parent Object: [MeshFeatureEntitiesEnumerator](../MeshFeatureEntitiesEnumerator/MeshFeatureEntitiesEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

MeshFeatureEntitiesEnumerator.**Item**( ***Index*** As Long ) As [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Property Value

This is a read only property whose value is a [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the MeshFeatureEntity to return. |

## Version

Introduced in version 2017
