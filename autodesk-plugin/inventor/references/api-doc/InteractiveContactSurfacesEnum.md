# InteractiveContactSurfacesEnum Enumerator

## Description

Enum indicating the type of surfaces for interactive contact analysis.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAllSurfacesInteractiveContact | 65025 | Takes into account all surfaces. |
| kGeneralSurfacesInteractiveContact | 65026 | Takes into account most surfaces except for some fillets(blends). |
| kSimpleSurfacesInteractiveContact | 65027 | Ignores non-analytic surfaces. |

## Version

Introduced in version 2008
