# PositionalRepresentation.RemoveRelationshipSuppressionOverride Method

Parent Object: [PositionalRepresentation](../PositionalRepresentation/PositionalRepresentation.md)

## Description

Method that removes the overridden suppression state of the input assembly relationship.

## Syntax

PositionalRepresentation.**RemoveRelationshipSuppressionOverride**( ***Relationship*** As Object )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Relationship | Object | Input assembly constraint or connection object for which the override should be removed. |

## Version

Introduced in version 2014
