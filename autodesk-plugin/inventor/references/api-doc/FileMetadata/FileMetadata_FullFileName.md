# FileMetadata.FullFileName Property

Parent Object: [FileMetadata](../FileMetadata/FileMetadata.md)

## Description

Gets and sets the full file name (with the extension and the path).

## Syntax

FileMetadata.**FullFileName**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
