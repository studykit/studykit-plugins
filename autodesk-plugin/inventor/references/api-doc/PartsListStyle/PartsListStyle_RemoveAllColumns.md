# PartsListStyle.RemoveAllColumns Method

Parent Object: [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Description

Method that removes all columns from the style.

## Syntax

PartsListStyle.**RemoveAllColumns**()

## Version

Introduced in version 2011
