# PartsListStyle.SortSettings Property

Parent Object: [PartsListStyle](../PartsListStyle/PartsListStyle.md)

## Description

Gets the parts list sort settings.

## Syntax

PartsListStyle.**SortSettings**() As [PartsListSortSettings](../PartsListSortSettings/PartsListSortSettings.md)

## Property Value

This is a read only property whose value is a [PartsListSortSettings](../PartsListSortSettings/PartsListSortSettings.md).

## Version

Introduced in version 2025
