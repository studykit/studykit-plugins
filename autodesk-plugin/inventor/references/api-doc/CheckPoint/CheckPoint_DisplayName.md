# CheckPoint.DisplayName Property

Parent Object: [CheckPoint](../CheckPoint/CheckPoint.md)

## Description

Gets the user-friendly display name of this check point.

## Syntax

CheckPoint.**DisplayName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 5
