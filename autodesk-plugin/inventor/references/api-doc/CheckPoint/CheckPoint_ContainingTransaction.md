# CheckPoint.ContainingTransaction Property

Parent Object: [CheckPoint](../CheckPoint/CheckPoint.md)

## Description

Gets the transaction in which this check point is defined.

## Syntax

CheckPoint.**ContainingTransaction**() As [Transaction](../Transaction/Transaction.md)

## Property Value

This is a read only property whose value is a [Transaction](../Transaction/Transaction.md).

## Version

Introduced in version 5
