# ModelFeatureControlFrameDefinition.AnnotationPlaneDefinition Property

Parent Object: [ModelFeatureControlFrameDefinition](../ModelFeatureControlFrameDefinition/ModelFeatureControlFrameDefinition.md)

## Description

Gets and sets the annotation plane definition for the feature control frame.

## Syntax

ModelFeatureControlFrameDefinition.**AnnotationPlaneDefinition**() As [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Property Value

This is a read/write property whose value is an [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md).

## Version

Introduced in version 2018
