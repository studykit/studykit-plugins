# ProfilePathProxy.AddsMaterial Property

Parent Object: [ProfilePathProxy](../ProfilePathProxy/ProfilePathProxy.md)

## Description

Gets or sets a Boolean indicating whether the path adds or removes material from the entire area.

## Syntax

ProfilePathProxy.**AddsMaterial**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
