# ProfilePathProxy.Parent Property

Parent Object: [ProfilePathProxy](../ProfilePathProxy/ProfilePathProxy.md)

## Description

Property that returns the parent Profile.

## Syntax

ProfilePathProxy.**Parent**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read only property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 6
