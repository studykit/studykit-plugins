# PointHolePlacementDefinition.Point Property

Parent Object: [PointHolePlacementDefinition](../PointHolePlacementDefinition/PointHolePlacementDefinition.md)

## Description

Property that indicates the point that defines the hole center. This can only be a WorkPoint.

## Syntax

PointHolePlacementDefinition.**Point**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
