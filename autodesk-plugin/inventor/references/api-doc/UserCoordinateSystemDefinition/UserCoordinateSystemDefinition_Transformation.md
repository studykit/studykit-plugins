# UserCoordinateSystemDefinition.Transformation Property

Parent Object: [UserCoordinateSystemDefinition](../UserCoordinateSystemDefinition/UserCoordinateSystemDefinition.md)

## Description

Gets and puts the transformation matrix.

## Syntax

UserCoordinateSystemDefinition.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2011
