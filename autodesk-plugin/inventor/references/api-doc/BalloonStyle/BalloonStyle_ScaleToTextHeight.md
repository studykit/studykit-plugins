# BalloonStyle.ScaleToTextHeight Property

Parent Object: [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Description

Property that gets and sets whether to define the balloon size by text height. This property is not applicable if the balloon type is kNoneBalloonType or kLinearBalloonType.

## Syntax

BalloonStyle.**ScaleToTextHeight**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
