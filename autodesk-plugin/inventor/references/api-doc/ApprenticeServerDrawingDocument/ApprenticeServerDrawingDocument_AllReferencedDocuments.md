# ApprenticeServerDrawingDocument.AllReferencedDocuments Property

Parent Object: [ApprenticeServerDrawingDocument](../ApprenticeServerDrawingDocument/ApprenticeServerDrawingDocument.md)

## Description

Property that returns all the document references of this document along with all of the recursively nested references.

## Syntax

ApprenticeServerDrawingDocument.**AllReferencedDocuments**() As [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md)

## Property Value

This is a read only property whose value is an [ApprenticeServerDocuments](../ApprenticeServerDocuments/ApprenticeServerDocuments.md).

## Version

Introduced in version 11
