# AssemblyJointProxy.DriveSettings Property

Parent Object: [AssemblyJointProxy](../AssemblyJointProxy/AssemblyJointProxy.md)

## Description

Read-only property that returns the DriveSettings object associated with this joint.

## Syntax

AssemblyJointProxy.**DriveSettings**() As [DriveSettings](../DriveSettings/DriveSettings.md)

## Property Value

This is a read only property whose value is a [DriveSettings](../DriveSettings/DriveSettings.md).

## Version

Introduced in version 2014
