# ParallelToYAxisConstraint3DProxy.Deletable Property

Parent Object: [ParallelToYAxisConstraint3DProxy](../ParallelToYAxisConstraint3DProxy/ParallelToYAxisConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

ParallelToYAxisConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2017
