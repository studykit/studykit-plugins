# ContentTableCell.Column Property

Parent Object: [ContentTableCell](../ContentTableCell/ContentTableCell.md)

## Description

Returns the column index of this cell within the parent ContentFamily.

## Syntax

ContentTableCell.**Column**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2010
