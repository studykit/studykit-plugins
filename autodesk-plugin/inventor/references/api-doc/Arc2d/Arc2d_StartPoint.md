# Arc2d.StartPoint Property

Parent Object: [Arc2d](../Arc2d/Arc2d.md)

## Description

Property that gets the start point of the arc.

## Syntax

Arc2d.**StartPoint**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 5
