# DirectionAndDistanceMoveDefinition.DirectionReversed Property

Parent Object: [DirectionAndDistanceMoveDefinition](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition.md)

## Description

Gets and sets whether the direction defined by the entity in the Direction property should be reversed.

## Syntax

DirectionAndDistanceMoveDefinition.**DirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
