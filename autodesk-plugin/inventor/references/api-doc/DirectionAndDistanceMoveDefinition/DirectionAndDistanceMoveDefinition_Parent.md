# DirectionAndDistanceMoveDefinition.Parent Property

Parent Object: [DirectionAndDistanceMoveDefinition](../DirectionAndDistanceMoveDefinition/DirectionAndDistanceMoveDefinition.md)

## Description

Property that returns the parent MoveFaceDefinition object.

## Syntax

DirectionAndDistanceMoveDefinition.**Parent**() As [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md)

## Property Value

This is a read only property whose value is a [MoveFaceDefinition](../MoveFaceDefinition/MoveFaceDefinition.md).

## Version

Introduced in version 2011
