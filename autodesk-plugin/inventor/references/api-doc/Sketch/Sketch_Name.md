# Sketch.Name Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Gets and sets name of the sketch.

## Syntax

Sketch.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 5.3
