# Sketch.Delete Method

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Method that deletes the sketch. This method is only valid for sketches that are not used by a feature.

## Syntax

Sketch.**Delete**()

## Version

Introduced in version 5.3
