# Sketch.SketchCircles Property

Parent Object: [Sketch](../Sketch/Sketch.md)

## Description

Property that returns the SketchArcs collection object.

## Syntax

Sketch.**SketchCircles**() As [SketchCircles](../SketchCircles/SketchCircles.md)

## Property Value

This is a read only property whose value is a [SketchCircles](../SketchCircles/SketchCircles.md).
