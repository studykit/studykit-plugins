# ModelHoleThreadNote.Faces Property

Parent Object: [ModelHoleThreadNote](../ModelHoleThreadNote/ModelHoleThreadNote.md)

## Description

Returns the model hole or thread feature faces associated with the hole or thread note.

## Syntax

ModelHoleThreadNote.**Faces**() As [Faces](../Faces/Faces.md)

## Property Value

This is a read only property whose value is a [Faces](../Faces/Faces.md).

## Version

Introduced in version 2023
