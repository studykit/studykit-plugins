# AttributeSets Object

## Description

Provides access to the objects associated with a specific object and provides the ability to create additional sets on the object. See the article in the overviews section.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../AttributeSets/AttributeSets_Add.md) | Method that creates a new for the object the AttributeSets collection was obtained from. The created AttributeSet object is returned. |
| [AddTransient](../AttributeSets/AttributeSets_AddTransient.md) | Adds a transient into this collection. This set is transient as it is available only during this session. |
| [ParentAvailable](../AttributeSets/AttributeSets_ParentAvailable.md) | Method that returns whether the parent can be found. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../AttributeSets/AttributeSets_Count.md) | Property that returns the number of items in this collection. |
| [DataIO](../AttributeSets/AttributeSets_DataIO.md) | Property that returns the object that allows you to write the contents of all of the AttributeSets attached to object as XML. The DataIO object returned supports the WriteDataToFile method with the format "XML." The AttributeSet names are used as the tag names in the XML data. Because of this, the names must be valid XML tag names. This means they must begin with a letter and can contain the following characters '\_', '-', '.'. Spaces are not allowed. |
| [Item](../AttributeSets/AttributeSets_Item.md) | Returns the specified object from the collection. This is the default property of the AttributeSets collection object. |
| [NameIsUsed](../AttributeSets/AttributeSets_NameIsUsed.md) | Property that returns whether an existing attribute set has the specified name or not. Multiple AttributeSets with the same name are not allowed within this collection. Returns True if the name is already being used. |
| [Parent](../AttributeSets/AttributeSets_Parent.md) | Property that returns the object this attribute set is attached to. |
| [Type](../AttributeSets/AttributeSets_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
