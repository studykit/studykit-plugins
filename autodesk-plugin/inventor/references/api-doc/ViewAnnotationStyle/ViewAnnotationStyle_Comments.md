# ViewAnnotationStyle.Comments Property

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Gets and sets comments associated with the style.

## Syntax

ViewAnnotationStyle.**Comments**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025.1
