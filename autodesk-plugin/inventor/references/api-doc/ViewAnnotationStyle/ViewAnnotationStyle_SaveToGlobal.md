# ViewAnnotationStyle.SaveToGlobal Method

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Method that saves this style to the global repository. If a style with the same name is found in the repository, that style is updated.

## Syntax

ViewAnnotationStyle.**SaveToGlobal**()

## Version

Introduced in version 2025.1
