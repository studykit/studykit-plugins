# ViewAnnotationStyle.ConvertToLocal Method

Parent Object: [ViewAnnotationStyle](../ViewAnnotationStyle/ViewAnnotationStyle.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

ViewAnnotationStyle.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2025.1
