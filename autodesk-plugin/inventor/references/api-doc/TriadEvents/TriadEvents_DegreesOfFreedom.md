# TriadEvents.DegreesOfFreedom Property

Parent Object: [TriadEvents](../TriadEvents/TriadEvents.md)

## Description

Gets/sets the allowed translation and rotation directions.

## Syntax

TriadEvents.**DegreesOfFreedom**() As [TriadSegmentEnum](../TriadSegmentEnum.md)

## Property Value

This is a read/write property whose value is a [TriadSegmentEnum](../TriadSegmentEnum.md).

## Version

Introduced in version 9
