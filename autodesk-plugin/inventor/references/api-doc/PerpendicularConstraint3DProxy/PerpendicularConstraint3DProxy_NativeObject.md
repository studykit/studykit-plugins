# PerpendicularConstraint3DProxy.NativeObject Property

Parent Object: [PerpendicularConstraint3DProxy](../PerpendicularConstraint3DProxy/PerpendicularConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

PerpendicularConstraint3DProxy.**NativeObject**() As [PerpendicularConstraint3D](../PerpendicularConstraint3D/PerpendicularConstraint3D.md)

## Property Value

This is a read only property whose value is a [PerpendicularConstraint3D](../PerpendicularConstraint3D/PerpendicularConstraint3D.md).

## Version

Introduced in version 10
