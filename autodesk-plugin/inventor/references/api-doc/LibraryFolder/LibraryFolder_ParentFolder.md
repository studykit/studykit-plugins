# LibraryFolder.ParentFolder Property

Parent Object: [LibraryFolder](../LibraryFolder/LibraryFolder.md)

## Description

Read-only property that returns the parent LibraryFolder for this library folder object.

## Syntax

LibraryFolder.**ParentFolder**() As [LibraryFolder](../LibraryFolder/LibraryFolder.md)

## Property Value

This is a read only property whose value is a [LibraryFolder](../LibraryFolder/LibraryFolder.md).

## Version

Introduced in version 2016
