# VbaApplication Object

## Description

The VBAApplication object represents the currently available VBA application.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [ThisApplication](../VbaApplication/VbaApplication_ThisApplication.md) | Gets the VBA application interface. |
| [ThisServer](../VbaApplication/VbaApplication_ThisServer.md) | Gets the VBA InventorServer interface. |

## Version

Introduced in version 5
