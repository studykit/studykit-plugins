# SectionDrawingView.InheritSection Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether the view should inherit the corresponding section cut from the parent view.

## Syntax

SectionDrawingView.**InheritSection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
