# SectionDrawingView.ModelToDrawingViewTransform Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that returns the transformation from model space to the drawing view coordinate space.

## Syntax

SectionDrawingView.**ModelToDrawingViewTransform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 9
