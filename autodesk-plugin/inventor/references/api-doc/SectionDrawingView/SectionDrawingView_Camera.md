# SectionDrawingView.Camera Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that returns a Camera object that defines the orientation of the model within the view.

## Syntax

SectionDrawingView.**Camera**() As [Camera](../Camera/Camera.md)

## Property Value

This is a read only property whose value is a [Camera](../Camera/Camera.md).

## Version

Introduced in version 9
