# SectionDrawingView.ScaleString Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets or sets the Model-to-PaperSpace scale in string format of this particular Drawing View.

## Syntax

SectionDrawingView.**ScaleString**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2008
