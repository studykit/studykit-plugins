# SectionDrawingView.ShowEntireLine Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that indicates whether to show the entire section line or hide parts of it.

## Syntax

SectionDrawingView.**ShowEntireLine**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
