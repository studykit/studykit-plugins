# SectionDrawingView.IsUpdateComplete Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Indicates if the view has finished updating.

## Syntax

SectionDrawingView.**IsUpdateComplete**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2012
