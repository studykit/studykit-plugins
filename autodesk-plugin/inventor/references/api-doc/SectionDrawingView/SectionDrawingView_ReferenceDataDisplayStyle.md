# SectionDrawingView.ReferenceDataDisplayStyle Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Read-write property that gets and sets the reference data display style.

## Syntax

SectionDrawingView.**ReferenceDataDisplayStyle**() As [ReferenceDataDisplayStyleEnum](../ReferenceDataDisplayStyleEnum.md)

## Property Value

This is a read/write property whose value is a [ReferenceDataDisplayStyleEnum](../ReferenceDataDisplayStyleEnum.md).

## Version

Introduced in version 2024
