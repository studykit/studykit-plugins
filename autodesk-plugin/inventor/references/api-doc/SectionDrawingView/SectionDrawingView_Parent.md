# SectionDrawingView.Parent Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property returning the parent Sheet object.

## Syntax

SectionDrawingView.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 9
