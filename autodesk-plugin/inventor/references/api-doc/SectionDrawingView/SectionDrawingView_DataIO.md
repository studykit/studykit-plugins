# SectionDrawingView.DataIO Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that returns the associated DataIO object. This can be used in Apprentice to obtain an associated bitmap, if it exists.

## Syntax

SectionDrawingView.**DataIO**() As [DataIO](../DataIO/DataIO.md)

## Property Value

This is a read only property whose value is a [DataIO](../DataIO/DataIO.md).

## Version

Introduced in version 9
