# SectionDrawingView.DisplayForeshortenedTangentEdges Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether to display shortened tangent edges in order to distinguish them from visible edges.

## Syntax

SectionDrawingView.**DisplayForeshortenedTangentEdges**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
