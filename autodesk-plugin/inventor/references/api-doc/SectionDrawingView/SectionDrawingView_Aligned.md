# SectionDrawingView.Aligned Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets whether the view is aligned with another view.

## Syntax

SectionDrawingView.**Aligned**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2010
