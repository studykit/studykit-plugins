# SectionDrawingView.Left Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that returns the position of the left edge of the drawing view.

## Syntax

SectionDrawingView.**Left**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 9
