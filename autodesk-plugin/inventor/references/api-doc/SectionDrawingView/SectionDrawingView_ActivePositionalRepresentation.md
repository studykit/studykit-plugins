# SectionDrawingView.ActivePositionalRepresentation Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Gets and sets the name of the active Positional Representation for a drawing view of an assembly.

## Syntax

SectionDrawingView.**ActivePositionalRepresentation**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 11
