# SectionDrawingView.AuxiliaryOrientationEdge Property

Parent Object: [SectionDrawingView](../SectionDrawingView/SectionDrawingView.md)

## Description

Property that returns drawing curve that the auxiliary view is aligned to. This property returns Nothing if this is not an auxiliary view or if the orientation edge (drawing curve) has been removed from the parent view during a subsequent update.

## Syntax

SectionDrawingView.**AuxiliaryOrientationEdge**() As [DrawingCurve](../DrawingCurve/DrawingCurve.md)

## Property Value

This is a read only property whose value is a [DrawingCurve](../DrawingCurve/DrawingCurve.md).

## Version

Introduced in version 2010
