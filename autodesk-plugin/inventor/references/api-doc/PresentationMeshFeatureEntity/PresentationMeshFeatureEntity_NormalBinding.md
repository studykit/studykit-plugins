# PresentationMeshFeatureEntity.NormalBinding Property

Parent Object: [PresentationMeshFeatureEntity](../PresentationMeshFeatureEntity/PresentationMeshFeatureEntity.md)

## Description

Read-only property that returns how the normals are defined for the mesh entity.

## Syntax

PresentationMeshFeatureEntity.**NormalBinding**() As [NormalBindingEnum](../NormalBindingEnum.md)

## Property Value

This is a read only property whose value is a [NormalBindingEnum](../NormalBindingEnum.md).

## Version

Introduced in version 2018
