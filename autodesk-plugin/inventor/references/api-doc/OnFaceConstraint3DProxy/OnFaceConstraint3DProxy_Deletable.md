# OnFaceConstraint3DProxy.Deletable Property

Parent Object: [OnFaceConstraint3DProxy](../OnFaceConstraint3DProxy/OnFaceConstraint3DProxy.md)

## Description

Indicates whether this object can be deleted.

## Syntax

OnFaceConstraint3DProxy.**Deletable**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2017
