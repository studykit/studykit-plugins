# CircularOccurrencePatternProxy.NativeObject Property

Parent Object: [CircularOccurrencePatternProxy](../CircularOccurrencePatternProxy/CircularOccurrencePatternProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

CircularOccurrencePatternProxy.**NativeObject**() As [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md)

## Property Value

This is a read only property whose value is a [CircularOccurrencePattern](../CircularOccurrencePattern/CircularOccurrencePattern.md).

## Version

Introduced in version 2010
