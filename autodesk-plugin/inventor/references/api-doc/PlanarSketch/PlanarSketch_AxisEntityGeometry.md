# PlanarSketch.AxisEntityGeometry Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that gets the geometry that describes the axis entity.

## Syntax

PlanarSketch.**AxisEntityGeometry**() As [Line](../Line/Line.md)

## Property Value

This is a read only property whose value is a [Line](../Line/Line.md).
