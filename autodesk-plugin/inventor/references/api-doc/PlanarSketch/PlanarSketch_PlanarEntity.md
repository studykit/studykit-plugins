# PlanarSketch.PlanarEntity Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets and sets the planar object that defines the planar object the sketch is to be built on.

## Syntax

PlanarSketch.**PlanarEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.
