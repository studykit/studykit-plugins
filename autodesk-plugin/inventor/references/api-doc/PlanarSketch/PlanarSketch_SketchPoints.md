# PlanarSketch.SketchPoints Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the SketchPoints collection object.

## Syntax

PlanarSketch.**SketchPoints**() As [SketchPoints](../SketchPoints/SketchPoints.md)

## Property Value

This is a read only property whose value is a [SketchPoints](../SketchPoints/SketchPoints.md).
