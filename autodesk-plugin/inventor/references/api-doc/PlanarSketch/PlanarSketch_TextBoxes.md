# PlanarSketch.TextBoxes Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets the TextBoxes collection associated with this Sketch.

## Syntax

PlanarSketch.**TextBoxes**() As [TextBoxes](../TextBoxes/TextBoxes.md)

## Property Value

This is a read only property whose value is a [TextBoxes](../TextBoxes/TextBoxes.md).
