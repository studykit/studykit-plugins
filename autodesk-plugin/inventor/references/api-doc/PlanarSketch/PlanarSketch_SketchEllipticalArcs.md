# PlanarSketch.SketchEllipticalArcs Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns the SketchEllipticalArcs collection object.

## Syntax

PlanarSketch.**SketchEllipticalArcs**() As [SketchEllipticalArcs](../SketchEllipticalArcs/SketchEllipticalArcs.md)

## Property Value

This is a read only property whose value is a [SketchEllipticalArcs](../SketchEllipticalArcs/SketchEllipticalArcs.md).

## Version

Introduced in version 5
