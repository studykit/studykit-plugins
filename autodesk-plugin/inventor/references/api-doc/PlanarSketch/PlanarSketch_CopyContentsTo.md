# PlanarSketch.CopyContentsTo Method

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Method that copies all the contents of the sketch to the \input target sketch.

## Syntax

PlanarSketch.**CopyContentsTo**( ***TargetSketch*** As [Sketch](../Sketch/Sketch.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| TargetSketch | [Sketch](../Sketch/Sketch.md) | Sketch object that specifies the target sketch into which the contents of the sketch should be copied. |

## Version

Introduced in version 2011
