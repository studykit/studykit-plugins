# PlanarSketch.SketchImages Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Property that returns a collection of all images on the sketch.

## Syntax

PlanarSketch.**SketchImages**() As [SketchImages](../SketchImages/SketchImages.md)

## Property Value

This is a read only property whose value is a [SketchImages](../SketchImages/SketchImages.md).

## Version

Introduced in version 11
