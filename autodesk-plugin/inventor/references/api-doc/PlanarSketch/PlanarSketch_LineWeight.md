# PlanarSketch.LineWeight Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets and Sets the line weight override for the sketch.

## Syntax

PlanarSketch.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2010
