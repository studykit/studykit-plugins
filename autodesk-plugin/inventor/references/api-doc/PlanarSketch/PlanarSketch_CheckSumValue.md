# PlanarSketch.CheckSumValue Property

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Gets sketch checksum value.

## Syntax

PlanarSketch.**CheckSumValue**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2020
