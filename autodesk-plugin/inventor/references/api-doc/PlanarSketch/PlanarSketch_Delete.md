# PlanarSketch.Delete Method

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Method that deletes the sketch. This method is only valid for sketches that are not used by a feature.

## Syntax

PlanarSketch.**Delete**()
