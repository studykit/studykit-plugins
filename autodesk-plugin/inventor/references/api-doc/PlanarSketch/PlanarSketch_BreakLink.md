# PlanarSketch.BreakLink Method

Parent Object: [PlanarSketch](../PlanarSketch/PlanarSketch.md)

## Description

Method that breaks the link to the source sketch.

## Syntax

PlanarSketch.**BreakLink**()

## Version

Introduced in version 2011
