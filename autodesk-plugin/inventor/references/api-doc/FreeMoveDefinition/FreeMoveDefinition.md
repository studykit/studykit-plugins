# FreeMoveDefinition Object

## Description

The FreeMoveDefinition object defines all of the inputs that are unique to a move face feature defined using a transform.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../FreeMoveDefinition/FreeMoveDefinition_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Parent](../FreeMoveDefinition/FreeMoveDefinition_Parent.md) | Property that returns the parent MoveFaceDefinition object. |
| [Transformation](../FreeMoveDefinition/FreeMoveDefinition_Transformation.md) | Gets and sets the Matrix defining the free move transform of the faces. |
| [Type](../FreeMoveDefinition/FreeMoveDefinition_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2011
