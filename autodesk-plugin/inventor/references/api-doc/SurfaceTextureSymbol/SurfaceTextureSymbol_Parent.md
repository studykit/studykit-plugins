# SurfaceTextureSymbol.Parent Property

Parent Object: [SurfaceTextureSymbol](../SurfaceTextureSymbol/SurfaceTextureSymbol.md)

## Description

Property that returns the parent Sheet object.

## Syntax

SurfaceTextureSymbol.**Parent**() As [Sheet](../Sheet/Sheet.md)

## Property Value

This is a read only property whose value is a [Sheet](../Sheet/Sheet.md).

## Version

Introduced in version 2009
