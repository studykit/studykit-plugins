# ObjectDefaultsStyle.BalloonStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for balloons.

## Syntax

ObjectDefaultsStyle.**BalloonStyle**() As [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Property Value

This is a read/write property whose value is a [BalloonStyle](../BalloonStyle/BalloonStyle.md).

## Version

Introduced in version 2010
