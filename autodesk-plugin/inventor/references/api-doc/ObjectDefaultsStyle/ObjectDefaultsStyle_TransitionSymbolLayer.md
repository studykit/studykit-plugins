# ObjectDefaultsStyle.TransitionSymbolLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default layer for transition symbols.

## Syntax

ObjectDefaultsStyle.**TransitionSymbolLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2025.1
