# ObjectDefaultsStyle.TransitionSymbolStyle Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Gets and sets the default style for transition symbols.

## Syntax

ObjectDefaultsStyle.**TransitionSymbolStyle**() As [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md)

## Property Value

This is a read/write property whose value is a [TransitionSymbolStyle](../TransitionSymbolStyle/TransitionSymbolStyle.md).

## Version

Introduced in version 2025.1
