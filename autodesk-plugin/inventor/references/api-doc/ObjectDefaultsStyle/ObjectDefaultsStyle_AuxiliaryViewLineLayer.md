# ObjectDefaultsStyle.AuxiliaryViewLineLayer Property

Parent Object: [ObjectDefaultsStyle](../ObjectDefaultsStyle/ObjectDefaultsStyle.md)

## Description

Read-write property that gets and sets the default layer for auxiliary view line.

## Syntax

ObjectDefaultsStyle.**AuxiliaryViewLineLayer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 2025.1
