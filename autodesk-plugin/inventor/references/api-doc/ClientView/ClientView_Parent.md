# ClientView.Parent Property

Parent Object: [ClientView](../ClientView/ClientView.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

ClientView.**Parent**() As [ClientViews](../ClientViews/ClientViews.md)

## Property Value

This is a read only property whose value is a [ClientViews](../ClientViews/ClientViews.md).

## Version

Introduced in version 6
