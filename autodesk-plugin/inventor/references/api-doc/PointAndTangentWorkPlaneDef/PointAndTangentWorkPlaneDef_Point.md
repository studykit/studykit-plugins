# PointAndTangentWorkPlaneDef.Point Property

Parent Object: [PointAndTangentWorkPlaneDef](../PointAndTangentWorkPlaneDef/PointAndTangentWorkPlaneDef.md)

## Description

Property that returns the point of a work plane that passes through a point and is tangent to a surface. This object can be a Vertex, WorkPoint, SketchPoint, or SketchPoint3D object.

## Syntax

PointAndTangentWorkPlaneDef.**Point**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 10
