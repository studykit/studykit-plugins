# iPartTableRows Object

## Description

TheiPartTableRows collection object provides access to all existing objects in the iPart factory table.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../iPartTableRows/iPartTableRows_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../iPartTableRows/iPartTableRows_Count.md) | Property that returns the number of items in the collection. |
| [Item](../iPartTableRows/iPartTableRows_Item.md) | Returns the specified iPartTableRow object from the collection. This is the default property of the iPartTableRows collection object. |
| [Type](../iPartTableRows/iPartTableRows_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[iPartFactory.TableRows](../iPartFactory/iPartFactory_TableRows.md)

## Version

Introduced in version 6
