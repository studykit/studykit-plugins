# Polyline2d.Evaluator Property

Parent Object: [Polyline2d](../Polyline2d/Polyline2d.md)

## Description

Property that returns an evaluator object for the curve. The evaluator supports general curve evaluation functions.

## Syntax

Polyline2d.**Evaluator**() As [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md)

## Property Value

This is a read only property whose value is a [Curve2dEvaluator](../Curve2dEvaluator/Curve2dEvaluator.md).

## Version

Introduced in version 2009
