# PathEntityProxy.NativeObject Property

Parent Object: [PathEntityProxy](../PathEntityProxy/PathEntityProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

PathEntityProxy.**NativeObject**() As [PathEntity](../PathEntity/PathEntity.md)

## Property Value

This is a read only property whose value is a [PathEntity](../PathEntity/PathEntity.md).

## Version

Introduced in version 9
