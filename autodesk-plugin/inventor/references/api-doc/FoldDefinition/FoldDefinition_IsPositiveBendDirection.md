# FoldDefinition.IsPositiveBendDirection Property

Parent Object: [FoldDefinition](../FoldDefinition/FoldDefinition.md)

## Description

Gets and sets whether the direction of the fold is in the positive or negative direction.

## Syntax

FoldDefinition.**IsPositiveBendDirection**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
