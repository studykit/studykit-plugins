# CheckPointsEnumerator.Item Property

Parent Object: [CheckPointsEnumerator](../CheckPointsEnumerator/CheckPointsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

CheckPointsEnumerator.**Item**( ***Index*** As Long ) As [CheckPoint](../CheckPoint/CheckPoint.md)

## Property Value

This is a read only property whose value is a [CheckPoint](../CheckPoint/CheckPoint.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 5
