# SnapFitFeature.Name Property

Parent Object: [SnapFitFeature](../SnapFitFeature/SnapFitFeature.md)

## Description

Gets/Sets the name of this Part Feature within the scope of this Document.

## Syntax

SnapFitFeature.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2010
