# SketchFillRegion.Profile Property

Parent Object: [SketchFillRegion](../SketchFillRegion/SketchFillRegion.md)

## Description

Property to gets the profile object associated with this sketch fill region.

## Syntax

SketchFillRegion.**Profile**() As [Profile](../Profile/Profile.md)

## Property Value

This is a read only property whose value is a [Profile](../Profile/Profile.md).

## Version

Introduced in version 10
