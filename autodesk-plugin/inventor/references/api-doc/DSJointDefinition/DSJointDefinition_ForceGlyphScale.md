# DSJointDefinition.ForceGlyphScale Property

Parent Object: [DSJointDefinition](../DSJointDefinition/DSJointDefinition.md)

## Description

Gets and sets the scale of the force glyph.

## Syntax

DSJointDefinition.**ForceGlyphScale**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2013
