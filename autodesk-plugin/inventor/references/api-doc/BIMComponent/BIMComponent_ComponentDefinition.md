# BIMComponent.ComponentDefinition Property

Parent Object: [BIMComponent](../BIMComponent/BIMComponent.md)

## Description

Read-only property that returns the Inventor PartComponentDefinition or AssemblyComponentDefinition that this BIMComponent object is associated with.

## Syntax

BIMComponent.**ComponentDefinition**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2011
