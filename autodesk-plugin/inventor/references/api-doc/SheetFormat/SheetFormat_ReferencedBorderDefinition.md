# SheetFormat.ReferencedBorderDefinition Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that returns the BorderDefinition that this format references.

## Syntax

SheetFormat.**ReferencedBorderDefinition**() As [BorderDefinition](../BorderDefinition/BorderDefinition.md)

## Property Value

This is a read only property whose value is a [BorderDefinition](../BorderDefinition/BorderDefinition.md).

## Version

Introduced in version 2009
