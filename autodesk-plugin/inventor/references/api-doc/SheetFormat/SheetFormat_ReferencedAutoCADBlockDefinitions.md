# SheetFormat.ReferencedAutoCADBlockDefinitions Property

Parent Object: [SheetFormat](../SheetFormat/SheetFormat.md)

## Description

Property that returns an enumerator of all the AutoCADBlockDefinition objects that this format references. The property returns Nothing if the sheet format does not reference any AutoCAD block definitions.

## Syntax

SheetFormat.**ReferencedAutoCADBlockDefinitions**() As [AutoCADBlockDefinitionsEnumerator](../AutoCADBlockDefinitionsEnumerator/AutoCADBlockDefinitionsEnumerator.md)

## Property Value

This is a read only property whose value is an [AutoCADBlockDefinitionsEnumerator](../AutoCADBlockDefinitionsEnumerator/AutoCADBlockDefinitionsEnumerator.md).

## Version

Introduced in version 2011
