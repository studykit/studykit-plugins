# PartsListRow.Remove Method

Parent Object: [PartsListRow](../PartsListRow/PartsListRow.md)

## Description

Method that removes this row from the parts list. This method works only for custom rows.

## Syntax

PartsListRow.**Remove**()

## Version

Introduced in version 9
