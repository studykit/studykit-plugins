# PartsListRow.Ballooned Property

Parent Object: [PartsListRow](../PartsListRow/PartsListRow.md)

## Description

Property that returns whether the item represented by this row in the parts list has been ballooned.

## Syntax

PartsListRow.**Ballooned**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 10
