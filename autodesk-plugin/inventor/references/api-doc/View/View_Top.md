# View.Top Property

Parent Object: [View](../View/View.md)

## Description

Gets/Sets the distance between the top of the view window and top of the frame window.

## Syntax

View.**Top**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 6
