# View.Left Property

Parent Object: [View](../View/View.md)

## Description

Gets/Sets the distance between the left edge of the view window and left edge of the frame window.

## Syntax

View.**Left**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 6
