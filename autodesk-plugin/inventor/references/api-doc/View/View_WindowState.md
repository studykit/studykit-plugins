# View.WindowState Property

Parent Object: [View](../View/View.md)

## Description

Gets/Sets this windows state.

## Syntax

View.**WindowState**() As [WindowsSizeEnum](../WindowsSizeEnum.md)

## Property Value

This is a read/write property whose value is a [WindowsSizeEnum](../WindowsSizeEnum.md).

## Version

Introduced in version 6
