# View.DisplaySeparateColorsType Property

Parent Object: [View](../View/View.md)

## Description

Read-write property that gets and sets which type of display separate colors will be applied.

## Syntax

View.**DisplaySeparateColorsType**() As [DisplaySeparateColorsTypeEnum](../DisplaySeparateColorsTypeEnum.md)

## Property Value

This is a read/write property whose value is a [DisplaySeparateColorsTypeEnum](../DisplaySeparateColorsTypeEnum.md).

## Version

Introduced in version 2025
