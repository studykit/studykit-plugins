# View.RayTracingQuality Property

Parent Object: [View](../View/View.md)

## Description

Gets and sets the quality used when ray tracing is enabled.

## Syntax

View.**RayTracingQuality**() As [RayTracingQualityEnum](../RayTracingQualityEnum.md)

## Property Value

This is a read/write property whose value is a [RayTracingQualityEnum](../RayTracingQualityEnum.md).

## Version

Introduced in version 2012
