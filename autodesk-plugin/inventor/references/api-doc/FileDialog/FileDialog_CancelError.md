# FileDialog.CancelError Property

Parent Object: [FileDialog](../FileDialog/FileDialog.md)

## Description

Generate error when user cancels dialog if true.

## Syntax

FileDialog.**CancelError**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.
