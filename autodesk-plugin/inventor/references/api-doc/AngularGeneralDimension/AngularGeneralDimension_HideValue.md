# AngularGeneralDimension.HideValue Property

Parent Object: [AngularGeneralDimension](../AngularGeneralDimension/AngularGeneralDimension.md)

## Description

Gets and sets the HideValue setting.

## Syntax

AngularGeneralDimension.**HideValue**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
