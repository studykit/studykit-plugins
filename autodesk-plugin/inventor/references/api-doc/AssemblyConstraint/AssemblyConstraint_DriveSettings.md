# AssemblyConstraint.DriveSettings Property

Parent Object: [AssemblyConstraint](../AssemblyConstraint/AssemblyConstraint.md)

## Description

Returns the DriveSettings object for the constraint.

## Syntax

AssemblyConstraint.**DriveSettings**() As [DriveSettings](../DriveSettings/DriveSettings.md)

## Property Value

This is a read only property whose value is a [DriveSettings](../DriveSettings/DriveSettings.md).

## Version

Introduced in version 2014
