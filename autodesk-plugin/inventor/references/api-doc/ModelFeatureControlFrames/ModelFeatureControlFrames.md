# ModelFeatureControlFrames Object

## Description

The ModelFeatureControlFrames collection object provides access to all the model feature control frame symbols in a part or assembly and provides methods to create additional feature control frames.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../ModelFeatureControlFrames/ModelFeatureControlFrames_Add.md) | Method that creates a feature control frame symbol. |
| [CreateDefinition](../ModelFeatureControlFrames/ModelFeatureControlFrames_CreateDefinition.md) | Method that creates a feature control frame definition. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ModelFeatureControlFrames/ModelFeatureControlFrames_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../ModelFeatureControlFrames/ModelFeatureControlFrames_Count.md) | Gets the number of items in this collection. |
| [Item](../ModelFeatureControlFrames/ModelFeatureControlFrames_Item.md) | Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well. |
| [Type](../ModelFeatureControlFrames/ModelFeatureControlFrames_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[ModelAnnotations.ModelFeatureControlFrames](../ModelAnnotations/ModelAnnotations_ModelFeatureControlFrames.md)

## Version

Introduced in version 2018
