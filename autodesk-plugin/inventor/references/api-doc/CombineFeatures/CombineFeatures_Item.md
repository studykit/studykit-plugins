# CombineFeatures.Item Property

Parent Object: [CombineFeatures](../CombineFeatures/CombineFeatures.md)

## Description

Method that returns the specified CombineFeature object from the collection.

## Syntax

CombineFeatures.**Item**( ***Index*** As Variant ) As [CombineFeature](../CombineFeature/CombineFeature.md)

## Property Value

This is a read only property whose value is a [CombineFeature](../CombineFeature/CombineFeature.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the object to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the feature name. If an out of range index or a name of a non-existent combine feature is provided, an error occurs. |

## Version

Introduced in version 2010
