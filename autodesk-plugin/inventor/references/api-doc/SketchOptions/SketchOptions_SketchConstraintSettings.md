# SketchOptions.SketchConstraintSettings Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Returns the SketchConstraintSettings object.

## Syntax

SketchOptions.**SketchConstraintSettings**() As [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Property Value

This is a read only property whose value is a [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md).

## Version

Introduced in version 2015
