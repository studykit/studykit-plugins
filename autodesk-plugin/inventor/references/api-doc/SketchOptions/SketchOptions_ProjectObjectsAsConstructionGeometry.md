# SketchOptions.ProjectObjectsAsConstructionGeometry Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Enables/disables whether project objects as construction geometry.

## Syntax

SketchOptions.**ProjectObjectsAsConstructionGeometry**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
