# SketchOptions.DisplayCoordinateSystemIndicator Property

Parent Object: [SketchOptions](../SketchOptions/SketchOptions.md)

## Description

Enables/disables the display of the sketch plane coordinate system indicator (triad).

## Syntax

SketchOptions.**DisplayCoordinateSystemIndicator**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 9
