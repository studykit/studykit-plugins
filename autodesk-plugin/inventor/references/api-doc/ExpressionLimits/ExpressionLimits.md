# ExpressionLimits Object

## Description

The ExpressionLimits object provides access to the various range values that can be specified when defining a custom expression for a column.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ExpressionLimits/ExpressionLimits_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [DefaultValue](../ExpressionLimits/ExpressionLimits_DefaultValue.md) | Gets/Sets the default value for this expression. |
| [IncrementValue](../ExpressionLimits/ExpressionLimits_IncrementValue.md) | Gets/Sets the increment value for this expression. |
| [MaximumValue](../ExpressionLimits/ExpressionLimits_MaximumValue.md) | Gets/Sets the maximum value for this expression. |
| [MaximumValueComparisonType](../ExpressionLimits/ExpressionLimits_MaximumValueComparisonType.md) | Gets/Sets the type of comparison to perform against the maximum value. |
| [MinimumValue](../ExpressionLimits/ExpressionLimits_MinimumValue.md) | Gets/Sets the minimum value for this expression. |
| [MinimumValueComparisonType](../ExpressionLimits/ExpressionLimits_MinimumValueComparisonType.md) | Gets/Sets the type of comparison to perform against the minimum value. |
| [Type](../ExpressionLimits/ExpressionLimits_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[ContentTableColumns.CreateExpressionLimits](../ContentTableColumns/ContentTableColumns_CreateExpressionLimits.md)

## Version

Introduced in version 2010
