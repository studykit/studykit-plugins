# LanguageTools.CurrentPrimaryModelStateString Property

Parent Object: [LanguageTools](../LanguageTools/LanguageTools.md)

## Description

Read-only property that returns the primary ModelState string in current language.

## Syntax

LanguageTools.**CurrentPrimaryModelStateString**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2023
