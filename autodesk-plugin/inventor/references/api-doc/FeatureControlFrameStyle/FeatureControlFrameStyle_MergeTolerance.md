# FeatureControlFrameStyle.MergeTolerance Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to merge tolerance cells when the data is the same.

## Syntax

FeatureControlFrameStyle.**MergeTolerance**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
