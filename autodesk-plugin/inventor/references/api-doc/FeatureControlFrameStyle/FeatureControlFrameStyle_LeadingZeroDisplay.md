# FeatureControlFrameStyle.LeadingZeroDisplay Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets whether to display the leading zeroes for primary units.

## Syntax

FeatureControlFrameStyle.**LeadingZeroDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
