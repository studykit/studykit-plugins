# FeatureControlFrameStyle.EnableInlineNotes Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Gets and sets whether the inline notes is enabled or not.

## Syntax

FeatureControlFrameStyle.**EnableInlineNotes**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
