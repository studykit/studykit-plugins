# FeatureControlFrameStyle.AlternateUnitsDisplayType Property

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Property that gets and sets the arrangement of the alternate units display with respect to the primary units. The primary unit always precedes the alternate unit. This property is used only when the AlternateUnitsDisplay property is set to true.

## Syntax

FeatureControlFrameStyle.**AlternateUnitsDisplayType**() As [AlternateUnitsDisplayTypeEnum](../AlternateUnitsDisplayTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AlternateUnitsDisplayTypeEnum](../AlternateUnitsDisplayTypeEnum.md).

## Version

Introduced in version 2009
