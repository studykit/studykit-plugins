# FeatureControlFrameStyle.GetAvailableAdditionalSymbols Method

Parent Object: [FeatureControlFrameStyle](../FeatureControlFrameStyle/FeatureControlFrameStyle.md)

## Description

Gets the array of FeatureControlFrameAdditionalSymbolsEnum indicating the additional symbols to be made available in the feature control frame creation and edit dialog.

## Syntax

FeatureControlFrameStyle.**GetAvailableAdditionalSymbols**( ***AvailaleAdditionalSymbols***() As [FeatureControlFrameAdditionalSymbolsEnum](../FeatureControlFrameAdditionalSymbolsEnum.md) )

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| AvailaleAdditionalSymbols | [FeatureControlFrameAdditionalSymbolsEnum](../FeatureControlFrameAdditionalSymbolsEnum.md) | Output array of FeatureControlFrameAdditionalSymbolsEnum indicating the additional symbols to be made available in the feature control frame creation and edit dialog. |

## Version

Introduced in version 2016
