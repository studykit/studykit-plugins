# DWGBlockReferenceProxy.NativeObject Property

Parent Object: [DWGBlockReferenceProxy](../DWGBlockReferenceProxy/DWGBlockReferenceProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGBlockReferenceProxy.**NativeObject**() As [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md)

## Property Value

This is a read only property whose value is a [DWGBlockReference](../DWGBlockReference/DWGBlockReference.md).

## Version

Introduced in version 2016
