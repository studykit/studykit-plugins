# DWGBlockReferenceProxy.Transformation Property

Parent Object: [DWGBlockReferenceProxy](../DWGBlockReferenceProxy/DWGBlockReferenceProxy.md)

## Description

Read-only property that returns the transformation matrix for the DWGBlockReference instance.

## Syntax

DWGBlockReferenceProxy.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2016
