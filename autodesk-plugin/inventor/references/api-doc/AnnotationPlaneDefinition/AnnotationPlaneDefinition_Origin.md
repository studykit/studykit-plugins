# AnnotationPlaneDefinition.Origin Property

Parent Object: [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Description

Read-only property that returns origin of the annotation plane in model space.

## Syntax

AnnotationPlaneDefinition.**Origin**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2018
