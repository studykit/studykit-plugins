# AnnotationPlaneDefinition.Transformation Property

Parent Object: [AnnotationPlaneDefinition](../AnnotationPlaneDefinition/AnnotationPlaneDefinition.md)

## Description

Read-only property that returns a matrix indicating the position and orientation of the annotation plane in model space.

## Syntax

AnnotationPlaneDefinition.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read only property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2018
