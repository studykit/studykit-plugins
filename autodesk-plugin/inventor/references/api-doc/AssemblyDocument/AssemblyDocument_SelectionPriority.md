# AssemblyDocument.SelectionPriority Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Gets and sets the current selection priority for the document.

## Syntax

AssemblyDocument.**SelectionPriority**() As [SelectionPriorityEnum](../SelectionPriorityEnum.md)

## Property Value

This is a read/write property whose value is a [SelectionPriorityEnum](../SelectionPriorityEnum.md).

## Version

Introduced in version 2008
