# AssemblyDocument.ReferencedOLEFileDescriptors Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Property that returns the collection of linked and embedded files in the document.

## Syntax

AssemblyDocument.**ReferencedOLEFileDescriptors**() As [ReferencedOLEFileDescriptors](../ReferencedOLEFileDescriptors/ReferencedOLEFileDescriptors.md)

## Property Value

This is a read only property whose value is a [ReferencedOLEFileDescriptors](../ReferencedOLEFileDescriptors/ReferencedOLEFileDescriptors.md).

## Version

Introduced in version 4
