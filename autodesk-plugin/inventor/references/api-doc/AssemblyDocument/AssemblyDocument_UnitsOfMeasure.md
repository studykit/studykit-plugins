# AssemblyDocument.UnitsOfMeasure Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Property that returns the UnitsOfMeasure object.

## Syntax

AssemblyDocument.**UnitsOfMeasure**() As [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md)

## Property Value

This is a read only property whose value is a [UnitsOfMeasure](../UnitsOfMeasure/UnitsOfMeasure.md).
