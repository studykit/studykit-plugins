# AssemblyDocument.DocumentSubType Property

Parent Object: [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Description

Property that returns the subtype of the document.

## Syntax

AssemblyDocument.**DocumentSubType**() As [DocumentSubType](DocumentSubType.md)

## Property Value

This is a read only property whose value is a [DocumentSubType](DocumentSubType.md).

## Version

Introduced in version 6
