# WireEdgeDefinitions.Item Property

Parent Object: [WireEdgeDefinitions](../WireEdgeDefinitions/WireEdgeDefinitions.md)

## Description

Returns the specified WireEdgeDefinition object from the collection.

## Syntax

WireEdgeDefinitions.**Item**( ***Index*** As Long ) As [WireEdgeDefinition](../WireEdgeDefinition/WireEdgeDefinition.md)

## Property Value

This is a read only property whose value is a [WireEdgeDefinition](../WireEdgeDefinition/WireEdgeDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the WireEdgeDefinition object within the collection to return. |

## Version

Introduced in version 2013
