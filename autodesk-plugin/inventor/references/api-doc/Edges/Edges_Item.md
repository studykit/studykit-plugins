# Edges.Item Property

Parent Object: [Edges](../Edges/Edges.md)

## Description

Allows integer-indexed access to objects in the collection.

## Syntax

Edges.**Item**( ***Index*** As Long ) As [Edge](../Edge/Edge.md)

## Property Value

This is a read only property whose value is an [Edge](../Edge/Edge.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |
