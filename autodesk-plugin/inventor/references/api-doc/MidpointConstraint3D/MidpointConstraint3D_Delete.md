# MidpointConstraint3D.Delete Method

Parent Object: [MidpointConstraint3D](../MidpointConstraint3D/MidpointConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

MidpointConstraint3D.**Delete**()

## Version

Introduced in version 2009
