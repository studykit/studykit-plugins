# ConcentricConstraint.CenterPointConstraint Property

Parent Object: [ConcentricConstraint](../ConcentricConstraint/ConcentricConstraint.md)

## Description

Gets a Boolean indicating if the constraint is between the arc and its centerpoint or not.

## Syntax

ConcentricConstraint.**CenterPointConstraint**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
