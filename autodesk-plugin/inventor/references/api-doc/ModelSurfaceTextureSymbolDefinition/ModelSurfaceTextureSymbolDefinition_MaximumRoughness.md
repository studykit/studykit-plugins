# ModelSurfaceTextureSymbolDefinition.MaximumRoughness Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets the roughness value, roughness value Ra maximum, maximum roughness value, or grade number. Denoted by A' in the surface texture symbol dialog.

## Syntax

ModelSurfaceTextureSymbolDefinition.**MaximumRoughness**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2018
