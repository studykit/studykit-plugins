# ModelSurfaceTextureSymbolDefinition.SurfaceProfileUpperTolerance Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Gets and sets the surface profile upper tolerance.

## Syntax

ModelSurfaceTextureSymbolDefinition.**SurfaceProfileUpperTolerance**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025
