# ModelSurfaceTextureSymbolDefinition.AllAroundSymbol Property

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Read-write property that gets and sets whether to add the all-around indicator to the symbol.

## Syntax

ModelSurfaceTextureSymbolDefinition.**AllAroundSymbol**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2018
