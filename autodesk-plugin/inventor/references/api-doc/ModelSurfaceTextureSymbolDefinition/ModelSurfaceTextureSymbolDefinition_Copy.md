# ModelSurfaceTextureSymbolDefinition.Copy Method

Parent Object: [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Description

Method that creates a copy of this ModelSurfaceTextureSymbolDefinition object. The new ModelSurfaceTextureSymbolDefinition object is independent of any surface texture symbols. It can edited and used as input to edit an existing surface texture symbol or to create a new surface texture symbol.

## Syntax

ModelSurfaceTextureSymbolDefinition.**Copy**() As [ModelSurfaceTextureSymbolDefinition](../ModelSurfaceTextureSymbolDefinition/ModelSurfaceTextureSymbolDefinition.md)

## Version

Introduced in version 2018
