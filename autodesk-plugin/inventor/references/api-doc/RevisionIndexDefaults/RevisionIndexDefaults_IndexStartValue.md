# RevisionIndexDefaults.IndexStartValue Property

Parent Object: [RevisionIndexDefaults](../RevisionIndexDefaults/RevisionIndexDefaults.md)

## Description

Read-write property that gets and sets the index start value.

## Syntax

RevisionIndexDefaults.**IndexStartValue**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2025
