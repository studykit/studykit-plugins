# HorizontalConstraintProxy.Entity Property

Parent Object: [HorizontalConstraintProxy](../HorizontalConstraintProxy/HorizontalConstraintProxy.md)

## Description

Property that returns the sketch entity being constrained. This can be a sketch line, ellipse, or elliptical arc.

## Syntax

HorizontalConstraintProxy.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
