# HorizontalConstraintProxy.NativeObject Property

Parent Object: [HorizontalConstraintProxy](../HorizontalConstraintProxy/HorizontalConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

HorizontalConstraintProxy.**NativeObject**() As [HorizontalConstraint](../HorizontalConstraint/HorizontalConstraint.md)

## Property Value

This is a read only property whose value is a [HorizontalConstraint](../HorizontalConstraint/HorizontalConstraint.md).

## Version

Introduced in version 6
