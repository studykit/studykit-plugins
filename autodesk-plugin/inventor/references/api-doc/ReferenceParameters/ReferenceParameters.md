# ReferenceParameters Object

## Description

Provides access to the objects of the component definition the collection was obtained from.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddByExpression](../ReferenceParameters/ReferenceParameters_AddByExpression.md) | Method that creates a new reference parameter given a name and expression. |
| [AddByValue](../ReferenceParameters/ReferenceParameters_AddByValue.md) | Method that creates a new reference parameter given a name and equation. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../ReferenceParameters/ReferenceParameters_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ReferenceParameters/ReferenceParameters_Item.md) | Returns the specified ReferenceParameter object from the collection. This is the default property of the ReferenceParameters collection object. |
| [Type](../ReferenceParameters/ReferenceParameters_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Parameters.ReferenceParameters](../Parameters/Parameters_ReferenceParameters.md)

## Version

Introduced in version 4
