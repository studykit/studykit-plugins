# PatternBoundarySetting.BoundaryInclusionType Property

Parent Object: [PatternBoundarySetting](../PatternBoundarySetting/PatternBoundarySetting.md)

## Description

Read-write property that gets and sets boundary inclusion type. Use the SetBoundarySettingData method when set this to kOccurrenceBasePointInclusionType so the OccurrenceBasePoint can be specified also.

## Syntax

PatternBoundarySetting.**BoundaryInclusionType**() As [PatternBoundaryInclusionEnum](../PatternBoundaryInclusionEnum.md)

## Property Value

This is a read/write property whose value is a [PatternBoundaryInclusionEnum](../PatternBoundaryInclusionEnum.md).

## Version

Introduced in version 2025
