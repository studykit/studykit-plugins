# PartFeatures.ThickenFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the ThickenFeatures collection object. This collection provides access to existing ThickenFeature objects.

## Syntax

PartFeatures.**ThickenFeatures**() As [ThickenFeatures](../ThickenFeatures/ThickenFeatures.md)

## Property Value

This is a read only property whose value is a [ThickenFeatures](../ThickenFeatures/ThickenFeatures.md).

## Version

Introduced in version 6
