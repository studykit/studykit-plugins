# PartFeatures.GrillFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Gets the collection object that lists the subset of part features that are GrillFeatures.

## Syntax

PartFeatures.**GrillFeatures**() As [GrillFeatures](../GrillFeatures/GrillFeatures.md)

## Property Value

This is a read only property whose value is a [GrillFeatures](../GrillFeatures/GrillFeatures.md).

## Version

Introduced in version 2010
