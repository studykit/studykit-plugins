# PartFeatures.ThreadFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the ThreadFeatures collection object. This collection provides access to existing ThreadFeature objects and provides functionality to create new ThreadFeature objects.

## Syntax

PartFeatures.**ThreadFeatures**() As [ThreadFeatures](../ThreadFeatures/ThreadFeatures.md)

## Property Value

This is a read only property whose value is a [ThreadFeatures](../ThreadFeatures/ThreadFeatures.md).
