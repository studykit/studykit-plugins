# PartFeatures.EmbossFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the EmbossFeatures collection object. This collection provides access to existing EmbossFeature objects.

## Syntax

PartFeatures.**EmbossFeatures**() As [EmbossFeatures](../EmbossFeatures/EmbossFeatures.md)

## Property Value

This is a read only property whose value is an [EmbossFeatures](../EmbossFeatures/EmbossFeatures.md).

## Version

Introduced in version 6
