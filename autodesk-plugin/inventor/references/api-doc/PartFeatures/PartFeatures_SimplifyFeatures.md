# PartFeatures.SimplifyFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Read-only property that returns the SimplifyFeatures collection object.

## Syntax

PartFeatures.**SimplifyFeatures**() As [SimplifyFeatures](../SimplifyFeatures/SimplifyFeatures.md)

## Property Value

This is a read only property whose value is a [SimplifyFeatures](../SimplifyFeatures/SimplifyFeatures.md).

## Version

Introduced in version 2026
