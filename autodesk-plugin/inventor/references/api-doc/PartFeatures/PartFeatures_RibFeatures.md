# PartFeatures.RibFeatures Property

Parent Object: [PartFeatures](../PartFeatures/PartFeatures.md)

## Description

Property that returns the RibFeatures collection object. This collection provides access to existing RibFeature objects.

## Syntax

PartFeatures.**RibFeatures**() As [RibFeatures](../RibFeatures/RibFeatures.md)

## Property Value

This is a read only property whose value is a [RibFeatures](../RibFeatures/RibFeatures.md).

## Version

Introduced in version 5
