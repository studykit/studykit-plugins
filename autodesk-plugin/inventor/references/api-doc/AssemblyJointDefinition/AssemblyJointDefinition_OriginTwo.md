# AssemblyJointDefinition.OriginTwo Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-write property that gets and sets the second origin associated with the assembly joint object.

## Syntax

AssemblyJointDefinition.**OriginTwo**() As [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Property Value

This is a read/write property whose value is a [GeometryIntent](../GeometryIntent/GeometryIntent.md).

## Version

Introduced in version 2014
