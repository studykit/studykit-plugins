# AssemblyJointDefinition.JointType Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-write property that gets and sets the type of AssemblyJoint.

## Syntax

AssemblyJointDefinition.**JointType**() As [AssemblyJointTypeEnum](../AssemblyJointTypeEnum.md)

## Property Value

This is a read/write property whose value is an [AssemblyJointTypeEnum](../AssemblyJointTypeEnum.md).

## Version

Introduced in version 2014
