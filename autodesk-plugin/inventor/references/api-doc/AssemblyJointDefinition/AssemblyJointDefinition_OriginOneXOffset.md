# AssemblyJointDefinition.OriginOneXOffset Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-only property that returns the offset of the OriginOne along origin X-axis direction. This returns Empty if the OriginOneDefinitonType is not equal to kOffsetOriginDefinitionType.

## Syntax

AssemblyJointDefinition.**OriginOneXOffset**() As Variant

## Property Value

This is a read only property whose value is a Variant.

## Version

Introduced in version 2016
