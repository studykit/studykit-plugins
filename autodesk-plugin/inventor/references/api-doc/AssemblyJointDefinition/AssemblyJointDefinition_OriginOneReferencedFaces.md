# AssemblyJointDefinition.OriginOneReferencedFaces Property

Parent Object: [AssemblyJointDefinition](../AssemblyJointDefinition/AssemblyJointDefinition.md)

## Description

Read-only property that returns the two referenced planar faces to define the projecting plane for OriginOne. This returns Nothing if the OriginOneDefinitonType is not equal to kBetweenTwoFacesOriginDefinitionType.

## Syntax

AssemblyJointDefinition.**OriginOneReferencedFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read only property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2016
