# RipFeatures Object

## Description

The RipFeatures collection object provides access to the RipFeature objects

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../RipFeatures/RipFeatures_Add.md) | Method that creates a new rip feature. The newly created RipFeature object is returned. |
| [CreateRipDefinition](../RipFeatures/RipFeatures_CreateRipDefinition.md) | Method that creates a new RipDefinition object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../RipFeatures/RipFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../RipFeatures/RipFeatures_Count.md) | Property that returns the number of items in this collection. |
| [Item](../RipFeatures/RipFeatures_Item.md) | Returns the specified RipFeature object from the collection. |
| [Type](../RipFeatures/RipFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
