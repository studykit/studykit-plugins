# iFeatureTable.Delete Method

Parent Object: [iFeatureTable](../iFeatureTable/iFeatureTable.md)

## Description

Method that deletes the iFeatureTable. This converts a table driven iFeature into a standard iFeature.

## Syntax

iFeatureTable.**Delete**()

## Version

Introduced in version 2010
