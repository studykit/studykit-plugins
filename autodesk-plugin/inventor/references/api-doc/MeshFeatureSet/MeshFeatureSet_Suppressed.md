# MeshFeatureSet.Suppressed Property

Parent Object: [MeshFeatureSet](../MeshFeatureSet/MeshFeatureSet.md)

## Description

Read-write property that gets and sets the whether all of the mesh features within this set are suppressed or not.

## Syntax

MeshFeatureSet.**Suppressed**() As [MeshFeaturesSuppressionEnum](../MeshFeaturesSuppressionEnum.md)

## Property Value

This is a read/write property whose value is a [MeshFeaturesSuppressionEnum](../MeshFeaturesSuppressionEnum.md).

## Version

Introduced in version 2017
