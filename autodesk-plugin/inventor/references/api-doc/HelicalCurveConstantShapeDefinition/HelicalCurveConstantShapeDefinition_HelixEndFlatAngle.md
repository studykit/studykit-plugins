# HelicalCurveConstantShapeDefinition.HelixEndFlatAngle Property

Parent Object: [HelicalCurveConstantShapeDefinition](../HelicalCurveConstantShapeDefinition/HelicalCurveConstantShapeDefinition.md)

## Description

Gets and sets the helix end flat angle.

## Syntax

HelicalCurveConstantShapeDefinition.**HelixEndFlatAngle**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2019
