# PresentationComponent.Name Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that returns the presentation component name.

## Syntax

PresentationComponent.**Name**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
