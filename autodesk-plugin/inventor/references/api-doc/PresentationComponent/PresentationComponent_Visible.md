# PresentationComponent.Visible Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that gets the visibility of the presentation component.

## Syntax

PresentationComponent.**Visible**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2018
