# PresentationComponent.Appearance Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that gets and sets the current appearance for this presentation component.

## Syntax

PresentationComponent.**Appearance**() As [Asset](../Asset/Asset.md)

## Property Value

This is a read only property whose value is an [Asset](../Asset/Asset.md).

## Version

Introduced in version 2018
