# PresentationComponent.Opacity Property

Parent Object: [PresentationComponent](../PresentationComponent/PresentationComponent.md)

## Description

Read-only property that gets the opacity of this presentation component.

## Syntax

PresentationComponent.**Opacity**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2018
