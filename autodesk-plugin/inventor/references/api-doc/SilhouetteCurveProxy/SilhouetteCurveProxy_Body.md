# SilhouetteCurveProxy.Body Property

Parent Object: [SilhouetteCurveProxy](../SilhouetteCurveProxy/SilhouetteCurveProxy.md)

## Description

Read-write property that defines the surfaces the silhouette is calculated for.

## Syntax

SilhouetteCurveProxy.**Body**() As [SurfaceBody](../SurfaceBody/SurfaceBody.md)

## Property Value

This is a read/write property whose value is a [SurfaceBody](../SurfaceBody/SurfaceBody.md).

## Version

Introduced in version 2021
