# SilhouetteCurveProxy.BreakLink Method

Parent Object: [SilhouetteCurveProxy](../SilhouetteCurveProxy/SilhouetteCurveProxy.md)

## Description

Method that breaks the link between the silhouette curve and the model.

## Syntax

SilhouetteCurveProxy.**BreakLink**()

## Version

Introduced in version 2012
