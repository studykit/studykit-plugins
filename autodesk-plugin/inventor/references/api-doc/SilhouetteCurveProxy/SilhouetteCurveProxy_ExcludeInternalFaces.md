# SilhouetteCurveProxy.ExcludeInternalFaces Property

Parent Object: [SilhouetteCurveProxy](../SilhouetteCurveProxy/SilhouetteCurveProxy.md)

## Description

Read-write property that specifies whether to exclude the internal faces from the silhouette curve projection.

## Syntax

SilhouetteCurveProxy.**ExcludeInternalFaces**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2021
