# SilhouetteCurveProxy.ExcludedFaces Property

Parent Object: [SilhouetteCurveProxy](../SilhouetteCurveProxy/SilhouetteCurveProxy.md)

## Description

Read-write property that gets and sets the faces that are excluded in the silhouette curve projection.

## Syntax

SilhouetteCurveProxy.**ExcludedFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2021
