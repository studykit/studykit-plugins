# RadiusDimConstraintProxy.Entity Property

Parent Object: [RadiusDimConstraintProxy](../RadiusDimConstraintProxy/RadiusDimConstraintProxy.md)

## Description

Property that returns the entity being constrained to.

## Syntax

RadiusDimConstraintProxy.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
