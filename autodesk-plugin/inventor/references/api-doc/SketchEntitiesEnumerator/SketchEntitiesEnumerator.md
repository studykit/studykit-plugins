# SketchEntitiesEnumerator Object

## Description

The SketchEntitiesEnumerator object provides access to all of the geometric entities on the sketch regardless of their type. See the article in the overviews section.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../SketchEntitiesEnumerator/SketchEntitiesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../SketchEntitiesEnumerator/SketchEntitiesEnumerator_Item.md) | Returns the specified SketchEntity object from the collection. |
| [Type](../SketchEntitiesEnumerator/SketchEntitiesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
