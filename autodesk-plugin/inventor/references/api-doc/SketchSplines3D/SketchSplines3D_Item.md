# SketchSplines3D.Item Property

Parent Object: [SketchSplines3D](../SketchSplines3D/SketchSplines3D.md)

## Description

Returns the specified SketchSpline3D object from the collection.

## Syntax

SketchSplines3D.**Item**( ***Index*** As Long ) As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
