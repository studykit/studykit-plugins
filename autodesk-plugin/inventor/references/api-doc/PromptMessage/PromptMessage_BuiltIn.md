# PromptMessage.BuiltIn Property

Parent Object: [PromptMessage](../PromptMessage/PromptMessage.md)

## Description

Read-only property that returns whether the prompt message is built-in or not.

## Syntax

PromptMessage.**BuiltIn**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2025.1
