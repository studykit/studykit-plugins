# PromptMessage Object

## Description

PromptMessage object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../PromptMessage/PromptMessage_Delete.md) | Method that deletes this prompt message. This is applicable for custom prompt messages. |
| [Display](../PromptMessage/PromptMessage_Display.md) | Method that displays this prompt message. This is applicable for custom prompt messages. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PromptMessage/PromptMessage_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [BuiltIn](../PromptMessage/PromptMessage_BuiltIn.md) | Read-only property that returns whether the prompt message is built-in or not. |
| [Buttons](../PromptMessage/PromptMessage_Buttons.md) | Read-only property that returns the buttons that are used for the prompt message. |
| [DefaultAnswer](../PromptMessage/PromptMessage_DefaultAnswer.md) | Read-write property that gets and sets the default response for the prompt message. |
| [DisplayedPromptText](../PromptMessage/PromptMessage_DisplayedPromptText.md) | Read-only property that returns the displayed prompt text of the prompt message. |
| [Icon](../PromptMessage/PromptMessage_Icon.md) | Read-only property that returns the icon type of the prompt message. Values that can be used: vbCritical(16), vbExclamation(48), vbInformation(64), vbQuestion(32). |
| [Id](../PromptMessage/PromptMessage_Id.md) | Read-only property that returns the Id of the prompt message. |
| [PromptText](../PromptMessage/PromptMessage_PromptText.md) | Read-only property that returns the prompt text of the prompt message. |
| [Restrictions](../PromptMessage/PromptMessage_Restrictions.md) | Read-write property that gets and sets the restrictions for the prompt. |
| [Title](../PromptMessage/PromptMessage_Title.md) | Read-only property that returns the title of the prompt message. |
| [Type](../PromptMessage/PromptMessage_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[PromptMessages.Add](../PromptMessages/PromptMessages_Add.md), [PromptMessages.Item](../PromptMessages/PromptMessages_Item.md)
