# GraphicsColorSet.Color Property

Parent Object: [GraphicsColorSet](../GraphicsColorSet/GraphicsColorSet.md)

## Description

Property that gets and sets the color at a given index in the set.

## Syntax

GraphicsColorSet.**Color**( ***Index*** As Long ) As [Color](../Color/Color.md)

## Property Value

This is a read/write property whose value is a [Color](../Color/Color.md).

| --- | --- | --- |
| Name | Type | Description |
