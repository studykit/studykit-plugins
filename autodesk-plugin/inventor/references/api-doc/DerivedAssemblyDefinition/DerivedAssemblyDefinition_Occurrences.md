# DerivedAssemblyDefinition.Occurrences Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Property that gets the available in the selected assembly. This returns the top-level occurrences in the assembly. You can use the SubOccurrences of the DerivedAssemblyOccurrence object to traverse through the entire occurrence tree.

## Syntax

DerivedAssemblyDefinition.**Occurrences**() As [DerivedAssemblyOccurrences](../DerivedAssemblyOccurrences/DerivedAssemblyOccurrences.md)

## Property Value

This is a read only property whose value is a [DerivedAssemblyOccurrences](../DerivedAssemblyOccurrences/DerivedAssemblyOccurrences.md).

## Version

Introduced in version 5.3
