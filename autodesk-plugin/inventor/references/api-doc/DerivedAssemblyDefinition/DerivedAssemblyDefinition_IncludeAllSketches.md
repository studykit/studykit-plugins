# DerivedAssemblyDefinition.IncludeAllSketches Method

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Method that includes all sketches from the source assembly. This includes top level sketches in the source assembly as well as sketches in sub-assemblies and parts.

## Syntax

DerivedAssemblyDefinition.**IncludeAllSketches**()

## Version

Introduced in version 2011
