# DerivedAssemblyDefinition.IncludeAllTopLevelWorkFeatures Property

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Read-write property that defines whether all top level work features in the source assembly are included in the derived assembly.

## Syntax

DerivedAssemblyDefinition.**IncludeAllTopLevelWorkFeatures**() As [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md)

## Property Value

This is a read/write property whose value is a [DerivedComponentOptionEnum](../DerivedComponentOptionEnum.md).
