# DerivedAssemblyDefinition.IncludeAllWorkFeatures Method

Parent Object: [DerivedAssemblyDefinition](../DerivedAssemblyDefinition/DerivedAssemblyDefinition.md)

## Description

Method that includes all work features from the source assembly. This includes top level work features in the source assembly as well as work features in sub-assemblies and parts.

## Syntax

DerivedAssemblyDefinition.**IncludeAllWorkFeatures**()

## Version

Introduced in version 2011
