# CornerRoundFeatureProxy.Parent Property

Parent Object: [CornerRoundFeatureProxy](../CornerRoundFeatureProxy/CornerRoundFeatureProxy.md)

## Description

Property that returns the parent of the feature.

## Syntax

CornerRoundFeatureProxy.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 9
