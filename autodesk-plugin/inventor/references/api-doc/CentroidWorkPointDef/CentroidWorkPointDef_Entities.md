# CentroidWorkPointDef.Entities Property

Parent Object: [CentroidWorkPointDef](../CentroidWorkPointDef/CentroidWorkPointDef.md)

## Description

Property that returns either an Edge, an EdgeLoop or an EdgeCollection object.

## Syntax

CentroidWorkPointDef.**Entities**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2008
