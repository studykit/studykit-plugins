# SketchEllipseProxy.Length Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Gets the length of the ellipse.

## Syntax

SketchEllipseProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2010
