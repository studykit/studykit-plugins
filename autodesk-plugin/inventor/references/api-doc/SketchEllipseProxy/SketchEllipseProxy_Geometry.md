# SketchEllipseProxy.Geometry Property

Parent Object: [SketchEllipseProxy](../SketchEllipseProxy/SketchEllipseProxy.md)

## Description

Property that returns an EllipseFull2d geometry object. The object returned represents a snapshot view of the current state of the sketch ellipse.

## Syntax

SketchEllipseProxy.**Geometry**() As [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md)

## Property Value

This is a read only property whose value is an [EllipseFull2d](../EllipseFull2d/EllipseFull2d.md).

## Version

Introduced in version 6
