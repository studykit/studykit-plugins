# ColorSchemes Object

## Description

The ColorSchemes object provides access to all of the existing objects. This is somewhat equivalent to the Color tab of the Application Options dialog.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ColorSchemes/ColorSchemes_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [BackgroundType](../ColorSchemes/ColorSchemes_BackgroundType.md) | Gets and sets the type of background. |
| [Count](../ColorSchemes/ColorSchemes_Count.md) | Property that returns the number of items in the collection. |
| [EnableEnhancedHighlighting](../ColorSchemes/ColorSchemes_EnableEnhancedHighlighting.md) | Gets and sets whether the enhanced highlighting option is enabled. |
| [EnablePrehighlight](../ColorSchemes/ColorSchemes_EnablePrehighlight.md) | Gets and sets whether enables or disables pre-highlighting during selection. |
| [HighlightHiddenFaces](../ColorSchemes/ColorSchemes_HighlightHiddenFaces.md) | Gets and sets whether enables or disables hidden faces highlighting. |
| [IconsColor](../ColorSchemes/ColorSchemes_IconsColor.md) | Gets and sets the color theme for the icons in the ribbon interface. |
| [Item](../ColorSchemes/ColorSchemes_Item.md) | Returns the specified ColorScheme object from the collection. |
| [Type](../ColorSchemes/ColorSchemes_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Application.ColorSchemes](../Application/Application_ColorSchemes.md)

## Version

Introduced in version 9
