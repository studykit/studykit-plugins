# ColorSchemes.BackgroundType Property

Parent Object: [ColorSchemes](../ColorSchemes/ColorSchemes.md)

## Description

Gets and sets the type of background.

## Syntax

ColorSchemes.**BackgroundType**() As [BackgroundTypeEnum](../BackgroundTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BackgroundTypeEnum](../BackgroundTypeEnum.md).

## Version

Introduced in version 10
