# CameraEvents.Type Property

Parent Object: [CameraEvents](../CameraEvents/CameraEvents.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CameraEvents.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 2011
