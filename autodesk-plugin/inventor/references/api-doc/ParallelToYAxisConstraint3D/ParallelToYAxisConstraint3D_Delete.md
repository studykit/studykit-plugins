# ParallelToYAxisConstraint3D.Delete Method

Parent Object: [ParallelToYAxisConstraint3D](../ParallelToYAxisConstraint3D/ParallelToYAxisConstraint3D.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

ParallelToYAxisConstraint3D.**Delete**()

## Version

Introduced in version 2017
