# DerivedFuturePartComponents.Parent Property

Parent Object: [DerivedFuturePartComponents](../DerivedFuturePartComponents/DerivedFuturePartComponents.md)

## Description

Read-only property that returns the ComponentDefinition object.

## Syntax

DerivedFuturePartComponents.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2018
