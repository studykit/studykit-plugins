# AssemblyJoint.Parent Property

Parent Object: [AssemblyJoint](../AssemblyJoint/AssemblyJoint.md)

## Description

Read-only property that returns the parent AssemblyComponentDefinition object.

## Syntax

AssemblyJoint.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).

## Version

Introduced in version 2014
