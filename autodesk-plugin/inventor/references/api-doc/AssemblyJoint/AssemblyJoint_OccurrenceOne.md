# AssemblyJoint.OccurrenceOne Property

Parent Object: [AssemblyJoint](../AssemblyJoint/AssemblyJoint.md)

## Description

Read-only property that returns the first of the two components this assembly joint is between.

## Syntax

AssemblyJoint.**OccurrenceOne**() As [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md)

## Property Value

This is a read only property whose value is a [ComponentOccurrence](../ComponentOccurrence/ComponentOccurrence.md).

## Version

Introduced in version 2014
