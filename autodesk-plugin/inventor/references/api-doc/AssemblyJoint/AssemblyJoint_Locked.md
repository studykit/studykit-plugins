# AssemblyJoint.Locked Property

Parent Object: [AssemblyJoint](../AssemblyJoint/AssemblyJoint.md)

## Description

Read-write property that gets and sets whether the degree of freedom of the assembly joint is locked or not.

## Syntax

AssemblyJoint.**Locked**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
