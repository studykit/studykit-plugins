# UserInterfaceManager.Environments Property

Parent Object: [UserInterfaceManager](../UserInterfaceManager/UserInterfaceManager.md)

## Description

Property that returns the Environments collection object.

## Syntax

UserInterfaceManager.**Environments**() As [Environments](../Environments/Environments.md)

## Property Value

This is a read only property whose value is an [Environments](../Environments/Environments.md).
