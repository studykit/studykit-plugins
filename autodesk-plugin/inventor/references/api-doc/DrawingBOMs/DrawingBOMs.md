# DrawingBOMs Object

## Description

The DrawingBOMs collection object returns the collection of all locally stored BOMs in the drawing document.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [IsDrawingBOMDefined](../DrawingBOMs/DrawingBOMs_IsDrawingBOMDefined.md) | Method that returns whether BOM properties have been defined in the drawing for the input model. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DrawingBOMs/DrawingBOMs_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../DrawingBOMs/DrawingBOMs_Count.md) | Property that returns the number of items in the collection. |
| [Item](../DrawingBOMs/DrawingBOMs_Item.md) | Returns the specified DrawingBOM object from the collection. |
| [Type](../DrawingBOMs/DrawingBOMs_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingDocument.DrawingBOMs](../DrawingDocument/DrawingDocument_DrawingBOMs.md)

## Version

Introduced in version 2009
