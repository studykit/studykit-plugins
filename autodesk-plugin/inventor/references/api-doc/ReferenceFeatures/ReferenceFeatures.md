# ReferenceFeatures Object

## Description

The ReferenceFeatures collection object provides access to all of the existing objects in a part component definition.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ReferenceFeatures/ReferenceFeatures_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ReferenceFeatures/ReferenceFeatures_Count.md) | Property that returns the number of items in the collection. |
| [Item](../ReferenceFeatures/ReferenceFeatures_Item.md) | Returns the specified ReferenceFeature object from the collection. |
| [Type](../ReferenceFeatures/ReferenceFeatures_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[PartFeatures.ReferenceFeatures](../PartFeatures/PartFeatures_ReferenceFeatures.md), [SheetMetalFeatures.ReferenceFeatures](../SheetMetalFeatures/SheetMetalFeatures_ReferenceFeatures.md)

## Version

Introduced in version 5.3
