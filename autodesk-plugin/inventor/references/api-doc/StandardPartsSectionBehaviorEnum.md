# StandardPartsSectionBehaviorEnum Enumerator

## Description

Enum indicating the sectioning behavior for standard parts when section views are created in the drawing.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kAlwaysSectionStandardParts | 66561 | Standard parts should always be sectioned irrespective of the settings. |
| kNeverSectionStandardParts | 66562 | Standard parts should never be sectioned irrespective of the settings. |
| kObeyBrowserSettingsSectionStandardParts | 66563 | The drawing browser settings should be obeyed when sectioning standard parts. |

## Version

Introduced in version 2008
