# ControlDefinition.DisplayName Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Property that returns the display name of the ControlDefinition.

## Syntax

ControlDefinition.**DisplayName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 7
