# ControlDefinition.AutoAddToGUI Method

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Method that automatically adds a control based on this definition to the General panel of the Add-Ins tab in the ribbon interface.

## Syntax

ControlDefinition.**AutoAddToGUI**()

## Version

Introduced in version 6
