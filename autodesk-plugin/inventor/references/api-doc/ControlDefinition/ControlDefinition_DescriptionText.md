# ControlDefinition.DescriptionText Property

Parent Object: [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Description

Gets/Sets Description Text. For MacroControlDefinition this property is read-only.

## Syntax

ControlDefinition.**DescriptionText**() As String

## Property Value

This is a read/write property whose value is a String.
