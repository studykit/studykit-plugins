# OnFaceConstraint3D.Face Property

Parent Object: [OnFaceConstraint3D](../OnFaceConstraint3D/OnFaceConstraint3D.md)

## Description

Read-only property that returns the Face which the 3D sketch entity is constrained to.

## Syntax

OnFaceConstraint3D.**Face**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2017
