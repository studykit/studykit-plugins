# InsertConstraintProxy.NativeObject Property

Parent Object: [InsertConstraintProxy](../InsertConstraintProxy/InsertConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

InsertConstraintProxy.**NativeObject**() As [InsertConstraint](../InsertConstraint/InsertConstraint.md)

## Property Value

This is a read only property whose value is an [InsertConstraint](../InsertConstraint/InsertConstraint.md).

## Version

Introduced in version 11
