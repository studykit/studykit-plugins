# ModelLeaderNoteDefinition.ArrowheadType Property

Parent Object: [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md)

## Description

Read-write property that gets and sets the arrowhead type of the leader.

## Syntax

ModelLeaderNoteDefinition.**ArrowheadType**() As [ArrowheadTypeEnum](../ArrowheadTypeEnum.md)

## Property Value

This is a read/write property whose value is an [ArrowheadTypeEnum](../ArrowheadTypeEnum.md).

## Version

Introduced in version 2018
