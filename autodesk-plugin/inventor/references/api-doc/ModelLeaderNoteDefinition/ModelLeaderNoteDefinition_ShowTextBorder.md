# ModelLeaderNoteDefinition.ShowTextBorder Property

Parent Object: [ModelLeaderNoteDefinition](../ModelLeaderNoteDefinition/ModelLeaderNoteDefinition.md)

## Description

Read-write property that gets and sets whether to show the text border or not.

## Syntax

ModelLeaderNoteDefinition.**ShowTextBorder**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2023
