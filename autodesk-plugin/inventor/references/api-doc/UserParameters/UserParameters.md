# UserParameters Object

## Description

Provides access to the user parameters (  objects) associated with the object the collection was obtained from.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddByExpression](../UserParameters/UserParameters_AddByExpression.md) | Method that creates a new parameter given a name and expression. |
| [AddByValue](../UserParameters/UserParameters_AddByValue.md) | Method that creates a new parameter given a name and equation. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Count](../UserParameters/UserParameters_Count.md) | Property that returns the number of items in this collection. |
| [Item](../UserParameters/UserParameters_Item.md) | Returns the specified UserParameter object from the collection. This is the default property of the UserParameters collection object. |
| [Type](../UserParameters/UserParameters_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
