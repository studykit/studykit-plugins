# ObjectDefaultsStylesEnumerator Object

## Description

The ObjectDefaultsStylesEnumerator object allows iteration through the collection.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../ObjectDefaultsStylesEnumerator/ObjectDefaultsStylesEnumerator_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../ObjectDefaultsStylesEnumerator/ObjectDefaultsStylesEnumerator_Count.md) | Property that returns the number of items in this collection. |
| [Item](../ObjectDefaultsStylesEnumerator/ObjectDefaultsStylesEnumerator_Item.md) | Returns the specified object in the collection. |
| [Type](../ObjectDefaultsStylesEnumerator/ObjectDefaultsStylesEnumerator_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingStylesManager.ObjectDefaultsStyles](../DrawingStylesManager/DrawingStylesManager_ObjectDefaultsStyles.md)

## Version

Introduced in version 9
