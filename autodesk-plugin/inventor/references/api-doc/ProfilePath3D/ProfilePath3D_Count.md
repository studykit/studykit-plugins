# ProfilePath3D.Count Property

Parent Object: [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Description

Property that returns the number of items in the collection.

## Syntax

ProfilePath3D.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 6
