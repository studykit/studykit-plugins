# ProfilePath3D.Item Property

Parent Object: [ProfilePath3D](../ProfilePath3D/ProfilePath3D.md)

## Description

Method that returns the specified ProfileEntity3D object from the collection.

## Syntax

ProfilePath3D.**Item**( ***Index*** As Long ) As [ProfileEntity3D](../ProfileEntity3D/ProfileEntity3D.md)

## Property Value

This is a read only property whose value is a [ProfileEntity3D](../ProfileEntity3D/ProfileEntity3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
