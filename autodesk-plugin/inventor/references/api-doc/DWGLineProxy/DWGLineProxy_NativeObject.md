# DWGLineProxy.NativeObject Property

Parent Object: [DWGLineProxy](../DWGLineProxy/DWGLineProxy.md)

## Description

Get the object in the context of the definition instead of the containing assembly.

## Syntax

DWGLineProxy.**NativeObject**() As [DWGLine](../DWGLine/DWGLine.md)

## Property Value

This is a read only property whose value is a [DWGLine](../DWGLine/DWGLine.md).

## Version

Introduced in version 2016
