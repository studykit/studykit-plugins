# DWGLineProxy Object

Derived from: [DWGLine](../DWGLine/DWGLine.md) Object

## Description

DWGLineProxy Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../DWGLineProxy/DWGLineProxy_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [ContainingOccurrence](../DWGLineProxy/DWGLineProxy_ContainingOccurrence.md) | Get the component occurrence context through which the native object is being seen through. |
| [Direction](../DWGLineProxy/DWGLineProxy_Direction.md) | Read-only property that returns the UnitVector indicating the direction of the line. |
| [EndPoint](../DWGLineProxy/DWGLineProxy_EndPoint.md) | Read-only property that returns the point that defines the position of the end of the line. |
| [HandleID](../DWGLineProxy/DWGLineProxy_HandleID.md) | Read-only property that returns the HandleID for the DWGEntity. |
| [NativeObject](../DWGLineProxy/DWGLineProxy_NativeObject.md) | Get the object in the context of the definition instead of the containing assembly. |
| [Parent](../DWGLineProxy/DWGLineProxy_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [ParentEntity](../DWGLineProxy/DWGLineProxy_ParentEntity.md) | Read-only property that returns the parent DWGEntity of the entity. |
| [StartPoint](../DWGLineProxy/DWGLineProxy_StartPoint.md) | Read-only property that returns the point that defines the position of the start of the line. |
| [Type](../DWGLineProxy/DWGLineProxy_Type.md) | Gets the constant that indicates the type of this object. |

## Version

Introduced in version 2016
