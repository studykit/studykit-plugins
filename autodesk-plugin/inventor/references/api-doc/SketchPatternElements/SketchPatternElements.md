# SketchPatternElements Object

## Description

SketchPatternElements Collection Object.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchPatternElements/SketchPatternElements_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [Count](../SketchPatternElements/SketchPatternElements_Count.md) | Gets the number of items in this collection. |
| [Item](../SketchPatternElements/SketchPatternElements_Item.md) | Allows integer-indexed access to items in the collection. |
| [Type](../SketchPatternElements/SketchPatternElements_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[SketchCircularPattern.Elements](../SketchCircularPattern/SketchCircularPattern_Elements.md), [SketchRectangularPattern.Elements](../SketchRectangularPattern/SketchRectangularPattern_Elements.md)

## Version

Introduced in version 2025.1
