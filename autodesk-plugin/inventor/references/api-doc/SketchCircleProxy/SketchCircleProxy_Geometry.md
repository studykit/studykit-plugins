# SketchCircleProxy.Geometry Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Property that returns a Circle2d geometry object. The object returned represents a snapshot view of the current state of the sketch circle.

## Syntax

SketchCircleProxy.**Geometry**() As [Circle2d](../Circle2d/Circle2d.md)

## Property Value

This is a read only property whose value is a [Circle2d](../Circle2d/Circle2d.md).

## Version

Introduced in version 6
