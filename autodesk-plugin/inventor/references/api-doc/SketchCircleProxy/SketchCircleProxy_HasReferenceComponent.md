# SketchCircleProxy.HasReferenceComponent Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchCircleProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
