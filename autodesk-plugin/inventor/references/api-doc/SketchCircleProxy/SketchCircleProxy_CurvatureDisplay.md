# SketchCircleProxy.CurvatureDisplay Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Specifies whether curvature information is displayed for the circle.

## Syntax

SketchCircleProxy.**CurvatureDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
