# SketchCircleProxy.Radius Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Gets and sets the radius of the circle.

## Syntax

SketchCircleProxy.**Radius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 6
