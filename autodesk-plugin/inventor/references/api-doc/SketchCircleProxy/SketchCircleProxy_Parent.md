# SketchCircleProxy.Parent Property

Parent Object: [SketchCircleProxy](../SketchCircleProxy/SketchCircleProxy.md)

## Description

Property that returns the parent sketch of the entity.

## Syntax

SketchCircleProxy.**Parent**() As [Sketch](../Sketch/Sketch.md)

## Property Value

This is a read only property whose value is a [Sketch](../Sketch/Sketch.md).

## Version

Introduced in version 6
