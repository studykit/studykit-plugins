# ThreadInfo.ThreadDesignation Property

Parent Object: [ThreadInfo](../ThreadInfo/ThreadInfo.md)

## Description

Property that returns a string that contains the thread designation. This is the full thread designation that is used in a drawing for the thread callout.

## Syntax

ThreadInfo.**ThreadDesignation**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 9
