# UnfoldMethod.ConvertToLocal Method

Parent Object: [UnfoldMethod](../UnfoldMethod/UnfoldMethod.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

UnfoldMethod.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2009
