# SurfaceGraphicsEdge.Edge Property

Parent Object: [SurfaceGraphicsEdge](../SurfaceGraphicsEdge/SurfaceGraphicsEdge.md)

## Description

Property that returns the Edge object associated with the SurfaceGraphicsEdge.

## Syntax

SurfaceGraphicsEdge.**Edge**() As [Edge](../Edge/Edge.md)

## Property Value

This is a read only property whose value is an [Edge](../Edge/Edge.md).

## Version

Introduced in version 2009
