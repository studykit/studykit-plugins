# SurfaceGraphicsEdge.Index Property

Parent Object: [SurfaceGraphicsEdge](../SurfaceGraphicsEdge/SurfaceGraphicsEdge.md)

## Description

Property that returns the index of the SurfaceGraphicsEdge object within the SurfaceGraphicsEdgeList.

## Syntax

SurfaceGraphicsEdge.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2010
