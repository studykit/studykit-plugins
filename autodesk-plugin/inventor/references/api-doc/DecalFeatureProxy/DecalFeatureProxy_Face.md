# DecalFeatureProxy.Face Property

Parent Object: [DecalFeatureProxy](../DecalFeatureProxy/DecalFeatureProxy.md)

## Description

Gets and sets the primary face on which the decal is applied.

## Syntax

DecalFeatureProxy.**Face**() As [Face](../Face/Face.md)

## Property Value

This is a read/write property whose value is a [Face](../Face/Face.md).

## Version

Introduced in version 2008
