# CustomTable.Style Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the style associated with this object.

## Syntax

CustomTable.**Style**() As [TableStyle](../TableStyle/TableStyle.md)

## Property Value

This is a read/write property whose value is a [TableStyle](../TableStyle/TableStyle.md).

## Version

Introduced in version 2011
