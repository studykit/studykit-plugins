# CustomTable.RowLineSpacing Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the spacing between the rows.

## Syntax

CustomTable.**RowLineSpacing**() As [LineSpacingEnum](../LineSpacingEnum.md)

## Property Value

This is a read/write property whose value is a [LineSpacingEnum](../LineSpacingEnum.md).

## Version

Introduced in version 2009
