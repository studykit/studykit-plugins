# CustomTable.TableSource Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Property that returns the source type for this table.

## Syntax

CustomTable.**TableSource**() As [TableSourceTypeEnum](../TableSourceTypeEnum.md)

## Property Value

This is a read only property whose value is a [TableSourceTypeEnum](../TableSourceTypeEnum.md).

## Version

Introduced in version 2009
