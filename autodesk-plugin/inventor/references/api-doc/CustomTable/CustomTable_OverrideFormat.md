# CustomTable.OverrideFormat Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the table format.

## Syntax

CustomTable.**OverrideFormat**() As [TableFormat](../TableFormat/TableFormat.md)

## Property Value

This is a read/write property whose value is a [TableFormat](../TableFormat/TableFormat.md).
