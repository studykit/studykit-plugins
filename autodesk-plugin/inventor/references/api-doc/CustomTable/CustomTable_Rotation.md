# CustomTable.Rotation Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the absolute rotation angle of the table in radians.

## Syntax

CustomTable.**Rotation**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2009
