# CustomTable.ShowTitle Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets whether to show the title of the table.

## Syntax

CustomTable.**ShowTitle**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
