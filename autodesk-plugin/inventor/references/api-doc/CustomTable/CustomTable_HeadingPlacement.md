# CustomTable.HeadingPlacement Property

Parent Object: [CustomTable](../CustomTable/CustomTable.md)

## Description

Gets and sets the placement position of the table heading.

## Syntax

CustomTable.**HeadingPlacement**() As [HeadingPlacementEnum](../HeadingPlacementEnum.md)

## Property Value

This is a read/write property whose value is a [HeadingPlacementEnum](../HeadingPlacementEnum.md).

## Version

Introduced in version 9
