# DraftAnalysis.Name Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Gets and sets the name of the draft analysis.

## Syntax

DraftAnalysis.**Name**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2009
