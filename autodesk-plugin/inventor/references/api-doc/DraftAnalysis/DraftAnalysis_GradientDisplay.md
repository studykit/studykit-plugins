# DraftAnalysis.GradientDisplay Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Gets/Sets whether the draft analysis results should be displayed as a color gradient.

## Syntax

DraftAnalysis.**GradientDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2009
