# DraftAnalysis.PullDirection Property

Parent Object: [DraftAnalysis](../DraftAnalysis/DraftAnalysis.md)

## Description

Gets/Sets the object that specifies the direction in which the draft is applied.

## Syntax

DraftAnalysis.**PullDirection**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2009
