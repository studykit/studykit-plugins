# UniqueOccurrencesBehaviorEnum Enumerator

## Description

Constants that define the special behaviors to assign to the occurrences provided by the UniqueOccurrences argument.

## Methods

|  |  |  |
| --- | --- | --- |
| Name | Value | Description |
| kLockDOF | 4 | Lock DOF. |
| kRetainDOF | 2 | Retain DOF. |
| kSinglePartRigidBody | 1 | Single Part Rigid Body. |

## Version

Introduced in version 2008
