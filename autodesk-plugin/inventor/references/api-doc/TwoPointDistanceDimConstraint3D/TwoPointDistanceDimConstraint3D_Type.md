# TwoPointDistanceDimConstraint3D.Type Property

Parent Object: [TwoPointDistanceDimConstraint3D](../TwoPointDistanceDimConstraint3D/TwoPointDistanceDimConstraint3D.md)

## Description

## Syntax

TwoPointDistanceDimConstraint3D.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
