# LinearModelDimensionProxy.NativeObject Property

Parent Object: [LinearModelDimensionProxy](../LinearModelDimensionProxy/LinearModelDimensionProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

LinearModelDimensionProxy.**NativeObject**() As [LinearModelDimension](../LinearModelDimension/LinearModelDimension.md)

## Property Value

This is a read only property whose value is a [LinearModelDimension](../LinearModelDimension/LinearModelDimension.md).

## Version

Introduced in version 2018
