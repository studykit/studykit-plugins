# InventorVBAProject.InventorVBAComponents Property

Parent Object: [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Description

Gets the InventorVBAComponents contained in this project.

## Syntax

InventorVBAProject.**InventorVBAComponents**() As [InventorVBAComponents](../InventorVBAComponents/InventorVBAComponents.md)

## Property Value

This is a read only property whose value is an [InventorVBAComponents](../InventorVBAComponents/InventorVBAComponents.md).

## Version

Introduced in version 6
