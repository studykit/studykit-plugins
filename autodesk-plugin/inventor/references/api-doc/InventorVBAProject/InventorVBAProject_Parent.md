# InventorVBAProject.Parent Property

Parent Object: [InventorVBAProject](../InventorVBAProject/InventorVBAProject.md)

## Description

Property that returns the parent object from whom this object can logically be reached.

## Syntax

InventorVBAProject.**Parent**() As [Application](../Application/Application.md)

## Property Value

This is a read only property whose value is an [Application](../Application/Application.md).

## Version

Introduced in version 6
