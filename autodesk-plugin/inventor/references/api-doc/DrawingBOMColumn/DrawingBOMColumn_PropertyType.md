# DrawingBOMColumn.PropertyType Property

Parent Object: [DrawingBOMColumn](../DrawingBOMColumn/DrawingBOMColumn.md)

## Description

Property that returns the property type associated with this column.

## Syntax

DrawingBOMColumn.**PropertyType**() As [PropertyTypeEnum](../PropertyTypeEnum.md)

## Property Value

This is a read only property whose value is a [PropertyTypeEnum](../PropertyTypeEnum.md).

## Version

Introduced in version 2009
