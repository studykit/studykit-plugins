# SearchBox.SearchBoxEvents Property

Parent Object: [SearchBox](../SearchBox/SearchBox.md)

## Description

Read-only property that returns a SearchBoxEvents object.

## Syntax

SearchBox.**SearchBoxEvents**() As [SearchBoxEvents](../SearchBoxEvents/SearchBoxEvents.md)

## Property Value

This is a read only property whose value is a [SearchBoxEvents](../SearchBoxEvents/SearchBoxEvents.md).

## Version

Introduced in version 2018
