# FaceDefinitions.Item Property

Parent Object: [FaceDefinitions](../FaceDefinitions/FaceDefinitions.md)

## Description

Property that returns an item from the collection. The index of the first item in the collection is 1.

## Syntax

FaceDefinitions.**Item**( ***Index*** As Long ) As [FaceDefinition](../FaceDefinition/FaceDefinition.md)

## Property Value

This is a read only property whose value is a [FaceDefinition](../FaceDefinition/FaceDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the FaceDefinition object within the collection to return. |

## Version

Introduced in version 2011
