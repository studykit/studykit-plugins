# DerivedFuturePartComponentProxy.SurfaceBodies Property

Parent Object: [DerivedFuturePartComponentProxy](../DerivedFuturePartComponentProxy/DerivedFuturePartComponentProxy.md)

## Description

Read-only property that returns the ReferenceFeature objects that represent the work surfaces that were created as a result of this derived part.

## Syntax

DerivedFuturePartComponentProxy.**SurfaceBodies**() As [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md)

## Property Value

This is a read only property whose value is a [ReferenceFeaturesEnumerator](../ReferenceFeaturesEnumerator/ReferenceFeaturesEnumerator.md).

## Version

Introduced in version 2018
