# ImportedDataExchangeComponent.Urn Property

Parent Object: [ImportedDataExchangeComponent](../ImportedDataExchangeComponent/ImportedDataExchangeComponent.md)

## Description

Read-only property that returns the DataExchange urn.

## Syntax

ImportedDataExchangeComponent.**Urn**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2023
