# MidpointConstraintProxy.NativeObject Property

Parent Object: [MidpointConstraintProxy](../MidpointConstraintProxy/MidpointConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

MidpointConstraintProxy.**NativeObject**() As [MidpointConstraint](../MidpointConstraint/MidpointConstraint.md)

## Property Value

This is a read only property whose value is a [MidpointConstraint](../MidpointConstraint/MidpointConstraint.md).

## Version

Introduced in version 6
