# CornerChamferFeature.Definition Property

Parent Object: [CornerChamferFeature](../CornerChamferFeature/CornerChamferFeature.md)

## Description

Property that gets the CornerChamferDefinition object associated with this fold feature.

## Syntax

CornerChamferFeature.**Definition**() As [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md)

## Property Value

This is a read only property whose value is a [CornerChamferDefinition](../CornerChamferDefinition/CornerChamferDefinition.md).

## Version

Introduced in version 2009
