# PrintManager.ColorMode Property

Parent Object: [PrintManager](../PrintManager/PrintManager.md)

## Description

Gets/Sets the color mode to be used for printing. Defaults to using kPrintGrayScale.

## Syntax

PrintManager.**ColorMode**() As [PrintColorModeEnum](../PrintColorModeEnum.md)

## Property Value

This is a read/write property whose value is a [PrintColorModeEnum](../PrintColorModeEnum.md).
