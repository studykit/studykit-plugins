# PrintManager.Orientation Property

Parent Object: [PrintManager](../PrintManager/PrintManager.md)

## Description

Gets/Sets the orientation of the printer. Defaults to the printer's default.

## Syntax

PrintManager.**Orientation**() As [PrintOrientationEnum](../PrintOrientationEnum.md)

## Property Value

This is a read/write property whose value is a [PrintOrientationEnum](../PrintOrientationEnum.md).
