# OccurrencePatterns Object

## Description

The OccurrencePatterns collection object provides access to the existing assembly patterns ( objects) and supports methods to create new patterns.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [AddCircularPattern](../OccurrencePatterns/OccurrencePatterns_AddCircularPattern.md) | Method that creates a new circular occurrence pattern of input component(s). |
| [AddFeatureBasedPattern](../OccurrencePatterns/OccurrencePatterns_AddFeatureBasedPattern.md) | Method that creates a new occurrence pattern of the input component(s) using a pattern feature to define the positions of the elements within the pattern. |
| [AddRectangularPattern](../OccurrencePatterns/OccurrencePatterns_AddRectangularPattern.md) | Method that creates a new rectangular occurrence pattern of input component(s). |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../OccurrencePatterns/OccurrencePatterns_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../OccurrencePatterns/OccurrencePatterns_Count.md) | Property that returns the number of items in the collection. |
| [Item](../OccurrencePatterns/OccurrencePatterns_Item.md) | Method that returns the specified OccurrencePattern object from the collection. |
| [Parent](../OccurrencePatterns/OccurrencePatterns_Parent.md) | Property that returns the parent of the object. |
| [Type](../OccurrencePatterns/OccurrencePatterns_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[AssemblyComponentDefinition.OccurrencePatterns](../AssemblyComponentDefinition/AssemblyComponentDefinition_OccurrencePatterns.md), [WeldmentComponentDefinition.OccurrencePatterns](../WeldmentComponentDefinition/WeldmentComponentDefinition_OccurrencePatterns.md)

## Version

Introduced in version 6
