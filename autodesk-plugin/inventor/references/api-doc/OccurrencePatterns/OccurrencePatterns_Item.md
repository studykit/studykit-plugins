# OccurrencePatterns.Item Property

Parent Object: [OccurrencePatterns](../OccurrencePatterns/OccurrencePatterns.md)

## Description

Method that returns the specified OccurrencePattern object from the collection.

## Syntax

OccurrencePatterns.**Item**( ***Index*** As Variant ) As [OccurrencePattern](../OccurrencePattern/OccurrencePattern.md)

## Property Value

This is a read only property whose value is an [OccurrencePattern](../OccurrencePattern/OccurrencePattern.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the OccurrencePattern to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the pattern name. This is the name that is displayed to the user in the assembly browser. If an out of range index or a name of a non-existent pattern name is provided, an error occurs. |

## Version

Introduced in version 6
