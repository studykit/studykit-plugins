# Centermark.CentermarkType Property

Parent Object: [Centermark](../Centermark/Centermark.md)

## Description

Property that gets the centermark type.

## Syntax

Centermark.**CentermarkType**() As [CentermarkTypeEnum](../CentermarkTypeEnum.md)

## Property Value

This is a read only property whose value is a [CentermarkTypeEnum](../CentermarkTypeEnum.md).

## Version

Introduced in version 2008
