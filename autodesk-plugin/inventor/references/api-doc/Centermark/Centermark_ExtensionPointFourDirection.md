# Centermark.ExtensionPointFourDirection Property

Parent Object: [Centermark](../Centermark/Centermark.md)

## Description

Property that returns the direction of the extension point. This can be used to understand the direction the fourth extension point can be moved.

## Syntax

Centermark.**ExtensionPointFourDirection**() As [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Property Value

This is a read only property whose value is a [UnitVector2d](../UnitVector2d/UnitVector2d.md).

## Version

Introduced in version 2009
