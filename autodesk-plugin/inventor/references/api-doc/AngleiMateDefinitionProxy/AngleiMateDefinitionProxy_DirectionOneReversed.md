# AngleiMateDefinitionProxy.DirectionOneReversed Property

Parent Object: [AngleiMateDefinitionProxy](../AngleiMateDefinitionProxy/AngleiMateDefinitionProxy.md)

## Description

Property that gets whether the direction is reversed.

## Syntax

AngleiMateDefinitionProxy.**DirectionOneReversed**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 6
