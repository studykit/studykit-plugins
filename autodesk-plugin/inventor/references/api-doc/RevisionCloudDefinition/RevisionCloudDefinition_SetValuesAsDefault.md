# RevisionCloudDefinition.SetValuesAsDefault Property

Parent Object: [RevisionCloudDefinition](../RevisionCloudDefinition/RevisionCloudDefinition.md)

## Description

Gets and sets whether to set the arc radius values as default for next time when create a new RevisionCloudDefinition.

## Syntax

RevisionCloudDefinition.**SetValuesAsDefault**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2025
