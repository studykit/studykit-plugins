# RevisionCloudDefinition.MinimumArcRadius Property

Parent Object: [RevisionCloudDefinition](../RevisionCloudDefinition/RevisionCloudDefinition.md)

## Description

Gets and sets the minimum arc radius.

## Syntax

RevisionCloudDefinition.**MinimumArcRadius**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2025
