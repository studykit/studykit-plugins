# Sketches3D.Item Property

Parent Object: [Sketches3D](../Sketches3D/Sketches3D.md)

## Description

Returns the specified Sketch3D object from the collection.

## Syntax

Sketches3D.**Item**( ***Index*** As Variant ) As [Sketch3D](../Sketch3D/Sketch3D.md)

## Property Value

This is a read only property whose value is a [Sketch3D](../Sketch3D/Sketch3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the 3D sketch to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the 3D sketch name. If an out-of-range index or a name of a non-existent sketch is provided, an error occurs. |

## Version

Introduced in version 6
