# CornerRoundDefinition.EdgeSetCount Property

Parent Object: [CornerRoundDefinition](../CornerRoundDefinition/CornerRoundDefinition.md)

## Description

Property that returns the number of edge sets defined within this CornerRoundDefinition object.

## Syntax

CornerRoundDefinition.**EdgeSetCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2009
