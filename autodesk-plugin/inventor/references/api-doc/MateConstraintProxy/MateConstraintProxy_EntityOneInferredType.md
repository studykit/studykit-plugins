# MateConstraintProxy.EntityOneInferredType Property

Parent Object: [MateConstraintProxy](../MateConstraintProxy/MateConstraintProxy.md)

## Description

Property that returns an enum indicating how the geometry of entity one is interpreted.

## Syntax

MateConstraintProxy.**EntityOneInferredType**() As [InferredTypeEnum](../InferredTypeEnum.md)

## Property Value

This is a read only property whose value is an [InferredTypeEnum](../InferredTypeEnum.md).

## Version

Introduced in version 11
