# SketchEllipticalArcs3D.Item Property

Parent Object: [SketchEllipticalArcs3D](../SketchEllipticalArcs3D/SketchEllipticalArcs3D.md)

## Description

Returns the specified SketchEllipticalArc3D object from the collection.

## Syntax

SketchEllipticalArcs3D.**Item**( ***Index*** As Long ) As [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md)

## Property Value

This is a read only property whose value is a [SketchEllipticalArc3D](../SketchEllipticalArc3D/SketchEllipticalArc3D.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 6
