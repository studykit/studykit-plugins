# SketchEllipticalArcs3D Object

## Description

The SketchEllipticalArcs3D object provides access to all the objects in a sketch.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchEllipticalArcs3D/SketchEllipticalArcs3D_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchEllipticalArcs3D/SketchEllipticalArcs3D_Count.md) | Property that returns the number of items in the collection. |
| [Item](../SketchEllipticalArcs3D/SketchEllipticalArcs3D_Item.md) | Returns the specified SketchEllipticalArc3D object from the collection. |
| [Type](../SketchEllipticalArcs3D/SketchEllipticalArcs3D_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[Sketch3D.SketchEllipticalArcs3D](../Sketch3D/Sketch3D_SketchEllipticalArcs3D.md), [Sketch3DProxy.SketchEllipticalArcs3D](../Sketch3DProxy/Sketch3DProxy_SketchEllipticalArcs3D.md)

## Version

Introduced in version 6
