# CommandControl.ControlDefinition Property

Parent Object: [CommandControl](../CommandControl/CommandControl.md)

## Description

Property that returns the ControlDefinition associated with this control.

## Syntax

CommandControl.**ControlDefinition**() As [ControlDefinition](../ControlDefinition/ControlDefinition.md)

## Property Value

This is a read only property whose value is a [ControlDefinition](../ControlDefinition/ControlDefinition.md).

## Version

Introduced in version 2010
