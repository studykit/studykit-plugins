# CommandControl.DisplayName Property

Parent Object: [CommandControl](../CommandControl/CommandControl.md)

## Description

Property that returns the display name of the control.

## Syntax

CommandControl.**DisplayName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2010
