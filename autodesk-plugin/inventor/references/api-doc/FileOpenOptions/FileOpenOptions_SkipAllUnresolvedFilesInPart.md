# FileOpenOptions.SkipAllUnresolvedFilesInPart Property

Parent Object: [FileOpenOptions](../FileOpenOptions/FileOpenOptions.md)

## Description

Enables/disables skip all unresolved files during openning in part.

## Syntax

FileOpenOptions.**SkipAllUnresolvedFilesInPart**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2011
