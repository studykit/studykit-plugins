# GeometryIntent.Point Property

Parent Object: [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Description

Read-only property that returns a 3d point define by the geometry intent. This property returns nothing there is not a 3d point intent.

## Syntax

GeometryIntent.**Point**() As [Point](../Point/Point.md)

## Property Value

This is a read only property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 2014
