# GeometryIntent.PointOnSheet Property

Parent Object: [GeometryIntent](../GeometryIntent/GeometryIntent.md)

## Description

Property that returns the point on sheet represented by a point intent. This property returns Nothing if this is not a point intent.

## Syntax

GeometryIntent.**PointOnSheet**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read only property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 11
