# SurfaceTextureJISDefinition.FirstRequirement Property

Parent Object: [SurfaceTextureJISDefinition](../SurfaceTextureJISDefinition/SurfaceTextureJISDefinition.md)

## Description

Gets and sets the first requirement value for the surface.

## Syntax

SurfaceTextureJISDefinition.**FirstRequirement**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2016
