# SimulationManager.Parent Property

Parent Object: [SimulationManager](../SimulationManager/SimulationManager.md)

## Description

Gets the parent assembly or part component definition object.

## Syntax

SimulationManager.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2013
