# Wires.Count Property

Parent Object: [Wires](../Wires/Wires.md)

## Description

Property that returns the number of items in this collection.

## Syntax

Wires.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2009
