# FaceCollection.CollectionType Property

Parent Object: [FaceCollection](../FaceCollection/FaceCollection.md)

## Description

Property returning a constant indicating the Face Collection type.

## Syntax

FaceCollection.**CollectionType**() As [FaceCollectionEnum](../FaceCollectionEnum.md)

## Property Value

This is a read only property whose value is a [FaceCollectionEnum](../FaceCollectionEnum.md).

## Version

Introduced in version 9
