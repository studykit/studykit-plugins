# SketchRectangularPatternDefinition.SuppressedElements Property

Parent Object: [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Description

Gets and sets the suppressed elements.

## Syntax

SketchRectangularPatternDefinition.**SuppressedElements**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2025.1
