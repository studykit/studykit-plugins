# SketchRectangularPatternDefinition.Copy Method

Parent Object: [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Description

Creates a copy of this SketchRectangularPatternDefinition object. The new SketchRectangularPatternDefinition object is independent of any pattern.

## Syntax

SketchRectangularPatternDefinition.**Copy**() As [SketchRectangularPatternDefinition](../SketchRectangularPatternDefinition/SketchRectangularPatternDefinition.md)

## Version

Introduced in version 2025.1
