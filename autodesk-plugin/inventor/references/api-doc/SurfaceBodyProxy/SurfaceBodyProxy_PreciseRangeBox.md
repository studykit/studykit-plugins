# SurfaceBodyProxy.PreciseRangeBox Property

Parent Object: [SurfaceBodyProxy](../SurfaceBodyProxy/SurfaceBodyProxy.md)

## Description

Gets a tight fitting bounding box for this body.

## Syntax

SurfaceBodyProxy.**PreciseRangeBox**() As [Box](../Box/Box.md)

## Property Value

This is a read only property whose value is a [Box](../Box/Box.md).

## Version

Introduced in version 2022
