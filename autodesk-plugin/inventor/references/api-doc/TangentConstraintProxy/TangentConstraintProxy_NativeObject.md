# TangentConstraintProxy.NativeObject Property

Parent Object: [TangentConstraintProxy](../TangentConstraintProxy/TangentConstraintProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

TangentConstraintProxy.**NativeObject**() As [TangentConstraint](../TangentConstraint/TangentConstraint.md)

## Property Value

This is a read only property whose value is a [TangentConstraint](../TangentConstraint/TangentConstraint.md).

## Version

Introduced in version 11
