# SketchConstraintSettings.DisplayConstraintsOnCreation Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets whether to display constraints on creation.

## Syntax

SketchConstraintSettings.**DisplayConstraintsOnCreation**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
