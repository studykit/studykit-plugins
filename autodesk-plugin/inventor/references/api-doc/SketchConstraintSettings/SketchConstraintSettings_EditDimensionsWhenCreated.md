# SketchConstraintSettings.EditDimensionsWhenCreated Property

Parent Object: [SketchConstraintSettings](../SketchConstraintSettings/SketchConstraintSettings.md)

## Description

Gets and sets whether to edit dimensions just after creation.

## Syntax

SketchConstraintSettings.**EditDimensionsWhenCreated**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2015
