# SketchConstraintSettings Object

## Description

Provides access to sketch constraints application settings.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchConstraintSettings/SketchConstraintSettings_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [ConstraintInferencePriority](../SketchConstraintSettings/SketchConstraintSettings_ConstraintInferencePriority.md) | Gets and sets the priority of inferred constraints when placing. |
| [CreateDimensionsOnValueInput](../SketchConstraintSettings/SketchConstraintSettings_CreateDimensionsOnValueInput.md) | Gets and sets whether to create dimensions when inputting values into heads-up display fields. |
| [DisplayCoincidentConstraintsInSketch](../SketchConstraintSettings/SketchConstraintSettings_DisplayCoincidentConstraintsInSketch.md) | Gets and sets whether to display coincident constraints in sketch. |
| [DisplayConstraintsOnCreation](../SketchConstraintSettings/SketchConstraintSettings_DisplayConstraintsOnCreation.md) | Gets and sets whether to display constraints on creation. |
| [EditDimensionsWhenCreated](../SketchConstraintSettings/SketchConstraintSettings_EditDimensionsWhenCreated.md) | Gets and sets whether to edit dimensions just after creation. |
| [EnableInferConstraints](../SketchConstraintSettings/SketchConstraintSettings_EnableInferConstraints.md) | Gets and sets whether to enable inferred constraints. |
| [EnablePersistConstraints](../SketchConstraintSettings/SketchConstraintSettings_EnablePersistConstraints.md) | Gets and sets whether to persist inferred constraints. |
| [EnableRelaxMode](../SketchConstraintSettings/SketchConstraintSettings_EnableRelaxMode.md) | Gets and sets whether to enable relax mode. |
| [GeometricConstraintsToRemoveInRelaxMode](../SketchConstraintSettings/SketchConstraintSettings_GeometricConstraintsToRemoveInRelaxMode.md) | Gets and sets which geometric constraints can be removed in relax mode when dragging sketch entitiess. |
| [KeepDimensionsWithEquationInRelaxMode](../SketchConstraintSettings/SketchConstraintSettings_KeepDimensionsWithEquationInRelaxMode.md) | Gets and sets whether to keep the dimensions with an equation in relax dragging and over-constraint solving. |
| [OverConstrainedDimensionBehavior](../SketchConstraintSettings/SketchConstraintSettings_OverConstrainedDimensionBehavior.md) | Gets and sets the behavior when create an over-constrained dimension. |
| [Parent](../SketchConstraintSettings/SketchConstraintSettings_Parent.md) | Gets the parent object from whom this object can logically be reached. |
| [SelectionForGeometricConstraintsInference](../SketchConstraintSettings/SketchConstraintSettings_SelectionForGeometricConstraintsInference.md) | Gets and sets which geometric constraints have their inference indicators shown to assist creating sketch entities. |
| [ShowConstraintsForSelectedObjects](../SketchConstraintSettings/SketchConstraintSettings_ShowConstraintsForSelectedObjects.md) | Gets and sets whether to show constraints for the selected objects. |
| [Type](../SketchConstraintSettings/SketchConstraintSettings_Type.md) | Gets the constant that indicates the type of this object. |

## Accessed From

[SketchOptions.SketchConstraintSettings](../SketchOptions/SketchOptions_SketchConstraintSettings.md)

## Version

Introduced in version 2015
