# ComponentOccurrenceProxy.ActiveDesignViewRepresentation Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets the active Design View Representation for an assembly occurrence. This property returns Nothing in the case where a design view representation was set for the occurrence non-associatively (i.e. with the 'Associative' flag turned off).

## Syntax

ComponentOccurrenceProxy.**ActiveDesignViewRepresentation**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2026
