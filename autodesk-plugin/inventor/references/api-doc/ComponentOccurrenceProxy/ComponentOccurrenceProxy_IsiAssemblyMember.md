# ComponentOccurrenceProxy.IsiAssemblyMember Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Property that returns whether the ComponentOccurrence is an iAssemblyMember. The property returns True if the ComponentOccurrence is an iAssemblyMember. This property always returns False for occurrences of part documents.

## Syntax

ComponentOccurrenceProxy.**IsiAssemblyMember**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 11
