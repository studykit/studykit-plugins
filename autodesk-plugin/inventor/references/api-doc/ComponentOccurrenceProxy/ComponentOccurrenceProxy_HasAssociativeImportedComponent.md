# ComponentOccurrenceProxy.HasAssociativeImportedComponent Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Read-only property that returns whether the ComponentOccurrence has an associative imported component.

## Syntax

ComponentOccurrenceProxy.**HasAssociativeImportedComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2016
