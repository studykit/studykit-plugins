# ComponentOccurrenceProxy.BOMStructure Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Gets and sets the BOM structure override.

## Syntax

ComponentOccurrenceProxy.**BOMStructure**() As [BOMStructureEnum](../BOMStructureEnum.md)

## Property Value

This is a read/write property whose value is a [BOMStructureEnum](../BOMStructureEnum.md).

## Version

Introduced in version 10
