# ComponentOccurrenceProxy.OccurrencePropertySetsEnabled Property

Parent Object: [ComponentOccurrenceProxy](../ComponentOccurrenceProxy/ComponentOccurrenceProxy.md)

## Description

Read-write property that gets and sets whether this occurrence has a PropertySets object.

## Syntax

ComponentOccurrenceProxy.**OccurrencePropertySetsEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2022
