# DerivedAssemblyOccurrences.Item Property

Parent Object: [DerivedAssemblyOccurrences](../DerivedAssemblyOccurrences/DerivedAssemblyOccurrences.md)

## Description

Returns the specified object from the collection.

## Syntax

DerivedAssemblyOccurrences.**Item**( ***Index*** As Variant ) As [DerivedAssemblyOccurrence](../DerivedAssemblyOccurrence/DerivedAssemblyOccurrence.md)

## Property Value

This is a read only property whose value is a [DerivedAssemblyOccurrence](../DerivedAssemblyOccurrence/DerivedAssemblyOccurrence.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the index of the to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the derived occurrence's name. If an out of range index or a name of a non-existent occurrence is provided, an error occurs. |

## Version

Introduced in version 5.3
