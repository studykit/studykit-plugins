# DSResult.Parent Property

Parent Object: [DSResult](../DSResult/DSResult.md)

## Description

Gets the parent degree of freedom that this results object is associated with.

## Syntax

DSResult.**Parent**() As [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md)

## Property Value

This is a read only property whose value is a [DSDegreeOfFreedom](../DSDegreeOfFreedom/DSDegreeOfFreedom.md).

## Version

Introduced in version 2013
