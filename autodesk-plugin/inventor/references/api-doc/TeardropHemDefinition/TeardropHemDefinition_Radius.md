# TeardropHemDefinition.Radius Property

Parent Object: [TeardropHemDefinition](../TeardropHemDefinition/TeardropHemDefinition.md)

## Description

Property that returns the parameter controlling the radius of a hem feature defined by a tear drop style of bend.

## Syntax

TeardropHemDefinition.**Radius**() As [Parameter](../Parameter/Parameter.md)

## Property Value

This is a read only property whose value is a [Parameter](../Parameter/Parameter.md).

## Version

Introduced in version 2009
