# ReplaceFaceDefinition.NewFaces Property

Parent Object: [ReplaceFaceDefinition](../ReplaceFaceDefinition/ReplaceFaceDefinition.md)

## Description

Read-write property that gets and sets the new faces that are replacing the other faces.

## Syntax

ReplaceFaceDefinition.**NewFaces**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read/write property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 2024.1
