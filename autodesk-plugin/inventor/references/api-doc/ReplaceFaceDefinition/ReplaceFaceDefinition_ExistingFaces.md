# ReplaceFaceDefinition.ExistingFaces Property

Parent Object: [ReplaceFaceDefinition](../ReplaceFaceDefinition/ReplaceFaceDefinition.md)

## Description

Read-write property that gets and sets the faces that are being replaced.

## Syntax

ReplaceFaceDefinition.**ExistingFaces**() As [FaceCollection](../FaceCollection/FaceCollection.md)

## Property Value

This is a read/write property whose value is a [FaceCollection](../FaceCollection/FaceCollection.md).

## Version

Introduced in version 2024.1
