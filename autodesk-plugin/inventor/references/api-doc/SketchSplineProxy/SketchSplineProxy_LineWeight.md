# SketchSplineProxy.LineWeight Property

Parent Object: [SketchSplineProxy](../SketchSplineProxy/SketchSplineProxy.md)

## Description

Gets and sets the LineWeight applied to this sketch spline.

## Syntax

SketchSplineProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 9
