# SketchSplineProxy.Geometry Property

Parent Object: [SketchSplineProxy](../SketchSplineProxy/SketchSplineProxy.md)

## Description

Property that returns a geometry object. The object returned represents a snapshot view of the current state of the sketch spline.

## Syntax

SketchSplineProxy.**Geometry**() As [BSplineCurve2d](../BSplineCurve2d/BSplineCurve2d.md)

## Property Value

This is a read only property whose value is a [BSplineCurve2d](../BSplineCurve2d/BSplineCurve2d.md).

## Version

Introduced in version 6
