# SketchSplineProxy.Type Property

Parent Object: [SketchSplineProxy](../SketchSplineProxy/SketchSplineProxy.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchSplineProxy.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 6
