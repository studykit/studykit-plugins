# SketchControlPointSpline.Construction Property

Parent Object: [SketchControlPointSpline](../SketchControlPointSpline/SketchControlPointSpline.md)

## Description

Gets and sets whether the entity behaves as a construction entity or not.

## Syntax

SketchControlPointSpline.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2014
