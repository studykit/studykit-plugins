# LibraryFolders.Add Method

Parent Object: [LibraryFolders](../LibraryFolders/LibraryFolders.md)

## Description

creates a new library folder.

## Syntax

LibraryFolders.**Add**( [***Name***] As Variant ) As [LibraryFolder](../LibraryFolder/LibraryFolder.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Name | Variant | Optional input String value to specify the name of the library folder. |

## Version

Introduced in version 2016
