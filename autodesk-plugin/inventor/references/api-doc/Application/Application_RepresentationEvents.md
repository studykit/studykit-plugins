# Application.RepresentationEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the RepresentationEvents object. This object supports a set of events related to design view and positional representations that are assembly centric.

## Syntax

Application.**RepresentationEvents**() As [RepresentationEvents](../RepresentationEvents/RepresentationEvents.md)

## Property Value

This is a read only property whose value is a [RepresentationEvents](../RepresentationEvents/RepresentationEvents.md).

## Version

Introduced in version 11
