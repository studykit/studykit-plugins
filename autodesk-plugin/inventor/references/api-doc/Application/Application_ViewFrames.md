# Application.ViewFrames Property

Parent Object: [Application](../Application/Application.md)

## Description

Read-only property that returns the ViewFramesEnumerator collection object.

## Syntax

Application.**ViewFrames**() As [ViewFramesEnumerator](../ViewFramesEnumerator/ViewFramesEnumerator.md)

## Property Value

This is a read only property whose value is a [ViewFramesEnumerator](../ViewFramesEnumerator/ViewFramesEnumerator.md).

## Version

Introduced in version 2021
