# Application.FileUIEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the FileUIEvents object. This object supports a set of events that are fired in reaction to certain user interface actions.

## Syntax

Application.**FileUIEvents**() As [FileUIEvents](../FileUIEvents/FileUIEvents.md)

## Property Value

This is a read only property whose value is a [FileUIEvents](../FileUIEvents/FileUIEvents.md).

## Version

Introduced in version 4
