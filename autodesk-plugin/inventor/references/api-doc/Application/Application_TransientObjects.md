# Application.TransientObjects Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the object through which all general transient objects are created.

## Syntax

Application.**TransientObjects**() As [TransientObjects](../TransientObjects/TransientObjects.md)

## Property Value

This is a read only property whose value is a [TransientObjects](../TransientObjects/TransientObjects.md).
