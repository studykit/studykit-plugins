# Application.FileManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the FileManager object.

## Syntax

Application.**FileManager**() As [FileManager](../FileManager/FileManager.md)

## Property Value

This is a read only property whose value is a [FileManager](../FileManager/FileManager.md).
