# Application.VBAProjects Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns Autodesk Inventor's VBA projects collection.

## Syntax

Application.**VBAProjects**() As [InventorVBAProjects](../InventorVBAProjects/InventorVBAProjects.md)

## Property Value

This is a read only property whose value is an [InventorVBAProjects](../InventorVBAProjects/InventorVBAProjects.md).

## Version

Introduced in version 6
