# Application.RefreshRibbonForComparison Method

Parent Object: [Application](../Application/Application.md)

## Description

Method that refreshes the ribbon controls display when the comparison is set.

## Syntax

Application.**RefreshRibbonForComparison**()

## Version

Introduced in version 2020
