# Application.PromptsOptions Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the Prompts Options object.

## Syntax

Application.**PromptsOptions**() As [PromptsOptions](../PromptsOptions/PromptsOptions.md)

## Property Value

This is a read only property whose value is a [PromptsOptions](../PromptsOptions/PromptsOptions.md).

## Version

Introduced in version 2025.1
