# Application.MaterialDisplayUnits Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets and sets the units that are used in the user interface when working with materials.

## Syntax

Application.**MaterialDisplayUnits**() As [MaterialDisplayUnitsEnum](../MaterialDisplayUnitsEnum.md)

## Property Value

This is a read/write property whose value is a [MaterialDisplayUnitsEnum](../MaterialDisplayUnitsEnum.md).

## Version

Introduced in version 2014
