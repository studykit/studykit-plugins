# Application.CommandManager Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the CommandManager object.

## Syntax

Application.**CommandManager**() As [CommandManager](../CommandManager/CommandManager.md)

## Property Value

This is a read only property whose value is a [CommandManager](../CommandManager/CommandManager.md).
