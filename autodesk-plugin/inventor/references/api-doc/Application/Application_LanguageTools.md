# Application.LanguageTools Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the LanguageTools object.

## Syntax

Application.**LanguageTools**() As [LanguageTools](../LanguageTools/LanguageTools.md)

## Property Value

This is a read only property whose value is a [LanguageTools](../LanguageTools/LanguageTools.md).

## Version

Introduced in version 2008
