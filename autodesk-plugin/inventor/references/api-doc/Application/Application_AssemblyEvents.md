# Application.AssemblyEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Property that returns the AssemblyEvents object. This object supports a set of events that are assembly centric.

## Syntax

Application.**AssemblyEvents**() As [AssemblyEvents](../AssemblyEvents/AssemblyEvents.md)

## Property Value

This is a read only property whose value is an [AssemblyEvents](../AssemblyEvents/AssemblyEvents.md).

## Version

Introduced in version 11
