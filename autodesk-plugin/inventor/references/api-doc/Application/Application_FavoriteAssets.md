# Application.FavoriteAssets Property

Parent Object: [Application](../Application/Application.md)

## Description

Gets the set of favorite assets. This includes both appearance and material assets.

## Syntax

Application.**FavoriteAssets**() As [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md)

## Property Value

This is a read only property whose value is an [AssetsEnumerator](../AssetsEnumerator/AssetsEnumerator.md).

## Version

Introduced in version 2014
