# Application.ModelStateEvents Property

Parent Object: [Application](../Application/Application.md)

## Description

Read-only property that returns the ModelStateEvents object.

## Syntax

Application.**ModelStateEvents**() As [ModelStateEvents](../ModelStateEvents/ModelStateEvents.md)

## Property Value

This is a read only property whose value is a [ModelStateEvents](../ModelStateEvents/ModelStateEvents.md).

## Version

Introduced in version 2022
