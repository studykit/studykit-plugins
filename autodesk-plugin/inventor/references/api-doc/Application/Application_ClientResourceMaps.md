# Application.ClientResourceMaps Property

Parent Object: [Application](../Application/Application.md)

## Description

Read-only property that returns ClientResourceMaps object.

## Syntax

Application.**ClientResourceMaps**() As [ClientResourceMaps](../ClientResourceMaps/ClientResourceMaps.md)

## Property Value

This is a read only property whose value is a [ClientResourceMaps](../ClientResourceMaps/ClientResourceMaps.md).

## Version

Introduced in version 2022
