# CornerOptions Object

## Description

The CornerOptions object defines the settings that influence the creation of a corner in a sheet metal part.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../CornerOptions/CornerOptions_Copy.md) | Function that creates a copy of this CornerOptions object. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [CornerReliefPlacement](../CornerOptions/CornerOptions_CornerReliefPlacement.md) | Defines the two bend relief placement to use for the associated sheet metal feature (valid only for round and square corners). |
| [CornerReliefShape](../CornerOptions/CornerOptions_CornerReliefShape.md) | Defines the type of two bend relief shape to use for the associated sheet metal feature. |
| [CornerReliefSize](../CornerOptions/CornerOptions_CornerReliefSize.md) | Provides access to the two bend corner relief size of a sheet metal feature. |
| [IsTwoBend](../CornerOptions/CornerOptions_IsTwoBend.md) | Property that indicates if the corner this CornerOptions object is associated with is a two or three bend corner. |
| [Parent](../CornerOptions/CornerOptions_Parent.md) | Property that returns the parent object of this CornerOptions object. |
| [Type](../CornerOptions/CornerOptions_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
