# BendDefinition.BendOptions Property

Parent Object: [BendDefinition](../BendDefinition/BendDefinition.md)

## Description

Gets and sets the BendOptions object that defines the options for a bend.

## Syntax

BendDefinition.**BendOptions**() As [BendOptions](../BendOptions/BendOptions.md)

## Property Value

This is a read/write property whose value is a [BendOptions](../BendOptions/BendOptions.md).

## Version

Introduced in version 2009
