## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../WebBrowserDialog/WebBrowserDialog_Application.md) | Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server. |
| [DisabledWindowState](../WebBrowserDialog/WebBrowserDialog_DisabledWindowState.md) | Read-write property that gets and sets a bit-mask defining the disabled window states of this web browser dialog. |
| [Height](../WebBrowserDialog/WebBrowserDialog_Height.md) | Read-write property that gets and sets the height of this web browser dialog. |
| [HWND](../WebBrowserDialog/WebBrowserDialog_HWND.md) | Read-only property that returns the HWND of this web browser dialog. |
| [InternalName](../WebBrowserDialog/WebBrowserDialog_InternalName.md) | Read-only property that returns the internal name of the web browser dialog. |
| [Left](../WebBrowserDialog/WebBrowserDialog_Left.md) | Read-write property that gets and sets the left edge of this web browser dialog. |
| [Modal](../WebBrowserDialog/WebBrowserDialog_Modal.md) | Read-only property that returns whether this dialog is modal dialog or not. |
| [Title](../WebBrowserDialog/WebBrowserDialog_Title.md) | Read-write property that gets and sets the title of this web browser dialog. |
| [Top](../WebBrowserDialog/WebBrowserDialog_Top.md) | Read-write property that gets and sets the top edge of this web browser dialog. |
| [Type](../WebBrowserDialog/WebBrowserDialog_Type.md) | Gets the constant that indicates the type of this object. |
| [Visible](../WebBrowserDialog/WebBrowserDialog_Visible.md) | Read-write property that gets and sets the visibility of this web browser dialog. |
| [WebBrowserDialogEvents](../WebBrowserDialog/WebBrowserDialog_WebBrowserDialogEvents.md) | Read-only property that gets the WebBrowserDialogEvents object. |
| [Width](../WebBrowserDialog/WebBrowserDialog_Width.md) | Read-write property that gets and sets the width of this web browser dialog. |
| [WindowState](../WebBrowserDialog/WebBrowserDialog_WindowState.md) | Read-write property that gets and sets the window state of this web browser dialog. |
