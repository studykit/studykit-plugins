# BIMConnectorLinks.Add Method

Parent Object: [BIMConnectorLinks](../BIMConnectorLinks/BIMConnectorLinks.md)

## Description

Method that creates a new BIMConnectorLink.The new BIMConnectorLink is returned.

## Syntax

BIMConnectorLinks.**Add**( ***ConnectorOne*** As [BIMConnector](../BIMConnector/BIMConnector.md), ***ConnectorTwo*** As [BIMConnector](../BIMConnector/BIMConnector.md) ) As [BIMConnectorLink](../BIMConnectorLink/BIMConnectorLink.md)

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| ConnectorOne | [BIMConnector](../BIMConnector/BIMConnector.md) | Input BIMConnector that defines the first of two connectors to be connected by the link. |
| ConnectorTwo | [BIMConnector](../BIMConnector/BIMConnector.md) | Input BIMConnector that defines the second of two connectors to be connected by the link. |

## Version

Introduced in version 2011
