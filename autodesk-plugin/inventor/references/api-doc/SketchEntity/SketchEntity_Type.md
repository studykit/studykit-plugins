# SketchEntity.Type Property

Parent Object: [SketchEntity](../SketchEntity/SketchEntity.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

SketchEntity.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).
