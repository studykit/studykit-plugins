# SmoothConstraint3DProxy.Delete Method

Parent Object: [SmoothConstraint3DProxy](../SmoothConstraint3DProxy/SmoothConstraint3DProxy.md)

## Description

Method that deletes the constraint. In the case of coincident constraint, all the connected lines are disconnected and the constraint is deleted.

## Syntax

SmoothConstraint3DProxy.**Delete**()

## Version

Introduced in version 11
