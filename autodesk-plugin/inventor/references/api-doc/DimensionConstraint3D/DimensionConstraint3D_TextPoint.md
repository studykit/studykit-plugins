# DimensionConstraint3D.TextPoint Property

Parent Object: [DimensionConstraint3D](../DimensionConstraint3D/DimensionConstraint3D.md)

## Description

Gets and sets the position of the dimension text.

## Syntax

DimensionConstraint3D.**TextPoint**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 11
