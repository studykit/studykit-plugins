# AssetLibrary.MaterialAssetCategories Property

Parent Object: [AssetLibrary](../AssetLibrary/AssetLibrary.md)

## Description

Gets the AssetCategories collection which provides access to the material related categories defined within this library.

## Syntax

AssetLibrary.**MaterialAssetCategories**() As [AssetCategories](../AssetCategories/AssetCategories.md)

## Property Value

This is a read only property whose value is an [AssetCategories](../AssetCategories/AssetCategories.md).

## Version

Introduced in version 2014
