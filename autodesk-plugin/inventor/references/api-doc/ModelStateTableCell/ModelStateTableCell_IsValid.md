# ModelStateTableCell.IsValid Property

Parent Object: [ModelStateTableCell](../ModelStateTableCell/ModelStateTableCell.md)

## Description

Read-only property that returns whether the contents of this cell are valid or not.

## Syntax

ModelStateTableCell.**IsValid**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2022
