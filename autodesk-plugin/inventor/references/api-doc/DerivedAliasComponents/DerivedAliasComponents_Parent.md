# DerivedAliasComponents.Parent Property

Parent Object: [DerivedAliasComponents](../DerivedAliasComponents/DerivedAliasComponents.md)

## Description

Property that returns the ComponentDefinition object.

## Syntax

DerivedAliasComponents.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2010
