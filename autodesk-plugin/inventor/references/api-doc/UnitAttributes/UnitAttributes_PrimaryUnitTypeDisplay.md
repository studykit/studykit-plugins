# UnitAttributes.PrimaryUnitTypeDisplay Property

Parent Object: [UnitAttributes](../UnitAttributes/UnitAttributes.md)

## Description

Gets and sets whether display the primary unit type.

## Syntax

UnitAttributes.**PrimaryUnitTypeDisplay**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
