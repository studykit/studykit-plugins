# iMateDefinitionsEnumerator.Item Property

Parent Object: [iMateDefinitionsEnumerator](../iMateDefinitionsEnumerator/iMateDefinitionsEnumerator.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

iMateDefinitionsEnumerator.**Item**( ***Index*** As Long ) As [iMateDefinition](../iMateDefinition/iMateDefinition.md)

## Property Value

This is a read only property whose value is an [iMateDefinition](../iMateDefinition/iMateDefinition.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the to return. |

## Version

Introduced in version 6
