# PointInferenceEnumerator.Item Property

Parent Object: [PointInferenceEnumerator](../PointInferenceEnumerator/PointInferenceEnumerator.md)

## Description

Returns the specified PointInference object from the collection.

## Syntax

PointInferenceEnumerator.**Item**( ***Index*** As Long ) As [PointInference](../PointInference/PointInference.md)

## Property Value

This is a read only property whose value is a [PointInference](../PointInference/PointInference.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the PointInference to return. |

## Version

Introduced in version 5
