# HelicalCurveShapeDefinitionRow.Index Property

Parent Object: [HelicalCurveShapeDefinitionRow](../HelicalCurveShapeDefinitionRow/HelicalCurveShapeDefinitionRow.md)

## Description

Returns the index of the row.

## Syntax

HelicalCurveShapeDefinitionRow.**Index**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2019
