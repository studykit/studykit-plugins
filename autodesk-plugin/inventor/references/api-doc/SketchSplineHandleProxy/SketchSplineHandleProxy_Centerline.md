# SketchSplineHandleProxy.Centerline Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Gets and sets whether the line behaves as a centerline or not.

## Syntax

SketchSplineHandleProxy.**Centerline**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
