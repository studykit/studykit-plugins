# SketchSplineHandleProxy.Length Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Double property that returns the length of the entity in centimeters.

## Syntax

SketchSplineHandleProxy.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 2008
