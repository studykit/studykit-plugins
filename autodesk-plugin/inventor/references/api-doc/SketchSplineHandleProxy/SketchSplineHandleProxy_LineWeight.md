# SketchSplineHandleProxy.LineWeight Property

Parent Object: [SketchSplineHandleProxy](../SketchSplineHandleProxy/SketchSplineHandleProxy.md)

## Description

Gets and sets the LineWeight applied to this sketch line.

## Syntax

SketchSplineHandleProxy.**LineWeight**() As Double

## Property Value

This is a read/write property whose value is a Double.

## Version

Introduced in version 2008
