# ReferenceParameter.BuiltIn Property

Parent Object: [ReferenceParameter](../ReferenceParameter/ReferenceParameter.md)

## Description

Property that returns whether this is an Inventor-generated parameter or an API-created one. Returns True if it is Inventor-generated.

## Syntax

ReferenceParameter.**BuiltIn**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 10
