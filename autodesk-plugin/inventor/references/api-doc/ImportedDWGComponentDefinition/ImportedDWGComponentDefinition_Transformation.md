# ImportedDWGComponentDefinition.Transformation Property

Parent Object: [ImportedDWGComponentDefinition](../ImportedDWGComponentDefinition/ImportedDWGComponentDefinition.md)

## Description

Read-write property that gets and sets the translation and rotation of the imported DWG in part space.

## Syntax

ImportedDWGComponentDefinition.**Transformation**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).
