# CircularPatternFeatureDefinition.PositioningMethod Property

Parent Object: [CircularPatternFeatureDefinition](../CircularPatternFeatureDefinition/CircularPatternFeatureDefinition.md)

## Description

Read-write property that gets and sets the enum indicating the positioning method used for pattern creation.

## Syntax

CircularPatternFeatureDefinition.**PositioningMethod**() As [PatternPositioningMethodEnum](../PatternPositioningMethodEnum.md)

## Property Value

This is a read/write property whose value is a [PatternPositioningMethodEnum](../PatternPositioningMethodEnum.md).

## Version

Introduced in version 2017
