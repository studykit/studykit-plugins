# SketchArc.HasReferenceComponent Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchArc.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 5.3
