# SketchArc.Geometry3d Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Read-only property that returns arc geometry that represents this arc in model space.

## Syntax

SketchArc.**Geometry3d**() As [Arc3d](../Arc3d/Arc3d.md)

## Property Value

This is a read only property whose value is an [Arc3d](../Arc3d/Arc3d.md).

## Version

Introduced in version 2013
