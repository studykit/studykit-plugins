# SketchArc.LineType Property

Parent Object: [SketchArc](../SketchArc/SketchArc.md)

## Description

Gets and sets the lineType applied to this sketch line.

## Syntax

SketchArc.**LineType**() As [LineTypeEnum](../LineTypeEnum.md)

## Property Value

This is a read/write property whose value is a [LineTypeEnum](../LineTypeEnum.md).

## Version

Introduced in version 9
