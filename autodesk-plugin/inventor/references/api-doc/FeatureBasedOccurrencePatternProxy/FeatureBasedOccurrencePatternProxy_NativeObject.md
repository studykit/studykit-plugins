# FeatureBasedOccurrencePatternProxy.NativeObject Property

Parent Object: [FeatureBasedOccurrencePatternProxy](../FeatureBasedOccurrencePatternProxy/FeatureBasedOccurrencePatternProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

FeatureBasedOccurrencePatternProxy.**NativeObject**() As [FeatureBasedOccurrencePattern](../FeatureBasedOccurrencePattern/FeatureBasedOccurrencePattern.md)

## Property Value

This is a read only property whose value is a [FeatureBasedOccurrencePattern](../FeatureBasedOccurrencePattern/FeatureBasedOccurrencePattern.md).

## Version

Introduced in version 2010
