# ImportedDataExchangeComponentProxy.HealthStatus Property

Parent Object: [ImportedDataExchangeComponentProxy](../ImportedDataExchangeComponentProxy/ImportedDataExchangeComponentProxy.md)

## Description

Property that returns an enum indicating the current state of the object.

## Syntax

ImportedDataExchangeComponentProxy.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2023
