# SketchImages Object

## Description

The SketchImages collection object provides access to all the images in a sketch.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../SketchImages/SketchImages_Add.md) | Method that creates an image based on an input picture file name. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../SketchImages/SketchImages_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../SketchImages/SketchImages_Count.md) | Property that specifies the number of items in the collection. |
| [Item](../SketchImages/SketchImages_Item.md) | Method that returns the specified Image object from the collection. |
| [Type](../SketchImages/SketchImages_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
