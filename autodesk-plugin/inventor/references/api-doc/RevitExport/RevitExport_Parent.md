# RevitExport.Parent Property

Parent Object: [RevitExport](../RevitExport/RevitExport.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

RevitExport.**Parent**() As [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Property Value

This is a read only property whose value is an [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md).

## Version

Introduced in version 2022
