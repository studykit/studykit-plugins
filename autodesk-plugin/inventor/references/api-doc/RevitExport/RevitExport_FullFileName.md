# RevitExport.FullFileName Property

Parent Object: [RevitExport](../RevitExport/RevitExport.md)

## Description

Read-only property that returns the full filename of the Revit model file that relates with this RevitExport object.

## Syntax

RevitExport.**FullFileName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
