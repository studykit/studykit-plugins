# SketchHatchRegion.Pattern Property

Parent Object: [SketchHatchRegion](../SketchHatchRegion/SketchHatchRegion.md)

## Description

Read-write property that gets and sets the DrawingHatchPattern of the sketch hatch region.

## Syntax

SketchHatchRegion.**Pattern**() As [DrawingHatchPattern](../DrawingHatchPattern/DrawingHatchPattern.md)

## Property Value

This is a read/write property whose value is a [DrawingHatchPattern](../DrawingHatchPattern/DrawingHatchPattern.md).

## Version

Introduced in version 2021
