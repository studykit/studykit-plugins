# SplineFitPointConstraint.Spline Property

Parent Object: [SplineFitPointConstraint](../SplineFitPointConstraint/SplineFitPointConstraint.md)

## Description

Property that returns the spline entity being constrained.

## Syntax

SplineFitPointConstraint.**Spline**() As [SketchSpline](../SketchSpline/SketchSpline.md)

## Property Value

This is a read only property whose value is a [SketchSpline](../SketchSpline/SketchSpline.md).

## Version

Introduced in version 5
