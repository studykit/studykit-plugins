# PunchNotes Object

## Description

The PunchNotes collection object provides access to all of the punch notes on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../PunchNotes/PunchNotes_Add.md) | Method that creates a punch note on the sheet based on the \input punch edge. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../PunchNotes/PunchNotes_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../PunchNotes/PunchNotes_Count.md) | Property that returns the number of items in the collection. |
| [Item](../PunchNotes/PunchNotes_Item.md) | Returns the specified PunchNote object from the collection. |
| [Type](../PunchNotes/PunchNotes_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |
