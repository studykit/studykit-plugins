# ModelDatumDefinition.DatumTargetIndices Property

Parent Object: [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Description

Gets and sets the model datum targets’ indices.

## Syntax

ModelDatumDefinition.**DatumTargetIndices**() As Long

## Property Value

This is a read/write property whose value is a Long.

## Version

Introduced in version 2023
