# ModelDatumDefinition.Parent Property

Parent Object: [ModelDatumDefinition](../ModelDatumDefinition/ModelDatumDefinition.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ModelDatumDefinition.**Parent**() As [ModelDatum](../ModelDatum/ModelDatum.md)

## Property Value

This is a read only property whose value is a [ModelDatum](../ModelDatum/ModelDatum.md).

## Version

Introduced in version 2023
