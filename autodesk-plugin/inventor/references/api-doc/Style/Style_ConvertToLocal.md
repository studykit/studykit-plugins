# Style.ConvertToLocal Method

Parent Object: [Style](../Style/Style.md)

## Description

Method that creates a local cached copy of a global style and returns the local style.

## Syntax

Style.**ConvertToLocal**() As [Style](../Style/Style.md)

## Version

Introduced in version 2008
