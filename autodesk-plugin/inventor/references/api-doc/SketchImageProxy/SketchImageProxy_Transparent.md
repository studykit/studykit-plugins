# SketchImageProxy.Transparent Property

Parent Object: [SketchImageProxy](../SketchImageProxy/SketchImageProxy.md)

## Description

Gets and sets whether the image is transparent.

## Syntax

SketchImageProxy.**Transparent**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
