# FilletConstantRadiusEdgeSet.InvertedFillet Property

Parent Object: [FilletConstantRadiusEdgeSet](../FilletConstantRadiusEdgeSet/FilletConstantRadiusEdgeSet.md)

## Description

Gets and sets whether or not to use inverted fillet for the edge set.

## Syntax

FilletConstantRadiusEdgeSet.**InvertedFillet**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2019
