# DesignProject.Owner Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Gets and sets the string identifying the owner of the project.

## Syntax

DesignProject.**Owner**() As String

## Property Value

This is a read/write property whose value is a String.

## Version

Introduced in version 2011
