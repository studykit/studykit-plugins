# DesignProject.Parent Property

Parent Object: [DesignProject](../DesignProject/DesignProject.md)

## Description

Property that returns the parent DesignProjectManager object.

## Syntax

DesignProject.**Parent**() As [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md)

## Property Value

This is a read only property whose value is a [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md).

## Version

Introduced in version 2011
