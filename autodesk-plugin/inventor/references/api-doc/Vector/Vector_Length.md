# Vector.Length Property

Parent Object: [Vector](../Vector/Vector.md)

## Description

Get the length of this vector.

## Syntax

Vector.**Length**() As Double

## Property Value

This is a read only property whose value is a Double.

## Version

Introduced in version 4
