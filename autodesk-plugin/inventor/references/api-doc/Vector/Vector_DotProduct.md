# Vector.DotProduct Method

Parent Object: [Vector](../Vector/Vector.md)

## Description

Determine the dot product of this vector to the specified vector.

## Syntax

Vector.**DotProduct**( ***Argument*** As [Vector](../Vector/Vector.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Argument | [Vector](../Vector/Vector.md) | Input  used for comparison. |

## Version

Introduced in version 4
