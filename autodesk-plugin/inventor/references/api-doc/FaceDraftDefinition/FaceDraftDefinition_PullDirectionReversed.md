# FaceDraftDefinition.PullDirectionReversed Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-write property that gets and sets whether reverse the pull direction.

## Syntax

FaceDraftDefinition.**PullDirectionReversed**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2016
