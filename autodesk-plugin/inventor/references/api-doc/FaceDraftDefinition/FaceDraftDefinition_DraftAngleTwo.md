# FaceDraftDefinition.DraftAngleTwo Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-write property that gets and sets the draft angle two of the face draft. This property is applicable only when the DraftAngleConstraintType is set to kAsymmetricDraftAngles.

## Syntax

FaceDraftDefinition.**DraftAngleTwo**() As Variant

## Property Value

This is a read/write property whose value is a Variant.

## Version

Introduced in version 2016
