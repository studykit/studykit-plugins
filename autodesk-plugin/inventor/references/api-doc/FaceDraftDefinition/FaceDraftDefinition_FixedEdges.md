# FaceDraftDefinition.FixedEdges Property

Parent Object: [FaceDraftDefinition](../FaceDraftDefinition/FaceDraftDefinition.md)

## Description

Read-write property that gets and sets the fixed edges on the input faces. This property is applicable when the DefinitionType is kFixedEdgeFaceDraftDefinitionType or kPartingToolFaceDraftDefinitionType and the CutMaterial is False.

## Syntax

FaceDraftDefinition.**FixedEdges**() As [EdgeCollection](../EdgeCollection/EdgeCollection.md)

## Property Value

This is a read/write property whose value is an [EdgeCollection](../EdgeCollection/EdgeCollection.md).

## Version

Introduced in version 2016
