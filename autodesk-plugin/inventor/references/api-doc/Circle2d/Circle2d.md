# Circle2d Object

## Description

The mathematical Circle2d object - one of the transient objects.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Copy](../Circle2d/Circle2d_Copy.md) | Creates a copy of this Circle2d object. The result is entirely independent and can be edited without affecting the original Circle2d object. |
| [GetCircleData](../Circle2d/Circle2d_GetCircleData.md) | Get the data defining this circle. |
| [PutCircleData](../Circle2d/Circle2d_PutCircleData.md) | Method that sets the data defining this circle. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Center](../Circle2d/Circle2d_Center.md) | The center of the 2D circle. |
| [Evaluator](../Circle2d/Circle2d_Evaluator.md) | Gets the Curve2dEvaluator for this circle. |
| [Radius](../Circle2d/Circle2d_Radius.md) | The radius of the 2D circle. |
