# Circle2d.Center Property

Parent Object: [Circle2d](../Circle2d/Circle2d.md)

## Description

The center of the 2D circle.

## Syntax

Circle2d.**Center**() As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Version

Introduced in version 4
