# SketchOffsetSplines.Item Property

Parent Object: [SketchOffsetSplines](../SketchOffsetSplines/SketchOffsetSplines.md)

## Description

Method that returns the specified SketchOffsetSpline object from the collection.

## Syntax

SketchOffsetSplines.**Item**( ***Index*** As Long ) As [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md)

## Property Value

This is a read only property whose value is a [SketchOffsetSpline](../SketchOffsetSpline/SketchOffsetSpline.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 9
