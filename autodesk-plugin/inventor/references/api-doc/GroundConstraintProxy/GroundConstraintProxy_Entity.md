# GroundConstraintProxy.Entity Property

Parent Object: [GroundConstraintProxy](../GroundConstraintProxy/GroundConstraintProxy.md)

## Description

Property that returns entity being grounded.

## Syntax

GroundConstraintProxy.**Entity**() As [SketchEntity](../SketchEntity/SketchEntity.md)

## Property Value

This is a read only property whose value is a [SketchEntity](../SketchEntity/SketchEntity.md).

## Version

Introduced in version 6
