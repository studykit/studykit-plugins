# GroundConstraintProxy Object

Derived from: [GroundConstraint](../GroundConstraint/GroundConstraint.md) Object

## Description

This is an assembly-context proxy object derived from its native definition-context object.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Delete](../GroundConstraintProxy/GroundConstraintProxy_Delete.md) | Method that deletes the constraint. |
| [GetReferenceKey](../GroundConstraintProxy/GroundConstraintProxy_GetReferenceKey.md) | Method that generates and returns the reference key for this entity. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../GroundConstraintProxy/GroundConstraintProxy_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [AttributeSets](../GroundConstraintProxy/GroundConstraintProxy_AttributeSets.md) | Property that returns the AttributeSets collection object associated with this object. |
| [ContainingOccurrence](../GroundConstraintProxy/GroundConstraintProxy_ContainingOccurrence.md) | Property that returns the ComponentOccurrence that the native object is being referenced through. The returned occurrence is the containing occurrence. |
| [Deletable](../GroundConstraintProxy/GroundConstraintProxy_Deletable.md) | Indicates whether this object is deletable. |
| [Entity](../GroundConstraintProxy/GroundConstraintProxy_Entity.md) | Property that returns entity being grounded. |
| [NativeObject](../GroundConstraintProxy/GroundConstraintProxy_NativeObject.md) | Gets the object in the context of the definition instead of the containing assembly. |
| [Parent](../GroundConstraintProxy/GroundConstraintProxy_Parent.md) | Property that returns the parent sketch of the object. |
| [Type](../GroundConstraintProxy/GroundConstraintProxy_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 6
