# ModelGeneralNoteProxy.InternalName Property

Parent Object: [ModelGeneralNoteProxy](../ModelGeneralNoteProxy/ModelGeneralNoteProxy.md)

## Description

Gets the internal name (GUID) of the model annotation.

## Syntax

ModelGeneralNoteProxy.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2022
