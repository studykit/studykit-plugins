# SplineFitPointsConstraint3DProxy.NativeObject Property

Parent Object: [SplineFitPointsConstraint3DProxy](../SplineFitPointsConstraint3DProxy/SplineFitPointsConstraint3DProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

SplineFitPointsConstraint3DProxy.**NativeObject**() As [SplineFitPointsConstraint3D](../SplineFitPointsConstraint3D/SplineFitPointsConstraint3D.md)

## Property Value

This is a read only property whose value is a [SplineFitPointsConstraint3D](../SplineFitPointsConstraint3D/SplineFitPointsConstraint3D.md).

## Version

Introduced in version 8
