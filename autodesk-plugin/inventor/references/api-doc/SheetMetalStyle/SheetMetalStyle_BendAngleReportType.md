# SheetMetalStyle.BendAngleReportType Property

Parent Object: [SheetMetalStyle](../SheetMetalStyle/SheetMetalStyle.md)

## Description

Gets and sets which angle is to be reported in the flat pattern.

## Syntax

SheetMetalStyle.**BendAngleReportType**() As [BendAngleTypeEnum](../BendAngleTypeEnum.md)

## Property Value

This is a read/write property whose value is a [BendAngleTypeEnum](../BendAngleTypeEnum.md).

## Version

Introduced in version 2010
