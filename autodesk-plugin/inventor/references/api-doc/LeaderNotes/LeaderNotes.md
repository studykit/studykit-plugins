# LeaderNotes Object

## Description

The LeaderNotes collection object provides access to all of the leader notes on the sheet.

## Methods

|  |  |
| --- | --- |
| Name | Description |
| [Add](../LeaderNotes/LeaderNotes_Add.md) | Method that creates a leader note. |

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../LeaderNotes/LeaderNotes_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [Count](../LeaderNotes/LeaderNotes_Count.md) | Property that returns the number of items in the collection. |
| [Item](../LeaderNotes/LeaderNotes_Item.md) | Returns the specified LeaderNote object from the collection. |
| [Type](../LeaderNotes/LeaderNotes_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Accessed From

[DrawingNotes.LeaderNotes](../DrawingNotes/DrawingNotes_LeaderNotes.md)

## Version

Introduced in version 10
