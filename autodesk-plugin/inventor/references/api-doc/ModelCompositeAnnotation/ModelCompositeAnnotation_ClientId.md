# ModelCompositeAnnotation.ClientId Property

Parent Object: [ModelCompositeAnnotation](../ModelCompositeAnnotation/ModelCompositeAnnotation.md)

## Description

Returns the string that uniquely identifies the client.

## Syntax

ModelCompositeAnnotation.**ClientId**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 2018
