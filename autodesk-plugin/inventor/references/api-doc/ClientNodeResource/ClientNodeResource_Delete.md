# ClientNodeResource.Delete Method

Parent Object: [ClientNodeResource](../ClientNodeResource/ClientNodeResource.md)

## Description

Method that gets rid of this object. If it is being used currently by some node definition object, the method fails.

## Syntax

ClientNodeResource.**Delete**()

## Version

Introduced in version 8
