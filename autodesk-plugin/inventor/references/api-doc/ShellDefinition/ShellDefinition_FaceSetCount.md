# ShellDefinition.FaceSetCount Property

Parent Object: [ShellDefinition](../ShellDefinition/ShellDefinition.md)

## Description

Property that specifies the number of face sets currently defined in the defintion.

## Syntax

ShellDefinition.**FaceSetCount**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 9
