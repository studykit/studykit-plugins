# WorkPlane.Dependents Property

Parent Object: [WorkPlane](../WorkPlane/WorkPlane.md)

## Description

Property that returns the collection of objects that have a direct dependency on the work plane.

## Syntax

WorkPlane.**Dependents**() As [ObjectCollection](../ObjectCollection/ObjectCollection.md)

## Property Value

This is a read only property whose value is an [ObjectCollection](../ObjectCollection/ObjectCollection.md).

## Version

Introduced in version 4
