# WorkPlane.Plane Property

Parent Object: [WorkPlane](../WorkPlane/WorkPlane.md)

## Description

Property that returns a Plane's geometry. The Plane object returned provides information about the position and normal of the work plane.

## Syntax

WorkPlane.**Plane**() As [Plane](../Plane/Plane.md)

## Property Value

This is a read only property whose value is a [Plane](../Plane/Plane.md).

## Version

Introduced in version 4
