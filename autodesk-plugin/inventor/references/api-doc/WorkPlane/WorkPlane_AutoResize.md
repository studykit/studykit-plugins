# WorkPlane.AutoResize Property

Parent Object: [WorkPlane](../WorkPlane/WorkPlane.md)

## Description

Gets and sets whether this work plane should be resized automatically based on the component size.

## Syntax

WorkPlane.**AutoResize**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 2008
