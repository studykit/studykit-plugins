# ImportedComponentProxy.HealthStatus Property

Parent Object: [ImportedComponentProxy](../ImportedComponentProxy/ImportedComponentProxy.md)

## Description

Gets the health status of this object.

## Syntax

ImportedComponentProxy.**HealthStatus**() As [HealthStatusEnum](../HealthStatusEnum.md)

## Property Value

This is a read only property whose value is a [HealthStatusEnum](../HealthStatusEnum.md).

## Version

Introduced in version 2016
