# ImportedComponentProxy.ReferencedDocumentDescriptor Property

Parent Object: [ImportedComponentProxy](../ImportedComponentProxy/ImportedComponentProxy.md)

## Description

Gets the the descriptor of the document that was used to create this Reference Component.

## Syntax

ImportedComponentProxy.**ReferencedDocumentDescriptor**() As [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md)

## Property Value

This is a read only property whose value is a [DocumentDescriptor](../DocumentDescriptor/DocumentDescriptor.md).

## Version

Introduced in version 2016
