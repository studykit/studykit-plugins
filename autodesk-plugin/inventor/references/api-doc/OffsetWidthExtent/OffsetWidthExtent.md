# OffsetWidthExtent Object

Derived from: [PartFeatureExtent](../PartFeatureExtent/PartFeatureExtent.md) Object

## Description

The OffsetWidthExtent is used by sheet metal features when the width of the feature is defined with respect to two entities.

## Properties

|  |  |
| --- | --- |
| Name | Description |
| [Application](../OffsetWidthExtent/OffsetWidthExtent_Application.md) | Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned. |
| [OffsetDistanceOne](../OffsetWidthExtent/OffsetWidthExtent_OffsetDistanceOne.md) | Property that returns the parameter controlling the distance of the feature from this offset entity. |
| [OffsetDistanceTwo](../OffsetWidthExtent/OffsetWidthExtent_OffsetDistanceTwo.md) | Property that returns the parameter controlling the distance of the feature from this offset entity. |
| [OffsetEntityOne](../OffsetWidthExtent/OffsetWidthExtent_OffsetEntityOne.md) | Property that gets and set the entity the width is measured relative to. |
| [OffsetEntityTwo](../OffsetWidthExtent/OffsetWidthExtent_OffsetEntityTwo.md) | Property that gets and set the entity the width is measured relative to. |
| [Parent](../OffsetWidthExtent/OffsetWidthExtent_Parent.md) | Property that returns the parent PartFeature of the definition. |
| [Type](../OffsetWidthExtent/OffsetWidthExtent_Type.md) | Returns an ObjectTypeEnum indicating this object's type. |

## Version

Introduced in version 2009
