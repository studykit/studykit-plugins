# AssemblyComponentDefinition.Parent Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Property that returns the parent of the object.

## Syntax

AssemblyComponentDefinition.**Parent**() As [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md)

## Property Value

This is a read only property whose value is an [AssemblyDocument](../AssemblyDocument/AssemblyDocument.md).

## Version

Introduced in version 4
