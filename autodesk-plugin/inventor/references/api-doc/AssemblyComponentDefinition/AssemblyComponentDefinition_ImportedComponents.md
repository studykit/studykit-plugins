# AssemblyComponentDefinition.ImportedComponents Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Read-only property that returns the ImportedComponents collection object.

## Syntax

AssemblyComponentDefinition.**ImportedComponents**() As [ImportedComponents](../ImportedComponents/ImportedComponents.md)

## Property Value

This is a read only property whose value is an [ImportedComponents](../ImportedComponents/ImportedComponents.md).

## Version

Introduced in version 2016
