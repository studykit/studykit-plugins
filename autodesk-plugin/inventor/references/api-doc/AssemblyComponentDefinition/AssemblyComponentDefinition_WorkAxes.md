# AssemblyComponentDefinition.WorkAxes Property

Parent Object: [AssemblyComponentDefinition](../AssemblyComponentDefinition/AssemblyComponentDefinition.md)

## Description

Property that returns the WorkAxes collection object that encapsulates all of the work axes defined in this ComponentDefinition.

## Syntax

AssemblyComponentDefinition.**WorkAxes**() As [WorkAxes](../WorkAxes/WorkAxes.md)

## Property Value

This is a read only property whose value is a [WorkAxes](../WorkAxes/WorkAxes.md).

## Version

Introduced in version 6
