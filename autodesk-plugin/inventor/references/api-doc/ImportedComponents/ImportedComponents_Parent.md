# ImportedComponents.Parent Property

Parent Object: [ImportedComponents](../ImportedComponents/ImportedComponents.md)

## Description

Gets the parent object from whom this object can logically be reached.

## Syntax

ImportedComponents.**Parent**() As [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md)

## Property Value

This is a read only property whose value is a [ComponentDefinition](../ComponentDefinition/ComponentDefinition.md).

## Version

Introduced in version 2016
