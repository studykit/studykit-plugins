# DrawingBOMColumns.Count Property

Parent Object: [DrawingBOMColumns](../DrawingBOMColumns/DrawingBOMColumns.md)

## Description

Property that returns the number of items in the collection.

## Syntax

DrawingBOMColumns.**Count**() As Long

## Property Value

This is a read only property whose value is a Long.

## Version

Introduced in version 2009
