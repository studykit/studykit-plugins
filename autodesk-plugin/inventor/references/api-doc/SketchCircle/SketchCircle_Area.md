# SketchCircle.Area Property

Parent Object: [SketchCircle](../SketchCircle/SketchCircle.md)

## Description

Double property that returns the area of the entity in square centimeters.

## Syntax

SketchCircle.**Area**() As Double

## Property Value

This is a read only property whose value is a Double.
