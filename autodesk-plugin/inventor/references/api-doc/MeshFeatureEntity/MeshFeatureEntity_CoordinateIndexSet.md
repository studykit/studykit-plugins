# MeshFeatureEntity.CoordinateIndexSet Property

Parent Object: [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Description

Read-only property that returns the coordinate index set of the mesh feature entity.

## Syntax

MeshFeatureEntity.**CoordinateIndexSet**() As [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md)

## Property Value

This is a read only property whose value is a [GraphicsIndexSet](../GraphicsIndexSet/GraphicsIndexSet.md).

## Version

Introduced in version 2017
