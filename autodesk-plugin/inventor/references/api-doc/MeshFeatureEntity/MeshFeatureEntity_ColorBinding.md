# MeshFeatureEntity.ColorBinding Property

Parent Object: [MeshFeatureEntity](../MeshFeatureEntity/MeshFeatureEntity.md)

## Description

Read-only property that returns the color binding of the mesh feature entity.

## Syntax

MeshFeatureEntity.**ColorBinding**() As [ColorBindingEnum](../ColorBindingEnum.md)

## Property Value

This is a read only property whose value is a [ColorBindingEnum](../ColorBindingEnum.md).

## Version

Introduced in version 2017
