# ManipulatorEvents.SelectedElement Property

Parent Object: [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md)

## Description

Read-write property that gets and sets the selected manipulator element. If the kAllManipulatorElements is set that no element will be selected.

## Syntax

ManipulatorEvents.**SelectedElement**() As [ManipulatorElementEnum](../ManipulatorElementEnum.md)

## Property Value

This is a read/write property whose value is a [ManipulatorElementEnum](../ManipulatorElementEnum.md).

## Version

Introduced in version 2017
