# ManipulatorEvents.Transform Property

Parent Object: [ManipulatorEvents](../ManipulatorEvents/ManipulatorEvents.md)

## Description

Gets and sets the current position and orientation of the manipulator in model space.

## Syntax

ManipulatorEvents.**Transform**() As [Matrix](../Matrix/Matrix.md)

## Property Value

This is a read/write property whose value is a [Matrix](../Matrix/Matrix.md).

## Version

Introduced in version 2017
