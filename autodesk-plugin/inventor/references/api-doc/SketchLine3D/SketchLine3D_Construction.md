# SketchLine3D.Construction Property

Parent Object: [SketchLine3D](../SketchLine3D/SketchLine3D.md)

## Description

Gets / sets the Sketch3D Construction Property.

## Syntax

SketchLine3D.**Construction**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
