# CustomTables.Type Property

Parent Object: [CustomTables](../CustomTables/CustomTables.md)

## Description

Returns an ObjectTypeEnum indicating this object's type.

## Syntax

CustomTables.**Type**() As [ObjectTypeEnum](../ObjectTypeEnum.md)

## Property Value

This is a read only property whose value is an [ObjectTypeEnum](../ObjectTypeEnum.md).

## Version

Introduced in version 9
