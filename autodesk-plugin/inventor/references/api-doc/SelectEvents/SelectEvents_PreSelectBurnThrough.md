# SelectEvents.PreSelectBurnThrough Property

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Gets/Sets the flag indicating whether the preselected items will burn through.

## Syntax

SelectEvents.**PreSelectBurnThrough**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 7
