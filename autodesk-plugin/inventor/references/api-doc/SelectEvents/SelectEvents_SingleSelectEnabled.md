# SelectEvents.SingleSelectEnabled Property

Parent Object: [SelectEvents](../SelectEvents/SelectEvents.md)

## Description

Gets/Sets the flag indicating whether single select is enabled.

## Syntax

SelectEvents.**SingleSelectEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 6
