# FixedWorkPointDef.Point Property

Parent Object: [FixedWorkPointDef](../FixedWorkPointDef/FixedWorkPointDef.md)

## Description

Specifies the origin point of the work point.

## Syntax

FixedWorkPointDef.**Point**() As [Point](../Point/Point.md)

## Property Value

This is a read/write property whose value is a [Point](../Point/Point.md).

## Version

Introduced in version 4
