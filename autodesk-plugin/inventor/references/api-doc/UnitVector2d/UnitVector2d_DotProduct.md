# UnitVector2d.DotProduct Method

Parent Object: [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Description

Determine the dot product of this unit vector2d to the specified unit vector2d.

## Syntax

UnitVector2d.**DotProduct**( ***Vector*** As [UnitVector2d](../UnitVector2d/UnitVector2d.md) ) As Double

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [UnitVector2d](../UnitVector2d/UnitVector2d.md) | 2nd vector to use to calculate dot product. |

## Version

Introduced in version 11
