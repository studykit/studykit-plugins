# UnitVector2d.IsPerpendicularTo Method

Parent Object: [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Description

Determine if this unit vector2d is perpendicular to the specified unit vector2d.

## Syntax

UnitVector2d.**IsPerpendicularTo**( ***Vector*** As [UnitVector2d](../UnitVector2d/UnitVector2d.md), [***Tolerance***] As Double ) As Boolean

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Vector | [UnitVector2d](../UnitVector2d/UnitVector2d.md) | Vector with which to test perpendicular status. |
| Tolerance | Double | Tolerance for test. |

## Version

Introduced in version 11
