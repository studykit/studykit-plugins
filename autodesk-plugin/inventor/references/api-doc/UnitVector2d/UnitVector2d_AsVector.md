# UnitVector2d.AsVector Method

Parent Object: [UnitVector2d](../UnitVector2d/UnitVector2d.md)

## Description

Get the vector2d equivalent of this unit vector2d.

## Syntax

UnitVector2d.**AsVector**() As [Vector2d](../Vector2d/Vector2d.md)

## Version

Introduced in version 11
