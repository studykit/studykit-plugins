# Balloon.Style Property

Parent Object: [Balloon](../Balloon/Balloon.md)

## Description

Gets and sets the associated BalloonStyle object.

## Syntax

Balloon.**Style**() As [BalloonStyle](../BalloonStyle/BalloonStyle.md)

## Property Value

This is a read/write property whose value is a [BalloonStyle](../BalloonStyle/BalloonStyle.md).

## Version

Introduced in version 2009
