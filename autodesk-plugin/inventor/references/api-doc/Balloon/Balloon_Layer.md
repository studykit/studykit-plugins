# Balloon.Layer Property

Parent Object: [Balloon](../Balloon/Balloon.md)

## Description

Gets and sets the layer applied to this balloons.

## Syntax

Balloon.**Layer**() As [Layer](../Layer/Layer.md)

## Property Value

This is a read/write property whose value is a [Layer](../Layer/Layer.md).

## Version

Introduced in version 10
