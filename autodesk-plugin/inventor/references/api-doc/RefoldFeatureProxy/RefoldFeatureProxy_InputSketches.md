# RefoldFeatureProxy.InputSketches Property

Parent Object: [RefoldFeatureProxy](../RefoldFeatureProxy/RefoldFeatureProxy.md)

## Description

Property that returns the sketches that were specifies as \input when creating this feature.

## Syntax

RefoldFeatureProxy.**InputSketches**() As [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchesEnumerator](../SketchesEnumerator/SketchesEnumerator.md).

## Version

Introduced in version 2010
