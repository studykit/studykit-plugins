# MouseEvents.PointInferenceEnabled Property

Parent Object: [MouseEvents](../MouseEvents/MouseEvents.md)

## Description

Gets/Sets the Boolean flag indicating that Inventor should turn its special point inferencing on.

## Syntax

MouseEvents.**PointInferenceEnabled**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 5
