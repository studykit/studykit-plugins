# MouseEvents.PointInferences Property

Parent Object: [MouseEvents](../MouseEvents/MouseEvents.md)

## Description

Gets the inferencing object.

## Syntax

MouseEvents.**PointInferences**() As [PointInferenceEnumerator](../PointInferenceEnumerator/PointInferenceEnumerator.md)

## Property Value

This is a read only property whose value is a [PointInferenceEnumerator](../PointInferenceEnumerator/PointInferenceEnumerator.md).

## Version

Introduced in version 5
