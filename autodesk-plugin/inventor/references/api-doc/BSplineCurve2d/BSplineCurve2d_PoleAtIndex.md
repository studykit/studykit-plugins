# BSplineCurve2d.PoleAtIndex Property

Parent Object: [BSplineCurve2d](../BSplineCurve2d/BSplineCurve2d.md)

## Description

Indicates the pole coordinate point at the specified index into the spline's pole vector.

## Syntax

BSplineCurve2d.**PoleAtIndex**( ***Index*** As Long ) As [Point2d](../Point2d/Point2d.md)

## Property Value

This is a read/write property whose value is a [Point2d](../Point2d/Point2d.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Long |  |

## Version

Introduced in version 4
