# SketchPoint3D.HoleCenter Property

Parent Object: [SketchPoint3D](../SketchPoint3D/SketchPoint3D.md)

## Description

Defines if the sketch point is being displayed as a hole center.

## Syntax

SketchPoint3D.**HoleCenter**() As Boolean

## Property Value

This is a read/write property whose value is a Boolean.

## Version

Introduced in version 11
