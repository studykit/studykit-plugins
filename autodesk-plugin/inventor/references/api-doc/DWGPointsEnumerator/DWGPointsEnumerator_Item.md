# DWGPointsEnumerator.Item Property

Parent Object: [DWGPointsEnumerator](../DWGPointsEnumerator/DWGPointsEnumerator.md)

## Description

Allows VARIANT-indexed access to items in the collection. You can use names as indexes as well.

## Syntax

DWGPointsEnumerator.**Item**( ***Index*** As Variant ) As [DWGPoint](../DWGPoint/DWGPoint.md)

## Property Value

This is a read only property whose value is a [DWGPoint](../DWGPoint/DWGPoint.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Long value that specifies the index of the object to return. |

## Version

Introduced in version 2016
