# ChangeDefinition.InternalName Property

Parent Object: [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md)

## Description

Gets the internal name of the ChangeDefinition. This is a unique string that may be used to reliably retrieve a ChangeDefinition object.

## Syntax

ChangeDefinition.**InternalName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 9
