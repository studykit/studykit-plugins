# ChangeDefinition.CommandName Property

Parent Object: [ChangeDefinition](../ChangeDefinition/ChangeDefinition.md)

## Description

Gets the command name associated with this ChangeDefinition. This localized string shows up in the undo list.

## Syntax

ChangeDefinition.**CommandName**() As String

## Property Value

This is a read only property whose value is a String.

## Version

Introduced in version 9
