# WebBrowserDialogs.Item Property

Parent Object: [WebBrowserDialogs](../WebBrowserDialogs/WebBrowserDialogs.md)

## Description

Allows integer-indexed access to items in the collection.

## Syntax

WebBrowserDialogs.**Item**( ***Index*** As Variant ) As [WebBrowserDialog](../WebBrowserDialog/WebBrowserDialog.md)

## Property Value

This is a read only property whose value is a [WebBrowserDialog](../WebBrowserDialog/WebBrowserDialog.md).

## Parameters

|  |  |  |
| --- | --- | --- |
| Name | Type | Description |
| Index | Variant | Input Variant value that specifies the WebBrowserDialog to return. This can be either a numeric value indicating the index of the item in the collection or it can be a string indicating the WebBrowserDialog’s internal name. If an out of range index or a name of a non-existent WebBrowserDialog is provided, an error occurs. |

## Version

Introduced in version 2017
