# RotateAboutLineMoveOperation.AxisEntity Property

Parent Object: [RotateAboutLineMoveOperation](../RotateAboutLineMoveOperation/RotateAboutLineMoveOperation.md)

## Description

Read-write property that gets and sets the entity used to define the axis or rotation.

## Syntax

RotateAboutLineMoveOperation.**AxisEntity**() As Object

## Property Value

This is a read/write property whose value is an Object.

## Version

Introduced in version 2013
