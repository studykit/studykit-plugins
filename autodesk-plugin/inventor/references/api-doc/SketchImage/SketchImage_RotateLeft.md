# SketchImage.RotateLeft Method

Parent Object: [SketchImage](../SketchImage/SketchImage.md)

## Description

Method that rotates the image counter-clockwise by 90 degrees.

## Syntax

SketchImage.**RotateLeft**()

## Version

Introduced in version 11
