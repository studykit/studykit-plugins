# SketchImage.MirrorVertical Method

Parent Object: [SketchImage](../SketchImage/SketchImage.md)

## Description

Method that mirrors the image vertically.

## Syntax

SketchImage.**MirrorVertical**()

## Version

Introduced in version 11
