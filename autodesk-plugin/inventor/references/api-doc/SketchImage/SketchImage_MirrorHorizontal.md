# SketchImage.MirrorHorizontal Method

Parent Object: [SketchImage](../SketchImage/SketchImage.md)

## Description

Method that mirrors the image horizontally.

## Syntax

SketchImage.**MirrorHorizontal**()

## Version

Introduced in version 11
