# SketchImage.Parent Property

Parent Object: [SketchImage](../SketchImage/SketchImage.md)

## Description

Property that returns the parent sketch.

## Syntax

SketchImage.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 11
