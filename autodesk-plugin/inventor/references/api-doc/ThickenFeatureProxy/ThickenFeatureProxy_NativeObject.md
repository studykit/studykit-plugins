# ThickenFeatureProxy.NativeObject Property

Parent Object: [ThickenFeatureProxy](../ThickenFeatureProxy/ThickenFeatureProxy.md)

## Description

Gets the object in the context of the definition instead of the containing assembly.

## Syntax

ThickenFeatureProxy.**NativeObject**() As [ThickenFeature](../ThickenFeature/ThickenFeature.md)

## Property Value

This is a read only property whose value is a [ThickenFeature](../ThickenFeature/ThickenFeature.md).

## Version

Introduced in version 9
