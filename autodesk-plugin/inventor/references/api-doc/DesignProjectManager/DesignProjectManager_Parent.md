# DesignProjectManager.Parent Property

Parent Object: [DesignProjectManager](../DesignProjectManager/DesignProjectManager.md)

## Description

Property that returns the parent Application or ApprenticeServerComponent object.

## Syntax

DesignProjectManager.**Parent**() As Object

## Property Value

This is a read only property whose value is an Object.

## Version

Introduced in version 2011
