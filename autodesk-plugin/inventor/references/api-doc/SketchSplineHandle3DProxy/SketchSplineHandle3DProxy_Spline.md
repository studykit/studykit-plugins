# SketchSplineHandle3DProxy.Spline Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Property that returns the spline associated with the handle.

## Syntax

SketchSplineHandle3DProxy.**Spline**() As [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md)

## Property Value

This is a read only property whose value is a [SketchSpline3D](../SketchSpline3D/SketchSpline3D.md).

## Version

Introduced in version 2008
