# SketchSplineHandle3DProxy.Constraints3D Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Property that returns a collection of sketch constraints that are tied directly to this entity. This collection consists of geometric and bend constraints.

## Syntax

SketchSplineHandle3DProxy.**Constraints3D**() As [SketchConstraints3DEnumerator](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator.md)

## Property Value

This is a read only property whose value is a [SketchConstraints3DEnumerator](../SketchConstraints3DEnumerator/SketchConstraints3DEnumerator.md).

## Version

Introduced in version 2008
