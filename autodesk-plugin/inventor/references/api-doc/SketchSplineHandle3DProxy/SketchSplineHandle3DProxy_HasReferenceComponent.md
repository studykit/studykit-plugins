# SketchSplineHandle3DProxy.HasReferenceComponent Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Property that specifies if the object was created as the result of a derived part.

## Syntax

SketchSplineHandle3DProxy.**HasReferenceComponent**() As Boolean

## Property Value

This is a read only property whose value is a Boolean.

## Version

Introduced in version 2008
