# SketchSplineHandle3DProxy.Geometry Property

Parent Object: [SketchSplineHandle3DProxy](../SketchSplineHandle3DProxy/SketchSplineHandle3DProxy.md)

## Description

Gets and sets a LineSegment geometry object. The object returned represents a 'snapshot' view of the current state of the sketch line.

## Syntax

SketchSplineHandle3DProxy.**Geometry**() As [LineSegment](../LineSegment/LineSegment.md)

## Property Value

This is a read/write property whose value is a [LineSegment](../LineSegment/LineSegment.md).

## Version

Introduced in version 2008
