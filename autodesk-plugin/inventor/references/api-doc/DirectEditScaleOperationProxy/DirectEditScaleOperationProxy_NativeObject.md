# DirectEditScaleOperationProxy.NativeObject Property

Parent Object: [DirectEditScaleOperationProxy](../DirectEditScaleOperationProxy/DirectEditScaleOperationProxy.md)

## Description

Get the source object in the context of the definition instead of the containing assembly.

## Syntax

DirectEditScaleOperationProxy.**NativeObject**() As [DirectEditScaleOperation](../DirectEditScaleOperation/DirectEditScaleOperation.md)

## Property Value

This is a read only property whose value is a [DirectEditScaleOperation](../DirectEditScaleOperation/DirectEditScaleOperation.md).

## Version

Introduced in version 2016
