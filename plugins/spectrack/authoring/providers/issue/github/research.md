## Scope

Use this binding for the GitHub issue side of `research`. The final body structure is listed below.

## Final body structure

Use this final GitHub body structure for `research` issues.

Common required sections are defined by `../../../contracts/issue/research.md`:

- `## Description`
- `## Research Question`
- `## Mode`
- `## Options`

Common optional sections are defined by `../../../contracts/issue/research.md` and `../../../contracts/issue/body.md`:

- `## Scope`
- `## Sources To Check`
- `## Resume`

GitHub label is automatically set to `research`.
