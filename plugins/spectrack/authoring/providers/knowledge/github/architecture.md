# GitHub Knowledge Architecture Authoring

Provider-specific binding for `architecture` knowledge pages stored as repository Markdown files.

Read after:

- `../../../contracts/knowledge/architecture.md`
- `./convention.md`

## Final body structure

Common required sections are defined by `../../../contracts/knowledge/architecture.md`:

- `## Overview`
- `## Components`
- `## Technology Stack`
- `## Test Strategy`

Common optional sections are defined by `../../../contracts/knowledge/architecture.md` and `../../../contracts/knowledge/body.md`:

- `## Component Diagram`
- `## External Dependencies`
- `## Related Work`
- `## Change Log`

Diagrams may be Mermaid, ASCII, or linked files in the repository.
