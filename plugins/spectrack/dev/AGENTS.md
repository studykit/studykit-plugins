# SpecTrack Contributor Docs

This AGENTS.md file is for SpecTrack plugin contributors. It is not runtime
guidance for LLMs using an installed SpecTrack plugin.

This directory collects contributor-facing reference documents for maintaining
the SpecTrack plugin. These files describe implementation boundaries, testing
principles, injected-context architecture, and local cache/projection behavior.

Do not treat files in this directory as user-facing plugin documentation or as
runtime instructions for installed-plugin users.
