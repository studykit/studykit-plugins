# SpecTrack Tests

This AGENTS.md file is for SpecTrack plugin contributors. It is not runtime
guidance for LLMs using an installed SpecTrack plugin.

Before adding or changing SpecTrack tests, read `../dev/testing-principles.md`.

Keep tests focused on executable behavior and stable layer boundaries. Do not
freeze prose wording, heading preferences, or authoring judgment unless the
wording is intentionally part of a contract.
